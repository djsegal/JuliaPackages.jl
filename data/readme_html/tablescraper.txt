<p dir="auto">In this package there is only one function</p>
<div class="snippet-clipboard-content notranslate position-relative overflow-auto" data-snippet-clipboard-copy-content="scrape_tables(url)"><pre class="notranslate"><code>scrape_tables(url)
</code></pre></div>
<p dir="auto">which lets you scrape for tables wrapped in <code>&lt;table&gt;</code> tags and return them in a vector of <a href="https://github.com/JuliaData/Tables.jl">Tables.jl</a> compatible row-tables.</p>
<p dir="auto">By default the function uses <code>Cascadia.nodeText</code> to extract the text from each <code>&lt;td&gt;</code> node.</p>
<p dir="auto">However, if you wish to extract more than the text node you may want to use</p>
<div class="snippet-clipboard-content notranslate position-relative overflow-auto" data-snippet-clipboard-copy-content="scrape_tables(url, identity)"><pre class="notranslate"><code>scrape_tables(url, identity)
</code></pre></div>
<p dir="auto">to keep the cells as <code>Gumbo.HTMLNode</code>s and do more advanced extraction.</p>
<p dir="auto">Also, you can put any callable into the <code>cell_transform</code> argument to do custom transformation of the <code>&lt;td&gt;</code> nodes before returning.</p>
<p dir="auto">E.g.</p>
<div class="snippet-clipboard-content notranslate position-relative overflow-auto" data-snippet-clipboard-copy-content="scrape_tables(url, cell_transform)"><pre class="notranslate"><code>scrape_tables(url, cell_transform)
</code></pre></div>
