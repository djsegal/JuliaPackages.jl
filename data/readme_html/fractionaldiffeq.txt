<p dir="auto">FractionalDiffEq.jl provides FDE solvers to <a href="https://diffeq.sciml.ai/dev/" rel="nofollow">DifferentialEquations.jl</a> ecosystem, including FODE(Fractional Ordianry Differential Equations), FDDE(Fractional Delay Differential Equations) and many more. There are many performant solvers available, capable of solving many kinds of fractional differential equations.</p>

<p dir="auto">FractionalDiffEq.jl is a powerful tool to solve system of fractional differential equations, if you are familiar with <a href="https://github.com/SciML/DifferentialEquations.jl">DifferentialEquations.jl</a>, it would be just like out of the box.</p>
<p dir="auto">Let's see if we have a Chua chaos system:</p>
<p dir="auto"><math-renderer class="js-display-math" style="display: block" data-static-url="https://github.githubassets.com/static" data-run-id="197854de3dea7253a977c668ea0f4a18">$$ \begin{cases}D^{\alpha_1}x=10.725[y-1.7802x-[0.1927(|x+1|-|x-1|)]\\
D^{\alpha_2}y=x-y+z\\
D^{\alpha_3}z=-10.593y-0.268z\end{cases} $$</math-renderer></p>
<p dir="auto">By using the <code>NonLinearAlg</code> algorithms to solve this problem:</p>
<div class="highlight highlight-source-julia notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="using FractionalDiffEq, Plots
function chua!(du, x, p, t)
    a, b, c, m0, m1 = p
    du[1] = a*(x[2]-x[1]-(m1*x[1]+0.5*(m0-m1)*(abs(x[1]+1)-abs(x[1]-1))))
    du[2] = x[1]-x[2]+x[3]
    du[3] = -b*x[2]-c*x[3]
end

p = [10.725, 10.593, 0.268, -1.1726, -0.7872]

sol = solve(prob, h, NonLinearAlg())
plot(sol, vars=(1, 2), title=&quot;Chua System&quot;, legend=:bottomright)"><pre><span class="pl-k">using</span> FractionalDiffEq, Plots
