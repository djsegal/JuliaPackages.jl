<p dir="auto">A library of functions for polynomial bases used in spectral element methods using the quadrature rules from
<a href="https://github.com/ajt60gaibb/FastGaussQuadrature.jl">FastGaussQuadrature.jl</a> for <code>Float64</code> and root finding
via the Newton algorithm for other scalar types (such as <code>BigFloat</code>). The algorithms for interpolation and
differentiation use barycentric weights as described in the book "Implementing Spectral Methods for PDEs"
by David Kopriva. If <a href="https://github.com/JuliaPy/SymPy.jl">SymPy.jl</a>/<a href="https://github.com/symengine/symengine">SymEngine.jl</a>
is loaded, symbolic computations using <code>SymPy.Sym</code>/<code>SymEngine.Basic</code> are supported.</p>
<p dir="auto">A brief tutorial is given as
<a href="http://nbviewer.ipython.org/github/ranocha/PolynomialBases.jl/blob/main/notebooks/Tutorial.ipynb" rel="nofollow">notebook</a>.</p>
