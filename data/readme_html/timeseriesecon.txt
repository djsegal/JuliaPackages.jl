<p>A Julia package for working with discrete time-series data.</p>
<p><code>TimeSeriesEcon</code> package is part of the
<a href="https://github.com/bankofcanada/StateSpaceEcon.jl">StateSpaceEcon</a> ecosystem,
which also includes
<a href="https://github.com/bankofcanada/ModelBaseEcon.jl">ModelBaseEcon</a></p>
