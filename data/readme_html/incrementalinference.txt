<p dir="auto">Optimization routines for incremental non-parametric and parametric solutions based on factor graphs and the Bayes (Junction) tree implemented in the <a href="http://www.julialang.org/" rel="nofollow">Julia language</a> (and <a href="http://www.juliacomputing.com" rel="nofollow">JuliaPro</a>).</p>
<h1 dir="auto"><a id="user-content-introduction" class="anchor" aria-hidden="true" href="#introduction"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path d="m7.775 3.275 1.25-1.25a3.5 3.5 0 1 1 4.95 4.95l-2.5 2.5a3.5 3.5 0 0 1-4.95 0 .751.751 0 0 1 .018-1.042.751.751 0 0 1 1.042-.018 1.998 1.998 0 0 0 2.83 0l2.5-2.5a2.002 2.002 0 0 0-2.83-2.83l-1.25 1.25a.751.751 0 0 1-1.042-.018.751.751 0 0 1-.018-1.042Zm-4.69 9.64a1.998 1.998 0 0 0 2.83 0l1.25-1.25a.751.751 0 0 1 1.042.018.751.751 0 0 1 .018 1.042l-1.25 1.25a3.5 3.5 0 1 1-4.95-4.95l2.5-2.5a3.5 3.5 0 0 1 4.95 0 .751.751 0 0 1-.018 1.042.751.751 0 0 1-1.042.018 1.998 1.998 0 0 0-2.83 0l-2.5 2.5a1.998 1.998 0 0 0 0 2.83Z"></path></svg></a>Introduction</h1>
<p dir="auto">This package implements a few different non-Gaussian factor graph inference algorithms, primarily</p>
<ul dir="auto">
<li>Multi-Modal iSAM (MM-iSAM) (<a href="http://www.juliarobotics.org/Caesar.jl/latest/refs/literature/" rel="nofollow">see references</a>) which does hybrid non-parametric and parametric inference/state-estimation over large factor graphs.</li>
<li>Batch Parametric (akin to conventional "non-linear least squares"),</li>
<li>Max-mixtures parametric,</li>
<li>Other multiparametric and non-Gaussian algorithms are in the works and will be announced in due course.</li>
</ul>
<p dir="auto">Fundamentally, inference is performed via the Bayes (junction) tree where Chapman-Kolmogorov transit integral solutions are based on marginal-joint belief estimation (a sum-product / belief-propagation approximation algorithm).  Many benefits such as clique recycling are also available.  See the common Caesar.jl documenation for more details.  <a href="http://juliarobotics.github.io/Caesar.jl/latest/" rel="nofollow"><img src="https://camo.githubusercontent.com/5cd28800ffafbf6aad3e61c1811ea12811d856e254265e5552b2a309b0d54554/68747470733a2f2f696d672e736869656c64732e696f2f62616467652f436165736172446f63732d6c61746573742d626c75652e737667" alt="caesar-docs" data-canonical-src="https://img.shields.io/badge/CaesarDocs-latest-blue.svg" style="max-width: 100%;"></a></p>
<p dir="auto">This package forms a cardinal piece of the <a href="https://github.com/JuliaRobotics/Caesar.jl">Caesar.jl</a> robotics toolkit, including 3D visualization and database interaction, which can serve as a base station for a robotic platform. A standalone Robot Motion Estimate (<a href="https://github.com/JuliaRobotics/RoME.jl">RoME.jl</a>) package extends the available variables, factors, and utilities for use in robotic navigation.  <a href="https://juliahub.com/ui/Packages/IncrementalInference/NrVw2??page=2" rel="nofollow"><img src="https://camo.githubusercontent.com/e660e1ac246dcd1ac1908700448e841d425f9abe44b728b43a4085ccf6b2889e/68747470733a2f2f6a756c69616875622e636f6d2f646f63732f496e6372656d656e74616c496e666572656e63652f646570732e737667" alt="iif-deps-img" data-canonical-src="https://juliahub.com/docs/IncrementalInference/deps.svg" style="max-width: 100%;"></a></p>
<p dir="auto">Note, that IncrementalInference.jl <strong>does not</strong> have to be used with RoME.jl / Caesar.jl -- IncrementalInference.jl only implements the algebraic inference operations against mathematical abstractions such as Manifolds.jl.</p>
<p dir="auto">Furthermore, please contact <a href="mailto:info@navability.io">info@navability.io</a> for more formal support on this package, <a href="https://www.navability.io" rel="nofollow">NavAbility.io</a>.</p>
