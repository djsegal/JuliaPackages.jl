<div id="readme" class="md" data-path="README.md"><article class="markdown-body entry-content container-lg" itemprop="text"><h1><a id="user-content-f16model" class="anchor" aria-hidden="true" href="#f16model"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path fill-rule="evenodd" d="M7.775 3.275a.75.75 0 001.06 1.06l1.25-1.25a2 2 0 112.83 2.83l-2.5 2.5a2 2 0 01-2.83 0 .75.75 0 00-1.06 1.06 3.5 3.5 0 004.95 0l2.5-2.5a3.5 3.5 0 00-4.95-4.95l-1.25 1.25zm-4.69 9.64a2 2 0 010-2.83l2.5-2.5a2 2 0 012.83 0 .75.75 0 001.06-1.06 3.5 3.5 0 00-4.95 0l-2.5 2.5a3.5 3.5 0 004.95 4.95l1.25-1.25a.75.75 0 00-1.06-1.06l-1.25 1.25a2 2 0 01-2.83 0z"></path></svg></a>F16Model</h1>
<p>This is a Julia package for a nonlinear model of the F16 aircraft. The aerodynamics included in this model come from the NASA Technical Report 1538, <em>Simulator Study of Stall/Post-Stall Characteristics of a Fighter Airplane with Relaxed Longitudinal Static Stability</em>, by Nguyen, Ogburn, Gilbert, Kibler, Brown, and Deal, Dec 1979. The model is based on <em>Aircraft Control and Simulations</em>, by Brian Stevens and Frank Lewis, Wiley Inter-Science, New York, 1992.</p>
<p>This Julia package is inspired by the <a href="https://dept.aem.umn.edu/~balas/darpa_sec/SEC.Software.html" rel="nofollow">MATLAB/Simulink package</a>, and currently has the following features:</p>
<ol>
<li>Nonlinear dynamics for simulations.</li>
<li>Trim routine for various flight conditions.</li>
<li>Linearization about a trim point (x0,u0) using ForwardDiff.</li>
</ol>
<p>More features will be added as we continue to develop this package.</p>
<h2><a id="user-content-model-details" class="anchor" aria-hidden="true" href="#model-details"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path fill-rule="evenodd" d="M7.775 3.275a.75.75 0 001.06 1.06l1.25-1.25a2 2 0 112.83 2.83l-2.5 2.5a2 2 0 01-2.83 0 .75.75 0 00-1.06 1.06 3.5 3.5 0 004.95 0l2.5-2.5a3.5 3.5 0 00-4.95-4.95l-1.25 1.25zm-4.69 9.64a2 2 0 010-2.83l2.5-2.5a2 2 0 012.83 0 .75.75 0 001.06-1.06 3.5 3.5 0 00-4.95 0l-2.5 2.5a3.5 3.5 0 004.95 4.95l1.25-1.25a.75.75 0 00-1.06-1.06l-1.25 1.25a2 2 0 01-2.83 0z"></path></svg></a>Model Details</h2>
<p>Detailed information is available <a href="https://dept.aem.umn.edu/~balas/darpa_sec/software/F16Manual.pdf" rel="nofollow">here</a>.</p>
<p>The 12 states of the system are as follows:</p>
<ol>
<li>N: North position in ft</li>
<li>E: East position in ft</li>
<li>h: Altitude in ft, min: 5000 ft, max: 40000 ft</li>
<li>phi: Roll angle in rad</li>
<li>theta: Pitch angle in rad</li>
<li>psi: Yaw angle in rad</li>
<li>Vt: Magnitude of total velocity in ft/s, min: 300 ft/s, max: 900 ft/s</li>
<li>alpha: Angle of attack in rad, min: -20 deg, max: 45 deg</li>
<li>beta: Side slip angle in rad, min: -30 deg, max: 30 deg</li>
<li>p: Roll rate in rad/s</li>
<li>q: Pitch rate in rad/s</li>
<li>r: Yaw rate in rad/s</li>
</ol>
<p>The 5 control variables are:</p>
<ol>
<li>T: Thrust in lbs, min: 1000, max: 19000</li>
<li>dele: Elevator angle in deg, min:-25, max: 25</li>
<li>dail: Aileron angle in deg, min:-21.5, max: 21.5</li>
<li>drud: Rudder angle in deg, min: -30, max: 30</li>
<li>dlef: Leading edge flap in deg, min: 0, max: 25</li>
</ol>
<p>Actuator models are defined as:</p>
<ol>
<li>T: max |rate|: 10,000 lbs/s</li>
<li>dele: max |rate|: 60 deg/s</li>
<li>dail: max |rate|: 80 deg/s</li>
<li>drud: max |rate|: 120 deg/s</li>
<li>dlef: max |rate|: 25 deg/s</li>
</ol>
<p>The nonlinear model of the aircraft does not include actuator dynamics.
The actuator dynamics need to be modeled as LTI systems and added to the system.
For example, for dele the low pass filter 1/(s/60+1) would model the actuator dynamics.</p>
