<p dir="auto">A simple package to merge PDF (Portable Document Format) files on
Linux, OS X and Windows. The following functions are exported:</p>
<ul dir="auto">
<li><code>merge_pdfs</code></li>
<li><code>append_pdf!</code></li>
<li><code>split_pdf</code></li>
</ul>

<p dir="auto">Appending with <code>append_pdf!</code> is particularly useful to create a single PDF
that contains multiple plots on separate pages:</p>
<div class="highlight highlight-source-julia notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="using Plots

for i in 1:5
    p = plot(rand(33), title=&quot;$i&quot;);
    savefig(p, &quot;temp.pdf&quot;)
    append_pdf!(&quot;allplots.pdf&quot;, &quot;temp.pdf&quot;, cleanup=true)
end"><pre><span class="pl-k">using</span> Plots

<span class="pl-k">for</span> i <span class="pl-k">in</span> <span class="pl-c1">1</span><span class="pl-k">:</span><span class="pl-c1">5</span>
    p <span class="pl-k">=</span> <span class="pl-c1">plot</span>(<span class="pl-c1">rand</span>(<span class="pl-c1">33</span>), title<span class="pl-k">=</span><span class="pl-s"><span class="pl-pds">"</span><span class="pl-v">$i</span><span class="pl-pds">"</span></span>);
    <span class="pl-c1">savefig</span>(p, <span class="pl-s"><span class="pl-pds">"</span>temp.pdf<span class="pl-pds">"</span></span>)
    <span class="pl-c1">append_pdf!</span>(<span class="pl-s"><span class="pl-pds">"</span>allplots.pdf<span class="pl-pds">"</span></span>, <span class="pl-s"><span class="pl-pds">"</span>temp.pdf<span class="pl-pds">"</span></span>, cleanup<span class="pl-k">=</span><span class="pl-c1">true</span>)
<span class="pl-k">end</span></pre></div>
<p dir="auto">All five plots are contained in <code>allplots.pdf</code> and the temporary file is deleted.</p>
