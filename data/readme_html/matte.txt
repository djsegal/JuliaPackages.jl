<p dir="auto">Matte is a julia library for julia-powered dashboards, inspired by Google's material design.
Matte makes it easy to convert your existing julia code into a beautiful and powerful dashboard
for end users.</p>
<p dir="auto"><strong>Please note</strong>: Matte is still highly experimental.</p>

<p dir="auto">Matte comes with some example apps. Let's create and run one of the examples:</p>
<div class="snippet-clipboard-content notranslate position-relative overflow-auto" data-snippet-clipboard-copy-content="julia&gt; matte_example(&quot;hello_world&quot;, &quot;hello_world&quot;)"><pre class="notranslate"><code>julia&gt; matte_example("hello_world", "hello_world")
</code></pre></div>
<p dir="auto">Matte will create a new folder and julia project at your chosen path and create all the files
for the example. Matte will change your working directory and activate the project.</p>
<h3 dir="auto"><a id="user-content-running-your-new-app" class="anchor" aria-hidden="true" href="#running-your-new-app"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path d="m7.775 3.275 1.25-1.25a3.5 3.5 0 1 1 4.95 4.95l-2.5 2.5a3.5 3.5 0 0 1-4.95 0 .751.751 0 0 1 .018-1.042.751.751 0 0 1 1.042-.018 1.998 1.998 0 0 0 2.83 0l2.5-2.5a2.002 2.002 0 0 0-2.83-2.83l-1.25 1.25a.751.751 0 0 1-1.042-.018.751.751 0 0 1-.018-1.042Zm-4.69 9.64a1.998 1.998 0 0 0 2.83 0l1.25-1.25a.751.751 0 0 1 1.042.018.751.751 0 0 1 .018 1.042l-1.25 1.25a3.5 3.5 0 1 1-4.95-4.95l2.5-2.5a3.5 3.5 0 0 1 4.95 0 .751.751 0 0 1-.018 1.042.751.751 0 0 1-1.042.018 1.998 1.998 0 0 0-2.83 0l-2.5 2.5a1.998 1.998 0 0 0 0 2.83Z"></path></svg></a>Running your new app</h3>
<p dir="auto">Let's run the app.</p>
<p dir="auto">But before we do: a note about Revise. I <em>highly</em> recommend using Revise with Matte. This
lets you edit your app and see the changes immediately, without having to restart your app.
Revise is included by default when you create a new Matte app, so all we need to do is:</p>
<div class="snippet-clipboard-content notranslate position-relative overflow-auto" data-snippet-clipboard-copy-content="julia&gt; using Revise"><pre class="notranslate"><code>julia&gt; using Revise
</code></pre></div>
<p dir="auto">Back to our app.</p>
<p dir="auto">To run the app, we need to first load Matte, and then load our module MyApp:</p>
<div class="snippet-clipboard-content notranslate position-relative overflow-auto" data-snippet-clipboard-copy-content="julia&gt; using Matte
julia&gt; includet(&quot;app.jl&quot;)"><pre class="notranslate"><code>julia&gt; using Matte
julia&gt; includet("app.jl")
</code></pre></div>
<p dir="auto">(Note that I use <code>includet</code> from Revise, to track changes to the file. If you aren't using
Revise, just use <code>include</code>. But you should <em>definitely</em> use Revise!)</p>
<p dir="auto">All we need to do now is run our app. We do this using the Matte function <code>run_app</code>, which
takes the module that defines our app as argument:</p>
<div class="snippet-clipboard-content notranslate position-relative overflow-auto" data-snippet-clipboard-copy-content="julia&gt; run_app(HelloWorldApp)"><pre class="notranslate"><code>julia&gt; run_app(HelloWorldApp)
</code></pre></div>
<p dir="auto">Matte will take a little while to start. Your app is up and running once you see:</p>
<div class="snippet-clipboard-content notranslate position-relative overflow-auto" data-snippet-clipboard-copy-content="Web Server starting at http://127.0.0.1:8000"><pre class="notranslate"><code>Web Server starting at http://127.0.0.1:8000
</code></pre></div>
<p dir="auto">Navigate your favourite web browser to the address listed above and (after a few seconds)
viola! Your app is ready!</p>
