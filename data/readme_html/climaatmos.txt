<p dir="auto">ClimaAtmos.jl is the atmosphere components of the CliMA software stack. We strive for a user interface that makes ClimaAtmos.jl as friendly and intuitive to use as possible, allowing users to focus on the science.</p>

<p dir="auto">Recommended Julia: Stable release v1.8.5</p>
<p dir="auto">Download the <code>ClimaAtmos</code>
<a href="https://github.com/CliMA/ClimaAtmos.jl">source</a> with:</p>
<div class="snippet-clipboard-content notranslate position-relative overflow-auto" data-snippet-clipboard-copy-content="$ git clone https://github.com/CliMA/ClimaAtmos.jl.git"><pre class="notranslate"><code>$ git clone https://github.com/CliMA/ClimaAtmos.jl.git
</code></pre></div>
<p dir="auto">Now change into the <code>ClimaAtmos.jl</code> directory with</p>
<div class="snippet-clipboard-content notranslate position-relative overflow-auto" data-snippet-clipboard-copy-content="$ cd ClimaAtmos.jl"><pre class="notranslate"><code>$ cd ClimaAtmos.jl
</code></pre></div>
<p dir="auto">To use ClimaAtmos, you need to instantiate all dependencies with:</p>
<div class="snippet-clipboard-content notranslate position-relative overflow-auto" data-snippet-clipboard-copy-content="$ julia --project
julia&gt;]
(v1.8) pkg&gt; instantiate"><pre class="notranslate"><code>$ julia --project
julia&gt;]
(v1.8) pkg&gt; instantiate
</code></pre></div>

<p dir="auto">If you're interested in contributing to the development of ClimaAtmos we want your help no matter how big or small a contribution you make! It's always great to have new people look at the code with fresh eyes: you will see errors that other developers have missed.</p>
<p dir="auto">Let us know by <a href="https://github.com/CliMA/ClimaAtmos.jl/issues/new">opening an issue</a> if you'd like to work on a new feature.</p>
