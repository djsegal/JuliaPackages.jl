<p dir="auto">Install the package,</p>
<div class="snippet-clipboard-content notranslate position-relative overflow-auto" data-snippet-clipboard-copy-content="] add QuantumAnnealing"><pre class="notranslate"><code>] add QuantumAnnealing
</code></pre></div>
<p dir="auto">Load the package and build a two spin ferromagnetic Ising model for simulation,</p>
<div class="snippet-clipboard-content notranslate position-relative overflow-auto" data-snippet-clipboard-copy-content="using QuantumAnnealing

ising_model = Dict((1,) =&gt; 0.1, (1,2) =&gt; -1.0)"><pre class="notranslate"><code>using QuantumAnnealing

ising_model = Dict((1,) =&gt; 0.1, (1,2) =&gt; -1.0)
</code></pre></div>
<p dir="auto">Perform a basic simulation with an annealing time of <code>1.0</code> and a linear annealing schedule,</p>
<div class="snippet-clipboard-content notranslate position-relative overflow-auto" data-snippet-clipboard-copy-content="ρ = simulate(ising_model, 1.0, AS_LINEAR)
print_z_state_probabilities(ρ)"><pre class="notranslate"><code>ρ = simulate(ising_model, 1.0, AS_LINEAR)
print_z_state_probabilities(ρ)
</code></pre></div>
<p dir="auto">Increase the annealing time to approach the adiabatic limit,</p>
<div class="snippet-clipboard-content notranslate position-relative overflow-auto" data-snippet-clipboard-copy-content="ρ = simulate(ising_model, 5.0, AS_LINEAR)
print_z_state_probabilities(ρ)"><pre class="notranslate"><code>ρ = simulate(ising_model, 5.0, AS_LINEAR)
print_z_state_probabilities(ρ)
</code></pre></div>
<p dir="auto">Change the annealing schedule and observe different state probabilities,</p>
<div class="snippet-clipboard-content notranslate position-relative overflow-auto" data-snippet-clipboard-copy-content="ρ = simulate(ising_model, 5.0, AS_QUADRATIC)
print_z_state_probabilities(ρ)"><pre class="notranslate"><code>ρ = simulate(ising_model, 5.0, AS_QUADRATIC)
print_z_state_probabilities(ρ)
</code></pre></div>
