<p dir="auto">With <strong>Trixi2Vtk.jl</strong> you can convert the HDF5-based output files created by
<a href="https://github.com/trixi-framework/Trixi.jl">Trixi.jl</a> (solution or restart
files) to VTK files. The converted files can then be further processed with
<a href="https://www.paraview.org" rel="nofollow">ParaView</a> or <a href="https://visit.llnl.gov" rel="nofollow">VisIt</a> to
generate publication-quality visualizations. Trixi2Vtk is part of the
<a href="https://github.com/trixi-framework">Trixi framework</a>.</p>

<p dir="auto">If you have not yet installed Julia, please <a href="https://julialang.org/downloads/platform/" rel="nofollow">follow the instructions for your
operating system</a>. Trixi2Vtk works
with Julia v1.7.</p>
<p dir="auto">Trixi2Vtk is a registered Julia package. Hence, you can install it by executing
the following commands in the Julia REPL:</p>
<div class="highlight highlight-source-julia notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="julia&gt; import Pkg; Pkg.add(&quot;Trixi2Vtk&quot;)"><pre>julia<span class="pl-k">&gt;</span> <span class="pl-k">import</span> Pkg; Pkg<span class="pl-k">.</span><span class="pl-c1">add</span>(<span class="pl-s"><span class="pl-pds">"</span>Trixi2Vtk<span class="pl-pds">"</span></span>)</pre></div>
<p dir="auto">To update an existing installation of Trixi2Vtk to the
<a href="https://github.com/trixi-framework/Trixi2Vtk.jl/releases/latest">latest release</a>,
execute</p>
<div class="highlight highlight-source-julia notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="julia&gt; import Pkg; Pkg.update(&quot;Trixi2Vtk&quot;); Pkg.status(&quot;Trixi2Vtk&quot;)"><pre>julia<span class="pl-k">&gt;</span> <span class="pl-k">import</span> Pkg; Pkg<span class="pl-k">.</span><span class="pl-c1">update</span>(<span class="pl-s"><span class="pl-pds">"</span>Trixi2Vtk<span class="pl-pds">"</span></span>); Pkg<span class="pl-k">.</span><span class="pl-c1">status</span>(<span class="pl-s"><span class="pl-pds">"</span>Trixi2Vtk<span class="pl-pds">"</span></span>)</pre></div>
<p dir="auto">in the Julia REPL. A brief list of notable changes to Trixi2Vtk is available in
<a href="NEWS.md"><code>NEWS.md</code></a>.</p>

<p dir="auto">In the Julia REPL, first load the package Trixi2Vtk</p>
<div class="highlight highlight-source-julia notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="julia&gt; using Trixi2Vtk"><pre>julia<span class="pl-k">&gt;</span> <span class="pl-k">using</span> Trixi2Vtk</pre></div>
<p dir="auto">To process an HDF5 file generated by Trixi.jl, execute</p>
<div class="highlight highlight-source-julia notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="julia&gt; trixi2vtk(joinpath(&quot;out&quot;, &quot;solution_000000.h5&quot;), output_directory=&quot;out&quot;)"><pre>julia<span class="pl-k">&gt;</span> <span class="pl-c1">trixi2vtk</span>(<span class="pl-c1">joinpath</span>(<span class="pl-s"><span class="pl-pds">"</span>out<span class="pl-pds">"</span></span>, <span class="pl-s"><span class="pl-pds">"</span>solution_000000.h5<span class="pl-pds">"</span></span>), output_directory<span class="pl-k">=</span><span class="pl-s"><span class="pl-pds">"</span>out<span class="pl-pds">"</span></span>)</pre></div>
<p dir="auto">This will create two unstructured VTK files in the <code>out/</code> subdirectory that can
be opened with ParaView or VisIt: <code>solution_000000.vtu</code> contains the
discontinuous Galerkin solution data while <code>solution_000000_celldata.vtu</code> holds
any cell-based values such as the current AMR indicator or the cell refinement
level.</p>
