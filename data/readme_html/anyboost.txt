<p>A julia based machine learning package for boosting any loss, activation and constraint.</p>
<p>This package implements the first order methods discussed in my <a href="https://stacks.stanford.edu/file/druid:dz563hn6234/Thesis_Draft_Rakesh_v4-augmented.pdf" rel="nofollow">PhD thesis</a>
from the Department of Statistics at Stanford University under Prof. Trevor Hastie.
The Thesis is titled “Boosting Like Path Algorithms for L1 Regularized Infinite Dimensional Convex Neural Networks”.</p>
<p>This package can be used to solve any Sparse Inverse Problem, once the required subroutines are implemented.</p>
