<p dir="auto">This package implements a continuous-time differential inequality approach to
computing state bounds and relaxations using the DynamicBounds.jl framework. These methods solve auxiliary ODE/DAEs to generate relaxations of parametric pODES/pDAEs at specific points in time. Full documentation of this functionality may be found <a href="https://psorlab.github.io/DynamicBounds.jl/dev/pODEsIneq/pODEsIneq" rel="nofollow">here</a> in the DynamicBounds.jl website.</p>
