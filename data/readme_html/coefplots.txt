<p dir="auto"><a href="https://github.com/caibengbu/Coefplots.jl">Coefplots</a> aims to make available in Julia (part of) the functionalities of <a href="http://repec.sowi.unibe.ch/stata/coefplot/getting-started.html" rel="nofollow">the Stata command <code>coefplot</code></a>. <a href="https://github.com/caibengbu/Coefplots.jl">Coefplots</a> is built on <a href="https://github.com/KristofferC/PGFPlotsX.jl/tree/ada03510396af592e05b2e382a0c12ce37ee3cc8">PGFPlotsX</a>.</p>
