<p dir="auto">VariantVisualization.jl is a package we built specifically to power the genetics visualization tool, <em>VIVA</em>.</p>
<p dir="auto"><em>VIVA</em> is a user-friendly command line tool for creating publication quality graphics from Variant Call Format (VCF) files. It has been designed for clinicians and bioinformaticians to explore their VCF files visually. In a single command, users can extract genotype or read depth information and plot trends in interactive categorical heatmaps and scatter plots of average read depth values. VIVA offers a robust set of filters to select variants and samples of interest for analysis. VIVA is especially useful in early data exploration for identifying batch effect and sources of poor read depth in sequencing experiments, as well as identifying distribution of disease causing variants in a set of clinical samples.</p>

<p dir="auto">To add Julia to the PATH on Windows 7 or Windows 10:</p>
<p dir="auto">Add the path to the Julia binaries (C:\Program Files\Julia\bin) to the PATH following the concise instructions <a href="https://www.java.com/en/download/help/path.xml" rel="nofollow">found here</a></p>
<p dir="auto">To add Julia to the PATH on Mac run the following line in the Terminal:</p>
<blockquote>
<p dir="auto">sudo ln -s /Applications/Julia-1.2.app/Contents/Resources/julia/bin/julia /usr/local/bin/julia</p>
</blockquote>
<p dir="auto">Be sure to replace "/Applications/Julia-1.2.app/..." to reflect the version of Julia you've downloaded.</p>
<p dir="auto"><em>Linux Note</em>: To run on remote compute clusters, you may need to load the opengl and julia modules.</p>

<p dir="auto">VIVA Jupyter notebook and the VIVA the command line tool are built with functions contained in our VariantVisualization.jl package.</p>
<p dir="auto">Developers may contribute to these open source tools by using <a href="https://github.com/compbiocore/VariantVisualization.jl/tree/master/src/">functions contained within VariantVisualization.jl</a> which are carefully documented with docstrings.</p>
<p dir="auto">We have included in-line comments within the code for the <a href="https://github.com/compbiocore/VariantVisualization.jl/tree/master/viva">VIVA command line tool</a>.</p>
<p dir="auto">The <em><strong>VIVA Jupyter notebook</strong></em> is powered by a <a href="https://github.com/compbiocore/VariantVisualization.jl/tree/master/src/new_notebook_utils.jl">main function</a> which takes arguments defined by the user in the notebook. We welcome users to post in issues to request a new feature or bug fix.</p>
