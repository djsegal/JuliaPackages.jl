<p dir="auto">A Julia package that provides functionality for state space sets.
These are collections of points of fixed, and known by type, size (called dimension).
It is used in several projects in the JuliaDynamics organization,
such as <a href="https://juliadynamics.github.io/DynamicalSystems.jl/dev/" rel="nofollow">DynamicalSystems.jl</a>
or <a href="https://juliadynamics.github.io/CausalityTools.jl/dev/" rel="nofollow">CausalityTools.jl</a>.</p>
<p dir="auto">The main export of <code>StateSpaceSets</code> is the concrete type <code>StateSpaceSet</code>.
The package also provides functionality for distances, neighbor searches,
sampling, and normalization.</p>
<p dir="auto">To install it you may run <code>import Pkg; Pkg.add("StateSpaceSets")</code>,
however, there is no real reason to install this package directly
as it is re-exported by all downstream packages that use it.</p>
<p dir="auto"><em>previously StateSpaceSets.jl was part of DelayEmbeddings.jl</em></p>
