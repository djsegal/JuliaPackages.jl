<p dir="auto">MacroTools provides a library of tools for working with Julia code and expressions. This includes a powerful template-matching system and code-walking tools that let you do deep transformations of code in a few lines. See the <a href="https://fluxml.github.io/MacroTools.jl/" rel="nofollow">docs</a> for more info.</p>
<p dir="auto">Suggestions, issues and pull requests are welcome.</p>
