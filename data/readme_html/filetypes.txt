<ul dir="auto">
<li>Supports a <a href="#supported-types">wide range</a> of file types</li>
<li>Provides file extension and proper MIME type</li>
<li>File discovery by extension or MIME type</li>
<li>File discovery by class (image, video, audio...)</li>
<li>Provides a bunch of helpers and file matching shortcuts</li>
<li>Simple and semantic API</li>
</ul>
