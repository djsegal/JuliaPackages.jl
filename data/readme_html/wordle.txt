<p dir="auto"><a href="https://www.nytimes.com/games/wordle/index.html" rel="nofollow">Wordle</a> is a daily word guessing game where the objective is to guess the daily, five-letter "wordle" within six guesses. Each time you make a guess, you are given the following feedback:</p>
<ul dir="auto">
<li>Letters that are in the correct position will be highlighted in green;</li>
<li>Letters that are in the wordle but not in the correct position are highlighted in yellow;</li>
<li>And, the rest are highlighted in gray.</li>
</ul>

<p dir="auto">It turns out that the full lists of valid words and wordles can be found in the game source code. Once extracted, we will be able to use these lists of words to do all sorts of fun stuff. Each time you load the package <code>using Wordle</code>, we download the list of words and only keep the ones that have been seen before (include today's; beware!).</p>
<p dir="auto">However, we should take care in the way we do this extraction to avoid spoilers. Ideally, the code we write to do this extraction requires no explicit knowledge of past or future wordles. In this way, no wordles are specifically "hard-coded" into the extraction routines.</p>
