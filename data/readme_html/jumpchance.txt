<p><strong>JuMPChance is no longer actively maintained. It is compatible with JuMP 0.18 and earlier.</strong></p>
<p>JuMPChance (formerly CCJuMP) is an extension to <a href="https://github.com/JuliaOpt/JuMP.jl">JuMP</a> for formulating and solving optimization problems with chance constraints (also known as probabilistic constraints). JuMPChance currently supports only a particular class of chance constraints involving affine combinations of jointly normal random variables, a classical formulation that's known to be efficiently solvable by using second-order conic programming (SOCP) (although JuMPChance also provides an outer-approximation algorithm which solves a sequence of linear problems).</p>
<p>JuMPChance supports an extension of the classical model to distributionally robust (or ambiguous) chance constraints where the parameters of the normal distributions are known to fall in a symmetric interval or more general uncertainty set.</p>
<p>See the <a href="http://jumpchance.readthedocs.org/" rel="nofollow">documentation</a> for installation installation instructions, a quick start guide, and a more detailed discussion of the methods implemented.</p>
