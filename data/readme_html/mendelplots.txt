<p dir="auto">MendelPlots.jl is a Julia package for generating common plots from results of genome-wide association studies (GWAS). It can currently create Manhattan Plots and QQ Plots.</p>
<p dir="auto"><strong>MendelPlots</strong> is a plotting and data visualization system written in <a href="http://julialang.org/" rel="nofollow">Julia</a>. It uses <a href="https://github.com/GiovineItalia/Gadfly.jl">Gadfly</a> as the backend, which is influenced by Grammar of Graphics Style programming and Hadley Wickham's <a href="http://ggplot2.org/" rel="nofollow">ggplot2</a></p>
<p dir="auto">This package is registered in the default Julia package registry, and can be installed through standard package installation procedure: e.g., running the following code in Julia REPL.</p>
<div class="highlight highlight-source-julia notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="using Pkg
