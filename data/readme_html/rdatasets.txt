<p dir="auto"><a href="https://github.com/JuliaStats/RDatasets.jl/actions?query=workflow%3ACI+branch%3Amaster"><img src="https://github.com/JuliaStats/RDatasets.jl/workflows/CI/badge.svg" alt="Build status" style="max-width: 100%;"></a></p>
<p dir="auto">The RDatasets package provides an easy way for Julia users to experiment with most of the standard data sets that are available in the core of R as well as datasets included with many of R's most popular packages. This package is essentially a simplistic port of the Rdatasets repo created by Vincent Arelbundock, who conveniently gathered data sets from many of the standard R packages in one convenient location on GitHub at <a href="https://github.com/vincentarelbundock/Rdatasets">https://github.com/vincentarelbundock/Rdatasets</a></p>
<p dir="auto">In order to load one of the data sets included in the RDatasets package, you will need to have the <code>DataFrames</code> package installed. This package is automatically installed as a dependency of the <code>RDatasets</code> package if you install <code>RDatasets</code> as follows:</p>
<div class="snippet-clipboard-content notranslate position-relative overflow-auto" data-snippet-clipboard-copy-content="Pkg.add(&quot;RDatasets&quot;)"><pre class="notranslate"><code>Pkg.add("RDatasets")
</code></pre></div>
<p dir="auto">After installing the RDatasets package, you can then load data sets using the <code>dataset()</code> function, which takes the name of a package and a data set as arguments:</p>
<div class="snippet-clipboard-content notranslate position-relative overflow-auto" data-snippet-clipboard-copy-content="using RDatasets
iris = dataset(&quot;datasets&quot;, &quot;iris&quot;)
neuro = dataset(&quot;boot&quot;, &quot;neuro&quot;)"><pre class="notranslate"><code>using RDatasets
iris = dataset("datasets", "iris")
neuro = dataset("boot", "neuro")
</code></pre></div>
