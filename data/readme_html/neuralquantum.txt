<p><strong>NeuralQuantum</strong> is a numerical framework written in <a href="http://julialang.org" rel="nofollow">Julia</a> to investigate
Neural-Network representations of mixed quantum states and to find the Steady-
State of dissipative Quantum Systems with variational Montecarlo schemes.
It can also compute the ground state of hermitian hamiltonians.</p>
<p>This code has been developed while working on <a href="https://arxiv.org/abs/1902.10104" rel="nofollow">Variational neural network ansatz for steady states in open quantum systems</a>, by <a href="mailto:filippo.vicentini@univ-paris-diderot.fr">F. Vicentini</a> et al. <a href="https://link.aps.org/doi/10.1103/PhysRevLett.122.250503" rel="nofollow">Phys Rev Lett 122, 250503 (2019)</a>.</p>

<p>To Install <code>NeuralQuantum.jl</code>, run the following commands to install it's dependcy. Please note that we require julia &gt;= 1.3, and relatively recent versions of several packages.</p>
<pre><code>using Pkg
pkg"add https://github.com/PhilipVinc/NeuralQuantum.jl"
</code></pre>
<p>Alternatively you may activate the project included in the manifest that comes with NeuralQuantum.</p>
