<p dir="auto">This package implements the functions of <a href="https://github.com/dgilford">Daniel Gilford</a>'s <a href="https://github.com/dgilford/tcpyPI/blob/master/tcpyPI/pi.py">TCPyPI</a> for the <a href="https://github.com/JuliaLang/julia">JuliaLanguage</a>. This are widely used routines in the Tropical Cyclone community. I hope this helps widen the atmospheric science ecosystem for Julia.</p>
