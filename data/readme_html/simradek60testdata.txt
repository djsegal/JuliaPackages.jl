<p dir="auto">Sample Simrad EK60 scientific echosounder data for testing and
demonstration purposes.</p>
<p dir="auto">Get the name of a RAW file like this:</p>
<div class="snippet-clipboard-content notranslate position-relative overflow-auto" data-snippet-clipboard-copy-content="using SimradEK60TestData

</code></pre></div>

<div class="snippet-clipboard-content notranslate position-relative overflow-auto" data-snippet-clipboard-copy-content="cal = ECS_SAMPLE"><pre class="notranslate"><code>cal = ECS_SAMPLE

<p dir="auto">The data directory also includes corresponding CSV data exported from
EchoView to allow comparison and testing.</p>
