<p dir="auto">Julia package for causal inference, graphical models and structure learning.</p>
<p dir="auto">This package contains code for the stable version of the PC algorithm and the extended FCI algorithm.</p>
<p dir="auto">See the <a href="https://mschauer.github.io/CausalInference.jl/latest/" rel="nofollow">documentation</a> for implemented functionality and <a href="https://github.com/mschauer/CausalInference.jl/issues/1" data-hovercard-type="issue" data-hovercard-url="/mschauer/CausalInference.jl/issues/1/hovercard">issue #1 (Roadmap/Contribution)</a> for coordination of the development.</p>

<p dir="auto"><strong>Q:</strong> I looked for "causal inference" and found <a href=".">CausalInference.jl</a> and <a href="http://www.zenna.org/Omega.jl/latest/causal/" rel="nofollow">Omega.jl</a>...
<strong>A:</strong> CausalInference.jl is about causal discovery; you observe the data and want to infer the causal structure. Omega lets you reason what happens then: when you intervene ("do calculus") and want to cause changes.</p>
