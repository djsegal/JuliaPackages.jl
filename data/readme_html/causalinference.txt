<p>Julia package for causal inference, graphical models and structure learning.</p>
<p>This package contains code for the stable version of the PC algorithm and the FCI algorithm as described in Zhang's article.</p>
<p>The algorithms use the Julia packagea <a href="https://github.com/JuliaGraphs/LightGraphs.jl">LightGraphs</a> and <a href="https://github.com/JuliaGraphs/MetaGraphs.jl">MetaGraphs</a>. Graphs are represented by sorted adjacency lists (vectors in the implemention). CPDAGs are just <code>DiGraph</code>s where unoriented edges are represented by both a forward and a backward directed edge. PAGs are <code>MetaDiGraph</code>s where every edge is represented by a forward and a backward edge. The marks of the endpoint of an edge are stored in the <code>:marks</code> property. Marks can be checked and set using the <code>has_marks</code> and <code>set_marks!</code> functions (see documentation for details).</p>
<p>The PC algorithm is tested on random DAGs by comparing the result of the PC algorithm using the <em>d</em>-separation oracle with CPDAGs computed with Chickering's DAG-&gt;CPDAG conversion algorithm (implemented as <code>dsep</code> and <code>cpdag</code> in this package).</p>
<p>See the <a href="https://mschauer.github.io/CausalInference.jl/latest/" rel="nofollow">documentation</a> for other implemented functionality and <a href="https://github.com/mschauer/CausalInference.jl/issues/1">issue #1 (Roadmap/Contribution)</a> for coordination of the development.</p>
