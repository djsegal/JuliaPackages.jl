<p>NeRCA (<strong>Ne</strong>utrino <strong>R</strong>esearch with <strong>C</strong>osmics in the <strong>A</strong>byss) is a Julia
package to access and analyse KM3NeT related data.
It can read the official KM3NeT ROOT and formats, calibration information (DETX
files) and also live data from running detectors.
The ROyFit reconstruction is also included and can be found in <code>src/fit.jl</code>.</p>
