<p dir="auto">This package provides tools and structures to manipulate and apply gradient impulse response functions (GIRFs) to acquired imaging data using the MRIReco.jl datatypes.</p>
<p dir="auto">To use this package, clone the repo into your <code>.julia/dev/</code> folder, and add the package to your environment using <code>dev MRIGradients</code> in the package manager mode in the Julia REPL.</p>
<p dir="auto">Then, in your script, proceed with the <code>using MRIGradients</code> statement as usual.</p>
