<p>This module provides an interface for generating .pomdpx files that can be used with the <a href="https://github.com/JuliaPOMDP/SARSOP.jl">SARSOP.jl</a>. This module leverages the API defined in <a href="https://github.com/JuliaPOMDP/POMDPs.jl">POMDPs.jl</a>.</p>
