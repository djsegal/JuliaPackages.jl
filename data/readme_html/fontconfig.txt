<p><code>Pattern</code> corresponds to the fontconfig type <code>FcPattern</code>. It respresents a set
of font properties used to match specific fonts.</p>
<p>It can be constructed in two ways. First with zero or more keyword arguments
corresponding to fontconfig font properties.</p>
<div class="highlight highlight-source-julia position-relative" data-snippet-clipboard-copy-content="Fontconfig.Pattern(; args...)
"><pre>Fontconfig<span class="pl-k">.</span><span class="pl-c1">Pattern</span>(; args<span class="pl-k">...</span>)</pre></div>
<p>For example</p>
<div class="highlight highlight-source-julia position-relative" data-snippet-clipboard-copy-content="Fontconfig.Pattern(family=&quot;Helvetica&quot;, size=10, hinting=true)
"><pre>Fontconfig<span class="pl-k">.</span><span class="pl-c1">Pattern</span>(family<span class="pl-k">=</span><span class="pl-s"><span class="pl-pds">"</span>Helvetica<span class="pl-pds">"</span></span>, size<span class="pl-k">=</span><span class="pl-c1">10</span>, hinting<span class="pl-k">=</span><span class="pl-c1">true</span>)</pre></div>

<h1><a id="user-content-match" class="anchor" aria-hidden="true" href="#match"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path fill-rule="evenodd" d="M7.775 3.275a.75.75 0 001.06 1.06l1.25-1.25a2 2 0 112.83 2.83l-2.5 2.5a2 2 0 01-2.83 0 .75.75 0 00-1.06 1.06 3.5 3.5 0 004.95 0l2.5-2.5a3.5 3.5 0 00-4.95-4.95l-1.25 1.25zm-4.69 9.64a2 2 0 010-2.83l2.5-2.5a2 2 0 012.83 0 .75.75 0 001.06-1.06 3.5 3.5 0 00-4.95 0l-2.5 2.5a3.5 3.5 0 004.95 4.95l1.25-1.25a.75.75 0 00-1.06-1.06l-1.25 1.25a2 2 0 01-2.83 0z"></path></svg></a>Match</h1>
<p>The primary functionality fontconfig provides is matching font patterns. In
Fontconfig.jl this is done with the <code>match</code> function, corresponding to <code>FcMatch</code>
in fontconfig.</p>
<div class="highlight highlight-source-julia position-relative" data-snippet-clipboard-copy-content="match(pat::Pattern)
"><pre><span class="pl-c1">match</span>(pat<span class="pl-k">::</span><span class="pl-c1">Pattern</span>)</pre></div>
<p>It takes a <code>Pattern</code> and return a <code>Pattern</code> corresponding to the nearest
matching installed font.</p>

function, which wraps <code>FcPatternFormat</code>.</p>

function will return a vector of <code>Pattern</code>s</p>
