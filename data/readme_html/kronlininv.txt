<p dir="auto">Kronecker-product-based linear inversion of geophysical (or other kinds of) data under Gaussian and separability assumptions.
The code computes the posterior mean model and the posterior covariance matrix (or subsets of it) in an efficient manner (parallel algorithm) taking into account 3-D correlations both in the model parameters and in the observed data.</p>
<p dir="auto">If you use this code for research or else, please cite the related paper:</p>
<p dir="auto">Andrea Zunino, Klaus Mosegaard,
<strong>An efficient method to solve large linearizable inverse problems under Gaussian and separability assumptions</strong>,
Computers &amp; Geosciences, 2018
ISSN 0098-3004, <a href="https://doi.org/10.1016/j.cageo.2018.09.005" rel="nofollow">https://doi.org/10.1016/j.cageo.2018.09.005</a>.</p>
