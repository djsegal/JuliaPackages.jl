<p>This Julia package is for to solve mathematical optimisation problems of both unconstrained as well as constrained kind. The package uses the approach named <em>Charibde</em> in which it uses two parallel running algorithms <em>Differential Evolution</em> and <em>Interval Branch &amp; Contract algorithm</em> to achieve solution of any difficult problem.</p>
<p>The package makes these two algorithms run in parallel either on same worker (processor) or on two different workers while maintaining contact with one another through <code>channels</code> or <code>remotechannels</code>.</p>
<p>The package also allows us to use just <em>Interval Branch &amp; Contract</em> algorithm to solve the problems..</p>
<p>The package can also be used through JuMP syntax.</p>
