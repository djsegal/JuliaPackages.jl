<p dir="auto">This packages defines an interface for defining and working with <a href="https://en.wikipedia.org/wiki/Hybrid_system" rel="nofollow">Hybrid Systems</a>.
It also includes an implementation of this interface.</p>
<p dir="auto">The goal of this package is twofold</p>
<ul dir="auto">
<li>help making algorithms on Hybrid Systems independent of the particular data structure used to represent them.</li>
<li>help users to try many different tools for Hybrid Systems without needed to deal with different interfaces.</li>
</ul>
<p dir="auto">The following package implements hybrid systems representations using this interface:</p>
<ul dir="auto">
<li><a href="https://github.com/JuliaReach/SpaceExParser.jl">SpaceExParser</a> : <a href="http://spaceex.imag.fr/" rel="nofollow">SpaceEx</a> modeling language parser</li>
</ul>
<p dir="auto">The following package implements algorithms using this interface:</p>
<ul dir="auto">
<li><a href="https://github.com/blegat/SwitchOnSafety.jl">SwitchOnSafety</a> : Computing invariant sets of hybrid systems.</li>
<li><a href="https://github.com/JuliaReach/ReachabilityAnalysis.jl">ReachabilityAnalysis</a> : Computing reachable states of hybrid dynamical systems.</li>
</ul>
