<h2 dir="auto"><a id="user-content-features" class="anchor" aria-hidden="true" href="#features"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path d="m7.775 3.275 1.25-1.25a3.5 3.5 0 1 1 4.95 4.95l-2.5 2.5a3.5 3.5 0 0 1-4.95 0 .751.751 0 0 1 .018-1.042.751.751 0 0 1 1.042-.018 1.998 1.998 0 0 0 2.83 0l2.5-2.5a2.002 2.002 0 0 0-2.83-2.83l-1.25 1.25a.751.751 0 0 1-1.042-.018.751.751 0 0 1-.018-1.042Zm-4.69 9.64a1.998 1.998 0 0 0 2.83 0l1.25-1.25a.751.751 0 0 1 1.042.018.751.751 0 0 1 .018 1.042l-1.25 1.25a3.5 3.5 0 1 1-4.95-4.95l2.5-2.5a3.5 3.5 0 0 1 4.95 0 .751.751 0 0 1-.018 1.042.751.751 0 0 1-1.042.018 1.998 1.998 0 0 0-2.83 0l-2.5 2.5a1.998 1.998 0 0 0 0 2.83Z"></path></svg></a>Features</h2>
<ul dir="auto">
<li>
<p dir="auto">Chemical structure file I/O</p>
<ul dir="auto">
<li>2D structure image drawing and export to SVG</li>
<li>3D structure drawing (<a href="https://github.com/MakieOrg/Makie.jl">Makie.jl</a>)</li>
<li>SDFile reader/writer (.sdf, .mol)</li>
<li>SMILES/SMARTS parser (only reader)</li>
<li>Coordinates generation (<a href="https://github.com/schrodinger/coordgenlibs">coordgenlibs</a>)</li>
</ul>
</li>
<li>
<p dir="auto">Properties and descriptors</p>
<ul dir="auto">
<li>H-bond donor/acceptor</li>
<li>rotatable bonds</li>
<li>Aromaticity</li>
<li>Wildman-Crippen logP</li>
</ul>
</li>
<li>
<p dir="auto">Substructure and query</p>
<ul dir="auto">
<li>InChI (<a href="https://www.inchi-trust.org/" rel="nofollow">InChI</a>)</li>
<li>Serialization (molecule object &lt;-&gt; JSON)</li>
<li>Subgraph isomorphism detection with VF2 algorithm
<ul dir="auto">
<li>SMARTS query match</li>
<li>Monomorphism, node-induced and edge-induced match</li>
<li>Constraints (mandatory/forbidden vertex mapping)</li>
</ul>
</li>
<li>Functional group query set</li>
<li>Structural alerts detection with ChEMBL dataset</li>
</ul>
</li>
<li>
<p dir="auto">Atomic mass</p>
<ul dir="auto">
<li>standard atomic/molecular weight</li>
<li>relative atomic/molecular mass</li>
<li>isotopic composition</li>
</ul>
</li>
<li>
<p dir="auto">Molecular graph topology</p>
<ul dir="auto">
<li>Ring, scaffold, connected components</li>
<li>Minimum cycle basis (de Pina algorithm)
<ul dir="auto">
<li>Smallest set of smallest rings (SSSR)</li>
</ul>
</li>
<li>Planarity (left-right planarity test)</li>
<li>Maximum matching
<ul dir="auto">
<li>Kekulization</li>
</ul>
</li>
<li>Graph traversal</li>
</ul>
</li>
<li>
<p dir="auto">Maximum common substructure (MCS)</p>
<ul dir="auto">
<li>By clique detection algorithm</li>
<li>Node-induced (MCIS) and edge-induced (MCES)</li>
<li>Connected and disconnected</li>
<li>Topological constraint (known as tdMCS)</li>
<li>Diameter restriction (MCS-DR) and graph-based local similarity (GLS)</li>
</ul>
</li>
</ul>
