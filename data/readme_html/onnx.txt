<p dir="auto">ONNX.jl is in the process of a total reconstruction and currently supports saving &amp; loading graphs as a <a href="https://dfdx.github.io/Umlaut.jl/dev/reference/#Umlaut.Tape" rel="nofollow"><code>Umlaut.Tape</code></a>. When possible, functions from <code>NNlib</code> or standard library are used, but no conversion to Flux is implemented yet. See <a href="examples/resnet18.jl">resnet18.jl</a> for a practical example of graph loading.</p>
<p dir="auto">Not all ONNX operators are implemented. See <a href="CONTRIBUTE.md">How to contribute</a> for details of adding new operators.</p>
