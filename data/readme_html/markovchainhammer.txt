  MarkovChainHammer.jl
</h1>
<p dir="auto">A toolkit for analyzing, generating, and constructing both continuous and discrete markov chains with a finite state space. See the <a href="https://sandreza.github.io/MarkovChainHammer.jl/dev/" rel="nofollow">documentation</a> for the mathematical underpinnings, repository functionality, and examples.</p>

<p dir="auto">By default MarkovChainHammer exports no functions and has modules that</p>
<ol dir="auto">
<li>Construct <a href="https://en.wikipedia.org/wiki/Transfer_operator" rel="nofollow">transfer operators</a> from data</li>
<li>Construct <a href="https://en.wikipedia.org/wiki/Markov_chain" rel="nofollow">Markov chains</a> from transfer operators</li>
<li>Detects <a href="https://en.wikipedia.org/wiki/Community_structure" rel="nofollow">communities</a> from transfer operators</li>
<li>Exports utilities for plotting with <a href="https://github.com/MakieOrg/Makie.jl">Makie</a></li>
</ol>
<p dir="auto">See the <a href="https://github.com/sandreza/MarkovChainHammer.jl/tree/main/examples">examples</a> for inspiration on how the utilities can be used.</p>
