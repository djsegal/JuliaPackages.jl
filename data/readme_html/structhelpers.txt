<p dir="auto">Sometimes defining a new struct is accompanied by a bit of boilerplate. For instance the default definition of <code>Base.hash</code> is by object id. Often a <code>hash</code> that is based on the object struture is preferable however. Similar for <code>==</code>.
StructHelpers aims to simplify the boilerplate required for such common tweaks.</p>
