<p dir="auto"><a href="https://github.com/jump-dev/MosekTools.jl">MosekTools.jl</a> is the
<a href="https://github.com/jump-dev/MathOptInterface.jl">MathOptInterface.jl</a>
implementation for the MOSEK solver.</p>
<p dir="auto">The low-level solver API for MOSEK is found in the package <a href="https://github.com/MOSEK/Mosek.jl">Mosek.jl</a>.</p>

the latest release of Mosek.jl (which uses MOSEK v10).</p>
<p dir="auto">To use MOSEK v9 (resp. v8), use the v0.12.x (resp. v0.7.x) releases of this
package, and the <code>mosekv9</code> (resp. <code>mosekv8</code>) branch and v1.2.x (resp. v0.9.x)
releases of Mosek.jl.</p>

<p dir="auto">The parameter <code>QUIET</code> is a special parameter that when set to <code>true</code>
disables all Mosek printing output.</p>
<p dir="auto">All other parameters can be found in the <a href="https://docs.mosek.com/8.1/capi/param-groups.html#doc-param-groups" rel="nofollow">Mosek documentation</a>.</p>
<p dir="auto">Note that the prefix <code>MSK_IPAR_</code> (for integer parameters), <code>MSK_DPAR_</code> (for
floating point parameters) or <code>MSK_SPAR_</code> (for string parameters) are optional.
If they are not given, they are inferred from the type of the value. For
example, in the example above, as <code>1e-7</code> is a floating point number, the
parameters name used is <code>MSK_DPAR_INTPNT_CO_TOL_DFEAS</code>.</p>
