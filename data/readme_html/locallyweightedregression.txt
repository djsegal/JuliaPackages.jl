<p>This package provides an implementation of locally weighted regression (a.k.a. LOESS) introduced by
<a href="http://www.stat.washington.edu/courses/stat527/s13/readings/Cleveland_JASA_1979.pdf" rel="nofollow">Cleveland 1979</a>.
It is the most natural generalization of <a href="https://github.com/JuliaEarth/InverseDistanceWeighting.jl">InverseDistanceWeighting.jl</a>
in which one is allowed to use a custom weight function instead of distance-based weights.</p>
<p>Like in the inverse distance weighting scheme, this package makes use of k-d trees from the
<a href="https://github.com/KristofferC/NearestNeighbors.jl">NearestNeighbors.jl</a> package for fast data
lookup. Locally weighted regression (LWR or LOESS) is a popular non-parametric method in machine
learning, however it still has poor statistical properties compared to other estimation methods
such as Kriging that explicitly model spatial correlation.</p>
<p>In the current implementation, the estimation variance is computed assuming Gaussian residuals.</p>
