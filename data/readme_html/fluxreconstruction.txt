<p dir="auto"><strong>FluxReconstruction</strong> is a lightweight <a href="https://julialang.org" rel="nofollow">Julia</a> implementation of the <a href="https://arc.aiaa.org/doi/10.2514/6.2007-4079" rel="nofollow">flux reconstruction</a> method proposed by Huynh.
It is built in conjunction with the <a href="https://github.com/SciML/DifferentialEquations.jl">SciML</a> and <a href="https://github.com/vavrines/Kinetic.jl">Kinetic</a> ecosystems.</p>

<p dir="auto">FluxReconstruction focuses on numerical solutions of transport equations.
Any advection-diffusion-type equation can be solved within the framework.
A partial list of current supported models include</p>
<ul dir="auto">
<li>advection-diffusion equation</li>
<li>Burgers equation</li>
<li>Euler equations</li>
<li>Navier-Stokes equations</li>
<li>Boltzmann equation</li>
</ul>
