<p dir="auto">This module aims to solve multistage stochastic problems by randomized versions of the progressive hedging algorithm. It comes with the companion paper "<a href="https://hal.archives-ouvertes.fr/hal-02946615/document" rel="nofollow">Randomized Progressive Hedging methods for Multi-stage Stochastic Programming</a>" published in the journal Annals of Operations Research.</p>

<p dir="auto">The package is installed with the following command:</p>
<div class="highlight highlight-source-julia notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="]add RandomizedProgressiveHedging.jl"><pre>]add RandomizedProgressiveHedging<span class="pl-k">.</span>jl</pre></div>
<p dir="auto">GLPK and Ipopt are installed along as default solvers. Other solvers can of course be used, see <a href="http://www.juliaopt.org/JuMP.jl/v0.19.0/installation/#Getting-Solvers-1" rel="nofollow">JuMP doc</a>. for installation and <code>example/</code> scripts for use of the Juniper and Mosek solvers. If you don't have them already, simply <code>]add Juniper Mosek MosekTools</code>.</p>

<p dir="auto">Many example scripts are available in the <code>example/</code> folder. A good place to start is:</p>
<ul dir="auto">
<li>the <a href="https://yassine-laguel.github.io/RandomizedProgressiveHedging.jl/stable" rel="nofollow">documentation</a>!</li>
<li>the <a href="https://github.com/yassine-laguel/RandomizedProgressiveHedging.jl/blob/master/examples/tutorial.ipynb"><code>examples/tutorial.ipynb</code></a> jupyter notebook</li>
<li>the script <a href="https://github.com/yassine-laguel/RandomizedProgressiveHedging.jl/blob/master/examples/tutorial.jl"><code>tutorial.jl</code></a></li>
</ul>
<p dir="auto">For distributed solve on local machine, launch e.g. julia as <code>julia -p 3</code> for 2 workers and a master thread. On a cluster, add workers with the package <code>Distributed</code> either <code>addprocs</code> on ssh connection, or use an adapted <a href="https://github.com/JuliaParallel/ClusterManagers.jl">cluster manager</a>.</p>
