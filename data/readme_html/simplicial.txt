<p dir="auto">This package provides data structures and algorithms for combinatorial topology. Currently, it can handle filtered simplicial complexes, directed complexes, and combinatorial codes. The package is written in <a href="http://julialang.org" rel="nofollow">Julia</a>.  The long-term goal of this project is to be a <em>"swiss-knife"</em>  for manipulating (<em>very</em> <em>large</em>) combinatorial structures, with an eye towards topological data analysis.</p>
<p dir="auto">This software is free under the terms of the GNU General Public License (<a href="http://www.gnu.org/licenses/gpl.html" rel="nofollow">GPLv3</a>).
The work was supported by the ARO award W911NF-15-1-0084 and NIH R01GM117592.</p>
<p dir="auto">DISCLAIMER: This software is still in development. The documentation is currently very sparse. Use at your own risk! Please let us know if you'd like to contribute.</p>
<p dir="auto">This package interfaces with existing TDA software for homology computations. Currently it uses <a href="https://github.com/blazs/phat">PHAT</a> and <a href="http://people.maths.ox.ac.uk/nanda/perseus/index.html" rel="nofollow">Perseus</a>.  In the future, <code>Simplicial</code> will interface with other TDA software as well.</p>
