<p>Bayesian Inference At Scale</p>
<p><a href="https://travis-ci.org/adham/BIAS.jl" rel="nofollow"><img src="https://camo.githubusercontent.com/220273ff26f6e1705087ce6b3b92c683954b783b/68747470733a2f2f7472617669732d63692e6f72672f616468616d2f424941532e6a6c2e7376673f6272616e63683d6d6173746572" alt="Build Status" data-canonical-src="https://travis-ci.org/adham/BIAS.jl.svg?branch=master" style="max-width:100%;"></a></p>
<p><em>BIAS</em> is an open source implementation of several Bayesian parametric and non-parametric
models in <a href="http://julialang.org/" rel="nofollow">julia</a>. The implemented models include:</p>
<ol>
<li>Bayesian Mixture Model (BMM)</li>
<li>Latent Dirichlet Allocation (LDA)</li>
<li>Dirichlet Process Mixture Model (DPM)</li>
<li>Hierarchical Dielectric Mixture Model (HDP)</li>
<li>dynamic Hierarchical Dirichlet Mixture Model (dHDP)</li>
<li>Temporal Dirichlet Process Mixture model (TDPM) also called Recurrent Chinese Restaurant Process (RCRP)</li>
<li>Recurrent Chinese Restaurant Franchise (RCRF)</li>
</ol>
<p>See the <a href="http://biasjl.readthedocs.org" rel="nofollow">Package Documentation</a> for details and examples.</p>
