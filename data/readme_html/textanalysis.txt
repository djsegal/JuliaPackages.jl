<h2 dir="auto"><a id="user-content-features" class="anchor" aria-hidden="true" href="#features"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path d="m7.775 3.275 1.25-1.25a3.5 3.5 0 1 1 4.95 4.95l-2.5 2.5a3.5 3.5 0 0 1-4.95 0 .751.751 0 0 1 .018-1.042.751.751 0 0 1 1.042-.018 1.998 1.998 0 0 0 2.83 0l2.5-2.5a2.002 2.002 0 0 0-2.83-2.83l-1.25 1.25a.751.751 0 0 1-1.042-.018.751.751 0 0 1-.018-1.042Zm-4.69 9.64a1.998 1.998 0 0 0 2.83 0l1.25-1.25a.751.751 0 0 1 1.042.018.751.751 0 0 1 .018 1.042l-1.25 1.25a3.5 3.5 0 1 1-4.95-4.95l2.5-2.5a3.5 3.5 0 0 1 4.95 0 .751.751 0 0 1-.018 1.042.751.751 0 0 1-1.042.018 1.998 1.998 0 0 0-2.83 0l-2.5 2.5a1.998 1.998 0 0 0 0 2.83Z"></path></svg></a>Features</h2>
<ul dir="auto">
<li>Container type for <em>Document</em> and <em>Corpus</em></li>
<li>DocumentTermMatrix and TF/IDF</li>
<li>LSA/LDA</li>
<li>Vocabulary and statistical Language Model</li>
<li>Co-occurance matrix</li>
<li>NaiveBayes classifier</li>
<li>ROUGE evaluation metrics</li>
</ul>
<p dir="auto">This package also incorporates features from the <a href="https://juliahub.com/ui/Packages/Languages/w1H1r" rel="nofollow">Languages</a> and <a href="https://juliahub.com/ui/Packages/WordTokenizers/wKkKC" rel="nofollow">WordTokenizers</a> packages within the <a href="https://github.com/JuliaText">JuliaText</a> ecosystem.</p>

<p dir="auto">The <a href="https://github.com/JuliaText/TextModels.jl">TextModels</a> package enhances this library with the additon of practical neural network based models. Some of that code used to live in this package, but was moved to simplify installation and reduce the number of dependencies.</p>
