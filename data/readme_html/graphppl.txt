<p dir="auto">GraphPPL.jl is a probabilistic programming language focused on probabilistic graphical models. This repository is aimed for advanced users, please refer to the <a href="https://github.com/biaslab/ReactiveMP.jl">ReactiveMP.jl</a> repository for more comprehensive and self-contained documentation and usages examples.</p>

<p dir="auto">GraphPPL.jl does not export any Bayesian inference backend. It provides a simple DSL parser, model generation, constraints specification and meta specification helpers. To run inference on
generated models user needs to have a Bayesian inference backend with GraphPPL.jl support (e.g. <a href="https://github.com/biaslab/ReactiveMP.jl">ReactiveMP.jl</a>).</p>
