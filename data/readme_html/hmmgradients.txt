<p>This package enables computing the gradient of the parameters of <a href="https://en.wikipedia.org/wiki/Hidden_Markov_model" rel="nofollow">Hidden Markov Models (HMMs)</a>.
This makes it possible to perform HMM training using gradient based methods like stochastic gradient descent, which is necessary for example when neural networks are involved, e.g. in modern automatic speech recognition systems. Check out this <a href="https://github.com/idiap/TIDIGITSRecipe.jl">TIDIGITS recipe</a> for an example.</p>
<p>Formally, this package extends <a href="https://github.com/JuliaDiff/ChainRulesCore.jl">ChainRulesCore</a>
making it possible to train HMM models using the automatic differentiation frameworks of Julia,
for example using <a href="https://github.com/FluxML/Zygote.jl">Zygote</a> and machine learning libraries like <a href="https://github.com/FluxML/Flux.jl">Flux</a>.
The package also provides numerical stable algorithms to compute forward, backward and posterior probabilities of HMMs.</p>
