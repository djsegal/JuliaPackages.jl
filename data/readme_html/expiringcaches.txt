<div class="snippet-clipboard-content notranslate position-relative overflow-auto" data-snippet-clipboard-copy-content="ExpiringCaches.Cache{K, V}(timeout::Dates.Period; purge_on_timeout::Bool=false)"><pre class="notranslate"><code>ExpiringCaches.Cache{K, V}(timeout::Dates.Period; purge_on_timeout::Bool=false)
</code></pre></div>
<p dir="auto">Create a thread-safe, expiring cache where values older than <code>timeout</code>
are "invalid" and will be deleted.</p>
<p dir="auto">An <code>ExpiringCaches.Cache</code> is an <code>AbstractDict</code> and tries to emulate a regular
<code>Dict</code> in all respects. It is most useful when the cost of retrieving or
calculating a value is expensive and is able to be "cached" for a certain
amount of time. To avoid using the cache (i.e. to invalidate the cache),
a <code>Cache</code> supports the <code>delete!</code> and <code>empty!</code> methods to remove values
manually.</p>

<div class="snippet-clipboard-content notranslate position-relative overflow-auto" data-snippet-clipboard-copy-content="@cacheable timeout function_definition::ReturnType"><pre class="notranslate"><code>@cacheable timeout function_definition::ReturnType
</code></pre></div>
<p dir="auto">For a function definition (<code>function_definition</code>, either short-form
or full), create an <code>ExpiringCaches.Cache</code> and store results for <code>timeout</code>
(hashed by the exact input arguments obviously).</p>
<p dir="auto">Note that the function definition <em>MUST</em> include the <code>ReturnType</code> declartion
as this is used as the value (<code>V</code>) type in the <code>Cache</code>.</p>
