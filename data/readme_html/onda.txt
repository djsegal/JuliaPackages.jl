<p>Onda.jl is a Julia package for high-throughput manipulation of structured LPCM signal data across arbitrary domain-specific encodings, file formats and storage layers via <a href="https://github.com/beacon-biosignals/OndaFormat">the Onda Format</a>.</p>
<p>This package follows the <a href="https://github.com/jrevels/YASGuide">YASGuide</a>.</p>
