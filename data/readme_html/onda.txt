<p>To install Onda for development, run:</p>
<pre><code>julia -e 'using Pkg; Pkg.develop(PackageSpec(url="https://github.com/beacon-biosignals/Onda.jl"))'
</code></pre>
<p>This will install Onda to the default package development directory, <code>~/.julia/dev/Onda</code>.</p>
