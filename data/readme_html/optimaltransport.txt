<p>This package provides some <a href="https://julialang.org/" rel="nofollow">Julia</a> implementations of algorithms for computational <a href="https://optimaltransport.github.io/" rel="nofollow">optimal transport</a>, including the Earth-Mover's (Wasserstein) distance, Sinkhorn scaling algorithm for entropy-regularised transport as well as some variants or extensions.</p>
<h2><a id="user-content-overview-of-supported-functionality" class="anchor" aria-hidden="true" href="#overview-of-supported-functionality"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path fill-rule="evenodd" d="M7.775 3.275a.75.75 0 001.06 1.06l1.25-1.25a2 2 0 112.83 2.83l-2.5 2.5a2 2 0 01-2.83 0 .75.75 0 00-1.06 1.06 3.5 3.5 0 004.95 0l2.5-2.5a3.5 3.5 0 00-4.95-4.95l-1.25 1.25zm-4.69 9.64a2 2 0 010-2.83l2.5-2.5a2 2 0 012.83 0 .75.75 0 001.06-1.06 3.5 3.5 0 00-4.95 0l-2.5 2.5a3.5 3.5 0 004.95 4.95l1.25-1.25a.75.75 0 00-1.06-1.06l-1.25 1.25a2 2 0 01-2.83 0z"></path></svg></a>Overview of supported functionality</h2>
<ul>
<li>Monge-Kantorovich (Earth-Mover's) distance (<code>emd</code>, <code>emd2</code>)</li>
<li>Sinkhorn algorithm for entropy-regularised optimal transport (<code>sinkhorn</code>, <code>sinkhorn2</code>)</li>
<li>Log-stabilized Sinkhorn algorithm (<code>sinkhorn_stabilized</code>)</li>
<li>Epsilon-scaling stabilized Sinkhorn algorithm (<code>sinkhorn_stabilized_epsscaling</code>)</li>
<li>Unbalanced Sinkhorn algorithm (<code>sinkhorn_unbalanced</code>)</li>
<li>Entropy-regularised barycenters (Sinkhorn barycenters) (<code>sinkhorn_barycenter</code>)</li>
<li>Quadratically regularised optimal transport via semismooth Newton scheme [Lorenz, 2019] (<code>quadreg</code>)</li>
</ul>
<p>See the documentation pages linked below for further information. Most calling conventions are analogous to those in the <a href="https://optimaltransport.github.io/" rel="nofollow">Python Optimal Transport</a> library, which formed the inspiration for this library.</p>
