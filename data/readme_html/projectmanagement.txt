<p dir="auto">ProjectManagement.jl is a julia package based around traditional project management.
In particular it focuses on <a href="https://en.wikipedia.org/wiki/Program_evaluation_and_review_technique" rel="nofollow">PERT charts</a>.</p>

<ul dir="auto">
<li>PERT Chart drawing</li>
<li>Critical path identification and cost computation</li>
<li>Monte Carlo duration sampling (and thus probability distributions of duration and derived statistics)</li>
</ul>
