<p>JuliaChem is an electronic structure theory program written in Julia, designed to combine
Julia's dynamic and interactive nature with its high-performance capabilities to offer users the best of both worlds for quantum chemistry computations.</p>
<h1><a id="user-content-building-juliachem-for-users" class="anchor" aria-hidden="true" href="#building-juliachem-for-users"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path fill-rule="evenodd" d="M4 9h1v1H4c-1.5 0-3-1.69-3-3.5S2.55 3 4 3h4c1.45 0 3 1.69 3 3.5 0 1.41-.91 2.72-2 3.25V8.59c.58-.45 1-1.27 1-2.09C10 5.22 8.98 4 8 4H4c-.98 0-2 1.22-2 2.5S3 9 4 9zm9-3h-1v1h1c1 0 2 1.22 2 2.5S13.98 12 13 12H9c-.98 0-2-1.22-2-2.5 0-.83.42-1.64 1-2.09V6.25c-1.09.53-2 1.84-2 3.25C6 11.31 7.55 13 9 13h4c1.45 0 3-1.69 3-3.5S14.5 6 13 6z"></path></svg></a>Building JuliaChem (for Users)</h1>
<p>If you are simply interested in using JuliaChem for your calculations, you
can build JuliaChem as follows.</p>
<p>The first step to building JuliaChem is setting up SIMINT. This is needed to
form the JuliaChem/SIMINT interface, currently called Julia Electron Repulsion
Integrals (JERI). This is done as follows:</p>
<ol>
<li>
<p>Download and install SIMINT. Note that SIMINT must be compiled with the
-DSIMINT_C_FLAGS=-fPIC flag as one of the flag options. Also note that you must
compile SIMINT yourself - the pregenerated version of SIMINT is not compatible with
JuliaChem. Since JuliaChem only supports up to L shells, though,
you can build SIMINT with only the smaller-AM shells, which greatly speeds
up the process.</p>
</li>
<li>
<p>Define the environmental variable SIMINT as the directory of your
SIMINT installation.</p>
</li>
<li>
<p>Then, the other GitHub dependencies for JuliaChem must be downloaded.
This can be done with the following Julia commands:</p>
</li>
</ol>
<p>a. using Pkg</p>
<p>b. Pkg.add(PackageSpec(url="<a href="https://github.com/davpoolechem/MATH.jl.git">https://github.com/davpoolechem/MATH.jl.git</a>"))</p>
<p>c. Pkg.add(PackageSpec(url="<a href="https://github.com/davpoolechem/JCModules.jl.git">https://github.com/davpoolechem/JCModules.jl.git</a>"))</p>
<p>Finally, JuliaChem itself can be downloaded and installed:</p>
<p>d. Pkg.add(PackageSpec(url="<a href="https://github.com/davpoolechem/JuliaChem.jl">https://github.com/davpoolechem/JuliaChem.jl</a>"))</p>
<p>This will download the JuliaChem package to your computer and build JERI.
If all goes well, this should be all that is needed. Now, you can use
JuliaChem in any script you wish simply by importing the JuliaChem module:</p>
<p>using JuliaChem</p>
<h1><a id="user-content-building-juliachem-for-developers" class="anchor" aria-hidden="true" href="#building-juliachem-for-developers"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path fill-rule="evenodd" d="M4 9h1v1H4c-1.5 0-3-1.69-3-3.5S2.55 3 4 3h4c1.45 0 3 1.69 3 3.5 0 1.41-.91 2.72-2 3.25V8.59c.58-.45 1-1.27 1-2.09C10 5.22 8.98 4 8 4H4c-.98 0-2 1.22-2 2.5S3 9 4 9zm9-3h-1v1h1c1 0 2 1.22 2 2.5S13.98 12 13 12H9c-.98 0-2-1.22-2-2.5 0-.83.42-1.64 1-2.09V6.25c-1.09.53-2 1.84-2 3.25C6 11.31 7.55 13 9 13h4c1.45 0 3-1.69 3-3.5S14.5 6 13 6z"></path></svg></a>Building JuliaChem (for Developers)</h1>
<p>If you are interested in actively developing JuliaChem, you will want to build and
install JuliaChem as follows.</p>
<ol>
<li>
<p>As with above, you will need to build and install SIMINT. This follows the
first two steps as above.</p>
</li>
<li>
<p>Git clone the JuliaChem repo into your directory and cd into it.</p>
</li>
<li>
<p>Download the necessary dependencies. This can be done by inputting the following commands into
the Julia REPL:</p>
</li>
</ol>
<p>a. using Pkg</p>
<p>b. Pkg.add(PackageSpec(url="<a href="https://github.com/davpoolechem/MATH.jl.git">https://github.com/davpoolechem/MATH.jl.git</a>"))</p>
<p>c. Pkg.add(PackageSpec(url="<a href="https://github.com/davpoolechem/JCModules.jl.git">https://github.com/davpoolechem/JCModules.jl.git</a>"))</p>
<ol start="4">
<li>Next, JuliaChem itself can be set up for development. This can be done in the Julia REPL
by first pushing the ] button on your keyboard, and then inputting the following command into
the REPL:</li>
</ol>
<p>a. dev .</p>
<ol start="5">
<li>Finally, JuliaChem needs to be built. This can be done by inputting the following
command into the Julia REPL:</li>
</ol>
<p>a. build JuliaChem</p>
<p>This will build both the JuliaChem package and JERI.
If all goes well, this should be all that is needed. Now, you can use
JuliaChem in any script you wish simply by importing the JuliaChem module:</p>
<p>using JuliaChem</p>

<p>JuliaChem uses the Documenter.jl package to allow for the generation of its
documentation. Much of the information regarding JuliChem, its usage, and its
inner workings is contained within the documentation. The documentation is
generated by going into the docs/ directory and running</p>
<p>julia make_user.jl</p>
<p>in the shell.</p>
