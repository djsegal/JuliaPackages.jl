<h2 dir="auto"><a id="user-content-quick-start" class="anchor" aria-hidden="true" href="#quick-start"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path d="m7.775 3.275 1.25-1.25a3.5 3.5 0 1 1 4.95 4.95l-2.5 2.5a3.5 3.5 0 0 1-4.95 0 .751.751 0 0 1 .018-1.042.751.751 0 0 1 1.042-.018 1.998 1.998 0 0 0 2.83 0l2.5-2.5a2.002 2.002 0 0 0-2.83-2.83l-1.25 1.25a.751.751 0 0 1-1.042-.018.751.751 0 0 1-.018-1.042Zm-4.69 9.64a1.998 1.998 0 0 0 2.83 0l1.25-1.25a.751.751 0 0 1 1.042.018.751.751 0 0 1 .018 1.042l-1.25 1.25a3.5 3.5 0 1 1-4.95-4.95l2.5-2.5a3.5 3.5 0 0 1 4.95 0 .751.751 0 0 1-.018 1.042.751.751 0 0 1-1.042.018 1.998 1.998 0 0 0-2.83 0l-2.5 2.5a1.998 1.998 0 0 0 0 2.83Z"></path></svg></a>Quick Start</h2>
<p dir="auto">Assume you want to solve the following minimization problem.</p>
<p dir="auto"><a target="_blank" rel="noopener noreferrer nofollow" href="https://raw.githubusercontent.com/jmejia8/Metaheuristics.jl/master/docs/src/figs/rastrigin.png"><img src="https://raw.githubusercontent.com/jmejia8/Metaheuristics.jl/master/docs/src/figs/rastrigin.png" alt="Rastrigin Surface" style="max-width: 100%;"></a></p>
<p dir="auto">Minimize:</p>
<p dir="auto"><math-renderer class="js-display-math" style="display: block" data-static-url="https://github.githubassets.com/static" data-run-id="f42c4d8b3ff6ce7c8e7280599e5a5d1d">$$f(x) = 10D + \sum_{i=1}^D x_i^2 - 10\cos(2\pi x_i)$$</math-renderer></p>
<p dir="auto">where <math-renderer class="js-inline-math" style="display: inline" data-static-url="https://github.githubassets.com/static" data-run-id="f42c4d8b3ff6ce7c8e7280599e5a5d1d">$x\in [-5, 5]^D$</math-renderer>, that is, each coordinate in <math-renderer class="js-inline-math" style="display: inline" data-static-url="https://github.githubassets.com/static" data-run-id="f42c4d8b3ff6ce7c8e7280599e5a5d1d">$x$</math-renderer> is between -5 and 5. Use <math-renderer class="js-inline-math" style="display: inline" data-static-url="https://github.githubassets.com/static" data-run-id="f42c4d8b3ff6ce7c8e7280599e5a5d1d">$D=10$</math-renderer>.</p>
<h3 dir="auto">
<a id="user-content-solution" class="anchor" aria-hidden="true" href="#solution"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path d="m7.775 3.275 1.25-1.25a3.5 3.5 0 1 1 4.95 4.95l-2.5 2.5a3.5 3.5 0 0 1-4.95 0 .751.751 0 0 1 .018-1.042.751.751 0 0 1 1.042-.018 1.998 1.998 0 0 0 2.83 0l2.5-2.5a2.002 2.002 0 0 0-2.83-2.83l-1.25 1.25a.751.751 0 0 1-1.042-.018.751.751 0 0 1-.018-1.042Zm-4.69 9.64a1.998 1.998 0 0 0 2.83 0l1.25-1.25a.751.751 0 0 1 1.042.018.751.751 0 0 1 .018 1.042l-1.25 1.25a3.5 3.5 0 1 1-4.95-4.95l2.5-2.5a3.5 3.5 0 0 1 4.95 0 .751.751 0 0 1-.018 1.042.751.751 0 0 1-1.042.018 1.998 1.998 0 0 0-2.83 0l-2.5 2.5a1.998 1.998 0 0 0 0 2.83Z"></path></svg></a>Solution</h3>
<p dir="auto">Firstly, import the Metaheuristics package:</p>
<div class="highlight highlight-source-julia notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="using Metaheuristics"><pre><span class="pl-k">using</span> Metaheuristics</pre></div>
<p dir="auto">Code the objective function:</p>
<div class="highlight highlight-source-julia notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="f(x) = 10length(x) + sum( x.^2 - 10cos.(2π*x)  )"><pre><span class="pl-en">f</span>(x) <span class="pl-k">=</span> <span class="pl-c1">10</span><span class="pl-c1">length</span>(x) <span class="pl-k">+</span> <span class="pl-c1">sum</span>( x<span class="pl-k">.^</span><span class="pl-c1">2</span> <span class="pl-k">-</span> <span class="pl-c1">10</span><span class="pl-c1">cos</span>.(<span class="pl-c1">2</span>π<span class="pl-k">*</span>x)  )</pre></div>
<p dir="auto">Instantiate the bounds:</p>
<div class="highlight highlight-source-julia notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="D = 10
bounds = boxconstraints(lb = -5ones(D), ub = 5ones(D))"><pre>D <span class="pl-k">=</span> <span class="pl-c1">10</span>
bounds <span class="pl-k">=</span> <span class="pl-c1">boxconstraints</span>(lb <span class="pl-k">=</span> <span class="pl-k">-</span><span class="pl-c1">5</span><span class="pl-c1">ones</span>(D), ub <span class="pl-k">=</span> <span class="pl-c1">5</span><span class="pl-c1">ones</span>(D))</pre></div>
<p dir="auto">Also, <code>bounds</code> can be a <math-renderer class="js-inline-math" style="display: inline" data-static-url="https://github.githubassets.com/static" data-run-id="f42c4d8b3ff6ce7c8e7280599e5a5d1d">$2\times 10$</math-renderer> <code>Matrix</code> where the first row corresponds to the
lower bounds whilst the second row corresponds to the upper bounds.</p>
<p dir="auto">Approximate the optimum using the function <code>optimize</code>.</p>
<div class="highlight highlight-source-julia notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="result = optimize(f, bounds)"><pre>result <span class="pl-k">=</span> <span class="pl-c1">optimize</span>(f, bounds)</pre></div>
<p dir="auto">Optimize returns a <code>State</code> datatype which contains some information about the approximation.
For instance, you may use mainly two functions to obtain such an approximation.</p>
<div class="highlight highlight-source-julia notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="@show minimum(result)
@show minimizer(result)"><pre><span class="pl-c1">@show</span> <span class="pl-c1">minimum</span>(result)
<span class="pl-c1">@show</span> <span class="pl-c1">minimizer</span>(result)</pre></div>
