<h2><a id="user-content-algorithms" class="anchor" aria-hidden="true" href="#algorithms"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path fill-rule="evenodd" d="M7.775 3.275a.75.75 0 001.06 1.06l1.25-1.25a2 2 0 112.83 2.83l-2.5 2.5a2 2 0 01-2.83 0 .75.75 0 00-1.06 1.06 3.5 3.5 0 004.95 0l2.5-2.5a3.5 3.5 0 00-4.95-4.95l-1.25 1.25zm-4.69 9.64a2 2 0 010-2.83l2.5-2.5a2 2 0 012.83 0 .75.75 0 001.06-1.06 3.5 3.5 0 00-4.95 0l-2.5 2.5a3.5 3.5 0 004.95 4.95l1.25-1.25a.75.75 0 00-1.06-1.06l-1.25 1.25a2 2 0 01-2.83 0z"></path></svg></a>Algorithms</h2>
<ul>
<li>ECA: Evolutionary Centers Algorithm</li>
<li>DE:  Differential Evolution</li>
<li>PSO: Particle Swarm Optimization</li>
<li>ABC: Artificial Bee Colony</li>
<li>MOEA/D-DE: Multi-objective Evolutionary Algorithm based on Decomposition</li>
<li>GSA: Gravitational Search Algorithm</li>
<li>SA:  Simulated Annealing</li>
<li>WOA: Whale Optimization Algorithm</li>
<li>NSGA-II:  A fast and elitist multi-objective genetic algorithm: NSGA-II</li>
<li>NSGA-III: Evolutionary Many-Objective Optimization Algorithm Using Reference-Point-Based
Nondominated Sorting Approach</li>
</ul>

<h3><a id="user-content-solution" class="anchor" aria-hidden="true" href="#solution"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path fill-rule="evenodd" d="M7.775 3.275a.75.75 0 001.06 1.06l1.25-1.25a2 2 0 112.83 2.83l-2.5 2.5a2 2 0 01-2.83 0 .75.75 0 00-1.06 1.06 3.5 3.5 0 004.95 0l2.5-2.5a3.5 3.5 0 00-4.95-4.95l-1.25 1.25zm-4.69 9.64a2 2 0 010-2.83l2.5-2.5a2 2 0 012.83 0 .75.75 0 001.06-1.06 3.5 3.5 0 00-4.95 0l-2.5 2.5a3.5 3.5 0 004.95 4.95l1.25-1.25a.75.75 0 00-1.06-1.06l-1.25 1.25a2 2 0 01-2.83 0z"></path></svg></a>Solution</h3>
<p>Firstly, import the Metaheuristics package:</p>
<div class="highlight highlight-source-julia position-relative" data-snippet-clipboard-copy-content="using Metaheuristics
"><pre><span class="pl-k">using</span> Metaheuristics</pre></div>
<p>Code the objective function:</p>
<div class="highlight highlight-source-julia position-relative" data-snippet-clipboard-copy-content="f(x) = 10length(x) + sum( x.^2 - 10cos.(2π*x)  )
"><pre><span class="pl-en">f</span>(x) <span class="pl-k">=</span> <span class="pl-c1">10</span><span class="pl-c1">length</span>(x) <span class="pl-k">+</span> <span class="pl-c1">sum</span>( x<span class="pl-k">.^</span><span class="pl-c1">2</span> <span class="pl-k">-</span> <span class="pl-c1">10</span><span class="pl-c1">cos</span>.(<span class="pl-c1">2</span>π<span class="pl-k">*</span>x)  )</pre></div>
<p>Instantiate the bounds, note that <code>bounds</code> should be a $2\times 10$ <code>Matrix</code> where
the first row corresponds to the lower bounds whilst the second row corresponds to the
upper bounds.</p>
<div class="highlight highlight-source-julia position-relative" data-snippet-clipboard-copy-content="D = 10
bounds = [-5ones(D) 5ones(D)]'
"><pre>D <span class="pl-k">=</span> <span class="pl-c1">10</span>
bounds <span class="pl-k">=</span> [<span class="pl-k">-</span><span class="pl-c1">5</span><span class="pl-c1">ones</span>(D) <span class="pl-c1">5</span><span class="pl-c1">ones</span>(D)]'</pre></div>
<p>Approximate the optimum using the function <code>optimize</code>.</p>
<div class="highlight highlight-source-julia position-relative" data-snippet-clipboard-copy-content="result = optimize(f, bounds)
"><pre>result <span class="pl-k">=</span> <span class="pl-c1">optimize</span>(f, bounds)</pre></div>
<p>Optimize returns a <code>State</code> datatype which contains some information about the approximation.
For instance, you may use mainly two functions to obtain such approximation.</p>
<div class="highlight highlight-source-julia position-relative" data-snippet-clipboard-copy-content="@show minimum(result)
@show minimizer(result)
"><pre><span class="pl-c1">@show</span> <span class="pl-c1">minimum</span>(result)
<span class="pl-c1">@show</span> <span class="pl-c1">minimizer</span>(result)</pre></div>
