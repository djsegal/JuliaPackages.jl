<p>McCormick.jl is a component package in the EAGO ecosystem and is reexported by <a href="https://github.com/PSORLab/EAGO.jl">EAGO.jl</a>. It contains a library of forward nonsmooth and differentiable McCormick operators. Documentation for this is included in the <a href="https://github.com/PSORLab/EAGO.jl">EAGO.jl</a> package and additional usage examples are included <a href="https://github.com/PSORLab/EAGO-notebooks">EAGO-notebooks</a> in the form of Jupyter notebooks.</p>
