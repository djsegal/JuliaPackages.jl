<p dir="auto">This package contains functionality to represent, manipulate and optimize algorithms for matrix functions using directed acyclic graphs. It also contains code generate features for other languages (Julia, MATLAB, C/BLAS).</p>
<ul dir="auto">
<li>Features and examples are described in the journal article <a href="https://doi.org/10.1145/3568991" rel="nofollow">Computational graphs for matrix functions</a> (a preprint of the accepted version is <a href="https://arxiv.org/abs/2107.12198" rel="nofollow">available on ArXiv</a>).</li>
<li>Functions and usage is described in the <a href="https://matrixfunctions.github.io/GraphMatFun.jl/dev/" rel="nofollow">online package documentation</a>.</li>
<li>Data files for various matrix functions are available in <a href="https://github.com/matrixfunctions/GraphMatFunData">the package GraphMatFunData</a>.</li>
</ul>
