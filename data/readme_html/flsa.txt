<p dir="auto">Computing a graph induced <em>Fused LASSO Signal Approximator</em>.
You can use it to denoise data.
The package includes some utility methods to assess the algorithms and
apply it to <em>images</em> as well es <em>ion-mobilty spectrometry</em> (IMS) data sets.</p>
