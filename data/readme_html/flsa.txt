<p>Computing a graph induced <em>Fused LASSO Signal Approximator</em>.
You can use it to denoise data.
The package includes some utility methods to assess the algorithms and
apply it to <em>images</em> as well es <em>ion-mobilty spectrometry</em> (IMS) data sets.</p>

(see <a href="/generate_hdf5.py">generate_hdf5.py</a> for a working python example):</p>
<pre><code>             1 2 3 ... n
nodes/input
     /weight

             1 2 3 ... m
edges/head
     /tail
     /weight

algorithm/@name
         /@param1
         /@param2
</code></pre>
