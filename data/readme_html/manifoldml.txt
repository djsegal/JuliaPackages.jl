<p>The Package <code>ManifoldML.jl</code> provides methods to do Machine Learning methods on Riemannian
manifolds. It is based on <a href="https://juliamanifolds.github.io/Manifolds.jl/stable/interface.html" rel="nofollow"><code>ManifoldsBase</code></a>
in order to work with all manifolds from <a href="https://juliamanifolds.github.io/Manifolds.jl/stable/index.html" rel="nofollow"><code>Manifolds.jl</code></a>.</p>

<li><a href="https://marco-congedo.github.io/PosDefManifoldML.jl/dev/" rel="nofollow"><code>PosDefManifoldML.jl</code></a> for the specific case on symmetric positive definite matrices.</li>
</ul>
