<div id="user-content-preamble" dir="auto">
<div dir="auto">
<div dir="auto">
<p dir="auto">This package provides recipes for <a href="https://github.com/JuliaPlots/Plots.jl">Plots.jl</a> for plotting contour lines and pseudocolor plots on unstructured triangular meshes (<code>tricontour</code> and <code>tripcolor</code>).
The mesh itself can be shown with <code>trimesh</code>.
The contour line computation and pseudocolor rasterization is performed with <a href="https://github.com/pazner/TriplotBase.jl">TriplotBase.jl</a>.</p>
</div>
<div dir="auto">
<p dir="auto">Filled contour plots are not currently supported (cf. <a href="https://github.com/JuliaPlots/Plots.jl/issues/972" data-hovercard-type="issue" data-hovercard-url="/JuliaPlots/Plots.jl/issues/972/hovercard">this issue</a> in Plots.jl).</p>
</div>
</div>
</div>

<div dir="auto">
<div dir="auto">
<p dir="auto">The main functionality of TriplotRecipes.jl is provided in the following functions:</p>
</div>
<div dir="auto">
<ul dir="auto">
<li>
<p dir="auto"><code>tricontour</code></p>
</li>
<li>
<p dir="auto"><code>tripcolor</code> (and the closely related <code>dgtripcolor</code>)</p>
</li>
<li>
<p dir="auto"><code>trimesh</code></p>
</li>
</ul>
</div>
<div dir="auto">
<p dir="auto">The functions <code>tricontour</code> and <code>tripcolor</code> take arrays of vertex coordinates (<code>x</code> and <code>y</code>), vertex funciton values (<code>z</code>), and a triangulation (<code>t</code>).
<code>tricontour</code> also takes a <code>levels</code> argument, which is either an integer (number of contour lines) or a list of contour values.
The variant <code>dgtripcolor</code> is intended for drawing pseudocolor plots of discontinuous fields.
In this case, the <code>z</code> array has shape <code>(3, nt)</code>, where <code>nt</code> is the number of triangles.</p>
</div>
<div dir="auto">
<p dir="auto">The function <code>trimesh</code> takes vertex coordinates (<code>x</code> and <code>y</code>) and a triangulation (<code>t</code>) and draws the mesh.
The line color and fill color (potentially transparent) can be specified.</p>
</div>
<div dir="auto">
<p dir="auto">All of these functions accept keyword arguments for <a href="http://docs.juliaplots.org/latest/attributes/" rel="nofollow">Plots.jl attributes</a>.</p>
</div>
</div>
