<p>Archive of functions that emulate R's d-p-q-r functions for probability distributions.</p>
<p>This package will download <a href="https://github.com/staticfloat/RmathBuilder">libRmath-julia</a>
on Julia 0.6 and newer where it is no longer built as part of Julia. Note that
libRmath-julia is licensed under the GPLv2, see <a href="https://github.com/JuliaLang/Rmath-julia/blob/master/COPYING">https://github.com/JuliaLang/Rmath-julia/blob/master/COPYING</a>.
The Julia bindings here are licensed under <a href="LICENSE.md">MIT</a>.</p>
<p>If you only want to use this package for the sake of building the Rmath library, you can do</p>
<div class="highlight highlight-source-julia"><pre>Pkg<span class="pl-k">.</span><span class="pl-c1">add</span>(<span class="pl-s"><span class="pl-pds">"</span>Rmath<span class="pl-pds">"</span></span>)
