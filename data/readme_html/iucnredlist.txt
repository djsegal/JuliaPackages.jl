<p dir="auto">API Wrapper for IUCN Red List.</p>
<p dir="auto">Are you a data scientist and you are working in biodiversity? you might find this useful.</p>
<h1 dir="auto"><a id="user-content-what-this-does" class="anchor" aria-hidden="true" href="#what-this-does"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path d="m7.775 3.275 1.25-1.25a3.5 3.5 0 1 1 4.95 4.95l-2.5 2.5a3.5 3.5 0 0 1-4.95 0 .751.751 0 0 1 .018-1.042.751.751 0 0 1 1.042-.018 1.998 1.998 0 0 0 2.83 0l2.5-2.5a2.002 2.002 0 0 0-2.83-2.83l-1.25 1.25a.751.751 0 0 1-1.042-.018.751.751 0 0 1-.018-1.042Zm-4.69 9.64a1.998 1.998 0 0 0 2.83 0l1.25-1.25a.751.751 0 0 1 1.042.018.751.751 0 0 1 .018 1.042l-1.25 1.25a3.5 3.5 0 1 1-4.95-4.95l2.5-2.5a3.5 3.5 0 0 1 4.95 0 .751.751 0 0 1-.018 1.042.751.751 0 0 1-1.042.018 1.998 1.998 0 0 0-2.83 0l-2.5 2.5a1.998 1.998 0 0 0 0 2.83Z"></path></svg></a>What this does</h1>
<p dir="auto">Implements all of the functionalities of the IUCN Red List API in an easy to use way.</p>
<p dir="auto">In some cases, you can call 4 different endpoints from only 1 function, thanks to multiple dispatch.</p>
<h1 dir="auto"><a id="user-content-how-it-works" class="anchor" aria-hidden="true" href="#how-it-works"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path d="m7.775 3.275 1.25-1.25a3.5 3.5 0 1 1 4.95 4.95l-2.5 2.5a3.5 3.5 0 0 1-4.95 0 .751.751 0 0 1 .018-1.042.751.751 0 0 1 1.042-.018 1.998 1.998 0 0 0 2.83 0l2.5-2.5a2.002 2.002 0 0 0-2.83-2.83l-1.25 1.25a.751.751 0 0 1-1.042-.018.751.751 0 0 1-.018-1.042Zm-4.69 9.64a1.998 1.998 0 0 0 2.83 0l1.25-1.25a.751.751 0 0 1 1.042.018.751.751 0 0 1 .018 1.042l-1.25 1.25a3.5 3.5 0 1 1-4.95-4.95l2.5-2.5a3.5 3.5 0 0 1 4.95 0 .751.751 0 0 1-.018 1.042.751.751 0 0 1-1.042.018 1.998 1.998 0 0 0-2.83 0l-2.5 2.5a1.998 1.998 0 0 0 0 2.83Z"></path></svg></a>How it works</h1>
<p dir="auto">Set your token, then use all of the wonderful functionality.</p>
<div class="highlight highlight-source-julia notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="set_token(&quot;Your Token&quot;)"><pre><span class="pl-c1">set_token</span>(<span class="pl-s"><span class="pl-pds">"</span>Your Token<span class="pl-pds">"</span></span>)</pre></div>
<p dir="auto">(If you don't have a token, you can get it <a href="https://apiv3.iucnredlist.org/api/v3/token" rel="nofollow">here</a>)</p>
<p dir="auto">Check the multiple methods of the functions. If region is not added as an argument then it will return a global assessment.</p>
<p dir="auto">You can use ID or Name (scientific) of the species.</p>
<p dir="auto">For more details, you can check the official <a href="https://apiv3.iucnredlist.org/api/v3/docs" rel="nofollow">API Reference</a>. Nevertheless the functions are pretty well commented in the source code, so pretty much everything you need is here.</p>
