<p>Julia for handling BDF+ and EDF+ EEG and similar signal data files.</p>
<p>Heavily influenced by the C EEG library edflib.</p>
<p>License: 2-clause BSD.</p>
<p>Installation:</p>
<p>To install from a Julia REPL command line session:</p>
<pre><code>using Pkg
Pkg.add("EDFPlus"))
</code></pre>
<p>Note that the test files include a 23 mb test file. You may need to allow extra time for that to download when installing.</p>
