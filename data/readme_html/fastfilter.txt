<p dir="auto">Fast filters to check whether an element is in the set. These data structures are
probabilistic, meaning they have certain probability for false positives, but
they are never wrong when item was actually in the initial set. What is more,
space usage and speed of access is excellent for these data structures.</p>
<p dir="auto">This Julia package is based of <a href="https://github.com/FastFilter/xor_singleheader/blob/master/include/binaryfusefilter.h">binaryfusefilter.h</a>.
Currently only binary fuse filters are re-implemented, but we support UInt8, UInt16 and UInt32 fingerprints.</p>
<p dir="auto">Feel free to make a PR with other implementations.</p>
