<p dir="auto">The package <strong>SpatialDatasets.jl</strong> contians a set of popular spatial datasets for spatial data analysis.</p>
<p dir="auto">The datasets are storead as lazy Pkg Artifacts and are downloaded and installed once first used. See <a href="https://javierbarbero.github.io/SpatialDatasets.jl/dev/sdatasets/" rel="nofollow">List of Spatial Datasets</a> for a list of all available datasets and their sources.</p>
<p dir="auto">The following code loads the popular GuerryÂ´s Moral statistics of France dataset:</p>
<div class="highlight highlight-source-julia notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="using SpatialDatasets

guerry = sdataset(&quot;Guerry&quot;)"><pre><span class="pl-k">using</span> SpatialDatasets

guerry <span class="pl-k">=</span> <span class="pl-c1">sdataset</span>(<span class="pl-s"><span class="pl-pds">"</span>Guerry<span class="pl-pds">"</span></span>)</pre></div>
<p dir="auto">Spatial datasets are loaded as Julia <a href="https://dataframes.juliadata.org/stable/" rel="nofollow">DataFrames</a> and contains a column named <code>geometry</code> with the geometry information (Points or Polygons).</p>
