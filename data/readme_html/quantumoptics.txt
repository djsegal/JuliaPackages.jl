<p><strong>QuantumOptics.jl</strong> is a numerical framework written in <a href="http://julialang.org" rel="nofollow">Julia</a> that makes it easy to simulate various kinds of quantum systems. It is inspired by the <a href="http://qo.phy.auckland.ac.nz/toolbox" rel="nofollow">Quantum Optics Toolbox</a> for MATLAB and the Python framework <a href="http://qutip.org" rel="nofollow">QuTiP</a>.</p>
<p>More information, documentation and examples can be found on our website <a href="http://qojulia.org" rel="nofollow">http://qojulia.org</a>.</p>
