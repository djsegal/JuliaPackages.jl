<h3 dir="auto"><a id="user-content-implemented" class="anchor" aria-hidden="true" href="#implemented"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path d="m7.775 3.275 1.25-1.25a3.5 3.5 0 1 1 4.95 4.95l-2.5 2.5a3.5 3.5 0 0 1-4.95 0 .751.751 0 0 1 .018-1.042.751.751 0 0 1 1.042-.018 1.998 1.998 0 0 0 2.83 0l2.5-2.5a2.002 2.002 0 0 0-2.83-2.83l-1.25 1.25a.751.751 0 0 1-1.042-.018.751.751 0 0 1-.018-1.042Zm-4.69 9.64a1.998 1.998 0 0 0 2.83 0l1.25-1.25a.751.751 0 0 1 1.042.018.751.751 0 0 1 .018 1.042l-1.25 1.25a3.5 3.5 0 1 1-4.95-4.95l2.5-2.5a3.5 3.5 0 0 1 4.95 0 .751.751 0 0 1-.018 1.042.751.751 0 0 1-1.042.018 1.998 1.998 0 0 0-2.83 0l-2.5 2.5a1.998 1.998 0 0 0 0 2.83Z"></path></svg></a>Implemented</h3>
<ul dir="auto">
<li>Fast convolutions adapted to the needs of Physical Optics. Methods like <code>conv_psf</code> and <code>conv_otf</code> are wrappers for the more general <code>conv</code> and <code>conv_v_ft</code>.</li>
<li>Light propagation with Fresnel and Rayleigh-Sommerfeld (<code>propagate</code>)</li>
<li>2D jinc PSF</li>
<li>some conversion methods</li>
<li>Light propagation with Fresnel</li>
<li>Optical elements like lenses, apertures</li>
<li>some optical conversions</li>
<li>micro lenses</li>
<li>Light propagation with Fraunhofer</li>
<li>Focused and defocused PSFs (3D) with Debye integral.</li>
</ul>
