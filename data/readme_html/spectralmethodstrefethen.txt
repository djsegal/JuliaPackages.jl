<p dir="auto">This is a Julia 1.x rewrite of the codes in Trefethen's <em>Spectral Methods in MATLAB.</em> It's in alpha and the codes are not at all thoroughly checked yet, much less made beautiful.</p>
<p dir="auto">The graphics is done by <code>PyPlot</code>, which is the most MATLAB-like interface (but not the one most used in Julia).</p>

<p dir="auto">To get the package loaded, activate the dedicated environment, then enter</p>
<div class="highlight highlight-source-julia notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="using SpectralMethodsTrefethen,PyPlot

<p dir="auto">Then each of the book's program scripts can be run as functions, e.g. <code>p1()</code>, <code>p2()</code>, etc. Some of the functions have additional versions with a suffix like <code>b</code> or <code>u</code>.</p>
<p dir="auto">The book functions <code>cheb</code>, <code>chebfft</code>, <code>clencurt</code> and <code>gauss</code> are also available.</p>
