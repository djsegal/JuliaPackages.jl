<p dir="auto"><a href="https://jchitpin.github.io/MarkovWeightedEFMs.jl/dev/" rel="nofollow"><img src="https://camo.githubusercontent.com/7fcec4b2d3ab291529fce8ef6a4fcd4129a0683b2f5d5fe2f5c648f02db8b616/68747470733a2f2f696d672e736869656c64732e696f2f62616467652f646f63732d6465762d626c75652e737667" alt="Dev" data-canonical-src="https://img.shields.io/badge/docs-dev-blue.svg" style="max-width: 100%;"></a></p>
<p dir="auto">This package is used to decompose steady state metabolic fluxes onto elementary flux mode weights. Currently, this package only supports closed-loop networks of unimolecular reactions. Click the "docs" badge above to access package the documentation.</p>
