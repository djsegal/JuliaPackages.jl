<p dir="auto">A Julia Framework for Quantum Dynamics and Control.</p>
<p dir="auto">The <a href="https://github.com/JuliaQuantumControl/QuantumControl.jl"><code>QuantumControl</code></a> package collects the <a href="https://github.com/JuliaQuantumControl#packages">packages</a> in the <a href="https://github.com/JuliaQuantumControl">JuliaQuantumControl</a> organization and provides a single coherent <a href="https://juliaquantumcontrol.github.io/QuantumControl.jl/dev/api/quantum_control/#QuantumControlAPI" rel="nofollow">API</a> for solving quantum control problems. See the <a href="https://github.com/JuliaQuantumControl#readme">organization README</a> for details.</p>
<p dir="auto">The <a href="https://github.com/JuliaQuantumControl/QuantumControl.jl"><code>QuantumControl</code></a> package provides a framework for solving quantum control problems. It exposes the <a href="https://github.com/JuliaQuantumControl#packages">packages</a> in the <a href="https://github.com/JuliaQuantumControl">JuliaQuantumControl</a> organization in a single coherent <a href="https://juliaquantumcontrol.github.io/QuantumControl.jl/dev/api/quantum_control/#QuantumControlAPI" rel="nofollow">API</a>. Specifically, it subsumes the <a href="https://github.com/JuliaQuantumControl/QuantumPropagators.jl"><code>QuantumPropagators</code></a> package to simulate quantum dynamics.</p>
