<p dir="auto">It's glitching time!</p>
<p dir="auto"><code>JpegGlitcher</code> only exports the <code>glitch</code> function.
<code>glitch</code> takes an image, compresses it using the JPEG encoding, safely modifies
some bytes of the compressed version and return the decoded version.</p>
<p dir="auto">Here is a basic example, using the default parameters.</p>
<div class="highlight highlight-source-julia notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="using JpegGlitcher
using Random, FileIO, TestImages

img = testimage(&quot;mountainview&quot;)
glitch(img)"><pre><span class="pl-k">using</span> JpegGlitcher
<span class="pl-k">using</span> Random, FileIO, TestImages

img <span class="pl-k">=</span> <span class="pl-c1">testimage</span>(<span class="pl-s"><span class="pl-pds">"</span>mountainview<span class="pl-pds">"</span></span>)
<span class="pl-c1">glitch</span>(img)</pre></div>
