<p dir="auto">Graph layout and visualization algorithms based on <a href="https://github.com/dcjones/Compose.jl">Compose.jl</a> and inspired by <a href="https://github.com/IainNZ/GraphLayout.jl">GraphLayout.jl</a>.</p>
<p dir="auto">The <code>spring_layout</code> and <code>stressmajorize_layout</code> function are copy from <a href="https://github.com/IainNZ">IainNZ</a>'s <a href="https://github.com/IainNZ/GraphLayout.jl">GraphLayout.jl</a>.</p>
<p dir="auto">Other layout algorithms are wrapped from <a href="https://github.com/networkx/networkx">NetworkX</a>.</p>
<p dir="auto"><code>gadfly.js</code> is copied from <a href="https://github.com/dcjones/Gadfly.jl">Gadfly.jl</a></p>

<ul dir="auto">
<li><code>layout</code> Layout algorithm: <code>random_layout</code>, <code>circular_layout</code>, <code>spring_layout</code>, <code>shell_layout</code>, <code>stressmajorize_layout</code>, <code>spectral_layout</code>. Default: <code>spring_layout</code></li>
<li><code>NODESIZE</code> Max size for the nodes. Default: <code>3.0/sqrt(N)</code></li>
<li><code>nodesize</code> Relative size for the nodes, can be a Vector. Default: <code>1.0</code></li>
<li><code>nodelabel</code> Labels for the vertices, a Vector or nothing. Default: <code>nothing</code></li>
<li><code>nodelabelc</code> Color for the node labels, can be a Vector. Default: <code>colorant"black"</code></li>
<li><code>nodelabeldist</code> Distances for the node labels from center of nodes. Default: <code>0.0</code></li>
<li><code>nodelabelangleoffset</code> Angle offset for the node labels. Default: <code>π/4.0</code></li>
<li><code>NODELABELSIZE</code> Largest fontsize for the vertice labels. Default: <code>4.0</code></li>
<li><code>nodelabelsize</code> Relative fontsize for the vertice labels, can be a Vector. Default: <code>1.0</code></li>
<li><code>nodefillc</code> Color to fill the nodes with, can be a Vector. Default: <code>colorant"turquoise"</code></li>
<li><code>nodestrokec</code> Color for the nodes stroke, can be a Vector. Default: <code>nothing</code></li>
<li><code>nodestrokelw</code> Line width for the nodes stroke, can be a Vector. Default: <code>0.0</code></li>
<li><code>edgelabel</code> Labels for the edges, a Vector or nothing. Default: <code>[]</code></li>
<li><code>edgelabelc</code> Color for the edge labels, can be a Vector. Default: <code>colorant"black"</code></li>
<li><code>edgelabeldistx, edgelabeldisty</code> Distance for the edge label from center of edge. Default: <code>0.0</code></li>
<li><code>EDGELABELSIZE</code> Largest fontsize for the edge labels. Default: <code>4.0</code></li>
<li><code>edgelabelsize</code> Relative fontsize for the edge labels, can be a Vector. Default: <code>1.0</code></li>
<li><code>EDGELINEWIDTH</code> Max line width for the edges. Default: <code>0.25/sqrt(N)</code></li>
<li><code>edgelinewidth</code> Relative line width for the edges, can be a Vector. Default: <code>1.0</code></li>
<li><code>edgestrokec</code> Color for the edge strokes, can be a Vector. Default: <code>colorant"lightgray"</code></li>
<li><code>arrowlengthfrac</code> Fraction of line length to use for arrows. Equal to 0 for undirected graphs. Default: <code>0.1</code> for the directed graphs</li>
<li><code>arrowangleoffset</code> Angular width in radians for the arrows. Default: <code>π/9 (20 degrees)</code></li>
<li><code>linetype</code> Type of line used for edges ("straight", "curve"). Default: "straight"</li>
<li><code>outangle</code> Angular width in radians for the edges (only used if <code>linetype = "curve</code>). Default: <code>π/5 (36 degrees)</code></li>
</ul>
