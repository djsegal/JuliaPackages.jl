<p>PackageCompilerX is a Julia package with two main purposes:</p>
<ol>
<li>
<p>Creating custom sysimages for reduced latency when working locally with packages that has a high startup time.</p>
</li>
<li>
<p>Creating "apps" which are a bundle of files including an executable that can be sent and run on other machines without Julia being installed on that machine.</p>
</li>
</ol>
<p>For installation and usage instructions, see the <a href="https://JuliaComputing.github.io/PackageCompilerX.jl/dev" rel="nofollow">documentation</a>.</p>
