<p dir="auto">The package is a Julia interface to the test suite TestU01 of Pierre l'Ecuyer. All the tests included in the SmallCrush and BigCrush test batteries can be called as Julia functions.
The first argument to the test function must be either</p>
<ul dir="auto">
<li>
<p dir="auto">a function without arguments, which must return a <code>Float64</code> between zero and one, or</p>
</li>
<li>
<p dir="auto">a wrapped <code>AbstractRNG</code> obtained via the function <code>wrap(rng, T)</code>
where <code>T</code> is the type of the variates produced by <code>rng</code> that one
wants tested (currently <code>T</code> must be one of the standard
finite-precision Julia <code>Integer</code> or <code>FloatingPoint</code> types).</p>
</li>
</ul>
<p dir="auto">The output from the test is a p-value.
The package also includes the Small- and the BigCrush batteries. Some examples:</p>
