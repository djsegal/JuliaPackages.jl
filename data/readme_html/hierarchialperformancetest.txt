<p>This is a Julia implementation of the Non-parametric Hierarchial Performance Testing statistical technique described in <a href="https://parsec.cs.princeton.edu/publications/chen14ieeetc.pdf" rel="nofollow">Statistical Performance Comparisons of Computers</a>. It is meant to compare the performance of different computers on a common set of benchmarks with high statistical confidence, rather than .</p>
<p>For more information, check the <a href="http://novel.ict.ac.cn/tchen/hpt/" rel="nofollow">official website</a> maintained by one of the technique's creators.</p>
<h2><a id="user-content-example-usage" class="anchor" aria-hidden="true" href="#example-usage"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path fill-rule="evenodd" d="M7.775 3.275a.75.75 0 001.06 1.06l1.25-1.25a2 2 0 112.83 2.83l-2.5 2.5a2 2 0 01-2.83 0 .75.75 0 00-1.06 1.06 3.5 3.5 0 004.95 0l2.5-2.5a3.5 3.5 0 00-4.95-4.95l-1.25 1.25zm-4.69 9.64a2 2 0 010-2.83l2.5-2.5a2 2 0 012.83 0 .75.75 0 001.06-1.06 3.5 3.5 0 00-4.95 0l-2.5 2.5a3.5 3.5 0 004.95 4.95l1.25-1.25a.75.75 0 00-1.06-1.06l-1.25 1.25a2 2 0 01-2.83 0z"></path></svg></a>Example usage</h2>
<p>Each set of benchmark results should be provided as a 2D matrix where each row is comprised of the same benchmark's results.</p>
