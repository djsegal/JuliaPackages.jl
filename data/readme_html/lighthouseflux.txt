<p>LighthouseFlux provides a <code>FluxClassifier</code> wrapper that implements the <a href="https://github.com/beacon-biosignals/Lighthouse.jl">Lighthouse</a> <code>AbstractClassifier</code> interface, thus enabling Flux models to easily utilize Lighthouse's training/testing harness. Assuming your model obeys normal Flux model conventions, hooking it up to LighthouseFlux generally only requires a single method overload (<code>LighthouseFlux.loss</code>).</p>
<p>See this package's tests for example usage.</p>
