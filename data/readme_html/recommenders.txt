<p dir="auto">This package aims to provide light-weight recommendation models, mainly for implicit feedback data. We want to provide</p>
<ul dir="auto">
<li>consistent interface for model training and inference</li>
<li>flexibility for input data with <a href="https://github.com/JuliaData/Tables.jl"><code>Tables.jl</code></a> package, which offers simple, but powerful abstract interface for tabular data</li>
<li>robust baseline metrics for classic datasets. The comparison of advanced recommendation models to these baselines turns out to be challenge.</li>
</ul>
<p dir="auto">See <a href="https://yng87.github.io/Recommenders.jl/stable/getting_started/" rel="nofollow">Getting started</a> and <a href="https://github.com/yng87/Recommenders.jl/tree/master/examples">examples</a>.</p>
