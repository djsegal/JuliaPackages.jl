<p dir="auto">Rigorous moment propagation with partial information about <a href="https://en.wikipedia.org/wiki/Moment_(mathematics)" rel="nofollow">moments</a> and <a href="https://en.wikipedia.org/wiki/Copula_(probability_theory)" rel="nofollow">dependencies</a> in Julia.</p>
<p dir="auto">Generalisation of First Order Error Propagation with:</p>
<ul dir="auto">
<li>Independence between variables need not be assumed (but can be)</li>
<li>Moments may be partially known (intervals)</li>
<li>Input distribution assumptions no longer necessary</li>
<li>Distributional information can be obtained from moment and range information in non-linear models</li>
</ul>
<p dir="auto">May be viewed as a form of <em>distribution-free risk analysis</em>.</p>
<p dir="auto"><a href="https://www.sciencedirect.com/science/article/pii/S0888613X22000512?via%3Dihub" rel="nofollow">Open access IJAR paper</a></p>
<p dir="auto"><a href="http://ww2new.unime.it/REC2021/presentations/REC2021-37_Presentation.pdf" rel="nofollow">REC21 Presentation</a></p>
<h2 dir="auto"><a id="user-content-features" class="anchor" aria-hidden="true" href="#features"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path d="m7.775 3.275 1.25-1.25a3.5 3.5 0 1 1 4.95 4.95l-2.5 2.5a3.5 3.5 0 0 1-4.95 0 .751.751 0 0 1 .018-1.042.751.751 0 0 1 1.042-.018 1.998 1.998 0 0 0 2.83 0l2.5-2.5a2.002 2.002 0 0 0-2.83-2.83l-1.25 1.25a.751.751 0 0 1-1.042-.018.751.751 0 0 1-.018-1.042Zm-4.69 9.64a1.998 1.998 0 0 0 2.83 0l1.25-1.25a.751.751 0 0 1 1.042.018.751.751 0 0 1 .018 1.042l-1.25 1.25a3.5 3.5 0 1 1-4.95-4.95l2.5-2.5a3.5 3.5 0 0 1 4.95 0 .751.751 0 0 1-.018 1.042.751.751 0 0 1-1.042.018 1.998 1.998 0 0 0-2.83 0l-2.5 2.5a1.998 1.998 0 0 0 0 2.83Z"></path></svg></a>Features</h2>
<ul dir="auto">
<li>9 univariate transformations: scalar multiplication, scalar translation, exp, log, ln, 1/x, x^2, sqrt and |x|</li>
<li>7 binary operations: +, -, *, /, ^, min, max</li>
<li>Independence and no dependence assumptions (Fréchet)</li>
<li>Moments consitancy checking</li>
<li>Bounding <a href="https://en.wikipedia.org/wiki/Probability_box" rel="nofollow">p-box</a> from Moments</li>
</ul>
