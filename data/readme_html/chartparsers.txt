<p>This package contains a simple implementation of a chart parser for context-free grammars, with both bottom-up and top-down prediction rules available.</p>
<p>The algorithms implemented here are based on the excellent descriptions by Blackburn and Striegnitz, available online at <a href="http://cs.union.edu/~striegnk/courses/nlp-with-prolog/html/index.html" rel="nofollow">http://cs.union.edu/~striegnk/courses/nlp-with-prolog/html/index.html</a></p>
<p>This package was developed to support <a href="https://github.com/rdeits/CrypticCrosswords.jl">CrypticCrosswords.jl</a>.</p>
