<p>This package allows you to specify required keyword arguments, as suggusted in <a href="https://github.com/JuliaLang/julia/issues/5111">https://github.com/JuliaLang/julia/issues/5111</a>.</p>
<p>As of Julia version 0.7.0, this functionality is built in. From this version forward, the <code>@required_keywords</code> macro won't do anything.</p>
