<p dir="auto">This Julia language repo
provides methods
for regularized estimation of fieldmaps in MRI.</p>
<p dir="auto">Currently there are methods for B0 fieldmap estimation
from a water-only image signal model,
or from a water-fat model
with 1 or more lipid peaks.</p>

<p dir="auto">The algorithm in function <code>b0map</code>
for B0 field map estimation is based on the paper:
C Y Lin, J A Fessler,
"Efficient Regularized Field Map Estimation in 3D MRI", IEEE TCI 2020
[<a href="http://doi.org/10.1109/TCI.2020.3031082" rel="nofollow">http://doi.org/10.1109/TCI.2020.3031082</a>]
[<a href="http://arxiv.org/abs/2005.08661" rel="nofollow">http://arxiv.org/abs/2005.08661</a>]
Please cite this paper if you use this method.</p>
<p dir="auto">The internal algorithm details are a bit different
(and faster)
because here we perform coil combination
before starting the iterations,
whereas
<a href="https://github.com/ClaireYLin/regularized-field-map-estimation">the original Matlab code</a>
had loops over coils
within each iteration.</p>
