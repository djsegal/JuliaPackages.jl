<p dir="auto">This repository contains experimental code for solving linear and quadratic
programming problems using first-order methods. It provides basic utilities and
data structures for reading MPS files, rescaling, and implementing saddle-point
methods. Specialized implementations are present for Mirror Prox and
Primal-Dual Hybrid Gradient.</p>
<p dir="auto"><strong>A C++ implementation of the PDLP algorithm is included in
<a href="https://github.com/google/or-tools">OR-Tools</a>. We recommend the OR-Tools
version of PDLP to all potential users. This repository is intended exclusively
for supporting experiments and publications.</strong></p>

<p dir="auto">To solve a test instance with PDHG:</p>
<div class="highlight highlight-source-shell notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="$ julia --project=scripts scripts/solve_qp.jl \

--method pdhg --output_dir /tmp/first_order_lp_solve"><pre>$ julia --project=scripts scripts/solve_qp.jl \

<p dir="auto">This script provides an interface similar to <code>solve_qp</code> but for calling
external solvers for baselines. This script does not support quadratic
objectives.</p>
<p dir="auto">To solve a test instance with SCS's indirect mode:</p>
<div class="highlight highlight-source-shell notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="$ julia --project=scripts scripts/solve_lp_external.jl \

--solver scs-indirect --tolerance 1e-7 --output_dir /tmp/scs_solve"><pre>$ julia --project=scripts scripts/solve_lp_external.jl \
--instance_path test/trivial_lp_model.mps --iteration_limit 5000 \
--solver scs-indirect --tolerance 1e-7 --output_dir /tmp/scs_solve</pre></div>
<p dir="auto">To solve a test with HiGHS's interior-point mode:</p>
