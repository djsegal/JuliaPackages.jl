<ol dir="auto">
<li>Create a new folder in your <code>.julia</code> directory called <code>config</code></li>
<li>In the <code>.julia\config</code> folder create a new text file: <code>Right click  → New → Text Document</code></li>
<li>Inside the new text document, type:</li>
</ol>
<blockquote>
<p dir="auto">ENV["PATH_LICENSE_STRING"] = "<em>licence number provided in the above
link</em>"</p>
</blockquote>
<ol dir="auto">
<li>In the text document, go <code>File → Save as</code></li>
<li>In the dropdown <code>Save as type</code> option at the bottom of the pop-up window select <code>All files</code></li>
<li>In the field <code>File name</code> write <code>startup.jl</code></li>
<li>Press <code>Save</code></li>
</ol>
