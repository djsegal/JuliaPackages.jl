<p>This package is a wrapper around the <em>new</em> API for the mangal ecological
interactions database. It uses <a href="https://julialang.org/" rel="nofollow">Julia 1.1</a> to provide a programmatic
interface to <em>read</em> the data. Development of this package and the underlying
database was funded by the <a href="https://www.innovation.ca/" rel="nofollow">Canadian Foundation for Innovation</a> and
<a href="http://www.nserc-crsng.gc.ca/index_eng.asp" rel="nofollow">NSERC</a>.</p>
