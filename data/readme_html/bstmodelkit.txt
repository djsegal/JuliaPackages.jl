<p dir="auto"><code>BSTModelKit.jl</code> is a package for constructing, solving, and analyzing Biochemical Systems Theory (BST) models of biochemical networks written in the <a href="https://julialang.org" rel="nofollow">Julia</a> programming language.</p>

<p dir="auto"><code>BSTModelKit.jl</code> can be installed, updated, or removed using the <a href="https://docs.julialang.org/en/v1/stdlib/Pkg/" rel="nofollow">Julia package management system</a>. To access the package management interface, open the <a href="https://docs.julialang.org/en/v1/stdlib/REPL/" rel="nofollow">Julia REPL</a>, and start the package mode by pressing <code>]</code>.
While in package mode, to install <code>BSTModelKit.jl</code>, issue the command:</p>
<div class="snippet-clipboard-content notranslate position-relative overflow-auto" data-snippet-clipboard-copy-content="(@v1.8.x) pkg&gt; add BSTModelKit"><pre class="notranslate"><code>(@v1.8.x) pkg&gt; add BSTModelKit
</code></pre></div>
<p dir="auto">To use <code>BSTModelKit.jl</code> in your projects, issue the command:</p>
<div class="snippet-clipboard-content notranslate position-relative overflow-auto" data-snippet-clipboard-copy-content="julia&gt; using BSTModelKit"><pre class="notranslate"><code>julia&gt; using BSTModelKit
</code></pre></div>
