<p dir="auto">Inverse Ising inference for General Boltzmann Machines [GBM].</p>
<h2 dir="auto"><a id="user-content-detail" class="anchor" aria-hidden="true" href="#detail"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path d="m7.775 3.275 1.25-1.25a3.5 3.5 0 1 1 4.95 4.95l-2.5 2.5a3.5 3.5 0 0 1-4.95 0 .751.751 0 0 1 .018-1.042.751.751 0 0 1 1.042-.018 1.998 1.998 0 0 0 2.83 0l2.5-2.5a2.002 2.002 0 0 0-2.83-2.83l-1.25 1.25a.751.751 0 0 1-1.042-.018.751.751 0 0 1-.018-1.042Zm-4.69 9.64a1.998 1.998 0 0 0 2.83 0l1.25-1.25a.751.751 0 0 1 1.042.018.751.751 0 0 1 .018 1.042l-1.25 1.25a3.5 3.5 0 1 1-4.95-4.95l2.5-2.5a3.5 3.5 0 0 1 4.95 0 .751.751 0 0 1-.018 1.042.751.751 0 0 1-1.042.018 1.998 1.998 0 0 0-2.83 0l-2.5 2.5a1.998 1.998 0 0 0 0 2.83Z"></path></svg></a>Detail</h2>
<p dir="auto">The inverse Ising problem is to estimate the spin-spin interaction from the spin configurations. <em>InverseIsing.jl</em>, based on julia language, contains a machine for generating spin configurations (Simulated Annealing Machine:SA) and a solver for the inverse ising problem.</p>

Simulated Annealing Machine</h3>
<p dir="auto">For ising case, input the magnetic field <code>h</code> and interaction <code>J</code>.</p>
<div class="snippet-clipboard-content notranslate position-relative overflow-auto" data-snippet-clipboard-copy-content="julia&gt; using InverseIsing

julia&gt; h = Dict(1 =&gt; -1) # Longitudinal magnetic field

julia&gt; J = Dict((1, 2) =&gt; 1) # Ferromagnetic-bond

julia&gt; result = anneal(h, J)"><pre class="notranslate"><code>julia&gt; using InverseIsing

julia&gt; h = Dict(1 =&gt; -1) # Longitudinal magnetic field

julia&gt; J = Dict((1, 2) =&gt; 1) # Ferromagnetic-bond

julia&gt; result = anneal(h, J)
</code></pre></div>
<p dir="auto">The result of the annealing is output to the response structure.</p>
<div class="snippet-clipboard-content notranslate position-relative overflow-auto" data-snippet-clipboard-copy-content="julia&gt; result.states

 [-1, -1]"><pre class="notranslate"><code>julia&gt; result.states

 [-1, -1]
</code></pre></div>

Inverse Ising Estimater</h3>
<p dir="auto">Train GBM parameters:</p>
<div class="snippet-clipboard-content notranslate position-relative overflow-auto" data-snippet-clipboard-copy-content="julia&gt; using InverseIsing

julia&gt; samples = [1 -1 -1;] # Spin configuration sample.

julia&gt; model = GBM(3) # Set the number of units.

julia&gt; fit(model, samples)"><pre class="notranslate"><code>julia&gt; using InverseIsing

julia&gt; samples = [1 -1 -1;] # Spin configuration sample.

julia&gt; model = GBM(3) # Set the number of units.

julia&gt; fit(model, samples)
</code></pre></div>
<p dir="auto">After model is fitted, you can estimate GBM parameters known as weights:</p>
<div class="snippet-clipboard-content notranslate position-relative overflow-auto" data-snippet-clipboard-copy-content="julia&gt; W = infer(model)
3×3 Array{Int64,2}:
  0  -1  -1
 -1   0   1
 -1   1   0"><pre class="notranslate"><code>julia&gt; W = infer(model)
3×3 Array{Int64,2}:
  0  -1  -1
 -1   0   1
 -1   1   0
</code></pre></div>
<p dir="auto">The output can be transformed to make the display easier to read:</p>
<div class="snippet-clipboard-content notranslate position-relative overflow-auto" data-snippet-clipboard-copy-content="julia&gt; decode(W)
OrderedCollections.OrderedDict{Tuple{Int64,Int64},Int64} with 3 entries:
  (1, 2) =&gt; -1
  (1, 3) =&gt; -1
  (2, 3) =&gt; 1"><pre class="notranslate"><code>julia&gt; decode(W)
OrderedCollections.OrderedDict{Tuple{Int64,Int64},Int64} with 3 entries:
  (1, 2) =&gt; -1
  (1, 3) =&gt; -1
  (2, 3) =&gt; 1
</code></pre></div>
<p dir="auto">The above example means that the interaction between (1, 2) and (1, 3) is antiferromagnetic bond and only (2, 3) is ferromagnetic bond.</p>
