<ul dir="auto">
<li>
<p dir="auto"><code>FermionXYChain</code>: The quantum XY model as a chain of fermions.</p>
<p dir="auto"><strong>Arguments:</strong></p>
<ul dir="auto">
<li>
<code>L::Int</code>: The chain length.</li>
<li>
<code>J::Real</code>: The coupling constant.</li>
<li>
<code>h::Real</code>: The magnetic field.</li>
<li>
<code>gamma::Real</code>: The anisotropy constant.</li>
<li>
<code>start::Symbol</code>: The initial state. Can be one of <code>:rand</code>, <code>:vacuum</code>, <code>:filled</code>
</li>
<li>
<code>parity::Int=-1</code>: Can be -1 or 1.</li>
</ul>
</li>
<li>
<p dir="auto"><code>FermionXXChain</code>: The quantum XY model for <math-renderer class="js-inline-math" style="display: inline" data-static-url="https://github.githubassets.com/static" data-run-id="ee4ffa4de73009a3f9700b1cb632027e">$\gamma=0$</math-renderer>.</p>
</li>
<li>
<p dir="auto"><code>FermionIsingChain</code>: The quantum XY model for <math-renderer class="js-inline-math" style="display: inline" data-static-url="https://github.githubassets.com/static" data-run-id="ee4ffa4de73009a3f9700b1cb632027e">$\gamma=1$</math-renderer>.</p>
</li>
</ul>

<ul dir="auto">
<li>
<p dir="auto"><code>FermionBasis</code>: An Iterator over the binary basis of fermion chains. It accepts the following arguments:</p>
<p dir="auto"><strong>Arguments:</strong></p>
<ul dir="auto">
<li>
<code>n_sites::Int</code>: The number of sites in fermion chain.</li>
<li>
<code>states::Tuple=(-1, 1)</code>: A keyword argument taking the integer representation of filled and unfilled sites.</li>
</ul>
</li>
</ul>
<h2 dir="auto">
<a id="user-content-correlations-and-probabilities" class="anchor" aria-hidden="true" href="#correlations-and-probabilities"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path d="m7.775 3.275 1.25-1.25a3.5 3.5 0 1 1 4.95 4.95l-2.5 2.5a3.5 3.5 0 0 1-4.95 0 .751.751 0 0 1 .018-1.042.751.751 0 0 1 1.042-.018 1.998 1.998 0 0 0 2.83 0l2.5-2.5a2.002 2.002 0 0 0-2.83-2.83l-1.25 1.25a.751.751 0 0 1-1.042-.018.751.751 0 0 1-.018-1.042Zm-4.69 9.64a1.998 1.998 0 0 0 2.83 0l1.25-1.25a.751.751 0 0 1 1.042.018.751.751 0 0 1 .018 1.042l-1.25 1.25a3.5 3.5 0 1 1-4.95-4.95l2.5-2.5a3.5 3.5 0 0 1 4.95 0 .751.751 0 0 1-.018 1.042.751.751 0 0 1-1.042.018 1.998 1.998 0 0 0-2.83 0l-2.5 2.5a1.998 1.998 0 0 0 0 2.83Z"></path></svg></a>Correlations and Probabilities</h2>
<ul dir="auto">
<li>
<p dir="auto"><code>correlation_matrix</code>: Calculates the correlation matrix of the chain given by <math-renderer class="js-inline-math" style="display: inline" data-static-url="https://github.githubassets.com/static" data-run-id="ee4ffa4de73009a3f9700b1cb632027e">$G_{ij} = \langle a_i^\dagger a_j\rangle$</math-renderer>. Has two methods, one takes a <code>FermionXYChain</code> as argument. Other takes the agruments:</p>
<ul dir="auto">
<li>
<code>L::Int</code>: The chain length.</li>
<li>
<code>J::Real</code>: The coupling constant.</li>
<li>
<code>h::Real</code>: The magnetic field.</li>
<li>
<code>gamma::Real</code>: The anisotropy constant.</li>
<li>
<code>parity::Int=-1</code>: Can be -1 or 1.</li>
<li>
<code>float_type::Type=Float64</code>: The floating point type (default is <code>Float64</code>, can be <code>BigFloat</code> for greater precision).</li>
</ul>
</li>
<li>
<p dir="auto"><code>probability_matrix</code>: Calculates the probability matrix of the chain. The probability of the particular configuration is then given by <math-renderer class="js-inline-math" style="display: inline" data-static-url="https://github.githubassets.com/static" data-run-id="ee4ffa4de73009a3f9700b1cb632027e">$\det(P)$</math-renderer> where <math-renderer class="js-inline-math" style="display: inline" data-static-url="https://github.githubassets.com/static" data-run-id="ee4ffa4de73009a3f9700b1cb632027e">$P$</math-renderer> is the said matrix. Has two methods, one takes a <code>FermionXYChain</code> as argument. Other takes the agruments:</p>
<ul dir="auto">
<li>
<code>sites::Vector{Int}</code>: The sites of the chain having value -1 for no fermion and 1 for a fermion.</li>
<li>
<code>L::Int</code>: The chain length.</li>
<li>
<code>J::Real</code>: The coupling constant.</li>
<li>
<code>h::Real</code>: The magnetic field.</li>
<li>
<code>gamma::Real</code>: The anisotropy constant.</li>
<li>
<code>parity::Int=-1</code>: Can be -1 or 1.</li>
<li>
<code>float_type::Type=Float64</code>: The floating point type (default is <code>Float64</code>, can be <code>BigFloat</code> for greater precision).</li>
</ul>
</li>
</ul>
<h2 dir="auto">
<a id="user-content-monte-carlo-simulation" class="anchor" aria-hidden="true" href="#monte-carlo-simulation"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path d="m7.775 3.275 1.25-1.25a3.5 3.5 0 1 1 4.95 4.95l-2.5 2.5a3.5 3.5 0 0 1-4.95 0 .751.751 0 0 1 .018-1.042.751.751 0 0 1 1.042-.018 1.998 1.998 0 0 0 2.83 0l2.5-2.5a2.002 2.002 0 0 0-2.83-2.83l-1.25 1.25a.751.751 0 0 1-1.042-.018.751.751 0 0 1-.018-1.042Zm-4.69 9.64a1.998 1.998 0 0 0 2.83 0l1.25-1.25a.751.751 0 0 1 1.042.018.751.751 0 0 1 .018 1.042l-1.25 1.25a3.5 3.5 0 1 1-4.95-4.95l2.5-2.5a3.5 3.5 0 0 1 4.95 0 .751.751 0 0 1-.018 1.042.751.751 0 0 1-1.042.018 1.998 1.998 0 0 0-2.83 0l-2.5 2.5a1.998 1.998 0 0 0 0 2.83Z"></path></svg></a>Monte-Carlo Simulation</h2>
<ul dir="auto">
<li>
<code>metropolis_update!(model::FermionXYChain)</code>: Generates a new configuration for the chain by performing single site updates using acceptance rate <math-renderer class="js-inline-math" style="display: inline" data-static-url="https://github.githubassets.com/static" data-run-id="ee4ffa4de73009a3f9700b1cb632027e">$A(P'|P) = \cfrac{P'}{P}$</math-renderer>, where <math-renderer class="js-inline-math" style="display: inline" data-static-url="https://github.githubassets.com/static" data-run-id="ee4ffa4de73009a3f9700b1cb632027e">$P'$</math-renderer> is the probability of new configuration and <math-renderer class="js-inline-math" style="display: inline" data-static-url="https://github.githubassets.com/static" data-run-id="ee4ffa4de73009a3f9700b1cb632027e">$P$</math-renderer> is that of old configuration..</li>
<li>
<code>equilibrate!(model::FermionXYChain, steps::Int)</code>: Equilibrates the chain by performing <math-renderer class="js-inline-math" style="display: inline" data-static-url="https://github.githubassets.com/static" data-run-id="ee4ffa4de73009a3f9700b1cb632027e">$N$</math-renderer> updates.</li>
</ul>
<p dir="auto"><strong>Note: The above methods for monte-carlo sampling are useless from my experience this is probably because single site updates can not capture the transformation from quantum spins to spinless fermions. If anyone knows more about this please inform me by opening an issue</strong></p>
