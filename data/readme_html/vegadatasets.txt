<p dir="auto">This package provides an easy way to load the datasets in <a href="https://github.com/vega/vega-datasets">vega-datasets</a> from julia.</p>
<h2 dir="auto"><a id="user-content-installation" class="anchor" aria-hidden="true" href="#installation"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path d="m7.775 3.275 1.25-1.25a3.5 3.5 0 1 1 4.95 4.95l-2.5 2.5a3.5 3.5 0 0 1-4.95 0 .751.751 0 0 1 .018-1.042.751.751 0 0 1 1.042-.018 1.998 1.998 0 0 0 2.83 0l2.5-2.5a2.002 2.002 0 0 0-2.83-2.83l-1.25 1.25a.751.751 0 0 1-1.042-.018.751.751 0 0 1-.018-1.042Zm-4.69 9.64a1.998 1.998 0 0 0 2.83 0l1.25-1.25a.751.751 0 0 1 1.042.018.751.751 0 0 1 .018 1.042l-1.25 1.25a3.5 3.5 0 1 1-4.95-4.95l2.5-2.5a3.5 3.5 0 0 1 4.95 0 .751.751 0 0 1-.018 1.042.751.751 0 0 1-1.042.018 1.998 1.998 0 0 0-2.83 0l-2.5 2.5a1.998 1.998 0 0 0 0 2.83Z"></path></svg></a>Installation</h2>

<div class="highlight highlight-source-julia notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="Pkg.add(&quot;VegaDatasets&quot;)"><pre>Pkg<span class="pl-k">.</span><span class="pl-c1">add</span>(<span class="pl-s"><span class="pl-pds">"</span>VegaDatasets<span class="pl-pds">"</span></span>)</pre></div>
<h2 dir="auto"><a id="user-content-getting-started" class="anchor" aria-hidden="true" href="#getting-started"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path d="m7.775 3.275 1.25-1.25a3.5 3.5 0 1 1 4.95 4.95l-2.5 2.5a3.5 3.5 0 0 1-4.95 0 .751.751 0 0 1 .018-1.042.751.751 0 0 1 1.042-.018 1.998 1.998 0 0 0 2.83 0l2.5-2.5a2.002 2.002 0 0 0-2.83-2.83l-1.25 1.25a.751.751 0 0 1-1.042-.018.751.751 0 0 1-.018-1.042Zm-4.69 9.64a1.998 1.998 0 0 0 2.83 0l1.25-1.25a.751.751 0 0 1 1.042.018.751.751 0 0 1 .018 1.042l-1.25 1.25a3.5 3.5 0 1 1-4.95-4.95l2.5-2.5a3.5 3.5 0 0 1 4.95 0 .751.751 0 0 1-.018 1.042.751.751 0 0 1-1.042.018 1.998 1.998 0 0 0-2.83 0l-2.5 2.5a1.998 1.998 0 0 0 0 2.83Z"></path></svg></a>Getting started</h2>
<p dir="auto">The package only exports one function that takes the name of a dataset and returns a <code>VegaDataset</code> with that data:</p>
<div class="highlight highlight-source-julia notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="using VegaDatasets

vg = dataset(&quot;iris&quot;)"><pre><span class="pl-k">using</span> VegaDatasets

vg <span class="pl-k">=</span> <span class="pl-c1">dataset</span>(<span class="pl-s"><span class="pl-pds">"</span>iris<span class="pl-pds">"</span></span>)</pre></div>
<p dir="auto"><code>VegaDataset</code> implements the <a href="https://github.com/queryverse/IterableTables.jl">iterable tables</a> interface, so it can be passed to any sink that accepts iterable tables.</p>
<p dir="auto">For example, to convert a dataset into a <code>DataFrame</code>, you can write:</p>
<div class="highlight highlight-source-julia notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="using VegaDatasets, DataFrames

df = DataFrame(dataset(&quot;iris&quot;))"><pre><span class="pl-k">using</span> VegaDatasets, DataFrames

df <span class="pl-k">=</span> <span class="pl-c1">DataFrame</span>(<span class="pl-c1">dataset</span>(<span class="pl-s"><span class="pl-pds">"</span>iris<span class="pl-pds">"</span></span>))</pre></div>
<p dir="auto">You can pipe a <code>VegaDataset</code> directly into a <a href="https://github.com/fredo-dedup/VegaLite.jl">VegaLite.jl</a> plot:</p>
<div class="highlight highlight-source-julia notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="using VegaLite, VegaDatasets

dataset(&quot;iris&quot;) |&gt; @vlplot(:point, x=:sepalLength, y=:petalWidth)"><pre><span class="pl-k">using</span> VegaLite, VegaDatasets

<span class="pl-c1">dataset</span>(<span class="pl-s"><span class="pl-pds">"</span>iris<span class="pl-pds">"</span></span>) <span class="pl-k">|&gt;</span> <span class="pl-c1">@vlplot</span>(<span class="pl-c1">:point</span>, x<span class="pl-k">=</span><span class="pl-c1">:sepalLength</span>, y<span class="pl-k">=</span><span class="pl-c1">:petalWidth</span>)</pre></div>

<p dir="auto">The following datasets are not tabular in nature:</p>
<ul dir="auto">
<li>earthquakes</li>
<li>graticule</li>
<li>londonBoroughs</li>
<li>londonTubeLines</li>
<li>miserables</li>
<li>us_10m</li>
<li>world_110m</li>
</ul>
<p dir="auto">In those cases an object is returned that can be lowered to JSON, or one can access the <code>.path</code> property for such an object to get a path to the location of the dataset.</p>
