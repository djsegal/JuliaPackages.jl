<p dir="auto">This repository contains algorithms and examples from our paper: <a href="https://arxiv.org/abs/2107.05616" rel="nofollow">Fast Contact-Implicit Model-Predictive Control</a>.</p>
<p dir="auto">A collection of examples are pre-generated in notebooks with the package, please try: <a href="examples/flamingo/flat.jl">flamingo</a>, <a href="examples/pushbot/push_recovery.jl">pushbot</a>, <a href="examples/hopper/flat.jl">hopper</a>, and <a href="examples/quadruped/flat.jl">quadruped</a>. Additional notebooks with examples from the paper can be <a href="examples/README.md">generated</a>.</p>
