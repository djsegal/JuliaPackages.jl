<p dir="auto">WiSER stands for <strong>wi</strong>thin-<strong>s</strong>ubject variance <strong>e</strong>stimation by <strong>r</strong>obust regression. It is a regression aproach for estimating the effects of predictors on the within-subject variation in a longitudinal setting.</p>
<p dir="auto">WiSER.jl requires Julia v1.0 or later. See documentation for usage. I
This package is registered in the default Julia package registry, and can be installed through standard package installation procedure: e.g., running the following code in Julia REPL.</p>
<div class="highlight highlight-source-julia notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="using Pkg
