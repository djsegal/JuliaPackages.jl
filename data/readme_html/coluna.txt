<p>Coluna is a branch-and-price-and-cut framework written in Julia.
The user introduces an original MIP that models his problem using the
<a href="https://github.com/JuliaOpt/JuMP.jl">JuMP</a> modeling language and our specific extension
<a href="https://github.com/atoptima/BlockDecomposition.jl">BlockDecomposition</a> that offers a syntax
to specify the problem decomposition. Then, Coluna reformulates the original MIP and
optimizes the reformulation using the algorithms chosen by the user.
Coluna aims to be very modular and tweakable so that any user can define the behavior of
his customized branch-and-price-and-cut algorithm.</p>

<p>Coluna is a <a href="https://julialang.org/" rel="nofollow">Julia Language</a> package.</p>
<p>You can install Coluna through the Julia package manager.
Open Julia's interactive session (REPL) and type:</p>
<pre><code>   ] add Coluna
</code></pre>
<p>The documentation is under construction.
You can browse the <a href="https://atoptima.github.io/Coluna.jl/stable" rel="nofollow">stable documentation</a> for an introductory example
or the <a href="https://atoptima.github.io/Coluna.jl/latest" rel="nofollow">dev documentation</a> if you are working with the master version of Coluna.</p>

branch-and-cut-and-price algorithms. We look for beta users as Coluna is under
active development.</p>

<p>Contributions are welcomed !</p>
<p>If you encounter a bug or something unexpected happens while using Coluna,
please open an issue via the GitHub issues tracker or chat with us on the
<a href="https://discord.gg/cg77wFW" rel="nofollow">discord</a> dedicated to Coluna.</p>
<p>You can also suggest and add new models in <a href="https://github.com/atoptima/ColunaDemos.jl">ColunaDemos</a>
for tests and benchmarks.</p>
<p>See the list of <a href="https://github.com/atoptima/Coluna.jl/graphs/contributors">contributors</a>
who make Coluna possible.</p>
