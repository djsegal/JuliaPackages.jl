<p dir="auto">Coluna is a branch-and-price-and-cut framework written in Julia.
You write an original MIP that models your problem using the
<a href="https://github.com/jump-dev/JuMP.jl">JuMP</a> modeling language and our specific extension
<a href="https://github.com/atoptima/BlockDecomposition.jl">BlockDecomposition</a> offers a syntax
to specify the problem decomposition. Then, Coluna reformulates the original MIP and
optimizes the reformulation using the algorithms you choose.
Coluna aims to be very modular and tweakable so that you can define the behavior of
your customized branch-and-price-and-cut algorithm.</p>

<p dir="auto">Coluna is a <a href="https://julialang.org/" rel="nofollow">Julia Language</a> package.</p>
<p dir="auto">You can install Coluna through the Julia package manager.
Open Julia's interactive session (REPL) and type:</p>
<div class="snippet-clipboard-content notranslate position-relative overflow-auto" data-snippet-clipboard-copy-content="   ] add Coluna"><pre class="notranslate"><code>   ] add Coluna
</code></pre></div>
<p dir="auto">The documentation is still under construction but provides examples to run advanced branch-cut-and-price.
You can browse the <a href="https://atoptima.github.io/Coluna.jl/stable" rel="nofollow">stable documentation</a> if you work with the latest release
or the <a href="https://atoptima.github.io/Coluna.jl/latest" rel="nofollow">dev documentation</a> if you work with the master version of Coluna.</p>

<p dir="auto">We aim to integrate into Coluna the state-of-the-art techniques used for
branch-and-cut-and-price algorithms.</p>
<p dir="auto">The latest minor release <code>0.6.0</code> suffers from regressions as some features are not available anymore. Indeed, we focused on the correctness of the algorithms and all the missing features will be back soon, well implemented and well tested.</p>
