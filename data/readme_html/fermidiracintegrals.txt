<p dir="auto"><a href="https://github.com/feanor12/FermiDiracIntegral.jl/actions/workflows/CI.yml?query=branch%3Amain"><img src="https://github.com/feanor12/FermiDiracIntegral.jl/actions/workflows/CI.yml/badge.svg?branch=main" alt="Build Status" style="max-width: 100%;"></a></p>
<p dir="auto">Implements the complete Fermi-Dirac integral (<a href="https://en.wikipedia.org/wiki/Complete_Fermi%E2%80%93Dirac_integral" rel="nofollow">Wikipedia</a>)</p>
<p dir="auto">The general implementaion uses <a href="https://github.com/mroughan/Polylogarithms.jl">Polylogarithms.jl</a>, but there is also an approximaton for <code>F(1/2,x)</code> available.</p>
<p dir="auto">The approximated version can be called like this:</p>
<div class="highlight highlight-source-julia notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="julia&gt; using FermiDiracIntegrals

julia&gt; F(Val(1/2),1)

julia<span class="pl-k">&gt;</span> <span class="pl-c1">F</span>(<span class="pl-c1">Val</span>(<span class="pl-c1">1</span><span class="pl-k">/</span><span class="pl-c1">2</span>),<span class="pl-c1">1</span>)
<span class="pl-c1">1.5676943564187247</span></pre></div>
