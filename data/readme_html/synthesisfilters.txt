<p>SynthesisFilters.jl provides waveform generation filters for speech synthesis, especially from mel-generalized cepstrum.</p>
<p>Note that this package is built on top of <a href="https://github.com/r9y9/SPTK.jl">SPTK.jl</a>. The package is designed to provide a Julia-like interface with a focus on speech waveform synthesis. A part of the core is re-writen in Julia language from <a href="http://sp-tk.sourceforge.net/" rel="nofollow">Speech Signal Processing Toolkit (SPTK)</a>.</p>
<h2><a id="user-content-features" class="anchor" aria-hidden="true" href="#features"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path fill-rule="evenodd" d="M7.775 3.275a.75.75 0 001.06 1.06l1.25-1.25a2 2 0 112.83 2.83l-2.5 2.5a2 2 0 01-2.83 0 .75.75 0 00-1.06 1.06 3.5 3.5 0 004.95 0l2.5-2.5a3.5 3.5 0 00-4.95-4.95l-1.25 1.25zm-4.69 9.64a2 2 0 010-2.83l2.5-2.5a2 2 0 012.83 0 .75.75 0 001.06-1.06 3.5 3.5 0 00-4.95 0l-2.5 2.5a3.5 3.5 0 004.95 4.95l1.25-1.25a.75.75 0 00-1.06-1.06l-1.25 1.25a2 2 0 01-2.83 0z"></path></svg></a>Features</h2>
<ul>
<li><strong>LMADF</strong>: Log magnitude approximation digital filter for synthesis from cepstrum</li>
<li><strong>MLSADF</strong>: Mel-log spectrum approximation digital filter for synthesis from mel-cepstrum</li>
<li><strong>MGLSADF</strong>: Mel generalized-log spectrum approximation digital filter for synthesis from mel-generalized cepstrum</li>
<li><strong>AllPoleDF</strong>: All-pole digital filter for synthesis from LPC</li>
<li><strong>AllPoleLatticeDF</strong>: All-pole lattice digital filter for synthesis from PARCOR</li>
<li><strong>LSPDF</strong>: LSP digital filter for synthesis from LSP</li>
</ul>
