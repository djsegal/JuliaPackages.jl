<p dir="auto">This package provides tools for working with colorschemes and colormaps. It's a companion to the ColorSchemes.jl package.</p>
<p dir="auto">For example, you can extract colorschemes from images, and replace an image colorscheme with another. There are also functions for creating color schemes from lists, dictionaries, and functions.</p>
<p dir="auto">This package relies on:</p>
<ul dir="auto">
<li><a href="https://github.com/JuliaGraphics/Colors.jl">Colors.jl</a></li>
<li><a href="https://github.com/JuliaGraphics/ColorSchemes.jl">ColorSchemes.jl</a></li>
<li><a href="https://github.com/JuliaImages/Images.jl">Images.jl</a></li>
<li><a href="https://github.com/JuliaStats/Clustering.jl">Clustering.jl</a></li>
<li><a href="https://github.com/JuliaMath/Interpolations.jl">Interpolations.jl</a></li>
