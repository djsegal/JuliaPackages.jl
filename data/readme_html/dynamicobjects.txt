<p dir="auto">Implements a dynamic object type which can be redefined with convenient/confusing <code>.</code>-syntax and caching.</p>
<p dir="auto">See <a href="https://nsiccha.github.io/examples/DynamicObjects.jl/" rel="nofollow">https://nsiccha.github.io/examples/DynamicObjects.jl/</a> for examples.</p>

A fancy rectangle.
&quot;&quot;&quot;
@dynamic_object Rectangle height width
