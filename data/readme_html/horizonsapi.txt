<p dir="auto">The <a href="https://www.jpl.nasa.gov" rel="nofollow">NASA Jet Propulsion Laboratory</a> (JPL), located at
the <a href="https://www.caltech.edu" rel="nofollow">California Institute of Technology</a>, provides a
solar system ephemeris service called <em>HORIZONS</em>. It's fantastic! It can
<a href="https://ssd.jpl.nasa.gov/tools/periodic_orbits.html" rel="nofollow">find</a> and
<a href="https://ssd.jpl.nasa.gov/tools/orbit_diagram.html" rel="nofollow">visualize</a> periodic orbits,
allow you to <a href="https://ssd.jpl.nasa.gov/ephem.html" rel="nofollow">download</a> position and
velocity data for bodies in our solar system, list
<a href="https://ssd.jpl.nasa.gov/sats/phys_par/" rel="nofollow">physical parameters</a> for celestial
bodies, and more!</p>
<p dir="auto">You may utilize these tools using several methods, including through a
<a href="https://ssd.jpl.nasa.gov/horizons/app.html" rel="nofollow">web interface</a>, with telnet via
<code>telnet ssd.jpl.nasa.gov 6775</code>, and even through
<a href="horizons@ssd.jpl.nasa.gov?subject=BATCH-LONG">email</a>! You can consult the
HORIZONS <a href="https://ssd.jpl.nasa.gov/horizons/manual.html" rel="nofollow">documentation</a> for more
information.</p>
<p dir="auto">In 2021, JPL released a
<a href="https://www.redhat.com/en/topics/api/what-is-a-rest-api" rel="nofollow">REST</a> API which allows
users to programatically interact with HORIZONS through HTTP requests. This API
is <strong>also</strong> described in the HORIZONS
<a href="https://ssd-api.jpl.nasa.gov/doc/horizons.html" rel="nofollow">documentation</a>. This Julia
package provides an interface to the HORIZONS REST API.</p>

<p dir="auto">Note that JPL Horizons ephemeris accuracy for major planetary bodies in the
solar system
<a href="https://ssd.jpl.nasa.gov/horizons/manual.html#limits" rel="nofollow">may have errors larger than 1000km</a>
at some times!</p>

<p dir="auto">This package matches the precise interace, which matches the syntax of the
requests described in the HORIZONS
<a href="https://ssd-api.jpl.nasa.gov/doc/horizons.html" rel="nofollow">documentation</a>. In addition, a
more Julian syntax for ephemeris downloading and processing is provided in a
separate interface:
<a href="https://github.com/cadojo/HorizonsEphemeris.jl"><code>HorizonsEphemeris.jl</code></a>.</p>
