<p dir="auto">Evaluating only <code>PV</code> and <code>Storage</code> requires a linear program solver. Adding a generator and/or multiple outages makes the problem mixed-integer linear, and thus requires a MILP solver. See <a href="https://jump.dev/JuMP.jl/stable/installation/" rel="nofollow">https://jump.dev/JuMP.jl/stable/installation/</a> for a list of solvers. The REopt package has been tested with <code>Xpress</code>, <code>Cbc</code>, <code>HiGHS</code> and <code>CPLEX</code>.</p>
<p dir="auto">Note that not all solvers support indicator constraints and special order sets (such as HiGHS), and so not all REopt problems can be solved with solvers lacking these capabilities.</p>
