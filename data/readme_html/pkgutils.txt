<p dir="auto">Some small utilities to help Julia package users and authors. This package is basically a smorgasbord of code that others have submitted and I could never have written myself. <a href="https://github.com/ianshmean">https://github.com/ianshmean</a> and <a href="https://github.com/harryscholes">https://github.com/harryscholes</a> also provided lots of emotional support.</p>
<h2 dir="auto"><a id="user-content-environments" class="anchor" aria-hidden="true" href="#environments"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path d="m7.775 3.275 1.25-1.25a3.5 3.5 0 1 1 4.95 4.95l-2.5 2.5a3.5 3.5 0 0 1-4.95 0 .751.751 0 0 1 .018-1.042.751.751 0 0 1 1.042-.018 1.998 1.998 0 0 0 2.83 0l2.5-2.5a2.002 2.002 0 0 0-2.83-2.83l-1.25 1.25a.751.751 0 0 1-1.042-.018.751.751 0 0 1-.018-1.042Zm-4.69 9.64a1.998 1.998 0 0 0 2.83 0l1.25-1.25a.751.751 0 0 1 1.042.018.751.751 0 0 1 .018 1.042l-1.25 1.25a3.5 3.5 0 1 1-4.95-4.95l2.5-2.5a3.5 3.5 0 0 1 4.95 0 .751.751 0 0 1-.018 1.042.751.751 0 0 1-1.042.018 1.998 1.998 0 0 0-2.83 0l-2.5 2.5a1.998 1.998 0 0 0 0 2.83Z"></path></svg></a>Environments</h2>
<ul dir="auto">
<li><code>manifest!"..."</code> creates and activates a throwaway environment based on the contents of a <code>Manifest.toml</code>.</li>
<li><code>project!"..."</code> creates and activates a throwaway environment based on the contents of a <code>Project.toml</code>.</li>
</ul>
<p dir="auto">Thanks to <a href="https://github.com/christopher-DG">https://github.com/christopher-DG</a> and <a href="https://github.com/oxinabox">https://github.com/oxinabox</a> for this feature.</p>
<h2 dir="auto"><a id="user-content-dependencies-and-dependents" class="anchor" aria-hidden="true" href="#dependencies-and-dependents"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path d="m7.775 3.275 1.25-1.25a3.5 3.5 0 1 1 4.95 4.95l-2.5 2.5a3.5 3.5 0 0 1-4.95 0 .751.751 0 0 1 .018-1.042.751.751 0 0 1 1.042-.018 1.998 1.998 0 0 0 2.83 0l2.5-2.5a2.002 2.002 0 0 0-2.83-2.83l-1.25 1.25a.751.751 0 0 1-1.042-.018.751.751 0 0 1-.018-1.042Zm-4.69 9.64a1.998 1.998 0 0 0 2.83 0l1.25-1.25a.751.751 0 0 1 1.042.018.751.751 0 0 1 .018 1.042l-1.25 1.25a3.5 3.5 0 1 1-4.95-4.95l2.5-2.5a3.5 3.5 0 0 1 4.95 0 .751.751 0 0 1-.018 1.042.751.751 0 0 1-1.042.018 1.998 1.998 0 0 0-2.83 0l-2.5 2.5a1.998 1.998 0 0 0 0 2.83Z"></path></svg></a>Dependencies and Dependents</h2>
<ul dir="auto">
<li>
<p dir="auto"><code>get_dependents("MyPackage", n = 1)</code> returns n-th order dependents of <code>MyPackage</code> (in the <code>General</code> registry.)</p>
</li>
<li>
<p dir="auto"><code>get_dependencies("SomePackage", n = 1)</code> returns n-th order dependencies of <code>SomePackage</code>.</p>
</li>
<li>
<p dir="auto"><code>get_latest_version("SomePackage")</code> will get the latest version of that package.</p>
</li>
</ul>
<p dir="auto">Thanks to <a href="https://github.com/sbromberger">https://github.com/sbromberger</a> for most of this code.</p>

<ul dir="auto">
<li>
<p dir="auto"><code>snapshot!()</code> will save a copy of your current project and manifest file in the relevant <code>environments/</code> folder.</p>
</li>
<li>
<p dir="auto"><code>undo!()</code> will revert the current project and manifest to the most recent snapshot.</p>
</li>
</ul>
