<p dir="auto">Umlaut.jl is a code tracer for the Julia programming language. It lets you trace the function execution, recording all primitive operations onto a linearized tape.</p>

limitations, but eventually the codebase has diverged so much that the new package was
born. Although the two have pretty similar API, there are several notable differences.
See <a href="https://dfdx.github.io/Umlaut.jl/dev/ghost/" rel="nofollow">Migration from Ghost</a> for details.</p>
