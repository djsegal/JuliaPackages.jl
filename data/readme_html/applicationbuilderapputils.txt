<p dir="auto">A small package providing utilities for applications built with
<a href="https://github.com/NHDaly/ApplicationBuilder.jl"><code>ApplicationBuilder.jl</code></a>.</p>
<p dir="auto">This is split out into a separate package so that users' application code does not need to
have a dependency on the <code>ApplicationBuilder</code> module itself.</p>
<p dir="auto">Users should <code>Pkg.add("ApplicationBuilderAppUtils")</code> for their application's Project if they
want to use these utilities.</p>
