<p dir="auto">This package provides different kind of 2D and 3D viewers for kite power system.</p>
<p dir="auto">It is part of Julia Kite Power Tools, which consist of the following packages:</p>
<p align="center" dir="auto"><a target="_blank" rel="noopener noreferrer" href="./docs/kite_power_tools.png"><img src="./docs/kite_power_tools.png" width="500" style="max-width: 100%;"></a></p>
<h2 dir="auto"><a id="user-content-what-to-install" class="anchor" aria-hidden="true" href="#what-to-install"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path d="m7.775 3.275 1.25-1.25a3.5 3.5 0 1 1 4.95 4.95l-2.5 2.5a3.5 3.5 0 0 1-4.95 0 .751.751 0 0 1 .018-1.042.751.751 0 0 1 1.042-.018 1.998 1.998 0 0 0 2.83 0l2.5-2.5a2.002 2.002 0 0 0-2.83-2.83l-1.25 1.25a.751.751 0 0 1-1.042-.018.751.751 0 0 1-.018-1.042Zm-4.69 9.64a1.998 1.998 0 0 0 2.83 0l1.25-1.25a.751.751 0 0 1 1.042.018.751.751 0 0 1 .018 1.042l-1.25 1.25a3.5 3.5 0 1 1-4.95-4.95l2.5-2.5a3.5 3.5 0 0 1 4.95 0 .751.751 0 0 1-.018 1.042.751.751 0 0 1-1.042.018 1.998 1.998 0 0 0-2.83 0l-2.5 2.5a1.998 1.998 0 0 0 0 2.83Z"></path></svg></a>What to install</h2>
<p dir="auto">If you want to run simulations and see the results in 3D, please install the meta package  <a href="https://github.com/aenarete/KiteSimulators.jl">KiteSimulators</a> . If you just want to replay log files
or implement a real-time viewer for a system outside of Julia this package will be sufficient. When you have KiteSimulators installed, please replace
any statement <code>using KiteViewers</code> in the examples with <code>using KiteSimulators</code>.</p>
