<p dir="auto">AtomicGraphNets.jl implements graph-based models for machine learning on atomic systems, such as <a href="https://arxiv.org/abs/1710.10324" rel="nofollow">Crystal Graph Convolutional Neural Nets</a>, in Julia. It makes use of the <a href="https://fluxml.ai" rel="nofollow">Flux</a> ecosystem for model building and the <a href="https://github.com/JuliaGraphs">JuliaGraphs</a> ecosystem for graph representation and visualization, as well as adapting some features from <a href="https://github.com/yuehhua/GeometricFlux.jl">GeometricFlux</a>.</p>
<p dir="auto">Documentation is in progress <a href="https://atomicgraphnets.chemellia.org/dev/" rel="nofollow">over here</a>.</p>

<ol dir="auto">
<li>
<p dir="auto">To install the latest tagged version, in your Julia REPL, do <code>]add AtomicGraphNets</code>. However, you can also play with the latest version on the <code>main</code> branch by skipping to step 2 and then doing <code>]add /path/to/repo</code> where you replace the dummy path with the location of your clone.</p>
</li>
<li>
<p dir="auto">Clone this package to wherever you want to play.</p>
</li>
<li>
<p dir="auto">Go and try out the example in examples/example1/ – it has its own README file with detailed instructions.</p>
</li>
</ol>
