<p>AtomicGraphNets.jl implements graph-based models for machine learning on atomic systems, such as <a href="https://arxiv.org/abs/1710.10324" rel="nofollow">Crystal Graph Convolutional Neural Nets</a>, in Julia. It makes use of the <a href="https://fluxml.ai" rel="nofollow">Flux</a> ecosystem for model building and the <a href="https://github.com/JuliaGraphs">JuliaGraphs</a> ecosystem for graph representation and visualization, as well as adapting some features from <a href="https://github.com/yuehhua/GeometricFlux.jl">GeometricFlux</a>.</p>
<p>Documentation is in progress <a href="https://aced-differentiate.github.io/AtomicGraphNets.jl/dev/" rel="nofollow">over here</a>.</p>

<ol>
<li>
<p>To install the latest tagged version, in your Julia REPL, do <code>]add AtomicGraphNets</code>. However, you can also play with the latest version on the <code>main</code> branch by skipping to step 2 and then doing <code>]add /path/to/repo</code> where you replace the dummy path with the location of your clone.</p>
</li>
<li>
<p>Clone this package to wherever you want to play.</p>
</li>
<li>
<p>Go and try out the example in examples/example1/ – it has its own README file with detailed instructions.</p>
</li>
</ol>
