<p dir="auto">This package provides a number of navigation functions.
The functions are based on the Aviation Formulary V1.46 of Ed Williams and the
latlong scripts of Movable-type.</p>
<p dir="auto">Implemented Functions:</p>
<ul dir="auto">
<li>distance</li>
<li>bearing</li>
<li>final_bearing</li>
<li>midpoint</li>
<li>intermediate_point</li>
<li>destination_point</li>
<li>intersection_point</li>
<li>along_track_distance</li>
<li>cross_track_distance</li>
<li>Vground</li>
<li>head_wind</li>
<li>cross_wind</li>
<li>normalize</li>
<li>closest_point_to_pole</li>
<li>isinside</li>
</ul>
<p dir="auto">Implemented Types:</p>
<ul dir="auto">
<li>Point(ϕ, λ)</li>
<li>Airspace</li>
</ul>
<p dir="auto">Implemented constants:</p>
<ul dir="auto">
<li>Rₑ_m    Radius Earth in [m]</li>
</ul>
<p dir="auto">The calculations are done using the Point struct for specifying points as radians</p>
<p dir="auto">All angles are in degrees, distances are in meters, and speeds are in m/s.</p>
<ul dir="auto">
<li>Source: <a href="https://www.movable-type.co.uk/scripts/latlong.html" rel="nofollow">https://www.movable-type.co.uk/scripts/latlong.html</a></li>
<li>Source: <a href="http://edwilliams.org/avform.htm" rel="nofollow">http://edwilliams.org/avform.htm</a></li>
</ul>
