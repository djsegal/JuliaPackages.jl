<p>PlanarConvexHulls provides a <code>ConvexHull</code> type, which represents the convex hull of a set
of 2D points by its extreme points. Functionality includes:</p>
<ul>
<li>convexity test</li>
<li>construction of a convex hull given a set of points</li>
<li>area</li>
<li>centroid</li>
<li>point-in-convex-hull test</li>
<li>closest point within convex hull</li>
<li>equivalent halfspace representation of the convex hull</li>
</ul>
