<div id="readme" class="md" data-path="README.md"><article class="markdown-body entry-content container-lg" itemprop="text"><p dir="auto">WORK IN PROGRESS</p>
<p dir="auto"><code>LinearRationalExpectations</code> provides tools to solve economic linear
rational expectations models.</p>
<p dir="auto">Linear rational expectations models have the general form</p>
<p dir="auto"><math-renderer class="js-display-math" style="display: block" data-static-url="https://github.githubassets.com/static" data-run-id="db25b0d1ba740b7417921117156283b3">$$E_t { A y_{t+1} + B y_t + C y_{t-1} + D u_t + e} = 0$$</math-renderer></p>
<p dir="auto">The deterministic steady state of the model is defined as</p>
<p dir="auto"><math-renderer class="js-display-math" style="display: block" data-static-url="https://github.githubassets.com/static" data-run-id="db25b0d1ba740b7417921117156283b3">$$\bar y = -(I - A - B - C)^{-1}e$$</math-renderer></p>
<p dir="auto">The solution takes the form</p>
<p dir="auto"><math-renderer class="js-display-math" style="display: block" data-static-url="https://github.githubassets.com/static" data-run-id="db25b0d1ba740b7417921117156283b3">$$y_t - \bar y= G_y (y_{t-1} - \bar y) + G_u u_t$$</math-renderer></p>
<p dir="auto">where <math-renderer class="js-inline-math" style="display: inline" data-static-url="https://github.githubassets.com/static" data-run-id="db25b0d1ba740b7417921117156283b3">$G_y$</math-renderer>  is the solution of the polynomial matrix equation</p>
<p dir="auto"><math-renderer class="js-display-math" style="display: block" data-static-url="https://github.githubassets.com/static" data-run-id="db25b0d1ba740b7417921117156283b3">$$A G_y G_y + B G_y + C = 0$$</math-renderer></p>
<p dir="auto">Two different algorithms are provided by package
<code>PolynomialMatrixEquations</code>:
one based on generalized Schur decomposition and one based on cyclic
reduction.</p>
<p dir="auto"><math-renderer class="js-display-math" style="display: block" data-static-url="https://github.githubassets.com/static" data-run-id="db25b0d1ba740b7417921117156283b3">$$G_u = -(A G_y + B)^{-1}Du_t$$</math-renderer></p>
<p dir="auto">In addition the <code>LinearRationalExpectations</code> package provides
functions to reduce the problem size by eliminating static variables.</p>
