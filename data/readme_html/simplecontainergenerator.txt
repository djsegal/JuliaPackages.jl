<p dir="auto">SimpleContainerGenerator automates the process of creating container images for using Julia packages.</p>
<p dir="auto">The Julia packages inside the container image are automatically compiled by <a href="https://github.com/JuliaLang/PackageCompiler.jl">PackageCompiler</a> into a custom Julia system image (sysimage) for faster load times.</p>
<p dir="auto">These container images are especially useful for using Julia packages on systems without Internet access. But they are not limited to that use case. You can use these container images anywhere, as long as you have access to a tool such as Docker, Singularity, etc.</p>
<p dir="auto">When building the Docker images, make sure that Docker Desktop is set to use at least 4 GB of memory (RAM). If you run into errors, you should try further increasing the amount of available memory.</p>
