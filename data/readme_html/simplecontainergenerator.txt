<p>SimpleContainerGenerator automates the process of creating container images for using Julia packages on systems without Internet access. The Julia packages inside the container are automatically compiled into a custom sysimage for faster load times.</p>
<p>To use Julia inside the continer, you should use the <code>stopgap_julia</code> command (or, equivalently, <code>/usr/bin/stopgap_julia</code>) instead of the <code>julia</code> command.</p>
