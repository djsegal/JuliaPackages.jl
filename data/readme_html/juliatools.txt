<p dir="auto">Julietta is an integrated developement environment (IDE) for the programming language Julia. It is written in Julia using Gtk.jl. In its current form Julietta is a prototype and should not be used for real developement as several thing are not yet implemented (and due to several bugs).</p>

<p dir="auto">Currently it is a lot complicated to get Julietta running. Here is a list of instructions:</p>
<ul dir="auto">
<li>Install Gtk.jl from my "listView" branch: <a href="https://github.com/tknopp/Gtk.jl/tree/listView">https://github.com/tknopp/Gtk.jl/tree/listView</a></li>
<li>Install GtkSourceWidget.jl: <a href="https://github.com/tknopp/GtkSourceWidget.jl">https://github.com/tknopp/GtkSourceWidget.jl</a></li>
</ul>
<p dir="auto">Furthermore Julietta requires the following Julia packages to be installed: REPLCompletions, Winston</p>
<p dir="auto">The complicated part is not the installation of the Julia packages (using Pkg.clone) but the installation of the dependencies. Under linux this should be most simple as the libraries Gtk+ and GtkSourceView are part of the distributions package manager.</p>
