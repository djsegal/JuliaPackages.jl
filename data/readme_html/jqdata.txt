<p dir="auto">JQData是聚宽数据团队专门为金融机构、学术研究和量化研究者们提供的本地量化金融数据服务。</p>
<p dir="auto">JQData Api 是基于JQData数据，提供的开放式网络接口。用户可以不受语言和开发环境限制，以更低成本，更高效率，更简单的方式使用聚宽数据。</p>

<p dir="auto">请求数据接口前需要调用get_token方法获取token
请求方式为post，请求格式为json格式的body字符串</p>
<p dir="auto">返回：csv格式文本数据，少数接口为json格式字符串</p>
<p dir="auto">请求示例
Python
官网详细案例：【JQData-HTTP】示例教程(Python)</p>
<div class="highlight highlight-source-python notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="# coding=utf-8
import requests,json
url=&quot;https://dataapi.joinquant.com/apis&quot;
#获取调用凭证
body={
    &quot;method&quot;: &quot;get_token&quot;,
    &quot;mob&quot;: &quot;135xxxxxxx&quot;,  #mob是申请JQData时所填写的手机号
    &quot;pwd&quot;: &quot;xxxxxxxxxx&quot;,  #Password为聚宽官网登录密码，新申请用户默认为手机号后6位
}
response = requests.post(url, data = json.dumps(body))
token=response.text
#调用get_security_info获取单个标的信息
body={
    &quot;method&quot;: &quot;get_security_info&quot;,
    &quot;token&quot;: token,
    &quot;code&quot;: &quot;502050.XSHG&quot;,
}
response = requests.post(url, data = json.dumps(body))
print response.text

<h2 dir="auto"><a id="user-content-常见问题" class="anchor" aria-hidden="true" href="#常见问题"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path d="m7.775 3.275 1.25-1.25a3.5 3.5 0 1 1 4.95 4.95l-2.5 2.5a3.5 3.5 0 0 1-4.95 0 .751.751 0 0 1 .018-1.042.751.751 0 0 1 1.042-.018 1.998 1.998 0 0 0 2.83 0l2.5-2.5a2.002 2.002 0 0 0-2.83-2.83l-1.25 1.25a.751.751 0 0 1-1.042-.018.751.751 0 0 1-.018-1.042Zm-4.69 9.64a1.998 1.998 0 0 0 2.83 0l1.25-1.25a.751.751 0 0 1 1.042.018.751.751 0 0 1 .018 1.042l-1.25 1.25a3.5 3.5 0 1 1-4.95-4.95l2.5-2.5a3.5 3.5 0 0 1 4.95 0 .751.751 0 0 1-.018 1.042.751.751 0 0 1-1.042.018 1.998 1.998 0 0 0-2.83 0l-2.5 2.5a1.998 1.998 0 0 0 0 2.83Z"></path></svg></a>常见问题</h2>
<p dir="auto">请求数据时返回error: invalid token或者error: token expired：</p>
<p dir="auto">请使用get_token方法重新获取token
token的有效期是多久：</p>
<p dir="auto">当天获取的token有效期截止到当天夜里23:59:59。每次获取token后，之前的token都会失效
为什么返回数据是CSV格式：</p>
<ol dir="auto">
<li>
<p dir="auto">减少非必要的返回字符</p>
</li>
<li>
<p dir="auto">方便用户保存数据到文本</p>
</li>
<li>
<p dir="auto">由于sdk中的get_fund_info方法返回数据是嵌套结构，所以get_fund_info接口返回了json格式数据
如何解析CSV格式数据：</p>
</li>
<li>
<p dir="auto">先用换行符"\n"切割文本数据，生成数组，再将每条数据用","切割成字段数据</p>
</li>
<li>
<p dir="auto">Python开发者可以使用csv包或者pandas的read_csv()方法进行读取csv数据</p>
</li>
</ol>
<p dir="auto">请求返回空、状态码504怎么办：</p>
<p dir="auto">504意为请求超时，可以稍后再试。如果出现在含有count参数的接口，可以减小count值重新请求。</p>
<p dir="auto">请求返回空、状态码500怎么办：</p>
<p dir="auto">500意为服务器报错，可以先检查参数是否正确。也可能是服务器负载过高，稍后再试。</p>
<p dir="auto">为什么会返回“请求频率过高，请稍后再试”</p>
<p dir="auto">为了防止大量高频率的请求导致服务器负载过高，增加网络请求限制，
目前限制规则为每个账号每分钟1800次请求。后续看情况可逐步放宽限制。</p>
<p dir="auto">为什么get_price和get_ticks等接口不能支持多标的请求</p>
<p dir="auto">为了保证http接口的性能，需要限制每个接口的数据返回量。多标的情况下上述接口数据量不可控，所以暂时不支持。</p>
<p dir="auto">建议有需求的用户使用并发方式同时请求数据，或者使用JQData SDK获取。
JQData现有流量增加活动，详情请咨询JQData管理员，微信号：JQData02</p>
