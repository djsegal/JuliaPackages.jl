<p dir="auto">This package provides optimization models based on <a href="https://github.com/JuliaSmoothOptimizers/NLPModels.jl">NLPModels</a>.
The API can be found on their <a href="https://juliasmoothoptimizers.github.io/NLPModels.jl/dev/api" rel="nofollow">documentation page</a>.</p>
<p dir="auto">The models in this package specialize on modifying existing methods.</p>

<p dir="auto">The following models are implemented.</p>
<ul dir="auto">
<li><a href="@ref"><code>FeasibilityFormNLS</code></a>: Creates residual variables and constraints, so that the residual is linear.</li>
<li><a href="@ref"><code>FeasibilityResidual</code></a>: Creates a nonlinear least squares model from an equality constrained problem in which the residual function is the constraints function.</li>
<li><a href="@ref"><code>LBFGSModel</code></a>: Creates a model using a LBFGS approximation to the Hessian using an existing NLPModel.</li>
<li><a href="@ref"><code>LSR1Model</code></a>: Creates a model using a LSR1 approximation to the Hessian using an existing NLPModel.</li>
<li><a href="@ref"><code>SlackModel</code></a>: Creates an equality constrained problem with bounds on the variables using an existing NLPModel.</li>
<li><a href="@ref"><code>SlackNLSModel</code></a>: Creates an equality constrained nonlinear least squares problem with bounds on the variables using an existing NLSModel.</li>
</ul>
<p dir="auto">See the <a href="https://JuliaSmoothOptimizers.github.io/NLPModels.jl/dev/" rel="nofollow">documentation</a> for details on the models.</p>
