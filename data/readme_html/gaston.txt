<p dir="auto">Gaston is a <a href="https://julialang.org" rel="nofollow">Julia</a> package for plotting. It provides an interface to <a href="http://gnuplot.info" rel="nofollow">gnuplot</a>, a powerful plotting package available on all major platforms.</p>
<p dir="auto">Current stable release is v1.1.0, and it has been tested with Julia LTS (1.6) and stable (1.8), on
Linux. Gaston <em>should</em> work on any platform that runs gnuplot.</p>

<p dir="auto">Why use Gaston, when there are powerful alternatives such as Plots.jl and Makie.jl? These are some Gaston features that may be attractive to you:</p>
<ul dir="auto">
<li>Gaston can plot:
<ul dir="auto">
<li>Using graphical windows, and keeping multiple plots active at a time, with mouse interaction</li>
<li>Directly to the REPL, using text (ASCII) or sixels</li>
<li>In Jupyter, Juno and other IDEs</li>
</ul>
</li>
<li>Supports popular 2-D plots: regular function plots, stem, step, histograms, images, etc.</li>
<li>Supports surface, contour and heatmap 3-D plots.</li>
<li>Can save plots to multiple formats, including pdf, png and svg.</li>
<li>Provides a simple interface for knowledgeable users to access gnuplot features.</li>
<li>Fast: time to load package, plot, and save to pdf is around six seconds.</li>
</ul>
<p dir="auto">Knowledge of gnuplot is not required. Users familiar with gnuplot, however, will be able to take advantage of Gaston's facilities to access gnuplot's vast feature set.</p>
