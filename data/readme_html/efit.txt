<p dir="auto"><a href="https://fusion.gat.com/theory/Efit" rel="nofollow">EFIT (Equilibrium Fitting)</a> is a computer code developed to translate measurements from plasma diagnostics into useful information like plasma geometry, stored energy, and current profiles.</p>
<p dir="auto">EFIT.jl provides basic functionality for reading <a href="https://fusion.gat.com/theory/Efitgeqdsk" rel="nofollow">EFIT GEQDSK</a> files.</p>
