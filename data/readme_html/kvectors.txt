<p dir="auto">The <code>KVectors</code> <a href="http://julialang.org" rel="nofollow">Julia</a> package defines the <code>KVector</code> Type
to represent linear combinations of <a href="https://en.wikipedia.org/wiki/Blade_(geometry)" rel="nofollow">k-blades</a>.</p>
<p dir="auto">All Blades in a given KVector have the same grade.</p>
<p dir="auto">This is the main object in <a href="https://en.wikipedia.org/wiki/Exterior_algebra" rel="nofollow">Grassmann Algebra</a>.</p>
<p dir="auto">KVectors essentially extends the algebras and Types defined in <a href="https://github.com/mewertd2/Blades.jl">Blades</a> with the <code>+</code> operator.  This allows for a vector space of Blades.</p>
<p dir="auto">Most operators that act on Blades can act on KVectors.  Notable exceptions are the inner product <code>â‹…</code> and geometric product <code>*</code>.
These can not, in general, operate on KVectors as they could result in a mixed grade vector.  The KVectors algebra is not closed under such operators.  For mixed grades you need <a href="https://github.com/mewertd2/Multivectors.jl">Multivectors</a>.</p>
<p dir="auto">See the documentation of <a href="https://github.com/mewertd2/Blades.jl">Blades</a> for more information.</p>
