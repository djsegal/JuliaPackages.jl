<p dir="auto"><code>LuxCore.jl</code> defines the abstract layers for Lux. Allows users to be compatible with the
entirely of <code>Lux.jl</code> without having such a heavy dependency. If you are depending on
<code>Lux.jl</code> directly, you do not need to depend on <code>LuxCore.jl</code> (all the functionality is
exported via <code>Lux.jl</code>).</p>
