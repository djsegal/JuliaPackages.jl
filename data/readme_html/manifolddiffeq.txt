<p dir="auto">The package <strong>ManifoldDiffEq.jl</strong> provides solvers for differential equations on Riemannian
manifolds using the interfaces of <a href="https://github.com/JuliaManifolds/ManifoldsBase.jl"><code>ManifoldsBase.jl</code></a>
and <a href="https://github.com/SciML/OrdinaryDiffEq.jl"><code>OrdinaryDiffEq.jl</code></a> from within the
<a href="https://sciml.ai" rel="nofollow">SciML</a> suite.</p>
