<p>Statistical block bootstrap library for Julia. Created by William Davis and Maggie Avery.</p>
<p><code>BlockBootstrap.jl</code> is a Julia package for calculating and bootstrapping uncertainties statistics on time-series data.</p>
<h1><a id="user-content-theory" class="anchor" aria-hidden="true" href="#theory"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path fill-rule="evenodd" d="M4 9h1v1H4c-1.5 0-3-1.69-3-3.5S2.55 3 4 3h4c1.45 0 3 1.69 3 3.5 0 1.41-.91 2.72-2 3.25V8.59c.58-.45 1-1.27 1-2.09C10 5.22 8.98 4 8 4H4c-.98 0-2 1.22-2 2.5S3 9 4 9zm9-3h-1v1h1c1 0 2 1.22 2 2.5S13.98 12 13 12H9c-.98 0-2-1.22-2-2.5 0-.83.42-1.64 1-2.09V6.25c-1.09.53-2 1.84-2 3.25C6 11.31 7.55 13 9 13h4c1.45 0 3-1.69 3-3.5S14.5 6 13 6z"></path></svg></a>Theory</h1>
<p>The theory used in this package is based off the book “Resampling methods for dependent data” by S.N. Lahiri, 2003, Springer.</p>

<p>This example is mostly from <code>./notebooks/BootstrapSDE.ipynb</code>. For a time-series observation, such as a realization of the Ornstein-Uhlenbeck stochastic differential equation,</p>
<p><a target="_blank" rel="noopener noreferrer" href="https://camo.githubusercontent.com/c21648bcc8739469319e881d32462db82b937114/68747470733a2f2f6c617465782e636f6465636f67732e636f6d2f6769662e6c617465783f64752673706163653b3d2673706163653b2d253543616c7068612673706163653b752673706163653b64742673706163653b2b2673706163653b253543626574612673706163653b64575f742e"><img src="https://camo.githubusercontent.com/c21648bcc8739469319e881d32462db82b937114/68747470733a2f2f6c617465782e636f6465636f67732e636f6d2f6769662e6c617465783f64752673706163653b3d2673706163653b2d253543616c7068612673706163653b752673706163653b64742673706163653b2b2673706163653b253543626574612673706163653b64575f742e" alt="equation" data-canonical-src="https://latex.codecogs.com/gif.latex?du&amp;space;=&amp;space;-%5Calpha&amp;space;u&amp;space;dt&amp;space;+&amp;space;%5Cbeta&amp;space;dW_t." style="max-width:100%;"></a></p>
<p>This solution will look something like the figure below.</p>
<p><a target="_blank" rel="noopener noreferrer" href="/figures/SDEsolution.png"><img src="/figures/SDEsolution.png" height="400" style="max-width:100%;"></a></p>
<p>Say we want to estimate the parameter <a target="_blank" rel="noopener noreferrer" href="https://camo.githubusercontent.com/e5c5578a54815d3be59cae2b1d48f9492ce872cc/68747470733a2f2f6c617465782e636f6465636f67732e636f6d2f6769662e6c617465783f253543616c706861"><img src="https://camo.githubusercontent.com/e5c5578a54815d3be59cae2b1d48f9492ce872cc/68747470733a2f2f6c617465782e636f6465636f67732e636f6d2f6769662e6c617465783f253543616c706861" alt="equation" data-canonical-src="https://latex.codecogs.com/gif.latex?%5Calpha" style="max-width:100%;"></a> from the realization, and we also want to find the incertainties on that estimation. Define a function handle that returns the statistic of choice (in this example it is <code>slopeStatisticSet()</code>, and pass it to the block bootstrapping function <code>bootstrapStatistic()</code>.</p>
<div class="highlight highlight-source-julia"><pre><span class="pl-c"><span class="pl-c">#</span># Calculate the statistic and bootstrap the uncertainties</span>
<span class="pl-c"><span class="pl-c">#</span> Settings</span>
inputData <span class="pl-k">=</span> U;
statisticHandle <span class="pl-k">=</span> slopeStatisticSet;
bootstrapSampleHandle <span class="pl-k">=</span> CBBsample;
blockLength <span class="pl-k">=</span> <span class="pl-c1">1500</span>;
NbootstrapReplicates <span class="pl-k">=</span> <span class="pl-c1">200</span>;

fullDataEstimate, replicateEstimate, resampleIndexBB, resampleDataBB <span class="pl-k">=</span> <span class="pl-c1">bootstrapStatistic</span>(
    inputData, statisticHandle, bootstrapSampleHandle, blockLength, NbootstrapReplicates);</pre></div>
<p>Settings are defined as functions to the argument <code>bootstrapStatistic()</code>, see the header of the function for descriptions. The output <code>resampleIndexBB</code> shows how the time indices have been resampled, e.g. see the figure below.</p>
<p><a target="_blank" rel="noopener noreferrer" href="/figures/MBBindices.png"><img src="/figures/MBBindices.png" height="400" style="max-width:100%;"></a></p>
<p>The output <code>fullDataEstimate</code> is the estimate of the statistics, and <code>replicateEstimate</code> is a vector of the statistics for bootstrap replicants, e.g. see the figure below.</p>
<p><a target="_blank" rel="noopener noreferrer" href="/figures/bootstrapDriftSlope.png"><img src="/figures/bootstrapDriftSlope.png" height="400" style="max-width:100%;"></a></p>
