<h2><a id="user-content-description" class="anchor" aria-hidden="true" href="#description"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path fill-rule="evenodd" d="M4 9h1v1H4c-1.5 0-3-1.69-3-3.5S2.55 3 4 3h4c1.45 0 3 1.69 3 3.5 0 1.41-.91 2.72-2 3.25V8.59c.58-.45 1-1.27 1-2.09C10 5.22 8.98 4 8 4H4c-.98 0-2 1.22-2 2.5S3 9 4 9zm9-3h-1v1h1c1 0 2 1.22 2 2.5S13.98 12 13 12H9c-.98 0-2-1.22-2-2.5 0-.83.42-1.64 1-2.09V6.25c-1.09.53-2 1.84-2 3.25C6 11.31 7.55 13 9 13h4c1.45 0 3-1.69 3-3.5S14.5 6 13 6z"></path></svg></a>Description</h2>
<p>Shearlab is a Julia Library with toolbox for two- and threedimensional data processing using the Shearlet system as basis functions which generates an sparse representation of cartoon-like functions. It is based in the Matlab Library Shearlab3D, developed by the <a href="http://www.math.tu-berlin.de/fachgebiete_ag_modnumdiff/angewandtefunktionalanalysis/v_menue/afg/" rel="nofollow">Applied Functional Analysis Research Group</a> in the Technical University of Berlin lead by <a href="http://www.tu-berlin.de/?108957" rel="nofollow">Professor Gitta Kutyniok</a>, for further information of the Matlab Implementation you can visit the link <a href="http://www3.math.tu-berlin.de/numerik/www.shearlab.org/" rel="nofollow">Shearlab3D</a>.</p>
<p>The Julia implementation has visible efficiency improvements which can be seen in the carpet <a href="https://github.com/arsenal9971/Shearlab.jl/tree/master/benchmarks">Benchmarks</a>, and some examples in <a href="https://github.com/arsenal9971/Shearlab.jl/tree/master/examples">Examples</a>.</p>
<p>For the 2D version one has three important functions:</p>
<ul>
<li>Generate the Shearlet System.</li>
</ul>
<div class="highlight highlight-source-julia"><pre><span class="pl-c1">SLgetShearletSystem2D</span>(rows,cols,nScales,shearLevels,full<span class="pl-k">=</span> <span class="pl-c1">0</span>,directionalFilter, quadratureMirrorFilter) </pre></div>
<ul>
<li>Decoding of a signal X.</li>
</ul>
<div class="highlight highlight-source-julia"><pre><span class="pl-c1">SLsheardec2D</span>(X,shearletSystem) </pre></div>
<ul>
<li>Reconstruction of a signal X.</li>
</ul>
<div class="highlight highlight-source-julia"><pre><span class="pl-c1">SLshearrec2D</span>(coeffs,shearletSystem) </pre></div>
<p>For more detailed usage functionalities check the original <a href="http://shearlab.org/files/documents/ShearLab3Dv10_Manual.pdf" rel="nofollow">Shearlab manual</a>, for scientific reference one can also read <a href="http://www.math.tu-berlin.de/fileadmin/i26_fg-kutyniok/Kutyniok/Papers/ShearLab3D.pdf" rel="nofollow">"ShearLab 3D: Faithful Digital Shearlet Transforms Based on Compactly Supported Shearlets"</a>.</p>
