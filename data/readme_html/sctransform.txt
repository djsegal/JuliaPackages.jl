<p dir="auto">This is a Julia implementation of <a href="https://github.com/satijalab/sctransform">sctransform</a>, based on the paper <a href="https://doi.org/10.1186/s13059-019-1874-1" rel="nofollow">Normalization and variance stabilization of single-cell RNA-seq data using regularized negative binomial regression</a> by Hafemaister and Satija.</p>
<p dir="auto">This is an independent implementation, the original authors were not involved.
Here is <a href="https://github.com/satijalab/sctransform">the original R implementation of sctransform</a>.</p>

<p dir="auto">The doc strings for <code>scparams</code> and <code>sctransform</code> describe the usage.
Also see the <a href="https://github.com/rasmushenningsson/SingleCellProjections.jl">SingleCellProjections.jl</a> package which provides a framework for working with Single Cell expression data and supports <code>SCTransform.jl</code>.</p>
<p dir="auto">Note that <a href="https://docs.julialang.org/en/v1/manual/multi-threading/" rel="nofollow">threading</a> should be enabled in Julia for <code>SCTransform.jl</code> to work optimally.</p>
