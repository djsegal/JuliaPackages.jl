<p dir="auto"><a href="https://github.com/JinraeKim/FSimPlots.jl">FSimPlots.jl</a> is
the plotting package for predefined environments exported from <a href="https://github.com/JinraeKim/FlightSims.jl">FlightSims.jl</a>.</p>
<p dir="auto">FSimPlots.jl is detached from FlightSims.jl to improve the precompilation overload.
See <a href="https://github.com/JinraeKim/FSimBase.jl">FSimBase.jl</a> for the lightweight version of FlightSims.jl.</p>
