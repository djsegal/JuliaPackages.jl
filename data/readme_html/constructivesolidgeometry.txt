<p dir="auto">You can install this package with the following Julia command:</p>
<div class="highlight highlight-source-julia notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="Pkg.add(&quot;ConstructiveSolidGeometry&quot;)"><pre>Pkg<span class="pl-k">.</span><span class="pl-c1">add</span>(<span class="pl-s"><span class="pl-pds">"</span>ConstructiveSolidGeometry<span class="pl-pds">"</span></span>)</pre></div>
<p dir="auto">The package can then be loaded and used in a Julia script or a Jupyter Notebook by:</p>
<div class="highlight highlight-source-julia notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="using ConstructiveSolidGeometry"><pre><span class="pl-k">using</span> ConstructiveSolidGeometry</pre></div>
<p dir="auto">Note that plotting functionality for this package is dependent on Plots package and the PyPlot backend. If you don't have these already, install them by:</p>
<div class="highlight highlight-source-julia notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="Pkg.add(&quot;Plots&quot;)
Pkg.add(&quot;PyPlot&quot;)"><pre>Pkg<span class="pl-k">.</span><span class="pl-c1">add</span>(<span class="pl-s"><span class="pl-pds">"</span>Plots<span class="pl-pds">"</span></span>)
Pkg<span class="pl-k">.</span><span class="pl-c1">add</span>(<span class="pl-s"><span class="pl-pds">"</span>PyPlot<span class="pl-pds">"</span></span>)</pre></div>
<p dir="auto">The PyPlot package requires that matplotlib (a python package) be installed on your system, which can be done using apt-get, pip, or by following the full instructions <a href="http://matplotlib.org/faq/installing_faq.html#python-org-python" rel="nofollow">here</a>.</p>
