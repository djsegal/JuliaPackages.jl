<p>This package implements three different Bayesian non-parametric methods for
inferring the number of clusters in a mixture model, and cluster assignemnts.</p>
<ol>
<li>A Gibbs sampler which makes multiple passes over the data (batch).</li>
<li>A particle filter using the resampling scheme described by <a href="https://doi.org/10.1023/B:STCO.0000009418.04621.cd" rel="nofollow">Fearnhead
(2004)</a> (online).</li>
<li>A particle filter using the resampling scheme of <a href="https://doi.org/10.1111/1467-9868.00246" rel="nofollow">Chen and Liu
(2000)</a> (online).</li>
</ol>
<p>These can fit variants of a Dirichlet-process mixture model, with a conjugate
prior for the cluster parameters.  Currently uni- or multivariate normal
distribution components are supported (via a <code>NormalInverseChisq</code> and
<code>NormalInverseWishart</code> prior).  In principle the code could be generalized to
use any of the distributions in
<a href="https://github.com/JuliaStats/ConjugatePriors.jl">ConjugatePriors.jl</a> but this
requires some additional methods (for doing online updates of the necessary
sufficient statistics and calculating the collapsed/marginal likelihoods).</p>
