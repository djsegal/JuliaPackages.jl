<p>FastHistograms declares and implements a minimal histogram interface with a focus on speed.
This package does not aim to implement general histogram algorithms that work for a wide array of data types; for that
purpose, consider <a href="https://github.com/JuliaStats/StatsBase.jl">StatsBase</a>.</p>
<p>Currently, the only implemented algorithm is for fixed-width bins on (small) 2D data.</p>
