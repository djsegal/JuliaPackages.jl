<p dir="auto">This package allows to use machine learning in Julia through a graphical user interface.</p>
<p dir="auto">NB! This is a beta version. Bugs and breaking changes should be expected.</p>
<p dir="auto">The package will be updated to Julia 1.7 as soon as <a href="https://github.com/barche/QML.jl">QML.jl</a> is updated.</p>
<p dir="auto">If you are interested in the package and would like to contribute then write here: <a href="mailto:a_ill@outlook.com">a_ill@outlook.com</a>.</p>

<p dir="auto">It is possible to:</p>
<ul dir="auto">
<li>Design a neural network</li>
<li>Train a neural network</li>
<li>Validate a neural network</li>
<li>Apply a neural network to new data</li>
</ul>
<p dir="auto">Classification, regression and segmentation on images are currently supported.</p>
<p dir="auto"><a href="https://github.com/FluxML/Flux.jl">Flux.jl</a> machine learning library is used under the hood.</p>
