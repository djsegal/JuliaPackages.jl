<p dir="auto"><code>PooksoftAlphaVantageDataStore.jl</code> is an application programming interface (API) for <a href="https://www.alphavantage.co" rel="nofollow">AlphaVantage</a>, a leading provider of realtime and historical stock, forex (FX) data, and digital/crypto currency data feeds written in the <a href="https://julialang.org" rel="nofollow">Julia</a> programming language.</p>
<h2 dir="auto"><a id="user-content-installation-and-requirements" class="anchor" aria-hidden="true" href="#installation-and-requirements"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path d="m7.775 3.275 1.25-1.25a3.5 3.5 0 1 1 4.95 4.95l-2.5 2.5a3.5 3.5 0 0 1-4.95 0 .751.751 0 0 1 .018-1.042.751.751 0 0 1 1.042-.018 1.998 1.998 0 0 0 2.83 0l2.5-2.5a2.002 2.002 0 0 0-2.83-2.83l-1.25 1.25a.751.751 0 0 1-1.042-.018.751.751 0 0 1-.018-1.042Zm-4.69 9.64a1.998 1.998 0 0 0 2.83 0l1.25-1.25a.751.751 0 0 1 1.042.018.751.751 0 0 1 .018 1.042l-1.25 1.25a3.5 3.5 0 1 1-4.95-4.95l2.5-2.5a3.5 3.5 0 0 1 4.95 0 .751.751 0 0 1-.018 1.042.751.751 0 0 1-1.042.018 1.998 1.998 0 0 0-2.83 0l-2.5 2.5a1.998 1.998 0 0 0 0 2.83Z"></path></svg></a>Installation and Requirements</h2>
<p dir="auto"><code>PooksoftAlphaVantageDataStore.jl</code> can be installed, updated, or removed using the <a href="https://docs.julialang.org/en/v1/stdlib/Pkg/" rel="nofollow">Julia package management system</a>. To access the package management interface, open the <a href="https://docs.julialang.org/en/v1/stdlib/REPL/" rel="nofollow">Julia REPL</a>, and start the package mode by pressing <code>]</code>.
While in package mode, to install <code>PooksoftAlphaVantageDataStore.jl</code>, issue the command:</p>
<div class="snippet-clipboard-content notranslate position-relative overflow-auto" data-snippet-clipboard-copy-content="(@v1.6) pkg&gt; add PooksoftAlphaVantageDataStore"><pre class="notranslate"><code>(@v1.6) pkg&gt; add PooksoftAlphaVantageDataStore
</code></pre></div>
<p dir="auto">To use <code>PooksoftAlphaVantageDataStore.jl</code> in your project issue the command:</p>
<div class="snippet-clipboard-content notranslate position-relative overflow-auto" data-snippet-clipboard-copy-content="julia&gt; using PooksoftAlphaVantageDataStore"><pre class="notranslate"><code>julia&gt; using PooksoftAlphaVantageDataStore
</code></pre></div>
<h2 dir="auto"><a id="user-content-utility-functions" class="anchor" aria-hidden="true" href="#utility-functions"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path d="m7.775 3.275 1.25-1.25a3.5 3.5 0 1 1 4.95 4.95l-2.5 2.5a3.5 3.5 0 0 1-4.95 0 .751.751 0 0 1 .018-1.042.751.751 0 0 1 1.042-.018 1.998 1.998 0 0 0 2.83 0l2.5-2.5a2.002 2.002 0 0 0-2.83-2.83l-1.25 1.25a.751.751 0 0 1-1.042-.018.751.751 0 0 1-.018-1.042Zm-4.69 9.64a1.998 1.998 0 0 0 2.83 0l1.25-1.25a.751.751 0 0 1 1.042.018.751.751 0 0 1 .018 1.042l-1.25 1.25a3.5 3.5 0 1 1-4.95-4.95l2.5-2.5a3.5 3.5 0 0 1 4.95 0 .751.751 0 0 1-.018 1.042.751.751 0 0 1-1.042.018 1.998 1.998 0 0 0-2.83 0l-2.5 2.5a1.998 1.998 0 0 0 0 2.83Z"></path></svg></a>Utility functions</h2>
<p dir="auto">The utility functions construct two important composite data types, <a href="https://github.com/Pooksoft/PooksoftAlphaVantageDataStore.jl/blob/master/src/base/Types.jl">PSUserModel</a> which encapsulates information about your <a href="https://www.alphavantage.co" rel="nofollow">AlphaVantage</a> account and <a href="https://github.com/Pooksoft/PooksoftAlphaVantageDataStore.jl/blob/master/src/base/Types.jl">PSDataStoreAPICallModel</a> which can be used along with the high-level interface to make <a href="https://www.alphavantage.co" rel="nofollow">AlphaVantage</a> application programming interface (API) calls.</p>
<p dir="auto">Utility functions:</p>
<ul dir="auto">
<li><a href="https://github.com/Pooksoft/PooksoftAlphaVantageDataStore.jl/blob/master/src/base/User.jl">build_api_user_model</a> | Function to build a user model object which requires an <a href="https://www.alphavantage.co/support/#api-key" rel="nofollow">AlphaVantage API key</a></li>
<li><a href="https://github.com/Pooksoft/PooksoftAlphaVantageDataStore.jl/blob/master/src/base/Datastore.jl">build_datastore_apicall_model</a> | Utility function to build an api call model which is required for the high-level api call interface. This function returns a <a href="https://github.com/Pooksoft/PooksoftAlphaVantageDataStore.jl/blob/master/src/base/Types.jl">PSDataStoreAPICallModel</a> object wrapped in a <code>PSResult</code> type; the <a href="https://github.com/Pooksoft/PooksoftAlphaVantageDataStore.jl/blob/master/src/base/Types.jl">PSDataStoreAPICallModel</a> object can be accessed from the <code>value</code> field of the <code>PSResult</code> type.</li>
</ul>
<h2 dir="auto"><a id="user-content-stock-time-series-sts-functions-low-level-interface" class="anchor" aria-hidden="true" href="#stock-time-series-sts-functions-low-level-interface"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path d="m7.775 3.275 1.25-1.25a3.5 3.5 0 1 1 4.95 4.95l-2.5 2.5a3.5 3.5 0 0 1-4.95 0 .751.751 0 0 1 .018-1.042.751.751 0 0 1 1.042-.018 1.998 1.998 0 0 0 2.83 0l2.5-2.5a2.002 2.002 0 0 0-2.83-2.83l-1.25 1.25a.751.751 0 0 1-1.042-.018.751.751 0 0 1-.018-1.042Zm-4.69 9.64a1.998 1.998 0 0 0 2.83 0l1.25-1.25a.751.751 0 0 1 1.042.018.751.751 0 0 1 .018 1.042l-1.25 1.25a3.5 3.5 0 1 1-4.95-4.95l2.5-2.5a3.5 3.5 0 0 1 4.95 0 .751.751 0 0 1-.018 1.042.751.751 0 0 1-1.042.018 1.998 1.998 0 0 0-2.83 0l-2.5 2.5a1.998 1.998 0 0 0 0 2.83Z"></path></svg></a>Stock Time Series (STS) functions (low-level interface)</h2>
<p dir="auto">STS functions allow the user to download daily, weekly or monthly stock price data (or adjusted data) with a frequency depending upon your <a href="https://www.alphavantage.co/support/#support" rel="nofollow">AlphaVantage</a> account privileges. These functions take the form:</p>
<div class="snippet-clipboard-content notranslate position-relative overflow-auto" data-snippet-clipboard-copy-content="execute_sts_{*}_api_call"><pre class="notranslate"><code>execute_sts_{*}_api_call
</code></pre></div>
<p dir="auto">where <code>{*}</code> denotes the time frame for the data.</p>
<ul dir="auto">
<li><a href="https://github.com/Pooksoft/PooksoftAlphaVantageDataStore.jl/blob/master/src/sts/STSDaily.jl">execute_sts_daily_api_call</a> | Download daily stock price information</li>
<li><a href="https://github.com/Pooksoft/PooksoftAlphaVantageDataStore.jl/blob/master/src/sts/STSDaily.jl">execute_sts_adjusted_daily_api_call</a> | Download adjusted daily stock price information</li>
<li><a href="https://github.com/Pooksoft/PooksoftAlphaVantageDataStore.jl/blob/master/src/sts/STSWeekly.jl">execute_sts_weekly_api_call</a> | Download weekly stock price information</li>
<li><a href="https://github.com/Pooksoft/PooksoftAlphaVantageDataStore.jl/blob/master/src/sts/STSWeekly.jl">execute_sts_adjusted_weekly_api_call</a> | Download adjusted weekly stock price information</li>
<li><a href="https://github.com/Pooksoft/PooksoftAlphaVantageDataStore.jl/blob/master/src/sts/STSMonthly.jl">execute_sts_monthly_api_call</a> | Download monthly stock price information</li>
<li><a href="https://github.com/Pooksoft/PooksoftAlphaVantageDataStore.jl/blob/master/src/sts/STSMonthly.jl">execute_sts_adjusted_monthly_api_call</a> | Download adjusted monthly stock price information</li>
</ul>
<h2 dir="auto"><a id="user-content-stock-time-series-sts-functions-high-level-interface" class="anchor" aria-hidden="true" href="#stock-time-series-sts-functions-high-level-interface"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path d="m7.775 3.275 1.25-1.25a3.5 3.5 0 1 1 4.95 4.95l-2.5 2.5a3.5 3.5 0 0 1-4.95 0 .751.751 0 0 1 .018-1.042.751.751 0 0 1 1.042-.018 1.998 1.998 0 0 0 2.83 0l2.5-2.5a2.002 2.002 0 0 0-2.83-2.83l-1.25 1.25a.751.751 0 0 1-1.042-.018.751.751 0 0 1-.018-1.042Zm-4.69 9.64a1.998 1.998 0 0 0 2.83 0l1.25-1.25a.751.751 0 0 1 1.042.018.751.751 0 0 1 .018 1.042l-1.25 1.25a3.5 3.5 0 1 1-4.95-4.95l2.5-2.5a3.5 3.5 0 0 1 4.95 0 .751.751 0 0 1-.018 1.042.751.751 0 0 1-1.042.018 1.998 1.998 0 0 0-2.83 0l-2.5 2.5a1.998 1.998 0 0 0 0 2.83Z"></path></svg></a>Stock Time Series (STS) functions (high-level interface)</h2>
<p dir="auto">There is also a high-level interface that calls the low-level functions. The high-level interface has the convenience function:</p>
<div class="snippet-clipboard-content notranslate position-relative overflow-auto" data-snippet-clipboard-copy-content="execute_api_call"><pre class="notranslate"><code>execute_api_call
</code></pre></div>
<p dir="auto">which takes a <a href="https://github.com/Pooksoft/PooksoftAlphaVantageDataStore.jl/blob/master/src/base/Types.jl">PSUserModel</a> and a <a href="https://github.com/Pooksoft/PooksoftAlphaVantageDataStore.jl/blob/master/src/base/Types.jl">PSDataStoreAPICallModel</a> object, along with an optional <a href="https://github.com/kmsquire/Logging.jl">logger</a> instance. Use the help system at the julia prompt for additional information on the <code>execute_api_call</code> function.</p>
