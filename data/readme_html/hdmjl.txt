<ul dir="auto">
<li>This package is a port of the <code>hdm</code> library in R. A collection of methods for estimation and quantification of uncertainty in high-dimensional approximately sparse models, based on:</li>
</ul>
<p dir="auto">Chernozhukov, V., Chetverikov, D., Demirer, M., Duflo, E., Hansen, C., &amp; Newey, W. (2017). Double/debiased/neyman machine learning of treatment effects. American Economic Review, 107(5), 261-65.</p>

<p dir="auto">The Post-Lasso procedure fits an OLS regression excluding the variables not previously selected by Lasso. The <code>rlasso</code> algorithm uses the standard errors of the residuals from this regression to evaluate whether there has been a gain in the goodness of the fit in the current iteration. Just like most of the functions in the package, <code>rlasso</code> returns a dictionary with the results of the regression.</p>

<p dir="auto">Following Chernozhukov, Hansen and Spindler (2015), the <code>HDMjl</code> package makes use of orthogonal estimating equations methods to reduce estimation bias. We can do this through the <code>rlassoEffect</code> function which does orthogonal estimation using <code>double selection</code> by default.</p>
<p dir="auto">We can use this method for the Barro &amp; Lee (1994) dataset, which has a large amount of covariates (61) relative to the sample size (90). Selecting covariates through Post-Lasso gives us more precise estimators.</p>
