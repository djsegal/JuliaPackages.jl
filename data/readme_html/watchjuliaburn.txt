<p>WatchJuliaBurn aims at destroying the look of your code by adding emojis like <g-emoji class="g-emoji" alias="smile" fallback-src="https://github.githubassets.com/images/icons/emoji/unicode/1f604.png">😄</g-emoji> and kaomojis like c╯°□°ↄ╯ instead of your favorite Julia functions.
For a serious use of unicode characters see also <a href="https://gitlab.com/ExpandingMan/Ueauty.jl" rel="nofollow">Ueauty.jl</a></p>
<h2><a id="user-content-add-your-own-awfulness" class="anchor" aria-hidden="true" href="#add-your-own-awfulness"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path fill-rule="evenodd" d="M7.775 3.275a.75.75 0 001.06 1.06l1.25-1.25a2 2 0 112.83 2.83l-2.5 2.5a2 2 0 01-2.83 0 .75.75 0 00-1.06 1.06 3.5 3.5 0 004.95 0l2.5-2.5a3.5 3.5 0 00-4.95-4.95l-1.25 1.25zm-4.69 9.64a2 2 0 010-2.83l2.5-2.5a2 2 0 012.83 0 .75.75 0 001.06-1.06 3.5 3.5 0 00-4.95 0l-2.5 2.5a3.5 3.5 0 004.95 4.95l1.25-1.25a.75.75 0 00-1.06-1.06l-1.25 1.25a2 2 0 01-2.83 0z"></path></svg></a>Add your own awfulness!</h2>
<p>Don't hesitate to add your worst creations via PR. All you need to do is to add the function and emoji to the <code>func_to_emojis</code> internal <code>Dict</code>. Don't touch the <code>README</code>!
It will be automatically generated after your PR is merged. Also tests are optional since tests are for losers!</p>
<h2><a id="user-content-emojify-your-code" class="anchor" aria-hidden="true" href="#emojify-your-code"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path fill-rule="evenodd" d="M7.775 3.275a.75.75 0 001.06 1.06l1.25-1.25a2 2 0 112.83 2.83l-2.5 2.5a2 2 0 01-2.83 0 .75.75 0 00-1.06 1.06 3.5 3.5 0 004.95 0l2.5-2.5a3.5 3.5 0 00-4.95-4.95l-1.25 1.25zm-4.69 9.64a2 2 0 010-2.83l2.5-2.5a2 2 0 012.83 0 .75.75 0 001.06-1.06 3.5 3.5 0 00-4.95 0l-2.5 2.5a3.5 3.5 0 004.95 4.95l1.25-1.25a.75.75 0 00-1.06-1.06l-1.25 1.25a2 2 0 01-2.83 0z"></path></svg></a>Emojify your code</h2>
<p>You can use the <code>emojify</code> function to recursively emojify all the files in a given path. <code>emojify</code> will replace all functions for which an alias is known
by the corresponding emoji (a random one is picked every time if multiple options are possible).
For example:</p>
<div class="highlight highlight-source-julia position-relative" data-snippet-clipboard-copy-content="vcat(round(log(pi)), broadcast(tan ∘ inv, rand(3)))
"><pre><span class="pl-c1">vcat</span>(<span class="pl-c1">round</span>(<span class="pl-c1">log</span>(<span class="pl-c1">pi</span>)), <span class="pl-c1">broadcast</span>(tan <span class="pl-k">∘</span> inv, <span class="pl-c1">rand</span>(<span class="pl-c1">3</span>)))</pre></div>

<div class="highlight highlight-source-julia position-relative" data-snippet-clipboard-copy-content="⬇️😹(🎠(🪵(🥧)), 📡(🧑🏻➡️🧑🏽 ∘ ↔, 🎰(3)))
