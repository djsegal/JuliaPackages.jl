<p><code>Powerlog</code> performs power analysis for logistic regression designs with a
continuous predictor variable.  Multiple predictor models can be accomodated using
the <code>rsq</code> option. This is a port of Stata powerlog.ado, which was originally ported
from the SAS macro program, POWERLOG.SAS.</p>
