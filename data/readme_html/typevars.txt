<p dir="auto">Sometimes it is useful to associate a data object with a parameterized type.
For example, the type <code>Zmod{m}</code> would represent integer calculations modulo <code>m</code>
where <code>m</code> is a samll integer.</p>
<p dir="auto">This tiny package provides a way to extend this if type parameters are not
feasible to keep the data.</p>

<p dir="auto">There are 2 functions that make the user interface:</p>
<p dir="auto"><code>settypevar(::Type, object)</code> is called once per type variant to define the association.</p>
<p dir="auto"><code>typevar(::Type)</code> makes the object available in an efficient way.</p>
