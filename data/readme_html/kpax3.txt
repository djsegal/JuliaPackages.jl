<h2><a id="user-content-about" class="anchor" aria-hidden="true" href="#about"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path fill-rule="evenodd" d="M4 9h1v1H4c-1.5 0-3-1.69-3-3.5S2.55 3 4 3h4c1.45 0 3 1.69 3 3.5 0 1.41-.91 2.72-2 3.25V8.59c.58-.45 1-1.27 1-2.09C10 5.22 8.98 4 8 4H4c-.98 0-2 1.22-2 2.5S3 9 4 9zm9-3h-1v1h1c1 0 2 1.22 2 2.5S13.98 12 13 12H9c-.98 0-2-1.22-2-2.5 0-.83.42-1.64 1-2.09V6.25c-1.09.53-2 1.84-2 3.25C6 11.31 7.55 13 9 13h4c1.45 0 3-1.69 3-3.5S14.5 6 13 6z"></path></svg></a>About</h2>
<p>Kpax3 is a <a href="http://julialang.org/" rel="nofollow">Julia</a> package for inferring the group structure of genetic sequences. In general, any multivariate categorical dataset (such as presence/absence data) can be analyzed by Kpax3. Output consists of a clustering of both the rows (statistical units) and columns (statistical variables) of the provided data matrix. It is an improved version of <a href="https://github.com/albertopessia/kpax2/">K-Pax2</a>, providing an MCMC algorithm for a proper Bayesian approach and a genetic algorithm for MAP estimation.</p>

<p>Kpax3 can be easily installed from within Julia:</p>
<ul>
<li>Enter the Pkg REPL-mode by pressing  <code>]</code> in the Julia REPL</li>
<li>Issue the command <code>add Kpax3</code></li>
<li>Press the Backspace key to return to the Julia REPL</li>
</ul>
