<p dir="auto">Kpax3 is a <a href="http://julialang.org/" rel="nofollow">Julia</a> package for inferring the group structure of genetic sequences. In general, any multivariate categorical dataset (such as presence/absence data) can be analyzed by Kpax3. Output consists of a clustering of both the rows (statistical units) and columns (statistical variables) of the provided data matrix. It is an improved version of <a href="https://github.com/albertopessia/kpax2/">K-Pax2</a>, providing an MCMC algorithm for a proper Bayesian approach and a genetic algorithm for MAP estimation.</p>

<p dir="auto">Kpax3 can be easily installed from within Julia:</p>
<ul dir="auto">
<li>Enter the Pkg REPL-mode by pressing  <code>]</code> in the Julia REPL</li>
<li>Issue the command <code>add Kpax3</code></li>
<li>Press the Backspace key to return to the Julia REPL</li>
</ul>
