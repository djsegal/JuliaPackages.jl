<p dir="auto"><code>RealPolyhedralHomotopy.jl</code> is a package for finding real roots of systems of polynomial equations using polyhedral homotopy.
The package implements the algorithm for the real polyhedral homotopy establised in <a href="https://arxiv.org/abs/1910.01957" rel="nofollow">A Polyhedral Homotopy Algorithm For Real Zeros</a>. The idea for the real polyhedral homotopy motivated from the celebrated article <a href="https://www.jstor.org/stable/2153370?seq=1" rel="nofollow">A Polyhedral Method for Solving Sparse Polynomial Systems</a> to apply the polyhedral homotopy method for real roots finding.</p>
