<p dir="auto"><a href="https://github.com/nickrobinson251/PowerFlowData.jl">PowerFlowData.jl</a>
provides a parser for PSS/E-format <code>.raw</code> Power Flow Data Files.</p>
<p dir="auto">To read a <code>.raw</code> file, use <code>parse_network</code>:</p>
<div class="highlight highlight-source-julia notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="using PowerFlowData
parse_network(&quot;file.raw&quot;)"><pre><span class="pl-k">using</span> PowerFlowData
<span class="pl-c1">parse_network</span>(<span class="pl-s"><span class="pl-pds">"</span>file.raw<span class="pl-pds">"</span></span>)</pre></div>
<p dir="auto">This will return a <code>Network</code> object, which contains the data parsed into dedicated structures matching the
<a href="https://en.wikipedia.org/wiki/Power_system_simulator_for_engineering" rel="nofollow">PSS/E</a>-format specification.</p>
<p dir="auto"><strong><a href="https://nickrobinson251.github.io/PowerFlowData.jl/dev" rel="nofollow">Documentation</a></strong></p>
<p dir="auto">The format specification is based on old PSS/E user-manuals and example files I could find online.
Currently v30 and v33 of the format are supported.
Please open an issue if you run into any problems or missing features.</p>
