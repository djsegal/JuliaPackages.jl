<p dir="auto">This defines two functions, <code>cpu</code> and <code>gpu</code> (similar to Flux.jl), to move <code>RBM</code> and layers to/from the CPU and GPU.</p>

using CudaRBMs: cpu, gpu

# copy to GPU
rbm_cu = gpu(rbm)

# ... do some things with rbm_cu on the GPU (e.g. training, sampling)

# copy back to CPU
rbm = cpu(rbm_cuda)"><pre><span class="pl-k">using</span> RestrictedBoltzmannMachines<span class="pl-k">:</span> BinaryRBM
