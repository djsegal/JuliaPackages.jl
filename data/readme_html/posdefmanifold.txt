<p dir="auto"><strong>PosDefManifold</strong> is a <a href="https://julialang.org/" rel="nofollow"><strong>Julia</strong></a> package for manipulating data in the manifold <strong>P</strong> of real or complex <a href="https://en.wikipedia.org/wiki/Definiteness_of_a_matrix" rel="nofollow"><strong>positive definite matrices</strong></a>. The package supports <strong>10 metrics</strong> acting on <strong>P</strong>, two of which define <a href="https://en.wikipedia.org/wiki/Riemannian_manifold" rel="nofollow"><strong>Riemannian manifolds</strong></a>.</p>
<p dir="auto">This package computes <strong>distances</strong>, <strong>geodesics</strong> and <strong>weighted Fr√©chet means</strong> for the supported metrics, <strong>graph Laplacians</strong>, <strong>Lapacian eigen maps</strong> and a special solution of the <a href="https://en.wikipedia.org/wiki/Orthogonal_Procrustes_problem" rel="nofollow"><strong>Procrustes problem</strong></a> in <strong>P</strong>. Key functions are multi-threaded.</p>
<p dir="auto">The documentation is rich and gives all mathematical details pertaining to the implemented functions (click the 'docs|dev' button).</p>
