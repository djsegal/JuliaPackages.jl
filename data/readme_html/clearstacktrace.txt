<p>An experimental package that hooks into <code>Base.show_backtrace</code> and replaces normal StackTrace printing behavior with a clearer version that indicates Modules, uses alignment and Crayon.jl colors to reduce visual clutter and expands base paths so they are clickable.</p>
<p>All you have to do is install the package and then execute:</p>
<div class="highlight highlight-source-julia"><pre><span class="pl-k">using</span> ClearStacktrace</pre></div>
<p>Example with <code>ClearStacktrace.jl</code>:</p>
<p><a target="_blank" rel="noopener noreferrer" href="img/after.png"><img src="img/after.png" alt="after" style="max-width:100%;"></a></p>
<p>And the default stacktrace:</p>
<p><a target="_blank" rel="noopener noreferrer" href="img/before.png"><img src="img/before.png" alt="before" style="max-width:100%;"></a></p>
<p>Untested on Windows!
Also not registered, yet.</p>
<p>Much of the visual style and behavior can be configured via const Ref values. You can, e.g., choose different preset colors for the modules by changing <code>MODULECRAYONS</code> like this:</p>
<div class="highlight highlight-source-julia"><pre>ClearStacktrace<span class="pl-k">.</span>MODULECRAYONS[] <span class="pl-k">=</span> [<span class="pl-s"><span class="pl-pds"><span class="pl-c1">crayon</span>"</span>blue<span class="pl-pds">"</span></span>, <span class="pl-s"><span class="pl-pds"><span class="pl-c1">crayon</span>"</span>green<span class="pl-pds">"</span></span>]</pre></div>
