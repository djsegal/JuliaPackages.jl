<p dir="auto"><code>ClearStacktrace.jl</code> is an experimental package that replaces <code>Base.show_backtrace</code> with a clearer version that uses alignment and colors to reduce visual clutter and indicate module boundaries, and expands base paths so they are clickable.</p>
