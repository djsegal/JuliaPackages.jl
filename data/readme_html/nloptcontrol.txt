<p>This software solves <strong>nonlinear control problems</strong> at a <strong>high-level</strong> very <strong>quickly</strong>.</p>
<p>Adds to <a href="http://www.juliaopt.org/" rel="nofollow">juliaOpt</a> community by:</p>
<ul>
<li>Providing an implementation of direct-collocation methods for solving optimal control problems in julia</li>
<li>Solving nonlinear optimal control problems at a high-level</li>
<li>Visualizing the solution</li>
</ul>

<p>If you are using <strong>Linux</strong> make sure that you have <strong>gfortran</strong> to run <strong>Ipopt</strong>:</p>
<pre><code>sudo apt-get update
sudo apt-get install gfortran
sudo apt-get install liblapack-dev
sudo apt-get install libblas-dev
</code></pre>
<p>Also, make sure that you are using julia 0.6.2</p>
<pre><code>sudo wget https://julialang-s3.julialang.org/bin/linux/x64/0.6/julia-0.6.2-linux-x86_64.tar.gz 
sudo tar -xvf julia-0.6.2-linux-x86_64.tar.gz -C /opt 
</code></pre>
<p>Then open up julia and install NLOptControl</p>
<div class="highlight highlight-source-julia"><pre>Pkg<span class="pl-k">.</span><span class="pl-c1">clone</span>(<span class="pl-s"><span class="pl-pds">"</span>https://github.com/JuliaMPC/NLOptControl.jl<span class="pl-pds">"</span></span>)
Pkg<span class="pl-k">.</span><span class="pl-c1">pin</span>(<span class="pl-s"><span class="pl-pds">"</span>KNITRO<span class="pl-pds">"</span></span>,<span class="pl-s"><span class="pl-pds"><span class="pl-c1">v</span>"</span>0.4<span class="pl-pds">"</span></span>)</pre></div>
