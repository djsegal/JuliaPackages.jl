<p dir="auto">Ai4EMetaPSE.jl can parse:</p>
<ul dir="auto">
<li>julia ASTs(abstract syntax tree) to a json file</li>
<li>json file to julia AST</li>
<li>a json file that include differential equations and generate julia solution code based on <a href="https://sciml.ai/" rel="nofollow">Sciml Ecosystem</a>.</li>
</ul>
