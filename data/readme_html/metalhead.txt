<p dir="auto"><a href="https://github.com/FluxML/Metalhead.jl">Metalhead.jl</a> provides standard machine learning vision models for use with <a href="https://fluxml.ai" rel="nofollow">Flux.jl</a>. The architectures in this package make use of pure Flux layers, and they represent the best-practices for creating modules like residual blocks, inception blocks, etc. in Flux. Metalhead also provides some building blocks for more complex models in the Layers module.</p>
