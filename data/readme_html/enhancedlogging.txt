<p><a href="https://travis-ci.com/adamslc/EnhancedLogging.jl" rel="nofollow"><img src="https://camo.githubusercontent.com/7bc4e31616dd424d2edf0276db929e82dd31e4b8/68747470733a2f2f7472617669732d63692e636f6d2f6164616d736c632f456e68616e6365644c6f6767696e672e6a6c2e7376673f6272616e63683d6d6173746572" alt="Build Status" data-canonical-src="https://travis-ci.com/adamslc/EnhancedLogging.jl.svg?branch=master" style="max-width:100%;"></a></p>
<p>Extends and improves the Julia language's built in <code>ConsoleLogger</code>.
Additionally adds a <code>WorkerLogger</code> that passes log records to the main process
before processing them.</p>

<h2><a id="user-content-usage" class="anchor" aria-hidden="true" href="#usage"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path fill-rule="evenodd" d="M4 9h1v1H4c-1.5 0-3-1.69-3-3.5S2.55 3 4 3h4c1.45 0 3 1.69 3 3.5 0 1.41-.91 2.72-2 3.25V8.59c.58-.45 1-1.27 1-2.09C10 5.22 8.98 4 8 4H4c-.98 0-2 1.22-2 2.5S3 9 4 9zm9-3h-1v1h1c1 0 2 1.22 2 2.5S13.98 12 13 12H9c-.98 0-2-1.22-2-2.5 0-.83.42-1.64 1-2.09V6.25c-1.09.53-2 1.84-2 3.25C6 11.31 7.55 13 9 13h4c1.45 0 3-1.69 3-3.5S14.5 6 13 6z"></path></svg></a>Usage</h2>
<p>To setup enhanced logging on all running processes, put the following <em>after</em>
all worker processes have been started:</p>
<div class="highlight highlight-source-julia"><pre><span class="pl-c1">@everywhere</span> <span class="pl-k">using</span> Logging, EnhancedLogging
<span class="pl-c1">global_logger</span>(<span class="pl-c1">EnhancedConsoleLogger</span>())
<span class="pl-c1">@everywhere</span> <span class="pl-c1">global_logger</span>(<span class="pl-c1">WorkerLogger</span>(<span class="pl-c1">global_logger</span>()))</pre></div>
<p>This will setup an <code>EnhancedLogger</code> on the master process, and <code>WorkerLogger</code>s
on the workers.</p>
<h2><a id="user-content-limitations" class="anchor" aria-hidden="true" href="#limitations"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path fill-rule="evenodd" d="M4 9h1v1H4c-1.5 0-3-1.69-3-3.5S2.55 3 4 3h4c1.45 0 3 1.69 3 3.5 0 1.41-.91 2.72-2 3.25V8.59c.58-.45 1-1.27 1-2.09C10 5.22 8.98 4 8 4H4c-.98 0-2 1.22-2 2.5S3 9 4 9zm9-3h-1v1h1c1 0 2 1.22 2 2.5S13.98 12 13 12H9c-.98 0-2-1.22-2-2.5 0-.83.42-1.64 1-2.09V6.25c-1.09.53-2 1.84-2 3.25C6 11.31 7.55 13 9 13h4c1.45 0 3-1.69 3-3.5S14.5 6 13 6z"></path></svg></a>Limitations</h2>
<p>Currently, the worker logging only works with global loggers, i.e. using
<code>with_logger</code> will prevent the log message from being forwarded to the master
process. This is mostly due to a limitation in how pointers are serialized for
interprocess communication. If you have ideas about how to circumvent this
problem, please open an issue so that we can discuss.</p>
