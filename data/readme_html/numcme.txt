<p dir="auto"><a href="https://voduchuy.github.io/NumCME.jl/dev/" rel="nofollow"><img src="https://camo.githubusercontent.com/7fcec4b2d3ab291529fce8ef6a4fcd4129a0683b2f5d5fe2f5c648f02db8b616/68747470733a2f2f696d672e736869656c64732e696f2f62616467652f646f63732d6465762d626c75652e737667" alt="" data-canonical-src="https://img.shields.io/badge/docs-dev-blue.svg" style="max-width: 100%;"></a></p>
<p dir="auto">An extensible toolkit for direct numerical solution of the Chemical Master Equation based on the <a href="https://www.atmos.colostate.edu/~munsky/Papers/JChemPhys_124_044104.pdf" rel="nofollow">Finite State Projection</a> and related algorithms.</p>

<p dir="auto">This package aims to offer dynamic, fast, and customizable methods for direct numerical integration of the Chemical Master Equation (CME) in Julia. Currently, it has:</p>
<ul dir="auto">
<li>Transient solution of the CME for time-homogeneous reaction rates/propensities as well as time-varying reaction rates/propensities using Finite State Projection and related variants.</li>
<li>Dynamic state spaces: The truncated state space is adapted on-the-fly to remove redundant states with low probabilities and add more states to ensure the truncation error is within user-specified tolerance.</li>
<li>Customizable: Users can choose how the dynamic state space is managed (by specifying parameters for existing <code>SpaceAdapter</code> subtypes or write their own <code>SpaceAdapter</code>) and how the reduced ODEs are solved (by choosing one among the multitude options offered by <a href="https://github.com/SciML/DifferentialEquations.jl"><code>DifferentialEquations.jl</code></a>). Advanced users can write their own dynamic state space management policy by subtyping <code>SpaceAdapter</code>.</li>
<li>Sensitivity analysis: Compute partial derivatives of the FSP solution with respect to model parameters. Users do not need to write analytic expressions of the propensity's partial derivatives. Instead, the package applies existing tools from <a href="https://github.com/SciML/ModelingToolkit.jl"><code>ModelingToolkit.jl</code></a> and <a href="https://github.com/JuliaDiff/ForwardDiff.jl"><code>ForwardDiff.jl</code></a> to generate those derivatives automatically.</li>
<li>Accept reaction systems defined using <a href="https://catalyst.sciml.ai/dev/" rel="nofollow"><code>Catalyst.jl</code></a> DSL.</li>
</ul>
