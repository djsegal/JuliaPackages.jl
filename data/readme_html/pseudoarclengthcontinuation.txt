<p>This Julia package aims at solving equations F(u,λ)=0 where λ∈ℝ starting from an initial guess (u0,λ0). It relies on the pseudo arclength continuation algorithm which provides a <em>predictor</em> (u1,λ1) from (u0,λ0). A Newton method is then used to correct this predictor.</p>
<p>The package actually does a little more. By leveraging on the above method, it can also seek for periodic orbits of Cauchy problems by casting them into an equation F(u,λ)=0 of high dimension. <strong>It is by now, one of the only softwares which provides shooting methods AND methods based on finite differences to compute periodic orbits.</strong></p>
<p>The current package focuses on large scale nonlinear problems and multiple hardwares. Hence, the goal is to use Matrix Free methods on <strong>GPU</strong> (see <a href="https://rveltz.github.io/PseudoArcLengthContinuation.jl/dev/tutorials2b/index.html#The-Swift-Hohenberg-equation-on-the-GPU-1" rel="nofollow">PDE example</a> and <a href="https://rveltz.github.io/PseudoArcLengthContinuation.jl/dev/tutorialsCGL/#Continuation-of-periodic-orbits-on-the-GPU-(Advanced)-1" rel="nofollow">Periodic orbit example</a>) or on a <strong>cluster</strong> to solve non linear PDE, nonlocal problems, compute sub-manifolds...</p>

<h2><a id="user-content-main-features" class="anchor" aria-hidden="true" href="#main-features"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path fill-rule="evenodd" d="M4 9h1v1H4c-1.5 0-3-1.69-3-3.5S2.55 3 4 3h4c1.45 0 3 1.69 3 3.5 0 1.41-.91 2.72-2 3.25V8.59c.58-.45 1-1.27 1-2.09C10 5.22 8.98 4 8 4H4c-.98 0-2 1.22-2 2.5S3 9 4 9zm9-3h-1v1h1c1 0 2 1.22 2 2.5S13.98 12 13 12H9c-.98 0-2-1.22-2-2.5 0-.83.42-1.64 1-2.09V6.25c-1.09.53-2 1.84-2 3.25C6 11.31 7.55 13 9 13h4c1.45 0 3-1.69 3-3.5S14.5 6 13 6z"></path></svg></a>Main features</h2>
<ul>
<li>Matrix Free Newton solver with generic linear / eigen <em>preconditioned</em> solver. Idem for the arc-length continuation.</li>
<li>Matrix Free Newton solver with deflation and preconditioner. It can be used for branch switching for example.</li>
<li>Bifurcation points are located using a bisection algorithm</li>
<li>Branch, Fold, Hopf bifurcation point detection of stationary solutions.</li>
<li>Fold / Hopf continuation based on Minimally Augmented formulation, with Matrix Free / Sparse Jacobian.</li>
<li>Periodic orbit computation and continuation using Shooting or Finite Differences.</li>
<li>Branch, Fold, Neimark-Sacker, Period Doubling bifurcation point detection of periodic orbits.</li>
<li>Computation and Continuation of Fold of periodic orbits</li>
</ul>
<p>Custom state means, we can use something else than <code>AbstractArray</code>, for example your own <code>struct</code>.</p>
<p><strong>Note that you can combine most of the solvers, like use Deflation for Periodic orbit computation or Fold of periodic orbits family.</strong></p>
