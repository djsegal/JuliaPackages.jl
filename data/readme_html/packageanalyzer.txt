<p dir="auto">Package to analyze the prevalence of documentation, testing and continuous
integration in Julia packages in a given registry.</p>

<p dir="auto">The package works on Julia v1.6 and following versions.</p>
<p dir="auto">To install the package, in Julia's REPL, press <code>]</code> to enter the Pkg mode and run
the command</p>

</code></pre></div>
