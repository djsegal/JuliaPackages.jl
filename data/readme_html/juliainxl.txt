<p dir="auto">JuliaInXL.jl is a Julia package that allows Julia functions to be called from within Microsoft Excel. This software
included Julia code that is present in this repository, and a corresponding <a href="https://github.com/JuliaComputing/JuliaInXL">native (ie XLL) plugin</a></p>

<p dir="auto">The Julia package can be installed directly using the package manager.</p>
<div class="snippet-clipboard-content notranslate position-relative overflow-auto" data-snippet-clipboard-copy-content="&gt; add JuliaInXL"><pre class="notranslate"><code>&gt; add JuliaInXL

<p dir="auto">As part of instaling the Julia package, the native plugin will be downloaded, and it's installer executed.</p>
<p dir="auto">Upon completion of the installer, press close to exit the installer.</p>

<p dir="auto">If JuliaInXL was installed via the package manager, then in most cases a Julia process should launch automatically when starting your Excel session.</p>
<p dir="auto"><a target="_blank" rel="noopener noreferrer" href="docs/src/assets/media/image77.png"><img src="docs/src/assets/media/image77.png" alt="" style="max-width: 100%;"></a></p>
<p dir="auto">A Julia tab will also be present in the Office Ribbon that contains a number of buttons and text boxes for controlling the connection between Julia and Excel, as well as loading functionality into the current Julia process.</p>
<p dir="auto"><a target="_blank" rel="noopener noreferrer" href="docs/src/assets/media/image78.png"><img src="docs/src/assets/media/image78.png" alt="" style="max-width: 100%;"></a></p>
<p dir="auto">Julia functions can be called from Excel using the jlcall worksheet function. The first argument to jlcall is a string, which is the name of the registered Julia function to be called. Subsequent arguments to the jlcall function are passed as parameters to the Julia function being called. These can be constant literals, or cell references</p>
