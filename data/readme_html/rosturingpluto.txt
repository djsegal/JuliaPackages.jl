<p dir="auto">This project will contain (work is in early stages of progress!) a set of Pluto notebooks that contain Julia versions of the examples in the R project <code>ROS-Examples</code> based on the book <a href="https://www.cambridge.org/highereducation/books/regression-and-other-stories/DD20DD6C9057118581076E54E40C372C#overview" rel="nofollow">"Regression and Other Stories" by A Gelman, J Hill and A Vehtari</a>.</p>
<p dir="auto">These notebooks are intended to be used in conjunction with above book.</p>
<p dir="auto">Each notebook contains a chapter.  In the <code>pdfs</code> directory there are also PDF versions of the chapters created with PlutoPDF (see notes.md).</p>

<p dir="auto">This project will take quite a while to complete, I expect at least a year. But it has a special meaning to me: When I started to work on Julia interfaces for Stan's cmdstan binary in 2011, I did that to work through the <a href="http://www.stat.columbia.edu/~gelman/arm/" rel="nofollow">"ARM" book</a>. The <a href="https://www.cambridge.org/highereducation/books/regression-and-other-stories/DD20DD6C9057118581076E54E40C372C#overview" rel="nofollow">"ROS" book</a> in a sense the successor to the ARM book.</p>

<div class="highlight highlight-source-julia notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="$ cd ~/.julia/dev
