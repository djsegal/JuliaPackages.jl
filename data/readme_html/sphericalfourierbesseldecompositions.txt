<p dir="auto"><em>SuperFaB</em> is a code for cosmological spherical Fourier-Bessel (SFB) analysis.
The details of the code are presented in <a href="https://arxiv.org/abs/2102.10079" rel="nofollow">2102.10079</a>.</p>
<p dir="auto">The name of this package is <strong>SphericalFourierBesselDecompositions.jl</strong>.</p>
