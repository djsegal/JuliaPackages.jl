<p dir="auto">TerminalLoggers provides a logger type <code>TerminalLogger</code> which can format your
log messages in a richer way than the default <code>ConsoleLogger</code> which comes with
the julia standard <code>Logging</code> library.</p>
<p dir="auto">Read the <a href="https://JuliaLogging.github.io/TerminalLoggers.jl/stable" rel="nofollow">documentation</a> for
more information.</p>
