<p>This package provides auditory filter implementations in Julia. The following implementations are available:</p>
<ul>
<li>A gammatone filterbank based on Malcolm Slaney's <a href="https://engineering.purdue.edu/~malcolm/interval/1998-010/" rel="nofollow">Auditory Toolbox</a>;</li>
<li>a gammatone-like spectrogram implementation, based on Dan Ellis' <a href="http://www.ee.columbia.edu/ln/rosa/matlab/gammatonegram/" rel="nofollow">implementation</a>;</li>
<li>a modulation filterbank based on the paper titled "Characterizing frequency selectivity for envelope
fluctuations" (2000), by Ewert and Dau.</li>
</ul>
