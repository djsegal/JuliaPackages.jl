<p dir="auto">Robot Motion Estimate (RoME.jl) is part of the overall <a href="https://github.com/JuliaRobotics/Caesar.jl">Caesar.jl</a> and provides a set of graph variables, factors, and utility features for robotics-related navigation, tracking, and mapping (i.e. SLAM).  RoME.jl helps build front-ends using the [Multi-modal iSAM] backend solver which is implemented over at <a href="https://github.com/JuliaRobotics/IncrementalInference.jl">IncrementalInference.jl</a>.  See <a href="http://www.juliarobotics.org/Caesar.jl/latest/refs/literature/" rel="nofollow">the related references of interest here</a>.  Most notably, this package provides common navigation-type variables and factors to be included in more general <a href="https://github.com/JuliaRobotics/DistributedFactorGraphs.jl">DistributedFactorGraphs.jl</a> graph objects.</p>
<p dir="auto"><a href="http://www.navability.io" rel="nofollow">NavAbility.io</a> helps the with administration and support of the Caesar.jl community, please reach out for any additional information (<a href="mailto:info@navability.io">info@navability.io</a>) or via the caesarjl Slack badge-link above.</p>
