<p>Robot Motion Estimate: A set of functions for developing front-ends for SLAM in <a href="www.julialang.org">Julia</a> which adds transform, visualization and convenience functions to the <a href="http://frc.ri.cmu.edu/~kaess/pub/Fourie16iros.pdf" rel="nofollow">Multi-modal iSAM</a> backend solver. The back-end solver is implemented in <a href="https://github.com/JuliaRobotics/IncrementalInference.jl">IncrementalInference.jl</a>.</p>
<h2><a id="user-content-introduction" class="anchor" aria-hidden="true" href="#introduction"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path fill-rule="evenodd" d="M7.775 3.275a.75.75 0 001.06 1.06l1.25-1.25a2 2 0 112.83 2.83l-2.5 2.5a2 2 0 01-2.83 0 .75.75 0 00-1.06 1.06 3.5 3.5 0 004.95 0l2.5-2.5a3.5 3.5 0 00-4.95-4.95l-1.25 1.25zm-4.69 9.64a2 2 0 010-2.83l2.5-2.5a2 2 0 012.83 0 .75.75 0 001.06-1.06 3.5 3.5 0 00-4.95 0l-2.5 2.5a3.5 3.5 0 004.95 4.95l1.25-1.25a.75.75 0 00-1.06-1.06l-1.25 1.25a2 2 0 01-2.83 0z"></path></svg></a>Introduction</h2>
<p>This package forms part of the <a href="https://github.com/JuliaRobotics/Caesar.jl">Caesar.jl</a> robot state estimate toolkit, towards robust solutions in robot navigation and mapping, which includes visualization and database interaction features. Robot style wrapper function and front-end factor graph generation functions are provided. Plot based visualization of robot belief based navigation variables is provided.</p>
