<p>Robot Motion Estimate: A set of functions for developing front-ends for SLAM in <a href="www.julialang.org">Julia</a> which adds transform, visualization and convenience functions to the <a href="http://frc.ri.cmu.edu/~kaess/pub/Fourie16iros.pdf" rel="nofollow">Multi-modal iSAM</a> backend solver. The back-end solver is implemented in <a href="https://github.com/JuliaRobotics/IncrementalInference.jl">IncrementalInference.jl</a>.</p>
<h2><a id="user-content-introduction" class="anchor" aria-hidden="true" href="#introduction"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path fill-rule="evenodd" d="M4 9h1v1H4c-1.5 0-3-1.69-3-3.5S2.55 3 4 3h4c1.45 0 3 1.69 3 3.5 0 1.41-.91 2.72-2 3.25V8.59c.58-.45 1-1.27 1-2.09C10 5.22 8.98 4 8 4H4c-.98 0-2 1.22-2 2.5S3 9 4 9zm9-3h-1v1h1c1 0 2 1.22 2 2.5S13.98 12 13 12H9c-.98 0-2-1.22-2-2.5 0-.83.42-1.64 1-2.09V6.25c-1.09.53-2 1.84-2 3.25C6 11.31 7.55 13 9 13h4c1.45 0 3-1.69 3-3.5S14.5 6 13 6z"></path></svg></a>Introduction</h2>
<p>This package forms part of the <a href="https://github.com/JuliaRobotics/Caesar.jl">Caesar.jl</a> robot state estimate toolkit, towards robust solutions in robot navigation and mapping, which includes visualization and database interaction features. Robot style wrapper function and front-end factor graph generation functions are provided. Plot based visualization of robot belief based navigation variables is provided.</p>

<p>You can directly install all the RoME and Multi-modal iSAM functionality in Julia 0.6+ with:</p>
<pre><code>julia&gt; Pkg.add("RoME")
</code></pre>
<p>If you are interested in a broader toolkit, which includes a visualizer and database interaction, please see <a href="https://github.com/dehann/Caesar.jl">Caesar.jl</a>.</p>
