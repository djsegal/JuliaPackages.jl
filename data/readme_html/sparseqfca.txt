<p dir="auto"><em>sparseQFCA</em> is a registered <a href="https://julialang.org/" rel="nofollow"><img src="https://camo.githubusercontent.com/f0be924fb2a870a1cb2f3f69bd70764f55a22562894bc014a2a056ca28e1004d/68747470733a2f2f6a756c69616c616e672e6f72672f6173736574732f696e6672612f6c6f676f2e737667" height="20" data-canonical-src="https://julialang.org/assets/infra/logo.svg" style="max-width: 100%;"></a> package for the sparse <a href="https://mtefagh.github.io/qfca/" rel="nofollow">Quantitative Flux Coupling Analysis</a>. Moreover, a Julia implementation of <a href="https://mtefagh.github.io/swiftcore/" rel="nofollow">Swift Consistency Checking</a> is also available as a preprocessing subroutine. The example data files <code>S.csv</code> and <code>rev.csv</code> are extracted from the <a href="http://systemsbiology.ucsd.edu/Downloads/EcoliCore" rel="nofollow">core <em>E. coli</em> model</a>.</p>

<ul dir="auto">
<li><code>S</code>: the associated sparse <strong>stoichiometric matrix</strong></li>

<h3 dir="auto"><a id="user-content-outputs" class="anchor" aria-hidden="true" href="#outputs"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path d="m7.775 3.275 1.25-1.25a3.5 3.5 0 1 1 4.95 4.95l-2.5 2.5a3.5 3.5 0 0 1-4.95 0 .751.751 0 0 1 .018-1.042.751.751 0 0 1 1.042-.018 1.998 1.998 0 0 0 2.83 0l2.5-2.5a2.002 2.002 0 0 0-2.83-2.83l-1.25 1.25a.751.751 0 0 1-1.042-.018.751.751 0 0 1-.018-1.042Zm-4.69 9.64a1.998 1.998 0 0 0 2.83 0l1.25-1.25a.751.751 0 0 1 1.042.018.751.751 0 0 1 .018 1.042l-1.25 1.25a3.5 3.5 0 1 1-4.95-4.95l2.5-2.5a3.5 3.5 0 0 1 4.95 0 .751.751 0 0 1-.018 1.042.751.751 0 0 1-1.042.018 1.998 1.998 0 0 0-2.83 0l-2.5 2.5a1.998 1.998 0 0 0 0 2.83Z"></path></svg></a>Outputs</h3>
<ul dir="auto">
<li><code>certificates</code>: the fictitious metabolites for the sparse <strong>positive certificates</strong></li>
<li><code>blocked</code>: the boolean vector with trues corresponding to the <strong>blocked reactions</strong></li>
<li><code>fctable</code>: the resulting <strong>flux coupling</strong> matrix</li>
</ul>
