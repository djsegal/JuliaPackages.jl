<p>This package provides support for loading and saving images using
native libraries on macOS.  This package was split off from
<a href="https://github.com/timholy/Images.jl">Images.jl</a> to make image I/O
more modular.</p>

<p>QuartzImageIO will be used as needed if you've said</p>
<pre><code>using FileIO
</code></pre>
<p>in your session or module. You should <strong>not</strong> generally say <code>using QuartzImageIO</code>.  See <a href="https://github.com/JuliaIO/FileIO.jl">FileIO</a> for
further details.</p>
<p>It's worth pointing out that packages such as Images load FileIO.</p>

<p>If QuartzImageIO does not provide the functionality you need, an
alternative is
<a href="https://github.com/JuliaIO/ImageMagick.jl">ImageMagick</a>. You can have
both packages installed, and FileIO will manage their interaction.</p>
