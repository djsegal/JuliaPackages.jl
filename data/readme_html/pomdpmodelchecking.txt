<p dir="auto">This package provides support for performing verification and policy synthesis in POMDPs from LTL formulas. It relies on <a href="https://github.com/JuliaPOMDP/POMDPs.jl">POMDPs.jl</a> for expressing the model and <a href="https://github.com/sisl/Spot.jl">Spot.jl</a> for manipulating LTL formulas.</p>
<p dir="auto">If this package is useful to you, consider citing: M. Bouton, J. Tumova, and M. J. Kochenderfer, "Point-Based Methods for Model Checking in Partially Observable Markov Decision Processes," in <em>AAAI Conference on Artificial Intelligence (AAAI)</em>, 2020.</p>

<p dir="auto">This package exports two solvers: <code>ReachabilitySolver</code> and <code>ModelCheckingSolver</code>. Those solvers are intended to be used on models implemented with <code>POMDPs.jl</code>, please refer to the <code>POMDPs.jl</code> documentation to learn how to implement a POMDP or MDP model using the correct interface.</p>
<p dir="auto"><strong>Interface with <a href="http://www.stormchecker.org/" rel="nofollow">Storm</a> :</strong></p>
<p dir="auto">A writer is already written to convert MDP to the good format, a solver interface has been prototyped, relying on the python library  <a href="https://moves-rwth.github.io/stormpy/" rel="nofollow">stormpy</a>. The files are in the <code>legacy/</code> folder but are only experimental for now.</p>
