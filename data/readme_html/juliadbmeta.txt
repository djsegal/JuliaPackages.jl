<p>JuliaDBMeta is a set of macros to simplify data manipulation with <a href="https://github.com/JuliaComputing/JuliaDB.jl">JuliaDB</a>, heavily inspired on <a href="https://github.com/JuliaStats/DataFramesMeta.jl">DataFramesMeta</a>. It exploits the technical advantages of JuliaDB:</p>
<ul>
<li>Fully typed tables with type stable column extraction</li>
<li>Fast row iteration</li>
<li>Parallel data storage and parallel computations</li>
</ul>
<p>Some ideas also come from <a href="https://github.com/davidanthoff/Query.jl">Query.jl</a>, in particular the curly bracket syntax is from there.</p>
<p>The macro packages <a href="https://github.com/MikeInnes/Lazy.jl">Lazy</a> and <a href="https://github.com/MikeInnes/MacroTools.jl">MacroTools</a> were also very useful in designing this package: the <code>@apply</code> macro is inspired by the concatenation macros in Lazy.</p>
