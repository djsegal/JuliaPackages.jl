<p dir="auto">This package implements a discretize-and-relax approaches to
computing state bounds and relaxations using the DynamicBounds.jl framework. These methods discretize the time domain over into a finite number of points and then compute valid
relaxations at these time-points. Full documentation of this functionality may be found <a href="https://psorlab.github.io/DynamicBounds.jl/dev/pODEsDiscrete/pODEsDiscrete" rel="nofollow">here</a> in the DynamicBounds.jl website.</p>
