<p dir="auto"><a href="https://travis-ci.com/Seelengrab/IOLogging.jl" rel="nofollow"><img src="https://camo.githubusercontent.com/f9cadb652be76ab2f6948250753adfe8004960851f9abd2ab97ed8ab43dd150f/68747470733a2f2f7472617669732d63692e636f6d2f5365656c656e677261622f494f4c6f6767696e672e6a6c2e7376673f6272616e63683d6d6173746572" alt="Build Status" data-canonical-src="https://travis-ci.com/Seelengrab/IOLogging.jl.svg?branch=master" style="max-width: 100%;"></a></p>
<p dir="auto">A simple, thin package providing basic loggers for logging to IO. As the logging functionality from Base might change in the future, so will this package.</p>

<p dir="auto">This package is registered with METADATA.jl, so you can just do <code>]add IOLogging</code> to install the package.</p>

<p dir="auto">We can also pass our own destinations for Logging:</p>
<div class="highlight highlight-source-julia notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="# default is stdout for everything above Info
julia&gt; logger = IOLogger(Dict(Logging.Info =&gt; stderr, Logging.Error =&gt; devnull))"><pre><span class="pl-c"><span class="pl-c">#</span> default is stdout for everything above Info</span>
julia<span class="pl-k">&gt;</span> logger <span class="pl-k">=</span> <span class="pl-c1">IOLogger</span>(<span class="pl-c1">Dict</span>(Logging<span class="pl-k">.</span>Info <span class="pl-k">=&gt;</span> <span class="pl-c1">stderr</span>, Logging<span class="pl-k">.</span>Error <span class="pl-k">=&gt;</span> <span class="pl-c1">devnull</span>))</pre></div>
<p dir="auto">The same as above applies to <code>FileLogger()</code> as well, but instead of giving destination IO, we specify a destination file.</p>
<div class="highlight highlight-source-julia notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="# default is default.log for everything above Info
julia&gt; logger = FileLogger(Dict(Logging.Info =&gt; &quot;info.log&quot;, Logging.Error =&gt; &quot;error.log&quot;))"><pre><span class="pl-c"><span class="pl-c">#</span> default is default.log for everything above Info</span>
julia<span class="pl-k">&gt;</span> logger <span class="pl-k">=</span> <span class="pl-c1">FileLogger</span>(<span class="pl-c1">Dict</span>(Logging<span class="pl-k">.</span>Info <span class="pl-k">=&gt;</span> <span class="pl-s"><span class="pl-pds">"</span>info.log<span class="pl-pds">"</span></span>, Logging<span class="pl-k">.</span>Error <span class="pl-k">=&gt;</span> <span class="pl-s"><span class="pl-pds">"</span>error.log<span class="pl-pds">"</span></span>))</pre></div>
<p dir="auto">For more information about the individual loggers, make sure to read <code>?IOLogger</code> and <code>?FileLogger</code>.</p>
