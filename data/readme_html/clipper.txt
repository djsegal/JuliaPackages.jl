<p>Clipper.jl is a Julia wrapper for <a href="http://www.angusj.com/delphi/clipper.php" rel="nofollow">Angus Johnson's Clipper library (ver. 6.4.2).</a>.</p>
<blockquote>
<p>Clipper - an open source freeware library for clipping and offsetting lines and polygons.</p>
</blockquote>
<blockquote>
<p>The Clipper library performs line &amp; polygon clipping - intersection, union, difference &amp; exclusive-or, and line &amp; polygon offsetting. The library is based on Vatti's clipping algorithm.</p>
</blockquote>
