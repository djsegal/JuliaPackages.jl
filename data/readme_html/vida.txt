<p><code>VIDA.jl</code> or the <em>Variational Image Domain Analysis</em> provides a interface to extracting features from fits images created for the EHT, using the notion of probability divergences similar to variational inference, hence the name. The currently implemented divergences are the Bhattacharyya distance/divergence as well as the Kullback-Leibler divergence. These are used to extract ring-like features from image reconstructions of black holes such as from M87.</p>
<p>For more information on how to use VIDA please refer to the <a href="https://ptiede.github.io/VIDA.jl/stable" rel="nofollow">documentation</a></p>
