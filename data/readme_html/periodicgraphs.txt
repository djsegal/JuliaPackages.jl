<p dir="auto">This module provides the new <code>PeriodicGraph{N}</code> type to manipulate <code>N</code>-dimensional periodic
graphs, extending the <code>AbstractGraph</code> API from <a href="https://github.com/JuliaGraphs/Graphs.jl/">Graphs.jl</a>.</p>
<p dir="auto">Various optimized algorithms are implemented as well, including:</p>
<ul dir="auto">
<li>Neighborhood exploration and coordination sequence computation</li>
<li>Separation by dimension of the connected components</li>
<li>Simple ring statistics</li>
</ul>
<p dir="auto">and more, see <a href="https://liozou.github.io/PeriodicGraphs.jl/" rel="nofollow">the documentation</a>.</p>
<p dir="auto">See also:</p>
<ul dir="auto">
<li><a href="https://github.com/Liozou/PeriodicGraphEmbeddings.jl">PeriodicGraphEmbeddings.jl</a>
for a dependent package specialized on the manipulation of periodic graph embeddings,
including symmetry computations for 3D graphs</li>
<li><a href="https://github.com/coudertlab/CrystalNets.jl">CrystalNets.jl</a> for a dependent package
specialized on crystal nets, including a periodic graph canonization algorithm for 3D,
2D and 1D periodic graphs.</li>
</ul>
