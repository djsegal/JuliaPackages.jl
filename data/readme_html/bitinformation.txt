<p dir="auto">BitInformation.jl is a package for bitwise information analysis and manipulation in Julia arrays.
Based on counting the occurrences of bits in floats (or generally any bits type) across various dimensions,
this package calculates quantities like the bitwise real information content, the mutual information, the
redundancy or preserved information between arrays. From v0.5 onwards masked arrays are also supported.</p>
<p dir="auto">For bitwise manipulation, BitInformation.jl also implements various rounding modes (IEEE round,shave,set_one, etc.)
efficiently with bitwise operations for any number of bits. E.g. <code>round(x,i)</code> implements IEEE's round to nearest
tie-to-even for any float retaining <code>i</code> mantissa bits. Furthermore, transormations like XOR-delta, bittranspose
(aka bit shuffle), or signed/biased exponents are implemented.</p>
<p dir="auto">If you'd like to propose changes, or contribute in any form create a

or raise an <a href="https://github.com/milankl/BitInformation.jl/issues">issue</a>.
Contributions are highly appreciated!</p>
