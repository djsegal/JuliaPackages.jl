<p dir="auto">The transition matrix method, or T-Matrix method, is one of the most powerful and widely used tools for rigorously computing electromagnetic scattering by single and compounded particles. As a package focusing on this method, <code>TransitionMatrices.jl</code> provides the following features:</p>
<ul dir="auto">
<li>Calculate the T-Matrix of various types of scatterers
<ul dir="auto">
<li>Homogeneous spheres (via <code>bhmie</code>)</li>
<li>Coated spheres (via <code>bhcoat</code>)</li>
<li>Homogeneous axisymmetric shapes (via EBCM and IITM)
<ul dir="auto">
<li>Spheroids</li>
<li>Cylinders</li>
<li>Chebyshev particles</li>
</ul>
</li>
<li>Arbitrary shapes (via IITM)
<ul dir="auto">
<li>Prisms</li>
</ul>
</li>
</ul>
</li>
<li>Calculate far-field scattering properties using the T-Matrix
<ul dir="auto">
<li>Cross sections and single scattering albedo (SSA)</li>
<li>Amplitude scattering matrix</li>
<li>Phase matrix</li>
<li>Scattering matrix</li>
</ul>
</li>
</ul>
<p dir="auto">Compared to existing packages, <code>TransitionMatrices.jl</code> is special in that it is generic and supports various floating-point types, e.g.:</p>
<ul dir="auto">
<li><code>Float64</code> and <code>BigFloat</code> from <a href="https://docs.julialang.org/en/v1/base/" rel="nofollow"><code>Base</code></a></li>
<li><code>Double64</code> from <a href="https://github.com/JuliaMath/DoubleFloats.jl"><code>DoubleFloats.jl</code></a></li>
<li><code>Float128</code> from <a href="https://github.com/JuliaMath/Quadmath.jl"><code>Quadmath.jl</code></a></li>
<li><code>Arb</code> from <a href="https://github.com/kalmarek/Arblib.jl"><code>Arblib.jl</code></a></li>
<li><code>ArbFloat</code> from <a href="https://github.com/JeffreySarnoff/ArbNumerics.jl"><code>ArbNumerics.jl</code></a></li>
</ul>
<p dir="auto">By using higher-precision floating-point types, the maximum size parameter that can be handled is greatly improved.</p>
