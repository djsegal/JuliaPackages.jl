<p dir="auto">Algorithms is an API for common search and sort algorithms.  The package would aim to be:</p>
<ul dir="auto">
<li><strong>Universal</strong> Serve as a one-stop library for implementation of common search and sort algorithms both in competitive programming, competitive data science and production</li>
<li><strong>Lightweight</strong> Have the minimal possible dependencies, which makes it super easy and lightweight to use</li>
<li><strong>Intuitive</strong> Use the API without worrying about the naming convention, and make the package very less documentation dependent.</li>
</ul>
