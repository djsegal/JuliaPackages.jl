<p>This package is created to register the <a href="https://github.com/adobe-type-tools/agl-aglfn">Adobe AGL &amp; AGLFN open source project</a> (also known as <code>agl-aglfn</code>) into the Julia package repository. Anyone using Adobe fonts for Julia applications can refer to this project to access the Adobe glyph names. Please note that this Julia package is not endorsed by Adobe. Hence, the README has been modified.</p>
<p>The README for Adobe AGL &amp; AGLFN open source project can be accessed from <a href="https://github.com/adobe-type-tools/agl-aglfn/blob/master/README.md">here</a></p>
