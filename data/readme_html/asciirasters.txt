<p dir="auto">Simple read and write functions for ASCII raster files. The ASCII format convention used is <a href="https://gdal.org/drivers/raster/aaigrid.html" rel="nofollow">AAIGrid</a> (the QGIS default for ASCII rasters).</p>

<p dir="auto"><code>read_ascii(filename::AbstractString)</code> reads a file and returns its contained data and corresponding header information in a <code>Tuple</code>.</p>
<p dir="auto"><code>write_ascii(filename, dat; kwargs...)</code> writes a file. Required keywords arguments are (see docs): <code>ncols</code>, <code>nrows</code>, <code>xll</code>, <code>yll</code>, <code>dx</code>, <code>dy</code> and <code>nodatavalue</code>.</p>
