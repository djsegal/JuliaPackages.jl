<p dir="auto">Tools for working with metadata of Tables.jl tables in Julia.</p>
<p dir="auto">Currently it defines and exports:</p>
<ul dir="auto">
<li><code>label</code>, <code>label!</code>, <code>labels</code>, and <code>findlabels</code> functions for convenient work
with column label metadata;</li>
<li><code>caption</code> and <code>caption!</code> functions for convenient work with table caption
metadata;</li>
<li><code>note</code> and <code>note!</code> functions for convenient work with note metadata both on
table and column level;</li>
<li><code>unit</code> (re-exported from Unitful.jl), <code>unit!</code>, and <code>units</code> functions for
convenient work with column unit metadata;</li>
<li><code>setmetadatastyle!</code>, <code>setcolmetadatastyle!</code>, <code>setallmetadatastyle!</code> for group
setting style for keys matching a passed pattern; usually needed when working
with metadata that initially has <code>:default</code> style set and one wants it to
have <code>:note</code> style (common when reading metadata from storage formats that do
not support metadata style information);</li>
<li><code>meta2toml</code> and <code>toml2meta!</code> for storing and loading metadata in TOML format;</li>
<li><code>dict2metadata!</code>, <code>dict2colmetadata!</code> for setting table and column level
metadata stored in a dictionary (e.g. earlier retrieved from some storage
format or by using <code>metadata</code> or <code>colmetadata</code> functions).</li>
<li>the <code>@track</code> macro for tracking operations applied to table and
the <code>tracklog</code> function for convenient printing of tracking metadata.</li>
</ul>

<p dir="auto">All exported functions have docstrings explaining their behavior.</p>
<p dir="auto">You can find a demo how the package can be used to work with metadata
in /docs/demo.ipynb Jupyter Notebook.</p>
<p dir="auto">To run it have your terminal in the /docs folder and do the following steps:</p>
<ul dir="auto">
<li>start Julia with <code>julia --project</code></li>
<li>to make sure all packages are instantiated correctly run
<code>using Pkg; Pkg.instantiate()</code> (this has to be done only the first time
you run the demo)</li>
<li>start Jupyter Notebook by writing <code>using IJulia; notebook(dir=pwd())</code>.</li>
<li>open <code>demo.ipynb</code> notebook.</li>
</ul>
