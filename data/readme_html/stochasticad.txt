<p dir="auto">StochasticAD is an experimental, research package for automatic differentiation (AD) of stochastic programs. It implements AD algorithms for handling programs that can contain <em>discrete</em> randomness, based on the methodology developed in <a href="https://doi.org/10.48550/arXiv.2210.08572" rel="nofollow">this NeurIPS 2022 paper</a>. We're still working on docs and code cleanup!</p>
