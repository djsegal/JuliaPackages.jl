<p dir="auto">Implementation of OpenAI's ES in julia using the native threads package and MPI. Based on two papers by uber AI labs <a href="https://arxiv.org/abs/1712.06567" rel="nofollow">here</a> and <a href="https://arxiv.org/abs/1712.06560" rel="nofollow">here</a>.<br>
This was created because my <a href="https://github.com/sash-a/es_pytorch">python implementation</a> was too slow, some very informal testing suggest that this repo is at least 10x faster on 8 cores.</p>
