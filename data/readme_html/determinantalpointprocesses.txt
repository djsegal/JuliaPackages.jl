<p dir="auto">An efficient implementation of Determinantal Point Processes (DPP) in Julia.</p>
<h3 dir="auto"><a id="user-content-current-features" class="anchor" aria-hidden="true" href="#current-features"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path d="m7.775 3.275 1.25-1.25a3.5 3.5 0 1 1 4.95 4.95l-2.5 2.5a3.5 3.5 0 0 1-4.95 0 .751.751 0 0 1 .018-1.042.751.751 0 0 1 1.042-.018 1.998 1.998 0 0 0 2.83 0l2.5-2.5a2.002 2.002 0 0 0-2.83-2.83l-1.25 1.25a.751.751 0 0 1-1.042-.018.751.751 0 0 1-.018-1.042Zm-4.69 9.64a1.998 1.998 0 0 0 2.83 0l1.25-1.25a.751.751 0 0 1 1.042.018.751.751 0 0 1 .018 1.042l-1.25 1.25a3.5 3.5 0 1 1-4.95-4.95l2.5-2.5a3.5 3.5 0 0 1 4.95 0 .751.751 0 0 1-.018 1.042.751.751 0 0 1-1.042.018 1.998 1.998 0 0 0-2.83 0l-2.5 2.5a1.998 1.998 0 0 0 0 2.83Z"></path></svg></a>Current features</h3>
<ul dir="auto">
<li>Exact sampling [1] from DPP and k-DPP (can be executed in parallel).</li>
<li>MCMC sampling [2] from DPP and k-DPP (parallelization will be added).</li>
<li><code>pdf</code> and <code>logpdf</code> evaluation functions [1] for DPP and k-DPP.</li>
</ul>
<h3 dir="auto"><a id="user-content-planned-features" class="anchor" aria-hidden="true" href="#planned-features"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path d="m7.775 3.275 1.25-1.25a3.5 3.5 0 1 1 4.95 4.95l-2.5 2.5a3.5 3.5 0 0 1-4.95 0 .751.751 0 0 1 .018-1.042.751.751 0 0 1 1.042-.018 1.998 1.998 0 0 0 2.83 0l2.5-2.5a2.002 2.002 0 0 0-2.83-2.83l-1.25 1.25a.751.751 0 0 1-1.042-.018.751.751 0 0 1-.018-1.042Zm-4.69 9.64a1.998 1.998 0 0 0 2.83 0l1.25-1.25a.751.751 0 0 1 1.042.018.751.751 0 0 1 .018 1.042l-1.25 1.25a3.5 3.5 0 1 1-4.95-4.95l2.5-2.5a3.5 3.5 0 0 1 4.95 0 .751.751 0 0 1-.018 1.042.751.751 0 0 1-1.042.018 1.998 1.998 0 0 0-2.83 0l-2.5 2.5a1.998 1.998 0 0 0 0 2.83Z"></path></svg></a>Planned features</h3>
<ul dir="auto">
<li>Exact sampling using dual representation [1].</li>
<li>Better integration with MCMC frameworks in Julia (such as <a href="https://github.com/JuliaStats/Lora.jl">Lora.jl</a> or <a href="https://github.com/TuringLang/AbstractMCMC.jl">AbstractMCMC.jl</a>).</li>
<li>Fitting DPP and k-DPP models to data [3, 4].</li>
<li>Reduced rank DPP and k-DPP.</li>
<li>Kronecker Determinantal Point Processes [5].</li>
</ul>
<p dir="auto">Any help on these topics would be highly appreciated</p>
