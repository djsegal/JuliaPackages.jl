<p dir="auto">This package provides an implementation of the FameSVD algorithm via the BLAS and LAPACK routines <code>syrk</code>, <code>syevr</code> and <code>gemm</code>, The provided method is faster than the SVD algorithm used in the Julia standard library and as shown in the paper faster than the Krylov-Method and Randomized-PCA.</p>
<p dir="auto"><a target="_blank" rel="noopener noreferrer" href="./benchmark/bench.png"><img src="./benchmark/bench.png" alt="alt text" style="max-width: 100%;"></a></p>
<p dir="auto">Please note that column size was kept contstant at 1000 and the machine used had 16GB DDR4 RAM and an Intel i7-8565U CPU running at 4.6GHz.</p>
