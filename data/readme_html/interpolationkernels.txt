<p dir="auto">The <code>InterpolationKernels</code> package provides a library of interpolation kernels
for <a href="https://julialang.org/" rel="nofollow">Julia</a>.  As suggested by their name,
interpolations kernels are mostly designed for interpolation, they can be
thought as smart functions that know their derivative and that implement
optimized computation of interpolation weights.</p>
<p dir="auto">You may have a look at some sections of the <a href="https://emmt.github.io/InterpolationKernels.jl/dev/" rel="nofollow">the latest
documentation</a>:</p>
<ul dir="auto">
<li><a href="https://emmt.github.io/InterpolationKernels.jl/dev/kernels/" rel="nofollow">List of provided kernels</a>.</li>
<li><a href="https://emmt.github.io/InterpolationKernels.jl/dev/interpolation/" rel="nofollow">Principles of interpolation</a>.</li>
<li><a href="https://emmt.github.io/InterpolationKernels.jl/dev/install/" rel="nofollow">Instructions for installation</a>.</li>
</ul>
<p dir="auto">The <code>InterpolationKernels</code> package is used by:</p>
<ul dir="auto">
<li>
<p dir="auto"><a href="https://github.com/emmt/FineShift.jl">FineShift</a>, a Julia package for
fast sub-sample shifting of arrays.</p>
</li>
<li>
<p dir="auto"><a href="https://github.com/emmt/LinearInterpolators.jl">LinearInterpolators</a> which implements
various interpolation methods for Julia as linear mappings.</p>
</li>
</ul>
