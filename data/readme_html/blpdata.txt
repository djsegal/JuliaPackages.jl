<p>Provides a wrapper for <a href="https://www.bloomberg.com/professional/support/api-library/" rel="nofollow">BLPAPI Library</a> to the Julia language.</p>
<p>The BLPAPI is the API Library provided by <a href="https://www.bloomberg.com/" rel="nofollow">Bloomberg L.P.</a>
to connect to Bloomberg Professional Services.</p>
<p>This package aims to wrap each BLPAPI "Class" into an equivalent Julia type.
Using this approach, the user can implement any query that BLPAPI provides
and also inspect message schemas to discover what requests and responses are
available from the API, without having to use the C API directly.</p>
