<p dir="auto">Enables a wide array of textual diagramming tools, such as
<a href="https://www.graphviz.org" rel="nofollow">Graphviz</a>, <a href="https://mermaidjs.github.io" rel="nofollow">Mermaid</a>,
<a href="https://plantuml.com" rel="nofollow">PlantUML</a>,
<a href="https://ivanceras.github.io/content/Svgbob.html" rel="nofollow">svgbob</a> and <a href="https://bauglir.github.io/Kroki.jl/stable/#diagram-support" rel="nofollow">many
more</a> within Julia
through the <a href="https://kroki.io" rel="nofollow">Kroki</a> service. It works in environments capable
of rendering images:</p>
<p dir="auto"><a target="_blank" rel="noopener noreferrer" href="./docs/src/kroki-demo-pluto.gif"><img src="./docs/src/kroki-demo-pluto.gif" alt="Kroki Pluto Demo" data-animated-image="" style="max-width: 100%;"></a></p>
<p dir="auto">and even in the REPL for a subset of the supported diagrams:</p>
<p dir="auto"><a target="_blank" rel="noopener noreferrer" href="./docs/src/kroki-demo-repl.gif"><img src="./docs/src/kroki-demo-repl.gif" alt="Kroki REPL Demo" data-animated-image="" style="max-width: 100%;"></a></p>
<p dir="auto">See the <a href="https://bauglir.github.io/Kroki.jl/stable" rel="nofollow">documentation</a> for more
information and please follow the <a href="CONTRIBUTING.md">contribution guidelines</a>
when contributing.</p>

<p dir="auto">Install Kroki through Julia's package manager</p>
<div class="snippet-clipboard-content notranslate position-relative overflow-auto" data-snippet-clipboard-copy-content="(v1.7) pkg&gt; add Kroki"><pre class="notranslate"><code>(v1.7) pkg&gt; add Kroki
</code></pre></div>
<p dir="auto">Construct diagrams using the
<a href="https://bauglir.github.io/Kroki.jl/stable/api/#Kroki.Diagram" rel="nofollow"><code>Diagram</code></a> type
or any of the available <a href="https://bauglir.github.io/Kroki.jl/stable/api/#String-Literals" rel="nofollow">string
literals</a>. Then
either rely on the available <code>Base.show</code> overloads, or call the
<a href="https://bauglir.github.io/Kroki.jl/stable/api/#Kroki.render" rel="nofollow"><code>render</code></a>
function with a specific output format, to visualize them.</p>

<p dir="auto"><em>Note:</em> ASCII art diagrams, as shown above, are only supported for a subset of
<a href="https://plantuml.com" rel="nofollow">PlantUML</a> diagrams. All other diagram types are better
supported in environments like
<a href="https://juliadocs.github.io/Documenter.jl/stable" rel="nofollow">Documenter.jl</a>,
<a href="https://github.com/fonsp/Pluto.jl">Pluto.jl</a> or
<a href="https://jupyter.org" rel="nofollow">Jupyter</a>.</p>
