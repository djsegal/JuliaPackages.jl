<p dir="auto">QDLDL is a factorisation routine for quasi-definite linear systems <code>Ax=b</code>. This is a pure Julia implementation of the C language QDLDL solver (<a href="https://github.com/osqp/qdldl">https://github.com/osqp/qdldl</a>) with some additional functionality implemented to support refactorisations.</p>

<p dir="auto">QDLDL requirers Julia v1.0 and can be added via the Julia package manager (type <code>]</code>): <code>pkg&gt; add QDLDL</code>. Make the package available in you project with <code>using QDLDL</code>.</p>
<h2 dir="auto"><a id="user-content-using-qdldl" class="anchor" aria-hidden="true" href="#using-qdldl"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path d="m7.775 3.275 1.25-1.25a3.5 3.5 0 1 1 4.95 4.95l-2.5 2.5a3.5 3.5 0 0 1-4.95 0 .751.751 0 0 1 .018-1.042.751.751 0 0 1 1.042-.018 1.998 1.998 0 0 0 2.83 0l2.5-2.5a2.002 2.002 0 0 0-2.83-2.83l-1.25 1.25a.751.751 0 0 1-1.042-.018.751.751 0 0 1-.018-1.042Zm-4.69 9.64a1.998 1.998 0 0 0 2.83 0l1.25-1.25a.751.751 0 0 1 1.042.018.751.751 0 0 1 .018 1.042l-1.25 1.25a3.5 3.5 0 1 1-4.95-4.95l2.5-2.5a3.5 3.5 0 0 1 4.95 0 .751.751 0 0 1-.018 1.042.751.751 0 0 1-1.042.018 1.998 1.998 0 0 0-2.83 0l-2.5 2.5a1.998 1.998 0 0 0 0 2.83Z"></path></svg></a>Using QDLDL</h2>
<p dir="auto">QDLDL can be used to solve linear systems <code>Ax = b</code>.
Given a quasidefinite matrix <code>A</code> and right-hand side vector <code>b</code>, compute the factorisation <code>F</code> with:</p>
<div class="highlight highlight-source-julia notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="F = qdldl(A)"><pre>F <span class="pl-k">=</span> <span class="pl-c1">qdldl</span>(A)</pre></div>
<p dir="auto">Solve the linear system for <code>x</code> with:</p>
<div class="highlight highlight-source-julia notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="x = solve(F, b)"><pre>x <span class="pl-k">=</span> <span class="pl-c1">solve</span>(F, b)</pre></div>
<p dir="auto">This will allocate new memory for <code>x</code>. To solve in-place and overwrite <code>b</code> with <code>x</code> use:</p>
<div class="highlight highlight-source-julia notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="solve!(F, b)"><pre><span class="pl-c1">solve!</span>(F, b)</pre></div>
