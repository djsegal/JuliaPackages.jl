<p dir="auto">Support Vector Regression (SVR) analysis in <a href="http://julialang.org" rel="nofollow">Julia</a> utilizing the <a href="https://www.csie.ntu.edu.tw/~cjlin/libsvm/" rel="nofollow">libSVM</a> library.</p>
<p dir="auto">SVR is a module of <a href="http://madsjulia.github.io/Mads.jl" rel="nofollow">MADS</a> (Model Analysis &amp; Decision Support).</p>
