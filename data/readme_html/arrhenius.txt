<ul dir="auto">
<li><strong>Sensitivity analysis for auto-ignition</strong> | <a href="https://github.com/DENG-MIT/ArrheniusActiveSubspace">repo</a> | Features: auto-differentiation, multi-threading, sensitivity to all of three Arrhenius params A, b and Ea, active subspace based uncertainty quantification</li>
<li><strong>Sensitivity analysis for one-dimensional flames</strong> | <a href="https://github.com/DENG-MIT/Arrhenius_Flame_1D">repo</a> | Features: auto-differentiation, multi-threading, sensitivity to all of three Arrhenius params A, b and Ea.</li>
<li><strong>Automonous learn kinetic mechanism using neural network</strong> | <a href="https://github.com/DENG-MIT/CRNN_HyChem">repo</a> | Features: Chemical Reaction Neural Network (CRNN), Neural Ordinary Differential Equations.</li>
<li><strong>Deep Reduction</strong> | <a href="https://github.com/DENG-MIT/DeepReduction">repo</a> | Features: Two-stages mechanism reduction with deep learning.</li>
</ul>
