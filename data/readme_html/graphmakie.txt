<p dir="auto">Plotting graphs¹... with <a href="https://github.com/MakieOrg/Makie.jl">Makie</a>!</p>
<p dir="auto">This package is just a plotting recipe, you still need to use one of the Makie backends.</p>
<p dir="auto"><code>GraphMakie</code> is in an early development stage and might break often. Any
contribution such as suggesting features, raising issues and opening PRs are
welcome!</p>
<p dir="auto">Starting from v0.3 <code>GraphMakie.jl</code> switches from <code>LightGraphs.jl</code> to <code>Graphs.jl</code> for the graph representation. See this <a href="https://discourse.julialang.org/t/lightgraphs-jl-transition/69526/17" rel="nofollow">discourse post</a> for more information. If you want to use <code>LightGraphs.jl</code> please specifically <code>] add GraphMakie@0.2</code>!</p>
