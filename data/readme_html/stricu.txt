<p dir="auto">This is a new wrapper for the ICU library, designed to work on Julia v1.0 and above, using the <a href="http://github.com/JuliaString/Strs.jl">Strs.jl</a> package to provide support for UTF-16 encoded strings.
The API has been redesigned to not pollute the namespace and to try to be a bit more "Julian"</p>
