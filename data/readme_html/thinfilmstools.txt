<p><a href="https://github.com/lnacquaroli/ThinFilmsTools.jl/wiki/Home">ThinFilmsTools.jl</a> provides tools for the design and characterisation of thin films written in Julia. Documentation, examples and details can be found in the <a href="https://github.com/lnacquaroli/ThinFilmsTools.jl/wiki/Home">Wiki</a> page.</p>
<p>Check <a href="https://julialang.org/" rel="nofollow">julia language</a> for how to install and setup the environment.</p>

<p><a href="https://github.com/lnacquaroli/ThinFilmsTools.jl/wiki/Home">ThinFilmsTools.jl</a> is compatible with Julia version 1.1 or later.</p>
<p>The package includes modules for the calculation of parameters of thin films: <a href="https://github.com/lnacquaroli/ThinFilmsTools.jl/wiki/TMMOptics">tmm_optics</a> for the simulation of optical properties of thin films and photonic crystals, <a href="https://github.com/lnacquaroli/ThinFilmsTools.jl/wiki/FitTMMOptics">fit_tmm_optics</a> to fit thin films spectrum and <a href="https://github.com/lnacquaroli/ThinFilmsTools.jl/wiki/ThreeOmega">three_omega</a> to model the thermal properties of thin films based on the 3Ï‰ method.</p>
<p>The package also contains a number of indices of refraction for different materials in a database <a href="https://github.com/lnacquaroli/ThinFilmsTools.jl/wiki/RefractiveIndicesDB.jl">RefractiveIndicesDB.jl</a> ready to use.</p>
<p>For the simulation of index of refraction mixtures, there is also available a database <a href="https://github.com/lnacquaroli/ThinFilmsTools.jl/wiki/RefractiveIndicesModels.jl">RefractiveIndicesModels.jl</a> that contains several mixing rules for this.</p>
<p>The <a href="https://github.com/lnacquaroli/ThinFilmsTools.jl/wiki/FitCurveModel">fit_curve_model</a> allows fitting the spectral line shapes of different processes using a simple function. It supports three models for an arbitrary number of peaks: Gaussian, Lorentzian, and Voigtian.</p>
<p>A bunch of functions and recipes are included (<a href="https://github.com/lnacquaroli/ThinFilmsTools.jl/wiki/PlottingTools.jl">PlottingTools.jl</a>) for convenience to plot results from fit_tmm_optics, tmm_optics and three_omega.</p>
