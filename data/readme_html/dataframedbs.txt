<p>The DateFrameDBs is the persistent, space efficient columnar database, inspired by <a href="https://github.com/JuliaData/DataFrames.jl">DataFrames</a> and columnar databases like <a href="https://github.com/ClickHouse/ClickHouse">ClickHouse</a>. It allows you to work with large amounts of data without loading them completely into memory.</p>
<p>See <a href="https://waralex.github.io/DataFrameDBs.jl/dev/" rel="nofollow">docs</a> for more details</p>
