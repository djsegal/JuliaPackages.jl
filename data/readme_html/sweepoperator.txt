<p dir="auto">The symmetric sweep operator is a powerful tool in computational statistics with uses in stepwise regression, conditional multivariate normal distributions, MANOVA, and more.  This package exports a single function:</p>

<p dir="auto">Perform the sweep operation (or inverse sweep if <code>inv=true</code>) on symmetric matrix <code>A</code> on
element <code>k</code> (or each element in <code>ks</code>).  <strong>Only the upper triangle is read/swept.</strong></p>
