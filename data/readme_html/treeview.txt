<p dir="auto">This is a small package to visualize a graph corresponding to an
abstract syntax tree (AST) of a Julia expression. It uses the <code>TikzGraphs.jl</code>
package to do the visualization.</p>

<p dir="auto">The package is designed to be used within the IJulia notebook.</p>
<p dir="auto">The simplest usage is <code>@tree &lt;EXPRESSION&gt;</code>, where <code>&lt;EXPRESSION&gt;</code> represents any
Julia expression, for example</p>
<div class="snippet-clipboard-content notranslate position-relative overflow-auto" data-snippet-clipboard-copy-content="@tree x^2 + y^2"><pre class="notranslate"><code>@tree x^2 + y^2
</code></pre></div>
<p dir="auto">which gives the following output:</p>
<p dir="auto"><a target="_blank" rel="noopener noreferrer" href="example_tree.png"><img src="example_tree.png" alt="example_tree" style="max-width: 100%;"></a></p>
<p dir="auto">See <a href="https://github.com/JuliaTeX/TreeView.jl/blob/master/examples/TreeView%20usage.ipynb">this notebook</a> for usage examples.</p>
