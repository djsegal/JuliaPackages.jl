<p dir="auto">Julia package to compute points and weights for Gauss quadrature rules
using the Golub-Welsch algorithm.</p>
<p dir="auto">Handles the classical Legendre, Chebyshev, Jacobi, Laguerre and Hermite
weights, as well as a logarithmic weight function.  In fact, the
Gauss rule is available for any custom weight function such that the
coefficients are known for the three-term recurrence relation satisfied
by the associated orthogonal polynomials.  The modified Chebyshev
algorithm is provided to determine these coefficients from the modified
moments of the weight function.</p>
<p dir="auto">The Lobatto and Radau variants of all these rules are also provided by
appropriate choice of the <code>endpt</code> argument: <code>neither</code> (the default),
<code>both</code>, <code>left</code> or <code>right</code>.</p>
<p dir="auto">For example, to obtain a plain 5-point Gauss-Legendre rule with weight
function <code>w(x)=1</code> on the interval <code>-1 &lt; x &lt; -1</code> do</p>
<div class="snippet-clipboard-content notranslate position-relative overflow-auto" data-snippet-clipboard-copy-content="julia&gt; using GaussQuadrature
julia&gt; x, w = legendre(5)"><pre class="notranslate"><code>julia&gt; using GaussQuadrature
julia&gt; x, w = legendre(5)
</code></pre></div>
<p dir="auto">whereas for the Lobatto version do</p>
<div class="snippet-clipboard-content notranslate position-relative overflow-auto" data-snippet-clipboard-copy-content="julia&gt; x, w = legendre(5, both)"><pre class="notranslate"><code>julia&gt; x, w = legendre(5, both)
</code></pre></div>
<p dir="auto">The package supports <code>BigFloat</code>s; for example,</p>
<div class="snippet-clipboard-content notranslate position-relative overflow-auto" data-snippet-clipboard-copy-content="julia&gt; x, w = legendre(BigFloat, 5)"><pre class="notranslate"><code>julia&gt; x, w = legendre(BigFloat, 5)
</code></pre></div>
<p dir="auto">gives a plain 5-point Gauss-Legendre rule accurate to about 75 significant
figures.</p>
<p dir="auto">Read the initial comments in the src/GaussQuadrature.jl module
for full details, or read the help documentation for the individual
functions called <code>legendre</code>, <code>chebyshev</code>, <code>jacobi</code>, <code>laguerre</code>, <code>hermite</code>,
<code>logweight</code> and <code>custom_gauss_rule</code>.</p>
