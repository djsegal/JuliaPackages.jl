<p>Sigma is a probabilistic programming environment implemented in Julia.
You can use it to specify probabilistic models as normal Julia programs, and perform inference.</p>

<p>Sigma is built on top of Julia.  Sigma currently runs on linux only. Sigma is currently highly unstable, beware.
Install from a REPL with</p>
<div class="highlight highlight-source-julia"><pre>Pkg<span class="pl-k">.</span><span class="pl-c1">add</span>(<span class="pl-s"><span class="pl-pds">"</span>Sigma<span class="pl-pds">"</span></span>)</pre></div>
<p>Sigma is then loaded with</p>
<div class="highlight highlight-source-julia"><pre><span class="pl-k">using</span> Sigma</pre></div>

<h1><a id="user-content-quick-start" class="anchor" aria-hidden="true" href="#quick-start"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path fill-rule="evenodd" d="M7.775 3.275a.75.75 0 001.06 1.06l1.25-1.25a2 2 0 112.83 2.83l-2.5 2.5a2 2 0 01-2.83 0 .75.75 0 00-1.06 1.06 3.5 3.5 0 004.95 0l2.5-2.5a3.5 3.5 0 00-4.95-4.95l-1.25 1.25zm-4.69 9.64a2 2 0 010-2.83l2.5-2.5a2 2 0 012.83 0 .75.75 0 001.06-1.06 3.5 3.5 0 00-4.95 0l-2.5 2.5a3.5 3.5 0 004.95 4.95l1.25-1.25a.75.75 0 00-1.06-1.06l-1.25 1.25a2 2 0 01-2.83 0z"></path></svg></a>Quick Start</h1>
<p>First we need to include Sigma</p>
<div class="highlight highlight-source-julia"><pre>julia<span class="pl-k">&gt;</span> <span class="pl-k">using</span> Sigma</pre></div>
<p>Then, we create a uniform distribution <code>x</code> and draw 100 samples from it using <code>rand</code>:</p>
<div class="highlight highlight-source-julia"><pre>julia<span class="pl-k">&gt;</span> x <span class="pl-k">=</span> <span class="pl-c1">uniform</span>(<span class="pl-c1">0</span>,<span class="pl-c1">1</span>)
RandVar{Float64}

julia<span class="pl-k">&gt;</span> <span class="pl-c1">rand</span>(x, <span class="pl-c1">100</span>)
<span class="pl-c1">100</span><span class="pl-k">-</span>element Array{Float64,<span class="pl-c1">1</span>}<span class="pl-k">:</span>
  <span class="pl-c1">0.376264</span>
  <span class="pl-c1">0.492391</span>
     <span class="pl-k">...</span></pre></div>
<p>Then we can find the probability that <code>x^2</code> is greater than 0.6:</p>
<div class="highlight highlight-source-julia"><pre>julia<span class="pl-k">&gt;</span> <span class="pl-c1">prob</span>(x<span class="pl-k">^</span><span class="pl-c1">2</span> <span class="pl-k">&gt;</span> <span class="pl-c1">0.6</span>)
[<span class="pl-c1">0.225463867187499</span> <span class="pl-c1">0.225463867187499</span>]</pre></div>
<p>Then we can introduce an exponentially distributed variable <code>y</code>, and find the probability that <code>x^2</code> is greater than 0.6 under the condition that the sum of <code>x</code> and <code>y</code> is less than 1</p>
<div class="highlight highlight-source-julia"><pre>julia<span class="pl-k">&gt;</span> y <span class="pl-k">=</span> <span class="pl-c1">exponential</span>(<span class="pl-c1">0.5</span>)
julia<span class="pl-k">&gt;</span> <span class="pl-c1">prob</span>(x<span class="pl-k">^</span><span class="pl-c1">2</span> <span class="pl-k">&gt;</span> <span class="pl-c1">0.6</span>, x <span class="pl-k">+</span> y <span class="pl-k">&lt;</span> <span class="pl-c1">1</span>)
[<span class="pl-c1">0.053548951048950494</span> <span class="pl-c1">0.06132144691466614</span>]</pre></div>
<p>Then, instead of computing conditional probabilities, we can sample from <code>x</code> under the same condition:</p>
<div class="highlight highlight-source-julia"><pre>julia<span class="pl-k">&gt;</span> <span class="pl-c1">rand</span>(x, x <span class="pl-k">+</span> y <span class="pl-k">&lt;</span> <span class="pl-c1">1</span>)
<span class="pl-c1">0.04740462764340371</span></pre></div>
