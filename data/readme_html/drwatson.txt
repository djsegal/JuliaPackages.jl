<p dir="auto">DrWatson is a <strong>scientific project assistant</strong> software.
It helps people manage their scientific projects (or any project for that matter).</p>
<p dir="auto">Specifically, it is a Julia package created to help people "deal" with their simulations, simulation parameters, where are files saved, experimental data, scripts, existing simulations, project source code, establishing reproducibility, and in general their scientific projects.
To install, simply type <code>] add DrWatson</code> in your Julia session.</p>
<p dir="auto">Please read the <a href="https://JuliaDynamics.github.io/DrWatson.jl/dev" rel="nofollow">documentation page</a> for more!</p>
<p dir="auto"><strong>DISCLAIMER</strong> - The Julia package DrWatson, contained in this GitHub repository, is in no way related with the proprietary <a href="https://en.wikipedia.org/wiki/Dr._Watson_(debugger)" rel="nofollow">Dr. Watson</a>, a debugger implemented by Microsoft. In addition, the Julia package DrWatson contained in this GitHub repository is offered for free, without any monetary charge, and is provided with an MIT license.</p>
