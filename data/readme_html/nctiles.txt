<p><strong>NCTiles.jl</strong> reads and writes <a href="https://en.wikipedia.org/wiki/NetCDF" rel="nofollow">NetCDF files</a> that represent e.g. subdivisions of Earth's surface (<code>tiles</code>). Inter-operability with popular climate model grids / output via <a href="https://github.com/JuliaClimate/MeshArrays.jl">MeshArrays.jl</a> and practical generation of <a href="http://cfconventions.org" rel="nofollow">CF-compliant</a> files are key goals of this package.</p>
