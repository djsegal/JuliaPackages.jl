<p dir="auto"><a href="https://github.com/gaelforget/NCTiles.jl">NCTiles.jl</a> mainly aims to make it easier to write (and read) <a href="https://en.wikipedia.org/wiki/NetCDF" rel="nofollow">NetCDF</a> files that represent e.g. the whole Earth surface or a subdomain (<code>tiles</code>) based on the <a href="http://cfconventions.org" rel="nofollow">CF conventions</a>.</p>
<p dir="auto">Goals of <a href="https://github.com/gaelforget/NCTiles.jl">NCTiles.jl</a> include (1) inter-operability with climate model grids via <a href="https://github.com/JuliaClimate/MeshArrays.jl">MeshArrays.jl</a>, <a href="https://github.com/gaelforget/MITgcmTools.jl">MITgcmTools.jl</a>, and <a href="https://github.com/gaelforget/ClimateModels.jl">ClimateModels.jl</a>; (2) generation of <a href="http://cfconventions.org" rel="nofollow">CF-compliant</a> files from suitable array formats.</p>
