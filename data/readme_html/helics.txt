<p dir="auto"><a href="https://github.com/GMLC-TDC/HELICS.jl">HELICS.jl</a> is a cross-platform Julia wrapper around the <a href="https://github.com/GMLC-TDC/HELICS">HELICS</a> library.</p>
<p dir="auto"><strong>This package is now available for Windows, Mac, and Linux.</strong></p>

<p dir="auto">Use the Julia package manager to install HELICS.jl</p>
<div class="highlight highlight-source-julia notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="julia&gt; ]
(v1.1)&gt; add HELICS"><pre>julia<span class="pl-k">&gt;</span> ]
(v1.<span class="pl-c1">1</span>)<span class="pl-k">&gt;</span> add HELICS</pre></div>
<p dir="auto">Open the package manager REPL (using <code>]</code>)</p>
<p dir="auto">To install the latest development version, use the following from within Julia:</p>
<div class="highlight highlight-source-julia notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="(v1.1) pkg&gt; add HELICS#master"><pre>(v1.<span class="pl-c1">1</span>) pkg<span class="pl-k">&gt;</span> add HELICS<span class="pl-c"><span class="pl-c">#</span>master</span></pre></div>
<p dir="auto">This package includes HELICS as a library. You do not have to install HELICS
separately.</p>
<p dir="auto">Note that this should work on 64-bit Windows, Mac and Linux systems.</p>
<p dir="auto">If you wish to develop <code>HELICS.jl</code> you may want to use the following:</p>
<div class="highlight highlight-source-julia notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="(v1.1) pkg&gt; dev HELICS"><pre>(v1.<span class="pl-c1">1</span>) pkg<span class="pl-k">&gt;</span> dev HELICS</pre></div>
<p dir="auto">You can also get a specific version,</p>
<div class="highlight highlight-source-julia notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="(v1.1) pkg&gt; add HELICS#33c98625"><pre>(v1.<span class="pl-c1">1</span>) pkg<span class="pl-k">&gt;</span> add HELICS<span class="pl-c"><span class="pl-c">#</span>33c98625</span></pre></div>
<p dir="auto">or specific branch,</p>
<div class="highlight highlight-source-julia notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="(v1.1) pkg&gt; add HELICS#kd/some-new-feature"><pre>(v1.<span class="pl-c1">1</span>) pkg<span class="pl-k">&gt;</span> add HELICS<span class="pl-c"><span class="pl-c">#</span>kd/some-new-feature</span></pre></div>
<p dir="auto">if these features haven't been merged to <code>master</code> yet.</p>

<p dir="auto">This package interfaces with HELICS, so a good understanding of HELICS will help troubleshooting.
There are plenty of useful resources located <a href="https://helics.readthedocs.io/en/latest/" rel="nofollow">here</a>.</p>
<p dir="auto">If you are having issues using this Julia interface, feel free to open an issue on GitHub <a href="https://github.com/GMLC-TDC/HELICS.jl/issues/new">here</a>.</p>
