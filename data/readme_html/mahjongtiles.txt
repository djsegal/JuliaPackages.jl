<p dir="auto">Basic utilities for representing and working with Mahjong tiles, hands, and decks
(works in a similar way to
<a href="https://juliahub.com/ui/Packages/PlayingCards/I6MLG/0.3.0" rel="nofollow">PlayingCards.jl</a>).</p>
<p dir="auto">To get started, you can install from the Julia General repository in the usual way:</p>
<div class="highlight highlight-source-julia notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="(env) pkg&gt; add MahjongTiles"><pre>(env) pkg<span class="pl-k">&gt;</span> add MahjongTiles</pre></div>
<p dir="auto">See the <a href="https://tmthyln.github.io/MahjongTiles.jl/latest/" rel="nofollow">full documentation</a> for more details.</p>

<p dir="auto">Depending on the specific variety of Mahjong being played,
a deck is composed of 4 of each tile from the 3 standard "suits",
plus 4 of each of the winds, 4 of each of the dragons, and
sometimes the 8 flowers/seasons.
In the code, these are called <code>TilePile</code>s.
The easiest way to create one is to just use the standard constructor:</p>
<div class="highlight highlight-source-julia notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="using MahjongTiles: TilePile

deck = TilePile(:standard)"><pre><span class="pl-k">using</span> MahjongTiles<span class="pl-k">:</span> TilePile

deck <span class="pl-k">=</span> <span class="pl-c1">TilePile</span>(<span class="pl-c1">:standard</span>)</pre></div>

<p dir="auto">Again, depending on the specific variety being played,
a hand is usually composed of 13+1 or 16+1 tiles from a game deck.</p>
<p dir="auto">There are also functions to test if hands contain certain arrangements;
these are used to count points
(for gambling, sure, but also for bragging rights).</p>
