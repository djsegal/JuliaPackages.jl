<p dir="auto">This package provides an internal format to translate from/to other bibliographic format.</p>
<p dir="auto"><strong>!Warning</strong> The support for this package will move to Julia LTS once the next LTS release is available.</p>
<p dir="auto">All entries depend on an abstract super type <code>AbstractEntry</code>.
One generic entry <code>GenericEntry</code> is available to make entries without any specific rules.</p>
<p dir="auto">Currently, only one set of entries following the BibTeX rules is available. <em>Required</em> and <em>optional</em> BibTeX fields are checked by the constructor.</p>
<p dir="auto">Pull Requests to add more entries (or update the BibTeX rules) are welcome.</p>
<p dir="auto">Discussions are welcome either on this GitHub repository or on the <code>#modern-academics</code> channel of <a href="https://humansofjulia.org/" rel="nofollow">Humans of Julia</a> (to join the Discord server, please click the <code>chat</code> badge above).</p>
