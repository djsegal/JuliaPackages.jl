<p>A <code>RemoteSemaphore</code> is a <a href="https://www.quora.com/What-is-a-counting-semaphore" rel="nofollow">counting semaphore</a> that lives on a particular process in order to control access to a resource from multiple processes.
It is implemented using the unexported <code>Base.Semaphore</code> stored inside a <code>Future</code> which is only accessed on the process it was initialized on.
Like <code>Base.Semaphore</code>, it implements <code>acquire</code> and <code>release</code>, and is not thread-safe.</p>
