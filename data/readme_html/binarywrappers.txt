<p dir="auto"><a href="https://github.com/oscar-system/BinaryWrappers.jl/actions"><img src="https://github.com/oscar-system/BinaryWrappers.jl/workflows/CI/badge.svg" alt="Build Status" style="max-width: 100%;"></a></p>
<p dir="auto">This package provides a scratchspace with wrappers for the binaries and scripts
from JLL packages like <a href="https://github.com/JuliaBinaryWrappers/lib4ti2_jll.jl"><code>lib4ti2_jll</code></a>.
The wrappers allow non-julia code (like in <code>polymake_jll</code> or <code>Singular_jll</code>) to
execute those directly without having to adjust LIBPATH environment variables.</p>
<h2 dir="auto"><a id="user-content-usage" class="anchor" aria-hidden="true" href="#usage"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path d="m7.775 3.275 1.25-1.25a3.5 3.5 0 1 1 4.95 4.95l-2.5 2.5a3.5 3.5 0 0 1-4.95 0 .751.751 0 0 1 .018-1.042.751.751 0 0 1 1.042-.018 1.998 1.998 0 0 0 2.83 0l2.5-2.5a2.002 2.002 0 0 0-2.83-2.83l-1.25 1.25a.751.751 0 0 1-1.042-.018.751.751 0 0 1-.018-1.042Zm-4.69 9.64a1.998 1.998 0 0 0 2.83 0l1.25-1.25a.751.751 0 0 1 1.042.018.751.751 0 0 1 .018 1.042l-1.25 1.25a3.5 3.5 0 1 1-4.95-4.95l2.5-2.5a3.5 3.5 0 0 1 4.95 0 .751.751 0 0 1-.018 1.042.751.751 0 0 1-1.042.018 1.998 1.998 0 0 0-2.83 0l-2.5 2.5a1.998 1.998 0 0 0 0 2.83Z"></path></svg></a>Usage</h2>
<p dir="auto">The wrapper generation should be triggered during precompilation with</p>
<div class="snippet-clipboard-content notranslate position-relative overflow-auto" data-snippet-clipboard-copy-content="    using BinaryWrappers
    const lib4ti2_binpath = @generate_wrappers(lib4ti2_jll)"><pre class="notranslate"><code>    using BinaryWrappers
    const lib4ti2_binpath = @generate_wrappers(lib4ti2_jll)
</code></pre></div>
<p dir="auto">and the corresponding path might be used in <code>__init__()</code> as follows:</p>
<div class="snippet-clipboard-content notranslate position-relative overflow-auto" data-snippet-clipboard-copy-content="    ENV[&quot;PATH&quot;] = lib4ti2_binpath * &quot;:&quot; * ENV[&quot;PATH&quot;]"><pre class="notranslate"><code>    ENV["PATH"] = lib4ti2_binpath * ":" * ENV["PATH"]
</code></pre></div>

<ul dir="auto">
<li>Only Linux and macOS are supported.</li>
<li>This was primarily written to provide wrappers for <code>lib4ti2_jll</code> which comes with a bunch of shell scripts that need special care, but it might work for other JLL packages as well.</li>
