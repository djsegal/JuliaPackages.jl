<p><code>Chipmunk.jl</code> is a Julia binding of the popular physics engine [Chipmunk] (<a href="https://chipmunk-physics.net/" rel="nofollow">https://chipmunk-physics.net/</a>) ([Github page] (<a href="https://github.com/slembcke/Chipmunk2D">https://github.com/slembcke/Chipmunk2D</a>)).
This is currently very much in progress.</p>
<p>Take a look at the <code>examples/</code> folder to see what you can do with <code>Chipmunk.jl</code></p>
<p><code>Chipmunk.jl</code> uses [SFML.jl] (<a href="https://github.com/zyedidia/SFML.jl">https://github.com/zyedidia/SFML.jl</a>) to render the world to the screen.</p>
<p><code>Chipmunk.jl</code> also requires Julia 0.4.</p>

<p>Installation is quite simple. The package will clone and install chipmunk from source to <code>deps/</code>.
Make sure that you have <code>cmake</code> installed so that it can build chipmunk.</p>
<p>I have not been able to get cmake to compile Chipmunk on Windows yet.</p>
<p>If you get an SFML build error about <code>follow_symlinks</code> you should update your version of Julia 0.4.</p>
<pre><code>julia&gt; Pkg.clone("https://github.com/zyedidia/Chipmunk.jl")
julia&gt; Pkg.build("Chipmunk")
</code></pre>
