<p>AssetRegistry allows you to serve arbitrary files and
folders, using a global registry. Now packages like
<a href="https://github.com/JuliaWeb/Mux.jl"><code>Mux</code></a> and
<a href="https://github.com/JuliaLang/IJulia.jl"><code>IJulia</code></a> can
look up this registry and serve these files while
packages like <a href="https://github.com/JuliaGizmos/WebIO.jl"><code>WebIO</code></a>
and <a href="https://github.com/piever/InteractBase.jl"><code>InteractBase</code></a>
can register assets to be served to implement web-based UIs.</p>
<h2><a id="user-content-usage" class="anchor" aria-hidden="true" href="#usage"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path fill-rule="evenodd" d="M4 9h1v1H4c-1.5 0-3-1.69-3-3.5S2.55 3 4 3h4c1.45 0 3 1.69 3 3.5 0 1.41-.91 2.72-2 3.25V8.59c.58-.45 1-1.27 1-2.09C10 5.22 8.98 4 8 4H4c-.98 0-2 1.22-2 2.5S3 9 4 9zm9-3h-1v1h1c1 0 2 1.22 2 2.5S13.98 12 13 12H9c-.98 0-2-1.22-2-2.5 0-.83.42-1.64 1-2.09V6.25c-1.09.53-2 1.84-2 3.25C6 11.31 7.55 13 9 13h4c1.45 0 3-1.69 3-3.5S14.5 6 13 6z"></path></svg></a>Usage</h2>
<p>You can register an asset with the package by doing:</p>
<div class="highlight highlight-source-julia"><pre>key <span class="pl-k">=</span> AssetRegistry<span class="pl-k">.</span><span class="pl-c1">register</span>(<span class="pl-s"><span class="pl-pds">"</span>path/to/asset<span class="pl-pds">"</span></span>)</pre></div>
<p>This <code>key</code> is the unique URL where the asset
can be accessed. For example:</p>
<div class="highlight highlight-source-julia"><pre>julia<span class="pl-k">&gt;</span> key <span class="pl-k">=</span> AssetRegistry<span class="pl-k">.</span><span class="pl-c1">register</span>(<span class="pl-s"><span class="pl-pds">"</span>/Users/ranjan/.julia/v0.6/Tachyons/assets/tachyons.min.css<span class="pl-pds">"</span></span>)
<span class="pl-s"><span class="pl-pds">"</span>/assetserver/97a47bdda5bd9274ad1a9cd10a0337f3b033a790-tachyons.min.css<span class="pl-pds">"</span></span></pre></div>
