<p>This Julia package does one thing: gets you daily yield curves from the <a href="https://www.treasury.gov/resource-center/data-chart-center/interest-rates/Pages/TextView.aspx?data=yield" rel="nofollow">US Treasury</a>. The data is served via an XML feed, but this package cleans it up into a <code>DataFrame</code> so that you can use it.</p>

<p>Add the package via the Julia Package Manager.</p>
<div class="highlight highlight-source-julia position-relative" data-snippet-clipboard-copy-content="] add DailyTreasuryYieldCurve
"><pre>] add DailyTreasuryYieldCurve</pre></div>
<p>It is easy to get the historical yield curves:</p>
<div class="highlight highlight-source-julia position-relative" data-snippet-clipboard-copy-content="using DailyTreasuryYieldCurve

df_rates = getyieldcurves()

df_realrates = getyieldcurves(;realrates=true)
"><pre><span class="pl-k">using</span> DailyTreasuryYieldCurve

df_rates <span class="pl-k">=</span> <span class="pl-c1">getyieldcurves</span>()

df_realrates <span class="pl-k">=</span> <span class="pl-c1">getyieldcurves</span>(;realrates<span class="pl-k">=</span><span class="pl-c1">true</span>)</pre></div>
<p>You can also build a <code>RateInterpolator</code> which helps you interpolate/extrapolate using the data:</p>
<div class="highlight highlight-source-julia position-relative" data-snippet-clipboard-copy-content="using Dates

nominal_itp = createRateInterpolator(df_rates)

nominal_itp(45,Date(2020,4,13)) # gets the 45 day rate on 2020-4-13
"><pre><span class="pl-k">using</span> Dates

nominal_itp <span class="pl-k">=</span> <span class="pl-c1">createRateInterpolator</span>(df_rates)

<span class="pl-c1">nominal_itp</span>(<span class="pl-c1">45</span>,<span class="pl-c1">Date</span>(<span class="pl-c1">2020</span>,<span class="pl-c1">4</span>,<span class="pl-c1">13</span>)) <span class="pl-c"><span class="pl-c">#</span> gets the 45 day rate on 2020-4-13</span></pre></div>
