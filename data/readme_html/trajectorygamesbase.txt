<p dir="auto">A package that contains the problem interface and related types for trajectory games. The trajectory games considered here are played over continuous action spaces in <em>discrete time</em>. That is, this package currently does not support differential games.</p>
<p dir="auto">Note that this package does not contain any solver code. It merely serves as abstraction to share problem description and solvers.</p>
