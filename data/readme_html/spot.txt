<p dir="auto">This package provides Julia bindings to the <a href="https://spot.lrde.epita.fr/index.html" rel="nofollow">Spot</a> library for LTL and automata manipulation. It relies on <a href="https://github.com/JuliaInterop/CxxWrap.jl">CxxWrap.jl</a> to interface julia with the Spot c++ library. You can find the c++ code with the wrapped functions in <a href="https://github.com/MaximeBouton/spot_julia/">spot_julia</a>, and the build script for the <code>jll</code> file in <a href="https://github.com/JuliaPackaging/Yggdrasil/blob/master/S/Spot_julia/build_tarballs.jl">Yggdrasil</a>.</p>

Every function can be called using the Cxx interface.
If you need to wrap a function that has not been wrapped yet, feel free to submit a Pull Request.</p>
