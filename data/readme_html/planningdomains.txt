<p dir="auto">A (meta)-repository of PDDL domains and problems. The following repositories are supported:</p>
<ul dir="auto">
<li><code>JuliaPlannersRepo</code>: A built-in repository maintained by the <a href="https://github.com/JuliaPlanners">JuliaPlanners</a> GitHub organization. Domains and problem files can be found in <a href="repositories/julia-planners"><code>repositories/julia-planners</code></a>.</li>
<li><code>IPCInstancesRepo</code>: A repository of International Planning Competition (IPC) domains and problems, hosted at <a href="https://github.com/potassco/pddl-instances">https://github.com/potassco/pddl-instances</a>.</li>
<li><code>PlanningDomainsRepo</code>: A repository of domains and problems accessible via <a href="https://api.planning.domains" rel="nofollow">https://api.planning.domains</a>.</li>
</ul>

<p dir="auto">For repositories hosted online, PlanningDomains.jl maintains a local cache to reduce load times. However, this cache is not automatically updated if the remote repositories change. To manually clear the cache for a particular repository use <code>PlanningDomains.clear_cache!</code>. To clear all caches, use <code>PlanningDomains.clear_all_caches!</code>.</p>
