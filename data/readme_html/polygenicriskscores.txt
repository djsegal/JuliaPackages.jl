<p dir="auto">Julia's CSV reader supports multi-threaded CSV reading. In order to enable this, we need to load Julia with more than one thread. This is done through the <code>JULIA_NUM_THREADS</code> environment variable:</p>
<div class="snippet-clipboard-content notranslate position-relative overflow-auto" data-snippet-clipboard-copy-content="JULIA_NUM_THREADS=8 julia --project -e &quot;using PolygenicRiskScores; PolygenicRiskScores.main()&quot; ..."><pre class="notranslate"><code>JULIA_NUM_THREADS=8 julia --project -e "using PolygenicRiskScores; PolygenicRiskScores.main()" ...
</code></pre></div>
<p dir="auto">The above code would run PolygenicRiskScores.jl with 8 threads.</p>
