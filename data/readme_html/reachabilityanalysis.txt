<p>Reachability analysis is concerned with computing rigorous approximations of the set
of states reachable by a dynamical system. In the scope of this package are systems
modeled by ordinary differential equations (ODEs) with uncertain initial states,
uncertain parameters or non-deterministic inputs. The package also considers the
extension to so-called hybrid systems where the dynamics changes with discrete events.</p>

<p>The following types of systems are supported.</p>
<ul>
<li>Continuous ODEs with linear dynamics (<a href="https://juliareach.github.io/ReachabilityAnalysis.jl/dev/lib/algorithms/GLGM06/#ReachabilityAnalysis.GLGM06" rel="nofollow">GLGM06</a>, <a href="https://juliareach.github.io/ReachabilityAnalysis.jl/dev/lib/algorithms/LGG09/" rel="nofollow">LGG09</a>, <a href="https://juliareach.github.io/ReachabilityAnalysis.jl/dev/lib/algorithms/BFFPSV18/" rel="nofollow">BFFPSV18</a>) <g-emoji class="g-emoji" alias="heavy_check_mark" fallback-src="https://github.githubassets.com/images/icons/emoji/unicode/2714.png">✔️</g-emoji></li>
<li>Continuous ODEs with linear dynamics and parametric uncertainty (<a href="https://juliareach.github.io/ReachabilityAnalysis.jl/dev/lib/algorithms/ASB07/" rel="nofollow">ASB07</a>) <g-emoji class="g-emoji" alias="heavy_check_mark" fallback-src="https://github.githubassets.com/images/icons/emoji/unicode/2714.png">✔️</g-emoji></li>
<li>Continuous ODEs with non-linear dynamics (<a href="https://juliareach.github.io/ReachabilityAnalysis.jl/dev/lib/algorithms/TMJets/" rel="nofollow">TMJets</a>) <g-emoji class="g-emoji" alias="heavy_check_mark" fallback-src="https://github.githubassets.com/images/icons/emoji/unicode/2714.png">✔️</g-emoji></li>
<li>Hybrid systems with piecewise-affine dynamics <g-emoji class="g-emoji" alias="heavy_check_mark" fallback-src="https://github.githubassets.com/images/icons/emoji/unicode/2714.png">✔️</g-emoji></li>
<li>Hybrid systems with non-linear dynamics <g-emoji class="g-emoji" alias="heavy_check_mark" fallback-src="https://github.githubassets.com/images/icons/emoji/unicode/2714.png">✔️</g-emoji></li>
<li>Hybrid systems with clocked linear dynamics <g-emoji class="g-emoji" alias="heavy_check_mark" fallback-src="https://github.githubassets.com/images/icons/emoji/unicode/2714.png">✔️</g-emoji></li>
</ul>
