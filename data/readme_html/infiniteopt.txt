<p dir="auto">A <code>JuMP</code> extension for expressing and solving infinite-dimensional optimization
problems. Such areas include <a href="https://en.wikipedia.org/wiki/Stochastic_programming" rel="nofollow">stochastic programming</a>,
<a href="https://en.wikipedia.org/wiki/Dynamic_programming" rel="nofollow">dynamic programming</a>,
space-time optimization, and more. <code>InfiniteOpt</code> serves as an easy-to-use modeling
interface for these advanced problem types that can be used by those with little
to no background in these areas. It also it contains a wealth of capabilities
making it a powerful and convenient tool for advanced users.</p>

<p dir="auto">It builds upon <code>JuMP</code> to add support for many complex modeling objects which
include:</p>
<ul dir="auto">
<li>Infinite parameters (e.g., time, space, uncertainty, etc.)</li>
<li>Finite parameters (similar to <code>ParameterJuMP</code>)</li>
<li>Infinite variables (decision functions) (e.g., <code>y(t, x)</code>)</li>
<li>Derivatives (e.g., <code>‚àÇy(t, x)/‚àÇt</code>)</li>
<li>Measures (e.g., <code>‚à´y(t,x)dt</code>, <code>ùîº[y(Œæ)]</code>)</li>
<li><strong>1st class nonlinear modeling</strong></li>
</ul>
<p dir="auto">The unifying modeling abstraction behind <code>InfiniteOpt</code> captures a wide spectrum
of disciplines which include dynamic, PDE, stochastic, and semi-infinite
optimization. Moreover, we facilitate transferring techniques between these
to synthesize new optimization paradigms!</p>
