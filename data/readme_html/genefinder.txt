<blockquote>
<p dir="auto">This is a species-agnostic, algorithm extensible, sequence-anonymous (genome, metagenomes) <em>gene finder</em> library framework for the Julia Language.</p>
</blockquote>
<p dir="auto">The main goal of <code>GeneFinder</code> is to create a versatile module that enables apply different implemented algorithm to DNA sequences. See, for instance, BioAlignment implementations of different sequence alignment algorithms (local, global, edit-distance).</p>

<p dir="auto">The first implemented function is <code>findorfs</code> a very non-restrictive ORF finder function that will catch all ORFs in a dedicated structure. Note that this will catch random ORFs not necesarily genes since it has no ORFs size or overlapping condition contraints. Thus it might consider <code>aa"M*"</code> a posible encoding protein from the resulting ORFs.</p>
