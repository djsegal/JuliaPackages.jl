<p>A tiny package to make <code>html"&lt;input&gt;"</code> a bit more Julian. Use it with the <a href="https://github.com/fonsp/Pluto.jl"><code>@bind</code> macro in Pluto</a>.</p>
<p>For <strong>documentation</strong>, read the <strong>PlutoUI.jl sample notebook</strong> inside Pluto, or <a href="https://juliahub.com/docs/PlutoUI/" rel="nofollow">read the docstrings</a>.</p>
<blockquote>
<p>If you know a bit of JavaScript, this package is a great, low-entry way to contribute to Pluto! All fun UI elements are welcome here!</p>
</blockquote>
