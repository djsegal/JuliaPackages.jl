<p dir="auto">DICOMClient.jl
provides a Julia client for connecting to
servers that support the
<a href="https://www.dicomstandard.org/" rel="nofollow">Digital Imaging and Communications in Medicine (DICOM)</a>
standard by using the
<a href="https://www.dicomstandard.org/dicomweb" rel="nofollow">DICOMweb</a>
set of RESTful services.</p>
<p dir="auto">Please see the <a href="https://JuliaHealth.github.io/DICOMClient.jl/stable" rel="nofollow">documentation</a>.</p>
<p dir="auto">You may also be interested in the <a href="https://github.com/JuliaHealth/DICOM.jl">DICOM.jl</a> package.</p>
