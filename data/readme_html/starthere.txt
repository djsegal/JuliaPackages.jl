<td align="center">foo / bar / bazz</td>
<td align="center">These terms are used to give an example of what is being discussed, and the names don't actually mater. For example, <code>foo(bar.txt)</code> means that there is some function that will be called on a text file, but the names don't really matter at the moment.</td>
</tr>
<tr>
<td align="center">AFAIK</td>
<td align="center">As Far As I Know</td>
</tr>
<tr>
<td align="center">MWE</td>
<td align="center">Minimum Working Example. <a href="https://discourse.julialang.org/t/psa-make-it-easier-to-help-you/" rel="nofollow">A piece of code to reproduce your problem on someone else's machine</a> .</td>
</tr>
<tr>
<td align="center">LGTM</td>
<td align="center">Looks Good To Me</td>
</tr>
<tr>
<td align="center">YMMV</td>
<td align="center">Your Mileage May Vary, an abbrevaited way to say "Your experience may be different than mine".</td>
</tr>
<tr>
<td align="center">IIUC</td>
<td align="center">If I Understand Correctly</td>
</tr>
<tr>
<td align="center">AFAIK</td>
<td align="center">As Far As I Know</td>
</tr>
<tr>
<td align="center">IANAL</td>

</tr>
<tr>
<td align="center">PR</td>
<td align="center">Pull Request (see Git)</td>
</tr>
<tr>
<td align="center">REPL</td>
<td align="center">Read Evaluate Print Loop, also known as that terminal / dark window when you open up Julia for the first time. Very useful for experimenting with you Julia codes.</td>
</tr>
<tr>
<td align="center">TIL</td>
<td align="center">Today I Learned</td>
</tr>
<tr>
<td align="center">GC</td>
<td align="center">Garbage Collector. An algorithm that helps manage the memory needed for Julia to run.</td>
</tr>
<tr>
<td align="center">LLVM</td>
<td align="center">Julia's compiler. As a beginner, its enough to know that a compiler takes code, transforms it, and ideally makes it faster.</td>
</tr>
<tr>
<td align="center">GSOC</td>
<td align="center">Google Summer of Code. Happens once a year in June - August. Google pays students to code in Julia</td>
</tr>
<tr>
<td align="center">COC</td>
<td align="center">Julia's Code of Conduct.</td>
</tr>
<tr>
<td align="center">Slack</td>
<td align="center">An online chatting forum that permits talking in different #channels, with emojis and some code formatting. Messages get auto deleted after a while.</td>
</tr>
<tr>
<td align="center"><a href="https;//discourse.julialang.org">Discourse</a></td>
<td align="center">Usually suited towards longer form discussions.</td>
</tr>
<tr>
<td align="center">Zulip</td>
<td align="center">An online chatting forum like Slack, but open source, and permanent history.</td>
</tr>
<tr>
<td align="center">GitHub / issue tags</td>
<td align="center">A web page where most of Julia codes are freely hosted. People usually discuss code designs and submit Pull Requests here.</td>
</tr>
<tr>
<td align="center">Meetups</td>
<td align="center">When Julians meet each other in person. Usually location based with some scheduling. See <a href="">the calendar for your local ones</a></td>
</tr>
<tr>
<td align="center">Triage</td>
<td align="center">A call with Julia devs who focus on internal issues of how Julia works. Open to all, heavy technical knowledge.</td>
</tr>
<tr>
<td align="center">Community Calendar</td>
<td align="center">A place where meetups, calls, conventions and other Julia-related events are announced. <a href="https://julialang.org/community/#events" rel="nofollow">Link is here</a></td>
</tr>
<tr>
<td align="center">JuliaCon</td>
<td align="center">A once a year conference that happens somewhere in the world. People present posters, presentations and share their code. Almost all material is easily accessible on <a href="https://www.youtube.com/watch?v=rAxzR7lMGDM&amp;t=3s" rel="nofollow">youtube</a>.</td>
</tr>
<tr>
<td align="center">Ship it!</td>
<td align="center">A phrase to publish some code, usually with some bravery.</td>
</tr>
<tr>
<td align="center">#general</td>
<td align="center">A chatting channel on Slack/Zulip where things of general Julia interest are discussed.</td>
</tr>
<tr>
<td align="center">#random</td>
<td align="center">A chatting channel on Slack/Zulip where things are more casual and people post about random things, like the latest computer chips or a cool new video gets shared</td>
</tr>
<tr>
<td align="center">#introduction</td>
<td align="center"></td>

Very useful to read <a href="https://discourse.julialang.org/t/psa-make-it-easier-to-help-you/" rel="nofollow">how to get help</a> before. Also consider that people do this for free, so please be kind.</td>

<td align="center">Julia long time community members who take actions about faults with the CoC by community members. Not their day job, not their favorite job, not a thankful job, but rewarding for the community at large since this almost all online communities need to enforce some rules.</td>

<td align="center">bump (an issue)</td>
<td align="center">To recomment on a github issue or topic so that it gains attention again. Usually well intentioned if someone committed to working on the code and hasn't reported back in a while and it would be useful to know the current status of it.</td>
