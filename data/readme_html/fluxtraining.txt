<p dir="auto"><a href="https://fluxml.ai/FluxTraining.jl/dev/i" rel="nofollow">Docs (master)</a></p>
<p dir="auto">A Julia package for using and writing powerful, extensible training loops for deep learning models.</p>
<h2 dir="auto"><a id="user-content-what-does-it-do" class="anchor" aria-hidden="true" href="#what-does-it-do"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path d="m7.775 3.275 1.25-1.25a3.5 3.5 0 1 1 4.95 4.95l-2.5 2.5a3.5 3.5 0 0 1-4.95 0 .751.751 0 0 1 .018-1.042.751.751 0 0 1 1.042-.018 1.998 1.998 0 0 0 2.83 0l2.5-2.5a2.002 2.002 0 0 0-2.83-2.83l-1.25 1.25a.751.751 0 0 1-1.042-.018.751.751 0 0 1-.018-1.042Zm-4.69 9.64a1.998 1.998 0 0 0 2.83 0l1.25-1.25a.751.751 0 0 1 1.042.018.751.751 0 0 1 .018 1.042l-1.25 1.25a3.5 3.5 0 1 1-4.95-4.95l2.5-2.5a3.5 3.5 0 0 1 4.95 0 .751.751 0 0 1-.018 1.042.751.751 0 0 1-1.042.018 1.998 1.998 0 0 0-2.83 0l-2.5 2.5a1.998 1.998 0 0 0 0 2.83Z"></path></svg></a>What does it do?</h2>
<ul dir="auto">
<li>Implements a training loop to take the boilerplate out of training deep learning models</li>
<li>Lets you add features to training loops through reusable <a href="https://fluxml.ai/FluxTraining.jl/dev/documents/docs/callbacks/usage.md" rel="nofollow">callbacks</a></li>
<li>Comes with callbacks for many common use cases like <a href="https://fluxml.ai/FluxTraining.jl/dev/documents/docs/tutorials/hyperparameters.md" rel="nofollow">hyperparameter scheduling</a>, <a href="https://fluxml.ai/FluxTraining.jl/dev/references/FluxTraining.Metrics" rel="nofollow">metrics</a> tracking and <a href="https://fluxml.ai/FluxTraining.jl/dev/references/FluxTraining.LogMetrics" rel="nofollow">logging</a>, <a href="https://fluxml.ai/FluxTraining.jl/dev/references/FluxTraining.Checkpointer" rel="nofollow">checkpointing</a>, <a href="https://fluxml.ai/FluxTraining.jl/dev/references/FluxTraining.EarlyStopping" rel="nofollow">early stopping</a>, and <a href="https://fluxml.ai/FluxTraining.jl/dev/documents/docs/callbacks/reference.md" rel="nofollow">more...</a></li>
<li>Is extensible by creating <a href="https://fluxml.ai/FluxTraining.jl/dev/documents/docs/callbacks/custom.md" rel="nofollow">custom, reusable callbacks</a> or even <a href="https://fluxml.ai/FluxTraining.jl/dev/documents/docs/tutorials/training.md" rel="nofollow">custom training loops</a></li>
</ul>
<h2 dir="auto"><a id="user-content-when-should-you-use-fluxtrainingjl" class="anchor" aria-hidden="true" href="#when-should-you-use-fluxtrainingjl"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path d="m7.775 3.275 1.25-1.25a3.5 3.5 0 1 1 4.95 4.95l-2.5 2.5a3.5 3.5 0 0 1-4.95 0 .751.751 0 0 1 .018-1.042.751.751 0 0 1 1.042-.018 1.998 1.998 0 0 0 2.83 0l2.5-2.5a2.002 2.002 0 0 0-2.83-2.83l-1.25 1.25a.751.751 0 0 1-1.042-.018.751.751 0 0 1-.018-1.042Zm-4.69 9.64a1.998 1.998 0 0 0 2.83 0l1.25-1.25a.751.751 0 0 1 1.042.018.751.751 0 0 1 .018 1.042l-1.25 1.25a3.5 3.5 0 1 1-4.95-4.95l2.5-2.5a3.5 3.5 0 0 1 4.95 0 .751.751 0 0 1-.018 1.042.751.751 0 0 1-1.042.018 1.998 1.998 0 0 0-2.83 0l-2.5 2.5a1.998 1.998 0 0 0 0 2.83Z"></path></svg></a>When should you use FluxTraining.jl?</h2>
<ul dir="auto">
<li>You don't want to implement your own metrics tracking and hyperparameter scheduling or <em>insert common training feature here</em> for the 10th time</li>
<li>You want to use composable and reusable components that enhance your training loop</li>
<li>You want a simple training loop with reasonable defaults that can grow to the needs of your project</li>
</ul>
<h3 dir="auto"><a id="user-content-how-do-you-use-it" class="anchor" aria-hidden="true" href="#how-do-you-use-it"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path d="m7.775 3.275 1.25-1.25a3.5 3.5 0 1 1 4.95 4.95l-2.5 2.5a3.5 3.5 0 0 1-4.95 0 .751.751 0 0 1 .018-1.042.751.751 0 0 1 1.042-.018 1.998 1.998 0 0 0 2.83 0l2.5-2.5a2.002 2.002 0 0 0-2.83-2.83l-1.25 1.25a.751.751 0 0 1-1.042-.018.751.751 0 0 1-.018-1.042Zm-4.69 9.64a1.998 1.998 0 0 0 2.83 0l1.25-1.25a.751.751 0 0 1 1.042.018.751.751 0 0 1 .018 1.042l-1.25 1.25a3.5 3.5 0 1 1-4.95-4.95l2.5-2.5a3.5 3.5 0 0 1 4.95 0 .751.751 0 0 1-.018 1.042.751.751 0 0 1-1.042.018 1.998 1.998 0 0 0-2.83 0l-2.5 2.5a1.998 1.998 0 0 0 0 2.83Z"></path></svg></a>How do you use it?</h3>
<p dir="auto">Install like any other Julia package using the package manager:</p>
<div class="snippet-clipboard-content notranslate position-relative overflow-auto" data-snippet-clipboard-copy-content="]add FluxTraining"><pre lang="julia-repl" class="notranslate"><code>]add FluxTraining
</code></pre></div>
<p dir="auto">After installation, import it, create a <code>Learner</code> from a <a href="https://github.com/FluxML/Flux.jl">Flux.jl</a> model, data iterators, an optimizer, and a loss function. Finally train with <a href="#"><code>fit!</code></a>.</p>
