<h2><a id="user-content-overview" class="anchor" aria-hidden="true" href="#overview"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path fill-rule="evenodd" d="M7.775 3.275a.75.75 0 001.06 1.06l1.25-1.25a2 2 0 112.83 2.83l-2.5 2.5a2 2 0 01-2.83 0 .75.75 0 00-1.06 1.06 3.5 3.5 0 004.95 0l2.5-2.5a3.5 3.5 0 00-4.95-4.95l-1.25 1.25zm-4.69 9.64a2 2 0 010-2.83l2.5-2.5a2 2 0 012.83 0 .75.75 0 001.06-1.06 3.5 3.5 0 00-4.95 0l-2.5 2.5a3.5 3.5 0 004.95 4.95l1.25-1.25a.75.75 0 00-1.06-1.06l-1.25 1.25a2 2 0 01-2.83 0z"></path></svg></a>Overview</h2>
<p>This package implements different estimators of the expected calibration error
(ECE), the squared kernel calibration error calibration error (SKCE), and the
unnormalized calibration mean embedding (UCME) in the Julia language.</p>
<h2><a id="user-content-example" class="anchor" aria-hidden="true" href="#example"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path fill-rule="evenodd" d="M7.775 3.275a.75.75 0 001.06 1.06l1.25-1.25a2 2 0 112.83 2.83l-2.5 2.5a2 2 0 01-2.83 0 .75.75 0 00-1.06 1.06 3.5 3.5 0 004.95 0l2.5-2.5a3.5 3.5 0 00-4.95-4.95l-1.25 1.25zm-4.69 9.64a2 2 0 010-2.83l2.5-2.5a2 2 0 012.83 0 .75.75 0 001.06-1.06 3.5 3.5 0 00-4.95 0l-2.5 2.5a3.5 3.5 0 004.95 4.95l1.25-1.25a.75.75 0 00-1.06-1.06l-1.25 1.25a2 2 0 01-2.83 0z"></path></svg></a>Example</h2>
<p>Calibration errors can be estimated from a data set of predicted probabilities
and and a set of targets by executing</p>
<div class="highlight highlight-source-julia"><pre><span class="pl-c1">calibrationerror</span>(estimator, predictions, targets)</pre></div>
<p>The predictions can be provided as a vector of <code>n</code> vectors of predicted probabilities
of length <code>m</code> or as a matrix of size <code>(m, n)</code>, in which each of the <code>n</code> columns corresponds
to predicted probabilities of the targets <code>1,…,m</code>. The corresponding targets have to be
provided as a vector of length <code>n</code>, in which every element is from the set <code>1,…,m</code>.</p>
<p>Alternatively, it is possible to specify a tuple of predictions and targets or a vector of
tuples of predictions and targets.</p>
<p>This package implements the estimator <code>ECE</code> of the ECE, the estimators
<code>BiasedSKCE</code>, <code>UnbiasedSKCE</code>, and <code>BlockUnbiasedSKCE</code> for the SKCE, and <code>SUMCE</code> for the
SUMCE.</p>
