<p dir="auto">This package provides morphology operations for structure analysis and image processing.</p>
<p dir="auto">ImageMorphology is a sub-package of the umbrella package <a href="https://github.com/JuliaImages/Images.jl">Images.jl</a> -- either <code>using Images</code> or <code>using ImageMorphology</code> will give you access to this functionality.</p>
