<p>GibbsSeaWater.jl is a Julia wrapper for <a href="https://github.com/TEOS-10/GSW-C/">GSW-C#master</a>, which is the C implementation of the Thermodynamic Equation of Seawater 2010 (TEOS-10).</p>
