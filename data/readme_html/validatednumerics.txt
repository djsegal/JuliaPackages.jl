<p dir="auto"><code>ValidatedNumerics.jl</code> is a suite of Julia packages for performing <em>Validated Numerics</em> in Julia, i.e. <em>rigorous</em> computations with finite-precision floating-point arithmetic, using <strong>interval arithmetic</strong>: quantities are treated as intervals that are propagated throughout a calculation. The final result is an interval that is <em>guaranteed</em> to contain the correct result, starting from the given initial data.</p>
<p dir="auto">The aims of the package are both correctness and good performance.</p>

<p dir="auto">To install the package, from within Julia do</p>
<div class="highlight highlight-source-julia notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="julia&gt; Pkg.add(&quot;ValidatedNumerics&quot;)"><pre>julia<span class="pl-k">&gt;</span> Pkg<span class="pl-k">.</span><span class="pl-c1">add</span>(<span class="pl-s"><span class="pl-pds">"</span>ValidatedNumerics<span class="pl-pds">"</span></span>)</pre></div>
<p dir="auto">Since version 0.9, <code>ValidatedNumerics.jl</code> is a meta-package that automatically installs and reexports the following packages from the <a href="https://github.com/JuliaIntervals"><code>JuliaIntervals</code> GitHub organization</a>:</p>
<ul dir="auto">
<li>
<p dir="auto"><a href="https://github.com/JuliaIntervals/IntervalArithmetic.jl"><code>IntervalArithmetic.jl</code></a>: arithmetic and elementary functions on intervals</p>
</li>
<li>
<p dir="auto"><a href="https://github.com/JuliaIntervals/IntervalRootFinding.jl"><code>IntervalRootFinding.jl</code></a>: find roots of multidimensional functions in a guaranteed way</p>
</li>
<li>
<p dir="auto"><a href="https://github.com/JuliaIntervals/IntervalOptimisation.jl"><code>IntervalOptimisation.jl</code></a>: guaranteed global optimisation of multidimensional functions</p>
</li>
<li>
<p dir="auto"><a href="https://github.com/JuliaIntervals/IntervalConstraintProgramming.jl"><code>IntervalConstraintProgramming.jl</code></a>: characterization of feasible sets of equations and inequalities via constraint propagation</p>
</li>
<li>
<p dir="auto"><a href="https://github.com/JuliaIntervals/IntervalContractors.jl"><code>IntervalContractors.jl</code></a>: contractors and reverse (or inverse) functions</p>
</li>
<li>
<p dir="auto"><a href="https://github.com/JuliaIntervals/TaylorModels.jl"><code>TaylorModels.jl</code></a>: rigorous function approximation</p>
</li>
</ul>

<p dir="auto">The IEEE Std 1788-2015 - IEEE Standard for Interval Arithmetic was <a href="https://standards.ieee.org/findstds/standard/1788-2015.html" rel="nofollow">published</a> in June 2015. We are working towards having <code>ValidatedNumerics</code> be conformant with this standard.</p>
<p dir="auto">To do so, we have incorporated tests from the excellent <a href="https://github.com/oheim/ITF1788">ITF1788 test suite</a>, originally written by Marco Nehmeier and Maximilian Kiesner, and converted to a common format and to output tests for Julia by Oliver Heimlich.</p>
