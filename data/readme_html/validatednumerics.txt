<p><code>ValidatedNumerics.jl</code> is a suite of Julia packages for performing <em>Validated Numerics</em> in Julia, i.e. <em>rigorous</em> computations with finite-precision floating-point arithmetic, using <strong>interval arithmetic</strong>: quantities are treated as intervals that are propagated throughout a calculation. The final result is an interval that is <em>guaranteed</em> to contain the correct result, starting from the given initial data.</p>
<p>The aims of the package are both correctness and good performance.</p>

<p>To install the package, from within Julia do</p>
<div class="highlight highlight-source-julia"><pre>julia<span class="pl-k">&gt;</span> Pkg<span class="pl-k">.</span><span class="pl-c1">add</span>(<span class="pl-s"><span class="pl-pds">"</span>ValidatedNumerics<span class="pl-pds">"</span></span>)</pre></div>
<p>Since version 0.9, <code>ValidatedNumerics.jl</code> is a meta-package that automatically installs and reexports the following packages from the <a href="https://github.com/JuliaIntervals"><code>JuliaIntervals</code> GitHub organization</a>:</p>
<ul>
<li>
<p><a href="https://github.com/JuliaIntervals/IntervalArithmetic.jl"><code>IntervalArithmetic.jl</code></a>: arithmetic and elementary functions on intervals</p>
</li>
<li>
<p><a href="https://github.com/JuliaIntervals/IntervalRootFinding.jl"><code>IntervalRootFinding.jl</code></a>: find roots of multidimensional functions in a guaranteed way</p>
</li>
<li>
<p><a href="https://github.com/JuliaIntervals/IntervalOptimisation.jl"><code>IntervalOptimisation.jl</code></a>: guaranteed global optimisation of multidimensional functions</p>
</li>
<li>
<p><a href="https://github.com/JuliaIntervals/IntervalConstraintProgramming.jl"><code>IntervalConstraintProgramming.jl</code></a>: characterization of feasible sets of equations and inequalities via constraint propagation</p>
</li>
<li>
<p><a href="https://github.com/JuliaIntervals/IntervalContractors.jl"><code>IntervalContractors.jl</code></a>: contractors and reverse (or inverse) functions</p>
</li>
</ul>

<h2><a id="user-content-ieee-standard-1788-2015---ieee-standard-for-interval-arithmetic" class="anchor" aria-hidden="true" href="#ieee-standard-1788-2015---ieee-standard-for-interval-arithmetic"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path fill-rule="evenodd" d="M7.775 3.275a.75.75 0 001.06 1.06l1.25-1.25a2 2 0 112.83 2.83l-2.5 2.5a2 2 0 01-2.83 0 .75.75 0 00-1.06 1.06 3.5 3.5 0 004.95 0l2.5-2.5a3.5 3.5 0 00-4.95-4.95l-1.25 1.25zm-4.69 9.64a2 2 0 010-2.83l2.5-2.5a2 2 0 012.83 0 .75.75 0 001.06-1.06 3.5 3.5 0 00-4.95 0l-2.5 2.5a3.5 3.5 0 004.95 4.95l1.25-1.25a.75.75 0 00-1.06-1.06l-1.25 1.25a2 2 0 01-2.83 0z"></path></svg></a>IEEE Standard 1788-2015 - IEEE Standard for Interval Arithmetic</h2>
<p>The IEEE Std 1788-2015 - IEEE Standard for Interval Arithmetic was <a href="https://standards.ieee.org/findstds/standard/1788-2015.html" rel="nofollow">published</a> in June 2015. We are working towards having <code>ValidatedNumerics</code> be conformant with this standard.</p>
<p>To do so, we have incorporated tests from the excellent <a href="https://github.com/oheim/ITF1788">ITF1788 test suite</a>, originally written by Marco Nehmeier and Maximilian Kiesner, and converted to a common format and to output tests for Julia by Oliver Heimlich.</p>
