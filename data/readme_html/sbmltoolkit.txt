<p dir="auto">SBMLToolkit.jl is a lightweight tool to import models specified in the Systems Biology Markup Language (SBML) into the Julia SciML ecosystem. There are multiple ways to specify the same model in SBML. Please help us improving SBMLToolkit.jl by creating a GitHub issue if you experience errors when converting your SBML model.</p>
<p dir="auto">SBMLToolkit uses the <a href="https://github.com/LCSB-BioCore/SBML.jl">SBML.jl</a> wrapper of the <a href="https://model.caltech.edu/software/libsbml/" rel="nofollow">libSBML</a> library to lower dynamical SBML models into dynamical systems. If you would like to import BioNetGen models use the <code>writeSBML()</code> export function or import the <code>.net</code> file with <a href="https://github.com/SciML/ReactionNetworkImporters.jl">ReactionNetworkImporters.jl</a>. For constrained-based modelling, please have a look at <a href="https://github.com/LCSB-BioCore/COBREXA.jl">COBREXA.jl</a>.</p>

<p dir="auto">SBML models can be simulated with the following steps (note that <code>sol</code> is in absolute quantities rather than concentrations):</p>
<div class="highlight highlight-source-julia notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="using SBMLToolkit, ModelingToolkit, OrdinaryDiffEq

SBMLToolkit.checksupport_file(&quot;my_model.xml&quot;)
mdl = readSBML(&quot;my_model.xml&quot;, doc -&gt; begin
                   set_level_and_version(3, 2)(doc)
                   convert_promotelocals_expandfuns(doc)
               end)

rs = ReactionSystem(mdl)  # If you want to create a reaction system
odesys = convert(ODESystem, rs)  # Alternatively: ODESystem(mdl)

tspan = (0.0, 1.0)
prob = ODEProblem(odesys, [], tspan, [])
sol = solve(prob, Tsit5())"><pre><span class="pl-k">using</span> SBMLToolkit, ModelingToolkit, OrdinaryDiffEq

mdl <span class="pl-k">=</span> <span class="pl-c1">readSBML</span>(<span class="pl-s"><span class="pl-pds">"</span>my_model.xml<span class="pl-pds">"</span></span>, doc <span class="pl-k">-&gt;</span> <span class="pl-k">begin</span>
                   <span class="pl-c1">set_level_and_version</span>(<span class="pl-c1">3</span>, <span class="pl-c1">2</span>)(doc)

               <span class="pl-k">end</span>)

rs <span class="pl-k">=</span> <span class="pl-c1">ReactionSystem</span>(mdl)  <span class="pl-c"><span class="pl-c">#</span> If you want to create a reaction system</span>
odesys <span class="pl-k">=</span> <span class="pl-c1">convert</span>(ODESystem, rs)  <span class="pl-c"><span class="pl-c">#</span> Alternatively: ODESystem(mdl)</span>

tspan <span class="pl-k">=</span> (<span class="pl-c1">0.0</span>, <span class="pl-c1">1.0</span>)
