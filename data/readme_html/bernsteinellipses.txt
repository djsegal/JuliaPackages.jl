<p dir="auto">Convenience functions for working with Bernstein ellipses.</p>
<p dir="auto">Exported functions:</p>
<ul dir="auto">
<li><code>jouk(z)</code>: Forward Joukowsky map <code>jouk(z) = (z + inv(z))/2</code>.</li>
<li><code>ijouk(x)</code>: Inverse Joukowsky map. Different branches of this map can be selected using keywords arguments.</li>
</ul>
<p dir="auto">Unexported functions due to likely name clashes:</p>
<ul dir="auto">
<li><code>radius(x) = abs(ijouk(x))</code></li>
<li><code>semimajor(x)</code>: Semi-major axis of the Bernstein ellipse through <code>x</code>.</li>
<li><code>semiminor(x)</code>: Semi-minor axis of the Bernstein ellipse through <code>x</code>.</li>
<li><code>semiminor(x,xr)</code>: Imaginary component of the point in the upper half-plane where the line <code>{ x̃ | real(x̃) = xr }</code> intersects the Bernstein ellipse through <code>x</code>.</li>
</ul>
