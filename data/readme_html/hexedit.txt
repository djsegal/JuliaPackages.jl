<p>HexEdit is a package for editing and displaying data in binary files in
hexadecimal format.</p>

<h4><a id="user-content-dumpselfhexed-offset--0-nint---1" class="anchor" aria-hidden="true" href="#dumpselfhexed-offset--0-nint---1"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path fill-rule="evenodd" d="M7.775 3.275a.75.75 0 001.06 1.06l1.25-1.25a2 2 0 112.83 2.83l-2.5 2.5a2 2 0 01-2.83 0 .75.75 0 00-1.06 1.06 3.5 3.5 0 004.95 0l2.5-2.5a3.5 3.5 0 00-4.95-4.95l-1.25 1.25zm-4.69 9.64a2 2 0 010-2.83l2.5-2.5a2 2 0 012.83 0 .75.75 0 001.06-1.06 3.5 3.5 0 00-4.95 0l-2.5 2.5a3.5 3.5 0 004.95 4.95l1.25-1.25a.75.75 0 00-1.06-1.06l-1.25 1.25a2 2 0 01-2.83 0z"></path></svg></a>dump!(self::HexEd, offset = 0, n::Int = -1)</h4>
<p>Displays binary file data beginning at offset and ending at offset + n.</p>
<ul>
<li>offset defaults to 0</li>
<li>n defaults to file size - n.</li>
</ul>
<h4><a id="user-content-editselfhexed-datastrstring-offset--0" class="anchor" aria-hidden="true" href="#editselfhexed-datastrstring-offset--0"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path fill-rule="evenodd" d="M7.775 3.275a.75.75 0 001.06 1.06l1.25-1.25a2 2 0 112.83 2.83l-2.5 2.5a2 2 0 01-2.83 0 .75.75 0 00-1.06 1.06 3.5 3.5 0 004.95 0l2.5-2.5a3.5 3.5 0 00-4.95-4.95l-1.25 1.25zm-4.69 9.64a2 2 0 010-2.83l2.5-2.5a2 2 0 012.83 0 .75.75 0 001.06-1.06 3.5 3.5 0 00-4.95 0l-2.5 2.5a3.5 3.5 0 004.95 4.95l1.25-1.25a.75.75 0 00-1.06-1.06l-1.25 1.25a2 2 0 01-2.83 0z"></path></svg></a>edit!(self::HexEd, datastr::String, offset = 0)</h4>
<p>Edits targeted binary file by overwriting data beginning at offset.</p>
<ul>
<li>offset defaults to 0</li>
<li>datastr can be in ASCII or hexadecimal format (ie. "foobar" or "0x666f6f626172")</li>
</ul>
