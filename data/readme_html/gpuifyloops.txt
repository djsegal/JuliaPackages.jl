<p>In order to test this package locally you need to do:</p>
<div class="snippet-clipboard-content position-relative" data-snippet-clipboard-copy-content="julia --project=test/gpuenv
julia&gt; ]
(gpuenv) pkg&gt; resolve
(gpuenv) pkg&gt; instantiate
"><pre><code>julia --project=test/gpuenv
julia&gt; ]
(gpuenv) pkg&gt; resolve
(gpuenv) pkg&gt; instantiate
</code></pre></div>
<p>This will resolve the GPU environment, please do not checking changes to <code>test/gpuenv/</code>.
Then you can run the tests with <code>julia --project=test/gpuenv test/runtests.jl</code></p>
