<p>GPUifyLoops is a registered package, and can be installed using the Julia package
manager.</p>
<div class="highlight highlight-source-julia"><pre>julia<span class="pl-k">&gt;</span>]
(v1<span class="pl-c1">.1</span>) pkg<span class="pl-k">&gt;</span> add GPUifyLoops</pre></div>
<p><strong>Note</strong>: The current version of this package requires Julia 1.1.</p>

<p>Debugging failures to transforma a function for the GPU requires the use of <a href="https://github.com/JuliaDebug/Cthulhu.jl"><code>Cthulhu.jl</code></a>.</p>
<pre><code>using Cthulhu
using GPUifyLoops

# @launch CUDA() f(args...)
descend(GPUifyLoops.signature(f, args...)...)
</code></pre>

<p>In order to test this package locally you need to do:</p>
<pre><code>julia --project=test/gpuenv
julia&gt; ]
(gpuenv) pkg&gt; resolve
(gpuenv) pkg&gt; instantiate
</code></pre>
<p>This will resolve the GPU environment, please do not checking changes to <code>test/gpuenv/</code>.
Then you can run the tests with <code>julia --project=test/gpuenv test/runtests.jl</code></p>
