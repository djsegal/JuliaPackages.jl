<p dir="auto"><em>SoleData</em> provides a <strong>machine learning oriented</strong> data layer on top of DataFrames.jl/Tables.jl for:</p>
<ul dir="auto">
<li>Instantiating and manipulating <a href="https://en.wikipedia.org/wiki/Multimodal_learning" rel="nofollow"><em>multimodal</em></a> datasets for (un)supervised machine learning;</li>
<li>Dealing with <a href="https://en.wikipedia.org/wiki/Unstructured_data" rel="nofollow"><em>(un)structured</em> data</a> (e.g., graphs, images, time-series, etc.);</li>
<li>Describing datasets via basic statistical measures;</li>
<li>Saving to/loading from <em>npy/npz</em> format, as well as a custom CSV-based format (with interesting features such as <em>lazy loading</em> of datasets);</li>
<li>Performing basic data processing operations (e.g., windowing, moving average, etc.).</li>
</ul>
<p dir="auto">If you are used to dealing with unstructured/multimodal data, but cannot find the right tools in Julia,
you will find <a href="https://github.com/aclai-lab/SoleFeatures.jl/"><em>SoleFeatures.jl</em></a> useful!</p>
