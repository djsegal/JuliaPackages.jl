<p>A Julia package expanding the <a href="https://github.com/sisl/AutomotiveDrivingModels.jl">AutomotiveDrivingModels.jl</a> with intersections, enabling auto-converting roadway in AutomotiveDrivingModels.jl into <a href="http://www.opendrive.org/" rel="nofollow">OpenDrive format (.xodr)</a> used in <a href="https://vires.com/vtd-vires-virtual-test-drive/" rel="nofollow">VIRES VirtualTestDrive</a>. It also contains some driver models.</p>
