<p dir="auto">UC Irvine Machine Learning Repository is one the most popular collection of datasets that are avalaible for free.</p>
<p dir="auto">This Package provides functions for the user to easily download from the website directly into a DataFrame.</p>
<p dir="auto">Additionally, another function allows the user to view the accompanying metadata about the dataset.</p>
