<p><a href="https://github.com/queryverse/VegaLite.jl">VegaLite.jl</a> is a plotting package for the <a href="https://julialang.org/" rel="nofollow">julia</a> programming language. The package is based on <a href="https://vega.github.io/vega-lite/" rel="nofollow">Vega-Lite</a>, which extends a traditional <a href="https://doi.org/10.1007/0-387-28695-0" rel="nofollow">grammar of graphics</a> API into a <a href="https://doi.org/10.1109/TVCG.2016.2599030" rel="nofollow">grammar of interactive graphics</a>.</p>
<p>Additionally, <a href="https://github.com/queryverse/VegaLite.jl">VegaLite.jl</a> provides basic support for <a href="https://vega.github.io/vega" rel="nofollow">Vega</a>.</p>
<p><a href="https://github.com/queryverse/VegaLite.jl">VegaLite.jl</a> allows you to create a wide range of statistical plots. It exposes the full functionality of the underlying <a href="https://vega.github.io/vega-lite/" rel="nofollow">Vega-Lite</a> and is at the same time tightly integrated into the julia ecosystem. Here is an example of a scatter plot:</p>
<div class="snippet-clipboard-content position-relative" data-snippet-clipboard-copy-content="using VegaLite, VegaDatasets

<p><a href="https://github.com/queryverse/VegaLite.jl">VegaLite.jl</a> can be used in combination with many different front end clients including <a href="https://jupyterlab.readthedocs.io/en/stable/" rel="nofollow">JupyterLab</a>, <a href="https://github.com/fonsp/Pluto.jl">Pluto.jl</a>, and <a href="https://www.julia-vscode.org" rel="nofollow">Julia for VSCode</a>.</p>
<p><strong>Important:</strong> <a href="https://jupyterlab.readthedocs.io/en/stable/" rel="nofollow">JupyterLab</a> supplies its own version of <a href="https://vega.github.io/vega-lite/" rel="nofollow">Vega-Lite</a> and <em>cannot</em> be kept in perfect sync with <a href="https://github.com/queryverse/VegaLite.jl">VegaLite.jl</a>. Formatting discrepancies <strong>can</strong> and <strong>will</strong> arise on occasion for <a href="https://jupyterlab.readthedocs.io/en/stable/" rel="nofollow">JupyterLab</a> users. <a href="https://github.com/queryverse/VegaLite.jl">VegaLite.jl</a> targets the latest available minor release of <a href="https://vega.github.io/vega-lite/" rel="nofollow">Vega-Lite</a> which is also supported by <a href="https://jupyterlab.readthedocs.io/en/stable/" rel="nofollow">JupyterLab</a>.</p>

<p>JupyterLab support depends on Juperlab's built-in renderer; before updating to a new major version of vega-lite, check the <a href="https://github.com/jupyterlab/jupyterlab/blob/master/packages/vega5-extension/package.json#L42">JupyterLab project</a> for the latest supported version of vega-lite.</p>
