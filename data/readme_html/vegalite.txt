<p><a href="https://github.com/queryverse/VegaLite.jl">VegaLite.jl</a> is a plotting package for the <a href="https://julialang.org/" rel="nofollow">julia</a> programming language. The package is based on <a href="https://vega.github.io/vega-lite/" rel="nofollow">Vega-Lite</a>, which extends a traditional <a href="https://doi.org/10.1007/0-387-28695-0" rel="nofollow">grammar of graphics</a> API into a <a href="https://doi.org/10.1109/TVCG.2016.2599030" rel="nofollow">grammar of interactive graphics</a>.</p>
<p>Additionally, <a href="https://github.com/queryverse/VegaLite.jl">VegaLite.jl</a> provides basic support for <a href="https://vega.github.io/vega" rel="nofollow">Vega</a>.</p>
<p><a href="https://github.com/queryverse/VegaLite.jl">VegaLite.jl</a> allows you to create a wide range of statistical plots. It exposes the full functionality of the underlying <a href="https://vega.github.io/vega-lite/" rel="nofollow">Vega-Lite</a> and is at the same time tightly integrated into the julia ecosystem. Here is an example of a scatter plot:</p>
<pre lang="@example"><code>using VegaLite, VegaDatasets

dataset("cars") |&gt;
@vlplot(
    :point,
    x=:Horsepower,
    y=:Miles_per_Gallon,
    color=:Origin,
    width=400,
    height=400
)
</code></pre>
