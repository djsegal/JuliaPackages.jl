<p>This package implements a robust method to fit a given function
(described by its parameters) to input data. The method is based on
the LOVO algorithm [1] and also in a suitable voting strategy in order
automatically eliminate outliers.</p>
<p>Potential use of this package is in statistical  problems which depend on fitting functions or in more specific fields like computer vision for problems related shape detection.</p>
<p>For a brief description of the potential of this program and the difference between related strategies in the literature, like least square, consider the following dataset</p>
<p><code>A=[-2.0  5.0;  -1.5  3.25; -1.0  2.0 ; -0.5  1.25; 0.0  1.0 ; 0.5  2.55; 1.0  2.0 ; 1.5  3.25; 2.0  5.0 ;]</code></p>
<p>Let us assume that we wish to find a fitting function given by parameters <code>x[1], x[2]</code> and the model</p>
<p><code>f(t)=x[1]*t^2+x[2]</code></p>
<p>where <code>f(t[i])≈A[i,2]</code> in the least square sense. When we apply a least square strategy we obtain</p>
<p><a target="_blank" rel="noopener noreferrer" href="/docs/src/assets/figure_least_square_example.png"><img src="/docs/src/assets/figure_least_square_example.png" height="80%" width="80%" style="max-width:100%;"></a></p>
<p>and when we apply RAFF strategy we get</p>
<p><a target="_blank" rel="noopener noreferrer" href="/docs/src/assets/figure_RAFF_example.png"><img src="/docs/src/assets/figure_RAFF_example.png" height="80%" width="80%" style="max-width:100%;"></a></p>
<p>Note that RAFF detects (automatically) the model <code>f(t)=1.0*t^2 +1.0</code> that is, RAFF detects the point <code>(0.5,2.55)</code> like an outlier and excludes the influence of this point for determine <code>f(t)</code>.
Consequently, RAFF detects the number of possible outliers and in the same time, which their are. This fact distinguishes RAFF from other techniques like [1] or classical least square.</p>
<p>Other useful examples are found in the documentation and also in directory <a href="examples/"><code>examples/</code></a>.</p>
