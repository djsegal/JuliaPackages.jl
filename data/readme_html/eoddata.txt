<p>Call login. This will assign you a token which is needed to
make EodData web service calls.
The second call directly assigns a token, which is merely
an <code>::ASCIIString</code>.</p>
<div class="highlight highlight-source-julia"><pre>resp <span class="pl-k">=</span> <span class="pl-c1">login</span>(USERNAME, PASSWORD)
<span class="pl-c1">println</span>(resp<span class="pl-k">.</span>message)
<span class="pl-c1">println</span>(resp<span class="pl-k">.</span>token)
<span class="pl-c1">println</span>(<span class="pl-c1">typeof</span>(resp))

token <span class="pl-k">=</span> <span class="pl-c1">login</span>(USERNAME, PASSWORD)<span class="pl-k">.</span>token
<span class="pl-c1">println</span>(token)
<span class="pl-c1">println</span>(<span class="pl-c1">typeof</span>(token))</pre></div>
