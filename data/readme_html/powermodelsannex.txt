<p><a href="https://github.com/lanl-ansi/PowerModels.jl">PowerModels.jl</a> provides an implementation reference for <em>established</em> formulations and methods in power system optimization, and hence is is not an appropriate location for more exploratory work.  PowerModelsAnnex.jl is an extension of PowerModels.jl that provides a home for open-source sharing of preliminary and/or exploratory methods in power system optimization.</p>
<p>Due to the exploratory nature of PowerModelsAnnex,</p>
<ul>
<li>there is minimal documentation and testing</li>
<li>there are limited code quality and reliablity standards</li>
<li>anything goes in the annex, more-or-less</li>
</ul>
<p>Users should be prepared for features that break.</p>
<p>Pull Requests to PowerModelsAnnex are always welcome and not subject to significant scrutiny.</p>
