<p>DeepDiffs.jl provides the <code>deepdiff</code> function, which finds and displays differences (diffs) between Julia data structures. It supports <code>Vector</code>s, <code>Dict</code>s, and <code>String</code>s. When diffing dictionaries where values associated with a particular key may change, <code>deepdiff</code> will recurse into value to provide a more detailed diff.</p>
<p>Many users will likely only use the <code>deepdiff</code> function to interactively visualize diffs. For more advanced usage, the return value from <code>deepdiff</code> will be some subtype of the <code>DeepDiff</code> abstract type which can be further manipulated. These subtypes support the following functions:</p>
<ul>
<li><code>before(diff)</code>: returns the first original (left-hand-side) value that was diffed</li>
<li><code>after(diff)</code>: returns the modified (right-hand-side) value that was diffed</li>
<li><code>added(diff)</code>: returns a list of indices or dictionary keys that were new items. These indices correspond to the "after" value.</li>
<li><code>removed(diff)</code>: returns a list of indices or dictionary keys that were removed. These indices correspond to the "before" value.</li>
<li><code>changed(diff)</code>: returns a dictionary whose keys are indices or dictionary keys and whose values are themselves <code>DeepDiff</code>s that describe the modified value. Currently this is only meaningful when diffing dictionaries because the keys can be matched up between the original and modified values.</li>
</ul>

<p><code>Vector</code>s are diffed using a longest-subsequence algorithm that tries to minmize the number of additions and removals necessary to transform one <code>Vector</code> to another.</p>

<p><code>Dict</code>s are diffed by matching up the keys between the original and modified values, so it can recognize removed, added, or modified values.</p>

<p>Single-line strings are diffed character-by-character. The indices returned by <code>added</code> and <code>removed</code> correspond to indices in the <code>Vector</code> of characters returned by <code>collect(str::String)</code>.</p>

<p>Multi-line strings (strings with at least one newline) are diffed line-by-line. The indices returned by <code>added</code> and <code>removed</code> correspond to line numbers.</p>
