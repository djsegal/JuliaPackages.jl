<p dir="auto"><code>PeriodicSystems.jl</code> is intended to be a collection of Julia functions for numerical computations related to periodic system representations in the continuous-time form</p>
<div class="snippet-clipboard-content notranslate position-relative overflow-auto" data-snippet-clipboard-copy-content=" dx(t)/dt = A(t)x(t) + B(t)u(t) ,
 y(t)     = C(t)x(t) + D(t)u(t) ,"><pre class="notranslate"><code> dx(t)/dt = A(t)x(t) + B(t)u(t) ,
 y(t)     = C(t)x(t) + D(t)u(t) ,
</code></pre></div>
<p dir="auto">or in the discrete-time form</p>
<div class="snippet-clipboard-content notranslate position-relative overflow-auto" data-snippet-clipboard-copy-content=" x(t+1)  = A(t)x(t) + B(t)u(t) ,
 y(t)    = C(t)x(t) + D(t)u(t) ,"><pre class="notranslate"><code> x(t+1)  = A(t)x(t) + B(t)u(t) ,
 y(t)    = C(t)x(t) + D(t)u(t) ,
</code></pre></div>
<p dir="auto">where <code>x(t)</code>, <code>u(t)</code> and <code>y(t)</code> are the system state vector, system input vector and system output vector, respectively, and <code>t</code> is the continuous or discrete time variable. The system matrices satisfy <code>A(t) = A(t+T)</code>, <code>B(t) = B(t+T)</code>, <code>C(t) = C(t+T)</code>, <code>D(t) = D(t+T)</code>,  i.e., are periodic with period <code>T</code>.</p>
<p dir="auto">Many control applications are formulated as genuine
periodic control problems as for example, satellite attitude control, helicopter forward flight control, orbital stabilization of underactuated systems, etc. Besides
that, periodic systems represent a general framework to analyze and design multi-rate sampled-data systems.</p>
<p dir="auto">The targeted functionality of this package is described in [1].</p>
<p dir="auto">[1] A. Varga. <a href="https://elib.dlr.de/12283/1/varga_ifac2005p1.pdf" rel="nofollow">A Periodic Systems Toolbox for Matlab</a>. Proc. of IFAC 2005 World Congress, Prague, Czech Republic, 2005.</p>
