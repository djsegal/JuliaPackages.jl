<p dir="auto"><code>FlexibilityAnalysis.jl</code> is long out of date and is currently without a developer. It last worked with Julia <code>v1.1</code> and JuMP <code>v0.18</code> using Gurobi.jl <code>v.6</code> (wrapping Gurobi <code>0.7.x</code>). Please respond to <a href="https://github.com/pulsipher/FlexibilityAnalysis.jl/issues/2" data-hovercard-type="issue" data-hovercard-url="/pulsipher/FlexibilityAnalysis.jl/issues/2/hovercard">this issue</a> if you would like to help bring this up-to-date.</p>
