<p dir="auto">Zarr is a Julia package providing an implementation of chunked, compressed, N-dimensional arrays. <a href="https://zarr.readthedocs.io/en/stable/" rel="nofollow">Zarr</a> is originally a Python package. In Zarr we aim to implement the <a href="https://zarr.readthedocs.io/en/stable/spec/v2.html" rel="nofollow">zarr spec</a>.</p>

<p dir="auto">The package currently implements basic functionality for reading and writing zarr arrays. However, the package is under active development, since many compressors and backends supported by the python implementation are still missing.</p>
