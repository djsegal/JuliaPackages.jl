<p dir="auto">GraphNeuralNetworks.jl is a graph neural network library written in Julia and based on the deep learning framework <a href="https://github.com/FluxML/Flux.jl">Flux.jl</a>.</p>
<p dir="auto">Among its features:</p>
<ul dir="auto">
<li>Implements common graph convolutional layers.</li>
<li>Supports computations on batched graphs.</li>
<li>Easy to define custom layers.</li>
<li>CUDA support.</li>
<li>Integration with <a href="https://github.com/JuliaGraphs/Graphs.jl">Graphs.jl</a>.</li>
<li><a href="https://github.com/CarloLucibello/GraphNeuralNetworks.jl/tree/master/examples">Examples</a> of node, edge, and graph level machine learning tasks.</li>
</ul>
