<p dir="auto">A Julia package for representing system topologies for molecular simulations.</p>
<p dir="auto">The package currently only supporting reading atom names, residues names, and
residue indices from a GROMACS-format <code>.gro</code> file. Other formats may be
supported in the future.</p>
