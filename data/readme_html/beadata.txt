<p><em>BeaData.jl</em> ia a Julia interface for retrieving data from the U.S. Bureau of Economic Analysis (BEA) Data API.</p>
<p>A valid User ID key is required to use the BEA's API. A User ID can be obtained by registering  at the BEA website: <a href="https://apps.bea.gov/API/signup/index.cfm" rel="nofollow">https://apps.bea.gov/API/signup/index.cfm</a>.</p>
<p><em>BeaData.jl</em> is not affiliated with, officially maintained, or otherwise supported by the Bureau of Economic Analysis.</p>
