<p dir="auto">A <a href="https://julialang.org/" rel="nofollow">Julia package</a> to convert <a href="https://github.com/fonsp/Pluto.jl">Pluto notebooks</a> to static HTML.
Unlike <a href="https://github.com/JuliaPluto/PlutoSliderServer.jl">PlutoSliderServer</a>, the HTML files generated using PlutoStaticHTML are very minimal, and do not require JavaScript on the user side to render.
This makes it easier to style the output using CSS.
For example, with this package it is possible to create a website full with plots generated in Julia and all code hidden.
In effect, no one would be able to tell that the website was built with Julia.</p>
