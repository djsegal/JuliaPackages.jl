<p dir="auto">Individual.jl is a package which uses <a href="https://algebraicjulia.github.io/Catlab.jl/stable/" rel="nofollow">Catlab.jl</a>,
especially attributed C-Sets to create a set of data types and methods useful for building
individual-based epidemiological models. It is inspired by the R software <a href="https://mrc-ide.github.io/individual/" rel="nofollow">individual</a>.
Please read our <a href="https://slwu89.github.io/Individual.jl/dev/" rel="nofollow">documentation website</a> to
learn more about the package.</p>
