<li>Ensure Python is installed and working on your machine.</li>
<li><a href="https://matplotlib.org/users/installing.html" rel="nofollow">Install matplotlib for Python.</a> *</li>
<li>From the Julia prompt, press "]" to enter package mode, then type "add PlotSeis"</li>
<li>Exit package mode with "^C", then type "using PlotSeis"</li>
</ol>
<p>* for OS X, we recommend the <a href="https://brew.sh/" rel="nofollow">Homebrew</a> installation instructions of <a href="https://github.com/JuliaPy/PyPlot.jl">PyPlot.jl</a>:</p>
<pre><code>brew install python gcc freetype pyqt
brew link --force freetype
pip install numpy scipy matplotlib
</code></pre>
