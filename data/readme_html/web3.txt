<p dir="auto">A module for Ethereum connectivity.</p>
<p dir="auto">Parse ABI files, encode/decode ABI data, and make Ethereum <a href="https://github.com/ethereum/wiki/wiki/JSON-RPC">JSON-RPC</a> calls: low-level calls, smart contract function calls and transactions.</p>
<p dir="auto">This is a work in progress but work seems to be progressing...</p>
<p dir="auto">Here are some <a href="example/julia/dumpContract.jl">examples</a></p>

<li>readABI(con::Web3Connection, contractname::String, stream::IO): Read an ABI file for a contract</li>
