<p>A Julia package containing a number of algorithms for analyzing connected components
in binary images. In general, the input is an array of labelled components, and the output
is a <a href="https://github.com/JuliaData/DataFrames.jl">DataFrame</a> where each row
represents a connected component, and each column represents an attribute of
that connected component. When combined with <a href="https://github.com/queryverse/Query.jl">Query</a>
it affords a very flexible and easy way of filtering connected components
based on their attributes.</p>
<p>The general usage pattern of this package is:</p>
<div class="highlight highlight-source-julia"><pre>measurements <span class="pl-k">=</span> <span class="pl-c1">analyze_components</span>(components, algorithm<span class="pl-k">::</span><span class="pl-c1">AbstractComponentAnalysisAlgorithm</span>)</pre></div>
<p>For more detailed usage and a full list of algorithms, please check the <a href="https://zygmuntszpak.github.io/ImageComponentAnalysis.jl/stable" rel="nofollow">documentation</a>.</p>
