<h3 dir="auto"><a id="user-content-note-sequence-conventions" class="anchor" aria-hidden="true" href="#note-sequence-conventions"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path d="m7.775 3.275 1.25-1.25a3.5 3.5 0 1 1 4.95 4.95l-2.5 2.5a3.5 3.5 0 0 1-4.95 0 .751.751 0 0 1 .018-1.042.751.751 0 0 1 1.042-.018 1.998 1.998 0 0 0 2.83 0l2.5-2.5a2.002 2.002 0 0 0-2.83-2.83l-1.25 1.25a.751.751 0 0 1-1.042-.018.751.751 0 0 1-.018-1.042Zm-4.69 9.64a1.998 1.998 0 0 0 2.83 0l1.25-1.25a.751.751 0 0 1 1.042.018.751.751 0 0 1 .018 1.042l-1.25 1.25a3.5 3.5 0 1 1-4.95-4.95l2.5-2.5a3.5 3.5 0 0 1 4.95 0 .751.751 0 0 1-.018 1.042.751.751 0 0 1-1.042.018 1.998 1.998 0 0 0-2.83 0l-2.5 2.5a1.998 1.998 0 0 0 0 2.83Z"></path></svg></a>Note: sequence conventions</h3>
<p dir="auto">Sequences passed to RNAstructure use the following convention:</p>
<ul dir="auto">
<li>uppercase character: normal nucleotide, U equivalent to T</li>
<li>lowercase character: nucleotide cannot form basepairs</li>
<li>X or N character: unknown base or base that cannot interact with
others (cannot pair or stack)</li>
</ul>
<p dir="auto">See the <a href="https://rna.urmc.rochester.edu/Text/File_Formats.html#Sequence" rel="nofollow">RNAstructure manual section for
sequences</a>
for more details.</p>
<p dir="auto">Some programs make exceptions to these rules, check the manual pages
of the RNAstructure programs for details on any differences.</p>

<p dir="auto">The <code>mfe</code> function calculates the minimum free energy and the
corresponding minimum free energy structure of an RNA
sequence. Internally, this function calls the <code>Fold</code> program from
RNAstructure.</p>
<p dir="auto">Additional information on the <code>Fold</code> program and possible command-line
options that can be passed via <code>args</code> can be found at the
<a href="https://rna.urmc.rochester.edu/Text/Fold.html" rel="nofollow">RNAstructure Fold
documentation</a>.</p>
