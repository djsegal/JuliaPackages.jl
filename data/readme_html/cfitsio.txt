<ul>
<li>Function names closely mirror the C interface (e.g., <code>fits_open_file()</code>).</li>
<li>Functions operate on <code>FITSFile</code>, a thin wrapper for <code>fitsfile</code> C struct
(<code>FITSFile</code> has concept of "current HDU", as in CFITSIO).</li>
<li>Note that the wrapper functions <em>do</em> check the return status from CFITSIO
and throw an error with the appropriate message.</li>
</ul>
<p>For more information and usage examples, please visit the <a href="https://JuliaAstro.github.io/CFITSIO.jl/dev" rel="nofollow">documentation</a>.</p>
