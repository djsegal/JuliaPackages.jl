<p dir="auto"><strong>Futbolista.jl is a package to make some common football analytics tasks easier to do in Julia.</strong> You can</p>
<ul dir="auto">
<li>load <a href="https://github.com/statsbomb/open-data">Statsbomb's public dataset</a></li>
<li>plot pitches</li>
<li>plot <a href="https://sharmaabhishekk.github.io/projects/passmap" rel="nofollow">passmaps</a>.</li>
</ul>

<p dir="auto">Some of the things to be added to the package are:</p>
<ul dir="auto">
<li>Dataloaders for other free public sources (Wyscout, Metrica, Fbref)</li>

</ul>
<p dir="auto">If you want to help with any of them, please don't hesitate to fork, and PR! A great guide to contributing to Julia packages is <a href="https://www.youtube.com/watch?v=QVmU29rCjaA&amp;t=1796s" rel="nofollow">here</a>.</p>
