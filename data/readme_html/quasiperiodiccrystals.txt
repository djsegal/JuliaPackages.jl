<p dir="auto">This repository contains a reimplementation of the algorithm published in <a href="https://arxiv.org/pdf/2111.08128.pdf" rel="nofollow">Efficient algorithm for simulating particles in true quasiperiodic environments by Alan Rodrigo Mendoza and Atahualpa S. Kraemer</a> in collaboration with Miguel Raz Guzman Macedo.</p>
<p dir="auto">To use this package, run</p>
<div class="highlight highlight-source-julia notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="using Pkg
Pkg.add(&quot;QuasiPeriodicCrystals&quot;)"><pre><span class="pl-k">using</span> Pkg
Pkg<span class="pl-k">.</span><span class="pl-c1">add</span>(<span class="pl-s"><span class="pl-pds">"</span>QuasiPeriodicCrystals<span class="pl-pds">"</span></span>)</pre></div>
<p dir="auto">after opening a <a href="https://julialang.org/downloads/" rel="nofollow">JuliaLang</a> instance.</p>
<p dir="auto">You should then be able to run</p>
<div class="highlight highlight-source-julia notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="using QuasiPeriodicCrystals
