<p dir="auto">Yao is an open source framework that aims to empower quantum information research with software tools. It is designed with following in mind:</p>
<ul dir="auto">
<li>quantum algorithm design;</li>
<li>quantum <a href="https://medium.com/@karpathy/software-2-0-a64152b37c35" rel="nofollow">software 2.0</a>;</li>
<li>quantum computation education.</li>
</ul>
<p dir="auto"><strong>We are in an early-release beta. Expect some adventures and rough edges.</strong></p>
