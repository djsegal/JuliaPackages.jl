<ul dir="auto">
<li>Create an application on the Twitter's Developer Platform: <a href="https://developer.twitter.com/en" rel="nofollow">https://developer.twitter.com/en</a></li>
<li>Get CONSUMER_KEY(a.k.a API_KEY), CONSUMER_SECRET (a.k.a API_KEY_SECRET), ACCESS_TOKEN and ACCESS_TOKEN_SECRET from your dashboard of Twitter's Developer Platform.</li>
<li>Store them above in <code>.env</code> file so that you can access these secrets from <code>ENV</code>.</li>
</ul>
