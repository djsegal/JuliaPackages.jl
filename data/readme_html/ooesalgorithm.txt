<p dir="auto">A flexible, open-source package to optimize a Linear Function Over the Set of Efficient Solutions for BOMILP</p>
<p dir="auto">Follow this link for the <a href="http://eng.usf.edu/~amsierra/documents/Documentation_OOESAlg.pdf" rel="nofollow">Documentation</a>.</p>
<p dir="auto">This is a criterion space search for optimizing a linear function over the set of efficient solutions of bi-objective mixed integer linear programs. This project is a julia v1.0.2 project which is written in Linux (Ubuntu).</p>
<h3 dir="auto"><a id="user-content-the-following-problem-classes-are-supported" class="anchor" aria-hidden="true" href="#the-following-problem-classes-are-supported"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path d="m7.775 3.275 1.25-1.25a3.5 3.5 0 1 1 4.95 4.95l-2.5 2.5a3.5 3.5 0 0 1-4.95 0 .751.751 0 0 1 .018-1.042.751.751 0 0 1 1.042-.018 1.998 1.998 0 0 0 2.83 0l2.5-2.5a2.002 2.002 0 0 0-2.83-2.83l-1.25 1.25a.751.751 0 0 1-1.042-.018.751.751 0 0 1-.018-1.042Zm-4.69 9.64a1.998 1.998 0 0 0 2.83 0l1.25-1.25a.751.751 0 0 1 1.042.018.751.751 0 0 1 .018 1.042l-1.25 1.25a3.5 3.5 0 1 1-4.95-4.95l2.5-2.5a3.5 3.5 0 0 1 4.95 0 .751.751 0 0 1-.018 1.042.751.751 0 0 1-1.042.018 1.998 1.998 0 0 0-2.83 0l-2.5 2.5a1.998 1.998 0 0 0 0 2.83Z"></path></svg></a>The following problem classes are supported:</h3>
<p dir="auto">i. Objectives:    2 linear objectives.
ii. Constraints:  0 or more linear (both inequality and equality) constraints.
iii. Variables:
a. Binary
b. Integer variables
c. Continous variables
d. Any combination between previous types of variables.</p>
