<h3><a id="user-content-latest-version-v013" class="anchor" aria-hidden="true" href="#latest-version-v013"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path fill-rule="evenodd" d="M4 9h1v1H4c-1.5 0-3-1.69-3-3.5S2.55 3 4 3h4c1.45 0 3 1.69 3 3.5 0 1.41-.91 2.72-2 3.25V8.59c.58-.45 1-1.27 1-2.09C10 5.22 8.98 4 8 4H4c-.98 0-2 1.22-2 2.5S3 9 4 9zm9-3h-1v1h1c1 0 2 1.22 2 2.5S13.98 12 13 12H9c-.98 0-2-1.22-2-2.5 0-.83.42-1.64 1-2.09V6.25c-1.09.53-2 1.84-2 3.25C6 11.31 7.55 13 9 13h4c1.45 0 3-1.69 3-3.5S14.5 6 13 6z"></path></svg></a>(Latest version: v0.1.3)</h3>
<p>A flexible, open-source package to optimize a Linear Function Over the Set of Efficient Solutions for BOMILP</p>
<p>Follow this link for the <a href="http://eng.usf.edu/~amsierra/documents/Documentation_OOESAlg.pdf" rel="nofollow">Documentation</a>.</p>
<p>This is a criterion space search for optimizing a linear function over the set of efficient solutions of bi-objective mixed integer linear programs. This project is a julia v1.0.2 project which is written in Linux (Ubuntu).</p>
<h3><a id="user-content-the-following-problem-classes-are-supported" class="anchor" aria-hidden="true" href="#the-following-problem-classes-are-supported"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path fill-rule="evenodd" d="M4 9h1v1H4c-1.5 0-3-1.69-3-3.5S2.55 3 4 3h4c1.45 0 3 1.69 3 3.5 0 1.41-.91 2.72-2 3.25V8.59c.58-.45 1-1.27 1-2.09C10 5.22 8.98 4 8 4H4c-.98 0-2 1.22-2 2.5S3 9 4 9zm9-3h-1v1h1c1 0 2 1.22 2 2.5S13.98 12 13 12H9c-.98 0-2-1.22-2-2.5 0-.83.42-1.64 1-2.09V6.25c-1.09.53-2 1.84-2 3.25C6 11.31 7.55 13 9 13h4c1.45 0 3-1.69 3-3.5S14.5 6 13 6z"></path></svg></a>The following problem classes are supported:</h3>
<p>i. Objectives:    2 linear objectives.
ii. Constraints:  0 or more linear (both inequality and equality) constraints.
iii. Variables:
a. Binary
b. Integer variables
c. Continous variables
d. Any combination between previous types of variables.</p>
<h3><a id="user-content-a-multiobjective-mixed-integer-linear-instance-can-be-provided-as-a-input-in-3-ways" class="anchor" aria-hidden="true" href="#a-multiobjective-mixed-integer-linear-instance-can-be-provided-as-a-input-in-3-ways"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path fill-rule="evenodd" d="M4 9h1v1H4c-1.5 0-3-1.69-3-3.5S2.55 3 4 3h4c1.45 0 3 1.69 3 3.5 0 1.41-.91 2.72-2 3.25V8.59c.58-.45 1-1.27 1-2.09C10 5.22 8.98 4 8 4H4c-.98 0-2 1.22-2 2.5S3 9 4 9zm9-3h-1v1h1c1 0 2 1.22 2 2.5S13.98 12 13 12H9c-.98 0-2-1.22-2-2.5 0-.83.42-1.64 1-2.09V6.25c-1.09.53-2 1.84-2 3.25C6 11.31 7.55 13 9 13h4c1.45 0 3-1.69 3-3.5S14.5 6 13 6z"></path></svg></a>A multiobjective mixed integer linear instance can be provided as a input in 3 ways:</h3>
<pre><code>a. JuMP Model
b. LP file format
c. MPS file format
</code></pre>
<h3><a id="user-content-any-mixe-integer-programming-solver-supported-by-mathprogbasejl-can-be-used" class="anchor" aria-hidden="true" href="#any-mixe-integer-programming-solver-supported-by-mathprogbasejl-can-be-used"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path fill-rule="evenodd" d="M4 9h1v1H4c-1.5 0-3-1.69-3-3.5S2.55 3 4 3h4c1.45 0 3 1.69 3 3.5 0 1.41-.91 2.72-2 3.25V8.59c.58-.45 1-1.27 1-2.09C10 5.22 8.98 4 8 4H4c-.98 0-2 1.22-2 2.5S3 9 4 9zm9-3h-1v1h1c1 0 2 1.22 2 2.5S13.98 12 13 12H9c-.98 0-2-1.22-2-2.5 0-.83.42-1.64 1-2.09V6.25c-1.09.53-2 1.84-2 3.25C6 11.31 7.55 13 9 13h4c1.45 0 3-1.69 3-3.5S14.5 6 13 6z"></path></svg></a>Any mixe integer programming solver supported by MathProgBase.jl can be used.</h3>
<p>OOES.jl automatically installs GLPK by default. If the user desires to use any other MIP solver, it must be separately installed.</p>
<pre><code>a. OOES.jl has been successfully tested with:
    i.      GLPK - v4.61
    ii.     SCIP - v5.0.1 (Supports only SCIP.jl v0.6.1 and olders)
    iii.    Gurobi - v7.5
    iv.     CPLEX - v12.7.
    v.      Xpress - v8.5.
b. All parameters are already tuned.
c. Supports parallelization
</code></pre>
