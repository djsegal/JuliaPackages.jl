<p dir="auto">This package implements a flavour of the <em>Atomic Cluster Expansion</em>; i.e., parameterisation schemes for permutation and isometry invariant functions, primarily for the purpose of modelling invariant atomic properties. It provides constructions of symmetric polynomial bases, imposing permutation and isometry invariance.</p>
<p dir="auto">Most users should use the user-facing companian package <a href="https://github.com/ACEsuit/ACE1pack.jl"><code>ACE1pack.jl</code></a> which also provides documentation and tutorials on the usage of <code>ACE1.jl</code>.</p>
