<p>The <code>PCRE2</code> package implements a low-level API for accessing the PCRE libraries (8, 16, and 32-bit)
It is intended to replace <code>Base.PCRE</code>, which is not threadsafe, only supports UTF-8, and is using an old version of the PCRE library (10.30, current version is 10.31)</p>
