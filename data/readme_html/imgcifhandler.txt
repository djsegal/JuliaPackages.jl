<p dir="auto">This Julia package provides functions for working with ImgCIF files. imgCIF is a raw data standard developed
by the International Union of Crystallography, and recently enhanced with data
pointers so that the raw data can be preserved outside of the imgCIF file
itself.</p>

<p dir="auto">For some functions, the programs <code>tar</code>, <code>curl</code> and <code>awk</code> must be available in the environment,

<p dir="auto">To install this package, at the Julia package prompt (reached by typing <code>]</code> at the Julia prompt) type <code>add ImgCIFHandler</code>.
See also <a href="tools/README.md">additional instructions</a> in the <code>tools</code> directory for installing and updating the imgCIF checking tool.</p>

<p dir="auto">After installation, typing 'using ImgCIFHandler' will import the package. The following functions will be in scope:</p>
<ol dir="auto">
<li><code>imgload</code>. This function can be used to
access raw data stored using pointers within imgCIF files. The example file <a href="test/testfiles/b4_master.cif"><code>b4_master.cif</code></a> contains such pointers.
For example, to read the raw data array labelled <code>ext1</code> in <code>b4_master.cif</code> the following call is used:</li>
</ol>
<div class="highlight highlight-source-julia notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="x = imgload(&quot;b4_master.cif&quot;,&quot;ext1&quot;)"><pre>x <span class="pl-k">=</span> <span class="pl-c1">imgload</span>(<span class="pl-s"><span class="pl-pds">"</span>b4_master.cif<span class="pl-pds">"</span></span>,<span class="pl-s"><span class="pl-pds">"</span>ext1<span class="pl-pds">"</span></span>)</pre></div>
<p dir="auto">Note that this command will only work if the images referred to in <code>b4_master.cif</code> actually exist. To create those files locally, unpack the
<code>b4_mini.tar.bz2</code> contents into location <code>test/testfiles/test_cbf_unzipped</code>.</p>
<ol start="2" dir="auto">
<li><code>get_beam_centre(filename)</code>. This will calculate the beam centre in pixel and detector coordinates with all axes at zero.
<code>get_beam_centre(filename,scan_name,frameno)</code> will calculate the beam centre for the given frame number of the scan.</li>
</ol>

<p dir="auto">This release supports <code>HDF5</code>, <code>CBF</code>, and <code>SMV</code> (ADSC) formats, as well as Tar archives (optionally compressed using Bzip2, Gzip or Xzip) or
Zip archives containing files in those formats. In all cases the format stated by the imgCIF file is used.</p>
