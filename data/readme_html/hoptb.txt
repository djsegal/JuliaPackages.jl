<p dir="auto">HopTB.jl is a tight-binding package written in julia. The package has the ability of dealing with non-orthogonal tight-binding models and aims at both first principle calculations of real materials and model calculations.</p>
<p dir="auto">For real materials, HopTB.jl currently has interfaces with Wannier90, OpenMX and FHI-aims. Tight-binding systems are created with these density functional packages and HopTB.jl is a post-processing tool.</p>
<p dir="auto">For model calculation, HopTB.jl has a similar API as pythtb to construct tight-binding models.</p>
<p dir="auto">HopTB.jl provides infrastructure for analyzing response function and analyzing band structures. In addition, HopTB.jl contains out-of-box features including</p>
<ul dir="auto">
<li>Permittivity</li>
<li>Drude weight</li>
<li>Anomalous Hall effect</li>
<li>Spin Hall effect</li>
<li>Shift current conductivity</li>
<li>Second harmonic generation</li>
<li>Symmetrization of tight binding model</li>
<li>Intrinsic nonlinear Hall conductivity</li>
<li>Berry curvature dipole</li>
<li>Second order Drude weight</li>
<li>Fermi surface extraction</li>
</ul>
<p dir="auto">For more details, see <a href="https://hoptb.github.io/HopTB.jl/dev/" rel="nofollow">Documentation</a>.</p>
