<p>Beginnings of a package to display <strong>maps</strong> of all kinds, in various projections and with various configuragtions.</p>
<p>This package uses <a href="https://github.com/JuliaPlots/Makie.jl">Makie.jl</a> to plot maps and globes from source data.</p>
<p>This has only just started, but the idea would be to support many features such as <em>projections, scale, arbitrary resolutions, GPS data, points of interest, etc</em>.</p>
