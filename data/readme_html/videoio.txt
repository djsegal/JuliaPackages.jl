<p dir="auto"><a target="_blank" rel="noopener noreferrer" href="docs/src/assets/logo.png"><img align="right" width="90" src="docs/src/assets/logo.png" style="max-width: 100%;"></a></p>
<p dir="auto"><em>Reading and writing of video files in Julia.</em></p>
<p dir="auto">Functionality based on a dedicated build of ffmpeg via <a href="https://github.com/JuliaIO/FFMPEG.jl">FFMPEG.jl</a> and the <a href="https://github.com/JuliaPackaging/Yggdrasil/tree/master/F/FFMPEG">JuliaPackaging/Yggdrasil</a> cross-compiler.</p>

<p dir="auto">The package can be installed with the Julia package manager.
From the Julia REPL, type <code>]</code> to enter the Pkg REPL mode and run:</p>
<div class="snippet-clipboard-content notranslate position-relative overflow-auto" data-snippet-clipboard-copy-content="pkg&gt; add VideoIO"><pre class="notranslate"><code>pkg&gt; add VideoIO
</code></pre></div>
<p dir="auto">Or, equivalently, via the <code>Pkg</code> API:</p>
