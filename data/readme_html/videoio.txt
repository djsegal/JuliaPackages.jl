<p><em>Reading and writing of video files in Julia.</em></p>
<p>Functionality based on a dedicated build of ffmpeg via <a href="https://github.com/JuliaIO/FFMPEG.jl">FFMPEG.jl</a> and the <a href="https://github.com/JuliaPackaging/Yggdrasil/tree/master/F/FFMPEG">JuliaPackaging/Yggdrasil</a> cross-compiler.</p>

<p>The package can be installed with the Julia package manager.
From the Julia REPL, type <code>]</code> to enter the Pkg REPL mode and run:</p>
<div class="snippet-clipboard-content position-relative" data-snippet-clipboard-copy-content="pkg&gt; add VideoIO
"><pre><code>pkg&gt; add VideoIO
</code></pre></div>
<p>Or, equivalently, via the <code>Pkg</code> API:</p>
<div class="highlight highlight-source-julia position-relative" data-snippet-clipboard-copy-content="julia&gt; import Pkg; Pkg.add(&quot;VideoIO&quot;)
"><pre>julia<span class="pl-k">&gt;</span> <span class="pl-k">import</span> Pkg; Pkg<span class="pl-k">.</span><span class="pl-c1">add</span>(<span class="pl-s"><span class="pl-pds">"</span>VideoIO<span class="pl-pds">"</span></span>)</pre></div>

<p>Usage questions can be posted on the <a href="https://discourse.julialang.org/tags/videoio" rel="nofollow">Julia Discourse forum</a> under the <code>videoio</code> tag, and/or in the #video channel of the <a href="https://julialang.org/community/" rel="nofollow">Julia Slack</a>.</p>
