<p>This package provides the ability to generate Alizain Feerasta's Universally Unique Lexicographically
Sortable Identifiers (ULID) in Julia.
It's based on the original MIT-licensed <a href="https://github.com/alizain/ulid">JavaScript implementation</a>.
More information about ULIDs is available in the linked repository.</p>

<p>One function is exported: <code>ulid</code>.
Calling <code>ulid()</code> will generate a random 26-character ULID as a <code>String</code>.</p>
