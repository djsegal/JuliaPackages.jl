<p>This package contains two very useful diagnostics for Markov Chain Monte Carlo:</p>
<ol>
<li>
<p><code>potential_scale_reduction(chains...)</code>, which estimates the potential scale reduction factor, also known as <code>Rhat</code>, for multiple scalar chains,</p>
</li>
<li>
<p><code>effective_sample_size(chain)</code>, which calculates the effective sample size for scalar chains.</p>
</li>
</ol>
<p>These are intended as <em>building blocks</em>, to be used by other libraries, and were organized into a separate library for testing and DRY.</p>
