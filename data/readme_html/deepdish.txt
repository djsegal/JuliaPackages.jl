<p dir="auto">A simple package to load HDF5 files saved by the <a href="https://github.com/uchicago-cs/deepdish">DeepDish</a> python library in Julia. Currently, only lists of simple types, dictionaries, numpy arrays and <a href="https://pandas.pydata.org/" rel="nofollow">pandas</a> DataFrames are supported, as well as recursive structures of lists and dictionaries of the same.</p>
<p dir="auto">The only exported function is</p>
<p dir="auto"><code>load_deepdish(f)</code></p>
<p dir="auto">where f is the path to the h5 file.</p>
