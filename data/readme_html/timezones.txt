<p><a href="http://www.iana.org/time-zones" rel="nofollow">IANA time zone database</a> access for the <a href="http://julialang.org/" rel="nofollow">Julia</a> programming language. TimeZones.jl extends the Date/DateTime support for Julia to include a new time zone aware TimeType: ZonedDateTime.</p>
<h2><a id="user-content-features" class="anchor" aria-hidden="true" href="#features"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path fill-rule="evenodd" d="M4 9h1v1H4c-1.5 0-3-1.69-3-3.5S2.55 3 4 3h4c1.45 0 3 1.69 3 3.5 0 1.41-.91 2.72-2 3.25V8.59c.58-.45 1-1.27 1-2.09C10 5.22 8.98 4 8 4H4c-.98 0-2 1.22-2 2.5S3 9 4 9zm9-3h-1v1h1c1 0 2 1.22 2 2.5S13.98 12 13 12H9c-.98 0-2-1.22-2-2.5 0-.83.42-1.64 1-2.09V6.25c-1.09.53-2 1.84-2 3.25C6 11.31 7.55 13 9 13h4c1.45 0 3-1.69 3-3.5S14.5 6 13 6z"></path></svg></a>Features</h2>
<ul>
<li>A new time zone aware TimeType: ZonedDateTime</li>
<li>Support for all time zones in the IANA time zone database (also known as the tz/zoneinfo/Olson database)</li>
<li>ZonedDateTime-Period arithmetic <a href="https://docs.julialang.org/en/stable/manual/dates/#TimeType-Period-Arithmetic-1" rel="nofollow">similar to that of DateTime</a></li>
<li>Local system time zone information as a TimeZone</li>
<li>Current system time in any TimeZone</li>
<li>Support for reading the <a href="http://man7.org/linux/man-pages/man5/tzfile.5.html" rel="nofollow">tzfile</a> format</li>
<li>String parsing of ZonedDateTime using <a href="https://docs.julialang.org/en/stable/stdlib/dates/#Base.Dates.DateFormat" rel="nofollow">DateFormat</a></li>
</ul>
