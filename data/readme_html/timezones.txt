<p><a href="https://www.iana.org/time-zones" rel="nofollow">IANA time zone database</a> access for the <a href="https://julialang.org" rel="nofollow">Julia</a> programming language. TimeZones.jl extends the Date/DateTime support for Julia to include a new time zone aware TimeType: ZonedDateTime.</p>
<h2><a id="user-content-features" class="anchor" aria-hidden="true" href="#features"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path fill-rule="evenodd" d="M7.775 3.275a.75.75 0 001.06 1.06l1.25-1.25a2 2 0 112.83 2.83l-2.5 2.5a2 2 0 01-2.83 0 .75.75 0 00-1.06 1.06 3.5 3.5 0 004.95 0l2.5-2.5a3.5 3.5 0 00-4.95-4.95l-1.25 1.25zm-4.69 9.64a2 2 0 010-2.83l2.5-2.5a2 2 0 012.83 0 .75.75 0 001.06-1.06 3.5 3.5 0 00-4.95 0l-2.5 2.5a3.5 3.5 0 004.95 4.95l1.25-1.25a.75.75 0 00-1.06-1.06l-1.25 1.25a2 2 0 01-2.83 0z"></path></svg></a>Features</h2>
<ul>
<li>A new time zone aware TimeType: ZonedDateTime</li>
<li>Support for all time zones in the IANA time zone database (also known as the tz/zoneinfo/Olson database)</li>
<li>ZonedDateTime-Period arithmetic <a href="https://docs.julialang.org/en/stable/manual/dates/#TimeType-Period-Arithmetic-1" rel="nofollow">similar to that of DateTime</a></li>
<li>Local system time zone information as a TimeZone</li>
<li>Current system time in any TimeZone</li>
<li>Support for reading the <a href="https://man7.org/linux/man-pages/man5/tzfile.5.html" rel="nofollow">tzfile</a> format</li>
<li>String parsing of ZonedDateTime using <a href="https://docs.julialang.org/en/stable/stdlib/dates/#Base.Dates.DateFormat" rel="nofollow">DateFormat</a></li>
</ul>
