<p dir="auto">Information and uncertainty can be regarded as two sides of the same coin:
the more uncertainty there is, the more information we gain by removing that
uncertainty. In the context of information and probability theory, <em><strong>Entropy</strong></em>
quantifies that uncertainty.</p>
<p dir="auto">The concept of entropy has its origins in
<a href="http://www.scholarpedia.org/article/Entropy" title="Scholarpedia" rel="nofollow">classical physics</a>
under the second law of thermodynamics, a law
<a href="https://www.penguin.co.uk/books/301539/the-order-of-time/9780141984964.html" title="Rovelli" rel="nofollow">considered to underpin our fundamental understanding</a>
of <a href="https://en.wikipedia.org/wiki/Time_in_physics" title="Wiki Time" rel="nofollow">time in physics</a>.
Attempting to analyse the analog world around
us requires that we measure time in discrete steps, but doing so compromises
our ability to measure entropy accurately. Various measures have been derived
to estimate entropy (uncertainty) from discrete time series, each seeking to
best capture the uncertainty of the system under examination. This has resulted
in many entropy statistics from approximate entropy and sample entropy, to
multiscale sample entropy and refined-composite multiscale cross-sample entropy.</p>
<p dir="auto">As the number of statisitcal entropy measures grows, it becomes more difficult
to identify, contrast and compare the performance of each measure. To overcome
this, we have developed EntropyHub - an open-source toolkit designed to
integrate the many established entropy methods into one package. The goal of
EntropyHub is to provide a comprehensive set of functions with a simple and
consistent syntax that allows the user to augment parameters at the command
line, enabling a range from basic to advanced entropy methods to be implemented
with ease.</p>
<p dir="auto"><em><strong>It is important to clarify that the entropy functions herein described
estimate entropy in the context of probability theory and information theory as
defined by Shannon, and not thermodynamic or other entropies from classical physics.</strong></em></p>

<p dir="auto">There are two ways to install EntropyHub for Julia.</p>
<h4 dir="auto"><a id="user-content-method-1" class="anchor" aria-hidden="true" href="#method-1"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path d="m7.775 3.275 1.25-1.25a3.5 3.5 0 1 1 4.95 4.95l-2.5 2.5a3.5 3.5 0 0 1-4.95 0 .751.751 0 0 1 .018-1.042.751.751 0 0 1 1.042-.018 1.998 1.998 0 0 0 2.83 0l2.5-2.5a2.002 2.002 0 0 0-2.83-2.83l-1.25 1.25a.751.751 0 0 1-1.042-.018.751.751 0 0 1-.018-1.042Zm-4.69 9.64a1.998 1.998 0 0 0 2.83 0l1.25-1.25a.751.751 0 0 1 1.042.018.751.751 0 0 1 .018 1.042l-1.25 1.25a3.5 3.5 0 1 1-4.95-4.95l2.5-2.5a3.5 3.5 0 0 1 4.95 0 .751.751 0 0 1-.018 1.042.751.751 0 0 1-1.042.018 1.998 1.998 0 0 0-2.83 0l-2.5 2.5a1.998 1.998 0 0 0 0 2.83Z"></path></svg></a>Method 1:</h4>
<ol dir="auto">
<li>
<p dir="auto">In Julia, open the package REPL by typing <code>]</code>. The command line should appear as:</p>
<p dir="auto"><code>@vX.Y.  pkg&gt;        </code></p>
<p dir="auto">Where X and Y refer to your version of Julia.</p>
</li>
<li>
<p dir="auto">Type:</p>
<p dir="auto"><code>add EntropyHub</code></p>
<p dir="auto">(Note: this is <strong>case sensitive</strong>)</p>
</li>
</ol>
<p dir="auto">Alternatively, one can use the Pkg module to perform the same procedure:</p>
<p dir="auto"><code>using Pkg</code></p>
<p dir="auto"><code>Pkg.add("EntropyHub")</code></p>

<p dir="auto">There are several package dependencies which will be installed alongside EntropyHub (if not already installed):</p>
<p dir="auto">DSP, FFTW, HTTP, Random, Plots, StatsBase, StatsFuns, GroupSlices,
Statistics, DelimitedFiles, Combinatorics, LinearAlgebra, Dierckx, Clustering</p>
<p dir="auto">EntropyHub was designed using Julia 1.5 and is intended for use with Julia versions &gt;= 1.2.</p>
