<p dir="auto">This Julia package is used to represent infinities, including:</p>
<ol dir="auto">
<li><code>InfiniteCardinal{k}</code>, for the cardinality of an infinite set, e.g., <code>ℵ₀</code> for the cardinality of the integers and <code>ℵ₁</code> for the cardinality of the reals.</li>
<li><code>∞</code> to represent the positive real infinity.</li>
<li><code>RealInfinity</code> to represent <code>±∞</code>.</li>
<li><code>ComplexInfinity</code> to represent an oriented infinity  <code>exp(im*θ)∞</code></li>
</ol>
<p dir="auto">Note that we subtype based on interfaces, rather than strict mathematical definitions. For example,  <code>ℵ₀ isa Integer</code> as <code>Integer</code> is often used to represent the size of a set or vector. Similarly, <code>∞ isa Real</code>.</p>

<p dir="auto">This package is meant to eventually replace <a href="https://github.com/cjdoris/Infinity.jl">Infinity.jl</a> and the definitions of <code>∞</code> in <a href="https://github.com/JuliaArrays/InfiniteArrays.jl">InfiniteArrays.jl</a>. We do not yet support Infinity.jl's notions of <code>InfExtendedReal</code> but we hope to add this soon.</p>
