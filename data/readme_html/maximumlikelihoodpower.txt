<div class="highlight highlight-source-julia"><pre>    <span class="pl-c1">KSstatistic</span>(data<span class="pl-k">::</span><span class="pl-c1">AbstractVector</span>, alpha) <span class="pl-k">--&gt;</span> Float64</pre></div>
<p>Return the Kolmogorov-Smirnov statistic
comparing <code>data</code> to a power law with power <code>alpha</code>. The elements of <code>data</code> are
assumed to be unique. Minimizing the KS statistic over alpha is another way
to estimate the parameter of the sample distribution. See <code>testKS</code> in the
<code>test</code> directory.</p>

<p>Compute the Kolmogorov Smirnov statistic for several values of α in
the iterator <code>powers</code>. Return the value of α
that minimizes the KS statistic and the two neighboring values.</p>

<div class="highlight highlight-source-julia"><pre>    <span class="pl-c1">scanmle</span>(data<span class="pl-k">::</span><span class="pl-c1">AbstractVector</span>; ntrials<span class="pl-k">=</span><span class="pl-c1">100</span>, stderrcutoff<span class="pl-k">=</span><span class="pl-c1">0.1</span>, useKS<span class="pl-k">=</span><span class="pl-c1">false</span>)</pre></div>
<p>Perform <code>mle</code> approximately <code>ntrials</code> times on <code>data</code>, increasing <code>xmin</code>. Stop trials
if the <code>stderr</code> of the estimate <code>alpha</code> is greater than <code>stderrcutoff</code>. Return an object
containing statistics about the scan.</p>
