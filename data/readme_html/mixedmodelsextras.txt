<p dir="auto"><code>MixedModelsExtras.jl</code> is a Julia package providing extra capabilities for models fit with <a href="https://juliastats.org/MixedModels.jl/stable/" rel="nofollow"><code>MixedModels.jl</code></a>.</p>
<p dir="auto">Note that plotting functionality is planned for inclusion in separate packages.</p>
<p dir="auto"><a href="https://github.com/palday/MixedModelsPermutations.jl"><code>MixedModelsPermutations.jl</code></a> and <a href="https://github.com/RePsychLing/MixedModelsSim.jl/"><code>MixedModelsSim.jl</code></a> provide additional functionality for permutation tests and data/power simulation, respectively.</p>
