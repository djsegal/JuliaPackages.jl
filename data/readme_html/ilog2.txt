<p>This package provides a fast implementation of the integer-valued, base-2 logarithm.
<code>ilog2</code> supports other types of numbers, as well. It also provides <code>checkispow2</code>.</p>
<p><code>ilog2(n::Real)</code> compute the largest <code>m</code> such that <code>2^m &lt;= n</code>.</p>
<p><code>checkispow2(n::Number)</code> returns <code>ilog2(n)</code> if <code>n</code> is a power of two.
Otherwise throws a <code>DomainError</code>.</p>
