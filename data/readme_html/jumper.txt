<p dir="auto"><strong><a href="https://github.com/IainNZ/JuMPeR.jl">JuMPeR</a></strong> is a modeling language for <a href="http://en.wikipedia.org/wiki/Robust_optimization" rel="nofollow">robust optimization</a>.
It is embedded in the <a href="http://julialang.org/" rel="nofollow">Julia programming language</a>, and is an extension to the <a href="https://github.com/JuliaOpt/JuMP.jl">JuMP</a> modeling language.</p>
<ul dir="auto">
<li>Install with Julia package manager: <code>julia&gt; Pkg.add("JuMPeR")</code>.</li>
<li><a href="doc/index.md">Read the (limited) documentation to learn more</a>.
<ul dir="auto">
<li>Help with documentation is appreciated!</li>
<li>The JuMPeR chapter from my thesis is <a href="doc/JuMPeRChapter.pdf">also available</a>.</li>
<li>The source code is well-commented, if you'd like to understand more.</li>
</ul>
</li>

</ul>
