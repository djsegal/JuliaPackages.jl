<p dir="auto"><code>MKLSparse.jl</code> is a Julia package to seamlessly use the sparse functionality in MKL to speed up operations on sparse arrays in Julia.
In order to use <code>MKLSparse.jl</code> you do not need to install Intel's MKL library nor build Julia with MKL. <code>MKLSparse.jl</code> will automatically download and use the MKL library for you when installed.</p>

<p dir="auto">Solving linear systems with triangular sparse matrices is supported.
These matrices should be wrapped in their corresponding type, for example <code>LowerTriangular</code> for lower triangular matrices.</p>
<p dir="auto">For solving general sparse linear systems using MKL we refer to <a href="https://github.com/JuliaSparse/Pardiso.jl">Pardiso.jl</a>.</p>
