<p><code>MKLSparse.jl</code> is a Julia package to seamlessly use the sparse functionality in MKL to speed up operations on sparse arrays in Julia.
In order to use <code>MKLSparse.jl</code> you need to have MKL installed and the environment variables <code>MKLROOT</code> correctly set, see the <a href="https://software.intel.com/en-us/articles/intel-mkl-103-getting-started" rel="nofollow">MKL getting started guide</a> for a guide. You do not need to have built Julia with MKL as the used BLAS library to use the package.</p>

<p>Loading <code>MKLSparse.jl</code> will make sparse-dense matrix operations be computed using MKL.</p>
<h3><a id="user-content-solving-linear-systems" class="anchor" aria-hidden="true" href="#solving-linear-systems"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path fill-rule="evenodd" d="M4 9h1v1H4c-1.5 0-3-1.69-3-3.5S2.55 3 4 3h4c1.45 0 3 1.69 3 3.5 0 1.41-.91 2.72-2 3.25V8.59c.58-.45 1-1.27 1-2.09C10 5.22 8.98 4 8 4H4c-.98 0-2 1.22-2 2.5S3 9 4 9zm9-3h-1v1h1c1 0 2 1.22 2 2.5S13.98 12 13 12H9c-.98 0-2-1.22-2-2.5 0-.83.42-1.64 1-2.09V6.25c-1.09.53-2 1.84-2 3.25C6 11.31 7.55 13 9 13h4c1.45 0 3-1.69 3-3.5S14.5 6 13 6z"></path></svg></a>Solving linear systems</h3>
<p>Solving linear systems with triangular sparse matrices is supported.
These matrices should be wrapped in their corresponding type, for example <code>LowerTriangular</code> for lower triangular matrices.</p>
<p>For solving general sparse linear systems using MKL we refer to <a href="https://github.com/JuliaSparse/Pardiso.jl">Pardiso.jl</a>.</p>
