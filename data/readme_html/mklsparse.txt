<p><code>MKLSparse.jl</code> is a Julia package to seamlessly use the sparse functionality in MKL to speed up operations on sparse arrays in Julia.
In order to use <code>MKLSparse.jl</code> you do not need to install Intel's MKL library nor build Julia with MKL. <code>MKLSparse.jl</code> will automatically download and use the MKL library for you when installed.</p>

<p>Loading <code>MKLSparse.jl</code> will make sparse-dense matrix operations be computed using MKL.</p>
<h3><a id="user-content-solving-linear-systems" class="anchor" aria-hidden="true" href="#solving-linear-systems"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path fill-rule="evenodd" d="M7.775 3.275a.75.75 0 001.06 1.06l1.25-1.25a2 2 0 112.83 2.83l-2.5 2.5a2 2 0 01-2.83 0 .75.75 0 00-1.06 1.06 3.5 3.5 0 004.95 0l2.5-2.5a3.5 3.5 0 00-4.95-4.95l-1.25 1.25zm-4.69 9.64a2 2 0 010-2.83l2.5-2.5a2 2 0 012.83 0 .75.75 0 001.06-1.06 3.5 3.5 0 00-4.95 0l-2.5 2.5a3.5 3.5 0 004.95 4.95l1.25-1.25a.75.75 0 00-1.06-1.06l-1.25 1.25a2 2 0 01-2.83 0z"></path></svg></a>Solving linear systems</h3>
<p>Solving linear systems with triangular sparse matrices is supported.
These matrices should be wrapped in their corresponding type, for example <code>LowerTriangular</code> for lower triangular matrices.</p>
<p>For solving general sparse linear systems using MKL we refer to <a href="https://github.com/JuliaSparse/Pardiso.jl">Pardiso.jl</a>.</p>
