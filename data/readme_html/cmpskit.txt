<p dir="auto">This package is still work in progress, but is currently functional for running ground state simulations of one-dimensional non-relativistic quantum field theories using continuous matrix product states, both for infinite systems with homogeneous or periodic interactions, or for finite systems with inhomogeneous interactions.</p>
