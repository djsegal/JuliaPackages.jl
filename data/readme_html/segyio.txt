<p>SegyIO is a Julia package for reading and writing SEGY Rev 1 files. In addition to providing tools for reading/writing entire files, SegyIO provides a parallel scanner that reduces any number of files into a single object with direct out-of-core access to the underlying data.</p>
<p><a href="https://github.com/slimgroup/SegyIO.jl/actions?query=workflow%3ACI-tests"><img src="https://github.com/slimgroup/SegyIO.jl/workflows/CI-tests/badge.svg" alt="Build Status" style="max-width:100%;"></a></p>
<p>A video demonstrating the package's capabilities <a href="https://www.youtube.com/watch?v=tx530QOPeZo&amp;feature=youtu.be" rel="nofollow">has been made available here.</a></p>

<p>First switch to package manager prompt (using ']') and add SLIM registry:</p>
<div class="snippet-clipboard-content position-relative" data-snippet-clipboard-copy-content="	registry add https://github.com/slimgroup/SLIMregistryJL.git
"><pre><code>	registry add https://github.com/slimgroup/SLIMregistryJL.git
</code></pre></div>
<p>Then still from package manager prompt add SegyIO:</p>
<div class="snippet-clipboard-content position-relative" data-snippet-clipboard-copy-content="	add SegyIO
"><pre><code>	add SegyIO
</code></pre></div>
