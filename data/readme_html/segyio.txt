<p dir="auto">SegyIO is a Julia package for reading and writing SEGY Rev 1 files. In addition to providing tools for reading/writing entire files, SegyIO provides a parallel scanner that reduces any number of files into a single object with direct out-of-core access to the underlying data.</p>

<p dir="auto">A video demonstrating the package's capabilities <a href="https://www.youtube.com/watch?v=tx530QOPeZo&amp;feature=youtu.be" rel="nofollow">has been made available here.</a></p>
