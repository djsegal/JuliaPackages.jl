<p>The <code>dash</code> package contains some tools to build components and drive the bundles build process.
To avoid the circular dependency situation, we don't add <code>dash</code> as a required install in the <code>dash-core-components</code> setup.
But, in order to do development locally, you need to install <code>dash</code> before everything.</p>
<ol>
<li>Install the dependencies with:</li>
</ol>
