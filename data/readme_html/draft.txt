<p dir="auto">Package for faster prototyping without widespread implications for your Julia default installation.</p>
<ul dir="auto">
<li>Activates a temporary project when the module is loaded.</li>
<li>Enables offline mode when the module is loaded for a faster package adding. Locally installed packages will be reused.</li>
<li>Provides a macro to load and install a package: <code>@reuse PackageA</code> or <code>@reuse PackageA PackageB</code>.</li>
<li>Output of <code>@reuse</code> can be suppressed by calling <code>Draft.silent()</code>. Undo with <code>Draft.silent(false)</code>.</li>
<li>Provides a function to switch back to online mode: <code>Draft.online()</code>.</li>
<li>Provides a function to save the temporary Project.toml file to a target location: <code>save_environment(target_dir)</code>.</li>
</ul>
