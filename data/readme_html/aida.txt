<p dir="auto">This package implements simple readers for legacy AIDA files.
Pull requests for missing implementations of AIDA classes are welcome.</p>
<p dir="auto">There is no plan to implement writers. For writing out analysis data, we recommend the HDF5 format and the HDF5.jl or JDL.jl modules.</p>
