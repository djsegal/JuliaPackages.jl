<p dir="auto">This repository contains the LSPE/Strip simulation pipeline. LSPE/Strip is
a CMB microwave telescope to be installed at the Teide Observatory in
Tenerife. Stripeline is an instrument simulator that models the scanning
strategy of the telecope and the response of the detectors, up to the creation
of frequency maps.</p>
<p dir="auto">Stripeline is written in <a href="https://julialang.org" rel="nofollow">Julia</a>. The current version
requires Julia 1.3 or higher.</p>
