<p dir="auto">Encrypt and decrypt image files using Pseudo-Random Number Generators and various encryption techniques! <code>ChaoticEncryption.jl</code> comes loaded with Pseudo-Random Number Generators and various encryption techniques, which can be used to encrypt and decrypt any image file. The package is under active development, but the existing API is stable and might not change significantly.</p>
