<p dir="auto">DataEnvelopmentAnalysis.jl is a Julia package that provides functions for efficiency and productivity measurement using Data Envelopment Analysis (DEA). Particularly, it implements a variety of technical efficiency models, economic efficiency models and productivity change models.</p>
<p dir="auto">The package is being developed for Julia <code>1.6</code> and above on Linux, macOS, and Windows.</p>
<p dir="auto">The packes uses internally the <a href="https://github.com/JuliaOpt/JuMP.jl">JuMP</a> modelling language for mathematicall optimization with solvers <a href="http://www.gnu.org/software/glpk/" rel="nofollow">GLPK</a> and <a href="https://coin-or.github.io/Ipopt/" rel="nofollow">Ipopt</a>.</p>

<p dir="auto"><strong>Technical efficiency DEA models</strong></p>
<ul dir="auto">
<li>Radial input and output oriented model.</li>
<li>Directional distance function model.</li>
<li>Additive models: weighted additive model, measure of inefficiency proportions (MIP), normalized weighted additive model, range adjusted measure (RAM), bounded adjusted measure (BAM).</li>
<li>Generalized distance function model.</li>
<li>Russell graph and oriented model.</li>
<li>Enhanced Russell Graph Slack Based Measure.</li>
<li>Modified directional distance function.</li>
<li>Hölder distance function.</li>
<li>Reverse directional distance function.</li>
</ul>
<p dir="auto"><strong>Economic efficiency DEA models</strong></p>
<ul dir="auto">
<li>Cost model.</li>
<li>Revenue model.</li>
<li>Profit model.</li>
<li>Profitability model.</li>
</ul>
<p dir="auto"><strong>Productivity change models</strong></p>
<ul dir="auto">
<li>Mamlmquist index.</li>
</ul>
<p dir="auto"><strong>Statistical Analysis</strong></p>
<ul dir="auto">
<li>Bootstrap Radial DEA Model.</li>
<li>Returns to Scale (RTS) Test.</li>
</ul>
