<p dir="auto"><code>TopOpt</code> is a topology optimization package written in <a href="https://github.com/JuliaLang/julia">Julia</a>. To learn more and see some examples, visit the <a href="https://juliatopopt.github.io/TopOpt.jl/dev" rel="nofollow">documentation</a>. Numerous examples can also be found in the <code>test/examples</code> and <code>test/wcsmo14</code> directories.</p>

<ul dir="auto">
<li>2D and 3D truss domains</li>
<li>2D and 3D continuum domains</li>
<li>Unstructured ground mesh</li>
<li>Linear and quadratic triangle, quadrilateral, tetrahedron and hexahedron elements in ground mesh</li>

<p dir="auto">The following high level topology optimization algorithms and penalty types are available.</p>
<ul dir="auto">
<li>Solid isotropic material with penalization (SIMP)</li>
<li>Rational approximation of material properties (RAMP)</li>
<li>Continuation SIMP/RAMP</li>
<li>Bi-directional evolutionary structural optimization (BESO) with soft-kill</li>
<li>Topology optimization of binary structures (TOBS)</li>
</ul>
<h3 dir="auto"><a id="user-content-differentiable-functions" class="anchor" aria-hidden="true" href="#differentiable-functions"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path d="m7.775 3.275 1.25-1.25a3.5 3.5 0 1 1 4.95 4.95l-2.5 2.5a3.5 3.5 0 0 1-4.95 0 .751.751 0 0 1 .018-1.042.751.751 0 0 1 1.042-.018 1.998 1.998 0 0 0 2.83 0l2.5-2.5a2.002 2.002 0 0 0-2.83-2.83l-1.25 1.25a.751.751 0 0 1-1.042-.018.751.751 0 0 1-.018-1.042Zm-4.69 9.64a1.998 1.998 0 0 0 2.83 0l1.25-1.25a.751.751 0 0 1 1.042.018.751.751 0 0 1 .018 1.042l-1.25 1.25a3.5 3.5 0 1 1-4.95-4.95l2.5-2.5a3.5 3.5 0 0 1 4.95 0 .751.751 0 0 1-.018 1.042.751.751 0 0 1-1.042.018 1.998 1.998 0 0 0-2.83 0l-2.5 2.5a1.998 1.998 0 0 0 0 2.83Z"></path></svg></a>Differentiable functions</h3>
<p dir="auto">There are numerous functions in <code>TopOpt.jl</code> that are defined in a differentiable way and you can use them in the objectives or constraints in topology optimization formulations. In <code>TopOpt.jl</code>, you can build arbitrarily complex objective and constraint functions using these differentiable functions as building blocks or lego pieces chaining them in any arbitrary way. The gradient and jacobian of the aggregate Julia function defined is then obtained using <a href="https://www.youtube.com/watch?v=UqymrMG-Qi4" rel="nofollow">automatic differentiation</a>. For a detailed account of the functions available, see the relevant section in the <a href="https://juliatopopt.github.io/TopOpt.jl/dev/functions/" rel="nofollow">documentation</a>.</p>

<h3 dir="auto"><a id="user-content-optimization-algorithms" class="anchor" aria-hidden="true" href="#optimization-algorithms"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path d="m7.775 3.275 1.25-1.25a3.5 3.5 0 1 1 4.95 4.95l-2.5 2.5a3.5 3.5 0 0 1-4.95 0 .751.751 0 0 1 .018-1.042.751.751 0 0 1 1.042-.018 1.998 1.998 0 0 0 2.83 0l2.5-2.5a2.002 2.002 0 0 0-2.83-2.83l-1.25 1.25a.751.751 0 0 1-1.042-.018.751.751 0 0 1-.018-1.042Zm-4.69 9.64a1.998 1.998 0 0 0 2.83 0l1.25-1.25a.751.751 0 0 1 1.042.018.751.751 0 0 1 .018 1.042l-1.25 1.25a3.5 3.5 0 1 1-4.95-4.95l2.5-2.5a3.5 3.5 0 0 1 4.95 0 .751.751 0 0 1-.018 1.042.751.751 0 0 1-1.042.018 1.998 1.998 0 0 0-2.83 0l-2.5 2.5a1.998 1.998 0 0 0 0 2.83Z"></path></svg></a>Optimization algorithms</h3>
<p dir="auto">We use <a href="https://github.com/JuliaNonconvex/Nonconvex.jl">Nonconvex.jl</a> for the optimization problem definition and solving. The following algorithms are all available using <code>Nonconvex.jl</code>.</p>
<ul dir="auto">
<li>Method of moving asymptotes</li>
<li>All the algorithms in NLopt</li>
<li>Ipopt</li>
<li>First order augmented Lagrangian algorithm</li>
<li>Nonlinear semidefinite programming for buckling constrained optimization</li>
<li>Basic surrogate assisted optimization and Bayesian optimization</li>
<li>Integer nonlinear programming (design variables guaranteed to be integer)</li>
<li>Sequential integer linear programming in the topology optimization for binary structures (TOBS) algorithm</li>
</ul>
