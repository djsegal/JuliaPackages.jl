<p dir="auto">This is the user-facing entry-point for the skeletal <a href="https://github.com/tecosaur/DataToolkitBase.jl">DataToolkitBase</a> package and
  the set of useful transformers and plugins provided by <a href="https://github.com/tecosaur/DataToolkitCommon.jl">DataToolkitCommon</a>, along
  with a few trivial niceties.</p>
<p dir="auto">This aims to provide a foundation for robust data management.</p>
<p dir="auto">For now, this set of packages around the alpha stage of development. No major
  changes to the core functionality or structure are anticipated, but an expansion
  in the data-CLI functionality and set of transformers and plugins provided by
  <i>DataToolkitCommon</i> is wanted prior to the formal initial release.</p>
