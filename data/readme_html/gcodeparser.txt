<p dir="auto"><code>GcodeParser</code> is a simple package for parsing G-code files. It has been tested on 3d printing G-codes produced by some of the common slicers (mostly <code>Slic3r</code> and <code>Cura</code>).</p>
<h2 dir="auto"><a id="user-content-installation" class="anchor" aria-hidden="true" href="#installation"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path d="m7.775 3.275 1.25-1.25a3.5 3.5 0 1 1 4.95 4.95l-2.5 2.5a3.5 3.5 0 0 1-4.95 0 .751.751 0 0 1 .018-1.042.751.751 0 0 1 1.042-.018 1.998 1.998 0 0 0 2.83 0l2.5-2.5a2.002 2.002 0 0 0-2.83-2.83l-1.25 1.25a.751.751 0 0 1-1.042-.018.751.751 0 0 1-.018-1.042Zm-4.69 9.64a1.998 1.998 0 0 0 2.83 0l1.25-1.25a.751.751 0 0 1 1.042.018.751.751 0 0 1 .018 1.042l-1.25 1.25a3.5 3.5 0 1 1-4.95-4.95l2.5-2.5a3.5 3.5 0 0 1 4.95 0 .751.751 0 0 1-.018 1.042.751.751 0 0 1-1.042.018 1.998 1.998 0 0 0-2.83 0l-2.5 2.5a1.998 1.998 0 0 0 0 2.83Z"></path></svg></a>Installation</h2>

<p dir="auto">To parse a single line of g-code use the <code>parseLine</code> function.</p>
<div class="highlight highlight-source-julia notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="parseLine(&quot;G1 (move) X6.66 ; some command description&quot;)"><pre><span class="pl-c1">parseLine</span>(<span class="pl-s"><span class="pl-pds">"</span>G1 (move) X6.66 ; some command description<span class="pl-pds">"</span></span>)</pre></div>
<p dir="auto">This will return either an array of pairs (<code>["G" =&gt; "1", "X" =&gt; "6.66"]</code>) or an array of strings <code>["G1", "X6.66"]</code> if you call the method as <code>parseLine("G1...", false)</code>. Comments are striped out automatically.</p>

<p dir="auto">You can use the <code>parseFile</code> function to do more advanced calculations with your g-code.</p>
<div class="highlight highlight-source-julia notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="

# it will be passed as a second parameter to your callbacks
# here simple dictionary is used to store information during the print

myPrinter[&quot;numberOfMoves&quot;] = 0;
