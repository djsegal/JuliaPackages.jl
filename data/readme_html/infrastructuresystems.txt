<p dir="auto">The <code>InfrastructureSystems.jl</code> package provides utilities to support data models for infrastructure modeling in <a href="https://github.com/NREL-Sienna">NREL-Sienna</a>. The <code>InfrastructureSystems.jl</code> package is used to support functionalities in <a href="https://github.com/NREL-Sienna/PowerSystems.jl">PowerSystems.jl</a>, <a href="https://github.com/NREL-Sienna/PowerSimulations.jl">PowerSimulations.jl</a>, <a href="https://github.com/NREL-Sienna/PowerSimulationsDynamics.jl">PowerSimulationsDynamics.jl</a> and other modeling packages in the Sienna ecosystem.</p>
<p dir="auto">This package is only compatible with Julia 1.6 or higher.</p>
