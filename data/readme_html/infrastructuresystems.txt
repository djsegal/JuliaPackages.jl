<p>The <code>InfrastructureSystems.jl</code> package provides utilities to support data models for infrastructure modeling in <a href="https://github.com/NREL-SIIP">NREL-SIIP</a>. The <code>InfrastructureSystems.jl</code> package is used to support functionalities in <a href="https://github.com/NREL-SIIP/PowerSystems.jl">PowerSystems.jl</a>, <a href="https://github.com/NREL-SIIP/PowerSimulations.jl">PowerSimulations.jl</a> and <a href="https://github.com/NREL-SIIP/PowerSimulationsDynamics.jl">PowerSimulationsDynamics.jl</a>. This package is only compatible with Julia 1.6 or higher.</p>
