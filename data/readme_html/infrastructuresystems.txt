<p>The <code>InfrastructureSystems.jl</code> package provides utilities to support data models for infrastructure modeling data using Julia. The <code>InfrastructureSystems.jl</code> package is used to support functionalities in <a href="https://github.com/NREL-SIIP/PowerSystems.jl">PowerSystems.jl</a>. This package is only compatible with Julia 1.2 or higher.</p>
