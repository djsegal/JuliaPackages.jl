<p dir="auto">This application allows users to browse PowerSystems components and time series data in a web UI
via Plotly Dash. Here's how to start it:</p>
<div class="highlight highlight-source-julia notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="julia&gt; using PowerApps
julia&gt; run_system_explorer()

[ Info: Listening on: 0.0.0.0:8050"><pre>julia<span class="pl-k">&gt;</span> <span class="pl-k">using</span> PowerApps
julia<span class="pl-k">&gt;</span> <span class="pl-c1">run_system_explorer</span>()
[ Info<span class="pl-k">:</span> Navigate browser to<span class="pl-k">:</span> http<span class="pl-k">:</span><span class="pl-k">//</span><span class="pl-c1">0.0</span>.<span class="pl-c1">0.0</span><span class="pl-k">:</span><span class="pl-c1">8050</span>
[ Info<span class="pl-k">:</span> Listening on<span class="pl-k">:</span> <span class="pl-c1">0.0</span>.<span class="pl-c1">0.0</span><span class="pl-k">:</span><span class="pl-c1">8050</span></pre></div>
<p dir="auto">Open your browser to the IP address and port listed. In this case: <code>http://0.0.0.0:8050</code>.
The System Explorer app will appear with three tabs:</p>
<ul dir="auto">
<li>System: enter a path to a raw data file or serialized JSON and load the system.
Component data can be explored by type, and can be sorted and filtered.</li>
<li>Time Series: time series data for components selected on the "System" tab can be viewed
and visualized</li>
<li>Maps: a shapefile can be loaded (optional), and nodes (<code>Bus</code>) and <code>Branch</code>s can be plotted.
Several configuration options provide opportunities to visualize the geo-spatial data.
<em>note: geospatial layouts are significantly more useful when "latitude" and "longitude" is defined
in the <code>Bus.ext</code> fields. Without bus coordinates, an automatic layout will be applied</em></li>
</ul>
