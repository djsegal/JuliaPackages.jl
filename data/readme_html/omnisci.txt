<p>This package is an Apache Thrift-based client for OmniSci, with similar functionality to our Python package <a href="https://pymapd.readthedocs.io/en/latest/" rel="nofollow">pymapd</a>. Because of the rapid pace of <a href="https://github.com/omnisci/omniscidb">OmniSciDB</a> development, this package will attempt to always run against the most recent version of OmniSci. In most cases, OmniSci.jl should be backwards compatible to the last few OmniSciDB releases, but if you run into an issue, the first step would be to run the OmniSci.jl test suite against the <a href="https://hub.docker.com/r/omnisci/core-os-cpu" rel="nofollow">newest stable CPU Docker image</a> to validate the tests pass.</p>
