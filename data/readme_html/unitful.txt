<p dir="auto">Unitful is a Julia package for physical units. We want to support not only
SI units but also any other unit system. We also want to minimize or in some
cases eliminate the run-time penalty of units. There should be facilities
for dimensional analysis. All of this should integrate easily with the usual
mathematical operations and collections that are found in Julia base.</p>

<ul dir="auto">
<li><a href="https://github.com/PainterQubits/UnitfulUS.jl">UnitfulUS.jl</a>: U.S. customary units. Serves as an example for how to implement a units
package.</li>
<li><a href="https://github.com/mweastwood/UnitfulAstro.jl">UnitfulAstro.jl</a>: Astronomical units.</li>
<li><a href="https://github.com/yakir12/UnitfulAngles.jl">UnitfulAngles.jl</a>: More angular units, additional trigonometric functionalities, and clock-angle conversion.</li>
<li><a href="https://github.com/sostock/UnitfulAtomic.jl">UnitfulAtomic.jl</a>: Easy conversion from and to atomic units.</li>
<li><a href="https://github.com/invenia/PowerSystemsUnits.jl">PowerSystemsUnits.jl</a>: Common units for dealing with power systems.</li>
<li><a href="https://github.com/rafaqz/UnitfulMoles.jl">UnitfulMoles.jl</a> for defining mol units of chemical elements and compounds.</li>
</ul>

<ul dir="auto">
<li><a href="https://github.com/PainterQubits/UnitfulIntegration.jl">UnitfulIntegration.jl</a>: Enables use of Unitful quantities with <a href="https://github.com/JuliaMath/QuadGK.jl">QuadGK.jl</a>. PRs for other integration packages are welcome.</li>
<li><a href="https://github.com/sostock/UnitfulEquivalences.jl">UnitfulEquivalences.jl</a>: Enables conversion between equivalent quantities of different dimensions, e.g. between energy and wavelength of a photon.</li>
<li><a href="https://github.com/gustaphe/UnitfulLatexify.jl">UnitfulLatexify.jl</a>: Pretty print units and quantities in LaTeX format.</li>
<li><a href="https://github.com/rmsrosa/UnitfulBuckinghamPi.jl">UnitfulBuckinghamPi.jl</a>: Solves for the adimensional Pi groups in a list of Unitful parameters, according to the Buckingham-Pi Theorem.</li>
<li><a href="https://github.com/MasonProtter/NaturallyUnitful.jl">NaturallyUnitful.jl</a>: Convert to and from natural units in physics.</li>
<li><a href="https://github.com/SBuercklin/UnitfulChainRules.jl">UnitfulChainRules.jl</a>: Enables use of Unitful quantities with <a href="https://github.com/JuliaDiff/ChainRules.jl">ChainRules.jl</a>-compatible autodifferentiation systems</li>
<li><a href="https://github.com/cmichelenstrofer/DimensionfulAngles.jl">DimensionfulAngles.jl</a>: Adds angle as a dimension. This allows dispatching on angles and derived quantities.</li>
<li><a href="https://github.com/jw3126/UnitfulRecipes.jl">UnitfulRecipes.jl</a> (deprecated): Adds automatic labels and supports plot axes with units for <a href="https://github.com/JuliaPlots/Plots.jl">Plots.jl</a>. (UnitfulRecipes.jl is now included in Plots.jl.)</li>
</ul>
