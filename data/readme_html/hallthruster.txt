<p dir="auto">HallThruster.jl is an open-source, 1D fluid Hall thruster code written in Julia. It was initially developed by the University of Michigan's <a href="https://pepl.engin.umich.edu" rel="nofollow">Plasmadynamics and Electric Propulsion Laboratory</a> and is licensed under the MIT license.</p>

We recommend using the latest Julia release when possible. Once installed, launch Julia and type</p>
