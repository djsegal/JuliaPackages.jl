<p dir="auto">Geocoder.jl is a packag for geocoding. It is a part of <a href="https://github.com/arubhardwaj/JuliaMaps.jl">JuliaMaps.jl</a>. The idea of this project is to create something like <a href="https://github.com/dkahle/ggmap">ggmap</a> for plotting maps in Julia.</p>
<p dir="auto">Currently, <code>Geocoder.jl</code> it can only be used as a geocoding client for Google Maps API and Open Street Maps API. It is needed to have an API key for using this package, which can be requested from <a href="https://console.cloud.google.com/" rel="nofollow">Google Cloud Console</a> (for google maps) and from <a href="https://developer.mapquest.com/" rel="nofollow">Maps Quest</a> (for Open Streets Map).</p>
<p dir="auto">For more information on requesting an API visit:</p>
<ul dir="auto">
<li><a href="https://developers.google.com/maps/gmp-get-started" rel="nofollow">Google Maps Documentation</a></li>
<li>MapQuest Documentation</li>
</ul>

<p dir="auto">Right now, JuliaMaps.jl supports only two methods: <code>google</code> and <code>osm</code>. <code>google</code> as method supports api from google maps, and <code>osm</code> supports api from Open Streets Map.</p>
<p dir="auto">For example:</p>
<div class="highlight highlight-source-julia notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="geocode(&quot;1600 Amphitheatre Parkway, Mountain View, CA&quot;, &quot;API Key&quot;, &quot;method&quot;)"><pre><span class="pl-c1">geocode</span>(<span class="pl-s"><span class="pl-pds">"</span>1600 Amphitheatre Parkway, Mountain View, CA<span class="pl-pds">"</span></span>, <span class="pl-s"><span class="pl-pds">"</span>API Key<span class="pl-pds">"</span></span>, <span class="pl-s"><span class="pl-pds">"</span>method<span class="pl-pds">"</span></span>)</pre></div>
<p dir="auto">For a vector, it'd be required to use <a href="https://julia.guide/broadcasting" rel="nofollow">broadcasting</a> feature. For example:</p>
<div class="highlight highlight-source-julia notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="geocode.(addresses, &quot;API Key&quot;, &quot;method&quot;)"><pre><span class="pl-c1">geocode</span>.(addresses, <span class="pl-s"><span class="pl-pds">"</span>API Key<span class="pl-pds">"</span></span>, <span class="pl-s"><span class="pl-pds">"</span>method<span class="pl-pds">"</span></span>)</pre></div>

<div class="highlight highlight-source-julia notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="geocode_output = geocode(&quot;1600 Amphitheatre Parkway, Mountain View, CA&quot;, &quot;API Key&quot;, &quot;method&quot;)
