<p dir="auto">The communication value (CV) quantifies the performance of single-copy classical
communication.</p>
<h2 dir="auto"><a id="user-content-features" class="anchor" aria-hidden="true" href="#features"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path d="m7.775 3.275 1.25-1.25a3.5 3.5 0 1 1 4.95 4.95l-2.5 2.5a3.5 3.5 0 0 1-4.95 0 .751.751 0 0 1 .018-1.042.751.751 0 0 1 1.042-.018 1.998 1.998 0 0 0 2.83 0l2.5-2.5a2.002 2.002 0 0 0-2.83-2.83l-1.25 1.25a.751.751 0 0 1-1.042-.018.751.751 0 0 1-.018-1.042Zm-4.69 9.64a1.998 1.998 0 0 0 2.83 0l1.25-1.25a.751.751 0 0 1 1.042.018.751.751 0 0 1 .018 1.042l-1.25 1.25a3.5 3.5 0 1 1-4.95-4.95l2.5-2.5a3.5 3.5 0 0 1 4.95 0 .751.751 0 0 1-.018 1.042.751.751 0 0 1-1.042.018 1.998 1.998 0 0 0-2.83 0l-2.5 2.5a1.998 1.998 0 0 0 0 2.83Z"></path></svg></a>Features:</h2>
<ul dir="auto">
<li>Convex optimization methods bounding the communication value of a quantum channel.</li>
<li>Tools for certifying the non-multiplicativity of the communication value for
quantum channels.</li>
</ul>
<p dir="auto">This Julia package and numerical analysis support the findings in
<a href="https://arxiv.org/abs/2109.11144" rel="nofollow">The Communication Value of a Quantum Channel</a>.</p>

<p dir="auto">By default, CVChannel.jl uses <a href="https://github.com/cvxgrp/scs">SCS</a>
to solve semidefinite programs.
SCS is an open-source numerical solver easily accessible through Julia.</p>

<p dir="auto">It is recommended that you use <a href="https://timholy.github.io/Revise.jl/stable/" rel="nofollow">Revise.jl</a>
so that your saved changes are reflected in the working codebase used by Julia.
Please refer to the Revise.jl documentation for more details.</p>
<p dir="auto">To open the CVChannel.jl package for development,</p>
<ul dir="auto">
<li>Enter package mode from the Julia REPL by entering <code>]</code>.</li>
<li>Run <code>(@vx.x) pkg&gt; develop --local .</code></li>
</ul>
<p dir="auto">This tells Julia to run code from the local version of the CVChannel.jl module
rather than from the github repository.
Thus, <code>using CVChannel</code> will load the local version that you are modifying.</p>
