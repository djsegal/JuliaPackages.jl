<p dir="auto">This submodule provides a carbonate chemistry equilibrium solver originally based on Matlab CO2SYS v1.1 (van Heuven et al 2011, Lewis and Wallace 1998) and refactored for speed and extensibility.</p>
<p dir="auto">Please see <a href="https://github.com/jamesorr/CO2SYS-MATLAB">https://github.com/jamesorr/CO2SYS-MATLAB</a> for the current version of CO2SYS-MATLAB, history, and full citation information.</p>
<p dir="auto">The license for CO2SYS-MATLAB is included as the file LICENSE_CO2SYS.md.</p>
<h3 dir="auto"><a id="user-content-differences-from-co2sys-matlab" class="anchor" aria-hidden="true" href="#differences-from-co2sys-matlab"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path d="m7.775 3.275 1.25-1.25a3.5 3.5 0 1 1 4.95 4.95l-2.5 2.5a3.5 3.5 0 0 1-4.95 0 .751.751 0 0 1 .018-1.042.751.751 0 0 1 1.042-.018 1.998 1.998 0 0 0 2.83 0l2.5-2.5a2.002 2.002 0 0 0-2.83-2.83l-1.25 1.25a.751.751 0 0 1-1.042-.018.751.751 0 0 1-.018-1.042Zm-4.69 9.64a1.998 1.998 0 0 0 2.83 0l1.25-1.25a.751.751 0 0 1 1.042.018.751.751 0 0 1 .018 1.042l-1.25 1.25a3.5 3.5 0 1 1-4.95-4.95l2.5-2.5a3.5 3.5 0 0 1 4.95 0 .751.751 0 0 1-.018 1.042.751.751 0 0 1-1.042.018 1.998 1.998 0 0 0-2.83 0l-2.5 2.5a1.998 1.998 0 0 0 0 2.83Z"></path></svg></a>Differences from CO2SYS-MATLAB</h3>
<ul dir="auto">
<li>Refactored for speed, to allow use in numerical models.</li>
<li>Based on an older (2011) version 1.1 of CO2SYS Matlab</li>
<li>Uses CO2SYS Matlab equilibrium constants, with a reimplemented equilibrium calculation. NB: Although constants can be calculated for different pH scales, TAlk or pH calculations require constants to be on free pH scale.</li>
<li>H2S and NH3 added from Hofman etal (2010)</li>
</ul>
