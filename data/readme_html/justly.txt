<p dir="auto">You can use Justly to both compose and play music using any pitches you want.
Using staff notation, you can only write the items of the 12-tone scale.
Some intervals in any 12-tone scale are close to harmonic, but other intervals are not.
Johnston <a href="http://marsbat.space/pdfs/EJItext.pdf" rel="nofollow">expanded staff notation</a>, but relying on staff notation limited him.</p>

<p dir="auto">In Justly, you write intervals as a rational fraction (integer / integer) times a power of 2.
You can write the same ratio in multiple ways.
For example, you can write a fifth as <code>3/2</code>, or <code>3*2^-1</code>.</p>
<p dir="auto">You will likely only need to know 4 intervals:</p>
<ul dir="auto">
<li>Octave: <code>2/1</code></li>
<li>Perfect fifth: <code>3/2</code></li>
<li>Major third: <code>5/4</code></li>
<li>Harmonic seventh: <code>7/4</code></li>
</ul>
<p dir="auto">You can create other intervals by multiplying and dividing these intervals.
For example, a minor third is up a perfect fifth and down a major third: <code>(3/2) / (5/4)</code> = <code>6/5</code>.
A major second is up two fifths and down an octave: <code>(3/2) * (3/2) / 2</code> = <code>9/8</code>.</p>

<p dir="auto">You can edit the starting frequency, starting volume, and starting tempo using the sliders on the top.</p>
<ul dir="auto">
<li><code>Starting frequency</code> is the starting frequency, in Hz.</li>
<li><code>Starting volume</code> is the starting volume, between 0 and 100%. To avoid peaking, lower the volume for songs with many voices.</li>
<li><code>Starting tempo</code> is the starting tempo, in beats per minute. These beats are indivisible, so for songs which subdivide beats, you will need to multiply the tempo accordingly.</li>
</ul>
<h1 dir="auto"><a id="user-content-chords-vs-notes" class="anchor" aria-hidden="true" href="#chords-vs-notes"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path d="m7.775 3.275 1.25-1.25a3.5 3.5 0 1 1 4.95 4.95l-2.5 2.5a3.5 3.5 0 0 1-4.95 0 .751.751 0 0 1 .018-1.042.751.751 0 0 1 1.042-.018 1.998 1.998 0 0 0 2.83 0l2.5-2.5a2.002 2.002 0 0 0-2.83-2.83l-1.25 1.25a.751.751 0 0 1-1.042-.018.751.751 0 0 1-.018-1.042Zm-4.69 9.64a1.998 1.998 0 0 0 2.83 0l1.25-1.25a.751.751 0 0 1 1.042.018.751.751 0 0 1 .018 1.042l-1.25 1.25a3.5 3.5 0 1 1-4.95-4.95l2.5-2.5a3.5 3.5 0 0 1 4.95 0 .751.751 0 0 1-.018 1.042.751.751 0 0 1-1.042.018 1.998 1.998 0 0 0-2.83 0l-2.5 2.5a1.998 1.998 0 0 0 0 2.83Z"></path></svg></a>Chords vs. Notes</h1>
<p dir="auto">In Justly, there are "chords" and "notes".
A chord is a set of "notes" that will begin playing simulataneously.
You can think of a chord as a "key change" or modulation.
The interval, volume ratio, and tempo ratio changes in chords are cumulative.
The interval, volume ratio, and tempo ratio in a note are in reference to the chord, but only affect the note itself.
You can change the instrument of notes, but not chords.
Currently, only one instrument, "default", is supported, but that could change.</p>

<p dir="auto">This song starts with a key of frequency 220Hz, that is, a A3.
The key does not change in the first chord.
The three voices in the first chord play the tonic (≈A3), third (≈C#4), and fifth (≈E45).
All three voices play for <code>1</code> beat.</p>
<p dir="auto">After 1 beat, the key changes: you divide the key by <code>3/2</code>, so the key goes down by a fifth.
Now the key is close to D4.
The three voices play the fifth (≈A3), up one octave (≈D4), and up one octave and a third (≈F#4).</p>
<p dir="auto">After 1 more beat, you multiply the key by <code>3/2</code>, so the key goes up by a fifth. The voices repeat the items in the first chord, but play for <code>2</code> beats.</p>
