<p dir="auto">This package provides a framework for the method ANOVAapprox to approximate high-dimensional functions with a low superposition dimension or a sparse ANOVA decomposition from scattered data. The method has been dicussed and applied in the following articles/preprints:</p>
<ul dir="auto">
  <li>D. Potts and M. Schmischke <br> 
  <b>Interpretable transformed ANOVA approximation on the example of the prevention of forest fires</b> <br>
  <a href="https://arxiv.org/abs/2110.07353" rel="nofollow">arXiv</a>, <a href="https://www-user.tu-chemnitz.de/~mischmi/papers/transformedanova.pdf" rel="nofollow">PDF</a></li>
  <li>F. Bartel, D. Potts und M. Schmischke <br> 
  <b>Grouped transformations and Regularization in high-dimensional explainable ANOVA approximation</b> <br>
  SIAM Journal on Scientific Computing (accepted) <br>
  <a href="https://arxiv.org/abs/2010.10199" rel="nofollow">arXiv</a>, <a href="https://www-user.tu-chemnitz.de/~mischmi/papers/groupedtransforms.pdf" rel="nofollow">PDF</a></li>
  <li>D. Potts and M. Schmischke <br> 
  <b>Interpretable approximation of high-dimensional data</b> <br>
  SIAM Journal on Mathematics of Data Science (accepted) <br>
  <a href="https://arxiv.org/abs/2103.13787" rel="nofollow">arXiv</a>, <a href="https://www-user.tu-chemnitz.de/~mischmi/papers/attributeranking.pdf" rel="nofollow">PDF</a>, <a href="https://github.com/NFFT/AttributeRankingExamples">Software</a></li>
  <li>D. Potts and M. Schmischke <br> 
  <b>Learning multivariate functions with low-dimensional structures using polynomial bases</b><br>
  Journal of Computational and Applied Mathematics 403, 113821, 2021<br>
  <a href="https://doi.org/10.1016/j.cam.2021.113821" rel="nofollow">DOI</a>, <a href="https://arxiv.org/abs/1912.03195" rel="nofollow">arXiv</a>, <a href="https://www-user.tu-chemnitz.de/~mischmi/papers/anovacube.pdf" rel="nofollow">PDF</a></li>
  <li>D. Potts and M. Schmischke <br> 
  <b>Approximation of high-dimensional periodic functions with Fourier-based methods</b><br>
  SIAM Journal on Numerical Analysis 59 (5), 2393-2429, 2021<br>
  <a href="https://doi.org/10.1137/20M1354921" rel="nofollow">DOI</a>, <a href="https://arxiv.org/abs/1907.11412" rel="nofollow">arXiv</a>, <a href="https://www-user.tu-chemnitz.de/~mischmi/papers/anovafourier.pdf" rel="nofollow">PDF</a></li>
<li>L. Lippert, D. Potts and T. Ullrich <br> 
  <b>Fast Hyperbolic Wavelet Regression meets ANOVA</b><br>
  ArXiv: 2108.13197<br>
  <a href="https://arxiv.org/abs/2108.13197" rel="nofollow">arXiv</a>, <a href="https://www-user.tu-chemnitz.de/~lipl/paper/HWR.pdf" rel="nofollow">PDF</a></li>
</ul>
<p dir="auto"><code>ANOVAapprox.jl</code> provides the following functionality:</p>
<ul dir="auto">
<li>approximation of high-dimensional periodic and nonperiodic functions with a sparse ANOVA decomposition</li>
<li>analysis tools for interpretability (global sensitvitiy indices, attribute ranking, shapley values)</li>
</ul>
