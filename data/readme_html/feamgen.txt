<p dir="auto">This package can be used for the following missions.</p>
<ol dir="auto">
<li>Generate scalar integral (example: <code>test/TSI_Test.jl</code>)</li>
<li>Generate scalar integral with irreducible numerators (example: <code>test/IRD_Test.jl</code>)</li>
<li>Generate tensor integral (example: <code>test/AmpRed_Test.jl</code>)</li>
<li>Generate process amplitude (examples: <code>test/DrellYan_Test.jl</code> etc.)</li>
</ol>

<p dir="auto">The results contain amplitudes in file <code>amplitude_diagram&lt;number&gt;.out</code> and the Feynman diagrams in file <code>visual_diagram&lt;number&gt;.tex</code>.
