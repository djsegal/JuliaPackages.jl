<p dir="auto">A web client for the Hydrologic Reference Stations of the Australian Bureau of Meteorology in the Julia programming language. The website at <a href="http://www.bom.gov.au/water/hrs" rel="nofollow">http://www.bom.gov.au/water/hrs</a> provides high quality long-term streamflow data of unregulated catchments across Australia and their trend analysis results. With the package, you can download the data and analysis results directly to your Julia programming environment.</p>
<h2 dir="auto"><a id="user-content-installation" class="anchor" aria-hidden="true" href="#installation"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path d="m7.775 3.275 1.25-1.25a3.5 3.5 0 1 1 4.95 4.95l-2.5 2.5a3.5 3.5 0 0 1-4.95 0 .751.751 0 0 1 .018-1.042.751.751 0 0 1 1.042-.018 1.998 1.998 0 0 0 2.83 0l2.5-2.5a2.002 2.002 0 0 0-2.83-2.83l-1.25 1.25a.751.751 0 0 1-1.042-.018.751.751 0 0 1-.018-1.042Zm-4.69 9.64a1.998 1.998 0 0 0 2.83 0l1.25-1.25a.751.751 0 0 1 1.042.018.751.751 0 0 1 .018 1.042l-1.25 1.25a3.5 3.5 0 1 1-4.95-4.95l2.5-2.5a3.5 3.5 0 0 1 4.95 0 .751.751 0 0 1-.018 1.042.751.751 0 0 1-1.042.018 1.998 1.998 0 0 0-2.83 0l-2.5 2.5a1.998 1.998 0 0 0 0 2.83Z"></path></svg></a>Installation</h2>
<p dir="auto">The package can be installed with the Julia package manager. From the Julia REPL, type <code>]</code> to enter the Pkg REPL mode and run:</p>
<div class="highlight highlight-source-julia notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="pkg&gt; add HydroRefStations"><pre>pkg<span class="pl-k">&gt;</span> add HydroRefStations</pre></div>
<p dir="auto">If you want to install the package directly from its github development site,</p>
<div class="highlight highlight-source-julia notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="pkg&gt; add http://github.com/petershintech/HydroRefStations.jl"><pre>pkg<span class="pl-k">&gt;</span> add http<span class="pl-k">:</span><span class="pl-k">//</span>github<span class="pl-k">.</span>com<span class="pl-k">/</span>petershintech<span class="pl-k">/</span>HydroRefStations<span class="pl-k">.</span>jl</pre></div>

<p dir="auto">When you create an instance of the <code>HRS</code> structure, it downloads
site information, header and also stores available data types.</p>
<div class="highlight highlight-source-julia notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="julia&gt; hrs = HRS();"><pre>julia<span class="pl-k">&gt;</span> hrs <span class="pl-k">=</span> <span class="pl-c1">HRS</span>();</pre></div>
<p dir="auto">Once it is instantiated, the fields of <code>hrs</code> should be considered as read-only so don't try to change any values of the fields.</p>
