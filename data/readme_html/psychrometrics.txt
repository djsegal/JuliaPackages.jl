<p dir="auto">Psychrometrics provides a set of functions to compute the various variables related to water vapor humid air, providing the following functions:</p>
<ul dir="auto">
<li><strong>psychro</strong></li>
<li><strong>humidity</strong></li>
<li><strong>satPress</strong></li>
<li><strong>enthalpy</strong></li>
<li><strong>volume</strong></li>
<li><strong>adiabSat</strong></li>
<li><strong>dewTemp</strong></li>
</ul>
<h3 dir="auto"><a id="user-content-psychro" class="anchor" aria-hidden="true" href="#psychro"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path d="m7.775 3.275 1.25-1.25a3.5 3.5 0 1 1 4.95 4.95l-2.5 2.5a3.5 3.5 0 0 1-4.95 0 .751.751 0 0 1 .018-1.042.751.751 0 0 1 1.042-.018 1.998 1.998 0 0 0 2.83 0l2.5-2.5a2.002 2.002 0 0 0-2.83-2.83l-1.25 1.25a.751.751 0 0 1-1.042-.018.751.751 0 0 1-.018-1.042Zm-4.69 9.64a1.998 1.998 0 0 0 2.83 0l1.25-1.25a.751.751 0 0 1 1.042.018.751.751 0 0 1 .018 1.042l-1.25 1.25a3.5 3.5 0 1 1-4.95-4.95l2.5-2.5a3.5 3.5 0 0 1 4.95 0 .751.751 0 0 1-.018 1.042.751.751 0 0 1-1.042.018 1.998 1.998 0 0 0-2.83 0l-2.5 2.5a1.998 1.998 0 0 0 0 2.83Z"></path></svg></a><strong>psychro</strong></h3>
<p dir="auto">psychro computes</p>
<ul dir="auto">
<li>the dry bulb temperature,</li>
<li>the wet bulb temperature,</li>
<li>the dew point temperature,</li>
<li>the adiabatic saturation temperature,</li>
<li>the humidity,</li>
<li>the saturation humidity,</li>
<li>the saturation humidity at wet bulb temperature,</li>
<li>the adiabatic saturation humidity,</li>
<li>the specific enthalpy,</li>
<li>the specific volume,</li>
<li>the relative humidity,</li>
<li>the water vapor pressure,</li>
<li>the saturation pressure, the saturation</li>
<li>pressure at wet bulb temperature and</li>
<li>the density</li>
</ul>
<p dir="auto">given any two of the following parameters:</p>
<ul dir="auto">
<li>the dry bulb temperature,</li>
<li>the wet bulb temperature,</li>
<li>the dew point temperature,</li>
<li>the humidity,</li>
<li>the specific enthalpy,</li>
<li>the specific volume or</li>
<li>the relative humidity,</li>
</ul>
<p dir="auto">except the combination of humidity and dew point temperature, which are not independent. If a different number of parameters is given, execution will be aborted. If fig = true is given, a schematic psychrometric chart is plotted as a graphical representation of the solution.</p>
<p dir="auto">By default, stages plots a schematic diagram of the solution, fig = true.</p>
<p dir="auto">If fig = false is given, no plot is shown.</p>
