<p dir="auto">Cell-specific network inference using <em>loca</em>l <em>T</em>ransfer <em>E</em>ntropy.</p>
<p dir="auto">To get started, please see the <a href="https://zsteve.phatcode.net/locatedocs/" rel="nofollow">documentation</a>.</p>
<p dir="auto">Code for reproducing the manuscript figures can be found <a href="https://github.com/zsteve/locaTE-paper">here</a>.</p>
