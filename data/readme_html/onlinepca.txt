<p>OnlinePCA.jl binarizes CSV file, summarizes the information of data matrix and, performs some online-PCA functions for extreamly large scale matrix.</p>

</li>

<h2><a id="user-content-distributed-computing-with-multiple-stepsize-setting" class="anchor" aria-hidden="true" href="#distributed-computing-with-multiple-stepsize-setting"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path fill-rule="evenodd" d="M4 9h1v1H4c-1.5 0-3-1.69-3-3.5S2.55 3 4 3h4c1.45 0 3 1.69 3 3.5 0 1.41-.91 2.72-2 3.25V8.59c.58-.45 1-1.27 1-2.09C10 5.22 8.98 4 8 4H4c-.98 0-2 1.22-2 2.5S3 9 4 9zm9-3h-1v1h1c1 0 2 1.22 2 2.5S13.98 12 13 12H9c-.98 0-2-1.22-2-2.5 0-.83.42-1.64 1-2.09V6.25c-1.09.53-2 1.84-2 3.25C6 11.31 7.55 13 9 13h4c1.45 0 3-1.69 3-3.5S14.5 6 13 6z"></path></svg></a>Distributed Computing with Multiple Stepsize Setting</h2>
<p>The online PCA algorithms are performed until the reconstruction error is converged. In the default stopping criteria, the calculation is stopped when the relative change is bellow 1E-3 or above 0.03. These values can be changed by <em>lower</em> and <em>upper</em> options, respectively.</p>
<p>The convergence is depend on the step size parameter and default value is set as 1000. This value is tuned for single-cell RNA-Seq dataset, but the appropriate level may change according to the size and dynamic range of data matrix.</p>
<p>Combined with <a href="https://en.wikipedia.org/wiki/Oracle_Grid_Engine" rel="nofollow">Grid Engine</a>, this step is easily paralled, because each calculation of different step size are independently performed. For example, we firstly make the following template file (e.g., oja_template) containing the online PCA script,</p>
<div class="highlight highlight-source-shell"><pre><span class="pl-c"><span class="pl-c">#!</span>/bin/bash</span>

julia YOUR_HOME_DIR/.julia/v0.x/OnlinePCA/bin/oja \
--scale log \
--input Data.zst \
--outdir XXXXX \
--rowmeanlist Feature_LogMeans.csv \
--dim 10 \
--stepsize YYYYY \
--logdir XXXXX/log</pre></div>
<p>and then rewrite the template to set different step size by sed command and submit each job by qsub command.</p>
<div class="highlight highlight-source-shell"><pre><span class="pl-c"><span class="pl-c">#!</span>/bin/bash</span>

Steps=(1 10 100 1000 10000 100000 1000000)
<span class="pl-k">for</span> <span class="pl-smi">i</span> <span class="pl-k">in</span> <span class="pl-smi">${Step[@]}</span><span class="pl-k">;</span> <span class="pl-k">do</span>
	OUT=<span class="pl-s"><span class="pl-pds">"</span>Step<span class="pl-pds">"</span></span><span class="pl-smi">$i</span>
	mkdir -p <span class="pl-smi">$OUT</span>
	sed -e <span class="pl-s"><span class="pl-pds">"</span>s|XXXXX|<span class="pl-smi">$OUT</span>|g<span class="pl-pds">"</span></span> oja_template <span class="pl-k">&gt;</span> TMP_oja_scData.sh
	sed -e <span class="pl-s"><span class="pl-pds">"</span>s|YYYYY|<span class="pl-smi">$i</span>|g<span class="pl-pds">"</span></span> TMP_oja_scData.sh <span class="pl-k">&gt;</span> oja_scData.sh
	chmod +x oja_scData.sh
	qsub oja_scData.sh
<span class="pl-k">done</span></pre></div>
<p>Even if there are no distributed computational environment, background process is applicable (just adding &amp; in the end of command).</p>
<div class="highlight highlight-source-shell"><pre><span class="pl-c"><span class="pl-c">#!</span>/bin/bash</span>

Steps=(1 10 100 1000 10000 100000 1000000)
<span class="pl-k">for</span> <span class="pl-smi">i</span> <span class="pl-k">in</span> <span class="pl-smi">${Steps[@]}</span><span class="pl-k">;</span> <span class="pl-k">do</span>
	mkdir -p <span class="pl-s"><span class="pl-pds">"</span>Step<span class="pl-pds">"</span></span><span class="pl-smi">$i</span>
	julia YOUR_HOME_DIR/.julia/v0.x/OnlinePCA/bin/oja \
	--scale log \
	--input Data.zst \
	--outdir <span class="pl-s"><span class="pl-pds">"</span>Step<span class="pl-pds">"</span></span><span class="pl-smi">$i</span> \
	--rowmeanlist Feature_LogMeans.csv \
	--dim 10 \
	--stepsize <span class="pl-smi">$i</span> \
	--logdir <span class="pl-s"><span class="pl-pds">"</span>Step<span class="pl-pds">"</span></span><span class="pl-smi">$i</span>/log <span class="pl-k">&amp;</span>
<span class="pl-k">done</span>

ps <span class="pl-k">|</span> grep julia</pre></div>
