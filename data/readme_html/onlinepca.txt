<p dir="auto">OnlinePCA.jl binarizes CSV file, summarizes the information of data matrix and, performs some online-PCA functions for extreamly large scale matrix.</p>

<p dir="auto">The online PCA algorithms are performed until the reconstruction error is converged. In the default stopping criteria, the calculation is stopped when the relative change is bellow 1E-3 or above 0.03. These values can be changed by <em>lower</em> and <em>upper</em> options, respectively.</p>
<p dir="auto">The convergence is depend on the step size parameter and default value is set as 1000. This value is tuned for single-cell RNA-Seq dataset, but the appropriate level may change according to the size and dynamic range of data matrix.</p>
<p dir="auto">Combined with <a href="https://en.wikipedia.org/wiki/Oracle_Grid_Engine" rel="nofollow">Grid Engine</a>, this step is easily paralled, because each calculation of different step size are independently performed. For example, we firstly make the following template file (e.g., oja_template) containing the online PCA script,</p>
<div class="highlight highlight-source-shell notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="#!/bin/bash

--dim 10 \
--stepsize YYYYY \
--logdir XXXXX/log"><pre><span class="pl-c"><span class="pl-c">#!</span>/bin/bash</span>
