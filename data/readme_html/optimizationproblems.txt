<a href="https://github.com/JuliaOpt/JuMP.jl">JuMP</a> syntax.</p>

<p>OptimizationProblems can be installed and tested through the Julia package manager:</p>
<div class="highlight highlight-source-julia position-relative" data-snippet-clipboard-copy-content="julia&gt; ]
pkg&gt; add OptimizationProblems
pkg&gt; test OptimizationProblems
"><pre>julia<span class="pl-k">&gt;</span> ]
pkg<span class="pl-k">&gt;</span> add OptimizationProblems
pkg<span class="pl-k">&gt;</span> test OptimizationProblems</pre></div>
<p>You can obtain the list of problems currently defined with
<code>names(OptimizationProblems)</code>. The first symbol in the list is
<code>:OptimizationProblems</code> itselfâ€“the name of the module.</p>
<p>Currently, only a few unconstrained problems are implemented. Some are
available in variable size.</p>
<p>This module is particularly useful in conjunction with
<a href="https://github.com/JuliaSmoothOptimizers/NLPModels.jl">NLPModels</a> to facilitate evaluating
objective, constraints and their derivatives.</p>
