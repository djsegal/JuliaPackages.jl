<p dir="auto">I like to write up notes of how to use my packages in plain <code>.jl</code> files,
which the package <a href="https://github.com/fredrikekre/Literate.jl">Literate.jl</a> can run,
producing <code>.ipynb</code> notebooks, with graphs embedded.
This little package automates that slightly. Typing this:</p>
<div class="snippet-clipboard-content notranslate position-relative overflow-auto" data-snippet-clipboard-copy-content="folder(&quot;~/.julia/dev/MyPackage/notes/&quot;)"><pre class="notranslate"><code>folder("~/.julia/dev/MyPackage/notes/")
</code></pre></div>
<p dir="auto">or (roughly) equivalently</p>
<div class="snippet-clipboard-content notranslate position-relative overflow-auto" data-snippet-clipboard-copy-content="import MyPackage
package(MyPackage)"><pre class="notranslate"><code>import MyPackage
package(MyPackage)
</code></pre></div>
<p dir="auto">will run all files in that folder.
Then it will convert the resulting notebooks to <code>.html</code>
to make it easier to preview last week's graphs, unless you say <code>html=false</code>.
You can also pass <code>all=false</code> to update only files that have changed.</p>
<p dir="auto">By default everything is saved in the same folder,
but to avoid saving for all eternity every day's versions of all figures etc,
I add to <code>.gitignore</code> these lines:</p>
<div class="snippet-clipboard-content notranslate position-relative overflow-auto" data-snippet-clipboard-copy-content=".DS_Store
*.ipynb
*.html"><pre class="notranslate"><code>.DS_Store
*.ipynb
*.html
</code></pre></div>
<p dir="auto">Then I can happily include this in the package's tests, it returns <code>true</code> if there are no errors.
In fact if you type <code>] test RunMyNotes</code> this will happen for this package itself,
for a simple <a href="notes/note.jl">note.jl</a>. Or else:</p>
<div class="snippet-clipboard-content notranslate position-relative overflow-auto" data-snippet-clipboard-copy-content="using RunMyNotes
package(RunMyNotes, &quot;~/Desktop&quot;)"><pre class="notranslate"><code>using RunMyNotes
package(RunMyNotes, "~/Desktop")
</code></pre></div>
