<p>Lint.jl is a tool that uses static analysis to assist in the development process by detecting common bugs and potential issues.</p>

<p>There are 3 functions you can use to lint your code.</p>
<ul>
<li><code>lintpkg("MyPackage")</code> for linting an entire package</li>
<li><code>lintfile("my_file.jl")</code> for linting a file</li>
<li><code>lintstr("my string")</code> for linting a string</li>
</ul>
