<p dir="auto">Install with the Julia package manager <a href="https://pkgdocs.julialang.org/" rel="nofollow">Pkg</a>, just like any other registered Julia package:</p>
<div class="highlight highlight-source-julia notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="pkg&gt; add PkgTemplates  # Press ']' to enter the Pkg REPL mode."><pre>pkg<span class="pl-k">&gt;</span> add PkgTemplates  <span class="pl-c"><span class="pl-c">#</span> Press ']' to enter the Pkg REPL mode.</span></pre></div>
<p dir="auto">or</p>
<div class="highlight highlight-source-julia notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="julia&gt; using Pkg; Pkg.add(&quot;PkgTemplates&quot;)"><pre>julia<span class="pl-k">&gt;</span> <span class="pl-k">using</span> Pkg; Pkg<span class="pl-k">.</span><span class="pl-c1">add</span>(<span class="pl-s"><span class="pl-pds">"</span>PkgTemplates<span class="pl-pds">"</span></span>)</pre></div>

<p dir="auto">Creating a <code>Template</code> is as simple as:</p>
<div class="highlight highlight-source-julia notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="using PkgTemplates
tpl = Template()"><pre><span class="pl-k">using</span> PkgTemplates
tpl <span class="pl-k">=</span> <span class="pl-c1">Template</span>()</pre></div>
<p dir="auto">The no-keywords constructor assumes the existence of some preexisting Git configuration (set with <code>git config --global</code>):</p>
<ul dir="auto">
<li><code>user.name</code>: Your real name, e.g. John Smith.</li>
<li><code>user.email</code>: Your email address, eg. <a href="mailto:john.smith@acme.corp">john.smith@acme.corp</a>.</li>
<li><code>github.user</code>: Your GitHub username: e.g. john-smith.</li>
</ul>
<p dir="auto">Once you have a <code>Template</code>, use it to generate a package:</p>
<div class="highlight highlight-source-julia notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="tpl(&quot;MyPkg&quot;)"><pre><span class="pl-c1">tpl</span>(<span class="pl-s"><span class="pl-pds">"</span>MyPkg<span class="pl-pds">"</span></span>)</pre></div>
<p dir="auto">However, it's probably desirable to customize the template to your liking with various options and plugins:</p>
<div class="highlight highlight-source-julia notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="tpl = Template(;
    dir=&quot;~/code&quot;,
    plugins=[

        Documenter{TravisCI}(),
    ],
)"><pre>tpl <span class="pl-k">=</span> <span class="pl-c1">Template</span>(;

    plugins<span class="pl-k">=</span>[
        <span class="pl-c1">Git</span>(; manifest<span class="pl-k">=</span><span class="pl-c1">true</span>, ssh<span class="pl-k">=</span><span class="pl-c1">true</span>),
        <span class="pl-c1">Codecov</span>(),
        <span class="pl-c1">TravisCI</span>(; x86<span class="pl-k">=</span><span class="pl-c1">true</span>),
        <span class="pl-c1">Documenter</span><span class="pl-c1">{TravisCI}</span>(),
    ],
)</pre></div>
