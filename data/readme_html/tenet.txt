<p dir="auto">This is a Julia package for the Variational Uniform Matrix product states(VUMPS) to contract infinite two-dimension square lattice tensor network.</p>
<p dir="auto">In this package we implemented the VUMPS algorithms including the following:</p>
<ul dir="auto">
<li>Complex number forward and backward propagation</li>
<li>NixNj Big Unit Cell</li>
<li>U1-symmmetry and Z2-symmmetry</li>
</ul>
<p dir="auto">This package a updated version of the original <a href="https://github.com/XingyuZhang2018/ADVUMPS.jl">ADVUMPS.jl</a> package, so for more easily to learn and use, we recommend you to use ADVUMPS.jl package.</p>
