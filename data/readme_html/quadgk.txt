<p dir="auto">This package provides support for one-dimensional numerical integration in Julia using adaptive
Gauss-Kronrod quadrature.
The code was originally part of Base Julia.  It supports integration of arbitrary numeric types,
including arbitrary precision (<code>BigFloat</code>), and even integration of arbitrary normed vector spaces
(e.g. matrix-valued integrands).</p>
<p dir="auto">The package provides three basic functions: <code>quadgk</code>, <code>gauss</code>, and <code>kronrod</code>.
<code>quadgk</code> performs the integration, <code>gauss</code> computes Gaussian quadrature points and weights for integrating
over the interval [a, b], and <code>kronrod</code> computes Kronrod points, weights, and embedded Gaussian quadrature
weights for integrating over [-1, 1].   Typical usage looks like:</p>
<div class="highlight highlight-source-julia notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="using QuadGK
integral, err = quadgk(x -&gt; exp(-x^2), 0, 1, rtol=1e-8)"><pre><span class="pl-k">using</span> QuadGK
integral, err <span class="pl-k">=</span> <span class="pl-c1">quadgk</span>(x <span class="pl-k">-&gt;</span> <span class="pl-c1">exp</span>(<span class="pl-k">-</span>x<span class="pl-k">^</span><span class="pl-c1">2</span>), <span class="pl-c1">0</span>, <span class="pl-c1">1</span>, rtol<span class="pl-k">=</span><span class="pl-c1">1e-8</span>)</pre></div>
<p dir="auto">which computes the integral of exp(–x²) from x=0 to x=1 to a relative tolerance of 10⁻⁸, and returns the approximate <code>integral = 0.746824132812427</code> and error estimate <code>err = 7.887024366937112e-13</code> (which is actually smaller than the requested tolerance: convergence was very rapid because the integrand is smooth).</p>
<p dir="auto">For more information, see the <a href="https://JuliaMath.github.io/QuadGK.jl/stable" rel="nofollow">documentation</a>.</p>
