<p dir="auto">P4estTypes.jl provides a <a href="http://julialang.org/" rel="nofollow">Julia</a> type-based interface to
the distributed (via <a href="http://www.mpi-forum.org/" rel="nofollow">MPI</a>) forest-of-octrees
library <a href="https://p4est.org" rel="nofollow">p4est</a> by leveraging
<a href="https://github.com/trixi-framework/P4est.jl">P4est.jl</a> (which provides
low-level Julia bindings to the p4est API).</p>

<p dir="auto">Contributions are encouraged. In particular only a small part of p4est is
currently exposed. If there are additional functions you would like to use,
please open an <a href="https://github.com/lcw/P4estTypes.jl/issues">issue</a> or <a href="https://github.com/lcw/P4estTypes.jl/pulls">pull
request</a>.</p>
<p dir="auto">Additional examples and documentation improvements are also welcome.</p>
