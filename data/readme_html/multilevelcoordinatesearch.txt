<p>This is an implementation of <a href="http://www.mat.univie.ac.at/~Neum/ms/mcs.pdf" rel="nofollow">Multilevel Coordinate Search</a> (MCS), an algorithm by Waltraud Huyer and Arnold Neumaier for global minimization over a possibly-bounded domain. In a <a href="https://link.springer.com/article/10.1007/s10898-012-9951-y" rel="nofollow">recent comparison of global optimization routines that do not require derivatives</a>, MCS scored highest among non-commercial algorithms.</p>
<p>This implementation is "fresh" from the original paper, and is not based on the Matlab code provided <a href="https://www.mat.univie.ac.at/~neum/software/mcs/" rel="nofollow">by the authors</a>.</p>
<p>Status: work in progress. Until it's finished (if it ever will be), check out my <a href="https://github.com/timholy/QuadDIRECT.jl">QuadDIRECT</a>.</p>
