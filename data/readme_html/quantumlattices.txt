<p dir="auto"><em>Julia package for the construction of quantum lattice systems.</em></p>
<p dir="auto">Welcome to <strong><a href="https://github.com/Quantum-Many-Body/QuantumLattices.jl">QuantumLattices</a></strong>. Here we provide a general framework to construct the <strong>operator-formed Hamiltonian of any quantum lattice system</strong>, with the inputs as simple as its description by the natural language. This operator-formed Hamiltonian supports <strong>complete symbolic computations</strong> when combined with <strong><a href="https://github.com/JuliaPy/SymPy.jl">SymPy</a></strong>, and can serve as a convenient <strong>frontend of quantum many-body algorithms</strong>, such as <strong><a href="https://github.com/Quantum-Many-Body/TightBindingApproximation.jl">TBA (tight-bind approximation)</a></strong>, <strong><a href="https://github.com/Quantum-Many-Body/SpinWaveTheory.jl">LSWT (linear spin wave theory)</a></strong>, <strong>SCMF (self-consistent mean field theory)</strong>, <strong><a href="https://github.com/Quantum-Many-Body/ExactDiagonalization.jl">ED (exact diagonalization)</a></strong>, <strong><a href="https://github.com/Quantum-Many-Body/QuantumClusterTheories.jl">CPT/VCA (cluster perturbation theory / variational cluster approach)</a></strong>, <strong>DMRG (density matrix renormalization group)</strong>, etc. Generic interfaces are defined to provide a unified access to these algorithms with automatic project management.</p>

<p dir="auto">The mathematical foundations of our package is that the operators in a lattice Hamiltonian:</p>
<ul dir="auto">
<li><strong>act on local Hilbert spaces</strong>, and</li>
<li><strong>form an algebra over the complex field</strong>.</li>
</ul>
<p dir="auto">Based on this, the package has the following features:</p>
<ul dir="auto">
<li>
<p dir="auto"><strong>Unitcell Description Framework</strong>: the Hamiltonian can be constructed based on the unitcell of a lattice with the information of the local algebra acting on the local Hilbert space living on each point and the terms that couples different degrees of freedom on the same or different points. Such information can be input into the program as simple as describing the quantum system in a usual research paper.</p>
</li>
<li>
<p dir="auto"><strong>Complete Symbolic Computation</strong>: with only this package, symbolic computation between operators is realized while the coefficient of any operator remains numeric; by integrating it with <a href="https://github.com/JuliaPy/SymPy.jl">SymPy</a>, complete symbolic computation can be achieved and no modifications need be made on the methods in this package.</p>
</li>
<li>
<p dir="auto"><strong>Generic Frontend of Many-Body Algorithms</strong>: with the operator-formed Hamiltonian as the foothold, quantum many-body algorithms can be initialized in quite similar ways with only minor modifications needed. Moreover, automatic project management is realized, including that of result recording, data caching, parameter updating, information logging, dependency managing, etc.</p>
</li>
</ul>

<p dir="auto">Four common categories of quantum lattice systems in condensed matter physics are supported:</p>
<ul dir="auto">
<li><strong>canonical complex fermionic systems</strong></li>
<li><strong>canonical complex and hard-core bosonic systems</strong></li>
<li><strong>SU(2) spin systems</strong></li>
<li><strong>Phononic systems</strong></li>
</ul>
<p dir="auto">Furthermore, other systems can be supported easily by extending the generic protocols provided in this package.</p>
<h2 dir="auto"><a id="user-content-supported-algorithms" class="anchor" aria-hidden="true" href="#supported-algorithms"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path d="m7.775 3.275 1.25-1.25a3.5 3.5 0 1 1 4.95 4.95l-2.5 2.5a3.5 3.5 0 0 1-4.95 0 .751.751 0 0 1 .018-1.042.751.751 0 0 1 1.042-.018 1.998 1.998 0 0 0 2.83 0l2.5-2.5a2.002 2.002 0 0 0-2.83-2.83l-1.25 1.25a.751.751 0 0 1-1.042-.018.751.751 0 0 1-.018-1.042Zm-4.69 9.64a1.998 1.998 0 0 0 2.83 0l1.25-1.25a.751.751 0 0 1 1.042.018.751.751 0 0 1 .018 1.042l-1.25 1.25a3.5 3.5 0 1 1-4.95-4.95l2.5-2.5a3.5 3.5 0 0 1 4.95 0 .751.751 0 0 1-.018 1.042.751.751 0 0 1-1.042.018 1.998 1.998 0 0 0-2.83 0l-2.5 2.5a1.998 1.998 0 0 0 0 2.83Z"></path></svg></a>Supported Algorithms</h2>
<p dir="auto">Concrete algorithms could be considered as the "backend" of quantum lattice systems. They are developed in separate packages (still in progress):</p>
<ul dir="auto">
<li><strong><a href="https://github.com/Quantum-Many-Body/TightBindingApproximation.jl">TBA</a></strong>: tight-binding approximation for complex-fermionic/complex-bosonic/phononic systems;</li>
<li><strong>SCMF</strong>: self-consistent mean field theory for complex fermionic systems;</li>
<li><strong><a href="https://github.com/Quantum-Many-Body/ExactDiagonalization.jl">ED</a></strong>: exact diagonalization for complex-fermionic/hard-core-bosonic/local-spin systems;</li>
<li><strong><a href="https://github.com/Quantum-Many-Body/QuantumClusterTheories.jl">CPT/VCA</a></strong>: cluster perturbation theory and variational cluster approach for complex fermionic and local spin systems;</li>
<li><strong>DMRG</strong>: density matrix renormalization group for complex-fermionic/hard-core-bosonic/local-spin systems;</li>
<li><strong><a href="https://github.com/Quantum-Many-Body/SpinWaveTheory.jl">LSWT</a></strong>: linear spin wave theory for magnetically ordered local-spin systems.</li>
</ul>
