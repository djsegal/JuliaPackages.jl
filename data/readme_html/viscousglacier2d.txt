<p dir="auto">Concise CPU and GPU iterative Stokes solvers for viscous flow - 2D glacier examples.</p>
<h2 dir="auto"><a id="user-content-description" class="anchor" aria-hidden="true" href="#description"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path d="m7.775 3.275 1.25-1.25a3.5 3.5 0 1 1 4.95 4.95l-2.5 2.5a3.5 3.5 0 0 1-4.95 0 .751.751 0 0 1 .018-1.042.751.751 0 0 1 1.042-.018 1.998 1.998 0 0 0 2.83 0l2.5-2.5a2.002 2.002 0 0 0-2.83-2.83l-1.25 1.25a.751.751 0 0 1-1.042-.018.751.751 0 0 1-.018-1.042Zm-4.69 9.64a1.998 1.998 0 0 0 2.83 0l1.25-1.25a.751.751 0 0 1 1.042.018.751.751 0 0 1 .018 1.042l-1.25 1.25a3.5 3.5 0 1 1-4.95-4.95l2.5-2.5a3.5 3.5 0 0 1 4.95 0 .751.751 0 0 1-.018 1.042.751.751 0 0 1-1.042.018 1.998 1.998 0 0 0-2.83 0l-2.5 2.5a1.998 1.998 0 0 0 0 2.83Z"></path></svg></a>Description</h2>
<p dir="auto">This repository contains GPU and CPU implementations of a 2D Stokes solver to resolve glacier flow on an inclined plane with a stress-free surface. The routines exemplify the implementation of a fast and concise (60 lines of code) iterative solving strategy referred to as Pseudo-Transient. The core of the Pseudo-Transient approach relies in using physics-motivated transient terms within differential equations in order to iteratively converge to an accurate solution.</p>
<p dir="auto">The routines relate to a seminar on <em>"Simply solving PDEs"</em>. The seminar presentation is accessible <a href="#seminar-presentation">here</a>. Further material relating to the pseudo-transient method and pseudo-transient solvers can be found <a href="https://ptsolvers.github.io" rel="nofollow">here</a>.</p>

<p dir="auto">The <a href="/scripts/">/scripts</a> folder contains the following PT routines:</p>
<ul dir="auto">
<li><a href="scripts/viscous_gl2D.jl"><code>viscous_gl2D.jl</code></a></li>

<p dir="auto">The keywords in the code naming stand for:</p>
<ul dir="auto">
<li>gpu: Nvidia GPU ready routines using the <a href="https://github.com/JuliaGPU/CUDA.jl">CUDA.jl</a> package</li>
<li>nondim: non-dimensional version of the code, using natural scaling</li>
</ul>
