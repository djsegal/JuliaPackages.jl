<p dir="auto"><code>@safetestset</code> puts <code>@testset</code> into a module.</p>
<p dir="auto">You can use it by</p>
<div class="highlight highlight-source-julia notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="using SafeTestsets
@safetestset &quot;Benchmark Tests&quot; begin include(&quot;benchmark_tests.jl&quot;) end"><pre><span class="pl-k">using</span> SafeTestsets
<span class="pl-c1">@safetestset</span> <span class="pl-s"><span class="pl-pds">"</span>Benchmark Tests<span class="pl-pds">"</span></span> <span class="pl-k">begin</span> <span class="pl-c1">include</span>(<span class="pl-s"><span class="pl-pds">"</span>benchmark_tests.jl<span class="pl-pds">"</span></span>) <span class="pl-k">end</span></pre></div>
<p dir="auto">which makes a new module whose name is generated by <code>gensym()</code>, or</p>
<div class="highlight highlight-source-julia notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="using SafeTestsets
@safetestset BenchmarkTests = &quot;Benchmark Tests&quot; begin include(&quot;benchmark_tests.jl&quot;) end"><pre><span class="pl-k">using</span> SafeTestsets
<span class="pl-c1">@safetestset</span> BenchmarkTests <span class="pl-k">=</span> <span class="pl-s"><span class="pl-pds">"</span>Benchmark Tests<span class="pl-pds">"</span></span> <span class="pl-k">begin</span> <span class="pl-c1">include</span>(<span class="pl-s"><span class="pl-pds">"</span>benchmark_tests.jl<span class="pl-pds">"</span></span>) <span class="pl-k">end</span></pre></div>
<p dir="auto">which makes a module named <code>BenchmarkTests</code>.</p>
