<p dir="auto">The package can be installed with the Julia package manager.
From the Julia REPL, type <code>]</code> to enter the Pkg REPL mode and run:</p>
<div class="snippet-clipboard-content notranslate position-relative overflow-auto" data-snippet-clipboard-copy-content="pkg&gt; add ExpressCommands"><pre class="notranslate"><code>pkg&gt; add ExpressCommands
</code></pre></div>
<p dir="auto">Or, equivalently, via the <a href="https://pkgdocs.julialang.org/v1/getting-started/" rel="nofollow"><code>Pkg</code> API</a>:</p>
<div class="highlight highlight-source-julia notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="julia&gt; import Pkg; Pkg.add(&quot;ExpressCommands&quot;)"><pre>julia<span class="pl-k">&gt;</span> <span class="pl-k">import</span> Pkg; Pkg<span class="pl-k">.</span><span class="pl-c1">add</span>(<span class="pl-s"><span class="pl-pds">"</span>ExpressCommands<span class="pl-pds">"</span></span>)</pre></div>
<h2 dir="auto"><a id="user-content-command-line-interface" class="anchor" aria-hidden="true" href="#command-line-interface"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path d="m7.775 3.275 1.25-1.25a3.5 3.5 0 1 1 4.95 4.95l-2.5 2.5a3.5 3.5 0 0 1-4.95 0 .751.751 0 0 1 .018-1.042.751.751 0 0 1 1.042-.018 1.998 1.998 0 0 0 2.83 0l2.5-2.5a2.002 2.002 0 0 0-2.83-2.83l-1.25 1.25a.751.751 0 0 1-1.042-.018.751.751 0 0 1-.018-1.042Zm-4.69 9.64a1.998 1.998 0 0 0 2.83 0l1.25-1.25a.751.751 0 0 1 1.042.018.751.751 0 0 1 .018 1.042l-1.25 1.25a3.5 3.5 0 1 1-4.95-4.95l2.5-2.5a3.5 3.5 0 0 1 4.95 0 .751.751 0 0 1-.018 1.042.751.751 0 0 1-1.042.018 1.998 1.998 0 0 0-2.83 0l-2.5 2.5a1.998 1.998 0 0 0 0 2.83Z"></path></svg></a>Command Line Interface</h2>
<p dir="auto">Add <code>~/.julia/bin</code> to your <code>PATH</code> to enable command line interface. Or run
<code>ExpressCommands.comonicon_install_path()</code> to install everything automatically.</p>
<p dir="auto">Sometimes, you won't trigger the package <code>build</code> of Julia. You can install the command line interface
manually via <code>ExpressCommands.comonicon_install()</code>.</p>
<h3 dir="auto"><a id="user-content-completions" class="anchor" aria-hidden="true" href="#completions"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path d="m7.775 3.275 1.25-1.25a3.5 3.5 0 1 1 4.95 4.95l-2.5 2.5a3.5 3.5 0 0 1-4.95 0 .751.751 0 0 1 .018-1.042.751.751 0 0 1 1.042-.018 1.998 1.998 0 0 0 2.83 0l2.5-2.5a2.002 2.002 0 0 0-2.83-2.83l-1.25 1.25a.751.751 0 0 1-1.042-.018.751.751 0 0 1-.018-1.042Zm-4.69 9.64a1.998 1.998 0 0 0 2.83 0l1.25-1.25a.751.751 0 0 1 1.042.018.751.751 0 0 1 .018 1.042l-1.25 1.25a3.5 3.5 0 1 1-4.95-4.95l2.5-2.5a3.5 3.5 0 0 1 4.95 0 .751.751 0 0 1-.018 1.042.751.751 0 0 1-1.042.018 1.998 1.998 0 0 0-2.83 0l-2.5 2.5a1.998 1.998 0 0 0 0 2.83Z"></path></svg></a>Completions</h3>
<p dir="auto">If you are using ZSH, you can enable the auto-completion by <code>ExpressCommands.comonicon_install_path()</code>. Or add the <code>FPATH</code>
to your <code>.zshrc</code></p>
<div class="highlight highlight-source-shell notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="export FPATH=&quot;$HOME/.julia/completions:$FPATH&quot;"><pre><span class="pl-k">export</span> FPATH=<span class="pl-s"><span class="pl-pds">"</span><span class="pl-smi">$HOME</span>/.julia/completions:<span class="pl-smi">$FPATH</span><span class="pl-pds">"</span></span></pre></div>
<p dir="auto">if you do not have <a href="https://github.com/ohmyzsh/ohmyzsh">oh-my-zsh</a> installed, you need to add</p>
<div class="highlight highlight-source-shell notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="autoload -Uz compinit &amp;&amp; compinit"><pre>autoload -Uz compinit <span class="pl-k">&amp;&amp;</span> compinit</pre></div>
<p dir="auto">to your <code>.zshrc</code> as well.</p>
