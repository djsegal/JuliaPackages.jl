<p>TLDR: Python / C# <code>yield</code> with performance matching plain Julia iterators  (i.e. unbelievably fast)</p>
<p>Continuables are generator-like higher-order functions which take a continuation as an extra argument. The key macro provided by the package is <code>@cont</code> which will give access to the special function <code>cont</code> within its scope and wraps the computation in a special Type <code>Continuables.Continuable</code>.
It is best to think of <code>cont</code> in the sense of <code>yield</code> from Python's Generators. It generates values and takes feedback from the outer process as return value.</p>
<p>If you come from Python, use Continuables wherever you would use generators. If you are Julia-native, Continuables can be used instead of Julia's Channels in many place with drastic performance-improvements (really drastic: in the little benchmark example below it is 20 million times faster!).</p>
<p>This package implements all standard functions like e.g. <code>collect</code>, <code>reduce</code>, <code>any</code> and others. As well as functionalities known from <code>Base.Iterators</code> and <a href="https://github.com/JuliaCollections/IterTools.jl"><code>IterTools.jl</code></a> like <code>take</code>, <code>dropwhile</code>, <code>groupby</code>, <code>partition</code>, <code>nth</code> and others.</p>
<p>For convenience, all methods also work for plain iterables.</p>
