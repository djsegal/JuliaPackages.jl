<p>Emulate directed rounding using only the default rounding mode.</p>
<p><a href="https://travis-ci.com/matsueushi/RoundingEmulator.jl" rel="nofollow"><img src="https://camo.githubusercontent.com/42d7733695b6fae5b326eafd8a1803213be095a7e62819d33104d46da3ea3ea4/68747470733a2f2f7472617669732d63692e636f6d2f6d6174737565757368692f526f756e64696e67456d756c61746f722e6a6c2e7376673f6272616e63683d6d6173746572" alt="Build Status" data-canonical-src="https://travis-ci.com/matsueushi/RoundingEmulator.jl.svg?branch=master" style="max-width:100%;"></a> <a href="https://matsueushi.github.io/RoundingEmulator.jl/dev/" rel="nofollow"><img src="https://camo.githubusercontent.com/7fcec4b2d3ab291529fce8ef6a4fcd4129a0683b2f5d5fe2f5c648f02db8b616/68747470733a2f2f696d672e736869656c64732e696f2f62616467652f646f63732d6465762d626c75652e737667" alt="Dev Docs" data-canonical-src="https://img.shields.io/badge/docs-dev-blue.svg" style="max-width:100%;"></a></p>
<p>This package is meant to produce the exact same results of <code>Rounding.setrounding</code> (<a href="https://github.com/JuliaLang/julia/pull/27166">deprecated</a>) without switching rounding modes.</p>
