<p dir="auto">The <code>SkewLinearAlgebra</code> package provides specialized matrix types, optimized methods of <code>LinearAlgebra</code> functions, and a few entirely new functions for dealing with linear algebra on <a href="https://en.wikipedia.org/wiki/Skew-Hermitian_matrix" rel="nofollow">skew-Hermitian matrices</a>, especially for the case of <a href="https://en.wikipedia.org/wiki/Skew-symmetric_matrix" rel="nofollow">real skew-symmetric matrices</a>.</p>
<p dir="auto">In particular, it defines new <code>SkewHermitian</code> and <code>SkewHermTridiagonal</code> matrix types supporting optimized eigenvalue/eigenvector,
Hessenberg factorization, and matrix exponential/trigonometric functions.  It also provides functions to compute the
<a href="https://en.wikipedia.org/wiki/Pfaffian" rel="nofollow">Pfaffian</a> of real skew-symmetric matrices, along with a Cholesky-like factorization.</p>
<p dir="auto">See the <a href="https://julialinearalgebra.github.io/SkewLinearAlgebra.jl/dev/" rel="nofollow">Documentation</a> for details.</p>
