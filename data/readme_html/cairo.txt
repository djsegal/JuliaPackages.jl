<p>Adaptation to <a href="https://www.cairographics.org/" rel="nofollow">Cairo</a>, a 2D graphics library with support for multiple output devices.</p>
<p>Some of the functions implemented by this wrapper may be documented in <a href="https://juliagraphics.github.io/Graphics.jl/stable/" rel="nofollow">Graphics.jl</a>.</p>
<p>There is an extensive set of <a href="samples/Samples.md">examples</a>.</p>
