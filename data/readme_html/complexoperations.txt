<p dir="auto">Perform basic operations on complex numbers. Complex numbers are represented as 2-element vectors. Functions here can return StaticArrays for extra speed.
Operations include:</p>
<ul dir="auto">
<li>complex scalar operations
<ul dir="auto">
<li>scalar multiplications (<code>complex_multiply</code>, <code>complex_multiply_SMatrix</code>)</li>
<li>scalar division (<code>complex_divide</code>, <code>complex_divide_SMatrix</code>)</li>
</ul>
</li>
<li>complex vector operations
<ul dir="auto">
<li>vector dot product (<code>complex_vector_dot_product</code>, <code>complex_vector_dot_product_SMatrix</code>)</li>
<li>vector cross product (<code>complex_vector_cross_product</code>, <code>complex_vector_cross_product_SMatrix</code>)</li>
</ul>
</li>
<li>complex matrix operations
<ul dir="auto">
<li>matrix multiplication (<code>complex_matrix_multiplication</code>)</li>
<li>matrix inversion (<code>complex_matrix_inversion</code>)</li>
</ul>
</li>
</ul>
