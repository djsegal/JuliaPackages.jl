<p dir="auto">Routines for reading and manipulating compressed storage of biallelic <a href="https://en.wikipedia.org/wiki/Single-nucleotide_polymorphism" rel="nofollow">SNP (Single-Nucleotide Polymorphism)</a> data.</p>
<p dir="auto">Data from <a href="https://en.wikipedia.org/wiki/Genome-wide_association_study" rel="nofollow"><em>genome-wide association studies (GWAS)</em></a> are often saved as a <a href="https://www.cog-genomics.org/plink2/formats#bed" rel="nofollow"><strong>PLINK binary biallelic genotype table</strong></a> or <code>.bed</code> file.
To be useful, such files should be accompanied by a <code>.fam</code> file, containing metadata on the rows of the table, and a <code>.bim</code> file, containing metadata on the columns. The <code>.fam</code> and <code>.bim</code> files are in tab-separated format.</p>
<p dir="auto">Linear algebra operations on the PLINK formatted data now support multi-threading and GPU (CUDA) computing.</p>

<p dir="auto">This package requires Julia v1.5 or later, which can be obtained from
<a href="https://julialang.org/downloads/" rel="nofollow">https://julialang.org/downloads/</a> or by building Julia from the sources in the
<a href="https://github.com/JuliaLang/julia">https://github.com/JuliaLang/julia</a> repository.</p>
<p dir="auto">This package is registered in the default Julia package registry, and can be installed through standard package installation procedure.</p>
