<p dir="auto">Create a file at <code>.github/workflows/CompatHelper.yml</code> with the contents of the <a href=".github/workflows/CompatHelper.yml">CompatHelper.yml</a> that is included in this repository.</p>
<p dir="auto">If you need to use any special arguments for the <code>main</code> function, you can modify this file to add them.</p>

<p dir="auto">Similarly to the GitHub setup, you can modify the <code>main</code> call here if you need to change any of the default arguments.
You must also remember to add the <code>GITLAB_TOKEN</code> and <code>COMPATHELPER_PRIV</code> CI secrets to the project so that CompatHelper can find them.</p>
