<p><code>GpABC</code> provides algorithms for likelihood - free <a href="https://tanhevg.github.io/GpABC.jl/latest/overview-abc/" rel="nofollow">parameter inference</a> and <a href="https://tanhevg.github.io/GpABC.jl/latest/overview-ms/" rel="nofollow">model selection</a> using Approximate Bayesian Computation (<strong>ABC</strong>). Two sets of algorithms are available:</p>
<ul>
<li>Simulation based - full simulations of the model(s) is done on each step of ABC.</li>
<li>Emulation based - a small number of simulations can be used to train a regression model (the <em>emulator</em>), which is then used to approximate model simulation results during ABC.</li>
</ul>
<p><code>GpABC</code> offers <a href="https://tanhevg.github.io/GpABC.jl/latest/overview-gp/" rel="nofollow">Gaussian Process Regression</a> (<strong>GPR</strong>) as an emulator, but custom emulators can also be used. GPR can also be used standalone, for any regression task.</p>
<p>Stochastic models, that don't conform to Gaussian Process Prior assumption, are supported via <a href="https://tanhevg.github.io/GpABC.jl/latest/overview-lna/" rel="nofollow">Linear Noise Approximation</a> (<strong>LNA</strong>).</p>
