<p dir="auto"><strong>SimpleConstants</strong> defines the values of some fundamental constants of physics in SI units.  <a href="https://github.com/JuliaPhysics/PhysicalConstants.jl">PhysicalConstants</a> provides a complete set of fundamental constants with units, but in many cases of writing scientific code we care only about the values of the fundamental constants without units.  SimpleConstants is a lightweight package defining those values.</p>
<p dir="auto">SimpleConstants utilizes Unicode symbols as much as possible to define compact symbols to store the fundamental constants (e.g., <code>ε₀</code> instead of <code>eps_0</code> or <code>ε_0</code>).  It further promotes compactness by defining <em>compound constants</em> that are products of fundamental constants that frequently appear in equations.  For example,</p>
<ul dir="auto">
<li>the package defines <code>πc₀ = π * c₀</code>, with which the angular frequency of an electromagnetic wave of wavelength <code>λ</code> can be written <code>ω = 2πc₀ / λ</code> instead of <code>ω = 2π * c₀ / λ</code>;</li>
<li>the package defines <code>i2π = im * 2π</code>, with which the complex exponential of an angle 120° can be written <code>exp(i2π/3)</code> instead of <code>exp(im * 2π / 3)</code>;
etc.</li>
</ul>
<p dir="auto">SimpleConstants also defines all the SI prefixes (e.g., <code>kilo</code> and <code>milli</code>).  <a href="https://github.com/PainterQubits/Unitful.jl">Unitful</a> provides a complete set of those prefixes together with other units, but again, SimpleConstants is a lightweight package defining those prefixes.</p>
