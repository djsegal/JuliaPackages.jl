<p dir="auto">[EdgeList]: a simple list of sources and dests separated by whitespace and/or comma, one pair per line.
[GML]: <a href="https://en.wikipedia.org/wiki/Graph_Modelling_Language" rel="nofollow">https://en.wikipedia.org/wiki/Graph_Modelling_Language</a>

[GEXF]: <a href="https://gephi.org/gexf/format/" rel="nofollow">https://gephi.org/gexf/format/</a>
[DOT]: <a href="https://en.wikipedia.org/wiki/DOT_(graph_description_language)" rel="nofollow">https://en.wikipedia.org/wiki/DOT_(graph_description_language)</a></p>
<p dir="auto">Graphs are read using either the <code>loadgraph</code> function or, for formats that support multiple graphs in a single file,
the <code>loadgraphs</code> functions. <code>loadgraph</code> returns a Graph object, while <code>loadgraphs</code> returns a dictionary of Graph objects.</p>
<p dir="auto">For example, an edgelist file could be loaded as:</p>
