<p dir="auto">Collection of little helpers to simplify various Machine Learning tasks
(esp. building neural networks with Knet).</p>
<p dir="auto">The German word <em>Helferlein</em> means something like <em>little helper</em>;
please pronounce it like <code>hell-fur-line</code>.</p>
<p dir="auto">The package follows mainly the Knet-style; i.e. all networks can be trained with the Knet-iterators, all layers can be used together with Knet-style quickly-self-written layers, all Knet-networks can be trained with tb_train!(), all data providers can be used together, ...</p>
<p dir="auto">The high-level API makes its possible to define and train neural networks in less
lines of code and even more intuitively as with high-level (Python-) frameworks.</p>
<p dir="auto">See documentation and examples for a first intro.</p>
