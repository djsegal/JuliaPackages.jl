<p dir="auto">Wflow is a <a href="https://julialang.org/" rel="nofollow">Julia</a> package that provides a hydrological modeling
framework, as well as several different vertical and lateral concepts that can be used to
run hydrological simulations. It is a continuation of the work available
<a href="https://github.com/openstreams/wflow">here</a>.</p>
