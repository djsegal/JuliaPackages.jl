<p dir="auto">SaguaroTrader.jl is a modular schedule-driven backtesting library for long-short equities and ETF based systematic trading strategies. This library originated as a Julia port of the Python library <a href="https://github.com/mhallsmoore/qstrader">qstrader</a>. The library is currently experimental, and is under active development.</p>
