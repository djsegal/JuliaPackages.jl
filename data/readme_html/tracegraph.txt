<pre><code>using TraceGraph
using GraphPlot
</code></pre>
<p>Declare your functions:</p>
<pre><code>function foo(a)
  b = a+10
  c = b+10
  return (a,b,c)
end
</code></pre>
<p>Call <code>generategraph</code> as follows</p>
<pre><code>tg = tracegraph(foo, 10)
</code></pre>
<p><code>tg</code> is an object of type <code>TGraph</code> which consists of following fields:</p>
<ul>
<li><code>:g</code> : Object of type SimpleDiGraph</li>
<li><code>:nodelabel</code> : Vector of strings of unique names to the names</li>
<li><code>:nodeop</code> : Vector consisting of operation of each node</li>
<li><code>:nodevalue</code> : Vector of values of each node</li>
</ul>
<p>Then call <code>gplot</code> or <code>gplothtml</code> to plot the graph</p>
<pre><code>gplot(tg.g, nodelabel = tg.nodelabel)
#OR
gplothtml(tg.g, nodelabel = tg.nodelabel)
</code></pre>
