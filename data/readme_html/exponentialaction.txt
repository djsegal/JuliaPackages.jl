<p dir="auto">ExponentialAction is a lightweight package that implements the action of the <a href="https://en.wikipedia.org/wiki/Matrix_exponential" rel="nofollow">Matrix exponential</a> using the algorithm of Al-Mohy and Higham <sup><a href="#user-content-fn-AlMohyHigham2011-23993d15ddd85744c6e7a6d39e875f87" id="user-content-fnref-almohyhigham2011-23993d15ddd85744c6e7a6d39e875f87" data-footnote-ref="" aria-describedby="footnote-label">1</a></sup><sup><a href="#user-content-fn-Expmv-23993d15ddd85744c6e7a6d39e875f87" id="user-content-fnref-expmv-23993d15ddd85744c6e7a6d39e875f87" data-footnote-ref="" aria-describedby="footnote-label">2</a></sup>.</p>
<p dir="auto">The API of this package is a single function <code>expv(t, A, B)</code> that computes <code>exp(t * A) * B</code> for a scalar <code>t</code>, matrix <code>A</code>, and matrix or vector <code>B</code>, without computing <code>exp(t * A)</code>.
For large matrices <code>A</code>, this is significantly less expensive than calling <code>exp(t * A) * B</code> directly.</p>
<p dir="auto">See the <a href="https://sethaxen.github.io/ExponentialAction.jl/stable" rel="nofollow">Documentation</a> for details.</p>
