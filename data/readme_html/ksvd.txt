<p dir="auto"><a href="https://en.wikipedia.org/wiki/K-SVD" rel="nofollow">K-SVD</a> is an algorithm for creating overcomplete dictionaries for sparse representations.</p>
<p dir="auto">This package implements:</p>
<ul dir="auto">
<li>K-SVD as described in the original paper: <a href="http://www.cs.technion.ac.il/~freddy/papers/120.pdf" rel="nofollow">K-SVD: An Algorithm for Designing Overcomplete Dictionaries for Sparse Representation</a></li>
<li><a href="https://en.wikipedia.org/wiki/Matching_pursuit" rel="nofollow">Matching Pursuit</a> for representing signals using a given dictionary.</li>
</ul>

<p dir="auto">Assume that each column of Y represents a feature vector (or an input signal from some system).<br>
D is a dictionary. Each column of D represents an atom.<br>
K-SVD derives D and X such that DX â‰ˆ Y from only Y.</p>

<p dir="auto"><a href="https://en.wikipedia.org/wiki/Matching_pursuit" rel="nofollow">Matching Pursuit</a> derives X from D and Y such that DX = Y in constraint that X be as sparse as possible.</p>
