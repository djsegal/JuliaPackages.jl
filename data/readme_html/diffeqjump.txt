<p>DiffEqJump.jl is a component package in the <a href="https://sciml.ai/" rel="nofollow">SciML</a> ecosystem. It
holds the utilities for building jump equations, like stochastic simulation algorithms (SSAs), Gillespie methods or Kinetic Monte Carlo methods; and for building jump
diffusions. It is one of the core solver libraries included in <a href="https://github.com/JuliaDiffEq/DifferentialEquations.jl">DifferentialEquations.jl</a>.
Users interested in using this functionality should see the
<a href="https://diffeq.sciml.ai/latest/" rel="nofollow">DifferentialEquations.jl documentation</a>.</p>
