<p dir="auto">Julia wrapper for <a href="https://github.com/Cyan4973/xxHash">xxHash</a> C library.<br>
Initial xxHash algorithm has variants for 32 and 64 bit hashes.<br>
The XXH3 algorithm is faster and supports 64 and 128 bit hashes.<br>
All hash variants can be used directly, or incrementally using stream states.</p>
