<p><strong>Kinetic</strong> is a lightweight <a href="https://julialang.org" rel="nofollow">Julia</a> toolbox for the study of computational fluid dynamics and scientific machine learning.
Depending on the focus, the main module is divided into portable components:</p>
<ul>
<li><a href="https://github.com/vavrines/KitBase.jl">KitBase.jl</a>: basic physics and numerics</li>
<li><a href="https://github.com/vavrines/KitML.jl">KitML.jl</a>: neural dynamics and machine learning</li>
</ul>
<p>The high-performance Fortran library is embedded in <a href="https://github.com/vavrines/KitFort.jl">KitFort.jl</a>.
Although not the default option, it can be manually imported into the current ecosystem seamlessly when the ultimate executing efficiency is pursued.
A Python wrapper <a href="https://github.com/vavrines/kineticpy">kineticpy</a> has been built as well to call the structs and methods remotely through <a href="https://github.com/JuliaPy/pyjulia">pyjulia</a>.</p>

<p>Kinetic.jl is interested in theoretical and numerical studies of many-particle systems of gases, photons, plasmas, neutrons, etc.
It employs the finite volume method (FVM) to conduct 1-3 dimensional numerical simulations on CPUs and GPUs.
Any advection-diffusion-type equation can be solved within the framework.
Special attentions have been paid on Hilbert's sixth problem, i.e. to build the numerical passage between kinetic theory of gases, e.g. the Boltzmann equation</p>

<p>A partial list of current supported models and equations include</p>
<ul>
<li>linear Boltzmann equation</li>
<li>nonlinear Boltzmann equation</li>
<li>multi-component Boltzmann equations</li>
<li>Fokker-Planck-Landau equation</li>
<li>direct simulation Monte Carlo</li>
<li>advection-diffusion equation</li>
<li>Burgers equation</li>
<li>Euler equations</li>
<li>Navier-Stokes equations</li>
<li>Extended hydrodynamical equations from gas kinetic expansion</li>
<li>Magnetohydrodynamical equations</li>
<li>Maxwell's equations</li>
</ul>
