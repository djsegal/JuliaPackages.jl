<p>This package is designed to make it easier for Julia packages to build binary dependencies that use CMake. It automatically downloads a modern version of CMake (3.X.Y, instead of the Ubuntu 14.04 default of 2.8.12), and it exports a constant <code>cmake</code> giving the path of this executable.</p>
<p>For a <a href="https://github.com/JuliaLang/BinDeps.jl">BinDeps.jl</a>-compatible <code>CMakeProcess</code> class for automatically building CMake dependencies,
see the <a href="https://github.com/JuliaPackaging/CMakeWrapper.jl">CMakeWrapper.jl</a> package.</p>
