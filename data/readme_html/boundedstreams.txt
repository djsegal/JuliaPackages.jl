<p dir="auto">The <code>BoundedStreams</code> package describe a defined area in a source stream, which is defined
by an offset in the source stream and a length.</p>
<p dir="auto">The <code>BoundedStream</code> objects may be understood as views to a section of their source streams.</p>
<p dir="auto">All feasible access to stream (interface  <code>IO</code>) is supported.</p>

<p dir="auto">The public API of <code>BoundedStreams</code> includes the structure:</p>
<ul dir="auto">
<li><code>BoundedInputStream  &lt;: IO</code> — defines an bounded input stream in its initial state</li>
<li><code>BoundedOutputStream &lt;: IO</code> — defines an bounded output stream in its initial state</li>
</ul>
<p dir="auto">and corresponding construcors. All access is via the <code>IO</code> functions (<code>read</code>/<code>write</code>,
<code>skip</code>, <code>seek</code>, <code>mark</code>, <code>reset</code>, <code>isreadable</code>, <code>iswritable</code>, <code>eof</code>,
<code>close</code>, <code>position</code>, <code>bytesavailable</code>). They may be wrapped in other wrapping streams
as well. Some functions may be restricted due to backing the source stream.</p>

<div class="snippet-clipboard-content notranslate position-relative overflow-auto" data-snippet-clipboard-copy-content="BoundedInputStream(source::IO, nbytes::Integer; offset=0, close=nbytes)
BoundedOutputStream(source::IO, nbytes::Integer; offset=0, close=nbytes)"><pre class="notranslate"><code>BoundedInputStream(source::IO, nbytes::Integer; offset=0, close=nbytes)
BoundedOutputStream(source::IO, nbytes::Integer; offset=0, close=nbytes)
</code></pre></div>
<p dir="auto">Provide the <code>IO</code> interface for reading/writing the source stream <code>source</code>. Restrict the
number of bytes to to <code>nbytes</code>.</p>
<p dir="auto">The optional integer argument <code>offset</code> shifts the starting point off the
current position of the source stream.</p>
<p dir="auto">The optional argument <code>close</code> determines the position of the source stream after
this stream is closed. The special value <code>BoundedStreams.CLOSE</code> closes
the source stream in this case.</p>
