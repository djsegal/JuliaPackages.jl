<p dir="auto">A thin Julia wrapper of <a href="http://hmmer.org" rel="nofollow">hmmer</a>. Functions have the same name as the HMMER (and Easel miniapps) programs, and keyword arguments are named consistently with the command line arguments of the respective programs. The package creates temporary files to save output and returns their path.</p>
