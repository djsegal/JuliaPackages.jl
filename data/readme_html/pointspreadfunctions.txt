<p dir="auto">Toolbox for calculating optical PointSpreadFunctions (PSFs). It incorporates aberrations (specified though Zernike coefficients) and supports a number of
models for calculation (<code>PointSpreadFunctions.MethodSincR</code>, <code>PointSpreadFunctions.MethodPropagate</code>, <code>PointSpreadFunctions.MethodPropagateIterative</code>), some are faster and some more accurate.</p>
<p dir="auto">For detailed documentation, see the docs/dev (or docs/stable) link below.
You may also find the file <code>PSFs_demo.ipnb</code> in the examples folder helpful.</p>
