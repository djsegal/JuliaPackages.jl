<p>PackageCompatUI is a terminal text interface to the <code>[compat]</code> section
of a Julia <code>Project.toml</code> file.</p>

<ul>
<li>Red: No compat information available for the package.</li>
<li>Yellow: Compat declared for some versions of the package, but not the latest.</li>
<li>Green: Compatible with the latest version of the package.</li>
<li>Gray: Package is not registered.</li>
</ul>
