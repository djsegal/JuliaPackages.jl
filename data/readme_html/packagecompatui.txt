<p dir="auto">PackageCompatUI is a terminal text interface to the <code>[compat]</code> section
of a Julia <code>Project.toml</code> file.</p>

<p dir="auto"><em>Note</em>: add PackageCompatUI to your default environment, not to the
project you want to set compat for.</p>
<p dir="auto"><em>Note</em>: As of Julia v1.8, the <code>pkg&gt;</code>-REPL-mode
<a href="https://pkgdocs.julialang.org/dev/repl/#repl-compat" rel="nofollow">natively supports editing of the compat section</a>.
This is very rudimentary compared to PackageCompatUI but starts faster.</p>
