<p dir="auto">This package allows creating, fetching, and parsing TLEs (two-line elements).</p>

<p dir="auto">The TLE, or two-line elements, is a fixed-width format that express the mean
elements of a object in Earth's orbit. They are used as input for the Simplified
General Perturbation Model 4 (SGP4 / SDP4) to propagate satellite orbits.</p>
<p dir="auto">For more information about the TLE, see
<a href="https://en.wikipedia.org/wiki/Two-line_element_set" rel="nofollow">Two-line element set</a>.</p>
