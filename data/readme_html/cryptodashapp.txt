<p dir="auto">Interactive visualization of historical price (currently only in EUR), health metrics, candlestick
and volume data for various cryptocurrencies. Data are obtained via API queries to Alpha Vantage.
The app is written in Julia and makes use of Dash.jl along with its react framework to generate a
dashboard style view accessible via a browser. Different moving averages are also calculated and
plotted along with the daily average price data. Averaging window can be selected by the user.</p>
<p dir="auto">Functionality of the app will be extended in the future by adding more technical indicators.
<strong>Contributions are most welcome!</strong></p>

<p dir="auto">This package is available via the General registry. Add it to your working Julia environment by
doing the following in the REPL:</p>
<ul dir="auto">
<li>Press ']' to enter Pkg prompt</li>
</ul>
<div class="highlight highlight-source-julia notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="pkg&gt; add CryptoDashApp"><pre>pkg<span class="pl-k">&gt;</span> add CryptoDashApp</pre></div>
<p dir="auto">Once all the packages are downloaded, you can make use of the <strong>CryptoDashApp</strong> module from the
REPL itself or within a script by executing:</p>
<div class="highlight highlight-source-julia notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="julia&gt; using CryptoDashApp
julia&gt; run_app(8056, &quot;your key&quot;) # where 8056 is the port"><pre>julia<span class="pl-k">&gt;</span> <span class="pl-k">using</span> CryptoDashApp
julia<span class="pl-k">&gt;</span> <span class="pl-c1">run_app</span>(<span class="pl-c1">8056</span>, <span class="pl-s"><span class="pl-pds">"</span>your key<span class="pl-pds">"</span></span>) <span class="pl-c"><span class="pl-c">#</span> where 8056 is the port</span></pre></div>
<p dir="auto">Free API key can be obtained from <a href="https://www.alphavantage.co/support/#api-key" rel="nofollow">here.</a></p>
<p dir="auto">Now, open a new tab in your browser and check if you can access <a href="http://0.0.0.0:8056/" rel="nofollow">http://0.0.0.0:8056/</a> or <a href="http://127.0.0.1:8056/" rel="nofollow">http://127.0.0.1:8056/</a></p>
<p dir="auto">First plot will take a few seconds to appear, wait patiently! If you are running this app for the
first time, or on a different date, new data will be retrieved from Alpha Vantage, and then saved
to CSV files on disk. Keep in mind that the free API key imposes a limit of five calls/minute,
which means you won't be able to load data for more than five currencies in quick succession. In
case you see an error, wait for a while and then try again.</p>
