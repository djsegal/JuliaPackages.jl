<p dir="auto">JuDGE stands for: Julia Decomposition for Generalized Expansion. Functionally,
it is a solver which leverages the syntax of the JuMP modelling language to
solve complex multi-stage (and multi-horizon) capacity expansion problems.</p>
<p dir="auto">JuDGE supports generalized set partitioning problems. This, however, is
experimental.</p>
<p dir="auto">Please see the <a href="https://EPOC-NZ.github.io/JuDGE.jl/" rel="nofollow">documentation</a>
for details about installing JuDGE.jl, and examples showing how to set up a
stochastic capacity example model using the JuDGE.jl package.</p>
<p dir="auto">For more details see our working paper: <a href="http://www.optimization-online.org/DB_HTML/2020/11/8086.html" rel="nofollow">JuDGE.jl: a Julia package for optimizing capacity expansion</a>.</p>
