<p dir="auto">A Julia Package for handling Spherical Geometry. Spherical geometry is the
geometry of the two-dimensional surface of a sphere. This package only
handles geometries generated using great circle sections (arcs).</p>
<p dir="auto">Angles are specified in [deg].</p>
<p dir="auto">Types:</p>
<ul dir="auto">
<li>Point: specified by latitude ϕ [deg] and longitude λ [deg]</li>
<li>Arc: specified as the shortest great circle between two points.</li>
<li>Line: is a great circle line specified by a point and azimuth [deg]</li>
<li>Arcs: is a string of continuous line sections defined by a set of points.</li>
<li>Polygon: is a spherical polygon defined by a set of points.</li>
</ul>
<p dir="auto">It includes the calculation of:</p>
<ul dir="auto">
<li>The angular distance to a point, line, arc, multi-arc, or polygon border</li>
<li>The along line angular distance between a point and a line.</li>
<li>The intersection points between lines, arcs, multi-arcs, and polygon borders.</li>
<li>The self intersection points of multi-arcs and polygon borders</li>
<li>The bounding box of a given polygon or set of arcs.</li>
<li>The convexhull of a given polygon.</li>
<li>The normalized point</li>
<li>The (final) azimuth [deg] between two points</li>
<li>The spherical angle [deg] and spherical excess [deg] between three points</li>
<li>The midpoint between two points, of an arc, of an arcs</li>
<li>The intermediate point at a given fraction between two points, of an arc, of an arcs</li>
<li>The destination point given a start point, a direction and distance.</li>
<li>The intersection points of arcs, and polygons</li>
<li>The self intersection points of arcs or a polygon</li>
<li>The highest/lowest latitude (point) of a great circle</li>
<li>The area of a polygon/spherical triangle given a radius</li>
</ul>
<p dir="auto">And testing if:</p>
<ul dir="auto">
<li>a point, arc, arcs or polygon is inside a polygon</li>
<li>a point is on a line, arc, arcs or polygon border (within tolerance)</li>
<li>a polygon/arcs is self-isselfintersecting</li>
<li>a polygon is simple, complex, convex or concave</li>
</ul>
