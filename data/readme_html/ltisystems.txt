<p>Core functionality for <strong>identifying</strong>, <strong>analyzing</strong> and <strong>designing</strong> control
systems in <code>Julia</code>.</p>
<h3><a id="user-content-description" class="anchor" aria-hidden="true" href="#description"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path fill-rule="evenodd" d="M7.775 3.275a.75.75 0 001.06 1.06l1.25-1.25a2 2 0 112.83 2.83l-2.5 2.5a2 2 0 01-2.83 0 .75.75 0 00-1.06 1.06 3.5 3.5 0 004.95 0l2.5-2.5a3.5 3.5 0 00-4.95-4.95l-1.25 1.25zm-4.69 9.64a2 2 0 010-2.83l2.5-2.5a2 2 0 012.83 0 .75.75 0 001.06-1.06 3.5 3.5 0 00-4.95 0l-2.5 2.5a3.5 3.5 0 004.95 4.95l1.25-1.25a.75.75 0 00-1.06-1.06l-1.25 1.25a2 2 0 01-2.83 0z"></path></svg></a>Description</h3>
<p>This repository is meant to provide a basic set of tools, <em>i.e.</em>, <em>transfer
function</em> and <em>state space</em> types as well as the basic mathematical operations
defined on them, in a way that other tools for <a href="https://github.com/JuliaSystems/IdentificationToolbox.jl">identifying</a> or
<a href="https://github.com/JuliaSystems/ControlToolbox.jl">analyzing</a> control systems would use the same set of functionality
in a transparent, coherent way.</p>
