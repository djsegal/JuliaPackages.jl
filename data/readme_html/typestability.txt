<p dir="auto">This package provides functions to automate checking functions for type stability.  The checks are only run when enabled, which allows the function signatures that need to perform well to be located with the actual code without hurting runtime performance.</p>
<p dir="auto">TypeStability.jl is tested on Julia versions 0.6 through 1.1.</p>

<p dir="auto">Run <code>Pkg.add(TypeStability)</code> to install the latest stable version of TypeStability.  Then TypeStability can be <code>using</code>ed or <code>import</code>ed.</p>

<p dir="auto">The function <code>enable_inline_stability_checks(::Bool)</code> enables running the stability checks, while the macro <code>@stablefunction(signatures, function)</code> handles running the checks.</p>
