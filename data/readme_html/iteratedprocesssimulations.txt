<p dir="auto"><code>IteratedProcessSimulations.jl</code> is a package which makes it easier to orchestrate generative statistical models (mockups of data generating processes) machine learning models and decision rules in an iterated framework so that the effects of digital learning systems can be more easily studied.</p>
<p dir="auto">An example of how to use the package is provided in the following <a href="https://www.jeremiahlewis.me/IteratedProcessSimulations.jl/dev/Vignette-Recommender/" rel="nofollow">vignette</a>.</p>
