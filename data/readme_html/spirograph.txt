<p dir="auto">Julia implementation of the classic <a href="https://en.wikipedia.org/wiki/Spirograph" rel="nofollow">Spirograph</a> toy.</p>
<h2 dir="auto"><a id="user-content-creating-spirograph-drawings" class="anchor" aria-hidden="true" href="#creating-spirograph-drawings"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path d="m7.775 3.275 1.25-1.25a3.5 3.5 0 1 1 4.95 4.95l-2.5 2.5a3.5 3.5 0 0 1-4.95 0 .751.751 0 0 1 .018-1.042.751.751 0 0 1 1.042-.018 1.998 1.998 0 0 0 2.83 0l2.5-2.5a2.002 2.002 0 0 0-2.83-2.83l-1.25 1.25a.751.751 0 0 1-1.042-.018.751.751 0 0 1-.018-1.042Zm-4.69 9.64a1.998 1.998 0 0 0 2.83 0l1.25-1.25a.751.751 0 0 1 1.042.018.751.751 0 0 1 .018 1.042l-1.25 1.25a3.5 3.5 0 1 1-4.95-4.95l2.5-2.5a3.5 3.5 0 0 1 4.95 0 .751.751 0 0 1-.018 1.042.751.751 0 0 1-1.042.018 1.998 1.998 0 0 0-2.83 0l-2.5 2.5a1.998 1.998 0 0 0 0 2.83Z"></path></svg></a>Creating Spirograph Drawings</h2>
<p dir="auto">To make a spirograph drawing, use the function</p>
<div class="highlight highlight-source-julia notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="spirograph(a,b,offset,args...)"><pre><span class="pl-c1">spirograph</span>(a,b,offset,args<span class="pl-k">...</span>)</pre></div>
<p dir="auto">where</p>
<ul dir="auto">
<li><code>a</code> is the radius of the fixed wheel,</li>
<li><code>b</code> is the radius of the moving wheel, and</li>
<li><code>offset</code> is the distance from the center of the moving wheel to the pen location.</li>
</ul>
<p dir="auto">Here <code>a</code> and <code>b</code> are integers. The argument <code>b</code> may be negative, in which case the moving wheel glides along the inside of the fixed wheel. This is shown in the following diagram in which the red dot shows the pen location.</p>
<p dir="auto"><a target="_blank" rel="noopener noreferrer" href="diagram.png"><img src="diagram.png" alt="" style="max-width: 100%;"></a></p>
<p dir="auto">See also <a href="diagram2.pdf">this diagram</a> that shows the spirograph with <code>a=14</code>, <code>b=-5</code>, and <code>offset=4</code> overlayed with a picture of the two wheels with a red dot for the pen location. (This picture was created using the <code>mechanism</code> function defined in the file <code>diagram/mechanism.jl</code>.) See also <a href="animation.gif">this animation</a>.</p>

<p dir="auto">We also provide the function <code>spirograph!</code> that operates exactly like <code>spirograph</code> but does not first clear the drawing window. In this way, multiple spirograph images can be combined.</p>
