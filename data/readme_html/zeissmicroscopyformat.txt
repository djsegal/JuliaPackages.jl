<p dir="auto">This package provides support for the <a href="https://www.zeiss.com/microscopy/int/products/microscope-software/zen/czi.html" rel="nofollow">CZI image format for microscopy</a> in the Julia programming language. It is registered with the <a href="https://github.com/JuliaIO/FileIO.jl">FileIO</a> package.</p>
