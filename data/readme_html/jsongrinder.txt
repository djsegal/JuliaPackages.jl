<p dir="auto"><strong>JsonGrinder</strong> is a collection of routines that facilitates conversion of JSON documents into structures used by
<a href="https://github.com/CTUAvastLab/Mill.jl"><code>Mill.jl</code></a> project.</p>
<p dir="auto">It provides schema estimation from data, extraction of various data types to numeric representation with
reasonable defaults, suggestion of NN model structure based on data and interactive HTML visualization of estimated schema.
For more details, see <a href="https://CTUAvastLab.github.io/JsonGrinder.jl/stable" rel="nofollow">the documentation</a>.</p>
<p dir="auto"><a href="https://www.youtube.com/watch?v=Bf0CvltIDbE" rel="nofollow"><strong>Watch our introductory talk from JuliaCon 2021</strong> </a></p>
