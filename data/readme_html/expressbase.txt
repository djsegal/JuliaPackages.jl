<p dir="auto">This package defines some basic types and functions used by the
<a href="https://github.com/MineralsCloud/Express.jl"><code>Express.jl</code></a> package.</p>
<p dir="auto">The code is <a href="https://github.com/MineralsCloud/ExpressBase.jl">hosted on GitHub</a>,
with some continuous integration services to test its validity.</p>
<p dir="auto">This repository is created and maintained by <a href="https://github.com/singularitti">@singularitti</a>.
You are very welcome to contribute.</p>
