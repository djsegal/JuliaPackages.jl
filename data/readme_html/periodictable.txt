<p dir="auto">PeriodicTable.jl provides a Julia interface to a small database of element
properties for all of the elements in the periodic table. In particular <code>PeriodicTable</code> exports a global variable called <code>elements</code>, which is a collection of
<code>Element</code> data structures.</p>

<p dir="auto">You can look up elements by name (case-insensitive)
via <code>elements["oxygen"]</code>, by symbol via <code>elements[:O]</code>, or by number via
<code>elements[8]</code>, for example.</p>
<p dir="auto">Each element has fields <code>name</code>, <code>appearance</code>, <code>atomic_mass</code>, <code>boil</code>, <code>category</code>, <code>color</code>, <code>density</code>, <code>discovered_by</code>, <code>melt</code>, <code>molar_heat</code>, <code>named_by</code>, <code>number</code>, <code>period</code>, <code>phase</code>, <code>source</code>, <code>spectral_img</code>, <code>summary</code>, <code>symbol</code>, <code>xpos</code>, <code>ypos</code>, <code>shells</code>.</p>
<p dir="auto">All physical quantities are <a href="https://github.com/ajkeller34/Unitful.jl">unitful</a>.</p>
<p dir="auto">The data is pretty-printed when you look up an element in the Julia REPL.
For example:</p>
