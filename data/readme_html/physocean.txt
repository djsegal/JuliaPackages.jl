<p>Your need <a href="http://julialang.org" rel="nofollow">Julia</a> to use <code>PhysOcean</code>. The command line version is sufficient for <code>PhysOcean</code>.
Inside Julia, you can download and install the package by issuing:</p>
<div class="highlight highlight-source-julia"><pre><span class="pl-k">using</span> Pkg
Pkg<span class="pl-k">.</span><span class="pl-c1">add</span>(<span class="pl-s"><span class="pl-pds">"</span>PhysOcean<span class="pl-pds">"</span></span>)</pre></div>
<p>Or if you want to use the latest version, you can use the following command:</p>
<div class="highlight highlight-source-julia"><pre><span class="pl-k">using</span> Pkg
Pkg<span class="pl-k">.</span><span class="pl-c1">add</span>(<span class="pl-c1">PackageSpec</span>(name<span class="pl-k">=</span><span class="pl-s"><span class="pl-pds">"</span>PhysOcean<span class="pl-pds">"</span></span>, rev<span class="pl-k">=</span><span class="pl-s"><span class="pl-pds">"</span>master<span class="pl-pds">"</span></span>))</pre></div>
