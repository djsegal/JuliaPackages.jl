<p dir="auto">A quick introduction was given at the JuliaCon 2021.</p>
<p dir="auto"><a href="https://www.youtube.com/watch?v=qYgJDb_Ko2E" rel="nofollow"><img src="docs/src/assets/julia_con.jpg" width="300" style="max-width: 100%;"></a></p>
<p dir="auto">The main features are:</p>
<ul dir="auto">
<li>sinc interpolation allows to up and downsample a (bandlimited) signal</li>
<li>FFT based convolutions</li>
<li>array/image rotation</li>
<li>array/image shifting (including noteworthy subpixel shifts)</li>
<li>array/image shearing</li>
<li>convenient wrappers of <a href="https://github.com/JuliaMath/NFFT.jl">NFFT.jl</a></li>
<li>several tools like <code>ffts</code>, <code>ft</code>, <code>fftshift_view</code> etc. allowing simpler use with Fourier transforms</li>
<li>Chirp Z-Transform</li>
<li>Fractional Fourier Transform</li>
<li>reexports <a href="https://github.com/JuliaMath/FFTW.jl">FFTW.jl</a></li>
</ul>
<p dir="auto">Have a look in the <a href="examples/">examples folder</a> for interactive examples. The <a href="https://bionanoimaging.github.io/FourierTools.jl/dev/" rel="nofollow">documentation</a> offers a quick overview.</p>

<p dir="auto">By default we set 4 Threads. Use <code>FFTW.set_num_threads(N)</code> to set <code>N</code> threads.</p>

<p dir="auto">There are numerous packages related to Fourier transforms which offer similar functions or which this package is based on:</p>
<ul dir="auto">
<li><a href="https://github.com/JuliaMath/FFTW.jl">FFTW.jl</a> for FFTs</li>
<li><a href="https://github.com/JuliaMath/NFFT.jl">NFFT.jl</a> for non-uniform FFTs</li>
<li><a href="https://github.com/SciFracX/FractionalTransforms.jl">FractionalTransforms.jl</a> offers 1D fractional Fourier transforms</li>
</ul>
