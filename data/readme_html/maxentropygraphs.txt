<p dir="auto">A Julia package to use maximum entropy networks using the JuliaGraphs ecosystem.</p>
<p dir="auto">It provides the following functionalities (currently only for the UBCM and DBCM model):</p>
<ul dir="auto">
<li>sampling from a networks ensemble</li>
<li>computing expected value and variance for any metric that can be defined using the adjacence matrix of the network. This is done by using the automatic differentiation capabilities of Julia. In doing so, there is no need to compute the gradient of the metric explicitly and its also tends to be faster than doing it manually.</li>
</ul>
<p dir="auto">The computation of the parameters that maximize the likelihood is done using the <a href="https://nemtropy.readthedocs.io/en/master/index.html" rel="nofollow">NEMtropy package</a> (making use of PyCall).</p>
<p dir="auto">More detailed information is available in the documentation and some demo notebooks are available in the examples subfolder. These can be run locally, or you can just look at the html rendering of the notebook.</p>
