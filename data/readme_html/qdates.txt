<p dir="auto">The <strong>QDates</strong> package provides Japanese "Kyūreki (旧暦)" calendrical calculations into Julia.<br>
"Kyūreki (旧暦)" is one of the Lunisolar Calendar that has been once used in Japan.</p>

<p dir="auto">To install the release version, simply run on the Julia Pkg REPL-mode:</p>
<div class="highlight highlight-source-julia notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="pkg&gt; add QDates"><pre>pkg<span class="pl-k">&gt;</span> add QDates</pre></div>
<p dir="auto">To install the latest development version, run the following command instead:</p>
<div class="highlight highlight-source-julia notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="pkg&gt; add QDates#master"><pre>pkg<span class="pl-k">&gt;</span> add QDates<span class="pl-c"><span class="pl-c">#</span>master</span></pre></div>
<p dir="auto">Then you can run the built-in unit tests with</p>
<div class="highlight highlight-source-julia notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="pkg&gt; test QDates"><pre>pkg<span class="pl-k">&gt;</span> test QDates</pre></div>
<p dir="auto">to verify that everything is functioning properly on your machine.</p>

<p dir="auto"><code>QDates</code> has APIs almost compatible with the standard <code>Dates</code> package.</p>
<div class="highlight highlight-source-julia notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="using QDates

### construct Kyūreki Date
qdt = QDate(2017, 5, 1)
# =&gt; 旧2017年05月01日

### get year/month/day values
using Dates
Dates.yearmonthday(qdt)
# =&gt; (2017,5,1)

### check leapmonth or not
QDates.isleapmonth(qdt)
# =&gt; false

### arithmetic
qdt1 = qdt + Dates.Month(1)
# =&gt; 旧2017年閏05月01日

### check leapmonth or not
QDates.isleapmonth(qdt1)
# =&gt; true

### get year/month/isleapmonth/day values
QDates.yearmonthleapday(qdt1)
# =&gt; (2017,5,true,1)

### comparison (with constructing leapmonth-date)
qdt1 == QDate(2017, 5, true, 1)
# =&gt; true

### conversion to Gregorian Date
dt = Date(qdt)
# =&gt; 2017-05-26

### conversion from Gregorian Date
qdt0 = QDate(dt)
# =&gt; 旧2017年05月01日

### get today
qtoday = QDates.today()
# =&gt; 旧2017年04月10日
qtoday == QDate(Dates.today())
# =&gt; true

### six-day week system
[QDates.dayname(d) for d=qdt:QDate(2017,5,7)]
# =&gt; 7-element Array{String,1}:
#     &quot;大安&quot;
#     &quot;赤口&quot;
#     &quot;先勝&quot;
#     &quot;友引&quot;
#     &quot;先負&quot;
#     &quot;仏滅&quot;
#     &quot;大安&quot;

[d for d=qdt:qdt1 if QDates.is大安(d)]
# =&gt; 6-element Array{QDates.QDate,1}:
#     旧2017年05月01日
#     旧2017年05月07日
#     旧2017年05月13日
#     旧2017年05月19日
#     旧2017年05月25日
#     旧2017年閏05月01日

### month names (old-fashioned)
[QDates.monthname(m) for m=QDate(2017,1):Dates.Month(1):QDate(2017,12)]
# =&gt; 13-element Array{String,1}:
#     &quot;睦月&quot;
#     &quot;如月&quot;
#     &quot;弥生&quot;
#     &quot;卯月&quot;
#     &quot;皐月&quot;
#     &quot;閏皐月&quot; # &lt;- means leapmonth(&quot;閏月&quot;) of &quot;皐月&quot;
#     &quot;水無月&quot;
#     &quot;文月&quot;
#     &quot;葉月&quot;
#     &quot;長月&quot;
#     &quot;神無月&quot;
#     &quot;霜月&quot;
#     &quot;師走&quot;

### The number of days in a month (29 or 30, not constant).
[QDates.daysinmonth(m) for m=QDate(2017,1):Dates.Month(1):QDate(2017,12)]
# =&gt; 13-element Array{Int64,1}:
#     29 # 1月
#     30 # 2月
#     29 # 3月
#     30 # 4月
#     29 # 5月
#     29 # 閏5月
#     30 # 6月
#     29 # 7月
#     30 # 8月
#     29 # 9月
#     30 # 10月
#     30 # 11月
#     30 # 12月

#      true # 2012
#     false # 2013
#      true # 2014
