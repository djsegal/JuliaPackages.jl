<p dir="auto">This package is a Julia frontend to the Fortran-based <em>High Order Hex-Quad Mesher</em>
(a.k.a. <a href="https://github.com/trixi-framework/HOHQMesh"><strong>HOHQMesh</strong></a>) created and developed by
<a href="https://www.math.fsu.edu/~kopriva/" rel="nofollow">David A. Kopriva</a>. It augments HOHQMesh with
<a href="https://trixi-framework.github.io/HOHQMesh.jl/stable/interactive_overview/" rel="nofollow">interactive functionality</a>
that gives a user the ability to create, visualize,
and generate high-order meshes.
It further allows one to seamlessly integrate meshes generated by HOHQMesh into a Julia-based simulation workflow.
For example, running a simulation on an unstructured quadrilateral mesh
with <a href="https://trixi-framework.github.io/Trixi.jl/stable/tutorials/hohqmesh_tutorial/" rel="nofollow">Trixi.jl</a>.
HOHQMesh.jl is available on Linux, MacOS, and Windows.</p>
<h2 dir="auto"><a id="user-content-installation" class="anchor" aria-hidden="true" href="#installation"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path d="m7.775 3.275 1.25-1.25a3.5 3.5 0 1 1 4.95 4.95l-2.5 2.5a3.5 3.5 0 0 1-4.95 0 .751.751 0 0 1 .018-1.042.751.751 0 0 1 1.042-.018 1.998 1.998 0 0 0 2.83 0l2.5-2.5a2.002 2.002 0 0 0-2.83-2.83l-1.25 1.25a.751.751 0 0 1-1.042-.018.751.751 0 0 1-.018-1.042Zm-4.69 9.64a1.998 1.998 0 0 0 2.83 0l1.25-1.25a.751.751 0 0 1 1.042.018.751.751 0 0 1 .018 1.042l-1.25 1.25a3.5 3.5 0 1 1-4.95-4.95l2.5-2.5a3.5 3.5 0 0 1 4.95 0 .751.751 0 0 1-.018 1.042.751.751 0 0 1-1.042.018 1.998 1.998 0 0 0-2.83 0l-2.5 2.5a1.998 1.998 0 0 0 0 2.83Z"></path></svg></a>Installation</h2>
<p dir="auto">If you have not yet installed Julia, please <a href="https://julialang.org/downloads/platform/" rel="nofollow">follow the instructions for your
operating system</a>. HOHQMesh.jl works
with Julia v1.6 and above.</p>
<p dir="auto">HOHQMesh.jl is a registered Julia package. Hence, you can install it by executing
the following commands in the Julia REPL:</p>
<div class="highlight highlight-source-julia notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="julia&gt; import Pkg; Pkg.add(&quot;HOHQMesh&quot;)"><pre>julia<span class="pl-k">&gt;</span> <span class="pl-k">import</span> Pkg; Pkg<span class="pl-k">.</span><span class="pl-c1">add</span>(<span class="pl-s"><span class="pl-pds">"</span>HOHQMesh<span class="pl-pds">"</span></span>)</pre></div>
<p dir="auto">HOHQMesh.jl depends on the binary distribution of the
<a href="https://github.com/trixi-framework/HOHQMesh">HOHQMesh</a>
mesh generator, which is available via the Julia package
<a href="https://github.com/JuliaBinaryWrappers/HOHQMesh_jll.jl">HOHQMesh_jll.jl</a>
and which is automatically installed as a dependency.</p>
<h2 dir="auto"><a id="user-content-usage" class="anchor" aria-hidden="true" href="#usage"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path d="m7.775 3.275 1.25-1.25a3.5 3.5 0 1 1 4.95 4.95l-2.5 2.5a3.5 3.5 0 0 1-4.95 0 .751.751 0 0 1 .018-1.042.751.751 0 0 1 1.042-.018 1.998 1.998 0 0 0 2.83 0l2.5-2.5a2.002 2.002 0 0 0-2.83-2.83l-1.25 1.25a.751.751 0 0 1-1.042-.018.751.751 0 0 1-.018-1.042Zm-4.69 9.64a1.998 1.998 0 0 0 2.83 0l1.25-1.25a.751.751 0 0 1 1.042.018.751.751 0 0 1 .018 1.042l-1.25 1.25a3.5 3.5 0 1 1-4.95-4.95l2.5-2.5a3.5 3.5 0 0 1 4.95 0 .751.751 0 0 1-.018 1.042.751.751 0 0 1-1.042.018 1.998 1.998 0 0 0-2.83 0l-2.5 2.5a1.998 1.998 0 0 0 0 2.83Z"></path></svg></a>Usage</h2>
<p dir="auto">In the Julia REPL, you can load HOHQMesh with</p>
<div class="highlight highlight-source-julia notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="julia&gt; using HOHQMesh"><pre>julia<span class="pl-k">&gt;</span> <span class="pl-k">using</span> HOHQMesh</pre></div>
<p dir="auto">and then happily generate away!</p>
<p dir="auto">Two 2D examples <code>GingerbreadMan</code> and <code>NACA0012</code> and a 3D example <code>Snake</code> (all
from HOHQMesh itself) come delivered with this package. You can generate a
mesh for them by executing</p>
<div class="highlight highlight-source-julia notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="julia&gt; control_file = joinpath(HOHQMesh.examples_dir(), &quot;GingerbreadMan.control&quot;)

julia&gt; output = generate_mesh(control_file)"><pre>julia<span class="pl-k">&gt;</span> control_file <span class="pl-k">=</span> <span class="pl-c1">joinpath</span>(HOHQMesh<span class="pl-k">.</span><span class="pl-c1">examples_dir</span>(), <span class="pl-s"><span class="pl-pds">"</span>GingerbreadMan.control<span class="pl-pds">"</span></span>)

julia<span class="pl-k">&gt;</span> output <span class="pl-k">=</span> <span class="pl-c1">generate_mesh</span>(control_file)</pre></div>
<p dir="auto">You will then find the resulting output files (mesh, plot file, statistics) in
the designated output directory, which defaults to <code>out</code>. The
<code>GingerbreadMan.control</code> file will yield the following mesh,</p>
<p dir="auto"><a target="_blank" rel="noopener noreferrer nofollow" href="https://user-images.githubusercontent.com/3637659/117241938-80f4ee80-ae34-11eb-854a-ebebcd0b9d88.png"><img src="https://user-images.githubusercontent.com/3637659/117241938-80f4ee80-ae34-11eb-854a-ebebcd0b9d88.png" alt="gingerbreadman_with_edges_400px" style="max-width: 100%;"></a></p>
<p dir="auto">while the 3D file <code>Snake.control</code> produces this mesh:</p>
<p dir="auto"><a target="_blank" rel="noopener noreferrer nofollow" href="https://user-images.githubusercontent.com/3637659/117241963-8ce0b080-ae34-11eb-9b79-d091807d9a23.png"><img src="https://user-images.githubusercontent.com/3637659/117241963-8ce0b080-ae34-11eb-9b79-d091807d9a23.png" alt="snake_400px" style="max-width: 100%;"></a></p>
<p dir="auto">Examples scripts of interactive mesh generation tools are available in the
<a href="https://github.com/trixi-framework/HOHQMesh.jl/tree/main/examples">examples/</a> subdirectory.
These example scripts are prefaced with the phrase <code>interactive_</code>.
There is a brief summary at the top of each <code>interactive</code> example script that describes
the mesh it will create and the features of HOHQMesh it uses.
An example script can be executed from a Julia REPL session with an <code>include(...)</code> statement, e.g.,</p>
<div class="highlight highlight-source-julia notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="julia&gt; include(joinpath(HOHQMesh.examples_dir(), &quot;interactive_outer_box_two_circles.jl&quot;))"><pre>julia<span class="pl-k">&gt;</span> <span class="pl-c1">include</span>(<span class="pl-c1">joinpath</span>(HOHQMesh<span class="pl-k">.</span><span class="pl-c1">examples_dir</span>(), <span class="pl-s"><span class="pl-pds">"</span>interactive_outer_box_two_circles.jl<span class="pl-pds">"</span></span>))</pre></div>
<p dir="auto">The resulting output mesh and plot files are saved in the output directory <code>out</code> as
designated in the example script. Mesh statistics are printed to the screen.</p>
<p dir="auto">The interactive functionality uses <a href="https://github.com/JuliaPlots/Makie.jl/">Makie.jl</a>
to visualize the boundary curves and mesh from the interactive tool. A Makie backend, such as
<a href="https://github.com/JuliaPlots/GLMakie.jl/">GLMakie</a>, can
be loaded in addition to HOHQMesh</p>
<div class="highlight highlight-source-julia notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="julia&gt; using Pkg; Pkg.add(&quot;GLMakie&quot;)
julia&gt; using GLMakie"><pre>julia<span class="pl-k">&gt;</span> <span class="pl-k">using</span> Pkg; Pkg<span class="pl-k">.</span><span class="pl-c1">add</span>(<span class="pl-s"><span class="pl-pds">"</span>GLMakie<span class="pl-pds">"</span></span>)
julia<span class="pl-k">&gt;</span> <span class="pl-k">using</span> GLMakie</pre></div>
<p dir="auto">Now, running the example script produces a figure in addition to the mesh and plot
files that are saved and the output of mesh statistics to the screen.</p>
<p dir="auto"><a target="_blank" rel="noopener noreferrer nofollow" href="https://user-images.githubusercontent.com/25242486/174244295-40d31df3-981e-4375-bc3a-af0a43737710.png"><img src="https://user-images.githubusercontent.com/25242486/174244295-40d31df3-981e-4375-bc3a-af0a43737710.png" alt="box_two_circles" style="max-width: 100%;"></a></p>
<p dir="auto">Further explanation of the interactive functionality can be found <a href="https://trixi-framework.github.io/HOHQMesh.jl/stable/interactive_overview/" rel="nofollow">here</a>.
Additional examples are available in the <a href="https://trixi-framework.github.io/HOHQMesh.jl/stable/tutorials/introduction/" rel="nofollow">Tutorials</a>.</p>
