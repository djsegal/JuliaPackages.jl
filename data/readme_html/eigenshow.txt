<p dir="auto">Interactive demonstrator of eigenvectors and singular vectors for Julia</p>
<h2 dir="auto"><a id="user-content-usage" class="anchor" aria-hidden="true" href="#usage"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path d="m7.775 3.275 1.25-1.25a3.5 3.5 0 1 1 4.95 4.95l-2.5 2.5a3.5 3.5 0 0 1-4.95 0 .751.751 0 0 1 .018-1.042.751.751 0 0 1 1.042-.018 1.998 1.998 0 0 0 2.83 0l2.5-2.5a2.002 2.002 0 0 0-2.83-2.83l-1.25 1.25a.751.751 0 0 1-1.042-.018.751.751 0 0 1-.018-1.042Zm-4.69 9.64a1.998 1.998 0 0 0 2.83 0l1.25-1.25a.751.751 0 0 1 1.042.018.751.751 0 0 1 .018 1.042l-1.25 1.25a3.5 3.5 0 1 1-4.95-4.95l2.5-2.5a3.5 3.5 0 0 1 4.95 0 .751.751 0 0 1-.018 1.042.751.751 0 0 1-1.042.018 1.998 1.998 0 0 0-2.83 0l-2.5 2.5a1.998 1.998 0 0 0 0 2.83Z"></path></svg></a>Usage</h2>
<ol dir="auto">
<li>
<p dir="auto"><a href="https://julialang.org/download" rel="nofollow">Install Julia.</a> Version 1.7 or greater is recommended.</p>
</li>
<li>
<p dir="auto">Start Julia and enter the command</p>
<div class="highlight highlight-source-julia notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="]add EigenShow"><pre>]add EigenShow</pre></div>
<p dir="auto">It may take a few minutes to download and install the necessary packages.</p>
</li>
<li>
<p dir="auto">Enter the commands</p>
<div class="highlight highlight-source-julia notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="using EigenShow
eigenshow()"><pre><span class="pl-k">using</span> EigenShow
<span class="pl-c1">eigenshow</span>()</pre></div>
<p dir="auto">(The <code>using</code> command is needed only once per Julia session.) After 20 seconds or so, a new window should open with the demonstrator. It runs independently and can simply be closed when you are done with it.</p>
</li>
</ol>
<p dir="auto"><strong>In EVD mode.</strong> Initially you see a vector <math-renderer class="js-inline-math" style="display: inline" data-static-url="https://github.githubassets.com/static" data-run-id="57f407fe96c2a0d33bdd9d376d09d352">$x$</math-renderer> on the unit circle and, in a different color, the vector <math-renderer class="js-inline-math" style="display: inline" data-static-url="https://github.githubassets.com/static" data-run-id="57f407fe96c2a0d33bdd9d376d09d352">$Ax$</math-renderer> resulting from matrix-vector multiplication using the current matrix <math-renderer class="js-inline-math" style="display: inline" data-static-url="https://github.githubassets.com/static" data-run-id="57f407fe96c2a0d33bdd9d376d09d352">$A$</math-renderer> chosen in the selection box. As you move the mouse, <math-renderer class="js-inline-math" style="display: inline" data-static-url="https://github.githubassets.com/static" data-run-id="57f407fe96c2a0d33bdd9d376d09d352">$x$</math-renderer> moves around the unit circle and <math-renderer class="js-inline-math" style="display: inline" data-static-url="https://github.githubassets.com/static" data-run-id="57f407fe96c2a0d33bdd9d376d09d352">$Ax$</math-renderer> traces out an ellipse (or, in a degenerate case, a line segment). When <math-renderer class="js-inline-math" style="display: inline" data-static-url="https://github.githubassets.com/static" data-run-id="57f407fe96c2a0d33bdd9d376d09d352">$Ax$</math-renderer> is parallel to <math-renderer class="js-inline-math" style="display: inline" data-static-url="https://github.githubassets.com/static" data-run-id="57f407fe96c2a0d33bdd9d376d09d352">$x$</math-renderer>, then <math-renderer class="js-inline-math" style="display: inline" data-static-url="https://github.githubassets.com/static" data-run-id="57f407fe96c2a0d33bdd9d376d09d352">$x$</math-renderer> is an eigenvector of <math-renderer class="js-inline-math" style="display: inline" data-static-url="https://github.githubassets.com/static" data-run-id="57f407fe96c2a0d33bdd9d376d09d352">$A$</math-renderer>, and the (signed) ratio of the vector lengths is the associated eigenvalue. When <math-renderer class="js-inline-math" style="display: inline" data-static-url="https://github.githubassets.com/static" data-run-id="57f407fe96c2a0d33bdd9d376d09d352">$x$</math-renderer> is an eigenvector, so is <math-renderer class="js-inline-math" style="display: inline" data-static-url="https://github.githubassets.com/static" data-run-id="57f407fe96c2a0d33bdd9d376d09d352">$-x$</math-renderer>. The matrix may have zero, one, or two distinct real eigenvectors.</p>
<p dir="auto"><strong>In SVD mode.</strong> As you move the mouse, a perpendicular pair of vectors <math-renderer class="js-inline-math" style="display: inline" data-static-url="https://github.githubassets.com/static" data-run-id="57f407fe96c2a0d33bdd9d376d09d352">$x,y$</math-renderer> move around the unit circle. In a different color, you also see <math-renderer class="js-inline-math" style="display: inline" data-static-url="https://github.githubassets.com/static" data-run-id="57f407fe96c2a0d33bdd9d376d09d352">$Ax$</math-renderer> and <math-renderer class="js-inline-math" style="display: inline" data-static-url="https://github.githubassets.com/static" data-run-id="57f407fe96c2a0d33bdd9d376d09d352">$Ay$</math-renderer> for the current matrix <math-renderer class="js-inline-math" style="display: inline" data-static-url="https://github.githubassets.com/static" data-run-id="57f407fe96c2a0d33bdd9d376d09d352">$A$</math-renderer> chosen in the selection box. When <math-renderer class="js-inline-math" style="display: inline" data-static-url="https://github.githubassets.com/static" data-run-id="57f407fe96c2a0d33bdd9d376d09d352">$Ax$</math-renderer> and <math-renderer class="js-inline-math" style="display: inline" data-static-url="https://github.githubassets.com/static" data-run-id="57f407fe96c2a0d33bdd9d376d09d352">$Ay$</math-renderer> are perpendicular to each other, then <math-renderer class="js-inline-math" style="display: inline" data-static-url="https://github.githubassets.com/static" data-run-id="57f407fe96c2a0d33bdd9d376d09d352">$x$</math-renderer> and <math-renderer class="js-inline-math" style="display: inline" data-static-url="https://github.githubassets.com/static" data-run-id="57f407fe96c2a0d33bdd9d376d09d352">$y$</math-renderer> are right singular vectors of <math-renderer class="js-inline-math" style="display: inline" data-static-url="https://github.githubassets.com/static" data-run-id="57f407fe96c2a0d33bdd9d376d09d352">$A$</math-renderer>, <math-renderer class="js-inline-math" style="display: inline" data-static-url="https://github.githubassets.com/static" data-run-id="57f407fe96c2a0d33bdd9d376d09d352">$Ax$</math-renderer> and <math-renderer class="js-inline-math" style="display: inline" data-static-url="https://github.githubassets.com/static" data-run-id="57f407fe96c2a0d33bdd9d376d09d352">$Ay$</math-renderer> are left singular vectors of <math-renderer class="js-inline-math" style="display: inline" data-static-url="https://github.githubassets.com/static" data-run-id="57f407fe96c2a0d33bdd9d376d09d352">$A$</math-renderer>, and the (unsigned) ratios of lengths of <math-renderer class="js-inline-math" style="display: inline" data-static-url="https://github.githubassets.com/static" data-run-id="57f407fe96c2a0d33bdd9d376d09d352">$Ax$</math-renderer> to <math-renderer class="js-inline-math" style="display: inline" data-static-url="https://github.githubassets.com/static" data-run-id="57f407fe96c2a0d33bdd9d376d09d352">$x$</math-renderer> and <math-renderer class="js-inline-math" style="display: inline" data-static-url="https://github.githubassets.com/static" data-run-id="57f407fe96c2a0d33bdd9d376d09d352">$Ay$</math-renderer> to <math-renderer class="js-inline-math" style="display: inline" data-static-url="https://github.githubassets.com/static" data-run-id="57f407fe96c2a0d33bdd9d376d09d352">$y$</math-renderer> are associated singular values. Aside from the trivial duplications <math-renderer class="js-inline-math" style="display: inline" data-static-url="https://github.githubassets.com/static" data-run-id="57f407fe96c2a0d33bdd9d376d09d352">$x ↦ -x$</math-renderer> and <math-renderer class="js-inline-math" style="display: inline" data-static-url="https://github.githubassets.com/static" data-run-id="57f407fe96c2a0d33bdd9d376d09d352">$y ↦ -y$</math-renderer>, every real <math-renderer class="js-inline-math" style="display: inline" data-static-url="https://github.githubassets.com/static" data-run-id="57f407fe96c2a0d33bdd9d376d09d352">$2\times 2$</math-renderer> matrix has a unique pair of real right singular vectors.</p>
<p dir="auto">In either mode, you can click the mouse button to mark a point for future reference.</p>
