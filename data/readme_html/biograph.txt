<p>BioGraph is a Julia package for handle genome graph in the GFA format. It reads information from GFA input, extract simple bidirected graphs and find the longest linear path in those graphs.</p>
<p><a href="https://nguyetdang.github.io/BioGraph.jl/stable" rel="nofollow">Complete documentation is available here</a></p>
