<p dir="auto">To install run <code>Pkg.add("GaussianDistributions")</code>.</p>
<p dir="auto">This package creates a simplistic alternative to <code>Distributions</code> to generate Gaussian or normally distributed random variables and deal with their distributions.</p>
<p dir="auto">The <em>raison d’être</em> of GaussianDistributions is that the type hierarchy layed out by distributions is not well suited to handle singular covariance matrices, and anything other than <code>Float64</code>s and <code>Vector{Float64}</code>s.</p>
<p dir="auto">This package contains enough functionality such that <code>Gaussian</code>s can be useful as Gaussian state variable for example in a Kalman
filter.</p>
<p dir="auto">It also provides cumulative distributions functions (<code>cdf</code>) for bivariate Normal distributions using an approximation (valid unless the correlation <code>ρ</code> is very strong and <code>x ≈ y*sign(ρ)</code>.)</p>
