<p dir="auto"><a href="https://github.com/blegat/SwitchOnSafety.jl/blob/master/examples/Finiteness_conjecture_counterexample.ipynb"><img src="examples/FinConjCounterEx.png" height="360" style="max-width: 100%;"></a></p>
<p dir="auto">This packages implements methods for computing invariant sets using <a href="https://github.com/JuliaOpt/SumOfSquares.jl">Sum Of Squares Programming</a>.
It supports:</p>
<ul dir="auto">
<li>Systems defined in <a href="https://github.com/JuliaReach/MathematicalSystems.jl">MathematicalSystems.jl</a>.</li>
<li>Hybrid Systems defined in <a href="https://github.com/blegat/HybridSystems.jl">HybridSystems.jl</a>.</li>
</ul>
<p dir="auto">It also includes utilities for approximation the <a href="https://link.springer.com/book/10.1007%2F978-3-540-95980-9" rel="nofollow">Joint Spectral Radius</a>.</p>
