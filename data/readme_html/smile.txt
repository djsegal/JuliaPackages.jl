<p>A Julia wrapper for the <strong><a href="https://dslpitt.org/genie/" rel="nofollow">Smile C++ Structural Modeling, Inference, and Learning Engine</a></strong>. This grants access to a wide variety of graphical decision-theoretical methods, such as Bayesian Networks and influence diagrams.</p>

<p>Smile.jl is no longer needed as of Julia 0.5. Julians can use the <a href="https://github.com/Keno/Cxx.jl">Cxx</a> library to call the Smile C++ .so files directly. There is also the <a href="https://github.com/sisl/BayesNets.jl">BayesNets</a> package which provides native Julia implementations.</p>
