<p dir="auto"><a href="https://github.com/agerlach/Flowstar.jl/actions?query=workflow%3ACI"><img src="https://github.com/agerlach/Flowstar.jl/workflows/CI/badge.svg" alt="Build Status" style="max-width: 100%;"></a></p>
<p dir="auto">This package is a wrapper to <a href="flowstar.org">Flow*</a>, a verification tool for cyber-physical systems. Currently only continuous reachability is supported.</p>
