<p dir="auto">MicrostructureNoise is a <a href="https://github.com/JuliaLang/julia">Julia</a> package for Bayesian volatility estimation in presence of market microstructure noise.</p>

<p dir="auto">MicrostructureNoise estimates the volatility function <a href="https://www.codecogs.com/eqnedit.php?latex=s" rel="nofollow"><img src="https://camo.githubusercontent.com/081d41cce4e94f1aa34a76156267321d733cef7105aa188c8b251cf75ab98448/68747470733a2f2f6c617465782e636f6465636f67732e636f6d2f7376672e6c617465783f73" title="s" data-canonical-src="https://latex.codecogs.com/svg.latex?s" style="max-width: 100%;"></a> of the stochastic differential equation</p>
<p dir="auto"><a href="https://www.codecogs.com/eqnedit.php?latex=dX_t&amp;space;=&amp;space;b(t,X_t)&amp;space;dt&amp;space;+&amp;space;s(t)&amp;space;dW_t,&amp;space;\quad&amp;space;X_0&amp;space;=&amp;space;x_0,&amp;space;\quad&amp;space;t&amp;space;\in&amp;space;[0,T]" rel="nofollow"><img src="https://camo.githubusercontent.com/f59b9f3616fc25ae4b520645ebb2c03b5eaaeed0bd2b770ef416c5f9898f42d0/68747470733a2f2f6c617465782e636f6465636f67732e636f6d2f7376672e6c617465783f64585f742673706163653b3d2673706163653b6228742c585f74292673706163653b64742673706163653b2b2673706163653b732874292673706163653b64575f742c2673706163653b5c717561642673706163653b585f302673706163653b3d2673706163653b785f302c2673706163653b5c717561642673706163653b742673706163653b5c696e2673706163653b5b302c545d" title="dX_t = b(t,X_t) dt + s(t) dW_t, \quad X_0 = x_0, \quad t \in [0,T]" data-canonical-src="https://latex.codecogs.com/svg.latex?dX_t&amp;space;=&amp;space;b(t,X_t)&amp;space;dt&amp;space;+&amp;space;s(t)&amp;space;dW_t,&amp;space;\quad&amp;space;X_0&amp;space;=&amp;space;x_0,&amp;space;\quad&amp;space;t&amp;space;\in&amp;space;[0,T]" style="max-width: 100%;"></a></p>
<p dir="auto">from noisy observations of its solution</p>
<p dir="auto"><a href="https://www.codecogs.com/eqnedit.php?latex=Y_i&amp;space;=&amp;space;X(t_i)&amp;space;+&amp;space;V_i,&amp;space;\quad&amp;space;0&amp;space;&lt;&amp;space;t_1&amp;space;&lt;&amp;space;\ldots&amp;space;&lt;&amp;space;t_n&amp;space;=&amp;space;T," rel="nofollow"><img src="https://camo.githubusercontent.com/7934653c9f58350869322f459dc1f84a3610cdc4ec94272c9dafd24b21060733/68747470733a2f2f6c617465782e636f6465636f67732e636f6d2f6769662e6c617465783f595f692673706163653b3d2673706163653b5828745f69292673706163653b2b2673706163653b565f692c2673706163653b5c717561642673706163653b302673706163653b3c2673706163653b745f312673706163653b3c2673706163653b5c6c646f74732673706163653b3c2673706163653b745f6e2673706163653b3d2673706163653b542c" title="Y_i = X(t_i) + V_i, \quad 0 &lt; t_1 &lt; \ldots &lt; t_n = T," data-canonical-src="https://latex.codecogs.com/gif.latex?Y_i&amp;space;=&amp;space;X(t_i)&amp;space;+&amp;space;V_i,&amp;space;\quad&amp;space;0&amp;space;&lt;&amp;space;t_1&amp;space;&lt;&amp;space;\ldots&amp;space;&lt;&amp;space;t_n&amp;space;=&amp;space;T," style="max-width: 100%;"></a></p>
<p dir="auto">where <a href="https://www.codecogs.com/eqnedit.php?latex=\{V_i\}" rel="nofollow"><img src="https://camo.githubusercontent.com/87bc0bf44340d491369b802f2060449e0c6e207c5c6367a1494a1c3c5393ce41/68747470733a2f2f6c617465782e636f6465636f67732e636f6d2f7376672e6c617465783f5c7b565f695c7d" title="\{V_i\}" data-canonical-src="https://latex.codecogs.com/svg.latex?\{V_i\}" style="max-width: 100%;"></a> denote unobservable stochastic disturbances. The method is minimalistic in its assumptions on the volatility function, which in particular can itself be a stochastic process.</p>
<p dir="auto">The estimation methodology is intuitive to understand, given that its ingredients are well-known statistical techniques. The posterior inference is performed via the Gibbs sampler, with the Forward Filtering Backward Simulation algorithm used to reconstruct unobservable states <a href="https://www.codecogs.com/eqnedit.php?latex=X(t_i)" rel="nofollow"><img src="https://camo.githubusercontent.com/5d984a62fe037dafa349ab9d87d7fe841d7ff63d73365c9e8b4d7f8fb1f8838a/68747470733a2f2f6c617465782e636f6465636f67732e636f6d2f7376672e6c617465783f5828745f6929" title="X(t_i)" data-canonical-src="https://latex.codecogs.com/svg.latex?X(t_i)" style="max-width: 100%;"></a>. This relies on the Kalman filter. The unknown squared volatility function is a priori modelled as piecewise constant and is assigned the inverse Gamma Markov chain prior, which induces smoothing among adjacent pieces of the function. The picture below gives an idea of the results obtainable with the method. Depicted is a reconstruction of the volatility function from the synthetic data generated according to the classical Heston stochastic volatility model (the unobserved true volatility curve is plotted in red). Note that next to the point estimate (posterior mean plotted in black), the method conducts automatic uncertainty quantification via the marginal Bayesian credible band (plotted in blue).</p>
<p dir="auto"><a target="_blank" rel="noopener noreferrer" href="./heston.png"><img src="./heston.png" width="600" style="max-width: 100%;"></a></p>
<p dir="auto">When <a href="https://www.codecogs.com/eqnedit.php?latex=X(t_i)" rel="nofollow"><img src="https://camo.githubusercontent.com/5d984a62fe037dafa349ab9d87d7fe841d7ff63d73365c9e8b4d7f8fb1f8838a/68747470733a2f2f6c617465782e636f6465636f67732e636f6d2f7376672e6c617465783f5828745f6929" title="X(t_i)" data-canonical-src="https://latex.codecogs.com/svg.latex?X(t_i)" style="max-width: 100%;"></a> is observed without noise, an option (<code>fixeta</code>) allows to perfom inference as described in the reference "Nonparametric Bayesian volatility estimation".</p>
