<p>The filter computes the minima and/or maxima of an array over sliding window with a given size.</p>
<p>This is a port of the Lemire min max filter as implemented by Bruno Luong.<br>
<a href="http://arxiv.org/abs/cs.DS/0610046" rel="nofollow">http://arxiv.org/abs/cs.DS/0610046</a><br>
<a href="http://lemire.me/" rel="nofollow">http://lemire.me/</a><br>
<a href="http://www.mathworks.com/matlabcentral/fileexchange/24705-min-max-filter" rel="nofollow">http://www.mathworks.com/matlabcentral/fileexchange/24705-min-max-filter</a></p>

<h3><a id="user-content-three-dimensional-peak-finding" class="anchor" aria-hidden="true" href="#three-dimensional-peak-finding"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path fill-rule="evenodd" d="M4 9h1v1H4c-1.5 0-3-1.69-3-3.5S2.55 3 4 3h4c1.45 0 3 1.69 3 3.5 0 1.41-.91 2.72-2 3.25V8.59c.58-.45 1-1.27 1-2.09C10 5.22 8.98 4 8 4H4c-.98 0-2 1.22-2 2.5S3 9 4 9zm9-3h-1v1h1c1 0 2 1.22 2 2.5S13.98 12 13 12H9c-.98 0-2-1.22-2-2.5 0-.83.42-1.64 1-2.09V6.25c-1.09.53-2 1.84-2 3.25C6 11.31 7.55 13 9 13h4c1.45 0 3-1.69 3-3.5S14.5 6 13 6z"></path></svg></a>Three dimensional peak finding</h3>
<p>Read in three dimensional data, plot and find peaks.<br>
Uses Winston and EEG packages for plotting.</p>
<div class="highlight highlight-source-julia"><pre><span class="pl-k">using</span> Winston
<span class="pl-k">using</span> EEG

filen <span class="pl-k">=</span> <span class="pl-c1">matopen</span>(<span class="pl-c1">joinpath</span>(<span class="pl-c1">dirname</span>(<span class="pl-c1">@__FILE__</span>), <span class="pl-s"><span class="pl-pds">"</span>data<span class="pl-pds">"</span></span>, <span class="pl-s"><span class="pl-pds">"</span>3d_array.mat<span class="pl-pds">"</span></span>))
A <span class="pl-k">=</span> <span class="pl-c1">read</span>(filen, <span class="pl-s"><span class="pl-pds">"</span>image<span class="pl-pds">"</span></span>)
A <span class="pl-k">=</span> <span class="pl-c1">convert</span>(Array{FloatingPoint}, A)
<span class="pl-c1">close</span>(filen)

p <span class="pl-k">=</span> <span class="pl-c1">plot_dat</span>(A)

minval, maxval <span class="pl-k">=</span> <span class="pl-c1">minmax_filter</span>(A, <span class="pl-c1">6</span>, verbose<span class="pl-k">=</span><span class="pl-c1">false</span>)

matching <span class="pl-k">=</span> A[<span class="pl-c1">2</span><span class="pl-k">:</span><span class="pl-c1">size</span>(maxval)[<span class="pl-c1">1</span>]<span class="pl-k">+</span><span class="pl-c1">1</span>, <span class="pl-c1">2</span><span class="pl-k">:</span><span class="pl-c1">size</span>(maxval)[<span class="pl-c1">2</span>]<span class="pl-k">+</span><span class="pl-c1">1</span>, <span class="pl-c1">2</span><span class="pl-k">:</span><span class="pl-c1">size</span>(maxval)[<span class="pl-c1">3</span>]<span class="pl-k">+</span><span class="pl-c1">1</span>]
matching <span class="pl-k">=</span> matching <span class="pl-k">.==</span> maxval

peaks <span class="pl-k">=</span> maxval[matching]
peaks <span class="pl-k">=</span> peaks[peaks <span class="pl-k">.&gt;=</span> <span class="pl-c1">0.1</span> <span class="pl-k">*</span> <span class="pl-c1">maximum</span>(peaks)]

<span class="pl-k">for</span> l <span class="pl-k">=</span> <span class="pl-c1">1</span><span class="pl-k">:</span><span class="pl-c1">length</span>(peaks)
    x, y, z <span class="pl-k">=</span> <span class="pl-c1">ind2sub</span>(<span class="pl-c1">size</span>(A), <span class="pl-c1">find</span>(A <span class="pl-k">.==</span> peaks[l]))
    <span class="pl-c1">println</span>(<span class="pl-s"><span class="pl-pds">"</span>x=<span class="pl-v">$x</span>, y=<span class="pl-v">$y</span>, z=<span class="pl-v">$z</span>, val=<span class="pl-v">$(A[x,y,z])</span><span class="pl-pds">"</span></span>)
    <span class="pl-c1">oplot</span>(p, x, y, z)
<span class="pl-k">end</span></pre></div>
