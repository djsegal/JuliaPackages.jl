<p dir="auto">A lightweight XML parser that is purely written in Julia.</p>
<p dir="auto">See the <a href="https://baxmittens.github.io/XMLParser.jl/dev/" rel="nofollow">documentation</a>:</p>
<p dir="auto"><a href="https://baxmittens.github.io/XMLParser.jl/dev/" rel="nofollow"><img src="https://camo.githubusercontent.com/7fcec4b2d3ab291529fce8ef6a4fcd4129a0683b2f5d5fe2f5c648f02db8b616/68747470733a2f2f696d672e736869656c64732e696f2f62616467652f646f63732d6465762d626c75652e737667" alt="" data-canonical-src="https://img.shields.io/badge/docs-dev-blue.svg" style="max-width: 100%;"></a></p>
<p dir="auto">For extensive use of XML features, consider using one of the following XML-libraries: <a href="https://github.com/JuliaIO/LightXML.jl">LightXML.jl</a>, <a href="https://github.com/JuliaIO/EzXML.jl">EzXML.jl</a>.</p>
<p dir="auto">There exists another XML library <a href="https://github.com/joshday/XML.jl">XML.jl</a> purely written in Julia.</p>
