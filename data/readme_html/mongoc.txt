<p dir="auto"><strong>Mongoc.jl</strong> is a <a href="https://www.mongodb.com/" rel="nofollow">MongoDB</a> driver for the Julia Language.</p>
<p dir="auto">It is implemented as a thin wrapper around <a href="http://mongoc.org/" rel="nofollow">libmongoc</a>, the official client library for C applications.</p>
<p dir="auto">Given that <a href="http://bsonspec.org/" rel="nofollow">BSON</a> is the document format for MongoDB,
this package also implements a wrapper around <a href="http://mongoc.org/libbson/current/index.html" rel="nofollow">libbson</a>,
which provides a way to create and manipulate BSON documents.</p>
