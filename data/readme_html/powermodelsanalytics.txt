<p>To use the <code>plot_network</code> function for example, one must load a network case, e.g. using <code>parse_file</code> in PowerModels or PowerModelsDistribution, and then</p>
<div class="highlight highlight-source-julia position-relative" data-snippet-clipboard-copy-content="using PowerModelsAnalytics

plot_network(network_case)
"><pre><span class="pl-k">using</span> PowerModelsAnalytics

<span class="pl-c1">plot_network</span>(network_case)</pre></div>
<p>should plot the network using the currently enabled backend, or e.g.</p>
<div class="highlight highlight-source-julia position-relative" data-snippet-clipboard-copy-content="plot_network(network_case; filename=&quot;network.pdf&quot;)
"><pre><span class="pl-c1">plot_network</span>(network_case; filename<span class="pl-k">=</span><span class="pl-s"><span class="pl-pds">"</span>network.pdf<span class="pl-pds">"</span></span>)</pre></div>
<p>will save a network plot to a file using Vega.jl.</p>
