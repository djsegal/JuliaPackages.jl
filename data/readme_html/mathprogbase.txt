<p><strong>This package is deprecated in favor of <a href="https://github.com/JuliaOpt/MathOptInterface.jl">MathOptInterface</a>.</strong></p>
<p>This package provides high-level one-shot functions for linear and mixed-integer
programming (<code>linprog</code> and <code>mixintprog</code>), as well as a solver-independent
low-level interface for implementing advanced techniques that require efficiently
solving a sequence of linear programming problems.</p>
<p>See <a href="http://www.juliaopt.org/" rel="nofollow">JuliaOpt.org</a> for an up-to-date list of supported solvers.</p>
<p>The interface is fully documented <strong><a href="http://mathprogbasejl.readthedocs.org/en/latest" rel="nofollow">here</a></strong>.</p>
