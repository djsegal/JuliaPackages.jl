<p>This package is an experimental refactoring of the module <code>jlgr</code> from <a href="https://github.com/jheinen/GR.jl">GR.jl</a>, intended to provide the main utilities of <code>jlgr</code> in a more "Julian" and modular style, easy to read, and facilitate code contributions by others.</p>
<p>Find all the information about how GRUtils works its <a href="https://heliosdrm.github.io/GRUtils.jl/" rel="nofollow">documentation pages</a>, and feel free to submit your <a href="https://github.com/heliosdrm/GRUtils.jl/issues">issues</a> and <a href="https://github.com/heliosdrm/GRUtils.jl/pulls">pull requests</a>!</p>
