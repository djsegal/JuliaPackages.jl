<p>LLVM.jl is a registered package, and can be installed using the Julia package manager:</p>
<div class="highlight highlight-source-julia"><pre>Pkg<span class="pl-k">.</span><span class="pl-c1">add</span>(<span class="pl-s"><span class="pl-pds">"</span>LLVM<span class="pl-pds">"</span></span>)</pre></div>
<p><strong>NOTE</strong>: the current version of this package requires Julia 0.7. Only older
versions of this package, v0.5.x, work with Julia 0.6, and require a
<strong>source-build</strong> of Julia.</p>
