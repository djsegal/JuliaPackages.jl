<p dir="auto">With this package you can generate the initial conditions and perform the dynamics for your nonadiabatic dynamics simulations.
Tight integration with <a href="https://diffeq.sciml.ai/stable/" rel="nofollow">DifferentialEquations.jl</a>
makes the implementation of new methods relatively simple since we
build upon an already successful package providing a vast array of features.
We hope that the package will be of use to new students and experienced researchers alike, acting as a tool for learning and for developing new methods.</p>
