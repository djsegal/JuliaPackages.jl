<p dir="auto">ExaTron.jl implements a trust-region Newton solver for batched nonlinear programming on GPUs.
Problems in the batch are solved in parallel by employing multiple thread blocks on GPUs.
Our basic algorithm to solve each problem on GPUs is an extension of the
algorithm by <a href="https://doi.org/10.1137/S1052623498345075" rel="nofollow">Lin and More</a>
and its code <a href="https://www.mcs.anl.gov/~more/tron" rel="nofollow">TRON</a>.</p>
