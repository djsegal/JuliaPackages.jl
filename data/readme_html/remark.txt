<p>A simple Julia package to create presentations from markdown using <a href="https://github.com/gnab/remark">Remark</a>.</p>
<h2><a id="user-content-example-use" class="anchor" aria-hidden="true" href="#example-use"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path fill-rule="evenodd" d="M4 9h1v1H4c-1.5 0-3-1.69-3-3.5S2.55 3 4 3h4c1.45 0 3 1.69 3 3.5 0 1.41-.91 2.72-2 3.25V8.59c.58-.45 1-1.27 1-2.09C10 5.22 8.98 4 8 4H4c-.98 0-2 1.22-2 2.5S3 9 4 9zm9-3h-1v1h1c1 0 2 1.22 2 2.5S13.98 12 13 12H9c-.98 0-2-1.22-2-2.5 0-.83.42-1.64 1-2.09V6.25c-1.09.53-2 1.84-2 3.25C6 11.31 7.55 13 9 13h4c1.45 0 3-1.69 3-3.5S14.5 6 13 6z"></path></svg></a>Example use</h2>
<p>To install type the following command in the Julia Pkg REPL</p>
<div class="highlight highlight-source-julia"><pre>(v1<span class="pl-c1">.0</span>) pkg<span class="pl-k">&gt;</span> add Remark</pre></div>
<p>Check out the <a href="https://github.com/gnab/remark/wiki/Markdown">Remark docs</a> on how to write the markdown for a Remark slideshow. The most important thing is to use <code>---</code> to separate slides; an example markdown file can be found <a href="https://github.com/piever/Remark.jl/blob/master/examples/example.md">here</a>.</p>
<div class="highlight highlight-source-julia"><pre><span class="pl-k">import</span> Remark

<span class="pl-c"><span class="pl-c">#</span> Generate a presentation (html+markdown) from the markdown file "example.md"</span>
<span class="pl-c"><span class="pl-c">#</span> and save it in the folder "presentation/mybeautifulslides".</span>
slideshowdir <span class="pl-k">=</span> Remark<span class="pl-k">.</span><span class="pl-c1">slideshow</span>(<span class="pl-s"><span class="pl-pds">"</span>example.md<span class="pl-pds">"</span></span>, <span class="pl-s"><span class="pl-pds">"</span>presentation/mybeautifulslides<span class="pl-pds">"</span></span>,
                                options <span class="pl-k">=</span> <span class="pl-c1">Dict</span>(<span class="pl-s"><span class="pl-pds">"</span>ratio<span class="pl-pds">"</span></span> <span class="pl-k">=&gt;</span> <span class="pl-s"><span class="pl-pds">"</span>16:9<span class="pl-pds">"</span></span>))

<span class="pl-c"><span class="pl-c">#</span> Open presentation in default browser.</span>
Remark<span class="pl-k">.</span><span class="pl-c1">open</span>(slideshowdir)</pre></div>
<h2><a id="user-content-features" class="anchor" aria-hidden="true" href="#features"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path fill-rule="evenodd" d="M4 9h1v1H4c-1.5 0-3-1.69-3-3.5S2.55 3 4 3h4c1.45 0 3 1.69 3 3.5 0 1.41-.91 2.72-2 3.25V8.59c.58-.45 1-1.27 1-2.09C10 5.22 8.98 4 8 4H4c-.98 0-2 1.22-2 2.5S3 9 4 9zm9-3h-1v1h1c1 0 2 1.22 2 2.5S13.98 12 13 12H9c-.98 0-2-1.22-2-2.5 0-.83.42-1.64 1-2.09V6.25c-1.09.53-2 1.84-2 3.25C6 11.31 7.55 13 9 13h4c1.45 0 3-1.69 3-3.5S14.5 6 13 6z"></path></svg></a>Features</h2>
<ul>
<li>
<p>The <code>slideshow</code> command creates a slideshow that uses local javascript libraries: the resulting presentation folder can be opened offline.</p>
</li>
<li>
<p>Using the <a href="https://github.com/fredrikekre/Literate.jl">Literate</a> package, it is possible to create a presentation from a Julia script. As explained in the Literate documentation, add a comment to slides corresponding to markdown. A slide separator is now <code># ---</code> for example.</p>
</li>
<li>
<p><a href="https://github.com/JuliaDocs/Documenter.jl">Documenter</a> is run on the markdown by default: to keep the markdown as is, use <code>documenter=false</code>.</p>
</li>
<li>
<p>All features of Documenter are automatically available here, for example use <code># hide</code> at the end of a line of code for it to not appear in the slideshow (useful for saving plots, see <a href="https://github.com/piever/Remark.jl/blob/master/examples/example.md">example</a>).</p>
</li>
</ul>
