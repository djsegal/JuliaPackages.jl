<p dir="auto">A Julia package for working with the Qur'an (Islam's Holy Book), computationally. QuranTree.jl is based on <a href="https://corpus.quran.com/" rel="nofollow">The Quranic Arabic Corpus</a> by Kais Dukes of University of Leeds, and is aimed at offerring a high-level API alternative to the Java package, <a href="https://corpus.quran.com/java/overview.jsp" rel="nofollow">JQuranTree</a>.</p>

<p dir="auto">The following are the features of the package:</p>
<ul dir="auto">
<li>Indexing
<ul dir="auto">
<li>Intuitive indexing for Chapters, Verses, Words and Parts</li>
</ul>
</li>
<li>Complete type for all Morphological Features and Part of Speech</li>
<li>Others
<ul dir="auto">
<li>Supports Tanzil data</li>
<li>Read-only array for raw datasets (Corpus and Tanzil)</li>
</ul>
</li>
</ul>
<h2 dir="auto"><a id="user-content-yunirjl-support" class="anchor" aria-hidden="true" href="#yunirjl-support"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path d="m7.775 3.275 1.25-1.25a3.5 3.5 0 1 1 4.95 4.95l-2.5 2.5a3.5 3.5 0 0 1-4.95 0 .751.751 0 0 1 .018-1.042.751.751 0 0 1 1.042-.018 1.998 1.998 0 0 0 2.83 0l2.5-2.5a2.002 2.002 0 0 0-2.83-2.83l-1.25 1.25a.751.751 0 0 1-1.042-.018.751.751 0 0 1-.018-1.042Zm-4.69 9.64a1.998 1.998 0 0 0 2.83 0l1.25-1.25a.751.751 0 0 1 1.042.018.751.751 0 0 1 .018 1.042l-1.25 1.25a3.5 3.5 0 1 1-4.95-4.95l2.5-2.5a3.5 3.5 0 0 1 4.95 0 .751.751 0 0 1-.018 1.042.751.751 0 0 1-1.042.018 1.998 1.998 0 0 0-2.83 0l-2.5 2.5a1.998 1.998 0 0 0 0 2.83Z"></path></svg></a>Yunir.jl support</h2>
<p dir="auto"><a href="https://github.com/alstat/Yunir.jl">Yunir.jl</a> is a lightweight Arabic NLP toolkit that well supports QuranTree.jl for the following functionalities:</p>
<ul dir="auto">
<li>Transliteration
<ul dir="auto">
<li>Buckwalter as default</li>
<li>Functionality for creating custom transliterator</li>
<li>Update transliteration in 1 line of code</li>
</ul>
</li>
<li>Seemless transition between Arabic and Buckwalter (or custom transliteration)</li>
<li>Simple Encoding (refer <a href="https://corpus.quran.com/java/simpleencoding.jsp" rel="nofollow">here</a>)</li>
<li>Character Normalization
<ul dir="auto">
<li>For both Arabic and Buckwalter (or custom transliteration)</li>
</ul>
</li>
<li>Character Dediacritization
<ul dir="auto">
<li>For both Arabic and Buckwalter (or custom transliteration)</li>
</ul>
</li>
<li>Orthographical Analysis</li>
</ul>
