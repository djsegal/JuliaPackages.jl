<p dir="auto">Implementation of <a href="https://arxiv.org/abs/1008.5126" rel="nofollow">Krotov's method of optimal control</a> enhanced with automatic differentiation.</p>
<p dir="auto">Part of <a href="https://github.com/JuliaQuantumControl/QuantumControl.jl#readme"><code>QuantumControl.jl</code></a> and the <a href="https://github.com/JuliaQuantumControl">JuliaQuantumControl</a> organization.</p>
<p dir="auto">This package is a port of the <a href="https://github.com/qucontrol/krotov#readme"><code>krotov</code> Python package</a>, adapted to the API  of <a href="https://github.com/JuliaQuantumControl/QuantumControl.jl#readme"><code>QuantumControl.jl</code></a>.</p>
