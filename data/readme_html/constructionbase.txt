<p>ConstructionBase is a very lightwight package, that provides primitive functions for construction of objects:</p>
<div class="highlight highlight-source-julia"><pre><span class="pl-c1">setproperties</span>(obj<span class="pl-k">::</span><span class="pl-c1">MyType</span>, patch<span class="pl-k">::</span><span class="pl-c1">NamedTuple</span>)
<span class="pl-c1">constructorof</span>(MyType)</pre></div>
<p>These functions can be overloaded and doing so provides interoperability with the following packages:</p>
<ul>
<li><a href="https://github.com/rafaqz/Flatten.jl">Flatten.jl</a></li>
<li><a href="https://github.com/jw3126/Setfield.jl">Setfield.jl</a></li>
<li><a href="https://github.com/tkf/BangBang.jl">BangBang.jl</a></li>
</ul>
