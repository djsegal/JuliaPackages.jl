<p dir="auto">This package provides a Julia wrapper for the <a href="http://webhotel4.ruc.dk/~keld/research/LKH-3/" rel="nofollow">LKH-3</a> library for solving Traveling Salesman Problems (TSP) and various Vehicle Routing Problems (VRPs).</p>
<p dir="auto">While this Julia package is under MIT License, the underlying LKH library comes in a different license. Check with the <a href="http://webhotel4.ruc.dk/~keld/research/LKH-3/" rel="nofollow">LKH library homepage</a>.</p>
