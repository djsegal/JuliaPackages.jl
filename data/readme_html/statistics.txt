<p dir="auto">If you want to develop this package, do the following steps:</p>
<ul dir="auto">
<li>Clone the repo anywhere.</li>
<li>In line 2 of the <code>Project.toml</code> file (the line that begins with <code>uuid = ...</code>), modify the UUID, e.g. change the <code>107</code> to <code>207</code>.</li>
<li>Change the current directory to the Statistics repo you just cloned and start julia with <code>julia --project</code>.</li>
<li><code>import Statistics</code> will now load the files in the cloned repo instead of the Statistics stdlib.</li>
<li>To test your changes, simply do <code>include("test/runtests.jl")</code>.</li>
</ul>
<p dir="auto">If you need to build Julia from source with a git checkout of Statistics, then instead use <code>make DEPS_GIT=Statistics</code> when building Julia. The <code>Statistics</code> repo is in <code>stdlib/Statistics</code>, and created initially with a detached <code>HEAD</code>. If you're doing this from a pre-existing Julia repository, you may need to <code>make clean</code> beforehand.</p>
