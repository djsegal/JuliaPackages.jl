<p dir="auto">MirrorUpdater.jl is a Julia application that provides functionality for
automatically mirroring Julia package repositories.</p>
<p dir="auto">MirrorUpdater.jl (and its sibling project, <a href="https://github.com/UnofficialJuliaMirrorSnapshots/Snapshots.jl">Snapshots.jl</a>) are used to maintain the
Julia package mirrors and snapshots hosted at:</p>

<h3 dir="auto"><a id="user-content-step-1" class="anchor" aria-hidden="true" href="#step-1"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path d="m7.775 3.275 1.25-1.25a3.5 3.5 0 1 1 4.95 4.95l-2.5 2.5a3.5 3.5 0 0 1-4.95 0 .751.751 0 0 1 .018-1.042.751.751 0 0 1 1.042-.018 1.998 1.998 0 0 0 2.83 0l2.5-2.5a2.002 2.002 0 0 0-2.83-2.83l-1.25 1.25a.751.751 0 0 1-1.042-.018.751.751 0 0 1-.018-1.042Zm-4.69 9.64a1.998 1.998 0 0 0 2.83 0l1.25-1.25a.751.751 0 0 1 1.042.018.751.751 0 0 1 .018 1.042l-1.25 1.25a3.5 3.5 0 1 1-4.95-4.95l2.5-2.5a3.5 3.5 0 0 1 4.95 0 .751.751 0 0 1-.018 1.042.751.751 0 0 1-1.042.018 1.998 1.998 0 0 0-2.83 0l-2.5 2.5a1.998 1.998 0 0 0 0 2.83Z"></path></svg></a>Step 1</h3>
<p dir="auto">If you do not already have a personal GitHub account, <a href="https://help.github.com/articles/signing-up-for-a-new-github-account/">create one</a>. For the remainder of this README, <code>MY_PERSONAL_GITHUB_USERNAME</code> refers to the username of your personal GitHub account.</p>
<p dir="auto"><em>For example, for me, <code>MY_PERSONAL_GITHUB_USERNAME</code> is equal to <code>DilumAluthge</code>.</em></p>
<h3 dir="auto"><a id="user-content-step-2" class="anchor" aria-hidden="true" href="#step-2"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path d="m7.775 3.275 1.25-1.25a3.5 3.5 0 1 1 4.95 4.95l-2.5 2.5a3.5 3.5 0 0 1-4.95 0 .751.751 0 0 1 .018-1.042.751.751 0 0 1 1.042-.018 1.998 1.998 0 0 0 2.83 0l2.5-2.5a2.002 2.002 0 0 0-2.83-2.83l-1.25 1.25a.751.751 0 0 1-1.042-.018.751.751 0 0 1-.018-1.042Zm-4.69 9.64a1.998 1.998 0 0 0 2.83 0l1.25-1.25a.751.751 0 0 1 1.042.018.751.751 0 0 1 .018 1.042l-1.25 1.25a3.5 3.5 0 1 1-4.95-4.95l2.5-2.5a3.5 3.5 0 0 1 4.95 0 .751.751 0 0 1-.018 1.042.751.751 0 0 1-1.042.018 1.998 1.998 0 0 0-2.83 0l-2.5 2.5a1.998 1.998 0 0 0 0 2.83Z"></path></svg></a>Step 2</h3>
<p dir="auto">Log in to GitHub as <code>MY_PERSONAL_GITHUB_USERNAME</code>.</p>
<h3 dir="auto"><a id="user-content-step-3" class="anchor" aria-hidden="true" href="#step-3"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path d="m7.775 3.275 1.25-1.25a3.5 3.5 0 1 1 4.95 4.95l-2.5 2.5a3.5 3.5 0 0 1-4.95 0 .751.751 0 0 1 .018-1.042.751.751 0 0 1 1.042-.018 1.998 1.998 0 0 0 2.83 0l2.5-2.5a2.002 2.002 0 0 0-2.83-2.83l-1.25 1.25a.751.751 0 0 1-1.042-.018.751.751 0 0 1-.018-1.042Zm-4.69 9.64a1.998 1.998 0 0 0 2.83 0l1.25-1.25a.751.751 0 0 1 1.042.018.751.751 0 0 1 .018 1.042l-1.25 1.25a3.5 3.5 0 1 1-4.95-4.95l2.5-2.5a3.5 3.5 0 0 1 4.95 0 .751.751 0 0 1-.018 1.042.751.751 0 0 1-1.042.018 1.998 1.998 0 0 0-2.83 0l-2.5 2.5a1.998 1.998 0 0 0 0 2.83Z"></path></svg></a>Step 3</h3>
<p dir="auto">While logged in as <code>MY_PERSONAL_GITHUB_USERNAME</code>, <a href="https://help.github.com/articles/configuring-two-factor-authentication/">enable two-factor authentication</a> on the <code>MY_PERSONAL_GITHUB_USERNAME</code> account.</p>
<p dir="auto"><strong>Make sure to store your two-factor recovery codes in a secure location!</strong></p>
<h3 dir="auto"><a id="user-content-step-4" class="anchor" aria-hidden="true" href="#step-4"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path d="m7.775 3.275 1.25-1.25a3.5 3.5 0 1 1 4.95 4.95l-2.5 2.5a3.5 3.5 0 0 1-4.95 0 .751.751 0 0 1 .018-1.042.751.751 0 0 1 1.042-.018 1.998 1.998 0 0 0 2.83 0l2.5-2.5a2.002 2.002 0 0 0-2.83-2.83l-1.25 1.25a.751.751 0 0 1-1.042-.018.751.751 0 0 1-.018-1.042Zm-4.69 9.64a1.998 1.998 0 0 0 2.83 0l1.25-1.25a.751.751 0 0 1 1.042.018.751.751 0 0 1 .018 1.042l-1.25 1.25a3.5 3.5 0 1 1-4.95-4.95l2.5-2.5a3.5 3.5 0 0 1 4.95 0 .751.751 0 0 1-.018 1.042.751.751 0 0 1-1.042.018 1.998 1.998 0 0 0-2.83 0l-2.5 2.5a1.998 1.998 0 0 0 0 2.83Z"></path></svg></a>Step 4</h3>
<p dir="auto">While logged in as <code>MY_PERSONAL_GITHUB_USERNAME</code>, <a href="https://help.github.com/articles/creating-a-new-organization-from-scratch/">create a free GitHub organization</a> that you will use only for hosting the mirrored repositories. For the remainder of this README, <code>MY_GITHUB_ORG</code> refers to the name of this organization. <code>MY_PERSONAL_GITHUB_USERNAME</code> should be an <code>owner</code> of the <code>MY_GITHUB_ORG</code> organization.</p>
<p dir="auto"><em>For example, for me, <code>MY_GITHUB_ORG</code> is equal to <code>UnofficialJuliaMirror</code>.</em></p>

<p dir="auto">While logged in as <code>MY_PERSONAL_GITHUB_USERNAME</code>, go to the <code>MY_GITHUB_ORG</code> organization security settings page (<code>https://github.com/organizations/MY_GITHUB_ORG/settings/security</code>).</p>
<p dir="auto">Next, make sure that the checkbox next to "Require two-factor authentication for everyone..." is CHECKED.</p>
<p dir="auto">Finally, click the "Save" button.</p>

<p dir="auto">While logged in as <code>MY_GITHUB_BOT_USERNAME</code>, <a href="https://help.github.com/articles/creating-a-personal-access-token-for-the-command-line/">create a personal access token</a> for the <code>MY_GITHUB_BOT_USERNAME</code> account and store it in a secure place (such as a password manager). For the remainder of this README, <code>my-github-bot-personal-access-token</code> refers to this personal access token.</p>
<p dir="auto"><strong>The personal access token should be treated as securely as a password. Do not share it with anyone. Do not save it in any unsecure location. Do not save it in a file. Do not commit it in a Git repository.</strong></p>
