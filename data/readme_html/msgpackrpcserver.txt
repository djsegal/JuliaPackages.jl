module RemoteFunctions
  function add(x, y)
    x+y
  end
end

<p>and run <code>ruby -r msgpack-rpc -e "p MessagePack::RPC::Client.new('127.0.0.1', 8080).call(:add, 2, 3)"</code> then you will get <code>5</code> calculated by Julia.</p>

<p>As of v0.0.1, only very basic synchronous rpc call on TCP Socket is implemented. Following features are to be supported in the future.</p>
<ul>
<li>Asynchronous Call</li>
<li>Multiple Transports such as UDP, UNIX domain socket</li>
<li>High Performance Capability such as single-threaded model or event-driven architecture etc.</li>
</ul>
