<p dir="auto">A simple and lightweight implementation of <a href="https://en.wikipedia.org/wiki/Receiver_operating_characteristic" rel="nofollow">ROC (Receiver operating characteristic)</a> curves in Julia.</p>
<p dir="auto">Exports the functions: <code>roc</code>, <code>auc</code>, <code>auc_roc</code> and <code>confusion_matrix</code>.
See their doc-strings for details, and the <code>example.jl</code> file for an usage example.</p>
<p dir="auto">For a more full-featured implementation see <a href="https://github.com/davidavdav/ROCAnalysis.jl">https://github.com/davidavdav/ROCAnalysis.jl</a>.</p>
