<p dir="auto">To run the online AWS Batch tests you must first set the environmental variables <code>TESTS</code> and
<code>AWS_STACKNAME</code>.</p>
<div class="highlight highlight-source-julia notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="ENV[&quot;TESTS&quot;] = &quot;batch&quot;

<span class="pl-c1">ENV</span>[<span class="pl-s"><span class="pl-pds">"</span>AWS_STACKNAME<span class="pl-pds">"</span></span>] <span class="pl-k">=</span> <span class="pl-s"><span class="pl-pds">"</span>aws-batch-manager-test<span class="pl-pds">"</span></span></pre></div>
<p dir="auto">To make an <code>aws-batch-manager-test</code> compatible stack you can use the CloudFormation template <a href="./test/batch.yml">test/resources/batch.yml</a>.</p>
