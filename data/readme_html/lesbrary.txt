<p>This package is a framework for building a library of large eddy simulations (LES) of ocean surface boundary layer turbulence — the <em>LESbrary</em> — with <a href="https://github.com/climate-machine/Oceananigans.jl">Oceananigans.jl</a>. The LESbrary will archive turbulence data for both idealized and realistic oceanic scenarios.</p>
<h2><a id="user-content-adding-lesbraryjl-to-a-julia-environment" class="anchor" aria-hidden="true" href="#adding-lesbraryjl-to-a-julia-environment"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path fill-rule="evenodd" d="M7.775 3.275a.75.75 0 001.06 1.06l1.25-1.25a2 2 0 112.83 2.83l-2.5 2.5a2 2 0 01-2.83 0 .75.75 0 00-1.06 1.06 3.5 3.5 0 004.95 0l2.5-2.5a3.5 3.5 0 00-4.95-4.95l-1.25 1.25zm-4.69 9.64a2 2 0 010-2.83l2.5-2.5a2 2 0 012.83 0 .75.75 0 001.06-1.06 3.5 3.5 0 00-4.95 0l-2.5 2.5a3.5 3.5 0 004.95 4.95l1.25-1.25a.75.75 0 00-1.06-1.06l-1.25 1.25a2 2 0 01-2.83 0z"></path></svg></a>Adding LESbrary.jl to a julia environment</h2>
<p>To add <code>LESbrary.jl</code> to your julia environment, type</p>
<div class="highlight highlight-source-julia position-relative" data-snippet-clipboard-copy-content="julia&gt; using Pkg

julia&gt; Pkg.add(url=&quot;https://github.com/CliMA/LESbrary.jl.git&quot;)
"><pre>julia<span class="pl-k">&gt;</span> <span class="pl-k">using</span> Pkg

julia<span class="pl-k">&gt;</span> Pkg<span class="pl-k">.</span><span class="pl-c1">add</span>(url<span class="pl-k">=</span><span class="pl-s"><span class="pl-pds">"</span>https://github.com/CliMA/LESbrary.jl.git<span class="pl-pds">"</span></span>)</pre></div>
<p>at the REPL. This will allow you to write <code>using LESbrary</code> in your scripts (and thereby run the scripts in <code>/examples</code>).</p>
