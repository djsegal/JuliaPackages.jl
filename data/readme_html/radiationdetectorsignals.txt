<p dir="auto">A Julia package for data types to represent radiation detector signals,
especially sparse hit data and digitized waveforms.</p>
<p dir="auto">RadiationDetectorSignals.jl is integrated with
<a href="https://github.com/JuliaArrays/StructArrays.jl">StructArrays.jl</a> for
contiguous memory layout of data. The package is also compatible with
<a href="https://github.com/oschulz/EncodedArrays.jl">EncodedArrays.jl</a> to handle
compressed data.</p>
