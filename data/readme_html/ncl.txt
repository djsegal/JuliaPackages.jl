<p dir="auto">An implementation of <a href="https://dx.doi.org/10.1007/978-3-319-90026-1_8" rel="nofollow">Algorithm NCL</a> in pure Julia using either <a href="http://www.coin-or.org/Ipopt/documentation/documentation.html" rel="nofollow">IPOPT</a> or <a href="https://www.artelys.com/knitro" rel="nofollow">Artelys KNITRO</a> to solve the subproblems.</p>
<p dir="auto">The <code>data</code> folder contains several tax models in <a href="http://www.ampl.com" rel="nofollow">AMPL</a> format that can be read with <a href="https://github.com/JuliaSmoothOptimizers/AmplNLReader.jl">AmplNLReader</a>.
Any model complying with the <a href="https://github.com/JuliaSmoothOptimizers/NLPModels.jl">NLPModels</a> API can be passed to the NCL solver, e.g., those from the <a href="https://github.com/JuliaSmoothOptimizers/CUTEst.jl">CUTEst</a> collection.</p>
<p dir="auto">The solver is still work in progress but is functional.
A similar solver is available for tax problems only in the AMPL scripting language: <a href="https://github.com/optimizers/ncl">https://github.com/optimizers/ncl</a>.</p>
