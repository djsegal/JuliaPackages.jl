<p dir="auto">This package implements an algorithm to find the <em>pole of inaccessibility</em> of a polygon, the most distant internal point from the polygon outline.  This algorithm was originally written (and taken from) <a href="https://github.com/mapbox/polylabel">mapbox/polylabel</a> - you can find a lot more information there!  To summarize, the algorithm is basically a quad-tree search across the polygon which finds the point which is most distant from any edge.</p>
<p dir="auto">In the plot above, this point is shown in orange, while the input polygon (multipolygon in this case) is shown in blue.</p>
<p dir="auto">The package is built on top of the GeoInterface, and is built to work with any polygons or multipolygon object which implements the GeoInterface, specifically the functions <code>GeoInterface.contains</code>, <code>GeoInterface.centroid</code>, and the <code>geointerface_geomtype</code> API for reverse conversion.</p>
<p dir="auto">The main entry point is the <code>polylabel(input; atol = nothing, rtol = 0.01)</code> function.  It returns a 2-Tuple of floats, representing the x and y coordinates of the found pole of inaccessibility.</p>
