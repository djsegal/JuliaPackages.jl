<p dir="auto">This is a Julia implementation of the ODE Hybrid package made for MATLAB.
This ported version tries to follow the MATLAB implementation closely so as to maintain the same functionality
and match the MATLAB output</p>
<p dir="auto">(NOTE that because the original code does not allow for Events to be used for the custom RK functions,
relying instead on the ODE suite of solvers in MATLAB like ODE45, Events are currently not supported).</p>
<p dir="auto">The original package is well-documented at <a href="http://www.anuncommonlab.com/doc/odehybrid" rel="nofollow">http://www.anuncommonlab.com/doc/odehybrid</a></p>
