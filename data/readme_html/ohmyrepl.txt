<p dir="auto">A package that hooks into the Julia REPL and gives it syntax highlighting, bracket highlighting, rainbow brackets and other goodies.
A (slightly outdated) video showing installation and features of the package is available <a href="https://www.youtube.com/watch?v=lTLPAOLLbTU" rel="nofollow">here</a>.</p>
<p dir="auto">If you like this package please give it a star. I like stars.</p>

<h3 dir="auto"><a id="user-content-features" class="anchor" aria-hidden="true" href="#features"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path d="m7.775 3.275 1.25-1.25a3.5 3.5 0 1 1 4.95 4.95l-2.5 2.5a3.5 3.5 0 0 1-4.95 0 .751.751 0 0 1 .018-1.042.751.751 0 0 1 1.042-.018 1.998 1.998 0 0 0 2.83 0l2.5-2.5a2.002 2.002 0 0 0-2.83-2.83l-1.25 1.25a.751.751 0 0 1-1.042-.018.751.751 0 0 1-.018-1.042Zm-4.69 9.64a1.998 1.998 0 0 0 2.83 0l1.25-1.25a.751.751 0 0 1 1.042.018.751.751 0 0 1 .018 1.042l-1.25 1.25a3.5 3.5 0 1 1-4.95-4.95l2.5-2.5a3.5 3.5 0 0 1 4.95 0 .751.751 0 0 1-.018 1.042.751.751 0 0 1-1.042.018 1.998 1.998 0 0 0-2.83 0l-2.5 2.5a1.998 1.998 0 0 0 0 2.83Z"></path></svg></a>Features</h3>
<ul dir="auto">
<li>Syntax highlighting - Highlighting of keyword, operators, symbols, strings etc. in different colors.</li>
<li>Bracket highlighting - Will make matching brackets highlighted when the cursor is between an opening and closing bracket.</li>
<li>Automatic bracket insertion - Will insert matching closing bracket and quotation symbols when suitable.</li>
<li>Prompt changing - Can change the text and color of the <code>julia&gt;</code> prompt as well as add a prompt for output.</li>
<li>Rainbow brackets - Colorizes matching brackets in the same color.</li>
<li>Fuzzy (fzf) history search - Search REPL history in any mode fuzzily with beloved <a href="https://github.com/junegunn/fzf">fzf</a>.</li>
</ul>
