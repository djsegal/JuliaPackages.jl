<p>A package that hooks into the Julia REPL and gives it syntax highlighting, bracket highlighting, rainbow brackets and other goodies.
A (slightly outdated) video showing installation and features of the package is available <a href="https://www.youtube.com/watch?v=lTLPAOLLbTU" rel="nofollow">here</a>.</p>
<p>If you like this package please give it a star. I like stars.</p>

<div class="highlight highlight-source-julia"><pre>Pkg<span class="pl-k">.</span><span class="pl-c1">add</span>(<span class="pl-s"><span class="pl-pds">"</span>OhMyREPL<span class="pl-pds">"</span></span>)</pre></div>

<h3><a id="user-content-features" class="anchor" aria-hidden="true" href="#features"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path fill-rule="evenodd" d="M7.775 3.275a.75.75 0 001.06 1.06l1.25-1.25a2 2 0 112.83 2.83l-2.5 2.5a2 2 0 01-2.83 0 .75.75 0 00-1.06 1.06 3.5 3.5 0 004.95 0l2.5-2.5a3.5 3.5 0 00-4.95-4.95l-1.25 1.25zm-4.69 9.64a2 2 0 010-2.83l2.5-2.5a2 2 0 012.83 0 .75.75 0 001.06-1.06 3.5 3.5 0 00-4.95 0l-2.5 2.5a3.5 3.5 0 004.95 4.95l1.25-1.25a.75.75 0 00-1.06-1.06l-1.25 1.25a2 2 0 01-2.83 0z"></path></svg></a>Features</h3>
<ul>
<li>Syntax highlighting - Highlighting of keyword, operators, symbols, strings etc. in different colors.</li>
<li>Bracket highlighting - Will make matching brackets highlighted when the cursor is between an opening and closing bracket.</li>
<li>Automatic bracket insertion - Will insert matching closing bracket and quotation symbols when suitable.</li>
<li>Prompt changing - Can change the text and color of the <code>julia&gt;</code> prompt as well as add a prompt for output.</li>
<li>Rainbow brackets - Colorizes matching brackets in the same color.</li>
</ul>
