<p dir="auto"><a target="_blank" rel="noopener noreferrer" href="fastai-julia-logo.png"><img src="fastai-julia-logo.png" alt="" style="max-width: 100%;"></a></p>
<p dir="auto">FastAI.jl is a Julia library for training state-of-the art deep learning models.</p>
<p dir="auto">From loading datasets and creating data preprocessing pipelines to training, FastAI.jl takes the boilerplate out of deep learning projects. It equips you with reusable components for every part of your project while remaining customizable at every layer. FastAI.jl comes with support for common computer vision and tabular data learning tasks, with more to come.</p>
<p dir="auto">FastAI.jl's high-level workflows combine functionality from many packages in the ecosystem, most notably <a href="https://github.com/FluxML/Flux.jl">Flux.jl</a>, <a href="https://github.com/FluxML/FluxTraining.jl">FluxTraining.jl</a>, <a href="https://github.com/lorenzoh/DataAugmentation.jl">DataAugmentation.jl</a> and <a href="https://github.com/JuliaML/MLUtils.jl">MLUtils.jl</a>.</p>
<p dir="auto">See our <a href="https://fluxml.ai/FastAI.jl" rel="nofollow"><strong>documentation</strong></a> to find out more.</p>
