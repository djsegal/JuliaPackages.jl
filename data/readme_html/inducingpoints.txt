<p>A package for selecting inducing points for sparse GPs</p>
<p>This package provide a collection of inducing point location selection algorithms, both offline and online.</p>
<p>For example</p>
<div class="highlight highlight-source-julia position-relative" data-snippet-clipboard-copy-content="using InducingPoints
Z = KmeansIP(X, 10, obsdim = 1)
"><pre><span class="pl-k">using</span> InducingPoints
Z <span class="pl-k">=</span> <span class="pl-c1">KmeansIP</span>(X, <span class="pl-c1">10</span>, obsdim <span class="pl-k">=</span> <span class="pl-c1">1</span>)</pre></div>
<p>will return 10 inducing points selected as clusters by the k-means algorithm</p>
<p>For online algorithms, one need to first create an empty object. For example</p>
<div class="highlight highlight-source-julia position-relative" data-snippet-clipboard-copy-content="Z = OIPS(200) # Method from unpublished paper (for now!)
"><pre>Z <span class="pl-k">=</span> <span class="pl-c1">OIPS</span>(<span class="pl-c1">200</span>) <span class="pl-c"><span class="pl-c">#</span> Method from unpublished paper (for now!)</span></pre></div>
<p>The first time data is met <code>init(Z, X, args...)</code> should be called (args depend of the algorithm). <code>init</code> will return a new object correctly parametrized.</p>
<p>In the subsequent calls <code>update!(Z, X, args...)</code> to add/remove/move points.</p>
