<p dir="auto">You first need to obtain an API token from Zotero.
Go to your <a href="https://www.zotero.org/settings/keys" rel="nofollow">Zotero account settings</a> and create a new API token.
You should also get your user ID from the same page.
Store both of these values in a <code>.env</code> file in the root of your project with
the following format:</p>
<div class="highlight highlight-source-toml notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="ZOTERO_API_TOKEN=&quot;[YOUR API TOKEN]&quot;
ZOTERO_USER_ID=&quot;[YOUR USER ID]&quot;"><pre><span class="pl-smi">ZOTERO_API_TOKEN</span>=<span class="pl-s"><span class="pl-pds">"</span>[YOUR API TOKEN]<span class="pl-pds">"</span></span>
<span class="pl-smi">ZOTERO_USER_ID</span>=<span class="pl-s"><span class="pl-pds">"</span>[YOUR USER ID]<span class="pl-pds">"</span></span></pre></div>
