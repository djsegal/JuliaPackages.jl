<p><a href="http://scikitlearnjl.readthedocs.org/en/latest/?badge=latest" rel="nofollow"><img src="https://camo.githubusercontent.com/bace7fb6479d29cd3dc5bf3c11b61878b4783db2/68747470733a2f2f72656164746865646f63732e6f72672f70726f6a656374732f7363696b69746c6561726e6a6c2f62616467652f3f76657273696f6e3d6c6174657374" alt="Documentation Status" data-canonical-src="https://readthedocs.org/projects/scikitlearnjl/badge/?version=latest" style="max-width:100%;"></a></p>
<p>ScikitLearn.jl implements the popular
<a href="http://scikit-learn.org/stable/" rel="nofollow">scikit-learn</a> interface and algorithms in
Julia. It supports both models from the Julia ecosystem and those of the
<a href="http://scikit-learn.org/stable/modules/classes.html" rel="nofollow">scikit-learn library</a>
(via PyCall.jl).</p>
<p>Would you rather use a machine-learning framework specially-designed for Julia? Check out <a href="https://github.com/alan-turing-institute/MLJ.jl">MLJ.jl</a>, from the Alan Turing institute.</p>
<p><strong>Disclaimer</strong>: ScikitLearn.jl borrows code and documentation from
<a href="http://scikit-learn.org/stable/" rel="nofollow">scikit-learn</a>, but <em>it is not an official part
of that project</em>. It is licensed under <a href="LICENSE">BSD-3</a>.</p>
<p>Main features:</p>
<ul>
<li>Around 150 <a href="http://scikitlearnjl.readthedocs.io/en/latest/models/#julia-models" rel="nofollow">Julia</a> and <a href="http://scikitlearnjl.readthedocs.io/en/latest/models/#python-models" rel="nofollow">Python</a> models accessed through a uniform <a href="http://scikitlearnjl.readthedocs.org/en/latest/api/" rel="nofollow">interface</a></li>
<li><a href="http://scikitlearnjl.readthedocs.org/en/latest/pipelines/" rel="nofollow">Pipelines and FeatureUnions</a></li>
<li><a href="http://scikitlearnjl.readthedocs.org/en/latest/cross_validation/" rel="nofollow">Cross-validation</a></li>
<li><a href="http://scikitlearnjl.readthedocs.org/en/latest/model_selection/" rel="nofollow">Hyperparameter tuning</a></li>
<li><a href="http://scikitlearnjl.readthedocs.org/en/latest/dataframes/" rel="nofollow">DataFrames support</a></li>
</ul>
<p>Check out the <a href="http://scikitlearnjl.readthedocs.org/en/latest/quickstart/" rel="nofollow">Quick-Start
Guide</a> for a
tour.</p>

<p>To install ScikitLearn.jl, type <code>]add ScikitLearn</code> at the REPL.</p>
<p>To import Python models (optional), ScikitLearn.jl requires <a href="http://scikitlearnjl.readthedocs.io/en/latest/models/#installation" rel="nofollow">the scikit-learn Python library</a>, which will be installed automatically when needed. Most of the examples use <a href="https://github.com/stevengj/PyPlot.jl">PyPlot.jl</a></p>

encounter any problem that is solved by that library but not this one, <a href="https://github.com/cstjean/ScikitLearn.jl/issues">file an
issue</a>.</p>
