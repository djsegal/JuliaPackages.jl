<p dir="auto">Package <strong>Manifolds.jl</strong> aims to provide both a unified interface to define and
use manifolds as well as a library of manifolds to use for your projects.
This package is mostly stable, see <a class="issue-link js-issue-link" data-error-text="Failed to load title" data-id="1022364819" data-permission-text="Title is private" data-url="https://github.com/JuliaManifolds/Manifolds.jl/issues/438" data-hovercard-type="issue" data-hovercard-url="/JuliaManifolds/Manifolds.jl/issues/438/hovercard" href="https://github.com/JuliaManifolds/Manifolds.jl/issues/438">#438</a> for planned upcoming changes.</p>

<p dir="auto">To install the package just type</p>
<div class="highlight highlight-source-julia notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="using Pkg; Pkg.add(&quot;Manifolds&quot;)"><pre><span class="pl-k">using</span> Pkg; Pkg<span class="pl-k">.</span><span class="pl-c1">add</span>(<span class="pl-s"><span class="pl-pds">"</span>Manifolds<span class="pl-pds">"</span></span>)</pre></div>
<p dir="auto">Then you can directly start, for example to stop half way from the north pole on the <a href="https://juliamanifolds.github.io/Manifolds.jl/stable/manifolds/sphere.html" rel="nofollow"><code>Sphere</code></a> to a point on the equator, you can generate the <a href="https://juliamanifolds.github.io/ManifoldsBase.jl/stable/functions.html#ManifoldsBase.shortest_geodesic-Tuple%7BAbstractManifold,%20Any,%20Any%7D" rel="nofollow"><code>shortest_geodesic</code></a>.
It internally employs <a href="https://juliamanifolds.github.io/ManifoldsBase.jl/stable/functions.html#Base.exp-Tuple%7BAbstractManifold,%20Any,%20Any%7D" rel="nofollow"><code>exp</code></a> and <a href="https://juliamanifolds.github.io/ManifoldsBase.jl/stable/functions.html#Base.log-Tuple%7BAbstractManifold,%20Any,%20Any%7D" rel="nofollow"><code>log</code></a>.</p>
<div class="highlight highlight-source-julia notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="using Manifolds

<h2 dir="auto"><a id="user-content-ecosystem-highlights" class="anchor" aria-hidden="true" href="#ecosystem-highlights"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path d="m7.775 3.275 1.25-1.25a3.5 3.5 0 1 1 4.95 4.95l-2.5 2.5a3.5 3.5 0 0 1-4.95 0 .751.751 0 0 1 .018-1.042.751.751 0 0 1 1.042-.018 1.998 1.998 0 0 0 2.83 0l2.5-2.5a2.002 2.002 0 0 0-2.83-2.83l-1.25 1.25a.751.751 0 0 1-1.042-.018.751.751 0 0 1-.018-1.042Zm-4.69 9.64a1.998 1.998 0 0 0 2.83 0l1.25-1.25a.751.751 0 0 1 1.042.018.751.751 0 0 1 .018 1.042l-1.25 1.25a3.5 3.5 0 1 1-4.95-4.95l2.5-2.5a3.5 3.5 0 0 1 4.95 0 .751.751 0 0 1-.018 1.042.751.751 0 0 1-1.042.018 1.998 1.998 0 0 0-2.83 0l-2.5 2.5a1.998 1.998 0 0 0 0 2.83Z"></path></svg></a>Ecosystem highlights</h2>
<ul dir="auto">
<li>A wide selection of Riemannian manifolds like spheres, hyperbolic spaces, Stiefel and Grassmann manifolds.</li>
<li>Support for optimization on manifolds using <a href="https://github.com/JuliaManifolds/Manopt.jl/">Manopt.jl</a>.</li>
<li>Support for many operations used in optimization and manifold-valued statistics with a focus on performance and ease of use.</li>
<li>Connection manifolds.</li>
<li>Lie groups.</li>
<li>Atlases, charts and custom metrics (work in progress).</li>
<li>A lightweight interface package: <a href="https://github.com/JuliaManifolds/ManifoldsBase.jl">ManifoldsBase.jl</a>.</li>
<li>Differential equations on manifolds: <a href="https://github.com/JuliaManifolds/ManifoldDiffEq.jl">ManifoldDiffEq.jl</a>.</li>
<li>Finite differences and automatic differentiation on manifolds using <a href="https://github.com/JuliaManifolds/ManifoldDiff.jl">ManifoldDiff.jl</a> (work in progress).</li>
<li>Integration and measures on manifolds: <a href="https://github.com/JuliaManifolds/ManifoldMeasures.jl">ManifoldMeasures.jl</a> (work in progress).</li>
<li>Functional manifolds: <a href="https://github.com/JuliaManifolds/FunManifolds.jl">FunManifolds.jl</a> (work in progress).</li>
</ul>
