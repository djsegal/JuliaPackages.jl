<p>Package <strong>Manifolds.jl</strong> aims to provide both a unified interface to define and
use manifolds as well as a library of manifolds to use for your projects.
This package is under development, and subject to changes as needed.</p>

<p>To install the package just type</p>
<div class="highlight highlight-source-julia position-relative" data-snippet-clipboard-copy-content="] add Manifolds
"><pre>] add Manifolds</pre></div>
<p>Then you can directly start, for example to stop half way from the north pole on the <a href="https://juliamanifolds.github.io/Manifolds.jl/stable/manifolds/sphere.html" rel="nofollow"><code>Sphere</code></a> to a point on the the equator, you can generate the <a href="https://juliamanifolds.github.io/Manifolds.jl/stable/interface.html#ManifoldsBase.shortest_geodesic-Tuple%7BManifold,Any,Any%7D" rel="nofollow"><code>shortest_geodesic</code></a>.
It internally employs <a href="https://juliamanifolds.github.io/Manifolds.jl/latest/interface.html#Base.exp-Tuple%7BManifold,Any,Any%7D" rel="nofollow"><code>exp</code></a> and <a href="https://juliamanifolds.github.io/Manifolds.jl/latest/interface.html#Base.log-Tuple%7BManifold,Any,Any%7D" rel="nofollow"><code>log</code></a>.</p>
