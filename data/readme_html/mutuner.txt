<p dir="auto">This package exports a method for tuning the chemical potential to achieve a target particle number in grand canonical Monte Carlo simulations.
The algorithm implemented in this package was introduced in the paper
<a href="https://journals.aps.org/pre/abstract/10.1103/PhysRevE.105.045311" rel="nofollow">Phys. Rev. E 105, 045311</a>.
If you use this package in work leading to a publication, please consider citing this paper.</p>
