<p dir="auto">More algorithms for the <a href="https://github.com/CarloLucibello/Erdos.jl">Erdos.jl</a> graph library.
This package contains the following functions:</p>
<ul dir="auto">
<li><code>solve_tsp</code>: solves the travelling salesman problem by linear programming + lazy addition of constraints.</li>
<li><code>minimum_weight_perfect_matching</code>: solve the minimum weight perfect matching  problem  with the <a href="https://github.com/mlewe/BlossomV.jl">BlossomV</a> algorithm.</li>
<li><code>minimum_weight_perfect_bmatching</code>: solve the minimum weight perfect b-matching problem by integer programming.</li>
</ul>
