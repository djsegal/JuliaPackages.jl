<p>Bcrypt.jl is an implementation <a href="http://www.usenix.org/event/usenix99/provos/provos.pdf" rel="nofollow">Provos and Mazi√®res's bcrypt adaptive hashing algorithm</a>.</p>
<p>This implementation was loosely transcribed from the <a href="https://golang.org" rel="nofollow">Go</a>(golang) implementation of the algorithm.
