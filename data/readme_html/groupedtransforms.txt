<p dir="auto"><code>GroupedTransforms.jl</code> provides the following fast algorithms:</p>
<ul dir="auto">
<li>nonequispaced fast transformation with exponential functions for grouped index sets based on the NFFT (non-equispaced fast Fourier transform)</li>
<li>nonequispaced fast transformation with cosine functions for grouped index sets based on the NFCT (non-equispaced fast cosine transform)</li>
</ul>
