<p dir="auto">ADMIXTURE.jl is a Julia wrapper of the popular <a href="http://dalexander.github.io/admixture/" rel="nofollow">ADMIXTURE program</a> for estimating ancestry in a model-based manner from large autosomal SNP genotype data sets.</p>
<p dir="auto">The methodology is described in following papers:</p>
<ol dir="auto">
<li>
<p dir="auto">D Alexander, J Novembre, and K Lange. (2009) Fast model-based estimation of ancestry in unrelated individuals, <em>Genome Research</em>, 19(9):1655â€“1664. <a href="https://dx.doi.org/10.1101/gr.094052.109" rel="nofollow">https://dx.doi.org/10.1101/gr.094052.109</a></p>
</li>
<li>
<p dir="auto">D Alexander and K Lange. (2011) Enhancements to the ADMIXTURE algorithm for individual
ancestry estimation. <em>BMC Bioinformatics</em>, 12:246. <a href="https://doi.org/10.1186/1471-2105-12-246" rel="nofollow">https://doi.org/10.1186/1471-2105-12-246</a></p>
</li>
<li>
<p dir="auto">S Shringarpure, C Bustamante, K Lange, and D Alexander. (2016) Efficient analysis of large datasets and sex bias with ADMIXTURE. <em>BMC Bioinformatics</em> 17:218. <a href="https://doi.org/10.1186/s12859-016-1082-x" rel="nofollow">https://doi.org/10.1186/s12859-016-1082-x</a></p>
</li>
</ol>
<p dir="auto">ADMIXTURE.jl requires Julia v1.3 or later. See documentation for usage. It is not yet registered and can be installed, in the Julia Pkg mode, by</p>
<div class="snippet-clipboard-content notranslate position-relative overflow-auto" data-snippet-clipboard-copy-content="(@v1.5) Pkg&gt; add https://github.com/OpenMendel/ADMIXTURE.jl"><pre lang="{julia}" class="notranslate"><code>(@v1.5) Pkg&gt; add https://github.com/OpenMendel/ADMIXTURE.jl
</code></pre></div>
<p dir="auto">The original ADMIXTURE software thus ADMIXTURE.jl only support Linux and MacOS.</p>
<p dir="auto">As a quick example, suppose a set of PLINK files <code>hapmap3.bed</code>, <code>hapmap3.bim</code>, <code>hapmap3.fam</code> are available in the current directory. Then</p>
<div class="highlight highlight-source-julia notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="P, Q = admixture(&quot;hapmap3.bed&quot;, 3)"><pre>P, Q <span class="pl-k">=</span> <span class="pl-c1">admixture</span>(<span class="pl-s"><span class="pl-pds">"</span>hapmap3.bed<span class="pl-pds">"</span></span>, <span class="pl-c1">3</span>)</pre></div>
<p dir="auto">returns the <code>P</code> matrix (allele frequencies of the inferred ancestral populations) and the <code>Q</code> matrix (the ancestry fractions). The output files by ADMIXTURE are saved in the working directory.</p>
<p dir="auto">For more details, check the documentation.</p>
