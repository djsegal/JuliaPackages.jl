<p dir="auto">This is a small package that renders LaTeX as SVG by utilizing the <code>dvisvgm</code> executable bundled with all major LaTeX distributions. It allows you to set custom preambles and is capable of rendering any LaTeX code that is considered valid by the <code>xelatex</code>, <code>pdflatex</code> or <code>lualatex</code> engine, and is thus much more flexible and powerful than javascript-based alternatives.</p>
<p dir="auto">Read <a href="https://ehbaozi.github.io/LatexSVG.jl/dev/" rel="nofollow">the documentation</a> for details.</p>
