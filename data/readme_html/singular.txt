<p>Julia package for using the <a href="https://www.singular.uni-kl.de/" rel="nofollow">Singular</a> library for commutative and
non-commutative algebra, algebraic geometry, and singularity theory.</p>
<p>To build Singular.jl, start julia and then type:</p>
<div class="highlight highlight-source-julia position-relative" data-snippet-clipboard-copy-content="julia&gt; using Pkg
julia&gt; Pkg.add(&quot;Singular&quot;)
"><pre>julia<span class="pl-k">&gt;</span> <span class="pl-k">using</span> Pkg

<div class="highlight highlight-source-julia position-relative" data-snippet-clipboard-copy-content="julia&gt; using Singular
