<p>Julia package for using the <a href="https://www.singular.uni-kl.de/" rel="nofollow">Singular</a> library for commutative and
non-commutative algebra, algebraic geometry, and singularity theory.</p>
<p>To build Singular.jl, start julia and then type:</p>
<div class="highlight highlight-source-julia"><pre>julia<span class="pl-k">&gt;</span> <span class="pl-k">using</span> Pkg
julia<span class="pl-k">&gt;</span> Pkg<span class="pl-k">.</span><span class="pl-c1">add</span>(<span class="pl-s"><span class="pl-pds">"</span>Singular<span class="pl-pds">"</span></span>)</pre></div>
<p>To use Singular.jl, start julia and then type:</p>
<div class="highlight highlight-source-julia"><pre>julia<span class="pl-k">&gt;</span> <span class="pl-k">using</span> Singular</pre></div>
