<p dir="auto">Provide various algorithms for approximate inference in latent Gaussian process models, currently focussing on non-conjugate (non-Gaussian) likelihoods and sparse approximations.</p>
<h2 dir="auto"><a id="user-content-structure" class="anchor" aria-hidden="true" href="#structure"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path d="m7.775 3.275 1.25-1.25a3.5 3.5 0 1 1 4.95 4.95l-2.5 2.5a3.5 3.5 0 0 1-4.95 0 .751.751 0 0 1 .018-1.042.751.751 0 0 1 1.042-.018 1.998 1.998 0 0 0 2.83 0l2.5-2.5a2.002 2.002 0 0 0-2.83-2.83l-1.25 1.25a.751.751 0 0 1-1.042-.018.751.751 0 0 1-.018-1.042Zm-4.69 9.64a1.998 1.998 0 0 0 2.83 0l1.25-1.25a.751.751 0 0 1 1.042.018.751.751 0 0 1 .018 1.042l-1.25 1.25a3.5 3.5 0 1 1-4.95-4.95l2.5-2.5a3.5 3.5 0 0 1 4.95 0 .751.751 0 0 1-.018 1.042.751.751 0 0 1-1.042.018 1.998 1.998 0 0 0-2.83 0l-2.5 2.5a1.998 1.998 0 0 0 0 2.83Z"></path></svg></a>Structure</h2>
<p dir="auto">Each approximation lives in its own submodule (<code>&lt;Approximation&gt;Module</code>), though
in general using the exported API is sufficient.</p>
<p dir="auto">The main API is:</p>
<ul dir="auto">
<li>
<p dir="auto"><code>posterior(approximation, lfx::LatentFiniteGP, ys)</code> to obtain the posterior
approximation to <code>lfx</code> conditioned on the observations <code>ys</code>.</p>
</li>
<li>
<p dir="auto"><code>approx_lml(approximation, lfx::LatentFiniteGP, ys)</code> which returns the
marginal likelihood approximation that can be used for hyperparameter
optimisation.</p>
</li>
</ul>
<p dir="auto">Currently implemented approximations:</p>
<ul dir="auto">
<li>
<p dir="auto"><code>LaplaceApproximation</code></p>
</li>
<li>
<p dir="auto"><code>SparseVariationalApproximation</code></p>
<p dir="auto">NOTE: requires optimisation of the variational distribution even for fixed
hyperparameters.</p>
</li>
</ul>
