<p dir="auto"><strong>QuantumCollocation.jl</strong> uses <a href="https://github.com/aarontrowbridge/NamedTrajectories.jl">NamedTrajectories.jl</a> to set up and solve direct collocation problems specific to quantum optimal control, i.e. problems of the form:</p>
<math-renderer class="js-display-math" style="display: block" data-static-url="https://github.githubassets.com/static" data-run-id="5a6455a21aa139b1a16c63eca30108c8">$$\begin{aligned}
\underset{U_{1:T}, a_{1:T-1}}{\text{minimize}} &amp; \quad \ell(U_T, U_{\text{goal}})\\\
\text{ subject to } &amp; \quad U_{t+1} = \exp(-i H(a_t)) U_t 
\end{aligned}$$</math-renderer>
<p dir="auto">QuantumCollocation.jl gives the user the ability to add other constraints and objective functions to this problem and solve it efficiently using <a href="https://github.com/jump-dev/Ipopt.jl">Ipopt.jl</a> and <a href="https://github.com/jump-dev/MathOptInterface.jl">MathOptInterface.jl</a> under the hood.</p>
