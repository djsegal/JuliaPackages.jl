<p dir="auto">A <a href="https://github.com/JuliaDocs/Documenter.jl">Documenter</a> plugin package that enables native support for <a href="https://mermaid.js.org/" rel="nofollow">mermaid.js</a> diagrams in Documenter documents (in HTML builds only).</p>
<p dir="auto">To enable the package, simply load it in the <code>make.jl</code> script. Any code blocks tagged as <code>mermaid</code> will be rendered as diagrams in the generated HTML.</p>
