<p>Es posible que desee compilar usted mismo el libro
a partir de los archivos fuente de este repositorio. Aquí
se describe el esquema general para ello:</p>
<ul>
<li>Descargue Julia y asegúrese de que puede iniciarlo desde una línea
de comandos como <code>bash</code>.</li>
<li>Asegúrese de tener tanto <code>ruby</code> (requerido por un asciidoctor) como<code> python</code> (usado por pygments).</li>
<li>Descargue asciidoctor, por ejemplo, utilizando <code>brew install asciidoctor</code> en MacOS.</li>
<li>Instale el paquete de python <code>pygments</code>, utilizando un comando como<code> pip install pygments</code>.</li>
<li>Clone este repositorio, por ejemplo con el comando:</li>
</ul>
<div class="highlight highlight-source-shell"><pre>git clone https://github.com/JuliaIntro/IntroAJulia.jl.git</pre></div>
<ul>
<li>En Julia, agregue el paquete <code>IntroAJulia</code> con un comando como el siguiente:</li>
</ul>
<div class="highlight highlight-source-julia"><pre>julia<span class="pl-k">&gt;</span> Pkg<span class="pl-k">.</span><span class="pl-c1">add</span>(<span class="pl-c1">PackageSpec</span>(url<span class="pl-k">=</span><span class="pl-s"><span class="pl-pds">"</span>https://github.com/JuliaIntro/IntroAJulia.jl.git<span class="pl-pds">"</span></span>))</pre></div>
<ul>
<li>Vaya al directorio <code>book</code> en el repositorio y ejecute el comando</li>
</ul>
<div class="highlight highlight-source-shell"><pre>julia --startup-file=no make.jl build html images</pre></div>
<p>Esto hará que el archivo <code>book/build/book.html</code>.</p>
