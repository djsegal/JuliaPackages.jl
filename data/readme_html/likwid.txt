<p dir="auto"><em>Like I Knew What I am Doing</em></p>
<p dir="auto">LIKWID.jl is a Julia wrapper for the performance monitoring and benchmarking suite <a href="https://github.com/RRZE-HPC/likwid">LIKWID</a>.</p>
<p dir="auto">

</p>

<p dir="auto">Prerequisites:</p>
<ul dir="auto">
<li>You must have <code>likwid</code> installed (see the <a href="https://github.com/RRZE-HPC/likwid#download-build-and-install">build &amp; install instructions</a>).</li>
<li><strong>You must be running Linux.</strong> (LIKWID doesn't support macOS or Windows.)</li>
</ul>
<p dir="auto">LIKWID.jl is a registered Julia package. Hence, you can simply add it to your Julia environment with the command</p>
