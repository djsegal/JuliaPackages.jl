<p>This package delivers <strong>bump functions</strong>, i.e. functions on the unit sphere that have infinitely many derivatives. Included functions cover the univariate and multivariate cases, gradients, and Hessians, with and without scale and location normalization.  For further information, see e.g. <a href="https://arxiv.org/abs/1508.04376" rel="nofollow">a paper by Steven Johnson</a>.</p>
<p>The formula used for the bump function without scale and location normalizations is simply</p>
<p>exp( 1 / ( ||x||^2 - 1 )  ) if ||x||&lt;1</p>
<p>and zero otherwise.</p>
