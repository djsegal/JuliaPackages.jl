<p>MelGeneralizedCepstrums.jl provides a mel generalized cepstrum anlysis for spectral envelope estimation that includes:</p>
<ul>
<li>linear predicition analysis (LPC)</li>
<li>generalized cepstrum analysis</li>
<li>mel-cepstrum analysis</li>
<li>mel-generalized cepstrum analysis</li>
</ul>
<p>The package also provides conversions, e.g, mel-generalized cepstrum to mel-cepstrum, mel-cepstrum to (linear frequency) cepstrum, mel-cesptrum to filter coefficients, and vise versa.</p>
<p>Note that this package is built on top of <a href="https://github.com/r9y9/SPTK.jl">SPTK.jl</a>. The package is designed to provide a Julia-like interface with a focus on the mel-generalized cepstrum analysis. A part of the core is re-writen in Julia from <a href="http://sp-tk.sourceforge.net/" rel="nofollow">Speech Signal Processing Toolkit (SPTK)</a>.</p>
