<p dir="auto">From the japanese word <a href="https://en.wikipedia.org/wiki/Maki-e" rel="nofollow"><em>Maki-e</em></a>, which is a technique to sprinkle lacquer with gold and silver powder.
Data is the gold and silver of our age, so let's spread it out beautifully on the screen!</p>
<p dir="auto"><a href="http://docs.makie.org/stable/" rel="nofollow">Check out the documentation here!</a></p>

<p dir="auto">Please consider using the backends directly. As explained in the documentation, they re-export all of Makie's functionality.
So, instead of installing Makie, just install e.g. GLMakie directly:</p>
<div class="highlight highlight-source-julia notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="julia&gt;]
pkg&gt; add GLMakie"><pre>julia<span class="pl-k">&gt;</span>]
pkg<span class="pl-k">&gt;</span> add GLMakie</pre></div>
<p dir="auto">You may check the installed version with:</p>
<div class="highlight highlight-source-julia notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="]st GLMakie"><pre>]st GLMakie</pre></div>
<p dir="auto">Start using the package:</p>
<div class="highlight highlight-source-julia notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="using GLMakie"><pre><span class="pl-k">using</span> GLMakie</pre></div>

<p dir="auto">Makie and its backends all live in the Makie monorepo.
This makes it easier to change code across all packages.
Therefore, dev'ing Makie almost works as with other Julia packages, just, that one needs to also dev the sub packages:</p>
<div class="highlight highlight-source-julia notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="]dev --local Makie # local will clone the repository at ./dev/Makie
]dev dev/Makie/MakieCore dev/Makie/GLMakie dev/Makie/CairoMakie dev/Makie/WGLMakie dev/Makie/RPRMakie"><pre>]dev <span class="pl-k">--</span><span class="pl-k">local</span> Makie <span class="pl-c"><span class="pl-c">#</span> local will clone the repository at ./dev/Makie</span>
]dev dev<span class="pl-k">/</span>Makie<span class="pl-k">/</span>MakieCore dev<span class="pl-k">/</span>Makie<span class="pl-k">/</span>GLMakie dev<span class="pl-k">/</span>Makie<span class="pl-k">/</span>CairoMakie dev<span class="pl-k">/</span>Makie<span class="pl-k">/</span>WGLMakie dev<span class="pl-k">/</span>Makie<span class="pl-k">/</span>RPRMakie</pre></div>
<p dir="auto">To run the tests, you also should add:</p>
<div class="highlight highlight-source-julia notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="]dev dev/Makie/ReferenceTests"><pre>]dev dev<span class="pl-k">/</span>Makie<span class="pl-k">/</span>ReferenceTests</pre></div>
<p dir="auto">For more info about ReferenceTests, check out its <a href="./ReferenceUpdater/README.md">README</a></p>
