<p dir="auto"><a href="https://rafaqz.github.io/Rasters.jl/dev/" rel="nofollow">Rasters.jl</a> provides excellent methods for reading, analysing and plotting for geospatial data.
This package contains modules that are useful for analysing oceanic data (either model output or gridded observations) and is designed to be used in conjunction with Rasters.jl.
The modules are:</p>
<ul dir="auto">
<li><a href="https://jbisits.github.io/OceanRasterConversions.jl/dev/modules/OceanVariableConversions/" rel="nofollow"><code>OceanVariableConversions</code></a>, conversion and computation (using <a href="https://www.teos-10.org/pubs/gsw/html/gsw_front_page.html" rel="nofollow">TEOS-10</a>) of ocean variables that are <code>Raster</code> data structures; and</li>
<li><a href="https://jbisits.github.io/OceanRasterConversions.jl/dev/modules/RasterHistograms/" rel="nofollow"><code>RasterHistograms</code></a>, empirical distribution fitting for <code>Raster</code> data structures aiming to provide functionality similar to python's <a href="https://xhistogram.readthedocs.io/en/latest/index.html" rel="nofollow">xhistogram</a> for <a href="https://docs.xarray.dev/en/stable/" rel="nofollow">xarray</a> in Julia.</li>
</ul>

<p dir="auto">The package is installed using Julia's package manager</p>
<div class="highlight highlight-source-julia notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="julia&gt; ]
(@v1.8) pkg&gt; add OceanRasterConversions"><pre>julia<span class="pl-k">&gt;</span> ]
(<span class="pl-c1">@v1</span>.<span class="pl-c1">8</span>) pkg<span class="pl-k">&gt;</span> add OceanRasterConversions</pre></div>
<p dir="auto">then press <code>backspace</code> to exit the package manager.
To start using the package you will also need to have <a href="https://github.com/rafaqz/Rasters.jl">Rasters.jl</a> installed (in the same manner as above but replace <code>OceanRasterConversions</code> with <code>Rasters</code>).
To then use the packages type</p>
<div class="highlight highlight-source-julia notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="julia&gt; using Rasters, OceanRasterConversions"><pre>julia<span class="pl-k">&gt;</span> <span class="pl-k">using</span> Rasters, OceanRasterConversions</pre></div>
<p dir="auto">into the repl.
The modules can also be loaded individually, to avoid loading in unnecessary funcitons into the workspace, by</p>
<div class="highlight highlight-source-julia notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="julia&gt; using OceanRasterConversions.RasterHistograms"><pre>julia<span class="pl-k">&gt;</span> <span class="pl-k">using</span> OceanRasterConversions<span class="pl-k">.</span>RasterHistograms</pre></div>
<p dir="auto">and similarly for <code>OceanRasterConversions.OceanVariableConversions</code>.</p>
