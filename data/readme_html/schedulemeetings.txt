<p dir="auto">Schedule weekly meetings for lists of presenters. The package is currently set up for two weekly meetings, internally called "lab meeting" and "journal club." It attempts to schedule with the following goals:</p>
<ul dir="auto">
<li>schedule each presenter once for each slated list (the lists of presenters in the two meetings do not have to be identical)</li>
<li>allow individuals to have target dates; for example, if a person is presenting in an external meeting, you might want to schedule a practice talk before (but not too far before) their official talk</li>
<li>avoid holidays and travel</li>
<li>try to avoid having the same person presenting in both meetings within the same 7-day period</li>
<li>try to preserve the order of presentation as passed in by the targets</li>
</ul>
<p dir="auto">It defines an objective function and uses evolutionary optimization where the mutation operation is to make swaps.</p>
