<p dir="auto"><strong>Important note</strong>: this is still a work on progress. The use of positive semidefinite matrices in linear equality constraints has not been implemented yet so only linear programs can be solved at the moment with DSDP.</p>
<p dir="auto">Julia wrapper for the <a href="http://www.mcs.anl.gov/hs/software/DSDP/" rel="nofollow">DSDP</a> semidefinite programming solver.</p>
