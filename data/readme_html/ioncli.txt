<p dir="auto">A CLI package manager for Julia.</p>
<p dir="auto"><strong>Disclaimer: It’s still at an early stage, even I have tested it myself in daily usage for a few weeks, I still expect it to have some glitches, and please file an issue to help improve it if you hit any.</strong></p>
<p dir="auto">For MacOS users, due to system image compilation issue, you will need to compile this package manually via <code>using IonCLI; IonCLI.comonicon_build()</code>.</p>
