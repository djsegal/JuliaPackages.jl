<p dir="auto">PDSampler.jl is a package designed to provide an efficient, flexible, and expandable framework for samplers based on <em>Piecewise Deterministic Markov Processes</em> and their applications.
This includes the <strong>Bouncy Particle Sampler</strong> and the <strong>Zig-Zag Sampler</strong>.</p>
<p dir="auto">Please refer to <a href="https://alan-turing-institute.github.io/PDSampler.jl/latest" rel="nofollow"><strong>the documentation</strong></a> for information on how to use/expand this package.
The project is hosted by the <a href="https://www.turing.ac.uk" rel="nofollow">Alan Turing Institute</a> (ATI). If you encounter problems, please <a href="https://github.com/alan-turing-institute/PDSampler.jl/issues">open an issue on Github</a>.
If you have comments or wish to collaborate, please send an email to <code>tlienart &gt; cpg σ gmail &gt; com</code>.</p>
<p dir="auto">If you find this toolbox useful please star the repo. If you use it in your work, please cite this code and send us an email so that we can cite your work here.</p>
<p dir="auto">If you want to make suggestions, if you want new features, please don't hesitate, open an issue or send an email.</p>

<p dir="auto">(This is explained in more details in the documentation)</p>
<p dir="auto">Requirements:</p>
<ul dir="auto">
<li>Julia ∈ <code>[0.7.*, 1.0.*]</code>, if you're on <code>0.6</code>, check out <a href="https://github.com/alan-turing-institute/PDSampler.jl/releases/tag/v0.1">the last legacy release</a>.</li>
</ul>
<p dir="auto">In the Julia REPL:</p>
<div class="highlight highlight-source-julia notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="] add PDSampler
using PDSampler"><pre>] add PDSampler
<span class="pl-k">using</span> PDSampler</pre></div>
<p dir="auto">Note that loading the package may take several seconds as some of the dependencies (in particular <a href="https://github.com/JuliaApproximation/ApproxFun.jl">ApproxFun.jl</a> are quite slow to load).</p>
