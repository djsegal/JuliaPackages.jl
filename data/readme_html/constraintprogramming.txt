<p>This Julia package allows us to specify a set of constraints on real-valued variables,
given by inequalities, and
rigorously calculate (inner and outer approximations to) the <em>feasible set</em>,
i.e. the set that satisfies the constraints.</p>
<p>The package is based on interval arithmetic using the
<a href="https://github.com/JuliaIntervals/IntervalArithmetic.jl"><code>IntervalArithmetic.jl</code></a> package (co-written by the author),
in particular multi-dimensional <code>IntervalBox</code>es (i.e. Cartesian products of one-dimensional intervals).</p>
