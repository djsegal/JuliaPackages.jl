<p dir="auto">This is a module written as a common sub module for other libraries under the <a href="https://github.com/EHTJulia">EHT Julia organization</a>. The module aims to implement</p>
<ul dir="auto">
<li>Quick shortcuts to popular physical or angular units and unit conversion functions based on <a href="https://github.com/PainterQubits/Unitful.jl">Unitful.Unit</a></li>
<li>Popular Conversion factors in EHT data analysis</li>
<li>Some useful shortcut functions for <a href="https://github.com/JuliaPy/PyPlot.jl">PyPlot.jl</a> or <a href="https://makie.juliaplots.org/stable/documentation/backends/glmakie/" rel="nofollow">Makie.jl</a></li>
</ul>
