<h2><a id="user-content-quickstart" class="anchor" aria-hidden="true" href="#quickstart"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path fill-rule="evenodd" d="M7.775 3.275a.75.75 0 001.06 1.06l1.25-1.25a2 2 0 112.83 2.83l-2.5 2.5a2 2 0 01-2.83 0 .75.75 0 00-1.06 1.06 3.5 3.5 0 004.95 0l2.5-2.5a3.5 3.5 0 00-4.95-4.95l-1.25 1.25zm-4.69 9.64a2 2 0 010-2.83l2.5-2.5a2 2 0 012.83 0 .75.75 0 001.06-1.06 3.5 3.5 0 00-4.95 0l-2.5 2.5a3.5 3.5 0 004.95 4.95l1.25-1.25a.75.75 0 00-1.06-1.06l-1.25 1.25a2 2 0 01-2.83 0z"></path></svg></a>Quickstart</h2>
<p>Fetch the latest <a href="https://www.iers.org/IERS/EN/DataProducts/EarthOrientationData/eop.html" rel="nofollow">IERS</a> tables:</p>
<div class="highlight highlight-source-julia"><pre><span class="pl-k">using</span> EarthOrientation
EarthOrientation<span class="pl-k">.</span><span class="pl-c1">update</span>()</pre></div>
<p>Get the current Earth orientation parameters, e.g. for polar motion:</p>
<div class="highlight highlight-source-julia"><pre>xp, yp <span class="pl-k">=</span> <span class="pl-c1">polarmotion</span>(<span class="pl-c1">now</span>()) <span class="pl-c"><span class="pl-c">#</span> arcseconds</span></pre></div>
<p>Or the current difference between UT1 and UTC and the associated prediction error:</p>
<div class="highlight highlight-source-julia"><pre>ΔUT1 <span class="pl-k">=</span> getΔ<span class="pl-c1">UT1</span>(<span class="pl-c1">now</span>()) <span class="pl-c"><span class="pl-c">#</span> seconds</span>
ΔUT1_err <span class="pl-k">=</span> getΔ<span class="pl-c1">UT1_err</span>(<span class="pl-c1">now</span>()) <span class="pl-c"><span class="pl-c">#</span> seconds</span></pre></div>
