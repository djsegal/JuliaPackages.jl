<div dir="auto">
<p dir="auto">This is a "backend" package that is used for computing contours, filled contours, and pseudocolor plots of data on unstructured triangular meshes.</p>
</div>
<div dir="auto">
<p dir="auto">Users should generally use one of the frontend packages that integrate with packages in the Julia plotting ecosystem:</p>
</div>
<div dir="auto">
<ul dir="auto">
<li>
<p dir="auto">For use with Plots.jl, use <a href="https://github.com/pazner/TriplotRecipes.jl">TriplotRecipes.jl</a>.</p>
</li>
<li>
<p dir="auto">For use with GR, use <a href="https://github.com/pazner/TriplotGR.jl">TriplotGR.jl</a>.</p>
</li>
</ul>
