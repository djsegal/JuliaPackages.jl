<p dir="auto">The API consists of the following functions:</p>
<ul dir="auto">
<li><code>cmd([args])</code> returns a command which calls MicroMamba, optionally with given arguments.</li>
<li><code>executable()</code> returns the path to a MicroMamba executable.</li>
<li><code>version()</code> returns the version of this executable.</li>
<li><code>available()</code> returns true if MicroMamba is available on this system. Use this to check if the above functions will succeed.</li>
</ul>
<p dir="auto">In all cases, MicroMamba will be downloaded and installed if required to a Julia-specific location.</p>
<p dir="auto">The command returned from <code>cmd()</code> includes the root prefix <code>-r</code> argument. By default this is some Julia-specific directory, but can be over-ridden with the environment variable <code>MAMBA_ROOT_PREFIX</code>.</p>
