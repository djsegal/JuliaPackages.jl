<p>To price an European option, simply create a new <code>EuropeanOption</code> and pass it to <code>evaluate</code> with the desired valuation model:</p>
<div class="highlight highlight-source-julia"><pre>julia<span class="pl-k">&gt;</span> <span class="pl-k">using</span> FinancialDerivatives

julia<span class="pl-k">&gt;</span> euro_put <span class="pl-k">=</span> <span class="pl-c1">EuropeanOption</span>(<span class="pl-c1">100.0</span>, <span class="pl-c1">90.0</span>, <span class="pl-c1">0.05</span>, <span class="pl-c1">0.3</span>, <span class="pl-c1">180</span><span class="pl-k">/</span><span class="pl-c1">365</span>, <span class="pl-k">-</span><span class="pl-c1">1</span>)

julia<span class="pl-k">&gt;</span> <span class="pl-c1">evaluate</span>(euro_put, <span class="pl-c1">BlackScholes</span>())
<span class="pl-c1">3.2281936525908073</span></pre></div>
