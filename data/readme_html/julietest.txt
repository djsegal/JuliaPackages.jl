<p>Run in Julia REPL:</p>
<div class="highlight highlight-source-julia position-relative" data-snippet-clipboard-copy-content="Pkg.add(&quot;JulieTest&quot;)
"><pre>Pkg<span class="pl-k">.</span><span class="pl-c1">add</span>(<span class="pl-s"><span class="pl-pds">"</span>JulieTest<span class="pl-pds">"</span></span>)</pre></div>
<p>This command will install JulieTest into <code>~/.julia/&lt;your-julia-version&gt;/JulieTest</code>.</p>
<p>If you want to access <code>julie</code> command from other folders, you may want to add <code>bin/julie</code> command to your <code>PATH</code>. You can do that by adding the following lines to your <code>~/.bashrc</code> file:</p>
<div class="highlight highlight-source-shell position-relative" data-snippet-clipboard-copy-content="# Julie test

PATH=&quot;$PATH:$JULIETEST_HOME&quot;

  it(&quot;above&quot;) do
    @is 10 =&gt; above 9
  end

  it(&quot;builtin functions&quot;) do
    @is 10 =&gt; isa Number
  end

  # _it will be ignored
  _it(&quot;should fail&quot;) do
    @is true =&gt; falsy
  end

  # iit will make the test ignore the it
  # and only run the iit
  iit(&quot;the test should only run this&quot;) do
    @is true =&gt; not not truthy
  end

  # describe can be recursive
  describe(&quot;Array&quot;) do
    it(&quot;should not empty&quot;) do
      @is [1:5] =&gt; not empty
    end
  end
end
