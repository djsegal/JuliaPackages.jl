<p><a href="https://github.com/Xh4H">Xh4H</a> and I previously implemented <a href="https://github.com/Xh4H/Discord.jl">another Discord.jl package</a>, but it was never published in a Julia registry for several reasons.
This fresh implementation aims to keep things as simple as possible and minimize extra features in order to maximize maintainability as the Discord API changes.
It also aims to be easily extensible, so that packages providing extra features can be written on top of it.</p>
