<p>This is a <a href="http://www.julialang.org" rel="nofollow">Julia</a> interface to the sequential version of the MUltifrontal Massively Parallel sparse direct Solver <a href="http://mumps.enseeiht.fr" rel="nofollow">MUMPS</a>.</p>
<p>A <a href="https://github.com/lruthotto/MUMPS.jl">MUMPS package</a> for Julia using the MPI interface is already registered.</p>
<p>Travis tests are currently failing, perhaps because the version of libdmumps_seq on Travis's Ubuntu 12.04 systems, is out-of-date?  Will need to create a more fully-featured <code>deps/build.jl</code>.</p>
