<p dir="auto">This package is a <a href="https://github.com/JuliaLang/julia">Julia</a> extension package to
<a href="https://github.com/JuliaDSP/Wavelets.jl">Wavelets.jl</a> (WaveletsExt is short for Wavelets
Extension). It contains additional functionalities that complement Wavelets.jl, namely</p>
<ul dir="auto">
<li>Multi-dimensional wavelet transforms</li>
<li>Redundant wavelet transforms
<ul dir="auto">
<li><a href="https://www.spiedigitallibrary.org/conference-proceedings-of-spie/1826/1/Wavelets-their-autocorrelation-functions-and-multiresolution-representations-of-signals/10.1117/12.131585.short" rel="nofollow">Autocorrelation Wavelet Transforms (ACWT)</a></li>
<li><a href="https://doi.org/10.1007/978-1-4612-2544-7_17" rel="nofollow">Stationary Wavelet Transforms (SWT)</a></li>
<li><a href="https://doi.org/10.1016/S0165-1684(97)00007-8" rel="nofollow">Shift Invariant Wavelet Transforms (SIWT)</a></li>
</ul>
</li>
<li>Best basis algorithms
<ul dir="auto">
<li><a href="https://ieeexplore.ieee.org/document/119732" rel="nofollow">Joint best basis (JBB)</a></li>
<li><a href="https://ieeexplore.ieee.org/document/750958" rel="nofollow">Least statistically dependent basis (LSDB)</a></li>
</ul>
</li>
<li>Denoising methods
<ul dir="auto">
<li><a href="https://ieeexplore.ieee.org/document/7752982" rel="nofollow">Relative Error Shrink (RelErrorShrink)</a></li>
<li><a href="https://www.tandfonline.com/doi/abs/10.1080/01621459.1995.10476626" rel="nofollow">Stein Unbiased Risk Estimator Shrink (SUREShrink)</a></li>
</ul>
</li>
<li>Wavelet transform based feature extraction techniques
<ul dir="auto">
<li><a href="https://www.spiedigitallibrary.org/conference-proceedings-of-spie/2303/1/Local-discriminant-bases/10.1117/12.188763.short" rel="nofollow">Local Discriminant Basis (LDB)</a></li>
</ul>
</li>
</ul>
