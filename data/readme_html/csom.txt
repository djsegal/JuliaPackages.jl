<h2><a id="user-content-the-programming-the-finite-element-method-toolkit" class="anchor" aria-hidden="true" href="#the-programming-the-finite-element-method-toolkit"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path fill-rule="evenodd" d="M7.775 3.275a.75.75 0 001.06 1.06l1.25-1.25a2 2 0 112.83 2.83l-2.5 2.5a2 2 0 01-2.83 0 .75.75 0 00-1.06 1.06 3.5 3.5 0 004.95 0l2.5-2.5a3.5 3.5 0 00-4.95-4.95l-1.25 1.25zm-4.69 9.64a2 2 0 010-2.83l2.5-2.5a2 2 0 012.83 0 .75.75 0 001.06-1.06 3.5 3.5 0 00-4.95 0l-2.5 2.5a3.5 3.5 0 004.95 4.95l1.25-1.25a.75.75 0 00-1.06-1.06l-1.25 1.25a2 2 0 01-2.83 0z"></path></svg></a>The "Programming the Finite Element Method" toolkit</h2>
<p>This Julia package currently contains the programs in chapters 4, 5 and early sections of 6 as described in "Programming the Finite Element Method" by I M Smith, D V Griffiths and L. Margetts (<a href="http://www.wiley.com/WileyCDA/WileyTitle/productCd-1119973341.html" rel="nofollow">PtFEM</a>).</p>
<p>I use PtFEM when referring to the book and PtFEM.jl when referring to the Julia package.  The authors and publisher have given permission to publish the Julia version of the PtFEM toolkit. Please refer to <a href="https://github.com/PtFEM/PtFEM.jl/blob/master/LICENSE.md">LICENSE</a> for more details.</p>

<p>Fundamental and great development work related to solving (partial) differential equations is done in several other Julia packages, e.g. <a href="https://github.com/JuliaApproximation/ApproxFun.jl">ApproxFun.jl</a>, <a href="https://github.com/JuliaDiffEq/DifferentialEquations.jl">DifferentialEquations.jl</a>, <a href="http://www.juliafem.org" rel="nofollow">JuliaFEM.jl</a> and  <a href="https://github.com/KristofferC/JuAFEM.jl">JuaFEM.jl</a> to name a few.</p>
<p>Outside of Julia at least 2 other toolkits should be mentioned, i.e.  <a href="http://dealii.org" rel="nofollow">deal.II</a> and <a href="https://fenicsproject.org" rel="nofollow">FEniCS</a>.</p>
