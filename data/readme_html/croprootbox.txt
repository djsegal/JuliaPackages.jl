<p dir="auto"><a href="https://github.com/cropbox/CropRootBox.jl">CropRootBox.jl</a> implements a root system architecture simulation algorithm described in <a href="https://plant-root-soil-interactions-modelling.github.io/CRootBox/" rel="nofollow">CRootBox</a> model. Our implementation is written in a domain-specific language based on Julia using <a href="https://github.com/cropbox/Cropbox.jl">Cropbox</a> framework. While Cropbox framework was primarily designed for helping development of conventional process-based crop models with less dynamic structural development in mind, it is still capable of handling complex structure as envisioned by functional-structural plant models (FSPM).</p>
