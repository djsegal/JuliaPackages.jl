<p dir="auto">This package provides data types and functions for working with <a href="https://en.wikipedia.org/wiki/B-spline" rel="nofollow">B-splines</a> as a means to approximate real functions.
For its usage, see the <a href="https://sostock.github.io/BSplines.jl/stable" rel="nofollow">documentation</a>.</p>
<p dir="auto"><strong>This package is not related to the package at <a href="https://github.com/gusl/BSplines.jl">https://github.com/gusl/BSplines.jl</a>.
While <a href="https://github.com/gusl/BSplines.jl">gusl/BSplines.jl</a> is only compatible with Julia ≤ 0.4, this package works with Julia ≥ 1.0.</strong></p>
