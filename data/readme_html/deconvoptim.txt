<br>
A package for microscopy image based deconvolution via Optim.jl. This package works with N dimensional Point Spread Functions and images.
The package was created with microscopy in mind but since the code base is quite general it is possible to deconvolve different kernels as well.

<p dir="auto">Have a quick look into the <a href="examples">examples folder</a>.
We demonstrate the effect of different regularizers. There is also a <a href="examples/cuda_2D.ipynb">CUDA example</a>.

<p dir="auto">For CUDA we only provide a Total variation regularizer via <code>TV_cuda</code>. The reason is that Tullio.jl is currently not very fast with <code>CuArray</code>s and especially
the derivative of such functions.</p>

<p dir="auto">The regularizers are generated with metaprogramming when <code>TV()</code> (or any other regularizer) is called. To prevent that the code
compile every time again, define the regularizer once and use it multiple times without newly defining it:</p>
<div class="highlight highlight-source-julia notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="reg = TV()"><pre>reg <span class="pl-k">=</span> <span class="pl-c1">TV</span>()</pre></div>
<p dir="auto">And in the new cell then use:</p>
<div class="highlight highlight-source-julia notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="res, o = deconvolution(img_n, psf, regularizer=reg)"><pre>res, o <span class="pl-k">=</span> <span class="pl-c1">deconvolution</span>(img_n, psf, regularizer<span class="pl-k">=</span>reg)</pre></div>
