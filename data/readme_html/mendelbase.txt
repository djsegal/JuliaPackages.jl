<p dir="auto">MendelBase includes all the base functions of <a href="https://openmendel.github.io" rel="nofollow">OpenMendel</a>. It includes useful utilities such as functions to process keywords that specify the data files to use and the analysis options to perform, and functions to read data from external files.</p>

<p dir="auto"><em>Note: The three OpenMendel packages (1) <a href="https://openmendel.github.io/SnpArrays.jl/latest/" rel="nofollow">SnpArrays</a>, (2) <a href="https://openmendel.github.io/MendelSearch.jl" rel="nofollow">MendelSearch</a>, and (3) <a href="https://openmendel.github.io/MendelBase.jl" rel="nofollow">MendelBase</a> must be installed before any other OpenMendel package will run. It is easiest if these three packages are installed in the above order and before any other OpenMendel package.</em></p>
<p dir="auto">Within Julia, use the package manager to install MendelBase:</p>
<div class="snippet-clipboard-content notranslate position-relative overflow-auto" data-snippet-clipboard-copy-content="pkg&gt; add https://github.com/OpenMendel/MendelBase.jl"><pre class="notranslate"><code>pkg&gt; add https://github.com/OpenMendel/MendelBase.jl
</code></pre></div>
<p dir="auto">This package supports Julia v1.0+</p>
