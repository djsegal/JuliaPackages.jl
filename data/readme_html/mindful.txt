<p dir="auto"><code>MINDFul.jl</code> is a young project aiming to research coordination algorithms of intent-driven multi-domain (MD) networks.
It offers interfaces for the development of resource-allocation algorithms and MD coordination mechanisms.</p>
<p dir="auto">For now, <code>MINDFul.jl</code> only supports simulation, but <a href="https://unistuttgart-ikr.github.io/MINDFul.jl/dev/roadmap/" rel="nofollow">future work</a> may bind <code>MINDFul.jl</code> with real-life Software Defined Network (SDN) controllers like <a href="https://www.opendaylight.org/" rel="nofollow">ODL</a> or <a href="https://opennetworking.org/onos/" rel="nofollow">ONOS</a></p>
<p dir="auto">This package is in an early development stage and might break often.</p>
<p dir="auto">Companion repositories:</p>
<ul dir="auto">
<li><a href="https://github.com/UniStuttgart-IKR/MINDFulMakie.jl">MINDFulMakie.jl</a> for visualization purposes</li>
<li><a href="https://github.com/UniStuttgart-IKR/MINDFulCompanion.jl">MINDFulCompanion.jl</a> planned for several algorithms and utilities.</li>
<li><a href="https://github.com/UniStuttgart-IKR/MINDFulNotebookExamples.jl">MINDFulNotebookExamples</a> for some examples in the form of <a href="https://github.com/fonsp/Pluto.jl">Pluto</a> notebooks.<br>
For example, <a href="https://unistuttgart-ikr.github.io/MINDFulNotebookExamples.jl/intentDAGinMD.html" rel="nofollow">have a look</a> in a notebook related to our latest work on using intent DAGs in multi-domain IP-optical networks.</li>
</ul>
<p dir="auto">Currently, as you will find in <a href="https://github.com/UniStuttgart-IKR/MINDFulNotebookExamples.jl">MINDFulNotebookExamples</a> there are some workflow dependencies on forked/unregistered packages due to pending PRs. These will be handled in time.</p>
