<h2><a id="user-content-example-usage" class="anchor" aria-hidden="true" href="#example-usage"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path fill-rule="evenodd" d="M7.775 3.275a.75.75 0 001.06 1.06l1.25-1.25a2 2 0 112.83 2.83l-2.5 2.5a2 2 0 01-2.83 0 .75.75 0 00-1.06 1.06 3.5 3.5 0 004.95 0l2.5-2.5a3.5 3.5 0 00-4.95-4.95l-1.25 1.25zm-4.69 9.64a2 2 0 010-2.83l2.5-2.5a2 2 0 012.83 0 .75.75 0 001.06-1.06 3.5 3.5 0 00-4.95 0l-2.5 2.5a3.5 3.5 0 004.95 4.95l1.25-1.25a.75.75 0 00-1.06-1.06l-1.25 1.25a2 2 0 01-2.83 0z"></path></svg></a>Example usage</h2>
<p>To use the NormalSplines package, begin your code with</p>
<div class="snippet-clipboard-content position-relative" data-snippet-clipboard-copy-content="using NormalSplines
"><pre><code>using NormalSplines
</code></pre></div>
<p>Construct a normal spline to some function values (knots <code>x</code> can be unevenly distributed):</p>
<div class="snippet-clipboard-content position-relative" data-snippet-clipboard-copy-content="using NormalSplines

x = [0.0, 1.0, 2.0]
u = [0.0, 1.0, 4.0]
interpolate(x, u, RK_W3())
"><pre><code>using NormalSplines

x = [0.0, 1.0, 2.0]
u = [0.0, 1.0, 4.0]
interpolate(x, u, RK_W3())
</code></pre></div>
<p>here <code>RK_W3()</code> is the reproducing kernel of Sobolev space <code>W_2^3[0,2]</code>.</p>
<p>Evaluate the spline, its first and second derivatives at some points:</p>
<div class="snippet-clipboard-content position-relative" data-snippet-clipboard-copy-content="p = [0.0, 0.5, 1.0, 1.5, 2.0]

σ = evaluate(p)       # result = [0.0, 0.26, 1.0, 2.24, 4.0]
σ = evaluate(1.5)     # result = 2.24

σ' = evaluate(p, 1)  # result = [0.08, 0.98, 1.96, 3.0, 4.04]

σ'' = evaluate(p, 2) # result = [1.71, 1.89, 2.02, 2.08, 2.08]
"><pre><code>p = [0.0, 0.5, 1.0, 1.5, 2.0]

σ = evaluate(p)       # result = [0.0, 0.26, 1.0, 2.24, 4.0]
σ = evaluate(1.5)     # result = 2.24

σ' = evaluate(p, 1)  # result = [0.08, 0.98, 1.96, 3.0, 4.04]

σ'' = evaluate(p, 2) # result = [1.71, 1.89, 2.02, 2.08, 2.08]
</code></pre></div>
<p>Construct a normal spline to some function and its first and second derivatives values:</p>
<div class="snippet-clipboard-content position-relative" data-snippet-clipboard-copy-content="using NormalSplines

x = [0.0, 1.0, 2.0] # Function knots
u = [0.0, 1.0, 4.0] # Function values 
s = [2.0]           # First derivative knot
v = [4.0]           # First derivative value
t = [0.0, 1.0]      # Second derivative knots
w = [2.0 ,2.0]      # Second derivative values
interpolate(x, u, s, v, t, w, RK_W3())
"><pre><code>using NormalSplines

x = [0.0, 1.0, 2.0] # Function knots
u = [0.0, 1.0, 4.0] # Function values 
s = [2.0]           # First derivative knot
v = [4.0]           # First derivative value
t = [0.0, 1.0]      # Second derivative knots
w = [2.0 ,2.0]      # Second derivative values
interpolate(x, u, s, v, t, w, RK_W3())
</code></pre></div>
<p>Evaluate the spline, its first and second derivatives at some points:</p>
<div class="snippet-clipboard-content position-relative" data-snippet-clipboard-copy-content="p = [0.0, 0.5, 1.0, 1.5, 2.0]

σ = evaluate(p)       # result = [0.0, 0.25, 1.0, 2.25, 4.0]

σ' = evaluate(p, 1) # result = [0.0, 1.0, 2.0, 3.0, 4.0]

σ'' = evaluate(p, 2) # result = [2.0, 2.0, 2.0, 2.0, 2.0]
"><pre><code>p = [0.0, 0.5, 1.0, 1.5, 2.0]

σ = evaluate(p)       # result = [0.0, 0.25, 1.0, 2.25, 4.0]

σ' = evaluate(p, 1) # result = [0.0, 1.0, 2.0, 3.0, 4.0]

σ'' = evaluate(p, 2) # result = [2.0, 2.0, 2.0, 2.0, 2.0]
</code></pre></div>
