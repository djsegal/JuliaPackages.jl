<p dir="auto">A Julia package for random variables and probabilities as an add-on to the
Distributions.jl package. Scope of the package:</p>
<ul dir="auto">
<li>Defining a random variable <code>X = RV(Normal(0, 1)); Y = RV(Poisson(4))</code></li>
<li>Transformation of random variables: <code>Z = log(Y + 1)</code></li>
<li>Convenient probability computation: <code>P(X &lt; 2)</code></li>
<li>Combination of events: <code>A = (X &lt; 2) &amp; (Y ≥ 2); B = (X ≤ 5) &amp; (Y &gt; 0)</code></li>
<li>Defining conditional events: <code>P(A|B)</code></li>
</ul>

<h4 dir="auto"><a id="user-content-transformation-of-random-variables" class="anchor" aria-hidden="true" href="#transformation-of-random-variables"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path d="m7.775 3.275 1.25-1.25a3.5 3.5 0 1 1 4.95 4.95l-2.5 2.5a3.5 3.5 0 0 1-4.95 0 .751.751 0 0 1 .018-1.042.751.751 0 0 1 1.042-.018 1.998 1.998 0 0 0 2.83 0l2.5-2.5a2.002 2.002 0 0 0-2.83-2.83l-1.25 1.25a.751.751 0 0 1-1.042-.018.751.751 0 0 1-.018-1.042Zm-4.69 9.64a1.998 1.998 0 0 0 2.83 0l1.25-1.25a.751.751 0 0 1 1.042.018.751.751 0 0 1 .018 1.042l-1.25 1.25a3.5 3.5 0 1 1-4.95-4.95l2.5-2.5a3.5 3.5 0 0 1 4.95 0 .751.751 0 0 1-.018 1.042.751.751 0 0 1-1.042.018 1.998 1.998 0 0 0-2.83 0l-2.5 2.5a1.998 1.998 0 0 0 0 2.83Z"></path></svg></a>Transformation of random variables</h4>
<p dir="auto">Random variables can be transformed. Say you transform a random variable X by
a function h(x), i.e. Z = f(X). The object <code>RVtransformed</code> stores the distribution
of the original random variable X, its ID, the function h and the inverse of the transformation
function. Implemented transformations for a real valued <code>y</code> are: <code>X + y</code>,
<code>X - y</code>, <code>X * y</code>, <code>X / y</code>, <code>inv(X)</code>, <code>abs(X)</code>, <code>exp(X)</code>, <code>log(X)</code>,
<code>sqrt(X)</code> and <code>X^y</code>.</p>

<p dir="auto">To compute probabilities, events can be defined as a combination of independent
random variables and bounding intervals. For example <code>X &gt; 1</code> creates an event
with random variable X and the interval (1, ∞). The bounding intervals can be
closed [a, b], open (a, b) or semi-closed [a, b) or (a, b]. Two events can be
combined with operators</p>
<ul dir="auto">
<li><code>&amp;</code> : both events occur</li>
<li><code>∨</code> : Either one event occurs or both</li>
<li><code>\</code> : Left event occurs but not the right event</li>
<li><code>⊻</code> : One event occurs but not the other</li>
<li><code>!</code> : The complementary event</li>
</ul>
<p dir="auto">Examples for events:</p>
<div class="snippet-clipboard-content notranslate position-relative overflow-auto" data-snippet-clipboard-copy-content="A = X &gt; 1
B = X ≤ 3
C = X ≤ 0"><pre class="notranslate"><code>A = X &gt; 1
B = X ≤ 3
C = X ≤ 0
</code></pre></div>
<ul dir="auto">
<li>A &amp; B : (1, 3]</li>
<li>A ∨ C : (-∞, 0] ∪ (1, ∞)</li>
<li>A \ B : (3, ∞)</li>
<li>A ⊻ B : (-∞, 1] ∪ (3, ∞)</li>
<li>!(A) : (-∞, 1]</li>
</ul>
<p dir="auto">Events can include multiple independent random variables. For n variables, the
bounding intervals will then be n dimensional rectangles (boxes). For example
<code>(X &lt; 1) &amp; (X ≥ 0) &amp; (Y ≤ 1) &amp; (Y &gt; 0)</code> creates an event with random variables
X and Y with bounding rectangle [0, 1) × (0, 1].</p>

<p dir="auto">With two events, conditional events can be defined. A|B gives the event that
event A occurs given that B occurs. The random variables in A do not necessarily
need to match the random variables in B.</p>

can also be applied to random variables. Further, a short notation for the mean
<code>E(X)</code> is added. The mean of a transformed random variable is computed by numerical integration / summation.</p>
