<h2><a id="user-content-purpose" class="anchor" aria-hidden="true" href="#purpose"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path fill-rule="evenodd" d="M7.775 3.275a.75.75 0 001.06 1.06l1.25-1.25a2 2 0 112.83 2.83l-2.5 2.5a2 2 0 01-2.83 0 .75.75 0 00-1.06 1.06 3.5 3.5 0 004.95 0l2.5-2.5a3.5 3.5 0 00-4.95-4.95l-1.25 1.25zm-4.69 9.64a2 2 0 010-2.83l2.5-2.5a2 2 0 012.83 0 .75.75 0 001.06-1.06 3.5 3.5 0 00-4.95 0l-2.5 2.5a3.5 3.5 0 004.95 4.95l1.25-1.25a.75.75 0 00-1.06-1.06l-1.25 1.25a2 2 0 01-2.83 0z"></path></svg></a><code>Purpose</code></h2>
<p>MathPhysicalConstants is a Julia package which has the values of a range of mathematical and physical constants updated with most recent available dataset from BIPM in 2018. Currently <a href="https://en.wikipedia.org/wiki/MKS_system_of_units" rel="nofollow">MKS</a> and <a href="https://en.wikipedia.org/wiki/Centimetre%E2%80%93gram%E2%80%93second_system_of_units" rel="nofollow">CGS</a> units and <a href="https://en.wikipedia.org/wiki/International_System_of_Units" rel="nofollow">International System of Units</a>. Updated CODATA 2017 and redefinition of SI base units following <a href="https://www1.bipm.org/utils/common/pdf/CGPM-2018/26th-CGPM-Resolutions.pdf" rel="nofollow">ISU, BIPM, CGPM, CIPM</a> approved resolutions and scheduled to come into force on 20 May 2019.
See <a href="https://en.wikipedia.org/wiki/2019_redefinition_of_SI_base_units" rel="nofollow">https://en.wikipedia.org/wiki/2019_redefinition_of_SI_base_units</a></p>
