<p><a href="https://github.com/PetrKryslUCSD/FinEtools.jl.git"><code>FinEtools</code></a> is a package
for basic operations on finite element meshes. <code>FinEtoolsAcoustics</code> is a package
using <code>FinEtools</code> to solve linear acoustics problems.</p>

<p>Clone the package to your working directory:</p>
<div class="snippet-clipboard-content position-relative" data-snippet-clipboard-copy-content="git clone https://github.com/PetrKryslUCSD/FinEtoolsAcoustics.jl.git
"><pre><code>git clone https://github.com/PetrKryslUCSD/FinEtoolsAcoustics.jl.git
</code></pre></div>
<p>Change your working directory to <code>FinEtoolsAcoustics</code>. Start Julia and run
the following:</p>
<div class="snippet-clipboard-content position-relative" data-snippet-clipboard-copy-content="using Pkg; Pkg.activate(&quot;.&quot;); Pkg.instantiate()
"><pre><code>using Pkg; Pkg.activate("."); Pkg.instantiate()
</code></pre></div>
<p>The tutorials are in the form of Julia scripts in the folder <code>FinEtoolsAcoustics.jl/docs/src/tutorials</code>. The markdown generated from these files is also in the same folder.</p>
<p>To view the markdown, follow the link to the documentation.</p>
<p>To run a tutorial, head over to the <code>tutorials</code> folder, open the tutorial script, and evaluate in Julia.</p>

<p>There are a number of examples covering modal analysis, steady-state, and
transient acoustics. The examples may be executed as described in the
<a href="https://petrkryslucsd.github.io/FinEtools.jl/latest" rel="nofollow">conceptual guide to
