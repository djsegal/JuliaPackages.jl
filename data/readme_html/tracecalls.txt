<p>TraceCalls.jl is a functional tracing package for <a href="http://nbviewer.jupyter.org/github/cstjean/TraceCalls.jl/blob/master/README.ipynb#Manipulating-traces" rel="nofollow">exploring</a>, <a href="http://nbviewer.jupyter.org/github/cstjean/TraceCalls.jl/blob/master/README.ipynb#Debugging-with-traces" rel="nofollow">debugging</a> and <a href="http://nbviewer.jupyter.org/github/cstjean/TraceCalls.jl/blob/master/README.ipynb#Profiling" rel="nofollow">profiling</a> Julia code, built on top of <a href="https://github.com/timholy/Revise.jl">Revise.jl</a>. It supports the REPL, <a href="https://github.com/JuliaLang/IJulia.jl">IJulia</a> and <a href="http://junolab.org/" rel="nofollow">Atom</a>.</p>

<p>TraceCalls.jl imports <a href="https://github.com/timholy/Revise.jl">Revise.jl</a>,
which triggers automatic reloading behaviour for subsequently-loaded packages. This
should be fine for most users, but you can <a href="https://github.com/timholy/Revise.jl#manual-revision">turn it
off</a> if it's an issue.</p>
