<p dir="auto">An attempt at a unified framework in Julia language for both event-driven backtesting and live trading. This package utilizes reactive programming elements implemented in <a href="https://github.com/JuliaLang/Reactive.jl">Reactive.jl</a>.</p>
<p dir="auto">The main objective is to be able to backtest and place real-time orders using the same trading strategy functions.</p>
<p dir="auto">NOTE: The current <code>master</code>-branch code requires <code>Julia &gt;=0.4.0</code>. See the <code>julia03</code> branch for the last version tested with Julia 0.3 before <a href="https://github.com/JuliaLang/julia/releases">0.4.0 final release</a>.</p>
