<p dir="auto">DiffEqBase.jl is a component package in the DiffEq ecosystem. It holds the
common types and utility functions which are shared by other solver packages
to promote code reuse in the differential equation solver code.
Users interested in using this
functionality in full should check out <a href="https://github.com/JuliaDiffEq/DifferentialEquations.jl">DifferentialEquations.jl</a></p>
<p dir="auto">The documentation for the interfaces here can be found in <a href="https://diffeq.sciml.ai/dev/" rel="nofollow">DiffEqDocs.jl</a> and <a href="https://devdocs.sciml.ai/dev" rel="nofollow">DiffEqDevDocs.jl</a>.</p>
