<p dir="auto">Open-source time-Temperature inversion of thermochronometric data.</p>
<p dir="auto">Implements a transdimensional Markov chain Monte Carlo (MCMC) time-Temperature inversion with optional Simulated Annealing (e.g., <a href="https://en.wikipedia.org/wiki/Simulated_annealing" rel="nofollow">1</a>, <a href="https://doi.org/10.1007/978-94-015-7744-1_2" rel="nofollow">2</a>) using a Crank-Nicholson diffusion model following the equations of <a href="https://doi.org/10.2138/rmg.2005.58.11" rel="nofollow">Ketcham, 2005</a>.</p>
<p dir="auto">This package currently supports zircon helium data with the damage and annealing model of <a href="https://doi.org/10.2475/03.2013.01" rel="nofollow">Guenthner et al., 2013</a>, with additional systems and models expected to be added in future releases.</p>
