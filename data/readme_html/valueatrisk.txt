<p dir="auto"><a href="https://en.wikipedia.org/wiki/Value_at_risk" rel="nofollow">Value-at-Risk</a> is a method for measuring the risk of extremal losses in financial investments. <a href="https://doi.org/10.1016/j.ijforecast.2015.08.003" rel="nofollow">Nieto and Ruiz (2016)</a> and <a href="https://doi.org/10.1093/jjfinec/nbj002" rel="nofollow">Kuester et al. (2005)</a> provide a good overview of the subject.</p>
<p dir="auto">This package is still at an experimental development stage. It has been written in a way to accomondate the needs of my thesis. (If you want to take a look at it you can find it <a href="https://www.pyxida.aueb.gr/index.php?op=view_object&amp;object_id=8462" rel="nofollow">here</a>. The model mentioned as EWMA in the thesis corresponds to the <code>EWMAHistoricalSimulationVaR</code> model in this package. The usual RiskMetrics EWMA model has  not been used in the thesis and has been added here as <code>EWMARiskMetricsVaR</code>. The code for the LASSO-GARCH model has not been uploaded at the moment since it was a novel approach that required/requires more research)</p>
<p dir="auto">Please check the <a href="https://chm-von-tla.github.io/ValueAtRisk.jl/dev" rel="nofollow">documentation</a> for more regarding use of the package and implementation concerns.</p>
<p dir="auto">Any feedback is appreciated!</p>
