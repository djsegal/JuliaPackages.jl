<p dir="auto">A somewhat minimal implementation of the auditory spectrogram (<code>Audiospect</code>)
and cortical model (<code>cortical</code>, <code>scalefitler</code>, <code>ratefilter</code>) as described in the following paper:</p>
<p dir="auto"><a href="http://doi.org/10.1121/1.1945807" rel="nofollow">Chi, T., Ru, P., &amp; Shamma, S. A. (2005). Multiresolution spectrotemporal
analysis of complex sounds. The Journal of the Acoustical Society of America,
118(2), 887–906.</a></p>
<p dir="auto">Refer to the documentation of each function (ala ? at the REPL) for more information.</p>
<p dir="auto">You can find the original MATLAB implementation of these models
<a href="http://nsl.isr.umd.edu/downloads.html" rel="nofollow">here</a>.</p>
