<p dir="auto">Plotting library for Climate fields. Right now, it is compatible with <a href="https://github.com/JuliaClimate/ClimateTools.jl">ClimateTools.jl</a>. The long term objective is to develop a generic climate mapping library in Julia.</p>

<p dir="auto">Under the hood, ClimatePlots use Cartopy for plotting gridded variables. Hence, PyCall should be configured so that the python distribution linked to PyCall is able to load cartopy and matplotlib.</p>
<p dir="auto">Installation steps are described <a href="https://juliaclimate.github.io/ClimateTools.jl/dev/installation/" rel="nofollow">here</a>.</p>
