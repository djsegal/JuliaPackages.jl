<h3><a id="user-content-factor-models-for-julia" class="anchor" aria-hidden="true" href="#factor-models-for-julia"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path fill-rule="evenodd" d="M4 9h1v1H4c-1.5 0-3-1.69-3-3.5S2.55 3 4 3h4c1.45 0 3 1.69 3 3.5 0 1.41-.91 2.72-2 3.25V8.59c.58-.45 1-1.27 1-2.09C10 5.22 8.98 4 8 4H4c-.98 0-2 1.22-2 2.5S3 9 4 9zm9-3h-1v1h1c1 0 2 1.22 2 2.5S13.98 12 13 12H9c-.98 0-2-1.22-2-2.5 0-.83.42-1.64 1-2.09V6.25c-1.09.53-2 1.84-2 3.25C6 11.31 7.55 13 9 13h4c1.45 0 3-1.69 3-3.5S14.5 6 13 6z"></path></svg></a>Factor Models for Julia</h3>
<p>[Factor models] (<a href="http://en.wikipedia.org/wiki/Dynamic_factor" rel="nofollow">http://en.wikipedia.org/wiki/Dynamic_factor</a>) or diffusion index models are statistical models which allow the estimation of a dependent variable using potentially very many regressors. Factor models are related to [factor analysis] (<a href="http://en.wikipedia.org/wiki/Factor_analysis" rel="nofollow">http://en.wikipedia.org/wiki/Factor_analysis</a>).</p>
<p>This package has a strong focus on the econometric literature related to factor models. This is because I wrote the package for my master thesis. Resultingly the predict method refers to time-series data and estimates a factor augmented regression. If you feel something relevant is missing please feel free to open an issue or a pull request.</p>
<p>As soon as I my thesis is handed in I will provide a proper README (and add some tests).</p>

<div class="highlight highlight-source-julia"><pre>x <span class="pl-k">=</span> <span class="pl-c1">randn</span>(<span class="pl-c1">50</span>, <span class="pl-c1">200</span>)  <span class="pl-c"><span class="pl-c">#</span> no problem to use more columns than rows, that is one of the nice features of factor models</span>

<p>predictions can be done using</p>
<div class="highlight highlight-source-julia"><pre><span class="pl-c1">predict</span>(fm<span class="pl-k">::</span><span class="pl-c1">FactorModel</span>, y<span class="pl-k">::</span><span class="pl-c1">Array{Float64, 1}</span>, h<span class="pl-k">::</span><span class="pl-c1">Int64</span><span class="pl-k">=</span><span class="pl-c1">1</span>, number_of_lags<span class="pl-k">::</span><span class="pl-c1">Int64</span><span class="pl-k">=</span><span class="pl-c1">5</span>, number_of_factors<span class="pl-k">::</span><span class="pl-c1">Int64</span><span class="pl-k">=</span><span class="pl-c1">0</span>)</pre></div>
<p>or</p>
<div class="highlight highlight-source-julia"><pre><span class="pl-c1">predict</span>(dfm<span class="pl-k">::</span><span class="pl-c1">DynamicFactorModel</span>, y<span class="pl-k">::</span><span class="pl-c1">Array{Float64, 1}</span>, h<span class="pl-k">::</span><span class="pl-c1">Int64</span><span class="pl-k">=</span><span class="pl-c1">1</span>, number_of_lags<span class="pl-k">::</span><span class="pl-c1">Int64</span><span class="pl-k">=</span><span class="pl-c1">5</span>, number_of_factors<span class="pl-k">::</span><span class="pl-c1">Int64</span><span class="pl-k">=</span><span class="pl-c1">0</span>)</pre></div>
<p>This estimates a linear model using number_of_lags lags of y, number_of_factors factors (and the number of lags thereof specified above in the case of dynamic factor models).</p>
<h2><a id="user-content-features" class="anchor" aria-hidden="true" href="#features"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path fill-rule="evenodd" d="M4 9h1v1H4c-1.5 0-3-1.69-3-3.5S2.55 3 4 3h4c1.45 0 3 1.69 3 3.5 0 1.41-.91 2.72-2 3.25V8.59c.58-.45 1-1.27 1-2.09C10 5.22 8.98 4 8 4H4c-.98 0-2 1.22-2 2.5S3 9 4 9zm9-3h-1v1h1c1 0 2 1.22 2 2.5S13.98 12 13 12H9c-.98 0-2-1.22-2-2.5 0-.83.42-1.64 1-2.09V6.25c-1.09.53-2 1.84-2 3.25C6 11.31 7.55 13 9 13h4c1.45 0 3-1.69 3-3.5S14.5 6 13 6z"></path></svg></a>Features</h2>
<ul>
<li>Simulate and Estimate Dynamic factor models</li>
<li>Estimate the number of Factors</li>
<li>Use the estimation for prediction</li>
<li>Preselect predictors using soft and hard thresholding (see Bai, Jushan, and Serena Ng. "Forecasting economic time series using targeted predictors." Journal of Econometrics 146.2 (2008): 304-317.)</li>
</ul>
