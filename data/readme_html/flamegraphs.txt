<p>FlameGraphs is a package that adds functionality to Julia's <code>Profile</code> standard library. It is directed primarily at the algorithmic side of producing <a href="http://www.brendangregg.com/flamegraphs.html" rel="nofollow">flame graphs</a>, but includes some "format agnostic" rendering code.</p>
<p>You might use FlameGraphs on its own, but users should consider one of its downstream packages:</p>
<ul>
<li><a href="https://github.com/timholy/ProfileView.jl">ProfileView</a>, a graphical user interface (GUI) based on <a href="https://github.com/JuliaGraphics/Gtk.jl">Gtk</a> for displaying and interacting with flame graphs</li>
<li><a href="https://github.com/JuliaPerf/PProf.jl">PProf</a>, an interactive, web-based profile GUI explorer, implemented as a wrapper around <a href="https://github.com/google/pprof"><code>google/pprof</code></a>. Has excellent support for interaction, filtering, aggregation, and viewing source code. But note that it loses the sample ordering information captured in FlameGraphs.</li>
<li><a href="https://junolab.org/" rel="nofollow">Juno</a>, a development environment that supports profile visualization</li>
<li><a href="https://github.com/davidanthoff/ProfileVega.jl">ProfileVega</a>, the recommended package for display in Jupyter notebooks</li>
<li><a href="https://github.com/timholy/ProfileSVG.jl">ProfileSVG</a>, a package for writing flame graphs to SVG format</li>
</ul>
<p>See the documentation for details.</p>
