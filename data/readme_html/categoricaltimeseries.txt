<p dir="auto">Toolbox helpful for the study of <em>categorical</em> time-series.
It contains methods used in <a href="https://categoricaltimeseriesjl.readthedocs.io/en/latest/Spectral_properties/" rel="nofollow">spectral envelope</a>  analysis, <a href="https://categoricaltimeseriesjl.readthedocs.io/en/latest/Correlations/" rel="nofollow">serial dependence</a> analysis, <a href="https://categoricaltimeseriesjl.readthedocs.io/en/latest/Data_clustering/" rel="nofollow">clustering</a>, and <a href="https://categoricaltimeseriesjl.readthedocs.io/en/latest/Motif_recognition/" rel="nofollow">motif recognition</a>.</p>
