<p dir="auto"><a href="https://femtomc.github.io/Jaynes.jl/" rel="nofollow"><code>Jaynes.jl</code> is a (research-oriented alpha) domain-specific compiler for probabilistic programming. Jaynes uses source-to-source IR transformations and contextual dispatch to implement the generative function interface of Gen.jl.</a></p>
