<p dir="auto">Julia introduced
<a href="https://docs.julialang.org/en/v1.10-dev/manual/code-loading/#man-extensions" rel="nofollow">package extensions</a>
in v1.9. This package makes these extensions backwards-compatible to earlier Julia versions,
with zero overhead on new versions.</p>
<p dir="auto">Internally, this uses
<a href="https://github.com/JuliaPackaging/Requires.jl">Requires.jl</a>
on earlier versions.</p>
<h2 dir="auto"><a id="user-content-usage" class="anchor" aria-hidden="true" href="#usage"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path d="m7.775 3.275 1.25-1.25a3.5 3.5 0 1 1 4.95 4.95l-2.5 2.5a3.5 3.5 0 0 1-4.95 0 .751.751 0 0 1 .018-1.042.751.751 0 0 1 1.042-.018 1.998 1.998 0 0 0 2.83 0l2.5-2.5a2.002 2.002 0 0 0-2.83-2.83l-1.25 1.25a.751.751 0 0 1-1.042-.018.751.751 0 0 1-.018-1.042Zm-4.69 9.64a1.998 1.998 0 0 0 2.83 0l1.25-1.25a.751.751 0 0 1 1.042.018.751.751 0 0 1 .018 1.042l-1.25 1.25a3.5 3.5 0 1 1-4.95-4.95l2.5-2.5a3.5 3.5 0 0 1 4.95 0 .751.751 0 0 1-.018 1.042.751.751 0 0 1-1.042.018 1.998 1.998 0 0 0-2.83 0l-2.5 2.5a1.998 1.998 0 0 0 0 2.83Z"></path></svg></a>Usage</h2>
<p dir="auto">Supposing you have a package called <code>Foo</code>:</p>
<ol dir="auto">
<li>
<p dir="auto">Set up package extensions for <code>Foo</code> as usual. This means adding <code>[weakdeps]</code> and
<code>[extensions]</code> to <code>Project.toml</code> and adding extension code to <code>ext/</code>.</p>
</li>
<li>
<p dir="auto">Add <code>PackageExtensionTools</code> as a dependency to <code>Foo</code>.</p>
</li>
<li>
<p dir="auto">Add the following code to <code>src/Foo.jl</code>:</p>
<div class="highlight highlight-source-julia notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="using PackageExensionTools
function __init__()
    @require_extensions
end"><pre><span class="pl-k">using</span> PackageExensionTools
<span class="pl-k">function</span> <span class="pl-en">__init__</span>()
    <span class="pl-c1">@require_extensions</span>
<span class="pl-k">end</span></pre></div>
</li>
</ol>
<p dir="auto">That's it! Your package extensions will now be loaded as expected on any Julia version!</p>
