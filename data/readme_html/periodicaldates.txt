<p dir="auto">This packages makes it easier to work with monthly, quarterly, or yearly dates. It defines three types:</p>
<ul dir="auto">
<li>
<p dir="auto">A <code>MonthlyDate &lt;: TimeType</code> type</p>
<div class="highlight highlight-source-julia notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content=" julia&gt; using Dates, PeriodicalDates
 julia&gt; dt = MonthlyDate(1990, 1)
 # Alternatively, construct from a date
 julia&gt; dtm = MonthlyDate(Date(1990, 1))
 # Alternatively, contruct from a string with default dateformat&quot;yyyy-mm&quot;
 julia&gt; dtm = MonthlyDate(&quot;1990-01&quot;)
 # or any other DateFormat
 julia&gt; dtm = MonthlyDate(&quot;1990/01&quot;, &quot;yyyy/mm&quot;)
 # Arithmetic works as expected
 julia&gt; dtm + Month(1)
 julia&gt; Date(dtm)
 # 1990-01-01
