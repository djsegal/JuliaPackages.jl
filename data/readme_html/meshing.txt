<p>This package provides a comprehensive suite of meshing (isosurface extraction) algorithms for use on distance fields and other structured datasets.</p>
<p>Algorithms included:</p>
<ul>
<li><a href="https://en.wikipedia.org/wiki/Marching_tetrahedra" rel="nofollow">Marching Tetrahedra</a></li>
<li><a href="https://en.wikipedia.org/wiki/Marching_cubes" rel="nofollow">Marching Cubes</a></li>
<li><a href="https://0fps.net/2012/07/12/smooth-voxel-terrain-part-2/" rel="nofollow">Naive Surface Nets</a></li>
</ul>
