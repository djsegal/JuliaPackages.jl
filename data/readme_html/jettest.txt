<p dir="auto"><g-emoji class="g-emoji" alias="warning" fallback-src="https://github.githubassets.com/images/icons/emoji/unicode/26a0.png">⚠️</g-emoji> <em><strong>Attention</strong></em> <g-emoji class="g-emoji" alias="warning" fallback-src="https://github.githubassets.com/images/icons/emoji/unicode/26a0.png">⚠️</g-emoji>:
From Julia v1.7, JETTest.jl is deprecated and its functionalities have been moved to <a href="https://github.com/aviatesk/JET.jl">JET.jl</a>.
In short, you use <code>JET.@report_opt</code>/<code>JET.report_opt</code>/<code>JET.@test_opt</code>/<code>JET.test_opt</code> instead of <code>JETTest.@report_dispatch</code>/<code>JETTest.@report_dispatch</code>/<code>JETTest.@test_nodispatch</code>/<code>JETTest.@test_dispatch</code>.
More detailed documentation can be found at <a href="https://aviatesk.github.io/JET.jl/dev/optanalysis/" rel="nofollow">https://aviatesk.github.io/JET.jl/dev/optanalysis/</a>.
JETTest.jl is still supported on Julia v1.6, but there won't be any more development.</p>
