<p>clone the project in your usual way, or using the following command which clones the project in <code>.julia\dev\MatLang</code></p>
<div class="highlight highlight-source-julia"><pre>] dev MatLang</pre></div>
<p>cd to cloned package. if you used dev command, run the following:</p>
<div class="highlight highlight-source-julia"><pre><span class="pl-c1">cd</span>(<span class="pl-s"><span class="pl-pds">"</span><span class="pl-v">$(<span class="pl-c1">homedir</span>())</span><span class="pl-cce">\\</span>.julia<span class="pl-cce">\\</span>dev<span class="pl-cce">\\</span>MatLang<span class="pl-pds">"</span></span>)</pre></div>
<p>Activate the package:</p>
<div class="highlight highlight-source-julia"><pre>] activate .</pre></div>
<p>Set <code>developing</code> variable to <code>true</code> in the files in <code>test</code> and <code>usage</code> folder.</p>
<div class="highlight highlight-source-julia"><pre>developing <span class="pl-k">=</span> <span class="pl-c1">true</span></pre></div>
<p>Set <code>developing</code> variable to <code>false</code> before committing (for a successful CI build).</p>
<p>The package uses <code>Revise</code> for quick testing without the need for restarting.</p>
