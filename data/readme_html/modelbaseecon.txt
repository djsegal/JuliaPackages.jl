<p dir="auto">This Julia package is part of the
<a href="https://github.com/bankofcanada/StateSpaceEcon.jl">StateSpaceEcon</a> ecosystem.
<a href="https://github.com/bankofcanada/ModelBaseEcon.jl">ModelBaseEcon</a> contains the
basic elements needed for model definition.
<a href="https://github.com/bankofcanada/StateSpaceEcon.jl">StateSpaceEcon</a> works with
model objects defined with ModelBaseEcon.</p>
