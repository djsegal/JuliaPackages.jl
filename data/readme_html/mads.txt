<p dir="auto"><a href="http://madsjulia.github.io/Mads.jl" rel="nofollow">MADS</a> is an integrated high-performance computational framework for data/model/decision analyses.</p>
<p dir="auto"><a href="http://madsjulia.github.io/Mads.jl" rel="nofollow">MADS</a> can be applied to perform:</p>
<ul dir="auto">
<li>Sensitivity Analysis</li>
<li>Parameter Estimation</li>
<li>Model Inversion and Calibration</li>
<li>Uncertainty Quantification</li>
<li>Model Selection and Averaging</li>
<li>Model Reduction and Surrogate Modeling</li>
<li>Decision Analysis and Support</li>
</ul>
<p dir="auto"><a href="http://madsjulia.github.io/Mads.jl" rel="nofollow">MADS</a> utilizes adaptive rules and techniques which allow the analyses to be performed efficiently with minimum user input.</p>
<p dir="auto"><a href="http://madsjulia.github.io/Mads.jl" rel="nofollow">MADS</a> provides a series of alternative algorithms to execute various types of data-based and model-based analyses.</p>
<p dir="auto"><a href="http://madsjulia.github.io/Mads.jl" rel="nofollow">MADS</a> can efficiently utilize available computational resources.</p>
<p dir="auto"><a href="http://madsjulia.github.io/Mads.jl" rel="nofollow">MADS</a> has been extensively tested and verified.</p>
