<p><strong>Update</strong>: this was for an <em>ancient</em> (and very outdated) version of Julia. These bindings are unlikely to be of use, so the repository has been archived. If you're interested in wrapping VTK for modern Julia, see <a href="https://github.com/JuliaPackaging/BinaryBuilder.jl">https://github.com/JuliaPackaging/BinaryBuilder.jl</a>, <a href="https://github.com/JuliaInterop/Clang.jl">https://github.com/JuliaInterop/Clang.jl</a>, and perhaps others. There's also a <code>#binarybuilder</code> channel on the <a href="https://julialang.org/slack/" rel="nofollow">Julia Slack</a>.</p>
