<p>PhyloPlots has utilities to plot tree and network objects
from <a href="https://github.com/crsl4/PhyloNetworks.jl">PhyloNetworks</a>
using plotting back-ends like <a href="http://gadflyjl.org/stable/" rel="nofollow">Gadfly</a>,
and through <a href="https://www.r-project.org" rel="nofollow">R</a>
via <a href="https://github.com/JuliaInterop/RCall.jl">RCall</a>.</p>
