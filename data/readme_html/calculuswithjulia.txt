<p><code>Julia</code> package to provide notes and features for using <code>Julia</code> to address typical problems from the undergraduate calculus sequence.</p>
<p>The notes may be read at <a href="http://juliahub.com/docs/CalculusWithJulia" rel="nofollow">Calculus with Julia</a>.</p>
<p>The notes expect that this package is installed, as it also provides a handful of conveniences for the task.</p>
<p>In addition to the html pages, there are other versions of the same material that can be generated, such as <code>ipynb</code> notebooks or <code>pdf</code> files. These are available by installing the package and running</p>
<div class="snippet-clipboard-content position-relative" data-snippet-clipboard-copy-content="CalculusWithJulia.WeaveSupport.weave_all()
"><pre><code>CalculusWithJulia.WeaveSupport.weave_all()
</code></pre></div>
<p>The named argument <code>build_list</code> defaults to <code>build_list=(:script,:html,:pdf,:github,:notebook)</code> which has all the output formats specified. This can be customized as desired.</p>
<p>The files are built in the package directory, available through the command <code>pathof(CalculusWithJulia)</code>.</p>

<p>This is a work in progress. To report an issue, make a comment, or suggest something new, please file an <a href="https://github.com/jverzani/CalculusWithJulia.jl/issues/">issue</a>. In your message add the tag <code>@jverzani</code> to ensure it is not overlooked. Otherwise, an email to <code>verzani</code> at <code>math.csi.cuny.edu</code> will also work.</p>
<p>To make edits to the documents directly, a pull request with the modified <code>*.jmd</code> files in the <code>CwJ</code> directory should be made. Minor edits to the <code>*.jmd</code> files should be possible through the GitHub web interface. The <code>*.html</code> files are generated using <code>Julia's</code> <code>Weave</code> package. This need not be done.</p>
