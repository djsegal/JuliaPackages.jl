<ul dir="auto">
<li>You need the Graphviz package to be installed (Debian: <code>apt install graphviz</code>, Fedora: <code>dnf install graphviz</code>, pick your poison). The <code>dot</code> command is expected to be somewhere in your <code>$PATH</code>.</li>
<li>Install the package the usual way by typing <code>]add GraphvizDotLang</code> in the REPL.</li>
</ul>
