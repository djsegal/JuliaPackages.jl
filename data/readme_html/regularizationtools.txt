<p>The package can be installed from the Julia package prompt with</p>
<div class="highlight highlight-source-julia position-relative" data-snippet-clipboard-copy-content="julia&gt; ]add RegularizationTools
"><pre>julia<span class="pl-k">&gt;</span> ]add RegularizationTools</pre></div>
<p>The closing square bracket switches to the package manager interface and the <code>add</code> command installs the package and any missing dependencies. To return to the Julia REPL hit the <code>delete</code> key.</p>
<p>To load the package run</p>
<div class="highlight highlight-source-julia position-relative" data-snippet-clipboard-copy-content="julia&gt; using RegularizationTools
"><pre>julia<span class="pl-k">&gt;</span> <span class="pl-k">using</span> RegularizationTools</pre></div>
