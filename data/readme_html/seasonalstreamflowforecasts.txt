<p dir="auto">A web client for the Seasonal Streamflow Forecasting Service  the Australian Bureau of Meteorology in the Julia programming language. The website at <a href="http://www.bom.gov.au/water/ssf" rel="nofollow">http://www.bom.gov.au/water/ssf</a> provides 3-month ahead monthly streamflow forecasts for catchments and unregulated total inflows to reservoirs across Australia.</p>

<p dir="auto">The package can be installed with the Julia package manager. From the Julia REPL, type <code>]</code> to enter the Pkg REPL mode and run:</p>
<div class="highlight highlight-source-julia notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="pkg&gt; add SeasonalStreamflowForecasts"><pre>pkg<span class="pl-k">&gt;</span> add SeasonalStreamflowForecasts</pre></div>
<p dir="auto">If you want to install the package directly from its github development site,</p>
<div class="highlight highlight-source-julia notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="pkg&gt; add http://github.com/petershintech/SeasonalStreamflowForecasts.jl"><pre>pkg<span class="pl-k">&gt;</span> add http<span class="pl-k">:</span><span class="pl-k">//</span>github<span class="pl-k">.</span>com<span class="pl-k">/</span>petershintech<span class="pl-k">/</span>SeasonalStreamflowForecasts<span class="pl-k">.</span>jl</pre></div>
<p dir="auto">And load the package using the command:</p>
<div class="highlight highlight-source-julia notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="using SeasonalStreamflowForecasts"><pre><span class="pl-k">using</span> SeasonalStreamflowForecasts</pre></div>

<p dir="auto">When you create an instance of the <code>SSF</code> structure, it downloads
site information.</p>
<div class="highlight highlight-source-julia notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="julia&gt; ssf = SSF();"><pre>julia<span class="pl-k">&gt;</span> ssf <span class="pl-k">=</span> <span class="pl-c1">SSF</span>();</pre></div>
<p dir="auto">Once it is instantiated, the fields of <code>ssf</code> should be considered as read-only so don't try to change any values of the fields.</p>

<p dir="auto"><code>get_forecasts()</code> returns forecast data as <code>DataFrames.DataFrame</code>. The method needs a site ID and a forecast date.
The returned forecast data has 5000 ensemble members and corresponding historical references (aka climatology).
The site ID of a station can be found in <code>ssf.sites</code>.</p>
