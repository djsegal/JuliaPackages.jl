<p><code>WorldVocoder.jl</code> is a wrapper package for the WORLD vocoder by mmorise <a href="https://github.com/mmorise/World">https://github.com/mmorise/World</a>.
It calls the original WORLD binary via the WORLD_jll.jl package.</p>
