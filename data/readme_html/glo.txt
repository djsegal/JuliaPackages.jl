<p>Julia wrapper of the <a href="https://gloapi.gitkraken.com/v1/docs/#/" rel="nofollow">Glo-board API</a>.
Developer documentation is <a href="https://support.gitkraken.com/developers/overview/" rel="nofollow">here</a>.</p>
<p>Endpoints become functions, if IDs are involved you have to wrap them in the corresponding ID type and pass them to the function.
POST methods take the post message as first argument, and DELETE methods take ! as first argument.
No names are exportet.</p>

<p>In order to use this API you have to pass a header specifying your authentification.</p>
<h3><a id="user-content-authentification-via-personal-access-token-pat" class="anchor" aria-hidden="true" href="#authentification-via-personal-access-token-pat"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path fill-rule="evenodd" d="M7.775 3.275a.75.75 0 001.06 1.06l1.25-1.25a2 2 0 112.83 2.83l-2.5 2.5a2 2 0 01-2.83 0 .75.75 0 00-1.06 1.06 3.5 3.5 0 004.95 0l2.5-2.5a3.5 3.5 0 00-4.95-4.95l-1.25 1.25zm-4.69 9.64a2 2 0 010-2.83l2.5-2.5a2 2 0 012.83 0 .75.75 0 001.06-1.06 3.5 3.5 0 00-4.95 0l-2.5 2.5a3.5 3.5 0 004.95 4.95l1.25-1.25a.75.75 0 00-1.06-1.06l-1.25 1.25a2 2 0 01-2.83 0z"></path></svg></a>Authentification via personal access token (PAT)</h3>
<p>A personal access token can be created <a href="https://app.gitkraken.com/pat/new" rel="nofollow">here</a></p>
