<p dir="auto">This package implements the determinant quantum Monte Carlo (DQMC) method for Hubbard,
and electron-phonon interactions, including both Holstein and Su-Schrieffer-Heeger (SSH) style
electron-phonon coupling.</p>
<p dir="auto">This package is currently in the experimental phase of development.</p>

<p dir="auto">The <a href="https://github.com/SmoQySuite/SmoQyDQMC.jl">SmoQyDQMC.jl</a> re-exports certain packages using
the <a href="https://github.com/simonster/Reexport.jl.git"><code>Reexport.jl</code></a> package in order to simplify the installation process.</p>
<ul dir="auto">
<li><a href="https://github.com/cohensbw/LatticeUtilities.jl.git"><code>LatticeUtilties.jl</code></a>: Used to represent arbitrary lattice geometries.</li>
<li><a href="https://github.com/SmoQySuite/JDQMCFramework.jl.git"><code>JDQMCFramework.jl</code></a>: Implements and exports the basic framework for running a DQMC simulation.</li>
<li><a href="https://github.com/SmoQySuite/JDQMCMeasurements.jl.git"><code>JDQMCMeasurements.jl</code></a>: Implements various global, local and correlation measurements for a DQMC simulation.</li>
<li><a href="https://github.com/cohensbw/MuTuner.jl.git"><code>MuTuner.jl</code></a>: Impelments and exports an algorithm for tuning the chemical potential to achieve a target density in grand canonical Monte Carlo simulations.</li>
</ul>
<h3 dir="auto"><a id="user-content-external-dependencies" class="anchor" aria-hidden="true" href="#external-dependencies"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path d="m7.775 3.275 1.25-1.25a3.5 3.5 0 1 1 4.95 4.95l-2.5 2.5a3.5 3.5 0 0 1-4.95 0 .751.751 0 0 1 .018-1.042.751.751 0 0 1 1.042-.018 1.998 1.998 0 0 0 2.83 0l2.5-2.5a2.002 2.002 0 0 0-2.83-2.83l-1.25 1.25a.751.751 0 0 1-1.042-.018.751.751 0 0 1-.018-1.042Zm-4.69 9.64a1.998 1.998 0 0 0 2.83 0l1.25-1.25a.751.751 0 0 1 1.042.018.751.751 0 0 1 .018 1.042l-1.25 1.25a3.5 3.5 0 1 1-4.95-4.95l2.5-2.5a3.5 3.5 0 0 1 4.95 0 .751.751 0 0 1-.018 1.042.751.751 0 0 1-1.042.018 1.998 1.998 0 0 0-2.83 0l-2.5 2.5a1.998 1.998 0 0 0 0 2.83Z"></path></svg></a>External Dependencies</h3>
<ul dir="auto">
<li><a href="https://github.com/cohensbw/StableLinearAlgebra.jl.git"><code>StableLinearAlgebra.jl</code></a>: Implements optimized numerical stabilizaiton methods required by DQMC simulations.</li>
<li><a href="https://github.com/cohensbw/Checkerboard.jl.git"><code>Checkerboard.jl</code></a>: Implements and exports the checkerboard method for approximating exponentiated hopping matrices by a sparse matrix.</li>
<li><a href="https://github.com/JuliaIO/JLD2.jl.git"><code>JLD2.jl</code></a>: Package used to write data to binary files in an HDF5 compatible format. It is also recommended this package be used at the scripting level to implement checkpointing in a simulation.</li>
<li><a href="https://github.com/carstenbauer/BinningAnalysis.jl.git"><code>BinningAnalysis.jl</code></a>: Export method impelementing the jackknife algorithm for calculating error bars.</li>
</ul>
