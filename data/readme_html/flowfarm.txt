<ul dir="auto">
<li>Compatible with ForwardDiff for gradiant-based optimization</li>
<li>Swap out models without changing anything else in the simulation setup</li>
<li>Smooth/continous model implementations</li>
<li>Runs on a single core, across multiple cores (threaded), or on multiple machines (distributed).</li>
<li>Designed so that new model implementations can be included by adding a single method</li>
<li>Allows for Wake Expansion Continuation (WEC) as described <a href="http://flowlab.groups.et.byu.net/preprints/Thomas2021.pdf" rel="nofollow">here</a></li>
</ul>

<p dir="auto">NaN Safe Mode must be enables in ForwardDiff for ForwardDiff to work properly with FLOWFarm.</p>
<div class="highlight highlight-source-julia notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="(v1.x) pkg&gt; dev ForwardDiff"><pre>(v1<span class="pl-k">.</span>x) pkg<span class="pl-k">&gt;</span> dev ForwardDiff</pre></div>
<div class="snippet-clipboard-content notranslate position-relative overflow-auto" data-snippet-clipboard-copy-content="$ cd ~/.julia/dev/ForwardDiff/src/"><pre class="notranslate"><code>$ cd ~/.julia/dev/ForwardDiff/src/
</code></pre></div>
<p dir="auto">In <code>prelude.jl</code>, on the first line, set <code>const NANSAFE_MODE_ENABLED = true</code> and save the file.
For more information see the ForwardDiff documentation at
<a href="http://www.juliadiff.org/ForwardDiff.jl/latest/user/advanced.html" rel="nofollow">http://www.juliadiff.org/ForwardDiff.jl/latest/user/advanced.html</a></p>
