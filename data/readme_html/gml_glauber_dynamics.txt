<p dir="auto"><code>GML_Glauber_Dynamics</code> is a julia package for learning graphical models from time correlated samples generated through Gibbs sampling (aka Glauber dynamics). It is built on top of the julia package <code>GraphicalModelLearning.jl</code> (<a href="https://github.com/lanl-ansi/GraphicalModelLearning.jl">see package here</a>).</p>

<p dir="auto">Let's start with a simple example where we generate samples through Glauber dynamics from an Ising model defined on a three node graph. The goal is to then check if the learned graph is close to the true graph from which the samples were generated.</p>
