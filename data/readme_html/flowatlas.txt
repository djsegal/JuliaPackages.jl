<p dir="auto"><em>an interactive explorer for flow cytometry data</em></p>
<p dir="auto">Flow Atlas is an interactive data explorer for <code>FCS</code> datasets and their accompanying FlowJo workspace files, such as those coming from the <a href="https://humancellatlas.org" rel="nofollow">Human Cell Atlas</a>. This tool is built in Julia and allows biologists to interactively inter-operate between FlowJo and modern computational libraries like <a href="https://github.com/LCSB-BioCore/GigaSOM.jl"><code>GigaSOM.jl</code></a></p>

<p dir="auto">To install Flow Atlas, Julia 1.5+ is required. Follow <a href="https://julialang.org/downloads/platform/" rel="nofollow">platform specific instructions</a>. In order to make the <code>julia</code> executable available from the command prompt / terminal follow the instructions on adding Julia to the <code>PATH</code>/enviroment; make sure this has been done according to the instructions linked above.</p>
<p dir="auto">Once julia has been added to the environment, open a command prompt / terminal and type <code>julia</code>. This opens a julis shell; you should see <code>julia&gt;</code>. Then type <code>]</code> to enter the package manager, and type <code>add FlowAtlas</code> and then hit Return ⏎. You should see <code>pkg&gt; add FlowAtlas</code>.</p>
<p dir="auto">This installs the most recent version of Flow Atlas and makes it availalbe in your global Julia enviroment. Finally to exit the package manager hit Backspace ⌫; you should see <code>julia&gt;</code> again.</p>
