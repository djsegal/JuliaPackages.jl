<p><em>DLPipelines.jl</em> is an interface for defining deep learning data pipelines. This includes every data transformation beside the model itself: preprocessing and augmenting the data before feeding it into the model, and decoding the model's outputs to targets.</p>
<p>It is used in <a href="https://www.github.com/FluxML/FastAI.jl">FastAI.jl</a> to provide an easy-to-use interface for common learning tasks.</p>
<p>See <a href="https://lorenzoh.github.io/DLPipelines.jl/dev/README.html" rel="nofollow">the documentation</a> for more information on implementing the interface.</p>
