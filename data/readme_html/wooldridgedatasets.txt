<p dir="auto">This package includes all the data sets used in the <strong>Introductory Econometrics: A Modern Approach</strong> by <strong>Jeffrey Wooldridge</strong>. It is similar to the <code>R</code> package <a href="https://cran.r-project.org/web/packages/wooldridge/wooldridge.pdf" rel="nofollow"><code>Wooldridge</code></a>.</p>
<p dir="auto">To download this package execute:</p>
<p dir="auto"><code>pkg&gt; add WooldridgeDatasets</code></p>
<p dir="auto">Then run</p>
<p dir="auto"><code>using WooldridgeDatasets</code></p>
<p dir="auto">Then use <code>wooldridge("Dataset name")</code> (any of these two), to use the dataset. It'd be better if these commands are used with <code>DataFrame()</code> function from <a href="https://github.com/JuliaData/DataFrames.jl"><code>DataFrames.jl</code></a> package.</p>
<p dir="auto">If you face any problem, feel free to raise it in the <a href="https://github.com/arubhardwaj/WooldridgeDatasets.jl/issues">issues</a>.</p>
