<p><code>LazySets.jl</code> is a registered Julia package and <a href="https://julialang.github.io/Pkg.jl/v1/managing-packages/" rel="nofollow">as such you can install it</a> by activating the <code>pkg</code> mode (type <code>]</code>, and to leave it, type <code>&lt;backspace&gt;</code>),
followed by</p>
<div class="highlight highlight-source-julia position-relative" data-snippet-clipboard-copy-content="pkg&gt; add LazySets
"><pre>pkg<span class="pl-k">&gt;</span> add LazySets</pre></div>
<p>See the <a href="https://juliareach.github.io/LazySets.jl/dev/man/getting_started/" rel="nofollow">Getting started</a> section of the manual for other options.</p>
