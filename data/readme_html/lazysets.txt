followed by</p>
<div class="highlight highlight-source-julia notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="pkg&gt; add LazySets"><pre>pkg<span class="pl-k">&gt;</span> add LazySets</pre></div>
<p dir="auto">See the <a href="https://juliareach.github.io/LazySets.jl/dev/man/getting_started/" rel="nofollow">Getting started</a> section of the manual for other options.</p>

<details>
<summary>Abstract</summary>
<p dir="auto">We present <a href="https://github.com/JuliaReach">JuliaReach</a>, a Julia ecosystem to perform reachability analysis of dynamical systems. JuliaReach builds on sound scientific approaches and was, in two occasions (2018 and 2020) the winner of the annual friendly competition on Applied Verification for Continuous and Hybrid Systems (<a href="https://cps-vo.org/group/ARCH" rel="nofollow">ARCH-COMP</a>).</p>
<p dir="auto">The workshop consists of three parts (respectively packages) in <a href="https://github.com/JuliaReach">JuliaReach</a>: our core package for set representations, our main package for reachability analysis, and a new package applying reachability analysis with potential use in domain of control, robotics and autonomous systems.</p>
<p dir="auto">In the first part we present <a href="https://github.com/JuliaReach/LazySets.jl">LazySets.jl</a>, which provides ways to symbolically represent sets of points as geometric shapes, with a special focus on convex sets and polyhedral approximations. <a href="https://github.com/JuliaReach/LazySets.jl">LazySets.jl</a> provides methods to apply common set operations, convert between different set representations, and efficiently compute with sets in high dimensions.</p>
<p dir="auto">In the second part we present <a href="https://github.com/JuliaReach/ReachabilityAnalysis.jl">ReachabilityAnalysis.jl</a>, which provides tools to approximate the set of reachable states of systems with both continuous and mixed discrete-continuous dynamics, also known as hybrid systems. It implements conservative discretization and set-propagation techniques at the state-of-the-art.</p>
<p dir="auto">In the third part we present <a href="https://github.com/JuliaReach/NeuralNetworkAnalysis.jl">NeuralNetworkAnalysis.jl</a>, which is an application of <a href="https://github.com/JuliaReach/ReachabilityAnalysis.jl">ReachabilityAnalysis.jl</a> to analyze dynamical systems that are controlled by neural networks. This package can be used to validate or invalidate specifications, for instance about the safety of such systems.</p>
<p dir="auto">Workshop materials are available here: <a href="https://github.com/JuliaReach/JuliaCon-2021-Workshop-Its-All-Set">https://github.com/JuliaReach/JuliaCon-2021-Workshop-Its-All-Set</a></p>
</details>
