<p dir="auto">This package provides an interface to the <a href="https://commonchemistry.cas.org/" rel="nofollow">CAS Common Chemistry</a> registry of chemical compounds.</p>
<p dir="auto">The database is maintained by the American Chemical Society. Quoting the description from the <a href="https://commonchemistry.cas.org/" rel="nofollow">home page</a></p>
<blockquote>
<p dir="auto">CAS Common Chemistry is an open community resource for accessing chemical information. Nearly 500,000 chemical substances from <a href="https://www.cas.org/cas-data/cas-registry" rel="nofollow">CAS REGISTRY®</a> cover areas of community interest, including common and frequently regulated chemicals, and those relevant to high school and undergraduate chemistry classes. This chemical information, curated by our expert scientists, is provided in alignment with our mission as a division of the American Chemical Society.</p>
</blockquote>

<p dir="auto">The main exported function is called <code>cas</code>.
It takes a query string and returns a dataframe of matching records.</p>
<p dir="auto">A wild-card query can return quite a lot of records, so by default only the first 10 are fetched.
The fetch more (or less), use the <code>fetch</code> keyword argument. Setting <code>fetch = :all</code> fetches all results.
Use with care.</p>
<p dir="auto">The number of found records are printed, so in an interactive session it is each to set fetch to a reasonable value.
To get the number of matched records programmatically, the <code>cas_search</code> function is exported.
This returns a Dict, where the value of the key "count" gives the number of found records.</p>

<p dir="auto">This package uses the Cas Common Chemistry web API, which is described here: <a href="https://commonchemistry.cas.org/api-overview" rel="nofollow">https://commonchemistry.cas.org/api-overview</a>. As a concequence, it only works with access to the internet.</p>
