<p dir="auto">This repo contains collection of useful functions for generating, solving, and plotting rectangular, triangular, and hexagonal 2D mazes.</p>
<p dir="auto">Most of the algorithms implemented in this repository are adapted from the excellent book, <a href="https://pragprog.com/titles/jbmaze/mazes-for-programmers/" rel="nofollow">Mazes for Programmers</a> by Jamis Buck.</p>
