<p dir="auto">ProjectAssigner.jl is a tool for assigning students from a large class (e.g. Senior Design) to projects based on their preferences, skills, and small friend groups.
It can be used via <a href="https://julialang.org/" rel="nofollow">Julia</a> or Python through <a href="https://github.com/JuliaPy/pyjulia">pyjulia</a>.</p>
<p dir="auto">Please see the documentation by clicking here for instructions and usage: <a href="https://zsunberg.github.io/ProjectAssigner.jl/dev" rel="nofollow"><img src="https://camo.githubusercontent.com/7fcec4b2d3ab291529fce8ef6a4fcd4129a0683b2f5d5fe2f5c648f02db8b616/68747470733a2f2f696d672e736869656c64732e696f2f62616467652f646f63732d6465762d626c75652e737667" alt="Dev" data-canonical-src="https://img.shields.io/badge/docs-dev-blue.svg" style="max-width: 100%;"></a>.</p>
<p dir="auto">More details about the approach can be found in our <a href="https://peer.asee.org/37187" rel="nofollow">ASEE conference paper</a>.</p>
<p dir="auto">This tool was originally developed for the senior design course in <a href="https://www.colorado.edu/aerospace" rel="nofollow">Aerospace Engineering Sciences at the University of Colorado Boulder</a>.</p>
