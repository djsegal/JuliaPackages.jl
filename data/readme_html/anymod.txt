<p><a href="https://github.com/leonardgoeke/AnyMOD.jl">AnyMOD.jl</a> is a <a href="https://julialang.org/" rel="nofollow">Julia</a> framework for creating large scale energy system models with multiple periods of capacity expansion formulated as linear optimization problems. It was developed to address the challenges in modelling high-levels of intermittent generation and sectoral integration. A comprehensive description of the framework's graph based methodology can found in the working paper <a href="https://arxiv.org/abs/2004.10184" rel="nofollow">Göke (2020), AnyMOD - A graph-based framework for energy system modelling with high levels of renewables and sector integration</a>. The software itself is seperately introduced in <a href="https://arxiv.org/abs/2011.00895" rel="nofollow">Göke (2020), AnyMOD.jl: A Julia package for creating energy system models</a>.</p>
<p>Any questions, suggestions, feature requests, or contributions are welcome. To get in touch open an <a href="https://github.com/leonardgoeke/AnyMOD.jl/issues">issue</a> or use the <a href="https://gitter.im/AnyMOD-jl/community" rel="nofollow">chat</a>.</p>
