<p dir="auto">This Julia package aims to provide a generic interface to handle various type of parametrized intensity models for radio and optical interfeometric data sets (in particular in the emphasis of VLBI data sets).</p>
<p dir="auto">The conceptial idea of the interface is largely based on the Lindy Blackburn's (<a href="https://github.com/lindyblackburn">@lindybackburn</a>) python library <a href="https://github.com/lindyblackburn/modmod">modmod</a>, which allows:</p>
<ul dir="auto">
<li>to compose a generic and hybrid model of various parameterized models including geometric and more generic pixeltized image models through basic primitives and transformations.</li>
<li>to have analytic or numerical correspondence between image and fourier domain.</li>
<li>delayed evaluation of image or uv products of a model through computational graph.</li>
</ul>
<p dir="auto">This package is a partial fork (or purge) of the Paul Tiede's (<a href="https://github.com/ptiede">@ptiede</a>) Julia package <a href="https://github.com/ptiede/Comrade.jl">Comrade.jl</a>, implementing the above interface for the parameterized model. The pure Julia imprementation of parametrized models offer efficient integration of these functions into various inference packages available in Julia.</p>
