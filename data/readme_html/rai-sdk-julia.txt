<p dir="auto">The RelationalAI (RAI) SDK for Julia enables developers to access the RAI REST APIs from Julia.</p>
<ul dir="auto">
<li>You can find RelationalAI Julia SDK documentation at <a href="https://docs.relational.ai/rkgms/sdk/julia-sdk" rel="nofollow">https://docs.relational.ai/rkgms/sdk/julia-sdk</a></li>
<li>You can find RelationalAI product documentation at <a href="https://docs.relational.ai" rel="nofollow">https://docs.relational.ai</a></li>
<li>You can learn more about RelationalAI at <a href="https://relational.ai" rel="nofollow">https://relational.ai</a></li>
</ul>

<p dir="auto">In order to run the examples you will need to create an SDK config file.
The default location for the file is <code>$HOME/.rai/config</code> and the file should
include the following:</p>
<p dir="auto">Sample configuration using OAuth client credentials:</p>
<div class="snippet-clipboard-content notranslate position-relative overflow-auto" data-snippet-clipboard-copy-content="[default]
host = azure.relationalai.com
port = &lt;api-port&gt;      # optional, default: 443
scheme = &lt;scheme&gt;      # optional, default: https
client_id = &lt;your client_id&gt;
client_secret = &lt;your client secret&gt;
client_credentials_url = &lt;account login URL&gt;  # optional
# default: https://login.relationalai.com/oauth/token"><pre lang="conf" class="notranslate"><code>[default]
host = azure.relationalai.com
port = &lt;api-port&gt;      # optional, default: 443
scheme = &lt;scheme&gt;      # optional, default: https
client_id = &lt;your client_id&gt;
client_secret = &lt;your client secret&gt;
client_credentials_url = &lt;account login URL&gt;  # optional
# default: https://login.relationalai.com/oauth/token
</code></pre></div>
<p dir="auto">Client credentials can be created using the RAI console at <a href="https://console.relationalai.com/login" rel="nofollow">https://console.relationalai.com/login</a></p>
<p dir="auto">You can copy <code>config.spec</code> from the root of this repo and modify as needed.</p>
