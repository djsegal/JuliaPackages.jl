<p dir="auto"><strong>This package is under development. Use at your own risk.</strong></p>
<p dir="auto">MadDiff.jl is an automatic differentiation and algebraic modeling package. MadDiff.jl constructs first and second derivative functions off-line (i.e., prior to calling the optimization solver) by applying operator overloading-based automatic differentiation on <em>functions</em>. The exact <em>derivative functions</em> can be obtained as results. A benchmark code can be found <a href="https://github.com/sshin23/MadDiff.jl/tree/main/benchmark">here</a>.</p>
