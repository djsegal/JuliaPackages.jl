<h1><a id="user-content-sarsop" class="anchor" aria-hidden="true" href="#sarsop"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path fill-rule="evenodd" d="M4 9h1v1H4c-1.5 0-3-1.69-3-3.5S2.55 3 4 3h4c1.45 0 3 1.69 3 3.5 0 1.41-.91 2.72-2 3.25V8.59c.58-.45 1-1.27 1-2.09C10 5.22 8.98 4 8 4H4c-.98 0-2 1.22-2 2.5S3 9 4 9zm9-3h-1v1h1c1 0 2 1.22 2 2.5S13.98 12 13 12H9c-.98 0-2-1.22-2-2.5 0-.83.42-1.64 1-2.09V6.25c-1.09.53-2 1.84-2 3.25C6 11.31 7.55 13 9 13h4c1.45 0 3-1.69 3-3.5S14.5 6 13 6z"></path></svg></a>SARSOP</h1>
<p>This Julia package wraps the <a href="http://bigbird.comp.nus.edu.sg/pmwiki/farm/appl/" rel="nofollow">SARSOP</a> software for offline POMDP planning.
It works with the <a href="https://github.com/JuliaPOMDP/POMDPs.jl">POMDPS.jl</a> interface.
A module for writing POMDPX files is provided through the <a href="https://github.com/JuliaPOMDP/POMDPXFile.jl">POMDPXFile.jl</a> package, and is a dependancy for SARSOP.jl.</p>

<p>You must have <a href="https://github.com/JuliaPOMDP/POMDPs.jl">POMDPs.jl</a> installed. To install SARSOP and its Julia wrapper run the following command:</p>
<div class="highlight highlight-source-julia"><pre><span class="pl-k">using</span> POMDPs
<span class="pl-k">using</span> Pkg
POMDPs<span class="pl-k">.</span><span class="pl-c1">add_registry</span>()
Pkg<span class="pl-k">.</span><span class="pl-c1">add</span>(<span class="pl-s"><span class="pl-pds">"</span>SARSOP<span class="pl-pds">"</span></span>)</pre></div>
