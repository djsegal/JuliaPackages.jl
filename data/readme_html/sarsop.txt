<h1><a id="user-content-sarsop" class="anchor" aria-hidden="true" href="#sarsop"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path fill-rule="evenodd" d="M7.775 3.275a.75.75 0 001.06 1.06l1.25-1.25a2 2 0 112.83 2.83l-2.5 2.5a2 2 0 01-2.83 0 .75.75 0 00-1.06 1.06 3.5 3.5 0 004.95 0l2.5-2.5a3.5 3.5 0 00-4.95-4.95l-1.25 1.25zm-4.69 9.64a2 2 0 010-2.83l2.5-2.5a2 2 0 012.83 0 .75.75 0 001.06-1.06 3.5 3.5 0 00-4.95 0l-2.5 2.5a3.5 3.5 0 004.95 4.95l1.25-1.25a.75.75 0 00-1.06-1.06l-1.25 1.25a2 2 0 01-2.83 0z"></path></svg></a>SARSOP</h1>
<p>This Julia package wraps the <a href="http://bigbird.comp.nus.edu.sg/pmwiki/farm/appl/" rel="nofollow">SARSOP</a> software for offline POMDP planning.
It works with the <a href="https://github.com/JuliaPOMDP/POMDPs.jl">POMDPS.jl</a> interface.
A module for writing POMDPX files is provided through the <a href="https://github.com/JuliaPOMDP/POMDPXFile.jl">POMDPXFile.jl</a> package, and is a dependancy for SARSOP.jl.</p>
