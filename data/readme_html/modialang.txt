<p dir="auto"><a href="https://github.com/ModiaSim/ModiaLang.jl/blob/master/LICENSE.md"><img src="https://camo.githubusercontent.com/ad8758fbaebbced78645b98e446c0bb5ec223676ed61700184320887cadbfb8e/68747470733a2f2f696d672e736869656c64732e696f2f62616467652f6c6963656e73652d4d49542d627269676874677265656e2e7376673f7374796c653d666c61742d737175617265" alt="The MIT License" data-canonical-src="https://img.shields.io/badge/license-MIT-brightgreen.svg?style=flat-square" style="max-width: 100%;"></a></p>
<p dir="auto">ModiaLang is part of <a href="https://modiasim.github.io/docs/" rel="nofollow">ModiaSim</a>.</p>
<p dir="auto">ModiaLang is usually used via <a href="https://github.com/ModiaSim/Modia.jl">Modia</a>.</p>
<p dir="auto">ModiaLang provides the core equation-based language of Modia, transformation of a model to ODE form dx/dt = f(x,t) and a thin interface to <a href="https://github.com/SciML/DifferentialEquations.jl">DifferentialEquations</a>.</p>
