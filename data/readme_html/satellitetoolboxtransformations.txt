<p dir="auto">This package contains transformations that are useful for the <strong>SatelliteToolbox.jl</strong>
ecosystem. Currently, the following algorithms are available:</p>
<ul dir="auto">
<li>Conversion between ECI and ECEF reference frames (IAU-76/FK5 and IAU-2006/2010A);</li>
<li>Conversion between ECEF and local frames (NED);</li>
<li>Conversion between geodetic and geocentric variables; and</li>
<li>Conversion between time epochs (UTC, UT1, TT).</li>
</ul>
