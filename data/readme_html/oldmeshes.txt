<p>This package is designed to make it easy to work with <a href="https://en.wikipedia.org/wiki/Polygon_mesh" rel="nofollow">polygon mesh data</a>.
It is a primarily a meta-package for
<a href="https://github.com/JuliaGeometry/Meshing.jl">Meshing</a>,
<a href="https://github.com/JuliaIO/MeshIO.jl">MeshIO</a>,
and <a href="https://github.com/JuliaGeometry/GeometryTypes.jl">GeometryTypes</a>.
In addition, it is a great (and recommended) place for experimental
development that is not yet congruent with the GeometryTypes type heirarchy.</p>
<p>The current release series corresponds to <code>v0.2.x</code>. The pre-GeometryTypes, et. al.
version, <a href="https://github.com/JuliaGeometry/Meshes.jl/tree/v0.1.x"><code>v0.1.x</code></a>,
is still supported by <a href="https://github.com/sjkelly/">@sjkelly</a>.</p>

<p>Meshes does not define many datatypes, but rather it uses those defined by
<a href="https://github.com/JuliaGeometry/GeometryTypes.jl">GeometryTypes</a> where possible.</p>

<p>This functionality is derived from the
<a href="https://github.com/JuliaGeometry/Meshing.jl">Meshing</a> package.
Full documentation is available there.
In combination with GeometryTypes it is easy to mesh implicit functions.</p>
<pre><code>using Meshes
using GeometryTypes

s = SignedDistanceField(HyperRectangle(Vec(0,0,0.),Vec(1,1,1.))) do v
           sqrt(sum(dot(v,v))) - 1 # sphere
       end
m = HomogenousMesh(s) # uses Marching Tetrahedra from Meshing.jl
save("eighth_sphere.ply",m)
</code></pre>

<h2><a id="user-content-history" class="anchor" aria-hidden="true" href="#history"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path fill-rule="evenodd" d="M4 9h1v1H4c-1.5 0-3-1.69-3-3.5S2.55 3 4 3h4c1.45 0 3 1.69 3 3.5 0 1.41-.91 2.72-2 3.25V8.59c.58-.45 1-1.27 1-2.09C10 5.22 8.98 4 8 4H4c-.98 0-2 1.22-2 2.5S3 9 4 9zm9-3h-1v1h1c1 0 2 1.22 2 2.5S13.98 12 13 12H9c-.98 0-2-1.22-2-2.5 0-.83.42-1.64 1-2.09V6.25c-1.09.53-2 1.84-2 3.25C6 11.31 7.55 13 9 13h4c1.45 0 3-1.69 3-3.5S14.5 6 13 6z"></path></svg></a>History</h2>
<p>This package started with an exclusive focus on Mesh geometry. For the first
year of existence it worked exclusively for computational geometry purposes. In
early 2015 an effort started to generalize for compatibilty with display
libraries such as OpenGL. This meant redesigning types and operations.
Ultimately it was broken up into other packages to facilitate collaboration.
Most notably
<a href="https://github.com/JuliaGeometry/Meshing.jl">Meshing</a>,
<a href="https://github.com/JuliaIO/MeshIO.jl">MeshIO</a>,
and <a href="https://github.com/JuliaGeometry/GeometryTypes.jl">GeometryTypes</a>
contain a significant portion of the original code. We plan to support this
package as a meta-package for domain focused usability and extensions.</p>
