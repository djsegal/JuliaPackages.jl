<p>HartreeFockBogoliubov.jl is a <a href="https://julialang.org" rel="nofollow">Julia</a> package for solving interacting fermion Hamiltonian using Hartree-Fock-Bogoliubov (HFB) approach. This project aims to assist fast development of HFB solver for a generic interacting Hamiltonian.</p>
