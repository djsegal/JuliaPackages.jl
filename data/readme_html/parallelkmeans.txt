<p>You can grab the latest stable version of this package by simply running in Julia.
Don't forget to Julia's package manager with <code>]</code></p>
<div class="highlight highlight-source-julia"><pre>pkg<span class="pl-k">&gt;</span> add ParallelKMeans</pre></div>
<p>For the few (and selected) brave ones, one can simply grab the current experimental features by simply adding the experimental branch to your development environment after invoking the package manager with <code>]</code>:</p>
<div class="highlight highlight-source-julia"><pre>pkg<span class="pl-k">&gt;</span> dev git<span class="pl-c1">@github</span><span class="pl-k">.</span>com<span class="pl-k">:</span>PyDataBlog<span class="pl-k">/</span>ParallelKMeans<span class="pl-k">.</span>jl<span class="pl-k">.</span>git</pre></div>
<p>Don't forget to checkout the experimental branch and you are good to go with bleeding edge features and breaks!</p>
<div class="highlight highlight-source-shell"><pre>git checkout experimental</pre></div>

<h3><a id="user-content-features" class="anchor" aria-hidden="true" href="#features"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path fill-rule="evenodd" d="M4 9h1v1H4c-1.5 0-3-1.69-3-3.5S2.55 3 4 3h4c1.45 0 3 1.69 3 3.5 0 1.41-.91 2.72-2 3.25V8.59c.58-.45 1-1.27 1-2.09C10 5.22 8.98 4 8 4H4c-.98 0-2 1.22-2 2.5S3 9 4 9zm9-3h-1v1h1c1 0 2 1.22 2 2.5S13.98 12 13 12H9c-.98 0-2-1.22-2-2.5 0-.83.42-1.64 1-2.09V6.25c-1.09.53-2 1.84-2 3.25C6 11.31 7.55 13 9 13h4c1.45 0 3-1.69 3-3.5S14.5 6 13 6z"></path></svg></a>Features</h3>
<ul>
<li>Lightening fast implementation of Kmeans clustering algorithm even on a single thread in native Julia.</li>
<li>Support for multi-theading implementation of K-Means clustering algorithm.</li>
<li>Kmeans++ initialization for faster and better convergence.</li>
<li>Implementation of all the variants of the K-Means algorithm.</li>
</ul>
