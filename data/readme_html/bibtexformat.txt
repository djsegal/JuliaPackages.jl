<p><code>BibTeXFormat.jl</code> is a port of the formatting submodule of the python library <a href="https://pybtex.org/" rel="nofollow">Pybtex!</a>. It allows to format a bibliography parsed with <a href="https://github.com/JuliaTeX/BibTeX.jl">BibTeX.jl</a> and convert it to HTML, LaTeX, Markdown or plain text</p>
