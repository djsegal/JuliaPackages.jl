<p>A package for rendering simple scenes primarily consisting of cars on roadways using Cairo.</p>
<p>AutomotiveVisualization is undergoing significant changes. If you are looking for the version before these changes that is designed around AutomotiveSimulator.jl, please checkout the v0.6.0 tag.</p>
