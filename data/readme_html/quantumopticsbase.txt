<p dir="auto"><strong>QuantumOpticsBase.jl</strong> provides the base functionality for <strong>QuantumOptics.jl</strong>. For the main project, see <a href="https://github.com/qojulia/QuantumOptics.jl">https://github.com/qojulia/QuantumOptics.jl</a> or the website of the @qojulia organization <a href="http://qojulia.org" rel="nofollow">http://qojulia.org</a>. Specifically, <strong>QuantumOpticsBase</strong> implements basic types such as different bases, states and operators defined on these bases, and fundamental operations (such as multiplication) on these states/operators.</p>
