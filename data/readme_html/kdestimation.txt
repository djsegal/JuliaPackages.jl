<p><a href="https://github.com/JuliaStats/Distributions.jl">Distributions.jl</a> defines <code>(loc,scale) = (a, b - a))</code> where <code>a</code> and <code>b</code> are the bounds lower and upper bounds, respectively.
This package accounts for this inconsistancy by evaluating the Uniform kernel as</p>
<p><a target="_blank" rel="noopener noreferrer" href="https://camo.githubusercontent.com/b91a1aaea31014b9ed13ebbd61d6297ab3b77d788070090d9f7213890b5bef26/68747470733a2f2f6c617465782e636f6465636f67732e636f6d2f6769662e6c617465783f5c696e6c696e652673706163653b5c746578747b7064667d285c746578747b556e69666f726d7d28785f692d5c667261637b687d7b327d2c785f692b5c667261637b687d7b327d292c7829"><img src="https://camo.githubusercontent.com/b91a1aaea31014b9ed13ebbd61d6297ab3b77d788070090d9f7213890b5bef26/68747470733a2f2f6c617465782e636f6465636f67732e636f6d2f6769662e6c617465783f5c696e6c696e652673706163653b5c746578747b7064667d285c746578747b556e69666f726d7d28785f692d5c667261637b687d7b327d2c785f692b5c667261637b687d7b327d292c7829" data-canonical-src="https://latex.codecogs.com/gif.latex?\inline&amp;space;\text{pdf}(\text{Uniform}(x_i-\frac{h}{2},x_i+\frac{h}{2}),x)" style="max-width:100%;"></a>.</p>

<p>The objective function to minimize is given by</p>
<p><a target="_blank" rel="noopener noreferrer" href="https://camo.githubusercontent.com/e8f2ed4526d415493f2c0b6209b62d4fe94fc774a261ff75a55d845b83c4f6f0/68747470733a2f2f6c617465782e636f6465636f67732e636f6d2f6769662e6c617465783f2673706163653b4c5343562868293d5c696e745c6861747b667d5e325f6828782964782d5c667261637b327d7b6e7d5c73756d5f695c6861747b667d5f7b682c2d697d28585f6929"><img src="https://camo.githubusercontent.com/e8f2ed4526d415493f2c0b6209b62d4fe94fc774a261ff75a55d845b83c4f6f0/68747470733a2f2f6c617465782e636f6465636f67732e636f6d2f6769662e6c617465783f2673706163653b4c5343562868293d5c696e745c6861747b667d5e325f6828782964782d5c667261637b327d7b6e7d5c73756d5f695c6861747b667d5f7b682c2d697d28585f6929" data-canonical-src="https://latex.codecogs.com/gif.latex?&amp;space;LSCV(h)=\int\hat{f}^2_h(x)dx-\frac{2}{n}\sum_i\hat{f}_{h,-i}(X_i)" style="max-width:100%;"></a></p>
<p>where</p>
<p><a target="_blank" rel="noopener noreferrer" href="https://camo.githubusercontent.com/9b2e1ab05bae6e236f6af97017cb997fd7b3ef8a3c8e8ecad569c05eb17ffdec/68747470733a2f2f6c617465782e636f6465636f67732e636f6d2f6769662e6c617465783f2673706163653b5c6861747b667d5f7b682c2d697d28585f69293d5c667261637b317d7b286e2d3129687d5c73756d5f7b6a5c6e652673706163653b697d4b5c6c656674285c667261637b585f692d585f6a7d7b687d5c726967687429"><img src="https://camo.githubusercontent.com/9b2e1ab05bae6e236f6af97017cb997fd7b3ef8a3c8e8ecad569c05eb17ffdec/68747470733a2f2f6c617465782e636f6465636f67732e636f6d2f6769662e6c617465783f2673706163653b5c6861747b667d5f7b682c2d697d28585f69293d5c667261637b317d7b286e2d3129687d5c73756d5f7b6a5c6e652673706163653b697d4b5c6c656674285c667261637b585f692d585f6a7d7b687d5c726967687429" data-canonical-src="https://latex.codecogs.com/gif.latex?&amp;space;\hat{f}_{h,-i}(X_i)=\frac{1}{(n-1)h}\sum_{j\ne&amp;space;i}K\left(\frac{X_i-X_j}{h}\right)" style="max-width:100%;"></a></p>
