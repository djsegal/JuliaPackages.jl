<p>The purpose of this package is to provide a general framework for implementing Kernel Density Estimation methods.</p>

<p>The density estimator</p>
<p><a target="_blank" rel="noopener noreferrer" href="https://camo.githubusercontent.com/fb577d475c9ff8e0cc4088748d3fa95b3f406368/68747470733a2f2f6c617465782e636f6465636f67732e636f6d2f7376672e6c617465783f2673706163653b253543686174253742662537442878293d25354366726163253742312537442537426e25374425354373756d5f253742693d312537442535456e4b2535436c6566742825354366726163253742782d785f6925374425374268253744253543726967687429"><img src="https://camo.githubusercontent.com/fb577d475c9ff8e0cc4088748d3fa95b3f406368/68747470733a2f2f6c617465782e636f6465636f67732e636f6d2f7376672e6c617465783f2673706163653b253543686174253742662537442878293d25354366726163253742312537442537426e25374425354373756d5f253742693d312537442535456e4b2535436c6566742825354366726163253742782d785f6925374425374268253744253543726967687429" alt="equation" data-canonical-src="https://latex.codecogs.com/svg.latex?&amp;space;%5Chat%7Bf%7D(x)=%5Cfrac%7B1%7D%7Bn%7D%5Csum_%7Bi=1%7D%5EnK%5Cleft(%5Cfrac%7Bx-x_i%7D%7Bh%7D%5Cright)" style="max-width:100%;"></a></p>
<p>where</p>
<ul>
<li><a target="_blank" rel="noopener noreferrer" href="https://camo.githubusercontent.com/a167fa997ba8190280c0c571f664c58e78e0ea1d/68747470733a2f2f6c617465782e636f6465636f67732e636f6d2f7376672e6c617465783f253543696e6c696e652673706163653b25354368617425374266253744287829"><img src="https://camo.githubusercontent.com/a167fa997ba8190280c0c571f664c58e78e0ea1d/68747470733a2f2f6c617465782e636f6465636f67732e636f6d2f7376672e6c617465783f253543696e6c696e652673706163653b25354368617425374266253744287829" alt="equation" data-canonical-src="https://latex.codecogs.com/svg.latex?%5Cinline&amp;space;%5Chat%7Bf%7D(x)" style="max-width:100%;"></a> is the estimator</li>
<li><a target="_blank" rel="noopener noreferrer" href="https://camo.githubusercontent.com/cf3976cff68d09f3ae8a4b99d1122c02ea0de0da/68747470733a2f2f6c617465782e636f6465636f67732e636f6d2f7376672e6c617465783f253543696e6c696e652673706163653b4b287529"><img src="https://camo.githubusercontent.com/cf3976cff68d09f3ae8a4b99d1122c02ea0de0da/68747470733a2f2f6c617465782e636f6465636f67732e636f6d2f7376672e6c617465783f253543696e6c696e652673706163653b4b287529" alt="equation" data-canonical-src="https://latex.codecogs.com/svg.latex?%5Cinline&amp;space;K(u)" style="max-width:100%;"></a> is the kernel function</li>
<li><a target="_blank" rel="noopener noreferrer" href="https://camo.githubusercontent.com/8510a9e63e9b9e1e190df0a65ccdb8113db12c7f/68747470733a2f2f6c617465782e636f6465636f67732e636f6d2f7376672e6c617465783f253543696e6c696e652673706163653b68"><img src="https://camo.githubusercontent.com/8510a9e63e9b9e1e190df0a65ccdb8113db12c7f/68747470733a2f2f6c617465782e636f6465636f67732e636f6d2f7376672e6c617465783f253543696e6c696e652673706163653b68" alt="equation" data-canonical-src="https://latex.codecogs.com/svg.latex?%5Cinline&amp;space;h" style="max-width:100%;"></a> is the bandwidth</li>
</ul>
<p>can be evaluated using one of three implemented methods.</p>
<ul>
<li><code>Direct() # O(N^2) where N is the sample size</code></li>
<li><code>Binned() # O(M^2) where M is the number of evaluation points (M=4096 by default)</code></li>
<li><code>Direct() # O(MlogM) where M is the number of evaluation points (M=4096 by default)</code></li>
</ul>
