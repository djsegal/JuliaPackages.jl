<p dir="auto">Provides a single function <code>why()</code> which returns a randomly generated String giving an explanation. Example outputs:</p>
<p dir="auto"><code>"Because some harmless and properly type-safe cockatoo and the fast type pirate decided so."</code></p>
<p dir="auto"><code>"Some deceitful and vaguely self aware person and the programmer wanted it."</code></p>
<p dir="auto"><code>"Some not at all self aware and very self aware compiler and some deceitful research software engineer decided so."</code></p>
<p dir="auto"><code>"The rudimentary struct and a self aware research software engineer and a vaguely scary config file could not prevent it."</code></p>
<p dir="auto">A follow up function <code>how()</code> is a current work in progress.</p>
<p dir="auto">(Initially a project from the JuliaCon 2018 hackathon)</p>
