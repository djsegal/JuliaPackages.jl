<h1><a id="user-content-whyjl" class="anchor" aria-hidden="true" href="#whyjl"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path fill-rule="evenodd" d="M4 9h1v1H4c-1.5 0-3-1.69-3-3.5S2.55 3 4 3h4c1.45 0 3 1.69 3 3.5 0 1.41-.91 2.72-2 3.25V8.59c.58-.45 1-1.27 1-2.09C10 5.22 8.98 4 8 4H4c-.98 0-2 1.22-2 2.5S3 9 4 9zm9-3h-1v1h1c1 0 2 1.22 2 2.5S13.98 12 13 12H9c-.98 0-2-1.22-2-2.5 0-.83.42-1.64 1-2.09V6.25c-1.09.53-2 1.84-2 3.25C6 11.31 7.55 13 9 13h4c1.45 0 3-1.69 3-3.5S14.5 6 13 6z"></path></svg></a>Why.jl</h1>
<p>Provides a single function <code>why()</code> which returns a randomly generated String giving an explanation. Example outputs:</p>
<p><code>"Because some harmless and properly type-safe cockatoo and the fast type pirate decided so."</code></p>
<p><code>"Some deceitful and vaguely self aware person and the programmer wanted it."</code></p>
<p><code>"Some not at all self aware and very self aware compiler and some deceitful research software engineer decided so."</code></p>
<p><code>"The rudimentary struct and a self aware research software engineer and a vaguely scary config file could not prevent it."</code></p>
<p>A follow up function <code>how()</code> is a current work in progress.</p>
<p>(Initially a project from the JuliaCon 2018 hackathon)</p>
