<p>This package implements OrderedDicts and OrderedSets, which are similar to containers in base Julia.
However, during iteration the Ordered* containers return items in the order in which they were added to the collection.
It also implements <code>LittleDict</code> which is a ordered dictionary, that is much faster than any other <code>AbstractDict</code> (ordered or not) for small collections.</p>
<p>This package was split out from <a href="https://github.com/JuliaCollections/DataStructures.jl">DataStructures.jl</a>.</p>
