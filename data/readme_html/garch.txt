<h2><a id="user-content-what-is-implemented" class="anchor" aria-hidden="true" href="#what-is-implemented"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path fill-rule="evenodd" d="M7.775 3.275a.75.75 0 001.06 1.06l1.25-1.25a2 2 0 112.83 2.83l-2.5 2.5a2 2 0 01-2.83 0 .75.75 0 00-1.06 1.06 3.5 3.5 0 004.95 0l2.5-2.5a3.5 3.5 0 00-4.95-4.95l-1.25 1.25zm-4.69 9.64a2 2 0 010-2.83l2.5-2.5a2 2 0 012.83 0 .75.75 0 001.06-1.06 3.5 3.5 0 00-4.95 0l-2.5 2.5a3.5 3.5 0 004.95 4.95l1.25-1.25a.75.75 0 00-1.06-1.06l-1.25 1.25a2 2 0 01-2.83 0z"></path></svg></a>What is implemented</h2>
<ul>
<li>garchFit - estimates parameters of univariate normal GARCH process.</li>
<li>predict - make n-step prediction using fitted object returned by garchFit</li>
<li>Jarque-Bera residuals test</li>
<li>Error analysis</li>
<li>Package test (compares model parameters and predictions with those obtained using R fGarch)</li>
</ul>
<p>Analysis of model residuals - currently only Jarque-Bera Test implemented.</p>

<h4><a id="user-content-returns" class="anchor" aria-hidden="true" href="#returns"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path fill-rule="evenodd" d="M7.775 3.275a.75.75 0 001.06 1.06l1.25-1.25a2 2 0 112.83 2.83l-2.5 2.5a2 2 0 01-2.83 0 .75.75 0 00-1.06 1.06 3.5 3.5 0 004.95 0l2.5-2.5a3.5 3.5 0 00-4.95-4.95l-1.25 1.25zm-4.69 9.64a2 2 0 010-2.83l2.5-2.5a2 2 0 012.83 0 .75.75 0 001.06-1.06 3.5 3.5 0 00-4.95 0l-2.5 2.5a3.5 3.5 0 004.95 4.95l1.25-1.25a.75.75 0 00-1.06-1.06l-1.25 1.25a2 2 0 01-2.83 0z"></path></svg></a>returns:</h4>
<p>Structure containing details of the GARCH fit with the fllowing fields:</p>
<ul>
<li>data - orginal data</li>
<li>params - vector of model parameters (omega,alpha,beta)</li>
<li>llh - likelihood</li>
<li>status - status of the solver</li>
<li>converged - boolean convergence status, true if constraints are satisfied</li>
<li>sigma - conditional volatility</li>
<li>hessian - Hessian matrix</li>
<li>secoef - standard errors</li>
<li>tval - t-statistics</li>
</ul>
