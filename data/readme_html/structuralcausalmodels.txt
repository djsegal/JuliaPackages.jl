<p dir="auto">StructuralCausalModels.jl is part of the StatisticalRethinkingJulia eco system and contains functionality to analyse directed acyclic graph (DAG) based causal models as described in <a href="https://xcelab.net/rm/statistical-rethinking/" rel="nofollow">StatisticalRethinking</a>, <a href="http://bcs.wiley.com/he-bcs/Books?action=index&amp;bcsId=10288&amp;itemId=1119186846" rel="nofollow">Causal Inference in Statistics</a> and <a href="https://www.cambridge.org/core/books/cause-and-correlation-in-biology/247799189B31939D24BC0F61FD59E9BB#" rel="nofollow">Cause and Correlation in Biology</a>.</p>
<p dir="auto">My initial goal for this package is to have a way to apply SCM ideas to the examples in <a href="https://github.com/StatisticalRethinkingJulia">StatisticalRethinking.jl</a>, i.e. a working version of <code>basis_set()</code>, <code>d_separation()</code>, <code>m_separations()</code> and <code>adjustment_sets()</code>.</p>
<p dir="auto">From the point of view of above functionality, I believe the package is close to R's <code>ggm</code> (including most of Sadeghi's additions). I'm hoping version 1.0.0 has a similar API but many more test cases, including more comparisons with R's <code>dagitty</code>.</p>
<p dir="auto">The status of the package remains experimental and is, as is StatisticalRethinking.jl, primarily intended for learning statistical modeling approaches and pitfalls.</p>
<p dir="auto">StructuralCausalModels.jl can be installed using
<code>] add StructuralCausalModels</code>.</p>
