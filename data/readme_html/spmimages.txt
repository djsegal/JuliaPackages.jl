<p dir="auto">A julia library to read and display SPM (scanning probe microscopy) images. Currently, only <a href="https://www.specs-group.com/nanonis/products/" rel="nofollow">Nanonis</a> files are supported.</p>
<p dir="auto">The commands are somewhat similar to the ones in the respective python version <a href="https://github.com/alexriss/imagex">imag<em>ex</em></a>.</p>
