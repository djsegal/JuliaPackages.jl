<p dir="auto">A Julia package for nonparametric regression with Cubic Smoothing Splines. The initial aim is to provide the same functionality as R's <code>smooth.spline</code> function and competitive computational performance. The implementation however is completely independent of the R function and based on the algorithm by Reinsch [1], as described in Chapter 2 of [2].</p>
