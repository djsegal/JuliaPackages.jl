<p dir="auto">This package provides the functions to correct quadratic phase aberration. It is easy to use. When the following code is executed, the phase <code>ϕ</code> is corrected by the least squares algorithm and stored in <code>ψ</code>.</p>
<div class="highlight highlight-source-julia notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="ψ = CorrectAberration(ϕ)"><pre>ψ <span class="pl-k">=</span> <span class="pl-c1">CorrectAberration</span>(ϕ)</pre></div>
<p dir="auto">When using the weighted least square algorithm, use the following function.</p>
<div class="highlight highlight-source-julia notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="ψ = WeightedCorrectAberration(ϕ, 10, 10)"><pre>ψ <span class="pl-k">=</span> <span class="pl-c1">WeightedCorrectAberration</span>(ϕ, <span class="pl-c1">10</span>, <span class="pl-c1">10</span>)</pre></div>
