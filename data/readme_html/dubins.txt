<p>Dubins.jl is a Julia package for computing the shortest path between two configurations for the Dubins' vehicle (see <a href="http://www.jstor.org/stable/2372560?seq=1#page_scan_tab_contents" rel="nofollow">Dubins, 1957</a>). The shortest path algorithm, implemented in this package, uses the algebraic solution approach in the paper "<a href="https://www.sciencedirect.com/science/article/pii/S0921889000001275" rel="nofollow">Classification of the Dubins set</a>" by Andrei M. Shkel and Vladimir Lumelsky.</p>
<p>All the unit tests and the data structures in this Julia implementation is borrowed from the <a href="https://github.com/AndrewWalker/Dubins-Curves">C implementation</a> and its corresponding <a href="https://github.com/AndrewWalker/pydubins">python wrapper</a>.</p>
