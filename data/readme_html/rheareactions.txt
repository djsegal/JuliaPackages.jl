<p dir="auto"><a href="https://www.repostatus.org/#active" rel="nofollow"><img src="https://camo.githubusercontent.com/2261082c77808ea734741b12e535d02d23c4101f6b8dfec807f4ddc5ef2eeec0/68747470733a2f2f7777772e7265706f7374617475732e6f72672f6261646765732f6c61746573742f6163746976652e737667" alt="repostatus-img" data-canonical-src="https://www.repostatus.org/badges/latest/active.svg" style="max-width: 100%;"></a></p>
<p dir="auto">This is a simple package you can use to query Rhea reactions and associated
annotations. Its primary use is in reconstructing metabolic models. It caches
all requests by default, speeding up repeated calls where appropriate.</p>
<div class="highlight highlight-source-julia notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="using RheaReactions # load module

get_reaction(11364) # Rhea reaction ID 11364"><pre><span class="pl-k">using</span> RheaReactions <span class="pl-c"><span class="pl-c">#</span> load module</span>

<span class="pl-c1">get_reaction</span>(<span class="pl-c1">11364</span>) <span class="pl-c"><span class="pl-c">#</span> Rhea reaction ID 11364</span></pre></div>
<p dir="auto">You can also get the metabolites associated with that reaction:</p>
<div class="highlight highlight-source-julia notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="# [(coefficient, metabolite), ...] 
coeff_mets = get_reaction_metabolites(11364) # Rhea reaction ID 11364"><pre><span class="pl-c"><span class="pl-c">#</span> [(coefficient, metabolite), ...] </span>
coeff_mets <span class="pl-k">=</span> <span class="pl-c1">get_reaction_metabolites</span>(<span class="pl-c1">11364</span>) <span class="pl-c"><span class="pl-c">#</span> Rhea reaction ID 11364</span></pre></div>
<p dir="auto">And look at each metabolite individually:</p>
<div class="highlight highlight-source-julia notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="coeff_mets[1][2] # metabolite"><pre>coeff_mets[<span class="pl-c1">1</span>][<span class="pl-c1">2</span>] <span class="pl-c"><span class="pl-c">#</span> metabolite</span></pre></div>
<p dir="auto">You can also look for all reactions that have a certain set of metabolite
substrates and products. This function looks for all reactions that have both
CHEBI:29985 (L-glutamate) and CHEBI:58359 (L-glutamine) on opposite sides
of the reaction:</p>
<div class="highlight highlight-source-julia notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="substrate_ids = [29985,]
product_ids = [58359,]
get_reactions_with_metabolites(
    substrate_ids,
    product_ids,
) # NB: not cached!"><pre>substrate_ids <span class="pl-k">=</span> [<span class="pl-c1">29985</span>,]
product_ids <span class="pl-k">=</span> [<span class="pl-c1">58359</span>,]

    substrate_ids,
    product_ids,

<div class="highlight highlight-source-julia notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="get_reactions_with_ec(&quot;2.5.1.49&quot;)"><pre><span class="pl-c1">get_reactions_with_ec</span>(<span class="pl-s"><span class="pl-pds">"</span>2.5.1.49<span class="pl-pds">"</span></span>)</pre></div>
<p dir="auto">Rhea reactions are typically broken into quartets.: one reference reaction, two
directional reactions, and one bidirectional reaction. You can find all four
reactions given any single reaction:</p>
<div class="highlight highlight-source-julia notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="get_reaction_quartet(11364) # Rhea reaction ID 11364"><pre><span class="pl-c1">get_reaction_quartet</span>(<span class="pl-c1">11364</span>) <span class="pl-c"><span class="pl-c">#</span> Rhea reaction ID 11364</span></pre></div>
