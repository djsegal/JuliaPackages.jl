<p dir="auto">PowerModelsInterface.jl is a Julia package for accessing routines from <a href="https://github.com/lanl-ansi/PowerModels.jl">PowerModels.jl</a> with data contained in <a href="https://github.com/nrel-siip/PowerSystems.jl">PowerSystems.jl</a>.</p>
<p dir="auto"><strong>This package is under development and is a work in progress.</strong> The 0.1.0 tag is being used in the development of PowerSimulations.jl and it is subject to change. Current features will remain supported, and if new features are particularly useful, please open an issue. As always, PRs are welcome.</p>

<p dir="auto"><code>PowerModelsInterface.jl</code> allows users to translate <a href="https://github.com/NREL-SIIP/PowerSystems.jl">PowerSystems.jl</a> data into <code>PowerModels.jl</code> format and provides interfaces to key <code>PowerModels.jl</code> modeling functions. In particular, <code>PowerModelsInterface.jl</code> has three main capabilities:</p>
