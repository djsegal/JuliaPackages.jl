<p dir="auto"><a href="https://github.com/mkhraijah/PowerModelsADA.jl"><code>PowerModelsADA.jl</code></a> (Power Models Alternating Distributed Algorithms) provides a framework to solve Optimal Power Flow (OPF) problems using alternating distributed algorithms. The package allows to use different distributed algorithms. <code>PowerModelsADA</code> is built on top of <a href="https://github.com/lanl-ansi/PowerModels.jl"><code>PowerModels.jl</code></a> and <a href="https://github.com/jump-dev/JuMP.jl"><code>JuMP.jl</code></a> to model and solve the subproblems.</p>
<h2 dir="auto"><a id="user-content-distributed-algorithms" class="anchor" aria-hidden="true" href="#distributed-algorithms"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path d="m7.775 3.275 1.25-1.25a3.5 3.5 0 1 1 4.95 4.95l-2.5 2.5a3.5 3.5 0 0 1-4.95 0 .751.751 0 0 1 .018-1.042.751.751 0 0 1 1.042-.018 1.998 1.998 0 0 0 2.83 0l2.5-2.5a2.002 2.002 0 0 0-2.83-2.83l-1.25 1.25a.751.751 0 0 1-1.042-.018.751.751 0 0 1-.018-1.042Zm-4.69 9.64a1.998 1.998 0 0 0 2.83 0l1.25-1.25a.751.751 0 0 1 1.042.018.751.751 0 0 1 .018 1.042l-1.25 1.25a3.5 3.5 0 1 1-4.95-4.95l2.5-2.5a3.5 3.5 0 0 1 4.95 0 .751.751 0 0 1-.018 1.042.751.751 0 0 1-1.042.018 1.998 1.998 0 0 0-2.83 0l-2.5 2.5a1.998 1.998 0 0 0 0 2.83Z"></path></svg></a>Distributed Algorithms</h2>
<p dir="auto">The <code>PowerModelsADA</code> framework is designed to easily incorporate new alternating distributed algorithms. The framework provides means to decompose a test case into multiple areas, model the subproblems associated with each area using <code>PowerModels</code>, solve the supropblems in parallel using multi-threading, communicate the shared data between the areas, and calculate the mismatches to decide if the termination criteria are satisfied.</p>
<p dir="auto">The current version of <code>PowerModelsADA</code> implements four distributed algorithms:</p>
<ul dir="auto">
<li>Alternating Direction Method of Multipliers (ADMM)</li>
<li>Analytical Target Cascading (ATC)</li>
<li>Auxiliary Problem Principle (APP)</li>
<li>Augmented Lagrangian Alternating Direction Inexact Newton (ALADIN)</li>
</ul>
<p dir="auto"><code>PowerModelsADA</code> can be extended to include variations of the existing algorithms or new user-defined algorithms. More details about the formulations and algorithm implementations are shown in <a href="https://mkhraijah.github.io/PowerModelsADA.jl/dev/specification/" rel="nofollow">Technical Specifications</a></p>
