<p dir="auto">This is a package for executing the same instruction on multiple datapoints.
This is in-particular required for batched forward mode AD, where you must pushforward each basis element of the tangent space to get the jacobian.</p>
<h2 dir="auto"><a id="user-content-representations" class="anchor" aria-hidden="true" href="#representations"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path d="m7.775 3.275 1.25-1.25a3.5 3.5 0 1 1 4.95 4.95l-2.5 2.5a3.5 3.5 0 0 1-4.95 0 .751.751 0 0 1 .018-1.042.751.751 0 0 1 1.042-.018 1.998 1.998 0 0 0 2.83 0l2.5-2.5a2.002 2.002 0 0 0-2.83-2.83l-1.25 1.25a.751.751 0 0 1-1.042-.018.751.751 0 0 1-.018-1.042Zm-4.69 9.64a1.998 1.998 0 0 0 2.83 0l1.25-1.25a.751.751 0 0 1 1.042.018.751.751 0 0 1 .018 1.042l-1.25 1.25a3.5 3.5 0 1 1-4.95-4.95l2.5-2.5a3.5 3.5 0 0 1 4.95 0 .751.751 0 0 1-.018 1.042.751.751 0 0 1-1.042.018 1.998 1.998 0 0 0-2.83 0l-2.5 2.5a1.998 1.998 0 0 0 0 2.83Z"></path></svg></a>Representations</h2>
<p dir="auto">We start with the notion of a <code>Batch</code>, which is a collection of basis elements.
In the case of the application to AD, a basis element is a <a href="https://juliadiff.org/ChainRulesCore.jl/dev/rule_author/tangents.html" rel="nofollow">tangent type</a>.</p>
<p dir="auto">At every stage of pushing forward there is a new <code>Batch</code> output.</p>
<p dir="auto">This in general answers the <a href="https://github.com/JuliaDiff/ChainRulesCore.jl/issues/66" data-hovercard-type="issue" data-hovercard-url="/JuliaDiff/ChainRulesCore.jl/issues/66/hovercard">question of the what the Jacobian representation for a function taking as input a composite type.</a>
It is a <code>Batch{Tangent{...}}</code>.
In the case of basis elements that are vectors, this <code>Batch{Vector{R}}</code> can be converted into the conventional <code>Matrix{R}</code> representation of a Jacobian.</p>
