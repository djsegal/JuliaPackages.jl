<p>This is a Julia package to use C++ Minuit2 package for function minimization.
The binaries for Minuit2 are built here: <a href="https://github.com/JuliaPackaging/Yggdrasil/tree/master/M/Minuit2">https://github.com/JuliaPackaging/Yggdrasil/tree/master/M/Minuit2</a></p>
<p>The interface code between C++ and Julia can be found here <a href="https://github.com/jstrube/Minuit2_Julia_Wrapper">https://github.com/jstrube/Minuit2_Julia_Wrapper</a> and needs to be installed locally for the time being.
If you cannot build the wrapper binaries using BinaryBuilder, you can download the artifacts from here: <a href="https://github.com/jstrube/Minuit2_Julia_Wrapper_jll.jl">https://github.com/jstrube/Minuit2_Julia_Wrapper_jll.jl</a></p>
<p>The Pkg documentation may have some helpful tips to achieve this:
<a href="https://julialang.github.io/Pkg.jl/v1/artifacts/#Overriding-artifact-locations" rel="nofollow">https://julialang.github.io/Pkg.jl/v1/artifacts/#Overriding-artifact-locations</a></p>
