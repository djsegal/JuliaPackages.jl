<p>The package DutyCycleNumbers provides a special type of "numbers" for
the programming language <a href="https://julialang.org" rel="nofollow">julia</a> that represent
piece-wise constant waveforms, e.g. values that are discontinuous in
time.  This can be rather convenient for modeling electrical equipment
if it is turned on and off with a given dutycycle.  More complex
switching patterns are also possible.</p>
<p>Documentation is avaliable
<a href="https://quantum-factory.de/open-source/DutyCycles.jl" rel="nofollow">online</a>.
Julia version 1.0 (or newer) is required.</p>
