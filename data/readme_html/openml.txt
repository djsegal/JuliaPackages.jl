<p dir="auto">Partial implementation of the <a href="https://www.openml.org" rel="nofollow">OpenML</a> API for
Julia. At present this package allows querying and
downloading of OpenML datasets.</p>
<p dir="auto">For further integration with the
<a href="https://JuliaAI.github.io/MLJ.jl/dev/" rel="nofollow">MLJ</a> machine
learning framework (such as uploading MLJ runs) see
<a href="https://github.com/JuliaAI/MLJOpenML.jl">MLJOpenML.jl</a>.</p>
<p dir="auto"><strong>Acknowledgements.</strong> The code in this repository is based on contributions of Diego Arenas
to <a href="https://github.com/JuliaAI/MLJBase.jl">MLJBase.jl</a> which do not
appear in the commit history of this repository.</p>
<p dir="auto">Package documentation is <a href="https://JuliaAI.github.io/OpenML.jl/stable" rel="nofollow">here</a>.</p>
