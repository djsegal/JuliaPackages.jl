<p dir="auto">This is a thin wrapper around the excellent <a href="https://github.com/achael/eht-imaging">eht-imaging</a> package.
<a href="https://github.com/cjdoris/PythonCall.jl">PythonCall</a> is used to interface with eht-imaging. The main export
is the <code>ehtim</code> object that contains the ehtim module. This means that all the usual <code>eht-imaging</code> commands
should work identically to how they work in Python.</p>
