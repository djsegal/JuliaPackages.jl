<p dir="auto">The <code>MATLABCluster.jl</code> package provides an interface for starting <a href="http://www.julialang.org/" rel="nofollow">Julia</a> worker processes using <a href="http://www.mathworks.com/help/distcomp/index.html" rel="nofollow">MATLAB™ Parallel Computing Toolbox</a> cluster profiles, including profiles configured to connect to a <a href="http://www.mathworks.com/products/distriben/index.html" rel="nofollow">MATLAB™ Distributed Computing Server</a>. You cannot use <code>MATLABCluster.jl</code> without having purchased and installed a copy of MATLAB™ and the MATLAB™ Parallel Computing Toolbox from <a href="http://www.mathworks.com/" rel="nofollow">MathWorks</a>. This package is available free of charge and in no way replaces or alters any functionality of MathWorks's MATLAB product.</p>
<p dir="auto">To connect to a MATLAB cluster:</p>
