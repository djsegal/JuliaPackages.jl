<div class="highlight highlight-source-julia"><pre>Pkg<span class="pl-k">.</span><span class="pl-c1">add</span>(<span class="pl-s"><span class="pl-pds">"</span>PlotThemes<span class="pl-pds">"</span></span>)</pre></div>

<p>Currently the following themes are available:</p>
<ul>
<li><code>:default</code></li>
<li><code>:dark</code></li>
<li><code>:ggplot2</code></li>
<li><code>:juno</code></li>
<li><code>:lime</code></li>
<li><code>:orange</code></li>
<li><code>:sand</code></li>
<li><code>:solarized</code></li>
<li><code>:solarized_light</code></li>
<li><code>:wong</code></li>
<li><code>:wong2</code></li>
<li><code>:gruvbox_dark</code></li>
<li><code>:gruvbox_light</code></li>
<li><code>:bright</code></li>
<li><code>:vibrant</code></li>
<li><code>:mute</code></li>
</ul>
<p>When using Plots, a theme can be set using the <code>theme</code> function:</p>
<div class="highlight highlight-source-julia"><pre><span class="pl-k">using</span> Plots
<span class="pl-c1">theme</span>(thm<span class="pl-k">::</span><span class="pl-c1">Symbol</span>; kwargs<span class="pl-k">...</span>)</pre></div>
<p><code>theme</code> accepts any Plots <a href="http://docs.juliaplots.org/attributes/" rel="nofollow">attribute</a> as keyword argument and sets its value as default for subsequent plots.</p>
<p>Themes can be previewed using <code>Plots.showtheme(thm::Symbol)</code>:</p>

<p>A theme specifies default values for different Plots <a href="http://docs.juliaplots.org/attributes/" rel="nofollow">attributes</a>.
At the moment these are typically colors, palettes and colorgradients, but any Plots attribute can be controlled by a theme in general.
PRs for new themes very welcome! Adding a new theme (e.g. <code>mytheme</code>) is as easy as adding a new file (mytheme.jl) that contains at least the following line:</p>
<div class="highlight highlight-source-julia"><pre>_themes[<span class="pl-c1">:mytheme</span>] <span class="pl-k">=</span> <span class="pl-c1">PlotTheme</span>(; kwargs<span class="pl-k">...</span>)</pre></div>
<p>The keyword arguments can be any collection of Plots attributes plus a colorgradient keyword argument.</p>
