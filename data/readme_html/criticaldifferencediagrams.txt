<p dir="auto">This package generates Tikz code for publication-ready vector graphics.</p>
<p dir="auto">Critical difference (CD) diagrams are a powerful tool to compare outcomes of multiple treatments over multiple observations. For instance, in machine learning research we often compare the performance (= outcome) of multiple methods (= treatments) over multiple data sets (= observations). A wrapper for Python is also available.</p>
<p dir="auto"><strong>Regular CD diagrams:</strong> indistinguishable methods are connected.</p>

<p dir="auto"><strong>2D sequences:</strong> sequences of multiple CD diagrams can be arranged in a single 2-dimensional axis.</p>
