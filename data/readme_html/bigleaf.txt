<p dir="auto"><strong>Bigleaf.jl</strong> is a partial Julia port of Jürgen Knauer's
<a href="https://bitbucket.org/juergenknauer/Bigleaf" rel="nofollow"><strong>bigleaf</strong> R package</a>
for the calculation of physical (e.g. aerodynamic conductance, surface temperature)
and physiological (e.g. canopy conductance, water-use efficiency) ecosystem properties
from eddy covariance data and accompanying meteorological measurements.
All calculations are based on a 'big-leaf' representation of the vegetation and
return representative bulk ecosystem/canopy variables.</p>

<p dir="auto">We are porting functionality of the <a href="https://bitbucket.org/juergenknauer/Bigleaf" rel="nofollow">R package</a> as needed. Please
file an <a href="https://github.com/bgctw/Bigleaf.jl/issues">issue</a> if you need a specific feature.</p>
<p dir="auto">At the current state we ported</p>
<ul dir="auto">
<li>Meteorological variables</li>
<li>Boundary layer and Aerodynamic conductance</li>
<li>Surface conductance</li>
<li>Evapotranspiration</li>
<li>Potential radiation</li>
<li>Unit conversions</li>
<li>Filtering data</li>
</ul>
