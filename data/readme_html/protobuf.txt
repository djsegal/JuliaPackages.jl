<p><a href="https://developers.google.com/protocol-buffers/docs/overview" rel="nofollow"><strong>Protocol buffers</strong></a> are a language-neutral, platform-neutral, extensible way of serializing structured data for use in communications protocols, data storage, and more.</p>
<p><strong>ProtoBuf.jl</strong> is a Julia implementation for protocol buffers.</p>
