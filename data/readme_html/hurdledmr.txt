<p dir="auto">HurdleDMR.jl is a Julia implementation of the Hurdle Distributed Multinomial Regression (HDMR), as described in:</p>
<p dir="auto"><a href="https://doi.org/10.1080/07350015.2021.1947843" rel="nofollow">Bryan Kelly, Asaf Manela &amp; Alan Moreira (2021)</a> Text Selection, Journal of Business &amp; Economic Statistics <a href="https://papers.ssrn.com/sol3/papers.cfm?abstract_id=3491942" rel="nofollow">(ungated preprint)</a>.</p>
<p dir="auto">It includes a Julia implementation of the Distributed Multinomial Regression (DMR) model of <a href="https://arxiv.org/abs/1311.6139" rel="nofollow">Taddy (2015)</a>.</p>
<p dir="auto">Read the <a href="https://asafmanela.github.io/HurdleDMR.jl/latest" rel="nofollow">documentation</a> and <a href="https://asafmanela.github.io/HurdleDMR.jl/latest/tutorials/" rel="nofollow">tutorials</a> to learn more.</p>
