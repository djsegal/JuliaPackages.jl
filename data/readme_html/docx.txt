<p dir="auto">A pure Julia package to work with Docx (Word) documents.</p>
<p dir="auto">This package aims to easily integrate with the packages of the <a href="https://github.com/JuliaText">JuliaText</a> organization.</p>
<ul dir="auto">
<li>This package is heavily inspired by <a href="https://github.com/python-openxml/python-docx">python-docx</a></li>
</ul>
