<p dir="auto">John D. Ramsdell and Moses D. Liskov</p>
<p dir="auto"><a href="https://www.electionguard.vote/" rel="nofollow">ElectionGuard</a> is a software system
designed to make voting more secure, transparent and accessible.
ElectionGuard uses cryptography to ensure that</p>
<ul dir="auto">
<li>
<p dir="auto">voters can verify that their own selections have been correctly
recorded, and</p>
</li>
<li>
<p dir="auto">anyone can verify that the recorded votes have been correctly
tallied.</p>
</li>
</ul>
<p dir="auto">The MITRE ElectionGuard Verifier provides the means to validate
ElectionGuard election records in an easy to use package.  It is
written in the <a href="https://julialang.org/" rel="nofollow">Julia</a> programming language.
Follow the instructions at the Julia web site to download and install
the system on your computer.  The user instructions are at
<a href="https://mitre.github.io/ElectionGuardVerifier.jl" rel="nofollow">ElectionGuardVerifier.jl</a>.</p>

<a href="https://github.com/JuliaIO/JSON.jl">JSON</a> package.  From the Julia
REPL, type <code>]</code> to enter the Pkg REPL mode and run</p>
<div class="snippet-clipboard-content notranslate position-relative overflow-auto" data-snippet-clipboard-copy-content="pkg&gt; add JSON.jl"><pre class="notranslate"><code>pkg&gt; add JSON.jl
</code></pre></div>
<p dir="auto">Type the delete key or cntl-C to exit the Pkg REPL mode.</p>

<p dir="auto">To develop code, a useful pattern is to create the <code>er</code> directory in
the directory containing this <code>README</code>, and place sample data within
it.</p>
<ol dir="auto">
<li>
<p dir="auto">Start Julia with the command</p>
<div class="highlight highlight-source-shell notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="$ julia --project=."><pre>$ julia --project=.</pre></div>
<p dir="auto">Unix OS users should look at the <code>ju</code> script.</p>
</li>
<li>
<p dir="auto">Load the software with</p>
<div class="highlight highlight-source-julia notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="julia&gt; using ElectionGuardVerifier"><pre>julia<span class="pl-k">&gt;</span> <span class="pl-k">using</span> ElectionGuardVerifier</pre></div>
</li>
<li>
<p dir="auto">Load your election records with</p>
<div class="highlight highlight-source-julia notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="julia&gt; er = load(&quot;er&quot;);"><pre>julia<span class="pl-k">&gt;</span> er <span class="pl-k">=</span> <span class="pl-c1">load</span>(<span class="pl-s"><span class="pl-pds">"</span>er<span class="pl-pds">"</span></span>);</pre></div>
</li>
<li>
<p dir="auto">Check your election records with</p>
<div class="highlight highlight-source-julia notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="julia&gt; check(er)"><pre>julia<span class="pl-k">&gt;</span> <span class="pl-c1">check</span>(er)</pre></div>
<p dir="auto">The final line of output is <code>true</code> if your election records pass
all tests, otherwise it is <code>false</code>.</p>
</li>
<li>
<p dir="auto">Exit Julia with <code>exit()</code> or type cntl-D.</p>
</li>
</ol>
