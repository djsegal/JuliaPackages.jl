<p>A <a href="http://julialang.org" rel="nofollow">Julia</a> client for the <a href="http://neo4j.org" rel="nofollow">Neo4j</a> graph
database.</p>
<p>Really easy to use, have a look at <code>test/runtests.jl</code> for the available methods.</p>

<div class="highlight highlight-source-julia"><pre>c <span class="pl-k">=</span> <span class="pl-c1">Connection</span>(<span class="pl-s"><span class="pl-pds">"</span>localhost<span class="pl-pds">"</span></span>; user<span class="pl-k">=</span><span class="pl-s"><span class="pl-pds">"</span>neo4j<span class="pl-pds">"</span></span>, password<span class="pl-k">=</span><span class="pl-s"><span class="pl-pds">"</span>neo4j<span class="pl-pds">"</span></span>)
tx <span class="pl-k">=</span> <span class="pl-c1">transaction</span>(c)
<span class="pl-c1">tx</span>(<span class="pl-s"><span class="pl-pds">"</span>CREATE (n:Lang) SET n.name = <span class="pl-cce">\$</span>name<span class="pl-pds">"</span></span>, <span class="pl-s"><span class="pl-pds">"</span>name<span class="pl-pds">"</span></span> <span class="pl-k">=&gt;</span> <span class="pl-s"><span class="pl-pds">"</span>Julia<span class="pl-pds">"</span></span>)
<span class="pl-c1">tx</span>(<span class="pl-s"><span class="pl-pds">"</span>MATCH (n:Lang) RETURN n LIMIT {limit}<span class="pl-pds">"</span></span>, <span class="pl-s"><span class="pl-pds">"</span>limit<span class="pl-pds">"</span></span> <span class="pl-k">=&gt;</span> <span class="pl-c1">10</span>)
results <span class="pl-k">=</span> <span class="pl-c1">commit</span>(tx)</pre></div>
<p>You can also submit a transaction to the server without committing it. This
will return a result set but will keep the transaction open both on the client
and server:</p>
<div class="highlight highlight-source-julia"><pre>results <span class="pl-k">=</span> <span class="pl-c1">tx</span>(<span class="pl-s"><span class="pl-pds">"</span>MATCH (n) RETURN n<span class="pl-pds">"</span></span>; submit<span class="pl-k">=</span><span class="pl-c1">true</span>)</pre></div>
<p>Rollbacks are also supported:</p>
<div class="highlight highlight-source-julia"><pre><span class="pl-c1">rollback</span>(tx)</pre></div>
<p>If the goal is to simply run a MATCH query and get the result in the form of a
<code>DataFrames.DataFrame</code> object, the <code>cypherQuery</code> function can be used.
The <code>cypherQuery</code> implementation performs the query in a single transaction which
automatically opens and closes the transaction:</p>
<div class="highlight highlight-source-julia"><pre>results <span class="pl-k">=</span> <span class="pl-c1">cypherQuery</span>(c, <span class="pl-s"><span class="pl-pds">"</span>MATCH (n) RETURN n.property AS Property<span class="pl-pds">"</span></span>)</pre></div>
