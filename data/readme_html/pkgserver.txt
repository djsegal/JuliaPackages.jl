<p>Reference implementation of a Julia Pkg server, providing advanced package serving and caching capabilities.</p>
<h2><a id="user-content-usage" class="anchor" aria-hidden="true" href="#usage"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path fill-rule="evenodd" d="M4 9h1v1H4c-1.5 0-3-1.69-3-3.5S2.55 3 4 3h4c1.45 0 3 1.69 3 3.5 0 1.41-.91 2.72-2 3.25V8.59c.58-.45 1-1.27 1-2.09C10 5.22 8.98 4 8 4H4c-.98 0-2 1.22-2 2.5S3 9 4 9zm9-3h-1v1h1c1 0 2 1.22 2 2.5S13.98 12 13 12H9c-.98 0-2-1.22-2-2.5 0-.83.42-1.64 1-2.09V6.25c-1.09.53-2 1.84-2 3.25C6 11.31 7.55 13 9 13h4c1.45 0 3-1.69 3-3.5S14.5 6 13 6z"></path></svg></a>Usage</h2>
<p>See <a href="bin/run_server.jl"><code>bin/run_server.jl</code></a> for an example on how to run a PkgServer locally.  For a quick, localhost-only setup for development purposes:</p>
<div class="highlight highlight-source-shell"><pre><span class="pl-c"><span class="pl-c">#</span> Enter package directory</span>
$ <span class="pl-c1">cd</span> PkgServer.jl
<span class="pl-c"><span class="pl-c">#</span> Launch server to run in foreground</span>
$ julia --project bin/run_server.jl</pre></div>
<p>Then, start a 1.4 or newer version of Julia, referring all Pkg operations to this package server::</p>
<pre><code>$ JULIA_PKG_SERVER=http://localhost:8000 julia
</code></pre>
