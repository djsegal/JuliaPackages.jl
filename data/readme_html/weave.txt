<p>Weave is a scientific report generator/literate programming tool for the <a href="https://julialang.org/" rel="nofollow">Julia programming language</a>.
It resembles
<a href="http://mpastell.com/pweave" rel="nofollow">Pweave</a>,
<a href="https://yihui.org/knitr/" rel="nofollow">knitr</a>,
<a href="https://rmarkdown.rstudio.com/" rel="nofollow">R Markdown</a>,
and <a href="https://stat.ethz.ch/R-manual/R-patched/library/utils/doc/Sweave.pdf" rel="nofollow">Sweave</a>.</p>
<p>You can write your documentation and code in input document using Markdown, Noweb or ordinal Julia script syntax,
and then use <code>weave</code> function to execute code and generate an output document while capturing results and figures.</p>
<p><strong>Current features</strong></p>
<ul>
<li>Publish markdown directly to HTML and PDF using Julia or <a href="https://pandoc.org/MANUAL.html" rel="nofollow">Pandoc</a></li>
<li>Execute code as in terminal or in a unit of code chunk</li>
<li>Capture <a href="https://github.com/JuliaPlots/Plots.jl">Plots.jl</a> or <a href="https://github.com/GiovineItalia/Gadfly.jl">Gadfly.jl</a> figures</li>
<li>Supports various input format: Markdown, <a href="https://www.cs.tufts.edu/~nr/noweb/" rel="nofollow">Noweb</a>, <a href="https://jupyter.org/" rel="nofollow">Jupyter Notebook</a>, and ordinal Julia script</li>
<li>Conversions between those input formats</li>
<li>Supports various output document formats: HTML, PDF, GitHub markdown, Jupyter Notebook, MultiMarkdown, Asciidoc and reStructuredText</li>
<li>Simple caching of results</li>
</ul>
<p><strong>Citing Weave:</strong> <em>Pastell, Matti. 2017. Weave.jl: Scientific Reports Using Julia. The Journal of Open Source Software. <a href="http://dx.doi.org/10.21105/joss.00204" rel="nofollow">http://dx.doi.org/10.21105/joss.00204</a></em></p>

<p>Install <a href="https://atom.io/packages/language-weave" rel="nofollow">language-weave</a> to add Weave support to Juno.
It allows running code from Weave documents with usual keybindings and allows preview of
html and pdf output.</p>
<p>The <a href="https://www.julia-vscode.org/" rel="nofollow">Julia extension for Visual Studio Code</a>
adds Weave support to <a href="https://code.visualstudio.com/" rel="nofollow">Visual Studio Code</a>.</p>
