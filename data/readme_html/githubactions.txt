<p>Utilities for working within GitHub Actions, modelled after <a href="https://github.com/actions/toolkit/tree/master/packages/core"><code>actions/core</code></a>.</p>
<p>Perhaps the most common use case is to set the global logger to one compatible with GitHub Actions' log format:</p>
