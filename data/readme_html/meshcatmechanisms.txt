<p>MeshCatMechanisms.jl adds support for visualizing mechanisms and robots from <a href="https://github.com/tkoolen/RigidBodyDynamics.jl">RigidBodyDynamics.jl</a> with <a href="https://github.com/rdeits/MeshCat.jl">MeshCat.jl</a>. All geometries are constructed using <a href="https://github.com/rdeits/MechanismGeometries.jl">MechanismGeometries.jl</a>.</p>
<p>Features:</p>
<ul>
<li>Parsing geometry directly from URDF files</li>
<li>Animation of robot trajectories from RigidBodyDynamics.jl simulations</li>
<li>Live rendering of simulation progress using the <code>OdeIntegrators.OdeResultsSink</code> interface</li>
<li>Interactive manipulation of the mechanism configuration using <a href="https://github.com/piever/InteractBase.jl">InteractBase.jl</a></li>
</ul>

<p>MeshCatMechanisms.jl provides similar functionality to <a href="https://github.com/rdeits/RigidBodyTreeInspector.jl">RigidBodyTreeInspector.jl</a>, but is built on top of the lighter-weight MeshCat viewer instead of <a href="https://github.com/rdeits/DrakeVisualizer.jl">DrakeVisualizer.jl</a>.</p>
