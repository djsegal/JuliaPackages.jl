<p dir="auto"><a href="https://github.com/JuliaDiff/TaylorDiff.jl">TaylorDiff.jl</a> is an automatic differentiation (AD) package for efficient and composable higher-order derivatives, implemented with operator-overloading on Taylor polynomials.</p>
<p dir="auto">Disclaimer: this project is still in early alpha stage, and APIs can change any time in the future. Discussions and potential use cases are extremely welcome!</p>

<p dir="auto">TaylorDiff.jl is designed with the following goals in head:</p>
<ul dir="auto">
<li>Linear scaling with the order of differentiation (while naively composing first-order differentiation would result in exponential scaling)</li>
<li>Same performance with <a href="https://github.com/JuliaDiff/ForwardDiff.jl">ForwardDiff.jl</a> on first order and second order, so there is no penalty in drop-in replacement</li>
<li>Capable for calculating exact derivatives in physical models with ODEs and PDEs</li>
<li>Composable with other AD systems like <a href="https://github.com/FluxML/Zygote.jl">Zygote.jl</a>, so that the above models evaluated with TaylorDiff can be further optimized with gradient-based optimization techniques</li>
</ul>
<p dir="auto">TaylorDiff.jl is fast! See our dedicated <a href="https://benchmark.tansongchen.com/TaylorDiff.jl" rel="nofollow">benchmarks</a> page for comparison with other packages in various tasks.</p>

<ul dir="auto">
<li><a href="https://github.com/JuliaDiff/TaylorSeries.jl">TaylorSeries.jl</a>: a systematic treatment of Taylor polynomials in one and several variables, but its mutating and scalar code isn't great for speed and composability with other packages</li>
<li><a href="https://github.com/JuliaDiff/ForwardDiff.jl">ForwardDiff.jl</a>: well-established and robust operator-overloading based forward-mode AD, where higher-order derivatives can be achieved by nesting first-order derivatives</li>
<li><a href="https://github.com/JuliaDiff/Diffractor.jl">Diffractor.jl</a>: next-generation source-code transformation based forward-mode and reverse-mode AD, designed with support for higher-order derivatives in mind; but the higher-order functionality is currently only a proof-of-concept</li>
<li><a href="https://jax.readthedocs.io/en/latest/jax.experimental.jet.html" rel="nofollow"><code>jax.jet</code></a>: an experimental (and unmaintained) implementation of Taylor-mode automatic differentiation in JAX, sharing the same underlying algorithm with this project</li>
</ul>
