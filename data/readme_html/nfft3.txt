<p dir="auto"><code>NFFT3.jl</code> provides the following fast algorithms:</p>
<ul dir="auto">
<li>nonequispaced fast Fourier transform (NFFT)</li>
<li>nonequispaced fast cosine transform (NFCT)</li>
<li>nonequispaced fast sine transform (NFST)</li>
<li>fast summation (fastsum)</li>
</ul>
