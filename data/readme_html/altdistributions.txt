<p>This is a collection of some probability distributions I find useful, primarily for Bayesian estimation. Eventually, they should be considered for contributing to <a href="https://github.com/JuliaStats/Distributions.jl">Distributions.jl</a>, but currently they reside in this package because</p>
<ol>
<li>I am experimenting with the interface,</li>
<li>I am experimenting with the implementation (making it friendly to automatic differentiation),</li>
<li>not all functionality is implemented (eg only the <code>logpdf</code>).</li>
</ol>
<p>When the name of distributions coincides with one in <code>Distributions</code>, it is prefixed with <code>Alt</code>, eg <code>AltMvNormal</code>.</p>
