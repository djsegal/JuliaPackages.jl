<p dir="auto">The GenomicFeatures package provides utilities for working with interval based genomic annotations.
It builds on <a href="https://github.com/biojulia/intervaltrees.jl">IntervalTrees</a> to provide a data-structures and algorithms for various formats such as <a href="https://github.com/biojulia/bed.jl">BED</a>, <a href="https://github.com/biojulia/gff3.jl">GFF3</a>, <a href="https://github.com/biojulia/bigwig.jl">bigWig</a> and <a href="https://github.com/biojulia/bigbed.jl">bigBed</a>.</p>

<p dir="auto">You can install the GenomicFeatures package from the <a href="https://docs.julialang.org/en/v1/manual/getting-started/" rel="nofollow">Julia REPL</a>.
Press <code>]</code> to enter <a href="https://docs.julialang.org/en/v1/stdlib/Pkg/" rel="nofollow">pkg mode</a>, then enter the following command:</p>
<div class="highlight highlight-source-julia notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="add GenomicFeatures"><pre>add GenomicFeatures</pre></div>
<p dir="auto">If you are interested in the cutting edge of the development, please check out the <a href="https://github.com/BioJulia/GenomicFeatures.jl/tree/develop">develop branch</a> to try new features before release.</p>
