<p>ChordalGraph is an extension of the Julia package LightGraphs to handle chordal graphs.</p>
<p>To use ChordalGraph in Julia, run</p>
<div class="highlight highlight-source-julia position-relative" data-snippet-clipboard-copy-content="pkg&gt; add https://github.com/wangjie212/ChordalGraph
"><pre>pkg<span class="pl-k">&gt;</span> add https<span class="pl-k">:</span><span class="pl-k">//</span>github<span class="pl-k">.</span>com<span class="pl-k">/</span>wangjie212<span class="pl-k">/</span>ChordalGraph</pre></div>

<h2><a id="user-content-usage" class="anchor" aria-hidden="true" href="#usage"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path fill-rule="evenodd" d="M7.775 3.275a.75.75 0 001.06 1.06l1.25-1.25a2 2 0 112.83 2.83l-2.5 2.5a2 2 0 01-2.83 0 .75.75 0 00-1.06 1.06 3.5 3.5 0 004.95 0l2.5-2.5a3.5 3.5 0 00-4.95-4.95l-1.25 1.25zm-4.69 9.64a2 2 0 010-2.83l2.5-2.5a2 2 0 012.83 0 .75.75 0 001.06-1.06 3.5 3.5 0 00-4.95 0l-2.5 2.5a3.5 3.5 0 004.95 4.95l1.25-1.25a.75.75 0 00-1.06-1.06l-1.25 1.25a2 2 0 01-2.83 0z"></path></svg></a>Usage</h2>
<p>The function CheckChordal checks whether a graph G is chordal. If G is chordal, then flag=true and a perfect elimination ordering <em>order</em> is returned; otherwise, flag=false.</p>
<div class="highlight highlight-source-julia position-relative" data-snippet-clipboard-copy-content="julia&gt; flag,order=CheckChordal(G)
"><pre>julia<span class="pl-k">&gt;</span> flag,order<span class="pl-k">=</span><span class="pl-c1">CheckChordal</span>(G)</pre></div>
<p>The function chordal_cliques! generates a chordal extension of G and enumerates all maximal cliques of the obtained graph.</p>
<div class="highlight highlight-source-julia position-relative" data-snippet-clipboard-copy-content="julia&gt; cliques,num_cliques,size_cliques=chordal_cliques!(G, method=&quot;MF&quot;, minimize=true)
"><pre>julia<span class="pl-k">&gt;</span> cliques,num_cliques,size_cliques<span class="pl-k">=</span><span class="pl-c1">chordal_cliques!</span>(G, method<span class="pl-k">=</span><span class="pl-s"><span class="pl-pds">"</span>MF<span class="pl-pds">"</span></span>, minimize<span class="pl-k">=</span><span class="pl-c1">true</span>)</pre></div>
<p>Options:<br>
method: MF (a chordal extension based on greedy minimal fill-in), MD (a chordal extension based on greedy minimal degree)<br>
minimize: true (minimizing the obtained chordal extension), false (not minimizing the obtained chordal extension)</p>
