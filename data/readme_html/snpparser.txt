<p dir="auto">Version 1.0 is planned to be supported in a future release. However, there is an easy work-around by following the necessary steps:</p>
<ul dir="auto">
<li>At the top of your Touchstone file add <code>[Version] 2.0</code></li>
<li>Right below the <code>#options</code> line add <code>[Number of Ports] N</code> where N is the number of ports. You can easily determine the number of ports from your network data: each row of your network data matrix should have N^2 + 1 columns</li>
<li>Above your network data add the <code>[Network Data]</code> tag.</li>
</ul>
