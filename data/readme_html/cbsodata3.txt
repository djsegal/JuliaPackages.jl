<p>This is a simple <a href="https://github.com/JuliaData/Tables.jl">Tables</a>-interface to use the <a href="https://opendata.cbs.nl/statline/portal.html?_la=nl&amp;_catalog=CBS" rel="nofollow">CBS</a> odata3 portal to download all CBS-datasets.</p>
<h2><a id="user-content-how-to-use-it" class="anchor" aria-hidden="true" href="#how-to-use-it"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path fill-rule="evenodd" d="M7.775 3.275a.75.75 0 001.06 1.06l1.25-1.25a2 2 0 112.83 2.83l-2.5 2.5a2 2 0 01-2.83 0 .75.75 0 00-1.06 1.06 3.5 3.5 0 004.95 0l2.5-2.5a3.5 3.5 0 00-4.95-4.95l-1.25 1.25zm-4.69 9.64a2 2 0 010-2.83l2.5-2.5a2 2 0 012.83 0 .75.75 0 001.06-1.06 3.5 3.5 0 00-4.95 0l-2.5 2.5a3.5 3.5 0 004.95 4.95l1.25-1.25a.75.75 0 00-1.06-1.06l-1.25 1.25a2 2 0 01-2.83 0z"></path></svg></a>How to use it</h2>
<p>If you know the exact dataset, you can directly use it:</p>
<div class="highlight highlight-source-julia position-relative" data-snippet-clipboard-copy-content="using DataFrames, CBSOData3

df = DataFrame(CBSOData3.ODataTable(&quot;82811NED&quot;))
"><pre><span class="pl-k">using</span> DataFrames, CBSOData3

df <span class="pl-k">=</span> <span class="pl-c1">DataFrame</span>(CBSOData3<span class="pl-k">.</span><span class="pl-c1">ODataTable</span>(<span class="pl-s"><span class="pl-pds">"</span>82811NED<span class="pl-pds">"</span></span>))</pre></div>
<p>You can use the keyword argument <code>columns</code> to give a list of column names to select (case sensitive). And you can use <code>filter</code> to give a filter expression for the rows (as a String) using the <a href="https://www.odata.org/documentation/odata-version-3-0/" rel="nofollow">Odata3</a> rules.</p>
<p>For a given table you can get information about the columns in the dataset and about the used classifications by using <code>get_meta</code>:</p>
<div class="highlight highlight-source-julia position-relative" data-snippet-clipboard-copy-content="df_meta = CBSOData3.get_meta(&quot;82811NED&quot;)
"><pre>df_meta <span class="pl-k">=</span> CBSOData3<span class="pl-k">.</span><span class="pl-c1">get_meta</span>(<span class="pl-s"><span class="pl-pds">"</span>82811NED<span class="pl-pds">"</span></span>)</pre></div>
<p>This gives a dict with DataProperties and the classifications.</p>
<p>Using <code>get_tables</code> you can get a list of dicts of all available tables. The <code>Identifier</code> can then be used to get the actual data.</p>
