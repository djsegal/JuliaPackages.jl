<ul dir="auto">
<li>TextTiling is a method for finding segment boundaries based on lexical cohesion and similarity between adjacent blocks.</li>
<li>window_size is the size of the block and smooth_window_size is set to smooth the depth score.</li>
<li>num_topics is the number of topics in the document. If this value is specified, segment boundaries are determined by the number of num_topics, starting with the highest depth score.</li>
<li>Utils.tokenize is a tokenizer for word segmentation of sentences; other languages can be supported by changing the tokenizer.</li>
</ul>

<ul dir="auto">
<li>TopicTiling is an extension of TextTiling that uses the topic IDs of words in a sentence to calculate the similarity between blocks.</li>
<li>lda_model uses gensim, a Python library.</li>
<li>TopicTiling requires a training document to build the lda_model in addition to the documents to be segmented. (The training document should be close in domain to the document to be segmented.)</li>
<li>See <a href="https://radimrehurek.com/gensim/models/ldamodel.html" rel="nofollow">models.ldamodel</a> for library details and parameter settings.</li>
<li>window_size is the size of the block and smooth_window_size is set to smooth the depth score.</li>
<li>num_topics is the number of topics in the document. If this value is specified, segment boundaries are determined by the number of num_topics, starting with the highest depth score.</li>
</ul>
