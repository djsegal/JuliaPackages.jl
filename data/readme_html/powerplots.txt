<p><code>PowerPlots.jl</code> is a registered julia package and can be added with the following command.</p>
<div class="highlight highlight-source-julia position-relative" data-snippet-clipboard-copy-content="Pkg&gt; add PowerPlots
"><pre>Pkg<span class="pl-k">&gt;</span> add PowerPlots</pre></div>
<p>To add the lastest development version of <code>PowerPlots.jl</code> use the command:</p>
<div class="highlight highlight-source-julia position-relative" data-snippet-clipboard-copy-content="Pkg&gt; add https://github.com/WISPO-POP/PowerPlots.jl.git
"><pre>Pkg<span class="pl-k">&gt;</span> add https<span class="pl-k">:</span><span class="pl-k">//</span>github<span class="pl-k">.</span>com<span class="pl-k">/</span>WISPO<span class="pl-k">-</span>POP<span class="pl-k">/</span>PowerPlots<span class="pl-k">.</span>jl<span class="pl-k">.</span>git</pre></div>
