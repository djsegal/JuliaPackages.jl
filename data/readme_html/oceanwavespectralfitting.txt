<p dir="auto">A Julia package for fitting parametric spectral wave models to recorded time series using Whittle and debiased Whittle likelihood inference. This package extends the functionality of <a href="https://github.com/JakeGrainger/WhittleLikelihoodInference.jl"><em>WhittleLikelihoodInference.jl</em></a> to include models for ocean waves.</p>
<p dir="auto">Includes the following models:</p>
<ul dir="auto">
<li><strong>JONSWAP</strong>: the JONSWAP spectra for vertical displacement,</li>
<li><strong>JS_BWG_HNE</strong>: a model for the heave, northwards and eastwards displacement of a particle on the water surface when the waves have a JONSWAP marginal spectral density function and bimodal wrapped Gaussian spreading function.</li>
<li><strong>JS_BWG_HNE_DL</strong>: a depth limited version of <strong>JS_BWG_HNE</strong>.</li>
<li><strong>JS_WG_HNE_DL</strong>: similar to <strong>JS_BWG_HNE_DL</strong> but with a single wrapped Gaussian spreading function (designed for swell modelling).</li>
</ul>
