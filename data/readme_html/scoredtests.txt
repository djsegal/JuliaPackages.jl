<p dir="auto">ScoredTests.jl is a tiny package providing</p>
<ul dir="auto">
<li><code>@scoredtest expr [award] [penalty] [name]</code> macro;</li>
<li>and <code>ScoredTestSet([description])</code>.</li>
</ul>
<p dir="auto">The macro evaluates an expression giving result of the test: was it passed, failed or errored during testing.</p>
<p dir="auto">When a test is passed, it gives <code>award</code>, when failed or errored it takes <code>penalty</code>.</p>
<p dir="auto">Tests (and testsets) can be grouped into <code>ScoredTestSet</code>-s via <code>*=</code> orepand.</p>

<p dir="auto">Originally, the package is supposed to be used by teachers and student in the following manner</p>
<p dir="auto">Teacher perspective</p>
<ul dir="auto">
<li>Create a repo (or template) of package, containing task and scored testset.</li>
</ul>
<p dir="auto">Student perspective</p>
<ul dir="auto">
<li>Clone the repo, complete task, check score via <code>julia --project=Project.toml test/runtests.jl</code> or from REPL <code>pkg&gt;</code> mode.</li>
</ul>
