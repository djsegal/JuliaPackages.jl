<p dir="auto"><code>Lowess</code> exports two functions, <code>lowess</code> and <code>lowess_model</code>. The <code>lowess</code> function returns the predict y-values for the input x-values. The <code>lowess_model</code> function returns a function that can be used to predict the y value for any given x value (which lies within the extrema of input x-values).<br>
The amount of smoothing is mainly controlled by the <code>f</code> keyword argument. E.g.:</p>
<div class="highlight highlight-source-julia notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="using Lowess, Plots

vs = model(us)

scatter(xs, ys)
plot!(us, vs, legend=false)"><pre><span class="pl-k">using</span> Lowess, Plots
