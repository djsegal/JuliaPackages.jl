<p dir="auto">DiscereteEventsLite is a lightweight package for running discrete event simulations. In DiscreteEventsLite, events are registered to a priority queue and executed in temporal sequence.
Some inspiration for this package was taken from <a href="https://github.com/pbayer/DiscreteEvents.jl">DiscreteEvents</a>, which is a well-designed, comprehensive framework for discrete event simulation.</p>
<p dir="auto">I created this package because I wanted a package that is simple and hackable. In addition, I needed a package with the ability to remove events based on id or other criteria, optionally print event traces, and optionally log events. The package you should use will depend on your goals. If you want a package that is lightweight and easily hackable, this package might be suitable for you. Otherwise, you might prefer a full-featured framework, such as <a href="https://github.com/pbayer/DiscreteEvents.jl">DiscreteEvents</a>. You might also checkout <a href="https://simjuliajl.readthedocs.io/en/stable/welcome.html" rel="nofollow">SimJulia</a>, or <a href="https://github.com/bkamins/EventSimulation.jl">EventStimulation</a> for other approaches.</p>
