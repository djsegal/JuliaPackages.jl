<p dir="auto">A library of of callbacks useful for monitoring/measuring simulations.</p>
<p dir="auto">This package is most useful in combination with <a href="http://github.com/zenna/Lens.jl">Lens</a>.</p>
<p dir="auto">The following example will update a UnicodePlot at every time step.</p>
<div class="highlight highlight-source-julia notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="using Callbacks, Lens

struct Loop end
function simulation()
  x = 0.0
  while true
    y = sin(x)
    lens(Loop, (x = x, y = y))
    x += rand()
  end
end

@leval Loop =&gt; plotscalar() simlulation()"><pre><span class="pl-k">using</span> Callbacks, Lens

    y <span class="pl-k">=</span> <span class="pl-c1">sin</span>(x)

<p dir="auto">This may be a bit more frequent than what I need.
Rather than update every time step, we can update every 1000000.</p>
