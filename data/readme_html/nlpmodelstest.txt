<p>During the development of your model, you might find the need for more robust tests.
This package provides problems and functions to that end.</p>
<p>The main usage of this package are the consistency checks, which runs a comparison of two or models on all API functions.</p>
<p>Check the <a href="https://JuliaSmoothOptimizers.github.io/NLPModelsTest.jl/dev" rel="nofollow">docs</a> for the complete usage.</p>
