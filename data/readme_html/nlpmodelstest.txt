<p dir="auto">During the development of your model, you might find the need for more robust tests.
This package provides problems and functions to that end.</p>
<p dir="auto">The main usage of this package are the consistency checks, which runs a comparison of two or models on all API functions.</p>
<p dir="auto">Check the <a href="https://JuliaSmoothOptimizers.github.io/NLPModelsTest.jl/dev" rel="nofollow">docs</a> for the complete usage.</p>
