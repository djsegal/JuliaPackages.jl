<p>Compiled versions for windows and linux are attached to the release.
ROMEO is a command line application.</p>
<p>Example usage on linux:</p>
<p><code>$ romeo ph.nii -m mag.ii -k nomask -o outputdir</code></p>
<p>On windows (cmd or powershell):</p>
<p><code>&gt;romeo.exe ph.nii -m mag.ii -k nomask -o outputdir</code></p>
