<p>This repository contains the ROMEO algorithm for 3D unwrapping.</p>
<p>For the specialized version for MR data in the nifti format (also 4D unwrapping), see <a href="https://github.com/korbinian90/MriResearchTools.jl">MriResearchTools</a>.</p>
