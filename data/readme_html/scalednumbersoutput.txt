<p dir="auto">This package provides a utility function for pretty printing
numbers.  The <code>to_SI(val; sigdigits)</code> takes a number and converts
it to the a string using SI scaling prefixes.  For example:</p>
<div class="highlight highlight-source-julia notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="julia&gt; to_SI(75.2e10)
&quot;752G&quot;

julia&gt; to_SI(0.000000021)
&quot;21n&quot;"><pre>julia<span class="pl-k">&gt;</span> <span class="pl-c1">to_SI</span>(<span class="pl-c1">75.2e10</span>)
<span class="pl-s"><span class="pl-pds">"</span>752G<span class="pl-pds">"</span></span>

julia<span class="pl-k">&gt;</span> <span class="pl-c1">to_SI</span>(<span class="pl-c1">0.000000021</span>)
<span class="pl-s"><span class="pl-pds">"</span>21n<span class="pl-pds">"</span></span></pre></div>
<p dir="auto">By default numbers are rounded to 5 digits but that can be chosen:</p>
<div class="highlight highlight-source-julia notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="julia&gt; to_SI(1/9)
&quot;0.11111&quot;

julia&gt; to_SI(1/9, sigdigits=7)
&quot;0.1111111&quot;"><pre>julia<span class="pl-k">&gt;</span> <span class="pl-c1">to_SI</span>(<span class="pl-c1">1</span><span class="pl-k">/</span><span class="pl-c1">9</span>)
<span class="pl-s"><span class="pl-pds">"</span>0.11111<span class="pl-pds">"</span></span>

julia<span class="pl-k">&gt;</span> <span class="pl-c1">to_SI</span>(<span class="pl-c1">1</span><span class="pl-k">/</span><span class="pl-c1">9</span>, sigdigits<span class="pl-k">=</span><span class="pl-c1">7</span>)
<span class="pl-s"><span class="pl-pds">"</span>0.1111111<span class="pl-pds">"</span></span></pre></div>
