<p dir="auto"><code>Underscores</code> provides a macro <code>@_</code> for passing closures to functions by
interpreting <code>_</code> <em>placeholders</em> as anonymous function arguments. For example
<code>@_ map(_+1, xs)</code> means <code>map(x-&gt;x+1, xs)</code>.</p>
<p dir="auto"><code>Underscores</code> is useful for writing anonymous functions succinctly and without
naming the arguments. This is particular useful for data processing pipelines
such as</p>
<div class="highlight highlight-source-julia notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="@_ people |&gt; filter(_.age &gt; 40, __) |&gt; map(_.name, __)"><pre><span class="pl-c1">@_</span> people <span class="pl-k">|&gt;</span> <span class="pl-c1">filter</span>(_<span class="pl-k">.</span>age <span class="pl-k">&gt;</span> <span class="pl-c1">40</span>, __) <span class="pl-k">|&gt;</span> <span class="pl-c1">map</span>(_<span class="pl-k">.</span>name, __)</pre></div>
<p dir="auto">Read the <a href="https://c42f.github.io/Underscores.jl/stable" rel="nofollow">documentation</a> for
more information, or see the online help for the <code>@_</code> macro.</p>
