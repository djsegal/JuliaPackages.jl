<p dir="auto">This is a Julia implementation of a robust data-preparation and -analysis package using data from eye tracking experiments.  This package is designed modelling an R package called <a href="https://github.com/jwdink/eyetrackingR"><code>eyetrackingR</code></a>.  This package is not to be confused with the "EyeTracking.jl" package (at time of writing; October, 2020), which <a href="https://github.com/dandandai/EyeTracking.jl/">seems to be a GUI-style experimental interface</a> (at first glance, perhaps similar to <a href="https://github.com/esdalmaijer/PyGaze">PyGaze</a>).</p>

<p dir="auto">This package has two main steps in the workflow, whose second step has three paths.  This workflow is drawn from <a href="http://www.eyetracking-r.com/workflow" rel="nofollow"><code>eyetrackingR</code></a>.</p>
<ol dir="auto">
<li><strong>Data cleaning</strong> — obtain data and information about data from various eye-tracking sources (this step should be extremely robust), and puts data into a standardised format for analyses.</li>
<li><strong>Analyses</strong> —:</li>
</ol>
<p dir="auto">		 a) <em>Overall Looking</em>;</p>
<p dir="auto">		 b) <em>Onset-Contingent</em>;</p>
<p dir="auto">		 c) <em>Time-Course of Looking</em>.</p>
