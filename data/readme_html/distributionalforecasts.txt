<p><a href="http://pkg.julialang.org/detail/GitHub" rel="nofollow"><img src="https://camo.githubusercontent.com/a2d93c527f864ab08e162b20760b3b884f95be4d/687474703a2f2f706b672e6a756c69616c616e672e6f72672f6261646765732f4769744875625f302e362e737667" alt="GitHub" data-canonical-src="http://pkg.julialang.org/badges/GitHub_0.6.svg" style="max-width:100%;"></a></p>
<p>The code has been developed in Julia 0.6.4. version, as a code accompanying the Anatolyev and Barunik (2019) paper, and provides an estimation and inference for a model forecasting conditional probability distributions of asset returns (henceforth AB model). For further details, see</p>
<p>Anatolyev, S. and Baruník, J., (2019): <em>Forecasting dynamic return distributions based on ordered binary choice</em>, International Journal of Forecasting, 35(3), pp.823-835. <a href="https://doi.org/10.1016/j.ijforecast.2019.01.005" rel="nofollow">DOI</a>, manuscript <a href="https://ideas.repec.org/p/arx/papers/1711.05681.html" rel="nofollow">available here for download</a> (Jan 2019)</p>
