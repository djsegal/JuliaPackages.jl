<p dir="auto">For the image that does not know the noise variance, we can use <a href="https://github.com/JuliaDSP/Wavelets.jl/">Wavelets.jl</a> to make noise estimation.</p>

noise_img_y = channelview(img_y)[1,:,:]	# Extract y channel.
# normalization
noise_img_y .-= 16.
noise_img_y ./= (235. - 16.)
