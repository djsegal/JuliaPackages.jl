<p dir="auto">This library allows one to create Tikz pictures and save in various formats. It integrates with IJulia, outputting SVG images to the notebook.</p>
<p dir="auto">This library will try to use the lualatex package already installed on the system. Lualatex may be installed through the texlive and miktex distributions. You should have dvisvgm installed. On Ubuntu, you can get these, if not already present, by running <code>sudo apt-get install texlive-latex-base</code> and <code>sudo apt-get install texlive-binaries</code>. If the library cannot run lualatex, it will fall back to trying to use tectonic for the compilation.</p>
<p dir="auto">Note: this package will attempt to turn off interpolation in the generated SVG, but this currently only works in Chrome.</p>

<p dir="auto">Compiling a standalone LaTeX file requires the Tikz code to be wrapped in a <code>tikzpicture</code> environment, which again is wrapped in a <code>document</code> environment. You can omit these wrappers if you intend to embed the output in a larger document, instead of compiling it as a standalone file.</p>
