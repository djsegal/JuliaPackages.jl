<p>This library allows one to create Tikz pictures and save in various formats. It integrates with IJulia, outputting SVG images to the notebook.</p>
<p>In order to use this library, lualatex must be installed. The texlive and miktex distributions include lualatex. You must also have dvisvgm installed. On Ubuntu, you can get these, if not already present, by running <code>sudo apt-get install texlive-latex-base</code> and <code>sudo apt-get install texlive-binaries</code>.</p>
<p>Note: this package will attempt to turn off interpolation in the generated SVG, but this currently only works in Chrome.</p>

<p>Compiling a standalone LaTeX file requires the Tikz code to be wrapped in a <code>tikzpicture</code> environment, which again is wrapped in a <code>document</code> environment. You can omit these wrappers if you intend to embed the output in a larger document, instead of compiling it as a standalone file.</p>

save(TEX(&quot;test&quot;; limit_to=:data), tp) # do not wrap the Tikz code, at all
