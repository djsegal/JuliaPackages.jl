<p dir="auto">Glimmer.jl is a <a href="https://julialang.org/" rel="nofollow">Julia</a> package for creating fast and simple user interface (UI) using the <a href="https://github.com/JuliaGizmos/Blink.jl">Blink</a> package. The UI is a dynamic <a href="https://angular.io/" rel="nofollow">Angular</a> application that runs inside a Blink window.</p>
<p dir="auto">It is designed to allow any Julia user to easily create simple UIs, perform quick code experiments where you can tweak the inputs, and visualize the results in the same window. You can update variable values using the interactive controls or using Julia assinments in the REPL or scripts, and Glimmer will make sure everything is synchronized.</p>
