<p dir="auto">The package can be installed with the Julia package manager.
From the Julia REPL, type <code>]</code> to enter the Pkg REPL mode and run:</p>
<div class="snippet-clipboard-content notranslate position-relative overflow-auto" data-snippet-clipboard-copy-content="pkg&gt; add MessyTimeSeries"><pre class="notranslate"><code>pkg&gt; add MessyTimeSeries
</code></pre></div>
