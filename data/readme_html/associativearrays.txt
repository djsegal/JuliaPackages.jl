<p dir="auto">This package implements associative arrays, the mathematical objects introduced in the book <a href="https://mitpress.mit.edu/books/mathematics-big-data" rel="nofollow">Mathematics of Big Data</a>.</p>
<p dir="auto">This is currently a proof of concept. Much more to do.</p>
<p dir="auto"><strong>Note:</strong> This package should not be relied upon; it is an experiment created during a course I took with the author of the book. The basic idea is to represent tabular data with one column per unique value, turning tables containing categorical columns into sparse numerical arrays. See the <a href="https://d4m.mit.edu/publications" rel="nofollow">D4M</a> publications for a description of the model.</p>
