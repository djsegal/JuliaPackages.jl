<p dir="auto">AlgebraicRewriting.jl is a Julia library for performing category-theoretic
rewrites over a broad class of data structures, defined in
<a href="https://github.com/AlgebraicJulia/Catlab.jl">Catlab.jl</a>.
This package defines DPO, SPO, and SqPO for C-Sets, as described in
[<a href="https://arxiv.org/abs/2111.03784" rel="nofollow">Brown 2022</a>]. See the <a href="https://algebraicjulia.github.io/AlgebraicRewriting.jl/dev/" rel="nofollow">documentation</a> for more details.</p>
