<p>The Tables.jl package provides simple, yet powerful interface functions for working with all kinds tabular data.</p>
<p><strong>Installation</strong>: at the Julia REPL, <code>import Pkg; Pkg.add("Tables")</code></p>
<p><strong>Maintenance</strong>: Tables is maintained collectively by the <a href="https://github.com/orgs/JuliaData/people">JuliaData collaborators</a>.
Responsiveness to pull requests and issues can vary, depending on the availability of key collaborators.</p>
