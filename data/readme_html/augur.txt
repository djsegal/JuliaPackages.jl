<h3><a id="user-content-usage" class="anchor" aria-hidden="true" href="#usage"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path fill-rule="evenodd" d="M7.775 3.275a.75.75 0 001.06 1.06l1.25-1.25a2 2 0 112.83 2.83l-2.5 2.5a2 2 0 01-2.83 0 .75.75 0 00-1.06 1.06 3.5 3.5 0 004.95 0l2.5-2.5a3.5 3.5 0 00-4.95-4.95l-1.25 1.25zm-4.69 9.64a2 2 0 010-2.83l2.5-2.5a2 2 0 012.83 0 .75.75 0 001.06-1.06 3.5 3.5 0 00-4.95 0l-2.5 2.5a3.5 3.5 0 004.95 4.95l1.25-1.25a.75.75 0 00-1.06-1.06l-1.25 1.25a2 2 0 01-2.83 0z"></path></svg></a>Usage</h3>
<p>Run simulations with default settings using all available cores:</p>
<pre><code>$ julia -p `nproc` test/controller.jl
</code></pre>
<p>Simulation results are automatically saved to <code>test/data</code>.  If <code>sim.SAVE_RAW_DATA = true</code>, in addition to the output data, full time traces will be saved to <code>test/data/raw</code>.  (Caution: this option both slows down the simulations and requires considerable storage space to store the results.)</p>
<p>Augur.jl includes plotters written for PyPlot and Gadfly.  To generate plots, just specify <code>pyplot</code> or <code>gadfly</code> when you run the simulations:</p>
<pre><code>$ julia -p `nproc` test/controller.jl pyplot
</code></pre>
<p>Plots are saved to <code>test/plots</code>.</p>
