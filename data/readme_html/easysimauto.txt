<p dir="auto"><code>EasySimauto.jl</code> is the Julia API for ESA, an easy-to-use power system analysis automation environment atop PowerWorld Simulator</p>

<p dir="auto">If you have installed ESA in an existing Python environment and do not want to reinstall it from conda, you can configure it in Pycall.</p>
<p dir="auto">You can manually set the Python path with the following commands in Julia:</p>
<div class="snippet-clipboard-content notranslate position-relative overflow-auto" data-snippet-clipboard-copy-content="ENV[&quot;PYTHON&quot;] = &quot;... path of the python executable ...&quot;

Pkg.build("PyCall")
</code></pre></div>
<p dir="auto">Check out the <a href="https://github.com/JuliaPy/PyCall.jl#specifying-the-python-version">documentation</a> of <code>PyCall.jl</code> for more details.</p>

<p dir="auto"><code>EasySimauto.jl</code> exposes all Python APIs under <code>esa.py</code>. Use the package with</p>
<div class="highlight highlight-source-julia notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="using EasySimauto"><pre><span class="pl-k">using</span> EasySimauto</pre></div>
<p dir="auto">Two classes <code>esa</code> and <code>SAW</code> are exposed once loading the module. Then all the methods in <code>esa.py</code> shall be available in Julia. For example, to load a case file and run power flow analysis, use</p>
<div class="highlight highlight-source-julia notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="using EasySimauto

case_path = raw&quot;C:\Users\myuser\git\ESA\tests\cases\ieee_14\IEEE 14 bus_pws_version_21.pwb&quot;
saw = SAW(case_path)
saw.SolvePowerFlow()
bus_data = saw.get_power_flow_results(&quot;bus&quot;)
print(bus_data)"><pre><span class="pl-k">using</span> EasySimauto

case_path <span class="pl-k">=</span> <span class="pl-s"><span class="pl-c1">raw</span><span class="pl-pds">"</span>C:\Users\myuser\git\ESA\tests\cases\ieee_14\IEEE 14 bus_pws_version_21.pwb<span class="pl-pds">"</span></span>
saw <span class="pl-k">=</span> <span class="pl-c1">SAW</span>(case_path)
saw<span class="pl-k">.</span><span class="pl-c1">SolvePowerFlow</span>()
bus_data <span class="pl-k">=</span> saw<span class="pl-k">.</span><span class="pl-c1">get_power_flow_results</span>(<span class="pl-s"><span class="pl-pds">"</span>bus<span class="pl-pds">"</span></span>)
<span class="pl-c1">print</span>(bus_data)</pre></div>
<p dir="auto">The example above is taken from <a href="https://mzy2240.github.io/ESA/html/quick_start.html#quick-start" rel="nofollow">Quick Start</a> and translated to Julia.</p>
