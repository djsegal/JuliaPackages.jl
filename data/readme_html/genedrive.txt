<p dir="auto"><a href="https://vnvasquez.github.io/GeneDrive.jl/dev/" rel="nofollow"><code>GeneDrive.jl</code></a> is a <a href="https://julialang.org" rel="nofollow">Julia</a> package designed for simulating biological dynamics and control. The objectives of the package include:</p>
<ul dir="auto">
<li>Provide data models that structure inputs to experimental setups and exploit the power of Julia's type system for <a href="https://docs.julialang.org/en/v1/manual/methods/" rel="nofollow">multiple dispatch</a>.</li>
<li>Enable the creation of dynamic models that build on the <a href="https://diffeq.sciml.ai/stable/" rel="nofollow"><code>DifferentialEquations.jl</code></a> platform.</li>
<li>Facilitate the formulation of decision models that employ <a href="https://jump.dev/JuMP.jl/stable/" rel="nofollow"><code>JuMP.jl</code></a>, the domain-specific modeling language for mathematical optimization embedded in Julia.</li>
</ul>
