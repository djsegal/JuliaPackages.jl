<p dir="auto">This is a minimal package for a pure Julia implementation of tools used in <a href="https://en.wikipedia.org/wiki/Computability_theory" rel="nofollow">Computability Theory</a>.  This is the science involving mathematical models of what it means to compute.  This naturally progresses to the idea of complexity: how much memory is used in computing?  How long will it take?  Does it halt?</p>
