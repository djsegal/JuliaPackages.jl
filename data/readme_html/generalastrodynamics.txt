<p>This package is currently being <strong>completely refactored</strong>! This is good news,
since the new implementation will do some cool things with Julia's type system.
The documentation and tests may frequently fail on the <code>main</code> branch on
this repo as I work out the kinks. No unstable versions will be pushed
to the general registry. All <code>stable</code> documentation and release versions
are still stable and work.</p>
<p>Here's a sneak peak at some features to come! The plot below shows a family of Halo orbits about Earth-Moon L1. Upcoming
package features will take advantage of manifolds about Halo orbits like these to find low-cost transfer designs
for interplanetary missions! <g-emoji class="g-emoji" alias="rocket" fallback-src="https://github.githubassets.com/images/icons/emoji/unicode/1f680.png">üöÄ</g-emoji></p>

<h2><a id="user-content-features" class="anchor" aria-hidden="true" href="#features"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path fill-rule="evenodd" d="M7.775 3.275a.75.75 0 001.06 1.06l1.25-1.25a2 2 0 112.83 2.83l-2.5 2.5a2 2 0 01-2.83 0 .75.75 0 00-1.06 1.06 3.5 3.5 0 004.95 0l2.5-2.5a3.5 3.5 0 00-4.95-4.95l-1.25 1.25zm-4.69 9.64a2 2 0 010-2.83l2.5-2.5a2 2 0 012.83 0 .75.75 0 001.06-1.06 3.5 3.5 0 00-4.95 0l-2.5 2.5a3.5 3.5 0 004.95 4.95l1.25-1.25a.75.75 0 00-1.06-1.06l-1.25 1.25a2 2 0 01-2.83 0z"></path></svg></a>Features</h2>
<ul>
<li>Restricted two-body problem equations, states, propagation, and plotting</li>
<li>Restricted three-body problem equations, states, propagation, and iterative Halo orbit solvers</li>
<li>N-body problem equations, states, propagation, and plotting</li>
<li>A collection of fairly accurate planetary constants from our solar system (pulled from <a href="https://naif.jpl.nasa.gov/pub/naif/generic_kernels/" rel="nofollow">SPICE</a> kernals)</li>
</ul>
<p>More to come! In the near term, additional features will include...</p>
<ul>
<li>Manifold-based transfer equations and states within the circular restricted three-body problem</li>
<li>Hohmann-based transfer equations and states within the restricted two-body problem</li>
<li>Zero-velocity curve plots for circular restricted three-body problem trajectories</li>
<li>Stability analysis for circular restricted three-body problem states</li>
</ul>
