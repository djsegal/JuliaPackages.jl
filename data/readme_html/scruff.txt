<p dir="auto">Scruff is an AI framework to build agents that sense, reason, and learn in the world using a variety of models.  It aims to integrate many different kinds of models in a coherent framework, provide flexibility in spatiotemporal modeling, and provide tools to compose, share, and reuse models and model components.</p>
<p dir="auto">Scruff is provided as a <a href="https://julialang.org/" rel="nofollow">Julia</a> package and is licensed under the BSD-3-Clause License.  It should be run using Julia v1.6 or v1.7.</p>
<blockquote>
<p dir="auto"><em>Warning</em>: Scruff is rapidly evolving beta research software. Although the software already has a lot of functionality, we intend to expand on this in the future and cannot promise stability of the code or the APIs at the moment.</p>
</blockquote>

<p dir="auto">The Scruff packages are split into four (4) main modules:  <code>Models</code>, <code>Algorithms</code>, <code>SFuncs</code>, and <code>Operators</code>.</p>
<ul dir="auto">
<li>To add to the <code>Models</code> module, add a <code>.jl</code> file to the <code>src/models/</code> directory and <code>include</code> it in the <code>src/models.jl</code> file</li>
<li>To add to the <code>Algorithms</code> module, add a <code>.jl</code> file to the <code>src/algorithms/</code> directory and <code>include</code> it in the <code>src/algorithms.jl</code> file</li>
<li>To add to the <code>SFuncs</code> module, add a <code>.jl</code> file to the <code>src/sfuncs/</code> directory and <code>include</code> it in the <code>src/sfuncs.jl</code> file</li>
<li>To add to the <code>Operators</code> module, add a <code>.jl</code> file to the <code>src/operators</code> directory and <code>include</code> it in the <code>src/operators.jl</code> file</li>
</ul>
