<p dir="auto"><code>@↓</code> and <code>@↑</code> provide <a href="https://github.com/carlobaldassi/ExtractMacro.jl">ExtractMacro.jl</a>-like features with <a href="https://github.com/mauro3/UnPack.jl">UnPack.jl</a>-like syntax and speed. For example,</p>
<div class="highlight highlight-source-julia notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="using ArrowMacros
mutable struct A; a; b; c; end
mutable struct B; d; e; end
s = A(1, [2, 3], B(4, [5, 6]))

@↓ a, b ← b .- a = s
# (a, b) == (1, [1, 2])

a += 1
@↑ s = a, b ← (@. 2b - 1)
# (s.a, s.b) == (2, [1, 3])"><pre><span class="pl-k">using</span> ArrowMacros

s <span class="pl-k">=</span> <span class="pl-c1">A</span>(<span class="pl-c1">1</span>, [<span class="pl-c1">2</span>, <span class="pl-c1">3</span>], <span class="pl-c1">B</span>(<span class="pl-c1">4</span>, [<span class="pl-c1">5</span>, <span class="pl-c1">6</span>]))
