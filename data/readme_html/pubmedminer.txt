<p><em>NOTE: This package is in a state of flux and currently just supports the <a href="https://bcbi.brown.edu/pubmedminer" rel="nofollow">PubMed Miner Web App</a> - DO NOT USE</em></p>
<p>This package provides a set of tools and examples to study co-occurrences/comorbidities in PubMed/Medline articles based on MeSH descriptors and UMLS concepts</p>
