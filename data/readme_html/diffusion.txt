<p>Diffusion.jl is a Julia package that solves the diffusion equation on a two-dimensional Cartesian domain, using
the Message Passing Interface (MPI) paradigm. For more information, please have a look at
<a href="http://www.claudiobellei.com/2018/09/30/julia-mpi/" rel="nofollow">this blog post</a>.</p>

<p>In your target folder, clone the repository with the command:
<code>git clone https://github.com/cbellei/Diffusion.jl.git</code></p>

<p>Diffusion.jl is tested on Julia 1.0. It assumes that an MPI installation (for example, Open MPI) is available on the system
and that the Julia package MPI.jl has been installed.</p>

<h3><a id="user-content-simulation-parameters" class="anchor" aria-hidden="true" href="#simulation-parameters"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path fill-rule="evenodd" d="M7.775 3.275a.75.75 0 001.06 1.06l1.25-1.25a2 2 0 112.83 2.83l-2.5 2.5a2 2 0 01-2.83 0 .75.75 0 00-1.06 1.06 3.5 3.5 0 004.95 0l2.5-2.5a3.5 3.5 0 00-4.95-4.95l-1.25 1.25zm-4.69 9.64a2 2 0 010-2.83l2.5-2.5a2 2 0 012.83 0 .75.75 0 001.06-1.06 3.5 3.5 0 00-4.95 0l-2.5 2.5a3.5 3.5 0 004.95 4.95l1.25-1.25a.75.75 0 00-1.06-1.06l-1.25 1.25a2 2 0 01-2.83 0z"></path></svg></a>Simulation parameters</h3>
<p>The simulation parameters are all specified in a yaml file, as in <code>src/example.yml</code>.
The parameters that are required are:</p>
<ul>
<li>number of cells in the x and y directions</li>
<li>number of processors in the x and y direction</li>
<li>maximum number of steps for the simulation</li>
<li>time step dt</li>
<li>criterion for terminating the simulation (if the solution changes by less than the <code>tol</code> value, the simulation is ended)</li>
<li>name of the output file</li>
</ul>

<p>If you want to run a simulation with a number of processes equal to <code>nproc</code>, and the simulation parameters are
contained in the file <code>src/example.yml</code>, then you should type on the Terminal</p>
<pre><code>mpirun -np nproc julia src/diffusion.jl src/example.yml
</code></pre>
