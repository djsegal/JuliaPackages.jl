<p dir="auto">A Julia package for easy access to the <a href="https://mba.tuck.dartmouth.edu/pages/faculty/ken.french/data_library.html" rel="nofollow">Ken French Data Library</a>. The Ken French Data Library is one of the most highly used publicly available data sources for financial investments and asset pricing research.</p>
<p dir="auto">Working with the data is sometimes tedious because the downloadable files come (1) compressed in <code>zip</code> archives and (2) having non-standard <code>csv</code> layouts. This package handles both of those hurdles, allowing users to get to the data faster.</p>

<p dir="auto">To add the package, type <code>] add FamaFrenchData</code> at the Julia REPL.</p>
<p dir="auto">Once added, type <code>using FamaFrenchData</code> to import the package.</p>
<p dir="auto">The package exports 3 functions: <code>readFamaFrench</code>, <code>downloadFamaFrench</code>, and <code>listFamaFrench</code>.</p>
<p dir="auto">For help with any of these functions, use <code>?</code> at the REPL (eg. <code>?readFamaFrench</code>).</p>
<p dir="auto">Please consult the online <a href="https://tbeason.github.io/FamaFrenchData.jl/stable" rel="nofollow">documentation</a> for additional detail.</p>
