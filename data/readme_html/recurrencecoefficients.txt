<p dir="auto">Compute recurrence coefficients for Chebyshev polynomials defined on multiple intervals.</p>
<p dir="auto">This respository contains the code for: <a href="https://arxiv.org/abs/2302.12930" rel="nofollow">arxiv.org/abs/2302.12930</a></p>
<p dir="auto">A basic example usage follows.</p>
<div class="snippet-clipboard-content notranslate position-relative overflow-auto" data-snippet-clipboard-copy-content="bands = [-3.0 -2.0; 2.0 3.0] #intervals of support

(avec,bvec) = get_n_coeffs(bands,5) #computes coefficients a₀,…,a₅,b₀,…,b₅"><pre class="notranslate"><code>bands = [-3.0 -2.0; 2.0 3.0] #intervals of support
(a,b) = get_coeffs(bands,5) #computes coefficients a₅,b₅
(avec,bvec) = get_n_coeffs(bands,5) #computes coefficients a₀,…,a₅,b₀,…,b₅
</code></pre></div>
<p dir="auto">Optional arguments include the choice of Chebyshev variant, a weight function perturbation, and the number of collocation points used. Example usage of these optional arguments follows.</p>
<div class="snippet-clipboard-content notranslate position-relative overflow-auto" data-snippet-clipboard-copy-content="bands = [-3.0 -2.0; 2.0 3.0]

<p dir="auto">To use different Chebyshev variants on each interval, different functions must be called.
