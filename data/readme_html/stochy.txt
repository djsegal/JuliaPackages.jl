<p>Stochy is a probabilistic programming language embedded within Julia. Models are specified using a functional subset of Julia, and inference can be performed by exhaustive enumeration, PMCMC<sup><a href="#citations">1</a></sup> or Metropolis-Hastings<sup><a href="#citations">2</a></sup>. The implementation follows the approach described in <a href="http://dippl.org/" rel="nofollow">The Design and Implementation of Probabilistic Programming Languages</a>.</p>

<p>This project is no longer under development.</p>
<p>If you're looking for an actively maintained and more fully featured system you might take a look at <a href="http://webppl.org" rel="nofollow">WebPPL</a>. (Stochy is basically a half-finished Julia port of WebPPL.) You'll find many more alternatives in this list of <a href="http://probabilistic-programming.org/wiki/Home#Existing_probabilistic_programming_systems" rel="nofollow">probabilistic programming systems</a>.</p>
