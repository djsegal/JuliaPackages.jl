<p dir="auto">This repository implements all optimal-transport based distances between spectra detailed in the following pre print</p>
<p dir="auto"><a href="http://arxiv.org/abs/2004.09152" rel="nofollow">"New Metrics Between Rational Spectra and their Connection to Optimal Transport", Bagge Carlson and Chitre 2020</a></p>
<p dir="auto"><strong>Supplementary material</strong> to the paper is provided in a <a href="https://github.com/baggepinnen/SpectralDistances_supplementary_material">separate repository</a>.</p>
<p dir="auto">The package also contains a number of generic solvers for optimal transport problems:</p>
<ul dir="auto">
<li>Fixed support in 1d (histograms)</li>
<li>Varying discrete support (atoms/dirac masses) with non-uniform masses in any dimension</li>
<li>Barycenters supported on fixed number of atoms, but possibly with non-uniform masses</li>
<li>Barycentric coordinates</li>
<li>Continuous support in 1d</li>
</ul>
<p dir="auto">See the <a href="https://baggepinnen.github.io/SpectralDistances.jl/latest" rel="nofollow">documentation</a> for instructions.</p>
