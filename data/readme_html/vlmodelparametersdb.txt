<p dir="auto"><code>VLModelParametersDB.jl</code> is a a <a href="https://github.com/JuliaLang/julia">Julia</a> package that wraps a <a href="https://github.com/JuliaDatabases/SQLite.jl">SQLite</a> database holding transcription (TX) and translation (TL) parameters (taken from <a href="https://bionumbers.hms.harvard.edu/search.aspx" rel="nofollow">BioNumbers</a> and other sources), and
enzyme kinetic information taken from the <a href="https://www.brenda-enzymes.org" rel="nofollow">BRENDA</a> enzyme database.</p>

<p dir="auto"><code>VLModelParametersDB.jl</code> can be installed using the <a href="https://docs.julialang.org/en/v1/stdlib/Pkg/" rel="nofollow">Julia package manager</a>. From the <a href="https://docs.julialang.org/en/v1/stdlib/REPL/" rel="nofollow">Julia REPL</a> enter package mode by pressing the ']' key and then issue the command:</p>
<div class="snippet-clipboard-content notranslate position-relative overflow-auto" data-snippet-clipboard-copy-content="(@v1.6) pkg&gt; add VLModelParametersDB"><pre class="notranslate"><code>(@v1.6) pkg&gt; add VLModelParametersDB
</code></pre></div>
<p dir="auto"><code>VLModelParametersDB.jl</code> can be used stand-alone, or as part of models generated using one of our various model code generators e.g., <a href="https://github.com/varnerlab/JUGRN.jl">JUGRN</a>.</p>
