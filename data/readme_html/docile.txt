<p dir="auto"><em>Docile</em> is a <a href="http://www.julialang.org" rel="nofollow">Julia</a> package documentation system.</p>
<p dir="auto">It provides a macro, <code>@doc</code>, for documenting arbitrary Julia objects. This
functionality is equivalent to the new <code>@doc</code> macro available from <code>Base</code> in
Julia <code>0.4</code>.</p>
<p dir="auto">Additionally, <em>Docile</em> has support for using plain strings for documentation
rather than using the <code>@doc</code> macro. See the package documentation for further
details.</p>

<p dir="auto">Package documentation is available for the <a href="http://docilejl.readthedocs.org/en/stable/" rel="nofollow">stable</a> and
