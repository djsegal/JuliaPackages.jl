<p><code>CorrelationFunctions.jl</code> is a library of correlation functions as described in
Salvatore Torquato's book "Random Heterogeneous Materials".</p>
<p>To build a documentation do <code>cd docs &amp;&amp; julia make.jl</code>. Also the documentation is available on
<a href="https://shamazmazum.github.io/CorrelationFunctions.jl/dev/" rel="nofollow">GitHub Pages</a>.</p>
