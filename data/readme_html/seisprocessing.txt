<p dir="auto"><a href="https://travis-ci.com/SeismicJulia/SeisProcessing.jl" rel="nofollow"><img src="https://camo.githubusercontent.com/b89cdc01dd661154036cabfe4cf85247b8cae9272a86a0f942c9cbff0ef287b7/68747470733a2f2f7472617669732d63692e636f6d2f536569736d69634a756c69612f5365697350726f63657373696e672e6a6c2e7376673f6272616e63683d6d6173746572" alt="Build Status" data-canonical-src="https://travis-ci.com/SeismicJulia/SeisProcessing.jl.svg?branch=master" style="max-width: 100%;"></a></p>
<p dir="auto">This package contains Processing tools for SeismicJulia project.</p>
<p dir="auto">At the moment, it is updated and tested against Julia v1.</p>
<h2 dir="auto"><a id="user-content-installation" class="anchor" aria-hidden="true" href="#installation"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path d="m7.775 3.275 1.25-1.25a3.5 3.5 0 1 1 4.95 4.95l-2.5 2.5a3.5 3.5 0 0 1-4.95 0 .751.751 0 0 1 .018-1.042.751.751 0 0 1 1.042-.018 1.998 1.998 0 0 0 2.83 0l2.5-2.5a2.002 2.002 0 0 0-2.83-2.83l-1.25 1.25a.751.751 0 0 1-1.042-.018.751.751 0 0 1-.018-1.042Zm-4.69 9.64a1.998 1.998 0 0 0 2.83 0l1.25-1.25a.751.751 0 0 1 1.042.018.751.751 0 0 1 .018 1.042l-1.25 1.25a3.5 3.5 0 1 1-4.95-4.95l2.5-2.5a3.5 3.5 0 0 1 4.95 0 .751.751 0 0 1-.018 1.042.751.751 0 0 1-1.042.018 1.998 1.998 0 0 0-2.83 0l-2.5 2.5a1.998 1.998 0 0 0 0 2.83Z"></path></svg></a>Installation</h2>
<p dir="auto">To use this package you must first install the <a href="http://julialang.org/downloads/" rel="nofollow">Julia</a> programming language.
Then, run the Julia application and add the package</p>
<div class="snippet-clipboard-content notranslate position-relative overflow-auto" data-snippet-clipboard-copy-content="julia&gt;using Pkg
julia&gt;Pkg.add(&quot;SeisProcessing&quot;)"><pre class="notranslate"><code>julia&gt;using Pkg
julia&gt;Pkg.add("SeisProcessing")
</code></pre></div>
<p dir="auto">or</p>
<div class="snippet-clipboard-content notranslate position-relative overflow-auto" data-snippet-clipboard-copy-content="julia&gt;]
pkg&gt; add SeisProcessing"><pre class="notranslate"><code>julia&gt;]
pkg&gt; add SeisProcessing
</code></pre></div>
<p dir="auto">Finally, in the julia prompt type</p>
<div class="snippet-clipboard-content notranslate position-relative overflow-auto" data-snippet-clipboard-copy-content="julia&gt;using SeisProcessing"><pre class="notranslate"><code>julia&gt;using SeisProcessing
</code></pre></div>
