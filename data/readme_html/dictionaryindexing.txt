<p dir="auto">Enables slicing dictionaries with <code>sliced</code>, or directly by using function syntax.</p>
<p dir="auto">Example using <code>sliced</code>:</p>

<p dir="auto">Ordinarily <code>keep = :last</code> occurs after filtering, but if for some reason you want it to happen before the filter use <code>keep = :lastbefore</code>. Likewise, if you want it to keep the first instance before filtering, do <code>keep = :firstbefore</code>. This is possibly desirable if your filter function involves the number of occurrences.</p>
<p dir="auto">Since we are working with dictionaries, we are often dealing with keys that are Symbols or Strings. These keys also work, instead of or in combination with numbers.</p>

<p dir="auto">The function syntax version only works with OrderedDict, so convert your dict to an OrderedDict to use this.
Usually this just means using the constructor like <code>newdict = OrderedDict(olddict)</code>.</p>
