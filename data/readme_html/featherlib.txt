<h2><a id="user-content-overview" class="anchor" aria-hidden="true" href="#overview"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path fill-rule="evenodd" d="M4 9h1v1H4c-1.5 0-3-1.69-3-3.5S2.55 3 4 3h4c1.45 0 3 1.69 3 3.5 0 1.41-.91 2.72-2 3.25V8.59c.58-.45 1-1.27 1-2.09C10 5.22 8.98 4 8 4H4c-.98 0-2 1.22-2 2.5S3 9 4 9zm9-3h-1v1h1c1 0 2 1.22 2 2.5S13.98 12 13 12H9c-.98 0-2-1.22-2-2.5 0-.83.42-1.64 1-2.09V6.25c-1.09.53-2 1.84-2 3.25C6 11.31 7.55 13 9 13h4c1.45 0 3-1.69 3-3.5S14.5 6 13 6z"></path></svg></a>Overview</h2>
<p>This is a low level package to read feather files. It is not meant to be used by end users, but rather as a building block for other packages that expose user friendly APIs for file IO.</p>
<p>End users are encouraged to use either <a href="https://github.com/queryverse/FeatherFiles.jl">FeatherFiles.jl</a> or <a href="https://github.com/JuliaData/Feather.jl">Feather.jl</a> to interact with feather files.</p>
<h2><a id="user-content-getting-started" class="anchor" aria-hidden="true" href="#getting-started"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path fill-rule="evenodd" d="M4 9h1v1H4c-1.5 0-3-1.69-3-3.5S2.55 3 4 3h4c1.45 0 3 1.69 3 3.5 0 1.41-.91 2.72-2 3.25V8.59c.58-.45 1-1.27 1-2.09C10 5.22 8.98 4 8 4H4c-.98 0-2 1.22-2 2.5S3 9 4 9zm9-3h-1v1h1c1 0 2 1.22 2 2.5S13.98 12 13 12H9c-.98 0-2-1.22-2-2.5 0-.83.42-1.64 1-2.09V6.25c-1.09.53-2 1.84-2 3.25C6 11.31 7.55 13 9 13h4c1.45 0 3-1.69 3-3.5S14.5 6 13 6z"></path></svg></a>Getting Started</h2>
<p>The package exports two functions: <code>featherread</code> and <code>featherwrite</code>.</p>
<p>Use the <code>featherread</code> function to read a feather file:</p>
<div class="highlight highlight-source-julia"><pre>data <span class="pl-k">=</span> <span class="pl-c1">featherread</span>(<span class="pl-s"><span class="pl-pds">"</span>testfile.feather<span class="pl-pds">"</span></span>)</pre></div>
<p><code>data</code> will then be of type <code>ResultSet</code>. The field <code>columns</code> is a vector of vectors and holds the actual data columns. The field <code>names</code> returns the names of the columns. The <code>description</code> and <code>metadata</code> fields return additional data from the feather file.</p>
<p>Use the <code>featherwrite</code> function to write a feather file:</p>
<div class="highlight highlight-source-julia"><pre><span class="pl-c1">featherwrite</span>(<span class="pl-s"><span class="pl-pds">"</span>testfile.feather<span class="pl-pds">"</span></span>, column_data, column_names)</pre></div>
<p><code>columns</code> should be a vector of vectors that holds the data to be written. <code>column_names</code> should be a vector of <code>Symbol</code>s with the column names.</p>
