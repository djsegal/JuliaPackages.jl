<p dir="auto">This is a low level package to read feather files. It is not meant to be used by end users, but rather as a building block for other packages that expose user friendly APIs for file IO.</p>
<p dir="auto">End users are encouraged to use either <a href="https://github.com/queryverse/FeatherFiles.jl">FeatherFiles.jl</a> or <a href="https://github.com/JuliaData/Feather.jl">Feather.jl</a> to interact with feather files.</p>
<h2 dir="auto"><a id="user-content-getting-started" class="anchor" aria-hidden="true" href="#getting-started"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path d="m7.775 3.275 1.25-1.25a3.5 3.5 0 1 1 4.95 4.95l-2.5 2.5a3.5 3.5 0 0 1-4.95 0 .751.751 0 0 1 .018-1.042.751.751 0 0 1 1.042-.018 1.998 1.998 0 0 0 2.83 0l2.5-2.5a2.002 2.002 0 0 0-2.83-2.83l-1.25 1.25a.751.751 0 0 1-1.042-.018.751.751 0 0 1-.018-1.042Zm-4.69 9.64a1.998 1.998 0 0 0 2.83 0l1.25-1.25a.751.751 0 0 1 1.042.018.751.751 0 0 1 .018 1.042l-1.25 1.25a3.5 3.5 0 1 1-4.95-4.95l2.5-2.5a3.5 3.5 0 0 1 4.95 0 .751.751 0 0 1-.018 1.042.751.751 0 0 1-1.042.018 1.998 1.998 0 0 0-2.83 0l-2.5 2.5a1.998 1.998 0 0 0 0 2.83Z"></path></svg></a>Getting Started</h2>
<p dir="auto">The package exports two functions: <code>featherread</code> and <code>featherwrite</code>.</p>
<p dir="auto">Use the <code>featherread</code> function to read a feather file:</p>
<div class="highlight highlight-source-julia notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="data = featherread(&quot;testfile.feather&quot;)"><pre>data <span class="pl-k">=</span> <span class="pl-c1">featherread</span>(<span class="pl-s"><span class="pl-pds">"</span>testfile.feather<span class="pl-pds">"</span></span>)</pre></div>
<p dir="auto"><code>data</code> will then be of type <code>ResultSet</code>. The field <code>columns</code> is a vector of vectors and holds the actual data columns. The field <code>names</code> returns the names of the columns. The <code>description</code> and <code>metadata</code> fields return additional data from the feather file.</p>
<p dir="auto">Use the <code>featherwrite</code> function to write a feather file:</p>
<div class="highlight highlight-source-julia notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="featherwrite(&quot;testfile.feather&quot;, column_data, column_names)"><pre><span class="pl-c1">featherwrite</span>(<span class="pl-s"><span class="pl-pds">"</span>testfile.feather<span class="pl-pds">"</span></span>, column_data, column_names)</pre></div>
<p dir="auto"><code>columns</code> should be a vector of vectors that holds the data to be written. <code>column_names</code> should be a vector of <code>Symbol</code>s with the column names.</p>
