<p>A Julia package for efficient storage and handling of nested arrays.</p>
<p>ArraysOfArrays provides two different types of nested arrays: <code>ArrayOfSimilarArrays</code> and <code>VectorOfArrays</code>.
An <code>ArrayOfSimilarArrays</code> offers a duality of view between representing the same data as both a flat multi-dimensional array and as an array of equally-sized arrays. A <code>VectorOfArrays</code> represents a vector of arrays of equal dimensionality but different size. Internally, both types store their data in flat arrays that are accessible to the user <code>flatview()</code>.</p>
