<p>Blade Element Momentum Method for Propellers and Turbines.<br>
Author: Andrew Ning</p>
<p>Originally based on my paper: <a href="http://dx.doi.org/10.1002/we.1636" rel="nofollow">http://dx.doi.org/10.1002/we.1636</a> and my earlier python code: <a href="https://github.com/wisdem/ccblade">https://github.com/wisdem/ccblade</a>.  But extended for propellers, wind reversals, no wind in one direction, etc.  See documentation for usage.  A theory document is also available for developers.</p>
