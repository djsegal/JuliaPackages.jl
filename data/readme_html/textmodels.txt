<p dir="auto">The TextModels package enhances the TextAnalysis package with end-user focussed, practical natural language models, typically based on neural networks (in this case, <a href="https://fluxml.ai/" rel="nofollow">Flux</a>).  Please see the <a href="https://juliahub.com/docs/TextModels" rel="nofollow">documentation</a> for more.</p>
