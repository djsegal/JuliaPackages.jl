<p><a href="https://github.com/junyuan-chen/ReadStatTables.jl">ReadStatTables.jl</a>
is a Julia package for loading data from Stata, SAS and SPSS files into
a <a href="https://github.com/JuliaData/Tables.jl">Tables.jl</a>-compatible table.
It relies on <a href="https://github.com/queryverse/ReadStat.jl">ReadStat.jl</a>,
which is a Julia interface of the
<a href="https://github.com/WizardMac/ReadStat">ReadStat</a> C library,
for parsing the data files.
The same C library is also the backend
for popular packages such as <a href="https://github.com/Roche/pyreadstat">pyreadstat</a>
and <a href="https://github.com/tidyverse/haven">haven</a>.</p>
<h2><a id="user-content-features" class="anchor" aria-hidden="true" href="#features"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path fill-rule="evenodd" d="M7.775 3.275a.75.75 0 001.06 1.06l1.25-1.25a2 2 0 112.83 2.83l-2.5 2.5a2 2 0 01-2.83 0 .75.75 0 00-1.06 1.06 3.5 3.5 0 004.95 0l2.5-2.5a3.5 3.5 0 00-4.95-4.95l-1.25 1.25zm-4.69 9.64a2 2 0 010-2.83l2.5-2.5a2 2 0 012.83 0 .75.75 0 001.06-1.06 3.5 3.5 0 00-4.95 0l-2.5 2.5a3.5 3.5 0 004.95 4.95l1.25-1.25a.75.75 0 00-1.06-1.06l-1.25 1.25a2 2 0 01-2.83 0z"></path></svg></a>Features</h2>
<p>ReadStatTables.jl adds the following features on top of the read support
from <a href="https://github.com/queryverse/ReadStat.jl">ReadStat.jl</a>.</p>
<ul>
<li>A lightweight <a href="https://github.com/JuliaData/Tables.jl">Tables.jl</a>-compatible column table <code>ReadStatTable</code> for basic operations.</li>
<li>Integration of value labels via a customized array type <code>LabeledArray</code>.</li>
<li>Translation of date and time values into Julia time types <code>Date</code> and <code>DateTime</code>.</li>
</ul>
