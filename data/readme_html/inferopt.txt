<p dir="auto">InferOpt.jl is a toolbox for using combinatorial optimization algorithms within machine learning pipelines.</p>
<p dir="auto">It allows you to create differentiable layers from optimization oracles that do not have meaningful derivatives.
Typical examples include mixed integer linear programs or graph algorithms.</p>
