The following Julia debuggers are actively maintained:</h1>
<ul dir="auto">
<li><a href="https://junolab.org" rel="nofollow">Juno</a> - the official integrated development environment (IDE) for the Julia language</li>
<li><a href="https://github.com/JuliaDebug/Debugger.jl">Debugger.jl</a> - the official Julia debugger</li>
<li><a href="https://github.com/timholy/Rebugger.jl">Rebugger.jl</a> - an expression-level debugger for Julia with a provocative command-line (REPL) user interface</li>
<li><a href="https://github.com/oxinabox/MagneticReadHead.jl">MagneticReadHead.jl</a> - a Julia debugger based on <a href="https://github.com/jrevels/Cassette.jl">Cassette.jl</a></li>
<li><a href="https://github.com/JuliaDebug/Infiltrator.jl">Infiltrator.jl</a> - allows you to set a breakpoint in a local context, inspect local variables and the call stack, and execute arbitrary statements in the context of the current function's module</li>
</ul>
<p dir="auto">You might also enjoy:</p>
<ul dir="auto">
<li><a href="https://github.com/JunoLab/Traceur.jl">Traceur.jl</a> - runs your code and tells you about any obvious performance traps</li>
<li><a href="https://github.com/JuliaDebug/Cthulhu.jl">Cthulhu.jl</a> - descend into Julia code and see it in all its representations.</li>
</ul>
