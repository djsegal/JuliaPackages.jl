<p dir="auto">NonlinearSolveMINPACK.jl is a wrapper library for bringing
<a href="https://github.com/sglyon/MINPACK.jl">MINPACK.jl</a> into the SciML common interface
for use with NonlinearSolve.jl. For more information on using this library, consult the
documentation for <a href="https://docs.sciml.ai/NonlinearSolve/stable/" rel="nofollow">NonlinearSolve.jl</a>.</p>
