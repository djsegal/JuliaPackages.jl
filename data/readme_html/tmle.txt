<p dir="auto">Targeted Minimum Loss-Based Estimation (TMLE) is a framework for efficient estimation of pathwise differentiable parameters. The goal of this package is to provide an implementation on top of the <a href="https://alan-turing-institute.github.io/MLJ.jl/dev/" rel="nofollow">MLJ</a> framework.</p>
<p dir="auto"><strong>New to TMLE.jl?</strong> The API design is still work in progress but you can already <a href="https://targene.github.io/TMLE.jl/stable/" rel="nofollow">get started now</a></p>
