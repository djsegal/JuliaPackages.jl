<p dir="auto">According to <a href="https://en.wikipedia.org/wiki/Genus_(mathematics)" rel="nofollow">this</a> and
<a href="https://www.sciencedirect.com/science/article/abs/pii/S152407032030031X" rel="nofollow">this</a>
articles, the Euler characteristic for a closed surface is defined as <code>χ = 2 - 2g</code> where <code>g</code> is genus of that surface. For a surface with boundary this
formula becomes <code>χ = 2 - 2g - b</code> where <code>b</code> is a number of connected components
of the boundary. For a surface embedded in three-dimensional space plus its
interior the Euler characteristic is equal to half of the Euler characteristic
of that surface.</p>

<p dir="auto">There is <code>euler_characteristic</code> function in the package which takes a binary
array and returns Euler characteristic. The value <code>true</code> counts as a vertex.</p>
<p dir="auto">There is also <code>EulerTracker</code> type which is a subtype of <code>AbstractArray</code> and
allows for fast recalculation of Euler characteristic when an element of
<code>EulerTracker</code> is changed.</p>
