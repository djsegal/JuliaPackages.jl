<p>The <code>PowerSystems.jl</code> package provides a rigorous data model using Julia structures to enable power systems analysis and modeling. In addition to stand-alone system analysis tools and data model building, the <code>PowerSystems.jl</code> package is used as the foundational data container for the <a href="https://github.com/NREL/PowerSimulations.jl">PowerSimulations.jl</a> package. <code>PowerSystems.jl</code> supports a limited number of data file formats for parsing.</p>
