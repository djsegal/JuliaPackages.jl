<p dir="auto">AdvancedPS provides an efficient implementation of common particle based Monte Carlo samplers using the <a href="https://github.com/TuringLang/AbstractMCMC.jl">AbstractMCMC</a> interface.
The package also relies on <a href="https://github.com/TuringLang/Libtask.jl">Libtask</a> for task manipulation.
AdvancedPS is part of the <a href="https://turing.ml/stable/" rel="nofollow">Turing</a> ecosystem.</p>
