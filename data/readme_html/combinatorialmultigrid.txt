<p>This package implements the Combinatorial Multigrid Preconditioner <em>[1]</em>. The code handles input matrices  that are symmetric diagonally dominant with negative off-diagonal entries, a class of matrices that includes graph Laplacians.</p>
<p>In order to install the package simply do the following:</p>
<div class="snippet-clipboard-content position-relative" data-snippet-clipboard-copy-content="Pkg.add(&quot;CombinatorialMultigrid&quot;) 
using CombinatorialMultigrid
"><pre><code>Pkg.add("CombinatorialMultigrid") 
using CombinatorialMultigrid
</code></pre></div>
<p>We present a quick example. Lets load a fairly large example matrix <code>X</code> from the <code>example</code> directory and build the <code>b</code> side. <code>X</code> is an adjacency matrix. <code>LX</code> is the corresponding Laplacian matrix.</p>
<div class="snippet-clipboard-content position-relative" data-snippet-clipboard-copy-content="## load example matrix

b = b .- sum(b)/length(b);
"><pre><code>## load example matrix
file = matopen("../example/X.mat"); X = read(file, "X"); close(file)

b = b .- sum(b)/length(b);
</code></pre></div>
<p>CMG outputs a preconditioner function that can be used to solve a linear system in the input system. This is done here:</p>

"><pre><code>t = @elapsed (pfunc, h) = cmg_preconditioner_lap(LX);

<p>The second output <code>h</code> is a hierarchy of graphs that is implicitly used in <code>pfunc</code> and it is exposed for its potential in other applications. Alternatively, one can bypass the computation of the Laplacian matrix as follows:</p>
<div class="snippet-clipboard-content position-relative" data-snippet-clipboard-copy-content="(pfunc, h) = cmg_preconditioner_adj(X);
"><pre><code>(pfunc, h) = cmg_preconditioner_adj(X);
</code></pre></div>

<p><code>CMG</code> builds the preconditioner in <code>3.26 seconds</code> compared to <code>26.31 seconds</code> with <code>approxchol_lap</code>. Solving the system takes <code>7.8 seconds</code> with <code>cmg</code>and <code>10.29 seconds</code> with <code>approxchol_lap</code>.</p>
