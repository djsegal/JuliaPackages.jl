<p dir="auto">This package is a Julia wrapper over <a href="https://min-api.cryptocompare.com/documentation" rel="nofollow">CryptoCompare</a> REST API. This package collects data from CryptoCompare and provides wide range of market data includes cryptocurrency trade data, order book data, blockchain data, social data and historical data. Please visit <a href="https://min-api.cryptocompare.com" rel="nofollow">https://min-api.cryptocompare.com</a> for more details.</p>
