<p>This is the Discord bot for Humans of Julia community server.</p>
<p><strong>All HoJ members are invited to contribute to this project.</strong></p>
<p>Please read the <a href="CONTRIBUTING.md">Contribution Guide</a> to get started.</p>
<h2><a id="user-content-features" class="anchor" aria-hidden="true" href="#features"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path fill-rule="evenodd" d="M7.775 3.275a.75.75 0 001.06 1.06l1.25-1.25a2 2 0 112.83 2.83l-2.5 2.5a2 2 0 01-2.83 0 .75.75 0 00-1.06 1.06 3.5 3.5 0 004.95 0l2.5-2.5a3.5 3.5 0 00-4.95-4.95l-1.25 1.25zm-4.69 9.64a2 2 0 010-2.83l2.5-2.5a2 2 0 012.83 0 .75.75 0 001.06-1.06 3.5 3.5 0 00-4.95 0l-2.5 2.5a3.5 3.5 0 004.95 4.95l1.25-1.25a.75.75 0 00-1.06-1.06l-1.25 1.25a2 2 0 01-2.83 0z"></path></svg></a>Features</h2>
<ol>
<li>
<p>The <em>happy reactor</em>: automatically reacts to your message with a smiley when your message sounds happy. Similarly, there are also <em>excited</em> and <em>disappointed</em> reactors.</p>
</li>
<li>
<p>The time zone command (<code>tz</code>): easily converts a date/time into multiple predefined timezones. This is perfect for setting up meeting time that works for our global community.</p>
</li>
</ol>
<p><a target="_blank" rel="noopener noreferrer" href="images/demo.gif"><img src="images/demo.gif" alt="tz demo" style="max-width:100%;"></a></p>
<ol start="3">
<li>The julia command (<code>j</code>): currently, only <code>j?&lt;name&gt;</code> or, alternatively, <code>j doc &lt;name&gt;</code> is implemented, returning the docstring for <code>&lt;name&gt;</code>.</li>
</ol>
