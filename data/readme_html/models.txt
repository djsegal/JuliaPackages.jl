<p dir="auto"><a href="https://github.com/invenia/Models.jl">Models.jl</a> defines the <code>Template</code> and <code>Model</code> types as well as a common API for constructing a generic model in downstream packages, including:</p>
<ul dir="auto">
<li>Calling <code>fit</code> on a <code>Template</code>.</li>
<li>Calling <code>predict</code> on a <code>Model</code>.</li>
<li>Assigning traits such as <code>EstimateTrait</code> and <code>OutputTrait</code>.</li>
<li>Testing interfaces and downstream dependencies with <code>TestUtils</code>.</li>
</ul>
