<ul dir="auto">
<li>Install <a href="https://github.com/JuliaPy/PyCall.jl">PyCall.jl</a> for julia
<ul dir="auto">
<li><code>using Pkg; Pkg.add("PyCall")</code></li>
</ul>
</li>
<li>Install <a href="https://gym.openai.com/docs/" rel="nofollow">OpenAI gym</a> in the same version of python used by <code>PyCall.jl</code>
<ul dir="auto">
<li><code>using Pkg; Pkg.add("Conda"); using Conda; Conda.add("gym")</code></li>
</ul>
</li>
<li>(Optional) Install <a href="http://www.mujoco.org/" rel="nofollow">MuJoCo</a></li>
<li>Install this package by opening julia and running <code>]add git@github.com:ancorso/POMDPGym.git</code></li>
</ul>
