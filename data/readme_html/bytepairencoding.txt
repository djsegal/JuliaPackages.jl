<p dir="auto">Pure Julia implementation of the Byte Pair Encoding (BPE) method.</p>
<p dir="auto">The design is inspired by the original python package <a href="https://github.com/rsennrich/subword-nmt">subword-nmt</a>
and the byte-level bpe use in <a href="https://github.com/openai/gpt-2">openai-gpt2</a>. <code>BytePairEncoding.jl</code> rely on
<a href="https://github.com/chengchingwen/TextEncodeBase.jl">TextEncodeBase.jl</a> and support different tokenization.</p>
