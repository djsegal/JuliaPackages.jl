<p dir="auto">Approximate Bayesian Computation (<strong>ABC</strong>) with differential evolution (<strong>de</strong>) moves and model evidence (<strong>Z</strong>) estimates.</p>
<p dir="auto">ABCdeZ.jl offers Bayesian parameter estimation and model comparison/selection for inference problems with an intractable likelihood. Models only need to be simulated (instead of calculating the likelihood). Please visit the documentation (<a href="https://mauricelanghinrichs.github.io/ABCdeZ.jl/dev/" rel="nofollow">dev</a>/<a href="https://mauricelanghinrichs.github.io/ABCdeZ.jl/stable/" rel="nofollow">stable</a>) to get started.</p>
<p dir="auto"><a target="_blank" rel="noopener noreferrer" href="docs/src/assets/abcdez_min_ex_post.png"><img src="docs/src/assets/abcdez_min_ex_post.png" width="539" style="max-width: 100%;"></a></p>
<p dir="auto">The documentation will go through a minimal example (code also found in <code>examples</code> folder above) computing the posterior samples (Figure above) and evidences for two different models. Model evidences can be used to derive posterior model probabilities (Figure below) or Bayes Factors.</p>
