<p dir="auto">A julia package providing implementation of common <a href="https://en.wikipedia.org/wiki/Semiring" rel="nofollow">semirings</a>.</p>
<p dir="auto">This package is largely inspired from <a href="https://github.com/JuliaComputing/SemiringAlgebra.jl">JuliaComputing/SemiringAlgebra</a>
and <a href="https://github.com/mcognetta/Semirings.jl">mcognetta/Semirings.jl</a>.</p>

<p dir="auto">The package can be installed with the Julia package manager.
From the Julia REPL, type ] to enter the Pkg REPL mode and run:</p>
