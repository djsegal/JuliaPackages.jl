<p dir="auto">Amazing package providing symbolic but explicit</p>
<ul dir="auto">
<li>decomposition of group representations into (semi-)simple representations and</li>
<li>the Wedderburn decompositions for endomorphisms of those representations.</li>
</ul>
<p dir="auto">We work with</p>
<ul dir="auto">
<li>a (linear) <strong>orthogonal</strong> actions of a <strong>finite group</strong> <code>G</code> on <strong>finite dimensional</strong> vector space <code>V</code> over <code>K = ‚Ñù</code> or <code>K = ‚ÑÇ</code> (i.e. a <code>KG</code>-module <code>V</code>) and</li>
<li>linear, <code>G</code>-equivariant maps (equivariant endomorphisms) <code>f : V ‚Üí V</code>.</li>
</ul>
<p dir="auto">These objects are of primary inportance in the study of the representation theory for finite groups, but also naturally arise from (non)commutative polynomial optimization with group symmetry. The aim of the package is to facilitate such uses.</p>
<h2 dir="auto"><a id="user-content-a-bit-of-theory" class="anchor" aria-hidden="true" href="#a-bit-of-theory"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path d="m7.775 3.275 1.25-1.25a3.5 3.5 0 1 1 4.95 4.95l-2.5 2.5a3.5 3.5 0 0 1-4.95 0 .751.751 0 0 1 .018-1.042.751.751 0 0 1 1.042-.018 1.998 1.998 0 0 0 2.83 0l2.5-2.5a2.002 2.002 0 0 0-2.83-2.83l-1.25 1.25a.751.751 0 0 1-1.042-.018.751.751 0 0 1-.018-1.042Zm-4.69 9.64a1.998 1.998 0 0 0 2.83 0l1.25-1.25a.751.751 0 0 1 1.042.018.751.751 0 0 1 .018 1.042l-1.25 1.25a3.5 3.5 0 1 1-4.95-4.95l2.5-2.5a3.5 3.5 0 0 1 4.95 0 .751.751 0 0 1-.018 1.042.751.751 0 0 1-1.042.018 1.998 1.998 0 0 0-2.83 0l-2.5 2.5a1.998 1.998 0 0 0 0 2.83Z"></path></svg></a>A bit of theory</h2>
<p dir="auto">By Maschke's theorem <code>V</code> can be decomposed uniquely <code>V ‚âÖ V‚ÇÅ ‚äï ‚ãØ ‚äï V·µ£</code> into isotypic/semisimple subspaces <code>V·µ¢</code> and each of <code>V·µ¢ ‚âÖ m·µ¢W·µ¢</code> is (in a non-canonical fashion) isomorphic to a direct sum of <code>m·µ¢</code> copies of irreducible/simple subspaces <code>W·µ¢</code>. By (symbolic) computation in (the group algebra) <code>KG</code>, <code>SymbolicWedderburn</code> is capable of producing the exact isomorphism <code>V ‚âÖ V‚ÇÅ ‚äï ‚ãØ ‚äï V·µ£</code> in the form of a collection of projections <code>œÄ·µ¢ : V ‚Üí V·µ¢</code> either in the group algebra (lazy, unevaluated form), or in terms of projection matrices, when a basis for <code>V</code> is explicitly given.</p>
<p dir="auto">The isomorphism produces a decomposition of <code>End_G(V)</code> (the set of linear <code>G</code>-equivariant self-maps of <code>V</code>) in the sense of Artin-Wedderburn theorem, i.e. the projections <code>œÄ·µ¢</code> block-diagonalize <code>f ‚âÖ f‚ÇÅ ‚äï ‚ãØ ‚äï f·µ£</code> where <code>f·µ¢ : V·µ¢ ‚Üí V·µ¢</code>. In terms of matrices if <code>f</code> is given by <code>n√ón</code>-matrix, then we can rewrite it as a block diagonal matrix with blocks of sizes <code>n·µ¢√ón·µ¢</code> where <code>Œ£·µ¢ n·µ¢ = n = dim V</code> and each <code>n·µ¢ = m·µ¢ ¬∑ dim W·µ¢</code>.</p>
<h3 dir="auto"><a id="user-content-semi-definite-constraints" class="anchor" aria-hidden="true" href="#semi-definite-constraints"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path d="m7.775 3.275 1.25-1.25a3.5 3.5 0 1 1 4.95 4.95l-2.5 2.5a3.5 3.5 0 0 1-4.95 0 .751.751 0 0 1 .018-1.042.751.751 0 0 1 1.042-.018 1.998 1.998 0 0 0 2.83 0l2.5-2.5a2.002 2.002 0 0 0-2.83-2.83l-1.25 1.25a.751.751 0 0 1-1.042-.018.751.751 0 0 1-.018-1.042Zm-4.69 9.64a1.998 1.998 0 0 0 2.83 0l1.25-1.25a.751.751 0 0 1 1.042.018.751.751 0 0 1 .018 1.042l-1.25 1.25a3.5 3.5 0 1 1-4.95-4.95l2.5-2.5a3.5 3.5 0 0 1 4.95 0 .751.751 0 0 1-.018 1.042.751.751 0 0 1-1.042.018 1.998 1.998 0 0 0-2.83 0l-2.5 2.5a1.998 1.998 0 0 0 0 2.83Z"></path></svg></a>Semi-definite constraints</h3>
<p dir="auto">For example, if a basis for a semi-definite constraint admits an action of a finite group, then the semi-definite matrix of a <strong>invariant solution</strong> is such an equivariant endomorphism.
In particular if <code>P</code> is a positive semidefinite constraint, when searching for an <code>G</code>-<strong>invariant</strong> solution we may replace</p>
<blockquote>
<p dir="auto"><code>0 ‚™Ø P[1:n, 1:n]</code></p>
</blockquote>
<p dir="auto">by a sequence of constraints</p>
<blockquote>
<p dir="auto"><code>0 ‚™Ø P[1:n·µ¢, 1:n·µ¢]</code> for <code>i = 1‚Ä¶r</code>,</p>
</blockquote>
<p dir="auto">greatly reducing the computational complexity (The size of psd constraint is reduced from <code>n¬≤</code> to <code>Œ£·µ¢ n·µ¢¬≤</code>). Such replacement can be justified if e.g. the objective is symmetric and the set of linear constraints follows a similar group-symmetric structure.</p>
<p dir="auto">If we are only interested in the <em>feasibility</em> of an optimization problem, then such replacement is always justified (i.e. an <em>invariant solution</em> is a <em>honest solution</em> which might not attain the same objective).</p>
<h3 dir="auto"><a id="user-content-rank-one-projection-to-simple-components" class="anchor" aria-hidden="true" href="#rank-one-projection-to-simple-components"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path d="m7.775 3.275 1.25-1.25a3.5 3.5 0 1 1 4.95 4.95l-2.5 2.5a3.5 3.5 0 0 1-4.95 0 .751.751 0 0 1 .018-1.042.751.751 0 0 1 1.042-.018 1.998 1.998 0 0 0 2.83 0l2.5-2.5a2.002 2.002 0 0 0-2.83-2.83l-1.25 1.25a.751.751 0 0 1-1.042-.018.751.751 0 0 1-.018-1.042Zm-4.69 9.64a1.998 1.998 0 0 0 2.83 0l1.25-1.25a.751.751 0 0 1 1.042.018.751.751 0 0 1 .018 1.042l-1.25 1.25a3.5 3.5 0 1 1-4.95-4.95l2.5-2.5a3.5 3.5 0 0 1 4.95 0 .751.751 0 0 1-.018 1.042.751.751 0 0 1-1.042.018 1.998 1.998 0 0 0-2.83 0l-2.5 2.5a1.998 1.998 0 0 0 0 2.83Z"></path></svg></a>Rank one projection to simple components</h3>
<p dir="auto">Sometimes even stronger reduction is possible when the acting group <code>G</code> is <em>sufficiently complicated</em> and we have a <em>minimal projection system</em> at our disposal (The package tries to compute such system by a heuristic algorithm. If the approach fails please open an issue!). In such case we can (often) find subsequent projections <code>V·µ¢ ‚Üí K^{m·µ¢}</code> (depending only on the multiplicity of the irreducible, <strong>not</strong> on its dimension!). This leads to an equivalent formulation for the psd constraint with <code>n·µ¢ = m·µ¢</code> further reducing its size.</p>
<p dir="auto">Moreover in the case of symmetric optimization problems it's possible to use the symmetry to reduce the number of linear constraints (since in that case only one constraint <strong>per orbit</strong> is needed). <code>SymbolicWedderburn</code> facilitates also this simplification.</p>

<p dir="auto">In <a href="https://arxiv.org/abs/1712.07167" rel="nofollow"><em>Aut(ùîΩ‚ÇÖ) has property (T)</em></a> we use the trick above to successfully simplify and solve a large semidefinite problem coming from sum-of-squares optimization.</p>
<p dir="auto">The original problem had one (symmetric) psd constraint of size <code>4641√ó4641</code> and <code>11_154_301</code> linear constraints. By exploiting its (admittedly -- pretty large) symmetry group (of order <code>3840</code>) we can reduce this problem to <code>20</code> (symmetric) psd constraints of sizes</p>
<div class="snippet-clipboard-content notranslate position-relative overflow-auto" data-snippet-clipboard-copy-content="[56  38  34  32  27  27  23  23  22  22  18  17  9  8  6  2  1  1  1  1]"><pre class="notranslate"><code>[56  38  34  32  27  27  23  23  22  22  18  17  9  8  6  2  1  1  1  1]
</code></pre></div>
<p dir="auto">which correspond to (the simple) <code>W·µ¢</code> blocks above. In particular, the number of variables in psd constraints was reduced from <code>10_771_761</code> to just <code>5_707</code>.</p>
<p dir="auto">Moreover, the symmetry group has just <code>7 229</code> orbits (when acting on the subspace of linear constraints), so the symmetrized problem has equal number of (a bit denser) linear constraints.</p>
<p dir="auto">The symmetrized problem is solvable in ~20 minutes on an average office laptop (with <code>16GB</code> of RAM).</p>
<p dir="auto">For more examples you may have a look at <a href="https://github.com/kalmarek/SymbolicWedderburn.jl/blob/master/examples/ex_robinson_form.jl">dihedral action example</a>, or different <a href="https://github.com/kalmarek/SymbolicWedderburn.jl/blob/master/examples/sos_problem.jl">sum of squares formulations</a>.</p>

<p dir="auto">This package is used by <a href="https://github.com/jump-dev/SumOfSquares.jl">SumOfSquares</a> to perform exactly this reduction, for an example use see its <a href="https://jump.dev/SumOfSquares.jl/latest/generated/Symmetry/dihedral_symmetry_of_the_robinson_form/" rel="nofollow">documentation</a>.</p>
<p dir="auto">The software for sum of (hermitian) squares computations in a non-commutative setting (group algebra of a infinite group) using <code>SymbolicWedderburn</code> is my project <a href="https://github.com/kalmarek/PropertyT.jl/"><code>PropertyT.jl</code></a> (unregistered). There we used the sum of squares optimization to prove Property (T) for special automorphisms group of the free group. It's a cool result, <a href="https://annals.math.princeton.edu/2021/193-2/p03" rel="nofollow">check it out!</a>.</p>
<h2 dir="auto"><a id="user-content-other-symbolic-decompositions" class="anchor" aria-hidden="true" href="#other-symbolic-decompositions"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path d="m7.775 3.275 1.25-1.25a3.5 3.5 0 1 1 4.95 4.95l-2.5 2.5a3.5 3.5 0 0 1-4.95 0 .751.751 0 0 1 .018-1.042.751.751 0 0 1 1.042-.018 1.998 1.998 0 0 0 2.83 0l2.5-2.5a2.002 2.002 0 0 0-2.83-2.83l-1.25 1.25a.751.751 0 0 1-1.042-.018.751.751 0 0 1-.018-1.042Zm-4.69 9.64a1.998 1.998 0 0 0 2.83 0l1.25-1.25a.751.751 0 0 1 1.042.018.751.751 0 0 1 .018 1.042l-1.25 1.25a3.5 3.5 0 1 1-4.95-4.95l2.5-2.5a3.5 3.5 0 0 1 4.95 0 .751.751 0 0 1-.018 1.042.751.751 0 0 1-1.042.018 1.998 1.998 0 0 0-2.83 0l-2.5 2.5a1.998 1.998 0 0 0 0 2.83Z"></path></svg></a>Other symbolic decompositions</h2>
<p dir="auto">The main aim of <code>GAP</code> package <a href="https://www.gap-system.org/Manuals/pkg/wedderga/doc/chap0.html" rel="nofollow"><code>Wedderga</code></a> is to</p>
<blockquote>
<p dir="auto">compute the simple components of the Wedderburn decomposition of semisimple group algebras of finite groups over finite fields and over subfields of finite cyclotomic extensions of the rationals.</p>
</blockquote>
<p dir="auto">The focus is thus on symbolic computations and identifying <em>isomorphism type</em> of the simple components.
<code>SymbolicWedderburn</code> makes no efforts to compute the types or defining fields,
it's primary goal is to compute symbolic/numerical Wedderburn-Artin isomorphism in a form usable for (polynomial) optimization. <code>Wedderga</code> also contains much more sophisticated methods for computing <em>a complete set of orthogonal primitive idempotents</em> (i.e. a minimal projection system) through Shoda pairs.
In principle those idempotents could be computed using <a href="https://github.com/oscar-system/Oscar.jl"><code>Oscar</code></a> and used in <code>SymbolicWedderburn</code>.</p>
