<p>Blowfish.jl is an implementation of Bruce Schneier's Blowfish encryption algorithm.</p>
<p>This implementation was loosely transcribed from the <a href="https://golang.org" rel="nofollow">Go</a>(golang) port of <a href="https://www.schneier.com/blowfish.html" rel="nofollow">Bruce Schneier's C implementation</a>.<br>
See <a href="https://pkg.go.dev/golang.org/x/crypto/blowfish" rel="nofollow">crpto/blowfish</a></p>
