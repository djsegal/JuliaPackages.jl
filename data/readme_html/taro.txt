<p>Taro is a utility belt of functions to work with document files in Julia. It uses <a href="http://tika.apache.org/" rel="nofollow">Apache Tika</a>, <a href="http://poi.apache.org" rel="nofollow">Apache POI</a> and <a href="https://xmlgraphics.apache.org/fop/" rel="nofollow">Apache FOP</a>  (via <a href="http://aviks.github.io/JavaCall.jl/" rel="nofollow">JavaCall</a>) to work with Word, Excel and PDF files.</p>
<p>Documentation is available at <a href="http://aviks.github.io/Taro.jl/" rel="nofollow">http://aviks.github.io/Taro.jl/</a></p>
