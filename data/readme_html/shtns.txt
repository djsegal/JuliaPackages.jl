<p dir="auto">For the moment not a lot is tested. There are wrappers generated by Clang.jl to call most functions exported by the C library. The <em>more Julian</em> API is under development. The documentation will come, for the time being I refer to the <a href="https://nschaeff.bitbucket.io/shtns/" rel="nofollow">documentation of the C library</a>.</p>
