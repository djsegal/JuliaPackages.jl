<p dir="auto">A simple interface for interpolations on timeseries with first order discontinuities (using <a href="https://github.com/kbarbary/Dierckx.jl">Dierckx.jl</a>).</p>
<p dir="auto">The module exports one new type, <code>PiecewiseSpline1D</code>, and extends one function, <code>append!()</code>.</p>
<div class="snippet-clipboard-content notranslate position-relative overflow-auto" data-snippet-clipboard-copy-content="PiecewiseSpline1D(times::Vector, values::Vector, jumps::Vector)"><pre class="notranslate"><code>PiecewiseSpline1D(times::Vector, values::Vector, jumps::Vector)
</code></pre></div>
<p dir="auto">Construct a spline interpolation which takes into account discontinuties listed in <code>jumps</code>.
Any <code>PiecewiseSpline1D</code> object <code>p</code> can be called as a function, i.e. <code>p(t)</code>.</p>
<div class="snippet-clipboard-content notranslate position-relative overflow-auto" data-snippet-clipboard-copy-content="append!(p::PiecewiseSpline1D, new_times, new_values, new_jumps)"><pre class="notranslate"><code>append!(p::PiecewiseSpline1D, new_times, new_values, new_jumps)
</code></pre></div>
<p dir="auto">Extend an existing spline <code>p</code> with new data.</p>
<p dir="auto"><strong>A practical example can be found in our <a href="demo.ipynb">demo notebook</a>.</strong></p>
