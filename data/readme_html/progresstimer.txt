<p dir="auto">This package implements a light-weight and easy to use timing functionality for Julia when you know the number of iterations that need to take place.</p>
<p dir="auto">If you want more advanced features please see the package <a href="https://github.com/timholy/ProgressMeter.jl">ProgressMeter</a>.</p>
<p dir="auto">Activate the package as usual using</p>
<div class="highlight highlight-source-julia notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="using ProgressTimer"><pre><span class="pl-k">using</span> ProgressTimer</pre></div>
<p dir="auto">To start the timer call <code>TimingInit()</code> and store the output into a variable</p>
<div class="highlight highlight-source-julia notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="TS=TimingInit()"><pre>TS<span class="pl-k">=</span><span class="pl-c1">TimingInit</span>()</pre></div>
<p dir="auto">To trigger printing simply by calling <code>TimingProgress(TS,Nr,NSamples)</code> and store the output again.</p>
<div class="highlight highlight-source-julia notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="for n in 1:NSamples
    ###
    ### Some computation
    ###
    TS=TimingProgress(TS,n,NSamples)
end"><pre><span class="pl-k">for</span> n <span class="pl-k">in</span> <span class="pl-c1">1</span><span class="pl-k">:</span>NSamples
    <span class="pl-c"><span class="pl-c">#</span>##</span>
    <span class="pl-c"><span class="pl-c">#</span>## Some computation</span>
    <span class="pl-c"><span class="pl-c">#</span>##</span>
    TS<span class="pl-k">=</span><span class="pl-c1">TimingProgress</span>(TS,n,NSamples)
<span class="pl-k">end</span></pre></div>
