<p>This package implements basic functionality for a type of Functional Reactive Programming (FRP).
This is a style of DataFlow programming where updates propagate to other objects automatically.
It follows the same basic API as <a href="http://juliagizmos.github.io/Reactive.jl/" rel="nofollow">Reactive.jl</a>.</p>
