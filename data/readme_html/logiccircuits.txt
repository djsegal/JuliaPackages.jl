<h2><a id="user-content-installation" class="anchor" aria-hidden="true" href="#installation"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path fill-rule="evenodd" d="M7.775 3.275a.75.75 0 001.06 1.06l1.25-1.25a2 2 0 112.83 2.83l-2.5 2.5a2 2 0 01-2.83 0 .75.75 0 00-1.06 1.06 3.5 3.5 0 004.95 0l2.5-2.5a3.5 3.5 0 00-4.95-4.95l-1.25 1.25zm-4.69 9.64a2 2 0 010-2.83l2.5-2.5a2 2 0 012.83 0 .75.75 0 001.06-1.06 3.5 3.5 0 00-4.95 0l-2.5 2.5a3.5 3.5 0 004.95 4.95l1.25-1.25a.75.75 0 00-1.06-1.06l-1.25 1.25a2 2 0 01-2.83 0z"></path></svg></a>Installation</h2>
<p>To install the latest stable release, run:</p>
<div class="highlight highlight-source-shell"><pre>julia -e <span class="pl-s"><span class="pl-pds">'</span>using Pkg; Pkg.add("LogicCircuits")<span class="pl-pds">'</span></span></pre></div>
<p>To install the package with the latest commits on master branch, run:</p>
<div class="highlight highlight-source-shell"><pre>julia -e <span class="pl-s"><span class="pl-pds">'</span>using Pkg; Pkg.add(PackageSpec(url="https://github.com/Juice-jl/LogicCircuits.jl.git"))<span class="pl-pds">'</span></span></pre></div>
<p>These will automatically install all dependencies described in <code>Project.toml</code>.
The first time you run <code>using LogicCircuits</code> in Julia, it will precompile the package and all its dependencies.</p>
