<p>To install the latest stable release, run:</p>
<pre><code>julia -e 'using Pkg; Pkg.add("LogicCircuits")'
</code></pre>
<p>To install the package with the latest commits on master branch, run:</p>
<pre><code>julia -e 'using Pkg; Pkg.add(PackageSpec(url="https://github.com/Juice-jl/LogicCircuits.jl.git"))'
</code></pre>
<p>These will automatically install all dependencies described in <code>Project.toml</code>.
The first time you run <code>using LogicCircuits</code> in Julia, it will precompile the package and all its dependencies.</p>
