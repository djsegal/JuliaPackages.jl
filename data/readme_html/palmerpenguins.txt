<p dir="auto">The <a href="https://allisonhorst.github.io/palmerpenguins/" rel="nofollow">Palmer penguins dataset</a> by Allison Horst, Alison Hill, and Kristen Gorman is a dataset for data exploration &amp; visualization, as an alternative to the Iris dataset.</p>
<p dir="auto">The dataset contains data for 344 penguins.
There are 3 different species of penguins in this dataset, collected from 3 islands in the Palmer Archipelago, Antarctica.</p>
<p dir="auto">Data were collected and made available by <a href="https://www.uaf.edu/cfos/people/faculty/detail/kristen-gorman.php" rel="nofollow">Dr. Kristen Gorman</a> and the <a href="https://pal.lternet.edu/" rel="nofollow">Palmer Station, Antarctica LTER</a>, a member of the <a href="https://lternet.edu/" rel="nofollow">Long Term Ecological Research Network</a>.</p>
<p dir="auto">Data are available by <a href="https://creativecommons.org/share-your-work/public-domain/cc0/" rel="nofollow">CC-0</a> license in accordance with the <a href="http://pal.lternet.edu/data/policies" rel="nofollow">Palmer Station LTER Data Policy</a> and the <a href="https://lternet.edu/data-access-policy/" rel="nofollow">LTER Data Access Policy for Type I data</a>.</p>
<p dir="auto">More information about the dataset is available in <a href="https://allisonhorst.github.io/palmerpenguins/" rel="nofollow">its official documentation</a>.</p>

<p dir="auto">The simplified version of the raw data can be loaded by</p>
<div class="highlight highlight-source-julia notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="julia&gt; using PalmerPenguins

julia&gt; table = PalmerPenguins.load()"><pre>julia<span class="pl-k">&gt;</span> <span class="pl-k">using</span> PalmerPenguins

julia<span class="pl-k">&gt;</span> table <span class="pl-k">=</span> PalmerPenguins<span class="pl-k">.</span><span class="pl-c1">load</span>()</pre></div>
<p dir="auto">This will download the data (both the simplified and the raw version) once using <a href="https://github.com/oxinabox/DataDeps.jl">DataDeps.jl</a>, displaying the information above and making it available for future use.</p>
<p dir="auto">The prompt and the confirmation before downloading the data for the first time can be bypassed by setting the <a href="https://www.oxinabox.net/DataDeps.jl/stable/z10-for-end-users/#Configuration-1" rel="nofollow"><code>DATADEPS_ALWAYS_ACCEPT</code> environment variable</a>.
In particular, this is required for <a href="https://github.com/devmotion/PalmerPenguins.jl/issues/9" data-hovercard-type="issue" data-hovercard-url="/devmotion/PalmerPenguins.jl/issues/9/hovercard">non-interactive use</a> such as in continuous integration tests.</p>
<p dir="auto">The function <code>PalmerPenguins.load()</code> returns a <a href="https://juliadata.github.io/CSV.jl/stable/#CSV.File" rel="nofollow"><code>CSV.File</code></a> object that supports the <a href="https://github.com/JuliaData/Tables.jl">Tables.jl</a> interface.
For instance, it can be converted to a <a href="https://juliadata.github.io/DataFrames.jl/stable/man/getting_started/#The-DataFrame-Type-1" rel="nofollow"><code>DataFrame</code></a> by executing</p>
<div class="highlight highlight-source-julia notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="julia&gt; using DataFrames

julia&gt; df = DataFrame(table)"><pre>julia<span class="pl-k">&gt;</span> <span class="pl-k">using</span> DataFrames

julia<span class="pl-k">&gt;</span> df <span class="pl-k">=</span> <span class="pl-c1">DataFrame</span>(table)</pre></div>
<p dir="auto">The raw data can be loaded with</p>
<div class="highlight highlight-source-julia notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="julia&gt; tableraw = PalmerPenguins.load(; raw = true)"><pre>julia<span class="pl-k">&gt;</span> tableraw <span class="pl-k">=</span> PalmerPenguins<span class="pl-k">.</span><span class="pl-c1">load</span>(; raw <span class="pl-k">=</span> <span class="pl-c1">true</span>)</pre></div>
<p dir="auto">If <code>raw = false</code> (the default), then the simplified version is returned.</p>
