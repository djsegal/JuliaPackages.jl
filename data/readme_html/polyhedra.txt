<p><a href="https://github.com/JuliaPolyhedra/Polyhedra.jl/tree/master/examples/drakeperm.jl"><img src="examples/drakeperm.png" height="240" style="max-width:100%;"></a>
<a href="https://github.com/JuliaPolyhedra/Polyhedra.jl/tree/master/examples/glvizperm.jl"><img src="examples/glvizperm.png" height="240" style="max-width:100%;"></a></p>
<p>Polyhedra provides an unified interface for Polyhedral Computation Libraries such as <a href="https://github.com/JuliaPolyhedra/CDDLib.jl">CDDLib.jl</a>.
These manipulation notably include the transformation from (resp. to) an inequality representation of a polyhedron to (resp. from) its generator representation (convex hull of points + conic hull of rays) and projection/elimination of a variable with e.g. <a href="https://en.wikipedia.org/wiki/Fourier%E2%80%93Motzkin_elimination" rel="nofollow">Fourier-Motzkin</a>.</p>
<p>It defines the abstract type <code>Polyhedron</code> and splits the operations on this type in two categories:</p>
<ul>
<li>Mandatory: Operations that needs to be implemented by the Polyhedral Computation Libraries: e.g. Transformation between the two representations described above and variable elimination.</li>
<li>Optional: Operations that can be implemented using the other operations and hence have a default implementation: e.g. creation of the polyhedron from the feasible set of a <a href="https://github.com/JuliaOpt/JuMP.jl">JuMP</a> model, linear transformation, intersection, <a href="https://en.wikipedia.org/wiki/Minkowski_addition" rel="nofollow">Minkowski addition</a>, decomposition into points and faces for e.g. 3D visualization using <a href="https://github.com/rdeits/DrakeVisualizer.jl">DrakeVisualizer</a> or <a href="https://github.com/JuliaGL/GLVisualize.jl">GLVisualize.jl</a>...</li>
</ul>
