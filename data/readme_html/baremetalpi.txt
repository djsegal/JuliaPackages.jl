<p dir="auto">This package has the purpose to create a Julia interface with the peripherals of
the Raspberry Pi without using any external libraries. Avoiding middlewares can
help to decrease latency for real-time applications.</p>

<p dir="auto">This package should be considered alpha. It has been tested only against the
Raspberry Pi W Zero. Help is needed to improve the reliability of the package.</p>
