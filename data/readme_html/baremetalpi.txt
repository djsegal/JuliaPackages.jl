<p>This package has the purpose to create a Julia interface with the peripherals of
the Raspberry Pi without using any external libraries. Avoiding middlewares can
help to decrease latency for real-time applications.</p>

<p>This package should be considered alpha. It has been tested only against the
Raspberry Pi W Zero. Help is needed to improve the reliability of the package.</p>

<p>The following peripherals are currently supported:</p>
<ul>
<li>GPIO (without PWM).</li>
<li>SPI.</li>
<li>I2C (SMBUS).</li>
</ul>
