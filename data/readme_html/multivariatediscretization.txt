MultivariateDiscretization.jl is a Julia package that can be used for mulitvariate, interaction preserving discretization. Currently the following methods are implemented:</p>
<ul dir="auto">
<li>Uniform width binning</li>
<li>Bayesian Blocks using <a href="https://github.com/Tchanders/Discretizers.jl">Discretizers.jl</a>.</li>
<li><a href="https://ieeexplore.ieee.org/document/1490525" rel="nofollow">Correlation preserving discretization</a></li>
<li><a href="https://link.springer.com/article/10.1007/s10618-014-0350-5" rel="nofollow">Interaction preserving discretization</a></li>
</ul>
<p dir="auto">For testing purposes a function to evaluate the <a href="http://eda.mmci.uni-saarland.de/prj/cjs/" rel="nofollow">Cumulative Jensen Shannon divergence</a> is included as well.</p>
<p dir="auto">Documentation can be found <a href="https://github.com/niclaspopp/MultivariateDiscretization.jl/blob/master/MultivariateDiscretization/Doc%20v0.1.0.ipynb">here</a></p>
