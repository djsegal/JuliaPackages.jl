<p dir="auto"><em>A documentation for the McmcHermes package.</em></p>
<p dir="auto">McmcHermes provides a simple but efficient way to generate <a href="https://en.wikipedia.org/wiki/Markov_chain_Monte_Carlo" rel="nofollow">Markov Chain Monte-Carlo</a> algorithms in order to sample a probability density distribution.</p>

<p dir="auto">The major functions in this module are:</p>
<p dir="auto">run_mcmc: run multiple chains with a specific number of walkers.</p>
<p dir="auto">get_flat_chain: get the stored chain of MCMC samples.</p>
<p dir="auto">get_gelman_rubin: get the Gelman Rubin convergence diagnostic of the chains.</p>
