<p><a href="https://pik-icon.github.io/NetworkDynamics.jl/dev" rel="nofollow"><img src="https://camo.githubusercontent.com/7fcec4b2d3ab291529fce8ef6a4fcd4129a0683b2f5d5fe2f5c648f02db8b616/68747470733a2f2f696d672e736869656c64732e696f2f62616467652f646f63732d6465762d626c75652e737667" alt="" data-canonical-src="https://img.shields.io/badge/docs-dev-blue.svg" style="max-width:100%;"></a></p>

<p>A package for working with dynamical systems on complex networks. NetworkDynamics.jl provides an interface between <a href="https://github.com/JuliaGraphs/LightGraphs.jl">LightGraphs.jl</a> and <a href="https://github.com/JuliaDiffEq/DifferentialEquations.jl">DifferentialEquations.jl</a>. It allows to define several types of dynamic and static nodes and edges and to link them up in order to create complex network dynamics.</p>
<p>The behavior of a node or an edge can be described by algebraic equations, by differential algebraic equation (DAEs) in mass matrix form, by ordinary differential equations (ODE) or by delay differential equations (DDE). Stochastic ordinary differential equations (SDE) can be implemented as a <a href="https://github.com/pik-icon/NetworkDynamics.jl/blob/master/examples/sde.jl">two-layer network</a>. For details see the <a href="https://pik-icon.github.io/NetworkDynamics.jl/dev" rel="nofollow">docs</a>.</p>
