<p>A package for working with dynamical systems on complex networks. NetworkDynamics.jl provides an interface between <a href="https://github.com/JuliaGraphs/LightGraphs.jl">LightGraphs.jl</a> and <a href="https://github.com/JuliaDiffEq/DifferentialEquations.jl">DifferentialEquations.jl</a>. It allows to define several types of dynamic and static nodes and edges and to link them up in order to create complex network dynamics.</p>
<p>At the moment the behaviour of a node or an edge can be described by algebraic equations, by ordinary differential equations (ODE) or by delay differential equations(DDE). Stochastic ordinary differential equations (SDE) can be implemented as a <a href="https://github.com/FHell/NetworkDynamics.jl/blob/master/examples/sde.jl">two-layer network</a> - this is still an experimental feature, though.</p>
