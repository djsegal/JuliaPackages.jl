<p dir="auto">Similar to <a href="https://github.com/JuliaLang/julia/tree/master/stdlib/Dates">Dates</a>' <code>firstdayof*</code> and <code>lastdayof*</code>, I introduce to you <code>penultimatedayof*</code>; a useful function to find the second-to-last day of a measure of time.</p>
<p dir="auto">Unlike Dates' <code>*dayof*</code> functions, I have extended functionality to find, for example, the penultimate <em>Tuesday</em> of a measure of time.  We have also extended some of Dates' <code>*dayof*</code> functions to allow this functionality.</p>
<p dir="auto">See also <a href="https://github.com/jakewilliami/pt">jakewilliami/pt</a> for a command-line implementation using logic from this package.</p>
