<p><code>StippleCharts</code> is a library of reactive charts for <a href="https://github.com/GenieFramework/Stipple.jl">Stipple.jl</a>.</p>
<p><code>StippleCharts.jl</code>, together with
<a href="https://github.com/GenieFramework/Stipple.jl">Stipple.jl</a>,
<a href="https://github.com/GenieFramework/StippleUI.jl">StippleUI.jl</a> and
<a href="https://github.com/GenieFramework/Genie.jl">Genie.jl</a> provide a powerful and complete solution for building
beautiful, responsive, reactive, high performance interactive data dashboards in pure Julia.</p>
<p><code>StippleCharts</code> provides access to multiple multiple types of reactive charts components, including line, bar, scatter, bubble, pie, donut, and column.</p>
