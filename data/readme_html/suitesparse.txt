<p dir="auto">SuiteSparse.jl provides Julia wrappers for the <a href="https://github.com/DrTimothyAldenDavis/SuiteSparse">SuiteSparse</a> library, and provides Julia's sparse linear algebra capabilities - specifically the solvers. This package has been merged into <a href="https://github.com/JuliaSparse/SparseArrays.jl">SparseArrays.jl</a>, and further development continues there.</p>
<p dir="auto">If you use this package in an academic work please cite <a href="https://github.com/JuliaLang/julia/blob/master/CITATION.bib">Julia</a>, <a href="https://github.com/DrTimothyAldenDavis/SuiteSparse/blob/master/CITATION.bib">SuiteSparse, as well as the individual components you used</a>.</p>
