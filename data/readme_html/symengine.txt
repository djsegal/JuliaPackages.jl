<p>One can define variables in a few ways. The following three examples are equivalent.</p>
<p>Defining two symbolic variables with the names <code>a</code> and <code>b</code>, and assigning them to julia variables with the same name.</p>
<div class="highlight highlight-source-julia"><pre>julia<span class="pl-k">&gt;</span> a<span class="pl-k">=</span><span class="pl-c1">symbols</span>(<span class="pl-c1">:a</span>); b<span class="pl-k">=</span><span class="pl-c1">symbols</span>(<span class="pl-c1">:b</span>)
b

julia<span class="pl-k">&gt;</span> a,b <span class="pl-k">=</span> <span class="pl-c1">symbols</span>(<span class="pl-s"><span class="pl-pds">"</span>a b<span class="pl-pds">"</span></span>)
(a, b)

julia<span class="pl-k">&gt;</span> <span class="pl-c1">@vars</span> a b
(a, b)</pre></div>
