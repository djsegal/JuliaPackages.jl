<p dir="auto">This package provides a standardized API for multivariate polynomial bases
based on the <a href="https://github.com/JuliaAlgebra/MultivariatePolynomials.jl">MultivariatePolynomials</a> API.</p>
<p dir="auto">It defines the following basis:</p>
<ul dir="auto">
<li><code>FixedPolynomialBasis</code>: A polynomial basis described by a list of polynomials.</li>
<li>Monomial bases: <code>MonomialBasis</code> and <code>ScaledMonomialBasis</code>.</li>
<li>Orthogonal bases:
<ul dir="auto">
<li>Hermite bases: <code>ProbabilistsHermiteBasis</code> and <code>PhysicistsHermiteBasis</code>.</li>
<li><code>LaguerreBasis</code>.</li>
<li>Gegenbauer bases:
<ul dir="auto">
<li><code>LegendreBasis</code>.</li>
<li>Chebyshev bases: <code>ChebyshevBasisFirstKind</code> and <code>ChebyshevBasisSecondKind</code></li>
</ul>
</li>
</ul>
</li>
</ul>
<p dir="auto">See the documentation for more details.</p>
