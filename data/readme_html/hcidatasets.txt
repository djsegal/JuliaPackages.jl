<p dir="auto">This repository provides a common location for high-contrast imaging datasets in an easy-to-consume interface.</p>

<p dir="auto">The following datasets are provided:</p>
<ul dir="auto">
<li><code>HR8799</code> - HR8799 taken with NIRC2/Vortex Coronagraph</li>
<li><code>BetaPictoris</code> - Beta Pictoris taken with NaCo IRDIS</li>
<li><code>V471Tau</code> - V 471 Tau IFS data cube taken with VLT/SPHERE</li>
</ul>

<p dir="auto">The datasets provided here are supplied via <a href="https://github.com/oxinabox/DataDeps.jl">DataDeps.jl</a>. The first time you try accessing a dataset you will prompted to download the necessary files. For more information about options for where this data is saved and how it is loaded, see <a href="https://www.oxinabox.net/DataDeps.jl/stable/z10-for-end-users/" rel="nofollow">the documentation</a>.</p>
<p dir="auto">Note: for package writers using this for testing with CI- DataDeps.jl can be set to automatically download via the environment variable <code>DATADEPS_ALWAYS_ACCEPT=true</code>, which will need to be set in your CI environment.</p>

<p dir="auto">If you would like to add a dataset, please upload it to zenodo and then file a <a href="https://github.com/JuliaHCI/HCIDatasets.jl/pulls">pull request</a> with the corresponding code additions. The code behind HCIDatasets.jl is very simple: browse the <code>HR8799.jl</code> file to get an idea of the process.</p>
