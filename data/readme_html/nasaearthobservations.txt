<p>This is a simple package which allows download of data from NASA's Earth Observation program.</p>
<p>We reference this URL for the data, and you should examine the directory structure there to find datasets to download: <a href="https://neo.sci.gsfc.nasa.gov/archive" rel="nofollow">https://neo.sci.gsfc.nasa.gov/archive</a></p>

<p>The main function exported by this package is <code>observations(path::String)</code>.  Use it to download observations from folders in the NASA FTP site above.</p>
