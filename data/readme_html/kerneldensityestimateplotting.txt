<p dir="auto">This package provides the plotting functionality for the <a href="https://github.com/JuliaRobotics/KernelDensityEstimate.jl">KernelDensityEstimate.jl</a> package, and currently only supports the Gadfly back-end.</p>
<p dir="auto">This package is also used extensively by the <a href="https://github.com/dehann/RoMEPlotting.jl">RoMEPlotting.jl</a> package.</p>
