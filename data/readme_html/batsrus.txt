<p dir="auto">Fast <a href="https://github.com/MSTEM-QUDA/BATSRUS">BATSRUS</a>/<a href="https://github.com/MSTEM-QUDA/SWMF">SWMF</a> data reading, converting, and visualizing using Julia, successor of <a href="https://github.com/henry2004y/VisAnaMatlab">VisAnaMatlab</a>.</p>
<p dir="auto">This package provides the following functionalities:</p>
<ul dir="auto">
<li>simulation data reader</li>
<li>2D/3D region cut from the whole domain</li>
<li>interpolation from unstructured to structured data</li>
<li>data format conversion to VTK</li>
<li>simulation data visualization through multiple plotting libraries</li>
</ul>
<p dir="auto">For more details, please check the <a href="https://henry2004y.github.io/Batsrus.jl/dev" rel="nofollow">document</a>.</p>
