<p dir="auto">ContaminationPlots.jl is a <a href="https://github.com/zemjulia">ZEM</a> module.</p>
<p dir="auto">The modeule can be applied to create an empty plot with the background image without data; or plot data using linear interpolation, kriging, or inverse weighted distance; or plot matrix data.</p>
