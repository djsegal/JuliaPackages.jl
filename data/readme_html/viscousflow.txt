<p dir="auto">The purpose of this package is to enable easy setup and solution of viscous incompressible flows about bodies of various shapes. Documentation can be found at <a href="https://JuliaIBPM.github.io/ViscousFlow.jl/latest" rel="nofollow">https://JuliaIBPM.github.io/ViscousFlow.jl/latest</a>.</p>
<p dir="auto"><strong>ViscousFlow.jl</strong> is registered in the general Julia registry. To install, type
e.g.,</p>
<div class="highlight highlight-source-julia notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="] add ViscousFlow"><pre>] add ViscousFlow</pre></div>
<p dir="auto">Then, in any version, type</p>
<div class="highlight highlight-source-julia notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="julia&gt; using ViscousFlow"><pre>julia<span class="pl-k">&gt;</span> <span class="pl-k">using</span> ViscousFlow</pre></div>
<p dir="auto">For examples, consult the documentation or see the example Jupyter notebooks in the Examples folder.</p>
<p dir="auto"><a target="_blank" rel="noopener noreferrer" href="https://github.com/jdeldre/ViscousFlow.jl/raw/master/cylinderRe400.gif"><img src="https://github.com/jdeldre/ViscousFlow.jl/raw/master/cylinderRe400.gif" alt="" data-animated-image="" style="max-width: 100%;"></a></p>
<p dir="auto">Many of the core aspects of the fluid-body interaction are based on the Method of Immersed Layers [1], which is an extension of the immersed boundary projection method [2].</p>
<p dir="auto">[1]: Eldredge, J. D. (2021) "A method of immersed layers on Cartesian grids, with application to incompressible flows," <a href="https://arxiv.org/abs/2103.04521" rel="nofollow">arXiv:2103.04521</a>.</p>
<p dir="auto">[2]: Taira, K. and Colonius, T. (2007) "The immersed boundary method: a projection approach," <em>J. Comput. Phys.</em>, 225, 2118--2137.</p>
