<p dir="auto">Basic definitions for Engineering Thermodynamics in Julia.</p>
<h1 dir="auto"><a id="user-content-description" class="anchor" aria-hidden="true" href="#description"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path d="m7.775 3.275 1.25-1.25a3.5 3.5 0 1 1 4.95 4.95l-2.5 2.5a3.5 3.5 0 0 1-4.95 0 .751.751 0 0 1 .018-1.042.751.751 0 0 1 1.042-.018 1.998 1.998 0 0 0 2.83 0l2.5-2.5a2.002 2.002 0 0 0-2.83-2.83l-1.25 1.25a.751.751 0 0 1-1.042-.018.751.751 0 0 1-.018-1.042Zm-4.69 9.64a1.998 1.998 0 0 0 2.83 0l1.25-1.25a.751.751 0 0 1 1.042.018.751.751 0 0 1 .018 1.042l-1.25 1.25a3.5 3.5 0 1 1-4.95-4.95l2.5-2.5a3.5 3.5 0 0 1 4.95 0 .751.751 0 0 1-.018 1.042.751.751 0 0 1-1.042.018 1.998 1.998 0 0 0-2.83 0l-2.5 2.5a1.998 1.998 0 0 0 0 2.83Z"></path></svg></a>Description</h1>
<p dir="auto">The <code>EngThermBase.jl</code> package aims at providing a common platform for engineering
thermodynamics packages and case calculations by implementing:</p>
<ul dir="auto">
<li>Engineering thermodynamics quantities <strong>tagging</strong>, such as <code>P</code>, <code>T</code>, <code>v</code>, <code>u</code>, <code>h</code>, <code>s</code>,
etc...</li>
<li>Default (SI) <strong>units</strong> for tagged quantities—through
<a href="https://github.com/PainterQubits/Unitful.jl">Unitful.jl</a>;</li>
<li><strong>Uncertainty propagation</strong>—through
<a href="https://github.com/JuliaPhysics/Measurements.jl">Measurements.jl</a>.</li>
<li>Somewhat configurable thermodynamic amount <strong>pretty-printing</strong>, such as:
<ul dir="auto">
<li><code>P₆₄: 101.35 kPa</code>,</li>
<li><code>v₆₄: 1.1800 m³/kg</code>,</li>
<li><code>R̄₆₄: (8.3145 ± 1.5e-05 kJ/K/kmol)</code>,</li>
<li><code>Ma₆₄: 1.0333 –</code>,</li>
<li><code>ṁ₆₄: 3.4560 kg/s</code>, etc...</li>
</ul>
</li>
<li>Automatic <strong>re-tagging</strong>, through Julia's multiple dispatch system, such as:
<ul dir="auto">
<li><code>u + P * v --&gt; h</code>,</li>
<li><code>u - T * s --&gt; a</code>,</li>
<li><code>(P * v) / (R * T) --&gt; Z</code>, and the like;</li>
</ul>
</li>
<li>Thermodynamic <strong>bases</strong>, such as:
<ul dir="auto">
<li><code>MA</code> (mass),</li>
<li><code>MO</code> (molar), etc... ones;</li>
</ul>
</li>
<li>Automatic <strong>re-basing</strong>, such as:
<ul dir="auto">
<li><code>u * m --&gt; U</code>,</li>
<li><code>R̄ / M --&gt; R</code>,</li>
<li><code>ṁ * q --&gt; Q̇</code>, etc..., and</li>
</ul>
</li>
<li>Exports an <strong>abstract type hyerarchy</strong> so as to provide <strong>hooks</strong> for thermodynamic models of
heat capacity, pure substance (by equation of state, or EoS), mixtures, etc... such as the
<a href="https://github.com/JEngTherm/IdealGasLib.jl">IdealGasLib.jl</a>.</li>
</ul>
<p dir="auto">For additional information and examples, please refer to the package's documentation.</p>
