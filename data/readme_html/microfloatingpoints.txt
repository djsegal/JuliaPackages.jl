<div id="readme" class="md" data-path="README.md"><article class="markdown-body entry-content container-lg" itemprop="text"><h1><a id="user-content-microfloatingpoints" class="anchor" aria-hidden="true" href="#microfloatingpoints"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path fill-rule="evenodd" d="M7.775 3.275a.75.75 0 001.06 1.06l1.25-1.25a2 2 0 112.83 2.83l-2.5 2.5a2 2 0 01-2.83 0 .75.75 0 00-1.06 1.06 3.5 3.5 0 004.95 0l2.5-2.5a3.5 3.5 0 00-4.95-4.95l-1.25 1.25zm-4.69 9.64a2 2 0 010-2.83l2.5-2.5a2 2 0 012.83 0 .75.75 0 001.06-1.06 3.5 3.5 0 00-4.95 0l-2.5 2.5a3.5 3.5 0 004.95 4.95l1.25-1.25a.75.75 0 00-1.06-1.06l-1.25 1.25a2 2 0 01-2.83 0z"></path></svg></a>MicroFloatingPoints</h1>
<p><em>The MicroFloatingPoints package defines the <code>Floatmu</code> parametric type to manipulate <a href="https://en.wikipedia.org/wiki/IEEE_754" rel="nofollow">IEEE 754 standard</a>-compliant floating-point numbers with a very small size (from 5 bits up to 32 bits).</em></p>
<h3><a id="user-content-example-of-use" class="anchor" aria-hidden="true" href="#example-of-use"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path fill-rule="evenodd" d="M7.775 3.275a.75.75 0 001.06 1.06l1.25-1.25a2 2 0 112.83 2.83l-2.5 2.5a2 2 0 01-2.83 0 .75.75 0 00-1.06 1.06 3.5 3.5 0 004.95 0l2.5-2.5a3.5 3.5 0 00-4.95-4.95l-1.25 1.25zm-4.69 9.64a2 2 0 010-2.83l2.5-2.5a2 2 0 012.83 0 .75.75 0 001.06-1.06 3.5 3.5 0 00-4.95 0l-2.5 2.5a3.5 3.5 0 004.95 4.95l1.25-1.25a.75.75 0 00-1.06-1.06l-1.25 1.25a2 2 0 01-2.83 0z"></path></svg></a>Example of use</h3>
<p>Adding <code>0.25</code> and <code>2.0</code> in a floating-point format with 1 bit for the sign (implied), 2 bits for the exponent, and 2 bits for the fractional part (3 bits for the whole significand with the <em>hidden bit</em>), and testing whether the result had to be rounded:</p>
<div class="highlight highlight-source-julia position-relative" data-snippet-clipboard-copy-content="julia&gt; using MicroFloatingPoints

julia&gt; Floatmu{2,2}(0.25)+Floatmu{2,2}(2.0)
2.0

julia&gt; inexact()
true
"><pre>julia<span class="pl-k">&gt;</span> <span class="pl-k">using</span> MicroFloatingPoints

julia<span class="pl-k">&gt;</span> <span class="pl-c1">Floatmu</span><span class="pl-c1">{2,2}</span>(<span class="pl-c1">0.25</span>)<span class="pl-k">+</span><span class="pl-c1">Floatmu</span><span class="pl-c1">{2,2}</span>(<span class="pl-c1">2.0</span>)
<span class="pl-c1">2.0</span>

julia<span class="pl-k">&gt;</span> <span class="pl-c1">inexact</span>()
<span class="pl-c1">true</span></pre></div>

<p>The package can be installed with the Julia package manager. From the Julia REPL, type <code>]</code> to enter the Pkg REPL mode and run:</p>
<div class="highlight highlight-source-julia position-relative" data-snippet-clipboard-copy-content="pkg&gt; add MicroFloatingPoints
"><pre>pkg<span class="pl-k">&gt;</span> add MicroFloatingPoints</pre></div>
<p>Or, equivalently, via the Pkg API:</p>
<div class="highlight highlight-source-julia position-relative" data-snippet-clipboard-copy-content="julia&gt; import Pkg; Pkg.add(&quot;MicroFloatingPoints&quot;)
"><pre>julia<span class="pl-k">&gt;</span> <span class="pl-k">import</span> Pkg; Pkg<span class="pl-k">.</span><span class="pl-c1">add</span>(<span class="pl-s"><span class="pl-pds">"</span>MicroFloatingPoints<span class="pl-pds">"</span></span>)</pre></div>
<p>Note that the <code>matplotlib</code> Python package must be available through <code>PyCall</code>.</p>
