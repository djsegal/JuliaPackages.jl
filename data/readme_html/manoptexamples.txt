<p dir="auto">This package provides examples of optimizations problems on Riemannian manifolds.</p>
<p dir="auto">It uses the manifolds from <a href="https://juliamanifolds.github.io/Manifolds.jl/" rel="nofollow">Manifolds.jl</a> and the structures provided by <a href="https://manoptjl.org/" rel="nofollow">Manopt.jl</a>.
to state the problems.
Furthermore the problems are illustrated in their usage in <a href="https://quarto.org" rel="nofollow">Quarto</a> Markdown files, that
are pre-rendered into the documentation. This way, the package also provides examples of
the provided examples “in action”.</p>
