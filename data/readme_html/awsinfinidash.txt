<p dir="auto">AWS Infinidash Interface for Julia.</p>
<p dir="auto">This package wraps up the standard Infinidash SDK from <a href="https://github.com/JuliaCloud/AWS.jl">AWS.jl</a> in to a more idiomatic Julia interface.</p>
<p dir="auto">Right now it is still very bare bones.
<strong>PRs welcome!</strong></p>
