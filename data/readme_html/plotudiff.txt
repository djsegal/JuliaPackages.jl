<p dir="auto">This Julia package implements a set of diffing utilities to more easily <em>visualize differences</em> between two inputs. Currently, this package focuses on differences between strings but the goal is to expand to all kind of Julia objects like dictionnaries, images and more in a composable manner.</p>
