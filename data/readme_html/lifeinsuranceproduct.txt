<p dir="auto">This project is by now just a pilot to investigate possible interface designs for
LifeInsuranceProducts that should be pluggable into <a href="https://github.com/Actuarial-Sciences-for-Africa-ASA/LifeInsuranceContracts.jl">our project LifeInsuranceContracts</a></p>
<p dir="auto">It provides a very simple calculator API example providing</p>
<ul dir="auto">
<li>functions leveraging <a href="https://github.com/JuliaActuary/LifeContingencies.jl">the LifeContingencies library</a>
comprising computation of
<ul dir="auto">
<li>present value and</li>
<li>net premium with varying payment frequencies where applicable for</li>
<li>annuity, lifelong and temporary</li>
<li>Life Risk</li>
<li>Two Life Risk</li>
</ul>
</li>
<li>as well as function meta data in JSON format <a href="https://github.com/Actuarial-Sciences-for-Africa-ASA/GenieBuiltLifeProto">usable for calling these functions from HTML forms in a life insurance web app</a></li>
</ul>
<p dir="auto"><a href="src/LifeInsuranceProduct.jl">LifeInsuranceProduct</a> bundles and dispatches calls for tariff metadata and computations as of now to
- <a href="src/SingleLifeRiskTariff.jl">SingleLifeRiskTariff</a>
- <a href="src/JointLifeRiskTariff.jl">JointLifeRiskTariff</a>
- <a href="src/PensionTariff.jl">PensionTariff</a>
- <a href="src/ProfitParticipationTariff.jl">ProfitParticipationTariff</a></p>
<p dir="auto">To explore usage of this package
use the debugger on these scripts:</p>
<ul dir="auto">
<li><a href="debugcalcSLR.jl">SingleLifeRiskTariff</a></li>
<li><a href="debugcalcSLR.jl">JointLifeRiskTariff</a></li>
<li><a href="debugcalcPEN.jl">PensionTariff</a></li>
</ul>
