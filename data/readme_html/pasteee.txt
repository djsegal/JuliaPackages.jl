<p dir="auto"><a href="https://paste.ee/" rel="nofollow">Paste.ee</a> is a free version of <a href="https://pastebin.com/" rel="nofollow">Pastebin</a> with SSL, IPv6, and an easy to use API.
This package provides a Julia wrapper around the Paste.ee API (see <a href="https://pastee.github.io/docs/" rel="nofollow">https://pastee.github.io/docs/</a>).</p>
