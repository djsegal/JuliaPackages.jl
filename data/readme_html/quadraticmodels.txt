<p>A package to model linear and quadratic optimization problems using the <a href="https://github.com/JuliaSmoothOptimizers/NLPModels.jl">NLPModels.jl</a> data structures.</p>
<p>The problems represented have the form</p>
<p align="center">
optimize   c₀ + cᵀ x + ½ xᵀ Q x
  
subject to   L ≤ Ax ≤ U and ℓ ≤ x ≤ u,
</p>
<p>where the square symmetric matrix Q is zero for linear optimization problems.</p>
