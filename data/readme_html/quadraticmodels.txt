<p dir="auto">A package to model linear and quadratic optimization problems using the <a href="https://github.com/JuliaSmoothOptimizers/NLPModels.jl">NLPModels.jl</a> data structures.</p>
<p dir="auto">The problems represented have the form</p>
<p align="center" dir="auto">
optimize   c₀ + cᵀ x + ½ xᵀ Q x
  
subject to   L ≤ Ax ≤ U and ℓ ≤ x ≤ u,
</p>
<p dir="auto">where the square symmetric matrix Q is zero for linear optimization problems.</p>
