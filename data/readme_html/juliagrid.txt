<p dir="auto"><a href="https://mcosovic.github.io/JuliaGrid.jl/stable/" rel="nofollow"><img src="https://github.com/mcosovic/JuliaGrid.jl/workflows/Documentation/badge.svg" alt="Documentation" style="max-width: 100%;"></a> <a target="_blank" rel="noopener noreferrer" href="https://github.com/mcosovic/JuliaGrid.jl/workflows/Build/badge.svg"><img src="https://github.com/mcosovic/JuliaGrid.jl/workflows/Build/badge.svg" alt="Build" style="max-width: 100%;"></a></p>
<p dir="auto"><a href="https://mcosovic.github.io/JuliaGrid.jl/stable/" rel="nofollow"><img align="right" width="145" src="/docs/src/assets/logo2.png" style="max-width: 100%;"></a></p>
<p dir="auto">JuliaGrid is an open-source, easy-to-use simulation tool/solver for researchers and educators provided as a Julia package, with source code released under MIT License. JuliaGrid is inspired by the Matpower, an open-source steady-state power system solver, and allows a variety of display and manipulation options.</p>
<p dir="auto">We have tested and verified simulation tool using different scenarios to the best of our ability. As a user of this simulation tool, you can help us to improve future versions, we highly appreciate your feedback about any errors, inaccuracies, and bugs. For more information, please visit <a href="https://mcosovic.github.io/JuliaGrid.jl/stable/" rel="nofollow">documentation</a> site.</p>
<p dir="auto">The software package provides the solution of the AC and DC power flow and optimal power flow, non-linear and DC state estimation, as well as the state estimation with PMUs, with standalone measurement generator.</p>
