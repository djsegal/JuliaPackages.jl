<p dir="auto">Experimental forward-mode AD library for tensor networks.</p>
<p dir="auto">Hadagote = Handai + Zygote. This is because the library was created
when the author was in Handai (Osaka University) in Japan.</p>
<p dir="auto">Compatible with OMEinsum.jl, Zygote.jl, etc.
The implementation is following HyperDualNumbers.jl.
<a href="https://github.com/JuliaDiff/HyperDualNumbers.jl">https://github.com/JuliaDiff/HyperDualNumbers.jl</a></p>

<p dir="auto">In order to avoid type piracy, the functionality of OMEinsum.jl is duplicated.
Currently only supports the <code>@dualein_str</code> notation <code>dualein"ij, jk, klm -&gt; ilm"(A, B, C)</code>,
and <code>@hyperdualein_str</code> notation <code>hyperdualein"ij, jk, klm -&gt; ilm"(A, B, C)</code>.</p>
