<p dir="auto">Fast implementations of root finding algorithms in Julia that satisfy the SciML common interface.
SimpleNonlinearSolve.jl focuses on low-dependency implementations of very fast methods for
very small and simple problems. For the full set of solvers, see
<a href="https://github.com/SciML/NonlinearSolve.jl">NonlinearSolve.jl</a>, of which
SimpleNonlinearSolve.jl is just one solver set.</p>
<p dir="auto">For information on using the package,
<a href="https://docs.sciml.ai/NonlinearSolve/stable/" rel="nofollow">see the stable documentation</a>. Use the
<a href="https://docs.sciml.ai/NonlinearSolve/dev/" rel="nofollow">in-development documentation</a> for the version of
the documentation which contains the unreleased features.</p>
