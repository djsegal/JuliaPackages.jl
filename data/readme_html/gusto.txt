<p>This is a Julia suite for trajectory optimization using the Guaranteed Sequential Trajectory Optimization (GuSTO) framework. Details can be found in <a href="https://arxiv.org/abs/1903.00155" rel="nofollow">this paper</a>.</p>
<p>GuSTO.jl runs on julia v1.X, although an older version running on <a href="https://julialang.org/downloads/oldreleases.html" rel="nofollow">julia v0.6.4</a> can be found in the julia-v0.6 branch.</p>
<p>Also required are the <a href="https://github.com/StanfordASL/BulletCollision.jl">BulletCollision.jl</a> and <a href="https://github.com/StanfordASL/AstrobeeRobot.jl">AstrobeeRobot.jl</a> packages. GuSTO.jl performs optimization through the <a href="https://github.com/JuliaOpt/JuMP.jl">JuMP.jl</a> interface, and Gurobi and Ipopt are currently used in examples.</p>
