<p dir="auto">The Networks package is a library for working with Graphs in Julia. A Networks.jl <code>Graph</code> is a datastructure where the nodes and edges can be arbitrary Julia types, with the only restriction that nodes should be hashable. This allows the user to use meaningful types as nodes and edges. The design is borrowed from the Python NetworkX package</p>

<p dir="auto">The <code>Graph</code> type is used to represent undirected graphs. It has one field, named <code>adj</code>, which is of type <code>Dict{Any, Dict{Any, Any}}</code>. The keys of <code>adj</code> correspond to the nodes of the graph (this is why node objects should be hashable), and the values consist of other dictionaries containing the incident node objects (as keys) and edge objects (as values). If <code>g</code> is a graph containing both nodes <code>u</code> and <code>v</code>, then we always have that <code>g[u][v]</code> is equal to <code>g[v][u]</code>.</p>
