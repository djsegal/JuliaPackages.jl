<p dir="auto">A Julia module for working with an editorial repository organized following HCMID conventions.</p>
<p dir="auto">This package provides most of the functionality used in HC MID's Pluto notebook for validating editorial work.  You can find documentation and a template editorial repository including the current release version of the Pluto notebook for validation at <a href="https://github.com/HCMID/validatormodel">https://github.com/HCMID/validatormodel</a>.</p>
