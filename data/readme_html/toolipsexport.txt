This module allows you to compile <a href="https://github.com/ChifiSource/Toolips.jl">toolips</a> apps into executable files and shared libraries. Future plans are also to port this to ARM64 and mobile. Currently available exports are:</li>
<li><strong>so</strong>: A shared library</li>
<li><strong>app</strong>: An Electron-based application</li>
<li><strong>server</strong>: An executable server</li>
</ul>
