<p dir="auto"><a target="_blank" rel="noopener noreferrer" href="docs/src/figures/banner/banner.svg"><img src="docs/src/figures/banner/banner.svg" alt="APS" style="max-width: 100%;"></a></p>
<p dir="auto"><strong>MakiePublication.jl</strong> is a Julia package for producing publication quality figures based on <a href="https://github.com/JuliaPlots/Makie.jl">Makie.jl</a>. It aims to provide equivalent functionalities as the Python package <a href="https://github.com/liuyxpp/mpltex">mpltex</a>.</p>
<p dir="auto">Read the <a href="http://www.yxliu.group/MakiePublication.jl/dev/" rel="nofollow">full documentation here</a>.</p>

<ul dir="auto">
<li>Provide a collection of custom themes for journal publishers: ACS, APS, RSC.</li>
<li>Custom theme for making figures suitable for web pages.</li>
<li>15 color palettes based on well-known quality color schemes with special tweaked ordering for scientific publishing. (since v0.3.0)</li>
<li>Support hollow markers. (since v0.3.1)</li>
</ul>

<p dir="auto">MakiePublication provides a number of <code>theme_*</code> methods to generate themes (<code>Makie.Theme</code> instances) which can be given to <a href="https://docs.makie.org/stable/documentation/theming/index.html" rel="nofollow">Makie's theming methods</a>: <code>set_theme!</code>, <code>update_theme!</code>, and <code>with_theme</code>. A typical usage pattern is</p>
<div class="highlight highlight-source-julia notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="using MakiePublication
using CairoMakie

with_theme(theme_web()) do
    lines(0..10, sin)
    lines!(0..10, cos)
    current_figure()
end"><pre><span class="pl-k">using</span> MakiePublication
<span class="pl-k">using</span> CairoMakie

<span class="pl-c1">with_theme</span>(<span class="pl-c1">theme_web</span>()) <span class="pl-k">do</span>
    <span class="pl-c1">lines</span>(<span class="pl-c1">0</span><span class="pl-k">..</span><span class="pl-c1">10</span>, sin)
    <span class="pl-c1">lines!</span>(<span class="pl-c1">0</span><span class="pl-k">..</span><span class="pl-c1">10</span>, cos)
    <span class="pl-c1">current_figure</span>()
<span class="pl-k">end</span></pre></div>
<p dir="auto">Or if you have a more complicated plotting function,</p>

<h2 dir="auto"><a id="user-content-colors" class="anchor" aria-hidden="true" href="#colors"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path d="m7.775 3.275 1.25-1.25a3.5 3.5 0 1 1 4.95 4.95l-2.5 2.5a3.5 3.5 0 0 1-4.95 0 .751.751 0 0 1 .018-1.042.751.751 0 0 1 1.042-.018 1.998 1.998 0 0 0 2.83 0l2.5-2.5a2.002 2.002 0 0 0-2.83-2.83l-1.25 1.25a.751.751 0 0 1-1.042-.018.751.751 0 0 1-.018-1.042Zm-4.69 9.64a1.998 1.998 0 0 0 2.83 0l1.25-1.25a.751.751 0 0 1 1.042.018.751.751 0 0 1 .018 1.042l-1.25 1.25a3.5 3.5 0 1 1-4.95-4.95l2.5-2.5a3.5 3.5 0 0 1 4.95 0 .751.751 0 0 1-.018 1.042.751.751 0 0 1-1.042.018 1.998 1.998 0 0 0-2.83 0l-2.5 2.5a1.998 1.998 0 0 0 0 2.83Z"></path></svg></a>Colors</h2>
<p dir="auto">The default color cycle is 10-color <a href="https://juliagraphics.github.io/ColorSchemes.jl/stable/catalogue/#Seaborn" rel="nofollow">seanborn_deep</a> (since v0.2.3). MakiePublication also provides 15 carefully chosen color palettes based on well-known palettes, such as <code>tab10</code>, <code>tableau_10</code>, <code>Makie.wong_colors</code>, <code>okabe_ito</code>, etc. The colors in each palette has been reordered if necessary to be consistent with the order of the default matplotlib color palette, <code>tab10</code>. The order is: blue, orange, green, red, purple, brown, pink, gray, yellow, and cyan. You can choose it from the array <code>MakiePublication.COLORS</code> and use it like this:</p>
<div class="highlight highlight-source-julia notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="with_theme(myplot, theme_web(colors=MakiePublication.COLORS[5]))"><pre><span class="pl-c1">with_theme</span>(myplot, <span class="pl-c1">theme_web</span>(colors<span class="pl-k">=</span>MakiePublication<span class="pl-k">.</span>COLORS[<span class="pl-c1">5</span>]))</pre></div>
<p dir="auto">Or obtain the color palette by calling a function:</p>
<div class="highlight highlight-source-julia notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="with_theme(myplot, theme_web(colors=MakiePublication.tol_bright()))"><pre><span class="pl-c1">with_theme</span>(myplot, <span class="pl-c1">theme_web</span>(colors<span class="pl-k">=</span>MakiePublication<span class="pl-k">.</span><span class="pl-c1">tol_bright</span>()))</pre></div>
<p dir="auto">The demonstration of available color palettes can be found in the Pluto notebook <code>doc/makiepub_colors.jl.html</code>.</p>
