<p>This <a href="http://julialang.org" rel="nofollow">Julia</a> package allows to wrap long lines of text to fit within a given width.</p>
<h3><a id="user-content-quick-example" class="anchor" aria-hidden="true" href="#quick-example"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path fill-rule="evenodd" d="M7.775 3.275a.75.75 0 001.06 1.06l1.25-1.25a2 2 0 112.83 2.83l-2.5 2.5a2 2 0 01-2.83 0 .75.75 0 00-1.06 1.06 3.5 3.5 0 004.95 0l2.5-2.5a3.5 3.5 0 00-4.95-4.95l-1.25 1.25zm-4.69 9.64a2 2 0 010-2.83l2.5-2.5a2 2 0 012.83 0 .75.75 0 001.06-1.06 3.5 3.5 0 00-4.95 0l-2.5 2.5a3.5 3.5 0 004.95 4.95l1.25-1.25a.75.75 0 00-1.06-1.06l-1.25 1.25a2 2 0 01-2.83 0z"></path></svg></a>Quick example</h3>
<pre><code>julia&gt; text = "This text is going to be wrapped around in lines no longer than 20 characters.";

julia&gt; println_wrapped(text, width=20)
This text is going
to be wrapped around
in lines no longer
than 20 characters.
</code></pre>
<p>The other exported functions are <code>wrap</code> (returns a string) and <code>print_wrapped</code>.
See the documentation for more advanced settings.</p>
<h3><a id="user-content-installation" class="anchor" aria-hidden="true" href="#installation"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path fill-rule="evenodd" d="M7.775 3.275a.75.75 0 001.06 1.06l1.25-1.25a2 2 0 112.83 2.83l-2.5 2.5a2 2 0 01-2.83 0 .75.75 0 00-1.06 1.06 3.5 3.5 0 004.95 0l2.5-2.5a3.5 3.5 0 00-4.95-4.95l-1.25 1.25zm-4.69 9.64a2 2 0 010-2.83l2.5-2.5a2 2 0 012.83 0 .75.75 0 001.06-1.06 3.5 3.5 0 00-4.95 0l-2.5 2.5a3.5 3.5 0 004.95 4.95l1.25-1.25a.75.75 0 00-1.06-1.06l-1.25 1.25a2 2 0 01-2.83 0z"></path></svg></a>Installation</h3>
<p>To install the module, use Julia's package manager: start pkg mode by pressing <kbd>]</kbd> and then enter:</p>
<pre><code>(v1.3) pkg&gt; add TextWrap
</code></pre>
<p>Dependencies will be installed automatically.
The module can then be loaded like any other Julia module:</p>
<pre><code>julia&gt; using TextWrap
</code></pre>
