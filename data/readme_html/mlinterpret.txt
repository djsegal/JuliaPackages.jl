<p dir="auto">You can interpret any machine learning model from Python which has a property <code>.predict</code> by calling</p>
<div class="highlight highlight-source-julia notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="interpret(model, X, y)"><pre><span class="pl-c1">interpret</span>(model, X, y)</pre></div>
<p dir="auto">If your model dosen't have a property '.predict' (like Julia models), you can still interpret its predictions by</p>
<div class="highlight highlight-source-julia notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="ŷ = model.predict(X)
interpret(X, ŷ)"><pre>ŷ <span class="pl-k">=</span> model<span class="pl-k">.</span><span class="pl-c1">predict</span>(X)
<span class="pl-c1">interpret</span>(X, ŷ)</pre></div>
