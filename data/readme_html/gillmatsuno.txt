<h2><a id="user-content-introduction" class="anchor" aria-hidden="true" href="#introduction"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path fill-rule="evenodd" d="M4 9h1v1H4c-1.5 0-3-1.69-3-3.5S2.55 3 4 3h4c1.45 0 3 1.69 3 3.5 0 1.41-.91 2.72-2 3.25V8.59c.58-.45 1-1.27 1-2.09C10 5.22 8.98 4 8 4H4c-.98 0-2 1.22-2 2.5S3 9 4 9zm9-3h-1v1h1c1 0 2 1.22 2 2.5S13.98 12 13 12H9c-.98 0-2-1.22-2-2.5 0-.83.42-1.64 1-2.09V6.25c-1.09.53-2 1.84-2 3.25C6 11.31 7.55 13 9 13h4c1.45 0 3-1.69 3-3.5S14.5 6 13 6z"></path></svg></a><strong>Introduction</strong></h2>
<p><code>GillMatsuno.jl</code> is a Julia package that:</p>
<ul>
<li>numerically solves the Shallow-Water Equations on a $\beta$-plane</li>
<li>allows the user to define custom heat-forcing $Q$</li>
</ul>
<p><code>GillMatsuno.jl</code> can be installed via</p>
<pre><code>] add GillMatsuno
</code></pre>
<h2><a id="user-content-equations" class="anchor" aria-hidden="true" href="#equations"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path fill-rule="evenodd" d="M4 9h1v1H4c-1.5 0-3-1.69-3-3.5S2.55 3 4 3h4c1.45 0 3 1.69 3 3.5 0 1.41-.91 2.72-2 3.25V8.59c.58-.45 1-1.27 1-2.09C10 5.22 8.98 4 8 4H4c-.98 0-2 1.22-2 2.5S3 9 4 9zm9-3h-1v1h1c1 0 2 1.22 2 2.5S13.98 12 13 12H9c-.98 0-2-1.22-2-2.5 0-.83.42-1.64 1-2.09V6.25c-1.09.53-2 1.84-2 3.25C6 11.31 7.55 13 9 13h4c1.45 0 3-1.69 3-3.5S14.5 6 13 6z"></path></svg></a>Equations</h2>
<p>The Shallow-Water Equations are as described by <em>Vallis [2018]</em> (Atmospheric and Oceanic Fluid Dynamics, 2nd Edition):</p>
<pre><code>∂u/∂t + αu + ∂ϕ/∂x - βyv = 0
∂v/∂t + αu + ∂ϕ/∂y + βyu = 0
∂ϕ/∂t + αϕ + gH(∂u/∂x + ∂v/∂y) = -Q
</code></pre>
<p>We solve the equations numerically using the Finite-Difference Method.  The backend solvers can be found in <code>src/numdefault.jl</code>, but this README will not go through the details.</p>
<h2><a id="user-content-running-the-gill-matsuno-numerical-model" class="anchor" aria-hidden="true" href="#running-the-gill-matsuno-numerical-model"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path fill-rule="evenodd" d="M4 9h1v1H4c-1.5 0-3-1.69-3-3.5S2.55 3 4 3h4c1.45 0 3 1.69 3 3.5 0 1.41-.91 2.72-2 3.25V8.59c.58-.45 1-1.27 1-2.09C10 5.22 8.98 4 8 4H4c-.98 0-2 1.22-2 2.5S3 9 4 9zm9-3h-1v1h1c1 0 2 1.22 2 2.5S13.98 12 13 12H9c-.98 0-2-1.22-2-2.5 0-.83.42-1.64 1-2.09V6.25c-1.09.53-2 1.84-2 3.25C6 11.31 7.55 13 9 13h4c1.45 0 3-1.69 3-3.5S14.5 6 13 6z"></path></svg></a>Running the Gill-Matsuno Numerical Model</h2>
<p>The numerical solution to the Gill-Matusno Model is called using <code>GMcalc()</code>.  This function accepts the following keyword arguments as modifications to the initial model:</p>
<ul>
<li><code>xmin</code> and <code>xmax</code> define the left and right boundaries of the domain</li>
<li><code>ymin</code> and <code>ymax</code> define the lower and upper boundaries of the domain</li>
<li><code>δx</code> and <code>δy</code> define the grid-spacing in the x- and y- directions respectively</li>
<li><code>nt</code> and <code>δt</code> define the number of timesteps, and the time between each step, respectively</li>
<li><code>A</code> and <code>L</code> represent the amplitude and length of the heat-forcing <code>Q</code></li>
<li><code>α</code> represents the damping coefficient on the winds induced by the heat-forcing (default: <code>α = 0.1</code>)</li>
<li><code>β</code> is the Coriolis Factor (nondimensionalized to <code>β = 0.5</code> as the default)</li>
<li><code>g</code> and <code>H</code> represent gravity and the height of the domain (both nondimensionalized to 1 as default)</li>
</ul>
<p>An example is given below, with the default values as follows:</p>
<pre><code>GMcalc(xmin=-25.0,xmax=50.0,δx=0.1,ymin=-10.0,ymax=10.0,δy=0.1,nt=5000,δt=0.001,
	   A=1,L=2,α=0.1,β=0.5,g=1.0,H=1.0)
</code></pre>
