<p>This package is a growing collection of Lie Group/Algebra, Quaternion, Euler, AxisAngle representations. Including convert functions between each, and overloading operators for those sets. The package already includes exponential, logarithm maps and Jacobians for Lie SO(3). SE(3) mostly complete.</p>
<h3><a id="user-content-interesting-usage" class="anchor" aria-hidden="true" href="#interesting-usage"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path fill-rule="evenodd" d="M7.775 3.275a.75.75 0 001.06 1.06l1.25-1.25a2 2 0 112.83 2.83l-2.5 2.5a2 2 0 01-2.83 0 .75.75 0 00-1.06 1.06 3.5 3.5 0 004.95 0l2.5-2.5a3.5 3.5 0 00-4.95-4.95l-1.25 1.25zm-4.69 9.64a2 2 0 010-2.83l2.5-2.5a2 2 0 012.83 0 .75.75 0 001.06-1.06 3.5 3.5 0 00-4.95 0l-2.5 2.5a3.5 3.5 0 004.95 4.95l1.25-1.25a.75.75 0 00-1.06-1.06l-1.25 1.25a2 2 0 01-2.83 0z"></path></svg></a>Interesting usage</h3>
<p>Supports mangle products, for example (using identity constructors):</p>
<pre><code>julia&gt; pp = SO3(0) * Quaternion(0) * so3(0.1*randn(3)) * AngleAxis(0)
</code></pre>
<p>Or maybe you want to compare against another rotation</p>
<pre><code>julia&gt; compare(SO3(0), pp) # returns true or false
</code></pre>
