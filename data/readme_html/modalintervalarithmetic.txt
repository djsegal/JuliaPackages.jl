<p>Instantiate modal intervals of the form <code>ModalInterval{T&lt;:Real} &lt;: Number</code> using the following constructors</p>
<p><code>ModalInterval(a::Real, b::Real)</code> constructs a modal interval <code>[a, b]</code><br>
<code>ModalInterval{T}(a::Real)</code> constructs a modal interval <code>[a, a]</code> representing a real number, cast to type <code>T</code><br>
<code>ModalInterval(a::Real)</code> constructs a modal interval <code>[a, a]</code> representing a real number<br>
<code>ModalInterval(X::Interval)</code> constructs a proper modal interval from classical interval <code>X</code><br>
<code>ModalInterval{T}(X::Interval)</code> constructs a proper modal interval from classical interval <code>X</code>, with endpoints cast to type <code>T</code><br>
<code>ModalInterval{T}(X::ModalInterval)</code></p>
<p>Julia operations are overloaded to support <code>ModalInterval</code>.</p>
