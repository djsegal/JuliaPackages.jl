<p dir="auto">Julia package for nuclear structure calculations covering:</p>
<ul dir="auto">
<li>generating Chiral EFT interactions</li>
<li>many-body calculations (HFMBPT, IMSRG/VS-IMSRG, valence shell-model, etc.)</li>
</ul>
<p dir="auto">Note: Of course, 'for structural calculations' simply means that the author (SY) is not familiar with reaction theories.
Contributions and suggestions from reaction theory and experimental researchers are very welcome. Thanks.</p>

<p dir="auto">Assuming that you have already installed Julia (v&gt;=1.7.0),</p>
<div class="snippet-clipboard-content notranslate position-relative overflow-auto" data-snippet-clipboard-copy-content="julia&gt;using Pkg; Pkg.add(&quot;NuclearToolkit&quot;)"><pre lang="jldoctest" class="notranslate"><code>julia&gt;using Pkg; Pkg.add("NuclearToolkit")
</code></pre></div>
<p dir="auto">Pkg (Julia's builtin package manager) intalls packages in <code>$JULIA_DEPOT_PATH</code>, which is by default <code>~/.julia</code>.<br>
When working on a working node (w/o permissions to access <code>~/</code>), overwrite the <code>JULIA_DEPOT_PATH</code> by <code>export JULIA_DEPOT_PATH="PATH_TO_JULIA_DEPOT"</code>.</p>

<p dir="auto">Execute <code>example/sample_script.jl</code> like</p>
<div class="snippet-clipboard-content notranslate position-relative overflow-auto" data-snippet-clipboard-copy-content="julia -t 10 example/sample_script.jl"><pre class="notranslate"><code>julia -t 10 example/sample_script.jl
</code></pre></div>
<p dir="auto">This sample script performs:</p>
<ul dir="auto">
<li>generating NN potential with hw=20, emax=4</li>
<li>HFMBPT(3) and IMSRG/VS-IMSRG calculation using the NN potential</li>
<li>shell-model calculation using the effective interaction derived by VS-IMSRG</li>
</ul>
