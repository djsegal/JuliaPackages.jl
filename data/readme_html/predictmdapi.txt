<p>The PredictMDAPI package provides the abstract types, traits, and functions that define the <a href="https://predictmd.net" rel="nofollow">PredictMD</a> application programming interface (API).</p>
<p>This is a very lightweight package. It has no dependencies. The only lines of code are:</p>
<ul>
<li>Abstract types, i.e. <code>abstract type Foo end</code></li>
<li>Immutable composite types with no fields, i.e. <code>struct Bar &lt;: Foo end</code></li>
<li>Function stubs, i.e. <code>function hello end</code></li>
</ul>
