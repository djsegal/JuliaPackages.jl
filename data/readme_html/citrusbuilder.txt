<p dir="auto">This Julia package provides utilities to construct surveys that can be uploaded to a running <a href="https://www.limesurvey.org/" rel="nofollow">LimeSurvey</a> server.</p>
<h2 dir="auto"><a id="user-content-getting-started" class="anchor" aria-hidden="true" href="#getting-started"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path d="m7.775 3.275 1.25-1.25a3.5 3.5 0 1 1 4.95 4.95l-2.5 2.5a3.5 3.5 0 0 1-4.95 0 .751.751 0 0 1 .018-1.042.751.751 0 0 1 1.042-.018 1.998 1.998 0 0 0 2.83 0l2.5-2.5a2.002 2.002 0 0 0-2.83-2.83l-1.25 1.25a.751.751 0 0 1-1.042-.018.751.751 0 0 1-.018-1.042Zm-4.69 9.64a1.998 1.998 0 0 0 2.83 0l1.25-1.25a.751.751 0 0 1 1.042.018.751.751 0 0 1 .018 1.042l-1.25 1.25a3.5 3.5 0 1 1-4.95-4.95l2.5-2.5a3.5 3.5 0 0 1 4.95 0 .751.751 0 0 1-.018 1.042.751.751 0 0 1-1.042.018 1.998 1.998 0 0 0-2.83 0l-2.5 2.5a1.998 1.998 0 0 0 0 2.83Z"></path></svg></a>Getting started</h2>
<p dir="auto">A minimal survey must contain a survey id and a title.</p>

<p dir="auto">To add question groups and questions to a survey the <code>do ... end</code> syntax can be used. Note that LimeSurvey requires that questions must be nested within question groups.</p>
<p dir="auto">If we want to create a basic survey asking for the name (using a short text question) and gender of the survey participants (using a dropdown select) we can use the following constructor,</p>
<div class="highlight highlight-source-julia notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="gender_options = response_scale([
    response_option(&quot;f&quot;, &quot;female&quot;),
    response_option(&quot;m&quot;, &quot;male&quot;)
])

    question_group(1, &quot;Basic participant information&quot;) do
        short_text_question(&quot;name&quot;, &quot;Please state your full name.&quot;, mandatory=true),
        dropdown_list_question(&quot;gender&quot;, &quot;Please select a gender.&quot;, gender_options, other=true, mandatory=true)
    end
end"><pre>gender_options <span class="pl-k">=</span> <span class="pl-c1">response_scale</span>([

<p dir="auto">To export your survey simply call <code>write</code>,</p>
<div class="highlight highlight-source-julia notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="write(&quot;my_basic_survey.lss&quot;, basic_survey)"><pre><span class="pl-c1">write</span>(<span class="pl-s"><span class="pl-pds">"</span>my_basic_survey.lss<span class="pl-pds">"</span></span>, basic_survey)</pre></div>
<p dir="auto">The resulting xml file can be imported on the server using the <a href="https://manual.limesurvey.org/Surveys_-_introduction#Import_a_survey" rel="nofollow">LimeSurvey import function</a> or <a href="https://github.com/p-gw/CitrusAPI.jl">CitrusAPI.jl</a>.</p>
<p dir="auto">For more details on how to use this package, please refer to the <a href="https://p-gw.github.io/CitrusBuilder.jl/dev" rel="nofollow">documentation</a>.</p>
