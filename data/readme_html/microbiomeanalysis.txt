<p dir="auto">This package provides common methods for microbiome analysis based on the <a href="https://ltla.github.io/SummarizedExperiments.jl/dev/" rel="nofollow">SummarizedExperiment</a> and <a href="https://ltla.github.io/MultiAssayExperiments.jl/dev/" rel="nofollow">MultiAssayExperiment</a> data containers. This work in Julia language reflects <a href="https://github.com/microbiome/mia/">related developments in R</a> and has its own presentation at <a href="https://www.youtube.com/watch?v=3PYFqwEQAb8&amp;t=158s" rel="nofollow">JuliaCon 2022</a>.</p>
