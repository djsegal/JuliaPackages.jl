<p dir="auto">Wastime exposes two C api. The first one is the common Web Assembly runtime C api with names starting with <code>Wasm</code> in Wasmtime.jl.
The second one is a wasmtime specific api which provides more functionality like WASI imports, fine-grained configuration
of the store features like fuel.</p>
<p dir="auto">See the <a href="https://github.com/Pangoraw/Wasmtime.jl/tree/main/test"><code>test</code> folder</a> for usage examples.</p>
