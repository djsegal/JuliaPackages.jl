<p dir="auto">This is a julia library which provides an interface between <strong>Singular</strong>
(<a href="https://github.com/Singular/Singular">https://github.com/Singular/Singular</a>) and <strong>gb</strong> (<a href="https://github.com/ederc/gb">https://github.com/ederc/gb</a>).</p>
<p dir="auto">It depends on <strong>Singular.jl</strong> (<a href="https://github.com/oscar-system/Singular.jl">https://github.com/oscar-system/Singular.jl</a>) for using
Singular in julia.</p>
<p dir="auto">The package also provides a benchmark suite for various different Groebner basis
computations, see <strong>Benchmarks.jl</strong>.</p>
