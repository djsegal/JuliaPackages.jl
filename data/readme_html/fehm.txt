<p><a href="http://madsjulia.github.io/FEHM.jl" rel="nofollow">FEHM.jl</a> provides interface to the code <a href="http://fehm.lanl.gov" rel="nofollow">FEHM</a>.</p>
<p><a href="http://fehm.lanl.gov" rel="nofollow">FEHM</a> is used to simulate single- and multi-phase flow fractured and un-fractured porous media including groundwater flow and contaminant transport.</p>
<p><a href="http://madsjulia.github.io/FEHM.jl" rel="nofollow">FEHM.jl</a> is a module of the <a href="http://mads.lanl.gov" rel="nofollow">Mads</a> and <a href="http://github.com/zemjulia">ZEM</a> frameworks.</p>
<p><a href="http://madsjulia.github.io/Mads.jl" rel="nofollow">MADS</a> is an integrated high-performance computational framework for data/model/decision analyses.</p>
<p><a href="http://github.com/zemjulia">ZEM</a> is an integrated open-source high-performance computational (HPC) framework for environmental management in <a href="http://julialang.org" rel="nofollow">Julia</a>.</p>
<p><a href="http://github.com/zemjulia">ZEM</a> includes a series of modules (FEHM.jl, mysql.jl, etc.).</p>
<p><a href="http://madsjulia.github.io" rel="nofollow">Mads</a> and <a href="http://github.com/zemjulia">ZEM</a> have been tested to perform HPC simulations on a wide-range multi-processor clusters and parallel environments (Moab, Slurm, etc.).</p>
