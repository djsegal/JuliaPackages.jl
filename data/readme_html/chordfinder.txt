<p dir="auto">This package grew out of my attempt to annotate chords to Bach's <a href="https://www.mutopiaproject.org/cgibin/piece-info.cgi?id=5" rel="nofollow">Praeludium I</a> of Das Wohltemperierte Clavier.</p>
<p dir="auto">Along the way I expanded my understanding of harmonies a bit.</p>
<h1 dir="auto"><a id="user-content-how-many-chords-are-there" class="anchor" aria-hidden="true" href="#how-many-chords-are-there"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path d="m7.775 3.275 1.25-1.25a3.5 3.5 0 1 1 4.95 4.95l-2.5 2.5a3.5 3.5 0 0 1-4.95 0 .751.751 0 0 1 .018-1.042.751.751 0 0 1 1.042-.018 1.998 1.998 0 0 0 2.83 0l2.5-2.5a2.002 2.002 0 0 0-2.83-2.83l-1.25 1.25a.751.751 0 0 1-1.042-.018.751.751 0 0 1-.018-1.042Zm-4.69 9.64a1.998 1.998 0 0 0 2.83 0l1.25-1.25a.751.751 0 0 1 1.042.018.751.751 0 0 1 .018 1.042l-1.25 1.25a3.5 3.5 0 1 1-4.95-4.95l2.5-2.5a3.5 3.5 0 0 1 4.95 0 .751.751 0 0 1-.018 1.042.751.751 0 0 1-1.042.018 1.998 1.998 0 0 0-2.83 0l-2.5 2.5a1.998 1.998 0 0 0 0 2.83Z"></path></svg></a>How many chords are there?</h1>
<p dir="auto">To get a list of chords, I started with <a href="http://lilypond.org/doc/v2.20/Documentation/notation/chord-name-chart" rel="nofollow">this list</a> and <a href="http://lilypond.org/doc/v2.20/Documentation/notation/common-chord-modifiers" rel="nofollow">this list</a> from Lilypond.
Lilypond knows a lot about music and notation, so that looked like a good place to start.</p>
<p dir="auto">Wikipedia also has a <a href="https://en.wikipedia.org/wiki/List_of_chords" rel="nofollow">list of chords</a>, and I'll need to cross reference the lists at some point.</p>
<p dir="auto">For notation, I use names based on <a href="http://lilypond.org/doc/v2.20/Documentation/notation/common-chord-modifiers" rel="nofollow">Lilypond</a> and try to replicate the notation as well as possible in Unicode. I still need to find superscripted versions of ♭, ♯ and ø.</p>
<p dir="auto">The <a href="https://en.wikipedia.org/wiki/List_of_chords" rel="nofollow">Wikipedia list</a> introduces "pitch classes" as a notation of pitch mod 12: 0, 1, ..., 9, t, e.
This is a useful way of printing chords.</p>
<h2 dir="auto"><a id="user-content-when-are-chords-equal" class="anchor" aria-hidden="true" href="#when-are-chords-equal"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path d="m7.775 3.275 1.25-1.25a3.5 3.5 0 1 1 4.95 4.95l-2.5 2.5a3.5 3.5 0 0 1-4.95 0 .751.751 0 0 1 .018-1.042.751.751 0 0 1 1.042-.018 1.998 1.998 0 0 0 2.83 0l2.5-2.5a2.002 2.002 0 0 0-2.83-2.83l-1.25 1.25a.751.751 0 0 1-1.042-.018.751.751 0 0 1-.018-1.042Zm-4.69 9.64a1.998 1.998 0 0 0 2.83 0l1.25-1.25a.751.751 0 0 1 1.042.018.751.751 0 0 1 .018 1.042l-1.25 1.25a3.5 3.5 0 1 1-4.95-4.95l2.5-2.5a3.5 3.5 0 0 1 4.95 0 .751.751 0 0 1-.018 1.042.751.751 0 0 1-1.042.018 1.998 1.998 0 0 0-2.83 0l-2.5 2.5a1.998 1.998 0 0 0 0 2.83Z"></path></svg></a>When are chords equal</h2>
<p dir="auto">I define chords to be identical (==) if their vector of pitch-classes are identical (same order of notes), and I call the <em>similar</em> (≈) if the <em>set</em> of pitch classes are identical (not considering order).</p>
<p dir="auto">With this in place, we can take all 44 chords from Lilypond, and make all 11 transpositions by semitones, and check how many are similar:</p>

<p dir="auto">Note that all the 44*12 = 528 chords are different, as the 44 c-based ones from Lilypond are different, and the transpositions give a new root note.</p>
<p dir="auto">The result is at  102 out of 528 chords are similar (19.3%).</p>
