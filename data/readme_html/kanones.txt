<p dir="auto">Kanones includes an extensive suite of unit tests that take some time to run.  To run all of all them, just use the standard <code>test</code> command from package mode (and be prepared to wait).</p>
<p dir="auto">If you want to run a selection of tests, you can do that from the <code>tests</code> directory by running <code>julia select_tests.jl [FILENAMES]</code>.  <code>select_tests.jl</code> uses the <code>TestSetExtensions.jl</code> package; if you prefer its reporting format, you can run all the tests with  <code>julia select_tests.jl test* */test*jl</code>.</p>
