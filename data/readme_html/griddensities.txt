<p dir="auto">This packages allows users to define a piecewise uniform density over a hypergrid and draw samples from it.</p>
<p dir="auto">The figure below shows an example defined on a two-dimensional grid. The left half of the figure shows a plot of the probability density function (pdf) and the right half of the figure shows a two-dimensional histogram of 1,000,000 samples. Darker colors indicate higher values.</p>

<p dir="auto">A grid density is defined by calling <code>d = GridDensity(data, lo, hi, bins)</code> where <code>data</code> is the relative density within each grid cell (does not need to be normalized), <code>lo</code> is a vector of lower bounds for each dimension, <code>hi</code> is a vector of upper bounds for each dimension, and <code>bins</code> is a vector containing the number of bins for each dimension. The following line will create the density shown in the figure above.</p>
<div class="highlight highlight-source-julia notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="d = GridDensity(collect(1:8), [0.0, 0.0], [2.0, 4.0], [2, 4])"><pre>d <span class="pl-k">=</span> <span class="pl-c1">GridDensity</span>(<span class="pl-c1">collect</span>(<span class="pl-c1">1</span><span class="pl-k">:</span><span class="pl-c1">8</span>), [<span class="pl-c1">0.0</span>, <span class="pl-c1">0.0</span>], [<span class="pl-c1">2.0</span>, <span class="pl-c1">4.0</span>], [<span class="pl-c1">2</span>, <span class="pl-c1">4</span>])</pre></div>

<p dir="auto">To draw a sample from the distribution for grid density <code>d</code>, run:</p>
<div class="highlight highlight-source-julia notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="sample = rand(d)"><pre>sample <span class="pl-k">=</span> <span class="pl-c1">rand</span>(d)</pre></div>
<p dir="auto">To draw <code>n</code> samples from the distribution for grid density <code>d</code>, run:</p>
<div class="highlight highlight-source-julia notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="samples = rand(d, n)"><pre>samples <span class="pl-k">=</span> <span class="pl-c1">rand</span>(d, n)</pre></div>
