<h3><a id="user-content-a-fresh-approach-to-numerical-linear-algebra-in-julia" class="anchor" aria-hidden="true" href="#a-fresh-approach-to-numerical-linear-algebra-in-julia"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path fill-rule="evenodd" d="M7.775 3.275a.75.75 0 001.06 1.06l1.25-1.25a2 2 0 112.83 2.83l-2.5 2.5a2 2 0 01-2.83 0 .75.75 0 00-1.06 1.06 3.5 3.5 0 004.95 0l2.5-2.5a3.5 3.5 0 00-4.95-4.95l-1.25 1.25zm-4.69 9.64a2 2 0 010-2.83l2.5-2.5a2 2 0 012.83 0 .75.75 0 001.06-1.06 3.5 3.5 0 00-4.95 0l-2.5 2.5a3.5 3.5 0 004.95 4.95l1.25-1.25a.75.75 0 00-1.06-1.06l-1.25 1.25a2 2 0 01-2.83 0z"></path></svg></a>A fresh approach to numerical linear algebra in Julia</h3>
<p>The purpose of this package is partly to extend linear algebra functionality in base to cover generic element types, e.g. <code>BigFloat</code> and <code>Quaternion</code>, and partly to be a place to experiment with fast linear algebra routines written in Julia (except for optimized BLAS). It is my hope that it is possible to have implementations that are generic, fast, and readable.</p>
<p>So far, this has mainly been my playground but you might find some of the functionality here useful. The package has a generic implementation of a singular value solver which will make it possible to compute <code>norm</code> and <code>cond</code> of matrices of <code>BigFloat</code>. Hence</p>

<p>The package also includes functions for the blocked Cholesky and QR factorization, the self-adjoint (symmetric) and the general eigenvalue problem. These routines can be accessed by fully qualifying the names</p>
<div class="highlight highlight-source-julia position-relative" data-snippet-clipboard-copy-content="julia&gt; using GenericLinearAlgebra

julia&gt; A = randn(1000,1000); A = A'A;

julia&gt; cholesky(A);

julia&gt; @time cholesky(A);
  0.013036 seconds (16 allocations: 7.630 MB)

julia&gt; GenericLinearAlgebra.cholRecursive!(copy(A), Val{:L});

julia&gt; @time GenericLinearAlgebra.cholRecursive!(copy(A), Val{:L});
  0.012098 seconds (7.00 k allocations: 7.934 MB)
