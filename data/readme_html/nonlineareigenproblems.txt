<p dir="auto">A nonlinear eigenvalue problem is the problem to determine a scalar <em>λ</em> and a vector <em>v</em> such that
<em></em></p><p align="center" dir="auto"><em>M(λ)v=0</em></p>
where <em>M</em> is an <em>nxn</em>-matrix depending on a parameter. This package aims to provide state-of-the-art algorithms to solve this problem, as well as a framework to formulate applications and easy access to benchmark problems. This currently includes (but is not restricted to) Newton-type methods, Subspace methods, Krylov methods, contour integral methods, block methods, companion matrix approaches. Problem transformation techniques such as scaling, shifting, deflating are also natively supported by the package.<p dir="auto"></p>

<p dir="auto">Features and solvers (see documentation <a href="https://nep-pack.github.io/NonlinearEigenproblems.jl/methods/" rel="nofollow">https://nep-pack.github.io/NonlinearEigenproblems.jl/methods/</a> for further information and references):</p>
<ul dir="auto">
<li>Arnoldi/Krylov type
<ul dir="auto">
<li>NLEIGS</li>
<li>Infinite Arnoldi method: (iar)</li>
<li>Tensor infinite Arnoldi method  (tiar)</li>
<li>Infinite bi-Lanczos (infbilanczos)</li>
<li>Infinite Lanczos (ilan)</li>
<li>AAA CORK (AAAeigs)</li>
</ul>
</li>
<li>Projection methods
<ul dir="auto">
<li>Jacobi-Davidson (jd_effenberger)</li>
<li>Jacobi-Davidson (jd_betcke)</li>
<li>Nonlinear Arnoldi method (nlar)</li>
<li>Common Rayleigh-Ritz projection interface</li>
</ul>
</li>
<li>Contour integral methods
<ul dir="auto">
<li>Beyn's contour integral method</li>
<li>Block SS (Higher moments) contour integral method of Asakura &amp; Sakurai</li>
<li>Common quadrature interface for parallelization</li>
</ul>
</li>
<li>Newton &amp; Rayleigh type:
<ul dir="auto">
<li>Classical Newton-Raphson</li>
<li>Augmented Newton</li>
<li>Residual inverse iteration</li>
<li>Quasi-Newton</li>
<li>Block Newton</li>
<li>Rayleigh functional iteration (RFI a, b)</li>
<li>Newton-QR</li>
<li>Implicit determinant method</li>
<li>Broyden's method</li>
</ul>
</li>
<li>Companion matrices
<ul dir="auto">
<li>First companion form</li>
<li>Companion form for Chebyshev polynomials</li>
</ul>
</li>
<li>Other
<ul dir="auto">
<li>Chebyshev interpolation</li>
<li>Transformations</li>
<li>Rayleigh-Ritz (<code>ProjNEP</code> and <code>inner_solve</code>)</li>
<li>Problem gallery (including access to the NLEVP-gallery)</li>
<li>Deflation (Effenberger style)</li>
</ul>
</li>
</ul>
