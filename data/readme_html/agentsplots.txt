<p>Plotting functionality for <a href="https://github.com/JuliaDynamics/Agents.jl"><code>Agents.jl</code></a> agent-based modeling in Julia.</p>
<p>Agents.jl is a <a href="https://julialang.org/" rel="nofollow">Julia</a> framework for agent-based modeling (ABM). It provides a structure and components for quickly implementing agent-based models, run them in batch, collect data, and visualize them.</p>
<p>AgentsPlots.jl is plotting functionality for 2D grid models in Agents.jl</p>
