<p>The <a href="http://en.wikipedia.org/wiki/Jet_Propulsion_Laboratory_Development_Ephemeris" rel="nofollow">JPL Development Ephemerides</a> are the results of simulations of the Solar System used for spacecraft navigation and astronomical purposes. They are published as <a href="http://naif.jpl.nasa.gov/pub/naif/generic_kernels/spk/" rel="nofollow">SPK kernel files</a> which contain sets of Chebyshev polynomial coefficients with which the position and velocity of the Solar System's planets can be interpolated with high precision for all dates covered by the ephemeris.</p>
<p>This package provides functionality to read SPK files and compute the position and velocity of the planets directly from Julia.</p>

<div class="highlight highlight-source-julia"><pre><span class="pl-k">using</span> JPLEphemeris

<span class="pl-c"><span class="pl-c">#</span> Load the DE430 SPK kernel</span>

<span class="pl-c"><span class="pl-c">#</span> Velocity of Earth w.r.t. Earth's barycentre at 2016-01-01T00:00</span>
<span class="pl-c"><span class="pl-c">#</span> [km/s]</span>
vel <span class="pl-k">=</span> <span class="pl-c1">velocity</span>(spk, <span class="pl-s"><span class="pl-pds">"</span>earth barycenter<span class="pl-pds">"</span></span>, <span class="pl-s"><span class="pl-pds">"</span>earth<span class="pl-pds">"</span></span>, jd)

<span class="pl-c"><span class="pl-c">#</span> Compute the state vector (position and velocity) of Earth's barycenter (NAIF ID: 3)</span>
<span class="pl-c"><span class="pl-c">#</span> w.r.t. to the Solar System's barycenter (NAIF ID: 0) for a range of Julian days</span>
st <span class="pl-k">=</span> <span class="pl-c1">state</span>(spk, <span class="pl-c1">0</span>, <span class="pl-c1">3</span>, jd<span class="pl-k">:</span>jd<span class="pl-k">+</span><span class="pl-c1">100</span>)

<span class="pl-c"><span class="pl-c">#</span> Two-part Julian dates (day number and fraction) can be used for higher precision.</span>

st <span class="pl-k">=</span> <span class="pl-c1">state</span>(spk, <span class="pl-c1">0</span>, <span class="pl-c1">3</span>, jd, <span class="pl-c1">0.5</span>)</pre></div>
