<p>This package allows users to use Freudenthal triangulation for <a href="https://sisl.github.io/FreudenthalTriangulation.jl/concepts/#Concepts" rel="nofollow">functional value approximation</a>.</p>
<p>Specifically it allows the user to find the simplex of a point <code>x</code> in Freudenthal space and the barycentric coordinates of that point with respect to the its simplex.</p>
