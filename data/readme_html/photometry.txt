<p dir="auto">This is a package for performing astronomical photometry using modern and efficient algorithms.</p>
<p dir="auto">Inspired by <a href="https://github.com/astropy/photutils">photutils</a>, <a href="https://github.com/kbarbary/sep">SEP</a>, and <a href="https://github.com/kbarbary/AperturePhotometry.jl">AperturePhotometry.jl</a>.</p>

<p dir="auto">Here is a basic example to do some aperture photometry using <code>CircularAperture</code>. The <code>photometry</code> function performs the photometry using a given method. Please see <a href="https://JuliaAstro.github.io/Photometry.jl/dev" rel="nofollow">the documentation</a> for more examples and reference material.</p>

@assert results.aperture_sum[1] ≈ 9π
@assert results.aperture_sum[2] ≈ 25π / 4

results
