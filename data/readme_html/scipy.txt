<p>A Julia interface for <a href="https://www.scipy.org/scipylib/index.html" rel="nofollow">SciPy</a> using <a href="https://github.com/JuliaPy/PyCall.jl">PyCall.jl</a>.</p>
<p>You can use many useful scientific functions of SciPy from Julia codes.</p>

<div class="highlight highlight-source-julia"><pre><span class="pl-k">using</span> Pkg;Pkg<span class="pl-k">.</span><span class="pl-c1">add</span>(<span class="pl-s"><span class="pl-pds">"</span>SciPy<span class="pl-pds">"</span></span>)</pre></div>
<p>and then just import it with <code>using SciPy</code>.</p>
<p>If you want use latest development code, clone this repo.</p>
<p>Then, command this in Pkg mode:</p>
<blockquote>
<p>Pkg&gt; dev /path/to/module</p>
</blockquote>
