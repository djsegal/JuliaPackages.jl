<p>LifeContingencies is a package enabling actuarial life contingent calculations.
The benefits are:</p>
<ul>
<li>Integration with other JuliaActuary packages such as <a href="https://github.com/JuliaActuary/MortalityTables.jl">MortalityTables.jl</a></li>
<li>Fast calculations, with some parts utilizing parallel processing power automatically</li>
<li>Use functions that look more like the math you are used to (e.g. <code>Ax</code>, <code>ä</code>)
with <a href="https://docs.julialang.org/en/v1/manual/unicode-input/index.html" rel="nofollow">Unicode support</a></li>
<li>All of the power, speed, convenience, tooling, and ecosystem of Julia</li>
<li>Flexible and modular modeling approach</li>
</ul>
<h2><a id="user-content-package-overview" class="anchor" aria-hidden="true" href="#package-overview"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path fill-rule="evenodd" d="M7.775 3.275a.75.75 0 001.06 1.06l1.25-1.25a2 2 0 112.83 2.83l-2.5 2.5a2 2 0 01-2.83 0 .75.75 0 00-1.06 1.06 3.5 3.5 0 004.95 0l2.5-2.5a3.5 3.5 0 00-4.95-4.95l-1.25 1.25zm-4.69 9.64a2 2 0 010-2.83l2.5-2.5a2 2 0 012.83 0 .75.75 0 001.06-1.06 3.5 3.5 0 00-4.95 0l-2.5 2.5a3.5 3.5 0 004.95 4.95l1.25-1.25a.75.75 0 00-1.06-1.06l-1.25 1.25a2 2 0 01-2.83 0z"></path></svg></a>Package Overview</h2>
<ul>
<li>Leverages <a href="https://github.com/JuliaActuary/MortalityTables.jl">MortalityTables.jl</a> for
the mortality calculations</li>
<li>Contains common insurance calculations such as:
<ul>
<li><code>A(x)</code>: Whole life</li>
<li><code>A(x,n)</code>: Term life for <code>n</code> years</li>
<li><code>ä(x)</code>: Life contingent annuity due</li>
<li><code>ä(x,n)</code>: Life contingent annuity due for <code>n</code> years</li>
</ul>
</li>
<li>Contains various commutaion functions such as <code>D(x)</code>,<code>M(x)</code>,<code>C(x)</code>, etc.</li>
<li>Various interest rate mechanics (e.g. stochastic, constant, etc.)</li>
<li>More documentation available by clicking the DOCS bages at the top of this README</li>
</ul>
<h2><a id="user-content-examples" class="anchor" aria-hidden="true" href="#examples"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path fill-rule="evenodd" d="M7.775 3.275a.75.75 0 001.06 1.06l1.25-1.25a2 2 0 112.83 2.83l-2.5 2.5a2 2 0 01-2.83 0 .75.75 0 00-1.06 1.06 3.5 3.5 0 004.95 0l2.5-2.5a3.5 3.5 0 00-4.95-4.95l-1.25 1.25zm-4.69 9.64a2 2 0 010-2.83l2.5-2.5a2 2 0 012.83 0 .75.75 0 001.06-1.06 3.5 3.5 0 00-4.95 0l-2.5 2.5a3.5 3.5 0 004.95 4.95l1.25-1.25a.75.75 0 00-1.06-1.06l-1.25 1.25a2 2 0 01-2.83 0z"></path></svg></a>Examples</h2>
<p>Calculate the whole life insurance rate for a 30-year-old male nonsmoker using
2015 VBT base table and a 5% interest rate</p>
<div class="highlight highlight-source-julia"><pre><span class="pl-k">using</span> LifeContingencies, MortalityTables

tbls <span class="pl-k">=</span> MortalityTables<span class="pl-k">.</span><span class="pl-c1">tables</span>()
vbt2001 <span class="pl-k">=</span> tbls[<span class="pl-s"><span class="pl-pds">"</span>2001 VBT Residual Standard Select and Ultimate - Male Nonsmoker, ANB<span class="pl-pds">"</span></span>]
issue_age <span class="pl-k">=</span> <span class="pl-c1">30</span>
l <span class="pl-k">=</span> <span class="pl-c1">LifeContingency</span>(
    vbt2001<span class="pl-k">.</span>select,
    <span class="pl-c1">InterestRate</span>(<span class="pl-c1">0.05</span>),
    issue_age
    )

start_time <span class="pl-k">=</span> <span class="pl-c1">0</span>
<span class="pl-c1">A</span>(l,start_time) <span class="pl-c"><span class="pl-c">#</span> 0.111...</span></pre></div>
<p>Use a stochastic interest rate calculation to price a term policy:</p>
<div class="highlight highlight-source-julia"><pre><span class="pl-k">using</span> LifeContingencies, MortalityTables
<span class="pl-k">using</span> Distributions

tbls <span class="pl-k">=</span> MortalityTables<span class="pl-k">.</span><span class="pl-c1">tables</span>()
vbt2001 <span class="pl-k">=</span> tbls[<span class="pl-s"><span class="pl-pds">"</span>2001 VBT Residual Standard Select and Ultimate - Male Nonsmoker, ANB<span class="pl-pds">"</span></span>]

<span class="pl-c"><span class="pl-c">#</span> use an interest rate that's normally distirbuted</span>
μ <span class="pl-k">=</span> <span class="pl-c1">0.05</span>
σ <span class="pl-k">=</span> <span class="pl-c1">0.01</span>
int <span class="pl-k">=</span> <span class="pl-c1">InterestRate</span>(t <span class="pl-k">-&gt;</span> <span class="pl-c1">rand</span>(<span class="pl-c1">Normal</span>(μ,σ)))

l <span class="pl-k">=</span> <span class="pl-c1">LifeContingency</span>(
    vbt2001<span class="pl-k">.</span>select,
    int,
    <span class="pl-c1">30</span> <span class="pl-c"><span class="pl-c">#</span> issue age</span>
    )

start_time <span class="pl-k">=</span> <span class="pl-c1">0</span>
term <span class="pl-k">=</span> <span class="pl-c1">10</span>
<span class="pl-c1">A</span>(l,start_time,term) <span class="pl-c"><span class="pl-c">#</span> somewhere around 0.055</span></pre></div>
<p>You can use autocorrelated interest rates - substitute the following in the prior example
using the ability to self reference:</p>
<div class="highlight highlight-source-julia"><pre>σ <span class="pl-k">=</span> <span class="pl-c1">0.01</span>
initial_rate <span class="pl-k">=</span> <span class="pl-c1">0.05</span>
int <span class="pl-k">=</span> <span class="pl-c1">InterestRate</span>(
    <span class="pl-k">function</span> <span class="pl-en">intAR</span>(time)
        <span class="pl-k">if</span> time <span class="pl-k">&lt;=</span> <span class="pl-c1">1</span>
            initial_rate
        <span class="pl-k">else</span>
            i′ <span class="pl-k">=</span> <span class="pl-c1">last</span>(int<span class="pl-k">.</span>rate_vector)
            <span class="pl-c1">rand</span>(<span class="pl-c1">Normal</span>(i′,σ))
        <span class="pl-k">end</span>
    <span class="pl-k">end</span>
)
</pre></div>
<p>Compare the cost of annual premium, whole life insurance between multiple tables visually:</p>
<div class="highlight highlight-source-julia"><pre><span class="pl-k">using</span> LifeContingencies, MortalityTables, Plots

tbls <span class="pl-k">=</span> MortalityTables<span class="pl-k">.</span><span class="pl-c1">tables</span>()
tables <span class="pl-k">=</span> [
    tbls[<span class="pl-s"><span class="pl-pds">"</span>1980 CET - Male Nonsmoker, ANB<span class="pl-pds">"</span></span>],
    tbls[<span class="pl-s"><span class="pl-pds">"</span>2001 VBT Residual Standard Select and Ultimate - Male Nonsmoker, ANB<span class="pl-pds">"</span></span>],
    tbls[<span class="pl-s"><span class="pl-pds">"</span>2015 VBT Male Non-Smoker RR100 ANB<span class="pl-pds">"</span></span>],
    ]

issue_ages <span class="pl-k">=</span> <span class="pl-c1">30</span><span class="pl-k">:</span><span class="pl-c1">90</span>
int <span class="pl-k">=</span> <span class="pl-c1">InterestRate</span>(<span class="pl-c1">0.05</span>)

whole_life_costs <span class="pl-k">=</span> <span class="pl-c1">map</span>(tables) <span class="pl-k">do</span> t
    <span class="pl-c1">map</span>(issue_ages) <span class="pl-k">do</span> ia
        lc <span class="pl-k">=</span> <span class="pl-c1">LifeContingency</span>(
            t<span class="pl-k">.</span>ultimate,
            int,
            ia
            )

        <span class="pl-c1">A</span>(lc,<span class="pl-c1">0</span>) <span class="pl-k">/</span> ä(lc,<span class="pl-c1">0</span>)

    <span class="pl-k">end</span>
<span class="pl-k">end</span>

plt <span class="pl-k">=</span> <span class="pl-c1">plot</span>(ylabel<span class="pl-k">=</span><span class="pl-s"><span class="pl-pds">"</span>Annual Premium per unit<span class="pl-pds">"</span></span>, xlabel<span class="pl-k">=</span><span class="pl-s"><span class="pl-pds">"</span>Issue Age<span class="pl-pds">"</span></span>,
            legend<span class="pl-k">=</span><span class="pl-c1">:topleft</span>, legendfontsize<span class="pl-k">=</span><span class="pl-c1">8</span>)
<span class="pl-k">for</span> (i,t) <span class="pl-k">in</span> <span class="pl-c1">enumerate</span>(tables)
    <span class="pl-c1">plot!</span>(plt,issue_ages,whole_life_costs[i], label<span class="pl-k">=</span><span class="pl-s"><span class="pl-pds">"</span><span class="pl-v">$(t<span class="pl-k">.</span>d<span class="pl-k">.</span>name)</span><span class="pl-pds">"</span></span>)
<span class="pl-k">end</span>
<span class="pl-c1">display</span>(plt)</pre></div>
