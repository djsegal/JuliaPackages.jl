<h2><a id="user-content-adding-an-example" class="anchor" aria-hidden="true" href="#adding-an-example"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path fill-rule="evenodd" d="M7.775 3.275a.75.75 0 001.06 1.06l1.25-1.25a2 2 0 112.83 2.83l-2.5 2.5a2 2 0 01-2.83 0 .75.75 0 00-1.06 1.06 3.5 3.5 0 004.95 0l2.5-2.5a3.5 3.5 0 00-4.95-4.95l-1.25 1.25zm-4.69 9.64a2 2 0 010-2.83l2.5-2.5a2 2 0 012.83 0 .75.75 0 001.06-1.06 3.5 3.5 0 00-4.95 0l-2.5 2.5a3.5 3.5 0 004.95 4.95l1.25-1.25a.75.75 0 00-1.06-1.06l-1.25 1.25a2 2 0 01-2.83 0z"></path></svg></a>Adding an example</h2>
<p>MakieGallery hosts a lot of examples describing how to use Makie.jl.<br>
To add a standalone example, find the file where it fits best, and add a new <code>@cell</code> entry
(or a new <code>@block</code> entry if you want to add more examples in the future).<br>
If you want to add multiple examples, you probably want to add a new <code>@block</code>, or even a new file.</p>
<h2><a id="user-content-building-the-documentation" class="anchor" aria-hidden="true" href="#building-the-documentation"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path fill-rule="evenodd" d="M7.775 3.275a.75.75 0 001.06 1.06l1.25-1.25a2 2 0 112.83 2.83l-2.5 2.5a2 2 0 01-2.83 0 .75.75 0 00-1.06 1.06 3.5 3.5 0 004.95 0l2.5-2.5a3.5 3.5 0 00-4.95-4.95l-1.25 1.25zm-4.69 9.64a2 2 0 010-2.83l2.5-2.5a2 2 0 012.83 0 .75.75 0 001.06-1.06 3.5 3.5 0 00-4.95 0l-2.5 2.5a3.5 3.5 0 004.95 4.95l1.25-1.25a.75.75 0 00-1.06-1.06l-1.25 1.25a2 2 0 01-2.83 0z"></path></svg></a>Building the documentation</h2>
<p><code>MakieGallery</code> hosts documentation for Makie.jl.<br>
However, this documentation requires the reference images to be downloaded in the home directory
(configurability via an environment variable is planned).<br>
You can get the reference images from <a href="https://github.com/JuliaPlots/MakieReferenceImages">https://github.com/JuliaPlots/MakieReferenceImages</a> -
it's a large repository, so you may want to shallow clone.</p>

<p>You can upgrade the reference images by using the script <code>test/upgrade_recordings.jl</code>.  Then, you can create a pull request to <a href="https://github.com/JuliaPlots/MakieReferenceImages">MakieReferenceImages</a>.</p>
