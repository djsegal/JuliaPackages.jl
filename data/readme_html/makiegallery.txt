<h2><a id="user-content-adding-an-example" class="anchor" aria-hidden="true" href="#adding-an-example"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path fill-rule="evenodd" d="M4 9h1v1H4c-1.5 0-3-1.69-3-3.5S2.55 3 4 3h4c1.45 0 3 1.69 3 3.5 0 1.41-.91 2.72-2 3.25V8.59c.58-.45 1-1.27 1-2.09C10 5.22 8.98 4 8 4H4c-.98 0-2 1.22-2 2.5S3 9 4 9zm9-3h-1v1h1c1 0 2 1.22 2 2.5S13.98 12 13 12H9c-.98 0-2-1.22-2-2.5 0-.83.42-1.64 1-2.09V6.25c-1.09.53-2 1.84-2 3.25C6 11.31 7.55 13 9 13h4c1.45 0 3-1.69 3-3.5S14.5 6 13 6z"></path></svg></a>Adding an example</h2>
<p>MakieGallery hosts a lot of examples describing how to use Makie.jl.<br>
To add a standalone example, find the file where it fits best, and add a new <code>@cell</code> entry
(or a new <code>@block</code> entry if you want to add more examples in the future).<br>
If you want to add multiple examples, you probably want to add a new <code>@block</code>, or even a new file.</p>
<h2><a id="user-content-building-the-documentation" class="anchor" aria-hidden="true" href="#building-the-documentation"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path fill-rule="evenodd" d="M4 9h1v1H4c-1.5 0-3-1.69-3-3.5S2.55 3 4 3h4c1.45 0 3 1.69 3 3.5 0 1.41-.91 2.72-2 3.25V8.59c.58-.45 1-1.27 1-2.09C10 5.22 8.98 4 8 4H4c-.98 0-2 1.22-2 2.5S3 9 4 9zm9-3h-1v1h1c1 0 2 1.22 2 2.5S13.98 12 13 12H9c-.98 0-2-1.22-2-2.5 0-.83.42-1.64 1-2.09V6.25c-1.09.53-2 1.84-2 3.25C6 11.31 7.55 13 9 13h4c1.45 0 3-1.69 3-3.5S14.5 6 13 6z"></path></svg></a>Building the documentation</h2>
<p><code>MakieGallery</code> hosts documentation for Makie.jl.<br>
However, this documentation requires the reference images to be downloaded in the home directory
(configurability via an environment variable is planned).<br>
You can get the reference images from <a href="https://github.com/JuliaPlots/MakieReferenceImages">https://github.com/JuliaPlots/MakieReferenceImages</a> -
it's a large repository, so you may want to shallow clone.</p>
