<p dir="auto"><a href="https://travis-ci.org/matteoacrossi/ExpmV.jl" rel="nofollow"><img src="https://camo.githubusercontent.com/30202cab8578906113fa43f25348da0efbebbbef03ba39f493bb92571762cd12/68747470733a2f2f7472617669732d63692e6f72672f6d617474656f6163726f7373692f4578706d562e6a6c2e7376673f6272616e63683d6d6173746572" alt="Build Status" data-canonical-src="https://travis-ci.org/matteoacrossi/ExpmV.jl.svg?branch=master" style="max-width: 100%;"></a></p>
<blockquote>
<p dir="auto">This is a fork from <a href="https://github.com/marcusps/ExpmV.jl">https://github.com/marcusps/ExpmV.jl</a>, implementing the evaluation for multiple values of the parameter <code> t</code>.</p>
</blockquote>
<p dir="auto">This is a Julia translation of the MATLAB implementation of Al-Mohy and Higham's
function for computing <code>expm(t*A)*v</code> when <code>A</code> is sparse, without explicitly computing <code>expm(A)</code>.</p>
<p dir="auto">If <code>t</code> is a <code>StepRangeLen</code> object (i. e. a <code>linspace</code>), use an optimized algorithm to output the result for all <code>t</code>.</p>
<p dir="auto">The original code can be found at (<a href="https://github.com/higham/expmv">https://github.com/higham/expmv</a>), and the theory is explained in the following article:</p>
<p dir="auto"><em>Computing the Action of the Matrix Exponential, with an Application to Exponential Integrators</em>, Awad H. Al-Mohy and Nicholas J. Higham, SIAM Journal on Scientific Computing 2011 33:2, 488-511. (<a href="http://eprints.ma.man.ac.uk/1426/" rel="nofollow">preprint</a>)</p>
