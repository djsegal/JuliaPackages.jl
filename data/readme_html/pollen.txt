<p dir="auto"><a href="https://lorenzoh.github.io/Pollen.jl" rel="nofollow">Documentation</a></p>
<p dir="auto">A document generation system based on tree rewriting, built for interactive work.</p>
<p dir="auto">Pollen.jl generates documentation for Julia packages that comes with many features:</p>
<ul dir="auto">
<li>support for many input and output formats: Markdown, Jupyter notebooks, HTML, JSON, and Julia source code</li>
<li>a modern frontend with Sliding Panes</li>
<li>automatic hyperreferencing of code variables for discoverability</li>
<li>code execution to ensure your code examples stay up-to-date</li>
<li>automated builds, deployments and PR previews with GitHub Actions and GitHub Pages</li>
<li>local preview with incremental builds for an improved developer experience</li>
<li>full support for multiple packages and linking between them</li>
</ul>
<p dir="auto">First, find out <a href="https://lorenzoh.github.io/Pollen.jl/dev/Pollen@dev/doc/docs/howto/install.md.html" rel="nofollow">how to install Pollen</a> and then <a href="https://lorenzoh.github.io/Pollen.jl/dev/Pollen@dev/doc/docs/tutorials/setup.md.html" rel="nofollow">how to setup package documentation</a>.</p>
