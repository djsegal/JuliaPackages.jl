<p dir="auto">2D plotting functionality for the RoME.jl package (presently only using Gadfly).  This package is part of the <a href="http://www.github.com/JuliaRobotics/Caesar.jl">Caesar.jl</a> suite of tools.  This package contains all the plotting functions relating to the <a href="http://www.github.com/JuliaRobotics/IncrementalInference.jl">IncrementalInference.jl</a> and <a href="http://www.github.com/JuliaRobotics/RoME.jl">RoME.jl</a> packages.</p>
