<h3><a id="user-content-prerequisites-for-plotting" class="anchor" aria-hidden="true" href="#prerequisites-for-plotting"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path fill-rule="evenodd" d="M7.775 3.275a.75.75 0 001.06 1.06l1.25-1.25a2 2 0 112.83 2.83l-2.5 2.5a2 2 0 01-2.83 0 .75.75 0 00-1.06 1.06 3.5 3.5 0 004.95 0l2.5-2.5a3.5 3.5 0 00-4.95-4.95l-1.25 1.25zm-4.69 9.64a2 2 0 010-2.83l2.5-2.5a2 2 0 012.83 0 .75.75 0 001.06-1.06 3.5 3.5 0 00-4.95 0l-2.5 2.5a3.5 3.5 0 004.95 4.95l1.25-1.25a.75.75 0 00-1.06-1.06l-1.25 1.25a2 2 0 01-2.83 0z"></path></svg></a>Prerequisites for plotting</h3>
<p><code>Simplehypergraphs</code> can optionally use (among other options) the Python's <a href="https://github.com/pnnl/HyperNetX">hypernetx</a> library so if you want the <code>hypernetx</code> plotting to work you need to install it first along with its all dependent libraries. In Julia command line REPL press <code>]</code> for the Julia package manager and run:</p>
<div class="snippet-clipboard-content position-relative" data-snippet-clipboard-copy-content="(v1.5) pkg&gt; add PyCall Conda
"><pre><code>(v1.5) pkg&gt; add PyCall Conda
</code></pre></div>
<p>Now press <em>backspace</em> to exit the package manager and run the following installation code:</p>
<div class="highlight highlight-source-julia position-relative" data-snippet-clipboard-copy-content="using PyCall
using Conda
Conda.runconda(`install matplotlib --yes`)
Conda.runconda(`install networkx --yes`)
Conda.runconda(`install pandas --yes`)
run(`$(PyCall.python) -m pip install hypernetx`)
"><pre><span class="pl-k">using</span> PyCall
<span class="pl-k">using</span> Conda
Conda<span class="pl-k">.</span><span class="pl-c1">runconda</span>(<span class="pl-s"><span class="pl-pds">`</span>install matplotlib --yes<span class="pl-pds">`</span></span>)
Conda<span class="pl-k">.</span><span class="pl-c1">runconda</span>(<span class="pl-s"><span class="pl-pds">`</span>install networkx --yes<span class="pl-pds">`</span></span>)
Conda<span class="pl-k">.</span><span class="pl-c1">runconda</span>(<span class="pl-s"><span class="pl-pds">`</span>install pandas --yes<span class="pl-pds">`</span></span>)
<span class="pl-c1">run</span>(<span class="pl-s"><span class="pl-pds">`</span>$(PyCall.python) -m pip install hypernetx<span class="pl-pds">`</span></span>)</pre></div>
<p>In case of throubleshooting to check whether <code>hypernetx</code> is properly configured with Julia you can always run the following test:</p>
<div class="snippet-clipboard-content position-relative" data-snippet-clipboard-copy-content="using PyCall
pyimport(&quot;hypernetx&quot;)
"><pre><code>using PyCall
pyimport("hypernetx")
</code></pre></div>
<p>For plotting we also use the Jupyter notebook. The installation instructions are <a href="https://github.com/JuliaLang/IJulia.jl#installation">here</a></p>
