<p dir="auto"><a href="https://github.com/qecsim/Qecsim.jl">Qecsim.jl</a> is a Julia package for
simulating quantum error correction using stabilizer codes.</p>
<p dir="auto"><strong>NOTE</strong>: <em>Qecsim.jl</em> is a ground-up rewrite of the Python package
<a href="https://github.com/qecsim/qecsim">qecsim</a>.</p>

manager <a href="https://pkgdocs.julialang.org/" rel="nofollow">Pkg</a>:</p>
