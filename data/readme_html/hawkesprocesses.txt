<p>A Julia package for fitting, analysing and plotting Hawkes processes.</p>
<p>You can find the maths behind the algorithm <a href="https://dm13450.github.io/assets/hawkesprocesses.pdf" rel="nofollow">here</a>. The Bayesian sampling algorithm is a generic version of the <code>bayesianETAS</code> package from CRAN which can be found <a href="https://cran.r-project.org/web/packages/bayesianETAS/index.html" rel="nofollow">here</a>.</p>
