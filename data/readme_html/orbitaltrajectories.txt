<p dir="auto"><em>OrbitalTrajectories.jl is a modern orbital trajectory design, optimisation, and analysis library for Julia, providing methods and tools for designing spacecraft orbits and transfers via high-performance simulations of astrodynamical models.</em></p>

<p dir="auto">Install the package with <code>add OrbitalTrajectories</code> in Julia's package
mode (run the Julia console and press <code>]</code>, or alternatively <code>using Pkg; Pkg.add("OrbitalTrajectories")</code>).</p>
<p dir="auto">To later update to the newest release, simply do <code>update OrbitalTrajectories</code>.</p>
