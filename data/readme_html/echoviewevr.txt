<p>The Echoview 2D Region definition file format is described on
the <a href="http://bit.ly/2uH0O4a" rel="nofollow">Echoview support web site</a>.</p>
<p>This project is a <a href="https://julialang.org/" rel="nofollow">Julia</a> package that reads
Echoview EVR files.</p>
