<p>To install, open a Julia REPL and run the following:</p>
<div class="highlight highlight-source-julia"><pre>]add UnitfulCurrency</pre></div>
<h2><a id="user-content-adding-custom-currencies" class="anchor" aria-hidden="true" href="#adding-custom-currencies"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path fill-rule="evenodd" d="M7.775 3.275a.75.75 0 001.06 1.06l1.25-1.25a2 2 0 112.83 2.83l-2.5 2.5a2 2 0 01-2.83 0 .75.75 0 00-1.06 1.06 3.5 3.5 0 004.95 0l2.5-2.5a3.5 3.5 0 00-4.95-4.95l-1.25 1.25zm-4.69 9.64a2 2 0 010-2.83l2.5-2.5a2 2 0 012.83 0 .75.75 0 001.06-1.06 3.5 3.5 0 00-4.95 0l-2.5 2.5a3.5 3.5 0 004.95 4.95l1.25-1.25a.75.75 0 00-1.06-1.06l-1.25 1.25a2 2 0 01-2.83 0z"></path></svg></a>Adding Custom Currencies</h2>
<p>To add custom currencies, use the <code>currencyunit</code> macro:</p>
<div class="highlight highlight-source-julia"><pre><span class="pl-c1">@currencyunit</span> SYM NewCurrencyName</pre></div>
<p>See <a href="https://github.com/PainterQubits/Unitful.jl">Unitful.jl</a> documentation for more detail on custom units.</p>
<h2><a id="user-content-currency-pairs--markets" class="anchor" aria-hidden="true" href="#currency-pairs--markets"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path fill-rule="evenodd" d="M7.775 3.275a.75.75 0 001.06 1.06l1.25-1.25a2 2 0 112.83 2.83l-2.5 2.5a2 2 0 01-2.83 0 .75.75 0 00-1.06 1.06 3.5 3.5 0 004.95 0l2.5-2.5a3.5 3.5 0 00-4.95-4.95l-1.25 1.25zm-4.69 9.64a2 2 0 010-2.83l2.5-2.5a2 2 0 012.83 0 .75.75 0 001.06-1.06 3.5 3.5 0 00-4.95 0l-2.5 2.5a3.5 3.5 0 004.95 4.95l1.25-1.25a.75.75 0 00-1.06-1.06l-1.25 1.25a2 2 0 01-2.83 0z"></path></svg></a>Currency Pairs / Markets</h2>
<p>To define currency pairs (or currency markets), use the <code>CurrencyPair</code> (or <code>CurrencyMarket</code>) type:</p>
<div class="highlight highlight-source-julia"><pre><span class="pl-c1">CurrencyPair</span>(EUR, USD)
<span class="pl-c1">CurrencyMarket</span>(EUR, USD, <span class="pl-c1">1.0126</span>)</pre></div>
