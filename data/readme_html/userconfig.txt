<p dir="auto">This package provides helper functions to get, store and retrieve user-specific data to/from disk. All data is stored in the <code>[HOME]/.julia/config/</code> folder.</p>
<p dir="auto">The functions are currently:</p>

<div class="highlight highlight-source-julia notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="    path = localpath(title, checkfun, isfolder=false)"><pre>    path <span class="pl-k">=</span> <span class="pl-c1">localpath</span>(title, checkfun, isfolder<span class="pl-k">=</span><span class="pl-c1">false</span>)</pre></div>
<p dir="auto">Get the path to a local file or folder (if <code>isfolder</code> is true), and ask the user if they haven't given it before. <code>checkfun(path)</code> is run, and only returns the path if true. The path is stored in a .txt file, the filename of which is <code>string2key(title).txt</code>; <code>title</code> should therefore be a suitably unique string.</p>

The string is stored in a .txt file, the filename of which is <code>string2key(title).txt</code>; <code>title</code> should therefore be a suitably unique string.</p>
