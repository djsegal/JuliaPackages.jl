<p dir="auto">A collection of background subtraction algorithms for spectroscopic data</p>

<p dir="auto">To install the package, simply type "<code>]</code>" followed up by "<code>add BackgroundSubtraction</code>" in the Julia REPL.</p>
<p dir="auto">The main function is based on the <a href="https://rdcu.be/b32TH" rel="nofollow">multi-component background learning model (MCBL)</a>, with the corresponding function <code>mcbl</code>:</p>
<div class="snippet-clipboard-content notranslate position-relative overflow-auto" data-snippet-clipboard-copy-content="mcbl(A::AbstractMatrix, k::Int, x::AbstractVector, l::Real)"><pre class="notranslate"><code>mcbl(A::AbstractMatrix, k::Int, x::AbstractVector, l::Real)
</code></pre></div>
<ul dir="auto">
<li><code>A</code> is the data matrix, each column of which is assumed to be a spectrogram.</li>
<li><code>k</code> is the number of components in the multi-component background model.</li>
<li><code>x</code> is the index vector corresponding to rows of <code>A</code>.
For example, if a column of <code>A</code> is an X-ray diffraction spectrogram, <code>x</code> should be the angle of diffraction of each row.</li>
<li><code>l</code> is the length scale of the background component. It controls how quickly the
background model is allowed to vary with <code>x</code>.
This functions as an important regularization for medium-sized data (100s-1000s spectrograms).</li>
</ul>
<p dir="auto">There are 3 parameters controlling the algorithm, which can optionally be passed as keyword arguments:</p>
<ul dir="auto">
<li><code>minres</code> is the minimum residual standard deviation after which the algorithms terminates.</li>
<li><code>nsigma</code> is the number of standard deviations above the noise level after which a data point is classified as a peak. A smaller number will be more agressive in classifying points as peaks.</li>
<li><code>maxiter</code> is the maximum number of iterations between updating the noise and background model.</li>
</ul>
