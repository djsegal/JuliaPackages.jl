<p dir="auto">For a function <math-renderer class="js-inline-math" style="display: inline" data-static-url="https://github.githubassets.com/static" data-run-id="b60e733a24a578645b324e9c14d868e3">$f: ℳ → ℝ$</math-renderer>  that maps from a <a href="https://en.wikipedia.org/wiki/Riemannian_manifold" rel="nofollow">Riemannian manifold</a>
ℳ to the real line, we aim to solve</p>
<blockquote>
<p dir="auto">Find the minimizer p on ℳ, i.e. the (or a) point where f attains its minimum.</p>
</blockquote>
<p dir="auto"><code>Manopt.jl</code> provides</p>
<ul dir="auto">
<li>A framework to implement arbitrary optimization algorithms on Riemannian Manifolds</li>
<li>A library of optimization algorithms on Riemannian manifolds</li>
<li>an easy-to-use interface for (debug) output and recording values during an algorithm run.</li>
<li>several tools to investigate the algorithms, gradients, and optimality criteria</li>
</ul>

<p dir="auto">Manopt.jl is based on <a href="https://juliamanifolds.github.io/ManifoldsBase.jl/stable/" rel="nofollow"><code>ManifoldsBase.jl</code></a>,
hence the algorithms can be used with <em>any</em> manifold following this interface for defining
a Riemannian manifold.</p>
<p dir="auto">The following packages are related to <code>Manopt.jl</code></p>
<ul dir="auto">
<li>
<a href="https://juliamanifolds.github.io/Manifolds.jl/stable/" rel="nofollow"><code>Manifolds.jl</code></a> – a library of manifolds implemented using <a href="https://juliamanifolds.github.io/ManifoldsBase.jl/stable/" rel="nofollow"><code>ManifoldsBase.jl</code></a> <img class="emoji" title=":octocat:" alt=":octocat:" src="https://github.githubassets.com/images/icons/emoji/octocat.png" height="20" width="20" align="absmiddle"> <a href="https://github.com/JuliaManifolds/Manifolds.jl">GitHub repository</a>
</li>
<li>
<a href="https://juliamanifolds.github.io/ManifoldDiff.jl/stable/" rel="nofollow"><code>ManifoldsDiff.jl</code></a> – a package to use (Euclidean) AD tools on manifolds, that also provides several differentials and gradients. <img class="emoji" title=":octocat:" alt=":octocat:" src="https://github.githubassets.com/images/icons/emoji/octocat.png" height="20" width="20" align="absmiddle"> <a href="https://github.com/JuliaManifolds/ManifoldDiff.jl">GitHub repository</a>
</li>
</ul>
