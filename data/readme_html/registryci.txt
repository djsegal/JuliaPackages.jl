<p>RegistryCI.jl
provides continuous integration (CI) tools for Julia package registries, including registry consistency testing, automatic merging (automerge) of pull requests, and automatic TagBot triggers.</p>
<p>Please see the <a href="https://JuliaRegistries.github.io/RegistryCI.jl/stable" rel="nofollow">documentation</a>.</p>
