<p dir="auto">This package contains Julia translations of original scripts by Elizabeth Dolan, Jorge Moré and Stefan Wild.
See <a href="http://www.mcs.anl.gov/~wild/dfo/benchmarking" rel="nofollow">http://www.mcs.anl.gov/~wild/dfo/benchmarking</a>.</p>
<p dir="auto">The original code was not accompanied by an open-source license. Jorge Moré and Stefan Wild have kindly provided their consent in writing to allow distribution of this Julia translation.
See the <code>consent</code> folder for a full transcription.</p>
<p dir="auto">It appears that performance profiles date back to <a href="https://dx.doi.org/10.1109/9.539425" rel="nofollow">at least 1996</a>!</p>
<p dir="auto">Watch out for the <a href="https://dl.acm.org/citation.cfm?id=2950048" rel="nofollow">pitfalls</a> of profiles!</p>

<div class="highlight highlight-source-julia notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="julia&gt; Pkg.add(&quot;BenchmarkProfiles&quot;)"><pre>julia<span class="pl-k">&gt;</span> Pkg<span class="pl-k">.</span><span class="pl-c1">add</span>(<span class="pl-s"><span class="pl-pds">"</span>BenchmarkProfiles<span class="pl-pds">"</span></span>)</pre></div>
<p dir="auto">No plotting backend is loaded by default so the user can choose among several available plotting backends.
Currently, <a href="https://github.com/JuliaPlots/Plots.jl">Plots.jl</a> and <a href="https://github.com/Evizero/UnicodePlots.jl">UnicodePlots.jl</a> are supported.
Backends become available when the corresponding package is imported.</p>
