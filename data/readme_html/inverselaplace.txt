<p>This implements some numerical methods for computing inverse Laplace transforms in Julia.</p>
<p>InverseLaplace v0.1.0 is the last version that supports Julia v0.6</p>
<p>See the documentation <a href="https://jlapeyre.github.io/InverseLaplace.jl/latest" rel="nofollow">https://jlapeyre.github.io/InverseLaplace.jl/latest</a> .</p>
<p>Note: A small part of <code>InverseLaplace.jl</code> depends on <code>Optim.jl</code>, which is currently
broken on Julia v0.7. So optimization is disabled for <code>InverseLaplace.jl</code> versions
greater than v0.1.0</p>
<p>Note: the last version of this module supporting Julia v0.4 is tagged v0.0.2.
the last version of this module supporting Julia v0.6 is tagged v0.1.0.</p>
