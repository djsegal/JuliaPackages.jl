<h1><a id="user-content-finetools-finite-element-tools-in-julia" class="anchor" aria-hidden="true" href="#finetools-finite-element-tools-in-julia"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path fill-rule="evenodd" d="M7.775 3.275a.75.75 0 001.06 1.06l1.25-1.25a2 2 0 112.83 2.83l-2.5 2.5a2 2 0 01-2.83 0 .75.75 0 00-1.06 1.06 3.5 3.5 0 004.95 0l2.5-2.5a3.5 3.5 0 00-4.95-4.95l-1.25 1.25zm-4.69 9.64a2 2 0 010-2.83l2.5-2.5a2 2 0 012.83 0 .75.75 0 001.06-1.06 3.5 3.5 0 00-4.95 0l-2.5 2.5a3.5 3.5 0 004.95 4.95l1.25-1.25a.75.75 0 00-1.06-1.06l-1.25 1.25a2 2 0 01-2.83 0z"></path></svg></a>FinEtools: Finite Element tools in Julia</h1>
<p><code>FinEtools</code> is a package for basic operations on finite element meshes: Construction, modification, selection, and evaluation of quantities defined on a mesh. Utilities are provided for maintaining mesh-based data (fields), for defining normals and loads, for working with physical units and coordinate systems, and for integrating over finite element meshes. <a target="_blank" rel="noopener noreferrer" href="https://camo.githubusercontent.com/13828e57777c63497d79af33dd6cc42c6039882215424d84add192a253d7ae30/687474703a2f2f686f6777617274732e756373642e6564752f7e706b7279736c2f736974652e696d616765732f626166666c65642d706973746f6e2d61612e706e67"><img src="https://camo.githubusercontent.com/13828e57777c63497d79af33dd6cc42c6039882215424d84add192a253d7ae30/687474703a2f2f686f6777617274732e756373642e6564752f7e706b7279736c2f736974652e696d616765732f626166666c65642d706973746f6e2d61612e706e67" alt="Alt Visualization of acoustic pressure" title="FinEtools.jl" data-canonical-src="http://hogwarts.ucsd.edu/~pkrysl/site.images/baffled-piston-aa.png" style="max-width:100%;"></a></p>
<p>The package supports application packages, for instance:</p>
<ul>
<li><a href="https://github.com/PetrKryslUCSD/FinEtoolsMeshing.jl">Meshing</a>;</li>
<li><a href="https://github.com/PetrKryslUCSD/FinEtoolsVoxelMesher.jl">Meshing of biomedical images</a>;</li>
<li><a href="https://github.com/PetrKryslUCSD/FinEtoolsAcoustics.jl">Linear acoustics</a>;</li>
<li><a href="https://github.com/PetrKryslUCSD/FinEtoolsHeatDiff.jl">Heat conduction</a>;</li>
<li><a href="https://github.com/PetrKryslUCSD/FinEtoolsDeforLinear.jl">Linear stress analysis</a>;</li>
<li><a href="https://github.com/PetrKryslUCSD/FinEtoolsDeforNonlinear.jl">Nonlinear stress analysis</a>;</li>
<li><a href="https://github.com/PetrKryslUCSD/FinEtoolsFlexBeams.jl">Nonlinear analysis of flexible beam structures</a>;</li>
<li><a href="https://github.com/PetrKryslUCSD/FinEtoolsVibInFluids.jl">Vibration in fluids</a>.</li>
</ul>
