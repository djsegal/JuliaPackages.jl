<h1><a id="user-content-finetools-finite-element-tools-in-julia" class="anchor" aria-hidden="true" href="#finetools-finite-element-tools-in-julia"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path fill-rule="evenodd" d="M4 9h1v1H4c-1.5 0-3-1.69-3-3.5S2.55 3 4 3h4c1.45 0 3 1.69 3 3.5 0 1.41-.91 2.72-2 3.25V8.59c.58-.45 1-1.27 1-2.09C10 5.22 8.98 4 8 4H4c-.98 0-2 1.22-2 2.5S3 9 4 9zm9-3h-1v1h1c1 0 2 1.22 2 2.5S13.98 12 13 12H9c-.98 0-2-1.22-2-2.5 0-.83.42-1.64 1-2.09V6.25c-1.09.53-2 1.84-2 3.25C6 11.31 7.55 13 9 13h4c1.45 0 3-1.69 3-3.5S14.5 6 13 6z"></path></svg></a>FinEtools: Finite Element tools in Julia</h1>
<p><code>FinEtools</code> is a package for basic operations on finite element meshes: Construction, modification, selection, and evaluation of quantities defined on a mesh. Utilities are provided for maintaining mesh-based data (fields), for defining normals and loads, for working with physical units and coordinate systems, and for integrating over finite element meshes. <a target="_blank" rel="noopener noreferrer" href="https://camo.githubusercontent.com/e928bd673f655fa53c493fab76f7f4f5625cd853/687474703a2f2f686f6777617274732e756373642e6564752f7e706b7279736c2f736974652e696d616765732f626166666c65642d706973746f6e2d61612e706e67"><img src="https://camo.githubusercontent.com/e928bd673f655fa53c493fab76f7f4f5625cd853/687474703a2f2f686f6777617274732e756373642e6564752f7e706b7279736c2f736974652e696d616765732f626166666c65642d706973746f6e2d61612e706e67" alt="Alt Visualization of acoustic pressure" title="FinEtools.jl" data-canonical-src="http://hogwarts.ucsd.edu/~pkrysl/site.images/baffled-piston-aa.png" style="max-width:100%;"></a></p>
<p>The package supports application packages, for instance:</p>
<ul>
<li><a href="https://github.com/PetrKryslUCSD/FinEtoolsMeshing.jl">Meshing</a>;</li>
<li><a href="https://github.com/PetrKryslUCSD/FinEtoolsVoxelMesher.jl">Meshing of biomedical images</a>;</li>
<li><a href="https://github.com/PetrKryslUCSD/FinEtoolsAcoustics.jl">Linear acoustics</a>;</li>
<li><a href="https://github.com/PetrKryslUCSD/FinEtoolsHeatDiff.jl">Heat conduction</a>;</li>
<li><a href="https://github.com/PetrKryslUCSD/FinEtoolsDeforLinear.jl">Linear stress analysis</a>;</li>
<li><a href="https://github.com/PetrKryslUCSD/FinEtoolsDeforNonlinear.jl">Nonlinear stress analysis</a>;</li>
<li><a href="https://github.com/PetrKryslUCSD/FinEtoolsFlexBeams.jl">Nonlinear analysis of flexible beam structures</a>;</li>
<li><a href="https://github.com/PetrKryslUCSD/FinEtoolsVibInFluids.jl">Vibration in fluids</a>.</li>
</ul>

<p>Tutorials in the form
of marked-down Julia source files using the
<a href="https://github.com/fredrikekre/Literate.jl">Literate</a> workflow are available
and more will  be added in the near future. (Each application package has some tutorials. For instance for the <a href="https://github.com/PetrKryslUCSD/FinEtoolsDeforLinear.jl/tree/master/tutorials">linear deformation</a>.)</p>
<p>The package has been used to build applications for various purposes. In
addition to the list at the top, there are also examples of various  <a href="https://github.com/PetrKryslUCSD/FinEtoolsMeshGenerationExamples.git">mesh
generation
tasks</a>).</p>
<p>The documentation  is published as <a href="https://petrkryslucsd.github.io/FinEtools.jl/latest/" rel="nofollow">Github pages</a>.</p>
