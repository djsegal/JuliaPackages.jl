<p dir="auto"><code>FinEtools</code> is a package for basic operations on finite element meshes: Construction, modification, selection, and evaluation of quantities defined on a mesh. Utilities are provided for maintaining mesh-based data (fields), for defining normals and loads, for working with physical units and coordinate systems, and for integrating over finite element meshes. <a target="_blank" rel="noopener noreferrer nofollow" href="https://camo.githubusercontent.com/13828e57777c63497d79af33dd6cc42c6039882215424d84add192a253d7ae30/687474703a2f2f686f6777617274732e756373642e6564752f7e706b7279736c2f736974652e696d616765732f626166666c65642d706973746f6e2d61612e706e67"><img src="https://camo.githubusercontent.com/13828e57777c63497d79af33dd6cc42c6039882215424d84add192a253d7ae30/687474703a2f2f686f6777617274732e756373642e6564752f7e706b7279736c2f736974652e696d616765732f626166666c65642d706973746f6e2d61612e706e67" alt="Alt Visualization of acoustic pressure" title="FinEtools.jl" data-canonical-src="http://hogwarts.ucsd.edu/~pkrysl/site.images/baffled-piston-aa.png" style="max-width: 100%;"></a></p>
<p dir="auto">The package supports application packages, for instance:</p>
<ul dir="auto">
<li><a href="https://github.com/PetrKryslUCSD/FinEtoolsMeshing.jl">Meshing</a>;</li>
<li><a href="https://github.com/PetrKryslUCSD/FinEtoolsVoxelMesher.jl">Meshing of biomedical images</a>;</li>
<li><a href="https://github.com/PetrKryslUCSD/FinEtoolsAcoustics.jl">Linear acoustics</a>;</li>
<li><a href="https://github.com/PetrKryslUCSD/FinEtoolsHeatDiff.jl">Heat conduction</a>;</li>
<li><a href="https://github.com/PetrKryslUCSD/FinEtoolsDeforLinear.jl">Linear stress analysis</a>;</li>
<li><a href="https://github.com/PetrKryslUCSD/FinEtoolsDeforNonlinear.jl">Nonlinear stress analysis</a>;</li>
<li><a href="https://github.com/PetrKryslUCSD/FinEtoolsFlexStructures.jl">Analysis of flexible beam and shell structures</a>;</li>
<li><a href="https://github.com/PetrKryslUCSD/FinEtoolsVibInFluids.jl">Vibration in fluids</a>.</li>
</ul>

<ul dir="auto">
<li>04/20/2023: Make all types in the library generic.</li>
<li>04/18/2023: Implemented resizing of assembly buffers. Makematrix! resets pointers.</li>
<li>04/16/2023: Enabled creation of finite element sets from arbitrary arrays.</li>
<li>03/15/2023: Changed strategy when assembling into the COO format.</li>
<li>03/10/2023: Genericity of arguments enabled in the assembly module.</li>
<li>12/27/2022: Added T4 and T10 cylinder meshing routines.</li>
<li>11/03/2022: Added extraction of outer surface of solid.</li>
<li>06/29/2022: Reconfigured documentation.</li>
<li>06/15/2022: Added point partitioning.</li>
<li>06/15/2022: Added dual connection matrix implementation.</li>
<li>05/03/2022: Updated project to Julia 1.7.2: Julia 1.6 is not supported from version 5.4.0 on.</li>
<li>01/26/2022: Added assembler for sparse diagonal matrices (such as the mass
matrix for explicit finite element methods).</li>
<li>01/25/2022: Added export of a ParaView collection of a time sequence of data.</li>
