<p dir="auto">SMLMFrameConnection performs frame-connection on localization microscopy data organized in an SMLMData.SMLD2D structure (<a href="https://github.com/JuliaSMLM/SMLMData.jl">https://github.com/JuliaSMLM/SMLMData.jl</a>).
Specifically, SMLMFrameConnection connects repeated localizations of a single blinking event of an emitter into a single higher precision localization.  This is done
using the algorithm(s) presented in <a href="https://doi.org/10.3389/fbinf.2021.724325" rel="nofollow">https://doi.org/10.3389/fbinf.2021.724325</a>.  Localizations which were connected will share the same unique integer value for the field
SMLMData.SMLD2D.connectID.</p>

<p dir="auto">Once an SMLMData.SMLD2D structure is fully populated, the user only needs to run a single high-level method from the package: <code>frameconnect()</code>.
All fields of the SMLMData.SMLD2D structure must be populated with either meaningful values (e.g., for fields like <code>x</code>, <code>y</code>, <code>ﾏダx</code>, <code>ﾏダy</code>, and <code>framenum</code>,
which the algorithm depends on) or by placeholders with a meaningful size (e.g., fields like <code>bg</code> and <code>ﾏダbg</code>, which may not be available, should be set to something like
<code>smld.bg = zeros(Float64, length(smld.framenum))</code>, and <code>ﾏダbg = fill(Inf64, length(smld.framenum))</code>).</p>
<p dir="auto">The algorithm can be run on the fully populated <code>smld::SMLMData.SMLD2D</code> with default parameters as</p>
<p dir="auto"><code>smld_connected, smld_preclustered, smld_combined, params = SMLMFrameConnection.frameconnect(smld)</code></p>
<p dir="auto">The output <code>smld_connected</code> is a copy of <code>smld</code> with the field <code>smld.connectID</code> updated to associate connected localizations.  <code>smld_preclustered</code> is a copy of <code>smld</code>
with the field <code>smld_preclustered.connectID</code> updated to associate localizations that belonged to the same precluster.  <code>smld_combined</code> contains the combined higher
precision localizations (i.e., <code>smld_combined=SMLMFrameConnection.combinelocalizations(smld_connected)</code>) and is considered the main output of this package.  <code>params</code>
is a structure containing the user-defined and internally-defined parameters used in the algorithm.</p>
<p dir="auto">Several user-defined parameters can be changed as optional keyword arguments to <code>frameconnect()</code>:</p>
