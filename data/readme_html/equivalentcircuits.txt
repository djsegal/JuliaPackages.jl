<p>This Julia module allows users to analyse their electrochemical impedance spectroscopy data using Equivalent electrical circuits. EquivalentCircuits.jl can be used to either find the optimal paramters of a given equivalent electrical circuit , or to get recommendations for an appropriate equivalent electrical circuit configuration. The latter is based on a genetic programming approach.</p>
