<p dir="auto"><code>RaySolver</code> is a differentiable 2½D Gaussian beam tracer for use with <a href="https://github.com/org-arl/UnderwaterAcoustics.jl"><code>UnderwaterAcoustics.jl</code></a>.
It is similar to <a href="http://oalib.hlsresearch.com/AcousticsToolbox/" rel="nofollow">Bellhop</a>, but fully written in Julia to be compatible with automatic differentiation (AD)
tool such as <code>ForwardDiff</code> (compatibility with other AD packages such as <code>ReverseDiff</code> and <code>Zygote</code> is not fully tested).</p>

<p dir="auto">The propagation modeling API is detailed in the <a href="https://org-arl.github.io/UnderwaterAcoustics.jl/stable/" rel="nofollow">UnderwaterAcoustics</a> documentation.
We assume that the reader is familiar with it. This documentation only provides guidance on specific use of <code>RaySolver</code> propagation model.</p>
<p dir="auto">Additional options available with <code>RaySolver</code>:</p>
<ul dir="auto">
<li><code>nbeams</code> -- number of beams used for ray tracing (default: auto)</li>
<li><code>minangle</code> -- minimum beam angle in radians (default: -80°)</li>
<li><code>maxangle</code> -- maximum beam angle in radians (default: 80°)</li>
<li><code>ds</code> -- ray trace step size in meters (default: 1.0)</li>
<li><code>atol</code> -- absolute tolerance of differential equation solver (default: 1e-4)</li>
<li><code>rugocity</code> -- measure of small-scale variations of surfaces (default: 1.5)</li>
<li><code>athreshold</code> -- relative amplitude threshold below which rays are dropped (default: 1e-5)</li>
<li><code>solver</code> -- differential equation solver (default: auto)</li>
<li><code>solvertol</code> -- differential equation solver tolerance (default: 1e-4)</li>
</ul>
