<p dir="auto">Features:</p>
<ul dir="auto">
<li>Matching against almost any data type with a first-match policy</li>
<li>Deep matching within data types and matrices</li>
<li>Variable binding within matches</li>
</ul>
<p dir="auto">For alternatives to <code>Match</code>, check out</p>
<ul dir="auto">
<li>toivoh's <a href="https://github.com/toivoh/PatternDispatch.jl"><code>PatternDispatch.jl</code></a> for a more Julia-like function dispatch on patterns.</li>
</ul>

<p dir="auto">The package provides one macro, <code>@match</code>, which can be used as:</p>
<div class="snippet-clipboard-content notranslate position-relative overflow-auto" data-snippet-clipboard-copy-content="using Match

@match item begin
    pattern1              =&gt; result1
    pattern2, if cond end =&gt; result2
    pattern3 || pattern4  =&gt; result3
    _                     =&gt; default_result
end"><pre class="notranslate"><code>using Match

@match item begin
    pattern1              =&gt; result1
    pattern2, if cond end =&gt; result2
    pattern3 || pattern4  =&gt; result3
    _                     =&gt; default_result
end
</code></pre></div>
<p dir="auto">It is possible to supply variables inside pattern, which will be bound
to corresponding values.</p>
<p dir="auto">See the <a href="http://kmsquire.github.io/Match.jl/latest/" rel="nofollow">documentation</a>
for examples of this and other features.</p>
