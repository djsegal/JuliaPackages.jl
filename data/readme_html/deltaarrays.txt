<p dir="auto">This Julia library provides <code>DeltaArray</code>, an efficient N-dimensional <code>Diagonal</code> array type. If your array <math-renderer class="js-inline-math" style="display: inline" data-static-url="https://github.githubassets.com/static" data-run-id="abb4bc74488381984cbb767fd72fe378">$A$</math-renderer> is of the form</p>
<p dir="auto"><math-renderer class="js-display-math" style="display: block" data-static-url="https://github.githubassets.com/static" data-run-id="abb4bc74488381984cbb767fd72fe378">$$
A = a_i \delta_{i \dots j} = \begin{cases}
a_i, &amp;\text{if} ~~ i=\dots=j \\
0, &amp;\text{otherwise}
\end{cases}
$$</math-renderer></p>
<p dir="auto">then it can be represented by a <code>DeltaArray</code>.</p>
<p dir="auto">For compatibility, <code>DeltaArrays{T,2}</code> should just behave like <code>Diagonal{T}</code>.</p>
