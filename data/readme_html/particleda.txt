<p dir="auto"><code>ParticleDA.jl</code> is a Julia package to run data assimilation with particle filter
distributed using MPI.</p>

<p dir="auto">To install the latest stable version of the package, open the <a href="https://docs.julialang.org/en/v1/stdlib/REPL/" rel="nofollow">Julia
REPL</a>, enter the package manager
with <code>]</code>, then run the command</p>
<div class="snippet-clipboard-content notranslate position-relative overflow-auto" data-snippet-clipboard-copy-content="add ParticleDA"><pre class="notranslate"><code>add ParticleDA
</code></pre></div>
<p dir="auto">If you plan to develop the package (make changes, submit pull requests, etc), in
the package manager mode run this command</p>
<div class="snippet-clipboard-content notranslate position-relative overflow-auto" data-snippet-clipboard-copy-content="dev ParticleDA"><pre class="notranslate"><code>dev ParticleDA
</code></pre></div>
<p dir="auto">This will automatically clone the repository to your local directory
<code>~/.julia/dev/ParticleDA</code>.</p>
<p dir="auto">You can exit from the package manager mode by pressing <code>CTRL + C</code> or,
alternatively, the backspace key when there is no input in the prompt.</p>
