<p><a target="_blank" rel="noopener noreferrer" href="https://camo.githubusercontent.com/3a6e722dd761f09591839eda47cc97c404abc701631f0c5ae26cd439de07d47f/68747470733a2f2f7472617669732d63692e6f72672f6d69742d6e6c702f4f6c6c616d2e6a6c2e7376673f6272616e63683d6d6173746572"><img align="right" src="https://camo.githubusercontent.com/3a6e722dd761f09591839eda47cc97c404abc701631f0c5ae26cd439de07d47f/68747470733a2f2f7472617669732d63692e6f72672f6d69742d6e6c702f4f6c6c616d2e6a6c2e7376673f6272616e63683d6d6173746572" alt="Build Status" data-canonical-src="https://travis-ci.org/mit-nlp/Ollam.jl.svg?branch=master" style="max-width:100%;"></a></p>
<p>This package is an implementation of:</p>
<ol>
<li>MIRA and PA algorithms (Krammer and Singer 2003)</li>
<li>Averaged Perceptron (Collins 2001)</li>
</ol>
<p>It is also contains a LIBSVM wrapper for linear models (mainly for
comparison and initialization).</p>
