<li>Support classical spin dynamics and micromagnetic simulations.</li>
<li>Support both CPU and GPU, which can be easily switched.</li>
<li>Easily switch between single and double using <code>JuMag.cuda_using_double(false)</code></li>
<li>...</li>
</ul>

<p dir="auto">Install JuMag is straightforward as long as  Julia is installed, and it is equally easy in Windows, Linux and Mac.</p>
<p dir="auto">Requirements:</p>
<ul dir="auto">
<li>Julia 1.4 (or above) (<a href="http://julialang.org/downloads/" rel="nofollow">http://julialang.org/downloads/</a>)</li>
<li>Some packages such as FFTW, WriteVTK and NPZ</li>
<li>CUDA.jl (<a href="https://github.com/JuliaGPU/CUDA.jl">https://github.com/JuliaGPU/CUDA.jl</a>) (Needed for GPU support)</li>
</ul>
<p dir="auto">In <a href="http://julialang.org" rel="nofollow">Julia</a>, packages can be easily installed using</p>
<div class="snippet-clipboard-content notranslate position-relative overflow-auto" data-snippet-clipboard-copy-content="using Pkg;
Pkg.add(&quot;FFTW&quot;)"><pre class="notranslate"><code>using Pkg;
Pkg.add("FFTW")
</code></pre></div>
<p dir="auto">or</p>
<div class="snippet-clipboard-content notranslate position-relative overflow-auto" data-snippet-clipboard-copy-content="julia&gt; ]
(v1.4) pkg&gt; add FFTW"><pre class="notranslate"><code>julia&gt; ]
(v1.4) pkg&gt; add FFTW
</code></pre></div>
<p dir="auto">We don't have to install these packages for now, since the packages will be installed automatically when we install JuMag.
To install <a href="https://github.com/ww1g11/JuMag.jl">JuMag.jl</a>, simply using</p>
<div class="snippet-clipboard-content notranslate position-relative overflow-auto" data-snippet-clipboard-copy-content="(v1.4) pkg&gt; add https://github.com/ww1g11/JuMag.jl"><pre class="notranslate"><code>(v1.4) pkg&gt; add https://github.com/ww1g11/JuMag.jl
</code></pre></div>
<p dir="auto">Now we will see similar messages if we type <code>using JuMag</code></p>
