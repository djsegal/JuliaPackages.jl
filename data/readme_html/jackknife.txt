<p dir="auto">None of the functions here are exported, so you'll have to call them with the prefix
<code>Jackknife.</code> or explicitly import them.</p>
<p dir="auto">Each function takes the following two arguments:</p>
<ul dir="auto">
<li>
<p dir="auto">A point estimator, given as a <code>Function</code>.
The function must return a scalar when passed a vector.</p>
</li>
<li>
<p dir="auto">A real-valued vector of length &gt; 1.</p>
</li>
</ul>

<div class="highlight highlight-source-julia notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="leaveoneout(estimator, x)"><pre><span class="pl-c1">leaveoneout</span>(estimator, x)</pre></div>
<p dir="auto">Compute a vector of point estimates based on systematic subsamples of <code>x</code> wherein
each index is omitted one at a time.
These are the "leave-one-out" estimates.
The resulting vector will have length <code>length(x) - 1</code>.</p>
