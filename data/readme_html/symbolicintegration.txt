<p dir="auto">This package provides Julia implementations of symbolic integration algorithms.</p>
<p dir="auto">The front-end (i.e., the user interface) requires <a href="https://symbolicutils.juliasymbolics.org/" rel="nofollow">SymbolicUtils.jl</a>.
The actual integration algorithms are implemented in a generic way using <a href="https://nemocas.github.io/AbstractAlgebra.jl/dev/" rel="nofollow">AbstractAlgebra.jl</a>.
Some algorithms require <a href="https://nemocas.github.io/Nemo.jl/dev/" rel="nofollow">Nemo.jl</a> for calculations with algebraic numbers.</p>
<p dir="auto"><code>SymbolicIntegration.jl</code> is based on the algorithms from the book</p>
<blockquote>
<p dir="auto">Manuel Bronstein, <a href="https://link.springer.com/book/10.1007/b138171" rel="nofollow">Symbolic Integration I: Transcentental Functions</a>, 2nd ed, Springer 2005,</p>
</blockquote>
<p dir="auto">for which a pretty complete set of reference implementations is provided.
Currently, <code>SymbolicIntegration.jl</code> can integrate rational functions and integrands involving transcendental elementary
functions like <code>exp</code>, <code>log</code>, <code>sin</code>, etc.
As in the book, integrands involving algebraic functions like <code>sqrt</code> and non-integer powers are not treated.</p>
<p dir="auto">Note that <code>SymbolicIntegration.jl</code> is still in an early stage of development and should not be expected to run stably in all situations.
It comes with absolutely no warranty whatsoever.</p>
