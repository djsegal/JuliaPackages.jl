<p>An incredibly simple logger!
All it does is to log all outputs of <code>print</code> and <code>println</code> into a file.</p>
<p>Usage:</p>
<div class="highlight highlight-source-julia"><pre>julia<span class="pl-k">&gt;</span> <span class="pl-k">using</span> PrintLog

julia<span class="pl-k">&gt;</span> <span class="pl-c1">@printlog</span> <span class="pl-s"><span class="pl-pds">"</span>junk.log<span class="pl-pds">"</span></span>
INFO<span class="pl-k">:</span> <span class="pl-s"><span class="pl-pds">`</span>print<span class="pl-pds">`</span></span> and <span class="pl-s"><span class="pl-pds">`</span>println<span class="pl-pds">`</span></span> will be logged into file <span class="pl-s"><span class="pl-pds">`</span>bf.log<span class="pl-pds">`</span></span>

julia<span class="pl-k">&gt;</span> <span class="pl-c1">println</span>(<span class="pl-s"><span class="pl-pds">"</span>junk<span class="pl-pds">"</span></span>)
junk

julia<span class="pl-k">&gt;</span> <span class="pl-c1">@noprintlog</span>
INFO<span class="pl-k">:</span> <span class="pl-s"><span class="pl-pds">`</span>print<span class="pl-pds">`</span></span> and <span class="pl-s"><span class="pl-pds">`</span>println<span class="pl-pds">`</span></span> are resumed.

julia<span class="pl-k">&gt;</span> <span class="pl-c1">println</span>(<span class="pl-s"><span class="pl-pds">"</span>junk<span class="pl-pds">"</span></span>)
junk</pre></div>
