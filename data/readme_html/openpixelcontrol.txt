<p dir="auto">This package provides a <a href="https://julialang.org" rel="nofollow">Julia</a> port of the <a href="https://raw.githubusercontent.com/zestyping/openpixelcontrol/master/python/opc.py" rel="nofollow">Python</a> of the <a href="https://github.com/zestyping/openpixelcontrol">openpixelcontrol</a> Client for the streaming protocol for controlling RGB LEDs.</p>

<p dir="auto">Then, to set the first pixel to red, the second to blue after activating interpolation,
you can use the following small code</p>
