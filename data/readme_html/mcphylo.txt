<p dir="auto">This package offers the functionality for phylogenetic computations in Julia. It is an extension of the <code>Mamba</code> package which implements different variants of Markov Chain Monte Carlo (MCMC) sampling for Bayesian analysis. (<a href="https://mambajl.readthedocs.io/en/latest/" rel="nofollow">https://mambajl.readthedocs.io/en/latest/</a>)
<strong>MCPhylo</strong> extends <code>Mamba</code> to perform phylogenetic computations.</p>
<p dir="auto"><strong>This module is build on a forked instance of Mamba 0.12.0</strong></p>
<p dir="auto">The documentation can be found <a href="https://erathorn.github.io/MCPhylo.jl/dev/" rel="nofollow">here</a>.</p>
