<p dir="auto">The <code>ChaosCommunications</code> package provides a framework for simulating chaos-based communication systems.</p>
<p dir="auto">At the moment the package provides a generic type hierarchy of communication systems and implements the
Chaos Shift-Keying (CSK) system. More functionality and systems will be added to the package.</p>
<p dir="auto">An example for simulating CSK and its Bit Error Rate (BER) can be found in <code>examples/UTURUniChannelCSK_example.jl</code>. This example
demonstrates the main interface of the package.</p>
