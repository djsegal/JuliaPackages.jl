<p dir="auto">The present package aims to support a querying framework that is</p>
<ul dir="auto">
<li>
<p dir="auto">Generic -- the framework should be able to support multiple backends.</p>
</li>
<li>
<p dir="auto">Modular -- the framework should encourage modularity of collection machinery.</p>
</li>
<li>
<p dir="auto">Extensible -- the framework should be easily extensible to represent (relatively) arbitrary manipulations.</p>
</li>
<li>
<p dir="auto">Ergonomic -- the framework should allow users to express their intentions easily.</p>
</li>
</ul>
