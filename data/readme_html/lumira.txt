<p dir="auto">Julia Library which allows developing Lumira extensions using Gadfly.</p>
<p dir="auto">Gadfly is a drawing library package written in julia which implements the grammar of graphics approach</p>
<p dir="auto">SAP Lumira is a visualization tool developed by SAP SE &amp; can be downloaded from                                                     <a href="https://websmp201.sap-ag.de/home" rel="nofollow">SAP Service Marketplace</a>.</p>
<p dir="auto">#Installation
- Pkg.add("Lumira")
- using Lumira</p>
<p dir="auto">#Examples</p>
<ul dir="auto">
<li>Check the examples in examples directory</li>
<li>On executing it, it generates a SAP Lumira Extension</li>
<li>Other sample .jpg files provided in images directory</li>
</ul>
