<p>This package provides example models for use with the <a href="https://github.com/awllee/SequentialMonteCarlo.jl">SequentialMonteCarlo.jl</a> package.</p>
<p>Some of the models in src/ are also used to test SequentialMonteCarlo.jl, particularly the Finite Feynman--Kac model which is otherwise not likely to be very interesting.</p>
<p>The demos in demo/ can be run as scripts or in the REPL. Some involve plotting. They are intended only to show how to run the algorithm and look at some of its outputs.</p>
<p>The tests in test/ also demonstrate some aspects of usage.</p>
