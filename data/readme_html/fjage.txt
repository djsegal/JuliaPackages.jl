<p dir="auto"><a href="https://github.com/org-arl/fjage">fjåge</a> provides a lightweight and
easy-to-learn framework for agent-oriented software development in Java and
Groovy. <code>Fjage.jl</code> is a Julia port of fjåge, which can be independently used,
or used in conjunction with a Java/Groovy container in a multi-language application.</p>

<ul dir="auto">
<li>Lightweight and fast</li>
<li>Easy to learn, and rapid agent development cycle</li>
<li>Agent development in Julia, Java or Groovy</li>
