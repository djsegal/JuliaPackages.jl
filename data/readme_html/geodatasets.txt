<p>The aim of this package is to give access to common geographics datasets. Currently it contains:</p>
<ul>
<li>the global land-sea-river mask from <a href="http://gmt.soest.hawaii.edu/" rel="nofollow">GMT</a></li>
<li>the <a href="https://www.soest.hawaii.edu/pwessel/gshhg/" rel="nofollow">Global Self-consistent, Hierarchical, High-resolution Geography Database (gshhg)</a>.</li>
</ul>
<p>Contributions are most welcome. Here are some guidelines:</p>
<ul>
<li>data sets should be global in scope</li>
<li>available under a license allowing redistribution</li>
<li>hosted on a publicly available server</li>
</ul>
