<p dir="auto">A Julia package that can estimate indicators of transitions (from one dynamic regime or stable state to another) in timeseries. Also bundles the indicators with significance testing via surrogate analysis using <a href="https://github.com/JuliaDynamics/TimeseriesSurrogates.jl">TimeseriesSurrogates.jl</a>. Alternative names for this package could have been: Early Warning Signals / Resilience Indicators / Regime-Shift Identifiers / Change-Point Detectors, or however else you want to call them!</p>
