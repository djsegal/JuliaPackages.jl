<p dir="auto">SpineOpt is designed to be used with <a href="https://github.com/spine-tools/Spine-Toolbox">Spine Toolbox</a>.</p>
<ol dir="auto">
<li>
<p dir="auto">Install Spine Toolbox as described <a href="https://github.com/spine-tools/Spine-Toolbox/blob/master/README.md#installation">here</a>.</p>
</li>
<li>
<p dir="auto">Download and install the latest version of Julia for your system as described <a href="https://julialang.org/downloads" rel="nofollow">here</a>.</p>
</li>
<li>
<p dir="auto">Install SpineOpt using <em>one</em> of the below options:</p>
<p dir="auto">a. If you want to <em>use</em> SpineOpt but not develop it,
we recommend installing it from the <a href="https://github.com/spine-tools/SpineJuliaRegistry">Spine Julia Registry</a>:</p>
<ol dir="auto">
<li>Start the <a href="https://github.com/spine-tools/SpineOpt.jl/raw/master/docs/src/figs/win_run_julia.png">Julia REPL</a>.</li>
<li>Copy and paste the following text into the julia prompt:
<div class="highlight highlight-source-julia notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="using Pkg
pkg&quot;registry add General https://github.com/spine-tools/SpineJuliaRegistry&quot;

<span class="pl-s"><span class="pl-pds"><span class="pl-c1">pkg</span>"</span>registry add General https://github.com/spine-tools/SpineJuliaRegistry<span class="pl-pds">"</span></span>
<span class="pl-s"><span class="pl-pds"><span class="pl-c1">pkg</span>"</span>add SpineOpt<span class="pl-pds">"</span></span></pre></div>
</li>
</ol>
<p dir="auto">b. If you want to both use and develop SpineOpt, we recommend installing it from sources:</p>
<ol dir="auto">
<li>Git-clone this repository into your local machine.</li>
<li>Git-clone the <a href="https://github.com/spine-tools/SpineInterface.jl">SpineInterface repository</a> into your local machine.</li>
<li>Start the <a href="https://github.com/spine-tools/SpineOpt.jl/raw/master/docs/src/figs/win_run_julia.png">Julia REPL</a>.</li>
<li>Run the following commands from the julia prompt, replacing your local SpineOpt and SpineInterface paths
<div class="highlight highlight-source-julia notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="using Pkg
pkg&quot;dev &lt;path-to-your-local-SpineInterface-repository&gt;&quot;
pkg&quot;dev &lt;path-to-your-local-SpineOpt-repository&gt;&quot;"><pre><span class="pl-k">using</span> Pkg
<span class="pl-s"><span class="pl-pds"><span class="pl-c1">pkg</span>"</span>dev &lt;path-to-your-local-SpineInterface-repository&gt;<span class="pl-pds">"</span></span>
<span class="pl-s"><span class="pl-pds"><span class="pl-c1">pkg</span>"</span>dev &lt;path-to-your-local-SpineOpt-repository&gt;<span class="pl-pds">"</span></span></pre></div>
</li>
</ol>
</li>
<li>
<p dir="auto">Configure Spine Toolbox to use your Julia:</p>
<p dir="auto">a. Run Spine Toolbox.</p>
<p dir="auto">b. Go to <strong>File</strong> -&gt; <strong>Settings</strong> -&gt; <strong>Tools</strong>.</p>
<p dir="auto">c. Under <strong>Julia</strong>, enter the path to your Julia executable. It should look something like <a href="https://github.com/spine-tools/SpineOpt.jl/raw/master/docs/src/figs/spinetoolbox_settings_juliaexe.png">this</a>.  In case you have multiple Julia's in your system, the path should point to the same Julia version as is in your environment PATH.</p>
<p dir="auto">d. Press <strong>Ok</strong>.</p>
</li>
</ol>

<p dir="auto">Sometimes it can be useful to 'compile' SpineOpt into a so-called system image. A system image is a binary library
that, roughly speaking, 'stores' all the compilation work from a previous Julia session.
If you start Julia with a system image, then Julia doesn't need to redo all that work and your code will be fast the
first time you run it.</p>
<p dir="auto"><strong>However</strong> if you upgrade your version of
SpineOpt, any system images you might have created will not reflect that change - you will need to re-generate them.</p>
<p dir="auto">To compile SpineOpt into a system image just do the following:</p>
<ol dir="auto">
<li>
<p dir="auto">Install <a href="https://github.com/JuliaLang/PackageCompiler.jl">PackageCompiler.jl</a>.</p>
</li>
<li>
<p dir="auto">Create a file with precompilation statements for SpineOpt:</p>
<p dir="auto">a. Start julia with <code>--trace-compile=file.jl</code>.</p>
<p dir="auto">b. Call <code>run_spineopt(url...)</code> with a nice DB - one that triggers most of SpineOpt's functionality you need.</p>
<p dir="auto">c. Quit julia.</p>
</li>
<li>
<p dir="auto">Create the sysimage using the precompilation statements file:</p>
<p dir="auto">a. Start julia normally.</p>
<p dir="auto">b. Create the sysimage with PackageCompiler:</p>
<div class="highlight highlight-source-julia notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="using PackageCompiler
