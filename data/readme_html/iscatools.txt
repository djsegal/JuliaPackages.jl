<p><code>IscaTools.jl</code> is a Julia package that aims to streamline the following processes:</p>
<ul>
<li>Management of output from the <a href="https://execlim.github.io/IscaWebsite/" rel="nofollow">Isca</a> GCM developed by the University of Exeter</li>
<li>Calculation of commonly-used variables (and saving back into original NetCDF output file)</li>
<li>Basic analysis of output (yearly/monthy means, etc.)</li>
</ul>
<p><code>IscaTools.jl</code> can be installed via</p>
<pre><code>] add IscaTools
</code></pre>
<p>Of course, before using <code>IscaTools.jl</code>, you need to download the GCM to generate the output from:</p>
<ul>
<li>The Isca GCM [<a href="https://github.com/ExeClim/Isca">GitHub</a>] [<a href="https://execlim.github.io/IscaWebsite/" rel="nofollow">Website</a>]</li>
</ul>
