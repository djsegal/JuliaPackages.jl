<div class="highlight highlight-source-julia"><pre><span class="pl-k">using</span> BrowseTables, Tables
<span class="pl-c"><span class="pl-c">#</span> make example table, but any table that supports Tables.jl will work</span>
table <span class="pl-k">=</span> Tables<span class="pl-k">.</span><span class="pl-c1">columntable</span>(<span class="pl-c1">collect</span>(i <span class="pl-k">==</span> <span class="pl-c1">5</span> ? (a <span class="pl-k">=</span> <span class="pl-c1">missing</span>, b <span class="pl-k">=</span> <span class="pl-s"><span class="pl-pds">"</span>string<span class="pl-pds">"</span></span>, c <span class="pl-k">=</span> <span class="pl-c1">nothing</span>) :
                                   (a <span class="pl-k">=</span> i, b <span class="pl-k">=</span> <span class="pl-c1">Float64</span>(i), c <span class="pl-k">=</span> <span class="pl-s"><span class="pl-pds">'</span>a<span class="pl-pds">'</span></span><span class="pl-k">-</span><span class="pl-c1">1</span><span class="pl-k">+</span>i) <span class="pl-k">for</span> i <span class="pl-k">in</span> <span class="pl-c1">1</span><span class="pl-k">:</span><span class="pl-c1">10</span>))

<p>The package exports four symbols:</p>
<ol>
<li><code>write_html_table</code> writes a table to a HTML file,</li>
<li><code>open_html_table</code> writes the table and opens it in a browser using <a href="https://github.com/tpapp/DefaultApplication.jl">DefaultApplication.jl</a>,</li>
<li><code>TableOptions</code> can be used to customize table appearance and HTML options for the above two functions.</li>
<li><code>HTMLTable</code> displays an HTML table using Julia's display system.  It needs a display supporting HTML output (e.g., <a href="https://github.com/JuliaLang/IJulia.jl">IJulia.jl</a>).</li>
</ol>
<p>Please read the docstrings for further information. That said, the primary design principle of this package is that it should “just work”, without further tweaking.</p>
