<p dir="auto">The package exports four symbols:</p>
<ol dir="auto">
<li><code>write_html_table</code> writes a table to a HTML file,</li>
<li><code>open_html_table</code> writes the table and opens it in a browser using <a href="https://github.com/tpapp/DefaultApplication.jl">DefaultApplication.jl</a>,</li>
<li><code>TableOptions</code> can be used to customize table appearance and HTML options for the above two functions.</li>
<li><code>HTMLTable</code> displays an HTML table using Julia's display system.  It needs a display supporting HTML output (e.g., <a href="https://github.com/JuliaLang/IJulia.jl">IJulia.jl</a>).</li>
</ol>
<p dir="auto">Please read the docstrings for further information. That said, the primary design principle of this package is that it should “just work”, without further tweaking.</p>
