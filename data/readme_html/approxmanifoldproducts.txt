<p dir="auto">Approximate the product between infinite functional objects on a manifold -- i.e. belief products.  ApproxManifoldProducts (AMP) is intended for use with the <a href="http://www.github.com/JuliaRobotics/Caesar.jl">Caesar.jl</a> framework.  AMP is intended for development and testing of various kernel embedding methods for approximating functional products.</p>
<p dir="auto">See <a href="http://www.juliarobotics.org/Caesar.jl/latest/" rel="nofollow">Caesar documentation</a> for more details.</p>

<p dir="auto"><strong>NOTE</strong> Current work on v0.3.x is transitioning to JuliaManifolds for some of the Manifold abstractions.  This package AMP then adds non-parametric probabilistic computations on those and user manifolds.</p>
<p dir="auto">The following on-manifold function approximations:</p>
<ul dir="auto">
<li>Euclidean (2DOF),</li>
<li>S1/Circular (1DOF), or SO(2) equivalent.</li>
<li>SE(2)</li>
<li>S2 (not implemented yet)</li>
</ul>
<p dir="auto">Work in progress on so-called 'subgroup' mixed-manifolds, where DOFs are bunched together for particular manifolds:</p>
<ul dir="auto">
<li>SO(3) / Quaternion,</li>
<li>Plucker coordinates (SP(3)) for rigid transforms,</li>
<li>SE(3) for rigid transforms.</li>
</ul>
<blockquote>
<p dir="auto">Implementation of multivariate methodology requires code to consider "double loops" that iterate over sub-groups, and then within each sub-group as required.  The supported list above can be implemented with "single loop" over all DOFs.</p>
</blockquote>
<p dir="auto">Any suggestions or issues welcome.</p>
