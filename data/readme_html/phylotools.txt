<p dir="auto">A julia package for phylogenetics that includes some visualization functions. <br>
The basic tree data structure and the plot recipes are based off of NewickTree.jl</p>
<h2 dir="auto"><a id="user-content-tree-manipulation" class="anchor" aria-hidden="true" href="#tree-manipulation"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path d="m7.775 3.275 1.25-1.25a3.5 3.5 0 1 1 4.95 4.95l-2.5 2.5a3.5 3.5 0 0 1-4.95 0 .751.751 0 0 1 .018-1.042.751.751 0 0 1 1.042-.018 1.998 1.998 0 0 0 2.83 0l2.5-2.5a2.002 2.002 0 0 0-2.83-2.83l-1.25 1.25a.751.751 0 0 1-1.042-.018.751.751 0 0 1-.018-1.042Zm-4.69 9.64a1.998 1.998 0 0 0 2.83 0l1.25-1.25a.751.751 0 0 1 1.042.018.751.751 0 0 1 .018 1.042l-1.25 1.25a3.5 3.5 0 1 1-4.95-4.95l2.5-2.5a3.5 3.5 0 0 1 4.95 0 .751.751 0 0 1-.018 1.042.751.751 0 0 1-1.042.018 1.998 1.998 0 0 0-2.83 0l-2.5 2.5a1.998 1.998 0 0 0 0 2.83Z"></path></svg></a>Tree manipulation</h2>
<p dir="auto">To read a tree:</p>
<div class="highlight highlight-source-julia notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="tree = readnw(readline(path_to_tree))"><pre>tree <span class="pl-k">=</span> <span class="pl-c1">readnw</span>(<span class="pl-c1">readline</span>(path_to_tree))</pre></div>
<p dir="auto">We can also root the tree automatically based on a midpoint rooting criterion
(at the node which has the minimal distance to the tip that is farthest away
from it):</p>
<div class="highlight highlight-source-julia notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="tree_rooted = root_midpoint(tree)"><pre>tree_rooted <span class="pl-k">=</span> <span class="pl-c1">root_midpoint</span>(tree)</pre></div>
<p dir="auto">Trees can be clustered based on the fraction of the total divergence allowed in
each subtree:</p>
<div class="highlight highlight-source-julia notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="clu = cluster_tree(tree_rooted,2)"><pre>clu <span class="pl-k">=</span> <span class="pl-c1">cluster_tree</span>(tree_rooted,<span class="pl-c1">2</span>)</pre></div>

<p dir="auto">Plotting is based on Plots.jl. <br>
To plot a tree (without showing its tip labels):</p>
<div class="highlight highlight-source-julia notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="plot(tree_rooted)"><pre><span class="pl-c1">plot</span>(tree_rooted)</pre></div>
<p dir="auto">or with tip labels:</p>
<div class="highlight highlight-source-julia notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="plot(tree_rooted; labels=true)"><pre><span class="pl-c1">plot</span>(tree_rooted; labels<span class="pl-k">=</span><span class="pl-c1">true</span>)</pre></div>
