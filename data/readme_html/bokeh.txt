<p dir="auto"><a href="https://julialang.org/" rel="nofollow">Julia</a> bindings for the <a href="https://bokeh.org/" rel="nofollow">Bokeh</a> plotting
library.</p>
<p dir="auto">Although Bokeh is mainly a Python library, all the actual plotting happens in the browser
using <a href="https://docs.bokeh.org/en/latest/docs/user_guide/bokehjs.html" rel="nofollow">BokehJS</a>. This package
wraps BokehJS directly without using Python.</p>
