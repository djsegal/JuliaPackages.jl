<p dir="auto">A trivial implementation of FILETIME style timestamps [1].</p>
<p dir="auto">FILETIME was apparently introduced in the Windows NT operating system
and is now used in a variety of file formats and equipment, including
Simrad Echosounders.</p>
<p dir="auto">A FILETIME is an "integer value stating the number of 100 nanosecond
intervals since January 1, 1601".</p>
