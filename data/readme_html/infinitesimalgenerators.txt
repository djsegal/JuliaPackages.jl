<ul dir="auto">
<li><code>X = DiffusionProcess(x::AbstractVector, μ::AbstractVector, σ::AbstractVector)</code> creates the discretized Markov Process with drift <code>μ</code> and volatility <code>σ</code>, on a grid <code>x</code> with reflecting boundaries.</li>
<li><code>generator(X)</code> returns its associated generator (i.e. the operator <code>f -&gt; ∂_tE[f(x_t)|x_0=x]</code>)</li>
<li><code>stationary_distribution(X)</code> returns its stationary distribution (i.e. the positive vector <code>g</code> such that <code>g * generator(X) = 0</code>)</li>
</ul>

<ul dir="auto">
<li><code>M = AdditiveFunctional(X, μm, σm)</code> creates, given a discretized Markov Process, the Additive Functional with drift  <code>μm</code> and volatility <code>σm</code></li>
<li><code>generator(M)</code> returns its associated generator (i.e. the operator <code>f -&gt; ∂_tE[e^{m}f(x_t)|x_0=x]</code>)</li>
<li><code>cgf(m)</code> returns the long run scaled CGF of <code>m</code></li>
<li><code>tail_index(m)</code> returns the tail index of the stationary distribution of <code>e^m</code></li>
</ul>
