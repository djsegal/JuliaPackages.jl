<p dir="auto"><a href="https://github.com/xzackli/PowerSpectra.jl">PowerSpectra.jl</a> is a package for power
spectrum analysis on the sphere. It computes mode-coupling matrices and analytic covariance
matrices for T, E, and B-mode spectra, using pseudo-Cl methods
(i.e. <a href="https://arxiv.org/abs/astro-ph/0105302" rel="nofollow">Hivon et al. 2002</a>,
<a href="https://arxiv.org/abs/astro-ph/0601107" rel="nofollow">Efstathiou 2006</a>,
<a href="https://arxiv.org/abs/0801.0554" rel="nofollow">Hamimeche and Lewis 2008</a>).</p>
<p dir="auto">Please consult the <a href="https://xzackli.github.io/PowerSpectra.jl/dev" rel="nofollow">documentation</a> for how to use this package. You can cite <a href="https://arxiv.org/abs/2112.13839" rel="nofollow">https://arxiv.org/abs/2112.13839</a> if you find this package useful.</p>
