<p dir="auto">An <em>enclosure</em> of the <a href="https://en.wikipedia.org/wiki/Range_of_a_function" rel="nofollow">range</a>
of a function <math-renderer class="js-inline-math" style="display: inline" data-static-url="https://github.githubassets.com/static" data-run-id="010a5d2a26ea5b587ee4a513c981bfcc">$f : D \subset \mathbb{R}^n \to \mathbb{R}$</math-renderer> is an interval that
contains the global minimum and maximum of <math-renderer class="js-inline-math" style="display: inline" data-static-url="https://github.githubassets.com/static" data-run-id="010a5d2a26ea5b587ee4a513c981bfcc">$f$</math-renderer> over its domain <math-renderer class="js-inline-math" style="display: inline" data-static-url="https://github.githubassets.com/static" data-run-id="010a5d2a26ea5b587ee4a513c981bfcc">$D$</math-renderer>.
<code>RangeEnclosures</code> offers an API to easily bound the range of <math-renderer class="js-inline-math" style="display: inline" data-static-url="https://github.githubassets.com/static" data-run-id="010a5d2a26ea5b587ee4a513c981bfcc">$f$</math-renderer> with different
algorithms. Here is a quick example:</p>
<div class="highlight highlight-source-julia notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="julia&gt; f(x) = -x^3/6 + 5x

julia&gt; dom = 1 .. 4
[1, 4]

julia&gt; enclose(f, dom, BranchAndBoundEnclosure())
[4.83333, 10.5709]"><pre>julia<span class="pl-k">&gt;</span> <span class="pl-en">f</span>(x) <span class="pl-k">=</span> <span class="pl-k">-</span>x<span class="pl-k">^</span><span class="pl-c1">3</span><span class="pl-k">/</span><span class="pl-c1">6</span> <span class="pl-k">+</span> <span class="pl-c1">5</span>x

julia<span class="pl-k">&gt;</span> dom <span class="pl-k">=</span> <span class="pl-c1">1</span> <span class="pl-k">..</span> <span class="pl-c1">4</span>
[<span class="pl-c1">1</span>, <span class="pl-c1">4</span>]

julia<span class="pl-k">&gt;</span> <span class="pl-c1">enclose</span>(f, dom, <span class="pl-c1">BranchAndBoundEnclosure</span>())
[<span class="pl-c1">4.83333</span>, <span class="pl-c1">10.5709</span>]</pre></div>
