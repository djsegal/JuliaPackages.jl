<p>A package to facilitate the calculation of epsilon (<a target="_blank" rel="noopener noreferrer" href="/tex/7ccca27b5ccc533a2dd72dc6fa28ed84.svg?invert_in_darkmode&amp;sanitize=true"><img src="/tex/7ccca27b5ccc533a2dd72dc6fa28ed84.svg?invert_in_darkmode&amp;sanitize=true" align="middle" width="6.672392099999992pt" height="14.15524440000002pt/" style="max-width:100%;"></a>) table structures, derived from Wynn's recursive epsilon algorithm. The components of the epsilon table are commonly used within the calculation of sequence transformations.</p>
<p>Suppose we are presented with a series, <a target="_blank" rel="noopener noreferrer" href="/tex/e257acd1ccbe7fcb654708f1a866bfe9.svg?invert_in_darkmode&amp;sanitize=true"><img src="/tex/e257acd1ccbe7fcb654708f1a866bfe9.svg?invert_in_darkmode&amp;sanitize=true" align="middle" width="11.027402099999989pt" height="22.465723500000017pt/" style="max-width:100%;"></a>, with component terms <a target="_blank" rel="noopener noreferrer" href="/tex/6512cbd0d448700a036bf3a691c37acc.svg?invert_in_darkmode&amp;sanitize=true"><img src="/tex/6512cbd0d448700a036bf3a691c37acc.svg?invert_in_darkmode&amp;sanitize=true" align="middle" width="16.81517804999999pt" height="14.15524440000002pt/" style="max-width:100%;"></a>, and partial sums <a target="_blank" rel="noopener noreferrer" href="/tex/d28140eda2d12e24b434e011b930fa23.svg?invert_in_darkmode&amp;sanitize=true"><img src="/tex/d28140eda2d12e24b434e011b930fa23.svg?invert_in_darkmode&amp;sanitize=true" align="middle" width="14.730823799999989pt" height="22.465723500000017pt/" style="max-width:100%;"></a>,</p>
<p><a target="_blank" rel="noopener noreferrer" href="/tex/305b00052b0f637b1f6b9cbbc20b6bd4.svg?invert_in_darkmode&amp;sanitize=true"><img src="/tex/305b00052b0f637b1f6b9cbbc20b6bd4.svg?invert_in_darkmode&amp;sanitize=true" align="middle" width="81.79511999999998pt" height="56.32434059999998pt/" style="max-width:100%;"></a>.</p>

<a target="_blank" rel="noopener noreferrer" href="/tex/018b65cf47992d70b1aef415dc34aa03.svg?invert_in_darkmode&amp;sanitize=true"><img src="/tex/018b65cf47992d70b1aef415dc34aa03.svg?invert_in_darkmode&amp;sanitize=true" align="middle" width="194.86075784999997pt" height="44.36012790000002pt/" style="max-width:100%;"></a>,</p>
<p>where</p>
<p><a target="_blank" rel="noopener noreferrer" href="/tex/a6d06ffa0c20fa59096509295f59021c.svg?invert_in_darkmode&amp;sanitize=true"><img src="/tex/a6d06ffa0c20fa59096509295f59021c.svg?invert_in_darkmode&amp;sanitize=true" align="middle" width="59.06766524999999pt" height="34.337843099999986pt/" style="max-width:100%;"></a>, for <a target="_blank" rel="noopener noreferrer" href="/tex/cb64b662810fa2e879c6c890c2c20026.svg?invert_in_darkmode&amp;sanitize=true"><img src="/tex/cb64b662810fa2e879c6c890c2c20026.svg?invert_in_darkmode&amp;sanitize=true" align="middle" width="93.33412439999998pt" height="21.68300969999999pt/" style="max-width:100%;"></a></p>
<p><a target="_blank" rel="noopener noreferrer" href="/tex/aed0ba5a615bf67526eca4fc5bb41642.svg?invert_in_darkmode&amp;sanitize=true"><img src="/tex/aed0ba5a615bf67526eca4fc5bb41642.svg?invert_in_darkmode&amp;sanitize=true" align="middle" width="54.45768899999999pt" height="34.337843099999986pt/" style="max-width:100%;"></a>, for <a target="_blank" rel="noopener noreferrer" href="/tex/cb64b662810fa2e879c6c890c2c20026.svg?invert_in_darkmode&amp;sanitize=true"><img src="/tex/cb64b662810fa2e879c6c890c2c20026.svg?invert_in_darkmode&amp;sanitize=true" align="middle" width="93.33412439999998pt" height="21.68300969999999pt/" style="max-width:100%;"></a></p>
<p><a target="_blank" rel="noopener noreferrer" href="/tex/dfa1956691e72694d10703fb035b88f5.svg?invert_in_darkmode&amp;sanitize=true"><img src="/tex/dfa1956691e72694d10703fb035b88f5.svg?invert_in_darkmode&amp;sanitize=true" align="middle" width="81.11019509999998pt" height="34.337843099999986pt/" style="max-width:100%;"></a>, for <a target="_blank" rel="noopener noreferrer" href="/tex/bfd898f3193f13c76d64173d28ac7a86.svg?invert_in_darkmode&amp;sanitize=true"><img src="/tex/bfd898f3193f13c76d64173d28ac7a86.svg?invert_in_darkmode&amp;sanitize=true" align="middle" width="95.38131569999997pt" height="21.68300969999999pt/" style="max-width:100%;"></a></p>
<p>The resulting table of <a target="_blank" rel="noopener noreferrer" href="/tex/f601aeaf6c36f343239c6ab5e365c738.svg?invert_in_darkmode&amp;sanitize=true"><img src="/tex/f601aeaf6c36f343239c6ab5e365c738.svg?invert_in_darkmode&amp;sanitize=true" align="middle" width="21.59732354999999pt" height="34.337843099999986pt/" style="max-width:100%;"></a> values is known as the epsilon table.</p>
<p align="center"><a target="_blank" rel="noopener noreferrer" href="/tex/6cf2c910e91ab031410942a0ddf527aa.svg?invert_in_darkmode&amp;sanitize=true"><img src="/tex/6cf2c910e91ab031410942a0ddf527aa.svg?invert_in_darkmode&amp;sanitize=true" align="middle" width="168.6075039pt" height="229.9926321pt/" style="max-width:100%;"></a></p>
<p>Epsilon table values with an even <a target="_blank" rel="noopener noreferrer" href="/tex/36b5afebdba34564d884d347484ac0c7.svg?invert_in_darkmode&amp;sanitize=true"><img src="/tex/36b5afebdba34564d884d347484ac0c7.svg?invert_in_darkmode&amp;sanitize=true" align="middle" width="7.710416999999989pt" height="21.68300969999999pt/" style="max-width:100%;"></a>-th index, i.e. <a target="_blank" rel="noopener noreferrer" href="/tex/6c0fb9bd50b8ed35149982975f93c8cd.svg?invert_in_darkmode&amp;sanitize=true"><img src="/tex/6c0fb9bd50b8ed35149982975f93c8cd.svg?invert_in_darkmode&amp;sanitize=true" align="middle" width="21.59732354999999pt" height="34.337843099999986pt/" style="max-width:100%;"></a>, are commonly used to compute rational sequence transformations and extrapolations, such as Shank's Transforms, and Pade Approximants.</p>

<p>Suppose we wanted to approximate <a target="_blank" rel="noopener noreferrer" href="/tex/559b96359a4653a6c35dbf27c11f68d2.svg?invert_in_darkmode&amp;sanitize=true"><img src="/tex/559b96359a4653a6c35dbf27c11f68d2.svg?invert_in_darkmode&amp;sanitize=true" align="middle" width="47.29464134999999pt" height="24.65753399999998pt/" style="max-width:100%;"></a> (around <a target="_blank" rel="noopener noreferrer" href="/tex/8436d02a042a1eec745015a5801fc1a0.svg?invert_in_darkmode&amp;sanitize=true"><img src="/tex/8436d02a042a1eec745015a5801fc1a0.svg?invert_in_darkmode&amp;sanitize=true" align="middle" width="39.53182859999999pt" height="21.18721440000001pt/" style="max-width:100%;"></a>) using a rational Pade Approximant <a target="_blank" rel="noopener noreferrer" href="/tex/8a9e0cd4c218dbb2d9e4be213d6f108e.svg?invert_in_darkmode&amp;sanitize=true"><img src="/tex/8a9e0cd4c218dbb2d9e4be213d6f108e.svg?invert_in_darkmode&amp;sanitize=true" align="middle" width="49.62157199999999pt" height="24.65753399999998pt/" style="max-width:100%;"></a>. The pade approximant <a target="_blank" rel="noopener noreferrer" href="/tex/8a9e0cd4c218dbb2d9e4be213d6f108e.svg?invert_in_darkmode&amp;sanitize=true"><img src="/tex/8a9e0cd4c218dbb2d9e4be213d6f108e.svg?invert_in_darkmode&amp;sanitize=true" align="middle" width="49.62157199999999pt" height="24.65753399999998pt/" style="max-width:100%;"></a> is known to correspond to the epsilon table value <a target="_blank" rel="noopener noreferrer" href="/tex/d7090fe83aa1e5aafcbd7cdaf76a596f.svg?invert_in_darkmode&amp;sanitize=true"><img src="/tex/d7090fe83aa1e5aafcbd7cdaf76a596f.svg?invert_in_darkmode&amp;sanitize=true" align="middle" width="43.10908139999999pt" height="34.337843099999986pt/" style="max-width:100%;"></a>.
