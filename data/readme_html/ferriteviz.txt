<p dir="auto">Small package to visualize your <a href="https://github.com/Ferrite-FEM/Ferrite.jl">Ferrite.jl</a> results. Currently supports only Makie,
but the goal is to extend it for different plotting packages.</p>
<p dir="auto">The package is highly experimental and breaking changes about the internal machinery are about to come.
Likely, only a small fraction of the interface will change over time.</p>

<ul dir="auto">
<li><code>solutionplot</code> FE solution contour plot on arbitrary finite element mesh (in Makie called <code>mesh</code> plots)</li>
<li><code>ferriteviewer</code> viewer with toggles and menus that update the plot</li>
<li><code>wireframe</code> plots the finite element mesh and optionally labels nodes and cells</li>
<li><code>arrows</code> - also called <code>quiver</code> plots, in paraview <code>glyph</code> filter</li>
<li><code>surface</code> 2D solutions in 3D space as surface, in paraview <code>warp by scalar</code> filter</li>
<li>synchronous plotting while your simulation runs with any of the above listed options</li>
<li>mutating versions of the above listed functions (except for the viewer)</li>
<li>deformed plots available for <code>solutionplot</code> and <code>wireframe</code></li>
<li>full integration into the Makie ecosystem, e.g. themes, layouts etc.</li>
<li>GPU powered plotting with GLMakie.jl, jupyter/pluto notebook plotting with WGLMakie.jl and vector graphics with CairoMakie.jl</li>
</ul>
