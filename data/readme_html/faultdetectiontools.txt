<p dir="auto"><code>FDITools</code> is a collection of Julia functions for the analysis and solution
of fault detection and model detection problems. The functions of this collection rely on
the <a href="https://github.com/andreasvarga/DescriptorSystems.jl">DescriptorSystems</a> package.</p>
<p dir="auto">The implemented functions are based on the computational procedures described in Chapters 5, 6 and 7 of the book:</p>
<p dir="auto"><strong>Andreas Varga</strong>, <a href="https://www.springer.com/us/book/9783319515588" rel="nofollow">Solving Fault Diagnosis Problems, Linear Synthesis Techniques</a>, vol. 84 of Studies in Systems, Decision and Control, Springer International Publishing, 2017.</p>
<p dir="auto">This book describes the mathematical background of solving synthesis problems of fault detection and model detection filters and gives detailed descriptions of the underlying synthesis procedures.</p>
<p dir="auto">The implemented functionality parallels the functionality of the MATLAB collection of tools <a href="https://github.com/andreasvarga/FDITools">FDITOOLS</a>, whose User's Guide of the version V1.0 is provided in the file <a href="https://github.com/andreasvarga/FDITools/blob/master/fditoolsdoc.pdf"><code>fditoolsdoc.pdf</code></a>.</p>
