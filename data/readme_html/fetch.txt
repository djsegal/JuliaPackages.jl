<p dir="auto">By using <code>Fetch.jl</code>, one can upload their dataset to Google drive,
and construct <code>DataDep</code> by setting <code>fetch_method=gdownload</code>.</p>
<div class="snippet-clipboard-content notranslate position-relative overflow-auto" data-snippet-clipboard-copy-content="using DataDeps

<p dir="auto">According to the document of <a href="https://github.com/Kaggle/kaggle-api#api-credentials">Kaggle-api</a>
one needs to set their environment variables <code>KAGGLE_USERNAME</code> and <code>KAGGLE_KEY</code>,
or simply download the api token from Kaggle, and place this file in the location <code>~/.kaggle/kaggle.json</code>
(on Windows in the location <code>C:\Users\&lt;Windows-username&gt;\.kaggle\kaggle.json</code>).</p>
