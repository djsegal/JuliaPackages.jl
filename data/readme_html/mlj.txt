</h2>
<p dir="auto">MLJ (Machine Learning in Julia) is a toolbox written in Julia
providing a common interface and meta-algorithms for selecting,
tuning, evaluating, composing and comparing about <a href="https://alan-turing-institute.github.io/MLJ.jl/dev/model_browser/#Model-Browser" rel="nofollow">200 machine learning
models</a>
written in Julia and other languages.</p>
<p dir="auto"><strong>New to MLJ?</strong> Start <a href="https://alan-turing-institute.github.io/MLJ.jl/dev/" rel="nofollow">here</a>.</p>
<p dir="auto"><strong>Integrating an existing machine learning model into the MLJ
framework?</strong> Start <a href="https://alan-turing-institute.github.io/MLJ.jl/dev/quick_start_guide_to_adding_models/" rel="nofollow">here</a>.</p>
<p dir="auto"><strong>Wanting to contribute?</strong> Start <a href="CONTRIBUTING.md">here</a>.</p>
<p dir="auto"><strong>PhD and Postdoc opportunies</strong> See <a href="https://sebastian.vollmer.ms/jobs/" rel="nofollow">here</a>.</p>
<p dir="auto">MLJ was initially created as a Tools, Practices and Systems project at
the <a href="https://www.turing.ac.uk/" rel="nofollow">Alan Turing Institute</a>
in 2019. Current funding is provided by a <a href="https://www.mbie.govt.nz/science-and-technology/science-and-innovation/funding-information-and-opportunities/investment-funds/strategic-science-investment-fund/ssif-funded-programmes/university-of-auckland/" rel="nofollow">New Zealand Strategic
Science Investment
Fund</a>
awarded to the University of Auckland.</p>
