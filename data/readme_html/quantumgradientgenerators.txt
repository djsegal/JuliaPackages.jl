<p dir="auto">Dynamic Gradients for Quantum Control.</p>
<p dir="auto">For a <a href="https://juliaquantumcontrol.github.io/QuantumPropagators.jl/stable/generators/" rel="nofollow">dynamical generator</a> <code>G</code> that depends on one more control function <code>ϵ₁(t)</code> … <code>ϵₗ(t)</code>, this package defines the "gradient generator" <code>G̃</code> and "gradient vector" <code>|Ψ̃⟩</code> derived from a quantum state <code>|Ψ⟩</code>, so that</p>
<math-renderer class="js-display-math" style="display: block" data-static-url="https://github.githubassets.com/static" data-run-id="3f95a85bd2d106d844276c57db54b683">$$\exp(± i \tilde{G} t) |\tilde{Ψ}⟩ = \begin{pmatrix}
    \frac{∂}{∂ϵ_1} e^{± i G t} |Ψ⟩ \\\
    \vdots \\\
    \frac{∂}{∂ϵ_n} e^{± i G t} |Ψ⟩ \\\
    e^{± i G dt} |Ψ⟩
\end{pmatrix}\,.$$</math-renderer>
<p dir="auto">This gradient generator is a core part of evaluating gradients in the <a href="https://github.com/JuliaQuantumControl/GRAPE.jl#readme">GRAPE</a> method of <a href="https://github.com/JuliaQuantumControl/QuantumControl.jl#readme">quantum control</a>.</p>
<p dir="auto">To evaluate the above equation, the <a href="https://github.com/JuliaQuantumControl/QuantumPropagators.jl#readme">QuantumPropagators</a> package should be used.</p>
