<p>WebIO provides a simple abstraction for displaying and interacting with web content. It works with:</p>
<ul>
<li><a href="http://junolab.org" rel="nofollow">Juno</a> - The hottest Julia IDE</li>
<li><a href="https://github.com/JuliaLang/IJulia.jl">IJulia</a> - Jupyter notebooks for Julia</li>
<li><a href="https://github.com/JunoLab/Blink.jl">Blink</a> - An <a href="http://electron.atom.io/" rel="nofollow">Electron</a> wrapper to make desktop apps</li>
<li><a href="https://github.com/JuliaWeb/Mux.jl">Mux</a> - A web server framework</li>
</ul>
