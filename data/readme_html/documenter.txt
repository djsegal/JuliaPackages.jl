<p>The package can be installed with the Julia package manager.
From the Julia REPL, type <code>]</code> to enter the Pkg REPL mode and run:</p>
<div class="snippet-clipboard-content position-relative" data-snippet-clipboard-copy-content="pkg&gt; add Documenter
"><pre><code>pkg&gt; add Documenter
</code></pre></div>
<p>Or, equivalently, via the <code>Pkg</code> API:</p>
<div class="highlight highlight-source-julia position-relative" data-snippet-clipboard-copy-content="julia&gt; import Pkg; Pkg.add(&quot;Documenter&quot;)

<h2><a id="user-content-project-status" class="anchor" aria-hidden="true" href="#project-status"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path fill-rule="evenodd" d="M7.775 3.275a.75.75 0 001.06 1.06l1.25-1.25a2 2 0 112.83 2.83l-2.5 2.5a2 2 0 01-2.83 0 .75.75 0 00-1.06 1.06 3.5 3.5 0 004.95 0l2.5-2.5a3.5 3.5 0 00-4.95-4.95l-1.25 1.25zm-4.69 9.64a2 2 0 010-2.83l2.5-2.5a2 2 0 012.83 0 .75.75 0 001.06-1.06 3.5 3.5 0 00-4.95 0l-2.5 2.5a3.5 3.5 0 004.95 4.95l1.25-1.25a.75.75 0 00-1.06-1.06l-1.25 1.25a2 2 0 01-2.83 0z"></path></svg></a>Project Status</h2>
<p>The package is tested against, and being developed for, Julia <code>1.0</code> and above on Linux, macOS, and Windows.</p>
<p>Support for Julia <code>0.4</code>, <code>0.5</code>, <code>0.6</code> and <code>0.7</code> has been dropped in the latest version, but older versions of Documenter may still work (Documenter versions <code>0.8</code>, <code>0.11</code>, <code>0.19</code>, and <code>0.22</code> respectively).</p>

<p>Usage questions can be posted on the <a href="https://discourse.julialang.org/tags/documenter" rel="nofollow">Julia Discourse forum</a> under the <code>documenter</code> tag, in the #documentation channel of the <a href="https://julialang.org/community/" rel="nofollow">Julia Slack</a> and/or in the <a href="https://gitter.im/juliadocs/users" rel="nofollow">JuliaDocs Gitter chat room</a>.</p>
<p>Contributions are very welcome, as are feature requests and suggestions. Please open an <a href="https://github.com/JuliaDocs/Documenter.jl/issues">issue</a> if you encounter any problems. The <a href="https://juliadocs.github.io/Documenter.jl/dev/contributing/" rel="nofollow">contributing page</a> has a few guidelines that should be followed when opening pull requests and contributing code.</p>
