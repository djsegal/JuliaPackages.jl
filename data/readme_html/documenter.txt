<p dir="auto">The package can be installed with the Julia package manager.
From the Julia REPL, type <code>]</code> to enter the Pkg REPL mode and run:</p>
<div class="snippet-clipboard-content notranslate position-relative overflow-auto" data-snippet-clipboard-copy-content="pkg&gt; add Documenter"><pre class="notranslate"><code>pkg&gt; add Documenter
</code></pre></div>
<p dir="auto">Or, equivalently, via the <code>Pkg</code> API:</p>
<div class="highlight highlight-source-julia notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="julia&gt; import Pkg; Pkg.add(&quot;Documenter&quot;)"><pre>julia<span class="pl-k">&gt;</span> <span class="pl-k">import</span> Pkg; Pkg<span class="pl-k">.</span><span class="pl-c1">add</span>(<span class="pl-s"><span class="pl-pds">"</span>Documenter<span class="pl-pds">"</span></span>)</pre></div>

<p dir="auto">Usage questions can be posted on the <a href="https://discourse.julialang.org/tags/documenter" rel="nofollow">Julia Discourse forum</a> under the <code>documenter</code> tag, in the #documentation channel of the <a href="https://julialang.org/community/" rel="nofollow">Julia Slack</a> and/or in the <a href="https://gitter.im/juliadocs/users" rel="nofollow">JuliaDocs Gitter chat room</a>.</p>
<p dir="auto">Contributions are very welcome, as are feature requests and suggestions. Please open an <a href="https://github.com/JuliaDocs/Documenter.jl/issues">issue</a> if you encounter any problems. The <a href="https://documenter.juliadocs.org/dev/contributing/" rel="nofollow">contributing page</a> has a few guidelines that should be followed when opening pull requests and contributing code.</p>
