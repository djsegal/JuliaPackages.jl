<p dir="auto">GalaxyBrain is a parse-time transpiler for the <a href="https://en.wikipedia.org/wiki/Brainfuck" rel="nofollow">Brainfuck</a> programming language. It allows you to embed Brainfuck code directly into your Julia code, using the string macro <code>bf""</code>. This macro unrolls into an anonymous function during parse time, accepting the following keyword arguments:</p>
<ul dir="auto">
<li>
<p dir="auto"><code>input::Union{AbstractString,IO}=""</code></p>
<ul dir="auto">
<li>The program's input, either as an IO source or as a string.</li>
<li>Input is read at a rate of one byte per <code>,</code> instruction.</li>
</ul>
</li>
<li>
<p dir="auto"><code>output::IO=stdout</code></p>
<ul dir="auto">
<li>The destination for <code>.</code> print calls.</li>
</ul>
</li>
<li>
<p dir="auto"><code>index::Int=1</code></p>
<ul dir="auto">
<li>The starting position of the program's memory pointer.</li>
</ul>
</li>
<li>
<p dir="auto"><code>memory_size::Int=30000</code></p>
<ul dir="auto">
<li>The size of the <code>UInt8</code> array allocated for running the program.</li>
<li>The default value is the same as that of the classic distribution.</li>
</ul>
</li>
</ul>
