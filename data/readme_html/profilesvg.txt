<h2><a id="user-content-purpose" class="anchor" aria-hidden="true" href="#purpose"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path fill-rule="evenodd" d="M7.775 3.275a.75.75 0 001.06 1.06l1.25-1.25a2 2 0 112.83 2.83l-2.5 2.5a2 2 0 01-2.83 0 .75.75 0 00-1.06 1.06 3.5 3.5 0 004.95 0l2.5-2.5a3.5 3.5 0 00-4.95-4.95l-1.25 1.25zm-4.69 9.64a2 2 0 010-2.83l2.5-2.5a2 2 0 012.83 0 .75.75 0 001.06-1.06 3.5 3.5 0 00-4.95 0l-2.5 2.5a3.5 3.5 0 004.95 4.95l1.25-1.25a.75.75 0 00-1.06-1.06l-1.25 1.25a2 2 0 01-2.83 0z"></path></svg></a>Purpose</h2>
<p>ProfileSVG allows you to export profiling data as an SVG file.
It can be used to display profiling results in Jupyter/IJulia notebooks,
although the <a href="https://github.com/davidanthoff/ProfileVega.jl">ProfileVega</a>
package is recommended as a better-maintained viewer for Jupyter notebooks.</p>
<p>An alternative visualization package is the GTK-based
<a href="https://github.com/timholy/ProfileView.jl">ProfileView</a>.
That package offers more interactive features.</p>
<p>ProfileSVG is essentially just an "export" package built on top of
<a href="https://github.com/timholy/FlameGraphs.jl">FlameGraphs</a>.
Among the Julia packages, ProfileView currently has the most comprehensive
tutorial on how to interpret a flame graph.</p>
