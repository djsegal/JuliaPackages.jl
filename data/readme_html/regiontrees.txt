<p>This Julia package is a lightweight framework for defining N-Dimensional region trees. In 2D, these are called <em>region quadtrees</em>, and in 3D they are typically referred to as <em>octrees</em>. A region tree is a simple data structure used to describe some kind of spatial data with varying resolution. Each element in the tree can be a leaf, representing an N-dimensional rectangle of space, or a node which is divided exactly in half along each axis into 2^N children. In addition, each element in a <code>RegionTrees.jl</code> tree can carry an arbitrary data payload. This makes it easy to use <code>RegionTrees</code> to approximate functions or describe other interesting spatial data.</p>

<li>Optimized for speed and for few memory allocations
<ul>
