<p>Julia implementation of Pullback Bundle Dynamical Systems (PBDS) framework for geometrically-consistent, multi-task reactive control on non-Euclidean robot configuration manifolds and task manifolds.</p>
<p>For more details on the PBDS framework, see:</p>
<p>A. Bylard, R. Bonalli, and M. Pavone, "Composable Geometric Motion Policies using Multi-Task Pullback Bundle Dynamical Systems," 2020. <a href="https://arxiv.org/abs/2101.01297" rel="nofollow">[pdf]</a></p>
