<p dir="auto">Package that provides an integrated workflow for molecular dynamics (MD) simulations. Defines an API for MD simulations that is compatible with the interatomic potential interface defined by <a href="https://github.com/cesmix-mit/InteratomicPotentials.jl">InteratomicPotentials.jl</a> and the atomic configuration interface defined by <a href="https://github.com/JuliaMolSim/AtomsBase.jl">AtomsBase.jl</a>.</p>
<p dir="auto">Developed as part of the <a href="https://computing.mit.edu/cesmix/" rel="nofollow">CESMIX</a> Julia package suite. See also <a href="https://github.com/cesmix-mit/ComposableWorkflows">ComposableWorkflows</a>, <a href="https://github.com/cesmix-mit/InteratomicPotentials.jl">InteratomicPotentials.jl</a>, and <a href="https://github.com/cesmix-mit/PotentialLearning.jl">PotentialLearning.jl</a>.</p>
