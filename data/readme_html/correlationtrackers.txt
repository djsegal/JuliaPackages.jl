<p dir="auto"><a href="https://github.com/shamazmazum/CorrelationTrackers.jl/actions/workflows/test.yml"><img src="https://github.com/shamazmazum/CorrelationTrackers.jl/actions/workflows/test.yml/badge.svg" alt="CI" style="max-width: 100%;"></a></p>
<p dir="auto"><code>CorrelationTrackers.jl</code> is a library which helps you to fastly update correlation
functions calculated by <code>CorrelationFunctions.jl</code> package when you
insignificantly change the input.</p>
<p dir="auto">To build a documentation do the following:</p>
<ol dir="auto">
<li>From Julia REPL: <code>import Pkg; Pkg.add("Documenter")</code></li>
<li>From shell, this directory being the working directory: <code>cd docs &amp;&amp; julia make.jl</code></li>
</ol>
<p dir="auto">Also the documentation for the most recent release is available on
<a href="https://fatimp.github.io/CorrelationTrackers.jl/stable/" rel="nofollow">GitHub Pages</a>.</p>
