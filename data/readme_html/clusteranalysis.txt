<p dir="auto">This package was <ins><strong>built from scratch</strong></ins>, entirely in <a href="https://julialang.org/" rel="nofollow">Julia Lang</a>, and implements a few popular clustering algorithms like K-Means and DBSCAN.</p>
<p dir="auto">This is mostly a learning experiment, but the package were also built and documented to be used by anyone, Plug-and-Play. Just input your data as an Array or a <a href="https://discourse.julialang.org/t/tables-jl-a-table-interface-for-everyone/14071" rel="nofollow">Tables.jl</a> type (like <a href="https://dataframes.juliadata.org/stable/" rel="nofollow">DataFrames.jl</a>), then start training your clusters algorithms and analyze your results.</p>
<p dir="auto">Documentation: <a href="https://augustocl.github.io/ClusterAnalysis.jl/" rel="nofollow">https://augustocl.github.io/ClusterAnalysis.jl/</a></p>

<p dir="auto">Currently we implemented two types of algorithms, a partitioned based (<a href="https://en.wikipedia.org/wiki/K-means_clustering" rel="nofollow">K-Means</a>) and a spatial density based (<a href="https://en.wikipedia.org/wiki/DBSCAN" rel="nofollow">DBSCAN</a>).</p>
<blockquote>
<p dir="auto">Go check the <code>Algorithms Overview</code> Section that contains all the details of how it works the algorithm and also got the bibliography and papers used during the research and development of the code.</p>
</blockquote>
<blockquote>
<p dir="auto">It's a great introduction to the algorithm and a good resource to read along with the source code.</p>
</blockquote>
