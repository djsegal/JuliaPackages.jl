<p>This library is developed independently of, but is heavily influenced in design by the <a href="https://github.com/frewsxcv/python-geojson">python-geojson</a> package. It contains:</p>
<ul>
<li>Functions for encoding and decoding GeoJSON formatted data</li>
<li>a type hierarchy (according to the <a href="http://geojson.org/geojson-spec.html" rel="nofollow">GeoJSON specification</a>)</li>
<li>An implementation of the <a href="https://gist.github.com/sgillies/2217756">_<em>geo_interface_</em></a>, a GeoJSON-like protocol for geo-spatial (GIS) vector data.</li>
</ul>
