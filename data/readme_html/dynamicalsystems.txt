<p dir="auto"><strong>DynamicalSystems.jl</strong> is an <a href="https://dsweb.siam.org/The-Magazine/Article/winners-of-the-dsweb-2018-software-contest" rel="nofollow">award-winning</a> Julia software library for nonlinear dynamics and nonlinear timeseries analysis. The current repository holds the documentation and exports all packages composing DynamicalSystems.jl.</p>
<p dir="auto">To install it, run <code>import Pkg; Pkg.add("DynamicalSystems")</code>.</p>
<p dir="auto"><strong>DynamicalSystems.jl</strong> is part of <a href="https://juliadynamics.github.io/JuliaDynamics/" rel="nofollow">JuliaDynamics</a>, an organization dedicated to creating high quality scientific software.</p>
<p dir="auto">All further information is provided in the documentation, which you can either find <a href="https://juliadynamics.github.io/DynamicalSystems.jl/dev/" rel="nofollow">online</a> or build locally by running the <code>docs/make.jl</code> file.</p>
