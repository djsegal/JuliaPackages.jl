<p dir="auto">TrajGWAS.jl is a Julia package for performing genome-wide association studies (GWAS) for continuous longitudinal phenotypes using a modified linear mixed effects model. It builds upon the <a href="https://github.com/OpenMendel/WiSER.jl">within-subject variance estimation by robust regression (WiSER) method</a> and can be used to identify variants associated with changes in the mean and within-subject variability of the longitduinal trait. The estimation procedure is robust to distributional misspecifications of both the random effects and the response. A saddlepoint approximation (SPA) option is implemented to provide improved power and calibrated type I error for rare variants. It runs efficiently and scales well to very large datasets. The package currently supports <a href="https://zzz.bwh.harvard.edu/plink/" rel="nofollow">PLINK</a>, <a href="https://en.wikipedia.org/wiki/Variant_Call_Format" rel="nofollow">VCF</a> (both dosage and genotype data), and <a href="https://www.well.ox.ac.uk/~gav/bgen_format/" rel="nofollow">BGEN</a> file formats. We plan to add <a href="https://www.cog-genomics.org/plink/2.0/formats#pgen" rel="nofollow">PGEN</a> support in the future.</p>
<p dir="auto">TrajGWAS.jl supports Julia v1.6 or later. See the <a href="https://openmendel.github.io/TrajGWAS.jl/dev/" rel="nofollow">documentation</a> for usage.</p>
