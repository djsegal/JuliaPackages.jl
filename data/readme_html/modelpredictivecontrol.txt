<p dir="auto">A <a href="https://en.wikipedia.org/wiki/Model_predictive_control" rel="nofollow">model predictive control</a> package
for Julia.</p>
<p dir="auto">The package depends on <a href="https://github.com/JuliaControl/ControlSystems.jl"><code>ControlSystemsBase.jl</code></a>
for the linear systems and <a href="https://github.com/jump-dev/JuMP.jl"><code>JuMP.jl</code></a> for the solvers.</p>
