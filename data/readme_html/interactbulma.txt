<p dir="auto">Package to create Bulma themes to style Interact apps with <a href="https://bulma.io/" rel="nofollow">Bulma</a> css.</p>
<p dir="auto">To learn how to use Interact, check out the <a href="https://juliagizmos.github.io/Interact.jl/latest/" rel="nofollow">Interact documentation</a>.</p>

<p dir="auto">InteractBulma provides a <code>compile_theme</code> function to create a theme (a CSS file) based on variables and overrides. Check out the Bulma <a href="https://bulma.io/documentation/customize/" rel="nofollow">documentation</a> to learn what variables can be used and how.</p>
<p dir="auto">The function <code>compile_theme(output)</code> has two optional keyword arguments (<code>variables</code> and <code>overrides</code>) with the path of the <code>scss</code> file you want to use to customize variables or to add overrides respectively. <code>output=mktempdir()</code> is the folder chosen to store the resulting css files (main.css and main_confined.css).</p>
