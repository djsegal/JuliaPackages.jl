<p>Package to create Bulma themes to style Interact apps with <a href="https://bulma.io/" rel="nofollow">Bulma</a> css.</p>
<p>To learn how to use Interact, check out the <a href="https://juliagizmos.github.io/Interact.jl/latest/" rel="nofollow">Interact documentation</a>.</p>
<h2><a id="user-content-theming-instruction" class="anchor" aria-hidden="true" href="#theming-instruction"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path fill-rule="evenodd" d="M4 9h1v1H4c-1.5 0-3-1.69-3-3.5S2.55 3 4 3h4c1.45 0 3 1.69 3 3.5 0 1.41-.91 2.72-2 3.25V8.59c.58-.45 1-1.27 1-2.09C10 5.22 8.98 4 8 4H4c-.98 0-2 1.22-2 2.5S3 9 4 9zm9-3h-1v1h1c1 0 2 1.22 2 2.5S13.98 12 13 12H9c-.98 0-2-1.22-2-2.5 0-.83.42-1.64 1-2.09V6.25c-1.09.53-2 1.84-2 3.25C6 11.31 7.55 13 9 13h4c1.45 0 3-1.69 3-3.5S14.5 6 13 6z"></path></svg></a>Theming instruction</h2>
<p>InteractBulma provides a <code>compile_theme</code> function to create a theme (a CSS file) based on variables and overrides. Check out the Bulma <a href="https://bulma.io/documentation/customize/" rel="nofollow">documentation</a> to learn what variables can be used and how.</p>
<p>The function <code>compile_theme(output)</code> has two optional keyword arguments (<code>variables</code> and <code>overrides</code>) with the path of the <code>scss</code> file you want to use to customize variables or to add overrides respectively. <code>output=mktempdir()</code> is the folder chosen to store the resulting css files (main.css and main_confined.css).</p>

<div class="highlight highlight-source-julia"><pre><span class="pl-k">using</span> Interact

<span class="pl-c1">button</span>() <span class="pl-c"><span class="pl-c">#</span> test the new looks of Interact widgets</span></pre></div>
