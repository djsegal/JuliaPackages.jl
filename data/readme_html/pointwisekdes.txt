<p dir="auto">One thing lacking currently in Julia is a generic Gaussian KDE in multiple dimensions, similar to <code>scipy</code>'s <code>gaussian_kde</code>.  This package provides exactly that: a constructor that takes a matrix of points in arbitrary dimension and constructs the Gaussian KDE density as a <code>MultivariateDistribution</code> from <a href="https://github.com/JuliaStats/Distributions.jl"><code>Distributions.jl</code></a>.  Bandwidth is chosen following Scott's rule.</p>
<p dir="auto">Unlike <a href="https://github.com/JuliaStats/KernelDensity.jl">KernelDensity.jl</a>, the cost to evaluate the density from a <code>PointwiseKDE</code> scales linearly with the number of input points; the cost to draw a point from the KDE is constant.</p>
