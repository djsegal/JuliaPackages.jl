<h1><a id="user-content-pdelibjl" class="anchor" aria-hidden="true" href="#pdelibjl"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path fill-rule="evenodd" d="M7.775 3.275a.75.75 0 001.06 1.06l1.25-1.25a2 2 0 112.83 2.83l-2.5 2.5a2 2 0 01-2.83 0 .75.75 0 00-1.06 1.06 3.5 3.5 0 004.95 0l2.5-2.5a3.5 3.5 0 00-4.95-4.95l-1.25 1.25zm-4.69 9.64a2 2 0 010-2.83l2.5-2.5a2 2 0 012.83 0 .75.75 0 001.06-1.06 3.5 3.5 0 00-4.95 0l-2.5 2.5a3.5 3.5 0 004.95 4.95l1.25-1.25a.75.75 0 00-1.06-1.06l-1.25 1.25a2 2 0 01-2.83 0z"></path></svg></a>PDELib.jl</h1>
<p>Work-In-Progress Julia implemenentation of  <a href="https://pdelib.org" rel="nofollow">pdelib</a>.</p>
<p>This  is  a  meta  package which  re-exports  several  Julia  packages
developed  and  maintained  by   the  <a href="https://www.wias-berlin.de" rel="nofollow">WIAS</a>  research  group  <a href="https://www.wias-berlin.de/research/rgs/fg3/" rel="nofollow">Numerical
Mathematics and Scientific Computing</a> and various coauthors.</p>
<h2><a id="user-content-packages-included" class="anchor" aria-hidden="true" href="#packages-included"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path fill-rule="evenodd" d="M7.775 3.275a.75.75 0 001.06 1.06l1.25-1.25a2 2 0 112.83 2.83l-2.5 2.5a2 2 0 01-2.83 0 .75.75 0 00-1.06 1.06 3.5 3.5 0 004.95 0l2.5-2.5a3.5 3.5 0 00-4.95-4.95l-1.25 1.25zm-4.69 9.64a2 2 0 010-2.83l2.5-2.5a2 2 0 012.83 0 .75.75 0 001.06-1.06 3.5 3.5 0 00-4.95 0l-2.5 2.5a3.5 3.5 0 004.95 4.95l1.25-1.25a.75.75 0 00-1.06-1.06l-1.25 1.25a2 2 0 01-2.83 0z"></path></svg></a>Packages included</h2>
<ul>
<li><a href="https://github.com/j-fu/VoronoiFVM.jl">VoronoiFVM.jl</a>: a finite volume solver for systems of nonlinear PDEs</li>
<li><a href="https://github.com/chmerdon/GradientRobustMultiPhysics.jl">GradientRobustMultiPhysics.jl</a>: finite element library implementing gradient robust FEM</li>
<li><a href="https://github.com/j-fu/ExtendableGrids.jl">ExtendableGrids.jl</a>: unstructured grid management library</li>
<li><a href="https://github.com/j-fu/GridVisualize.jl">GridVisualize.jl</a>: grid and function visualization related to ExtendableGrids.jl</li>
<li><a href="https://github.com/j-fu/SimplexGridFactory.jl">SimplexGridFactory.jl</a>: unified high level  mesh generator interface</li>
<li><a href="https://github.com/j-fu/ExtendableSparse.jl">ExtendableSparse.jl</a>: convenient and efficient sparse matrix assembly</li>
</ul>
<h2><a id="user-content-additional-packages-maintained" class="anchor" aria-hidden="true" href="#additional-packages-maintained"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path fill-rule="evenodd" d="M7.775 3.275a.75.75 0 001.06 1.06l1.25-1.25a2 2 0 112.83 2.83l-2.5 2.5a2 2 0 01-2.83 0 .75.75 0 00-1.06 1.06 3.5 3.5 0 004.95 0l2.5-2.5a3.5 3.5 0 00-4.95-4.95l-1.25 1.25zm-4.69 9.64a2 2 0 010-2.83l2.5-2.5a2 2 0 012.83 0 .75.75 0 001.06-1.06 3.5 3.5 0 00-4.95 0l-2.5 2.5a3.5 3.5 0 004.95 4.95l1.25-1.25a.75.75 0 00-1.06-1.06l-1.25 1.25a2 2 0 01-2.83 0z"></path></svg></a>Additional packages maintained</h2>
<p>Not part of PDELib.jl, but maintained as part of the project:</p>
<ul>
<li><a href="https://github.com/JuliaGeometry/Triangulate.jl">Triangulate.jl</a>,  <a href="https://github.com/JuliaBinaryWrappers/Triangle_jll.jl">Triangle_jll.jl</a>:  Julia wrapper and binary package of the <a href="https://www.cs.cmu.edu/~quake/triangle.html" rel="nofollow">Triangle</a> triangle mesh generator by J. Shewchuk</li>
<li><a href="https://github.com/JuliaGeometry/TetGen.jl">TetGen.jl</a>,<a href="https://github.com/JuliaBinaryWrappers/TetGen_jll.jl">TetGen_jll.jl</a>: (co-maintained with <a href="https://github.com/SimonDanisch">S. Danisch</a>):   Julia wrapper binary package of the <a href="http://www.tetgen.org" rel="nofollow">TetGen</a> tetrahedral mesh generator by H. Si.</li>
</ul>
