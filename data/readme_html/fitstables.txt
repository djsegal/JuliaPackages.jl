<p>Make <code>FITS</code> tables from <code>FITSIO.jl</code> conform to the <code>Tables.jl</code> interface.</p>
<p>With this package, you can do, e.g.</p>
<div class="highlight highlight-source-julia"><pre><span class="pl-k">using</span> FITSIO, FITSTables, DataFrames
df <span class="pl-k">=</span> <span class="pl-c1">FITS</span>(<span class="pl-s"><span class="pl-pds">"</span>table.fits<span class="pl-pds">"</span></span>) <span class="pl-k">do</span> f
    <span class="pl-c1">DataFrame</span>(f[<span class="pl-c1">2</span>])
<span class="pl-k">end</span></pre></div>
