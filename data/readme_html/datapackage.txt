<p>A library for working with <a href="http://specs.frictionlessdata.io/data-package/" rel="nofollow">Data Packages</a> in Julia.</p>
<p>Initially this package is primarily targeted and depends on <a href="https://github.com/frictionlessdata/TableSchema.jl">TableSchema.jl</a>, where you can find the common <a href="https://github.com/loleg/TableSchema.jl/blob/master/DESIGN.md">Design Document</a> for these libraries. Support for other types of data packages is to follow.</p>

<ul>
<li><code>Package</code> class for working with data packages</li>
<li><code>Resource</code> class for working with data resources</li>
<li><code>Profile</code> class for working with profiles</li>
<li><code>validate</code> function for validating data package descriptors</li>
<li><code>infer</code> function for inferring data package descriptors</li>
</ul>

<p>Install the <strong><a href="https://github.com/frictionlessdata/TableSchema.jl">TableSchema</a></strong> package.</p>
<p>Clone this repository and enter it:</p>
<p><code>$ git clone https://github.com/frictionlessdata/DataPackage.jl &amp;&amp; cd DataPackage.jl</code></p>
<p>See <em>examples</em> folder and unit tests in <a href="test/runtests.jl">runtests.jl</a> for current usage, e.g.:</p>
<p><code>$ julia examples/basic.jl</code></p>
<p>Running tests:</p>
<p><code>$ julia -L src/DataPackage.jl test/runtests.jl</code></p>
