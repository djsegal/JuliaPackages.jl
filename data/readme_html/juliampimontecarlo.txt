<p>Illustrates using Julia and MPI to do Monte Carlo. Release v1.0 contains the versions used in the paper "A Note on Julia and MPI, with Code Examples" (see <a href="http://link.springer.com/article/10.1007/s10614-015-9516-5" rel="nofollow">http://link.springer.com/article/10.1007/s10614-015-9516-5</a>), which used Julia v0.4. That code will no longer run, but it's left here in case someone wants to examine the code discussed in the paper.</p>
