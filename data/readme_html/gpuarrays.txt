<p dir="auto">This package is the counterpart of Julia's <code>AbstractArray</code> interface, but for GPU array
types: It provides functionality and tooling to speed-up development of new GPU array types.
<strong>This package is not intended for end users!</strong> Instead, you should use one of the packages
that builds on GPUArrays.jl, such as <a href="https://github.com/JuliaGPU/CUDA.jl">CUDA.jl</a>, <a href="https://github.com/JuliaGPU/oneAPI.jl">oneAPI.jl</a>, <a href="https://github.com/JuliaGPU/AMDGPU.jl">AMDGPU.jl</a>, or <a href="https://github.com/JuliaGPU/Metal.jl">Metal.jl</a>.</p>
