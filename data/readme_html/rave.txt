<p>You may occur errors on running the code if you are using packages with different versions from those I used to write the code. Luckily Julia has its build-in way to run codes in a "virtual environment". The file <code>Manifest.toml</code> I provide in this repository is for that purpose. Basically, if you already convert the notebook into a script using the provided <code>nb2script.sh</code>, you can then run the script by <code>julia --project=REPO_PATH scripts/bnpvae.jl ...</code> where <code>REPO_PATH</code> is the path to your clone of my repository.</p>
