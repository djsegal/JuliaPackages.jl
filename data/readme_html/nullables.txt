<p>This package provides the <code>Nullable</code> type from Julia 0.6, which was removed in
subsequent versions. It also defines the <code>unsafe_get</code> and <code>isnull</code> functions, and all
methods previously implemented in Julia Base: <code>get</code>, <code>eltype</code>, <code>convert</code>, <code>promote</code>,
<code>show</code>, <code>map</code>, <code>broadcast</code>, <code>filter</code>, <code>isequal</code>, <code>isless</code> and <code>hash</code>.</p>
