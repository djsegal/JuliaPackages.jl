<p>The state consists of position <a target="_blank" rel="noopener noreferrer" href="https://camo.githubusercontent.com/b2a453de419cfcf47f11affce55a29f15cc6eaed/68747470733a2f2f6c617465782e636f6465636f67732e636f6d2f6769662e6c617465783f253543696e6c696e652532302532387825324325323079253239"><img src="https://camo.githubusercontent.com/b2a453de419cfcf47f11affce55a29f15cc6eaed/68747470733a2f2f6c617465782e636f6465636f67732e636f6d2f6769662e6c617465783f253543696e6c696e652532302532387825324325323079253239" alt="(x,y)" data-canonical-src="https://latex.codecogs.com/gif.latex?%5Cinline%20%28x%2C%20y%29" style="max-width:100%;"></a> and heading <a target="_blank" rel="noopener noreferrer" href="https://camo.githubusercontent.com/ed6801acad814e8d5be5770293fa0d7eee9455e5/68747470733a2f2f6c617465782e636f6465636f67732e636f6d2f6769662e6c617465783f253543696e6c696e652532302535437468657461"><img src="https://camo.githubusercontent.com/ed6801acad814e8d5be5770293fa0d7eee9455e5/68747470733a2f2f6c617465782e636f6465636f67732e636f6d2f6769662e6c617465783f253543696e6c696e652532302535437468657461" alt="θ" data-canonical-src="https://latex.codecogs.com/gif.latex?%5Cinline%20%5Ctheta" style="max-width:100%;"></a> in the plane. The control inputs are speed <a target="_blank" rel="noopener noreferrer" href="https://camo.githubusercontent.com/227f041ada8e434d11f3a781074c3de5e08d45f2/68747470733a2f2f6c617465782e636f6465636f67732e636f6d2f6769662e6c617465783f253543696e6c696e6525323076"><img src="https://camo.githubusercontent.com/227f041ada8e434d11f3a781074c3de5e08d45f2/68747470733a2f2f6c617465782e636f6465636f67732e636f6d2f6769662e6c617465783f253543696e6c696e6525323076" alt="v" data-canonical-src="https://latex.codecogs.com/gif.latex?%5Cinline%20v" style="max-width:100%;"></a> and curvature <a target="_blank" rel="noopener noreferrer" href="https://camo.githubusercontent.com/5def7848ddfc8f2e87dcceced2f9e1e236426ca5/68747470733a2f2f6c617465782e636f6465636f67732e636f6d2f6769662e6c617465783f253543696e6c696e652532302535436b61707061"><img src="https://camo.githubusercontent.com/5def7848ddfc8f2e87dcceced2f9e1e236426ca5/68747470733a2f2f6c617465782e636f6465636f67732e636f6d2f6769662e6c617465783f253543696e6c696e652532302535436b61707061" alt="κ" data-canonical-src="https://latex.codecogs.com/gif.latex?%5Cinline%20%5Ckappa" style="max-width:100%;"></a>, or some derivative of each of them (e.g., curvature rate <a target="_blank" rel="noopener noreferrer" href="https://camo.githubusercontent.com/e7f4cef675c825b632607ca5dcf1916b854bf2c2/68747470733a2f2f6c617465782e636f6465636f67732e636f6d2f6769662e6c617465783f253543696e6c696e65253230253543646f742535436b61707061"><img src="https://camo.githubusercontent.com/e7f4cef675c825b632607ca5dcf1916b854bf2c2/68747470733a2f2f6c617465782e636f6465636f67732e636f6d2f6769662e6c617465783f253543696e6c696e65253230253543646f742535436b61707061" alt="κ̇" data-canonical-src="https://latex.codecogs.com/gif.latex?%5Cinline%20%5Cdot%5Ckappa" style="max-width:100%;"></a> as input ensures trajectories with continuous curvature) with the state and dynamics equation correspondingly augmented. This package exports the type <code>SimpleCarDynamics{Dv,Dκ} &lt;: DifferentialDynamics</code> to represent these dynamics, where <code>Dv</code> and <code>Dκ</code> denote the number of integrators in the speed and curvature control inputs respectively.</p>
<p>In addition to providing these dynamics and a few methods for exact propagation, this package also contains pure Julia implementations of minimum arc length <a href="http://planning.cs.uiuc.edu/node821.html" rel="nofollow">Dubins</a> and <a href="http://planning.cs.uiuc.edu/node822.html" rel="nofollow">Reeds-Shepp</a> steering for a simple car with minimum turning radius <code>r</code>. These implementations aim to be non-allocating and highly performant (e.g., for use in robotic motion planning where computing millions of steering connections in a few seconds may be necessary), and may be accessed through the <code>SteeringBVP</code> interface defined in <a href="https://github.com/schmrlng/DifferentialDynamicsModels.jl"><code>DifferentialDynamicsModels.jl</code></a>:</p>
<ul>
<li><code>DubinsSteering(; v=1, r=1)</code> returns a <code>DubinsSteering</code> instance which may be called with two <a href="https://github.com/schmrlng/SimpleCarModels.jl/blob/master/src/models.jl#L14"><code>SE2State</code></a>s as input, or any pair of <a href="https://github.com/JuliaArrays/StaticArrays.jl"><code>StaticVector</code></a>s of length 3.</li>
<li><code>ReedsSheppSteering(; v=1, r=1)</code> returns a <code>ReedsSheppSteering</code> instance which may be called as above.</li>
</ul>
<p>or through specialized functions:</p>
<ul>
<li><code>dubins_length(q0, qf; r=1)</code> and <code>dubins_waypoints(q0, qf, dt_or_N; v=1, r=1)</code> give the length and a <code>Vector</code> of states along the optimal Dubins steering curve respectively. <code>dt_or_N</code> may be an <code>AbstractFloat</code> or an <code>Int</code>, corresponding to a desired time spacing <code>dt</code> (with car speed <code>v</code>) or a desired total number of equally spaced waypoints <code>N</code>.</li>
<li><code>reedsshepp_length(q0, qf; r=1)</code> and <code>reedsshepp_waypoints(q0, qf, dt_or_N; v=1, r=1)</code> give the length and a <code>Vector</code> of states along the optimal Reeds-Shepp steering curve respectively.</li>
</ul>
