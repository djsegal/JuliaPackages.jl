For example:</p>
<ul>
<li>Generic index for tab-delimited files.</li>
<li>An index type for BGZFStream.</li>
</ul>
<h2><a id="user-content-installation" class="anchor" aria-hidden="true" href="#installation"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path fill-rule="evenodd" d="M7.775 3.275a.75.75 0 001.06 1.06l1.25-1.25a2 2 0 112.83 2.83l-2.5 2.5a2 2 0 01-2.83 0 .75.75 0 00-1.06 1.06 3.5 3.5 0 004.95 0l2.5-2.5a3.5 3.5 0 00-4.95-4.95l-1.25 1.25zm-4.69 9.64a2 2 0 010-2.83l2.5-2.5a2 2 0 012.83 0 .75.75 0 001.06-1.06 3.5 3.5 0 00-4.95 0l-2.5 2.5a3.5 3.5 0 004.95 4.95l1.25-1.25a.75.75 0 00-1.06-1.06l-1.25 1.25a2 2 0 01-2.83 0z"></path></svg></a>Installation</h2>
<p>The Indexes package is bundled into packages like <a href="https://github.com/BioJulia/BED.jl">BED</a>, <a href="https://github.com/BioJulia/GFF3.jl">GFF3</a>, and <a href="https://github.com/BioJulia/XAM.jl">XAM</a> to assist them with IO streams, so you may not need to install this package explicitly.
However, if you do, Indexes is made available to install through the [<a href="https://github.com/juliaregistries/General](General">https://github.com/juliaregistries/General](General</a> package registry).</p>
<p>You can install Indexes from the <a href="https://docs.julialang.org/en/v1/manual/getting-started/" rel="nofollow">Julia REPL</a>.
Press <code>]</code> to enter <a href="https://docs.julialang.org/en/v1/stdlib/Pkg/" rel="nofollow">pkg mode</a>, then enter the following command:</p>
<div class="highlight highlight-source-julia"><pre>add Indexes</pre></div>
<p>If you are interested in the cutting edge of the development, please check out the <a href="https://github.com/BioJulia/Indexes.jl/tree/develop">develop branch</a> to try new features before release.</p>
