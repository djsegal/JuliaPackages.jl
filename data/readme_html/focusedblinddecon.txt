<p><code>FocusedBlindDecon</code> is a Julia package that implements the following methods.</p>
<p><strong><code>lsbd!</code></strong> Least-squares blind deconvolution.</p>
<p><strong><code>fibd!</code></strong> Focused interferometric blind deconvolution.</p>
<p><strong><code>fpr!</code></strong> Focused phase retrieval.</p>
<p><strong><code>fbd!</code></strong>=<strong><code>fibd!</code></strong>+<strong><code>fpr!</code></strong> Focused blind deconvolution.</p>
<p>These methods are described in the article below; if you make use of this code, please cite it.</p>
<div class="highlight highlight-text-tex-latex"><pre>@article{bharadwaj2019focused,
  title={Focused blind deconvolution},
  author={Bharadwaj, Pawan and Demanet, Laurent and Fournier, Aim{<span class="pl-cce">\'</span>e}},
  journal={IEEE Transactions on Signal Processing},
  volume={67},
  number={12},
  pages={3168--3180},
  year={2019},
  publisher={IEEE}
}</pre></div>
