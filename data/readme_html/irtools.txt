<p dir="auto">IRTools aims to provide a simple and flexible IR format, expressive enough to work with both lowered and typed Julia code, as well as external IRs. It can be used with Julia metaprogramming tools such as <a href="https://github.com/jrevels/Cassette.jl">Cassette</a>.</p>

         r = 1

         end
         return r
       end

  br 2 (%7, %8)
4:
