<p dir="auto">A simplistic julia package implementing multi graphs.
This package aims to be fully compliant with <a href="https://github.com/JuliaGraphs/Graphs.jl">Graphs.jl</a>.</p>

<p dir="auto">Since the code needed is extremely similar to Graphs.SimpleGraphs we wrap a <code>SimpleGraph</code> type in a <code>MultiGraph</code> type.
Otherwise extreme code repetition is needed.
Instead we use composition and forward useful functions.
Use <code>graph()</code> to access the underlying wrapped graph.
Attentions: we also wrap <code>getproperty</code>, so the dot syntax returns fields of the wrapped struct.</p>
<p dir="auto">Basically, this package kills this <a href="https://github.com/JuliaGraphs/Graphs.jl/blob/c4360cfaca3936c3d3f784063ad312205cb4fdfe/src/SimpleGraphs/simplegraph.jl#L444">if statement</a> and deals with the repercussions.
The function <code>edges()</code> will return <code>AbstractEdge</code>that will repeat if there are multiple edges.</p>
