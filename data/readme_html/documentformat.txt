<p dir="auto">The main function to format code is <code>format</code>. When called with a string argument, that string is assumed to be code and a new string in which the code is formatted is returned. When called with an <code>AbstractPath</code> that points to a file, that file is being formatted. If called with an <code>AbstractPath</code> that points to a folder, all <code>*.jl</code> files in that folder are formatted.</p>
<p dir="auto">The function <code>isformatted</code> checks whether a piece of code is formatted. It can be called with a string (assumed to hold code) or an <code>AbstractPath</code>.</p>
