<p dir="auto">This package is NOT equal to the package <code>WorldBankData</code> from the
official Julia package repository! Although strongly related, the
package <code>WorldBankDataTd</code> returns data in <code>TimeData</code> format
(<code>DataFrames</code> are possible as well). Also, caching of country and
indicator data is more aligned to the concepts used in <code>EconDatasets</code>.</p>
