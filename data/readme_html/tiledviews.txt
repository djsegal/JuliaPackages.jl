<p dir="auto">This package allows to view an N-dimensional array as an 2N-dimensional <code>TiledView</code> being separated in overlapping tiles.
The tiled view has read and write access.
Via the <code>TiledWindowView</code> it is possible to imprint a weight-window onto the tiled view. By default the window is chosen such that
it sums up to one except in places very close to the border, where an insufficient number of contributions are generated.
However this can effect can easily be accounted for, since it optionally returns an overall weight distribution.</p>

<p dir="auto">The toolbox also offers support for iterators on the tiles via the functions <code>eachtile()</code>, <code>eachtilenumber()</code>, and <code>eachtilerelpos()</code>.
A very convenient way of processing all tiles with a user-supplied function and fusing the images automatically via window-based weighting is using
the function <code>tiled_processing()</code>.</p>
