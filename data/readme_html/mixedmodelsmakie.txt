<p dir="auto"><code>MixedModelsMakie.jl</code> is a Julia package providing plotting capabilities for models fit with <a href="https://juliastats.org/MixedModels.jl/stable/" rel="nofollow">MixedModels.jl</a>.
Plotting is performed using the <a href="https://makie.juliaplots.org/stable/" rel="nofollow">Makie ecoysystem</a>, with the interface defined using <code>Makie.jl</code> (<a href="https://discourse.julialang.org/t/ann-makie-v-0-13/61522" rel="nofollow">previously <code>AbstractPlotting.jl</code></a>) for compatibility across all Makie backends.</p>
<p dir="auto">Note that the functionality here is currently early development and so breaking changes are expected as we refine the interface.
Following <a href="https://semver.org/" rel="nofollow">SemVer</a>, these minor releases before 1.0 can introduce these breaking changes.
The release of 1.0 will be indication that we believe the interface is reasonably stable.
Minor refinements of graphical displays without changing the API are considered non-breaking.</p>
