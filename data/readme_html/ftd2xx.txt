<p><a href="https://travis-ci.org/cstook/FTD2XX.jl" rel="nofollow"><img src="https://camo.githubusercontent.com/5ffd90b243dc3a652f96a165ea41478ef4e1fed2/68747470733a2f2f7472617669732d63692e6f72672f6373746f6f6b2f4654443258582e6a6c2e7376673f6272616e63683d6d6173746572" alt="Build Status" data-canonical-src="https://travis-ci.org/cstook/FTD2XX.jl.svg?branch=master" style="max-width:100%;"></a></p>
<p>FTD2XX.jl is a thin wrapper for <a href="http://www.ftdichip.com" rel="nofollow">FTDI</a>'s <a href="http://www.ftdichip.com/Drivers/D2XX.htm" rel="nofollow">D2XX</a> driver.</p>
<p>##Installation</p>
<p>FTD2XX.jl is currently unregistered.  It can be installed using <code>Pkg.clone</code>.</p>
<div class="highlight highlight-source-julia"><pre>Pkg<span class="pl-k">.</span><span class="pl-c1">clone</span>(<span class="pl-s"><span class="pl-pds">"</span>https://github.com/cstook/FTD2XX.jl.git<span class="pl-pds">"</span></span>)</pre></div>
<p>The <a href="http://docs.julialang.org" rel="nofollow">julia documentation</a> section on installing unregistered <a href="http://docs.julialang.org/en/release-0.4/manual/packages/#packages" rel="nofollow">packages</a> provides more information.</p>
<h2><a id="user-content-uart-quick-start" class="anchor" aria-hidden="true" href="#uart-quick-start"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path fill-rule="evenodd" d="M7.775 3.275a.75.75 0 001.06 1.06l1.25-1.25a2 2 0 112.83 2.83l-2.5 2.5a2 2 0 01-2.83 0 .75.75 0 00-1.06 1.06 3.5 3.5 0 004.95 0l2.5-2.5a3.5 3.5 0 00-4.95-4.95l-1.25 1.25zm-4.69 9.64a2 2 0 010-2.83l2.5-2.5a2 2 0 012.83 0 .75.75 0 001.06-1.06 3.5 3.5 0 00-4.95 0l-2.5 2.5a3.5 3.5 0 004.95 4.95l1.25-1.25a.75.75 0 00-1.06-1.06l-1.25 1.25a2 2 0 01-2.83 0z"></path></svg></a>UART quick start</h2>
<p>The UART may be accessed as type IOftuart &lt;: IO</p>
<p>get information about devices</p>
<div class="highlight highlight-source-julia"><pre>device_infomation_list <span class="pl-k">=</span> <span class="pl-c1">ft_getdeviceinfolist</span>()</pre></div>
<p>open a device.</p>
<div class="highlight highlight-source-julia"><pre>io <span class="pl-k">=</span> <span class="pl-c1">open</span>(<span class="pl-c1">FT_SerialNumber</span>(<span class="pl-s"><span class="pl-pds">"</span>FTXRNZUJ<span class="pl-pds">"</span></span>),<span class="pl-c1">9600</span>,<span class="pl-c1">8</span>,<span class="pl-c1">1</span>,<span class="pl-s"><span class="pl-pds">"</span>n<span class="pl-pds">"</span></span>) <span class="pl-c"><span class="pl-c">#</span> open by serial number</span></pre></div>
<p>or</p>
<div class="highlight highlight-source-julia"><pre>io <span class="pl-k">=</span> <span class="pl-c1">open</span>(<span class="pl-c1">FT_Description</span>(<span class="pl-s"><span class="pl-pds">"</span>C232HM-EDHSL-0<span class="pl-pds">"</span></span>),<span class="pl-c1">9600</span>,<span class="pl-c1">8</span>,<span class="pl-c1">1</span>,<span class="pl-s"><span class="pl-pds">"</span>n<span class="pl-pds">"</span></span>) <span class="pl-c"><span class="pl-c">#</span> open by description</span></pre></div>
<p>or</p>
<div class="highlight highlight-source-julia"><pre>io <span class="pl-k">=</span> <span class="pl-c1">open</span>(<span class="pl-c1">FT_Location</span>(<span class="pl-c1">0x000001a2</span>),<span class="pl-c1">9600</span>,<span class="pl-c1">8</span>,<span class="pl-c1">1</span>,<span class="pl-s"><span class="pl-pds">"</span>n<span class="pl-pds">"</span></span>) <span class="pl-c"><span class="pl-c">#</span> open by location</span></pre></div>
<p>or</p>
<div class="highlight highlight-source-julia"><pre>io <span class="pl-k">=</span> <span class="pl-c1">open</span>(<span class="pl-c1">FT_DeviceIndex</span>(<span class="pl-c1">0</span>),<span class="pl-c1">9600</span>,<span class="pl-c1">8</span>,<span class="pl-c1">1</span>,<span class="pl-s"><span class="pl-pds">"</span>n<span class="pl-pds">"</span></span>) <span class="pl-c"><span class="pl-c">#</span> open by device index</span></pre></div>
<p>or</p>
<div class="highlight highlight-source-julia"><pre>uartconfig <span class="pl-k">=</span> <span class="pl-c1">UARTConfiguration</span>(<span class="pl-c1">9600</span>,<span class="pl-c1">8</span>,<span class="pl-c1">1</span>,<span class="pl-s"><span class="pl-pds">"</span>n<span class="pl-pds">"</span></span>,
							   readtimeout <span class="pl-k">=</span> <span class="pl-c1">1000</span>,		<span class="pl-c"><span class="pl-c">#</span> in milliseconds</span>
							   writetimeout <span class="pl-k">=</span> <span class="pl-c1">1000</span>,		<span class="pl-c"><span class="pl-c">#</span> in milliseconds</span>
							   flowcontrol <span class="pl-k">=</span> <span class="pl-s"><span class="pl-pds">"</span>none<span class="pl-pds">"</span></span>,	<span class="pl-c"><span class="pl-c">#</span> none, rts_cts, dtr_dsr, xon_xoff</span>
							   xon <span class="pl-k">=</span> <span class="pl-c1">0x11</span>,
							   xoff <span class="pl-k">=</span> <span class="pl-c1">0x13</span>)
io <span class="pl-k">=</span> <span class="pl-c1">open</span>(<span class="pl-c1">FT_DeviceIndex</span>(<span class="pl-c1">0</span>),uartconfig)</pre></div>
<p>use all the normal IO functions</p>
<div class="highlight highlight-source-julia"><pre><span class="pl-c1">write</span>(io, <span class="pl-c1">0x55</span>)
byteread <span class="pl-k">=</span> <span class="pl-c1">read</span>(io, UInt8)</pre></div>
<p>close the device</p>
<div class="highlight highlight-source-julia"><pre><span class="pl-c1">close</span>(io)</pre></div>

<p>##OS Compatibility</p>
<p>FTD2XX.jl is compatible with windows, linux, and osx.  See FTDI's <a href="http://www.ftdichip.com/Support/Documents/InstallGuides.htm" rel="nofollow">installation guides</a> for instructions on installing drivers for your operating system.  Drivers are assumed to be installed in the default locations.</p>
