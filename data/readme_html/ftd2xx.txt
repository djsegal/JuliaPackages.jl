<p dir="auto"><a href="https://travis-ci.org/cstook/FTD2XX.jl" rel="nofollow"><img src="https://camo.githubusercontent.com/b21c8a9e167c8c92b04e723465d2b6ed5983d1ea625f30ea23186d74dd79d752/68747470733a2f2f7472617669732d63692e6f72672f6373746f6f6b2f4654443258582e6a6c2e7376673f6272616e63683d6d6173746572" alt="Build Status" data-canonical-src="https://travis-ci.org/cstook/FTD2XX.jl.svg?branch=master" style="max-width: 100%;"></a></p>
<p dir="auto">FTD2XX.jl is a thin wrapper for <a href="http://www.ftdichip.com" rel="nofollow">FTDI</a>'s <a href="http://www.ftdichip.com/Drivers/D2XX.htm" rel="nofollow">D2XX</a> driver.</p>
<p dir="auto">##Installation</p>
<p dir="auto">FTD2XX.jl is currently unregistered.  It can be installed using <code>Pkg.clone</code>.</p>
<div class="highlight highlight-source-julia notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="Pkg.clone(&quot;https://github.com/cstook/FTD2XX.jl.git&quot;)"><pre>Pkg<span class="pl-k">.</span><span class="pl-c1">clone</span>(<span class="pl-s"><span class="pl-pds">"</span>https://github.com/cstook/FTD2XX.jl.git<span class="pl-pds">"</span></span>)</pre></div>
<p dir="auto">The <a href="http://docs.julialang.org" rel="nofollow">julia documentation</a> section on installing unregistered <a href="http://docs.julialang.org/en/release-0.4/manual/packages/#packages" rel="nofollow">packages</a> provides more information.</p>

<p dir="auto">##OS Compatibility</p>
<p dir="auto">FTD2XX.jl is compatible with windows, linux, and osx.  See FTDI's <a href="http://www.ftdichip.com/Support/Documents/InstallGuides.htm" rel="nofollow">installation guides</a> for instructions on installing drivers for your operating system.  Drivers are assumed to be installed in the default locations.</p>
