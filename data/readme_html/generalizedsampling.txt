<p>The examples use the <a href="https://github.com/tbreloff/Plots.jl">Plots package</a> for plotting examples.
<em>Plots</em> requires a backend as described in its <a href="https://juliaplots.github.io/backends" rel="nofollow">documentation</a>.
I have tested the <a href="https://github.com/jheinen/GR.jl">GR backend</a> with the examples in <em>GeneralizedSampling</em>;
the <em>Plots</em> interaction with GR is described in these <a href="https://juliaplots.github.io/examples/gr" rel="nofollow">Plots examples</a>.</p>
<p>The example file <code>examples/brain.jl</code> uses precomputed data from this <a href="http://bigwww.epfl.ch/algorithms/mriphantom" rel="nofollow">MRI phantom Matlab package</a> and the data is <em>not</em> included in the package as it takes up about 21 MB.
The data is available for download in the native Julia format <code>jld</code> from <a href="http://people.math.aau.dk/~robert/software/brain.zip" rel="nofollow">http://people.math.aau.dk/~robert/software/brain.zip</a>.
Unzip the three <code>jld</code> files in the <code>examples</code> folder.</p>
