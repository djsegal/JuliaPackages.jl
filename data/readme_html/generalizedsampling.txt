<p dir="auto">A Julia package implementing the generalized sampling framework by Anders Hansen, Ben Adcock et al. from Fourier measurements to Daubechies wavelets.</p>
<p dir="auto"><strong>Note</strong>: This package has not been updated in quite a while and will not run on current versions of Julia.
It is easy to try with an appropriate <a href="https://hub.docker.com/_/julia" rel="nofollow">Docker image</a>:</p>
<div class="snippet-clipboard-content notranslate position-relative overflow-auto" data-snippet-clipboard-copy-content="docker run --rm -it julia:0.4"><pre class="notranslate"><code>docker run --rm -it julia:0.4
</code></pre></div>
<p dir="auto">I hope to get around to making <em>GeneralizedSampling</em> work on newer versions of Julia.</p>

<p dir="auto">The examples use the <a href="https://github.com/tbreloff/Plots.jl">Plots package</a> for plotting examples.
<em>Plots</em> requires a backend as described in its <a href="https://juliaplots.github.io/backends" rel="nofollow">documentation</a>.
I have tested the <a href="https://github.com/jheinen/GR.jl">GR backend</a> with the examples in <em>GeneralizedSampling</em>;
the <em>Plots</em> interaction with GR is described in these <a href="https://juliaplots.github.io/examples/gr" rel="nofollow">Plots examples</a>.</p>
<p dir="auto">The example file <code>examples/brain.jl</code> uses precomputed data from this <a href="http://bigwww.epfl.ch/algorithms/mriphantom" rel="nofollow">MRI phantom Matlab package</a> and the data is <em>not</em> included in the package as it takes up about 21 MB.
The data is available for download in the native Julia format <code>jld</code> from <a href="http://dahl-jacobsen.dk/generalizedsampling/brain.zip" rel="nofollow">http://dahl-jacobsen.dk/generalizedsampling/brain.zip</a>.
Unzip the three <code>jld</code> files in the <code>examples</code> folder.</p>
