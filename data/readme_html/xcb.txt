<p>Wrapper around the <a href="https://xcb.freedesktop.org/" rel="nofollow">X protocol C-language Binding (XCB)</a> library for interacting with X window systems.
Includes a Clang-generated library wrapper along with a few abstractions including setting up connections, graphics contexts and windows, handling input events, and running a window. For more information, see the <a href="https://juliagl.github.io/XCB.jl/stable" rel="nofollow">package documentation</a>.
Users wishing to use the Clang wrapper directly will have to use <code>XCB.Libxcb</code>. For documentation, they should refer to the <a href="https://xcb.freedesktop.org/" rel="nofollow">XCB documentation</a>.</p>
