<p>Fancy terminal plotting for Julia using Braille characters.
Inspired by <a href="https://github.com/asciimoo/drawille">Drawille</a> but not based on it
due to its restrictive license (AGPL). TextPlots.jl is free for all under the MIT
license.</p>
<p>TextPlots.jl aims for as much elegance and aesthetics as possible for a terminal
(unicode) plotting library. It should be able to plot any continuous real-valued
function, and any small-to-medium collection of points.</p>
