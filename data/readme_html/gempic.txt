<p dir="auto">In a Julia session switch to <code>pkg&gt;</code> mode to add <code>GEMPIC</code>:</p>
<div class="highlight highlight-source-julia notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="julia&gt;] # switch to pkg&gt; mode
pkg&gt; add GEMPIC"><pre>julia<span class="pl-k">&gt;</span>] <span class="pl-c"><span class="pl-c">#</span> switch to pkg&gt; mode</span>
pkg<span class="pl-k">&gt;</span> add GEMPIC</pre></div>
<p dir="auto">When finished, make sure that you're back to the Julian prompt (<code>julia&gt;</code>)
and bring <code>GEMPIC</code> into scope:</p>
<div class="highlight highlight-source-julia notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="julia&gt; using GEMPIC"><pre>julia<span class="pl-k">&gt;</span> <span class="pl-k">using</span> GEMPIC</pre></div>
