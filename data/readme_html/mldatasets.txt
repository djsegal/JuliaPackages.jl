<p dir="auto">This package represents a community effort to provide a common interface for accessing common Machine Learning datasets.
In contrast to other data-related Julia packages, the focus of MLDatasets.jl is specifically on downloading, unpacking, and accessing benchmark datasets.
Functionality for the purpose of data processing or visualization is only provided to a degree that is special to some dataset.</p>
<p dir="auto">This package is a part of the
<a href="https://github.com/JuliaML">JuliaML</a> ecosystem.
Its functionality is built on top of the package
<a href="https://github.com/oxinabox/DataDeps.jl">DataDeps.jl</a>.</p>

<p dir="auto">Datasets are grouped into different categories. Click on the links below for a full list of datasets available in each category.</p>
<ul dir="auto">
<li><a href="https://juliaml.github.io/MLDatasets.jl/dev/datasets/graphs" rel="nofollow">Graphs</a> - Datasets with an underlying graph structure: Cora, PubMed, CiteSeer, ...</li>
<li><a href="https://juliaml.github.io/MLDatasets.jl/dev/datasets/misc/" rel="nofollow">Misc</a> - Datasets that do not fall into any of the other categories: Iris, BostonHousing, ...</li>
<li><a href="https://juliaml.github.io/MLDatasets.jl/dev/datasets/text/" rel="nofollow">Text</a> - Datasets for language models.</li>
<li><a href="https://juliaml.github.io/MLDatasets.jl/dev/datasets/vision/" rel="nofollow">Vision</a> - Vision related datasets such as MNIST, CIFAR10, CIFAR100, ...</li>
</ul>
