<p dir="auto">Fermi.jl is a quantum chemistry framework written in pure Julia. This code is developed at
the <a href="https://github.com/CCQC">Center for Computational Quantum Chemistry</a> at the University of Georgia under the supervision
of <a href="https://github.com/jturney">Dr. Justin M. Turney</a> and Prof. Henry F. Schaefer.</p>
<p dir="auto">This work is supported by the U.S. National Science Foundation under grant number CHE-1661604.</p>
<p dir="auto">For an academic overview of this project, check our paper published in the Journal of Chemical Theory and Computation:
<a href="https://pubs.acs.org/doi/10.1021/acs.jctc.1c00719" rel="nofollow">Fermi.jl: A Modern Design for Quantum Chemistry</a></p>
<p dir="auto">Fermi focuses on post Hartree--Fock methods. Currently, only restricted references are supported.
This is intended as a research code with an ever growing collection of methods implemented in
the package itself. However, the Fermi API is designed to make high performance pilot implementations
of methods achievable.</p>

<p dir="auto"><a href="https://github.com/FermiQC/Molecules.jl">Molecules.jl</a>: A package that deals with <code>Atom</code> objects. It can parse XYZ files and compute properties related to the position of nuclei.</p>
<p dir="auto"><a href="https://github.com/FermiQC/GaussianBasis.jl">GaussianBasis.jl</a>: A library for integrals and basis set objects. It can parse <code>.gbs</code> basis set files and create <code>BasisFunction</code> and <code>BasisSet</code> structures. Integrals over Gaussian basis are computed using <code>libcint</code>.</p>

<p dir="auto"><strong>1. New Methods for Energy Computations</strong></p>
<ul dir="auto">
<li>Unrestricted MP2</li>
<li>Unrestricted CC and High order CC methods (e.g. CCSDT, CCSDTQ)</li>
<li>Configuration Interaction (CID, CISD, FCI, selective CI, etc)</li>
<li>Relativistic Methods</li>
<li>Methods beyond Born-Oppenheimer approximation</li>
<li>Excited States</li>
</ul>
<p dir="auto"><strong>2. Properties</strong></p>
<ul dir="auto">
<li>Dipole Moments</li>
<li>Polarizability</li>
<li>IR intensities</li>
<li>Harmonic Frequencies</li>
<li>Anharmonic Frequencies</li>
</ul>
<p dir="auto"><strong>3. Computational schemes</strong></p>
<ul dir="auto">
