<p dir="auto">A Julia wrapper for <a href="https://github.com/algoim/algoim">algoim</a>, providing algorithms for implicitly defined geometry, level set methods, and Voronoi implicit interface methods</p>
<p dir="auto"><strong>If you use this package, please cite algoim, according to the guidelines on the <a href="https://algoim.github.io" rel="nofollow">algoim Github page</a>.</strong></p>
<p dir="auto">Currently, this code only uses algoim's quadrature algorithms for domains implicitly-defined by multivariate polynomials. Hence, the proper work to cite is <a href="https://algoim.github.io/#citing-1" rel="nofollow">this one</a>:</p>
<blockquote>
<p dir="auto"><a href="https://www.sciencedirect.com/science/article/pii/S002199912100615X?via%3Dihub" rel="nofollow">R. I. Saye, High-order quadrature on multi-component domains implicitly defined by multivariate polynomials, Journal of Computational Physics, 448, 110720 (2022).</a></p>
</blockquote>
