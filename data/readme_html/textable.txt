<p>This package aims to replicate functionality from the R package
<a href="https://www.rdocumentation.org/packages/xtable/versions/1.8-4/topics/xtable" rel="nofollow">xtable</a>.
It currently supports the creation of LaTeX tables from simple arrays and matrices.</p>
<h1><a id="user-content-quickstart" class="anchor" aria-hidden="true" href="#quickstart"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path fill-rule="evenodd" d="M7.775 3.275a.75.75 0 001.06 1.06l1.25-1.25a2 2 0 112.83 2.83l-2.5 2.5a2 2 0 01-2.83 0 .75.75 0 00-1.06 1.06 3.5 3.5 0 004.95 0l2.5-2.5a3.5 3.5 0 00-4.95-4.95l-1.25 1.25zm-4.69 9.64a2 2 0 010-2.83l2.5-2.5a2 2 0 012.83 0 .75.75 0 001.06-1.06 3.5 3.5 0 00-4.95 0l-2.5 2.5a3.5 3.5 0 004.95 4.95l1.25-1.25a.75.75 0 00-1.06-1.06l-1.25 1.25a2 2 0 01-2.83 0z"></path></svg></a>Quickstart</h1>
<div class="highlight highlight-source-julia"><pre><span class="pl-k">using</span> TeXTable;
A <span class="pl-k">=</span> [<span class="pl-c1">1</span> <span class="pl-c1">2</span> <span class="pl-c1">3</span>; <span class="pl-c1">4</span> <span class="pl-c1">5</span> <span class="pl-c1">6</span>; <span class="pl-c1">7</span> <span class="pl-c1">8</span> <span class="pl-c1">9</span>];
<span class="pl-c1">Table</span>(A)</pre></div>
<p>This results in</p>
<pre><code>\begin{table}[!h]
    \centering
    \begin{tabular}{lll}
    1 &amp; 2 &amp; 3 \\
    4 &amp; 5 &amp; 6 \\
    7 &amp; 8 &amp; 9 \\
    \end{tabular}
\end{table}
</code></pre>
<p>More options for table can be found from</p>
<div class="highlight highlight-source-julia"><pre>? Table</pre></div>
