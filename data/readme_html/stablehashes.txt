<p>An effort to preserve the julia hashing functionailty as it was under v.1.4.2.</p>
<p>To avoid conflict with the standard julia hashing machinery, <code>StableHashes</code> exports the function <code>shash</code>, which can be used as a drop-in replacement for the standard <code>hash</code> function, and will return hashes as though they were computed under julia v.1.4.2</p>
