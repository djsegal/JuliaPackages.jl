<ul dir="auto">
<li>
<p dir="auto"><a href="https://github.com/JuliaStats/StatsBase.jl">StatsBase.jl</a></p>
<p dir="auto">Contains a couple robust statistics, but has no overlapping functionality with this package.</p>
</li>
<li>
<p dir="auto"><a href="https://github.com/mrxiaohe/RobustStats.jl">RobustStats.jl</a></p>
<p dir="auto">Contains many more robust statistics primarily based on the WRS R package. Appears to be unmaintained and not updated to Julia v1. The <code>bivar</code> function is the same as this package's <a href="@ref"><code>midvar</code></a>, although <code>bivar</code> does not have definitions for the statistics across axes of an array.</p>
</li>
<li>
<p dir="auto"><a href="https://github.com/astropy/astropy">astropy.stats</a></p>
<p dir="auto">Python implementations of all the statistics presented here. There are some slight differences in the function signatures, including the default cutoff value <code>c</code> (for some statistics).</p>
</li>
</ul>
