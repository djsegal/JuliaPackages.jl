<p dir="auto">Implements Singular Value Decomposition for generic number types, such as <code>BigFloat</code>, <code>Complex{BigFloat}</code> or <a href="https://github.com/JuliaGeometry/Quaternions.jl"><code>Quaternion</code>s</a>. It internally overloads several Base functions such that existing methods (<code>svd</code>, <code>svdfact</code> and <code>svdvals</code>) should work directly.</p>
<p dir="auto">It uses a Golub-Kahan 2-stage algorithm of bidiagonalization with Householder reflections, followed by an implicit QR with shift.</p>
