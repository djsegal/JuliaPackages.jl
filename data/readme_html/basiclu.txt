<p dir="auto"><a href="https://github.com/ERGO-Code/basiclu">BasicLU</a> implements a sparse LU factorization and an update method that maintains the factorization after column changes to the matrix.
It is intended for use in simplex-type algorithms and has been tailored to hypersparse linear programming problems.
It provides routines for solving linear systems with a dense or sparse right-hand side.
BasicLU can be also used to compute a maximum volume basis and the row deficiency of a m-by-n matrix A.</p>

<p dir="auto"><strong>Note: BasicLU is already precompiled with Yggdrasil for all platforms.</strong></p>
<p dir="auto">To use your custom BasicLU, set the environment variable <code>JULIA_BASICLU_LIBRARY_PATH</code>
to point to the folder holding the basicLU shared libraries before <code>using BasicLU</code>.</p>
<p dir="auto">The <code>JULIA_BASICLU_LIBRARY_PATH</code> environment variable may be set permanently in the shell's startup file, or in <code>$HOME/.julia/config/startup.jl</code>.</p>
