<p dir="auto">The package will be periodically updated with the following objectives in mind, in order of decreasing priority:</p>
<ol dir="auto">
<li>Technical function bug fixes</li>
<li>Improved interoperability between functions and types of GeometricalOptics.jl</li>
<li>Improved consistency with base Julia (generalizing or narrowing type declarations, for example)</li>
<li>Improved intraoperability with other packages</li>
<li>Generalization of existing functions</li>
<li>New functions</li>
</ol>
