<p dir="auto">The package performs piecewise linear interpolation over an arbitrary number of dimensions.  There are only two functions that need mentioning.  The first function computes a set of nodes spaced uniformly over a specified domain for a variable.</p>
<div class="highlight highlight-source-julia notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="nodes = piecewise_linear_nodes(n,domain)"><pre>nodes <span class="pl-k">=</span> <span class="pl-c1">piecewise_linear_nodes</span>(n,domain)</pre></div>
<p dir="auto">where <code>n</code> is an integer representing the desired number of nodes and <code>domain</code> is a 1D array containing the upper and lower values for the domain.  If no domain is specified, then it defaults to [1.0,-1.0].</p>
<p dir="auto">To evaluate the piecewise linear approximation at an arbitrary point in the domain we use the command</p>
<div class="highlight highlight-source-julia notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="y_hat = piecewise_linear_evaluate(y,nodes,point)"><pre>y_hat <span class="pl-k">=</span> <span class="pl-c1">piecewise_linear_evaluate</span>(y,nodes,point)</pre></div>
<p dir="auto">where <code>y</code> is a multidimensional array, <code>nodes</code> is a tuple of 1D-arrays or an array of 1D-arrays, and <code>point</code> is a 1D array.</p>
<p dir="auto">If <code>point</code> resides outside the domain in any dimension, then linear extrapolation in that dimension is performed.</p>
