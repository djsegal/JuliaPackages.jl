<p>CasaCore.jl is a Julia wrapper of <a href="http://casacore.github.io/casacore/" rel="nofollow">CasaCore</a>, which is a
commonly used library in radio astronomy.</p>
<p>Functionality is divided into two submodules:</p>
<ul>
<li><code>CasaCore.Tables</code> for interfacing with tables (for example Casa measurement sets), and</li>
<li><code>CasaCore.Measures</code> for performing coordinate system conversions (for example calculating the
azimuth and elevation of an astronomical target).</li>
</ul>
