<p>The official name of this package is "Hamiltonian Open Quantum System Toolkit" (HOQST). To conform with the Julia package name <a href="https://julialang.github.io/Pkg.jl/v1/creating-packages/" rel="nofollow">guidelines</a> the code name of the package is <code>OpenQuantumTools</code>. It is a Julia toolkit for simulating the open quantum system dynamics. The package is still under heavy development, and changes in the interface(s) are more than likely to happen. Detailed documentation can be found <a href="https://uscqserver.github.io/OpenQuantumTools.jl/dev/" rel="nofollow">here</a>. Any pull requests are welcome.</p>

<p><code>OpenQuantumTools</code> requires Julia 1.4 or higher. Installing it on an older version of Julia will result in an unsatisfiable requirements error.</p>

<p>It is recommended to install the following external packages:</p>
<h3><a id="user-content-plotsjl" class="anchor" aria-hidden="true" href="#plotsjl"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path fill-rule="evenodd" d="M7.775 3.275a.75.75 0 001.06 1.06l1.25-1.25a2 2 0 112.83 2.83l-2.5 2.5a2 2 0 01-2.83 0 .75.75 0 00-1.06 1.06 3.5 3.5 0 004.95 0l2.5-2.5a3.5 3.5 0 00-4.95-4.95l-1.25 1.25zm-4.69 9.64a2 2 0 010-2.83l2.5-2.5a2 2 0 012.83 0 .75.75 0 001.06-1.06 3.5 3.5 0 00-4.95 0l-2.5 2.5a3.5 3.5 0 004.95 4.95l1.25-1.25a.75.75 0 00-1.06-1.06l-1.25 1.25a2 2 0 01-2.83 0z"></path></svg></a><a href="https://github.com/JuliaPlots/Plots.jl">Plots.jl</a></h3>
<p>Plots is a visualization interface and toolset for Julia. <code>OpenQuantumTools.jl</code> provides several plotting functionality by recipes to <code>Plots.jl</code>.</p>
<h3><a id="user-content-differentialequationsjl" class="anchor" aria-hidden="true" href="#differentialequationsjl"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path fill-rule="evenodd" d="M7.775 3.275a.75.75 0 001.06 1.06l1.25-1.25a2 2 0 112.83 2.83l-2.5 2.5a2 2 0 01-2.83 0 .75.75 0 00-1.06 1.06 3.5 3.5 0 004.95 0l2.5-2.5a3.5 3.5 0 00-4.95-4.95l-1.25 1.25zm-4.69 9.64a2 2 0 010-2.83l2.5-2.5a2 2 0 012.83 0 .75.75 0 001.06-1.06 3.5 3.5 0 00-4.95 0l-2.5 2.5a3.5 3.5 0 004.95 4.95l1.25-1.25a.75.75 0 00-1.06-1.06l-1.25 1.25a2 2 0 01-2.83 0z"></path></svg></a><a href="http://docs.juliadiffeq.org/latest/" rel="nofollow">DifferentialEquations.jl</a></h3>
<p>Even though <code>OpenQuantumTools.jl</code> can function without <code>DifferentialEquations.jl</code>, it needs to be loaded in order for the master equation solvers to work properly. For <a href="http://docs.juliadiffeq.org/stable/features/low_dep.html#Low-Dependency-Usage-1" rel="nofollow">low dependency usage</a>, replacing <code>DifferentialEquations</code> by <a href="https://github.com/JuliaDiffEq/OrdinaryDiffEq.jl">OrdinaryDiffEq.jl</a> will also work.</p>
