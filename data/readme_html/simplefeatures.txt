<p dir="auto">Fast reading, analyzing, manipulating, and writing vector GIS data in Julia. This package:</p>
<ul dir="auto">
<li>Works with feature data as a <a href="https://dataframes.juliadata.org/stable/" rel="nofollow"><strong>DataFrame</strong></a></li>
<li>Stores coordinate reference system (crs) info with the DataFrame and uses it automatically</li>
<li>Uses a custom geometry type to interface with <a href="https://gdal.org" rel="nofollow"><strong>GDAL</strong></a> and <a href="https://libgeos.org" rel="nofollow"><strong>GEOS</strong></a> for fast I/O and operations</li>
</ul>

<p dir="auto">The <code>SimpleFeatures</code> package aims to simplify working with geospatial data in Julia.</p>
<p dir="auto">This package currently offers some basic spatial operations, and more will be added in the future. <a href="https://acgold.github.io/SimpleFeatures.jl/dev/" rel="nofollow">See docs for more info and examples</a>.</p>
<p dir="auto">This package creates a <code>SimpleFeature</code> custom type that effectively functions as a DataFrame containing feature data but also contains projection and geometry type info.</p>
<p dir="auto">Geometries are represented as another custom type called <code>sfgeom</code>. The <code>sfgeom</code> type follows the <a href="https://juliageo.org/GeoInterface.jl/stable/" rel="nofollow">GeoInterface</a> standard, so it can be converted into other geometry types and GeoInterface functions will work.</p>
