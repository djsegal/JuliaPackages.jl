<p>GtkReactive is designed to simplify the creation of graphical user interfaces (GUIs) using <a href="https://github.com/JuliaGraphics/Gtk.jl">Gtk</a> and Julia.
GtkReactive descended from
<a href="https://github.com/jverzani/GtkInteract.jl">GtkInteract</a>, and mimics
aspects of the design of
<a href="https://github.com/JuliaGizmos/Interact.jl">Interact</a>. To learn how
to use this package, please see the documentation:</p>
