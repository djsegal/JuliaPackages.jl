<ul dir="auto">
<li><code>readFastq</code> - Import a .fastq file into Julia</li>
<li><code>readFasta</code> - Import a .fasta file into Julia</li>
<li><code>writeFasta</code> - Write a .fasta or .fasta.gz to file</li>
<li><code>FilterQuality_se</code> - Filter single-ended Illumina reads</li>
<li><code>FilterQuality_pe</code> - Filter paired-end Illumina reads</li>
<li><code>FastqtoFasta</code> - Convert a .fastq file or FastqRecord to a FastaRecord</li>
<li><code>potential_mismatches</code> - Create a list of potential mismatch barcodes (mutations + deletions)</li>
<li><code>demultiplex_se</code> - Demultiplex single-end Illumina reads</li>
<li><code>demultiplex_pe</code> - Demultiplex paired-end Illumina reads</li>
<li><code>PlotQuality</code> - Visualize quality of .fastq file</li>
</ul>
