<ol dir="auto">
<li>Install anaconda by following the instructions on this <a href="https://docs.anaconda.com/anaconda/install/index.html" rel="nofollow">website</a>.</li>
<li>Run <code>echo export JULIA_CONDAPKG_BACKEND="System" &gt; ~/.bashrc</code> in your terminal.</li>
<li>Run <code>source ~/.bashrc</code> in your terminal.</li>
<li>Open your Julia REPL by typing  <code>julia</code> in your terminal.</li>
<li>Press <code>]</code> on your keyboard to enter the package manager</li>
<li>Enter command <code>add https://github.com/adubredu/MuJoCo.jl</code> and press
<code>Enter</code> on your keyboard to install this package.</li>
<li>Press the <code>Backspace</code> key on your keyboard to return to the REPL</li>
</ol>

<p dir="auto">This package wraps DeepMind's <a href="https://github.com/deepmind/dm_control">dm_control</a> Python package that provides access to the core MuJoCo package as well as some useful environments for Reinforcement Learning. It also exports the entire numpy package for convenience.</p>
<p dir="auto">This package also wraps <a href="https://github.com/rohanpsingh">rohanpsingh's</a> amazing <a href="https://github.com/rohanpsingh/mujoco-python-viewer">mujoco-python-viewer</a> for interactive simulation.</p>
