<p><code>COBRA.jl</code> is a package written in <a href="http://julialang.org/downloads/" rel="nofollow">Julia</a> used to perform COBRA analyses such as Flux Balance Anlysis (FBA), Flux Variability Anlysis (FVA), or any of its associated variants such as <code>distributedFBA</code> [<a href="#References-1">1</a>].</p>
<p>FBA and FVA rely on the solution of LP problems. The package can be used with ease when the LP problem is defined in a <code>.mat</code> file according to the format outlined in the <a href="https://github.com/opencobra/cobratoolbox">COBRA Toolbox</a> [<a href="#References-1">2</a>].</p>
<h2><a id="user-content-installation-of-cobrajl" class="anchor" aria-hidden="true" href="#installation-of-cobrajl"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path fill-rule="evenodd" d="M7.775 3.275a.75.75 0 001.06 1.06l1.25-1.25a2 2 0 112.83 2.83l-2.5 2.5a2 2 0 01-2.83 0 .75.75 0 00-1.06 1.06 3.5 3.5 0 004.95 0l2.5-2.5a3.5 3.5 0 00-4.95-4.95l-1.25 1.25zm-4.69 9.64a2 2 0 010-2.83l2.5-2.5a2 2 0 012.83 0 .75.75 0 001.06-1.06 3.5 3.5 0 00-4.95 0l-2.5 2.5a3.5 3.5 0 004.95 4.95l1.25-1.25a.75.75 0 00-1.06-1.06l-1.25 1.25a2 2 0 01-2.83 0z"></path></svg></a>Installation of COBRA.jl</h2>
<blockquote>
<p>If you are new to Julia, you may find the <a href="http://opencobra.github.io/COBRA.jl/stable/beginnerGuide.html" rel="nofollow">Beginner's Guide</a> interesting. A working installation of Julia is required.</p>
</blockquote>
<p>At the Julia prompt, add the <code>COBRA</code> package:</p>
<div class="highlight highlight-source-julia position-relative" data-snippet-clipboard-copy-content="julia&gt; Pkg.add(&quot;COBRA&quot;)
"><pre>julia<span class="pl-k">&gt;</span> Pkg<span class="pl-k">.</span><span class="pl-c1">add</span>(<span class="pl-s"><span class="pl-pds">"</span>COBRA<span class="pl-pds">"</span></span>)</pre></div>
<p>Use the <code>COBRA.jl</code> module by running:</p>
<div class="highlight highlight-source-julia position-relative" data-snippet-clipboard-copy-content="julia&gt; using COBRA
"><pre>julia<span class="pl-k">&gt;</span> <span class="pl-k">using</span> COBRA</pre></div>
<p><code>COBRA.jl</code> has been tested on <code>Julia v0.5+</code> on <em>Ubuntu Linux 14.04+</em>, <em>MacOS 10.7+</em>, and <em>Windows 7 (64-bit)</em>. All solvers supported by <a href="https://github.com/JuliaOpt/MathProgBase.jl"><code>MathProgBase.jl</code></a> are supported by <code>COBRA.jl</code>, but must be installed <strong>separately</strong>. A COBRA model saved as a <code>.mat</code> file can be read in using <a href="https://github.com/JuliaIO/MAT.jl"><code>MAT.jl</code></a>. <code>MathProgBase.jl</code> and <code>MAT.jl</code> are automatically installed during the installation of the <code>COBRA.jl</code> package.</p>

<h2><a id="user-content-limitations" class="anchor" aria-hidden="true" href="#limitations"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path fill-rule="evenodd" d="M7.775 3.275a.75.75 0 001.06 1.06l1.25-1.25a2 2 0 112.83 2.83l-2.5 2.5a2 2 0 01-2.83 0 .75.75 0 00-1.06 1.06 3.5 3.5 0 004.95 0l2.5-2.5a3.5 3.5 0 00-4.95-4.95l-1.25 1.25zm-4.69 9.64a2 2 0 010-2.83l2.5-2.5a2 2 0 012.83 0 .75.75 0 001.06-1.06 3.5 3.5 0 00-4.95 0l-2.5 2.5a3.5 3.5 0 004.95 4.95l1.25-1.25a.75.75 0 00-1.06-1.06l-1.25 1.25a2 2 0 01-2.83 0z"></path></svg></a>Limitations</h2>
<ul>
<li>At present, a COBRA model in <code>.mat</code> format is loaded using the <code>MAT.jl</code> package. A valid MATLAB license is <strong>not</strong> required.</li>
<li>The inner layer parallelization is currently done within the solver. No log files of each spawned thread are generated.</li>
<li>The current benchmarks have been performed using default optimization and compilation options of Julia and a set of solver parameters. The performance may be further improved by tuning these parameters.</li>
</ul>
