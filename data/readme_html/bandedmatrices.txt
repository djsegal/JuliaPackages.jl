<p>This package supports representing banded matrices by only the entries on the
bands.</p>
<p>One can create banded matrices of type <code>BandedMatrix</code> as follows:</p>
<div class="highlight highlight-source-julia"><pre><span class="pl-c1">BandedMatrix</span>(<span class="pl-c1">Zeros</span>(m,n), (l,u))    <span class="pl-c"><span class="pl-c">#</span> creates a banded matrix of zeros, with l sub-diagonals and u super-diagonals</span>
<span class="pl-c1">brand</span>(m,n,l,u)     <span class="pl-c"><span class="pl-c">#</span> creates a random banded matrix, with l sub-diagonals and u super-diagonals</span>
<span class="pl-c1">BandedMatrix</span>(<span class="pl-c1">Ones</span>(m,n), (l,u))     <span class="pl-c"><span class="pl-c">#</span> creates a banded matrix of ones, with l sub-diagonals and u super-diagonals</span>
<span class="pl-c1">BandedMatrix</span>(<span class="pl-c1">Eye</span>(n), (l,u))        <span class="pl-c"><span class="pl-c">#</span> creates a banded  n x n identity matrix, with l sub-diagonals and u super-diagonals</span>
<span class="pl-c1">BandedMatrix</span>(<span class="pl-k">-</span><span class="pl-c1">1</span><span class="pl-k">=&gt;</span> <span class="pl-c1">1</span><span class="pl-k">:</span><span class="pl-c1">5</span>, <span class="pl-c1">2</span><span class="pl-k">=&gt;</span><span class="pl-c1">1</span><span class="pl-k">:</span><span class="pl-c1">3</span>)     <span class="pl-c"><span class="pl-c">#</span> creates a 5 x 5 banded matrix version of diagm(-1=&gt; 1:5, 2=&gt;1:3)</span>
<span class="pl-c1">BandedMatrix</span>((<span class="pl-k">-</span><span class="pl-c1">1</span><span class="pl-k">=&gt;</span> <span class="pl-c1">1</span><span class="pl-k">:</span><span class="pl-c1">5</span>, <span class="pl-c1">2</span><span class="pl-k">=&gt;</span><span class="pl-c1">1</span><span class="pl-k">:</span><span class="pl-c1">3</span>), (n,m))     <span class="pl-c"><span class="pl-c">#</span> creates an n x m banded matrix with 1 sub-diagonals and u super-diagonals with the specified diagonals</span>
<span class="pl-c1">BandedMatrix</span>((<span class="pl-k">-</span><span class="pl-c1">1</span><span class="pl-k">=&gt;</span> <span class="pl-c1">1</span><span class="pl-k">:</span><span class="pl-c1">5</span>, <span class="pl-c1">2</span><span class="pl-k">=&gt;</span><span class="pl-c1">1</span><span class="pl-k">:</span><span class="pl-c1">3</span>), (n,m), (l,u))     <span class="pl-c"><span class="pl-c">#</span> creates an n x m banded matrix with l sub-diagonals and u super-diagonals with the specified diagonals</span></pre></div>
<p>For more examples, see <a href="http://juliamatrices.github.io/BandedMatrices.jl/latest/#Creating-banded-matrices-1" rel="nofollow">the documentation</a>.</p>
<p>Specialized algebra routines are overriden, include <code>*</code> and <code>\</code>:</p>
<div class="highlight highlight-source-julia"><pre>A <span class="pl-k">=</span> <span class="pl-c1">brand</span>(<span class="pl-c1">10000</span>,<span class="pl-c1">10000</span>,<span class="pl-c1">4</span>,<span class="pl-c1">3</span>)  <span class="pl-c"><span class="pl-c">#</span> creates a 10000 x 10000 matrix with 4 sub-diagonals</span>
                            <span class="pl-c"><span class="pl-c">#</span> and 3 super-diagonals</span>
b <span class="pl-k">=</span> <span class="pl-c1">randn</span>(<span class="pl-c1">10000</span>)
A<span class="pl-k">*</span>b  <span class="pl-c"><span class="pl-c">#</span>   Calls optimized matrix*vector routine</span>
A<span class="pl-k">*</span>A  <span class="pl-c"><span class="pl-c">#</span>   Calls optimized matrix*matrix routine</span>
A<span class="pl-k">\</span>b  <span class="pl-c"><span class="pl-c">#</span>   Calls optimized matrix\vector routine</span></pre></div>
