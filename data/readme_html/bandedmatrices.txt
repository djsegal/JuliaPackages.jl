<p dir="auto">This package supports representing banded matrices by only the entries on the
bands.</p>
<p dir="auto">One can create banded matrices of type <code>BandedMatrix</code> as follows:</p>
<div class="highlight highlight-source-julia notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="BandedMatrix(Zeros(m,n), (l,u))    # creates a banded matrix of zeros, with l sub-diagonals and u super-diagonals
