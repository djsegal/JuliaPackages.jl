<p>This package contains Julia versions of the mcmc models contained in the R package "rethinking" associated with the book <a href="https://xcelab.net/rm/statistical-rethinking/" rel="nofollow">Statistical Rethinking</a> by Richard McElreath. It is part of the <a href="https://github.com/StatisticalRethinkingJulia">StatisticalRethinkingJulia</a> Github organization of packages.</p>
<p>This package contains the <a href="https://github.com/StanJulia">Stan</a> versions.</p>
