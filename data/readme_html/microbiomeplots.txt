<p>MicrobiomePlots.jl is a companion package for <a href="https://github.com/BioJulia/Microbiome.jl">Microbiome.jl</a>
for plotting microbiome and microbial community data.
Functionality will be ported to Microbiome.jl soon.
<strong>This package is Deprecated</strong></p>
