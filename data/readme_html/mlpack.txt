<p>This repository contains the <em>Julia bindings</em> for mlpack.  These bindings are
auto-generated, and so this repository is not really maintained or monitored.</p>
<p>If you are <em>looking for the documentation</em> for the Julia bindings, try here:</p>
<ul>
<li><a href="https://www.mlpack.org/doc/mlpack-git/julia_documentation.html" rel="nofollow">Julia binding documentation</a></li>
</ul>
<p>If you are <em>having trouble</em> or <em>want to learn more</em>, try looking in the main
mlpack repository:</p>
<ul>
<li><a href="https://github.com/mlpack/mlpack/">mlpack/mlpack</a></li>
</ul>
<p>Any issues with the Julia bindings should be filed there.</p>
