<p dir="auto">OrdinalGWAS.jl is a Julia package for performing genome-wide association studies (GWAS) for ordered categorical phenotypes. It is useful when the phenotype takes ordered discrete values, e.g., disease status (undiagnosed, pre-disease, mild, moderate, severe). It runs efficiently and scales well to very large datasets. The package currently supports <a href="https://zzz.bwh.harvard.edu/plink/" rel="nofollow">PLINK</a>, <a href="https://en.wikipedia.org/wiki/Variant_Call_Format" rel="nofollow">VCF</a> (both dosage and genotype data) file formats, and <a href="https://www.well.ox.ac.uk/~gav/bgen_format/" rel="nofollow">BGEN</a> file formats. We plan to add <a href="https://www.cog-genomics.org/plink/2.0/formats#pgen" rel="nofollow">PGEN</a> support in the future.</p>
<p dir="auto">OrdinalGWAS.jl supports Julia v1.6 or later. See the <a href="https://openmendel.github.io/OrdinalGWAS.jl/latest/" rel="nofollow">documentation</a> for usage.<br>
