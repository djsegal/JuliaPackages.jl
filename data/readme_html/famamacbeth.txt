<p dir="auto">The main function to use is <code>fmb()</code>, which accepts three arguments:</p>
<ul dir="auto">
<li><code>y</code>: outcome</li>
<li><code>X</code>: matrix with explanatory variables</li>
<li><code>g</code>: the grouping indicator</li>
</ul>
<div class="snippet-clipboard-content notranslate position-relative overflow-auto" data-snippet-clipboard-copy-content="using FamaMacBeth

y = rand(100,1)
X = rand(100,3)
g = repeat(collect(1:10),inner=10)
response = fmb(X,y,g)"><pre class="notranslate"><code>using FamaMacBeth

y = rand(100,1)
X = rand(100,3)
g = repeat(collect(1:10),inner=10)
response = fmb(X,y,g)
</code></pre></div>
<p dir="auto">You can use the functions <code>coef</code>, <code>stderror</code> and others to extract the relevant statistics. The interface follows <a href="https://github.com/JuliaStats/StatsBase.jl">https://github.com/JuliaStats/StatsBase.jl</a> as much as possible. See the <a href="https://zundertj.github.io/FamaMacBeth.jl/latest" rel="nofollow">docs</a> for the full API.</p>
