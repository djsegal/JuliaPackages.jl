<div id="readme" class="md" data-path="README.md"><article class="markdown-body entry-content container-lg" itemprop="text"><p dir="auto">VimBindings.jl is an experimental Julia package which brings vim emulation directly to the Julia REPL.</p>
<p dir="auto">The package is early in development and is not yet recommended for daily use. For the brave, feel free to give it a try!</p>

<p dir="auto">The REPL begins in Insert mode, which can be used in its familiar fashion.</p>
<p dir="auto">VimBindings.jl emulates Normal mode, which is accessed by striking the <code>Escape</code> key.</p>

<p dir="auto">Insert mode is similar to the standard REPL experience. Insert mode is indicated by replacing the terminal cursor, a block █, with a pipe ❘ cursor. If you are using a very old terminal application, it may not support the pipe cursor. However, most modern terminals do.</p>

<p dir="auto">Users may see warnings about method definitions being overwritten. VimBindings.jl overwrites some methods in the standard library in order to hook into REPL functionality. These warnings can be ignored.</p>
<h3 dir="auto"><a id="user-content-note-for-tmux-users-experiencing-freezing" class="anchor" aria-hidden="true" href="#note-for-tmux-users-experiencing-freezing"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path d="m7.775 3.275 1.25-1.25a3.5 3.5 0 1 1 4.95 4.95l-2.5 2.5a3.5 3.5 0 0 1-4.95 0 .751.751 0 0 1 .018-1.042.751.751 0 0 1 1.042-.018 1.998 1.998 0 0 0 2.83 0l2.5-2.5a2.002 2.002 0 0 0-2.83-2.83l-1.25 1.25a.751.751 0 0 1-1.042-.018.751.751 0 0 1-.018-1.042Zm-4.69 9.64a1.998 1.998 0 0 0 2.83 0l1.25-1.25a.751.751 0 0 1 1.042.018.751.751 0 0 1 .018 1.042l-1.25 1.25a3.5 3.5 0 1 1-4.95-4.95l2.5-2.5a3.5 3.5 0 0 1 4.95 0 .751.751 0 0 1-.018 1.042.751.751 0 0 1-1.042.018 1.998 1.998 0 0 0-2.83 0l-2.5 2.5a1.998 1.998 0 0 0 0 2.83Z"></path></svg></a>Note for tmux users experiencing freezing</h3>
<p dir="auto">A common practice for users of tmux is to set the <code>escape-time</code> setting of tmux to <code>0</code>, often after experiencing lag while using vim from within tmux.</p>
<p dir="auto">Tmux users wishing to use julia with <code>VimBindings.jl</code> should set <code>escape-time</code> to a value above <code>0</code>, for example:</p>
<div class="highlight highlight-source-shell notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="# .tmux.conf
set -g escape-time 5"><pre><span class="pl-c"><span class="pl-c">#</span> .tmux.conf</span>
<span class="pl-c1">set</span> -g escape-time 5</pre></div>
<p dir="auto">Using <code>escape-time</code> set to <code>0</code> may cause the VimBindings.jl library to sieze completely and become unresponsive. A higher value will fix this and won't cause noticeable lag. The technical reason for this can be seen in <a href="https://github.com/caleb-allen/VimBindings.jl/issues/18#issuecomment-1381018008" data-hovercard-type="issue" data-hovercard-url="/caleb-allen/VimBindings.jl/issues/18/hovercard">this issue</a>.</p>
