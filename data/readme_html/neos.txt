<p dir="auto"><a href="https://doi.org/10.5281/zenodo.5152449" rel="nofollow"><img src="https://camo.githubusercontent.com/868114f634e3c20004bbdaa3169b8948ee267fc16eb647f35c1e994bffb57c49/68747470733a2f2f7a656e6f646f2e6f72672f62616467652f444f492f31302e353238312f7a656e6f646f2e353135323434392e737667" alt="DOI" data-canonical-src="https://zenodo.org/badge/DOI/10.5281/zenodo.5152449.svg" style="max-width: 100%;"></a></p>
<p dir="auto"><code>NEOs.jl</code> is a Near-Earth Object orbital propagator and
fitter in Julia. <code>NEOs.jl</code> exploits jet transport techniques via
<a href="https://github.com/PerezHzTaylorIntegration.jl">TaylorIntegration.jl</a>.</p>

<p dir="auto">The <code>apophis.jl</code> file in the <code>pha</code> directory contains an example script. This
script may be called as:</p>
<p dir="auto"><code>julia --project=@. apophis.jl</code></p>
<p dir="auto"><code>NEOs.propagate</code> also supports multi-threading:</p>
<p dir="auto"><code>JULIA_NUM_THREADS=&lt;number-of-threads&gt; julia --project=@. apophis.jl</code></p>
