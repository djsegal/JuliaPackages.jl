<p dir="auto"><a href="https://mikeingold.github.io/UnitfulCoordinateSystems.jl/dev/" rel="nofollow"><img src="https://camo.githubusercontent.com/56f8252ba8e9d3f0b810769543f77823d2fe031ce560d4c2d69fb1fcad800383/68747470733a2f2f696d672e736869656c64732e696f2f62616467652f646f63732d6c61746573742d626c75652e737667" alt="" data-canonical-src="https://img.shields.io/badge/docs-latest-blue.svg" style="max-width: 100%;"></a></p>
<p dir="auto">This package is a simple and efficient implementation of several common coordinate
systems with <code>Unitful</code> dimensions. These systems, as defined by the ISO 80000-2:2019
standards, include:</p>
<ul dir="auto">
<li>Two-Dimensional <code>AbstractCoordinate{2}</code>:
<ul dir="auto">
<li>Rectangular <code>CoordinateRectangular</code> (x, y)</li>
<li>Polar <code>CoordinatePolar</code> (r, <math-renderer class="js-inline-math" style="display: inline" data-static-url="https://github.githubassets.com/static" data-run-id="85700bec54ca5b99f5bcf10352133816">$\varphi$</math-renderer>)</li>
</ul>
</li>
<li>Three-Dimensional <code>AbstractCoordinate{3}</code>:
<ul dir="auto">
<li>Cartesian <code>CoordinateCartesian</code> (x, y, z)</li>
<li>Cylindrical <code>CoordinateCylindrical</code> (<math-renderer class="js-inline-math" style="display: inline" data-static-url="https://github.githubassets.com/static" data-run-id="85700bec54ca5b99f5bcf10352133816">$\rho$</math-renderer>, <math-renderer class="js-inline-math" style="display: inline" data-static-url="https://github.githubassets.com/static" data-run-id="85700bec54ca5b99f5bcf10352133816">$\varphi$</math-renderer>, z)</li>
<li>Spherical <code>CoordinateSpherical</code> (r, <math-renderer class="js-inline-math" style="display: inline" data-static-url="https://github.githubassets.com/static" data-run-id="85700bec54ca5b99f5bcf10352133816">$\theta$</math-renderer>, <math-renderer class="js-inline-math" style="display: inline" data-static-url="https://github.githubassets.com/static" data-run-id="85700bec54ca5b99f5bcf10352133816">$\varphi$</math-renderer>)</li>
</ul>
</li>
</ul>
