<p dir="auto"><code>TextGraphs.jl</code> offers Graphs representations of Text, along with natural language proccessing (NLP) functionalities. Check the <a href="https://psyarxiv.com/nfze4/" rel="nofollow">white paper</a> including vignettes with examples.</p>
<p dir="auto">This package is inspired by <a href="https://repositorio.ufrn.br/jspui/handle/123456789/23273" rel="nofollow">SpeechGraphs</a>. <code>TextGraphs.jl</code> new features include pre-processing (e.g.lemmas), properties (e.g. centrality) and latent space embeddings (adding latent semantic information to graphs).</p>
<p dir="auto">Julia uses multiple dispatching, focusing on modular functions and high-performance computing.</p>

<h3 dir="auto"><a id="user-content-graph-types" class="anchor" aria-hidden="true" href="#graph-types"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path d="m7.775 3.275 1.25-1.25a3.5 3.5 0 1 1 4.95 4.95l-2.5 2.5a3.5 3.5 0 0 1-4.95 0 .751.751 0 0 1 .018-1.042.751.751 0 0 1 1.042-.018 1.998 1.998 0 0 0 2.83 0l2.5-2.5a2.002 2.002 0 0 0-2.83-2.83l-1.25 1.25a.751.751 0 0 1-1.042-.018.751.751 0 0 1-.018-1.042Zm-4.69 9.64a1.998 1.998 0 0 0 2.83 0l1.25-1.25a.751.751 0 0 1 1.042.018.751.751 0 0 1 .018 1.042l-1.25 1.25a3.5 3.5 0 1 1-4.95-4.95l2.5-2.5a3.5 3.5 0 0 1 4.95 0 .751.751 0 0 1-.018 1.042.751.751 0 0 1-1.042.018 1.998 1.998 0 0 0-2.83 0l-2.5 2.5a1.998 1.998 0 0 0 0 2.83Z"></path></svg></a>Graph types</h3>
<p dir="auto">You can build the following graphs from text (<code>AbstractString</code>):</p>
<p dir="auto">Raw</p>
<ul dir="auto">
<li>Naive (<code>naive_graph</code>) uses the original sequence of words.</li>
<li>Phrases Graph(<code>phrases_graph</code>): Uses the original sequence of phrases.</li>
</ul>
<p dir="auto">POS, Stems and Lemmas</p>
<ul dir="auto">
<li>Stem (<code>stem_graph</code>) uses stemmed words.</li>
<li>Lemma (<code>lemma_graph</code>): Uses lemmatized words.</li>
<li>Part of Speech Graph (POS, <code>pos_graph</code>) uses syntactical functions.</li>
</ul>
<p dir="auto">Latent space embeddings</p>
<ul dir="auto">
<li>Latent space embedding (LSE, <code>latent_space_graph</code>) graphs.</li>
<li>Latent space embeddings to target (<code>latent_space_graph</code>)</li>
</ul>
<h3 dir="auto"><a id="user-content-properties" class="anchor" aria-hidden="true" href="#properties"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path d="m7.775 3.275 1.25-1.25a3.5 3.5 0 1 1 4.95 4.95l-2.5 2.5a3.5 3.5 0 0 1-4.95 0 .751.751 0 0 1 .018-1.042.751.751 0 0 1 1.042-.018 1.998 1.998 0 0 0 2.83 0l2.5-2.5a2.002 2.002 0 0 0-2.83-2.83l-1.25 1.25a.751.751 0 0 1-1.042-.018.751.751 0 0 1-.018-1.042Zm-4.69 9.64a1.998 1.998 0 0 0 2.83 0l1.25-1.25a.751.751 0 0 1 1.042.018.751.751 0 0 1 .018 1.042l-1.25 1.25a3.5 3.5 0 1 1-4.95-4.95l2.5-2.5a3.5 3.5 0 0 1 4.95 0 .751.751 0 0 1-.018 1.042.751.751 0 0 1-1.042.018 1.998 1.998 0 0 0-2.83 0l-2.5 2.5a1.998 1.998 0 0 0 0 2.83Z"></path></svg></a>Properties</h3>
<p dir="auto">You can obtain several properties of the graphs:</p>
<p dir="auto">Direct measures<br>
<code>graph_props</code> returns values of density, # of self loops, # of SCCs, size of largest SCC, and mean centrality (betweeness, closeness and eigenvector methods).</p>
<p dir="auto">Erdős–Rényi ratios<br>
<code>rand_erdos_props</code> returns values as compared to random Erdõs-Rényi graph with identical number of vertices and edges through z-score or ratio to average.</p>
