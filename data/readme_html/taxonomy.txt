<p dir="auto">Taxonomy.jl is a julia package to handle the NCBI Taxonomy database.
The main features are:</p>
<ul dir="auto">
<li>Get various information on a given taxon (name, rank, parent-child relationships, etc.)</li>
<li>Convert a name to Taxids</li>
<li>Traverse taxonomic subtrees from a given taxon</li>
<li>Compute the lowest common ancestor (LCA) of given taxa</li>
<li>Evaluate ancestor-descendant relationships between two taxa</li>
<li>Filter taxa by a rank range</li>
<li>Construct taxonomic lineage of the given taxon</li>
<li>Reformat lineage according to canonical ranks</li>
<li>Construct a <code>DataFrame</code> from lineages</li>
</ul>
<p dir="auto">Now, this package only supports <code>scientific name</code>.</p>
