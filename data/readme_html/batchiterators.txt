<p dir="auto">A very small package providing the constructor <code>BatchIterator(X; batchsize=…, limit=…)</code> and the function <code>centered_batch_iterator(X; kwargs…)</code>, which allow iteration over blocks of columns of <code>X</code>, for any object <code>X</code> supporting 2d indexing and for which the function <code>size</code> is defined.
The function <code>choose_batchsize</code> helps finding a good batch size while controlling memory usage.</p>
<p dir="auto">The package was originally designed to iterate over samples of an out-of-core dataset.</p>
