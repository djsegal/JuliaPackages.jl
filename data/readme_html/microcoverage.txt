<p dir="auto">MicroCoverage.jl is a code coverage tool for Julia, implemented in pure Julia.</p>

<p dir="auto"><strong>Step 1:</strong> Create a file named <code>foo.jl</code> with the following contents:</p>
<div class="highlight highlight-source-julia notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="function foo(x)
    if x == 1 || x == 100
        return &quot;hello&quot;
    else
        return &quot;goodbye&quot;
    end
end"><pre><span class="pl-k">function</span> <span class="pl-en">foo</span>(x)
    <span class="pl-k">if</span> x <span class="pl-k">==</span> <span class="pl-c1">1</span> <span class="pl-k">||</span> x <span class="pl-k">==</span> <span class="pl-c1">100</span>
        <span class="pl-k">return</span> <span class="pl-s"><span class="pl-pds">"</span>hello<span class="pl-pds">"</span></span>
    <span class="pl-k">else</span>
        <span class="pl-k">return</span> <span class="pl-s"><span class="pl-pds">"</span>goodbye<span class="pl-pds">"</span></span>
    <span class="pl-k">end</span>
<span class="pl-k">end</span></pre></div>
<p dir="auto"><strong>Step 2:</strong> Create a file named <code>test_foo.jl</code> with the following contents:</p>
<div class="highlight highlight-source-julia notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="using Test

<p dir="auto"><strong>Step 3:</strong> Open <code>julia</code> and run the following commands:</p>
<div class="highlight highlight-source-julia notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="julia&gt; using MicroCoverage
