<p dir="auto">These kernels are designed for use in smoothing algorithms such as kernel regression and kernel density estimation. They are implemented in both unnormalized and normalized form.</p>
<h1 dir="auto"><a id="user-content-mathematical-form-of-implemented-kernels" class="anchor" aria-hidden="true" href="#mathematical-form-of-implemented-kernels"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path d="m7.775 3.275 1.25-1.25a3.5 3.5 0 1 1 4.95 4.95l-2.5 2.5a3.5 3.5 0 0 1-4.95 0 .751.751 0 0 1 .018-1.042.751.751 0 0 1 1.042-.018 1.998 1.998 0 0 0 2.83 0l2.5-2.5a2.002 2.002 0 0 0-2.83-2.83l-1.25 1.25a.751.751 0 0 1-1.042-.018.751.751 0 0 1-.018-1.042Zm-4.69 9.64a1.998 1.998 0 0 0 2.83 0l1.25-1.25a.751.751 0 0 1 1.042.018.751.751 0 0 1 .018 1.042l-1.25 1.25a3.5 3.5 0 1 1-4.95-4.95l2.5-2.5a3.5 3.5 0 0 1 4.95 0 .751.751 0 0 1-.018 1.042.751.751 0 0 1-1.042.018 1.998 1.998 0 0 0-2.83 0l-2.5 2.5a1.998 1.998 0 0 0 0 2.83Z"></path></svg></a>Mathematical Form of Implemented Kernels</h1>
<p dir="auto">Currently, the kernels implemented are those found in the <a href="http://en.wikipedia.org/wiki/Kernel_(statistics)" rel="nofollow">Wikipedia article on kernels in statistics</a>.</p>
<p dir="auto">In normalized form, the kernels are:</p>
<ul dir="auto">
<li>Uniform: <math-renderer class="js-inline-math" style="display: inline" data-static-url="https://github.githubassets.com/static" data-run-id="b6f4d59e3e0a2028e41f72f8a543d051">$K(u) = \frac{1}{2} I(|u| \leq 1)$</math-renderer>
</li>
<li>Triangular: <math-renderer class="js-inline-math" style="display: inline" data-static-url="https://github.githubassets.com/static" data-run-id="b6f4d59e3e0a2028e41f72f8a543d051">$K(u) = (1 - |u|) I(|u| \leq 1)$</math-renderer>
</li>
<li>Epanechnikov: <math-renderer class="js-inline-math" style="display: inline" data-static-url="https://github.githubassets.com/static" data-run-id="b6f4d59e3e0a2028e41f72f8a543d051">$K(u) = \frac{3}{4} (1 - |u|^2) I(|u| \leq 1)$</math-renderer>
</li>
<li>Biweight (Quartic): <math-renderer class="js-inline-math" style="display: inline" data-static-url="https://github.githubassets.com/static" data-run-id="b6f4d59e3e0a2028e41f72f8a543d051">$K(u) = \frac{15}{16} (1 - |u|^2)^2 I(|u| \leq 1)$</math-renderer>
</li>
<li>Triweight: <math-renderer class="js-inline-math" style="display: inline" data-static-url="https://github.githubassets.com/static" data-run-id="b6f4d59e3e0a2028e41f72f8a543d051">$K(u) = \frac{35}{32} (1 - |u|^2)^3 I(|u| \leq 1)$</math-renderer>
</li>
<li>Tricube: <math-renderer class="js-inline-math" style="display: inline" data-static-url="https://github.githubassets.com/static" data-run-id="b6f4d59e3e0a2028e41f72f8a543d051">$K(u) = \frac{70}{81} (1 - |u|^3)^3 I(|u| \leq 1)$</math-renderer>
</li>
<li>Gaussian: <math-renderer class="js-inline-math" style="display: inline" data-static-url="https://github.githubassets.com/static" data-run-id="b6f4d59e3e0a2028e41f72f8a543d051">$K(u) = \frac{1}{\sqrt{2 \pi}} e^{-\frac{1}{2}u^2}$</math-renderer>
</li>
<li>Cosine: <math-renderer class="js-inline-math" style="display: inline" data-static-url="https://github.githubassets.com/static" data-run-id="b6f4d59e3e0a2028e41f72f8a543d051">$K(u) = \frac{\pi}{4} \cos(\frac{\pi}{2} u) I(|u| \leq 1)$</math-renderer>
</li>
<li>Logistic: <math-renderer class="js-inline-math" style="display: inline" data-static-url="https://github.githubassets.com/static" data-run-id="b6f4d59e3e0a2028e41f72f8a543d051">$K(u) = \frac{1}{e^u + 2 + e^{-u}}$</math-renderer>
</li>
</ul>
