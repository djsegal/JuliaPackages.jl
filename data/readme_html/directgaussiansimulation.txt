<p>This package provides an implementation of direct Gaussian simulation (a.k.a. LU simulation)
as described in <a href="https://link.springer.com/article/10.1007/BF00897191" rel="nofollow">Alabert 1987</a>. In this
method, the full covariance matrix is built to include all locations of the simulation domain,
and samples from the multivariate Gaussian are drawn via LU factorization.</p>
<p>The method, which is widely implemented in many packages for Gaussian processes (e.g.
<a href="https://github.com/STOR-i/GaussianProcesses.jl">GaussianProcesses.jl</a>,
<a href="https://github.com/PieterjanRobbe/GaussianRandomFields.jl">GaussianRandomFields.jl</a>),
is appropriate for relatively small simulation domains (e.g. 100x100 grids) where it is feasible
to factorize the full covariance. For larger domains (e.g. 3D grids), other methods are available
such as sequential Gaussian simulation, spectral methods, and FFT moving averages.</p>
