<p dir="auto">This package provides for loading and parsing of <a href="https://docs.flowjo.com/flowjo/workspaces-and-samples/ws-savinganalysis" rel="nofollow">FlowJo Workspace files</a> in pure Julia. The gating strategy is parsed as a <code>DiMetaGraph</code> and group membership as <code>Dict</code>. Together with <a href="https://github.com/LCSB-BioCore/GigaSOM.jl">GigaSOM.jl</a> this metadata can be used to generate event-level labels for <code>FCS</code> datasets.</p>

<blockquote>
<p dir="auto"><g-emoji class="g-emoji" alias="warning" fallback-src="https://github.githubassets.com/images/icons/emoji/unicode/26a0.png">⚠️</g-emoji> <strong>FCS files under a workspace must have unique names</strong>. This limitation will be removed in future versions</p>

<p dir="auto">The <code>load</code> method returns a tuple of three <code>DataFrames</code> and one <code>Dict</code>:</p>
<ul dir="auto">
<li><code>data</code> transformed fluorescence values for each event</li>
<li><code>labels</code> boolean telling us which events have been gated</li>
<li><code>groups</code> booleans telling us which group each event belongs to</li>
<li><code>gating</code> Dictionary of graph representations of each gating strategy</li>
</ul>
