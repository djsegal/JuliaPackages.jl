<p dir="auto"><strong>DISCLAIMER</strong>: Even though we analyzed the theoretical framework which we are basing our privacy guarantees on, and extensively tested the implementation, we cannot entirely exclude the possibility of bugs or of privacy risks in a program which our typechecker claims to be safe. Use at your own risk. Feedback and questions are very welcome.</p>
<p dir="auto">The goal of this project is to create a type checker which can automatically analyze <a href="https://julialang.org/" rel="nofollow">Julia</a> programs for <a href="https://en.wikipedia.org/wiki/Differential_privacy" rel="nofollow">(ε, δ)-differential privacy</a> guarantees.</p>
<p dir="auto">This software is capable of inferring the ε and δ parameters of (ε, δ)-differentially private programs written in Julia, as long as they use only <a href="https://diffmu.github.io/DiffPrivacyInference.jl/dev/full_reference/syntax/" rel="nofollow">supported syntax</a> and our builtin <a href="https://diffmu.github.io/DiffPrivacyInference.jl/dev/tutorial/02_privacy_functions/" rel="nofollow">primitive privacy mechanisms</a>. Its main purpose is enabling the implementaion of novel private mechanisms, using primitives whose privacy guarantees are known from literature, without having to make manual proofs about their properties.</p>
<p dir="auto">We provide a verifiable implementation of <a href="https://arxiv.org/abs/1607.00133" rel="nofollow">Differentially Private Stochastic Gradient Descent</a> using the <a href="https://github.com/FluxML/Flux.jl">Flux.jl</a> machine learning framework. Head to the <a href="https://diffmu.github.io/DiffPrivacyInference.jl/dev/tutorial/03_flux_dp/" rel="nofollow">tutorial</a> for a walkthrough, and check out the <a href="https://github.com/DiffMu/DiffPrivacyInference.jl/tree/main/example/flux_dp">code</a>!</p>
<p dir="auto">Our type inference algorithm for Julia code is based on the <a href="https://arxiv.org/abs/1909.02481" rel="nofollow">Duet type system</a> and the corresponding <a href="https://github.com/uvm-plaid/duet">Haskell implementation</a>. The main part of our analysis is written in Haskell as well, head to <a href="https://github.com/DiffMu/DiffPrivacyInferenceHs">our Haskell repo</a>.</p>
<p dir="auto">If you have any questions, feel free to get in touch :)</p>
