<p>Compressed Sparse Row (CSR) sparse matrix implementations.</p>
<p>It includes <code>SparseMatrixCSR</code> and <code>SymSparseMatrixCSR</code> wrappers.</p>
<h2><a id="user-content-basic-usage" class="anchor" aria-hidden="true" href="#basic-usage"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path fill-rule="evenodd" d="M4 9h1v1H4c-1.5 0-3-1.69-3-3.5S2.55 3 4 3h4c1.45 0 3 1.69 3 3.5 0 1.41-.91 2.72-2 3.25V8.59c.58-.45 1-1.27 1-2.09C10 5.22 8.98 4 8 4H4c-.98 0-2 1.22-2 2.5S3 9 4 9zm9-3h-1v1h1c1 0 2 1.22 2 2.5S13.98 12 13 12H9c-.98 0-2-1.22-2-2.5 0-.83.42-1.64 1-2.09V6.25c-1.09.53-2 1.84-2 3.25C6 11.31 7.55 13 9 13h4c1.45 0 3-1.69 3-3.5S14.5 6 13 6z"></path></svg></a>Basic Usage</h2>
<p>CSR sparse matrix:</p>
<div class="highlight highlight-source-julia"><pre>julia<span class="pl-k">&gt;</span> <span class="pl-c1">push_coo!</span>(SparseMatrixCSR,I,J,V,i1,j1,v1)
julia<span class="pl-k">&gt;</span> <span class="pl-k">...</span>
julia<span class="pl-k">&gt;</span> <span class="pl-c1">push_coo!</span>(SparseMatrixCSR,I,J,V,in,jn,vn)
julia<span class="pl-k">&gt;</span> <span class="pl-c1">finalize_coo!</span>(SparseMatrixCSR,I,J,V,m,n)
julia<span class="pl-k">&gt;</span> CSR <span class="pl-k">=</span> <span class="pl-c1">sparsecsr</span>(I,J,V,m,n)</pre></div>
<p>Symmetric CSR sparse matrix:</p>
<div class="highlight highlight-source-julia"><pre>julia<span class="pl-k">&gt;</span> <span class="pl-c1">push_coo!</span>(SymSparseMatrixCSR,I,J,V,i1,j1,v1)
julia<span class="pl-k">&gt;</span> <span class="pl-k">...</span>
julia<span class="pl-k">&gt;</span> <span class="pl-c1">push_coo!</span>(SymSparseMatrixCSR,I,J,V,in,jn,vn)
julia<span class="pl-k">&gt;</span> <span class="pl-c1">finalize_coo!</span>(SymSparseMatrixCSR,I,J,V,m,n)
julia<span class="pl-k">&gt;</span> CSR <span class="pl-k">=</span> <span class="pl-c1">symsparsecsr</span>(I,J,V,m,n)</pre></div>

<p><strong>SparseMatricesCSR</strong> itself is installed when you add and use it into another project.</p>
<p>To include into your project from Julia REPL, use the following commands:</p>
<pre><code>pkg&gt; add SparseMatricesCSR
julia&gt; using SparseMatricesCSR
</code></pre>
