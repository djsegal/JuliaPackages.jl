<p dir="auto">MCMC sampler for inference for diffusion processes with the use of Guided
Proposals using the package <a href="https://github.com/mschauer/Bridge.jl">Bridge.jl</a>.
Currently under development.</p>

<p dir="auto">The main function introduced by this package is</p>
<div class="highlight highlight-source-julia notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="mcmc(setup)"><pre><span class="pl-c1">mcmc</span>(setup)</pre></div>
<p dir="auto">It finds the posterior distribution of the unknown parameters given discrete
time observations of the underlying process. Please see the
<a href="https://mmider.github.io/BridgeSDEInference.jl/dev" rel="nofollow">documentation (in development)</a>.</p>
