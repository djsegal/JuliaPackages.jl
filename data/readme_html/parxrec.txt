<ul dir="auto">
<li>
<p dir="auto">Supports PAR versions 3, 4, 4.1, 4.2.</p>
</li>
<li>
<p dir="auto">REC data is sorted and loaded into a single array. The dimensions are:</p>
<ul dir="auto">
<li>slices</li>
<li>echoes</li>
<li>dynamics</li>
<li>cardiac phases</li>
<li>diffusion b values</li>
<li>gradient orientations</li>
<li>label types</li>
<li>image types</li>
<li>sequences</li>
</ul>
<p dir="auto">Singleton dimensions (except for slices) are removed.</p>
</li>
</ul>
