<p dir="auto"><a href="https://travis-ci.org/randy3k/RCalling.jl" rel="nofollow"><img src="https://camo.githubusercontent.com/db5d41c09bcdc393b4464704ecc3edd9f3965e59c40fcbc5c15fd4fd701cf5a8/68747470733a2f2f7472617669732d63692e6f72672f72616e6479336b2f5243616c6c696e672e6a6c2e7376673f6272616e63683d6d6173746572" alt="Build Status" data-canonical-src="https://travis-ci.org/randy3k/RCalling.jl.svg?branch=master" style="max-width: 100%;"></a></p>
<p dir="auto">It is based one the work of <a href="https://github.com/lgautier/Rif.jl">Rif</a> and <a href="https://github.com/armgong/RJulia">RJulia</a>. Since the development of Rif has been slowed down and I wanted to learn more about the Julia and R, I started this project.</p>
<p dir="auto">This package was named <code>RCall.jl</code>. However a more promising package <a href="https://github.com/JuliaStats/RCall.jl">RCall.jl</a> is now alive at JuliaStats, I have renamed this to <code>RCalling.jl</code>.</p>
