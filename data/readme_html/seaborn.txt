<p dir="auto">A Julia wrapper around the <a href="http://seaborn.pydata.org/index.html" rel="nofollow">Seaborn data visualization library</a>:</p>
<blockquote>
<p dir="auto">Seaborn is a Python data visualization library based on <a href="https://matplotlib.org/" rel="nofollow">matplotlib</a>. It provides a high-level interface for drawing attractive and informative statistical graphics.</p>
</blockquote>
<blockquote>
<p dir="auto">For a brief introduction to the ideas behind the library, you can read the <a href="http://seaborn.pydata.org/introduction.html" rel="nofollow">introductory notes</a> or the <a href="https://joss.theoj.org/papers/10.21105/joss.03021" rel="nofollow">paper</a>. Visit the <a href="http://seaborn.pydata.org/installing.html" rel="nofollow">installation page</a> to see how you can download the package and get started with it. You can browse the <a href="http://seaborn.pydata.org/examples/index.html" rel="nofollow">example gallery</a> to see some of the things that you can do with seaborn, and then check out the <a href="http://seaborn.pydata.org/tutorial.html" rel="nofollow">tutorial</a> or <a href="http://seaborn.pydata.org/api.html" rel="nofollow">API reference</a> to find out how.</p>
</blockquote>
<blockquote>
<p dir="auto">To see the code or report a bug, please visit the <a href="https://github.com/mwaskom/seaborn">GitHub repository</a>. General support questions are most at home on <a href="https://stackoverflow.com/questions/tagged/seaborn/" rel="nofollow">stackoverflow</a> or <a href="https://discourse.matplotlib.org/c/3rdparty/seaborn/21" rel="nofollow">discourse</a>, which have dedicated channels for seaborn.</p>
</blockquote>

<p dir="auto">This Julia wrapper exposes all of the functions in the <a href="http://seaborn.pydata.org/api.html" rel="nofollow">Seaborn API</a>.</p>
<p dir="auto">You need to have a Python distribution installed with the <code>seaborn</code> Python package. If you do not already have it, run <code>pip install seaborn</code> form the command line.</p>
