<p>A Julia wrapper around the <a href="http://seaborn.pydata.org/index.html" rel="nofollow">Seaborn data visualization library</a>:</p>
<blockquote>
<p>Seaborn is a Python visualization library based on matplotlib. It provides a high-level interface for drawing attractive statistical graphics.</p>
</blockquote>
<blockquote>
<p>For a brief introduction to the ideas behind the package, you can read the <a href="http://seaborn.pydata.org/introduction.html#introduction" rel="nofollow">introductory notes</a>. More practical information is on the <a href="http://seaborn.pydata.org/installing.html" rel="nofollow">installation page</a>. You may also want to browse the <a href="http://seaborn.pydata.org/examples/index.html" rel="nofollow">example gallery</a> to get a sense for what you can do with seaborn and then check out the <a href="http://seaborn.pydata.org/tutorial.html" rel="nofollow">tutorial</a> and <a href="http://seaborn.pydata.org/api.html" rel="nofollow">API reference</a> to find out how.</p>
</blockquote>

<p>The API closely follows the Python original. Consult its <a href="http://seaborn.pydata.org/api.html" rel="nofollow">API reference</a>.</p>
<p>You need to have a Python distribution installed with the <code>seaborn</code> Python package. If you do not already have it, run <code>pip install seaborn</code> form the command line.</p>
