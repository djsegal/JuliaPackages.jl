<p>Deconvolution problems arise when the probability density function of a quantity is estimated even though this quantity cannot be measured directly. In this scenario, the density has to be inferred from related quantities which are measured instead.</p>
<p>Please read <a href="https://mirkobunse.github.io/CherenkovDeconvolution.jl/dev" rel="nofollow">the documentation</a> and <a href="https://sfb876.tu-dortmund.de/deconvolution" rel="nofollow">visit our website</a>.
You can install this package with the Julia package manager (Julia-0.7 and above):</p>
