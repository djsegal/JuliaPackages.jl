<p dir="auto"><a href="https://doi.org/10.5281/zenodo.5152451" rel="nofollow"><img src="https://camo.githubusercontent.com/2f5857bb47f89231fc738f277aadf201f9bc802dadc8dded7b7735d4b02da284/68747470733a2f2f7a656e6f646f2e6f72672f62616467652f444f492f31302e353238312f7a656e6f646f2e353135323435312e737667" alt="DOI" data-canonical-src="https://zenodo.org/badge/DOI/10.5281/zenodo.5152451.svg" style="max-width: 100%;"></a></p>
<p dir="auto"><code>PlanetaryEphemeris.jl</code> is a Taylor integrator for the JPL DE430 planetary
ephemeris dynamical model (Folkner et al., 2014), based on
<a href="https://github.com/PerezHzTaylorIntegration.jl">TaylorIntegration.jl</a>.</p>

<p dir="auto"><code>PlanetaryEphemeris.propagate_dense</code> is a high-level function which performs the
numerical integration. The file <code>integrate_ephemeris.jl</code> in the <code>scripts</code> directory
contains an example script. This script may be called as:</p>
<p dir="auto"><code>julia --project integrate_ephemeris.jl --help</code></p>
<p dir="auto"><code>PlanetaryEphemeris.propagate</code> also supports multi-threading:</p>
<p dir="auto"><code>julia -t &lt;number-of-threads&gt; --project integrate_ephemeris.jl --help</code></p>
