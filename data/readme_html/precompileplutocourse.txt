<p dir="auto">If you have a course with Pluto notebooks and you would like to ship it with custom sysimages, use this package as shown in the <a href="Demo/src/Demo.jl">demo</a> or in <a href="https://github.com/jbrea/MLCourse">MLCourse</a>.
The students, who study the notebooks in the <code>Demo</code> project, will be able to create a custom sysimage with</p>
<div class="highlight highlight-source-julia notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="using Demo

Demo<span class="pl-k">.</span><span class="pl-c1">create_sysimage</span>()</pre></div>
<p dir="auto">and use the custom sysimage with</p>

<li>Add this package to your project</li>

<li>
<p dir="auto">Create and customize a <code>warmup.jl</code> file. See <a href="Demo/precompile/warmup.jl">Demo/precompile/warmup.jl</a> for a minimal example or <a href="https://github.com/jbrea/MLCourse/blob/main/precompile/warmup.jl">this customized warmup file</a>. The <code>warmup.jl</code> file will be passed to the <code>precompile_execution_file</code> argument of the <a href="https://julialang.github.io/PackageCompiler.jl/stable/sysimages.html" rel="nofollow">PackageCompiler</a>.</p>
</li>
<li>
<p dir="auto">Configure <code>PrecompilePlutoCourse</code> in the <code>__init__()</code> function of your package as e.g.
in <a href="Demo/src/Demo.jl">Demo/src/Demo.jl</a> or <a href="https://github.com/jbrea/MLCourse/blob/main/src/MLCourse.jl#L10">here</a>.</p>
