<p dir="auto">This package extends the functionality of <code>Base.Fix1</code> and <code>Base.Fix2</code> to any function and provide a syntax to build <code>DelayEval</code> objects via <code>getindex</code> for callable objects, similar to <code>DataFrames</code> indexing. To achieve this, this package relies on type piracy, defining <code>Base.getindex</code> for <code>Base.Callable</code>.</p>

<p dir="auto">Simply index a function using a <code>!</code> as a placeholder where arguments will be supplied on call. Examples:</p>
