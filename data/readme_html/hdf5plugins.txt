<p dir="auto">This package formerly implemented compression, decompression plugins, and other filter plugins for <a href="https://github.com/JuliaIO/HDF5.jl">HDF5.jl</a>. bzip2, lz4, and zstd compression codecs were supported using <a href="https://github.com/JuliaIO/CodecBzip2.jl">CodecBzip2.jl</a>, <a href="https://github.com/JuliaIO/CodecLz4.jl">CodecLZ4.jl</a>, and <a href="https://github.com/JuliaIO/CodecZstd.jl">CodecZstd.jl</a>.</p>
<p dir="auto">The plugins themselves, which implement glue code between HDF5 and the individual codecs, are implemented in Julia after having been ported from C.</p>
<p dir="auto">Currently, the main contents of this package are now part of HDF5.jl as of version 0.16.0. This package now serves mainly as a convenience package to load all known HDF5 compression plugins implemented in Julia.</p>
