<p>A Julia <a href="http://www.amqp.org/" rel="nofollow">AMQP (Advanced Message Queuing Protocol)</a> Client.</p>
<p>Supports protocol version 0.9.1 and <a href="https://www.rabbitmq.com/" rel="nofollow">RabbitMQ</a> extensions.</p>
<p>This library has been tested with RabbitMQ, though it should also work with other AMQP 0.9.1 compliant systems.</p>
