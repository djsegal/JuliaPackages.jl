<p dir="auto">Bloqade is a package developed for quantum computation and quantum simulation based on the neutral-atom architecture with the ability to submit tasks to <a href="https://www.quera.com/aquila" rel="nofollow">QuEra's <em>Aquila</em> quantum processor</a>. Please refer to the <a href="https://queracomputing.github.io/Bloqade.jl/dev/" rel="nofollow">documentation</a> page to learn more about Bloqade.</p>
<p dir="auto"><strong>Bloqade is currently under public release beta. High-level APIs are stable, but please expect some further exploration and rough edges.</strong></p>
