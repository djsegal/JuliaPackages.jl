<ol dir="auto">
<li>Clone this repository using the command <code>git clone -b sim https://github.com/adubredu/KinodynamcFabrics.jl</code></li>
<li>Navigate to the parent directory of this repo and type  <code>julia</code> in your terminal to launch the Julia REPL.</li>
<li>Press <code>]</code> on your keyboard to enter the package manager</li>
<li>Enter command <code>activate .</code> then enter command <code>instantiate</code> to download all package dependencies</li>
<li>Once installation is complete, press the <code>Backspace</code> key on your keyboard to return to the REPL</li>
</ol>
