<p>Julia interface to <a href="https://plot.ly/javascript" rel="nofollow">plotly.js</a> visualization library.</p>
<p>This package constructs plotly graphics using all local resources. To interact or save graphics to the Plotly cloud, use the  <a href="https://github.com/plotly/Plotly.jl"><code>Plotly</code></a> package.</p>
<p>Check out the <a href="http://spencerlyon.com/PlotlyJS.jl/" rel="nofollow">docs</a>!</p>
<h2><a id="user-content-installation" class="anchor" aria-hidden="true" href="#installation"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path fill-rule="evenodd" d="M7.775 3.275a.75.75 0 001.06 1.06l1.25-1.25a2 2 0 112.83 2.83l-2.5 2.5a2 2 0 01-2.83 0 .75.75 0 00-1.06 1.06 3.5 3.5 0 004.95 0l2.5-2.5a3.5 3.5 0 00-4.95-4.95l-1.25 1.25zm-4.69 9.64a2 2 0 010-2.83l2.5-2.5a2 2 0 012.83 0 .75.75 0 001.06-1.06 3.5 3.5 0 00-4.95 0l-2.5 2.5a3.5 3.5 0 004.95 4.95l1.25-1.25a.75.75 0 00-1.06-1.06l-1.25 1.25a2 2 0 01-2.83 0z"></path></svg></a>Installation</h2>
<p>If you intend to use the <a href="http://spencerlyon.com/PlotlyJS.jl/syncplots/#electronplot" rel="nofollow">Electron display</a> or any of its features (recommended) you will need to enter the following at the Julia REPL:</p>
<div class="highlight highlight-source-julia"><pre><span class="pl-k">using</span> Blink
Blink<span class="pl-k">.</span>AtomShell<span class="pl-k">.</span><span class="pl-c1">install</span>()</pre></div>
<p>Note that this is a one time process.</p>
<p>Also, if you have issues building this package because of installation of the MbedTLS  package please see <a href="https://github.com/sglyon/PlotlyJS.jl/issues/83">this issue</a>.</p>
<h3><a id="user-content-jupyterlab" class="anchor" aria-hidden="true" href="#jupyterlab"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path fill-rule="evenodd" d="M7.775 3.275a.75.75 0 001.06 1.06l1.25-1.25a2 2 0 112.83 2.83l-2.5 2.5a2 2 0 01-2.83 0 .75.75 0 00-1.06 1.06 3.5 3.5 0 004.95 0l2.5-2.5a3.5 3.5 0 00-4.95-4.95l-1.25 1.25zm-4.69 9.64a2 2 0 010-2.83l2.5-2.5a2 2 0 012.83 0 .75.75 0 001.06-1.06 3.5 3.5 0 00-4.95 0l-2.5 2.5a3.5 3.5 0 004.95 4.95l1.25-1.25a.75.75 0 00-1.06-1.06l-1.25 1.25a2 2 0 01-2.83 0z"></path></svg></a>Jupyterlab</h3>
<p>If you will be using this package from within Jupyterlab, please also install the plotly jupyterlab extension by running:</p>
<div class="highlight highlight-source-shell"><pre>jupyter labextension install @jupyterlab/plotly-extension</pre></div>
<p>See the <a href="https://jupyterlab.readthedocs.io/en/stable/user/extensions.html" rel="nofollow">jupyterlab extension documentation</a> for more details.</p>
