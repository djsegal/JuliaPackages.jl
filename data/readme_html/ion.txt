<p dir="auto">If you would like to install the CLI, please run</p>
<div class="highlight highlight-source-julia notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="using Ion; Ion.comonicon_install()"><pre><span class="pl-k">using</span> Ion; Ion<span class="pl-k">.</span><span class="pl-c1">comonicon_install</span>()</pre></div>

<p dir="auto">It is recommended to use <a href="https://github.com/JuliaLang/juliaup">juliaup</a> for installation of Julia with Ion.</p>
<p dir="auto">For Linux &amp; MacOS users, copy paste the following command
in your terminal</p>
<div class="highlight highlight-source-shell notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="curl -fsSL https://install.julialang.org | sh"><pre>curl -fsSL https://install.julialang.org <span class="pl-k">|</span> sh</pre></div>
<p dir="auto">For Windows, one can use the following</p>
<div class="highlight highlight-source-shell notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="winget install julia -s msstore"><pre>winget install julia -s msstore</pre></div>

<p dir="auto">Each command has their own file or folder:</p>
<ul dir="auto">
<li>implemented with only one file, then <code>&lt;command&gt;.jl</code></li>
<li>implemented with more than one file, then <code>&lt;command&gt;/&lt;command&gt;.jl</code> is the main file contains <code>include</code></li>
</ul>
