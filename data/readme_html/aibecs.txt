<p dir="auto"><strong>AIBECS</strong> (for <strong>A</strong>lgebraic <strong>I</strong>mplicit <strong>B</strong>iogeochemical <strong>E</strong>lemental <strong>C</strong>ycling <strong>S</strong>ystem, pronounced like the cool <a href="https://en.wikipedia.org/wiki/Ibex" rel="nofollow">ibex</a>) is a Julia package that provides ocean biogeochemistry modellers with an easy-to-use interface for creating and running models of the ocean system.</p>
<p dir="auto">AIBECS is a system because it allows you to choose some biogeochemical tracers, define their interactions, select an ocean circulation and <em>Voilà!</em> — your model is ready to run.</p>

<h2 dir="auto"><a id="user-content-concept" class="anchor" aria-hidden="true" href="#concept"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path d="m7.775 3.275 1.25-1.25a3.5 3.5 0 1 1 4.95 4.95l-2.5 2.5a3.5 3.5 0 0 1-4.95 0 .751.751 0 0 1 .018-1.042.751.751 0 0 1 1.042-.018 1.998 1.998 0 0 0 2.83 0l2.5-2.5a2.002 2.002 0 0 0-2.83-2.83l-1.25 1.25a.751.751 0 0 1-1.042-.018.751.751 0 0 1-.018-1.042Zm-4.69 9.64a1.998 1.998 0 0 0 2.83 0l1.25-1.25a.751.751 0 0 1 1.042.018.751.751 0 0 1 .018 1.042l-1.25 1.25a3.5 3.5 0 1 1-4.95-4.95l2.5-2.5a3.5 3.5 0 0 1 4.95 0 .751.751 0 0 1-.018 1.042.751.751 0 0 1-1.042.018 1.998 1.998 0 0 0-2.83 0l-2.5 2.5a1.998 1.998 0 0 0 0 2.83Z"></path></svg></a>Concept</h2>
<p dir="auto">This package was developed to exploit linear-algebra tools and algorithms in Julia to efficiently simulate marine tracers.
AIBECS represents global biogeochemical cycles with a discretized system of nonlinear ordinary differential equations that takes the generic form</p>
<p dir="auto"><math-renderer class="js-display-math" style="display: block" data-static-url="https://github.githubassets.com/static" data-run-id="7290a2d67c6c5263cda7ce1d34ffa8bd">$$\frac{∂\boldsymbol{x}}{∂t} + \mathbf{T} \boldsymbol{x} = \boldsymbol{G}(\boldsymbol{x})$$</math-renderer></p>
<p dir="auto">where <math-renderer class="js-inline-math" style="display: inline" data-static-url="https://github.githubassets.com/static" data-run-id="7290a2d67c6c5263cda7ce1d34ffa8bd">$\boldsymbol{x}$</math-renderer> represents the model state variables, i.e., the marine tracer(s) concentration.
For a single tracer, <math-renderer class="js-inline-math" style="display: inline" data-static-url="https://github.githubassets.com/static" data-run-id="7290a2d67c6c5263cda7ce1d34ffa8bd">$\boldsymbol{x}$</math-renderer> can be interpreted as the 3D field of its concentration.
In AIBECS, <math-renderer class="js-inline-math" style="display: inline" data-static-url="https://github.githubassets.com/static" data-run-id="7290a2d67c6c5263cda7ce1d34ffa8bd">$\boldsymbol{x}$</math-renderer> is represented as a column vector (that's why it's <strong>bold</strong> and <em>italic</em>).</p>
<p dir="auto">The operator <math-renderer class="js-inline-math" style="display: inline" data-static-url="https://github.githubassets.com/static" data-run-id="7290a2d67c6c5263cda7ce1d34ffa8bd">$\mathbf{T}$</math-renderer> is a spatial differential operator that represents the transport of tracers.
For example, for a single tracer transported by ocean circulation,</p>
<p dir="auto"><math-renderer class="js-display-math" style="display: block" data-static-url="https://github.githubassets.com/static" data-run-id="7290a2d67c6c5263cda7ce1d34ffa8bd">$$\mathbf{T} = \nabla \cdot(\boldsymbol{u} - \mathbf{K}\nabla)$$</math-renderer></p>
<p dir="auto">represents the effects of advection and eddy-diffusion
(where <math-renderer class="js-inline-math" style="display: inline" data-static-url="https://github.githubassets.com/static" data-run-id="7290a2d67c6c5263cda7ce1d34ffa8bd">$\boldsymbol{u}$</math-renderer> is the 3D vector of the marine currents and <math-renderer class="js-inline-math" style="display: inline" data-static-url="https://github.githubassets.com/static" data-run-id="7290a2d67c6c5263cda7ce1d34ffa8bd">$\mathbf{K}$</math-renderer> is a 3×3 eddy-diffusivity matrix).
Thus, <math-renderer class="js-inline-math" style="display: inline" data-static-url="https://github.githubassets.com/static" data-run-id="7290a2d67c6c5263cda7ce1d34ffa8bd">$\mathbf{T}$</math-renderer> "acts" on <math-renderer class="js-inline-math" style="display: inline" data-static-url="https://github.githubassets.com/static" data-run-id="7290a2d67c6c5263cda7ce1d34ffa8bd">$\boldsymbol{x}$</math-renderer> such that <math-renderer class="js-inline-math" style="display: inline" data-static-url="https://github.githubassets.com/static" data-run-id="7290a2d67c6c5263cda7ce1d34ffa8bd">$\mathbf{T}\boldsymbol{x}$</math-renderer> is the flux divergence of that tracer.
In AIBECS, <math-renderer class="js-inline-math" style="display: inline" data-static-url="https://github.githubassets.com/static" data-run-id="7290a2d67c6c5263cda7ce1d34ffa8bd">$\mathbf{T}$</math-renderer> is represented by a matrix (that's why it's <strong>bold</strong> and upstraight).</p>
<p dir="auto">Lastly, the right-hand-side, <math-renderer class="js-inline-math" style="display: inline" data-static-url="https://github.githubassets.com/static" data-run-id="7290a2d67c6c5263cda7ce1d34ffa8bd">$\boldsymbol{G}(\boldsymbol{x}$</math-renderer>), represents the local sources minus sinks of each tracer, which must be provided as functions of the tracer(s) <math-renderer class="js-inline-math" style="display: inline" data-static-url="https://github.githubassets.com/static" data-run-id="7290a2d67c6c5263cda7ce1d34ffa8bd">$\boldsymbol{x}$</math-renderer>.</p>
<p dir="auto">To simulate tracers using the AIBECS, you just need to define the transport operators <math-renderer class="js-inline-math" style="display: inline" data-static-url="https://github.githubassets.com/static" data-run-id="7290a2d67c6c5263cda7ce1d34ffa8bd">$\mathbf{T}$</math-renderer> and the net sources and sinks <math-renderer class="js-inline-math" style="display: inline" data-static-url="https://github.githubassets.com/static" data-run-id="7290a2d67c6c5263cda7ce1d34ffa8bd">$\boldsymbol{G}$</math-renderer>.
That's pretty much the whole concept!</p>
