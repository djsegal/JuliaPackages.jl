<p>BiobakeryUtils.jl is a companion package for <a href="https://github.com/BioJulia/Microbiome.jl">Microbiome.jl</a>
for interacting with the <a href="https://bitbucket.org/biobakery/biobakery/wiki/Home" rel="nofollow">bioBakery</a>
family of computational tools
authored by the <a href="http://huttenhower.sph.harvard.edu/" rel="nofollow">Huttenhower Lab</a> at Harvard.</p>
