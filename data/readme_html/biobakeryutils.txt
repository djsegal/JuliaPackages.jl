<p dir="auto"><code>BiobakeryUtils.jl</code> is a companion package for <a href="https://github.com/EcoJulia/Microbiome.jl"><code>Microbiome.jl</code></a>
for interacting with the <a href="https://github.com/biobakery/biobakery/wiki">bioBakery</a>
family of computational tools
authored by the <a href="http://huttenhower.sph.harvard.edu/" rel="nofollow">Huttenhower Lab</a> at Harvard.</p>
<p dir="auto">BiobakeryUtils.jl reexports all functionality from <code>Microbiome.jl</code>,
so you never need to do both <code>using BiobakeryUtils</code> <em>and</em> <code>using Microbiome.jl</code>.</p>
