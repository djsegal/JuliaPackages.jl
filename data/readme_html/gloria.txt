<p dir="auto">Gloria internally runs multiple asynchronous tasks/threads to separate event handling (mouse and keyboard), update calls and rendering.  These tasks call <code>onevent!</code>, <code>update!</code> and <code>render!</code>, respectively, on objects put into the game world.  Using multiple dispatch, behaviour can be added to objects, just as in an object-oriented design.  The engine also comes with a (slightly experimental) rigid body collision system, allowing for game physics without external dependencies.</p>
