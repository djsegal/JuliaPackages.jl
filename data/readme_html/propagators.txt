<p>This is a Julia implementation of Radul and Sussman's propagator model, as
described in "Software Design for Flexibility". It's a fairly direct (and not
necessarily practical) port designed to run their demos, but it does try to be
Julian (eg using structs rather than closures for objects). There are some
walkthroughs in the demos folder.</p>
<p>See also Alexey Radul's
<a href="https://groups.csail.mit.edu/genesis/papers/radul%202009.pdf" rel="nofollow">thesis</a> or <a href="http://dspace.mit.edu/handle/1721.1/44215" rel="nofollow">The
Art of the Propagator</a>.</p>
