<p dir="auto">This package provides a core interface for working with <a href="https://en.wikipedia.org/wiki/Markov_decision_process" rel="nofollow">Markov decision processes (MDPs)</a> and <a href="https://en.wikipedia.org/wiki/Partially_observable_Markov_decision_process" rel="nofollow">partially observable Markov decision processes (POMDPs)</a>.
The <a href="https://juliapomdp.github.io/POMDPs.jl/stable/POMDPTools/#pomdptools_section" rel="nofollow">POMDPTools</a> package acts as a "standard library" for the POMDPs.jl interface, providing implementations of commonly-used components such as policies, belief updaters, distributions, and simulators.</p>
<p dir="auto">Our goal is to provide a common programming vocabulary for:</p>
<ol dir="auto">
<li><a href="http://juliapomdp.github.io/POMDPs.jl/stable/def_pomdp" rel="nofollow">Expressing problems as MDPs and POMDPs</a>.</li>
<li>Writing solver software.</li>
<li>Running simulations efficiently.</li>
</ol>
<p dir="auto">POMDPs.jl integrates with other ecosystems:</p>
<ul dir="auto">
<li>Python can be used to define and solve MDPs and POMDPs via the <a href="https://github.com/JuliaPOMDP/quickpomdps">quickpomdps</a> package or through tables directly via <a href="https://github.com/JuliaPy/pyjulia">pyjulia</a>.</li>
<li>POMDPTools provides <a href="https://juliapomdp.github.io/POMDPModelTools.jl/stable/common_rl/#CommonRLInterface-Integration" rel="nofollow">two-way integration</a> with <a href="https://github.com/JuliaReinforcementLearning/CommonRLInterface.jl">CommonRLInterface</a> and therefore with the <a href="https://github.com/JuliaReinforcementLearning/ReinforcementLearning.jl">JuliaReinforcementLearning packages</a>.</li>
<li>The <a href="https://github.com/JuliaPlanners/SymbolicMDPs.jl">SymbolicMDPs package</a> provides an interface to work with PDDL models.</li>
</ul>
<p dir="auto">For a detailed introduction, check out our <a href="https://juliaacademy.com/p/decision-making-under-uncertainty-with-pomdps-jl" rel="nofollow">Julia Academy course</a>! For help, please post in <a href="https://github.com/JuliaPOMDP/POMDPs.jl/discussions">GitHub Discussions tab</a>. We welcome contributions from anyone! See <a href="/CONTRIBUTING.md">CONTRIBUTING.md</a> for information about contributing.</p>

    initialstate = Uniform([&quot;left&quot;, &quot;right&quot;]),
    discount = 0.95,

        if a == &quot;listen&quot;
            return Deterministic(s) # tiger stays behind the same door
        else # a door is opened
            return Uniform([&quot;left&quot;, &quot;right&quot;]) # reset
        end
    end,

        if a == &quot;listen&quot;
            if sp == &quot;left&quot;
                return SparseCat([&quot;left&quot;, &quot;right&quot;], [0.85, 0.15]) # sparse categorical distribution
            else

        else # the tiger was escaped
            return 10.0
        end

<p dir="auto">Many packages use the POMDPs.jl interface, including MDP and POMDP solvers, support tools, and extensions to the POMDPs.jl interface. POMDPs.jl and all packages in the JuliaPOMDP project are fully supported on Linux. OSX and Windows are supported for all native solvers*, and most non-native solvers should work, but may require additional configuration.</p>
