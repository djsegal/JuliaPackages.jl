<p dir="auto">ZOOclient is the client part of Distributed <a href="https://github.com/eyounx/ZOOpt">ZOOpt</a>. In order to improve the efficiency of handling distributed computing, we use Julia language to code the client end for its high efficiency and Python-like features. Meanwhile, the servers are still coded in Python. Therefore, users programs their objective function in Python as usual, and only need to change a few lines of the client Julia codes (just as easy to understand as Python).</p>
<p dir="auto">Two  zeroth-order optimization methods are implemented in ZOOclient release 0.1, respectively are Asynchronous Sequential RACOS  (ASRacos) method and parallel pareto optimization for subset selection method (PPOSS, IJCAI'16)</p>
<p dir="auto"><strong>Documents:</strong> <a href="https://github.com/eyounx/ZOOpt/wiki/Tutorial-of-Distributed-ZOOpt">Wiki of Distributed ZOOpt</a></p>
<p dir="auto"><strong>Single-thread version:</strong> <a href="https://github.com/eyounx/ZOOpt">ZOOpt</a></p>
<p dir="auto"><strong>Server part of Distributed ZOOpt</strong>: <a href="https://github.com/eyounx/ZOOsrv">ZOOsrv</a></p>

<p dir="auto">To install ZOOclient, start Julia and run:</p>
