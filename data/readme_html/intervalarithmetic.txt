<p><code>IntervalArithmetic.jl</code> is a Julia package for performing <em>Validated Numerics</em> in Julia, i.e. <em>rigorous</em> computations with finite-precision floating-point arithmetic.</p>
<p>All calculations are carried out using <strong>interval arithmetic</strong>: all quantities are treated as intervals, which are propagated throughout a calculation. The final result is an interval that is <em>guaranteed</em> to contain the correct result, starting from the given initial data.</p>
<p>The aim of the package is correctness over speed, although performance considerations are also taken into account.</p>

<h2><a id="user-content-standard-for-interval-arithmetic--ieee-1788-2015" class="anchor" aria-hidden="true" href="#standard-for-interval-arithmetic--ieee-1788-2015"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path fill-rule="evenodd" d="M4 9h1v1H4c-1.5 0-3-1.69-3-3.5S2.55 3 4 3h4c1.45 0 3 1.69 3 3.5 0 1.41-.91 2.72-2 3.25V8.59c.58-.45 1-1.27 1-2.09C10 5.22 8.98 4 8 4H4c-.98 0-2 1.22-2 2.5S3 9 4 9zm9-3h-1v1h1c1 0 2 1.22 2 2.5S13.98 12 13 12H9c-.98 0-2-1.22-2-2.5 0-.83.42-1.64 1-2.09V6.25c-1.09.53-2 1.84-2 3.25C6 11.31 7.55 13 9 13h4c1.45 0 3-1.69 3-3.5S14.5 6 13 6z"></path></svg></a>Standard for Interval Arithmetic:  IEEE 1788-2015</h2>
<p>The IEEE Std 1788-2015 - IEEE Standard for Interval Arithmetic was <a href="https://standards.ieee.org/findstds/standard/1788-2015.html" rel="nofollow">published</a> in June 2015. We are working towards having <code>IntervalArithmetic.jl</code> be conformant with this standard.</p>
<p>To do so, we have incorporated tests from the excellent <a href="https://github.com/oheim/ITF1788">ITF1788 test suite</a>, originally written by Marco Nehmeier and Maximilian Kiesner, and converted to a common format and to output tests for Julia by Oliver Heimlich.</p>
