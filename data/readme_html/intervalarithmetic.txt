<p dir="auto"><code>IntervalArithmetic.jl</code> is a Julia package for performing <em>Validated Numerics</em> in Julia, i.e. <em>rigorous</em> computations with finite-precision floating-point arithmetic.</p>
<p dir="auto">All calculations are carried out using <strong>interval arithmetic</strong>: all quantities are treated as intervals, which are propagated throughout a calculation. The final result is an interval that is <em>guaranteed</em> to contain the correct result, starting from the given initial data.</p>
<p dir="auto">The aim of the package is correctness over speed, although performance considerations are also taken into account.</p>

<p dir="auto">The IEEE Std 1788-2015 - IEEE Standard for Interval Arithmetic was <a href="https://standards.ieee.org/findstds/standard/1788-2015.html" rel="nofollow">published</a> in June 2015. We are working towards having <code>IntervalArithmetic.jl</code> be conformant with this standard.</p>
<p dir="auto">To do so, we have incorporated tests from the excellent <a href="https://github.com/oheim/ITF1788">ITF1788 test suite</a>, originally written by Marco Nehmeier and Maximilian Kiesner, and converted to a common format and to output tests for Julia by Oliver Heimlich.</p>
