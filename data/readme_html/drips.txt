<p>This package provides a fast and robust method for solving LQG Dynamic Ratinoal Inattention Problems based on methods developed in <a href="http://www.afrouzi.com/dynamic_inattention.pdf" rel="nofollow">Afrouzi and Yang (2019): Dynamic Rational Inattention and the Phillips Curve</a>. For detailed documentation of the software, see the <a href="https://afrouzi.github.io/DRIPs.jl/dev" rel="nofollow">documentation page</a>.</p>

<p>The package is registered in the <a href="https://github.com/JuliaRegistries/General">General</a> registry and can be installed at the REPL with <code>] add DRIPs</code>.</p>
