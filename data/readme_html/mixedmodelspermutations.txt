<p dir="auto"><code>MixedModelsPermutations.jl</code> is a Julia package providing permutation and other resampling-based methods for<a href="https://juliastats.org/MixedModels.jl/stable/" rel="nofollow"><code>MixedModels.jl</code></a>.</p>
<p dir="auto">This package is alpha software in early development and results may not be accurate.
Nonetheless, it is registered in the Julia General Registry.
You can install it like this:</p>
<div class="highlight highlight-source-julia notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="julia&gt;]
pkg&gt; add MixedModelsPermutations.jl"><pre>julia<span class="pl-k">&gt;</span>]
pkg<span class="pl-k">&gt;</span> add MixedModelsPermutations<span class="pl-k">.</span>jl</pre></div>

<p dir="auto">If you find inaccurate results, please try the development version to see if the underlying problem has already been fixed before filing an issue.</p>

<p dir="auto"><a href="https://github.com/RePsychLing/MixedModelsSim.jl/"><code>MixedModelsSim.jl</code></a> provides additional functionality for data/power simulation and may also be of interest.</p>
