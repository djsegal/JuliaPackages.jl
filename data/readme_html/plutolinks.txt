<p dir="auto">This package contains a set of utilities built using <a href="https://github.com/JuliaPluto/PlutoHooks.jl">PlutoHooks.jl</a>.</p>
<ul dir="auto">
<li><code>@revise</code>: Watch a local package and update the cell when the package source changes.
See below for more information.</li>
<li><code>@use_task</code>: Run a process and relay it's output to the rest of your notebook.</li>
<li><code>@use_file</code>: Watch a file and reload the content when it changes.</li>
<li><code>@ingredients</code>: Watch a Julia file and automatically run the dependent cells when the code changes.</li>
<li><code>@use_debounce</code>: Wait for a variable value to stabilize before updating its output.</li>
<li><code>@use_process</code> and <code>@use_process_output</code>: to run a process asynchronously and restart it when the calling cell is explicitely run.</li>
</ul>
