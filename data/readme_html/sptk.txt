<p>SPTK.jl is a Julia wrapper for the <a href="http://sp-tk.sourceforge.net/" rel="nofollow">Speech Signal Processing Toolkit (SPTK)</a>, which provides a lot of functionalities for speech signal processing such as linear prediction analysis, mel-cepstrum analysis, generalized cepstrum analysis and mel-generalized cepstrum analysis to name a few. See the original project page for more details.</p>
<p><strong>NOTE</strong>: SPTK.jl is based on a modified version of SPTK (<a href="https://github.com/r9y9/SPTK">r9y9/SPTK</a>).</p>

<p>Functions that SPTK.jl provides are basically same as the SPTK, so if you are new to SPTK, please take a look at the original documentation first and then use SPTK.jl for your need. Also notice that there is no function exported, so you should explicitly import functions if you need.</p>
