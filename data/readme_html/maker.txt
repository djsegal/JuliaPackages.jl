<p><code>Maker</code> is a <a href="http://julialang.org/" rel="nofollow">Julia</a> package to set up tasks and
dependencies in a similar fashion to
<a href="https://en.wikipedia.org/wiki/Makefile" rel="nofollow">make</a> or
<a href="http://docs.seattlerb.org/rake/" rel="nofollow">Rake</a>. Like make, <code>Maker</code> evaluates file and
code dependencies to update processing. The focus here is on data analysis, not
build automation for code.</p>
