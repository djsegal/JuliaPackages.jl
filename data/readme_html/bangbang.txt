<p>BangBang.jl implements functions whose name ends with <code>!!</code>.  Those
functions provide a uniform interface for mutable and immutable data
structures.  Furthermore, those function implement the "widening"
fallback for the case the usual mutating function does not work (e.g.,
<code>push!!(Int[], 1.5)</code> creates a new array <code>Float64[1.5]</code>).</p>
<p>See the supported functions in the
<a href="https://juliafolds.github.io/BangBang.jl/dev" rel="nofollow">documentation</a></p>
