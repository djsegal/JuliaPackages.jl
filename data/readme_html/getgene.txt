<p dir="auto"><strong>GetGene.jl</strong> is a Julia package for getting the nearest locus to an inputted rsid (snpid) through the NCBI's dbSNP API located <a href="https://api.ncbi.nlm.nih.gov/variation/v0/" rel="nofollow">here</a></p>
