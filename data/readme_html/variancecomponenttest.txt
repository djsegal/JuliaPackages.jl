<p><strong>VarianceComponentTest.jl</strong> is a Julia package for performing exact variance component tests in genome-wide association study (GWAS). It provides three types of exact tests</p>
<ul>
<li>exact likelihood ratio test (<em>eLRT</em>)</li>
<li>exact restricted likelihood ratio test (<em>eRLRT</em>)</li>
<li>exact score test (<em>eScore</em>)</li>
</ul>
<p>Detailed documentation is <a href="http://variancecomponenttestjl.readthedocs.org" rel="nofollow"><em>here</em></a></p>
