<p dir="auto">KernelAbstractions (KA) is
a package that enables you to write GPU-like kernels targetting different
execution backends. KA is intended to be a minimal and
performant library that explores ways to write heterogeneous code.
Currently, the following backends are supported:</p>
<ul dir="auto">
<li><a href="https://github.com/JuliaGPU/CUDA.jl">NVIDIA CUDA</a></li>
<li><a href="https://github.com/JuliaGPU/AMDGPU.jl">AMD ROCm</a></li>
<li><a href="https://github.com/JuliaGPU/oneAPI.jl">Intel oneAPI</a></li>
<li><a href="https://github.com/JuliaGPU/Metal.jl">Apple Metal</a></li>
</ul>
