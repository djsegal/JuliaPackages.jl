<p>Create a dashboard with all your badges!</p>
<p>The exported functions are</p>
<ul>
<li><code>dashboard(target_users; output=:markdown, autoopen=false, stargazers=false, travisci=false, stars=true, activity=false)</code></li>
<li><code>pkgdashboard(packagenames; kwargs...)</code></li>
</ul>
<h3><a id="user-content-arguments" class="anchor" aria-hidden="true" href="#arguments"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path fill-rule="evenodd" d="M7.775 3.275a.75.75 0 001.06 1.06l1.25-1.25a2 2 0 112.83 2.83l-2.5 2.5a2 2 0 01-2.83 0 .75.75 0 00-1.06 1.06 3.5 3.5 0 004.95 0l2.5-2.5a3.5 3.5 0 00-4.95-4.95l-1.25 1.25zm-4.69 9.64a2 2 0 010-2.83l2.5-2.5a2 2 0 012.83 0 .75.75 0 001.06-1.06 3.5 3.5 0 00-4.95 0l-2.5 2.5a3.5 3.5 0 004.95 4.95l1.25-1.25a.75.75 0 00-1.06-1.06l-1.25 1.25a2 2 0 01-2.83 0z"></path></svg></a>Arguments:</h3>
<ul>
<li><code>target_users</code>: a string or a vector of strings with github usernames or org names</li>
<li><code>output</code>: <code>:markdown</code> or <code>:html</code></li>
<li><code>autoopen</code>: indicate whether or not the result is opened in editor or browser</li>
<li><code>travisci</code>: Also add a Badge for travis CI</li>
<li><code>activity</code>: display number of commits per month</li>
<li><code>stars</code>: Add github stars badge</li>
<li><code>stargazers</code>: include a plot with github stars over time</li>
</ul>
