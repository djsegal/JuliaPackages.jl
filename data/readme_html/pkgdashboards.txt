<p>Create a dashboard with all your badges!</p>
<p>The exported functions are</p>
<ul>
<li><code>dashboard(target_users; output=:markdown, autoopen=false, stargazers=false, githubci=false, stars=true, activity=false)</code></li>
<li><code>pkgdashboard(packagenames; kwargs...)</code></li>
</ul>
<h3><a id="user-content-arguments" class="anchor" aria-hidden="true" href="#arguments"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path fill-rule="evenodd" d="M4 9h1v1H4c-1.5 0-3-1.69-3-3.5S2.55 3 4 3h4c1.45 0 3 1.69 3 3.5 0 1.41-.91 2.72-2 3.25V8.59c.58-.45 1-1.27 1-2.09C10 5.22 8.98 4 8 4H4c-.98 0-2 1.22-2 2.5S3 9 4 9zm9-3h-1v1h1c1 0 2 1.22 2 2.5S13.98 12 13 12H9c-.98 0-2-1.22-2-2.5 0-.83.42-1.64 1-2.09V6.25c-1.09.53-2 1.84-2 3.25C6 11.31 7.55 13 9 13h4c1.45 0 3-1.69 3-3.5S14.5 6 13 6z"></path></svg></a>Arguments:</h3>
<ul>
<li><code>target_users</code>: a string or a vector of strings with github usernames or org names</li>
<li><code>output</code>: <code>:markdown</code> or <code>:html</code></li>
<li><code>autoopen</code>: indicate whether or not the result is opened in editor or browser</li>
<li><code>githubci</code>: Also add a Badge for github CI</li>
<li><code>activity</code>: display number of commits per month</li>
<li><code>stars</code>: Add github stars badge</li>
<li><code>stargazers</code>: include a plot with github stars over time</li>
</ul>
