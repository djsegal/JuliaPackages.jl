<p dir="auto">NativeSARSOP is a native julia implementation of the <a href="http://www.roboticsproceedings.org/rss04/p9.pdf" rel="nofollow">SARSOP POMDP algorithm</a>. It has comparable speed to the <a href="https://github.com/JuliaPOMDP/SARSOP.jl">wrapped C++ solver</a>, but avoids the bottleneck of writing to a pomdpx file, so it can often find a result in less total time.</p>
