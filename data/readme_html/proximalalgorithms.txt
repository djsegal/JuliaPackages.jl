<p dir="auto">A Julia package for non-smooth optimization algorithms.</p>
<p dir="auto">This package provides algorithms for the minimization of objective functions
that include non-smooth terms, such as constraints or non-differentiable penalties.
Implemented algorithms include:</p>
<ul dir="auto">
<li>(Fast) Proximal gradient methods</li>
<li>Douglas-Rachford splitting</li>
<li>Three-term splitting</li>
<li>Primal-dual splitting algorithms</li>
<li>Newton-type methods</li>
</ul>
<p dir="auto">This package works well in combination with <a href="https://github.com/JuliaFirstOrder/ProximalOperators.jl">ProximalOperators</a> (&gt;= 0.15),
which contains a wide range of functions that can be used to express cost terms.</p>
