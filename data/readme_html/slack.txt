<p dir="auto">The first thing you will need to use this package is a <a href="https://slack.com/create" rel="nofollow">Slack Workspace</a> to <a href="https://api.slack.com/start" rel="nofollow">install a Slack app into</a>.</p>
<p dir="auto">After that, you will need to add a <code>webhook</code> to your app such that you can send and retrieve data from the Slack Workspace.</p>
