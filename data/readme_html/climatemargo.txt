<p>The MARGO model is described in an accompanying (not yet peer-reviewed) manuscript, available to all at <a href="https://eartharxiv.org/5bgyc/" rel="nofollow">EarthArXiv.org/5bgyc</a>.</p>
<p>Try out the MARGO model by running through this Binder tutorial right in your browser (you don't need to download anything â€“ just click the <a href="https://mybinder.org/v2/gh/hdrake/ClimateMARGO.jl/master?filepath=examples%2Ftutorials%2Fbasic_tutorial.ipynb" rel="nofollow"><img src="https://camo.githubusercontent.com/483bae47a175c24dfbfc57390edd8b6982ac5fb3/68747470733a2f2f6d7962696e6465722e6f72672f62616467655f6c6f676f2e737667" alt="Binder" data-canonical-src="https://mybinder.org/badge_logo.svg" style="max-width:100%;"></a> button and, once the tutorial loads, click on the code cells and press <code>Enter</code> to run them).</p>
<p>MARGO is currently undocumented and still in alpha testing. Anyone interested in helping develop the model should contact post an Issue here or contact Henri F. Drake directly (henrifdrake <code>at</code> gmail.com), until explicit guidelines for contributing to the model are posted at a later date.</p>
