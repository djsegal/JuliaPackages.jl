<p dir="auto">Simply embed content in a <a href="https://github.com/fonsp/Pluto.jl">Pluto</a> notebook using short codes, inspired by the <a href="https://gohugo.io/content-management/shortcodes/" rel="nofollow">Hugo shortcodes</a>.</p>
<p dir="auto">The basic usage is shown below, check out the <a href="https://raw.githack.com/hellemo/ShortCodes.jl/main/examples/static-demo.html" rel="nofollow">example</a> to get an impression of the resulting page.</p>
