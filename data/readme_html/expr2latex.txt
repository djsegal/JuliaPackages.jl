<p>A particularly cool use of this is with <a href="https://github.com/symengine/SymEngine.jl">SymEngine.jl</a>.</p>
<p><code>repr</code>ing  a SymEngine expression, outputs basically julia code.
You can <code>parse</code> this to get a julia expression,
and then you can use Expr2LaTeX to <code>render</code> it.</p>
