<p dir="auto">IntervalTrees provides the type: <code>IntervalTree{K, V}</code>.
It implements an associative container mapping <code>(K, K)</code> pairs to to values of type <code>V</code>.
<code>K</code> may be any ordered type, but only pairs <code>(a, b)</code> where <code>a â‰¤ b</code> can be stored.
In other words they are associative contains that map intervals to values.</p>
