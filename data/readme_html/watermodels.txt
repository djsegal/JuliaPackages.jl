<p>WaterModels.jl is a Julia/JuMP package for steady state water network optimization.
It is designed to enable computational evaluation of historical and emerging water network formulations and algorithms using a common platform.
The code is engineered to decouple problem specifications (e.g., water flow, optimal water flow, network expansion) from water network optimization formulations (e.g., mixed-integer linear, mixed-integer nonlinear).
This decoupling enables the definition of a wide variety of water network optimization formulations and their comparison on common problem specifications.</p>
<p><strong>Core Problem Specifications</strong></p>
<ul>
<li>Water Flow (wf) - obtain feasible flows for a fixed network</li>
<li>Optimal Water Flow (owf) - minimize the cost required for network operation</li>
<li>Network Design (des) - minimize the cost of design (or expansion)</li>
</ul>
<p><strong>Core Network Formulations</strong></p>
<ul>
<li>NLP - nonconvex nonlinear program</li>
<li>MICP-R - relaxation-based mixed-integer convex program</li>
<li>MILP - mixed-integer linear program using piecewise linear approximations</li>
<li>MILP-R - relaxation-based mixed-integer linear program</li>
</ul>
