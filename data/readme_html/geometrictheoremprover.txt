<p dir="auto">A Julia package that can prove theorems in Euclidean geometry. Currently, it supports only Ritt-Wu method. For a brief overview check <a href="https://www.youtube.com/watch?v=q_08LE4UOU8" rel="nofollow">this video</a> at JuliaCon2022.</p>

<p dir="auto">Let us prove that Apollonius circle theorem. First we state hypothesis and thesis</p>
<div class="highlight highlight-source-julia notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="hp = @hp begin
    points(A, B, C)
    C := Segment(A, B) ⟂ Segment(A, C)
    M₁ := Midpoint(A, B)
    M₂ := Midpoint(A, C)
    M₃ := Midpoint(B, C)
    H := A ↓ Segment(B, C)
    O := Circle(M₁, M₂, M₃)
end"><pre>hp <span class="pl-k">=</span> <span class="pl-c1">@hp</span> <span class="pl-k">begin</span>
    <span class="pl-c1">points</span>(A, B, C)
    C <span class="pl-k">:=</span> <span class="pl-c1">Segment</span>(A, B) ⟂ <span class="pl-c1">Segment</span>(A, C)
    M₁ <span class="pl-k">:=</span> <span class="pl-c1">Midpoint</span>(A, B)
    M₂ <span class="pl-k">:=</span> <span class="pl-c1">Midpoint</span>(A, C)
    M₃ <span class="pl-k">:=</span> <span class="pl-c1">Midpoint</span>(B, C)
    H <span class="pl-k">:=</span> A ↓ <span class="pl-c1">Segment</span>(B, C)
    O <span class="pl-k">:=</span> <span class="pl-c1">Circle</span>(M₁, M₂, M₃)
<span class="pl-k">end</span></pre></div>
