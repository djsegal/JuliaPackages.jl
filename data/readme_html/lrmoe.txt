<p dir="auto"><strong>LRMoE.jl</strong> is an implementation of the Logit-Reduced Mixture-of-Experts model in <code>julia</code>.
This package is introduced in <a href="https://www.cambridge.org/core/journals/annals-of-actuarial-science/article/abs/lrmoejl-a-software-package-for-insurance-loss-modelling-using-mixture-of-experts-regression-model/18B8F5C17733C4DBAF2F921E08372DD8" rel="nofollow">Tseung et al. 2021</a>.
The website of full documentation is <a href="https://work.sparktseung.com/LRMoE.jl/dev/" rel="nofollow">here</a>.</p>
