<p dir="auto">As stated many times by the author in his <a href="https://www.youtube.com/playlist?list=PLDcUM9US4XdMROZ57-OIRtIK0aOynbgZN" rel="nofollow">online lectures</a>, StatisticalRethinking is a hands-on course. This project is intended to assist with the hands-on aspect of learning the key ideas in Statistical Rethinking.</p>
<p dir="auto">SR2TuringPluto is a Julia project that uses Pluto notebooks for this purpose. Max Lapan has a <a href="https://github.com/Shmuma/rethinking-2ed-julia">version using Jupyter</a>. Many of the Pluto notebooks have been derived from Max Lapan's work!</p>
<p dir="auto">This Julia project uses Turing as the underlying mcmc implementation.  A companion project ( <a href="https://github.com/StatisticalRethinkingJulia/SR2StanPluto.jl">SR2StanPluto.jl</a> ) uses Stan.</p>
<p dir="auto">Each notebook demonstrates Julia versions of <code>code snippets</code> and <code>mcmc models</code> contained in the R package "rethinking" associated with the book <a href="https://xcelab.net/rm/statistical-rethinking/" rel="nofollow">Statistical Rethinking</a> by Richard McElreath.</p>

<p dir="auto">All "rethinking" data files are stored and maintained in StatisticalRethinking.jl and can be accessed via <code>sr_datadir(...)</code>.</p>

<ol dir="auto">
<li>ppl5_1            : Turing model</li>
<li>m5_1t             : The instantiated Turing model (includes data)</li>
</ol>
<p dir="auto">Chain(s):</p>
<ol start="3" dir="auto">
<li>chns5_1t          : MCMCChains object (4000 samples from 4 chains)</li>
</ol>
<p dir="auto">Results as a DataFrame:</p>
<ol start="4" dir="auto">
<li>prior5_1t_df      : Prior samples (DataFrame)</li>
<li>post5_1t_df       : Posterior samples (DataFrame)</li>
<li>quap5_1t_df       : MAP approximation to posterior samples (DataFrame)</li>
<li>pred5_1t_df       : Posterior predictions (DataFrame)</li>
</ol>
<p dir="auto">As before, the <code>t</code> at the end of the model number indicates Turing.</p>
<p dir="auto"><strong>Note</strong>: Naming is not yet consistent through all notebooks. Work in progress!</p>

<p dir="auto">SR2TuringPluto.jl is compatible with the 2nd edition of the book.</p>
<p dir="auto">StructuralCausalModels.jl and ParetoSmoothedImportanceSampling.jl are included as experimental dependencies in the StatisticalRethinking.jl package. Definitely work in progress!</p>
<p dir="auto">Max Lapan added a package Dagitty.jl which covers options similar as available in StructuralCausalModels.jl. There is also a new package, ParetoSmooth.jl which overlaps with ParetoSmoothedImportanceSampling.jl.
As terminology differs from the terminology used in the Statistical Rethinking book, I have not used this package in the notebooks (yet?).</p>
<p dir="auto">Any feedback is appreciated. Please open an issue.</p>
