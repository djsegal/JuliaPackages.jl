<p dir="auto">MieScattering.jl is a pure-Julia package for calculation of Mie scattering. This package is basically a 1:1 port of the great <a href="https://github.com/scottprahl/miepython">miepython</a> Python package written by Scott Prahl. I highly recommed to read his <a href="https://miepython.readthedocs.io/en/latest/index.html" rel="nofollow">documentation</a> which also gives a lot of mathematical insights into the topic. This package basically follows the same syntax as <code>miepython</code> but with small tweaks to adapt to Julia specifics. This package gives comparable or even better (thanks to real multi-threading) performance than the JIT-version of <code>miepython</code> using <code>numba</code>.
The test suite has been adapted as well, so the whole package is numerically tested against Wiscombe's code and of course <code>miepython</code> itself.</p>
