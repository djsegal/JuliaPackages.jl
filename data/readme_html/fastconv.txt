<p>FastConv.jl makes two main contributions for computing convolutions in Julia.</p>
<ol>
<li>Accelerated convolutions for Machine Vision, Machine Learning applications.</li>
<li>Scalable implementation of <em>n-dimensional</em> convolutions that do not exist in current base Julia software.</li>
</ol>

<h3><a id="user-content-background" class="anchor" aria-hidden="true" href="#background"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path fill-rule="evenodd" d="M4 9h1v1H4c-1.5 0-3-1.69-3-3.5S2.55 3 4 3h4c1.45 0 3 1.69 3 3.5 0 1.41-.91 2.72-2 3.25V8.59c.58-.45 1-1.27 1-2.09C10 5.22 8.98 4 8 4H4c-.98 0-2 1.22-2 2.5S3 9 4 9zm9-3h-1v1h1c1 0 2 1.22 2 2.5S13.98 12 13 12H9c-.98 0-2-1.22-2-2.5 0-.83.42-1.64 1-2.09V6.25c-1.09.53-2 1.84-2 3.25C6 11.31 7.55 13 9 13h4c1.45 0 3-1.69 3-3.5S14.5 6 13 6z"></path></svg></a>Background</h3>
<p>Base Julia Convolutions are computed in the frequency domain -- yeilding high memory overhead for each use. While this overhead is suitable for applications where the two convolution input signals are very <strong>large</strong>, this package focuses on applications like Machine Vision, and Machine Learning where the input is convolved with a very <strong>small</strong> kernel. Using a direct computation of convolutions we can achieve a significant speedup of computation for these applications.</p>
<p>Furthermore, current versions of Julia (v0.5) support only 1D and 2D convolutions (no support for 3D or even higher dimensions. This package provides implementation of a generic <em>convn</em> function which can compute convolutions in any arbitrary dimension.</p>
