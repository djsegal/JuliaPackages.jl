<p dir="auto">We describe how to simulate simple reactions, in this case:</p>
<ul dir="auto">
<li>reaction1 describes transcription with a rate k1.</li>
<li>reaction2 describes mRNA decay with a rate k2.</li>
<li>reaction3 describes translation with a rate k3.</li>
<li>reaction4 describes protein decay with a rate k4.</li>
</ul>
<p dir="auto">Each reaction is a Named Tuple that has six entries:</p>
<div class="highlight highlight-source-julia notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="NamedTuple{(:name, :rate, :reactants, :products, :coeff_rea, :coeff_pro), Tuple{String, Int64, Vector{Symbol}, Vector{Symbol}, Vector{Int64}, Vector{Int64}}}"><pre>NamedTuple{(<span class="pl-c1">:name</span>, <span class="pl-c1">:rate</span>, <span class="pl-c1">:reactants</span>, <span class="pl-c1">:products</span>, <span class="pl-c1">:coeff_rea</span>, <span class="pl-c1">:coeff_pro</span>), Tuple{String, Int64, Vector{Symbol}, Vector{Symbol}, Vector{Int64}, Vector{Int64}}}</pre></div>
<ul dir="auto">
<li>name: the name of the reaction (String).</li>
<li>rate: rate of the reaction (Int64).</li>
<li>reactants: the reaction reactant(s) given as a vector of symbols.</li>
<li>products: the reaction product(s) given as a vector of symbols.</li>
<li>coeff_rea and coeff_pro refer to the coefficient of the reactants and the products respectively and given as vector of Int64.</li>
</ul>

<p dir="auto">The function Donne is then used to build a structure that contains information about the model and can be updated in time. The function Donne takes as inputs: the model, initial conditions, the simulation time (1000.0 in this example), time step (0.1 in this example), number of cells (100 in this example) and cells' growth rate (0.03 in this example).</p>
<div class="highlight highlight-source-julia notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="    data = Biomodelling.Donne(model,initiale_population,1000.0,0.1,100,0.03)"><pre>    data <span class="pl-k">=</span> Biomodelling<span class="pl-k">.</span><span class="pl-c1">Donne</span>(model,initiale_population,<span class="pl-c1">1000.0</span>,<span class="pl-c1">0.1</span>,<span class="pl-c1">100</span>,<span class="pl-c1">0.03</span>)</pre></div>
