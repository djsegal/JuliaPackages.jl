<p dir="auto">This Julia package exports the type <code>ImageGeom</code>
and corresponding constructor methods.
It is useful for describing
the pixel or voxel grid
for tomographic image reconstruction.</p>
<p dir="auto">For an explanation see the documentation
using the blue "docs" links above.</p>
