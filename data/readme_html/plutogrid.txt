<p dir="auto">A viewer for tabular data in Pluto. Input can be any Tables.jl compatible table.</p>
<p dir="auto">Compared to the standard Pluto table view, this viewer includes sorting and filtering options in the table GUI as well as (optional) pagination.</p>
<p dir="auto">This viewer is based on <a href="https://www.ag-grid.com/" rel="nofollow">AG-Grid</a> <a href="https://github.com/ag-grid/ag-grid">Community Edition</a>.</p>

<div class="snippet-clipboard-content notranslate position-relative overflow-auto" data-snippet-clipboard-copy-content="readonly_table(df; sortable=true, filterable=true, pagination=false)"><pre class="notranslate"><code>readonly_table(df; sortable=true, filterable=true, pagination=false)
</code></pre></div>
<p dir="auto">Shows a non-editable table in Pluto.</p>
<p dir="auto"><code>df</code>: DataFrame or any Tables.jl compatible data source (the latter is internally converted to a DataFrame)</p>
<p dir="auto"><code>sortable</code>: enables sorting in the table GUI for all columns (enabled by default)</p>
<p dir="auto"><code>filterable</code>: enables filtering in the table GUI for all columns (enabled by default)</p>
<p dir="auto"><code>pagination</code>: enables pagination of the table (disabled by default)</p>
