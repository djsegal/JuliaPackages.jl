<p>PowerModelsReliability.jl is a Julia/JuMP/PowerModels package with extensions for reliability management based on security-constrained OPF.
Building upon  PowerModels, the code is engineered to decouple problem specifications (e.g. Power Flow, Optimal Power Flow, ...) from the power network formulations (e.g. AC, DC-approximation, SOC-relaxation, ...).</p>
<p><strong>Core Problem Specifications</strong></p>
<ul>
<li>Optimal Power Flow with Phase Shifting Transformers and On-line Tap Changing Transformers (tfopf)</li>
<li>Optimal Power Flow with Phase Shifting Transformers and On-line Tap Changing Transformers and Load Shedding (unittfopf)</li>
<li>Security-Constrained Optimal Power Flow with Phase Shifting Transformers and On-line Tap Changing Transformers and Load Shedding (scunittfopf)</li>
</ul>
<p><strong>Core Network Formulations</strong></p>
<ul>
<li>See PowerModels documentation</li>
</ul>
<p><strong>Network Data Formats</strong></p>
<ul>
<li>Matpower ".m" files</li>
</ul>
<p>For further information, consult the PowerModels <a href="https://lanl-ansi.github.io/PowerModels.jl/stable/" rel="nofollow">documentation</a>.</p>
