<p dir="auto">This package exports a suite of types and methods useful for defining arbitrary lattice geometries,
and constructing neighbor tables.
The <a href="https://github.com/cohensbw/LatticeUtilities.jl"><code>LatticeUtilities.jl</code></a> package is intended
to be a lightweight utility package, useful for the development of other, more complicated packages.</p>
