<p dir="auto">This package version requires Julia 1.6 or above. In the Julia shell, switch to the
<code>Pkg</code> shell by pressing <code>]</code> and enter the following command:</p>
<div class="highlight highlight-source-shell notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="pkg&gt; add ImplicitArrays"><pre>pkg<span class="pl-k">&gt;</span> add ImplicitArrays</pre></div>
