<p dir="auto">A spatial analysis library written in Julia, ported from the great <a href="https://github.com/Turfjs/turf"><em>Turf.js</em></a>.</p>
<p dir="auto">Turf.jl uses <a href="https://github.com/JuliaGeo/GeoInterface.jl">GeoInterface.jl</a> and <a href="https://github.com/JuliaGeo/GeoJSON.jl">GeoJSON.jl</a> to create and handle all geographic data.</p>
<h2 dir="auto"><a id="user-content-installation" class="anchor" aria-hidden="true" href="#installation"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path d="m7.775 3.275 1.25-1.25a3.5 3.5 0 1 1 4.95 4.95l-2.5 2.5a3.5 3.5 0 0 1-4.95 0 .751.751 0 0 1 .018-1.042.751.751 0 0 1 1.042-.018 1.998 1.998 0 0 0 2.83 0l2.5-2.5a2.002 2.002 0 0 0-2.83-2.83l-1.25 1.25a.751.751 0 0 1-1.042-.018.751.751 0 0 1-.018-1.042Zm-4.69 9.64a1.998 1.998 0 0 0 2.83 0l1.25-1.25a.751.751 0 0 1 1.042.018.751.751 0 0 1 .018 1.042l-1.25 1.25a3.5 3.5 0 1 1-4.95-4.95l2.5-2.5a3.5 3.5 0 0 1 4.95 0 .751.751 0 0 1-.018 1.042.751.751 0 0 1-1.042.018 1.998 1.998 0 0 0-2.83 0l-2.5 2.5a1.998 1.998 0 0 0 0 2.83Z"></path></svg></a>Installation</h2>
<p dir="auto">Turf.jl can be installed either from the REPL using Pkg mode:</p>
<div class="snippet-clipboard-content notranslate position-relative overflow-auto" data-snippet-clipboard-copy-content="pkg&gt; add Turf"><pre class="notranslate"><code>pkg&gt; add Turf
</code></pre></div>
<p dir="auto">or via <code>Pkg</code>:</p>
<div class="snippet-clipboard-content notranslate position-relative overflow-auto" data-snippet-clipboard-copy-content="julia&gt; import Pkg; Pkg.add(&quot;Turf&quot;)"><pre class="notranslate"><code>julia&gt; import Pkg; Pkg.add("Turf")
</code></pre></div>
<p dir="auto">Alternatively if you want to use the latest version available, you can do:</p>
<div class="snippet-clipboard-content notranslate position-relative overflow-auto" data-snippet-clipboard-copy-content="pkg&gt; add Turf#master"><pre class="notranslate"><code>pkg&gt; add Turf#master
</code></pre></div>
