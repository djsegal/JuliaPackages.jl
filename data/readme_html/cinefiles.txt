<p dir="auto">Use <code>cf = CineFile("path/to/file.cine")</code> to create an object containing metadata and a frame cache for a video file in .cine format produced by a <a href="https://www.phantomhighspeed.com/" rel="nofollow">Phantom</a> high-speed camera. Individual frames can be loaded with, e.g. <code>cf[1]</code>, or ranges with <code>cf[1:3]</code> or <code>cf[end-5:end]</code>.</p>
