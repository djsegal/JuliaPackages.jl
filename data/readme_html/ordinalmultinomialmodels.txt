<p>OrdinalMultinomialModels.jl provides Julia utilities to fit ordered multinomial models, including <a href="https://en.wikipedia.org/wiki/Ordered_logit" rel="nofollow">proportional odds model</a> and <a href="https://en.wikipedia.org/wiki/Ordered_probit" rel="nofollow">ordered Probit model</a> as special cases.</p>
<p>OrdinalMultinomialModels.jl supports Julia v0.7 and later. See documentation for installation and usage.<br>
