<p dir="auto"><code>PQBaseCamp.jl</code> is <a href="https://julialang.org" rel="nofollow">Julia</a> package that encodes basic methods required
by other <a href="https://www.paliquant.com" rel="nofollow">Paliquant</a> packages.</p>

<p dir="auto"><code>PQBaseCamp.jl</code> can be installed, updated, or removed using the <a href="https://docs.julialang.org/en/v1/stdlib/Pkg/" rel="nofollow">Julia package management system</a>. To access the package management interface, open the <a href="https://docs.julialang.org/en/v1/stdlib/REPL/" rel="nofollow">Julia REPL</a>, and start the package mode by pressing <code>]</code>.
While in package mode, to install <code>PQBaseCamp.jl</code>, issue the command:</p>

<p dir="auto">Sometimes we may want to fit a distribution to historical price returns e.g., when constructing random
walk models for a particular asset or basket of assets. To facilitate this, <code>PQBaseCamp.jl</code>  encodes the
<code>ùíü</code> function(s):</p>
<div class="highlight highlight-source-julia notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="ùíü(distribution::Type{T}, data::DataFrame; 
    colkey::Symbol = :Œî) --&gt; UnivariateDistribution where {T&lt;:ContinuousUnivariateDistribution}"><pre><span class="pl-c1">ùíü</span>(distribution<span class="pl-k">::</span><span class="pl-c1">Type{T}</span>, data<span class="pl-k">::</span><span class="pl-c1">DataFrame</span>;

<p dir="auto">where <a href="https://juliastats.org/Distributions.jl/stable/univariate/#Continuous-Distributions" rel="nofollow">ContinuousUnivariateDistribution</a> is any type of continuous univariate probability density function

The optional <code>colkey</code> parameter holds the symbol of the column in the <code>data</code> <a href="https://dataframes.juliadata.org/stable/" rel="nofollow">DataFrame</a> corresponding to the log or simple return values.</p>

<code>ùíü</code> function:</p>

<h3 dir="auto"><a id="user-content-computing-beta" class="anchor" aria-hidden="true" href="#computing-beta"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path d="m7.775 3.275 1.25-1.25a3.5 3.5 0 1 1 4.95 4.95l-2.5 2.5a3.5 3.5 0 0 1-4.95 0 .751.751 0 0 1 .018-1.042.751.751 0 0 1 1.042-.018 1.998 1.998 0 0 0 2.83 0l2.5-2.5a2.002 2.002 0 0 0-2.83-2.83l-1.25 1.25a.751.751 0 0 1-1.042-.018.751.751 0 0 1-.018-1.042Zm-4.69 9.64a1.998 1.998 0 0 0 2.83 0l1.25-1.25a.751.751 0 0 1 1.042.018.751.751 0 0 1 .018 1.042l-1.25 1.25a3.5 3.5 0 1 1-4.95-4.95l2.5-2.5a3.5 3.5 0 0 1 4.95 0 .751.751 0 0 1-.018 1.042.751.751 0 0 1-1.042.018 1.998 1.998 0 0 0-2.83 0l-2.5 2.5a1.998 1.998 0 0 0 0 2.83Z"></path></svg></a>Computing beta</h3>
<p dir="auto"><a href="https://www.investopedia.com/ask/answers/070615/what-formula-calculating-beta.asp" rel="nofollow">Beta</a> is a measure of the volatility of an asset or portfolio relative to the overall market. Beta is defined as the covariance between an asset's return and the market return, dived by the variance of the market return:</p>
<div class="highlight highlight-source-julia notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="Œ≤(tickers::Array{String,1}, data::Dict{String,DataFrame};
    key::Symbol = :Œî, base::String = &quot;SPY&quot;) --&gt; Array{Float64,1}"><pre><span class="pl-c1">Œ≤</span>(tickers<span class="pl-k">::</span><span class="pl-c1">Array{String,1}</span>, data<span class="pl-k">::</span><span class="pl-c1">Dict{String,DataFrame}</span>;
    key<span class="pl-k">::</span><span class="pl-c1">Symbol</span> <span class="pl-k">=</span> <span class="pl-c1">:Œî</span>, base<span class="pl-k">::</span><span class="pl-c1">String</span> <span class="pl-k">=</span> <span class="pl-s"><span class="pl-pds">"</span>SPY<span class="pl-pds">"</span></span>) <span class="pl-k">--&gt;</span> Array{Float64,<span class="pl-c1">1</span>}</pre></div>
<p dir="auto">The <code>Œ≤(...)</code> function returns an array
of beta values (in the same order as the <code>tickers</code> array). The <code>tickers</code> array holds a list of ticker symbols, and <code>data</code> is a <a href="https://docs.julialang.org/en/v1/base/collections/#Dictionaries" rel="nofollow">Dict</a> holding ticker symbols as keys pointing to return <a href="https://dataframes.juliadata.org/stable/" rel="nofollow">DataFrames</a>. The optional argument <code>key</code> of type <code>Symbol</code> holds the column name for the return column in the <code>data</code>
<a href="https://dataframes.juliadata.org/stable/" rel="nofollow">DataFrame</a>, and <code>base</code> denotes the ticker symbol for
the market, taken to be <a href="https://www.google.com/finance/quote/SPY:NYSEARCA?sa=X&amp;ved=2ahUKEwj04c6Oiuv1AhVPmeAKHW-wBG4Q3ecFegQIERAU" rel="nofollow">SPY</a> by default.</p>

<p dir="auto"><a href="https://www.paliquant.com" rel="nofollow">Paliquant</a> relies heavily on <a href="https://en.wikipedia.org/wiki/Monte_Carlo_method" rel="nofollow">Monte-Carlo methods</a>. A key component of <a href="https://en.wikipedia.org/wiki/Monte_Carlo_method" rel="nofollow">Monte-Carlo approaches</a>
is sampling:</p>
<div class="highlight highlight-source-julia notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="sample(model::T, number_of_steps::Int64;
    number_of_sample_paths = 100, number_of_strata = 1) --&gt; Array{Float64,2} where {T&lt;:ContinuousUnivariateDistribution}"><pre><span class="pl-c1">sample</span>(model<span class="pl-k">::</span><span class="pl-c1">T</span>, number_of_steps<span class="pl-k">::</span><span class="pl-c1">Int64</span>;
    number_of_sample_paths <span class="pl-k">=</span> <span class="pl-c1">100</span>, number_of_strata <span class="pl-k">=</span> <span class="pl-c1">1</span>) <span class="pl-k">--&gt;</span> Array{Float64,<span class="pl-c1">2</span>} <span class="pl-k">where</span> {T<span class="pl-k">&lt;:</span><span class="pl-c1">ContinuousUnivariateDistribution</span>}</pre></div>
<p dir="auto">The <code>sample(...)</code> function generates a <code>number_of_steps</code> by <code>2 * number_of_sample_paths</code> array of random draws from the <code>model</code>, where <code>model</code> is any type of continuous univariate probability density function
encoded in the <a href="https://github.com/JuliaStats/Distributions.jl">Distributions.jl</a> package.
The optional argument <code>number_of_strata</code> (default: 1) is used for <a href="https://en.wikipedia.org/wiki/Stratified_sampling" rel="nofollow">stratified sampling</a>; the <code>number_of_strata</code> denotes the number of subpopulations to construct.
The <code>sample(...)</code> uses the <a href="https://en.wikipedia.org/wiki/Antithetic_variates" rel="nofollow">antithetic variates method</a> for variance reduction when generating samples.</p>
