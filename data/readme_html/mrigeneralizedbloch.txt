<p dir="auto">MRIgeneralizedBloch.jl is a Julia package that implements the <a href="https://doi.org/10.1002/mrm.29071" rel="nofollow">generalized Bloch equations</a> for modeling the dynamics of the semi-solid spin pool in magnetic resonance imaging (MRI), and its exchange with the free spin pool. It utilizes the <a href="https://diffeq.sciml.ai/stable/" rel="nofollow">DifferentialEquations.jl</a> package to solve its integro-differential equation. It also implements a linear approximation of the generalized Bloch equations that assumes rectangular radio frequency pulses and uses matrix exponentiation of <a href="https://github.com/JuliaArrays/StaticArrays.jl">static arrays</a>, which results in virtually non-allocating and extremely fast code.</p>
<p dir="auto">For more details, please refer to the <a href="https://doi.org/10.1002/mrm.29071" rel="nofollow">paper</a> and the above linked documentation, which also contains scripts that reproduce all figures in the paper.</p>
