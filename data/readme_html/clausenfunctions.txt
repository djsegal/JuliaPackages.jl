<p dir="auto"><a href="https://github.com/Expander/ClausenFunctions.jl/actions/workflows/build.yml"><img src="https://github.com/Expander/ClausenFunctions.jl/actions/workflows/build.yml/badge.svg" alt="test" style="max-width: 100%;"></a></p>
<p dir="auto">The ClausenFunctions.jl package provides Julia implementations of the
Standard Clausen functions and Glaisher-Clausen functions of integer
order for real arguments.</p>
