<p>GPlot is a plotting package wrapping the <a href="http://glx.sourceforge.net/index.html" rel="nofollow">Graphics Layout Engine</a> (and possibly GnuPlot in the future).
The focus is on speed, ease of use and high-quality output.</p>
<p>Have a look at <a href="https://tlienart.github.io/GPlot.jl/dev" rel="nofollow">the docs</a> if you wish to try this package out.</p>
<p><strong>Note</strong>: this package is very much WIP. If you would like to help, I'll be happy to hear from you.
Simply using the package and report issues or request features here would be very useful.</p>
