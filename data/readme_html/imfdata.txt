<p dir="auto"><strong>Get a list of datasets accessible from the API:</strong></p>
<div class="highlight highlight-source-julia notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="    get_imf_datasets()"><pre>    <span class="pl-c1">get_imf_datasets</span>()</pre></div>
<blockquote>
<p dir="auto"><strong>Note:</strong> While this function returns a list of all available datasets, currently the module only supports data requests from the International Financial Statistics (IFS) dataset.</p>
</blockquote>
<p dir="auto"><strong>Get the list of parameters ("dimensions") for a dataset and their values:</strong></p>

<p dir="auto"><strong>Retrieve data from the IFS dataset</strong></p>
<div class="highlight highlight-source-julia notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="    get_ifs_data(area, indicator, frequency, startyear, endyear)"><pre>    <span class="pl-c1">get_ifs_data</span>(area, indicator, frequency, startyear, endyear)</pre></div>
<ul dir="auto">
<li><code>area</code> and <code>indicator</code> must be <code>String</code>s or <code>Array</code>s of <code>String</code>s (to request multiple series with one call).</li>
<li><code>frequency</code> is limited to annual ("A"), quarterly ("Q"), or monthly ("M").</li>
<li><code>startyear</code> and <code>endyear</code> must be <code>Int</code>s.</li>
</ul>
<p dir="auto">The function returns an <code>IfsSeries</code> object; the data (if available) is in the <code>series</code> field.</p>
