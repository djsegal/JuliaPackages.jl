<p>This package provides nonlinear programming models as implemented by
<a href="https://github.com/JuliaSmoothOptimizers/NLPModels.jl">NLPModels</a>, through the use of
<a href="https://github.com/JuliaOpt/JuMP.jl">JuMP</a>.</p>
<p><strong>Disclaimer</strong>: NLPModelsJuMP is <em>not</em> developed or maintained by the JuMP developers.</p>
<p>See the documentation on NLPModels for the general description of NLPModels. Here, we
focus on the use of JuMP to create these.</p>
