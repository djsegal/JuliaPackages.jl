<p dir="auto">NLPModelsJuMP.jl provides conversion from <a href="https://github.com/jump-dev/JuMP.jl">JuMP.jl</a> / <a href="https://github.com/jump-dev/MathOptInterface.jl">MathOptInterface.jl</a> models to <a href="https://github.com/JuliaSmoothOptimizers/NLPModels.jl">NLPModels</a>.</p>
