<p dir="auto">This package uses the popular <a href="https://github.com/jshttp/mime-db">jshttp/mime-db</a> database. This database is an aggregation of the following sources:</p>
<ul dir="auto">
<li><a href="https://hg.nginx.org/nginx/raw-file/default/conf/mime.types" rel="nofollow">https://hg.nginx.org/nginx/raw-file/default/conf/mime.types</a></li>
<li><a href="https://svn.apache.org/repos/asf/httpd/httpd/trunk/docs/conf/mime.types" rel="nofollow">https://svn.apache.org/repos/asf/httpd/httpd/trunk/docs/conf/mime.types</a></li>
<li><a href="https://www.iana.org/assignments/media-types/media-types.xhtml" rel="nofollow">https://www.iana.org/assignments/media-types/media-types.xhtml</a></li>
</ul>
<p dir="auto">The function implementations, including resolution for conflicting extensions (nginx &gt; apache &gt; mime-db &gt; IANA), is based on <a href="https://github.com/jshttp/mime-types">jshttp/mime-types</a>.</p>

<p dir="auto">While mostly cold and technical, this package contains a couple of opinionated choices:</p>
<ul dir="auto">
<li>We use multiple sources for MIME information, and when these sources have conflicting/duplicate information, we</li>
<li><code>text/javascript</code> is used for <code>.js</code> and <code>.mjs</code> files, not <code>application/javascript</code>. This transition is still underway in official channels, but @fonsp prefers to start using <code>text/javascript</code> today. <a href="https://github.com/jshttp/mime-db/issues/194" data-hovercard-type="issue" data-hovercard-url="/jshttp/mime-db/issues/194/hovercard">More info →</a></li>
<li><code>text/julia</code> is used for <code>.jl</code> files. This MIME is not (yet) official: <a href="https://github.com/JuliaWeb/MIMEs.jl/issues/4" data-hovercard-type="issue" data-hovercard-url="/JuliaWeb/MIMEs.jl/issues/4/hovercard">More info →</a></li>
</ul>

<p dir="auto">Future goals of MIMEs.jl:</p>
<ul dir="auto">
<li>All things MIME! If you are writing a web application in Julia and you are missing MIME-related functionality, let us know! Issues and Pull Requests are welcome.</li>
<li>This package will be regularly updated to match the (monthly) updates to <a href="https://github.com/jshttp/mime-db">jshttp/mime-db</a>. Right now this is involves manually running the update script, but we might automate this in the future. <a href="https://github.com/JuliaWeb/MIMEs.jl/issues/1" data-hovercard-type="issue" data-hovercard-url="/JuliaWeb/MIMEs.jl/issues/1/hovercard">More info →</a></li>
</ul>
