<p dir="auto">A package for performing AC power flow with control actions using ANAREDE file.</p>
<p dir="auto">In large-scale power systems, controllable elements are included in the system to improve voltage profiles in stress situations.</p>
<p dir="auto">The default control options were inspired in ANAREDE software and they are:</p>
<ul dir="auto">
<li>Reactive Generation Limits (QLIM)</li>
<li>Voltage Magnitude Limits (VLIM)</li>
<li>Automatic Reactor Control (CSCA)</li>
<li>Automatic OLTC transformer control (CTAP/CTAF)</li>
<li>Automatic Phase-Shifting control (CPHS)</li>
</ul>
<p dir="auto">Additionally the package allow user to create its own control action.</p>
<p dir="auto">The control actions are achieved through modification in the original power flow formulations, provided by PowerModels.jl. The resultant model is a OPF which is solved through NLP Solvers.</p>
<p dir="auto">Fore more information visit ADD REFS</p>
