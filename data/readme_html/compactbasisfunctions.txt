<p>This package provides a set of basis functions, mostly compactly supported, which are implemented as <a href="https://github.com/JuliaApproximation/ContinuumArrays.jl">Continuum Arrays</a>. Bases are accessed like arrays with continuous dimensions, e.g. as <code>b[0.1, 2]</code> to evaluate the second basis function in the point <code>0.1</code>. Operations such as derivatives, inner products and mass matrices are implemented as <a href="https://github.com/JuliaArrays/LazyArrays.jl">Lazy Array</a> operations, providing a high-level linear algebra interface. Functions in a basis are represented by a lazy multiplication of the basis and a vector of coefficients, which materializes only upon evaluation of the product.</p>
