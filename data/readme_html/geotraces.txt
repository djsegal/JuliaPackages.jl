<blockquote>
<p dir="auto"><em><strong>Important note</strong></em></p>
<p dir="auto">In order to use this software, you must first download the GEOTRACES IDP21 data as a NetCDF file.</p>
<p dir="auto">You must place it in a <code>Data</code> directory in your local "home" directory.
For example, on OSX, the path of my GEOTRACES NetCDF file is:</p>
<div class="snippet-clipboard-content notranslate position-relative overflow-auto" data-snippet-clipboard-copy-content="$HOME/Data/GEOTRACES/GEOTRACES_IDP2021_v1/seawater/netcdf/GEOTRACES_IDP2021_Seawater_Discrete_Sample_Data_v1/GEOTRACES_IDP2021_Seawater_Discrete_Sample_Data_v1.nc"><pre class="notranslate"><code>$HOME/Data/GEOTRACES/GEOTRACES_IDP2021_v1/seawater/netcdf/GEOTRACES_IDP2021_Seawater_Discrete_Sample_Data_v1/GEOTRACES_IDP2021_Seawater_Discrete_Sample_Data_v1.nc
</code></pre></div>
<p dir="auto">The GEOTRACES data management committee does not allow third-party distribution of its data and does not provide a public URL pointing directly to the data, which prevents this package from downloading the data for you.
However, <strong>the GEOTRACES data are publicly accessible, but they <em>must be manually downloaded</em></strong>.</p>
</blockquote>
<p dir="auto">To use this package, like every other registered Julia package, you must <a href="https://julialang.github.io/Pkg.jl/v1/managing-packages/#Adding-registered-packages" rel="nofollow">add it to your environment</a>, and then</p>
<div class="highlight highlight-source-julia notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="julia&gt; using GEOTRACES"><pre>julia<span class="pl-k">&gt;</span> <span class="pl-k">using</span> GEOTRACES</pre></div>
<p dir="auto">should work.</p>

<p dir="auto">Most GEOTRACES variable names are not very explicit (e.g., <code>var70</code> for cadmium).
For this reason, GEOTRACES.jl provides shortcut names for common tracers/variables, like "Cd" for cadmium.
To check which variable they correspond to, you can do (sticking with cadmium as an example)</p>

But suggestions to add new shortcut names are more than welcome! Just start an issue to ask for it on the repository and I'll try to respond ASAP!
(PRs even better — check the <code>varname</code> function for the current list of predefined shortcuts.)</p>
