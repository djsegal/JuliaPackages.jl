<p>Julia wrapper for <a href="https://github.com/vurtun/nuklear">nuklear</a>, a minimal state immediate mode graphical user interface toolkit written in ANSI C. The bindings are auto-generated using
<a href="https://github.com/ihnorton/Clang.jl">Clang.jl</a>. Please refer to nuklear <a href="https://github.com/vurtun/nuklear">docs</a> for how to use those C-bindings.</p>
