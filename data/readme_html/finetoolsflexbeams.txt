<p>FinEtools used for the simulation of large-displacement response of three-dimensional flexible-beam structures. Linear static analysis, modal analysis, linear buckling analysis. Nonlinear statics and dynamics.</p>

<p>Clone the package to your working directory:</p>
<div class="snippet-clipboard-content position-relative" data-snippet-clipboard-copy-content="git clone https://github.com/PetrKryslUCSD/FinEtoolsFlexBeams.jl.git
"><pre><code>git clone https://github.com/PetrKryslUCSD/FinEtoolsFlexBeams.jl.git
</code></pre></div>
<p>Change your working directory to <code>FinEtoolsFlexBeams</code>. Start Julia and run
the following:</p>
<div class="snippet-clipboard-content position-relative" data-snippet-clipboard-copy-content="using Pkg; Pkg.activate(&quot;.&quot;); Pkg.instantiate()
"><pre><code>using Pkg; Pkg.activate("."); Pkg.instantiate()
</code></pre></div>
<p>The tutorials are in the form of Julia scripts in the folder <code>FinEtoolsFlexBeams.jl/docs/src/tutorials</code>. The markdown generated from these files is also in the same folder.</p>
<p>To view the markdown, follow the link to the documentation.</p>
<p>To run a tutorial, head over to the <code>tutorials</code> folder, open the tutorial script, and evaluate in Julia.</p>
