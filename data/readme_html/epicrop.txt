<p dir="auto">Epicrop.jl is a <a href="https://julialang.org" rel="nofollow">Julia</a> package that can be used to simulate disease epidemics.
If you have not yet installed Julia, <a href="https://julialang.org/downloads/platform/" rel="nofollow">please follow the instructions for your operating system</a>.</p>
<p dir="auto">This package provides a Healthy-Latent-Infectious-Postinfectious (HLIP) model for unmanaged plant disease epidemic modelling.
Originally this modelling framework was used by specific disease models in EPIRICE to simulate severity of several rice diseases (<a href="http://dx.doi.org/10.1016/j.cropro.2011.11.009" rel="nofollow">Savary <em>et al.</em> 2012</a>).
Given proper values it can be used with other pathosystems as well.</p>

<p dir="auto">Epicrop.jl is a package that can be used to simulate disease epidemics.
It provides a main function, <code>hlipmodel()</code> and five helper functions that enable users to quickly model bacterial blight, <code>bacterialblight()</code>; brown spot, <code>brownspot()</code>; leaf blast, <code>leafblast()</code>; sheath blight, <code>sheathblight()</code>; and tungro, <code>tungro()</code>, which require only three arguments to run, an object containing the weather data, the start of season date and duration of the season.</p>
<p dir="auto">Please see the <a href="https://adamhsparks.github.io/Epicrop.jl" rel="nofollow">documentation</a> for a full description of how to use the package.</p>
