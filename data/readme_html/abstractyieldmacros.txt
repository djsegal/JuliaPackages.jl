<p dir="auto">using
<a href="https://github.com/tkf/ContextualMacros.jl">ContextualMacros.jl</a>.
This package exists primary for sharing <code>@yield</code> and <code>@yieldfrom</code>
macro in
<a href="https://github.com/JuliaFolds/FGenerators.jl">FGenerators.jl</a> and
<a href="https://github.com/JuliaFolds/GeneratorsX.jl">GeneratorsX.jl</a>.
However, this package can be used for sharing <code>@yield</code> and
<code>@yieldfrom</code> macros between arbitrary packages.  See
<a href="https://github.com/tkf/ContextualMacros.jl">ContextualMacros.jl</a> for
more information.</p>
