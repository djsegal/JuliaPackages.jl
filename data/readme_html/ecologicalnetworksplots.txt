<p>This package provides plotting abilities for <code>EcologicalNetworks.jl</code>. It uses
<code>RecipesBase.jl</code> to do the plotting through <code>Plots.jl</code>. The package currently
generates force-directed layouts, circular layouts, bipartite layouts, heatmaps,
as well as layouts based on node properties.</p>
