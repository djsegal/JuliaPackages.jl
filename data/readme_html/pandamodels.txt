<p dir="auto"><a href="https://github.com/e2nIEE/PandaModels.jl">PandaModels.jl</a> is a <a href="https://julialang.org/" rel="nofollow">Julia</a> package which containing supplementary data and codes to prepare <a href="https://github.com/e2nIEE/pandapower">pandapower</a> networks in a compatible format for Julia packages which are based on <a href="https://lanl-ansi.github.io/InfrastructureModels.jl/dev/" rel="nofollow">InfrastructureModels</a>, such as <a href="https://github.com/lanl-ansi/PowerModels.jl">PowerModels.jl</a> to run and calculate steady-state power network optimization. These packages use <a href="https://github.com/JuliaOpt/JuMP.jl">JuMP</a> as optimization environment.</p>
