<p>DiffBase provides common utilities employed by various JuliaDiff packages, including:</p>
<ul>
<li>The <code>DiffResult</code> API for extracting multiple orders of derivatives simultaneously</li>
<li>The <code>diffrule</code> API for defining reusable derivative definitions</li>
<li>A suite of test functions for stressing the robustness of differentiation tools</li>
</ul>
