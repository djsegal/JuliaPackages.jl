<p dir="auto">A very basic Julia client for retrieving power prices from <a href="https://www.flickelectric.co.nz/" rel="nofollow">Flick Electric</a>.</p>
<p dir="auto">As flick doesn't (yet) allow developers to register applications you have to use
the app tokens embedded in their mobile client app. Internally this package just calls the same
endpoints as the mobile app.</p>
