<p dir="auto">TSFrames is a Julia package to handle timeseries data. It provides a
convenient interface for the commonly used timeseries data
manipulations. TSFrames is built on top of the powerful and mature
<a href="https://github.com/JuliaData/DataFrames.jl">DataFrames.jl</a> making
use of the many capabilities of the <code>DataFrame</code> type and being easily
extensible at the same time.</p>
