<p dir="auto">This Julia package implements the leave out correction of
<a href="https://eml.berkeley.edu/~pkline/papers/KSS2020.pdf" rel="nofollow">Kline, Saggio and Soelvsten (2020)</a> for estimating variance components in two-way fixed effects models. We provide the usual Julia Package as well as an executable/app that can be run in the terminal and does not require the installation of Julia or any other statistical software.</p>
<p dir="auto">For instructions on how to install and use the executable follow this <a href="https://highdimensionaleconlab.github.io/VarianceComponentsHDFE.jl/dev/Executable/" rel="nofollow">link</a>.</p>
<p dir="auto">For instructions on how to install the Julia Package and know about the exported methods follow this <a href="https://highdimensionaleconlab.github.io/VarianceComponentsHDFE.jl/dev/Package/" rel="nofollow">link</a>.</p>

<ul dir="auto">
<li>By default, the code runs a leave-out correction by leaving a match out as opposed to leaving an observation out. See Matlab's <a href="https://github.com/rsaggio87/LeaveOutTwoWay/blob/master/doc/VIGNETTE.pdf">vignette</a> for details.</li>
<li>New, optimized, random projection algorithm for calculation of the statistical leverages which scale extremely well to large datasets.</li>
<li>Controls are partialled out before performing the KSS routine.</li>
<li>High dimensional linear combination can be performed on the fixed effects based on the second identifier (e.g. firm effects).</li>
</ul>
