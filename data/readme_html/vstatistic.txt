<p>Implements the v-statistic, a measure that compares the estimation
accuracy of the ordinary least squares estimator against a random benchmark.</p>
<p>See the paper here: <a href="http://www.ncbi.nlm.nih.gov/m/pubmed/23661222/" rel="nofollow">http://www.ncbi.nlm.nih.gov/m/pubmed/23661222/</a> which features
an implementation in R.</p>
<p>I also wrote a python version, see <a href="https://github.com/dostodabsi/vstat.py">here</a>.</p>
<h1><a id="user-content-install-and-run" class="anchor" aria-hidden="true" href="#install-and-run"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path fill-rule="evenodd" d="M7.775 3.275a.75.75 0 001.06 1.06l1.25-1.25a2 2 0 112.83 2.83l-2.5 2.5a2 2 0 01-2.83 0 .75.75 0 00-1.06 1.06 3.5 3.5 0 004.95 0l2.5-2.5a3.5 3.5 0 00-4.95-4.95l-1.25 1.25zm-4.69 9.64a2 2 0 010-2.83l2.5-2.5a2 2 0 012.83 0 .75.75 0 001.06-1.06 3.5 3.5 0 00-4.95 0l-2.5 2.5a3.5 3.5 0 004.95 4.95l1.25-1.25a.75.75 0 00-1.06-1.06l-1.25 1.25a2 2 0 01-2.83 0z"></path></svg></a>install and run</h1>
<p>install with</p>
<pre><code>Pkg.add("VStatistic")
</code></pre>
<p>run with</p>
<pre><code>import VStatistic: vstat, sample_size

vstat(120, 4, .05) # get v-statistic
sample_size(3, .05, v=.9) # get needed (total n, n per predictor)
</code></pre>
