<p dir="auto">Package make support for general experiment outputs. The experiment result is
stored as row in CSV table. The cols may be different in every experiment.
There are functions prepared for 3D computer vision experiment support.</p>
<p dir="auto">The package is inspired by python <a href="https://github.com/mjirik/exsu"><code>exsu</code></a>.</p>
