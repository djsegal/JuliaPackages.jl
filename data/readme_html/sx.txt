<p>This package relies on the interfaces defined in <a href="https://github.com/blegat/HybridSystems.jl">HybridSystems.jl</a> and <a href="https://github.com/JuliaReach/MathematicalSystems.jl">MathematicalSystems.jl</a>.Â To handle XML files, we use <a href="https://github.com/bicycle1885/EzXML.jl">EzXML.jl</a>. Symbolic algebraic manipulations are performed with <a href="https://github.com/symengine/SymEngine.jl">SymEngine</a>.</p>
