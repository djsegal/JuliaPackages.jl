<p dir="auto">PowerAnalytics.jl is a Julia package that contains analytic routines for power system simulation results in the Sienna ecosystem, specifically from <a href="https://github.com/NREL-Sienna/PowerSimulations.jl">PowerSimulations.jl</a>.</p>

<p dir="auto"><code>PowerAnalytics.jl</code> uses <a href="https://github.com/NREL/PowerSystems.jl">PowerSystems.jl</a> and <a href="https://github.com/NREL/PowerSimulations.jl">PowerSimulations.jl</a> to handle the data and execution power system simulations.</p>
