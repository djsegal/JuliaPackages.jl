<p dir="auto"><code>PQPolygonSDK.jl</code> is a software development kit (SDK) for the <a href="https://polygon.io" rel="nofollow">Polygon.io</a> financial data warehouse in the <a href="https://julialang.org" rel="nofollow">Julia</a> programming language. <a href="https://polygon.io" rel="nofollow">Polygon.io</a> is a leading provider of realtime and historical stock, options, forex (FX) data, and digital/crypto currency data feeds.</p>

<p dir="auto"><code>PQPolygonSDK.jl</code> can be installed, updated, or removed using the <a href="https://docs.julialang.org/en/v1/stdlib/Pkg/" rel="nofollow">Julia package management system</a>. To access the package management interface, open the <a href="https://docs.julialang.org/en/v1/stdlib/REPL/" rel="nofollow">Julia REPL</a>, and start the package mode by pressing <code>]</code>.
While in package mode, to install <code>PQPolygonSDK.jl</code>, issue the command:</p>
<div class="snippet-clipboard-content notranslate position-relative overflow-auto" data-snippet-clipboard-copy-content="(@v1.7.x) pkg&gt; add PQPolygonSDK"><pre class="notranslate"><code>(@v1.7.x) pkg&gt; add PQPolygonSDK
</code></pre></div>
<p dir="auto">To use <code>PQPolygonSDK.jl</code> in your projects, issue the command:</p>
<div class="snippet-clipboard-content notranslate position-relative overflow-auto" data-snippet-clipboard-copy-content="julia&gt; using PQPolygonSDK"><pre class="notranslate"><code>julia&gt; using PQPolygonSDK
