<p dir="auto"><em>dash-bootstrap-components</em> doesn't come with CSS included. This is to give you
the freedom to use any Bootstrap v5 stylesheet of your choice. This means
however that in order for the components to be styled properly, you must link
to a stylesheet yourself.</p>
<p dir="auto">For convenience, links to <a href="https://www.bootstrapcdn.com/" rel="nofollow">BootstrapCDN</a> for each theme are
available through the <code>themes</code> module, which can be used as follows:</p>
<div class="highlight highlight-source-python notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="import dash
import dash_bootstrap_components as dbc

app = dash.Dash(external_stylesheets=[dbc.themes.BOOTSTRAP])"><pre><span class="pl-k">import</span> <span class="pl-s1">dash</span>
<span class="pl-k">import</span> <span class="pl-s1">dash_bootstrap_components</span> <span class="pl-k">as</span> <span class="pl-s1">dbc</span>

<span class="pl-s1">app</span> <span class="pl-c1">=</span> <span class="pl-s1">dash</span>.<span class="pl-v">Dash</span>(<span class="pl-s1">external_stylesheets</span><span class="pl-c1">=</span>[<span class="pl-s1">dbc</span>.<span class="pl-s1">themes</span>.<span class="pl-v">BOOTSTRAP</span>])</pre></div>
<p dir="auto">For more information on how to link local or external CSS, check out the
<a href="https://dash.plotly.com/external-resources" rel="nofollow">Dash documentation</a>.</p>
