<p><em>dash-bootstrap-components</em> is a library of <a href="https://getbootstrap.com/" rel="nofollow">Bootstrap</a>
components for use with <a href="https://dash.plot.ly/" rel="nofollow">Plotly Dash</a>, that makes it easier to
build consistently styled Dash apps with complex, responsive layouts.</p>

<p>To use <em>dash-bootstrap-components</em> you must do two things:</p>
<ul>
<li>Link a Bootstrap v4 compatible stylesheet</li>
<li>Incorporate <em>dash-bootstrap-components</em> into your layout</li>
</ul>
<h3><a id="user-content-linking-a-stylesheet" class="anchor" aria-hidden="true" href="#linking-a-stylesheet"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path fill-rule="evenodd" d="M7.775 3.275a.75.75 0 001.06 1.06l1.25-1.25a2 2 0 112.83 2.83l-2.5 2.5a2 2 0 01-2.83 0 .75.75 0 00-1.06 1.06 3.5 3.5 0 004.95 0l2.5-2.5a3.5 3.5 0 00-4.95-4.95l-1.25 1.25zm-4.69 9.64a2 2 0 010-2.83l2.5-2.5a2 2 0 012.83 0 .75.75 0 001.06-1.06 3.5 3.5 0 00-4.95 0l-2.5 2.5a3.5 3.5 0 004.95 4.95l1.25-1.25a.75.75 0 00-1.06-1.06l-1.25 1.25a2 2 0 01-2.83 0z"></path></svg></a>Linking a stylesheet</h3>
<p><em>dash-bootstrap-components</em> doesn't come with CSS included. This is to give you
the freedom to use any Bootstrap v4 stylesheet of your choice. This means however
that in order for the components to be styled properly, you must link to a
stylesheet yourself.</p>
<p>For convenience, links to <a href="https://www.bootstrapcdn.com/" rel="nofollow">BootstrapCDN</a> for each theme are
available through the <code>themes</code> module, which can be used as follows:</p>
<div class="highlight highlight-source-python position-relative" data-snippet-clipboard-copy-content="import dash
import dash_bootstrap_components as dbc

app = dash.Dash(external_stylesheets=[dbc.themes.BOOTSTRAP])
"><pre><span class="pl-k">import</span> <span class="pl-s1">dash</span>
<span class="pl-k">import</span> <span class="pl-s1">dash_bootstrap_components</span> <span class="pl-k">as</span> <span class="pl-s1">dbc</span>

<span class="pl-s1">app</span> <span class="pl-c1">=</span> <span class="pl-s1">dash</span>.<span class="pl-v">Dash</span>(<span class="pl-s1">external_stylesheets</span><span class="pl-c1">=</span>[<span class="pl-s1">dbc</span>.<span class="pl-s1">themes</span>.<span class="pl-v">BOOTSTRAP</span>])</pre></div>
<p>For more information on how to link local or external CSS, check out the
<a href="https:/dash.plot.ly/external-resources">Dash documentation</a>.</p>
