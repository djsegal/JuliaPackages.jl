<p dir="auto">ModiaPlot is part of <a href="https://modiasim.github.io/docs/" rel="nofollow">ModiaSim</a>.</p>
<p dir="auto">ModiaPlot provides a convenient interface to produce
line plots of time series data where a time series is identified by a String key.
The legends/labels of the plots are automatically constructed by the
keys and the units of the time series. Example:</p>
<div class="highlight highlight-source-julia notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="# result is a dictionary Dict{String,Any}.
ModiaPlot.plot(result, [ (&quot;phi&quot;, &quot;r&quot;)        (&quot;phi&quot;, &quot;phi2&quot;, &quot;w&quot;);
                         (&quot;w&quot;, &quot;w2&quot;, &quot;phi2&quot;) &quot;w&quot;                ],
               heading=&quot;Matrix of plots&quot;)"><pre><span class="pl-c"><span class="pl-c">#</span> result is a dictionary Dict{String,Any}.</span>
ModiaPlot<span class="pl-k">.</span><span class="pl-c1">plot</span>(result, [ (<span class="pl-s"><span class="pl-pds">"</span>phi<span class="pl-pds">"</span></span>, <span class="pl-s"><span class="pl-pds">"</span>r<span class="pl-pds">"</span></span>)        (<span class="pl-s"><span class="pl-pds">"</span>phi<span class="pl-pds">"</span></span>, <span class="pl-s"><span class="pl-pds">"</span>phi2<span class="pl-pds">"</span></span>, <span class="pl-s"><span class="pl-pds">"</span>w<span class="pl-pds">"</span></span>);
                         (<span class="pl-s"><span class="pl-pds">"</span>w<span class="pl-pds">"</span></span>, <span class="pl-s"><span class="pl-pds">"</span>w2<span class="pl-pds">"</span></span>, <span class="pl-s"><span class="pl-pds">"</span>phi2<span class="pl-pds">"</span></span>) <span class="pl-s"><span class="pl-pds">"</span>w<span class="pl-pds">"</span></span>                ],
               heading<span class="pl-k">=</span><span class="pl-s"><span class="pl-pds">"</span>Matrix of plots<span class="pl-pds">"</span></span>)</pre></div>
<p dir="auto">generates the following plot:</p>
<p dir="auto"><a target="_blank" rel="noopener noreferrer" href="docs/resources/images/matrix-of-plots.png"><img src="docs/resources/images/matrix-of-plots.png" alt="Matrix-of-Plots" style="max-width: 100%;"></a></p>
<p dir="auto">The underlying line plot is generated by GLMakie.</p>

<p dir="auto">The package is registered and is installed with (Julia &gt;= 1.5 is required):</p>
<div class="highlight highlight-source-julia notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="julia&gt; ]add ModiaPlot"><pre>julia<span class="pl-k">&gt;</span> ]add ModiaPlot</pre></div>
<p dir="auto">It is recommended to also add the following packages, in order that all tests can be executed
in an own environment (<code>]test ModiaPlot</code> works without adding these packages).</p>
