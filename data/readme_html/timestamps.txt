<p dir="auto">Timestamps provides the ability to work with time-based values without many of the constraints inherent in organizing
a time series. Multiple dates and values of different types can part of an <code>Vector{Timestamp}</code>.</p>
<p dir="auto"><em>WARNING</em>: versions <code>0.1.*</code> are not following semver and may contain breaking changes. Use at your own risk.</p>
<p dir="auto">Previous documentation can be found <a href="http://timestampsjl.readthedocs.org/en/latest/" rel="nofollow">here</a>.</p>
