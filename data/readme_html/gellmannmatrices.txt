<p dir="auto"><a href="https://github.com/thchr/GellMannMatrices.jl/actions/workflows/CI.yml?query=branch%3Amain"><img src="https://github.com/thchr/GellMannMatrices.jl/actions/workflows/CI.yml/badge.svg?branch=main" alt="Build status" style="max-width: 100%;"></a> <a href="https://codecov.io/gh/thchr/GellMannMatrices.jl" rel="nofollow"><img src="https://camo.githubusercontent.com/af4769ac340b8e602f32ad53c3cfa402015fbde7a5e0da8f1ab4d41b72ddc3b8/68747470733a2f2f636f6465636f762e696f2f67682f74686368722f47656c6c4d616e6e4d617472696365732e6a6c2f6272616e63682f6d61696e2f67726170682f62616467652e737667" alt="Coverage" data-canonical-src="https://codecov.io/gh/thchr/GellMannMatrices.jl/branch/main/graph/badge.svg" style="max-width: 100%;"></a></p>
<p dir="auto">A Julia package to compute the <a href="https://en.wikipedia.org/wiki/Generalizations_of_Pauli_matrices#Generalized_Gell-Mann_matrices_(Hermitian)" rel="nofollow">generalized Gell-Mann matrices</a> in <code>d</code> dimensions.</p>
<p dir="auto">The generalized Gell-Mann matrices provide a traceless basis for the space of Hermitian matrices, with elements that are orthogonal under the <a href="https://en.wikipedia.org/wiki/Frobenius_inner_product" rel="nofollow">Frobenius inner product</a>.</p>
<p dir="auto">Notable special cases include the <a href="https://en.wikipedia.org/wiki/Pauli_matrices" rel="nofollow">Pauli matrices σᵢ</a> (<code>d = 2</code>) and the <a href="https://en.wikipedia.org/wiki/Gell-Mann_matrices" rel="nofollow">standard Gell-Mann matrices Λᵢ</a> (<code>d = 3</code>).</p>

<p dir="auto">GellMannMatrices.jl exports a single function <code>gellmann(d)</code> which returns the <code>d^2 - 1</code> generalized Gell-Mann matrices in <code>d</code> dimensions.
The signature <code>gellmann(T, d)</code> allows specifying the matrix type <code>T</code> (which must be a mutable <code>AbstractMatrix</code> with complex element type).</p>
<p dir="auto">As examples, we can compute the Pauli matrices and the standard Gell-Mann matrices:</p>
