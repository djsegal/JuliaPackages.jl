<p dir="auto">This package provides basic infrastructure to work with <a href="https://en.wikipedia.org/wiki/Directed_acyclic_graph" rel="nofollow">Directed Acyclic Graphs</a> (DAGs) in Julia.
It forms the foundation for packages such as <a href="https://github.com/Juice-jl/LogicCircuits.jl">LogicCircuits</a> and <a href="https://github.com/Juice-jl/ProbabilisticCircuits.jl">ProbabilisticCircuits</a>, which define custom DAGs that represent logical or probabilistic computation graphs.</p>
<p dir="auto">Functionality includes:</p>
<ul dir="auto">
<li>applying <code>foreach</code> over the nodes of the DAG in topological or reverse topological order, linearize the DAG</li>
<li>computing the number of nodes, number of edges</li>
<li>propagating a value through the DAG, that is, a <code>foldup</code> operation</li>
<li><code>filter</code> the nodes in the DAG</li>
<li><code>iterate</code> over the nodes in the DAG, compute <code>maximum</code>, <code>minimum</code>, and other Base functions.</li>
<li>find the <code>lca</code> (lowest common ancestor) of nodes in a tree</li>
<li>arrange the DAG nodes in feedforward layers</li>
<li>collecting various statistics about the types of nodes and their in/out-degree</li>
</ul>
<p dir="auto">For example usage, please see the unit tests for <a href="https://github.com/Juice-jl/DirectedAcyclicGraphs.jl/blob/main/test/dags_test.jl">DAGs</a> and the special case of <a href="https://github.com/Juice-jl/DirectedAcyclicGraphs.jl/blob/main/test/trees_test.jl">trees</a>, or the source code of the dependent packages.
