<p dir="auto">Julia package for computing transfer entropy (TE) and mutual information (I).</p>
<p dir="auto">This package provides essential algorithms for the <a href="https://github.com/juliadynamics/CausalityTools.jl"><code>CausalityTools.jl</code></a> package, and uses entropy computations from <a href="https://github.com/juliadynamics/Entropies.jl"><code>Entropies.jl</code></a>.</p>
