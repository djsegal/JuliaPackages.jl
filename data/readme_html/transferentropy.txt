<p>Julia package for computing transfer entropy (TE), conditional mutual information (CMI) or any other information theoretic functional.</p>
<p>This package provides essential algorithms for the <a href="https://github.com/kahaaga/CausalityTools.jl"><code>CausalityTools.jl</code></a> package, which provides methods to detect causal relationship from time series, and tools for computating the transfer operator and invariant measures from time series. See the <a href="https://kahaaga.github.io/CausalityTools.jl/dev/" rel="nofollow"><code>CausalityTools.jl</code> docs</a> for more information.</p>
