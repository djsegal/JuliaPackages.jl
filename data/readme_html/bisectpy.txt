<p>This is a package that migrates Python's <a href="https://docs.python.org/3.7/library/bisect.html#module-bisect" rel="nofollow"><code>bisect</code> module</a> to Jula.</p>
<p>Note that since Julia's array index starts from <code>1</code> but Python starts from <code>0</code>, the returned index of either <code>bisect_left</code>
or <code>bisect_right</code> is always their Python's correspondence plus <code>1</code>!</p>
<p>Also, the behavior of Python's <code>a[:i]</code> where <code>a</code> is an array is also different from Julia: Julia array includes the <code>i</code>th item
but Python does not!</p>
