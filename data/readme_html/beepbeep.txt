<p dir="auto"><code>sound</code> character string or number specifying what sound to be played by either specifying one of the built in sounds, specifying the path to a wav file, or specifying a valid URL refering to a wav file. The default is 1. Possible sounds are:</p>
<ol dir="auto">
<li>"ping"</li>
<li>"coin"</li>
<li>"fanfare"</li>
<li>"complete"</li>
<li>"treasure"</li>
<li>"ready"</li>
<li>"shotgun"</li>
<li>"mario"</li>
<li>"wilhelm"</li>
<li>"facebook"</li>
<li>"sword"</li>
</ol>
<p dir="auto">If <code>sound</code> does not match any of the sounds above, or is not a valid path/URL, a random sound will be played.</p>
