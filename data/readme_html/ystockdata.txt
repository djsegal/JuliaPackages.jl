<p>Yahoo Finance generally supports two different mechanisms for querying security data.
The first returns current values for a set of fields, for a given ticker. In this package
these can be retrieved by using a series of <code>get_xxx</code> methods, which take a ticker as an
argument, and returns a <code>Nullable</code> value as a result. Each field supported by Yahoo has a
corresponding <code>get...</code> method. For example:</p>
<div class="highlight highlight-source-julia"><pre>julia<span class="pl-k">&gt;</span> <span class="pl-k">using</span> YStockData

<span class="pl-c1">Nullable</span>(<span class="pl-c1">136.75</span>)

<span class="pl-c1">Nullable</span>(<span class="pl-s"><span class="pl-pds">"</span>International Business Machines<span class="pl-pds">"</span></span>)</pre></div>

<p>The second mechanism returns OHLC time series values for a ticker and given time period.
This data is available via the <code>get_historical</code> function, which takes the ticker, a start
data and an end date as arguments. This function returns a <code>DataFrame</code> as its result.</p>
