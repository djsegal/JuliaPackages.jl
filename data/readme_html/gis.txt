<p dir="auto">A package for the visualization and manipulation of geographic data.</p>
<h2 dir="auto"><a id="user-content-concept" class="anchor" aria-hidden="true" href="#concept"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path d="m7.775 3.275 1.25-1.25a3.5 3.5 0 1 1 4.95 4.95l-2.5 2.5a3.5 3.5 0 0 1-4.95 0 .751.751 0 0 1 .018-1.042.751.751 0 0 1 1.042-.018 1.998 1.998 0 0 0 2.83 0l2.5-2.5a2.002 2.002 0 0 0-2.83-2.83l-1.25 1.25a.751.751 0 0 1-1.042-.018.751.751 0 0 1-.018-1.042Zm-4.69 9.64a1.998 1.998 0 0 0 2.83 0l1.25-1.25a.751.751 0 0 1 1.042.018.751.751 0 0 1 .018 1.042l-1.25 1.25a3.5 3.5 0 1 1-4.95-4.95l2.5-2.5a3.5 3.5 0 0 1 4.95 0 .751.751 0 0 1-.018 1.042.751.751 0 0 1-1.042.018 1.998 1.998 0 0 0-2.83 0l-2.5 2.5a1.998 1.998 0 0 0 0 2.83Z"></path></svg></a>Concept</h2>
<p dir="auto">JuliaGIS provides a frontend for interacting with geographic data in Julia. The backend is provided by <a href="https://github.com/wkearn/GDAL.jl">GDAL.jl</a> for raster data, the as-yet unwritten OGR.jl wrapper for vector data, and <a href="https://github.com/timholy/Images.jl">Images</a> for visualization.</p>

<div class="snippet-clipboard-content notranslate position-relative overflow-auto" data-snippet-clipboard-copy-content="using GIS"><pre class="notranslate"><code>using GIS
</code></pre></div>
<p dir="auto">allows you to use all of the GIS contained within GDAL, the fictional OGR and any other core GIS packages. Thus <code>using GIS</code> allows you to <code>open_raster("/path/to/raster/file")</code>.</p>
<div class="snippet-clipboard-content notranslate position-relative overflow-auto" data-snippet-clipboard-copy-content="display(raster)"><pre class="notranslate"><code>display(raster)
</code></pre></div>
<p dir="auto">converts the raster to a <code>RasterImage</code> type and displays the <code>RasterImage</code> using <a href="https://github.com/timholy/ImageView.jl">ImageView</a></p>
