<p>A package for the visualization and manipulation of geographic data.</p>
<h2><a id="user-content-concept" class="anchor" aria-hidden="true" href="#concept"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path fill-rule="evenodd" d="M4 9h1v1H4c-1.5 0-3-1.69-3-3.5S2.55 3 4 3h4c1.45 0 3 1.69 3 3.5 0 1.41-.91 2.72-2 3.25V8.59c.58-.45 1-1.27 1-2.09C10 5.22 8.98 4 8 4H4c-.98 0-2 1.22-2 2.5S3 9 4 9zm9-3h-1v1h1c1 0 2 1.22 2 2.5S13.98 12 13 12H9c-.98 0-2-1.22-2-2.5 0-.83.42-1.64 1-2.09V6.25c-1.09.53-2 1.84-2 3.25C6 11.31 7.55 13 9 13h4c1.45 0 3-1.69 3-3.5S14.5 6 13 6z"></path></svg></a>Concept</h2>
<p>JuliaGIS provides a frontend for interacting with geographic data in Julia. The backend is provided by <a href="https://github.com/wkearn/GDAL.jl">GDAL.jl</a> for raster data, the as-yet unwritten OGR.jl wrapper for vector data, and <a href="https://github.com/timholy/Images.jl">Images</a> for visualization.</p>

<h2><a id="user-content-usage" class="anchor" aria-hidden="true" href="#usage"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path fill-rule="evenodd" d="M4 9h1v1H4c-1.5 0-3-1.69-3-3.5S2.55 3 4 3h4c1.45 0 3 1.69 3 3.5 0 1.41-.91 2.72-2 3.25V8.59c.58-.45 1-1.27 1-2.09C10 5.22 8.98 4 8 4H4c-.98 0-2 1.22-2 2.5S3 9 4 9zm9-3h-1v1h1c1 0 2 1.22 2 2.5S13.98 12 13 12H9c-.98 0-2-1.22-2-2.5 0-.83.42-1.64 1-2.09V6.25c-1.09.53-2 1.84-2 3.25C6 11.31 7.55 13 9 13h4c1.45 0 3-1.69 3-3.5S14.5 6 13 6z"></path></svg></a>Usage</h2>
<pre><code>using GIS
</code></pre>
<p>allows you to use all of the GIS contained within GDAL, the fictional OGR and any other core GIS packages. Thus <code>using GIS</code> allows you to <code>open_raster("/path/to/raster/file")</code>.</p>
<pre><code>display(raster)
</code></pre>
<p>converts the raster to a <code>RasterImage</code> type and displays the <code>RasterImage</code> using <a href="https://github.com/timholy/ImageView.jl">ImageView</a></p>
