<p>A package for the visualization and manipulation of geographic data.</p>
<h2><a id="user-content-concept" class="anchor" aria-hidden="true" href="#concept"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path fill-rule="evenodd" d="M7.775 3.275a.75.75 0 001.06 1.06l1.25-1.25a2 2 0 112.83 2.83l-2.5 2.5a2 2 0 01-2.83 0 .75.75 0 00-1.06 1.06 3.5 3.5 0 004.95 0l2.5-2.5a3.5 3.5 0 00-4.95-4.95l-1.25 1.25zm-4.69 9.64a2 2 0 010-2.83l2.5-2.5a2 2 0 012.83 0 .75.75 0 001.06-1.06 3.5 3.5 0 00-4.95 0l-2.5 2.5a3.5 3.5 0 004.95 4.95l1.25-1.25a.75.75 0 00-1.06-1.06l-1.25 1.25a2 2 0 01-2.83 0z"></path></svg></a>Concept</h2>
<p>JuliaGIS provides a frontend for interacting with geographic data in Julia. The backend is provided by <a href="https://github.com/wkearn/GDAL.jl">GDAL.jl</a> for raster data, the as-yet unwritten OGR.jl wrapper for vector data, and <a href="https://github.com/timholy/Images.jl">Images</a> for visualization.</p>

<h2><a id="user-content-usage" class="anchor" aria-hidden="true" href="#usage"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path fill-rule="evenodd" d="M7.775 3.275a.75.75 0 001.06 1.06l1.25-1.25a2 2 0 112.83 2.83l-2.5 2.5a2 2 0 01-2.83 0 .75.75 0 00-1.06 1.06 3.5 3.5 0 004.95 0l2.5-2.5a3.5 3.5 0 00-4.95-4.95l-1.25 1.25zm-4.69 9.64a2 2 0 010-2.83l2.5-2.5a2 2 0 012.83 0 .75.75 0 001.06-1.06 3.5 3.5 0 00-4.95 0l-2.5 2.5a3.5 3.5 0 004.95 4.95l1.25-1.25a.75.75 0 00-1.06-1.06l-1.25 1.25a2 2 0 01-2.83 0z"></path></svg></a>Usage</h2>
<pre><code>using GIS
</code></pre>
<p>allows you to use all of the GIS contained within GDAL, the fictional OGR and any other core GIS packages. Thus <code>using GIS</code> allows you to <code>open_raster("/path/to/raster/file")</code>.</p>
<pre><code>display(raster)
</code></pre>
<p>converts the raster to a <code>RasterImage</code> type and displays the <code>RasterImage</code> using <a href="https://github.com/timholy/ImageView.jl">ImageView</a></p>
