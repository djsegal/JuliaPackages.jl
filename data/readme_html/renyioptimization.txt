<p>The general idea is to optimize the <em>RÃ©nyi free energy</em></p>

<p><a target="_blank" rel="noopener noreferrer" href="https://render.githubusercontent.com/render/math?math=%5CLarge%20F(%5Crho)%20%3D%20%5Cmathrm%7BTr%7D(H%5Crho)%20%2B%20%5Cfrac%7B1%7D%7B%5Cbeta%7D%5Clog(%5Crho%5E2)"><img src="https://render.githubusercontent.com/render/math?math=%5CLarge%20F(%5Crho)%20%3D%20%5Cmathrm%7BTr%7D(H%5Crho)%20%2B%20%5Cfrac%7B1%7D%7B%5Cbeta%7D%5Clog(%5Crho%5E2)" alt="formula" style="max-width:100%;"></a></p>
<p>over a matrix-product-state purification of the density matrix.
It leverages the <a href="https://github.com/Jutho/TensorKit.jl/">TensorKit.jl</a> package for all tensor manipulations, and <a href="https://github.com/jutho/OptimKit.jl">OptimKit.jl</a> for nonlinear optimization on Riemannian manifolds.
See the <a href="https://arxiv.org/abs/2012.12848" rel="nofollow">paper</a> for more information.</p>

<p>To run the optimization on the so-called XY model, you can use the scripts from the <code>scripts</code> directory.
They can be run directly from the command line, for example as <code>$ julia --project=@. routines/beta.jl [--options]</code>.
Note that you may need to install some additional packages.</p>
