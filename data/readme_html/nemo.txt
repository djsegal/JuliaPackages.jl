<p dir="auto">Nemo is a computer algebra package for the Julia programming language. It aims
to cover commutative algebra, number theory and group theory.</p>
<p dir="auto">It is currently developed by Dan Schultz, William Hart, Tommy Hofmann, Fredrik Johansson and
Claus Fieker with contributions from others.</p>
<ul dir="auto">
<li><a href="https://nemocas.github.io/Nemo.jl/dev/" rel="nofollow">https://nemocas.github.io/Nemo.jl/dev/</a> (Online documentation for development version)</li>
<li><a href="https://nemocas.github.io/Nemo.jl/stable/" rel="nofollow">https://nemocas.github.io/Nemo.jl/stable/</a> (Online documentation for latest stable release)</li>
</ul>
<p dir="auto">The official Nemo repository is at:</p>
<ul dir="auto">
<li><a href="https://github.com/Nemocas/Nemo.jl">https://github.com/Nemocas/Nemo.jl</a></li>
</ul>
<p dir="auto">Nemo currently provides:</p>
<ul dir="auto">
<li>Wrappers of MPIR, Flint, Arb and Antic</li>
<li>Generic polynomial rings, matrix spaces, fraction fields, residue rings, power series</li>
<li>finite fields, p-adics, integers, rationals, permutations and characters, number fields</li>
</ul>
