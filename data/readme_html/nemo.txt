<p>Nemo is a computer algebra package for the Julia programming language. It aims
to cover commutative algebra, number theory and group theory.</p>
<p>It is currently developed by William Hart, Tommy Hofmann, Fredrik Johansson and
Claus Fieker with contributions from others.</p>
<ul>
<li><a href="https://nemocas.org" rel="nofollow">https://nemocas.org</a> (Website)</li>
<li><a href="https://nemocas.github.io/Nemo.jl/dev/" rel="nofollow">https://nemocas.github.io/Nemo.jl/dev/</a> (Online documentation)</li>
</ul>
<p>The official Nemo repository is at:</p>
<ul>
<li><a href="https://github.com/Nemocas/Nemo.jl">https://github.com/Nemocas/Nemo.jl</a></li>
</ul>
<p>Nemo currently provides:</p>
<ul>
<li>Wrappers of MPIR, Flint, Arb and Antic</li>
<li>Generic polynomial rings, matrix spaces, fraction fields, residue rings, power series</li>
<li>finite fields, p-adics, integers, rationals, permutations and characters, number fields</li>
</ul>
