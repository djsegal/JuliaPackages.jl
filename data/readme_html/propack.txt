<p dir="auto">A Julia interface to <a href="http://sun.stanford.edu/~rmunk/PROPACK" rel="nofollow">PROPACK</a>, a library for the computation of the truncated singular value decomposition of real (Float32, Float64) and complex (ComplexF32, ComplexF64) matrices or linear operators.
PROPACK only requires operator-vector products to estimate singular values and singular vectors.</p>
