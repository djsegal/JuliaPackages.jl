<p dir="auto">Glue package between <a href="https://github.com/j-fu/VoronoiFVM.jl">VoronoiFVM.jl</a> and <a href="https://github.com/SciML/DifferentialEquations.jl">DifferentialEquations.jl</a></p>
<p dir="auto">The package extends the constructors for <code>ODEFunction</code> and <code>ODEProblem</code> by methods taking in a <code>VoronoiFVM.System</code>:</p>
<div class="snippet-clipboard-content notranslate position-relative overflow-auto" data-snippet-clipboard-copy-content="using VoronoiFVMDiffEq, DifferentialEquations
system = VoronoiFVM.System(...)

problem = ODEProblem(system,inival,tspan)
odesolution = DifferentialEquations.solve(problem, QNDF2())
voronoifvmsolution = reshape(odesolution, system)"><pre class="notranslate"><code>using VoronoiFVMDiffEq, DifferentialEquations
system = VoronoiFVM.System(...)
inival = unknowns(system)
problem = ODEProblem(system,inival,tspan)
odesolution = DifferentialEquations.solve(problem, QNDF2())
voronoifvmsolution = reshape(odesolution, system)
</code></pre></div>
<p dir="auto">Instead of <a href="https://sciml.ai/news/2021/05/24/QNDF/" rel="nofollow"><code>QNDF2</code></a> you can try all mass matrix form capable stiff ode solvers form the  <a href="https://github.com/SciML/DifferentialEquations.jl">DifferentialEquations.jl</a> suite.</p>
<p dir="auto">The package re-exports all of VoronoiFVM, so that it is sufficient to <code>use</code> this package instead of <a href="https://github.com/j-fu/VoronoiFVM.jl">VoronoiFVM.jl</a>.</p>
<p dir="auto">The package requires a recent Julia version (currently 1.8) due to significant advances in package precompilation.</p>
