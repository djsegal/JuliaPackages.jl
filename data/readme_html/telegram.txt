<p>Usage is straightforward, <a href="https://core.telegram.org/bots/api#available-methods" rel="nofollow">Telegram API methods</a> are in one to one correspondence with this Julia wrapper. You need to create connection and then simply call necessary methods</p>
<div class="highlight highlight-source-julia"><pre>julia<span class="pl-k">&gt;</span> token <span class="pl-k">=</span> <span class="pl-s"><span class="pl-pds">"</span>HERE SHOULD BE YOUR TOKEN<span class="pl-pds">"</span></span>
julia<span class="pl-k">&gt;</span> client <span class="pl-k">=</span> <span class="pl-c1">TelegramClient</span>(token)

julia<span class="pl-k">&gt;</span> <span class="pl-c1">getMe</span>(client)
JSON3<span class="pl-k">.</span>Object{Base<span class="pl-k">.</span>CodeUnits{UInt8,String},Array{UInt64,<span class="pl-c1">1</span>}} with <span class="pl-c1">2</span> entries<span class="pl-k">:</span>
  <span class="pl-c1">:ok</span>     <span class="pl-k">=&gt;</span> <span class="pl-c1">true</span>
  <span class="pl-c1">:result</span> <span class="pl-k">=&gt;</span> {â€¦</pre></div>
<p>You can necessary arguments in methods, but some subset can be set in <code>client</code> itself</p>
<div class="highlight highlight-source-julia"><pre>julia<span class="pl-k">&gt;</span> token <span class="pl-k">=</span> <span class="pl-s"><span class="pl-pds">"</span>HERE SHOULD BE YOUR TOKEN<span class="pl-pds">"</span></span>
julia<span class="pl-k">&gt;</span> client <span class="pl-k">=</span> <span class="pl-c1">TelegramClient</span>(token; chat_id <span class="pl-k">=</span> <span class="pl-s"><span class="pl-pds">"</span>HERE SHOULD BE CHAT_ID<span class="pl-pds">"</span></span>)

julia<span class="pl-k">&gt;</span> <span class="pl-c1">sendMessage</span>(client, text <span class="pl-k">=</span> <span class="pl-s"><span class="pl-pds">"</span>Hello, world!<span class="pl-pds">"</span></span>)</pre></div>
