<p>This packages makes it easier to work with monthly or quarterly dates. It defines two types:</p>
<ul>
<li>
<p>A <code>MonthlyDate &lt;: TimeType</code> type</p>
<div class="highlight highlight-source-julia"><pre> julia<span class="pl-k">&gt;</span> <span class="pl-k">using</span> Dates, MonthlyDates
 julia<span class="pl-k">&gt;</span> dt <span class="pl-k">=</span> <span class="pl-c1">Date</span>(<span class="pl-c1">1990</span>, <span class="pl-c1">1</span>)
 <span class="pl-c"><span class="pl-c">#</span> 1990-01-01</span>
 julia<span class="pl-k">&gt;</span> dtm <span class="pl-k">=</span> <span class="pl-c1">MonthlyDate</span>(dt)
 <span class="pl-c"><span class="pl-c">#</span> 1990m01</span>
 <span class="pl-c"><span class="pl-c">#</span> Alternatively, use the MonthlyDate constructor:</span>
 julia<span class="pl-k">&gt;</span> dtm <span class="pl-k">=</span> <span class="pl-c1">MonthlyDate</span>(<span class="pl-c1">1990</span>, <span class="pl-c1">1</span>)
 <span class="pl-c"><span class="pl-c">#</span> 1990m01</span>
 julia<span class="pl-k">&gt;</span> dtm <span class="pl-k">+</span> <span class="pl-c1">Month</span>(<span class="pl-c1">1</span>)
 <span class="pl-c"><span class="pl-c">#</span> 1991m01</span>
 julia<span class="pl-k">&gt;</span> <span class="pl-c1">Date</span>(dtm)
 <span class="pl-c"><span class="pl-c">#</span> 1990-01-01</span>
</pre></div>
</li>
<li>
<p>A <code>QuarterlyDate &lt;: TimeType</code> type</p>
<div class="highlight highlight-source-julia"><pre> julia<span class="pl-k">&gt;</span> <span class="pl-k">using</span> Dates, MonthlyDates
 julia<span class="pl-k">&gt;</span> dt <span class="pl-k">=</span> <span class="pl-c1">Date</span>(<span class="pl-c1">1990</span>, <span class="pl-c1">1</span>)
 <span class="pl-c"><span class="pl-c">#</span> 1990-01-01</span>
 julia<span class="pl-k">&gt;</span> dtq <span class="pl-k">=</span> <span class="pl-c1">QuarterlyDate</span>(dt)
 <span class="pl-c"><span class="pl-c">#</span> 1990q1</span>
 <span class="pl-c"><span class="pl-c">#</span> Alternatively, use the QuarterlyDate constructor:</span>
 julia<span class="pl-k">&gt;</span> dtq <span class="pl-k">=</span> <span class="pl-c1">QuarterlyDate</span>(<span class="pl-c1">1990</span>, <span class="pl-c1">1</span>)
 <span class="pl-c"><span class="pl-c">#</span> 1990q1</span>
 julia<span class="pl-k">&gt;</span> dtq <span class="pl-k">+</span> <span class="pl-c1">Quarter</span>(<span class="pl-c1">3</span>)
 <span class="pl-c"><span class="pl-c">#</span> 1991q4</span>
 julia<span class="pl-k">&gt;</span> <span class="pl-c1">Date</span>(dtq)
 <span class="pl-c"><span class="pl-c">#</span> 1990-01-01</span></pre></div>
</li>
</ul>
