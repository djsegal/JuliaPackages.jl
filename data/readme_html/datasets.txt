<p dir="auto">DataSets helps make data wrangling code more reusable.</p>
<ul dir="auto">
<li>We want to make it easy to <em>relocate</em> an algorithm between different <strong>data
storage environments</strong> without code changes. For example from your laptop to
the cloud, to another user's machine, or to an HPC system.</li>
<li>We want to reduce coupling between data and code, by storing <strong>rich type
information</strong> in metadata. Metadata bridges the gap between the ad hoc
implicit type system of data outside your program and the Julia data
structures within your program.</li>
</ul>
<p dir="auto">Watch <a href="https://www.youtube.com/watch?v=PJkf0CO5APs" rel="nofollow"><strong>DataSets.jl talk</strong></a> from
JuliaCon 2021, or read <a href="https://juliahub.com/docs/DataSets" rel="nofollow"><strong>the latest documentation</strong></a>
more information.</p>
