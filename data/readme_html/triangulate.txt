<p>Julia wrapper for Johnathan Richard Shewchuk's Triangle mesh generator. The package tries to
provide a 1:1 mapping of Triangle's functionality to Julia.</p>
<p>Useful information about Triangle:</p>
<ul>
<li>Triangle home page   <a href="https://www.cs.cmu.edu/~quake/triangle.html" rel="nofollow">https://www.cs.cmu.edu/~quake/triangle.html</a>
with instructions for using Triangle
<ul>
<li><a href="https://www.cs.cmu.edu/~quake/triangle.research.html" rel="nofollow">Research credit, references, and online papers</a></li>
<li><a href="https://www.cs.cmu.edu/~quake/triangle.trouble.html" rel="nofollow">Troubleshooting</a></li>
<li><a href="https://www.cs.cmu.edu/~quake/triangle.switch.html" rel="nofollow">Command line switches</a></li>
</ul>
</li>
</ul>

and <a href="https://github.com/cvdlab/Triangle.jl">Triangle.jl</a>.</p>
