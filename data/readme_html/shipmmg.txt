<p dir="auto"><strong>ShipMMG.jl</strong> is a unofficial Julia package of ship maneuvering simulation with respect to the research committee on “standardization of mathematical model for ship maneuvering predictions” was organized by the JASNAOE.</p>

<p dir="auto">If you want to use MCMC calculation package of this library, you should install from <strong>main</strong> or <strong>dev</strong> branch of this GitHub repository, not from JuliaHub.</p>
<p dir="auto"><strong>ShipMMG.jl</strong> is the project of Julia transportation of <a href="https://github.com/ShipMMG/shipmmg">ShipMMG</a>.</p>
