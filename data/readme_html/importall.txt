<p>The ImportAll.jl package provides the <code>@importall</code> macro to replace the <a href="https://docs.julialang.org/en/v0.6/manual/modules/" rel="nofollow"><code>importall</code></a> command from Julia 0.6, which was <a href="https://github.com/JuliaLang/julia/issues/22789">removed in Julia 1.0</a>.</p>

<pre><code>using ImportAll
@importall(Base)
</code></pre>
<p>This will import every function in Base.</p>
<p>This is generally not a good thing to do but is sometimes necessary.</p>
