<p dir="auto">Suppose we want to draw the next state in a <a href="https://en.wikipedia.org/wiki/Markov_chain_Monte_Carlo" rel="nofollow">MCMC</a> with target (unnormalized) pmf <code>p()</code> using Walker's algorithm with parameter <code>k</code>. Suppose further that the support of <code>p()</code> is ℕ and the current state is <code>x0</code>.</p>
<p dir="auto">The first step is to set up the environment:</p>
<div class="highlight highlight-source-julia notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="using Random, Walker2014Sampling
rng = MersenneTwister(1)"><pre><span class="pl-k">using</span> Random, Walker2014Sampling
rng <span class="pl-k">=</span> <span class="pl-c1">MersenneTwister</span>(<span class="pl-c1">1</span>)</pre></div>
<p dir="auto">Then, we create a sampler <code>s</code> using <code>Walker2014Sampler()</code>:</p>
<div class="highlight highlight-source-julia notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="s = Walker2014Sampler(k);"><pre>s <span class="pl-k">=</span> <span class="pl-c1">Walker2014Sampler</span>(k);</pre></div>
<p dir="auto">Finally, we draw the next state using <code>rand()</code>:</p>
<div class="highlight highlight-source-julia notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="x1 = rand(rng, p, s, x0);"><pre>x1 <span class="pl-k">=</span> <span class="pl-c1">rand</span>(rng, p, s, x0);</pre></div>
<p dir="auto">Be aware that both <code>rng</code> and <code>s</code> are modified in the process.</p>
