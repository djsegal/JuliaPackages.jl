<p>We use <a href="https://github.com/sjkelly/jocco">Jocco</a>. To enable the submodule, run <code>git submodule update --init</code>.
Then use jocco from the MeshSlicer directory with
<code>julia ./doc/jocco/jocco.jl ./src/MeshSlicer.jl</code>. This will generate the
documentation in <code>doc/</code>.</p>
