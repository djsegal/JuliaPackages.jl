<p dir="auto">Fast QCD evolution and convolution.</p>
<p dir="auto">QCDNUM.jl is a Julia wrapper for Michiel Botje's <a href="https://www.nikhef.nl/~h24/qcdnum/" rel="nofollow">QCDNUM</a>, written in Fortran77.
QCDNUM.jl is currently under development and more functionality and documentation will be added soon.</p>
<p dir="auto">Please check out the <a href="https://francescacapel.com/QCDNUM.jl/" rel="nofollow">documentation</a> for more information.</p>
