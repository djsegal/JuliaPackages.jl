<p>In Julia v1.0 (and v0.7) you can install JuAFEM from the Pkg REPL:</p>
<pre><code>pkg&gt; add https://github.com/KristofferC/JuAFEM.jl.git
</code></pre>
<p>which will track the <code>master</code> branch of the package.</p>
<p>In Julia v0.6 you need to checkout the <code>release-0.3</code> branch when installing:</p>
<pre><code>Pkg.clone("https://github.com/KristofferC/JuAFEM.jl.git")
Pkg.checkout("JuAFEM", "release-0.3")
</code></pre>
