<p dir="auto">Using <a href="https://github.com/JuliaNLSolvers/Optim.jl">Optim.jl</a> we can numerically maximize the blackbody function to see how closely it agrees with <a href="https://en.wikipedia.org/wiki/Wien%27s_displacement_law" rel="nofollow">Wien's law</a>.</p>
<div class="highlight highlight-source-julia notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="using Optim

# function to minimize
Teff = 5796
f(λ) = -blackbody(λ, Teff)

# bounded between 1 picometer and 10 m
res = Optim.optimize(f, 1e-12, 10)

λ = Optim.minimizer(res)
isapprox(λ, 2.898e-3 / Teff; rtol=1e-4)"><pre><span class="pl-k">using</span> Optim
