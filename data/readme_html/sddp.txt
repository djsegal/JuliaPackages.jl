<p><code>SDDP.jl</code> is a package for solving large multistage convex stochastic programming
problems using <em>stochastic dual dynamic programming</em>. You can find the documentation
at <a href="https://odow.github.io/SDDP.jl/latest/" rel="nofollow">https://odow.github.io/SDDP.jl/latest/</a>.</p>
