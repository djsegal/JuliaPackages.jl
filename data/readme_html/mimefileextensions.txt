<p dir="auto">Lookup MIME types from a file extension and vice versa.</p>
<p dir="auto">This is based on the mapping in Apache HTTP server project:
<a href="https://svn.apache.org/repos/asf/httpd/httpd/trunk/docs/conf/mime.types" rel="nofollow">https://svn.apache.org/repos/asf/httpd/httpd/trunk/docs/conf/mime.types</a></p>
