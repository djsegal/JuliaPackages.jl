<p dir="auto">This project provides a software interface in Julia for performing causal inference using Gaussian processes with structured latent confounders, as defined in the ICML 2020 <a href="http://proceedings.mlr.press/v119/witty20a/witty20a.pdf" rel="nofollow">paper</a>.</p>
