<p>Defines the distance of a point <code>v</code> to a set <code>s</code>. The distance is always 0 if <code>v âˆˆ s</code>.
The API consists of a single function <code>set_distance(d::D, v::V, s::S)</code>, with <code>S</code> a <code>MOI.AbstractSet</code>,
<code>v</code> a scalar or vector value and <code>d</code> a type of distance.</p>
<p>New sets should implement at least <code>set_distance(::DefaultDistance, v::V, s::S)</code>.</p>
