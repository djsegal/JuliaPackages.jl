<p><strong>ConstrainedDynamicsVis</strong> is a visualizer for the rigid body dynamics package <a href="https://github.com/janbruedigam/ConstrainedDynamics.jl">ConstrainedDynamics</a>. It is built on top of <a href="https://github.com/rdeits/MeshCat.jl">MeshCat</a> and provides an easy to use interface.</p>
