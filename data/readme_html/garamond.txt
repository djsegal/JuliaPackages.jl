<p>Installation can be performed by:</p>
<ul>
<li>first cloning with <code>git clone https://github.com/zgornel/Garamond.jl</code></li>
<li>then running <code>julia -e 'using Pkg; Pkg.activate("."); Pkg.instantiate()'</code> from the project root directory</li>
</ul>
<p>Binary executables of the search server and clients can be built by running <code>./make.jl</code> from the <code>build/</code> directory.</p>
