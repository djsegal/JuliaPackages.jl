<p dir="auto">VS Code has a <a href="https://code.visualstudio.com/docs/datascience/jupyter-notebooks" rel="nofollow">Jupyter notebook extension</a>, as do some other popular IDEs.</p>
<p dir="auto">You can also run the Jupyter notebooks on your browser through binder. Just click the binder icon <a href="https://mybinder.org/v2/gh/jared711/ThreeBodyProblem.jl/master" rel="nofollow"><img src="https://camo.githubusercontent.com/581c077bdbc6ca6899c86d0acc6145ae85e9d80e6f805a1071793dbe48917982/68747470733a2f2f6d7962696e6465722e6f72672f62616467655f6c6f676f2e737667" alt="Binder" data-canonical-src="https://mybinder.org/badge_logo.svg" style="max-width: 100%;"></a> and wait for a JupyterLab window to appear. It may take a few minutes to load. Make sure to <strong>turn off any adblocker</strong> for mybinder.org or it won't load. Pick the Jupyter notebook (filetype .ipynb) you want to run from the file tab on the left. Pressing CTRL+ENTER will run a block of code, while SHIFT+ENTER will run a block and move to the next one.</p>
