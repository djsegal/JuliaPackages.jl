<p dir="auto">DiffImages.jl is a Computer Vision library, which aims to make relavant parts of the <a href="https://github.com/JuliaImages">JuliaImages</a> ecosystem differentiable.</p>
<p dir="auto">We have started with making parts of <a href="https://github.com/JuliaImages/Images.jl">Images.jl</a> and <a href="https://github.com/JuliaImages/ImageTransformations.jl">ImageTransformations.jl</a> differentiable.</p>
<p dir="auto">Currently, we provide support to the following modules:</p>
<ul dir="auto">
<li>Warping modules inside of ImageTransformations.jl.
<ul dir="auto">
<li>which includes <a href="https://juliaimages.org/ImageTransformations.jl/stable/reference/#ImageTransformations.warp" rel="nofollow"><code>ImageTransformations.warp</code></a>, with support for transformations from <a href="https://github.com/JuliaGeometry/CoordinateTransformations.jl">CoordinateTransformations.jl</a>.</li>
</ul>
</li>
<li>Colorspace modules from <a href="https://github.com/JuliaImages/ImageCore.jl">ImageCore.jl</a>.</li>
</ul>
<p dir="auto">You can have a look at the tutorials in the tutorial section in our documentation!</p>
<p dir="auto">In the future, we aim to extend support to kernels from <a href="https://github.com/JuliaImages/ImageFiltering.jl">ImageFiltering.jl</a> as well as extend support to image derivatives which are not currently possible due to no implementations for different types of interpolants used in <a href="https://github.com/JuliaMath/Interpolations.jl">Interpolations.jl</a>.</p>
<p dir="auto">This library started off as a Julia Seasons of Code project. It continues to move ahead for being a very interesting, yet unexplored field of research. I am currently maintaining the library. After ticking off some things I had in my mind for implementations, I will welcome more contributions to the library.</p>
