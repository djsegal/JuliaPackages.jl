<p dir="auto">A Julia package for empirical Bayes estimation. See the <a href="https://nignatiadis.github.io/EBayes.jl/dev" rel="nofollow">documentation</a> for instructions on how to use it.</p>
<p dir="auto">The package implements the empirical Bayes cross-fit method [1], which estimates effect sizes of many experiments by optimally synthesizing experimental data and rich covariate information. Furthermore, the method may leverage any black-box predictive model: [1] provides theoretical guarantees that hold for <em>any</em> regression method and the package here allows usage of any supervised model that has implemented the <a href="https://github.com/alan-turing-institute/MLJ.jl">MLJ.jl</a> interface.</p>
