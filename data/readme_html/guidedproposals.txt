<h2><a id="user-content-key-features" class="anchor" aria-hidden="true" href="#key-features"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path fill-rule="evenodd" d="M7.775 3.275a.75.75 0 001.06 1.06l1.25-1.25a2 2 0 112.83 2.83l-2.5 2.5a2 2 0 01-2.83 0 .75.75 0 00-1.06 1.06 3.5 3.5 0 004.95 0l2.5-2.5a3.5 3.5 0 00-4.95-4.95l-1.25 1.25zm-4.69 9.64a2 2 0 010-2.83l2.5-2.5a2 2 0 012.83 0 .75.75 0 001.06-1.06 3.5 3.5 0 00-4.95 0l-2.5 2.5a3.5 3.5 0 004.95 4.95l1.25-1.25a.75.75 0 00-1.06-1.06l-1.25 1.25a2 2 0 01-2.83 0z"></path></svg></a>Key features</h2>
<p>Implementation of the <a href="https://arxiv.org/pdf/1712.03807.pdf" rel="nofollow">Backward Filtering–Forward Guiding <em>(BFFG)</em></a> algorithm. It is a generic computational framework for working with <a href="https://projecteuclid.org/euclid.bj/1494316837" rel="nofollow">Guided Proposals</a> introduced by M Schauer, F van der Meulen and H van Zanten [<a href="https://arxiv.org/pdf/1311.3606.pdf" rel="nofollow">arXiv</a>].</p>
<p>The main object introduced by this package is a struct <code>GuidProp</code> and it allows for sampling of guided proposals, computing log-likelihood functions of the sampled trajectories and embedding the samplers in smoothing or inference algorithms.</p>

<p>GuidedProposals.jl belongs to a suite of packages in <a href="https://github.com/JuliaDiffusionBayes">JuliaDiffusionBayes</a>, whose aim is to facilitate Bayesian inference for diffusion processes. Some other packages in this suite are as follows:</p>

<li><a href="https://github.com/JuliaDiffusionBayes/ExtensibleMCMC.jl">ExtensibleMCMC.jl</a>: a modular implementation of the Markov chain Monte Carlo (MCMC) algorithms</li>
<li><a href="https://github.com/JuliaDiffusionBayes/DiffusionMCMCTools.jl">DiffusionMCMCTools.jl</a>: utility methods that facilitate easier coding solutions for smoothing and inference algorithms for diffusions</li>
