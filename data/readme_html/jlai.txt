<p dir="auto">To run the code, you will need to first pull the <code>Docker</code> image by running the following command:</p>
<div class="highlight highlight-source-shell notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="docker pull abmhamdi/jlai"><pre>docker pull abmhamdi/jlai</pre></div>
<p dir="auto">This may take a while, as it will download and install all necessary dependencies.</p>

<ul dir="auto">
<li><code>docker-compose up</code> starts the container</li>
<li><code>docker-compose down</code> stops and destroys the container</li>
</ul>
<p dir="auto">Services can be run by typing the command <code>docker compose up</code>. This will start the <code>Jupyter Lab</code> on <a href="http://localhost:2468" rel="nofollow">http://localhost:2468</a> and you should be able to use <code>Julia</code> from within the notebook by starting a new <code>Julia</code> notebook. You can parallelly start <code>Pluto</code> on <a href="http://localhost:1234" rel="nofollow">http://localhost:1234</a>.</p>
