<p dir="auto">This package implements the Microsoft <a href="https://github.com/Microsoft/language-server-protocol">Language Server Protocol</a>
for the <a href="http://julialang.org/" rel="nofollow">Julia</a> programming language.</p>
<p dir="auto">Text editors with a client for the Language Server Protocol are able to
make use of the Julia Language Server for various code editing features:</p>
<ul dir="auto">
<li><a href="https://marketplace.visualstudio.com/items?itemName=julialang.language-julia" rel="nofollow">VS Code</a></li>
<li><a href="https://github.com/pfitzseb/atom-julia-lsp-client">Atom</a></li>
<li><a href="../../wiki/Vim-and-Neovim">Vim and Neovim</a></li>
<li><a href="../../wiki/Emacs">Emacs</a></li>
<li><a href="https://github.com/tomv564/LSP">Sublime Text</a></li>
<li><a href="../../wiki/Kakoune">Kakoune</a></li>
</ul>

<p dir="auto"><strong>Documentation</strong>: <a href="https://www.julia-vscode.org/LanguageServer.jl/dev" rel="nofollow"><img src="https://camo.githubusercontent.com/7fcec4b2d3ab291529fce8ef6a4fcd4129a0683b2f5d5fe2f5c648f02db8b616/68747470733a2f2f696d672e736869656c64732e696f2f62616467652f646f63732d6465762d626c75652e737667" alt="Dev" data-canonical-src="https://img.shields.io/badge/docs-dev-blue.svg" style="max-width: 100%;"></a></p>
<p dir="auto">To install LanguageServer.jl into the current environment:</p>
<div class="highlight highlight-source-julia notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="using Pkg
Pkg.add(&quot;LanguageServer&quot;)"><pre><span class="pl-k">using</span> Pkg
Pkg<span class="pl-k">.</span><span class="pl-c1">add</span>(<span class="pl-s"><span class="pl-pds">"</span>LanguageServer<span class="pl-pds">"</span></span>)</pre></div>
<p dir="auto">To run an instance of LanguageServer.jl on <code>env_path</code>, you can run
Julia as follows:</p>
<div class="highlight highlight-source-shell notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="julia --project=/path/to/LanguageServer.jl/environment \
  -e &quot;using LanguageServer; runserver()&quot; \
  &lt;env_path&gt;"><pre>julia --project=/path/to/LanguageServer.jl/environment \
  -e <span class="pl-s"><span class="pl-pds">"</span>using LanguageServer; runserver()<span class="pl-pds">"</span></span> \
  <span class="pl-k">&lt;</span>env_path<span class="pl-k">&gt;</span></pre></div>
<p dir="auto">If <code>env_path</code> is not specified, the language server will run on the
parent project of <code>pwd</code> or on the default <code>.julia/environments/v#.#</code>
if there is no parent project.</p>
