<p>A <a href="http://julialang.org" rel="nofollow">Julia</a> package for working with special matrix types.</p>
<p>This package extends the <code>LinearAlgebra</code> library with support for special
matrices which are used in linear algebra. Every special matrix has its own type.
The full matrix is accessed by the command <code>Matrix(A)</code>.</p>

<p>Riemann matrix is defined as <code>A = B[2:N+1, 2:N+1]</code>, where
<code>B[i,j] = i-1</code> if <code>i</code> divides <code>j</code>, and <code>-1</code> otherwise.
<a href="http://en.wikipedia.org/wiki/Riemann_hypothesis" rel="nofollow">Riemann hypothesis</a> holds
if and only if <code>det(A) = O( N! N^(-1/2+epsilon))</code> for every <code>epsilon &gt; 0</code>.</p>
