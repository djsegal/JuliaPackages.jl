<p dir="auto">This package implements the efficient RANSAC algorithm for point clouds.
Paper can be found <a href="https://cg.cs.uni-bonn.de/en/publications/paper-details/schnabel-2007-efficient/" rel="nofollow">here</a>.</p>
<blockquote>
<p dir="auto">R. Schnabel, R. Wahl, R. Klein
"Efficient RANSAC for Point-Cloud Shape Detection",
in Computer Graphics Forum, Vol. 26, No. 2, pages 214-226,
Blackwell Publishing, June 2007</p>
</blockquote>
<p dir="auto"><strong>WARNING: this implementation seems to be incorrect. It is slow and may not produce correct results. I'm sad, but almost certainly I won't revisit this project soon.</strong></p>

<p dir="auto">The efficient RANSAC algorithm is used to segment and fit primitive shapes (sphere, plane, cylinder, torus, cone) to point clouds.
Up to my knowledge, this is the first implementation in Julia.</p>

<ul dir="auto">
<li>easy-to-use primitive recognition</li>
<li>extensible: it's easy to add new primitive shapes</li>
<li>fast (work in progress)</li>
</ul>

<p dir="auto">Install the package by:</p>
<div class="highlight highlight-source-julia notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="] add https://github.com/cserteGT3/RANSAC.jl"><pre>] add https<span class="pl-k">:</span><span class="pl-k">//</span>github<span class="pl-k">.</span>com<span class="pl-k">/</span>cserteGT3<span class="pl-k">/</span>RANSAC<span class="pl-k">.</span>jl</pre></div>
<p dir="auto">The input of the algorithm is a point cloud with associated surface normals.
The output is a set of primitive shapes with corresponding sets of points, and the rest of the points that do not belong to any primitives.</p>
<p dir="auto">Follow the <a href="https://csertegt3.github.io/RANSAC.jl/stable/example/" rel="nofollow">detailed example</a> in the documentation.</p>
