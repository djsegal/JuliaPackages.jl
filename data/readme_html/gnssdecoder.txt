<p dir="auto">The data can be retrieved by</p>
<div class="highlight highlight-source-julia notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="decoder.data"><pre>decoder<span class="pl-k">.</span>data</pre></div>
<p dir="auto">Note that GNSSDecoder decodes each time a complete subframe has been retrieved.
<code>decoder.raw_data</code> holds the raw data. <code>decoder.data</code> hold data that has been checked for consistency.
<code>decoder.num_bits_after_valid_subframe</code> counts the number of bits after a valid subframe has been retrieved.</p>
