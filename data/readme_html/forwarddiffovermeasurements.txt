<p dir="auto">The package loads promote rules and some basic operations that favor a <code>ForwardDiff.Dual</code> over a <code>Measurements.Measurement</code>, so this holds:</p>
<div class="highlight highlight-source-julia notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="f(x::Dual,y::Measurement)::Dual"><pre><span class="pl-c1">f</span>(x<span class="pl-k">::</span><span class="pl-c1">Dual</span>,y<span class="pl-k">::</span><span class="pl-c1">Measurement</span>)<span class="pl-k">::</span><span class="pl-c1">Dual</span></pre></div>
<p dir="auto">The package does not export anything nor does define new functions.</p>
<p dir="auto">At the moment, <code>+</code>, <code>-</code>, <code>*</code>, and <code>/</code> are defined. if any Base function is missing, please open an issue.</p>
