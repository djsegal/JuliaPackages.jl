<p dir="auto">TuringGLM makes easy to specify Bayesian <strong>G</strong>eneralized <strong>L</strong>inear <strong>M</strong>odels using the formula syntax and returns an instantiated <a href="https://github.com/TuringLang/Turing.jl">Turing</a> model.</p>
<p dir="auto">Heavily inspired by <a href="https://github.com/paul-buerkner/brms/">brms</a> (uses RStan or CmdStanR) and <a href="https://github.com/bambinos/bambi">bambi</a> (uses PyMC3).</p>
<p dir="auto">The <code>@formula</code> macro is extended from <a href="https://github.com/JuliaStats/StatsModels.jl"><code>StatsModels.jl</code></a> along with <a href="https://github.com/JuliaStats/MixedModels.jl"><code>MixedModels.jl</code></a> for the random-effects (a.k.a. group-level predictors).</p>
