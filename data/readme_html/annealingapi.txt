<p dir="auto">This package provides an API used in <code>CorrelationTrackers.jl</code> and
<code>MaterialReconstruction.jl</code>. There are two kinds of functions and types.</p>
<p dir="auto">The first kind is used to make small modifications to annealed arrays and be
able to roll back a bad modification. This API consists of two functions:
<code>update_corrfns!</code> and <code>rollback!</code>. The first function must behave like
<code>setindex!</code> but return a rollback token of type <code>AbstractRollbackToken</code> which
can be used by the second function to bring an array to the previous state.</p>
<p dir="auto">The second kind contains types like <code>AbstractTracker</code>, <code>S2Tracker</code>, <code>L2Tracker</code>
and so on which are used to designate a correlation function plus phase which
must be used in the annealing process. There are also <code>tracked_data</code>,
<code>tracked_length</code> and <code>tracked_directions</code> functions which you may wish to
implement in correlation tracker.</p>
