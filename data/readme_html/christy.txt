<p dir="auto">Christy.jl is general automated conjecturing package currently in development that gives users the ability to produce conjectured relationships between invariants on a given type of mathematical object. To make conjectures you will need a <code>data.csv</code> file of precomputed invariant (and boolean data if you wish) data. The columns of <code>data.csv</code> should the names of the invariants, names of the boolean properties, and exactly one column called "name" containing the names of each instance of the objects. Thus, the rows of <code>data.csv</code> are the instances of your objects. An example <code>data.csv</code> file containing connected graph data is included with Christy.jl</p>
<p dir="auto">There are two modes for conjecturing with Christy.jl</p>
<ol dir="auto">
<li>Mi Amor</li>
<li>Package</li>
</ol>
<p dir="auto">To run the Mi Amor mode, which essentially calls the functions and modules of Christy.jl as a single Julia script, clone this repository in a directory of your choice. Then open the terminal and run:</p>
<p dir="auto"><code>julia --project mi_amor.jl</code></p>
<p dir="auto">This will prompt the user to enter a number associated with a invariant that Mi Amor will conjecture on.</p>
<p dir="auto">To use Christy.jl as a package (in its current form), clone this repository into a directory of your choice. Then open the Julia REPL, and include <code>Christy.jl</code> with:</p>
<div class="highlight highlight-source-julia notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="include(&quot;Christy.jl&quot;)"><pre><span class="pl-c1">include</span>(<span class="pl-s"><span class="pl-pds">"</span>Christy.jl<span class="pl-pds">"</span></span>)</pre></div>
<p dir="auto">This will give you access to two functions, <code>Christy.surprise_me()</code> and <code>Christy.write_to_me(target_invariant_name)</code> in your current Julia REPL session. For example,</p>
<div class="highlight highlight-source-julia notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="Christy.write_to_me(&quot;matching_number&quot;)"><pre>Christy<span class="pl-k">.</span><span class="pl-c1">write_to_me</span>(<span class="pl-s"><span class="pl-pds">"</span>matching_number<span class="pl-pds">"</span></span>)</pre></div>
<p dir="auto">will conjecture on the matching number invariant column in the provided <code>data.csv</code> file.</p>
