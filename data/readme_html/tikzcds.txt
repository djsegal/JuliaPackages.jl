<p dir="auto">A wrapper around <a href="https://github.com/JuliaTeX/TikzPictures.jl">TikzPictures.jl</a>
for easier drawing of commutative diagrams using
<a href="https://www.ctan.org/pkg/tikz-cd" rel="nofollow"><code>tikz-cd</code></a></p>

<p dir="auto">In order to use this library, <code>lualatex</code> must be installed.
You must also have <code>dvisvgm</code> installed.</p>
<p dir="auto">On Ubuntu, you can get these, if not already present, by running:</p>
<div class="snippet-clipboard-content notranslate position-relative overflow-auto" data-snippet-clipboard-copy-content="$ sudo apt-get install texlive-latex-base texlive-binaries texlive-luatex"><pre class="notranslate"><code>$ sudo apt-get install texlive-latex-base texlive-binaries texlive-luatex
</code></pre></div>

<p dir="auto">You can use <a href="q.uiver.app">Quiver</a> to draw your diagrams and then render them with TikzCD by taking the body of the tikzcd from the exported tex file.
The <code>QuiverCD</code> function will append the quiver.sty file to your TikzCD diagram.</p>
<div class="highlight highlight-source-julia notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="using TikzCDs: Styles
