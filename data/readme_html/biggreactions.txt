<p dir="auto">This is a simple package to automatically download BiGG reactions and metabolites from
the <a href="http://bigg.ucsd.edu/" rel="nofollow">BiGG website</a>. It automatically caches responses to speed up
repeated requests.</p>
<p dir="auto">You can get information about a reaction from the universal model using its ID:</p>
<div class="highlight highlight-source-julia notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="rxn = get_reaction(&quot;PFL&quot;)"><pre>rxn <span class="pl-k">=</span> <span class="pl-c1">get_reaction</span>(<span class="pl-s"><span class="pl-pds">"</span>PFL<span class="pl-pds">"</span></span>)</pre></div>
<p dir="auto">And you can get information about a metabolite from the universal model through its ID:</p>
<div class="highlight highlight-source-julia notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="met = get_metabolite(&quot;h2o&quot;)"><pre>met <span class="pl-k">=</span> <span class="pl-c1">get_metabolite</span>(<span class="pl-s"><span class="pl-pds">"</span>h2o<span class="pl-pds">"</span></span>)</pre></div>

<p dir="auto">The cache can be source of subtle issues. If you get errors or unexpected behavior try the following:</p>
<ol dir="auto">
<li><code>clear_cache!()</code>, restart the Julia session and, try again.</li>
<li>If you still get issues, manually delete the cache files at <code>BiGGReactions.cache_location</code>, restart the Julia session, and try again.</li>
<li>File an issue :/</li>
</ol>
