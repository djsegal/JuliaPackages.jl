<p dir="auto">SciMLSensitivity.jl is a component package in the <a href="https://sciml.ai/" rel="nofollow">SciML Scientific Machine Learning ecosystem</a>.
It holds the sensitivity analysis utilities. Users interested in using this
functionality should check out <a href="https://docs.sciml.ai/DiffEqDocs/stable/" rel="nofollow">DifferentialEquations.jl</a>.</p>
