<p dir="auto">This package provides the core functionality of <a href="https://github.com/RafaelArutjunjan/DerivableFunctions.jl"><strong>DerivableFunctions.jl</strong></a> but without loading backends such as <a href="https://github.com/JuliaDiff/ReverseDiff.jl"><strong>ReverseDiff.jl</strong></a> and <a href="https://github.com/FluxML/Zygote.jl"><strong>Zygote.jl</strong></a> to decrease load times.</p>
<p dir="auto">For explanations and examples, please see the <a href="https://RafaelArutjunjan.github.io/DerivableFunctions.jl/dev" rel="nofollow"><strong>documentation of the downsteam package DerivableFunctions.jl</strong></a>.</p>
