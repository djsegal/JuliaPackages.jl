<p dir="auto">Thebes.jl is a little extension package for vector-graphics package <a href="https://github.com/JuliaGraphics/Luxor.jl">Luxor.jl</a> that provides some rudimentary 3D (2.5D?) wireframe functionality.</p>

<h4 dir="auto"><a id="user-content-thebes-1" class="anchor" aria-hidden="true" href="#thebes-1"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path d="m7.775 3.275 1.25-1.25a3.5 3.5 0 1 1 4.95 4.95l-2.5 2.5a3.5 3.5 0 0 1-4.95 0 .751.751 0 0 1 .018-1.042.751.751 0 0 1 1.042-.018 1.998 1.998 0 0 0 2.83 0l2.5-2.5a2.002 2.002 0 0 0-2.83-2.83l-1.25 1.25a.751.751 0 0 1-1.042-.018.751.751 0 0 1-.018-1.042Zm-4.69 9.64a1.998 1.998 0 0 0 2.83 0l1.25-1.25a.751.751 0 0 1 1.042.018.751.751 0 0 1 .018 1.042l-1.25 1.25a3.5 3.5 0 1 1-4.95-4.95l2.5-2.5a3.5 3.5 0 0 1 4.95 0 .751.751 0 0 1-.018 1.042.751.751 0 0 1-1.042.018 1.998 1.998 0 0 0-2.83 0l-2.5 2.5a1.998 1.998 0 0 0 0 2.83Z"></path></svg></a>‘Thebes’?</h4>
<p dir="auto">Luxor.jl is built on <a href="https://github.com/JuliaGraphics/Cairo.jl">Cairo.jl</a>, and Thebes.jl provides a 3D context for Luxor.jl. Luxor, the modern city about 675km south of Cairo, is built on the site of Thebes, the ancient capital city of Egypt, which flourished 5000 years ago.</p>
