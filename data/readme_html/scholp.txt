<p dir="auto">This library supports multithreading through Julia's Base.Threads. However, in some cases, it can conflict with the multithreading in BLAS routines. Essentially, these are cases where an iterative solver is used to solve a large number of linear systems and there is simple parallelism over the linear systems being solved. To avoid the conflicts with BLAS threading, here is an example of how one should start Julia.</p>
<div class="highlight highlight-source-shell notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="export OPENBLAS_NUM_THREADS=1
