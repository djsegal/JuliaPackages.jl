<p dir="auto">A package to calculate CG-coefficient, Racah coefficient, and Wigner 3j, 6j, 9j symbols.</p>
<p dir="auto">One can get the exact result with <code>SqrtRational</code> type, which use <code>BigInt</code> to avoid overflow. And we also offer float version for numeric calculation, which is about twice faster than <a href="https://www.gnu.org/software/gsl/" rel="nofollow">GNU Scientific Library</a>.</p>
<p dir="auto">I also rewrite the float version with c++ for numeric calculation: <a href="https://github.com/0382/WignerSymbol">WignerSymbol</a>.</p>
<p dir="auto">For more details and the calculation formula, please see the document <a href="https://0382.github.io/CGcoefficient.jl/dev" rel="nofollow"><img src="https://camo.githubusercontent.com/7fcec4b2d3ab291529fce8ef6a4fcd4129a0683b2f5d5fe2f5c648f02db8b616/68747470733a2f2f696d672e736869656c64732e696f2f62616467652f646f63732d6465762d626c75652e737667" alt="" data-canonical-src="https://img.shields.io/badge/docs-dev-blue.svg" style="max-width: 100%;"></a>.</p>
