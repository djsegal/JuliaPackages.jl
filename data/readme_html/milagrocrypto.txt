<p dir="auto">Julia bindings for AMCL_jll.</p>
<p dir="auto">The objective of this project is to provide efficient and secure access to the
<a href="https://github.com/apache/incubator-milagro-crypto-c">AMCL C library</a> from Julia for all supported platforms (OS and CPU).</p>

<p dir="auto">If you are using the <code>csprng()</code> method, it will generate random seed from <code>/dev/random</code>,
which will "hang" by design if there is not enough entropy in the pool.</p>
<p dir="auto">Try re-running your app with the <code>JULIA_DEBUG=MilagroCrypto</code> environment variable and look for
the <code>got seed from OS</code> console log to confirm that your application is <strong>not</strong>
stuck waiting for more entropy.</p>
