<p><code>MiniLoggers.jl</code> are build around the idea of extremely simple and visually compact logger, with output format defined by string in the same manner as in <a href="https://docs.python.org/3/howto/logging.html#changing-the-format-of-displayed-messages" rel="nofollow">python logging module</a>. It uses <code>stdout</code> by default and intended to be use instead of the usual <code>print</code> commands.</p>
