<p>This is a experimental package that uses the interface from <a href="https://github.com/JuliaGraphs/LightGraphs.jl">LightGraphs.jl</a>.
It is similar to <a href="https://github.com/JuliaGraphs/MetaGraphs.jl">MetaGraphs.jl</a> and <a href="https://github.com/JuliaGraphs/SimpleWeightedGraphs.jl">SimpleWeightedGraphs.jl</a>.
It solves the following problem:</p>
<ul>
<li>The topology of <code>MetaGraphs</code> can be changed fast, but changing and querying edge values is slow.</li>
<li>Changing the topology of <code>SimpleWeightedGraphs</code> is slow, but changing and querying edge values is fast.</li>
</ul>
