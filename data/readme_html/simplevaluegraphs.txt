<p dir="auto">SimpleValueGraphs is a <a href="https://github.com/JuliaGraphs/Graphs.jl">Graphs.jl</a> compatible package for graphs with multiple, homogeneous vertex, edge and graph metadata. In particular it provides:</p>
<ul dir="auto">
<li>an abstract interface for graphs with metadata</li>
<li>concrete implementations of mutable graphs with metadata</li>
</ul>
<p dir="auto">Compared to <a href="https://github.com/JuliaGraphs/SimpleWeightedGraphs.jl">SimpleWeightedGraphs.jl</a> it has the following advantages:</p>
<ul dir="auto">
<li>vertex and graph metadata</li>
<li>multiple edge metadata</li>
<li>faster structural modifications of graphs</li>
</ul>
<p dir="auto">Compared to <a href="https://github.com/JuliaGraphs/MetaGraphs.jl">MetaGraphs.jl</a> it has the following advantages:</p>
<ul dir="auto">
<li>faster access and modifications of metadata</li>
<li>better type stability when accessing metadata</li>
</ul>
