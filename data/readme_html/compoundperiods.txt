<p dir="auto">This package enhances the CompoundPeriod type defined within Dates (Dates.CompoundPeriod). A CompoundPeriod is formed by attaching (adding) two or more distinct Periods:</p>
<div class="highlight highlight-source-julia notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="julia&gt; using Dates

julia&gt; typeof( Year(1999) ), typeof( Hour(15) )
Year, Hour

julia&gt; typeof( Year(1999) + Hour(15) )
Dates.CompoundPeriod"><pre>julia<span class="pl-k">&gt;</span> <span class="pl-k">using</span> Dates

julia<span class="pl-k">&gt;</span> <span class="pl-c1">typeof</span>( <span class="pl-c1">Year</span>(<span class="pl-c1">1999</span>) ), <span class="pl-c1">typeof</span>( <span class="pl-c1">Hour</span>(<span class="pl-c1">15</span>) )
Year, Hour

julia<span class="pl-k">&gt;</span> <span class="pl-c1">typeof</span>( <span class="pl-c1">Year</span>(<span class="pl-c1">1999</span>) <span class="pl-k">+</span> <span class="pl-c1">Hour</span>(<span class="pl-c1">15</span>) )
Dates<span class="pl-k">.</span>CompoundPeriod</pre></div>
<p dir="auto">Note that <code>typeof( compound_period )</code> is shown as <code>CompoundPeriod</code> rather than <code>Dates.CompoundPeriod</code>. This lets you know that enhanced CompoundPeriods are in use.</p>
