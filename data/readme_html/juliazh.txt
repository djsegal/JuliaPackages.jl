<h3><a id="user-content-在本地编译pdf版本" class="anchor" aria-hidden="true" href="#在本地编译pdf版本"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path fill-rule="evenodd" d="M7.775 3.275a.75.75 0 001.06 1.06l1.25-1.25a2 2 0 112.83 2.83l-2.5 2.5a2 2 0 01-2.83 0 .75.75 0 00-1.06 1.06 3.5 3.5 0 004.95 0l2.5-2.5a3.5 3.5 0 00-4.95-4.95l-1.25 1.25zm-4.69 9.64a2 2 0 010-2.83l2.5-2.5a2 2 0 012.83 0 .75.75 0 001.06-1.06 3.5 3.5 0 00-4.95 0l-2.5 2.5a3.5 3.5 0 004.95 4.95l1.25-1.25a.75.75 0 00-1.06-1.06l-1.25 1.25a2 2 0 01-2.83 0z"></path></svg></a>在本地编译pdf版本</h3>
<p>首先你需要安装 <a href="https://mg.readthedocs.io/latexmk.html" rel="nofollow">latexmk</a> 和相应平台的 LaTeX 编译器XeLaTeX。之后确认已安装 CTex 套装。然后在这个package的根目录下，在命令行中运行</p>
<div class="highlight highlight-source-shell position-relative" data-snippet-clipboard-copy-content="julia --project=doc doc/make.jl pdf
"><pre>julia --project=doc doc/make.jl pdf</pre></div>
<p>或者，如果你已经安装了docker，可以执行</p>
<div class="highlight highlight-source-shell position-relative" data-snippet-clipboard-copy-content="julia --project=doc/ doc/make.jl pdf texplatform=docker
"><pre>julia --project=doc/ doc/make.jl pdf texplatform=docker</pre></div>
<p>就会在 <code>doc/build</code> 中获得 <code>Julia中文文档.pdf</code> 文件。</p>
