<p>A web client for the 7-Day Streamflow Forecasting Service  the Australian Bureau of Meteorology in the Julia programming language. The website at <a href="http://www.bom.gov.au/water/7daystreamflow" rel="nofollow">http://www.bom.gov.au/water/7daystreamflow</a> provides 3-day ahead streamflow forecasts for catchments across Australia.</p>
<h2><a id="user-content-installation" class="anchor" aria-hidden="true" href="#installation"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path fill-rule="evenodd" d="M7.775 3.275a.75.75 0 001.06 1.06l1.25-1.25a2 2 0 112.83 2.83l-2.5 2.5a2 2 0 01-2.83 0 .75.75 0 00-1.06 1.06 3.5 3.5 0 004.95 0l2.5-2.5a3.5 3.5 0 00-4.95-4.95l-1.25 1.25zm-4.69 9.64a2 2 0 010-2.83l2.5-2.5a2 2 0 012.83 0 .75.75 0 001.06-1.06 3.5 3.5 0 00-4.95 0l-2.5 2.5a3.5 3.5 0 004.95 4.95l1.25-1.25a.75.75 0 00-1.06-1.06l-1.25 1.25a2 2 0 01-2.83 0z"></path></svg></a>Installation</h2>
<p>The package can be installed with the Julia package manager. From the Julia REPL, type <code>]</code> to enter the Pkg REPL mode and run:</p>
<div class="highlight highlight-source-julia position-relative" data-snippet-clipboard-copy-content="pkg&gt; add SevenDayStreamflowForecasts
"><pre>pkg<span class="pl-k">&gt;</span> add SevenDayStreamflowForecasts</pre></div>
<p>If you want to install the package directly from its github development site,</p>
<div class="highlight highlight-source-julia position-relative" data-snippet-clipboard-copy-content="pkg&gt; add http://github.com/petershintech/SevenDayStreamflowForecasts.jl
"><pre>pkg<span class="pl-k">&gt;</span> add http<span class="pl-k">:</span><span class="pl-k">//</span>github<span class="pl-k">.</span>com<span class="pl-k">/</span>petershintech<span class="pl-k">/</span>SevenDayStreamflowForecasts<span class="pl-k">.</span>jl</pre></div>
<p>And load the package using the command:</p>
<div class="highlight highlight-source-julia position-relative" data-snippet-clipboard-copy-content="using SevenDayStreamflowForecasts
"><pre><span class="pl-k">using</span> SevenDayStreamflowForecasts</pre></div>
<h2><a id="user-content-site-information" class="anchor" aria-hidden="true" href="#site-information"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path fill-rule="evenodd" d="M7.775 3.275a.75.75 0 001.06 1.06l1.25-1.25a2 2 0 112.83 2.83l-2.5 2.5a2 2 0 01-2.83 0 .75.75 0 00-1.06 1.06 3.5 3.5 0 004.95 0l2.5-2.5a3.5 3.5 0 00-4.95-4.95l-1.25 1.25zm-4.69 9.64a2 2 0 010-2.83l2.5-2.5a2 2 0 012.83 0 .75.75 0 001.06-1.06 3.5 3.5 0 00-4.95 0l-2.5 2.5a3.5 3.5 0 004.95 4.95l1.25-1.25a.75.75 0 00-1.06-1.06l-1.25 1.25a2 2 0 01-2.83 0z"></path></svg></a>Site Information</h2>
<p>When you create an instance of the <code>SDF</code> structure, it downloads
site information.</p>
<div class="highlight highlight-source-julia position-relative" data-snippet-clipboard-copy-content="julia&gt; sdf = SDF();
"><pre>julia<span class="pl-k">&gt;</span> sdf <span class="pl-k">=</span> <span class="pl-c1">SDF</span>();</pre></div>
<p>Once it is instantiated, the fields of <code>sdf</code> should be considered as read-only so don't try to change any values of the fields.</p>

<p><code>get_forecasts()</code> returns the most recent forecast data as <code>DataFrames.DataFrame</code>. The method needs a site ID.
The returned data have precentiles of ensemble forecasts and historical reference (streamflow climatology) along with the recent observation data.
The site ID of a station can be found in <code>awrc_id</code> columne of <code>sdf.sites</code>.
A new column called <code>DateTime</code> is created from <code>Time</code> strings but without considering the time zone.</p>
