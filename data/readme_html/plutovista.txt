<p dir="auto">Plot library for Pluto notebooks based on <a href="https://plotly.com/javascript/" rel="nofollow">plotly.js</a> for 1D data
and <a href="https://kitware.github.io/vtk-js/index.html" rel="nofollow">vtk.js</a> (thus using WebGL)  for 2/3D data.</p>
<p dir="auto">It uses the Pluto <a href="https://github.com/fonsp/Pluto.jl/pull/1124" data-hovercard-type="pull_request" data-hovercard-url="/fonsp/Pluto.jl/pull/1124/hovercard"><g-emoji class="g-emoji" alias="tipping_hand_person" fallback-src="https://github.githubassets.com/images/icons/emoji/unicode/1f481.png">üíÅ</g-emoji> API to make objects available inside JS</a>
to pass plot data from Julia to HTML5.</p>
<p dir="auto">It can serve as a backend for <a href="https://github.com/j-fu/GridVisualize.jl">GridVisualize.jl</a>.</p>

<a href="https://j-fu.github.io/PlutoVista.jl/dev/plutovista.html" rel="nofollow">html</a></p>
