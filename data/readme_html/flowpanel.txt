<ul dir="auto">
<li>Structured mesh generation through space transformation, lofting, or body of revolution</li>
<li>Source-panel or vortex-ring solver for non-lifting bodies</li>
<li>Vortex-ring solver for lifting bodies with rigid wake</li>
<li>Low-memory allocation</li>
<li>Direct and iterative Krylov (GMRES) <a href="http://flow.byu.edu/FLOWPanel.jl/dev/examples/sweptwing-solver/" rel="nofollow">solvers</a></li>
<li>Automatic differentiation for gradient-based optimization (supports <a href="https://github.com/JuliaDiff/ForwardDiff.jl">ForwardDiff</a>, <a href="https://github.com/JuliaDiff/ReverseDiff.jl">ReverseDiff</a>, and <a href="https://github.com/byuflowlab/ImplicitAD.jl">ImplicitAD</a>)</li>
</ul>
<p dir="auto">Developed and tested in Julia v1.6.</p>

<ol dir="auto">
<li>Download and install <a href="https://julialang.org/" rel="nofollow">Julia</a></li>
<li>In the Julia REPL: <code>] add FLOWPanel</code></li>
<li>(optional) For visualization, install <a href="https://www.paraview.org/" rel="nofollow">Paraview</a> and make sure that <a href="https://flow.byu.edu/FLOWUnsteady/tutorials/installation-instructions/#Paraview" rel="nofollow">it is callable from the terminal</a> typing <code>paraview</code></li>
</ol>
<p dir="auto">Copy and paste any of <a href="http://flow.byu.edu/FLOWPanel.jl/dev/examples/sweptwing-4p2aoa/" rel="nofollow">the examples</a> directly in the REPL to run your first simulation</p>
