<p dir="auto">This package provides support for CNC controllers running the <a href="https://github.com/synthetos/TinyG/wiki">TinyG firmware</a>,
principally developed by <a href="http://www.synthetos.com/" rel="nofollow">Synthetos</a>.</p>
<p dir="auto">Goals:</p>
<ul dir="auto">
<li>Discover and connect to TinyG devices</li>
<li>Firmware updating</li>
<li>Configuration load and save</li>
<li>High-level Command API</li>
</ul>
