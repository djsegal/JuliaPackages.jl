<p><strong>MIPLearn</strong> is an extensible open-source framework for solving discrete optimization problems using a combination of Mixed-Integer Linear Programming (MIP) and Machine Learning (ML). See the <a href="https://github.com/ANL-CEEESA/MIPLearn">main repository</a> for more information. This repository holds an experimental Julia interface for the package.</p>
