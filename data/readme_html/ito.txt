<p dir="auto">This package is now deprecated. This package started by emulating functionality from Quantlib, but the <a href="https://github.com/pazzo83/QuantLib.jl">Quantlib.jl</a> package now contains a much larger port of Quantlib. Some of the longer terms goals of this package are now implemented in <a href="http://juliacomputing.com/products/juliafin/miletus.html" rel="nofollow">Miletus</a>, a contract definition and modelling language from Julia Computing.</p>

<p dir="auto">Ito is a collection of Julia modules containing algorithms for efficient quantitative finance.
It includes functions and frameworks for holiday calendars, day count conventions, term structures,
stochastic processes and more.</p>
<p dir="auto">A lot of the functionality is work in progress, but there are hopefully some useful bits already.</p>
<p dir="auto">Detailed documentation is available at <a href="http://aviks.github.com/Ito.jl/">http://aviks.github.com/Ito.jl/</a></p>
