<p>This package is now deprecated. This package started by emulating functionality from Quantlib, but the <a href="https://github.com/pazzo83/QuantLib.jl">Quantlib.jl</a> package now contains a much larger port of Quantlib. Some of the longer terms goals of this package are now implemented in <a href="http://juliacomputing.com/products/juliafin/miletus.html" rel="nofollow">Miletus</a>, a contract definition and modelling language from Julia Computing.</p>
<h2><a id="user-content-ito-an-open-source-toolkit-for-financial-computing-in-julia" class="anchor" aria-hidden="true" href="#ito-an-open-source-toolkit-for-financial-computing-in-julia"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path fill-rule="evenodd" d="M4 9h1v1H4c-1.5 0-3-1.69-3-3.5S2.55 3 4 3h4c1.45 0 3 1.69 3 3.5 0 1.41-.91 2.72-2 3.25V8.59c.58-.45 1-1.27 1-2.09C10 5.22 8.98 4 8 4H4c-.98 0-2 1.22-2 2.5S3 9 4 9zm9-3h-1v1h1c1 0 2 1.22 2 2.5S13.98 12 13 12H9c-.98 0-2-1.22-2-2.5 0-.83.42-1.64 1-2.09V6.25c-1.09.53-2 1.84-2 3.25C6 11.31 7.55 13 9 13h4c1.45 0 3-1.69 3-3.5S14.5 6 13 6z"></path></svg></a>Ito, an open source toolkit for financial computing in Julia</h2>
<p>Ito is a collection of Julia modules containing algorithms for efficient quantitative finance.
It includes functions and frameworks for holiday calendars, day count conventions, term structures,
stochastic processes and more.</p>
<p>A lot of the functionality is work in progress, but there are hopefully some useful bits already.</p>
<p>Detailed documentation is available at <a href="http://aviks.github.com/Ito.jl/">http://aviks.github.com/Ito.jl/</a></p>
