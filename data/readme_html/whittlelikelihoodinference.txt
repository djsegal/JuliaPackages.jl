<p dir="auto">A julia package for Whittle and debiased Whittle likelihood inference. Provides the following functionality:</p>
<ul dir="auto">
<li>Simulating Gaussian processes with a given autocovariance.</li>
<li>Computation of the (negative) Whittle likelihood, its gradient and its Hessian (including with an arbitrary taper).</li>
<li>Computation of the (negative) debiased Whittle likelihood, its gradient and its Hessian (including with an arbitrary taper).</li>
<li>Support for reverse-mode automatic differentiation via <a href="https://github.com/JuliaDiff/ChainRules.jl"><em>ChainRules.jl</em></a>.</li>
<li>A fit function which uses solvers from <a href="https://github.com/JuliaNLSolvers/Optim.jl"><em>Optim.jl</em></a> with the option to use dpss tapers from <a href="https://github.com/JuliaDSP/DSP.jl"><em>DSP.jl</em></a>.</li>
<li>Plotting recipes for second-order properties of interest, including the spectral density function and autocovariance.</li>
<li>Approximation of the autocovariance from the spectral density function (and for gradients and Hessians).</li>
<li>Example models including 1D OU and 1D Matern, 2D correlated OU process and arbitrary dimensional Matern.</li>
<li>Basic non-parametric estimators including the periodogram and Bartlett's method with plotting recipes.</li>
</ul>
