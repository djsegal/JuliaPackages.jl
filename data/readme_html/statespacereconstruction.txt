<p>Julia package for state space reconstruction and partitioning. This package provides necessary functionality for the <a href="https://github.com/kahaaga/PerronFrobenius.jl">PerronFrobenius.jl</a>,
<a href="https://github.com/kahaaga/TransferEntropy.jl">TransferEntropy.jl</a> and
<a href="https://github.com/kahaaga/CausalityTools.jl">CausalityTools.jl</a> packages, which require fully flexible state space reconstructions, and partitioning tools for rectangular and simplex partitions.</p>
<p>This package works seamlessly with <code>Dataset</code> instances from <a href="https://github.com/JuliaDynamics/DynamicalSystems.jl">DynamicalSystems.jl</a>, which also provide methods for state space reconstruction.</p>
<p>See the <code>CausalityTools</code> documentation for more information.</p>
