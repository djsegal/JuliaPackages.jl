<p dir="auto">The <code>JUGRNModelGenerator.jl</code> package is a code generation system that transforms simple <a href="https://www.json.org/json-en.html" rel="nofollow">JSON</a> descriptions of the connectivity of gene regulatory networks into model code written in the <a href="http://julialang.org" rel="nofollow">Julia</a> programming language. <code>JUGRNModelGeneratorModelGenerator.jl</code> has been used in the publications:</p>
<ol dir="auto">
<li><a href="https://www.nature.com/articles/s41598-017-14523-5" rel="nofollow">Tasseff R, Jensen H, Congleton J, Dai W, Rogers K, Sagar A, Yen A and J. Varner (2017) An Effective Model of the Retinoic Acid Induced Differentiation Program, Sci Reports, 7:14327 doi:10.1038/s41598-017-14523-5</a></li>
<li><a href="http://journals.plos.org/ploscompbiol/article?id=10.1371/journal.pcbi.1005251" rel="nofollow">Gould R, Bassen DM, Chakrabarti A, Varner JD and Butcher J (2016) Population Heterogeneity in the Epithelial to Mesenchymal Transition Is Controlled by NFAT and Phosphorylated Sp1. PLoS Comput Biol 12(12): e1005251. doi:10.1371/journal.pcbi.1005251</a></li>
<li><a href="https://www.frontiersin.org/articles/10.3389/fbioe.2020.539081/full" rel="nofollow">Adhikari A, Vilkhovoy M, Vadhin S, Lim HE and Varner JD (2020) Effective Biophysical Modeling of Cell Free Transcription and Translation Processes. Front. Bioeng. Biotechnol. 8:539081. doi: 10.3389/fbioe.2020.539081</a></li>
</ol>
<p dir="auto">to generate gene regulatory network code.</p>

<p dir="auto"><code>JUGRNModelGenerator.jl</code> is organized as a <a href="http://julialang.org" rel="nofollow">Julia</a> package which can be installed in the <code>package mode</code> of Julia.</p>
<p dir="auto">Start of the <a href="https://docs.julialang.org/en/v1/stdlib/REPL/index.html" rel="nofollow">Julia REPL</a> and enter the <code>package mode</code> using the <code>]</code> key (to get back press the <code>backspace</code> or <code>^C</code> keys). Then, at the prompt enter:</p>
<div class="snippet-clipboard-content notranslate position-relative overflow-auto" data-snippet-clipboard-copy-content="(v1.1) pkg&gt; add JUGRNModelGenerator"><pre class="notranslate"><code>(v1.1) pkg&gt; add JUGRNModelGenerator
</code></pre></div>
<p dir="auto">This will install the <code>JUGRNModelGenerator.jl</code> package and the other required packages. <code>JUGRNModelGenerator.jl</code> requires Julia 1.6.x and above.
<code>JUGRNModelGenerator.jl</code> is open source. You can download this repository as a zip file, or clone or pull it by using the command (from the command-line):</p>
