You can still download and use this package,
but consider instead using the new packages with updated modules</strong></h2>
<p>This module provides tools to read, write, and process
seismic data.
The documentation can be found <a href="http://seismicjulia.github.io/Seismic.jl" rel="nofollow">here</a>
and <a href="http://seismic.physics.ualberta.ca" rel="nofollow">here</a>. At the moment, it is updated and tested against Julia 0.6.</p>
<h2><a id="user-content-installation" class="anchor" aria-hidden="true" href="#installation"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path fill-rule="evenodd" d="M7.775 3.275a.75.75 0 001.06 1.06l1.25-1.25a2 2 0 112.83 2.83l-2.5 2.5a2 2 0 01-2.83 0 .75.75 0 00-1.06 1.06 3.5 3.5 0 004.95 0l2.5-2.5a3.5 3.5 0 00-4.95-4.95l-1.25 1.25zm-4.69 9.64a2 2 0 010-2.83l2.5-2.5a2 2 0 012.83 0 .75.75 0 001.06-1.06 3.5 3.5 0 00-4.95 0l-2.5 2.5a3.5 3.5 0 004.95 4.95l1.25-1.25a.75.75 0 00-1.06-1.06l-1.25 1.25a2 2 0 01-2.83 0z"></path></svg></a>Installation</h2>
<p>To use this package you must first install the <a href="http://julialang.org/downloads/" rel="nofollow">Julia</a> programming language. Once you have Julia you can install the Seismic package by typing <code>Pkg.add("Seismic")</code> on the Julia command line and then run <code>Pkg.checkout("Seismic")</code> to stay updated to the last version in this repository.</p>
