<p>This is a 3D element-by-element program to compute the theoretical performance of a bottom hole assembly.</p>
<p>This is the base version of BHAtp and is intended to be used by CasePkgGenerator.jl to construct a new <code>case</code> package.</p>
<p>CasePkgGenerator will be updates to use PkgSkeleton.jl.</p>
