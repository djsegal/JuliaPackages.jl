<p dir="auto">This package provides plotting capabilities for item response models implementing the <a href="https://github.com/JuliaPsychometrics/AbstractItemResponseModels.jl">AbstractItemResponseModels.jl</a> interface.</p>
<p dir="auto">ItemResponsePlots.jl leverages the <a href="https://github.com/MakieOrg/Makie.jl">Makie.jl</a> ecosystem, making it easy to extend basic figures and combine them in complex plots.</p>

<p dir="auto">After sucessfull installation you can start plotting results of your item response model. Prerequisite is a fitted <code>ItemResponseModel</code>, e.g. via <a href="https://github.com/JuliaPsychometrics/RaschModels.jl">RaschModels.jl</a>.</p>
<div class="highlight highlight-source-julia notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="using RaschModels

data = rand(0:1, 100, 5)
rasch = fit(RaschModel, data, CML())"><pre><span class="pl-k">using</span> RaschModels

data <span class="pl-k">=</span> <span class="pl-c1">rand</span>(<span class="pl-c1">0</span><span class="pl-k">:</span><span class="pl-c1">1</span>, <span class="pl-c1">100</span>, <span class="pl-c1">5</span>)
rasch <span class="pl-k">=</span> <span class="pl-c1">fit</span>(RaschModel, data, <span class="pl-c1">CML</span>())</pre></div>
<p dir="auto">Once the parameters are estimated, simply call your desired plotting function.</p>
<p dir="auto">For example, item characteristic curves are implemented by the <code>item_characteristic_curve</code> function.
To plot the item characteristic curve for the first item, call</p>
<div class="highlight highlight-source-julia notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="item_characteristic_curve(rasch, 1)"><pre><span class="pl-c1">item_characteristic_curve</span>(rasch, <span class="pl-c1">1</span>)</pre></div>
<p dir="auto">All plotting functions in ItemResponsePlots.jl implement a variety of customization options. For details see the relevant plotting functions help page (e.g. <code>?item_characteristic_curve</code>).</p>

<p dir="auto">Currently ItemResponsePlots supports low-level plotting recipes for</p>
<ul dir="auto">
<li>Item characteristic curves</li>
<li>Item information curves</li>
<li>Test characteristic / expected score curves</li>
<li>Test information curves</li>
</ul>
<p dir="auto">as well as high-level figures for</p>
<ul dir="auto">
<li>items (item characteristic curve + item information curve)</li>
<li>(sub)tests (expected scores + test information curve)</li>
</ul>
