<p dir="auto">A Julia implementation of the Python package <a href="https://github.com/ajhynes7/scikit-spatial">scikit-spatial</a>.</p>
<p dir="auto">This package provides spatial types and computations between them.</p>
<p dir="auto">The following types are provided:</p>
<ul dir="auto">
<li>Line</li>
<li>Plane</li>
<li>Circle</li>
<li>Sphere</li>
</ul>
<p dir="auto">Most of the computations fall into the following categories:</p>
<ul dir="auto">
<li>Measurement</li>
<li>Comparison</li>
<li>Projection</li>
<li>Intersection</li>
<li>Transformation</li>
</ul>
