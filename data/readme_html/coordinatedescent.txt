<p dir="auto">Implements coordinate descent for a smooth function plus penalty that decomposes across coordinates.</p>
<p dir="auto">Curently a naive version of the active-set coordinate descent is implemented that works for L1 and weighted L1 penalty.</p>
<p dir="auto">Examples:</p>
<ul dir="auto">
<li>Lasso</li>
<li>Sqrt-Lasso</li>
<li>Scaled-Lasso</li>
</ul>
<p dir="auto">Package depends on <a href="https://github.com/mlakolar/ProximalBase.jl">ProximalBase.jl</a></p>
