<p dir="auto">A julia package for working with coupled fields. This is work in progress.
The main function <code>gradvecfield</code> calculates the gradient vector or gradient matrix for each instance of the coupled fields.</p>
<p dir="auto">For 𝒀 = g(𝑿), <code>CoupledFields.gradvecfield([a b], X, Y, kernelpars)</code> returns 𝑛 gradient matrices, for 𝑛 random points in 𝑿.
For parameters [𝑎 𝑏]: 𝑎 is a smoothness parameter, and 𝑏 is a ridge parameter.</p>
<div class="highlight highlight-source-julia notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="using CoupledFields

 kernelpars = GaussianKP(X)
 ∇g = gradvecfield([0.5 -7], X, Y[:,1:1], kernelpars)"><pre><span class="pl-k">using</span> CoupledFields

<p dir="auto">Also CoupledFields doesn’t require a closed-form function, it can be used if you have only the observed fields 𝑿 and 𝒀.</p>
