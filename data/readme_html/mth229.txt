<p>Helper files for using <code>Julia</code> with MTH229.</p>
<p>This is installed with</p>
<pre lang="noeval"><code>] add https://github.com/mth229/MTH229.jl
</code></pre>
<p>This should also install several other packages we make use of (<code>Roots</code>, <code>Plots</code>, <code>SymPy</code>)</p>
<p>To use this package we have:</p>
<pre><code>using MTH229
</code></pre>
<p>To find out what is in the package read the help page for the package:</p>
<pre><code>?MTH229
</code></pre>
<p>If installation fails, then this command will pull in the necessary functions:</p>
<pre lang="noeval"><code>include(download("https://raw.githubusercontent.com/mth229/MTH229.jl/master/src/229.jl"))
</code></pre>
