<p dir="auto">There are currently three supported coordinate systems. The following
immutable types are used to represent coordinates in each system:</p>
<ul dir="auto">
<li><code>ICRSCoords</code>: ICRS coordinates system</li>
<li><code>GalCoords</code>: Galactic coordinates system</li>
<li><code>FK5Coords</code>: FK5 coordinates system (with arbitrary equinox)</li>
<li><code>EclipticCoords</code>: Ecliptic coordinates system</li>
</ul>
