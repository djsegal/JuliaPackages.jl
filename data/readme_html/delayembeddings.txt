<p dir="auto">A Julia package that provides a generic interface for performing delay coordinate embeddings, as well as cutting edge algorithms for creating optimal embeddings given some data.
It can be used as a standalone package, or as part of
<a href="https://juliadynamics.github.io/DynamicalSystems.jl/dev/" rel="nofollow">DynamicalSystems.jl</a>.</p>
<p dir="auto">To install it, run <code>import Pkg; Pkg.add("DelayEmbeddings")</code>.</p>
<p dir="auto">All further information is provided in the documentation, which you can either find <a href="https://juliadynamics.github.io/DelayEmbeddings.jl/stable/" rel="nofollow">online</a> or build locally by running the <code>docs/make.jl</code> file.</p>
