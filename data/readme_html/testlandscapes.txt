<p dir="auto"><a href="https://gideonsimpson.github.io/TestLandscapes.jl/dev" rel="nofollow"><img src="https://camo.githubusercontent.com/7fcec4b2d3ab291529fce8ef6a4fcd4129a0683b2f5d5fe2f5c648f02db8b616/68747470733a2f2f696d672e736869656c64732e696f2f62616467652f646f63732d6465762d626c75652e737667" alt="" data-canonical-src="https://img.shields.io/badge/docs-dev-blue.svg" style="max-width: 100%;"></a></p>
<p dir="auto">Julia implementations of basic potential energy landscapes for testing sampling,
optimization, etc.</p>
<p dir="auto">This package can be added with the command:</p>
<div class="snippet-clipboard-content notranslate position-relative overflow-auto" data-snippet-clipboard-copy-content="(@v1.XYZ) pkg&gt; add TestLandscapes
"><pre class="notranslate"><code>(@v1.XYZ) pkg&gt; add TestLandscapes

</code></pre></div>
<p dir="auto">Currently, these landscapes are in dimensions one and two, but they allow for
exploration of multiple minima, along with energetic and entropic bottlenecks.</p>
<p dir="auto">These codes do <strong>not</strong> include derivatives.  These can be obtained using ForwardDiff,
<a href="https://github.com/JuliaDiff/ForwardDiff.jl">https://github.com/JuliaDiff/ForwardDiff.jl</a></p>
