<p><a href="https://travis-ci.org/BBN-Q/QuantumInfo.jl" rel="nofollow"><img src="https://camo.githubusercontent.com/5feb10c8ef6772e92eb4923ead22fc5465e25c11/68747470733a2f2f7472617669732d63692e6f72672f42424e2d512f5175616e74756d496e666f2e6a6c2e7376673f6272616e63683d6d6173746572" alt="Build Status" data-canonical-src="https://travis-ci.org/BBN-Q/QuantumInfo.jl.svg?branch=master" style="max-width:100%;"></a></p>
<p>This Julia package contains functions that are convenient for common
calculation in quantum information processing.</p>
<p>Install it in Julia v0.6 with the following command:</p>
<pre><code>  julia&gt; Pkg.add("QuantumInfo")
</code></pre>
<p>For Julia v1.0:</p>
<pre><code>  (v1.0) pkg&gt; add QuantumInfo
</code></pre>
