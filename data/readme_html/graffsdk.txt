<p>If you're starting from a completely fresh Linux image (e.g. Docker), you'll need some essential tools, such as make, unzip, and patch. You can install these independently, or you can just pull them with this command:</p>
<div class="highlight highlight-source-julia"><pre>sudo apt<span class="pl-k">-</span>get install unzip patch build<span class="pl-k">-</span>essential</pre></div>
<p>That should get you started! (You probably don't need the full 216Mb build-essential, but it's probably going to crop up as a dependency later during your endeavours)</p>
