<p dir="auto">This package is intended for developers of data transforms such as
the maintainers of <a href="https://github.com/JuliaML/TableTransforms.jl">TableTransforms.jl</a>.
It defines a simple API and basic transforms that work with any type of object such as
the <code>Identity</code> and <code>Sequential</code> transforms. Check dependant packages for more information.</p>
