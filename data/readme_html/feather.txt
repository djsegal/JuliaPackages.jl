<p>Feather.jl reads an older feather format now known as "Feather v1".  The current standard,
Feather v2, is simply the <a href="https://arrow.apache.org/" rel="nofollow">apache arrow</a> format written to
disk.  As such, you're probably looking for
<a href="https://github.com/JuliaData/Arrow.jl">Arrow.jl</a> which will allow you to read and write
Feather v2, but not feather v1.  We suggest that you upgrade any data you might have in
the v1 format to v2 using this package together with Arrow.jl, or with
<a href="https://pypi.org/project/pyarrow/" rel="nofollow"><code>pyarrow</code></a> which maintains support for the legacy
feather format.</p>
<p>Please note that the maintainers of this package have moved on to the new format, so it is
unlikely to receive updates and there may not be anyone available to review PR's.</p>
