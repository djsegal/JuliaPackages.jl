<p><code>Merlin</code> is a deep learning framework written in <a href="http://julialang.org/" rel="nofollow">Julia</a>.</p>
<p>It aims to provide a fast, flexible and compact deep learning library for machine learning.</p>
<p><code>Merlin</code> is tested against Julia <code>1.0</code> on Linux, OS X, and Windows (x64).<br>
It runs on CPU and CUDA GPU.</p>
