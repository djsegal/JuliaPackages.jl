<p>For more information see the <a href="https://vidriotech.gitlab.io/scanimagetiffreader-julia/" rel="nofollow">documentation</a>.</p>
<p>The ScanImageTiffReader is a <a href="https://julialang.org" rel="nofollow">Julia</a> library for extracting data from <a href="https://en.wikipedia.org/wiki/Tagged_Image_File_Format" rel="nofollow">Tiff</a> and <a href="http://bigtiff.org/" rel="nofollow">BigTiff</a> files recorded using <a href="http://scanimage.org" rel="nofollow">ScanImage</a>.  It is a very fast tiff reader and provides access to ScanImage-specific metadata.  It should read most tiff files, but as of now we don't support compressed or tiled data. This is the <a href="https://julialang.org" rel="nofollow">Julia</a> interface.  It is also available as a <a href="https://vidriotech.gitlab.io/scanimagetiffreader-matlab/" rel="nofollow">Matlab</a>, <a href="https://vidriotech.gitlab.io/scanimagetiffreader-python/" rel="nofollow">Python</a>,  or <a href="https://vidriotech.gitlab.io/scanimage-tiff-reader" rel="nofollow">C library</a>.  There's also a <a href="https://vidriotech.gitlab.io/scanimage-tiff-reader" rel="nofollow">command-line interface</a>.</p>
<p>Both <a href="http://scanimage.org" rel="nofollow">ScanImage</a> and this reader are products of <a href="http://vidriotechnologies.com/" rel="nofollow">Vidrio Technologies</a>.  If you
have questions or need support feel free to <a href="https://gitlab.com/vidriotech/scanimagetiffreader-julia/issues" rel="nofollow">submit an issue</a> or <a href="https://vidriotechnologies.com/contact-support/" rel="nofollow">contact us</a>.</p>
