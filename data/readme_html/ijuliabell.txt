<p>This adds the <code>@bell</code> macro which can be used in Jupyter notebooks to play a .wav file once a command completes. This is a Julia port of <a href="https://github.com/neighthan/jupyter-utils/blob/master/jupyter_utils/ipy_cell_completion_bell.py">ipy_cell_completion_bell</a>.</p>

<h2><a id="user-content-custom-wav-file" class="anchor" aria-hidden="true" href="#custom-wav-file"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path fill-rule="evenodd" d="M7.775 3.275a.75.75 0 001.06 1.06l1.25-1.25a2 2 0 112.83 2.83l-2.5 2.5a2 2 0 01-2.83 0 .75.75 0 00-1.06 1.06 3.5 3.5 0 004.95 0l2.5-2.5a3.5 3.5 0 00-4.95-4.95l-1.25 1.25zm-4.69 9.64a2 2 0 010-2.83l2.5-2.5a2 2 0 012.83 0 .75.75 0 001.06-1.06 3.5 3.5 0 00-4.95 0l-2.5 2.5a3.5 3.5 0 004.95 4.95l1.25-1.25a.75.75 0 00-1.06-1.06l-1.25 1.25a2 2 0 01-2.83 0z"></path></svg></a>Custom .wav file</h2>
<p>The package includes a .wav from <a href="https://freewavesamples.com/files/E-Mu-Proteus-FX-CosmoBel-C3.wav" rel="nofollow">Free Wave Samples</a> (slightly shortened to reduce file size), but you can replace the installed <code>bell.wav</code> file with another to use it instead (the file must be named <code>bell.wav</code>, though, and be in the same directory, unless you change the source code of the package).</p>
