<p dir="auto">This adds the <code>@bell</code> macro which can be used in Jupyter notebooks to play a .wav file once a command completes. This is a Julia port of <a href="https://github.com/neighthan/jupyter-utils/blob/master/jupyter_utils/ipy_cell_completion_bell.py">ipy_cell_completion_bell</a>.</p>

<p dir="auto">The package includes a .wav from <a href="https://freewavesamples.com/files/E-Mu-Proteus-FX-CosmoBel-C3.wav" rel="nofollow">Free Wave Samples</a> (slightly shortened to reduce file size), but you can replace the installed <code>bell.wav</code> file with another to use it instead (the file must be named <code>bell.wav</code>, though, and be in the same directory, unless you change the source code of the package).</p>
