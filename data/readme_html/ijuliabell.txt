<p>This adds the <code>@bell</code> macro which can be used in Jupyter notebooks to play a .wav file once a command completes. This is a Julia port of <a href="https://github.com/neighthan/jupyter-utils/blob/master/jupyter_utils/ipy_cell_completion_bell.py">ipy_cell_completion_bell</a>.</p>
