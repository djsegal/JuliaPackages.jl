<p dir="auto"><em>A Julia package for Bayesian inference with transport maps</em></p>
<p dir="auto"><a target="_blank" rel="noopener noreferrer" href="logo.png"><img src="logo.png" alt="logo" style="max-width: 100%;"></a></p>
<p dir="auto"><a href="https://github.com/mleprovost/TransportBasedInference.jl/actions"><img src="https://github.com/mleprovost/TransportBasedInference.jl/workflows/CI/badge.svg" alt="Build Status" style="max-width: 100%;"></a></p>
<p dir="auto">The objective of this package is to allow for easy and fast resolution of Bayesian inference problems using transport maps. The package provides tools for:</p>
<ul dir="auto">
<li>joint and conditional density estimation from limited samples of the target distribution using the adaptive transport map algorithm developed by Baptista et al. <sup><a href="#user-content-fn-1-02edf600593222163996015b2079c02a" id="user-content-fnref-1-02edf600593222163996015b2079c02a" data-footnote-ref="" aria-describedby="footnote-label">1</a></sup>.</li>
<li>sequential inference for state-space models using one of the following algorithms: the (localized) stochastic ensemble Kalman filter (Evensen <sup><a href="#user-content-fn-2-02edf600593222163996015b2079c02a" id="user-content-fnref-2-02edf600593222163996015b2079c02a" data-footnote-ref="" aria-describedby="footnote-label">2</a></sup>), the ensemble transform Kalman filter (Bishop et al. <sup><a href="#user-content-fn-3-02edf600593222163996015b2079c02a" id="user-content-fnref-3-02edf600593222163996015b2079c02a" data-footnote-ref="" aria-describedby="footnote-label">3</a></sup>) and a nonlinear generalization of the stochastic ensemble Kalman filter (Spantini et al. <sup><a href="#user-content-fn-4-02edf600593222163996015b2079c02a" id="user-content-fnref-4-02edf600593222163996015b2079c02a" data-footnote-ref="" aria-describedby="footnote-label">4</a></sup>).</li>
</ul>
