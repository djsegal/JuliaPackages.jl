<p dir="auto">RainFARM.jl is a Julia library and a collection of command-line interface tools implementing the RainFARM (Rainfall Filtered Autoregressive Model) stochastic precipitation downscaling method. Adapted for climate downscaling according to (D'Onofrio et al. 2018) and with fine-scale orographic weights (Terzago et al. 2018).</p>
<p dir="auto">RainFARM (Rebora et al. 2006) is a metagaussian stochastic downscaling procedure based on the extrapolation of the coarse-scale Fourier power spectrum of a spatio-temporal precipitation field to small scales.</p>

<p dir="auto">You will need an implementation of the <a href="https://julialang.org/" rel="nofollow">Julia language</a> on your machine.</p>
<p dir="auto">RainFARM is a registered package, so to install it just launch julia and from the REPL type the following:</p>
<ul dir="auto">
<li>in Julia &gt;= 0.7: <code>] add RainFARM</code></li>
<li>In Julia 0.6: <code>Pkg.add("RainFARM")</code></li>
<li>If you wish to try the very latest development features from this repository do:
<code>Using Pkg; Pkg.clone("https://github.com/jhardenberg/RainFARM.jl") </code>
(the <code>using Pkg</code>step is only needed for Julia versions &gt;= 0.7)</li>
</ul>
<p dir="auto">Test from the julia REPL calling <code>using RainFARM</code></p>
<p dir="auto">In the <code>tools</code> subdirectory (under .julia in your home) you will find the command line tools. Link or copy these to somewhere in you path in order to use them.</p>
