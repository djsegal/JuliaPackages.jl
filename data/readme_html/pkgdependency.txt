<p dir="auto"><a href="https://peng1999.github.io/PkgDependency.jl/dev/" rel="nofollow"><img src="https://camo.githubusercontent.com/56f8252ba8e9d3f0b810769543f77823d2fe031ce560d4c2d69fb1fcad800383/68747470733a2f2f696d672e736869656c64732e696f2f62616467652f646f63732d6c61746573742d626c75652e737667" alt="" data-canonical-src="https://img.shields.io/badge/docs-latest-blue.svg" style="max-width: 100%;"></a></p>
<p dir="auto">Print dependency tree of a project or package.</p>
<p dir="auto">Supported features include:</p>
<ul dir="auto">
<li>Reverse dependency tree</li>
<li>Show compat info in tree</li>
<li>Show repo link in tree (needs at least Julia 1.7)</li>
</ul>
