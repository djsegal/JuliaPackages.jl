<p dir="auto"><strong>RandomNumbers.jl</strong> is a package of <a href="http://julialang.org/" rel="nofollow">Julia</a>, in which several random number generators (RNGs)
are provided.</p>
<p dir="auto">If you use the Intel Math Kernel Library (MKL), <a href="https://github.com/JuliaRandom/VSL.jl"><code>VSL.jl</code></a> is also a good
choice for random number generation.</p>

<p dir="auto">There are four RNG families in this package:</p>
<ul dir="auto">
<li><a href="http://juliarandom.github.io/RandomNumbers.jl/stable/man/pcg/" rel="nofollow">PCG</a>:
A new family of RNG, based on <em>linear congruential generators</em>, using a <em>permuted function</em> to produce much
more random output.</li>
<li><a href="http://juliarandom.github.io/RandomNumbers.jl/stable/man/mersenne-twisters/" rel="nofollow">Mersenne Twister</a>:
The most widely used RNG, with long period.</li>
<li><a href="http://juliarandom.github.io/RandomNumbers.jl/stable/man/random123/" rel="nofollow">Random123</a>:
A family of good-performance <em>counter-based</em> RNG.</li>
<li><a href="http://juliarandom.github.io/RandomNumbers.jl/stable/man/xorshifts/" rel="nofollow">Xorshift</a>:
A class of RNG based on <em>exclusive or</em> and <em>bit shift</em>.</li>
</ul>
<p dir="auto">Note that <code>Random123</code> is now made a separate package as <a href="https://github.com/JuliaRandom/Random123.jl">Random123.jl</a>.
You can still use your old code with <code>RandomNumbers.Random123</code> as long as you import <code>Random123</code> manually.</p>
