<p dir="auto">This Work in Progress cadCAD implementation in Julia uses a Meta Engine approach. This represents a development initiative for a high performance engine for experienced modelers to run large experiments.</p>
<p dir="auto">A distinguishing feature of the Julia implementation is the focus on ease of integration of domain-specific packages for scientific computing into spaces.</p>

<p dir="auto">cadCAD.jl is a standallone CLI app, so it needs to be compiled to be used. To do so:</p>
<ol dir="auto">
<li>Install the latest stable release of <a href="https://julialang.org/downloads/" rel="nofollow">Julia</a>;</li>
<li>Clone this repository;</li>
<li>Do, from the root of the repository: <code>julia --project deps/build.jl app</code></li>
</ol>
<p dir="auto">This will create a <code>build</code> folder that contains a <code>cadCAD</code> folder that is the compiled app. You can <code>tar</code> the folder to distriute the app or just relocate it anywhere on your system. The <code>cadCAD</code> executable is in the <code>build/cadCAD/bin</code> folder.</p>
