<p dir="auto">LeafletJS maps for Julia.</p>
<p dir="auto">This package integrates with WebIO.jl to render leaflet maps for outputs like
Blink.jl, Mux.jl, and for Jupyter notebooks.</p>
<p dir="auto">All <a href="https://github.com/JuliaGeo/GeoInterface.jl">GeoInterface.jl</a> compatible geometries can be displayed as layers.</p>
<p dir="auto">A basic example, where we use GADM to download a country boundary shapefile,
and plot them over the CARTO <code>:dark_nolabels</code> base layers.</p>

m = Leaflet.Map(; layers, provider, zoom=3, height=1000, center=[30.0, 120.0]);
