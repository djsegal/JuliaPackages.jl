<h3><a id="user-content-usage" class="anchor" aria-hidden="true" href="#usage"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path fill-rule="evenodd" d="M4 9h1v1H4c-1.5 0-3-1.69-3-3.5S2.55 3 4 3h4c1.45 0 3 1.69 3 3.5 0 1.41-.91 2.72-2 3.25V8.59c.58-.45 1-1.27 1-2.09C10 5.22 8.98 4 8 4H4c-.98 0-2 1.22-2 2.5S3 9 4 9zm9-3h-1v1h1c1 0 2 1.22 2 2.5S13.98 12 13 12H9c-.98 0-2-1.22-2-2.5 0-.83.42-1.64 1-2.09V6.25c-1.09.53-2 1.84-2 3.25C6 11.31 7.55 13 9 13h4c1.45 0 3-1.69 3-3.5S14.5 6 13 6z"></path></svg></a>Usage:</h3>
<p>It can be used like any other common julia array:</p>
<div class="highlight highlight-source-julia"><pre>M<span class="pl-k">=</span><span class="pl-c1">DefaultArray</span>(<span class="pl-c1">0.0</span>,<span class="pl-c1">100</span>,<span class="pl-c1">100</span>) <span class="pl-c"><span class="pl-c">#</span>0.0 is the default value</span>
M<span class="pl-k">.=</span><span class="pl-c1">rand</span>([<span class="pl-c1">zeros</span>(<span class="pl-c1">50</span>);<span class="pl-c1">1</span>],<span class="pl-c1">100</span>,<span class="pl-c1">100</span>)

<span class="pl-k">for</span> i <span class="pl-k">in</span> <span class="pl-c1">eachnondefault</span>(M)
    M[i]<span class="pl-k">=</span><span class="pl-c1">rand</span>()
<span class="pl-k">end</span>
Q<span class="pl-k">=</span><span class="pl-c1">sin</span>.(M)

<span class="pl-c1">sum</span>(Q)  <span class="pl-c"><span class="pl-c">#</span>-&gt;  some random value</span>
<span class="pl-c1">length</span>(Q<span class="pl-k">.</span>elements) <span class="pl-c"><span class="pl-c">#</span>-&gt; MUCH less space occupied than 100*100 = 10000</span></pre></div>
