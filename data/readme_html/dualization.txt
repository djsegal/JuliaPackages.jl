<p dir="auto"><a href="https://github.com/jump-dev/Dualization.jl">Dualization.jl</a> is an extension
package for <a href="https://github.com/jump-dev/MathOptInterface.jl">MathOptInterface.jl</a>
that formulates the dual of conic optimization problems.</p>
<p dir="auto">Dualization.jl has two main features:</p>
<ul dir="auto">
<li>The <code>Dualization.dualize</code> function that computes the dual formulation of either
a <a href="https://github.com/jump-dev/MathOptInterface.jl">MathOptInterface.jl</a> or a
<a href="https://github.com/jump-dev/JuMP.jl">JuMP</a> model.</li>
<li>The <code>Dualization.dual_optimizer</code> function that creates a MathOptInterface-compatible
optimizer that solves the dual of the problem instead of the primal.</li>
</ul>

<div class="highlight highlight-source-julia notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="using JuMP, Dualization

model <span class="pl-k">=</span> <span class="pl-c1">Model</span>()
<span class="pl-c"><span class="pl-c">#</span> ... build model ...</span>
