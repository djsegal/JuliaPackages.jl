<p dir="auto">This little package wraps functions from the <a href="https://dandiarchive.org/" rel="nofollow">Dandi</a> <a href="https://github.com/dandi/dandi-cli">python</a> API.</p>
<p dir="auto">Functions and classes from the python API can be accessed with:</p>
<div class="highlight highlight-source-julia notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="using DANDIArchive
DANDIArchive.&lt;module&gt;.&lt;function&gt;"><pre><span class="pl-k">using</span> DANDIArchive
DANDIArchive<span class="pl-k">.&lt;</span><span class="pl-k">module</span><span class="pl-k">&gt;.&lt;</span><span class="pl-k">function</span><span class="pl-k">&gt;</span></pre></div>
<p dir="auto">where valid values for <code>&lt;module&gt;</code> and <code>&lt;function&gt;</code> generally follow the documentation for the python package, available <a href="https://dandi.readthedocs.io/en/latest/" rel="nofollow">here</a>.</p>
<p dir="auto">Additional functions have been added for convenience, such as:</p>
<div class="highlight highlight-source-julia notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="dandiurl(dandiset_id::String, filepath::String; version::{String})"><pre><span class="pl-c1">dandiurl</span>(dandiset_id<span class="pl-k">::</span><span class="pl-c1">String</span>, filepath<span class="pl-k">::</span><span class="pl-c1">String</span>; version<span class="pl-k">:</span>:{String})</pre></div>
<p dir="auto">which can be used to grab the url of a dandiset from the dandiset id and the filepath of the file within the dandiset.</p>
