<p dir="auto">A zero overhead game development framework for beginners.</p>

<p dir="auto">The aim of this package is to remove accidental complexity from the game development process. We therefore always choose simplicity and consistency over features. The users of this package will include young programmers learning their first language, maybe moving on from Scratch. While we aim to support reasonably sophisticated 2D games, our first priority will remain learners, and their teachers.</p>

<p dir="auto"><a href="https://juliahub.com/docs/GameZero/tTDGf/" rel="nofollow">Read the documentation</a> to get started. The best way to learn how to use this package is by looking at existing games created with it. There are some simple examples in the <a href="https://github.com/aviks/GameZero.jl/tree/master/example/BasicGame">example subdirectory</a>. More comprehensive examples are listed in the <a href="https://github.com/SquidSinker/GZExamples">GZExamples</a> repository. The documentation will also display the example sources.</p>
