<p><em>A Julia package for defining and working with linear maps.</em></p>
<p>Linear maps are also known as linear transformations or linear operators acting on vectors.
The only requirement for a <code>LinearMap</code> is that it can act on a vector (by multiplication) efficiently.</p>

From the Julia REPL, type <code>]</code> to enter the Pkg REPL mode and run:</p>

"><pre>pkg<span class="pl-k">&gt;</span> add LinearMaps</pre></div>
