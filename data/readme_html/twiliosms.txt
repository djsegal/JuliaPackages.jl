<p>TwilioSMS.jl is micro-package that provides a single function for sending text messages using the Twilio api. If there is interest, then further functionaliy can be added in the future. PRs are welcome.</p>
<h2><a id="user-content-setup" class="anchor" aria-hidden="true" href="#setup"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path fill-rule="evenodd" d="M4 9h1v1H4c-1.5 0-3-1.69-3-3.5S2.55 3 4 3h4c1.45 0 3 1.69 3 3.5 0 1.41-.91 2.72-2 3.25V8.59c.58-.45 1-1.27 1-2.09C10 5.22 8.98 4 8 4H4c-.98 0-2 1.22-2 2.5S3 9 4 9zm9-3h-1v1h1c1 0 2 1.22 2 2.5S13.98 12 13 12H9c-.98 0-2-1.22-2-2.5 0-.83.42-1.64 1-2.09V6.25c-1.09.53-2 1.84-2 3.25C6 11.31 7.55 13 9 13h4c1.45 0 3-1.69 3-3.5S14.5 6 13 6z"></path></svg></a>Setup</h2>
<p>Adding the package to Julia is easy, in the REPL simply type:</p>
<div class="highlight highlight-source-julia"><pre>]add TwilioSMS</pre></div>
<p>After that, an account has to be created on <a href="https://www.twilio.com/" rel="nofollow">Twilio.com</a> and a sid, an authentication token and a sender number has to be set up there. While this service costs, there is also a free trial that allows one to send sms. For the free trial it is also necessary that the receivers number is registered on their website.</p>
<p>The easiest way to store the the sid and auth token is in the environment variables <code>TWILIO_ACCOUNT_SID</code> and <code>TWILIO_AUTH_TOKEN</code>. It is also possible to pass those values explicetly in the code but for security reasons it is better not to have such values in code.</p>
