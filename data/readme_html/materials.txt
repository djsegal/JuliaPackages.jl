<p dir="auto">A computational material models package for JuliaFEM, concentrating on plasticity and viscoplasticity.</p>
<p dir="auto">The public API is defined in <a href="src/Materials.jl">src/Materials.jl</a>. For details, see the docstrings. For usage examples, see the <a href="test/">automated tests</a>.</p>
