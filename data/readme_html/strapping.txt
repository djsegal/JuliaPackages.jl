<p>"Strapping" stands for <strong>ST</strong>ruct <strong>R</strong>elational M<strong>APPING</strong>, and provides ORM-like functionality for Julia, including:</p>
<ul>
<li>automatically constructing Julia structs from any <a href="https://github.com/JuliaData/Tables.jl">Tables.jl</a>-compatible source (see <code>?Strapping.construct</code>)</li>
<li>ability to handle complicated aggregate types, with aggregate or collection fields</li>
<li>integration with the <a href="https://github.com/JuliaData/StructTypes.jl">StructTypes.jl</a> package for specifying struct and struct field options</li>
<li>transform any Julia struct (or vector of structs) into a 2D Tables.jl-compatible source, which can be stored in a database, file format, or other "sink" (see <code>?Strapping.deconstruct</code>)</li>
</ul>
