<p>JuMP extensions for constraint programming.</p>
<p>These extensions rely on <a href="https://github.com/dourouc05/ConstraintProgrammingExtensions.jl">ConstraintProgrammingExtensions</a>, an extension for <a href="https://github.com/JuliaOpt/MathOptInterface.jl">MathOptInterface</a> providing several constraint-programming-oriented sets.</p>
<p>For now, the new syntax can only be used with a patch applied on top of JuMP: <a href="https://github.com/JuliaOpt/JuMP.jl/pull/2051">PR 2051</a>.</p>
