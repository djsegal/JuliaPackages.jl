<p dir="auto">Qwind is a code that aims to simulate the launching and acceleration phase of line-driven winds in the context of AGN accretion discs. Refer to <a href="https://arxiv.org/abs/2111.02742" rel="nofollow">the paper</a> for an explanation on the physics of the code.</p>
