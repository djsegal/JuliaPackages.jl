<h3><a id="user-content-pointwiselearn" class="anchor" aria-hidden="true" href="#pointwiselearn"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path fill-rule="evenodd" d="M7.775 3.275a.75.75 0 001.06 1.06l1.25-1.25a2 2 0 112.83 2.83l-2.5 2.5a2 2 0 01-2.83 0 .75.75 0 00-1.06 1.06 3.5 3.5 0 004.95 0l2.5-2.5a3.5 3.5 0 00-4.95-4.95l-1.25 1.25zm-4.69 9.64a2 2 0 010-2.83l2.5-2.5a2 2 0 012.83 0 .75.75 0 001.06-1.06 3.5 3.5 0 00-4.95 0l-2.5 2.5a3.5 3.5 0 004.95 4.95l1.25-1.25a.75.75 0 00-1.06-1.06l-1.25 1.25a2 2 0 01-2.83 0z"></path></svg></a>PointwiseLearn</h3>
<p>A geostatistical learning solver that takes a classical (non-spatial)
statistical learning model from the literature and applies it to each
point of the domain. Although this solver is widely used in geospatial
problems, it has various issues as described in
<a href="https://arxiv.org/abs/2102.08791" rel="nofollow">Hoffimann et al. 2021</a>.</p>
