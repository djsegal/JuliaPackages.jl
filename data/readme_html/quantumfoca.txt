<p dir="auto">For any molecular calculations, you will need a basis set.</p>
<div class="highlight highlight-source-julia notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="sto3g = buildbasis(methane)"><pre>sto3g <span class="pl-k">=</span> <span class="pl-c1">buildbasis</span>(methane)</pre></div>
<p dir="auto">With this information, we can build the molecular integrals.</p>
