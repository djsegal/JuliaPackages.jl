<p dir="auto">Functions in the Julia programming language for supporting the writing of TextMate commands for the Go programming language.</p>
<p dir="auto">Presently it contains support for function and method completion based on <a href="https://github.com/nsf/gocode">gocode</a>.</p>
<p dir="auto">These functions depend on the <a href="https://github.com/ordovician/SimpleParser.jl">SimpleParser module</a> and <a href="https://github.com/ordovician/PList.jl">PList module</a>.</p>
