<p dir="auto">This package defines julia implementations for the common types <code>Option</code> (aka <code>Maybe</code>), <code>Either</code> and <code>Try</code>, as well as one extra type <code>ContextManager</code> which mimics Python's <code>with</code>-ContextManager.</p>
