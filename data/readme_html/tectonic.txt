<p dir="auto">Compile LaTeX files in Julia using the <a href="https://github.com/tectonic-typesetting/tectonic"><em>tectonic</em></a> typesetting
system. Also provides a <a href="https://github.com/plk/biber"><code>biber</code></a> binary that is compatible with the
version of the <code>biblatex</code> package provided with <em>tectonic</em>.</p>

<p dir="auto">If you need to compile a document that contains <code>biblatex</code> then you will need
to provide the <code>biber</code> program to <code>tectonic</code> by adding it to the <code>PATH</code>. This
can be done by calling <code>Biber.biber</code> as follows</p>
<div class="snippet-clipboard-content notranslate position-relative overflow-auto" data-snippet-clipboard-copy-content="julia&gt; biber() do _
           tectonic() do bin
               run(`$bin file.tex`)
           end
       end"><pre class="notranslate"><code>julia&gt; biber() do _
           tectonic() do bin
               run(`$bin file.tex`)
           end
       end
</code></pre></div>
<p dir="auto">If <code>biber</code> already exists on your <code>PATH</code> then calling <code>Tectonic.tectonic</code>
will find it automatically and use it.</p>
