<p dir="auto">This package produces pair plots, otherwise known as corner plots or scatter plot matrices: grids of 1D and 2D histograms that allow you to visualize high dimensional data.</p>
<p dir="auto">Pair plots are an excellent way to vizualize the results of MCMC simulations, but are also a useful way to vizualize correlations in general data tables.</p>
<p dir="auto">Read the documentation here: (<a href="https://sefffal.github.io/PairPlots.jl/dev/" rel="nofollow">https://sefffal.github.io/PairPlots.jl/dev/</a>)</p>
<p dir="auto">The default styles of this package roughly reproduce the output of the Python library <a href="https://corner.readthedocs.io/en/latest/index.html" rel="nofollow">corner.py</a> for a single series, and <a href="https://samreay.github.io/ChainConsumer/usage.html" rel="nofollow">chainconsumer.py</a> for multiple series.
If these are not to your tastes, this package is highly configurable.</p>
<p dir="auto">The current version of PairPlots.jl requires the <a href="https://makie.juliaplots.org/" rel="nofollow">Makie</a> plotting library. If instead you prefer to use Plots.jl, you can install the legacy version <code>0.6</code> of PairPlots (see <a href="https://github.com/sefffal/PairPlots.jl/blob/b632abd79c0dfbe7387d44393f4fb5b7f74ac5d8/README.md">archived documentation here</a>)</p>
<p dir="auto">For related functionality, see also: <a href="https://github.com/JuliaPlots/StatsPlots.jl#corrplot-and-cornerplot">StatsPlots.cornerplot</a> and <a href="https://github.com/kilianbreathnach/CornerPlot.jl">CornerPlot.jl</a> for the Gadfly plotting library.</p>
