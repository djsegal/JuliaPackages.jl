<p dir="auto">This package provides plotting functions for protein structures, multiple sequence alignments, and some other related plots like protein structure contact maps.</p>
<p dir="auto">So far, plotting methods exist for packages:</p>
<ul dir="auto">
<li>BioStructures.jl</li>
<li>MIToS.jl</li>
<li>FastaIO.jl</li>
<li>FASTX.jl</li>
</ul>
<p dir="auto">Plotting methods in development:</p>
<ul dir="auto">
<li>MolecularGraph.jl</li>
<li>ProtoSyn.jl</li>
</ul>

<p dir="auto">The main plotting functions are <strong>plotstruc</strong> and <strong>plotmsa</strong>, along with their mutating
versions, <strong>plotstruc!</strong> and <strong>plotmsa!</strong>. The mutating functions allow the user to add multiple plots to the same Figure, using grid positions.</p>
