<p dir="auto"><em>A Modular Multi-Fidelity Aeroelastic Analysis and Simulation Framework</em></p>
<p dir="auto">Author: Taylor McDonnell</p>
<p dir="auto"><strong>Aeroelasticity.jl</strong> is a modular multi-fidelity aeroelastic analysis and simulation package.  The intent of this package is to facilitate defining and simulating the behavior of both 2D and 3D aeroelastic systems, though this package may also be used to construct general monolithic coupled systems.</p>

<ul dir="auto">
<li>Facilitates defining and simulating the behavior of both 2D and 3D aeroelastic systems</li>
<li>Contains a number of predefined aerodynamic, structural, and aeroelastic models including:
<ul dir="auto">
<li>Aerodynamic Models
<ul dir="auto">
<li>Steady Thin Airfoil Theory (2D)</li>
<li>Quasi-Steady Thin Airfoil Theory (2D)</li>
<li>Wagner's Function (2D)</li>
<li>Peters' Finite State (2D)</li>
<li>Lifting Line (3D)</li>
</ul>
</li>
<li>Structural Models
<ul dir="auto">
<li>Two Degree of Freedom Typical Section Model (2D)</li>
<li>Rigid Body Model (3D)</li>
<li>Geometrically Exact Beam Theory Model (3D)</li>
</ul>
</li>
</ul>
</li>
<li>Supports multiple analysis types including:
<ul dir="auto">
<li>Steady State Analyses</li>
<li>Eigenvalue Analyses</li>
<li>Time-Marching Analyses (using <a href="https://github.com/SciML/DifferentialEquations.jl">DifferentialEquations</a>)</li>
</ul>
</li>
<li>Several verification cases for built-in models (see the <a href="https://flow.byu.edu/Aeroelasticity.jl/dev/examples/" rel="nofollow">examples</a>)</li>
<li>Provides a modular framework for constructing general monolithic coupled systems</li>
</ul>
