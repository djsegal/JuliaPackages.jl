<p dir="auto">The setup of the repository uses Literate.jl. If you are unfamiliar with it, it is worth taking a look at its documentation. Alternatively, have a look at existing examples source code in <code>docs/examples</code>.</p>
<p dir="auto">The process of adding a new example is simple:</p>
<ol dir="auto">
<li>Fork this repository and in your own version add a new <code>.jl</code> example file in <code>docs/examples</code>. As per Literate.jl, the example must be a valid, runnable Julia file.</li>
<li>In the file <code>docs/make.jl</code> add your new example in the variable <code>NEW_EXAMPLES</code> in the very first line of the file. You need to add a <code>Pair</code> of <code>String</code> to <code>String</code> such as <code>"My example title" =&gt; "example_file.jl"</code>.  This happens so that when you open a Pull Request only your example is compiled (saves a looot of time!).</li>
<li>Also mention your example in the <code>docs/src/index.md</code> file, which provides an overview of the examples and where they are useful.</li>
<li>Open a Pull Request with a summary of your example.</li>
<li>After the PR has been reviewed and accepted, simply move your example from the <code>NEW_EXAMPLES</code> variable to the <code>EXISTING_EXAMPLES</code> one (and please respect alphabetical sorting!).</li>
</ol>
