<p dir="auto"><code>VLOptionsModeling.jl</code> is a <a href="https://julialang.org/downloads/" rel="nofollow">Julia</a> package to model the price of <a href="https://www.investopedia.com/terms/e/europeanoption.asp" rel="nofollow">European</a> and <a href="https://www.investopedia.com/terms/a/americanoption.asp" rel="nofollow">American</a> style equity derivative contracts.</p>

<p dir="auto"><code>VLOptionsModeling.jl</code> is organized as a <a href="http://julialang.org" rel="nofollow">Julia</a> package which can be installed in the <code>package mode</code> of Julia.</p>
<p dir="auto">Start of the <a href="https://docs.julialang.org/en/v1/stdlib/REPL/index.html" rel="nofollow">Julia REPL</a> and enter the <code>package mode</code> using the <code>]</code> key (to get back press the <code>backspace</code> or <code>^C</code> keys). Then, at the prompt enter:</p>
<div class="snippet-clipboard-content notranslate position-relative overflow-auto" data-snippet-clipboard-copy-content="(v1.1) pkg&gt; add VLConstraintBasedModelGenerationUtlities"><pre class="notranslate"><code>(v1.1) pkg&gt; add VLConstraintBasedModelGenerationUtlities
</code></pre></div>
<p dir="auto">This will install the <code>VLOptionsModeling.jl</code> package and the other required packages. <code>VLOptionsModeling.jl</code> requires Julia 1.6.x and above.
<code>VLOptionsModeling.jl</code> is open source. You can download this repository as a zip file, or clone or pull it by using the command (from the command-line):</p>
<div class="snippet-clipboard-content notranslate position-relative overflow-auto" data-snippet-clipboard-copy-content="$ git pull https://github.com/varnerlab/VLOptionsModeling.jl.git"><pre class="notranslate"><code>$ git pull https://github.com/varnerlab/VLOptionsModeling.jl.git
</code></pre></div>
<p dir="auto">or</p>
