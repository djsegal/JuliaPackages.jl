<p dir="auto">Implements <a href="http://nlp.stanford.edu/projects/glove/" rel="nofollow">Global Word Vectors</a>.</p>
<div class="snippet-clipboard-content notranslate position-relative overflow-auto" data-snippet-clipboard-copy-content="using Pkg

</code></pre></div>
<p dir="auto">See <code>benchmark/perf.jl</code> for a usage example.</p>
<p dir="auto">Here's the rough idea:</p>
<ol dir="auto">
<li>
<p dir="auto">Take text and make a LookupTable. This is a dictionary that has a map
from words -&gt; ids and vice-versa. Preprocessing steps should be taken
prior to this.</p>
</li>
<li>
<p dir="auto">Use <code>weightedsums</code> to get the weighted co-occurence sum totals. This returns
a <code>CooccurenceDict</code>.</p>
</li>
<li>
<p dir="auto">Convert the <code>CooccurenceDict</code> to a <code>CooccurenceVector</code>. The reasoning for this is faster indexing when we train the model.</p>
</li>
<li>
<p dir="auto">Initialize a <code>Model</code> and train the model with the <code>CooccurenceVector</code> using
the <code>agagrad!</code> method.</p>
</li>
</ol>
<p dir="auto">It's pretty fast at this point. On a single core it's roughly 3x slower than the optimized C version.</p>
