<ul dir="auto">
<li>
<p dir="auto"><a href="https://github.com/queryverse/VegaLite.jl">VegaLite.jl</a> overloads
<code>getproperty</code> and lens API to manipulate JSON-based nested objects.</p>
</li>
<li>
<p dir="auto"><a href="https://github.com/tkf/Kaleido.jl">Kaleido.jl</a> is a library of
additional lenses.</p>
</li>
<li>
<p dir="auto"><a href="https://github.com/jw3126/PhaseSpaceIO.jl">PhaseSpaceIO.jl</a> overloads
<code>getproperty</code> and <code>setproperties</code> to get/set values from/in packed bits.</p>
</li>
</ul>
