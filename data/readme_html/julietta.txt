<p>Julietta is an integrated developement environment (IDE) for the programming language Julia. It is written in Julia using Gtk.jl. In its current form Julietta is a prototype and should not be used for real developement as several thing are not yet implemented (and due to several bugs).</p>

<h2><a id="user-content-installation" class="anchor" aria-hidden="true" href="#installation"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path fill-rule="evenodd" d="M4 9h1v1H4c-1.5 0-3-1.69-3-3.5S2.55 3 4 3h4c1.45 0 3 1.69 3 3.5 0 1.41-.91 2.72-2 3.25V8.59c.58-.45 1-1.27 1-2.09C10 5.22 8.98 4 8 4H4c-.98 0-2 1.22-2 2.5S3 9 4 9zm9-3h-1v1h1c1 0 2 1.22 2 2.5S13.98 12 13 12H9c-.98 0-2-1.22-2-2.5 0-.83.42-1.64 1-2.09V6.25c-1.09.53-2 1.84-2 3.25C6 11.31 7.55 13 9 13h4c1.45 0 3-1.69 3-3.5S14.5 6 13 6z"></path></svg></a>Installation</h2>
<p>Currently it is a lot complicated to get Julietta running. Here is a list of instructions:</p>
<ul>
<li>Install Gtk.jl from my "listView" branch: <a href="https://github.com/tknopp/Gtk.jl/tree/listView">https://github.com/tknopp/Gtk.jl/tree/listView</a></li>
<li>Install GtkSourceWidget.jl: <a href="https://github.com/tknopp/GtkSourceWidget.jl">https://github.com/tknopp/GtkSourceWidget.jl</a></li>
</ul>
<p>Furthermore Julietta requires the following Julia packages to be installed: REPLCompletions, Winston</p>
<p>The complicated part is not the installation of the Julia packages (using Pkg.clone) but the installation of the dependencies. Under linux this should be most simple as the libraries Gtk+ and GtkSourceView are part of the distributions package manager.</p>
