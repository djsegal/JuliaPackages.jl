<p dir="auto">A Julia package for uncertainty quantification. Current functionality includes:</p>
<ul dir="auto">
<li>Simulation-based reliability analysis
<ul dir="auto">
<li>Monte Carlo simulation</li>
<li>Quasi Monte Carlo simulation (Sobol, Halton, Latin Hypercube, Lattice Rule)</li>
<li>Line Sampling</li>
<li>Subset Simulation</li>
</ul>
</li>
<li>Sensitivity analysis
<ul dir="auto">
<li>Gradients</li>
<li>Sobol indices</li>
</ul>
</li>
<li>Metamodeling
<ul dir="auto">
<li>Polyharmonic splines</li>
<li>Polynomial Chaos Expansion</li>
<li>Response Surface Methodology</li>
</ul>
</li>
<li>Third-party solvers
<ul dir="auto">
<li>Connect to any solver by injecting random samples into source files</li>
</ul>
</li>
</ul>
