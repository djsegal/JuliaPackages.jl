<p dir="auto">First, Julia needs to be installed on your computer. If this is not the case, you can install it by following the instructions provided on <a href="https://julialang.org/downloads/" rel="nofollow">this page</a></p>
<p dir="auto">Within a Julia session, you can install the package by typing</p>
<div class="highlight highlight-source-julia notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="julia&gt; using Pkg
julia&gt; Pkg.add(&quot;PANDA&quot;)"><pre>julia<span class="pl-k">&gt;</span> <span class="pl-k">using</span> Pkg
julia<span class="pl-k">&gt;</span> Pkg<span class="pl-k">.</span><span class="pl-c1">add</span>(<span class="pl-s"><span class="pl-pds">"</span>PANDA<span class="pl-pds">"</span></span>)</pre></div>
<p dir="auto">PANDA uses R functions and packages for plotting. If you want to be able to use the plotting functions, <a href="https://www.r-project.org/" rel="nofollow">the R language</a> needs to be installed on your computer. You will also need a few R packages to be installed, including : <a href="https://CRAN.R-project.org/package=ape" rel="nofollow">ape</a>, <a href="https://CRAN.R-project.org/package=coda" rel="nofollow">coda</a>, <a href="https://CRAN.R-project.org/package=RColorBrewer" rel="nofollow">RColorBrewer</a>, <a href="https://CRAN.R-project.org/package=fields" rel="nofollow">fields</a>. You can install them from a R session by typing</p>
<div class="highlight highlight-source-r notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="&gt; install.packages(&quot;ape&quot;, &quot;coda&quot;, &quot;RColorBrewer&quot;, &quot;fields&quot;)"><pre><span class="pl-k">&gt;</span> install.packages(<span class="pl-s"><span class="pl-pds">"</span>ape<span class="pl-pds">"</span></span>, <span class="pl-s"><span class="pl-pds">"</span>coda<span class="pl-pds">"</span></span>, <span class="pl-s"><span class="pl-pds">"</span>RColorBrewer<span class="pl-pds">"</span></span>, <span class="pl-s"><span class="pl-pds">"</span>fields<span class="pl-pds">"</span></span>)</pre></div>
<p dir="auto">You will then be able to load PANDA to Julia by typing</p>
<div class="highlight highlight-source-julia notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="julia&gt; using PANDA"><pre>julia<span class="pl-k">&gt;</span> <span class="pl-k">using</span> PANDA</pre></div>
