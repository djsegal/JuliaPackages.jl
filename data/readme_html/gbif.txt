<p dir="auto"><em><strong>MAINTENANCE MODE</strong></em> - this repo is being moved to <a href="https://github.com/PoisotLab/SpeciesDistributionToolkit.jl">https://github.com/PoisotLab/SpeciesDistributionToolkit.jl</a></p>
<p dir="auto">This is project offers an interface to the <a href="http://gbif.org/" rel="nofollow">GBIF</a> search API from
Julia. Current efforts focus on querying and exporting occurrences through the
<code>occurrence/search</code> endpoint. There is partial support for the taxonomic API.</p>
