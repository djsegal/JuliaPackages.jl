<p dir="auto">If you want to develop this package do the following steps:</p>
<ul dir="auto">
<li>Make a fork and then clone the repo locally on your computer</li>
<li>In line 2 of the <code>Project.toml</code> file (the line that begins with <code>uuid = ...</code>), modify the UUID, e.g. change the <code>44cf...</code> to <code>54cf...</code>.</li>
<li>Change the current directory to the Pkg repo you just cloned and start julia with <code>julia --project</code>.</li>
<li><code>import Pkg</code> will now load the files in the cloned repo instead of the Pkg stdlib .</li>
<li>To test your changes, simply do <code>include("test/runtests.jl")</code>.</li>
<li>Before you commit and push your changes, remember to change the UUID in the <code>Project.toml</code> file back to the original UUID</li>
</ul>
<p dir="auto">If you need to build Julia from source with a Git checkout of Pkg, then instead use <code>make DEPS_GIT=Pkg</code> when building Julia. The <code>Pkg</code> repo is in <code>stdlib/Pkg</code>, and created initially with a detached <code>HEAD</code>. If you're doing this from a pre-existing Julia repository, you may need to <code>make clean</code> beforehand.</p>
<p dir="auto">If you need to build Julia from source with Git checkouts of two or more stdlibs, please see the instructions in the <a href="https://github.com/JuliaLang/julia/blob/master/doc/src/devdocs/build/build.md#building-julia-from-source-with-a-git-checkout-of-a-stdlib"><code>Building Julia from source with a Git checkout of a stdlib</code></a> section of the <a href="https://github.com/JuliaLang/julia/blob/master/doc/src/devdocs/build/build.md"><code>doc/src/devdocs/build/build.md</code></a> file within the Julia devdocs.</p>
