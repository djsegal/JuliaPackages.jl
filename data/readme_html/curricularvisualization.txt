<p><strong>CurricularVisualization.jl</strong> is an extension of the CurricularAnalytics.jl toolbox for visualizing curricula and degree plans.</p>

<p>Full documentation is available at CurricularAnalytics.jl <a href="https://curricularanalytics.github.io/CurricularAnalytics.jl/latest/" rel="nofollow">GitHub Pages</a>.</p>

<p>Installation is straightforward.  First, install the Julia programming language on your computer.  To do this, download Julia here: <a href="https://julialang.org" rel="nofollow">https://julialang.org</a>, and follow the instructions for your operating system.</p>
<p>Next, open the Julia application that you just installed. It should look similar to the image below. This interface is referred to as the <code>Julia REPL</code>.</p>
<p><a target="_blank" rel="noopener noreferrer" href="https://camo.githubusercontent.com/1563729cacc66273ee5f4aa3ec03c87c0138c9d2/68747470733a2f2f73332e616d617a6f6e6177732e636f6d2f637572726963756c6172616e616c79746963732e6a6c2f6a756c69612d636f6d6d616e642d6c696e652e706e67"><img src="https://camo.githubusercontent.com/1563729cacc66273ee5f4aa3ec03c87c0138c9d2/68747470733a2f2f73332e616d617a6f6e6177732e636f6d2f637572726963756c6172616e616c79746963732e6a6c2f6a756c69612d636f6d6d616e642d6c696e652e706e67" alt="Julia termain" data-canonical-src="https://s3.amazonaws.com/curricularanalytics.jl/julia-command-line.png" style="max-width:100%;"></a></p>
<p>Next, enter Pkg mode from within Julia by hitting the <code>]</code> key, and then type:</p>
<pre lang="julia-repl"><code>  pkg&gt; add CurricularAnalytics CurricularVisualization
</code></pre>
<p>This will install the CurricularAnalytics toolbox (if not already installed) and the visualization software, along with the other Julia packages needed to run it. To load and use the visualization, hit the <code>backspace</code> key to return to the Julia REPL. Now type:</p>
<pre lang="julia-repl"><code>  julia&gt; using CurricularAnalytics, CurricularVisualization
</code></pre>
<p>The visualization software must be loaded again via <code>using CurricularAnalytics, CurricularVisualization</code> every time you restart Julia.</p>
<p>For more information about installing the toolbox, including the steps neccessary to utilize degree plan optimization, please see the <a href="https://curricularanalytics.github.io/CurricularAnalytics.jl/latest/install.html" rel="nofollow">INSTALLING.md</a></p>

<ul>
<li>CurricularVisualization will be maintained/enhanced to work with the latest stable version of Julia.</li>
<li>Julia Version(s): CurricularVisualization v0.1.0 is the latest version guaranteed to work with Julia 1.4</li>
