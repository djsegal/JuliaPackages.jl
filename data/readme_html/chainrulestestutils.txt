<blockquote>
<p>Utilities for testing forward- and reverse-mode AD sensitivities.</p>
</blockquote>
<p><code>ChainRulesTestUtils.jl</code> is designed to help you test <code>ChainRulesCore.frule</code> and <code>ChainRulesCore.rrule</code> methods.
Currently this is done via testing the rules against numerical differentiation (using <a href="https://github.com/JuliaDiff/FiniteDifferences.jl"><code>FiniteDifferences.jl</code></a>).</p>
<p><code>ChainRulesTestUtils.jl</code> is separate from <a href="https://github.com/JuliaDiff/ChainRulesCore.jl"><code>ChainRulesCore.jl</code></a> so that it can be a test-only dependency, allowing it to have potentially heavy dependencies, while keeping <code>ChainRulesCore.jl</code> as light-weight as possible.</p>
