<blockquote>
<p dir="auto">Collection of utilities for testing forward- and reverse-mode AD sensitivities.</p>
</blockquote>
<p dir="auto"><code>ChainRulesTestUtils.jl</code> is designed to help you test <code>ChainRulesCore.frule</code> and <code>ChainRulesCore.rrule</code> methods.
The main entry points are <code>ChainRulesTestUtils.frule_test</code>, <code>ChainRulesTestUtils.rrule_test</code>, and <code>ChainRulesTestUtils.test_scalar</code>
Currently this is done via testing the rules against numerical differentiation (using <a href="https://github.com/JuliaDiff/FiniteDifferences.jl"><code>FiniteDifferences.jl</code></a>).</p>
<p dir="auto"><code>ChainRulesTestUtils.jl</code> is separate from <a href="https://github.com/JuliaDiff/ChainRulesCore.jl"><code>ChainRulesCore.jl</code></a> so that it can be a test-only dependency, allowing it to have potentially heavy dependencies, while keeping <code>ChainRulesCore.jl</code> as light-weight as possible.</p>
