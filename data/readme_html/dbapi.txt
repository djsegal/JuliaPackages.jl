<p dir="auto">This module defines an abstract interface for database drivers in Julia.</p>
<p dir="auto">This module contains abstract types, abstract required functions (which throw
a <code>NotImplementedError</code> by default), and abstract optional functions (which
throw a <code>NotSupportedError</code> by default).</p>
<p dir="auto">Database drivers should import this module, subtype its types, and create
methods for its functions which implement the signatures and behavioural
specifications contained within this module.</p>
<p dir="auto">This interface is largely inspired by Python's DB API 2.0, which is in the
public domain: <a href="https://www.python.org/dev/peps/pep-0249" rel="nofollow">PEP0249</a>.</p>
