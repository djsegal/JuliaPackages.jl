<p>This provides support for different types of characters, using a parameterized <code>Chr</code> type.
(and for v0.6.2, adds the AbstractChar type, which was later added to v0.7/master)
See the documentation in the <a href="https://github.com/JuliaString/Strs.jl">Strs</a> package</p>
