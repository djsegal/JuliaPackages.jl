<p dir="auto">The <code>play</code> function takes the crazyflie <a href="https://wiki.bitcraze.io/doc:crazyflie:api:python:index#uniform_resource_identifier_uri" rel="nofollow">uri</a> and the anonymous function in the <code>do...end</code> block as inputs:</p>
<div class="highlight highlight-source-julia notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="play(uri) do cf
  # send commands
end"><pre><span class="pl-c1">play</span>(uri) <span class="pl-k">do</span> cf
  <span class="pl-c"><span class="pl-c">#</span> send commands</span>
<span class="pl-k">end</span></pre></div>
<p dir="auto">It constructs the <a href="https://github.com/bitcraze/crazyflie-lib-python/blob/master/cflib/crazyflie/syncCrazyflie.py">SyncCrazyflie</a> python object, connects to the crazyflie, runs the provided algorithm, and disconnects from the crazyflie.</p>
