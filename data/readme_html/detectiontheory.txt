<p dir="auto"><code>DetectionTheory.jl</code> is a <a href="http://www.julialang.org" rel="nofollow">Julia</a> package to compute signal detection theory measures. It includes functions to compute <em>d'</em> for the following tasks:</p>
<ul dir="auto">
<li>ABX</li>
<li>Multiple alternatives forced choice</li>
<li>Odd-One-Out with 3 response alternatives (also known as oddity, or triangular method)</li>
<li>Same-Different</li>
<li>Yes-No</li>
</ul>
<p dir="auto">Most of the functions have been ported from the <a href="http://cran.r-project.org/web/packages/psyphy/index.html" rel="nofollow">psyphy</a> R package.</p>
<p dir="auto">Documentation is available <a href="http://samcarcagno.altervista.org/DetectionTheory/index.html" rel="nofollow">here</a></p>
