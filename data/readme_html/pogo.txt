<p dir="auto">The key functions provided by this package are:</p>
<ul dir="auto">
<li><code>approximate</code> this allows for the piecewise linear approximation of arbitrary functions. Including those that are discontinuous or otherwise non-smooth. Various methods are supported.</li>
<li><code>bilinear</code> this creates an approximation of the product of two variables. If either variable is integer or binary, then this will be modelled exactly.</li>
<li><code>power</code> this creates an approximation of one variable to the power of another.</li>
<li><code>interpolate</code> this creates a constraint forcing a set of variables to lie inside at least one set (from a group of sets), that may be defined in terms of other variables.</li>
<li><code>interpolate_fn</code> this creates one (or more) new variable(s) that is / are defined as a function of other variables at various points.</li>
<li><code>interpolate_points</code> this takes a set of multidimensional points and using multidimensional Delaunay triangulation over a subset of the dimensions, creates one (or more) new variable(s) that is / are defined by interpolation of the points over the space defined by the triangulation. If the triangulation is over all the dimensions, then rather than interpolating values for new variables, this function will define a feasible region for the variables.</li>
</ul>
