<p dir="auto">Simple GPE solver using FFTW. This package is intended to provide a julia implementation of the Gross-Pitaevskii equation based on Fourier spectral methods and <a href="https://github.com/SciML/DifferentialEquations.jl">DifferentialEquations.jl</a></p>
<p dir="auto">Aims</p>
<ul dir="auto">
<li>Simple, fast, and flexible modelling of Bose-Einstein condensate experiments.</li>
<li>Allocation free, adaptive evolution in kspace.</li>
<li>Establish some useful defaults and runnable examples for time evolution using <code>DifferentialEquations.jl</code></li>
<li>Arbitrary spatial dimensions.</li>
<li>Not de-aliased (projective methods available elsewhere)</li>
</ul>
