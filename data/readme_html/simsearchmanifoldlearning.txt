<p dir="auto">This package provides some support to use <code>SimilaritySearch</code> with manifold learning methods. In particular,
we implement the required methods to implement <code>knn</code> function for <code>ManifoldLearning</code> and also provides an <code>UMAP</code>
implementation that takes advantage of many <code>SimilaritySearch</code> features like multithreading and data independency; it supports string, sets, vectors, etc. under diverse distance functions.</p>
<p dir="auto">The <code>ManifoldLearning</code> support is limited to some structure specification due to the design of the package. See the <code>ManifoldKnnIndex</code> type in the documentation pages.</p>

<p dir="auto">This package also provides a pure Julia implementation of the <a href="https://arxiv.org/abs/1802.03426" rel="nofollow">Uniform Manifold Approximation and Projection</a> dimension reduction algorithm</p>
<blockquote>
<p dir="auto">McInnes, L, Healy, J, Melville, J, <em>UMAP: Uniform Manifold Approximation and Projection for
Dimension Reduction</em>. ArXiV 1802.03426, 2018</p>
</blockquote>
<p dir="auto">The implementation in this package is based on the <a href="https://github.com/dillondaudert/UMAP.jl">UMAP.jl</a> package by Dillon Gene Daudert and collaborators. Forked and adapted to work with <code>SimilaritySearch</code> and take advantage of multithreading systems.</p>
<p dir="auto">The provided implementation partially supports the <code>ManifoldLearning</code> API using <code>fit</code> and <code>predict</code> and similar arguments. It can use any distance function from <code>SimilaritySearch</code>, <a href="https://github.com/JuliaStats/Distances.jl">Distances.jl</a>, <a href="https://github.com/matthieugomez/StringDistances.jl">StringDistances.jl</a>, or any distance function implemented by the user.</p>
<p dir="auto">Additionally,  it improves multithreading support in the KNN and the UMAP projection.</p>
