<p dir="auto">This library provides a wide array of functions for dealing with color. This
includes conversion between colorspaces, measuring distance between colors,
simulating color blindness, parsing colors, and generating color scales for graphics.</p>
<p dir="auto">The core color types, along with some simple utilities, are defined and documented in <a href="https://github.com/JuliaGraphics/ColorTypes.jl">ColorTypes</a>.
You can use ColorTypes as a standalone package if you do not need the more extensive color-manipulation utilities defined here.</p>
<p dir="auto">Basic color arithmetic is available with <code>using ColorVectorSpace</code>; see <a href="https://github.com/JuliaGraphics/ColorVectorSpace.jl">its documentation</a> for why it's a separate package.</p>
