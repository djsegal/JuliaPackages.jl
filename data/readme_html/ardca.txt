<p dir="auto">The authors of this code are Jeanne Trinquier, Guido Uguzzoni, Andrea Pagnani, Francesco Zamponi, and Martin Weigt.</p>
<p dir="auto">See also <a href="https://en.wikipedia.org/wiki/Direct_coupling_analysis" rel="nofollow">this Wikipedia article</a> article for a general overview of the Direct Coupling Analysis technique.</p>
<p dir="auto">The code is written in <a href="https://www.julialang.org/" rel="nofollow">Julia</a>.</p>
