<p dir="auto"><a href="https://github.com/JuliaQuantumControl/QuantumCitations.jl#quantumcitationsjl">QuantumCitations.jl</a> is a fork of <a href="https://github.com/ali-ramadhan/DocumenterCitations.jl">DocumenterCitations.jl</a> adapted to the conventions of (quantum) physics. It adds support for BibTeX citations in documentation pages generated by <a href="https://github.com/JuliaDocs/Documenter.jl">Documenter.jl</a>, especially the documentation of the packages in the <a href="https://github.com/JuliaQuantumControl">JuliaQuantumControl organization</a>.</p>
<p dir="auto">By default, <a href="https://github.com/JuliaQuantumControl/QuantumCitations.jl#quantumcitationsjl">QuantumCitations.jl</a> uses the numeric citation style of the <a href="https://journals.aps.org" rel="nofollow">APS journals</a>, see the <a href="https://www.ctan.org/tex-archive/macros/latex/contrib/revtex/auguide" rel="nofollow">REVTeX author's guide</a>.  Citations are shown in-line, as a number enclosed in square brackets, e.g., "Optimal control is a cornerstone in the development of quantum technologiesÂ <a href="#screenshot">[1]</a>."</p>
<p dir="auto"><a target="_blank" rel="noopener noreferrer" href="docs/src/assets/references.png"><img id="user-content-screenshot" src="docs/src/assets/references.png" alt="Rendered bibliography of two references, [1] and [2]" width="830px" style="max-width: 100%;"></a></p>
<p dir="auto">Alternatively, author-year and alphabetic citations styles are available, see the <a href="https://juliaquantumcontrol.github.io/QuantumCitations.jl/dev/gallery/" rel="nofollow">Citation Style Gallery</a>. In addition, it is possible to define custom styles.</p>

<p dir="auto">The <code>QuantumCitations</code> package can be installed with <a href="https://pkgdocs.julialang.org/v1/" rel="nofollow">Pkg</a> as</p>
<div class="snippet-clipboard-content notranslate position-relative overflow-auto" data-snippet-clipboard-copy-content="pkg&gt; add QuantumCitations"><pre class="notranslate"><code>pkg&gt; add QuantumCitations
</code></pre></div>
<p dir="auto">In most cases, you will just want to have <code>QuantumCitations</code> in the project that builds your documentation (e.g. <a href="https://github.com/JuliaQuantumControl/QuantumCitations.jl/blob/master/test/Project.toml"><code>test/Project.toml</code></a>). Thus, you can also simply add</p>
<div class="snippet-clipboard-content notranslate position-relative overflow-auto" data-snippet-clipboard-copy-content="QuantumCitations = &quot;259add83-af3e-4603-9706-50e341d5574e&quot;"><pre class="notranslate"><code>QuantumCitations = "259add83-af3e-4603-9706-50e341d5574e"
</code></pre></div>
<p dir="auto">to the <code>[deps]</code> section of the relevant <code>Project.toml</code> file.</p>
<h2 dir="auto"><a id="user-content-usage" class="anchor" aria-hidden="true" href="#usage"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path d="m7.775 3.275 1.25-1.25a3.5 3.5 0 1 1 4.95 4.95l-2.5 2.5a3.5 3.5 0 0 1-4.95 0 .751.751 0 0 1 .018-1.042.751.751 0 0 1 1.042-.018 1.998 1.998 0 0 0 2.83 0l2.5-2.5a2.002 2.002 0 0 0-2.83-2.83l-1.25 1.25a.751.751 0 0 1-1.042-.018.751.751 0 0 1-.018-1.042Zm-4.69 9.64a1.998 1.998 0 0 0 2.83 0l1.25-1.25a.751.751 0 0 1 1.042.018.751.751 0 0 1 .018 1.042l-1.25 1.25a3.5 3.5 0 1 1-4.95-4.95l2.5-2.5a3.5 3.5 0 0 1 4.95 0 .751.751 0 0 1-.018 1.042.751.751 0 0 1-1.042.018 1.998 1.998 0 0 0-2.83 0l-2.5 2.5a1.998 1.998 0 0 0 0 2.83Z"></path></svg></a>Usage</h2>
<ul dir="auto">
<li>
<p dir="auto">Place a BibTeX <a href="https://github.com/JuliaQuantumControl/QuantumCitations.jl/blob/master/docs/src/refs.bib"><code>refs.bib</code></a> file in the <code>docs/src</code> folder of your project.  Then, in <a href="https://github.com/JuliaQuantumControl/QuantumCitations.jl/blob/master/docs/make.jl"><code>docs/make.jl</code></a>, instantiate the <code>CitationBibliography</code> plugin and pass it to <a href="https://documenter.juliadocs.org/stable/lib/public/#Documenter.makedocs" rel="nofollow"><code>makedocs</code></a>:</p>
<div class="highlight highlight-source-julia notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="using QuantumCitations

bib = CitationBibliography(joinpath(@__DIR__, &quot;src&quot;, &quot;refs.bib&quot;))
makedocs(bib, ...)"><pre><span class="pl-k">using</span> QuantumCitations

bib <span class="pl-k">=</span> <span class="pl-c1">CitationBibliography</span>(<span class="pl-c1">joinpath</span>(<span class="pl-c1">@__DIR__</span>, <span class="pl-s"><span class="pl-pds">"</span>src<span class="pl-pds">"</span></span>, <span class="pl-s"><span class="pl-pds">"</span>refs.bib<span class="pl-pds">"</span></span>))
<span class="pl-c1">makedocs</span>(bib, <span class="pl-k">...</span>)</pre></div>
</li>
<li>
<p dir="auto">Optional, but recommended: <a href="https://juliaquantumcontrol.github.io/QuantumCitations.jl/stable/styling/" rel="nofollow">add CSS to properly format the bibliography</a></p>
</li>
<li>
<p dir="auto">Somewhere in your documentation include a markdown block</p>
<div class="highlight highlight-source-gfm notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="```@bibliography
```"><pre><span class="pl-c1"><span class="pl-c1">```@bibliography</span></span>
<span class="pl-c1"><span class="pl-c1">```</span></span></pre></div>
<p dir="auto">that will expand into a bibliography for all citations in the documentation.</p>
</li>
<li>
<p dir="auto">Anywhere in the documentation or in docstrings, insert citations as, e.g., <code>[GoerzQ2022](@cite)</code>, which will be rendered as "<a href="#screenshot">[2]</a>" and link to the full reference in the bibliography.</p>
</li>
</ul>
<p dir="auto">See the <a href="https://juliaquantumcontrol.github.io/QuantumCitations.jl" rel="nofollow">documentation</a> for additional information.</p>
