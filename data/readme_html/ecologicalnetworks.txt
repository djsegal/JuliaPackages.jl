<p dir="auto">This <code>julia</code> package provides a common interface to analyze all types of data
on ecological networks. It is designed to be general, easy to expand, and work
on bipartite/unipartite as well as deterministic/quantitative/probabilistic
networks. The current version is compatible with <code>julia</code> version 1.5 and up.</p>

<p dir="auto">Install (from the package mode):</p>
<div class="snippet-clipboard-content notranslate position-relative overflow-auto" data-snippet-clipboard-copy-content="add EcologicalNetworks"><pre class="notranslate"><code>add EcologicalNetworks
</code></pre></div>
<p dir="auto">Optionally, if you want to visualize networks, install</p>
<div class="snippet-clipboard-content notranslate position-relative overflow-auto" data-snippet-clipboard-copy-content="add EcologicalNetworksPlots"><pre class="notranslate"><code>add EcologicalNetworksPlots
</code></pre></div>
<p dir="auto">To access network data stored in <a href="http://mangal.io" rel="nofollow">mangal.io</a>, install</p>
<div class="snippet-clipboard-content notranslate position-relative overflow-auto" data-snippet-clipboard-copy-content="add Mangal"><pre class="notranslate"><code>add Mangal
</code></pre></div>
<p dir="auto">That's it. Now head over to the
<a href="http://poisotlab.github.io/EcologicalNetworks.jl/stable/" rel="nofollow">documentation</a>.</p>
