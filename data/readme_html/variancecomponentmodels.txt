<p dir="auto">This <a href="http://julialang.org/" rel="nofollow">Julia</a> package provides computational routines for fitting and testing variance component models. VarianceComponentModels is one package of the umbrella <a href="https://openmendel.github.io" rel="nofollow">OpenMendel</a> project.</p>

<a href="https://julialang.org/downloads/" rel="nofollow">https://julialang.org/downloads/</a> or by building Julia from the sources in the
