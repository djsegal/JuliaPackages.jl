<p dir="auto">Collection of tools for processing <a href="https://www.khronos.org/spir/" rel="nofollow">SPIR-V</a> binaries. It intends to provide some features of <a href="https://github.com/KhronosGroup/SPIRV-Tools">SPIR-V Tools</a> and <a href="https://github.com/KhronosGroup/SPIRV-Cross">SPIR-V Cross</a>, rewritten in Julia.</p>
<p dir="auto">This is currently a work in progress. The <a href="https://serenity4.github.io/SPIRV.jl/dev" rel="nofollow">current documentation</a> may contain some information, but is still lacking. Potential users are encouraged to look at the tests and at the source code for more information.</p>
<p dir="auto">This library contains a very experimental Julia to SPIR-V compiler, built as an alternative SPIR-V backend to the Julia compilation pipeline. Some of the internal code used for integrating with the compilation pipeline was inspired from <a href="https://github.com/JuliaGPU/GPUCompiler.jl">GPUCompiler.jl</a>; however, since we do not go through LLVM at all, most other things from GPUCompiler are not applicable.</p>
