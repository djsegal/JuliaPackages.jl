<p>Installation is recommended via the <code>ImageIO</code> thin IO wrapper for <code>FileIO</code>:</p>
<div class="highlight highlight-source-julia"><pre>pkg<span class="pl-k">&gt;</span> add ImageIO  <span class="pl-c"><span class="pl-c">#</span> Press ']' to enter te Pkg REPL mode.</span></pre></div>
<h2><a id="user-content-usage" class="anchor" aria-hidden="true" href="#usage"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path fill-rule="evenodd" d="M4 9h1v1H4c-1.5 0-3-1.69-3-3.5S2.55 3 4 3h4c1.45 0 3 1.69 3 3.5 0 1.41-.91 2.72-2 3.25V8.59c.58-.45 1-1.27 1-2.09C10 5.22 8.98 4 8 4H4c-.98 0-2 1.22-2 2.5S3 9 4 9zm9-3h-1v1h1c1 0 2 1.22 2 2.5S13.98 12 13 12H9c-.98 0-2-1.22-2-2.5 0-.83.42-1.64 1-2.09V6.25c-1.09.53-2 1.84-2 3.25C6 11.31 7.55 13 9 13h4c1.45 0 3-1.69 3-3.5S14.5 6 13 6z"></path></svg></a>Usage</h2>
<p>Once <code>ImageIO</code> is installed, usage is as simple as:</p>
<div class="highlight highlight-source-julia"><pre><span class="pl-k">using</span> FileIO
<span class="pl-c1">save</span>(<span class="pl-s"><span class="pl-pds">"</span>img.png<span class="pl-pds">"</span></span>, <span class="pl-c1">rand</span>(Gray, <span class="pl-c1">100</span>, <span class="pl-c1">100</span>))
img <span class="pl-k">=</span> <span class="pl-c1">load</span>(<span class="pl-s"><span class="pl-pds">"</span>img.png<span class="pl-pds">"</span></span>)</pre></div>
<p>Or direct usage, if <code>PNGFiles</code> has been directly installed:</p>
<div class="highlight highlight-source-julia"><pre><span class="pl-k">using</span> PNGFiles
PNGFiles<span class="pl-k">.</span><span class="pl-c1">save</span>(<span class="pl-s"><span class="pl-pds">"</span>img.png<span class="pl-pds">"</span></span>, <span class="pl-c1">rand</span>(Gray, <span class="pl-c1">100</span>, <span class="pl-c1">100</span>))
img <span class="pl-k">=</span> PNGFiles<span class="pl-k">.</span><span class="pl-c1">load</span>(<span class="pl-s"><span class="pl-pds">"</span>img.png<span class="pl-pds">"</span></span>)</pre></div>
