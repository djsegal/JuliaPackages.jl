<p dir="auto">Kinematic1D.jl is a Julia registered package.
See the <a href="https://github.com/CliMA/CloudMicrophysics.jl/blob/main/Project.toml">Project.toml</a>
for a full list of dependencies.
When using the Kinematic1D.jl,
the easiest way to obtain the package dependencies
is to use the Julia built-in package manager
(accessed by pressing <code>]</code> in the Julia REPL):</p>
<div class="highlight highlight-source-shell notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="julia --project

julia&gt; ]
pkg&gt; instantiate"><pre>julia --project

julia<span class="pl-k">&gt;</span> ]
pkg<span class="pl-k">&gt;</span> instantiate</pre></div>
<p dir="auto">Additional dependencies related to plotting and NetCDF output are
included in the <code>test</code> environment.
See the <a href="https://docs.julialang.org/en/v1/stdlib/Pkg/" rel="nofollow">Pkg docs</a>
for an overview of basic package manager features.</p>
<p dir="auto">The <code>KiD_driver.jl</code> inside <code>test/experiments/KiD_driver</code> folder is
the main file for running the simulations.
It accepts some command line arguments,
see the <code>--help</code> for details.</p>
<div class="highlight highlight-source-shell notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="julia --project=test/ test/experiments/KiD_driver.jl --help"><pre>julia --project=test/ test/experiments/KiD_driver.jl --help</pre></div>
<p dir="auto">An example command to run the <code>Kid_driver.jl</code> from terminal:</p>
<div class="highlight highlight-source-shell notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="julia --color=yes --project=test test/experiments/KiD_driver/KiD_driver.jl --moisture_choice=NonEquilibriumMoisture --precipitation_choice=Precipitation1M"><pre>julia --color=yes --project=test test/experiments/KiD_driver/KiD_driver.jl --moisture_choice=NonEquilibriumMoisture --precipitation_choice=Precipitation1M</pre></div>
<p dir="auto">In addition to simulating the 1D rainshaft, Kinematic1D.jl provides tools for calibrating microphysics parameters against availabkle data. The main program for running calibrations is given inside <code>test/experiments/calibrations</code> folder. This program is accompanied by the <code>config.jl</code> file that defines all the settings for the dynamics, observations, optimization process, and parameters to be calibrated. To run calibrations of microphysics schemes by using Kinematic1D the <code>config.jl</code> file needs to be adjusted. Then the program can be called from terminal:</p>
<div class="highlight highlight-source-shell notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="julia --color=yes --project=test test/experiments/calibrations/run_calibrations.jl"><pre>julia --color=yes --project=test test/experiments/calibrations/run_calibrations.jl</pre></div>
