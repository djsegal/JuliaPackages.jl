<p dir="auto">Explainable AI in Julia using <a href="https://fluxml.ai" rel="nofollow">Flux.jl</a>.</p>
<p dir="auto">This package implements interpretability methods and visualizations for neural networks, similar to <a href="https://github.com/pytorch/captum">Captum</a> and <a href="https://github.com/chr5tphr/zennit">Zennit</a> for PyTorch and <a href="https://github.com/albermax/innvestigate">iNNvestigate</a> for Keras models.</p>
