<p>LTspice.jl provides a julia interface to <a href="http://www.linear.com/designtools/software/#LTspice" rel="nofollow">LTspice<sup>TM</sup></a>.  Several interfaces are provided.</p>
<ol>
<li>A dictionary like interface to access parameters and measurements by name.</li>
<li>An array interface, which is primarily for measurements of stepped simulations.</li>
<li>Simulations can be called like functions.</li>
</ol>

<h2><a id="user-content-installation" class="anchor" aria-hidden="true" href="#installation"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path fill-rule="evenodd" d="M4 9h1v1H4c-1.5 0-3-1.69-3-3.5S2.55 3 4 3h4c1.45 0 3 1.69 3 3.5 0 1.41-.91 2.72-2 3.25V8.59c.58-.45 1-1.27 1-2.09C10 5.22 8.98 4 8 4H4c-.98 0-2 1.22-2 2.5S3 9 4 9zm9-3h-1v1h1c1 0 2 1.22 2 2.5S13.98 12 13 12H9c-.98 0-2-1.22-2-2.5 0-.83.42-1.64 1-2.09V6.25c-1.09.53-2 1.84-2 3.25C6 11.31 7.55 13 9 13h4c1.45 0 3-1.69 3-3.5S14.5 6 13 6z"></path></svg></a>Installation</h2>
<p>LTspice.jl is currently unregistered.  It can be installed using <code>Pkg.clone</code>.</p>
<div class="highlight highlight-source-julia"><pre>Pkg<span class="pl-k">.</span><span class="pl-c1">clone</span>(<span class="pl-s"><span class="pl-pds">"</span>https://github.com/cstook/LTspice.jl.git<span class="pl-pds">"</span></span>)</pre></div>
<p>The <a href="http://docs.julialang.org" rel="nofollow">julia documentation</a> section on installing unregistered packages provides more information.</p>
<p>LTspice.jl is compatible with julia 1.0</p>
<h2><a id="user-content-example-1" class="anchor" aria-hidden="true" href="#example-1"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path fill-rule="evenodd" d="M4 9h1v1H4c-1.5 0-3-1.69-3-3.5S2.55 3 4 3h4c1.45 0 3 1.69 3 3.5 0 1.41-.91 2.72-2 3.25V8.59c.58-.45 1-1.27 1-2.09C10 5.22 8.98 4 8 4H4c-.98 0-2 1.22-2 2.5S3 9 4 9zm9-3h-1v1h1c1 0 2 1.22 2 2.5S13.98 12 13 12H9c-.98 0-2-1.22-2-2.5 0-.83.42-1.64 1-2.09V6.25c-1.09.53-2 1.84-2 3.25C6 11.31 7.55 13 9 13h4c1.45 0 3-1.69 3-3.5S14.5 6 13 6z"></path></svg></a><a href="https://github.com/cstook/LTspice.jl/blob/master/examples/example%201/example1.ipynb">Example 1</a></h2>
<p><a target="_blank" rel="noopener noreferrer" href="https://github.com/cstook/LTspice.jl/blob/master/examples/example%201/example1.jpg"><img src="https://github.com/cstook/LTspice.jl/raw/master/examples/example%201/example1.jpg" style="max-width:100%;"></a></p>
<p>Import the module.</p>
<div class="highlight highlight-source-julia"><pre><span class="pl-k">using</span> LTspice</pre></div>
<p>Create an instance of LTspiceSimulation.</p>
<div class="highlight highlight-source-julia"><pre>example1 <span class="pl-k">=</span> <span class="pl-c1">LTspiceSimulation</span>(<span class="pl-s"><span class="pl-pds">"</span>example1.asc<span class="pl-pds">"</span></span>,tempdir<span class="pl-k">=</span><span class="pl-c1">true</span>)</pre></div>
<p>Access parameters and measurements using their name as the key.</p>
<p>Set a parameter to a new value.</p>
<div class="highlight highlight-source-julia"><pre>example1[<span class="pl-s"><span class="pl-pds">"</span>Rload<span class="pl-pds">"</span></span>] <span class="pl-k">=</span> <span class="pl-c1">20.0</span>  <span class="pl-c"><span class="pl-c">#</span> set parameter Rload to 20.0</span></pre></div>
<p>Read the resulting measurement.</p>
<div class="highlight highlight-source-julia"><pre>loadpower <span class="pl-k">=</span> example1[<span class="pl-s"><span class="pl-pds">"</span>Pload<span class="pl-pds">"</span></span>] <span class="pl-c"><span class="pl-c">#</span> run simulation, return Pload</span></pre></div>
<p>Circuit can be called like a function</p>
<div class="highlight highlight-source-julia"><pre>loadpower <span class="pl-k">=</span> <span class="pl-c1">example1</span>(<span class="pl-c1">100.0</span>)  <span class="pl-c"><span class="pl-c">#</span> pass Rload, return Pload</span></pre></div>
<p>Use <a href="https://github.com/JuliaOpt/Optim.jl">Optim.jl</a> to perform an optimization on a LTspice simulation</p>
<div class="highlight highlight-source-julia"><pre><span class="pl-k">using</span> Optim
result <span class="pl-k">=</span> <span class="pl-c1">optimize</span>(rload <span class="pl-k">-&gt;</span> <span class="pl-k">-</span><span class="pl-c1">example1</span>(rload)[<span class="pl-c1">1</span>],<span class="pl-c1">10.0</span>,<span class="pl-c1">100.0</span>)
rload_for_maximum_power <span class="pl-k">=</span> example1[<span class="pl-s"><span class="pl-pds">"</span>Rload<span class="pl-pds">"</span></span>]</pre></div>
