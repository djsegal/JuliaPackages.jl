<h2 dir="auto"><a id="user-content-overview" class="anchor" aria-hidden="true" href="#overview"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path d="m7.775 3.275 1.25-1.25a3.5 3.5 0 1 1 4.95 4.95l-2.5 2.5a3.5 3.5 0 0 1-4.95 0 .751.751 0 0 1 .018-1.042.751.751 0 0 1 1.042-.018 1.998 1.998 0 0 0 2.83 0l2.5-2.5a2.002 2.002 0 0 0-2.83-2.83l-1.25 1.25a.751.751 0 0 1-1.042-.018.751.751 0 0 1-.018-1.042Zm-4.69 9.64a1.998 1.998 0 0 0 2.83 0l1.25-1.25a.751.751 0 0 1 1.042.018.751.751 0 0 1 .018 1.042l-1.25 1.25a3.5 3.5 0 1 1-4.95-4.95l2.5-2.5a3.5 3.5 0 0 1 4.95 0 .751.751 0 0 1-.018 1.042.751.751 0 0 1-1.042.018 1.998 1.998 0 0 0-2.83 0l-2.5 2.5a1.998 1.998 0 0 0 0 2.83Z"></path></svg></a>Overview</h2>
<p dir="auto">This package implements algorithms for solving <strong>Generalized Constrained Shortest Paths</strong> problems. It implements a simplified version of the framework from <a href="https://arxiv.org/abs/1504.07880" rel="nofollow">Parmentier 2017</a>, restricted to acyclic directed graphs.</p>
<p dir="auto">Let <math-renderer class="js-inline-math" style="display: inline" data-static-url="https://github.githubassets.com/static" data-run-id="92080ca1b6b9b9a2eebe5f47037afd6e">$D=(V, A)$</math-renderer> an <strong>acyclic directed graph</strong>, <math-renderer class="js-inline-math" style="display: inline" data-static-url="https://github.githubassets.com/static" data-run-id="92080ca1b6b9b9a2eebe5f47037afd6e">$o, d\in V$</math-renderer> <strong>origin</strong> and <strong>destination</strong> vertices, <math-renderer class="js-inline-math" style="display: inline" data-static-url="https://github.githubassets.com/static" data-run-id="92080ca1b6b9b9a2eebe5f47037afd6e">$c$</math-renderer> a <strong>cost function</strong>, and <math-renderer class="js-inline-math" style="display: inline" data-static-url="https://github.githubassets.com/static" data-run-id="92080ca1b6b9b9a2eebe5f47037afd6e">$\mathcal{P} \subset \mathcal{P}_{od}$</math-renderer> a subset of <math-renderer class="js-inline-math" style="display: inline" data-static-url="https://github.githubassets.com/static" data-run-id="92080ca1b6b9b9a2eebe5f47037afd6e">$o-d$</math-renderer> paths in <math-renderer class="js-inline-math" style="display: inline" data-static-url="https://github.githubassets.com/static" data-run-id="92080ca1b6b9b9a2eebe5f47037afd6e">$G$</math-renderer>. This package can compute the corresponding <strong>constrained shortest path</strong>:</p>
<p dir="auto"><math-renderer class="js-display-math" style="display: block" data-static-url="https://github.githubassets.com/static" data-run-id="92080ca1b6b9b9a2eebe5f47037afd6e">$$
\boxed{\begin{aligned}
P^\star = \arg\min\quad &amp; c(P)\\
\text{s.t.}\quad &amp; P\in \mathcal{P}
\end{aligned}}
$$</math-renderer></p>
<p dir="auto">See the <a href="https://batyleo.github.io/ConstrainedShortestPaths.jl" rel="nofollow">documentation</a> for more details.</p>
