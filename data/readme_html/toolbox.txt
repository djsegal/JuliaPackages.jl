<p dir="auto">Integrate vector using weighted mean of two parabolas on each side</p>
<ul dir="auto">
<li><code>integ(x, y)</code></li>
</ul>
<p dir="auto">Cumulative integral using <code>integ</code>. Also possible to start from zero using different extrapolations (default is <code>:lin</code>)</p>
<ul dir="auto">
<li><code>cuminteg(x, y[; extrapolate_zero=:lin, :quad, :plaw, :none])</code></li>
</ul>
<p dir="auto">First derivative using weighted parabolas</p>
<ul dir="auto">
<li><code>deriv(x, y)</code></li>
</ul>
<p dir="auto">Cubic or linear interpolation and linear extrapolation</p>
<ul dir="auto">
<li><code>interp(x, y, val[; method=:cubic, :lin])</code></li>
</ul>
<p dir="auto">Binary search for arrays</p>
<ul dir="auto">
<li><code>locate(x, val)</code></li>
</ul>
<p dir="auto">Smooth vector using Gaussian kernel <code>N</code> times. Also possible to define offset so that only <code>x[offs+1:end-offs-1]</code> is smoothed ensuring proper boundary conditions.</p>
<ul dir="auto">
<li><code>smooth(x[, N=1; offs=3])</code> and <code>smooth!(x[, N=1; offs=3])</code></li>
</ul>
<p dir="auto">Smooth vector with B-splines (DeBoor's algorithm)</p>
<ul dir="auto">
<li><code>şmooth_spline(x, [weights], smoothfactor)</code></li>
</ul>
<p dir="auto">Smooth vector that has a powerlaw behaviour by not smoothing the vector values directly but by smoothing the ratio between it and some reference vector <code>ref</code>.</p>
<ul dir="auto">
<li><code>şmooth_plaw(x, ref, smoothfactor/N[; offs=3, method=:kernel/:spline])</code></li>
</ul>
<p dir="auto">Nodes and weights for <code>N</code> point Gaussian quadrature. Returns tuple of <code>(nodes, weights)</code></p>
<ul dir="auto">
<li><code>gauss_laguerre_nw(N)</code></li>
<li><code>gauss_legendre_nw(N)</code></li>
</ul>
<p dir="auto">Exponential integral E_N(x) = int_1^infty e^(-x t) dt/ t^N for positive arguments</p>
<ul dir="auto">
<li><code>expi(N, x)</code></li>
</ul>
<p dir="auto">Indexes of an array fulfilling given criteria <code>expr</code></p>
<ul dir="auto">
<li><code>@where expr</code> (TODO)</li>
</ul>
