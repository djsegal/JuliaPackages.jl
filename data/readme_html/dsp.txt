<p>DSP.jl provides a number of common <a href="https://en.wikipedia.org/wiki/Digital_signal_processing" rel="nofollow">digital signal processing</a> routines in Julia. These include:</p>
<ul>
<li><a href="https://juliadsp.github.io/DSP.jl/stable/periodograms" rel="nofollow">Periodogram estimation</a></li>
<li><a href="https://juliadsp.github.io/DSP.jl/stable/filters" rel="nofollow">Filter design and filtering</a></li>
<li><a href="http://juliadsp.github.io/DSP.jl/stable/estimation/" rel="nofollow">Estimation</a></li>
<li><a href="https://juliadsp.github.io/DSP.jl/stable/convolutions" rel="nofollow">Convolution</a></li>
<li><a href="https://juliadsp.github.io/DSP.jl/stable/lpc" rel="nofollow">Linear Predictive Coding</a></li>
<li><a href="https://juliadsp.github.io/DSP.jl/stable/windows" rel="nofollow">Window functions</a></li>
<li><a href="https://juliadsp.github.io/DSP.jl/stable/util" rel="nofollow">Utility functions</a></li>
</ul>
<p>More details can be found in the <a href="https://juliadsp.github.io/DSP.jl/stable/contents/" rel="nofollow">Online Documentation</a>.</p>
<p>The documentation for pre-releases (master) can be found <a href="https://juliadsp.github.io/DSP.jl/latest/contents/" rel="nofollow">here</a>.</p>
