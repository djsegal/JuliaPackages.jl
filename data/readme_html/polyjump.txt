<p dir="auto"><a href="https://github.com/jump-dev/PolyJuMP.jl">PolyJuMP.jl</a> is a <a href="https://github.com/jump-dev/JuMP.jl">JuMP</a>
extension for formulating and solving polynomial optimization problems.</p>
<p dir="auto">These problems can then be solved using <a href="https://github.com/jump-dev/SumOfSquares.jl">SumOfSquares.jl</a>.</p>
