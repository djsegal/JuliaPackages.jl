<p>To use the lattice library in your code, install it as specified below.
To use it in your code, start by importing the library</p>
<pre lang="julia-REPL"><code>julia&gt; using LatticePhysics
</code></pre>
<p>For tutorials and examples, see the <a href="https://github.com/janattig/LatticePhysics_Tutorials">LatticePhysics Tutorials</a> repository.</p>
<h2><a id="user-content-installation" class="anchor" aria-hidden="true" href="#installation"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path fill-rule="evenodd" d="M4 9h1v1H4c-1.5 0-3-1.69-3-3.5S2.55 3 4 3h4c1.45 0 3 1.69 3 3.5 0 1.41-.91 2.72-2 3.25V8.59c.58-.45 1-1.27 1-2.09C10 5.22 8.98 4 8 4H4c-.98 0-2 1.22-2 2.5S3 9 4 9zm9-3h-1v1h1c1 0 2 1.22 2 2.5S13.98 12 13 12H9c-.98 0-2-1.22-2-2.5 0-.83.42-1.64 1-2.09V6.25c-1.09.53-2 1.84-2 3.25C6 11.31 7.55 13 9 13h4c1.45 0 3-1.69 3-3.5S14.5 6 13 6z"></path></svg></a>Installation</h2>
<p>You can install the package via the package mode in Julia (Pkg). However, since the package
is not listed in the Julia package repositories, you have to first install the unregistered
dependencies manually with</p>
<div class="highlight highlight-source-julia"><pre>(v1<span class="pl-c1">.0</span>) pkg<span class="pl-k">&gt;</span> add <span class="pl-s"><span class="pl-pds">"</span>https://github.com/janattig/LatPhysBase.jl<span class="pl-pds">"</span></span>
(v1<span class="pl-c1">.0</span>) pkg<span class="pl-k">&gt;</span> add <span class="pl-s"><span class="pl-pds">"</span>https://github.com/janattig/LatPhysUnitcellLibrary.jl<span class="pl-pds">"</span></span>
(v1<span class="pl-c1">.0</span>) pkg<span class="pl-k">&gt;</span> add <span class="pl-s"><span class="pl-pds">"</span>https://github.com/janattig/LatPhysLatticeConstruction.jl<span class="pl-pds">"</span></span>
(v1<span class="pl-c1">.0</span>) pkg<span class="pl-k">&gt;</span> add <span class="pl-s"><span class="pl-pds">"</span>https://github.com/janattig/LatPhysLatticeModification.jl<span class="pl-pds">"</span></span>
(v1<span class="pl-c1">.0</span>) pkg<span class="pl-k">&gt;</span> add <span class="pl-s"><span class="pl-pds">"</span>https://github.com/janattig/LatPhysReciprocal.jl<span class="pl-pds">"</span></span>
(v1<span class="pl-c1">.0</span>) pkg<span class="pl-k">&gt;</span> add <span class="pl-s"><span class="pl-pds">"</span>https://github.com/janattig/LatPhysBandstructures.jl<span class="pl-pds">"</span></span>
(v1<span class="pl-c1">.0</span>) pkg<span class="pl-k">&gt;</span> add <span class="pl-s"><span class="pl-pds">"</span>https://github.com/janattig/LatPhysLuttingerTisza.jl<span class="pl-pds">"</span></span></pre></div>
<p>to finally install the main package with</p>
<div class="highlight highlight-source-julia"><pre>(v1<span class="pl-c1">.0</span>) pkg<span class="pl-k">&gt;</span> add <span class="pl-s"><span class="pl-pds">"</span>https://github.com/janattig/LatticePhysics.jl<span class="pl-pds">"</span></span></pre></div>
<h2><a id="user-content-sub-modules" class="anchor" aria-hidden="true" href="#sub-modules"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path fill-rule="evenodd" d="M4 9h1v1H4c-1.5 0-3-1.69-3-3.5S2.55 3 4 3h4c1.45 0 3 1.69 3 3.5 0 1.41-.91 2.72-2 3.25V8.59c.58-.45 1-1.27 1-2.09C10 5.22 8.98 4 8 4H4c-.98 0-2 1.22-2 2.5S3 9 4 9zm9-3h-1v1h1c1 0 2 1.22 2 2.5S13.98 12 13 12H9c-.98 0-2-1.22-2-2.5 0-.83.42-1.64 1-2.09V6.25c-1.09.53-2 1.84-2 3.25C6 11.31 7.55 13 9 13h4c1.45 0 3-1.69 3-3.5S14.5 6 13 6z"></path></svg></a>Sub-modules</h2>
<p>The main code is divided into various sub-modules, which can be found here:</p>
<ol>
<li><a href="https://github.com/janattig/LatPhysBase.jl.git">LatPhysBase</a></li>
<li><a href="https://github.com/janattig/LatPhysUnitcellLibrary.jl.git">LatPhysUnitcellLibrary</a></li>
<li><a href="https://github.com/janattig/LatPhysLatticeConstruction.jl.git">LatPhysLatticeConstruction</a></li>
<li><a href="https://github.com/janattig/LatPhysLatticeModification.jl.git">LatPhysLatticeModification</a></li>
<li><a href="https://github.com/janattig/LatPhysReciprocal.jl.git">LatPhysReciprocal</a></li>
<li><a href="https://github.com/janattig/LatPhysBandstructures.jl.git">LatPhysBandstructures</a></li>
<li><a href="https://github.com/janattig/LatPhysLuttingerTisza.jl.git">LatPhysLuttingerTisza</a></li>
</ol>
<p>Additionally, there are modules to provide plotting functions, which can be found here:</p>
<ol>
<li><a href="https://github.com/janattig/LatPhysPlottingPyPlot.jl.git">LatPhysPlottingPyPlot</a></li>
<li><a href="https://github.com/janattig/LatPhysPlottingSVG.jl.git">LatPhysPlottingSVG</a></li>
<li><a href="https://github.com/janattig/LatPhysReciprocalPlottingPyPlot.jl.git">LatPhysReciprocalPlottingPyPlot</a></li>
<li><a href="https://github.com/janattig/LatPhysBandstructuresPlottingPyPlot.jl.git">LatPhysBandstructuresPlottingPyPlot</a></li>
<li><a href="https://github.com/janattig/LatPhysLuttingerTiszaPlottingPyPlot.jl.git">LatPhysLuttingerTiszaPlottingPyPlot</a></li>
</ol>
<p>The plotting modules have to be used explicitly inside user code as they are not a direct
part of <code>LatticePhysics.jl</code>.</p>
