<h2><a id="user-content-installation" class="anchor" aria-hidden="true" href="#installation"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path fill-rule="evenodd" d="M4 9h1v1H4c-1.5 0-3-1.69-3-3.5S2.55 3 4 3h4c1.45 0 3 1.69 3 3.5 0 1.41-.91 2.72-2 3.25V8.59c.58-.45 1-1.27 1-2.09C10 5.22 8.98 4 8 4H4c-.98 0-2 1.22-2 2.5S3 9 4 9zm9-3h-1v1h1c1 0 2 1.22 2 2.5S13.98 12 13 12H9c-.98 0-2-1.22-2-2.5 0-.83.42-1.64 1-2.09V6.25c-1.09.53-2 1.84-2 3.25C6 11.31 7.55 13 9 13h4c1.45 0 3-1.69 3-3.5S14.5 6 13 6z"></path></svg></a>Installation</h2>
<p>To install the latest stable release, run:</p>
<pre><code>julia -e 'using Pkg; Pkg.add("ProbabilisticCircuits")'
</code></pre>
<p>To install from the latest commits of the package (recommented to also use latest commits of <code>LogicCircuits.jl</code>), run:</p>
<pre><code>julia -e 'using Pkg; Pkg.add([PackageSpec(url="https://github.com/Juice-jl/LogicCircuits.jl.git"),PackageSpec(url="https://github.com/Juice-jl/ProbabilisticCircuits.jl.git")])'
</code></pre>
<p>This will automatically install all dependencies described in <code>Project.toml</code>.
The first time you run <code>using ProbabilisticCircuits</code> in Julia, it will precompile the package and all its dependencies.</p>
<p><strong>Note</strong>: Currently <code>ProbabilisticCircuits</code> installation and build fails on Windows due to one of our dependencies (see <a href="https://github.com/Juice-jl/ProbabilisticCircuits.jl/issues/3">Issue 3</a> for more details). Additionally, on Linux you need to have a C++ compiler installed due to the same dependency (See <a href="https://github.com/mlewe/BlossomV.jl">BlossomV.jl</a>).</p>
