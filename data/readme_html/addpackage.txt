<p>An <code>@add</code> macro to automatically add a missing package when calling <code>using Package</code> in Julia.</p>
<p>Example:</p>
<div class="highlight highlight-source-julia position-relative" data-snippet-clipboard-copy-content="@add using Distributions
"><pre><span class="pl-c1">@add</span> <span class="pl-k">using</span> Distributions</pre></div>
<p>This will run <code>using Distributions</code> if it's already installed, otherwise will install the package, then run <code>using Distributions</code>.</p>
