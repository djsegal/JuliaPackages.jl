<p dir="auto">This repository provides:</p>
<ul dir="auto">
<li>Wrapper for typed and object-based interface of <a href="https://github.com/flame/blis">BLIS</a>.</li>
<li>Overwrite of <code>LinearAlgebra.BLAS</code> functions so that matrix
multiplications can also be redirected to the BLIS backend.</li>
<li>As BLIS itself is using actively templates and macros,
this package also aims to maximize usage of Julia's
metaprogramming features.</li>
</ul>

<p dir="auto">Basically by <code>using BLIS</code> one replaces <code>LinearAlgebra</code> with BLIS routines as its backends. This should allow direct computation (computation without copying) of generic strided matrices and mixed-datatype matrices. In addition to that BLIS' typed and object-based API are exposed as backends. See <a href="https://JuliaLinearAlgebra.github.io/BLIS.jl/dev" rel="nofollow">documentation</a> for more details.</p>
