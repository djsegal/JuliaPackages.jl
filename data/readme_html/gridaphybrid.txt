<p dir="auto">GridapHybrid.jl is fresh approach, based on Julia, towards  a unified computational framework that supports R&amp;D in hybrid discretization methods on general polytopal meshes, including HDG (Hybridizable Discontinuous Galerkin), HHO (Hybrid High Order), and VEM (Virtual Element Method), and highly scalable parallel preconditioning techniques (based, e.g., on hybrid multiscale ideas) tailored to these discretization techniques.</p>

<p dir="auto">GridapHybrid is a collaborative project open to contributions. If you want to contribute, please take into account:</p>
<ul dir="auto">
<li>Before opening a PR with a significant contribution, contact the project administrators by <a href="https://github.com/gridap/GridapHybrid.jl/issues/new">opening an issue</a> describing what you are willing to implement. Wait for feed-back from other community members.</li>
<li>We adhere to the contribution and code-of-conduct instructions of the Gridap.jl project, available <a href="https://github.com/gridap/Gridap.jl/blob/master/CONTRIBUTING.md">here</a> and <a href="https://github.com/gridap/Gridap.jl/blob/master/CODE_OF_CONDUCT.md">here</a>, resp.  Please, carefully read and follow the instructions in these files.</li>
<li>Open a PR with your contribution.</li>
</ul>
<p dir="auto">Want to help? We have <a href="https://github.com/gridap/GridapHybrid.jl/labels/help%20wanted">issues waiting for help</a>. You can start contributing to the GridapHybrid project by solving some of those issues.</p>
