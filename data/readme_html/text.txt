<p><a target="_blank" rel="noopener noreferrer" href="https://camo.githubusercontent.com/17ee20dcda53189de7fc70e1d651a0eb2f290a71/68747470733a2f2f7472617669732d63692e6f72672f6d69742d6e6c702f546578742e6a6c2e7376673f6272616e63683d6d6173746572"><img align="right" src="https://camo.githubusercontent.com/17ee20dcda53189de7fc70e1d651a0eb2f290a71/68747470733a2f2f7472617669732d63692e6f72672f6d69742d6e6c702f546578742e6a6c2e7376673f6272616e63683d6d6173746572" alt="Build Status" data-canonical-src="https://travis-ci.org/mit-nlp/Text.jl.svg?branch=master" style="max-width:100%;"></a></p>
<p>This package is a julia implementation of:</p>
<ol>
<li>Text classification based on BoW models (e.g. topic/langauge id)</li>
<li>Language ID (training and processing) based on word and character n-grams</li>
<li>Lewis's SMART stop list for English</li>
<li>tfidf/tfllr text feature normalization</li>
<li>ngram feature extractors</li>
</ol>
