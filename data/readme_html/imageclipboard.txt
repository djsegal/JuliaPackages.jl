<p dir="auto">Just like <code>clipboard</code> in InteractiveUtils deals with string contents, this package provides <code>clipboard_img</code> to work with image contents:</p>
<div class="highlight highlight-source-julia notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="using Images, ImageClipboard

# Create a random image
img = rand(RGB{N0f8}, 100, 200)

# Copy the image to the clipboard
clipboard_img(img)

# Paste the image from the clipboard
img2 = clipboard_img()

# These are the same images
img == img2  # true"><pre><span class="pl-k">using</span> Images, ImageClipboard

<span class="pl-c"><span class="pl-c">#</span> Create a random image</span>
img <span class="pl-k">=</span> <span class="pl-c1">rand</span>(RGB{N0f8}, <span class="pl-c1">100</span>, <span class="pl-c1">200</span>)

<span class="pl-c"><span class="pl-c">#</span> Copy the image to the clipboard</span>
<span class="pl-c1">clipboard_img</span>(img)

<span class="pl-c"><span class="pl-c">#</span> Paste the image from the clipboard</span>
img2 <span class="pl-k">=</span> <span class="pl-c1">clipboard_img</span>()

<span class="pl-c"><span class="pl-c">#</span> These are the same images</span>
