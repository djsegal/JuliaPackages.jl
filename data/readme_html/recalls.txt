<p dir="auto">Recording calls:</p>
<ul dir="auto">
<li><code>@recall function f(...) ... end</code></li>
<li><code>@recall f(...)</code></li>
<li><code>recall()</code>: replay last record created by <code>@recall</code>. To debug last record,
use <code>@run recall()</code>.</li>
<li><code>Recalls.CALLS</code>: a vector of records created by <code>@recall</code>.</li>
</ul>
<p dir="auto">Recording variables:</p>
<ul dir="auto">
<li><code>@note v₁ v₂ ... vₙ</code>: record variables.</li>
<li><code>Recalls.NOTES</code>: a vector of records created by <code>@note</code>.</li>
</ul>
