<p dir="auto">ProgressiveHedging.jl is a basic implementation of the <a href="https://pdfs.semanticscholar.org/4ab8/028748c89b226fd46cf9f45de88218779572.pdf" rel="nofollow">Progressive Hedging algorithm</a> which is used to solve stochastic programs.</p>
<p dir="auto">ProgressiveHedging.jl makes use of the <a href="https://github.com/JuliaOpt/JuMP.jl">JuMP</a> framework to build and solve the subproblems.  It can be used with Julia's Distributed package to solve problems in parallel.</p>
<p dir="auto">See the <a href="https://NREL.github.io/ProgressiveHedging.jl/stable" rel="nofollow">documentation</a> or the examples directory for examples.</p>
