<p dir="auto">Compute <a href="https://en.wikipedia.org/wiki/Kullback%E2%80%93Leibler_divergence" rel="nofollow">KL Divergences</a> using <a href="https://github.com/JuliaStats/Distributions.jl">Distributions.jl</a> objects either analytically (PR are welcome to enrich the library) or via Monte Carlo sampling.</p>
<p dir="auto">There is only one function exported : <code>KL</code> which takes two arguments <code>p</code> and <code>q</code> for an analytical formulation and an additional argument <code>n_samples</code> for the sampling based approach.</p>
<p dir="auto">The list of the pair of distributions supported analytically is given here</p>
