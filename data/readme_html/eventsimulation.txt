<p><em>An event based Discrete Event Simulation engine for Julia.</em></p>
<p>Intended as a support library for teaching basic principles of Discrete Event Simulation.</p>
<p>The last package version that supports Julia 0.6.2 is 0.3.3. Versions starting with 0.4.0 support Julia 0.7+.
Starting from version 0.7.0 the package is migrated to Julua 1.0.</p>
