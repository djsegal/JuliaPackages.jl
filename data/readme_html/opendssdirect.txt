<p>OpenDSSDirect.jl is a cross-platform Julia package implements a "direct" library interface to <a href="http://smartgrid.epri.com/SimulationTool.aspx" rel="nofollow">OpenDSS</a> using <a href="https://github.com/dss-extensions/dss_capi/">DSS C-API</a>.</p>
<p>OpenDSS is an open-source distribution system simulator.</p>
<p>Originally, the package used the OpenDSSDirect library from the official OpenDSS repository. Since OpenDSS is officially Windows-only, this package was migrated to use DSS C-API in 2019. The DSS C-API library is a community implementation of the OpenDSS engine built with the Free Pascal compiler, targetting cross-platform compatibility and an extended API. See its repository <a href="https://github.com/dss-extensions/dss_capi">here</a> for the full Pascal/Delphi code, or <a href="https://dss-extensions.org/" rel="nofollow">https://dss-extensions.org/</a> for an overview of the projects around it.</p>
<p>See also <a href="https://github.com/dss-extensions/OpenDSSDirect.py">OpenDSSDirect.py</a> for a package in Python with similar calling style, or <a href="https://github.com/dss-extensions/dss_python">DSS_Python</a> and <a href="https://github.com/dss-extensions/dss_matlab">DSS_MATLAB</a> for packages that mimic the OpenDSS COM API organization and style.</p>
<p><strong>This package is now available for Windows, Mac, and Linux.</strong></p>
