<p dir="auto"><a href="https://github.com/dss-extensions/OpenDSSDirect.jl/actions/workflows/ci.yml"><img src="https://github.com/dss-extensions/OpenDSSDirect.jl/workflows/CI/badge.svg" alt="GitHub Actions: Tests" style="max-width: 100%;"></a></p>
<p dir="auto">OpenDSSDirect.jl is a cross-platform Julia package implements a "direct" library interface to <a href="http://smartgrid.epri.com/SimulationTool.aspx" rel="nofollow">OpenDSS</a> using <a href="https://github.com/dss-extensions/dss_capi/">DSS C-API</a>.</p>
<p dir="auto">OpenDSS is an open-source distribution system simulator.</p>
<p dir="auto">Originally, the package used the OpenDSSDirect library from the official OpenDSS repository. Since OpenDSS is officially Windows-only, this package was migrated to use DSS C-API in 2019. The DSS C-API library is a community implementation of the OpenDSS engine built with the Free Pascal compiler, targetting cross-platform compatibility and an extended API. See its repository <a href="https://github.com/dss-extensions/dss_capi">here</a> for the full Pascal/Delphi code, or <a href="https://dss-extensions.org/" rel="nofollow">https://dss-extensions.org/</a> for an overview of the projects around it.</p>
<p dir="auto">See also <a href="https://github.com/dss-extensions/OpenDSSDirect.py">OpenDSSDirect.py</a> for a package in Python with similar calling style, or <a href="https://github.com/dss-extensions/dss_python">DSS_Python</a> and <a href="https://github.com/dss-extensions/dss_matlab">DSS_MATLAB</a> for packages that mimic the OpenDSS COM API organization and style.</p>
<p dir="auto"><strong>This package is now available for Windows, Mac, and Linux.</strong></p>

<p dir="auto">This package interfaces with the OpenDSS engine using a library interface, so a good understanding of OpenDSS will help troubleshooting.
There are plenty of useful resources located <a href="https://sourceforge.net/p/electricdss/code/HEAD/tree/trunk/Version8/Doc/" rel="nofollow">here</a>.</p>
