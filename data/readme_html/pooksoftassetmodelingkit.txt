<p><code>PooksoftAssetModelingKit.jl</code> is a <a href="https://www.julialang.org" rel="nofollow">Julia</a> package to model underlying equity pricing as a function of time. <code>PooksoftAssetModelingKit.jl</code> is a member of the <a href="http://www.pooksoft.com" rel="nofollow">Project Serenity</a> collection of packages from <a href="http://www.pooksoft.com" rel="nofollow">Pooksoft</a>, along with <a href="https://github.com/Pooksoft/PooksoftOptionsKit.jl">PooksoftOptionsKit.jl</a> and
<a href="https://github.com/Pooksoft/PooksoftAlphaVantageDataStore.jl">PooksoftAlphaVantageDataStore.jl</a> packages.</p>
<h2><a id="user-content-installation-and-requirements" class="anchor" aria-hidden="true" href="#installation-and-requirements"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path fill-rule="evenodd" d="M7.775 3.275a.75.75 0 001.06 1.06l1.25-1.25a2 2 0 112.83 2.83l-2.5 2.5a2 2 0 01-2.83 0 .75.75 0 00-1.06 1.06 3.5 3.5 0 004.95 0l2.5-2.5a3.5 3.5 0 00-4.95-4.95l-1.25 1.25zm-4.69 9.64a2 2 0 010-2.83l2.5-2.5a2 2 0 012.83 0 .75.75 0 001.06-1.06 3.5 3.5 0 00-4.95 0l-2.5 2.5a3.5 3.5 0 004.95 4.95l1.25-1.25a.75.75 0 00-1.06-1.06l-1.25 1.25a2 2 0 01-2.83 0z"></path></svg></a>Installation and Requirements</h2>
<p><code>PooksoftAssetModelingKit.jl</code> requires <a href="https://julialang.org/downloads/" rel="nofollow">Julia 1.5.x</a> or above.
<code>PooksoftAssetModelingKit.jl</code> is organized as a <a href="http://julialang.org" rel="nofollow">Julia</a> package which
can be installed in the <code>package mode</code> of Julia.</p>
<p>To install <code>PooksoftAssetModelingKit.jl</code> in your project <code>my_project</code>,
start the <a href="https://docs.julialang.org/en/v1/stdlib/REPL/index.html" rel="nofollow">Julia REPL</a> in the root directory of
<code>my_project</code> and enter the <code>package mode</code> using the <code>]</code> key (to get back press the <code>backspace</code> or <code>^C</code> keys).
<code>Package mode</code> is the build in package management system in <a href="http://julialang.org" rel="nofollow">Julia</a>.
<code>Package mode</code> documentation can be found <a href="https://docs.julialang.org/en/v1/stdlib/Pkg/" rel="nofollow">here</a>.
Once in package mode prompt enter the commands:</p>
<div class="snippet-clipboard-content position-relative" data-snippet-clipboard-copy-content="(v1.5) pkg&gt; activate .
(v1.5) pkg&gt; add https://github.com/Pooksoft/PooksoftAssetModelingKit.jl.git
"><pre><code>(v1.5) pkg&gt; activate .
(v1.5) pkg&gt; add https://github.com/Pooksoft/PooksoftAssetModelingKit.jl.git
</code></pre></div>
<p>This will <a href="https://julialang.github.io/Pkg.jl/v1/api/#Pkg.activate" rel="nofollow">activate</a> <code>my_project</code>,
and install the <code>PooksoftAssetModelingKit.jl</code> package into the context of <code>my_project</code>.
To install other required packages, simply add them using the <code>add</code> command.</p>
