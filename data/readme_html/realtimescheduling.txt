<p dir="auto">Real-time systems modeling and schedulability analysis</p>
<p dir="auto"><a target="_blank" rel="noopener noreferrer" href="docs/src/assets/logo.svg"><img src="docs/src/assets/logo.svg" alt="Logo" style="max-width: 100%;"></a></p>
<p dir="auto">This package aims to provide useful tools for writing schedulability studies
with Julia.  It provides basic functionality for schedulability testing,
response time analysis, schedule simulation, and schedule plotting.
It is inspired by <a href="https://github.com/brandenburg/schedcat">SchedCAT</a> by BjÃ¶rn

<p dir="auto">To foster more research collaboration, and enable broader code reuse, you are
encouraged to submit your high-quality research code using this package to the
<code>RealTimeScheduling.Papers</code> submodule.  If your code is very broadly
applicable (e.g. new schedulability analysis for GEDF), then we would welcome
it directly to the <code>RealTimeScheduling</code> package instead.</p>
