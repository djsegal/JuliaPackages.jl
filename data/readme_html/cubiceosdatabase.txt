<p dir="auto"><em>Loading parameters of equations of state from plain text.</em></p>
<p dir="auto">The package provides interface to separated-values files describing <em>parameters for cubic equations of state</em>.
The package is <strong>free of third-party text parsers</strong> like <a href="https://github.com/JuliaData/CSV.jl">CSV.jl</a>.</p>
<p dir="auto">Currently, the most described equation of state is <strong>Brusilovsky</strong>'s (general) equation of state [<a href="https://doi.org/10.2118/20180-PA" rel="nofollow">Brusilovsky SPE Reservoir Engineering 1992</a>], which is easily transformed into common-known <strong>Peng-Robinson</strong> or <strong>Soave-Redlich-Kwong</strong> equations. Originally, the package is aimed at modelling pure alkanes and its mixtures.</p>
<p dir="auto">Package's <strong>documentation</strong> is available <a href="https://stepanzh.github.io/CubicEoSDatabase.jl/" rel="nofollow">here</a>.</p>
<p dir="auto">The package comes with <strong>Out-of-box database</strong> of gas properties and eos parameters.
For a quick preview of embedded substances see <a href="https://stepanzh.github.io/CubicEoSDatabase.jl/stable/outofboxdb/" rel="nofollow">this</a>.
The database can be accessed from <code>CubicEoSDatabase.Data</code> submodule. See <code>help?&gt; CubicEoSDatabase.Data</code>.</p>
<p dir="auto">For <strong>minimal working examples</strong> see <code>test/example.jl</code>.</p>

<ol dir="auto">
<li>Go to Julia REPL;</li>
<li>Activate packager mode (press <code>]</code>);</li>
