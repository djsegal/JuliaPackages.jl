<p>GtkObservables is designed to simplify the creation of graphical user interfaces (GUIs) using <a href="https://github.com/JuliaGraphics/Gtk.jl">Gtk</a> and Julia.
GtkObservables descended from
<a href="https://github.com/jverzani/GtkInteract.jl">GtkInteract</a>, and mimics
aspects of the design of
<a href="https://github.com/JuliaGizmos/Interact.jl">Interact</a>. To learn how
to use this package, please see the documentation:</p>
<p><a href="https://juliagizmos.github.io/GtkObservables.jl/stable" rel="nofollow"><img src="https://camo.githubusercontent.com/c97f0a5f2ae95755f64a27f1aa8d9a17462941fd3d6c907c7630abd5d3e60acf/68747470733a2f2f696d672e736869656c64732e696f2f62616467652f646f63732d737461626c652d626c75652e737667" alt="" data-canonical-src="https://img.shields.io/badge/docs-stable-blue.svg" style="max-width:100%;"></a>
<a href="https://juliagizmos.github.io/GtkObservables.jl/latest" rel="nofollow"><img src="https://camo.githubusercontent.com/56f8252ba8e9d3f0b810769543f77823d2fe031ce560d4c2d69fb1fcad800383/68747470733a2f2f696d672e736869656c64732e696f2f62616467652f646f63732d6c61746573742d626c75652e737667" alt="" data-canonical-src="https://img.shields.io/badge/docs-latest-blue.svg" style="max-width:100%;"></a></p>
<p>GtkObservables is probably a more future-oriented package than the similar <a href="https://github.com/JuliaGizmos/GtkReactive.jl">GtkReactive</a> upon which this started.</p>
