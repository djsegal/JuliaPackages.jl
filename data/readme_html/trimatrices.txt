<p dir="auto">Julia package for storing large triangular or symmetric matrices in non-redundant format.</p>
<p dir="auto"><code>TriMatrix</code> is a subtype of <code>AbstractMatrix</code> which stores only the values on one side of the
diagonal, thus reducing the memory required by about half.
The data is stored in an underlying 1D array.
It is intended for large matrices where the amount of memory saved is significant.</p>
