<p>This package contains distributed block operators and vectors for
<a href="https://github.com/ChevronETC/Jets.jl">Jets.jl</a>.  It builds on top
of the block operators in Jets.jl, providing a parallel distributed
version of block operators and block vectors that are used to orchestrate
distributed (in-memory) storage and compute.</p>
