<p dir="auto"><a href="https://github.com/JuliaApproximation/ApproxFun.jl">ApproxFun.jl</a> is a package for approximating functions. This package implements singularity based spaces to ApproxFun, including <code>JacobiWeight</code> and <code>LogWeight</code>.</p>
