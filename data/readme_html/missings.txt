<p dir="auto">Starting from Julia 1.0, the <code>Missing</code> type and basic related functionality are part of the language.
For documentation see <a href="https://docs.julialang.org/en/v1/manual/missing/index.html" rel="nofollow">the Julia manual section on missing values</a>.</p>
<p dir="auto">This package provides additional functionality for working with <code>missing</code> values:</p>
<ul dir="auto">
<li><code>nonmissingtype</code> to extract <code>T</code> from a <code>Union{T, Missing}</code> type</li>
<li><code>allowmissing</code> and <code>disallowmissing</code> to convert between <code>Vector{T}</code> and <code>Vector{Union{T, Missing}}</code></li>
<li><code>passmissing</code> to wrap a function so that it returns <code>missing</code> if any of its positional arguments is <code>missing</code></li>
<li><code>levels</code> to get the unique values in a vector excluding <code>missing</code> and in their preferred order</li>
<li><code>Missings.replace</code> to wrap a collection in a (possibly indexable) iterator replacing <code>missing</code> with another value</li>
<li><code>Missings.fail</code> to wrap a collection in a (possibly indexable) iterator throwing an error if <code>missing</code> is encountered</li>
<li><code>skipmissings</code> to loop through a collection of iterators excluding indices where any iterators are <code>missing</code></li>
</ul>
