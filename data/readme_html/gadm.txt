<p dir="auto">This package provides polygons/multipolygons for all countries and their sub-divisions from the <a href="https://gadm.org/" rel="nofollow">GADM dataset</a>.
It fetches the data dynamically from the officially hosted database using <a href="https://github.com/oxinabox/DataDeps.jl">DataDeps.jl</a>.</p>

<ul dir="auto">
<li><strong>Country Code</strong> follows the ISO 3166 Alpha 3 standard, you can find the code for your country <a href="https://en.wikipedia.org/wiki/ISO_3166-1_alpha-3" rel="nofollow">here</a>.</li>
<li>Other regions require "approximately" official names that are at least contained (case-insensitive) in the official name.</li>
</ul>
<p dir="auto">The coordinate reference system is longitude/latitude and the WGS84 datum.</p>

<p dir="auto">GADM, the Database of Global Administrative Areas, is a high-resolution database of country administrative areas, with a goal of "all countries, at all levels, at any time period." The database is available in a few export formats, including shapefiles that are used in most common GIS applications.</p>
<p dir="auto">Please read their license at <a href="https://gadm.org/license.html" rel="nofollow">https://gadm.org/license.html</a> which is different than the MIT license of the GADM.jl package.</p>
