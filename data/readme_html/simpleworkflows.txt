<p dir="auto">Build workflows from jobs. Run, monitor, and get results from them.
This package takes inspiration from packages like
<a href="https://github.com/cihga39871/JobSchedulers.jl">JobSchedulers</a> and
<a href="https://github.com/invenia/Dispatcher.jl">Dispatcher</a> (unmaintained).</p>
<p dir="auto">Please <a href="https://doi.org/10.1016/j.cpc.2022.108515" rel="nofollow">cite this package</a> as:</p>
<p dir="auto">Q. Zhang, C. Gu, J. Zhuang et al., <code>express</code>: extensible, high-level workflows for swifter <em>ab initio</em> materials modeling, <em>Computer Physics Communications</em>, 108515, doi: <a href="https://doi.org/10.1016/j.cpc.2022.108515" rel="nofollow">https://doi.org/10.1016/j.cpc.2022.108515</a>.</p>

<p dir="auto">The package can be installed with the Julia package manager.
From the Julia REPL, type <code>]</code> to enter the Pkg REPL mode and run:</p>
<div class="snippet-clipboard-content notranslate position-relative overflow-auto" data-snippet-clipboard-copy-content="pkg&gt; add SimpleWorkflows"><pre class="notranslate"><code>pkg&gt; add SimpleWorkflows
</code></pre></div>
<p dir="auto">Or, equivalently, via the <a href="https://pkgdocs.julialang.org/v1/getting-started/" rel="nofollow"><code>Pkg</code> API</a>:</p>
<div class="highlight highlight-source-julia notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="julia&gt; import Pkg; Pkg.add(&quot;SimpleWorkflows&quot;)"><pre>julia<span class="pl-k">&gt;</span> <span class="pl-k">import</span> Pkg; Pkg<span class="pl-k">.</span><span class="pl-c1">add</span>(<span class="pl-s"><span class="pl-pds">"</span>SimpleWorkflows<span class="pl-pds">"</span></span>)</pre></div>
