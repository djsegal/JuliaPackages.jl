<p>This package installs a private copy of <a href="https://nodejs.org/en/" rel="nofollow">Node.js</a> for use in julia packages. The package only exports two functions:</p>
<p><code>nodejs_cmd()</code> returns the full path of the node command.
<code>npm_cmd()</code> returns the full path of the npm command.</p>
<p>The return value of both functions can be interpolated directly into julia <code>Cmd</code> instances.</p>
