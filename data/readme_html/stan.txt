<p>V6.0.0 is a (mildly) breaking release primarily because StanSample now by default returns an array.
If e.g. an MCMCChains.Chains object is needed MCMCChains.jl must have been loaded and use:</p>
<pre><code>chns = read_samples(sm; output_format=:mcmcchains)
</code></pre>
<p>See <code>?read_samples</code> for other optional arguments.</p>
<p>Further work will happen on this package over the next few months, i.e. until at least April 2020 as the underlying StanJulia packages are being fine-tuned. Documentation - to be part of Stan.jl v6 - is also far from done. WORK IN PROGRESS!!!!</p>

<h2><a id="user-content-background-info" class="anchor" aria-hidden="true" href="#background-info"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path fill-rule="evenodd" d="M4 9h1v1H4c-1.5 0-3-1.69-3-3.5S2.55 3 4 3h4c1.45 0 3 1.69 3 3.5 0 1.41-.91 2.72-2 3.25V8.59c.58-.45 1-1.27 1-2.09C10 5.22 8.98 4 8 4H4c-.98 0-2 1.22-2 2.5S3 9 4 9zm9-3h-1v1h1c1 0 2 1.22 2 2.5S13.98 12 13 12H9c-.98 0-2-1.22-2-2.5 0-.83.42-1.64 1-2.09V6.25c-1.09.53-2 1.84-2 3.25C6 11.31 7.55 13 9 13h4c1.45 0 3-1.69 3-3.5S14.5 6 13 6z"></path></svg></a>Background info</h2>
<p>Stan.jl v4.x and earlier versions were the first generation of Julia tools to use Stan's <a href="https://mc-stan.org/users/interfaces/cmdstan.html" rel="nofollow">cmdstan executable</a>. CmdStan.jl v5.x and v6.x updated these for Julia v1.x and added features based on user feedback. Most of cmdstan's features are covered (but not all).</p>
<p>Stan.jl v6.x constitutes the third generation and extends Tamas Papp's approach taken in StanRun, StanDump and StanSamples. It covers all of cmdstan's features in separate modules. i.e. StanVariational, StanSample, etc., including an option to run <code>generate_quantities</code>.</p>
<p>Stan.jl v6.x will contain examples using the features available in <a href="https://github.com/StanJulia">StanJulia</a>.</p>
<p>My intention is to continue maintenance of CmdStan.jl for at least another two years.</p>

<p>A collection of documented examples to use Stan (as an external program) from Julia.</p>
<p>Stan's cmdstan executable needs to be installed separatedly. Please see <a href="https://stanjulia.github.io/Stan.jl/latest/INSTALLATION/" rel="nofollow">cmdstan installation</a>.</p>
<p>For more info on Stan, please go to <a href="http://mc-stan.org" rel="nofollow">http://mc-stan.org</a>.</p>
