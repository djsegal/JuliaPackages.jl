<h2><a id="user-content-purpose" class="anchor" aria-hidden="true" href="#purpose"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path fill-rule="evenodd" d="M7.775 3.275a.75.75 0 001.06 1.06l1.25-1.25a2 2 0 112.83 2.83l-2.5 2.5a2 2 0 01-2.83 0 .75.75 0 00-1.06 1.06 3.5 3.5 0 004.95 0l2.5-2.5a3.5 3.5 0 00-4.95-4.95l-1.25 1.25zm-4.69 9.64a2 2 0 010-2.83l2.5-2.5a2 2 0 012.83 0 .75.75 0 001.06-1.06 3.5 3.5 0 00-4.95 0l-2.5 2.5a3.5 3.5 0 004.95 4.95l1.25-1.25a.75.75 0 00-1.06-1.06l-1.25 1.25a2 2 0 01-2.83 0z"></path></svg></a>Purpose</h2>
<p>A collection of examples demonstrating to use Stan's cmdstan (as an external program) from Julia.</p>
<h2><a id="user-content-background-info" class="anchor" aria-hidden="true" href="#background-info"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path fill-rule="evenodd" d="M7.775 3.275a.75.75 0 001.06 1.06l1.25-1.25a2 2 0 112.83 2.83l-2.5 2.5a2 2 0 01-2.83 0 .75.75 0 00-1.06 1.06 3.5 3.5 0 004.95 0l2.5-2.5a3.5 3.5 0 00-4.95-4.95l-1.25 1.25zm-4.69 9.64a2 2 0 010-2.83l2.5-2.5a2 2 0 012.83 0 .75.75 0 001.06-1.06 3.5 3.5 0 00-4.95 0l-2.5 2.5a3.5 3.5 0 004.95 4.95l1.25-1.25a.75.75 0 00-1.06-1.06l-1.25 1.25a2 2 0 01-2.83 0z"></path></svg></a>Background info</h2>
<p>The first 2 generations of Stan.jl took a similar approach as the recently released <a href="https://mc-stan.org/cmdstanr/" rel="nofollow">CmdStanR</a> and <a href="https://github.com/stan-dev/cmdstanpy">CmdStanPy</a> options to use Stan's <a href="https://mc-stan.org/users/interfaces/cmdstan.html" rel="nofollow">cmdstan executable</a>.</p>
<p>Stan.jl v6.x constitutes the third generation and covers all of cmdstan's features in separate modules, i.e. StanVariational, StanSample, etc., including an option to run <code>generate_quantities</code>. In a sense, it extends Tamas Papp's approach taken in StanRun, StanDump and StanSamples.</p>
<p>Stan.jl v6.x will contain examples using the features available in <a href="https://github.com/StanJulia">StanJulia</a> component packages.</p>
<p>My intention is to continue maintenance of CmdStan.jl at least until late 2021.</p>

<p>Stan's cmdstan executable needs to be installed separatedly. Please see <a href="https://stanjulia.github.io/Stan.jl/latest/INSTALLATION/" rel="nofollow">cmdstan installation</a>.</p>
<p>For more info on Stan, please go to <a href="http://mc-stan.org" rel="nofollow">http://mc-stan.org</a>.</p>
