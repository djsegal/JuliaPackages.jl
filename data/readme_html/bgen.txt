<p dir="auto">Routines for reading compressed storage of genotyped or imputed markers</p>
<p dir="auto"><a href="https://en.wikipedia.org/wiki/Genome-wide_association_study" rel="nofollow"><em>Genome-wide association studies (GWAS)</em></a> data with imputed markers are often saved in the <a href="https://www.well.ox.ac.uk/~gav/bgen_format/" rel="nofollow"><strong>BGEN format</strong></a> or <code>.bgen</code> file.
It can store both hard calls and imputed data, unphased genotypes and phased haplotypes. Each variant is compressed separately to make indexing simple. An index file (<code>.bgen.bgi</code>) may be provided to access each variant easily. <a href="https://www.ukbiobank.ac.uk/" rel="nofollow">UK Biobank</a> uses this format for genome-wide imputed genotypes.</p>
