<p>This package solves least squares problem with high dimensional fixed effects. For a matrix <code>D</code> of high dimensional fixed effects, it finds <code>b</code> and <code>系</code> such that <code>y = D'b + 系</code> with <code>E[D系] = 0</code>.</p>
<p>It is the back end for the package <a href="https://github.com/matthieugomez/FixedEffectModels.jl">FixedEffectModels.jl</a>, that estimates linear models with high-dimensional fixed effect.</p>
<p>The package defines two functions <code>solve_coefficients!</code>, that returns the coefficients <code>b</code>, and <code>solve_residuals!</code>, that returns the residuals <code>系</code>. See the documentation <code>?solve_residuals!</code> or <code>?solve_coefficients!</code>.</p>

# define fixed effects:
p1 = FixedEffect(repeat(1:5, inner = 2))
# combine fixed effects
p2 = FixedEffect(repeat(1:2, outer = 5), repeat(1:2, inner = 5))
# define interacted fixed effects
p3 = FixedEffect(repeat(1:5, outer = 2), interaction = rand(10))

# find residuals
x = rand(10)
solve_residuals!(x, [p1, p2])

# find the fixed effect coefficients
solve_coefficients!(x, [p1, p3])
