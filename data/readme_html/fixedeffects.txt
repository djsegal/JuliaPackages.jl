<p dir="auto">This package solves least squares problem with high dimensional fixed effects. For a matrix <code>D</code> of high dimensional fixed effects, it finds <code>b</code> and <code>系</code> such that <code>y = D'b + 系</code> with <code>E[D系] = 0</code>.</p>
<p dir="auto">It is the back end for the package <a href="https://github.com/matthieugomez/FixedEffectModels.jl">FixedEffectModels.jl</a>, that estimates linear models with high-dimensional fixed effect.</p>
<p dir="auto">The package defines two functions <code>solve_coefficients!</code>, that returns the coefficients <code>b</code>, and <code>solve_residuals!</code>, that returns the residuals <code>系</code>. See the documentation <code>?solve_residuals!</code> or <code>?solve_coefficients!</code>.</p>
