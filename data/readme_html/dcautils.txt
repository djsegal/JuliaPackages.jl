<p>This package contains some utilities for <a href="https://en.wikipedia.org/wiki/Direct_coupling_analysis" rel="nofollow">Direct Coupling Analysis</a>,
an unsupervised technique to analyse Multiple Sequence Alignments of
protein families.</p>
<p>This code was originally written for <a href="https://github.com/carlobaldassi/GaussDCA.jl">GaussDCA.jl</a>.</p>
<p>See the <a href="https://carlobaldassi.github.io/DCAUtils.jl/stable" rel="nofollow">documentation</a> for details.</p>
