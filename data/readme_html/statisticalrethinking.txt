<h2><a id="user-content-purpose-of-this-package" class="anchor" aria-hidden="true" href="#purpose-of-this-package"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path fill-rule="evenodd" d="M7.775 3.275a.75.75 0 001.06 1.06l1.25-1.25a2 2 0 112.83 2.83l-2.5 2.5a2 2 0 01-2.83 0 .75.75 0 00-1.06 1.06 3.5 3.5 0 004.95 0l2.5-2.5a3.5 3.5 0 00-4.95-4.95l-1.25 1.25zm-4.69 9.64a2 2 0 010-2.83l2.5-2.5a2 2 0 012.83 0 .75.75 0 001.06-1.06 3.5 3.5 0 00-4.95 0l-2.5 2.5a3.5 3.5 0 004.95 4.95l1.25-1.25a.75.75 0 00-1.06-1.06l-1.25 1.25a2 2 0 01-2.83 0z"></path></svg></a>Purpose of this package</h2>
<p>The StatisticalRethinking.jl v3 <code>package</code> contains functions comparable to the functions in the R package "rethinking" associated with the book <a href="https://xcelab.net/rm/statistical-rethinking/" rel="nofollow">Statistical Rethinking</a> by Richard McElreath.</p>
<p>These functions are used in the Pluto notebooks <code>projects</code> specifically intended for hands-on use while studying the book or taking the course.</p>
<p>To work through the StatisticalRethinking book using Julia and Stan, download <code>project</code> <a href="https://github.com/StatisticalRethinkingJulia/StatisticalRethinkingStan.jl">StatisticalRethinkingStan.jl</a> and open one of the chapter Pluto notebooks.</p>
<p>To work through the StatisticalRethinking book using Julia and Turing, download <code>project</code> <a href="https://github.com/StatisticalRethinkingJulia/StatisticalRethinkingTuring.jl">StatisticalRethinkingTuring.jl</a> and open a Pluto notebook.</p>
<p>Time permitting I would love to see a StatisticalRethinkingDhmc.jl, which could be a combination of Soss.jl and DynamicHMC.jl, and a StatisticalRethinkingMamba.jl!</p>
<p>If interested in either of the last 2 projects, please contact me!</p>
<h2><a id="user-content-important-note-on-statisticalrethinking-v3" class="anchor" aria-hidden="true" href="#important-note-on-statisticalrethinking-v3"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path fill-rule="evenodd" d="M7.775 3.275a.75.75 0 001.06 1.06l1.25-1.25a2 2 0 112.83 2.83l-2.5 2.5a2 2 0 01-2.83 0 .75.75 0 00-1.06 1.06 3.5 3.5 0 004.95 0l2.5-2.5a3.5 3.5 0 00-4.95-4.95l-1.25 1.25zm-4.69 9.64a2 2 0 010-2.83l2.5-2.5a2 2 0 012.83 0 .75.75 0 001.06-1.06 3.5 3.5 0 00-4.95 0l-2.5 2.5a3.5 3.5 0 004.95 4.95l1.25-1.25a.75.75 0 00-1.06-1.06l-1.25 1.25a2 2 0 01-2.83 0z"></path></svg></a>Important note on StatisticalRethinking v3</h2>
<p>This is a breaking change from v2 of StatisticalRethinking.jl. I expect the first stage of this work to be completed by late Nov 2020.</p>
<p>If you prefer to use v2, you can use <code>] add StatisticalRethinking@v2.x.y</code> where 2.x.y is the latest released version for v2.</p>
<p>Given that Julia provides several very capable packages that support mcmc simulation, it only seemed appropriate to make StatisticalRethinking on Julia mcmc implementation independent.</p>
<p>The availablility of DynamicHMC, the huge progress made by the Turing.jl team over the last 2 years, the introduction of Julia <code>projects</code> in addition to Julia <code>packages</code>, the novel approach to notebooks in Pluto.jl and the work by <a href="https://github.com/karajan9/statisticalrethinking">Karajan</a> were a few of the ideas that triggered exploring a new setup for StatisticalRethinking.jl and the 'course' projects <a href="https://github.com/StatisticalRethinkingJulia/StatisticalRethinkingStan.jl">StatistcalRethinkingStan</a> and <a href="https://github.com/StatisticalRethinkingJulia/StatisticalRethinkingTuring.jl">StatisticalrethinkingTuring</a></p>
<p>An early, experimental version of <a href="https://github.com/StatisticalRethinkingJulia/StructuralCausalModels.jl">StructuralCausalModels.jl</a> is also included as a dependency in the StatisticalRethinking.jl v3 package. In the meantime I will definitely keep my eyes on <a href="https://github.com/zenna/Omega.jl">Omega.jl</a> and <a href="https://github.com/mschauer/CausalInference.jl">CausalInference.jl</a>. StructuralCausalModels does provide ways to convert DAGs to Daggity and ggm formats.</p>
<p>Also now added as a dependency is <a href="https://github.com/StatisticalRethinkingJulia/StatsModelComparisons.jl">StatsModelComparisons.jl</a> which provides PSIS and WAIC statistics. From chapter 7 onwards these are used in StatisticalRethinking.</p>
<p>Finally, for a good while I have been looking for a great statistics book using Julia as kind of an introductory text to StatisticalRethinking and I believe the first couple of chapters in an upcoming book <a href="https://statisticswithjulia.org/index.html" rel="nofollow">Statistics with Julia</a> by Yoni Nazarathy and Hayden Klok are exactly that.</p>
<p>As <a href="https://github.com/StatisticalRethinkingJulia">StatisticalRethinking</a> v3 is also (DrWatson &amp; Pkg) project based and uses Pluto notebooks, I have converted the book <code>listings</code> in the first 5 chapters to Pluto Notebooks in a new repository in StatisticalRethinkingJulia, i.e. <a href="https://github.com/StatisticalRethinkingJulia/StatisticsWithJuliaPlutoNotebooks.jl">StatisticsWithJuliaPlutoNotebooks</a>.</p>
<p>After chapter 4, <code>Statistics with Julia</code> follows the frequentionist approach while <code>Statistical Rethinking</code> opts for the Bayesian approach. Most of the material in chapters 5 and 6 of <code>Statistics with Julia</code> is therefore also covered using a more Bayesian perspective in the early chapters of the notebook projects <a href="https://github.com/StatisticalRethinkingJulia/StatisticalRethinkingStan.jl">StatisticalRethinkingStan</a> and <a href="https://github.com/StatisticalRethinkingJulia/StatisticalRethinkingTuring.jl">StatisticalRethinkingTuring</a>.</p>

<p>StatisticalRethinking.jl v3 is independent of the underlying mcmc package. All scripts previously in StatisticalRethinking.jl v2 holding the snippets have been replaced by Pluto notebooks in the above mentioned mcmc specific <code>project</code> repositories.</p>
<p>Initially StatisticalRethinkingTuring.jl will lag StatisticalRethinkingStan.jl somewhat but later this year both will cover the same chapters.</p>
<p>It is the intention to develop <em>tests</em> for StatisticalRethinking.jl v3 that work across the different mcmc implementations. This will limit dependencies to the <code>test/Project.toml</code>.</p>
