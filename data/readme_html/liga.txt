<p>This library is not well documented and updated in the metadata base. So the best way to install and use it in this moment is by cloning the directory and then using (just) the documentation API. Updates on documentation and metadata are coming soon. To start using just type</p>
<div class="snippet-clipboard-content position-relative" data-snippet-clipboard-copy-content="add https://github.com/evcastelani/Liga.jl

using Liga
"><pre><code>add https://github.com/evcastelani/Liga.jl

using Liga
</code></pre></div>
<p>and then</p>
<div class="snippet-clipboard-content position-relative" data-snippet-clipboard-copy-content="layout(3,1,&quot;Conformal&quot;)
"><pre><code>layout(3,1,"Conformal")
</code></pre></div>
<p>to work with G(3,1), for example. We are working to build a complete documentation and new functions.</p>
