<p>The upstream <a href="http://www.reduce-algebra.com" rel="nofollow">REDUCE</a> software was originally created by Anthony C. Hearn (currently maintained on <a href="https://sourceforge.net/p/reduce-algebra/" rel="nofollow">SourceForge</a>), the Julia packages included with this repository are based on the symbolic parser generator <a href="https://github.com/chakravala/Reduce.jl">Reduce.jl</a> created by Michael Reed, which extends the Julia language with REDUCE functionality.</p>
<p>In Reduce.jl, an upstream REDUCE package can be used with <code>load_package(::Symbol)</code>.</p>
<p>External packages provided by ReduceAlgebra.jl,</p>
<ul>
<li><a href="https://github.com/chakravala/Reduce.jl">Reduce.jl</a>: <em>core functionality (parser-generator)</em>, upstream docs (<a href="http://www.reduce-algebra.com/manual/manual.html" rel="nofollow">REDUCE</a> / <a href="http://www.reduce-algebra.com/manual/manual.pdf" rel="nofollow">pdf</a>), Julia docs (<a href="https://chakravala.github.io/Reduce.jl/stable" rel="nofollow">stable</a> / <a href="https://chakravala.github.io/Reduce.jl/latest" rel="nofollow">latest</a>)</li>
<li><a href="https://github.com/JuliaReducePkg/ReduceLinAlg.jl">ReduceLinAlg.jl</a>: <em>Linear algebra package</em>, upstream docs (<a href="http://www.reduce-algebra.com/manual/manualse127.html" rel="nofollow">LINALG</a> / <a href="http://www.reduce-algebra.com/manual/contributed/linalg.pdf" rel="nofollow">pdf</a>), Julia docs (<a href="https://JuliaReducePkg.github.io/ReduceLinAlg.jl/stable" rel="nofollow">stable</a> / <a href="https://JuliaReducePkg.github.io/ReduceLinAlg.jl/latest" rel="nofollow">latest</a>)</li>
</ul>
<p>What is included with the Reduce.jl core package? Julia docs (<a href="https://chakravala.github.io/Reduce.jl/stable/man/16-packages.html" rel="nofollow">stable</a> / <a href="https://chakravala.github.io/Reduce.jl/latest/man/16-packages.html" rel="nofollow">latest</a>)</p>
<ul>
<li><em>Integration of square roots</em>, upstream docs (<a href="http://www.reduce-algebra.com/manual/manualse91.html" rel="nofollow">ALGINT</a> / <a href="http://www.reduce-algebra.com/manual/contributed/algint.pdf" rel="nofollow">pdf</a>)</li>
<li><em>Package for compacting expressions</em>, upstream docs (<a href="http://www.reduce-algebra.com/manual/manualse105.html" rel="nofollow">COMPACT</a> / <a href="http://www.reduce-algebra.com/manual/contributed/compact.pdf" rel="nofollow">pdf</a>)</li>
<li><em>A package for finding limits</em>, upstream docs (<a href="http://www.reduce-algebra.com/manual/manualse126.html" rel="nofollow">LIMITS</a> / <a href="http://www.reduce-algebra.com/manual/contributed/limits.pdf" rel="nofollow">pdf</a>)</li>
<li><em>REDUCE LaTeX formula interface</em>, upstream docs (<a href="http://www.reduce-algebra.com/manual/manualse146.html" rel="nofollow">RLFI</a> / <a href="http://www.reduce-algebra.com/manual/contributed/rlfi.pdf" rel="nofollow">pdf</a>), supports <a href="https://github.com/JuliaLang/IJulia.jl">IJulia.jl</a> output</li>
<li><em>REDUCE source code optimization package</em>, upstream docs (<a href="http://www.reduce-algebra.com/manual/manualse150.html" rel="nofollow">SCOPE</a> / <a href="http://www.reduce-algebra.com/manual/contributed/scope.pdf" rel="nofollow">pdf</a>)</li>
<li><em>A package for series summation</em>, upstream docs (<a href="http://www.reduce-algebra.com/manual/manualse157.html" rel="nofollow">SUM</a> / <a href="http://www.reduce-algebra.com/manual/contributed/sum.pdf" rel="nofollow">pdf</a>)</li>
</ul>
<p>More packages will be added. See <a href="https://github.com/orgs/JuliaReducePkg/projects/1">Implement External REDUCE Packages</a> and <a href="https://github.com/chakravala/Reduce.jl/projects/1">Implement Reduce</a> for progress.</p>
