<p>The ReadStat.jl Julia package uses the <a href="https://github.com/WizardMac/ReadStat">ReadStat</a> C library to parse binary and transport files from Stata, SPSS and SAS. All functions return a tuple, with the first element an array of columns and the second element a vector of column names.</p>
<p>For integration with packages like <a href="https://github.com/JuliaData/DataFrames.jl">DataFrames.jl</a> you should use the <a href="https://github.com/queryverse/StatFiles.jl">StatFiles.jl</a> package.</p>
