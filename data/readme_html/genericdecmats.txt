<p dir="auto">This repository gives access to the generic decomposition matrices of various groups of Lie type.</p>
<p dir="auto">Currently the following use cases are supported.</p>
<ul dir="auto">
<li>
<p dir="auto">The <a href="https://github.com/jmichel7/Gapjm.jl">Gapjm.jl</a> package provides functionality for generic decomposition matrices.</p>
</li>
<li>
<p dir="auto">The generic decomposition matrices can be read into <a href="https://github.com/oscar-system/Oscar.jl">Oscar.jl</a>, its matrices and polynomials are used then.</p>
</li>
<li>
<p dir="auto">There is also a <a href="https://www.gap-system.org/" rel="nofollow">GAP 4</a> interface, read <code>init.g</code> into a GAP session in order to provide functions for loading, displaying, and testing the matrices.</p>
</li>
</ul>
<p dir="auto">The interfaces to Gapjm.jl and Oscar.jl can be used in the same Julia session.</p>
<p dir="auto">There is <a href="https://www.math.rwth-aachen.de/~Thomas.Breuer/GenericDecMats" rel="nofollow">a preliminary printout version of the data</a>; both the location and the format will be changed.</p>
<p dir="auto">The data (in the <code>data</code> subdirectory) have been provided by <a href="https://www.mathematik.uni-kl.de/~malle/de/index.html" rel="nofollow">Gunter Malle</a>.</p>
