<p dir="auto">This package is an interface from <a href="https://github.com/ModiaSim/SignalTables.jl">SignalTables</a> to
<a href="https://github.com/JuliaPlots/Makie.jl">WGLMakie</a> (= Julia interface to backend WGLMakie of Makie.jl)
in order to be able to perform convenient line plots in SignalTables via WGLMakie.</p>
<p dir="auto">This package is usually not directly used, but is activated via package
<a href="https://github.com/ModiaSim/SignalTables.jl">SignalTables</a>.
For details of the installation and the usage,
see the <a href="https://modiasim.github.io/SignalTables.jl/stable/index.html" rel="nofollow">SignalTables documentation</a>.</p>
