<p>Waveform is a flexible and modular approach to solving PDE-constrained inverse problems that aims to</p>
<ul>
<li>accurately reflect the underlying mathematics of the problem</li>
<li>produce <em>readable</em> code that reflects the underlying mathematics</li>
<li>scales easily from small 2D problems to large-scale 3D problems with <em>minimal</em> code modifications</li>
<li>allow users to integrate their own preconditioners, stencils, linear solvers, etc. easily in to the entire inversion framework</li>
</ul>
<p>For more details, as well as the full design of the software, see (<a href="https://arxiv.org/abs/1703.09268" rel="nofollow">https://arxiv.org/abs/1703.09268</a>) and for the original Matlab implementation, see (<a href="https://github.com/slimgroup/WAVEFORM">https://github.com/slimgroup/WAVEFORM</a>). All functions are written by Curt Da Silva (<a href="mailto:curt.dasilva@gmail.com">curt.dasilva@gmail.com</a>), unless indicated otherwise in the function documentation.</p>
