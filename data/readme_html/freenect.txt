<p dir="auto">Freenect.jl is a wrapper around the <a href="https://github.com/OpenKinect/libfreenect">libfreenect</a> open source Microsoft Kinect driver.</p>
<p dir="auto">This package only supports Kinect v1, which comes from the XBox 360 era.</p>
<p dir="auto">For installation and usage please visit the <a href="https://dmillard.github.io/Freenect.jl/dev" rel="nofollow">documentation</a>.</p>
