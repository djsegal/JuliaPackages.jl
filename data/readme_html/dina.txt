<p dir="auto">This package uses DataDeps.jl to download the US distributional national accounts (DINA) dataset from <a href="http://gabriel-zucman.eu/usdina/" rel="nofollow">Gabriel Zucman's website</a>. The data are downloaded in bulk (&gt;1GB) and stored in <code>.julia/datadeps/dina/</code>.</p>
<p dir="auto">The dataset is described in Piketty, T, E. Saez and G. Zucman: <strong>Distributional National Accounts: Methods and Estimates for the United States</strong>, <em>Quarterly Journal of Economics, 2018, 133 (2): 553-609.</em></p>
<p dir="auto">The variables are described in <a href="http://gabriel-zucman.eu/files/PSZCodebook.pdf" rel="nofollow">the codebook at Gabriel Zucman's website</a>.</p>

<p dir="auto">Load the DINA micro-data by year</p>
<div class="highlight highlight-source-julia notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="using DINA, DataFrames

tbl = get_dina(1980)
df80 = DataFrame(tbl)"><pre><span class="pl-k">using</span> DINA, DataFrames

tbl <span class="pl-k">=</span> <span class="pl-c1">get_dina</span>(<span class="pl-c1">1980</span>)
df80 <span class="pl-k">=</span> <span class="pl-c1">DataFrame</span>(tbl)</pre></div>
<p dir="auto">or load a quantile-year panel (takes about 1-2 minutes for 1962--2019).</p>
<div class="highlight highlight-source-julia notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="using DINA

wgt = :dweght

byvar = inc # compute deciles of `:peinc`

df = dina_quantile_panel(var, byvar, 10)"><pre><span class="pl-k">using</span> DINA

inc <span class="pl-k">=</span> <span class="pl-c1">:peinc</span>
wgt <span class="pl-k">=</span> <span class="pl-c1">:dweght</span>

var <span class="pl-k">=</span> [[<span class="pl-c1">:fiinc</span>, <span class="pl-c1">:fninc</span>, <span class="pl-c1">:ptinc</span>, inc, <span class="pl-c1">:poinc</span>, <span class="pl-c1">:ownerhome</span>, <span class="pl-c1">:rentalhome</span>]; dbt_var]
byvar <span class="pl-k">=</span> inc <span class="pl-c"><span class="pl-c">#</span> compute deciles of `:peinc`</span>
