<p dir="auto">SpatialGraphs.jl introduces the <code>AbstractSpatialGraph</code> and other subtypes. The <code>AbstractSpatialGraph</code>
is a <code>Graphs.AbstractGraph</code>, and can be weighted or directed. SpatialGraphs.jl extends the
Graphs interface, so methods from Graphs.jl work right out of the box.</p>
<p dir="auto">An <code>AbstractSpatialGraph</code> is a composite type that contains an <code>AbstractGraph</code> in addition to
metadata that details the spatial location of each vertex in the
graph. At this time, only raster-based graph types have been developed (and
vertex locations are stored in a <code>Rasters.Raster</code>), but there are plans to
implement graph types for vector data as well.</p>
