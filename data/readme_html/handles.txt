<p dir="auto">A lightweight (primitive), integer-like handle to a resource.  Useful for dispatch.
i.e. pretty much an integer, but with a unique dispatch signature.</p>
<p dir="auto">@HandleType(n)</p>
<p dir="auto">Create a new type named n to be used as a handle to your resource.</p>
<p dir="auto">Example:</p>
<div class="snippet-clipboard-content notranslate position-relative overflow-auto" data-snippet-clipboard-copy-content="julia&gt; @HandleType(Foo)
Foo

foo (generic function with 1 method)

julia&gt; foo(Foo(2))
4

julia&gt; foo(x::Foo) = x-x
foo (generic function with 2 methods)

julia&gt; foo(Foo(2))
0"><pre class="notranslate"><code>julia&gt; @HandleType(Foo)
