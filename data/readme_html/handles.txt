<p>A lightweight (primitive), integer-like handle to a resource.  Useful for dispatch.
i.e. pretty much an integer, but with a unique dispatch signature.</p>
<p>@HandleType(n)</p>
<p>Create a new type named n to be used as a handle to your resource.</p>
<p>Example:</p>
<pre><code>julia&gt; @HandleType(Foo)
Foo

julia&gt; foo(x::Integer) = x*x
foo (generic function with 1 method)

julia&gt; foo(Foo(2))
4

julia&gt; foo(x::Foo) = x-x
foo (generic function with 2 methods)

julia&gt; foo(Foo(2))
0
</code></pre>
