<p dir="auto">ISAData is a Julia package that can be used to calculate thermodynamic properties of air (density, pressure, temperature, and viscosity) at a certain altitude in the atmosphere  using the <strong>I</strong>nternational <strong>S</strong>tandard <strong>A</strong>tmosphere (ISA) model.</p>

<p dir="auto">The package contains only one exported function called <code>ISAdata</code>. The function is called with the altitude as the input parameter, and it returns a tuple containing the values for density, pressure, temperature, and dynamic viscosity. For example:</p>
<div class="snippet-clipboard-content notranslate position-relative overflow-auto" data-snippet-clipboard-copy-content="julia&gt; using ISAData

julia&gt; ρ, P, T, μ = ISAdata(10000)
(0.4136571526878796, 26510.092578167572, 223.25492665430897, 1.461852633009453e-5)"><pre class="notranslate"><code>julia&gt; using ISAData

julia&gt; ρ, P, T, μ = ISAdata(10000)
(0.4136571526878796, 26510.092578167572, 223.25492665430897, 1.461852633009453e-5)
</code></pre></div>
<p dir="auto">where all quantities are in SI units.</p>
<p dir="auto"><code>ISAdata</code> also supports the <code>Quantity</code> type provided by the <a href="https://github.com/PainterQubits/Unitful.jl">Unitful</a> package. For example:</p>
<div class="snippet-clipboard-content notranslate position-relative overflow-auto" data-snippet-clipboard-copy-content="julia&gt; using Unitful

julia&gt; ρ, P, T, μ = ISAdata(10u&quot;km&quot;)
(0.4136571526878796 kg m^-3, 26510.092578167572 Pa, 223.25492665430897 K, 1.461852633009453e-5 Pa s)"><pre class="notranslate"><code>julia&gt; using Unitful

julia&gt; ρ, P, T, μ = ISAdata(10u"km")
(0.4136571526878796 kg m^-3, 26510.092578167572 Pa, 223.25492665430897 K, 1.461852633009453e-5 Pa s)
</code></pre></div>
<p dir="auto">Here, output values are also of the type <code>Quantity</code>, and this provides the flexibility to convert quantities into different units:</p>
<div class="snippet-clipboard-content notranslate position-relative overflow-auto" data-snippet-clipboard-copy-content="julia&gt; P |&gt; u&quot;lbf/ft^2&quot;
553.6747950560401 lbf ft^-2"><pre class="notranslate"><code>julia&gt; P |&gt; u"lbf/ft^2"
553.6747950560401 lbf ft^-2
</code></pre></div>
