<p dir="auto">A package providing integration of <a href="https://www.openml.org" rel="nofollow">OpenML</a> with the
<a href="https://alan-turing-institute.github.io/MLJ.jl/dev/" rel="nofollow">MLJ</a> machine
learning framework.</p>
<p dir="auto">As of version 2.0, MLJOpenML.jl became a re-export of functionality
moved to the framework agnostic package
<a href="https://github.com/JuliaAI/OpenML.jl">OpenML.jl</a>. MLJ-specfic
functionality will appear in versions 2.1 onwards.</p>
