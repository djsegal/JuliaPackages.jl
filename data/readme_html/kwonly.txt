<p>Kwonly.jl provides a macro <code>@add_kwonly</code>.  It creates a keyword-only
version of the given function.  Example:</p>
<div class="highlight highlight-source-julia"><pre><span class="pl-k">using</span> Kwonly

<span class="pl-k">struct</span> A
    x
    y
    <span class="pl-c1">@add_kwonly</span> <span class="pl-en">A</span>(x, y<span class="pl-k">=</span><span class="pl-c1">2</span>) <span class="pl-k">=</span> <span class="pl-c1">new</span>(x, y)
<span class="pl-k">end</span></pre></div>
<p>This macro add a keyword-only constructor by expanding <code>A(x, y=2) = new(x, y)</code> into:</p>
<div class="highlight highlight-source-julia"><pre><span class="pl-en">A</span>(x, y) <span class="pl-k">=</span> <span class="pl-c1">new</span>(x, y)                                      <span class="pl-c"><span class="pl-c">#</span> original</span>
<span class="pl-en">A</span>(; x <span class="pl-k">=</span> <span class="pl-en">throw</span>(<span class="pl-en">UndefKeywordError</span>(<span class="pl-c1">:x</span>)), y<span class="pl-k">=</span><span class="pl-c1">2</span>) <span class="pl-k">=</span> <span class="pl-c1">new</span>(x, y)   <span class="pl-c"><span class="pl-c">#</span> keyword-only</span></pre></div>
<p>So, the struct <code>A</code> can also be constructed by using only keyword
arguments:</p>
<div class="highlight highlight-source-julia"><pre><span class="pl-c1">@test</span> <span class="pl-c1">A</span>(<span class="pl-c1">1</span>) <span class="pl-k">==</span> <span class="pl-c1">A</span>(x<span class="pl-k">=</span><span class="pl-c1">1</span>)</pre></div>
