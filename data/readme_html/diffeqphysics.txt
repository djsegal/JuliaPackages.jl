<h2><a id="user-content-simulation-of-gravitationally-interacting-bodies" class="anchor" aria-hidden="true" href="#simulation-of-gravitationally-interacting-bodies"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path fill-rule="evenodd" d="M4 9h1v1H4c-1.5 0-3-1.69-3-3.5S2.55 3 4 3h4c1.45 0 3 1.69 3 3.5 0 1.41-.91 2.72-2 3.25V8.59c.58-.45 1-1.27 1-2.09C10 5.22 8.98 4 8 4H4c-.98 0-2 1.22-2 2.5S3 9 4 9zm9-3h-1v1h1c1 0 2 1.22 2 2.5S13.98 12 13 12H9c-.98 0-2-1.22-2-2.5 0-.83.42-1.64 1-2.09V6.25c-1.09.53-2 1.84-2 3.25C6 11.31 7.55 13 9 13h4c1.45 0 3-1.69 3-3.5S14.5 6 13 6z"></path></svg></a>Simulation of gravitationally interacting bodies</h2>
<p>In order to create bodies/particles for the problem, one needs to use the MassBody structure and its constructor, which accepts mass, initial coordinates and velocity of the body.</p>
<div class="highlight highlight-source-julia"><pre>body1 <span class="pl-k">=</span> <span class="pl-c1">MassBody</span>(<span class="pl-c1">2.0</span>, <span class="pl-c1">SVector</span>(<span class="pl-c1">0.0</span>, <span class="pl-c1">1.0</span>, <span class="pl-c1">0.0</span>), <span class="pl-c1">SVector</span>( <span class="pl-c1">5.775e-6</span>, <span class="pl-c1">0.0</span>, <span class="pl-c1">0.0</span>))
body2 <span class="pl-k">=</span> <span class="pl-c1">MassBody</span>(<span class="pl-c1">2.0</span>, <span class="pl-c1">SVector</span>(<span class="pl-c1">0.0</span>,<span class="pl-k">-</span><span class="pl-c1">1.0</span>, <span class="pl-c1">0.0</span>), <span class="pl-c1">SVector</span>(<span class="pl-k">-</span><span class="pl-c1">5.775e-6</span>, <span class="pl-c1">0.0</span>, <span class="pl-c1">0.0</span>))</pre></div>
<p>Usually we solve an n-body problem for a certain period of time:</p>
<div class="highlight highlight-source-julia"><pre>tspan <span class="pl-k">=</span> (<span class="pl-c1">0.0</span>, <span class="pl-c1">1111150.0</span>);</pre></div>
<p>Solving gravitational problem one needs to specify the gravitational constant G.</p>
<div class="highlight highlight-source-julia"><pre>G <span class="pl-k">=</span> <span class="pl-c1">6.673e-11</span></pre></div>
<p>In fact, now we have enough parameters to create an NBodyGravProblem object:</p>
<div class="highlight highlight-source-julia"><pre>problem <span class="pl-k">=</span> <span class="pl-c1">NBodyGravProblem</span>([body1,body2], G, tspan)</pre></div>
<p>Solution to the problem might be evaluated using the standard <code>solve</code> function:</p>
<div class="highlight highlight-source-julia"><pre>solution <span class="pl-k">=</span> <span class="pl-c1">solve</span>(problem, <span class="pl-c1">Tsit5</span>());</pre></div>
<p>And, finally, we plot our solution showing two equal bodies rotating on the same orbit:</p>
<div class="highlight highlight-source-julia"><pre><span class="pl-c1">plot_xy_scattering</span>(solution,<span class="pl-s"><span class="pl-pds">"</span>./anim_two_boddies_scattering.gif<span class="pl-pds">"</span></span>)</pre></div>
