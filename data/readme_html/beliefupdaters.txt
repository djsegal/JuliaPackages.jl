<p>Collection of belief updaters and belief representation for POMDPs.jl. Particle filters are not included in this package, but an implementation can be found in <a href="https://github.com/JuliaPOMDP/ParticleFilters.jl">ParticleFilters.jl</a>.</p>
<p>This is a supported <a href="https://github.com/JuliaPOMDP">JuliaPOMDP</a> package.</p>
<p>The documentation can be found here: <a href="https://juliapomdp.github.io/BeliefUpdaters.jl/latest" rel="nofollow">https://juliapomdp.github.io/BeliefUpdaters.jl/latest</a></p>
