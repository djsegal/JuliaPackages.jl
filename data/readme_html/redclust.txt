<p dir="auto"><a href="https://github.com/abhinavnatarajan/RedClust.jl">RedClust</a> is a <a href="https://julialang.org/" rel="nofollow">Julia</a> package for Bayesian clustering of high-dimensional Euclidean data using pairwise dissimilarities instead of the raw observations. It uses an MCMC sampler to generate posterior samples from the space of all possible clustering structures on the data.</p>
