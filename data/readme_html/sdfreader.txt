<p><a href="https://cfsa-pmw.warwick.ac.uk/mediawiki/index.php/EPOCH:FAQ" rel="nofollow">EPOCH</a> is a code for plasma physics simulations using the Particle-in-Cell method. The simulation results are written is <code>.sdf</code> binary files. Several readers for this files are available at <a href="https://cfsa-pmw.warwick.ac.uk/SDF" rel="nofollow">https://cfsa-pmw.warwick.ac.uk/SDF</a>.
This package intends to be another reader for the <code>.sdf</code> file type providing low level acces to the data by following the <a href="https://cfsa-pmw.warwick.ac.uk/SDF/SDF_documentation" rel="nofollow">SDF file format documentation</a>.
For a more user firendly approach, please use <a href="https://github.com/SebastianM-C/SDFResults.jl">SDFResults</a>.</p>

<p>Install the package using</p>
<div class="snippet-clipboard-content position-relative" data-snippet-clipboard-copy-content="]add SDFReader
"><pre><code>]add SDFReader
</code></pre></div>
<p>The metadata in the <code>.sdf</code> files can be accessed with the <code>file_summary</code> function</p>
<div class="highlight highlight-source-julia position-relative" data-snippet-clipboard-copy-content="blocks = file_summary(filename)
"><pre>blocks <span class="pl-k">=</span> <span class="pl-c1">file_summary</span>(filename)</pre></div>
<p>This will return a dictionary that can be used to access the block headers corrsoponding to the data.
To read the data use</p>
<div class="highlight highlight-source-julia position-relative" data-snippet-clipboard-copy-content="ex = open(file) do f
    read(f, blocks[:ex])
end
"><pre>ex <span class="pl-k">=</span> <span class="pl-c1">open</span>(file) <span class="pl-k">do</span> f
    <span class="pl-c1">read</span>(f, blocks[<span class="pl-c1">:ex</span>])
<span class="pl-k">end</span></pre></div>
