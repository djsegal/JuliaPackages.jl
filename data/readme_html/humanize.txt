<p><a href="https://travis-ci.org/IainNZ/Humanize.jl" rel="nofollow"><img src="https://camo.githubusercontent.com/3dfb682e91da99eb9925c1e2ed959a9cb1cc07ad/68747470733a2f2f7472617669732d63692e6f72672f4961696e4e5a2f48756d616e697a652e6a6c2e7376673f6272616e63683d6d6173746572" alt="Build Status" data-canonical-src="https://travis-ci.org/IainNZ/Humanize.jl.svg?branch=master" style="max-width:100%;"></a></p>
<p>Humanize numbers, including</p>
<ul>
<li>data sizes (<code>3e6 -&gt; 3.0 MB or 2.9 MiB</code>).</li>
<li>Date/datetime differences (<code>Date(2014,2,3) - Date(2013,3,7) -&gt; 1 year, 1 month</code>)</li>
<li>Digit separator (<code>12345678 -&gt; 12,345,678</code>)</li>
</ul>
<p>This package is MIT licensed, and is based on <a href="https://github.com/jmoiron/humanize/">jmoiron's humanize Python library</a>.</p>
