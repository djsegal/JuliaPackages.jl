<p>To find the citation for a thing, e.g. function, type, or module.
use the <code>cite(thing)</code> command.</p>
<h3><a id="user-content-declaring-citations" class="anchor" aria-hidden="true" href="#declaring-citations"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path fill-rule="evenodd" d="M7.775 3.275a.75.75 0 001.06 1.06l1.25-1.25a2 2 0 112.83 2.83l-2.5 2.5a2 2 0 01-2.83 0 .75.75 0 00-1.06 1.06 3.5 3.5 0 004.95 0l2.5-2.5a3.5 3.5 0 00-4.95-4.95l-1.25 1.25zm-4.69 9.64a2 2 0 010-2.83l2.5-2.5a2 2 0 012.83 0 .75.75 0 001.06-1.06 3.5 3.5 0 00-4.95 0l-2.5 2.5a3.5 3.5 0 004.95 4.95l1.25-1.25a.75.75 0 00-1.06-1.06l-1.25 1.25a2 2 0 01-2.83 0z"></path></svg></a>Declaring citations</h3>
<h4><a id="user-content-function" class="anchor" aria-hidden="true" href="#function"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path fill-rule="evenodd" d="M7.775 3.275a.75.75 0 001.06 1.06l1.25-1.25a2 2 0 112.83 2.83l-2.5 2.5a2 2 0 01-2.83 0 .75.75 0 00-1.06 1.06 3.5 3.5 0 004.95 0l2.5-2.5a3.5 3.5 0 00-4.95-4.95l-1.25 1.25zm-4.69 9.64a2 2 0 010-2.83l2.5-2.5a2 2 0 012.83 0 .75.75 0 001.06-1.06 3.5 3.5 0 00-4.95 0l-2.5 2.5a3.5 3.5 0 004.95 4.95l1.25-1.25a.75.75 0 00-1.06-1.06l-1.25 1.25a2 2 0 01-2.83 0z"></path></svg></a>Function</h4>
<p>To declare a citation for a function attach it as a trait on the type of the function.
This is done</p>
<pre><code>function thing()
#...
end

Citations.cite(::typeof(thing) = "2001, Jane Doe, Considerations of a Thing"
</code></pre>
<p>Or you can use the helper macro <code>@declare_citation</code>:</p>
<pre><code>@declare_citation thing,  "2001, Jane Doe, Considerations of a Thing"
</code></pre>
<h4><a id="user-content-module-eg-package" class="anchor" aria-hidden="true" href="#module-eg-package"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path fill-rule="evenodd" d="M7.775 3.275a.75.75 0 001.06 1.06l1.25-1.25a2 2 0 112.83 2.83l-2.5 2.5a2 2 0 01-2.83 0 .75.75 0 00-1.06 1.06 3.5 3.5 0 004.95 0l2.5-2.5a3.5 3.5 0 00-4.95-4.95l-1.25 1.25zm-4.69 9.64a2 2 0 010-2.83l2.5-2.5a2 2 0 012.83 0 .75.75 0 001.06-1.06 3.5 3.5 0 00-4.95 0l-2.5 2.5a3.5 3.5 0 004.95 4.95l1.25-1.25a.75.75 0 00-1.06-1.06l-1.25 1.25a2 2 0 01-2.83 0z"></path></svg></a>Module (e.g. package)</h4>
<p>To declare a citation for a module,
declare a <code>cite</code> trait for the modules <code>eval</code> function.</p>
<pre><code>Citation.cite(::typeof(MyModule.eval)) = "2019, Author, Package"
</code></pre>
<p>Or use the <code>@declare_citation</code> macro, with only one argument.</p>
<pre><code>@declare_citation "2019, Author, Package"
</code></pre>
<h2><a id="user-content-using-citationsjl-in-your-packages" class="anchor" aria-hidden="true" href="#using-citationsjl-in-your-packages"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path fill-rule="evenodd" d="M7.775 3.275a.75.75 0 001.06 1.06l1.25-1.25a2 2 0 112.83 2.83l-2.5 2.5a2 2 0 01-2.83 0 .75.75 0 00-1.06 1.06 3.5 3.5 0 004.95 0l2.5-2.5a3.5 3.5 0 00-4.95-4.95l-1.25 1.25zm-4.69 9.64a2 2 0 010-2.83l2.5-2.5a2 2 0 012.83 0 .75.75 0 001.06-1.06 3.5 3.5 0 00-4.95 0l-2.5 2.5a3.5 3.5 0 004.95 4.95l1.25-1.25a.75.75 0 00-1.06-1.06l-1.25 1.25a2 2 0 01-2.83 0z"></path></svg></a>Using Citations.jl in your packages</h2>
<p>Suppose you are Jane Doe, the maintainer of SomeCoolPackage.jl and you want your users to be able to cite your package as well as your two functions, <code>awesomefunction1</code> and <code>awesomefunction2</code>. You can import Citations.jl into your package and export  citation methods as follows</p>
<div class="highlight highlight-source-julia"><pre><span class="pl-k">module</span> SomeCoolPackage

<span class="pl-k">using</span> Citations 

<span class="pl-en">awesomefunction1</span>(x) <span class="pl-k">=</span> <span class="pl-s"><span class="pl-pds">"</span>do cool stuff<span class="pl-pds">"</span></span>
<span class="pl-en">awesomefunction2</span>(x) <span class="pl-k">=</span> <span class="pl-s"><span class="pl-pds">"</span>do other cool stuff<span class="pl-pds">"</span></span>

<span class="pl-c1">@declare_citation</span> <span class="pl-s"><span class="pl-pds">"</span>2019, Jane Doe, SomeCoolPackage.jl<span class="pl-pds">"</span></span>
Citations<span class="pl-k">.</span><span class="pl-en">cite</span>(<span class="pl-k">::</span><span class="pl-c1">typeof</span>(awesomefunction1)) <span class="pl-k">=</span> <span class="pl-s"><span class="pl-pds">"</span>2019, Jane Doe, awesomefunction1, SomeCoolPackage.jl<span class="pl-pds">"</span></span>
Citations<span class="pl-k">.</span><span class="pl-en">cite</span>(<span class="pl-k">::</span><span class="pl-c1">typeof</span>(awesomefunction2)) <span class="pl-k">=</span> <span class="pl-s"><span class="pl-pds">"</span>2019, Jane Doe, awesomefunction2, SomeCoolPackage.jl<span class="pl-pds">"</span></span>


<span class="pl-k">export</span> cite, awesomefunction1, awesomefunction2

<span class="pl-k">end</span></pre></div>
<p>Your users can then easily generate citation info at the REPL</p>
<div class="highlight highlight-source-julia"><pre>julia<span class="pl-k">&gt;</span> <span class="pl-k">using</span> SomeCoolPackage

julia<span class="pl-k">&gt;</span> <span class="pl-c1">cite</span>(SomeCoolPackage)
<span class="pl-s"><span class="pl-pds">"</span>2019, Jane Doe, SomeCoolPackage.jl<span class="pl-pds">"</span></span>

julia<span class="pl-k">&gt;</span> <span class="pl-c1">cite</span>(awesomefunction1)
<span class="pl-s"><span class="pl-pds">"</span>2019, Jane Doe, awesomefunction1, SomeCoolPackage.jl<span class="pl-pds">"</span></span>

julia<span class="pl-k">&gt;</span> <span class="pl-c1">cite</span>(awesomefunction2)
<span class="pl-s"><span class="pl-pds">"</span>2019, Jane Doe, awesomefunction2, SomeCoolPackage.jl<span class="pl-pds">"</span></span></pre></div>
