<p dir="auto">A <a href="https://plutojl.org/" rel="nofollow">Pluto.jl</a> plugin for <a href="https://github.com/NyanCAD/Mosaic">Mosaic</a>, for interactive exploration of schematics.</p>
<p dir="auto">Mosaic is a web based schematic editor, and Pluto is a interactive Julia notebook server.
By binding the schematic editor to a Pluto variable, any modifications to the schematic will immeditely run any dependant cells.</p>
