<p dir="auto">A repository of Python-dependent tools for working with <a href="https://github.com/SimonEnsemble/Xtals.jl"><code>Xtals.jl</code></a></p>
<p dir="auto"><code>Xtals</code> is a pure-Julia package for reading/writing/editing crystal structures.
Some higher-level features were previously dependent on Python packages; those have been moved here.</p>
<p dir="auto">To obtain the package, simply add it via the REPL package manager (minimum Julia version 1.4):</p>
<p dir="auto"><code>pkg&gt; add XtalsPyTools</code></p>
<p dir="auto">The methods in this package require <code>scipy</code> and/or <code>pymatgen</code> to be installed.
To configure these dependencies automatically in a Julia-managed environment, use:</p>
<p dir="auto"><code>julia quick_setup.jl</code></p>
<p dir="auto">See the <a href="https://SimonEnsemble.github.io/Xtals.jl/stable" rel="nofollow"><code>Xtals.jl</code> docs</a> for further information.</p>
