<p dir="auto">Simple tool to select feature based on the statistical relationship between features to target variable. The currently implemented feature is based on:</p>
<ul dir="auto">
<li>Correlation</li>
<li>P-value, which can be obtained by either Chi-square or F test</li>
</ul>
