<p dir="auto"><code>Webviews.jl</code> requires <strong>Julia v1.8 or later on Windows and macOS</strong>,
and <strong>Julia v1.9 or later on Linux</strong>.</p>
<p dir="auto">You can install this package with Julia's package manager:</p>
<div class="highlight highlight-source-julia notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="(@v1.8) pkg&gt; add Webviews"><pre>(<span class="pl-c1">@v1</span>.<span class="pl-c1">8</span>) pkg<span class="pl-k">&gt;</span> add Webviews</pre></div>
<p dir="auto">For Linux users, you need to install the following dependencies:</p>
<div class="highlight highlight-source-shell notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="# Fedora
sudo dnf install gtk3-devel webkit2gtk3-devel
# Debian/Ubuntu
sudo apt install libwebkit2gtk-4.0-dev
# Arch Linux
sudo pacman -S webkit2gtk"><pre><span class="pl-c"><span class="pl-c">#</span> Fedora</span>
sudo dnf install gtk3-devel webkit2gtk3-devel
<span class="pl-c"><span class="pl-c">#</span> Debian/Ubuntu</span>
sudo apt install libwebkit2gtk-4.0-dev
<span class="pl-c"><span class="pl-c">#</span> Arch Linux</span>
sudo pacman -S webkit2gtk</pre></div>
<p dir="auto"><strong>Note</strong>: <code>Webviews.jl</code> downloads its own prebuilt binaries (only on Windows) and depends on libraries that are provided by the operating system, instead of using JLL packages.</p>
