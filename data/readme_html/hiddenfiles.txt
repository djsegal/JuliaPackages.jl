<p dir="auto">A simple tool to determine if a file directory is hidden.  Works on anyâ„¢ OS and file system!</p>
<p dir="auto">This library exports one function: <code>ishidden</code>.  Typically, a file or directory is classified as "hidden" if is it hidden from a normal directory listing.  See <a href="https://jakewilliami.github.io/HiddenFiles.jl/dev" rel="nofollow">the documentation</a> for notes on the behaviour of <code>ishidden</code>.</p>
