<p><strong>Bayesian inference for Discrete-state-space Partially Observed Markov Processes in Julia</strong></p>
<p><a target="_blank" rel="noopener noreferrer" href="https://github.com/mjb3/DiscretePOMP.jl/workflows/Documentation/badge.svg"><img src="https://github.com/mjb3/DiscretePOMP.jl/workflows/Documentation/badge.svg" alt="Documentation" style="max-width:100%;"></a>
<a target="_blank" rel="noopener noreferrer" href="https://github.com/mjb3/DiscretePOMP.jl/workflows/Tests/badge.svg"><img src="https://github.com/mjb3/DiscretePOMP.jl/workflows/Tests/badge.svg" alt="Package tests" style="max-width:100%;"></a></p>
<p>This package contains tools for Bayesian inference and simulation of DPOMP models. See the <a href="https://mjb3.github.io/DiscretePOMP.jl/stable" rel="nofollow">docs</a>.</p>
<h2><a id="user-content-features" class="anchor" aria-hidden="true" href="#features"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path fill-rule="evenodd" d="M7.775 3.275a.75.75 0 001.06 1.06l1.25-1.25a2 2 0 112.83 2.83l-2.5 2.5a2 2 0 01-2.83 0 .75.75 0 00-1.06 1.06 3.5 3.5 0 004.95 0l2.5-2.5a3.5 3.5 0 00-4.95-4.95l-1.25 1.25zm-4.69 9.64a2 2 0 010-2.83l2.5-2.5a2 2 0 012.83 0 .75.75 0 001.06-1.06 3.5 3.5 0 00-4.95 0l-2.5 2.5a3.5 3.5 0 004.95 4.95l1.25-1.25a.75.75 0 00-1.06-1.06l-1.25 1.25a2 2 0 01-2.83 0z"></path></svg></a>Features</h2>
<ul>
<li>Simulation and</li>
<li>Bayesian parameter inference for,</li>
<li>Discrete-state-space Partially Observed Markov Processes, in Julia.</li>
<li>Includes automated tools for convergence diagnosis and analysis.</li>
</ul>
<h3><a id="user-content-applications" class="anchor" aria-hidden="true" href="#applications"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path fill-rule="evenodd" d="M7.775 3.275a.75.75 0 001.06 1.06l1.25-1.25a2 2 0 112.83 2.83l-2.5 2.5a2 2 0 01-2.83 0 .75.75 0 00-1.06 1.06 3.5 3.5 0 004.95 0l2.5-2.5a3.5 3.5 0 00-4.95-4.95l-1.25 1.25zm-4.69 9.64a2 2 0 010-2.83l2.5-2.5a2 2 0 012.83 0 .75.75 0 001.06-1.06 3.5 3.5 0 00-4.95 0l-2.5 2.5a3.5 3.5 0 004.95 4.95l1.25-1.25a.75.75 0 00-1.06-1.06l-1.25 1.25a2 2 0 01-2.83 0z"></path></svg></a>Applications</h3>
<ul>
<li>Epidemiological modelling (e.g. SEIR models)</li>
<li>Ecology (e.g. predator-prey dynamics)</li>
<li>Many other potential use cases, e.g. physics; chemical reactions; social media.</li>
</ul>
<h3><a id="user-content-algorithms" class="anchor" aria-hidden="true" href="#algorithms"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path fill-rule="evenodd" d="M7.775 3.275a.75.75 0 001.06 1.06l1.25-1.25a2 2 0 112.83 2.83l-2.5 2.5a2 2 0 01-2.83 0 .75.75 0 00-1.06 1.06 3.5 3.5 0 004.95 0l2.5-2.5a3.5 3.5 0 00-4.95-4.95l-1.25 1.25zm-4.69 9.64a2 2 0 010-2.83l2.5-2.5a2 2 0 012.83 0 .75.75 0 001.06-1.06 3.5 3.5 0 00-4.95 0l-2.5 2.5a3.5 3.5 0 004.95 4.95l1.25-1.25a.75.75 0 00-1.06-1.06l-1.25 1.25a2 2 0 01-2.83 0z"></path></svg></a>Algorithms</h3>
<p>The package implements several different customisable algorithms for Bayesian parameter inference, including:</p>
<ul>
<li>Data-augmented MCMC</li>
<li>Particle filters (i.e. Sequential Monte Carlo)</li>
<li>Iterative-batch-importance sampling (e.g. 'SMC^2')</li>
</ul>
