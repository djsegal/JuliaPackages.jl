<p dir="auto"><strong>Bayesian inference for Discrete-state-space Partially Observed Markov Processes in Julia</strong></p>
<p dir="auto"><a target="_blank" rel="noopener noreferrer" href="https://github.com/mjb3/DiscretePOMP.jl/workflows/Documentation/badge.svg"><img src="https://github.com/mjb3/DiscretePOMP.jl/workflows/Documentation/badge.svg" alt="Documentation" style="max-width: 100%;"></a>
<a target="_blank" rel="noopener noreferrer" href="https://github.com/mjb3/DiscretePOMP.jl/workflows/Tests/badge.svg"><img src="https://github.com/mjb3/DiscretePOMP.jl/workflows/Tests/badge.svg" alt="Package tests" style="max-width: 100%;"></a></p>
<p dir="auto">This package contains tools for Bayesian inference and simulation of DPOMP models. See the <a href="https://mjb3.github.io/DiscretePOMP.jl/stable" rel="nofollow">docs</a>.</p>

<p dir="auto">The package implements several different customisable algorithms for Bayesian parameter inference, including:</p>
<ul dir="auto">
<li>Data-augmented MCMC</li>
<li>Particle filters (i.e. Sequential Monte Carlo)</li>
<li>Iterative-batch-importance sampling (e.g. 'SMC^2')</li>
</ul>
