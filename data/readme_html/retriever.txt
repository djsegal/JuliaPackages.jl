<p dir="auto">The Julia wrapper for the Data Retriever software.</p>
<p dir="auto">The Data Retriever automates the tasks of finding, downloading,
and cleaning up publicly available data, and then stores them in a local database or as .csv files.
Simply put, it's a package manager for data.
This enables data analysts to devote the majority of their time to analysis rather than data cleaning or management.</p>

<p dir="auto">You can change that path to a desired path as below.</p>

<p dir="auto">To run tests using docker</p>
<p dir="auto"><code>docker-compose run --service-ports retrieverj julia test/runtests.jl</code></p>
<p dir="auto">To run the image interactively</p>
<p dir="auto"><code>docker-compose run --service-ports retrieverj /bin/bash</code></p>

<p dir="auto"><code> docker-compose run --service-ports retrieverj bash -c "cd docs &amp;&amp; julia make.jl"</code></p>
