<p dir="auto">A minimum package for displaying a graph and configurations defined on graphs.
It is the <a href="https://github.com/JuliaGraphics/Luxor.jl"><code>Luxor</code></a> version of <a href="https://github.com/JuliaGraphs/GraphPlot.jl"><code>GraphPlot</code></a>.</p>
<p dir="auto">Install by typing <code>using Pkg; Pkg.add("LuxorGraphPlot")</code> in a julia REPL.</p>
<p dir="auto">(NOTE: After implementing this package, I noticed there is a similar package with more features: <a href="https://github.com/cormullion/Karnak.jl">https://github.com/cormullion/Karnak.jl</a>.
<code>LuxorGraphPlot</code> is light weighted version targeting combinatorial optimization and lattice plots.)</p>
