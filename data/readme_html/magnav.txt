<p dir="auto">MagNav.jl contains a full suite of tools for airborne Magnetic anomaly Navigation (MagNav), including flight path &amp; INS data import or simulation, mapping, aeromagnetic compensation, and navigation. Julia source code files are in the <a href="src"><code>src</code></a> folder and examples are in the <a href="runs"><code>runs</code></a> folder. This package was developed as part of the <a href="https://aia.mit.edu/" rel="nofollow">DAF-MIT Artificial Intelligence Accelerator</a>. More information on this effort, including a list of relevant publications, is provided on the <a href="https://magnav.mit.edu/" rel="nofollow">challenge problem website</a>. The package has been tested on the long-term support (LTS) and latest stable versions of Julia, which may be downloaded from <a href="https://julialang.org/downloads/" rel="nofollow">here</a>.</p>

<p dir="auto">Publicly available flight data can be automatically downloaded within the package itself. This dataset can also be directly downloaded from <a href="https://doi.org/10.5281/zenodo.4271803" rel="nofollow">here</a>. See the <a href="readmes/datasheet_sgl_2020_train.pdf">datasheet</a> for high-level information about this dataset. Details of the flight data are described in the readme files within the <a href="readmes"><code>readmes</code></a> folder.</p>
