<p dir="auto">This package can be used to reconstruct materials from measured correlation
functions (as described in Salvatore Torquato's book "Random Heterogeneous
Materials"
<a href="https://www.springer.com/us/book/9780387951676" rel="nofollow">ISBN 978-1-4757-6357-7</a>) using
a procedure called "Simulated annealing".</p>
<p dir="auto">Look <a href="https://fatimp.github.io/MaterialReconstruction.jl/stable/" rel="nofollow">here</a> for
documentation and examples.</p>
