<p>Provides methods for doing simple analysis of detached binary stars</p>
<ul>
<li>Determine (if any) the eclipse morphology of a system</li>
<li>If the system eclipses, determine the eclipse duration</li>
<li>determine if the system is experiencing Roche Lomb overflow</li>
<li>get a spherical star approximation of the transit flux depth (no limb darkening yet)</li>
</ul>
