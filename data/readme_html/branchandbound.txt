<p dir="auto">This Julia package provides an abstract framework for branch-and-bound methods. Examples of this package include:</p>
<ul dir="auto">
<li>a branch-and-bound method for a global solution of mixed-integer linear programming (see <code>./examples/milp.jl</code>)</li>
<li>a branch-and-bound method for a local solution of mixed-integer nonlinear programming (see <code>./examples/minlp.jl</code>)</li>
</ul>
