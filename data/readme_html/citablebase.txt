<p>A Julia module defining the two core abstractions of the CITE architecture:</p>
<ol>
<li>identifiers expressible using the syntax of the IETF URN specification</li>
<li>citable units, identified by URN, and including a human-readable label</li>
</ol>
<p>Although the package includes a few trivial convenience functions, its main purpose is to provide types that can be extended for specific types of URNs and specific types of citable entities.  See:</p>
<ul>
<li><a href="https://github.com/cite-architecture/CitableText.jl">CitableText.jl</a></li>
<li><a href="https://github.com/cite-architecture/CitableObject.jl">CitableObject.jl</a></li>
<li><a href="https://github.com/cite-architecture/CitablePhysicalText.jl">CitablePhysicalText.jl</a></li>
<li>CitableImage.jl (in progress)</li>
</ul>
