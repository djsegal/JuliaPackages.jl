<p dir="auto"><a href="https://lxvm.github.io/AutoSymPTR.jl/dev/" rel="nofollow">Documentation</a></p>
<p dir="auto">This package provides p-adaptive quadrature for periodic integrands using the
spectrally-convergent
<a href="https://en.wikipedia.org/wiki/Trapezoidal_rule#Periodic_and_peak_functions" rel="nofollow">Periodic Trapezoidal Rule (PTR)</a>
Convergence to a user-specified tolerance is automated by the algorithm.</p>
<p dir="auto">Additionally, symmetries can be used to reduce the number of distinct quadrature
points needed for the integration, which in computational solid state physics is
known as Monkhorst-Pack integration.</p>
