<p dir="auto">This package implements different hypothesis tests for calibration of
probabilistic models in the Julia language.</p>

<p dir="auto">The statistical tests in this package are based on the calibration error estimators
in the package <a href="https://github.com/devmotion/CalibrationErrors.jl">CalibrationErrors.jl</a>.</p>
<p dir="auto"><a href="https://github.com/devmotion/pycalibration">pycalibration</a> is a Python interface for CalibrationErrors.jl and CalibrationTests.jl.</p>
<p dir="auto"><a href="https://github.com/devmotion/rcalibration">rcalibration</a> is an R interface for CalibrationErrors.jl and CalibrationTests.jl.</p>
