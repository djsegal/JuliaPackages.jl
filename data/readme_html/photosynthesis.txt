<p dir="auto"><code>Photosynthesis.jl</code> has been refactored based on the ClimaCache.jl package. Functions has been renamed to be more specific, but it is getting more user-friendly. As <code>Photosynthesis.jl</code> is a module of CliMA Land, please cite <a href="https://github.com/CliMA/Land">CliMA Land</a> when you use <code>Photosynthesis.jl</code>. We have devote most of the time to developing the CliMA Land, and thus have minimum time to document the model. If you want to use the model, please make a request through <a href="https://github.com/Yujie-W/Photosynthesis.jl/issues/56" data-hovercard-type="issue" data-hovercard-url="/Yujie-W/Photosynthesis.jl/issues/56/hovercard">this Github Issue</a>.</p>
