<p>This Package provides methods for the rigorous approximation of Absolutely Continuous Invariant Measures for one dimensional dynamical systems,
using the results in <a href="#2">[2]</a> and subsequent works.</p>
<h2><a id="user-content-mathematical-background" class="anchor" aria-hidden="true" href="#mathematical-background"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path fill-rule="evenodd" d="M7.775 3.275a.75.75 0 001.06 1.06l1.25-1.25a2 2 0 112.83 2.83l-2.5 2.5a2 2 0 01-2.83 0 .75.75 0 00-1.06 1.06 3.5 3.5 0 004.95 0l2.5-2.5a3.5 3.5 0 00-4.95-4.95l-1.25 1.25zm-4.69 9.64a2 2 0 010-2.83l2.5-2.5a2 2 0 012.83 0 .75.75 0 001.06-1.06 3.5 3.5 0 00-4.95 0l-2.5 2.5a3.5 3.5 0 004.95 4.95l1.25-1.25a.75.75 0 00-1.06-1.06l-1.25 1.25a2 2 0 01-2.83 0z"></path></svg></a>Mathematical background</h2>
<p>By Birkhoff Ergodic Theorem we know that if a dynamical system T admits an ergodic invariant measure μ, for μ-almost every point x the frequence of the visits to a set E corresponds to the measure of the set with respect to μ.</p>
<p>Therefore, being able to approximate invariant measures with a large basin is interesting to investigate the statistical properties of the dynamical system T. <a href="#4">[4]</a></p>
<p>The existence of absolutely continuous invariant measures for one dimensional maps is a delicate topic; with this package we present approximation schemes for the invariant measures of system that satisfy a Lasota-Yorke inequality through the use of a coarse-fine scheme and a posteriori estimates on the mixing time, i.e., this means that the mixing time is estimated by our algorithm and we do not need an a priori estimate, which is usually difficult to obtain.</p>
<p>The Ulam approximation schemes works under relatively weak hypothesis on the dynamics and was used in <a href="#3">[3]</a> to approximate the invariant measure for the geometric Lorenz 1-dimensional map.</p>
<p>We are currently working on the implementation of the Ulam scheme for system with additive uniform noise, as the one used in
<a href="#1">[1]</a></p>

<p>Examples of usage are present in the directory examples.</p>
<div class="highlight highlight-source-julia position-relative" data-snippet-clipboard-copy-content="using InvariantMeasures
D = Mod1Dynamic(x -&gt; 4x + 0.01InvariantMeasures.sinpi(8x))
B = Ulam(1024)
Q = DiscretizedOperator(B, D)
"><pre><span class="pl-k">using</span> InvariantMeasures
D <span class="pl-k">=</span> <span class="pl-c1">Mod1Dynamic</span>(x <span class="pl-k">-&gt;</span> <span class="pl-c1">4</span>x <span class="pl-k">+</span> <span class="pl-c1">0.01</span>InvariantMeasures<span class="pl-k">.</span><span class="pl-c1">sinpi</span>(<span class="pl-c1">8</span>x))
B <span class="pl-k">=</span> <span class="pl-c1">Ulam</span>(<span class="pl-c1">1024</span>)
Q <span class="pl-k">=</span> <span class="pl-c1">DiscretizedOperator</span>(B, D)</pre></div>
<p>The code snippet above defines a dynamic obtained by reducing f(x) = 4x+0.01 sin(8πx) modulo 1, a basis B associated to the Ulam discretization on a partition of 1024 homogenous intervals, and computes the discretized operator Q, a Markov chain whose entries are P[T(x)∈ Iᵢ | x ∈ Iⱼ].</p>
<p>Note the usage of <code>InvariantMeasures.sinpi(8*x)</code> rather than <code>Base.sinpi</code> or <code>Base.sin(8\pi*x)</code>. This detail is required to ensure that f(1) == 4 exactly.</p>
<div class="highlight highlight-source-julia position-relative" data-snippet-clipboard-copy-content="norms = norms = powernormbounds(B, D; Q=Q)
"><pre>norms <span class="pl-k">=</span> norms <span class="pl-k">=</span> <span class="pl-c1">powernormbounds</span>(B, D; Q<span class="pl-k">=</span>Q)</pre></div>
<p>This function computes the L¹ norm of Q^k, for k = 1,2,...,k_max (up to a sufficiently large number of powers to observe decay) when restricted to the space U of average-0 vectors. This gives us the a posteriori estimate for the mixing time of the Markov chain and is used in our rigorous estimate.</p>
<div class="highlight highlight-source-julia position-relative" data-snippet-clipboard-copy-content="w = invariant_vector(B, Q)
distance_from_invariant(B, D, Q, w, norms)
"><pre>w <span class="pl-k">=</span> <span class="pl-c1">invariant_vector</span>(B, Q)
<span class="pl-c1">distance_from_invariant</span>(B, D, Q, w, norms)</pre></div>
<p>This computes a (non-rigorous) approximation of the invariant measure of D; then  <code>distance_from_invariant</code> computes an upper bound for the L¹ distance between w and the density of the absolutely continuous invariant measure of the system.</p>
<p>Inside the examples it is showed how to use the coarse-fine scheme to obtain better L¹ bounds and reduce the computational time.</p>

<p>The function <code>sinpi</code> in the interval arithmetic package that we are using relies on the <code>CRlibm.jl</code> package, which currently works only under Linux. So the examples that use trigonometric functions only work on this OS.</p>
