<p dir="auto">Effective field theory (EFT) models the correlation functions of quantum many-body systems with an effective action defined below a certain energy/momentum scale. EFT has been widely used in high-energy physics, nuclear physics, and condensed matter physics. Originally, EFT was developed assuming <strong>the principle of locality</strong>, which states that an object is influenced directly only by its immediate surroundings. If one observes a long-range interaction between two particles (say, the Coulomb interaction between two electrons), then such interaction must be mediated by an intermediate particle called a gauge boson (the photon in the previous example).</p>
<p dir="auto">All known theories of the elementary particles in our universe are local theories, but it does not mean the local EFT rules everything. Below the energy scales most relevant for human life, the speed of light can be safely assumed to be infinite so that effective physical laws are governed by an absolute Newtonian time. In those systems, an event at one point can cause a simultaneous result at another point. The EFTs describe such systems are then nonlocal.</p>
<p dir="auto">We create this package to extend the conventional EFT to incorporate the nonlocality. Such nonlocal EFTs are typically much more difficult to calculate compared to the conventional EFTs. Analytic solution is not feasible anymore, and numerical solution is usually the only option. We thus develop this numerical package to solve the problem and make nonlocal EFT a practical tool for real-world applications, such as the electron liquids in real materials, chiral EFT in the neutron-rich matter, and all kinds of emergent low-energy field theories in lattice models.</p>

<p dir="auto">The package <code>NumericalEFT.jl</code> is a collection of several independent components, which are organized in the following infrastructure:</p>
<p dir="auto"><a target="_blank" rel="noopener noreferrer" href="docs/src/assets/numericalEFT_clickable.svg"><img src="docs/src/assets/numericalEFT_clickable.svg" alt="NumericalEFT" style="max-width: 100%;"></a></p>
<p dir="auto">Most of the components have been published as independent packages, and you can test or use them separately. The packages are still fast-evolving, but many are production-ready.</p>
<h3 dir="auto"><a id="user-content-core-packages-a-set-of-self-contained-and-general-purpose-tools" class="anchor" aria-hidden="true" href="#core-packages-a-set-of-self-contained-and-general-purpose-tools"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path d="m7.775 3.275 1.25-1.25a3.5 3.5 0 1 1 4.95 4.95l-2.5 2.5a3.5 3.5 0 0 1-4.95 0 .751.751 0 0 1 .018-1.042.751.751 0 0 1 1.042-.018 1.998 1.998 0 0 0 2.83 0l2.5-2.5a2.002 2.002 0 0 0-2.83-2.83l-1.25 1.25a.751.751 0 0 1-1.042-.018.751.751 0 0 1-.018-1.042Zm-4.69 9.64a1.998 1.998 0 0 0 2.83 0l1.25-1.25a.751.751 0 0 1 1.042.018.751.751 0 0 1 .018 1.042l-1.25 1.25a3.5 3.5 0 1 1-4.95-4.95l2.5-2.5a3.5 3.5 0 0 1 4.95 0 .751.751 0 0 1-.018 1.042.751.751 0 0 1-1.042.018 1.998 1.998 0 0 0-2.83 0l-2.5 2.5a1.998 1.998 0 0 0 0 2.83Z"></path></svg></a>Core packages: a set of self-contained and general-purpose tools.</h3>
<ul dir="auto">
<li>
<p dir="auto"><a href="https://github.com/numericalEFT/Lehmann.jl">Lehmann.jl</a>: Discrete Lehmann representation (DLR) for imaginary-time/Matsubara frequency Green's function. For a generic Green's function at a temperature T, DLR is capable of representing it up to a given accuracy ϵ with a cost ~ log(1/T)log(1/ϵ), significantly cheaper than a naive approach with a cost ~ 1/(Tϵ).</p>
</li>
<li>
<p dir="auto"><a href="https://github.com/numericalEFT/FeynmanDiagram.jl">FeynmanDiagram.jl</a>: A mini-compiler that compiles generic multi-loop Feynman diagrams to a computational graph, which then can be efficiently manipulated or evaluated. The same type of computational graph has been the corner of neural-network-based machine learning.</p>
</li>
<li>
<p dir="auto"><a href="https://github.com/numericalEFT/MCIntegration.jl">MCIntegration.jl</a>: An adaptive Monte Carlo calculator for general high dimensional integral. It is one of the most robust and fastest Monte Carlo integration packages on the market.</p>
</li>
<li>
<p dir="auto"><a href="https://github.com/numericalEFT/CompositeGrids.jl">CompositeGrids.jl</a>: Composite Cheybeshev/Gaussian/logarithmic grid systems for highly efficient one-dimensional interpolation and integration. It allows the user to combine different grids to represent nontrivial functions.</p>
</li>
<li>
<p dir="auto"><a href="https://github.com/numericalEFT/BrillouinZoneMeshes.jl">BrillouinZoneMeshes.jl</a>: Compact mesh systems for representing and manipulating functions defined on generic Brillouin zone in 2D or 3D.</p>
</li>
</ul>
<h3 dir="auto"><a id="user-content-toolbox-packages-integrated-software-for-quantum-many-body-problems" class="anchor" aria-hidden="true" href="#toolbox-packages-integrated-software-for-quantum-many-body-problems"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path d="m7.775 3.275 1.25-1.25a3.5 3.5 0 1 1 4.95 4.95l-2.5 2.5a3.5 3.5 0 0 1-4.95 0 .751.751 0 0 1 .018-1.042.751.751 0 0 1 1.042-.018 1.998 1.998 0 0 0 2.83 0l2.5-2.5a2.002 2.002 0 0 0-2.83-2.83l-1.25 1.25a.751.751 0 0 1-1.042-.018.751.751 0 0 1-.018-1.042Zm-4.69 9.64a1.998 1.998 0 0 0 2.83 0l1.25-1.25a.751.751 0 0 1 1.042.018.751.751 0 0 1 .018 1.042l-1.25 1.25a3.5 3.5 0 1 1-4.95-4.95l2.5-2.5a3.5 3.5 0 0 1 4.95 0 .751.751 0 0 1-.018 1.042.751.751 0 0 1-1.042.018 1.998 1.998 0 0 0-2.83 0l-2.5 2.5a1.998 1.998 0 0 0 0 2.83Z"></path></svg></a>Toolbox packages: integrated software for quantum many-body problems</h3>
<ul dir="auto">
<li>
<p dir="auto"><a href="https://github.com/numericalEFT/GreenFunc.jl">GreenFunc.jl</a>: A software to calculate, manipulate and analyze the Green's functions and the vertex functions at the tree-level, which characterize the quantum field dynamics in the semiclassical limit ħ→0. It consists of:</p>
<ul dir="auto">
<li>A container of generic Green's functions based on the discrete Lehmann representation and adaptive mesh systems.</li>
<li>Several helper functions to generate typical Green's functions. For example, a set of functions to calculate the many-body Green's functions of a small cluster of Hubbard-type atoms is a small-scale exact diagonalization solver.</li>
<li>Import(export) Green's function from(to) the <a href="https://triqs.github.io/triqs/latest/" rel="nofollow">Triqs</a>.</li>
</ul>
</li>
<li>
<p dir="auto"><a href="https://github.com/numericalEFT/MultiLoop.jl">MultiLoop.jl</a> (WIP): A software to calculate and analyze the quantum many-body effects beyond the semiclassical limit. It consists of:</p>
<ul dir="auto">
<li>Diagrammatic Monte Carlo calculator of multi-loop Feynman diagrams of correlation functions and vertex functions.</li>
<li>Renormalization technique to improve the perturbation theory.</li>
<li>Renormalization Group analysis.</li>
</ul>
</li>
</ul>
