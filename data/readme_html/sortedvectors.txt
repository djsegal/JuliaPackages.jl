<h3><a id="user-content-standard-constructor" class="anchor" aria-hidden="true" href="#standard-constructor"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path fill-rule="evenodd" d="M4 9h1v1H4c-1.5 0-3-1.69-3-3.5S2.55 3 4 3h4c1.45 0 3 1.69 3 3.5 0 1.41-.91 2.72-2 3.25V8.59c.58-.45 1-1.27 1-2.09C10 5.22 8.98 4 8 4H4c-.98 0-2 1.22-2 2.5S3 9 4 9zm9-3h-1v1h1c1 0 2 1.22 2 2.5S13.98 12 13 12H9c-.98 0-2-1.22-2-2.5 0-.83.42-1.64 1-2.09V6.25c-1.09.53-2 1.84-2 3.25C6 11.31 7.55 13 9 13h4c1.45 0 3-1.69 3-3.5S14.5 6 13 6z"></path></svg></a>Standard constructor</h3>
<p>The only exported symbol is <code>SortedVector</code>. Use</p>
<div class="highlight highlight-source-julia"><pre><span class="pl-c1">SortedVector</span>([lt<span class="pl-k">=</span>isless], xs)</pre></div>
<p>to sort <code>xs</code> and save the result in a vector. <code>xs</code> can be any <code>&lt;: AbstractVector</code>. For immutable types (eg <code>StaticArrays.SVector</code> or <code>UnitRange</code>), <code>setindex!</code> will not work.</p>
<h3><a id="user-content-special-constructors-for-checking-or-skipping-sorting" class="anchor" aria-hidden="true" href="#special-constructors-for-checking-or-skipping-sorting"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path fill-rule="evenodd" d="M4 9h1v1H4c-1.5 0-3-1.69-3-3.5S2.55 3 4 3h4c1.45 0 3 1.69 3 3.5 0 1.41-.91 2.72-2 3.25V8.59c.58-.45 1-1.27 1-2.09C10 5.22 8.98 4 8 4H4c-.98 0-2 1.22-2 2.5S3 9 4 9zm9-3h-1v1h1c1 0 2 1.22 2 2.5S13.98 12 13 12H9c-.98 0-2-1.22-2-2.5 0-.83.42-1.64 1-2.09V6.25c-1.09.53-2 1.84-2 3.25C6 11.31 7.55 13 9 13h4c1.45 0 3-1.69 3-3.5S14.5 6 13 6z"></path></svg></a>Special constructors for checking or skipping sorting</h3>
<p>If your code emits sorted vectors, use the</p>
<div class="highlight highlight-source-julia"><pre><span class="pl-c1">SortedVector</span>(SortedVectors<span class="pl-k">.</span><span class="pl-c1">AssumeSorted</span>(), lt, sorted_contents)</pre></div>
<p>constructor. This will skip checks.</p>
<p>If your API accepts sorted vectors, and you want to check them, use the</p>
<div class="highlight highlight-source-julia"><pre><span class="pl-c1">SortedVector</span>(SortedVectors<span class="pl-k">.</span><span class="pl-c1">CheckSorted</span>(), lt, sorted_contents)</pre></div>
<p>constructor.</p>
<p><strong>In either case, you are responsible for ensuring that the argument vector is not modified later on. <code>copy</code> if you are unsure.</strong></p>
