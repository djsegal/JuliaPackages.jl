<h3><a id="user-content-standard-constructor" class="anchor" aria-hidden="true" href="#standard-constructor"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path fill-rule="evenodd" d="M7.775 3.275a.75.75 0 001.06 1.06l1.25-1.25a2 2 0 112.83 2.83l-2.5 2.5a2 2 0 01-2.83 0 .75.75 0 00-1.06 1.06 3.5 3.5 0 004.95 0l2.5-2.5a3.5 3.5 0 00-4.95-4.95l-1.25 1.25zm-4.69 9.64a2 2 0 010-2.83l2.5-2.5a2 2 0 012.83 0 .75.75 0 001.06-1.06 3.5 3.5 0 00-4.95 0l-2.5 2.5a3.5 3.5 0 004.95 4.95l1.25-1.25a.75.75 0 00-1.06-1.06l-1.25 1.25a2 2 0 01-2.83 0z"></path></svg></a>Standard constructor</h3>
<p>The only exported symbol is <code>SortedVector</code>. Use</p>
<div class="highlight highlight-source-julia position-relative" data-snippet-clipboard-copy-content="SortedVector([lt=isless], xs)
"><pre><span class="pl-c1">SortedVector</span>([lt<span class="pl-k">=</span>isless], xs)</pre></div>
<p>to sort <code>xs</code> and save the result in a vector. <code>xs</code> can be any <code>&lt;: AbstractVector</code>. For immutable types (eg <code>StaticArrays.SVector</code> or <code>UnitRange</code>), <code>setindex!</code> will not work.</p>
<h3><a id="user-content-special-constructors-for-checking-or-skipping-sorting" class="anchor" aria-hidden="true" href="#special-constructors-for-checking-or-skipping-sorting"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path fill-rule="evenodd" d="M7.775 3.275a.75.75 0 001.06 1.06l1.25-1.25a2 2 0 112.83 2.83l-2.5 2.5a2 2 0 01-2.83 0 .75.75 0 00-1.06 1.06 3.5 3.5 0 004.95 0l2.5-2.5a3.5 3.5 0 00-4.95-4.95l-1.25 1.25zm-4.69 9.64a2 2 0 010-2.83l2.5-2.5a2 2 0 012.83 0 .75.75 0 001.06-1.06 3.5 3.5 0 00-4.95 0l-2.5 2.5a3.5 3.5 0 004.95 4.95l1.25-1.25a.75.75 0 00-1.06-1.06l-1.25 1.25a2 2 0 01-2.83 0z"></path></svg></a>Special constructors for checking or skipping sorting</h3>
<p>If your code emits sorted vectors, use the</p>
<div class="highlight highlight-source-julia position-relative" data-snippet-clipboard-copy-content="SortedVector(SortedVectors.AssumeSorted(), lt, sorted_contents)
"><pre><span class="pl-c1">SortedVector</span>(SortedVectors<span class="pl-k">.</span><span class="pl-c1">AssumeSorted</span>(), lt, sorted_contents)</pre></div>
<p>constructor. This will skip checks.</p>
<p>If your API accepts sorted vectors, and you want to check them, use the</p>
<div class="highlight highlight-source-julia position-relative" data-snippet-clipboard-copy-content="SortedVector(SortedVectors.CheckSorted(), lt, sorted_contents)
"><pre><span class="pl-c1">SortedVector</span>(SortedVectors<span class="pl-k">.</span><span class="pl-c1">CheckSorted</span>(), lt, sorted_contents)</pre></div>
<p>constructor.</p>
<p><strong>In either case, you are responsible for ensuring that the argument vector is not modified later on. <code>copy</code> if you are unsure.</strong></p>
