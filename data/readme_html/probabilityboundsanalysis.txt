<p><a href="https://en.wikipedia.org/wiki/Probability_bounds_analysis" rel="nofollow">Probability bounds analysis</a> in Julia, a package for performing arithmetic between uncertain numbers. <code>ProbabilityBoundsAnalysis.jl</code> computes guaranteed bounds on functions of random variables, given only partial information about their marginals and dependence. Considered to be a form of rigorous computing with random variables.</p>
<p>The mayority of this code is a port from <a href="https://github.com/ScottFerson/pba.r">pba.r</a> by Scott Ferson and Jason O'Rawe, Applied Biomathematics (2006)</p>
<p>Supported uncertain numbers:</p>
<ul>
<li>scalars</li>
<li><a href="https://en.wikipedia.org/wiki/Cumulative_distribution_function" rel="nofollow">probability distributions</a></li>
<li><a href="https://en.wikipedia.org/wiki/Interval_arithmetic" rel="nofollow">intervals</a></li>
<li><a href="https://en.wikipedia.org/wiki/Probability_box" rel="nofollow">probability boxes</a> (p-boxes)</li>
</ul>

<h2><a id="user-content-uncertain-numbers" class="anchor" aria-hidden="true" href="#uncertain-numbers"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path fill-rule="evenodd" d="M7.775 3.275a.75.75 0 001.06 1.06l1.25-1.25a2 2 0 112.83 2.83l-2.5 2.5a2 2 0 01-2.83 0 .75.75 0 00-1.06 1.06 3.5 3.5 0 004.95 0l2.5-2.5a3.5 3.5 0 00-4.95-4.95l-1.25 1.25zm-4.69 9.64a2 2 0 010-2.83l2.5-2.5a2 2 0 012.83 0 .75.75 0 001.06-1.06 3.5 3.5 0 00-4.95 0l-2.5 2.5a3.5 3.5 0 004.95 4.95l1.25-1.25a.75.75 0 00-1.06-1.06l-1.25 1.25a2 2 0 01-2.83 0z"></path></svg></a>Uncertain numbers</h2>
<p>A probability distribution can be created by using it's shape and parameters:</p>
<div class="highlight highlight-source-julia"><pre>julia<span class="pl-k">&gt;</span> a <span class="pl-k">=</span> <span class="pl-c1">normal</span>(<span class="pl-c1">0</span>,<span class="pl-c1">1</span>)
Pbox<span class="pl-k">:</span> 	  <span class="pl-k">~</span> normal ( range<span class="pl-k">=</span>[<span class="pl-k">-</span><span class="pl-c1">3.090232</span>,<span class="pl-c1">3.090232</span>], mean<span class="pl-k">=</span><span class="pl-c1">0.0</span>, var<span class="pl-k">=</span><span class="pl-c1">1.0</span>)</pre></div>
<p>In <code>ProbabilityBoundsAnalysis.jl</code> probability distributions are p-boxes, but with equal bounds and precise moments.</p>
<p>The <code>IntervalArithmetic.jl</code> package is used to create intervals:</p>
<div class="highlight highlight-source-julia"><pre>julia<span class="pl-k">&gt;</span> b <span class="pl-k">=</span> <span class="pl-c1">interval</span>(<span class="pl-c1">0</span>,<span class="pl-c1">1</span>)
[<span class="pl-c1">0</span>, <span class="pl-c1">1</span>]</pre></div>
<p><code>IntervalArithmetic.jl</code> will work natively with <code>ProbabilityBoundsAnalysis.jl</code>, however an interval can be converted to a p-box in the following way:</p>
<div class="highlight highlight-source-julia"><pre>julia<span class="pl-k">&gt;</span> b <span class="pl-k">=</span> <span class="pl-c1">makepbox</span>(<span class="pl-c1">interval</span>(<span class="pl-c1">0</span>,<span class="pl-c1">1</span>))
Pbox<span class="pl-k">:</span> 	  <span class="pl-k">~</span>  ( range<span class="pl-k">=</span>[<span class="pl-c1">0.0</span>,<span class="pl-c1">1.0</span>], mean<span class="pl-k">=</span>[<span class="pl-c1">0.0</span>,<span class="pl-c1">1.0</span>], var<span class="pl-k">=</span>[<span class="pl-c1">0.0</span>,<span class="pl-c1">0.25</span>])</pre></div>
<p>There are a number of ways that p-boxes can be created. For example they are the result of arithemtic between random variables with unknown dependence. They can also be defined by using a distributions shape but with interval parameters:</p>
<div class="highlight highlight-source-julia"><pre>julia<span class="pl-k">&gt;</span> c <span class="pl-k">=</span> <span class="pl-c1">normal</span>(<span class="pl-c1">interval</span>(<span class="pl-c1">0</span>,<span class="pl-c1">1</span>),<span class="pl-c1">1</span>)
Pbox<span class="pl-k">:</span> 	  <span class="pl-k">~</span> normal ( range<span class="pl-k">=</span>[<span class="pl-k">-</span><span class="pl-c1">3.09023</span>,<span class="pl-c1">4.0902322</span>], mean<span class="pl-k">=</span>[<span class="pl-c1">0.0</span>,<span class="pl-c1">1.0</span>], var<span class="pl-k">=</span><span class="pl-c1">1.0</span>)</pre></div>
<p>or by taking the envelope over a number of uncertain numbers:</p>
<div class="highlight highlight-source-julia"><pre>julia<span class="pl-k">&gt;</span> d <span class="pl-k">=</span> <span class="pl-c1">normal</span>(<span class="pl-k">-</span><span class="pl-c1">1</span>,<span class="pl-c1">1</span>); 
julia<span class="pl-k">&gt;</span> e <span class="pl-k">=</span> <span class="pl-c1">normal</span>(<span class="pl-c1">1</span>, <span class="pl-c1">interval</span>(<span class="pl-c1">1</span>,<span class="pl-c1">2</span>));
julia<span class="pl-k">&gt;</span> f <span class="pl-k">=</span> <span class="pl-c1">env</span>(d,e)
Pbox<span class="pl-k">:</span> 	  <span class="pl-k">~</span> normal ( range<span class="pl-k">=</span>[<span class="pl-k">-</span><span class="pl-c1">5.18046</span>,<span class="pl-c1">7.1804646</span>], mean<span class="pl-k">=</span>[<span class="pl-k">-</span><span class="pl-c1">1.0</span>,<span class="pl-c1">1.0</span>], var<span class="pl-k">=</span>[<span class="pl-c1">1.0</span>,<span class="pl-c1">4.0</span>])</pre></div>
<p>and may be plotted as follows:</p>
<div class="highlight highlight-source-julia"><pre>julia<span class="pl-k">&gt;</span> <span class="pl-c1">plot</span>(f)</pre></div>
<p><a target="_blank" rel="noopener noreferrer" href="https://github.com/AnderGray/pba.jl/blob/master/doc/plots/PbaPlot1.png"><img src="https://github.com/AnderGray/pba.jl/raw/master/doc/plots/PbaPlot1.png" alt="alt text" title="a probability box" style="max-width:100%;"></a></p>

<p>The dependency package is not yet a part of the main module. To use</p>
<div class="highlight highlight-source-julia"><pre>julia<span class="pl-k">&gt;</span> <span class="pl-c1">include</span>(<span class="pl-s"><span class="pl-pds">"</span>directory/of/source/src/pbox/dependency.jl<span class="pl-pds">"</span></span>)</pre></div>
<p>Any possible multivariate dependence can be encoded in a copula. Copulas, sometimes called dependency functions, are a joint cdf with standard uniform marginals, and are a way to model dependency independently of marginal distributions.</p>
<p>The supported copulas are:</p>
<ul>
<li>π: independence</li>
<li>M: perfect dependence and the upper Frechét bound</li>
<li>W: opposite dependence and the lower Frechét bound</li>
<li>Gaussian: family with correlation coefficient r (-1 for W, 0 for π, 1 for M)</li>
<li>Frank: family with parameter s (0 for W, 1 for π, Inf for M)</li>
<li>Clayton: family with parameter t (-1 for W, 0 for π, Inf for M)</li>
</ul>
<p>A copula can be created and plotted in the following way:</p>
<div class="highlight highlight-source-julia"><pre>julia<span class="pl-k">&gt;</span> C <span class="pl-k">=</span> <span class="pl-c1">Gaussian</span>(<span class="pl-c1">0.7</span>)
Copula <span class="pl-k">~</span> <span class="pl-c1">Gau</span>(r<span class="pl-k">=</span><span class="pl-c1">0.7</span>)

julia<span class="pl-k">&gt;</span> <span class="pl-c1">plotCdf</span>(a)</pre></div>
<p><a target="_blank" rel="noopener noreferrer" href="https://github.com/AnderGray/ProbabilityBoundsAnalysis.jl/blob/master/doc/plots/GaussianCopula.png"><img src="https://github.com/AnderGray/ProbabilityBoundsAnalysis.jl/raw/master/doc/plots/GaussianCopula.png" alt="alt text" title="Gaussian copula with a correlation of 0.7" style="max-width:100%;"></a></p>
<p>The density can be plotted using the <code>plotDensity()</code> function. A copula C is the function C:[0,1]<sup>d</sup> →[0,1], where d is the dimension of the copula. Only bivariate copulas are considered so far in <code>ProbabilityBoundsAnalysis.jl</code>. A copula can be evaluated and sampled in the following way:</p>
<div class="highlight highlight-source-julia"><pre>julia<span class="pl-k">&gt;</span> <span class="pl-c1">C</span>(<span class="pl-c1">0.5</span>,<span class="pl-c1">0.5</span>)
<span class="pl-c1">0</span><span class="pl-k">-</span>dimensional Array{Float64,<span class="pl-c1">0</span>}<span class="pl-k">:</span>
<span class="pl-c1">0.37340834444668247</span>

julia<span class="pl-k">&gt;</span> <span class="pl-c1">C</span>([<span class="pl-c1">0.2</span>,<span class="pl-c1">0.6</span>],[<span class="pl-c1">0.3</span>,<span class="pl-c1">0.7</span>])
<span class="pl-c1">2</span><span class="pl-k">×</span><span class="pl-c1">2</span> Array{Float64,<span class="pl-c1">2</span>}<span class="pl-k">:</span>
 <span class="pl-c1">0.143315</span>  <span class="pl-c1">0.194937</span>
 <span class="pl-c1">0.273398</span>  <span class="pl-c1">0.52667</span> 
 
julia<span class="pl-k">&gt;</span> samps <span class="pl-k">=</span> <span class="pl-c1">sample</span>(C,<span class="pl-c1">10</span><span class="pl-k">^</span><span class="pl-c1">6</span>);</pre></div>
<p>Given any marginal distributions F<sub>X</sub>(x) and F<sub>Y</sub>(y) and a copla C( : ), a joint distribution can be created: H(x,y) = C(F<sub>X</sub>(x),F<sub>Y</sub>(y)).  In <code>ProbabilityBoundsAnalysis.jl</code> a joint distribution can be created by passing marginals to the copula. For example, a distribution with beta marginals and a gaussian copula:</p>
<div class="highlight highlight-source-julia"><pre>julia<span class="pl-k">&gt;</span> J <span class="pl-k">=</span> <span class="pl-c1">C</span>(<span class="pl-c1">Beta</span>(<span class="pl-c1">4</span>,<span class="pl-c1">2</span>),<span class="pl-c1">Beta</span>(<span class="pl-c1">4</span>,<span class="pl-c1">2</span>))
Joint <span class="pl-k">~</span> <span class="pl-c1">Gau</span>( r<span class="pl-k">=</span><span class="pl-c1">0.7</span>, <span class="pl-c1">Beta</span><span class="pl-c1">{Float64}</span>(α<span class="pl-k">=</span><span class="pl-c1">4.0</span>, β<span class="pl-k">=</span><span class="pl-c1">2.0</span>), <span class="pl-c1">Beta</span><span class="pl-c1">{Float64}</span>(α<span class="pl-k">=</span><span class="pl-c1">4.0</span>, β<span class="pl-k">=</span><span class="pl-c1">2.0</span>) )

julia<span class="pl-k">&gt;</span> <span class="pl-c1">plotDensity</span>(J)</pre></div>
<p><a target="_blank" rel="noopener noreferrer" href="https://github.com/AnderGray/ProbabilityBoundsAnalysis.jl/blob/master/doc/plots/JointDist1.png"><img src="https://github.com/AnderGray/ProbabilityBoundsAnalysis.jl/raw/master/doc/plots/JointDist1.png" alt="alt text" title="Bivariate Beta with gaussian copula with correlation of 0.7" style="max-width:100%;"></a></p>
