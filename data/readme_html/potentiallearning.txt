<p dir="auto">An open source Julia library for active learning of interatomic potentials in atomistic simulations of materials. It incorporates elements of bayesian inference, machine learning, differentiable programming,  software composability, and high-performance computing. This package is part of a software suite developed for the <a href="https://computing.mit.edu/cesmix/" rel="nofollow">CESMIX</a> project.</p>
