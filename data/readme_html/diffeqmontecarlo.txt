<p dir="auto">DiffEqMonteCarlo.jl is a component package in the DifferentialEquations ecosystem.
It holds methods for performing and analyzing Monte Carlo simulations.
Users interested in using this functionality should check out
<a href="https://github.com/JuliaDiffEq/DifferentialEquations.jl">DifferentialEquations.jl</a>.</p>
