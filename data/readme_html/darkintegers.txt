<p dir="auto"><code>DarkIntegers.jl</code> is a lightweight library for working with unsigned integers and polynomials. It includes:</p>
<ul dir="auto">
<li>Modulo arithmetic for primitive types;</li>
<li>Regular and modulo arithmetic for multi-limb (fixed size) unsigned integers;</li>
<li>Arithmetic for cyclic and negacyclic polynomials, including Karatsuba multiplication and multiplication via NTT</li>
</ul>
<p dir="auto">It is intended as a partial replacement for <a href="https://github.com/wbhart/Nemo.jl"><code>Nemo.jl</code></a>, which requires compilation of C libraries, and is often slower (in particular, because <code>DarkIntegers</code> does not need C calls and heap allocations for operations with multi-limb integers).</p>
<p dir="auto"><strong>Note:</strong> the library currently does not provide constant-time guarantees for arithmetic operations.</p>
