<p dir="auto">DocumenterCitations.jl uses <a href="https://github.com/Humans-of-Julia/Bibliography.jl">Bibliography.jl</a> to add support for BibTeX citations and references in documentation pages generated by <a href="https://github.com/JuliaDocs/Documenter.jl">Documenter.jl</a>.</p>
<p dir="auto">DocumenterCitations.jl is still in early development so please open issues if you encounter any bugs or pain points, would like
to see a new feature, or if you have any questions.</p>
