<p dir="auto">A fast computer helps to reduce the installation time, otherwise it works well even on dual core computers
with 4G RAM, even 2G RAM might be sufficient. OpenGL is a hard requirement. A dedicated graphics card is useful,
but not always required. If you have a Windows laptop please enable the dedicated graphics card in the settings.
The program was tested with a Logitech Extreme 3D pro Joystick, but most likely any Joystick will do. If not, please
create an issue on Github. The installation was tested on Windows 10 and Ubuntu 18.04 and 20.04, but should also work
on Mac.</p>

<p dir="auto">From the Julia prompt execute:</p>
<div class="highlight highlight-source-julia notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="using KiteSimulators

<div class="highlight highlight-source-julia notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="./bin/run_julia or bin\run_julia

<div class="highlight highlight-source-julia notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="edit(&quot;examples/joystick.jl&quot;)  # Linux

<p dir="auto">The x axis of the Joystick controls steering. By pressing the joystick forward you can depower the kite.</p>
<p dir="auto">Button 3 starts the reel-out, button 4 the reelin, and button 5 switches to parking mode. With button one you can start the simulation, with button two you can stop it.</p>
<p dir="auto">If you don't have a Joystick, try the following example:</p>
