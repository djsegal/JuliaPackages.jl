<p dir="auto">BVHFiles.jl is a package for working with BioVisionHierarchy files, a file format which stores motion capture data in a hierarchical structure.</p>
<p dir="auto">BVHFiles.jl uses <a href="https://github.com/JuliaGraphs/LightGraphs.jl">graphs</a> to represent and manipulate the data.</p>

<ul dir="auto">
<li>Load BVH files</li>
<li>Calculate global positions</li>
<li>Removal of joints</li>
<li>Adding joints</li>
<li>Removal of frames</li>
<li>Adding frames</li>
<li>Loss functions</li>
<li>Optimizing offsets</li>
<li>Optimizing rotations using gradient descent</li>
<li>Changing rotation sequences</li>
<li>Scaling</li>
<li>Transfer movements to different hierarchies</li>
<li>Interpolation between frames</li>
<li>Save BVH files</li>
</ul>
