<p dir="auto">Julia package containing utilities intended for Time Series analysis.</p>
<blockquote>
<p dir="auto"><g-emoji class="g-emoji" alias="warning" fallback-src="https://github.githubassets.com/images/icons/emoji/unicode/26a0.png">⚠️</g-emoji> This package is in an early development stage and its functionality has not been thoroughly tested. Please, consider to report issues if you find any.</p>
<p dir="auto"><g-emoji class="g-emoji" alias="warning" fallback-src="https://github.githubassets.com/images/icons/emoji/unicode/26a0.png">⚠️</g-emoji> This package is undergoing heavy refactoring and is in a feature freeze status.</p>
</blockquote>
<h2 dir="auto"><a id="user-content-featured-methods" class="anchor" aria-hidden="true" href="#featured-methods"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path d="m7.775 3.275 1.25-1.25a3.5 3.5 0 1 1 4.95 4.95l-2.5 2.5a3.5 3.5 0 0 1-4.95 0 .751.751 0 0 1 .018-1.042.751.751 0 0 1 1.042-.018 1.998 1.998 0 0 0 2.83 0l2.5-2.5a2.002 2.002 0 0 0-2.83-2.83l-1.25 1.25a.751.751 0 0 1-1.042-.018.751.751 0 0 1-.018-1.042Zm-4.69 9.64a1.998 1.998 0 0 0 2.83 0l1.25-1.25a.751.751 0 0 1 1.042.018.751.751 0 0 1 .018 1.042l-1.25 1.25a3.5 3.5 0 1 1-4.95-4.95l2.5-2.5a3.5 3.5 0 0 1 4.95 0 .751.751 0 0 1-.018 1.042.751.751 0 0 1-1.042.018 1.998 1.998 0 0 0-2.83 0l-2.5 2.5a1.998 1.998 0 0 0 0 2.83Z"></path></svg></a>Featured Methods</h2>
<ul dir="auto">
<li>Autocorrelated models for univariate and multivariate data (ar)</li>
<li>Autocorrelated model simulations for univariate and multivariate data (arsim)</li>
<li>Autocorrelated model forecasting with custom plots (forecast)</li>
<li>Autocorrelation &amp; Autocovariance function and custom plots (acf)</li>
<li>Boxcox Transformations and Inverse Boxcox Transformations (boxcox, iboxcox)</li>
<li>Crosscorrelation &amp; Crosscovariance function and custom plots (ccf)</li>
<li>Henderson Moving Average Filter (hma)</li>
<li>Lagged differences and Reverse lagged differences of a given order (d | p)</li>
<li>Locally Estimated Scatterplot Smoothing (loess)</li>
<li>Seasonal and Trend decomposition based on Loess and custom plot (stl)</li>
<li>Seasonal Plot for series and Time Series (splot)</li>
<li>Simple Moving Average (sma)</li>
</ul>

<ul dir="auto">
<li>Atmospheric Carbon Dioxide Dry Air Mole Fractions from quasi-continuous measurements at Mauna Loa, Hawaii.</li>
</ul>
<p dir="auto">K.W. Thoning, A.M. Crotwell, and J.W. Mund (2020), Atmospheric Carbon Dioxide Dry Air Mole Fractions from continuous measurements at Mauna Loa, Hawaii, Barrow, Alaska, American Samoa and South Pole. 1973-2019, Version 2020-08 National Oceanic and Atmospheric Administration (NOAA), Global Monitoring Laboratory (GML), Boulder, Colorado, USA <a href="https://doi.org/10.15138/yaf1-bk21" rel="nofollow">DOI</a></p>
<p dir="auto">FTP path: <code>ftp://aftp.cmdl.noaa.gov/data/greenhouse_gases/co2/in-situ/surface/</code></p>
<ul dir="auto">
<li>Airline Passengers</li>
</ul>
<p dir="auto">The classic Box &amp; Jenkins airline data. Monthly totals of international airline passengers from 1949 to 1960. Box, G. E. P., Jenkins, G. M. and Reinsel, G. C. (1976) <em>Time Series Analysis, Forecasting and Control.</em> Third Edition. Holden-Day. Series G.</p>
<ul dir="auto">
<li>London Crime and Weather</li>
</ul>
<p dir="auto">Ten years of monthly data about weather and crime in Greater London from 2008 to 2018. Data has been collected and joined from <a href="london.gov.uk">London Assembly</a> and <a href="metoffice.gov.uk">Met Office</a> (Heathrow Station).</p>
<ul dir="auto">
<li>Estimates of world seaborne trade from AIS data collected by MarineTraffic; available at UN COMTRADE Monitor.</li>
</ul>
<p dir="auto">Cerdeiro, Komaromi, Liu and Saeed (2020). Subset with imports and exports data for France, Germany and the United Kingdom from 2015-04-01 to 2021-05-02. <a href="https://comtrade.un.org/data/ais" rel="nofollow">UN Comtrade Database</a></p>
<ul dir="auto">
<li>Large Earthquakes</li>
</ul>
<p dir="auto">Number of earthquakes per year on earth with a magnitude higher or equal to six from 1950 to 2020.The data has been collected from the <a href="https://earthquake.usgs.gov/" rel="nofollow">U.S. Geological Survey
</a> and aggregated.</p>
