<p dir="auto">These IJulia notebooks provide the material used for the <a href="http://www.datagotham.com/tutorials/" rel="nofollow">Julia tutorial at DataGotham 2013</a>. To follow along, you'll need to install all of the software we use along the way:</p>
<p dir="auto"><em>Step 1</em>: Grab a binary of the latest Julia from <a href="http://julialang.org/downloads/#beta" rel="nofollow">http://julialang.org/downloads/#beta</a>. If the binaries do not work, please install from source. The initial source installation takes a long time because of the dependencies that have to be installed, but subsequent builds takes less than a minute on average.</p>
<p dir="auto"><em>Step 2</em>: For convenience, you can set up an alias to the newly installed Julia binary that you can use to invoke Julia from your normal shell prompt. On OS X, I use the following:</p>
<div class="snippet-clipboard-content notranslate position-relative overflow-auto" data-snippet-clipboard-copy-content="alias julia=&quot;/Applications/Julia-*.app/Contents/Resources/julia/bin/julia&quot;"><pre class="notranslate"><code>alias julia="/Applications/Julia-*.app/Contents/Resources/julia/bin/julia"
</code></pre></div>
<p dir="auto">I also setup an alias for IJulia to make it easier to launch:</p>
<div class="snippet-clipboard-content notranslate position-relative overflow-auto" data-snippet-clipboard-copy-content="alias ijulia=&quot;ipython notebook --profile=julia&quot;"><pre class="notranslate"><code>alias ijulia="ipython notebook --profile=julia"
</code></pre></div>
<p dir="auto"><em>Step 3</em>: To get all the packages we'll use, start a Julia REPL and then run the following commands in the Julia REPL:</p>
