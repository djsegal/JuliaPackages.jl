<p dir="auto">A basic API for getting Coinbase data into Julia.</p>
<p dir="auto">This package provides a Julia interface for the market data functions listed here: <a href="https://docs.pro.coinbase.com/" rel="nofollow">https://docs.pro.coinbase.com/</a></p>

<ul dir="auto">
<li><a href="https://docs.cloud.coinbase.com/exchange/reference/exchangerestapi_getproducts-1" rel="nofollow">products</a></li>
</ul>
<p dir="auto">Information on all currency pairs for trading or a specific currency pair.</p>
<div class="highlight highlight-source-julia notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="products()

<ul dir="auto">
<li><a href="https://docs.cloud.coinbase.com/exchange/reference/exchangerestapi_getproductbook-1" rel="nofollow">book</a></li>
</ul>
<p dir="auto">Get the orderbook for a product.</p>
<ul dir="auto">
<li>Level 1: Best Bid and Ask</li>
<li>Level 2: Full order book aggregated</li>
<li>Level 3: Full order book not aggregated.</li>
</ul>
<p dir="auto">Change the level with the 2nd argument to the function.</p>
<div class="highlight highlight-source-julia notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="#level 1
book(&quot;BTC-USD&quot;, 1)
#level 3

<span class="pl-c1">book</span>(<span class="pl-s"><span class="pl-pds">"</span>BTC-USD<span class="pl-pds">"</span></span>, <span class="pl-c1">1</span>)
<span class="pl-c"><span class="pl-c">#</span>level 3</span>
<span class="pl-c1">book</span>(<span class="pl-s"><span class="pl-pds">"</span>BTC-USD<span class="pl-pds">"</span></span>, <span class="pl-c1">3</span>)</pre></div>
<ul dir="auto">
<li><a href="https://docs.cloud.coinbase.com/exchange/reference/exchangerestapi_getproductticker-1" rel="nofollow">ticker</a>
Get the last trade, best bid/offer and 24 hour volume for a product.</li>
</ul>
<div class="highlight highlight-source-julia notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="ticker(&quot;BTC-USD&quot;)"><pre><span class="pl-c1">ticker</span>(<span class="pl-s"><span class="pl-pds">"</span>BTC-USD<span class="pl-pds">"</span></span>)</pre></div>
<ul dir="auto">
<li><a href="https://docs.cloud.coinbase.com/exchange/reference/exchangerestapi_getproducttrades-1" rel="nofollow">trades</a>
Get a list of trades of a product.
Paginated so will return both the data and a pagination cursor.
using <code>lasttrades</code> to get the last <code>n</code> trades of a product.</li>
</ul>
<div class="highlight highlight-source-julia notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="trades(&quot;BTC-USD&quot;)

<ul dir="auto">
<li><a href="https://docs.cloud.coinbase.com/exchange/reference/exchangerestapi_getproductcandles-1" rel="nofollow">candles</a>
Historical rates for a product in terms of open, high, low, and close (OHLC).<br>
1 minute, 5 minute, 15 minute, 1 hour, 6 hour and 1 day granularity available.</li>
</ul>
<div class="highlight highlight-source-julia notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="# 1 minute bars
