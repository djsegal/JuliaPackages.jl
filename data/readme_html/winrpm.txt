<p dir="auto">WinRPM is an installer for RPM packages provided by an RPM-md build system.
The default RPM-md provider is the <a href="https://build.opensuse.org/" rel="nofollow">OpenSUSE build service</a>,
which builds 32- and 64-bit DLLs for libraries used by
several Julia packages (note: builds are cross-compiled).</p>

<p dir="auto">RPM-md provides the following functions for general usage:
<code>update</code>, <code>whatprovides</code>, <code>search</code>, <code>lookup</code>, and <code>install</code></p>
<p dir="auto"><code>update()</code> -- download the new metadata from the hosts. Additional hosts can be added by editing the file <code>sources.list</code>.</p>
<p dir="auto"><code>whatprovides(file)</code> -- given a part of a filename or file-path, returns a list of packages that include</p>
<p dir="auto"><code>search(string)</code> -- search for a string in the package description, summary, or name fields and returns a list of matching packages</p>
<p dir="auto"><code>lookup(name)</code> -- search for a package by name</p>
<p dir="auto"><code>install(pkg)</code> -- install a package (by name or package identifier), including dependencies, into the <code>deps</code> folder</p>
<p dir="auto">The functions typically take a second parameter "arch" specifying the package architecture for search, defaulting to the current operating system.
It also offers the keyword argument <code>yes</code> which should be set to <code>true</code> if no prompt is desired.</p>
