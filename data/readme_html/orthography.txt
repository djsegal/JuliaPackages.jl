<p dir="auto">Work with rigorously defined scholarly editions of digital texts adhering to a specified orthography.  An orthography explicitly defines:</p>
<ul dir="auto">
<li>a set of token <em>types</em></li>
<li>tokens are composed from an explicitly defined character set or set of code points</li>
<li>the semantics of the character completely define how to tokenize text in the orthographic system. Each token is classified with one of the orthography's token types.</li>
</ul>
<p dir="auto">Documentation is available <a href="https://HCMID.github.io/Orthography.jl/stable/" rel="nofollow">here</a>.</p>
<p dir="auto">Orthography.jl was originally developed to support projects at the Holy Cross Manuscripts, Inscriptions and Documents Club (MID).</p>
