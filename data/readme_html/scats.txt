<p>This package allows you to construct correlograms and periodograms for a uniform time series.
If necessary, a time series can be generated, a correlogram can be weighted, and a periodogram can be smoothed.</p>
<p><strong>The package is designed exclusively for learning purposes and is not recommended for use in production.</strong></p>
