<p dir="auto">PianoFingering.jl is an automatic fingering generator for piano scores, written in the Julia language.</p>
<p dir="auto"><a target="_blank" rel="noopener noreferrer" href="./example.png"><img src="./example.png" alt="image" style="max-width: 100%;"></a></p>
<p dir="auto">There is no "best fingering", the fingerings generated by PianoFingering.jl are for reference only.</p>
<p dir="auto">THIS PACKAGE ONLY TESTED ON LINUX.</p>
