<p dir="auto">A small Julia package developed for a neuronal bifurcations minicourse mostly geared towards creating custom plots and running numerical simulations. Credit for the original course and Python code upon which this is based goes to <a href="https://github.com/nalewkoz">@nalewkoz</a>.</p>

<div class="snippet-clipboard-content notranslate position-relative overflow-auto" data-snippet-clipboard-copy-content="julia&gt; using Pkg

<p dir="auto">Build neuron or network models and study them in the phase plane or simulate their activity.</p>
<p dir="auto">Currently implemented models:</p>
<ul dir="auto">
<li>FitzHughNagumo</li>
<li>WilsonCowan</li>
<li>Simple Feedforward</li>
</ul>

<ol dir="auto">

<li>Create a pull request.</li>
</ol>
<p dir="auto">Alternatively see the GitHub <a href="https://help.github.com/en/github/collaborating-with-issues-and-pull-requests/creating-a-pull-request">documentation</a> on creating a pull request.</p>
