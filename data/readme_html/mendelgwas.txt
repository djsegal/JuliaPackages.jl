<p dir="auto">This <a href="http://julialang.org/" rel="nofollow">Julia</a> package performs standard Genome-Wide Association Studies. MendelGWAS is one component of the umbrella <a href="https://openmendel.github.io" rel="nofollow">OpenMendel</a> project.</p>

<p dir="auto"><em>Note: The three OpenMendel packages (1) <a href="https://openmendel.github.io/SnpArrays.jl/latest/" rel="nofollow">SnpArrays</a>, (2) <a href="https://openmendel.github.io/MendelSearch.jl" rel="nofollow">MendelSearch</a>, and (3) <a href="https://openmendel.github.io/MendelBase.jl" rel="nofollow">MendelBase</a> must be installed before any other OpenMendel package will run. It is easiest if these three packages are installed in the above order and before any other OpenMendel package.</em></p>
<p dir="auto">Within Julia, use the package manager to install MendelGWAS:</p>
<div class="snippet-clipboard-content notranslate position-relative overflow-auto" data-snippet-clipboard-copy-content="pkg&gt; add https://github.com/OpenMendel/MendelGWAS.jl"><pre class="notranslate"><code>pkg&gt; add https://github.com/OpenMendel/MendelGWAS.jl
</code></pre></div>
<p dir="auto">This package supports Julia v1.0+</p>
<h2 dir="auto"><a id="user-content-data-files" class="anchor" aria-hidden="true" href="#data-files"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path d="m7.775 3.275 1.25-1.25a3.5 3.5 0 1 1 4.95 4.95l-2.5 2.5a3.5 3.5 0 0 1-4.95 0 .751.751 0 0 1 .018-1.042.751.751 0 0 1 1.042-.018 1.998 1.998 0 0 0 2.83 0l2.5-2.5a2.002 2.002 0 0 0-2.83-2.83l-1.25 1.25a.751.751 0 0 1-1.042-.018.751.751 0 0 1-.018-1.042Zm-4.69 9.64a1.998 1.998 0 0 0 2.83 0l1.25-1.25a.751.751 0 0 1 1.042.018.751.751 0 0 1 .018 1.042l-1.25 1.25a3.5 3.5 0 1 1-4.95-4.95l2.5-2.5a3.5 3.5 0 0 1 4.95 0 .751.751 0 0 1-.018 1.042.751.751 0 0 1-1.042.018 1.998 1.998 0 0 0-2.83 0l-2.5 2.5a1.998 1.998 0 0 0 0 2.83Z"></path></svg></a>Data Files</h2>
<p dir="auto">To run this analysis package you will need to prepare a Control file and have your data files available. The Control file holds the names of your data files and any optional parameters for the analysis. Details on the general format and contents of the Control and data files can be found on the MendelBase <a href="https://openmendel.github.io/MendelBase.jl" rel="nofollow">documentation page</a>. Descriptions of the specific options available within the MendelGWAS analysis package are in its <a href="https://openmendel.github.io/MendelGWAS.jl" rel="nofollow">documentation page</a>.</p>
<p dir="auto">There are example data files in the "data" subfolder of each Mendel package.</p>
<h2 dir="auto"><a id="user-content-running-the-analysis" class="anchor" aria-hidden="true" href="#running-the-analysis"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path d="m7.775 3.275 1.25-1.25a3.5 3.5 0 1 1 4.95 4.95l-2.5 2.5a3.5 3.5 0 0 1-4.95 0 .751.751 0 0 1 .018-1.042.751.751 0 0 1 1.042-.018 1.998 1.998 0 0 0 2.83 0l2.5-2.5a2.002 2.002 0 0 0-2.83-2.83l-1.25 1.25a.751.751 0 0 1-1.042-.018.751.751 0 0 1-.018-1.042Zm-4.69 9.64a1.998 1.998 0 0 0 2.83 0l1.25-1.25a.751.751 0 0 1 1.042.018.751.751 0 0 1 .018 1.042l-1.25 1.25a3.5 3.5 0 1 1-4.95-4.95l2.5-2.5a3.5 3.5 0 0 1 4.95 0 .751.751 0 0 1-.018 1.042.751.751 0 0 1-1.042.018 1.998 1.998 0 0 0-2.83 0l-2.5 2.5a1.998 1.998 0 0 0 0 2.83Z"></path></svg></a>Running the Analysis</h2>
<p dir="auto">To run this analysis package, first launch Julia. Then load the package with the command:</p>
<div class="snippet-clipboard-content notranslate position-relative overflow-auto" data-snippet-clipboard-copy-content=" julia&gt; using MendelGWAS"><pre class="notranslate"><code> julia&gt; using MendelGWAS
</code></pre></div>
<p dir="auto">Next, if necessary, change to the directory containing your files, for example,</p>
<div class="snippet-clipboard-content notranslate position-relative overflow-auto" data-snippet-clipboard-copy-content=" julia&gt; cd(&quot;~/path/to/data/files/&quot;)"><pre class="notranslate"><code> julia&gt; cd("~/path/to/data/files/")
</code></pre></div>
<p dir="auto">Finally, to run the analysis using the parameters in the control file Control_file.txt use the command:</p>
<div class="snippet-clipboard-content notranslate position-relative overflow-auto" data-snippet-clipboard-copy-content=" julia&gt; GWAS(&quot;Control_file.txt&quot;)"><pre class="notranslate"><code> julia&gt; GWAS("Control_file.txt")
</code></pre></div>
<p dir="auto"><em>Note: The package is called</em> MendelGWAS <em>but the analysis function is called simply</em> GWAS.</p>
