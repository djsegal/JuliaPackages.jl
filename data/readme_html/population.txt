<p dir="auto"><a target="_blank" rel="noopener noreferrer" href="docs/figures/logo_sPop2.jpg"><img src="docs/figures/logo_sPop2.jpg" alt="logo" title="Climate impacts on vector-borne diseases" style="max-width: 100%;"></a></p>
<p dir="auto">This is the standalone Julia library of the dynamically-structured matrix population model sPop2. This version implements both age-dependent and accumulative processes.</p>

<p dir="auto">The following creates a pseudo-structured population with 10 individuals and iterates it one step with 0 mortality and an Erlang-distributed development time of 20±5 steps.</p>
<div class="highlight highlight-source-julia notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="    pop = sPop2(PopDataSto())
    AddProcess(pop, AccErlang())
    AddPop(pop, 10)
    pr = (devmn=20.0, devsd=5.0)
    size, completed, poptabledone = StepPop(pop, pr)"><pre>    pop <span class="pl-k">=</span> <span class="pl-c1">sPop2</span>(<span class="pl-c1">PopDataSto</span>())
    <span class="pl-c1">AddProcess</span>(pop, <span class="pl-c1">AccErlang</span>())
    <span class="pl-c1">AddPop</span>(pop, <span class="pl-c1">10</span>)
    pr <span class="pl-k">=</span> (devmn<span class="pl-k">=</span><span class="pl-c1">20.0</span>, devsd<span class="pl-k">=</span><span class="pl-c1">5.0</span>)
    size, completed, poptabledone <span class="pl-k">=</span> <span class="pl-c1">StepPop</span>(pop, pr)</pre></div>
