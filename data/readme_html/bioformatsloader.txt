<p>Requires <code>bioformats_package.jar</code></p>
<div class="highlight highlight-source-julia"><pre>Pkg<span class="pl-k">.</span><span class="pl-c1">clone</span>(<span class="pl-s"><span class="pl-pds">"</span>https://github.com/ahnlabb/BioformatsLoader.jl<span class="pl-pds">"</span></span>)</pre></div>
<div class="highlight highlight-source-julia"><pre><span class="pl-k">using</span> BioformatsLoader
<span class="pl-k">using</span> JavaCall

bfpkg_path <span class="pl-k">=</span> <span class="pl-s"><span class="pl-pds">"</span>/path/to/bioformats_package.jar<span class="pl-pds">"</span></span>

<span class="pl-k">try</span>
        JavaCall<span class="pl-k">.</span><span class="pl-c1">init</span>([<span class="pl-s"><span class="pl-pds">"</span>-ea<span class="pl-pds">"</span></span>, <span class="pl-s"><span class="pl-pds">"</span>-Xmx1024M<span class="pl-pds">"</span></span>, <span class="pl-s"><span class="pl-pds">"</span>-Djava.class.path=<span class="pl-v">$bfpkg_path</span><span class="pl-pds">"</span></span>])
<span class="pl-k">end</span></pre></div>
<p>To import the file <code>file.msr</code> you then do</p>
<div class="highlight highlight-source-julia"><pre>image <span class="pl-k">=</span> <span class="pl-c1">bf_import</span>(<span class="pl-s"><span class="pl-pds">"</span>file.msr<span class="pl-pds">"</span></span>)</pre></div>
<p>The variable <code>image</code> will contain an array of ImageMetadata, the data will be the type that the format specifies: <strong>Int8</strong>, <strong>UInt8</strong>, <strong>Int16</strong>, <strong>UInt16</strong>, <strong>Int32</strong>, <strong>UInt32</strong>, <strong>Float32</strong>, <strong>Float64</strong> or <strong>Bool</strong>.</p>
