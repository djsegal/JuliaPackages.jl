<p dir="auto">LITS v0.4 is the last release for this package and will be deprecated. The modeling capabilities and new features will be added
in <a href="https://github.com/NREL-SIIP/PowerSimulationsDynamics.jl">PowerSimulationsDynamics</a> to provide long-term support
under <a href="https://github.com/NREL-SIIP">NREL's SIIP Iniciative</a></p>
<p dir="auto">Old examples up to version v0.4 can be found in the <a href="https://github.com/Energy-MAC/LITS-Examples">LITS-Examples</a>  repository. Newer versions using <a href="https://github.com/NREL-SIIP/PowerSimulationsDynamics.jl">PowerSimulationsDynamics</a> can be found in <a href="https://github.com/NREL-SIIP/SIIPExamples.jl">SIIP-Examples</a></p>
<p dir="auto">The pre-print paper can be found in our <a href="https://arxiv.org/abs/2003.02957" rel="nofollow">arXiv preprint</a>.</p>
