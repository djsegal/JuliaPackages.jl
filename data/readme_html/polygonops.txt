<p>The objective of this package is to provide a set of generic polygon operations. There are two assumptions:</p>
<ul>
<li>the container and points are both vector-like.</li>
<li>the first and last elements are equal.</li>
</ul>
<p>This should provide generic implementations portable and extensible for packages such as GeometryTypes, GeometryBasics, and JuliaGeo.</p>
<p>Please see the <a href="https://juliageometry.github.io/PolygonOps.jl/stable" rel="nofollow">docs</a> for a list of exported functions.</p>
<p>Implements:</p>
<ul>
<li>point-in-polygon</li>
<li>area</li>
<li>centroid</li>
</ul>
<p>Planned:</p>
<ul>
<li>simplification</li>
<li>booleans (ported from PolygonClipping.jl)</li>
<li>offset/insetting</li>
</ul>
