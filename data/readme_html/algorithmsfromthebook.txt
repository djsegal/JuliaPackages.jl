<p dir="auto">AlgorithmsFromTheBook.jl collects the Julia code in Kenneth Lange's book <em>Algorithms from THE BOOK</em>. To access the functions in the book, simply install the package in Julia v1.0 or later</p>
<div class="highlight highlight-source-julia notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="using Pkg   
Pkg.clone(&quot;https://github.com/KennethLange/AlgorithmsFromTheBook.jl.git&quot;)"><pre><span class="pl-k">using</span> Pkg   
Pkg<span class="pl-k">.</span><span class="pl-c1">clone</span>(<span class="pl-s"><span class="pl-pds">"</span>https://github.com/KennethLange/AlgorithmsFromTheBook.jl.git<span class="pl-pds">"</span></span>)</pre></div>
<p dir="auto">For example, to test the Euclid algorithm in Chapter 1,</p>
<div class="highlight highlight-source-julia notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="using AlgorithmsFromTheBook  
gcd = euclid(600, 220)"><pre><span class="pl-k">using</span> AlgorithmsFromTheBook  
gcd <span class="pl-k">=</span> <span class="pl-c1">euclid</span>(<span class="pl-c1">600</span>, <span class="pl-c1">220</span>)</pre></div>
<p dir="auto">To read the source code of a function,</p>
<div class="highlight highlight-source-julia notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="@edit euclid(600, 220)"><pre><span class="pl-c1">@edit</span> <span class="pl-c1">euclid</span>(<span class="pl-c1">600</span>, <span class="pl-c1">220</span>)</pre></div>
