<p dir="auto">GeometricIntegrators.jl is a library of geometric integrators for ordinary differential equations, stochastic differential equations and differential algebraic equations in Julia. Its main aim is the democratization and proliferation of geometric integrators, providing a comprehensive collection of standard and structure-preserving algorithms under a unified interface. Furthermore it serves as testbed for the implementation and verification of novel geometric integrators, in particular their analysis with respect to long-time stability and conservation of geometric structures.
GeometricIntegrators.jl can be used either interactively or as computational core in other codes. It provides both, a high-level interface that requires only very few lines of code to solve an actual problem, and a lean low-level interface that allows for straightforward integration into application codes via the exchange of very small data structures.
Due to the modular structure and the use of the multiple dispatch paradigm, the library can easily be extended, e.g., towards new algorithms or new types of equations. GeometricIntegrators.jl is designed to minimize overhead and maximize performance in order to be able to perform simulations with millions or even billions of time steps to facilitate the study of the long-time behaviour of both numerical algorithms and dynamical systems.</p>

<p dir="auto">In the simplest cases, the use of <em>GeometricIntegrators.jl</em> requires the
construction of two objects, an equation and an integrator. For many standard
methods, the integrator can be implicitly selected by specifying an equation
and a tableau.</p>
<p dir="auto">Before any use, we need to load <code>GeometricIntegrators</code>,</p>
<div class="highlight highlight-source-julia notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="using GeometricIntegrators"><pre><span class="pl-k">using</span> GeometricIntegrators</pre></div>
<p dir="auto">Then we can create an <code>ODE</code> object for the equation ẋ(t) = x(t) with initial condition x(0)=1 by</p>
<div class="highlight highlight-source-julia notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="ode = ODE((t, x, ẋ) -&gt; ẋ[1] = x[1], [1.0]);"><pre>ode <span class="pl-k">=</span> <span class="pl-c1">ODE</span>((t, x, ẋ) <span class="pl-k">-&gt;</span> ẋ[<span class="pl-c1">1</span>] <span class="pl-k">=</span> x[<span class="pl-c1">1</span>], [<span class="pl-c1">1.0</span>]);</pre></div>
<p dir="auto">An integrator for this ODE, using the tableau for the explicit Euler method and a time step of Δt=0.1, is obtained by</p>
<div class="highlight highlight-source-julia notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="int = Integrator(ode, TableauExplicitEuler(), 0.1);"><pre>int <span class="pl-k">=</span> <span class="pl-c1">Integrator</span>(ode, <span class="pl-c1">TableauExplicitEuler</span>(), <span class="pl-c1">0.1</span>);</pre></div>
<p dir="auto">With that, the solution for nₜ=10 time steps is simply computed by</p>
<div class="highlight highlight-source-julia notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="sol = integrate(ode, int, 10);"><pre>sol <span class="pl-k">=</span> <span class="pl-c1">integrate</span>(ode, int, <span class="pl-c1">10</span>);</pre></div>
<p dir="auto">which returns an object holding the solution for all time steps.
With the help of the <em><a href="https://github.com/JuliaPlots/Plots.jl">Plots.jl</a></em> package we can visualise the result and compare with the exact solution:</p>
<div class="highlight highlight-source-julia notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="using Plots
plot(xlims=[0,1], xlab=&quot;t&quot;, ylab=&quot;x(t)&quot;, legend=:bottomright)

<span class="pl-c1">plot</span>(xlims<span class="pl-k">=</span>[<span class="pl-c1">0</span>,<span class="pl-c1">1</span>], xlab<span class="pl-k">=</span><span class="pl-s"><span class="pl-pds">"</span>t<span class="pl-pds">"</span></span>, ylab<span class="pl-k">=</span><span class="pl-s"><span class="pl-pds">"</span>x(t)<span class="pl-pds">"</span></span>, legend<span class="pl-k">=</span><span class="pl-c1">:bottomright</span>)
