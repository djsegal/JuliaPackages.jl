<p>This implements basic Unum and Ubound arithmetic, based on John Gustafson's <a href="https://www.crcpress.com/The-End-of-Error-Unum-Computing/Gustafson/9781482239867" rel="nofollow"><em>End of Error</em></a>.</p>
<p>See the <a href="https://github.com/JuliaComputing/Unums.jl/tree/master/examples">examples</a> directory for code samples.</p>
