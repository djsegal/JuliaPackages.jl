<p dir="auto">Julia package <strong><a href="https://github.com/sungsik-kong/PhyNEST.jl">PhyNEST.jl</a>: Phylogenetic Network Estimation using SiTe patterns</strong> can:</p>
<ul dir="auto">
<li>Read multi-locus sequence alignment (in a relaxed PHYLIP format) and parse site pattern frequencies for all permutations of four individuals (i.e., sequences).</li>
<li>Estimate maximum composite likelihood species network using the parsed site pattern frequencies using hill climbing or simulated annealing heuristic algorithms.</li>
</ul>

<p dir="auto">To use <a href="https://github.com/sungsik-kong/PhyNEST.jl">PhyNEST.jl</a>, Julia &gt;= v1.7 is recommended. This package was developed in Julia 1.7.2, and has been tested for Julia &gt;= v1.7 in OSX distributions.</p>
<p dir="auto">To install <a href="https://github.com/sungsik-kong/PhyNEST.jl">PhyNEST.jl</a> run the following command in Julia prompt:</p>
<div class="snippet-clipboard-content notranslate position-relative overflow-auto" data-snippet-clipboard-copy-content="using Pkg
Pkg.add(&quot;PhyNEST&quot;)"><pre lang="@julia" class="notranslate"><code>using Pkg
Pkg.add("PhyNEST")
</code></pre></div>
<p dir="auto">or</p>
<div class="snippet-clipboard-content notranslate position-relative overflow-auto" data-snippet-clipboard-copy-content="] add PhyNEST"><pre lang="@julia" class="notranslate"><code>] add PhyNEST
</code></pre></div>
<p dir="auto">Either command installs the package. To check the current version of installed <a href="https://github.com/sungsik-kong/PhyNEST.jl">PhyNEST.jl</a>, run the following command:</p>
<div class="snippet-clipboard-content notranslate position-relative overflow-auto" data-snippet-clipboard-copy-content="] status PhyNEST"><pre lang="@julia" class="notranslate"><code>] status PhyNEST
</code></pre></div>
