<h2 dir="auto"><a id="user-content-features" class="anchor" aria-hidden="true" href="#features"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path d="m7.775 3.275 1.25-1.25a3.5 3.5 0 1 1 4.95 4.95l-2.5 2.5a3.5 3.5 0 0 1-4.95 0 .751.751 0 0 1 .018-1.042.751.751 0 0 1 1.042-.018 1.998 1.998 0 0 0 2.83 0l2.5-2.5a2.002 2.002 0 0 0-2.83-2.83l-1.25 1.25a.751.751 0 0 1-1.042-.018.751.751 0 0 1-.018-1.042Zm-4.69 9.64a1.998 1.998 0 0 0 2.83 0l1.25-1.25a.751.751 0 0 1 1.042.018.751.751 0 0 1 .018 1.042l-1.25 1.25a3.5 3.5 0 1 1-4.95-4.95l2.5-2.5a3.5 3.5 0 0 1 4.95 0 .751.751 0 0 1-.018 1.042.751.751 0 0 1-1.042.018 1.998 1.998 0 0 0-2.83 0l-2.5 2.5a1.998 1.998 0 0 0 0 2.83Z"></path></svg></a>Features</h2>
<ul dir="auto">
<li><strong>Rich!</strong> Mamdani and Sugeno inference systems, both Type-1 and Type-2, several <a href="https://lucaferranti.github.io/FuzzyLogic.jl/stable/api/memberships" rel="nofollow">membership functions</a> and <a href="https://lucaferranti.github.io/FuzzyLogic.jl/stable/api/fis" rel="nofollow">algoritms options</a> available.</li>
<li><strong>Compatible!</strong> Read your models from <a href="https://ffll.sourceforge.net/fcl.htm" rel="nofollow">IEC 61131-7 Fuzzy Control Language</a>, <a href="https://en.wikipedia.org/wiki/Fuzzy_markup_language" rel="nofollow">IEEE 1855-2016 Fuzzy Markup Language</a> and Matlab Fuzzy toolbox <code>.fis</code> files.</li>
<li><strong>Expressive!</strong> Clear Domain Specific Language to write your model as human readable Julia code</li>
<li><strong>Productive!</strong> Several visualization tools to help debug and tune your model.</li>
<li><strong>Portable!</strong> Compile your final model to Julia code.</li>
</ul>
