<p dir="auto"><em>Julia wrapper for the 2D airfoil panel code XFOIL</em></p>
<p dir="auto">Author: Taylor McDonnell</p>
<p dir="auto"><strong>Xfoil.jl</strong> is a Julia wrapper for a Mark Drela's 2D airfoil panel code, <a href="https://web.mit.edu/drela/Public/web/xfoil/" rel="nofollow">XFOIL</a>, with the GUI features removed.</p>
<p dir="auto">This package was originally a fork of <a href="https://github.com/mdolab/pyXLIGHT">pyXLIGHT</a> (which has since been renamed to <a href="https://github.com/mdolab/CMPLXFOIL">CMPLXFOIL</a>), but has since undergone some development to introduce new features.  The version of XFOIL that this package wraps is that of the <a href="https://github.com/byuflowlab/xfoil_light">xfoil_light</a> project.</p>
<h2 dir="auto"><a id="user-content-package-features" class="anchor" aria-hidden="true" href="#package-features"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path d="m7.775 3.275 1.25-1.25a3.5 3.5 0 1 1 4.95 4.95l-2.5 2.5a3.5 3.5 0 0 1-4.95 0 .751.751 0 0 1 .018-1.042.751.751 0 0 1 1.042-.018 1.998 1.998 0 0 0 2.83 0l2.5-2.5a2.002 2.002 0 0 0-2.83-2.83l-1.25 1.25a.751.751 0 0 1-1.042-.018.751.751 0 0 1-.018-1.042Zm-4.69 9.64a1.998 1.998 0 0 0 2.83 0l1.25-1.25a.751.751 0 0 1 1.042.018.751.751 0 0 1 .018 1.042l-1.25 1.25a3.5 3.5 0 1 1-4.95-4.95l2.5-2.5a3.5 3.5 0 0 1 4.95 0 .751.751 0 0 1-.018 1.042.751.751 0 0 1-1.042.018 1.998 1.998 0 0 0-2.83 0l-2.5 2.5a1.998 1.998 0 0 0 0 2.83Z"></path></svg></a>Package Features</h2>
<ul dir="auto">
<li>Inputs airfoils and flow parameters directly into XFOIL, rather than through files.</li>
<li>Redistributes panels over the surface of an airfoil using XFOIL's PANE command.</li>
<li>Computes inviscid and viscous airfoil coefficients using XFOIL.</li>
<li>Can return pressure coefficients computed by XFOIL for inviscid and viscous analyses.</li>
<li>Can return boundary layer parameters computed by XFOIL for viscous analyses.</li>
<li>Provides full access to the internal state variables of XFOIL.</li>
<li>Provides a complex step enabled version of XFOIL for sensitivity analysis.</li>
</ul>

<p dir="auto">This code actually wraps two versions of Mark Drela's XFOIL code, one of which may be used to compute sensitivities using the complex step method.  The complex step enabled version of XFOIL may be called by appending <code>_cs</code> to the end of each functions name.  Note that there is no interaction between the two versions of XFOIL wrapped by this package, so if you wish to use the complex step version of the code you must use the functions that end with <code>_cs</code>.</p>
