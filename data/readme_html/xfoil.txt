<p><em>Julia Wrapper for a Simplified Version of XFOIL</em></p>
<p>Author: Taylor McDonnell</p>
<p><strong>Xfoil.jl</strong> is a Julia wrapper for a simplified version of Mark Drela's 2D airfoil panel code, <a href="https://web.mit.edu/drela/Public/web/xfoil/" rel="nofollow">XFOIL</a>.</p>
<p>This package was originally a fork of the <a href="https://github.com/mdolab/pyXLIGHT">pyXLIGHT</a>, but has since undergone some development to introduce new features.  The version of XFOIL that this package wraps is that of the <a href="https://github.com/byuflowlab/xfoil_light">xfoil_light</a> project.</p>
<h2><a id="user-content-package-features" class="anchor" aria-hidden="true" href="#package-features"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path fill-rule="evenodd" d="M7.775 3.275a.75.75 0 001.06 1.06l1.25-1.25a2 2 0 112.83 2.83l-2.5 2.5a2 2 0 01-2.83 0 .75.75 0 00-1.06 1.06 3.5 3.5 0 004.95 0l2.5-2.5a3.5 3.5 0 00-4.95-4.95l-1.25 1.25zm-4.69 9.64a2 2 0 010-2.83l2.5-2.5a2 2 0 012.83 0 .75.75 0 001.06-1.06 3.5 3.5 0 00-4.95 0l-2.5 2.5a3.5 3.5 0 004.95 4.95l1.25-1.25a.75.75 0 00-1.06-1.06l-1.25 1.25a2 2 0 01-2.83 0z"></path></svg></a>Package Features</h2>
<ul>
<li>Can handle any airfoil XFOIL can handle</li>
<li>Is able to redistribute panels over the surface of an airfoil using XFOIL's PANE command.</li>
<li>Able to perform viscous airfoil analysis using XFOIL.</li>
<li>Complex step method can be used to obtain gradients.</li>
</ul>
<h2><a id="user-content-obtaining-gradients-using-the-complex-step-method" class="anchor" aria-hidden="true" href="#obtaining-gradients-using-the-complex-step-method"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path fill-rule="evenodd" d="M7.775 3.275a.75.75 0 001.06 1.06l1.25-1.25a2 2 0 112.83 2.83l-2.5 2.5a2 2 0 01-2.83 0 .75.75 0 00-1.06 1.06 3.5 3.5 0 004.95 0l2.5-2.5a3.5 3.5 0 00-4.95-4.95l-1.25 1.25zm-4.69 9.64a2 2 0 010-2.83l2.5-2.5a2 2 0 012.83 0 .75.75 0 001.06-1.06 3.5 3.5 0 00-4.95 0l-2.5 2.5a3.5 3.5 0 004.95 4.95l1.25-1.25a.75.75 0 00-1.06-1.06l-1.25 1.25a2 2 0 01-2.83 0z"></path></svg></a>Obtaining Gradients using the Complex Step Method</h2>
<p>This code actually wraps two versions of Mark Drela's XFOIL code, one of which may be used to compute gradients using the complex step method.  The complex step enabled version of XFOIL may be called through using functions that end with <code>_cs</code>.  Note that there is no interaction between the two versions of XFOIL wrapped by this package, so if you wish to use the complex step version of the code you must use the functions that end with <code>_cs</code>.</p>
