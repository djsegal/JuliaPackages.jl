<p>This package provides AD-based model implementations that conform to the <a href="https://github.com/JuliaSmoothOptimizers/ADNLPModels.jl">NLPModels</a> API. The following packages are supported:</p>
<ul>
<li><code>ForwardDiff.jl</code>: default choice.</li>
<li><code>Zygote.jl</code>: you must load <code>Zygote.jl</code> separately and pass <code>ADNLPModels.ZygoteAD()</code> as the <code>adbackend</code> keyword argument to the <code>ADNLPModel</code> or <code>ADNLSModel</code> constructor.</li>
<li><code>ReverseDiff.jl</code>: you must load <code>ReverseDiff.jl</code> separately and pass <code>ADNLPModels.ReverseDiffAD()</code> as the <code>adbackend</code> keyword argument to the <code>ADNLPModel</code> or <code>ADNLSModel</code> constructor.</li>
</ul>
