<p>Solver for coupled nonlinear partial differential equations based on the Voronoi finite volume method.</p>
<p>This Julia package merges the ideas behind <a href="http://www.wias-berlin.de/software/pdelib/?lang=0" rel="nofollow">pdelib</a>/fvsys with the multiple dispatch paradigm of Julia. It allows to avoid the implementation of function derivatives.  It instead uses <a href="https://github.com/JuliaDiff/ForwardDiff.jl">ForwardDiff</a> and <a href="https://github.com/JuliaDiff/DiffResults.jl">DiffResults</a> to evaluate user functions along with their jacobians.</p>
<p>It requires Julia 1.x</p>
