<p>Bindings for the <a href="https://www.libsdl.org/" rel="nofollow">Simple DirectMedia Layer</a> library. The bindings were generated using <a href="https://github.com/ihnorton/Clang.jl">Clang.jl</a>.</p>
<p>It's recommended to declare a shorthand for the package after using it:</p>
<pre><code>using SimpleDirectMediaLayer
const SDL = SimpleDirectMediaLayer
</code></pre>
<p>The <code>SDL_</code> prefix was trimmed from names so functions and types can be accessed as <code>SDL.foo</code> instead of <code>SDL_foo</code>. Documentation can be found on the <a href="https://wiki.libsdl.org/FrontPage" rel="nofollow">SDL wiki</a>.</p>
<p>A few examples are available <a href="https://github.com/jonathanBieler/SimpleDirectMediaLayer.jl/blob/master/src/examples/">here</a>.</p>
