<p dir="auto">Parses a GraphQL executable document (that is, a query string) and partially validates it. Follows the <a href="https://spec.graphql.org/October2021" rel="nofollow">2021 specification</a>.</p>
<p dir="auto">Why only partial validation? Full validation (as per the GraphQL specification) requies knowledge of the schema, and GraphQLParser assumes no knowledge of the server and will therefore only perform some validation.</p>
<p dir="auto">For example, the validation provided by this package will fail if parsing fields, or if two variable definitions use the same name, but will not fail if a field is incorrectly named for a particularly query.
For more information about what is covered, see <a href="https://mmiller-max.github.io/GraphQLParser.jl/stable/#Validation" rel="nofollow">the documentation</a>.</p>
