<p dir="auto"><a href="https://github.com/nivupai/GrayCoding.jl">GrayCoding</a> is a formal Linear Algebraic framework for <code>q</code>-ry Gray Code.
Encoding and Decooding of Gray codes can be treated as a special case of algebraic block coding.</p>
<ul dir="auto">
<li>Encoding: <code>g=G*b </code></li>
<li>Decoding: <code>b=B*g </code></li>
</ul>
