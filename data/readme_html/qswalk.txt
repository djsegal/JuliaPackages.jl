<p dir="auto"><code>QSWalk</code> provides a package for <a href="https://julialang.org/" rel="nofollow">Julia programming language</a> which enables high-performance analysis of quantum stochastic walks. There are two main advantages of the presented packages over the existing software. First, it can be use to describe quantum stochastic walks in the local, as well as global regime. Second, it enables the user to seamlessly utilize parallel computing capabilities.</p>
