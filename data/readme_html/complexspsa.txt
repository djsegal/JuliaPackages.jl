<p dir="auto">This repository holds a <a href="https://julialang.org" rel="nofollow">Julia</a> source code implementation for the CSPSA optimization method for real-valued functions of complex variables.</p>
<p dir="auto">The CSPSA method and its variants correspond to complex generalizations of the <a href="https://www.jhuapl.edu/spsa/" rel="nofollow">SPSA</a> method.</p>
<p dir="auto">For more information, visit the <a href="https://jgidi.github.io/ComplexSPSA.jl/" rel="nofollow">official documentation</a>.</p>
