<h1><a id="user-content-nkftool-julia-package-for-converting-encodings-in-japanese-texts-using-nkf" class="anchor" aria-hidden="true" href="#nkftool-julia-package-for-converting-encodings-in-japanese-texts-using-nkf"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path fill-rule="evenodd" d="M7.775 3.275a.75.75 0 001.06 1.06l1.25-1.25a2 2 0 112.83 2.83l-2.5 2.5a2 2 0 01-2.83 0 .75.75 0 00-1.06 1.06 3.5 3.5 0 004.95 0l2.5-2.5a3.5 3.5 0 00-4.95-4.95l-1.25 1.25zm-4.69 9.64a2 2 0 010-2.83l2.5-2.5a2 2 0 012.83 0 .75.75 0 001.06-1.06 3.5 3.5 0 00-4.95 0l-2.5 2.5a3.5 3.5 0 004.95 4.95l1.25-1.25a.75.75 0 00-1.06-1.06l-1.25 1.25a2 2 0 01-2.83 0z"></path></svg></a>NKFtool: Julia package for converting encodings in Japanese texts using nkf</h1>
<p>NKFtool is a <a href="https://julialang.org" rel="nofollow">Julia</a> package for guessing and converting
encodings in Japanese text.  This is the wrapper module
to Network kanji (Japanese characters) filter,
aka <a href="https://osdn.net/projects/nkf/" rel="nofollow">nkf</a>,
and provides the interface to use nkf from Julia.</p>

<p>NKFtool requires nkf to be installed in your system. Major Unix-like operating systems offer its precompiled version in their package management systems.</p>
<p>NKFtool also requires Julia v1.0 or above.</p>
<p>To install NKFtool using Julia's packaging system, enter Julia's package manager prompt with <code>]</code>, and run</p>
<pre><code>(v1.1) pkg&gt; add NKFtool
</code></pre>
