<p>A minimal package that only defines <code>PersistenceDiagram</code>s.</p>
<p>If you want to create a package that needs access to persistence diagrams, you can make it
depend on this package.</p>
<p>To plot persistence diagrams, or do analysis on them, see
<a href="https://github.com/mtsch/PersistenceDiagrams.jl">PersistenceDiagrams.jl</a>. The package is
documented <a href="https://mtsch.github.io/PersistenceDiagrams.jl/dev/" rel="nofollow">here</a>.</p>
