<p>The package provides training and visualisation functions
for Kohonen's self-organising maps for Julia.
Training functions are implemented in pure Julia, without calling
binary libraries.</p>

<p>Because SOM.jl is a registered package, it can be installed for
Julia v0.7 or later with the package manager as</p>
<div class="highlight highlight-source-julia"><pre>pkg<span class="pl-k">&gt;</span> add SOM</pre></div>
<p>The current version 0.4.0 is tested with Julia v1.0.3 and 1.2.0 (use version 0.2.0 with Julia v0.7 and v1.0 because of changes in the PyPlot (i.e. matplotlib) package).</p>
<p>Please refer to the <a href="https://lisci-lab.github.io/SOM.jl/stable" rel="nofollow">documentation</a> for details!</p>
