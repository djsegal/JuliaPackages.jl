<p>This is an implementation of the <a href="https://en.wikipedia.org/wiki/Remez_algorithm" rel="nofollow">Remez algorithm</a> for computing minimax polynomial approximations to functions.</p>
<p>It is largely based on <a href="https://github.com/ARM-software/optimized-routines/blob/da55ef9510a53822b5706c61ad97795828999c80/auxiliary/remez.jl">code by ARM</a>, but updated for newer Julia versions and built into a package.</p>
<p>The main function is <code>ratfn_minimax</code>, see help for more details.</p>
