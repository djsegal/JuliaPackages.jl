<p>Julia wrapper for command line Git</p>
<p>This package provides Julia wrappers for some common Git operations,
as used by the Julia package manager in versions 0.4 and earlier.</p>
<p>If you do not already have <code>git</code> installed and on your system <code>PATH</code>, then
adding this package (or running <code>Pkg.build("Git")</code> will download a local binary
copy of command-line git if you are using Windows, Mac OS X via
<a href="https://github.com/JuliaLang/Homebrew.jl">Homebrew.jl</a>, or Linux on x86/amd64 architectures.</p>
