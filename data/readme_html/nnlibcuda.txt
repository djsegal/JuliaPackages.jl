<p dir="auto">This is a glue package which extends functions from <a href="https://github.com/FluxML/NNlib.jl">NNlib.jl</a> to work with <a href="https://github.com/JuliaGPU/CUDA.jl">CUDA.jl</a>. It should be loaded automatically when using <a href="https://github.com/FluxML/Flux.jl">Flux.jl</a>, but not when using NNlib.jl by itself.</p>
<p dir="auto">Julia gpu kernels are in <code>src/</code>, while wrappers around <code>cudnn</code> are in <code>src/cudnn/</code>.</p>
