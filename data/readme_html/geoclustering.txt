<p dir="auto">Simple Linear Iterative Clustering (SLIC) produces clusters
that are spatially connected based on a geospatial distance <code>dₛ</code>.
The samples in these clusters are similar in terms of their features
according to a distance <code>dᵥ</code>. The tradeoff is controlled with a
hyperparameter parameter <code>m</code> in an additive model <code>dₜ = √(dᵥ² + m²(dₛ/s)²)</code>.
The original method developed for images is described in
<a href="https://ieeexplore.ieee.org/document/6205760" rel="nofollow">Achanta et al. 2011</a>.
It has been generalized in this package for any geospatial data set
(e.g. point sets).</p>

<p dir="auto">Geostatistical Hierarchical Clustering (GHC) produces clusters
based on (cross-)variograms between covariates and on a kernel
function between geospatial coordinates. The method is described
in <a href="https://www.sciencedirect.com/science/article/abs/pii/S2211675316300367" rel="nofollow">Fouedjio, F. 2016</a>.</p>

<p dir="auto">Geostatistical Spectral Clustering (GSC) produces clusters
based on the spectral decomposition of the graph Laplacian
constructed with weights that are a function of features and
locations. The method is described in
<a href="https://www.sciencedirect.com/science/article/pii/S0098300415001314" rel="nofollow">Romary et al. 2015</a>.</p>
