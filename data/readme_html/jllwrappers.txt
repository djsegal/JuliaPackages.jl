<p>This package contains an array of macros that enable <a href="https://juliapackaging.github.io/BinaryBuilder.jl/stable/jll/" rel="nofollow">JLL packages</a> to generate themselves.  It is not intended to be used by users, but rather is used in autogenerated packages via <a href="https://github.com/JuliaPackaging/BinaryBuilder.jl"><code>BinaryBuilder.jl</code></a>.</p>
