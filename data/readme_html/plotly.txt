<h2><a id="user-content-usage" class="anchor" aria-hidden="true" href="#usage"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path fill-rule="evenodd" d="M7.775 3.275a.75.75 0 001.06 1.06l1.25-1.25a2 2 0 112.83 2.83l-2.5 2.5a2 2 0 01-2.83 0 .75.75 0 00-1.06 1.06 3.5 3.5 0 004.95 0l2.5-2.5a3.5 3.5 0 00-4.95-4.95l-1.25 1.25zm-4.69 9.64a2 2 0 010-2.83l2.5-2.5a2 2 0 012.83 0 .75.75 0 001.06-1.06 3.5 3.5 0 00-4.95 0l-2.5 2.5a3.5 3.5 0 004.95 4.95l1.25-1.25a.75.75 0 00-1.06-1.06l-1.25 1.25a2 2 0 01-2.83 0z"></path></svg></a>Usage</h2>
<p>Plotting functions provided by this package are identical to <a href="https://github.com/spencerlyon2/PlotlyJS.jl">PlotlyJS</a>. Please consult its <a href="http://spencerlyon.com/PlotlyJS.jl/" rel="nofollow">documentation</a>.</p>
<p>For example, this will display a basic scatter plot:</p>
<div class="snippet-clipboard-content position-relative" data-snippet-clipboard-copy-content="my_plot = plot([scatter(x=[1,2], y=[3,4])], Layout(title=&quot;My plot&quot;))

</code></pre></div>

<p>Use the <code>post</code> function to upload a local plot to the Plotly cloud:</p>
<div class="snippet-clipboard-content position-relative" data-snippet-clipboard-copy-content="&gt; my_plot  = plot([scatter(y=[1,2])])
&gt; remote_plot = post(my_plot)
Plotly.RemotePlot(URI(https://plot.ly/~malmaud/73))
"><pre><code>&gt; my_plot  = plot([scatter(y=[1,2])])
&gt; remote_plot = post(my_plot)
Plotly.RemotePlot(URI(https://plot.ly/~malmaud/73))
</code></pre></div>
<p>Visiting <a href="https://plot.ly/~malmaud/73" rel="nofollow">https://plot.ly/~malmaud/73</a> in a browser will show the plot.</p>
