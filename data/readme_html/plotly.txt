<p>Simply run <code>Pkg.add("Plotly")</code>.</p>
<h2><a id="user-content-usage" class="anchor" aria-hidden="true" href="#usage"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path fill-rule="evenodd" d="M4 9h1v1H4c-1.5 0-3-1.69-3-3.5S2.55 3 4 3h4c1.45 0 3 1.69 3 3.5 0 1.41-.91 2.72-2 3.25V8.59c.58-.45 1-1.27 1-2.09C10 5.22 8.98 4 8 4H4c-.98 0-2 1.22-2 2.5S3 9 4 9zm9-3h-1v1h1c1 0 2 1.22 2 2.5S13.98 12 13 12H9c-.98 0-2-1.22-2-2.5 0-.83.42-1.64 1-2.09V6.25c-1.09.53-2 1.84-2 3.25C6 11.31 7.55 13 9 13h4c1.45 0 3-1.69 3-3.5S14.5 6 13 6z"></path></svg></a>Usage</h2>
<p>Plotting functions provided by this package are identical to <a href="https://github.com/spencerlyon2/PlotlyJS.jl">PlotlyJS</a>. Please consult its <a href="http://spencerlyon.com/PlotlyJS.jl/" rel="nofollow">documentation</a>.</p>
<p>For example, this will display a basic scatter plot:</p>
<pre><code>my_plot = plot([scatter(x=[1,2], y=[3,4])], Layout(title="My plot"))
</code></pre>

<h3><a id="user-content-saving-a-plot-to-the-cloud" class="anchor" aria-hidden="true" href="#saving-a-plot-to-the-cloud"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path fill-rule="evenodd" d="M4 9h1v1H4c-1.5 0-3-1.69-3-3.5S2.55 3 4 3h4c1.45 0 3 1.69 3 3.5 0 1.41-.91 2.72-2 3.25V8.59c.58-.45 1-1.27 1-2.09C10 5.22 8.98 4 8 4H4c-.98 0-2 1.22-2 2.5S3 9 4 9zm9-3h-1v1h1c1 0 2 1.22 2 2.5S13.98 12 13 12H9c-.98 0-2-1.22-2-2.5 0-.83.42-1.64 1-2.09V6.25c-1.09.53-2 1.84-2 3.25C6 11.31 7.55 13 9 13h4c1.45 0 3-1.69 3-3.5S14.5 6 13 6z"></path></svg></a>Saving a plot to the cloud</h3>
<p>Use the <code>post</code> function to upload a local plot to the Plotly cloud:</p>
<pre><code>&gt; my_plot  = plot([scatter(y=[1,2])])
&gt; remote_plot = post(my_plot)
Plotly.RemotePlot(URI(https://plot.ly/~malmaud/73))
</code></pre>
<p>Visiting <a href="https://plot.ly/~malmaud/73" rel="nofollow">https://plot.ly/~malmaud/73</a> in a browser will show the plot.</p>
<h3><a id="user-content-download-a-plot-from-the-cloud" class="anchor" aria-hidden="true" href="#download-a-plot-from-the-cloud"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path fill-rule="evenodd" d="M4 9h1v1H4c-1.5 0-3-1.69-3-3.5S2.55 3 4 3h4c1.45 0 3 1.69 3 3.5 0 1.41-.91 2.72-2 3.25V8.59c.58-.45 1-1.27 1-2.09C10 5.22 8.98 4 8 4H4c-.98 0-2 1.22-2 2.5S3 9 4 9zm9-3h-1v1h1c1 0 2 1.22 2 2.5S13.98 12 13 12H9c-.98 0-2-1.22-2-2.5 0-.83.42-1.64 1-2.09V6.25c-1.09.53-2 1.84-2 3.25C6 11.31 7.55 13 9 13h4c1.45 0 3-1.69 3-3.5S14.5 6 13 6z"></path></svg></a>Download a plot from the cloud</h3>
<p>Use the <code>download</code> function with a remote plot object to download a plot stored on the Plotly cloud to a local <code>Plot</code> object:</p>
<pre><code>local_plot = download(RemotePlot("https://plot.ly/~malmaud/73"))
# or equivalently, local_plot = download_plot("https://plot.ly/~malmaud/73")
</code></pre>
