<h2 dir="auto"><a id="user-content-overview" class="anchor" aria-hidden="true" href="#overview"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path d="m7.775 3.275 1.25-1.25a3.5 3.5 0 1 1 4.95 4.95l-2.5 2.5a3.5 3.5 0 0 1-4.95 0 .751.751 0 0 1 .018-1.042.751.751 0 0 1 1.042-.018 1.998 1.998 0 0 0 2.83 0l2.5-2.5a2.002 2.002 0 0 0-2.83-2.83l-1.25 1.25a.751.751 0 0 1-1.042-.018.751.751 0 0 1-.018-1.042Zm-4.69 9.64a1.998 1.998 0 0 0 2.83 0l1.25-1.25a.751.751 0 0 1 1.042.018.751.751 0 0 1 .018 1.042l-1.25 1.25a3.5 3.5 0 1 1-4.95-4.95l2.5-2.5a3.5 3.5 0 0 1 4.95 0 .751.751 0 0 1-.018 1.042.751.751 0 0 1-1.042.018 1.998 1.998 0 0 0-2.83 0l-2.5 2.5a1.998 1.998 0 0 0 0 2.83Z"></path></svg></a>Overview</h2>
<p dir="auto"><code>BayesianNetworkRegression.jl</code> is a <a href="http://julialang.org/" rel="nofollow">Julia</a> package to perform (Bayesian) statistical inference of a regression model with networked covariates on a real response.</p>
<p dir="auto">The structure of the data assumes <math-renderer class="js-inline-math" style="display: inline" data-static-url="https://github.githubassets.com/static" data-run-id="daa28034fbf1e4b91986924e0450c39e">$n$</math-renderer> samples, each sample with a microbial network represented as an adjacency matrix <math-renderer class="js-inline-math" style="display: inline" data-static-url="https://github.githubassets.com/static" data-run-id="daa28034fbf1e4b91986924e0450c39e">$A_i$</math-renderer> and a measured biological phenotype <math-renderer class="js-inline-math" style="display: inline" data-static-url="https://github.githubassets.com/static" data-run-id="daa28034fbf1e4b91986924e0450c39e">$y_i$</math-renderer>. This package will identify influential nodes (microbes) associated with the phenotype, as well as significant edges (interactions among microbes) that are also associated with the phenotype.</p>
<p dir="auto">Examples of data that can be fit with this model:</p>
<ul dir="auto">
<li>Samples of soil microbiome data represented as <math-renderer class="js-inline-math" style="display: inline" data-static-url="https://github.githubassets.com/static" data-run-id="daa28034fbf1e4b91986924e0450c39e">$n$</math-renderer> microbial networks and measured yield of <math-renderer class="js-inline-math" style="display: inline" data-static-url="https://github.githubassets.com/static" data-run-id="daa28034fbf1e4b91986924e0450c39e">$n$</math-renderer> plants on that soil. The model will identify microbes and interactions associated with plant yield.</li>
<li>Samples of gut microbiome data represented as <math-renderer class="js-inline-math" style="display: inline" data-static-url="https://github.githubassets.com/static" data-run-id="daa28034fbf1e4b91986924e0450c39e">$n$</math-renderer> microbial networks, one per human patient, and BMI measurement per human patient. The model will identify microbes and interactions associated with BMI.</li>
</ul>
