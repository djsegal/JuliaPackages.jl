<p dir="auto"><code>PhyloDiamond.jl</code> is a <a href="http://julialang.org/" rel="nofollow">Julia</a> package for ultrafast learning of 4-node hybridization cycles in phylogenetic networks using algebraic invariants.</p>
<p dir="auto">Input data:</p>
<ul dir="auto">
<li>A concordance factor table, or</li>
<li>A file containing gene trees</li>
</ul>
