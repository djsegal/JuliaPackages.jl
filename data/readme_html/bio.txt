<strong>You might still download and use this package, as we don't want old scripts to break.</strong>
<strong>However going forward, know that this repository is archived and read only, no further updates or fixes will be committed. You should use the packages that replace Bio.jl - a list is available <a href="https://biojulia.net/post/biojl/" rel="nofollow">here</a> as well as below.</strong></p>
