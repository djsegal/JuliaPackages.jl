<ul dir="auto">
<li>
<p dir="auto"><code>maxRooks(p::Poly)</code>: Solves the maximal non-attacking rook set problem, O(n^4)</p>
</li>
<li>
<p dir="auto"><code>maxQueens(p::Poly)</code>: Solves the maximal non-attacking queen set problem, NP-complete</p>
</li>
<li>
<p dir="auto"><code>minRooks(p::Poly)</code>: Solves the minimal guarding rook set problem, NP-complete</p>
</li>
<li>
<p dir="auto"><code>minQueens(p::Poly)</code>: Solves the minimal guarding queen set problem, NP-complete</p>
</li>
</ul>
