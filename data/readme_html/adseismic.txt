<p dir="auto">ADSeismic is built for general seismic inversion problems, such as estimating velocity model, source location and time function. The package implements the forward FDTD (finite difference time domain) simulation of acoustic and elastic wavefields and enables flexible inversions of parameters in the wave equations using <strong>automatic differentiation</strong>. Several features of this package includes:</p>
<ul dir="auto">
<li><strong>Battery included</strong>: No adjoint state method codes required for building your own inversion models; Automatic differentiation is the work horse, making it unified approach to various seismic inversion problems such as full waveform inversion (FWI), earthquake location inversion, and source rupture inversion.</li>
<li><strong>Neural Networks</strong>: Easy to integrate Neural Networks (NN) into seismic inversion.</li>
<li><strong>High performance</strong>: Computational graph optimization using Tensorflow; Parallel computing using MPI.</li>
<li><strong>(Multi-)GPU support</strong>: Support for GPU acceleration as well as multi-GPU (in theory TPUs are also supported).</li>
</ul>
<p dir="auto">Examples can be found int these papers:</p>
<p dir="auto"><a href="https://arxiv.org/pdf/2003.06027.pdf" rel="nofollow">A General Approach to Seismic Inversion with Automatic Differentiation</a></p>
<p dir="auto"><a href="https://arxiv.org/pdf/2012.11149.pdf" rel="nofollow">Integrating Deep Neural Networks with Full-waveform Inversion: Reparametrization, Regularization, and Uncertainty Quantification</a></p>
<p dir="auto"><a href="https://arxiv.org/pdf/2011.01349.pdf" rel="nofollow">Distributed Machine Learning for Computational Engineering using MPI</a></p>
<p dir="auto"><a href="https://link.springer.com/article/10.1007/s40687-022-00329-z" rel="nofollow">Learning generative neural networks with physics knowledge</a></p>

<p dir="auto">This might take up to 20 minutes for complete installation.</p>
<p dir="auto">Then install this package (ADSeismic.jl) by</p>
<div class="highlight highlight-source-julia notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="using Pkg
