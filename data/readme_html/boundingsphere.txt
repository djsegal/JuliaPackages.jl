<p dir="auto">Package contains algorithms to calculate smallest enclosing sphere for a given
set of points in N dimensions.</p>
<p dir="auto">BoundingSphere.jl is a complete rewrite from scratch of <a href="https://github.com/JuliaFEM/Miniball.jl">Miniball.jl</a>.
See <a href="https://github.com/JuliaFEM/Miniball.jl/issues/28" data-hovercard-type="issue" data-hovercard-url="/ahojukka5/Miniball.jl/issues/28/hovercard">Miniball.jl issue #28</a>.</p>
