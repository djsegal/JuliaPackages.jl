<p>Performant, macro-only, Julia package for parsing command line arguments.<br>
Uses macros to generate the parsing code within your main function, directly storing
results in typed local variables.</p>

<p>Install ArgMacros using Julia's Pkg package manager. Enter the Pkg prompt
by typing <code>]</code> at the REPL and then install:</p>
<pre lang="julia-repl"><code>(@v1.4) pkg&gt; add ArgMacros
</code></pre>
<p>Then load ArgMacros into your script with <code>using ArgMacros</code>.</p>
