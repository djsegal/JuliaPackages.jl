<p>Performant, flexible, macro-based, Julia package for parsing command line arguments.<br>
Uses macros to generate the parsing code within your script, directly storing
results in typed local variables, an autogenerated custom <code>struct</code> type, <code>NamedTuple</code>, or <code>Dict</code></p>

<p>Install ArgMacros using Julia's Pkg package manager. Enter the Pkg prompt
by typing <code>]</code> at the REPL and then install:</p>
<div class="snippet-clipboard-content position-relative" data-snippet-clipboard-copy-content="(@v1.5) pkg&gt; add ArgMacros
"><pre lang="julia-repl"><code>(@v1.5) pkg&gt; add ArgMacros
</code></pre></div>
<p>Then load ArgMacros into your script with <code>using ArgMacros</code>.</p>
