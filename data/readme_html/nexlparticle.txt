<p dir="auto">SEM/EDS particle analysis tools</p>
<ul dir="auto">
<li>Read / Write Zeppelin-style automated particle analysis data sets</li>
<li>Diluvian Clustering</li>
<li>Particle data set registration (alignment)</li>
<li>Particle segregation algorithms</li>
<li>Multi-ternary plots</li>
</ul>
<p dir="auto"><code>NeXLParticle</code> is part of the NeXL collection of Julia libraries for X-ray microanalysis.  These consist of <a href="https://github.com/usnistgov/NeXLUncertainties.jl">NeXLUncertainties</a>, <a href="https://github.com/usnistgov/NeXLCore.jl">NeXLCore</a>, <a href="https://github.com/usnistgov/NeXLMatrixCorrection.jl">NeXLMatrixCorrection</a>, <a href="https://github.com/usnistgov/NeXLSpectrum.jl">NeXLSpectrum</a> and NeXLParticle.</p>
