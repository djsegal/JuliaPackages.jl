<p dir="auto"><code>AbInitioSoftwareBase.jl</code> is an interface package that defines some common API
shared by some packages that represent different <em>ab initio</em> software like
<a href="https://www.quantum-espresso.org/" rel="nofollow">Quantum ESPRESSO</a>, <a href="https://www.vasp.at/" rel="nofollow">VASP</a>, etc.
The API will be extended in
<a href="https://github.com/MineralsCloud/QuantumESPRESSOBase.jl"><code>QuantumESPRESSOBase.jl</code></a>, etc.</p>
<p dir="auto">The code is <a href="https://github.com/MineralsCloud/AbInitioSoftwareBase.jl">hosted on GitHub</a>,
with some continuous integration services to test its validity.</p>
