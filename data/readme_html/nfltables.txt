<p dir="auto">Functions to access NFL data. Please create a GitHub issue to request a new
dataset or report a bug.</p>
<p dir="auto"><code>NFLTables</code> provides enumerations and functions to make it <em>convenient</em> to access existing NFL data.
Where relevant, variable, module, and function names follow the conventions for the julia language.</p>

This is the recommended data for new analysis.</p>
