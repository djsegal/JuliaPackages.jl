<p dir="auto">This package provides an interface to the POSIX calls for tty I/O control.</p>
<blockquote>
<p dir="auto">The termios functions describe a general terminal interface that is
provided to control asynchronous communications ports.</p>
</blockquote>
<p dir="auto">For a complete description of these calls, see the <a href="http://man7.org/linux/man-pages/man3/termios.3.html" rel="nofollow">termios(3)</a> manual page.</p>
<p dir="auto">All functions in this package take a <code>RawFD</code> file descriptor <code>fd</code> as their first argument.
This can also be an integer file descriptor or a concrete instance of <code>Base.LibuvStream</code>, such as <code>stdin</code> or <code>stdout</code>.</p>
<p dir="auto">This package defines the constants needed to work with the functions provided.
You may need to refer to your system documentation for more information on using this package.</p>
