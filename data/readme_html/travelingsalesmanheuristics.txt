<p dir="auto"><code>TravelingSalesmanHeuristics</code> is a Julia package containing simple heuristics for the <a href="https://en.wikipedia.org/wiki/Travelling_salesman_problem" rel="nofollow">traveling salesman problem</a>.</p>
<p dir="auto">This package is both my first Julia package and my first effort in open source software, so I welcome any contributions, suggestions, feature requests, pull requests, criticisms, etc. The package has reached a stable usable state, and while ongoing development is sporadic, I attempt to keep the package well-maintained.</p>
