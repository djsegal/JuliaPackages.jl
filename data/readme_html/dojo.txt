<ul dir="auto">
<li>We are no longer actively developing Dojo, but pull requests are always welcome.</li>
<li>We have updated or removed examples to account for changes since the initial version of Dojo.</li>
<li>Additional developments on differentiable simulation:
<ul dir="auto">
<li>Differentiable collision detection (Kevin Tracy): <a href="https://arxiv.org/abs/2207.00202" rel="nofollow">capsules</a>, <a href="https://arxiv.org/abs/2207.00669" rel="nofollow">convex primitives</a></li>
<li>Single-level contact dynamics + collision detection (Simon Le Cleac'h): <a href="https://arxiv.org/pdf/2212.06764.pdf" rel="nofollow">Silico</a></li>
</ul>
</li>
</ul>
