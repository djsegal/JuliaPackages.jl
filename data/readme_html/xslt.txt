<p dir="auto">Transform XML documents to any other document type using XSLT stylesheets.</p>
<p dir="auto">This is a Julia wrapper for the <a href="http://xmlsoft.org/libxslt/" rel="nofollow">libxslt</a> library.</p>
<p dir="auto">XSLT.jl is fully compatible with <a href="https://github.com/JuliaIO/EzXML.jl">EzXML.jl</a> and <a href="https://github.com/JuliaIO/LightXML.jl">LightXML.jl</a>, so you can use it to transform XML documents to HTML, JSON, etc...</p>
