<p dir="auto">Measuring distances, synchrony and correlation between spike trains. Most methods implemented here are summarized in the relevant <a href="http://www.scholarpedia.org/article/Measures_of_spike_train_synchrony" rel="nofollow">Scholarpedia entry</a>, but the documentation strings also cite the papers.</p>
<p dir="auto">SpikeSynchrony.jl has a clear, small and easy-to-understand source code, which could be helpful in education purposes. Notice that the present code has not been optimized in any way for performance (as the functions are already fast enough that there is no reason to).</p>
<p dir="auto">To install, press <code>]</code> in the Julia console to enter package mode and then
do</p>
<div class="snippet-clipboard-content notranslate position-relative overflow-auto" data-snippet-clipboard-copy-content="add https://github.com/Datseris/SpikeSynchrony.jl"><pre class="notranslate"><code>add https://github.com/Datseris/SpikeSynchrony.jl
</code></pre></div>
<p dir="auto">To use it in your Julia sessions, do <code>using SpikeSynchrony</code>.</p>
