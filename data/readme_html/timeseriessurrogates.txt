<p dir="auto">A Julia package for generating timeseries surrogates. TimeseriesSurrogates.jl is the fastest and most featureful open source code for generating timeseries surrogates.
It can be used as a standalone package, or as part of
<a href="https://juliadynamics.github.io/DynamicalSystems.jl/dev/" rel="nofollow">DynamicalSystems.jl</a>.</p>
<p dir="auto">To install it, run <code>import Pkg; Pkg.add("TimeseriesSurrogates")</code>.</p>
<p dir="auto">All further information is provided in the documentation, which you can either find <a href="https://juliadynamics.github.io/TimeseriesSurrogates.jl/dev/" rel="nofollow">online</a> or build locally by running the <code>docs/make.jl</code> file.</p>
