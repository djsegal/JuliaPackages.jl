<p>This is a package designed to help get started with web APIs and some basic interaction between Julia and a browser.</p>
<h2><a id="user-content-installation" class="anchor" aria-hidden="true" href="#installation"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path fill-rule="evenodd" d="M7.775 3.275a.75.75 0 001.06 1.06l1.25-1.25a2 2 0 112.83 2.83l-2.5 2.5a2 2 0 01-2.83 0 .75.75 0 00-1.06 1.06 3.5 3.5 0 004.95 0l2.5-2.5a3.5 3.5 0 00-4.95-4.95l-1.25 1.25zm-4.69 9.64a2 2 0 010-2.83l2.5-2.5a2 2 0 012.83 0 .75.75 0 001.06-1.06 3.5 3.5 0 00-4.95 0l-2.5 2.5a3.5 3.5 0 004.95 4.95l1.25-1.25a.75.75 0 00-1.06-1.06l-1.25 1.25a2 2 0 01-2.83 0z"></path></svg></a>Installation</h2>
<p>To get the latest tagged version, try:</p>
<div class="highlight highlight-source-julia position-relative" data-snippet-clipboard-copy-content="pkg&gt; add Pages
"><pre>pkg<span class="pl-k">&gt;</span> add Pages</pre></div>
<p>However, this package is still in early development and is likely to change often. To get the latest, try:</p>
<div class="highlight highlight-source-julia position-relative" data-snippet-clipboard-copy-content="pkg&gt; add Pages#master
"><pre>pkg<span class="pl-k">&gt;</span> add Pages<span class="pl-c"><span class="pl-c">#</span>master</span></pre></div>
<h2><a id="user-content-introduction" class="anchor" aria-hidden="true" href="#introduction"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path fill-rule="evenodd" d="M7.775 3.275a.75.75 0 001.06 1.06l1.25-1.25a2 2 0 112.83 2.83l-2.5 2.5a2 2 0 01-2.83 0 .75.75 0 00-1.06 1.06 3.5 3.5 0 004.95 0l2.5-2.5a3.5 3.5 0 00-4.95-4.95l-1.25 1.25zm-4.69 9.64a2 2 0 010-2.83l2.5-2.5a2 2 0 012.83 0 .75.75 0 001.06-1.06 3.5 3.5 0 00-4.95 0l-2.5 2.5a3.5 3.5 0 004.95 4.95l1.25-1.25a.75.75 0 00-1.06-1.06l-1.25 1.25a2 2 0 01-2.83 0z"></path></svg></a>Introduction</h2>
<p>To get started, try the following:</p>
<div class="highlight highlight-source-julia position-relative" data-snippet-clipboard-copy-content="julia&gt; using Pages

julia&gt; @async Pages.start();
Listening on 0.0.0.0:8000...
"><pre>julia<span class="pl-k">&gt;</span> <span class="pl-k">using</span> Pages

julia<span class="pl-k">&gt;</span> <span class="pl-c1">@async</span> Pages<span class="pl-k">.</span><span class="pl-c1">start</span>();
Listening on <span class="pl-c1">0.0</span>.<span class="pl-c1">0.0</span><span class="pl-k">:</span><span class="pl-c1">8000.</span><span class="pl-k">..</span></pre></div>
<p>This starts a server that is listening at <a href="http://localhost:8000" rel="nofollow">http://localhost:8000</a> and exposes the <code>Endpoint</code> type with a few methods.</p>
<p>To create an <code>Endpoint</code>, try:</p>
<div class="highlight highlight-source-julia position-relative" data-snippet-clipboard-copy-content="julia&gt; Endpoint(&quot;/hello&quot;) do request::HTTP.Request
          &quot;Hello world&quot;
       end
"><pre>julia<span class="pl-k">&gt;</span> <span class="pl-c1">Endpoint</span>(<span class="pl-s"><span class="pl-pds">"</span>/hello<span class="pl-pds">"</span></span>) <span class="pl-k">do</span> request<span class="pl-k">::</span><span class="pl-c1">HTTP.Request</span>
          <span class="pl-s"><span class="pl-pds">"</span>Hello world<span class="pl-pds">"</span></span>
       <span class="pl-k">end</span></pre></div>
<p>This creates a web page at <a href="http://localhost:8000/hello" rel="nofollow">http://localhost:8000/hello</a> that says <code>Hello world</code>.</p>
<p>One nice thing about using <code>Pages</code> is that we can create pages whenever and wherever we want in our Julia code even while the server is running.</p>
<h2><a id="user-content-examples" class="anchor" aria-hidden="true" href="#examples"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path fill-rule="evenodd" d="M7.775 3.275a.75.75 0 001.06 1.06l1.25-1.25a2 2 0 112.83 2.83l-2.5 2.5a2 2 0 01-2.83 0 .75.75 0 00-1.06 1.06 3.5 3.5 0 004.95 0l2.5-2.5a3.5 3.5 0 00-4.95-4.95l-1.25 1.25zm-4.69 9.64a2 2 0 010-2.83l2.5-2.5a2 2 0 012.83 0 .75.75 0 001.06-1.06 3.5 3.5 0 00-4.95 0l-2.5 2.5a3.5 3.5 0 004.95 4.95l1.25-1.25a.75.75 0 00-1.06-1.06l-1.25 1.25a2 2 0 01-2.83 0z"></path></svg></a>Examples</h2>
<p>There are a few examples included. The fastest way to see the examples is to run:</p>
<div class="highlight highlight-source-julia position-relative" data-snippet-clipboard-copy-content="julia&gt; Pages.examples(launch=true);
"><pre>julia<span class="pl-k">&gt;</span> Pages<span class="pl-k">.</span><span class="pl-c1">examples</span>(launch<span class="pl-k">=</span><span class="pl-c1">true</span>);</pre></div>
<p>This will start a server and launch a browser open to a page with links to some simple examples. If you prefer not to open a browser to the page from the command line, you can instead run</p>
<div class="highlight highlight-source-julia position-relative" data-snippet-clipboard-copy-content="julia&gt; Pages.examples();
"><pre>julia<span class="pl-k">&gt;</span> Pages<span class="pl-k">.</span><span class="pl-c1">examples</span>();</pre></div>
<p>and then navigate to <a href="http://localhost:8000/examples" rel="nofollow">http://localhost:8000/examples</a>.</p>
<p>Current examples include:</p>
<ul>
<li>Requests - Send GET and POST requests from the browser to Julia and print the contents to the REPL.</li>
<li>Blank - You can use this for experimemntation, e.g. use <code>Pages.add_library</code> to insert your favorite JavaScript library.</li>
<li>Random Ping - For this example, click the "Blank" link and run</li>
</ul>
<div class="highlight highlight-source-julia position-relative" data-snippet-clipboard-copy-content="julia&gt; Pages.Examples.randomping.start()
"><pre>julia<span class="pl-k">&gt;</span> Pages<span class="pl-k">.</span>Examples<span class="pl-k">.</span>randomping<span class="pl-k">.</span><span class="pl-c1">start</span>()</pre></div>
<p>This puts you into an infinite loop inserting text to the webpage.</p>
<p>To stop the example, run</p>
<div class="highlight highlight-source-julia position-relative" data-snippet-clipboard-copy-content="julia&gt; Pages.Examples.randomping.stop()
"><pre>julia<span class="pl-k">&gt;</span> Pages<span class="pl-k">.</span>Examples<span class="pl-k">.</span>randomping<span class="pl-k">.</span><span class="pl-c1">stop</span>()</pre></div>
