<p dir="auto">Experimental Julia framework for multiphysics processes based on implicit finite-volume methods with automatic differentiation. See <a href="https://github.com/sintefmath/JutulDarcy.jl">JutulDarcy.jl</a> for the porous media simulator, and <a href="https://github.com/sintefmath/JutulViz.jl">JutulViz.jl</a> for visualization features and <a href="https://github.com/sintefmath/JutulExamples.jl">JutulExamples.jl</a> for examples that demonstrate usage.</p>
