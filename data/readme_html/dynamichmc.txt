<h2><a id="user-content-overview" class="anchor" aria-hidden="true" href="#overview"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path fill-rule="evenodd" d="M7.775 3.275a.75.75 0 001.06 1.06l1.25-1.25a2 2 0 112.83 2.83l-2.5 2.5a2 2 0 01-2.83 0 .75.75 0 00-1.06 1.06 3.5 3.5 0 004.95 0l2.5-2.5a3.5 3.5 0 00-4.95-4.95l-1.25 1.25zm-4.69 9.64a2 2 0 010-2.83l2.5-2.5a2 2 0 012.83 0 .75.75 0 001.06-1.06 3.5 3.5 0 00-4.95 0l-2.5 2.5a3.5 3.5 0 004.95 4.95l1.25-1.25a.75.75 0 00-1.06-1.06l-1.25 1.25a2 2 0 01-2.83 0z"></path></svg></a>Overview</h2>
<p>This package implements a modern version of the “No-U-turn sampler” in the Julia language, mostly as described in <a href="https://arxiv.org/abs/1701.02434" rel="nofollow">Betancourt (2017)</a>, with some tweaks.</p>
<p>In contrast to frameworks which utilize a directed acyclic graph to build a posterior for a Bayesian model from small components, this package requires that you code a <em>log-density function</em> of the posterior in Julia. Derivatives can be provided manually, or using <a href="http://www.juliadiff.org/" rel="nofollow">automatic differentiation</a>.</p>
<p>Consequently, this package requires that the user is comfortable with the basics of the theory of Bayesian inference, to the extent of coding a (log) posterior density in Julia. This approach allows the use of standard tools like <a href="https://docs.julialang.org/en/v1/manual/profile/" rel="nofollow">profiling</a> and <a href="https://github.com/JuliaCI/BenchmarkTools.jl">benchmarking</a> to optimize its <a href="https://docs.julialang.org/en/v1/manual/performance-tips/" rel="nofollow">performance</a>.</p>
<p>The building blocks of the algorithm are implemented using a <em>functional</em> (non-modifying) approach whenever possible, allowing extensive unit testing of components, while at the same time also intended to serve as a transparent, pedagogical introduction to the low-level mechanics of current Hamiltonian Monte Carlo samplers, and as a platform for research into MCMC methods.</p>
<p>Please start with the <a href="https://tamaspapp.eu/DynamicHMC.jl/dev/" rel="nofollow">documentation</a>.</p>
