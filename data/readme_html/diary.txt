<p>Diary.jl keeps a copy of what you type in the REPL in a file called <code>diary.jl</code>.  The file location defaults to the root of your current active project (the same location where your <code>Project.toml</code> and <code>Manifest.toml</code> files lie).  Please read the documentation (links above) for instructions on how to configure Diary.jl to your needs.</p>
<p>If you encounter any problems or have suggestions for how to improve the package, please open an issue or pull request.  All contributions from all people are welcome!</p>

<p>To try out Diary.jl, install it using <code>]add Diary</code>. Then,</p>
<div class="highlight highlight-source-julia position-relative" data-snippet-clipboard-copy-content="julia&gt; using Diary
"><pre>julia<span class="pl-k">&gt;</span> <span class="pl-k">using</span> Diary</pre></div>
<p>to get started.</p>
<p>If you want to enable Diary.jl by default, put the following in your <code>~/.julia/config/startup.jl</code> file:</p>
