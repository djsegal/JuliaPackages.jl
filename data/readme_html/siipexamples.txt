<p>This package contains examples and tutorials for <a href="https://www.nrel.gov/analysis/siip.html" rel="nofollow">Scalable Integrated Infrastructure Planning (SIIP) packages developed at the National Renewable Energy Laboratory (NREL)</a>. This repository uses a template established by <a href="https://github.com/JuliaOpt/JuMPTutorials.jl">JuMPTutorials.jl</a>.</p>
<p>The examples included here are intended as practical examples of using <a href="https://github.com/nrel-siip">SIIP packages</a>. However,
in many cases users may find the documentation pages for each package more informative:</p>
<ul>
<li><a href="https://nrel-siip.github.io/InfrastructureSystems.jl/stable/" rel="nofollow">InfrastructureSystems.jl docs</a></li>
<li><a href="https://nrel-siip.github.io/PowerSystems.jl/stable/" rel="nofollow">PowerSystems.jl docs</a></li>
<li><a href="https://nrel-siip.github.io/PowerSimulations.jl/latest/" rel="nofollow">PowerSimulations.jl docs</a></li>
<li><a href="https://nrel-siip.github.io/PowerGraphics.jl/latest/" rel="nofollow">PowerGraphics.jl docs</a></li>
<li><a href="https://nrel-siip.github.io/PowerSimulationsDynamics.jl/stable/" rel="nofollow">PowerSimulationsDynamics.jl docs</a></li>
</ul>

<p><a href="https://mybinder.org/v2/gh/NREL-SIIP/SIIPExamples.jl/master" rel="nofollow"><img src="https://camo.githubusercontent.com/581c077bdbc6ca6899c86d0acc6145ae85e9d80e6f805a1071793dbe48917982/68747470733a2f2f6d7962696e6465722e6f72672f62616467655f6c6f676f2e737667" alt="Binder" data-canonical-src="https://mybinder.org/badge_logo.svg" style="max-width:100%;"></a></p>
<p>To try out any of the tutorials in the browser without downloading Julia, click on the launch binder button above. Note that this functionality only supports open-source solvers which do not have additional requirements (for e.g. BLAS or MATLAB). This is also very slow and can take several minutes to start as it has to first install Julia and all the dependencies. Thus, you should download and run the notebooks on your local machine for the best experience.</p>

<p><em>Prerequisites:</em></p>
<ul>
<li>Install <a href="https://julialang.org" rel="nofollow">Julia</a></li>
</ul>
<p>To get started running the Jupyter notebooks included in this package, you can follow the process in <a href="https://www.youtube.com/watch?v=n1NvcnLczJ8&amp;feature=youtu.be" rel="nofollow">this video</a> demonstrating the following steps:</p>
<ol>
<li>Install SIIPExamples: <code>using Pkg; Pkg.add("SIIPExamples")</code></li>
<li>Launch a notebook server for any of the example categories (<code>JuliaExamples, PSYExamples, PSIExamples, PSDExamples</code>): <code>notebook(PSYExamples)</code></li>
</ol>
