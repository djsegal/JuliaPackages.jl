<p dir="auto">This Julia package provides a wrapper for the <a href="https://manual.limesurvey.org/RemoteControl_2_API" rel="nofollow">LimeSurvey Remote Control 2 API</a>.</p>

Simply install it by calling</p>
<div class="highlight highlight-source-julia notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="pkg&gt; add CitrusAPI"><pre>pkg<span class="pl-k">&gt;</span> add CitrusAPI</pre></div>

<p dir="auto">If your LimeSurvey server is set up correctly you can start by setting up a <code>CitrusClient</code> and connecting to the server.</p>
<div class="highlight highlight-source-julia notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="client = CitrusClient(&quot;https://your-limesurvey-server.com/index.php/admin/remotecontrol&quot;)
connect!(client, &quot;username&quot;, &quot;password&quot;)"><pre>client <span class="pl-k">=</span> <span class="pl-c1">CitrusClient</span>(<span class="pl-s"><span class="pl-pds">"</span>https://your-limesurvey-server.com/index.php/admin/remotecontrol<span class="pl-pds">"</span></span>)
<span class="pl-c1">connect!</span>(client, <span class="pl-s"><span class="pl-pds">"</span>username<span class="pl-pds">"</span></span>, <span class="pl-s"><span class="pl-pds">"</span>password<span class="pl-pds">"</span></span>)</pre></div>
<p dir="auto">Once the connection is established you can execute your desired API methods, e.g. getting a list of all surveys,</p>
<div class="highlight highlight-source-julia notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="surveys = list_surveys(client)"><pre>surveys <span class="pl-k">=</span> <span class="pl-c1">list_surveys</span>(client)</pre></div>
<p dir="auto">Close the session by disconnecting from the server.</p>
<div class="highlight highlight-source-julia notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="disconnect!(client)"><pre><span class="pl-c1">disconnect!</span>(client)</pre></div>

<p dir="auto">Note that CitrusAPI uses a more "julian" syntax for API methods. Methods that mutate data by either adding, deleting or modifying data are implemented as <a href="https://docs.julialang.org/en/v1/manual/style-guide/#bang-convention" rel="nofollow">bang functions</a>. For example the API method <code>add_survey</code> is implemented as the <code>add_survey!</code>.</p>
