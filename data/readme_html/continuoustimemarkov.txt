<p>A small package for working with continuous-time Markov transition rate matrices.</p>
<p>Currently exposes the following functionality:</p>
<ul>
<li>
<p>wrappers <code>TransitionRateMatrix</code> and <code>TransitionRateMatrix!</code> for creating transition rate matrices (square, nonnegative elements, rows sum to one),</p>
</li>
<li>
<p><code>steady_state</code> for calculating steady state distributions.</p>
</li>
</ul>
