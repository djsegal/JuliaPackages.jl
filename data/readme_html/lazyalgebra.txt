<p><strong>LazyAlgebra</strong> is a <a href="http://julialang.org/" rel="nofollow">Julia</a> package to generalize the
notion of matrices and vectors used in
<a href="https://en.wikipedia.org/wiki/Linear_algebra" rel="nofollow">linear algebra</a>.</p>
<p>This package is used in:</p>
<ul>
<li><a href="https://github.com/emmt/OptimPackNextGen.jl">OptimPackNextGen</a> which
provides optimization methods for large scale problems in pure Julia.</li>
<li><a href="https://github.com/emmt/LinearInterpolators.jl">LinearInterpolators</a> which
provides methods for interpolating multi-dimensional arrays.</li>
</ul>

<ul>
