<p>CoverageTools.jl provides the core functionality for processing code coverage and memory allocation results.</p>
<p>Most users will want to use <a href="https://github.com/JuliaCI/Coverage.jl">Coverage.jl</a> instead. <a href="https://github.com/JuliaCI/Coverage.jl">Coverage.jl</a> allows you to upload your coverage reports to online services such as Codecov.io and Coveralls.io.</p>
