<p dir="auto">MRFingerprintingRecon.jl is a Julia package that implements--so far--the <em>low rank inversion</em> reconstruction described in the paper <a href="https://doi.org/10.1002/mrm.26639" rel="nofollow">Low rank alternating direction method of multipliers reconstruction for MR fingerprinting</a>. This package is still work in progress and the interface will likely change over time. The ultimate goal of this package is to reproduce the <a href="https://bitbucket.org/asslaender/nyu_mrf_recon" rel="nofollow">Matlab code</a> of the ADMM algorithm to Julia.</p>
