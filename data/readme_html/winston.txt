<p dir="auto">Winston offers an easy to use <code>plot</code> command to create figures without any
fuss. After Winston is loaded by typing <code>using Winston</code>, the most basic
plot can be created by just writing</p>
<div class="highlight highlight-source-julia notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="julia&gt; plot(x, y)"><pre>julia<span class="pl-k">&gt;</span> <span class="pl-c1">plot</span>(x, y)</pre></div>
<p dir="auto">There is also an <code>oplot</code> command to add objects into already existing
plots. To add something to this, use</p>
<div class="highlight highlight-source-julia notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="julia&gt; oplot(x2, y2)"><pre>julia<span class="pl-k">&gt;</span> <span class="pl-c1">oplot</span>(x2, y2)</pre></div>
<p dir="auto">And finally save it with</p>
<div class="highlight highlight-source-julia notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="julia&gt; savefig(&quot;figure.png&quot;)   # .eps, .pdf, &amp; .svg are also supported"><pre>julia<span class="pl-k">&gt;</span> <span class="pl-c1">savefig</span>(<span class="pl-s"><span class="pl-pds">"</span>figure.png<span class="pl-pds">"</span></span>)   <span class="pl-c"><span class="pl-c">#</span> .eps, .pdf, &amp; .svg are also supported</span></pre></div>
<p dir="auto">More elaborate figures can be created by using the quick option for color,
line, and symbols</p>
<div class="highlight highlight-source-julia notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="julia&gt; plot(x, y, &quot;r--&quot;)"><pre>julia<span class="pl-k">&gt;</span> <span class="pl-c1">plot</span>(x, y, <span class="pl-s"><span class="pl-pds">"</span>r--<span class="pl-pds">"</span></span>)</pre></div>
<p dir="auto">This creates a red dashed curve. Abbreviations for colors and lines/symbols
are same as in matplotlib. The <code>plot</code> command can also take more then
one set of vectors and style options, like this</p>
<div class="highlight highlight-source-julia notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="julia&gt; plot(x, y, &quot;b:&quot;, x2, y2, &quot;g^&quot;)"><pre>julia<span class="pl-k">&gt;</span> <span class="pl-c1">plot</span>(x, y, <span class="pl-s"><span class="pl-pds">"</span>b:<span class="pl-pds">"</span></span>, x2, y2, <span class="pl-s"><span class="pl-pds">"</span>g^<span class="pl-pds">"</span></span>)</pre></div>
<p dir="auto">which creates a blue dotted line and green triangles.</p>
