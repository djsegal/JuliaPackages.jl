<h2><a id="user-content-getting-started" class="anchor" aria-hidden="true" href="#getting-started"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path fill-rule="evenodd" d="M7.775 3.275a.75.75 0 001.06 1.06l1.25-1.25a2 2 0 112.83 2.83l-2.5 2.5a2 2 0 01-2.83 0 .75.75 0 00-1.06 1.06 3.5 3.5 0 004.95 0l2.5-2.5a3.5 3.5 0 00-4.95-4.95l-1.25 1.25zm-4.69 9.64a2 2 0 010-2.83l2.5-2.5a2 2 0 012.83 0 .75.75 0 001.06-1.06 3.5 3.5 0 00-4.95 0l-2.5 2.5a3.5 3.5 0 004.95 4.95l1.25-1.25a.75.75 0 00-1.06-1.06l-1.25 1.25a2 2 0 01-2.83 0z"></path></svg></a>Getting started</h2>
<p>Winston offers an easy to use <code>plot</code> command to create figures without any
fuss. After Winston is loaded by typing <code>using Winston</code>, the most basic
plot can be created by just writing</p>
<div class="highlight highlight-source-julia"><pre>julia<span class="pl-k">&gt;</span> <span class="pl-c1">plot</span>(x, y)</pre></div>
<p>There is also an <code>oplot</code> command to add objects into already existing
plots. To add something to this, use</p>
<div class="highlight highlight-source-julia"><pre>julia<span class="pl-k">&gt;</span> <span class="pl-c1">oplot</span>(x2, y2)</pre></div>
<p>And finally save it with</p>
<div class="highlight highlight-source-julia"><pre>julia<span class="pl-k">&gt;</span> <span class="pl-c1">savefig</span>(<span class="pl-s"><span class="pl-pds">"</span>figure.png<span class="pl-pds">"</span></span>)   <span class="pl-c"><span class="pl-c">#</span> .eps, .pdf, &amp; .svg are also supported</span></pre></div>
<p>More elaborate figures can be created by using the quick option for color,
line, and symbols</p>
<div class="highlight highlight-source-julia"><pre>julia<span class="pl-k">&gt;</span> <span class="pl-c1">plot</span>(x, y, <span class="pl-s"><span class="pl-pds">"</span>r--<span class="pl-pds">"</span></span>)</pre></div>
<p>This creates a red dashed curve. Abbreviations for colors and lines/symbols
are same as in matplotlib. The <code>plot</code> command can also take more then
one set of vectors and style options, like this</p>
<div class="highlight highlight-source-julia"><pre>julia<span class="pl-k">&gt;</span> <span class="pl-c1">plot</span>(x, y, <span class="pl-s"><span class="pl-pds">"</span>b:<span class="pl-pds">"</span></span>, x2, y2, <span class="pl-s"><span class="pl-pds">"</span>g^<span class="pl-pds">"</span></span>)</pre></div>
<p>which creates a blue dotted line and green triangles.</p>
