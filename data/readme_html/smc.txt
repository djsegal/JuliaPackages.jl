<h1><a id="user-content-sequential-monte-carlo" class="anchor" aria-hidden="true" href="#sequential-monte-carlo"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path fill-rule="evenodd" d="M7.775 3.275a.75.75 0 001.06 1.06l1.25-1.25a2 2 0 112.83 2.83l-2.5 2.5a2 2 0 01-2.83 0 .75.75 0 00-1.06 1.06 3.5 3.5 0 004.95 0l2.5-2.5a3.5 3.5 0 00-4.95-4.95l-1.25 1.25zm-4.69 9.64a2 2 0 010-2.83l2.5-2.5a2 2 0 012.83 0 .75.75 0 001.06-1.06 3.5 3.5 0 00-4.95 0l-2.5 2.5a3.5 3.5 0 004.95 4.95l1.25-1.25a.75.75 0 00-1.06-1.06l-1.25 1.25a2 2 0 01-2.83 0z"></path></svg></a>Sequential Monte Carlo</h1>
<p>This package implements the Sequential Monte Carlo (SMC) sampling algorithm, an alternative to Metropolis Hastings Markov Chain Monte Carlo sampling for approximating posterior distributions. The SMC algorithm implemented here is based upon Edward Herbst and Frank Schorfheide's paper "<a href="http://dx.doi.org/10.1002/jae.2397" rel="nofollow">Sequential Monte Carlo Sampling for DSGE Models</a>" and the code accompanying their book, <em>Bayesian Estimation of DSGE Models</em>. More information and the original MATLAB scripts from which this code was derived can be found at Frank Schorfheide's <a href="https://sites.sas.upenn.edu/schorf/pages/bayesian-estimation-dsge-models" rel="nofollow">website</a>.</p>
<p>Our implementation features what we term <em>generalized tempering</em> for "online" estimation, as outlined in our recent paper, "<a href="https://papers.ssrn.com/sol3/papers.cfm?abstract_id=3426004" rel="nofollow">Online Estimation of DSGE Models</a>." For a broad overview of the algorithm, one may refer to the following <em>Liberty Street Economics</em> <a href="https://libertystreeteconomics.newyorkfed.org/2019/08/online-estimation-of-dsge-models.html" rel="nofollow">article</a>.</p>

<p><code>SMC.jl</code> is a registered Julia package in the <a href="https://github.com/JuliaRegistries/General"><code>General</code></a> registry, compatible with Julia <code>v0.7</code>, <code>v1.0</code>, and <code>v1.1</code>. To install it, open your Julia REPL, type <code>]</code> to enter the package manager, and run</p>
<div class="highlight highlight-source-julia"><pre>pkg<span class="pl-k">&gt;</span> add SMC</pre></div>
<h2><a id="user-content-usage" class="anchor" aria-hidden="true" href="#usage"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path fill-rule="evenodd" d="M7.775 3.275a.75.75 0 001.06 1.06l1.25-1.25a2 2 0 112.83 2.83l-2.5 2.5a2 2 0 01-2.83 0 .75.75 0 00-1.06 1.06 3.5 3.5 0 004.95 0l2.5-2.5a3.5 3.5 0 00-4.95-4.95l-1.25 1.25zm-4.69 9.64a2 2 0 010-2.83l2.5-2.5a2 2 0 012.83 0 .75.75 0 001.06-1.06 3.5 3.5 0 00-4.95 0l-2.5 2.5a3.5 3.5 0 004.95 4.95l1.25-1.25a.75.75 0 00-1.06-1.06l-1.25 1.25a2 2 0 01-2.83 0z"></path></svg></a>Usage</h2>
<p>The package requires our auxiliary package, <a href="https://github.com/FRBNY-DSGE/ModelConstructors.jl">ModelConstructors.jl</a>, which contains useful data structures for creating custom models (e.g. <code>Parameter</code>, <code>State</code>, <code>Observable</code>, <code>Setting</code> types).</p>
<p>For examples of how to set up a model in the form SMC can estimate, see scripts in the <a href="https://github.com/FRBNY-DSGE/SMC.jl/tree/master/examples"><code>examples/</code></a> folder.</p>
