<p dir="auto">This package implements the Sequential Monte Carlo (SMC) sampling algorithm, an alternative to Metropolis Hastings Markov Chain Monte Carlo sampling for approximating posterior distributions. The SMC algorithm implemented here is based upon and extends Edward Herbst and Frank Schorfheide's paper "<a href="http://dx.doi.org/10.1002/jae.2397" rel="nofollow">Sequential Monte Carlo Sampling for DSGE Models</a>" and the code accompanying their book, <em>Bayesian Estimation of DSGE Models</em>. More information and the original MATLAB scripts from which this code was derived can be found at Frank Schorfheide's <a href="https://sites.sas.upenn.edu/schorf/pages/bayesian-estimation-dsge-models" rel="nofollow">website</a>.</p>
<p dir="auto">Our implementation features an adaptive schedule and what we term <em>generalized tempering</em> for "online" estimation, as outlined in our recent paper, "<a href="https://papers.ssrn.com/sol3/papers.cfm?abstract_id=3426004" rel="nofollow">Online Estimation of DSGE Models</a>." For a broad overview of the algorithm, one may refer to the following <em>Liberty Street Economics</em> <a href="https://libertystreeteconomics.newyorkfed.org/2019/08/online-estimation-of-dsge-models.html" rel="nofollow">article</a>.</p>

<p dir="auto">The package requires our auxiliary package, <a href="https://github.com/FRBNY-DSGE/ModelConstructors.jl">ModelConstructors.jl</a>, which contains useful data structures for creating custom models (e.g. <code>Parameter</code>, <code>State</code>, <code>Observable</code>, <code>Setting</code> types).</p>
<p dir="auto">For examples of how to set up a model in the form SMC can estimate, see scripts in the <a href="https://github.com/FRBNY-DSGE/SMC.jl/tree/main/examples"><code>examples/</code></a> folder.</p>
