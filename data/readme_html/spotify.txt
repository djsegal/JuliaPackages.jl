<p dir="auto">You can</p>
<ul dir="auto">
<li>play music and prune your playlists through a mini-player REPL mode</li>
<li>organize playlists based on audio analysis or each track's popularity in Japan</li>
<li>fix your taste profile after that kid's party</li>
<li>build social graphs for artists, playlists or users, analyze popularity</li>
<li>clean duplicates</li>
<li>search podcast metadata</li>
<li>create your own visualizations from beat data</li>
<li>play with popularity and genre statistics</li>
</ul>
<p dir="auto">Easily find the right functions with the <code>select_calls</code> menu and the submodules:</p>
<ul dir="auto">
<li>Albums, Artists, Categories, Episodes, Genres,</li>
<li>Library, Markets, Player, Playlists, Search</li>
<li>Shows, Tracks, Users, Profile</li>
</ul>

<h2 dir="auto"><a id="user-content-wrapper-api" class="anchor" aria-hidden="true" href="#wrapper-api"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path d="m7.775 3.275 1.25-1.25a3.5 3.5 0 1 1 4.95 4.95l-2.5 2.5a3.5 3.5 0 0 1-4.95 0 .751.751 0 0 1 .018-1.042.751.751 0 0 1 1.042-.018 1.998 1.998 0 0 0 2.83 0l2.5-2.5a2.002 2.002 0 0 0-2.83-2.83l-1.25 1.25a.751.751 0 0 1-1.042-.018.751.751 0 0 1-.018-1.042Zm-4.69 9.64a1.998 1.998 0 0 0 2.83 0l1.25-1.25a.751.751 0 0 1 1.042.018.751.751 0 0 1 .018 1.042l-1.25 1.25a3.5 3.5 0 1 1-4.95-4.95l2.5-2.5a3.5 3.5 0 0 1 4.95 0 .751.751 0 0 1-.018 1.042.751.751 0 0 1-1.042.018 1.998 1.998 0 0 0-2.83 0l-2.5 2.5a1.998 1.998 0 0 0 0 2.83Z"></path></svg></a>Wrapper API</h2>
<p dir="auto">Wrapper functions require different permission scopes, which you grant as needed by pressing 'Accept' in a browser tab which pops up when needed. A granted scope lasts for one hour.</p>
<p dir="auto">The wrapper functions are duck-typed and will accept strings, vectors of strings or the types below.
Functions leniently convert input to these string-like types:</p>
<ul dir="auto">
<li><code>SpPlaylistId</code>, <code>SpArtistId</code>, <code>SpTrackId</code>, <code>SpAlbumId</code></li>
<li><code>SpShowId</code>, <code>SpEpisodeId</code>, <code>SpId</code>, <code>SpCategoryId</code></li>
</ul>
<p dir="auto">Ouput from all wrapper functions is a tuple: (<code>JSON3 object</code>, <code>wait_seconds</code>).</p>
<ul dir="auto">
<li><code>wait_seconds</code> is zero except if the API rate limit is temporarily exceeded</li>
<li><code>JSON3 object</code> is an <code>object</code> / <code>dictionary</code> which is easy and fast to inspect and use. See the examples for some ideas</li>
</ul>
<p dir="auto">Control of text feedback is provided, see <code>?LOGSTATE</code>.</p>
