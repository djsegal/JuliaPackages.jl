<p>This package will provide a library of functions useful for ML, such as softmax, sigmoid, convolutions and pooling. It doesn't provide any other "high-level" functionality like layers or AD.</p>
<p>Other packages can build on these functions as if they were defined in Base Julia; for example, <a href="https://github.com/JuliaGPU/CUDA.jl">CUDA.jl</a> provides GPU kernels, and <a href="https://github.com/FluxML/Zygote.jl">Zygote.jl</a> provides automatic differentiation; both can work together without explicitly being aware of each other.</p>
