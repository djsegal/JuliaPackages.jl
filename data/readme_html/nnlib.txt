<p dir="auto">This package provides a library of functions useful for neural networks, such as softmax, sigmoid, batched multiplication, convolutions and pooling. Many of these are used by <a href="https://github.com/FluxML/Flux.jl">Flux.jl</a>, which loads this package, but they may be used independently.</p>
<p dir="auto">For use with automatic differentiation, this package defines gradients using <a href="https://github.com/JuliaDiff/ChainRules.jl">ChainRules.jl</a>. These will be seen by various packages including <a href="https://github.com/FluxML/Zygote.jl">Zygote.jl</a>.</p>
<p dir="auto">To use these functions with <a href="https://github.com/JuliaGPU/CUDA.jl">CUDA.jl</a> you will need <a href="https://github.com/FluxML/NNlibCUDA.jl">NNlibCUDA.jl</a> as well.</p>
