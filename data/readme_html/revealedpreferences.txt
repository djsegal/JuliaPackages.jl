<p>A library for revealed preferences analysis using Julia.
The analysis can start from a choice function, a choice correspondence or prices and quantities.</p>
<p>The library implements classical axioms of revealed preferences (Sen's axioms, the Weak and Strong Axiom of Revealed Preferences, and so on)
It also implements some indices of rationality (Houtman-Maks and the swap index for now).</p>
