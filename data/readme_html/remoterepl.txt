<p dir="auto"><code>RemoteREPL</code> allows you to connect your local julia REPL to a separate Julia
process and run commands interactively:</p>
<ul dir="auto">
<li>Run code in the <code>Main</code> module of a remote Julia process</li>
<li>Standard REPL tab completion and help mode with <code>?</code></li>
<li>Transfer variables between processes with <code>@remote</code></li>
<li>Automatic ssh tunnel for network security. Reconnects dropped connections.</li>
</ul>
<p dir="auto">Read <a href="https://c42f.github.io/RemoteREPL.jl/dev" rel="nofollow"><strong>the latest documentation</strong></a> more information.</p>
<p dir="auto">(See also the <a href="https://juliahub.com/docs/RemoteREPL" rel="nofollow"><strong>docs on juliahub.com</strong></a>).</p>
