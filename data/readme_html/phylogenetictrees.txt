<p dir="auto">PhylogeneticTrees is a package implementing miqoGraph, a fast algorithm to fit admixture trees using mixed-integer quadratic optimization.  It was developed using the <a href="http://julialang.org/" rel="nofollow">Julia</a> language and the <a href="http://www.gurobi.com/" rel="nofollow">Gurobi</a> optimization solver.</p>

<p dir="auto">PhylogeneticTrees is implemented using the Gurobi optimization solver, which has a <a href="http://www.gurobi.com/registration/academic-license-reg" rel="nofollow">free academic license</a>.  The instructions for installation can be found <a href="http://www.gurobi.com/documentation/" rel="nofollow">here</a>.</p>
<p dir="auto">The Julia language can be downloaded <a href="https://julialang.org/downloads/" rel="nofollow">here</a>.  This package was developed using Julia 1.0.</p>
<p dir="auto">PhylogeneticTrees can be installed through the Julia <a href="https://docs.julialang.org/en/v1/stdlib/Pkg/index.html" rel="nofollow">package manager</a>:</p>
<div class="snippet-clipboard-content notranslate position-relative overflow-auto" data-snippet-clipboard-copy-content="(v1.0) pkg&gt; add &quot;https://github.com/juliayyan/PhylogeneticTrees.jl&quot; "><pre class="notranslate"><code>(v1.0) pkg&gt; add "https://github.com/juliayyan/PhylogeneticTrees.jl" 
</code></pre></div>
<p dir="auto">To check that the installation is working properly, run the following within the package manager:</p>
<div class="snippet-clipboard-content notranslate position-relative overflow-auto" data-snippet-clipboard-copy-content="(v1.0) pkg&gt; test PhylogeneticTrees"><pre class="notranslate"><code>(v1.0) pkg&gt; test PhylogeneticTrees
</code></pre></div>
<p dir="auto"><strong>NOTE</strong>.  The dependencies for PhylogeneticTrees include JuMP (v0.18.5), Gurobi, CSV (v0.4.3), DataFrames (v0.17.1), and LightGraphs.  Newer or older versions of these packages may cause errors.  To install a specific version of these packages, you can use the following commands:</p>
<div class="snippet-clipboard-content notranslate position-relative overflow-auto" data-snippet-clipboard-copy-content="(v1.0) pkg&gt; add JuMP@0.18.5"><pre class="notranslate"><code>(v1.0) pkg&gt; add JuMP@0.18.5
