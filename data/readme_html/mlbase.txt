<p>This package does not implement specific machine learning algorithms. Instead, it provides a collection of useful tools to support machine learning programs, including:</p>
<ul>
<li>Data manipulation &amp; preprocessing</li>
<li>Score-based classification</li>
<li>Performance evaluation (<em>e.g.</em> evaluating ROC)</li>
<li>Cross validation</li>
<li>Model tuning (<em>i.e.</em> search best settings of parameters)</li>
</ul>
<p><strong>Notes:</strong> This package depends on <a href="https://github.com/JuliaStats/StatsBase.jl">StatsBase</a> and reexports all names therefrom.</p>
