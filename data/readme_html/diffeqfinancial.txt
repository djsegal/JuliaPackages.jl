<p dir="auto">This repository holds problem definitions for common financial models for use in
the DifferentialEquations ecosystem. The goal is to be a feature-complete set of
solvers for the types of problems found in libraries like QuantLib. For example,
one can easily specify, solve, and plot the solution for a trajectory of a Heston
process via the commands:</p>
<div class="highlight highlight-source-julia notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="prob = HestonProblem(μ,κ,Θ,σ,ρ,u₀)
sol = solve(prob)
plot(sol)"><pre>prob <span class="pl-k">=</span> <span class="pl-c1">HestonProblem</span>(μ,κ,Θ,σ,ρ,u₀)
sol <span class="pl-k">=</span> <span class="pl-c1">solve</span>(prob)
<span class="pl-c1">plot</span>(sol)</pre></div>
<p dir="auto">Full documentation is in the
<a href="https://docs.sciml.ai/DiffEqDocs/stable/" rel="nofollow">DifferentialEquations.jl models documentation</a></p>
<p dir="auto">These solvers use DifferentialEquations.jl, meaning that high-performance and
high order algorithms are available.</p>
<p dir="auto">The project is looking for contributors who would like to implement more models.
If you're interested and need help getting started, talk to Chris Rackauckas in
the <a href="https://gitter.im/JuliaDiffEq/Lobby" rel="nofollow">Gitter chat</a>.</p>
