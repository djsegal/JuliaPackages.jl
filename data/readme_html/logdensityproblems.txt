<p>A common framework for implementing and using log densities for inference, providing the following functionality.</p>
<ol>
<li>
<p>The <a href="https://tamaspapp.eu/LogDensityProblems.jl/dev/#LogDensityProblems.logdensity" rel="nofollow"><code>logdensity</code></a> method with corresponding interface, which can be used by other packages that operate on (log) densities and need to evaluate the log densities or the gradients (eg <a href="https://en.wikipedia.org/wiki/Markov_chain_Monte_Carlo" rel="nofollow">MCMC</a>, <a href="https://en.wikipedia.org/wiki/Maximum_a_posteriori_estimation" rel="nofollow">MAP</a>, <a href="https://en.wikipedia.org/wiki/Maximum_likelihood_estimation" rel="nofollow">ML</a> or similar methods).</p>
</li>
<li>
<p>The <a href="https://tamaspapp.eu/LogDensityProblems.jl/dev/#LogDensityProblems.ADgradient" rel="nofollow"><code>ADgradient</code></a> which makes objects that support <code>logdensity</code> to calculate log density <em>values</em> calculate log density <em>gradients</em> using various automatic differentiation packages.</p>
</li>
<li>
<p>The wrapper <a href="https://tamaspapp.eu/LogDensityProblems.jl/dev/#LogDensityProblems.TransformedLogDensity" rel="nofollow"><code>TransformedLogDensity</code></a> using the <a href="https://github.com/tpapp/TransformVariables.jl">TransformVariables.jl</a> package, allowing callables that take a set of parameters transformed from a flat vector of real numbers to support the <code>logdensity</code> interface.</p>
</li>
<li>
<p>Various utility functions for debugging and testing log densities.</p>
</li>
</ol>
<p>See the <a href="https://tpapp.github.io/LogDensityProblems.jl/dev" rel="nofollow">documentation</a> for details.</p>
