<p dir="auto">A common framework for implementing and using log densities for inference, providing the following functionality.</p>
<ol dir="auto">
<li>
<p dir="auto">The <a href="https://tamaspapp.eu/LogDensityProblems.jl/dev/#LogDensityProblems.logdensity" rel="nofollow"><code>logdensity</code></a> method with corresponding interface, which can be used by other packages that operate on (log) densities and need to evaluate the log densities or the gradients (eg <a href="https://en.wikipedia.org/wiki/Markov_chain_Monte_Carlo" rel="nofollow">MCMC</a>, <a href="https://en.wikipedia.org/wiki/Maximum_a_posteriori_estimation" rel="nofollow">MAP</a>, <a href="https://en.wikipedia.org/wiki/Maximum_likelihood_estimation" rel="nofollow">ML</a> or similar methods).</p>
</li>
<li>
<p dir="auto">Various utility functions for debugging and testing log densities.</p>
</li>
</ol>
<p dir="auto"><strong>NOTE</strong> As of version 1.0, transformed log densities have been moved to <a href="https://github.com/tpapp/TransformedLogDensities.jl">TransformedLogDensities.jl</a>. Existing code that uses <code>TransformedLogDensity</code> should add</p>
<div class="snippet-clipboard-content notranslate position-relative overflow-auto" data-snippet-clipboard-copy-content="using TransformedLogDensities"><pre class="notranslate"><code>using TransformedLogDensities
</code></pre></div>
<p dir="auto">or equivalent.</p>
<p dir="auto"><strong>NOTE</strong>: As of version 2.0, automatic differentiation backends have been moved to <a href="https://github.com/tpapp/LogDensityProblemsAD.jl" title="LogDensityProblemsAD.jl">https://github.com/tpapp/LogDensityProblemsAD.jl</a>. If your code uses <code>ADgradient</code>, simply add</p>
<div class="highlight highlight-source-julia notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="using LogDensityProblemsAD"><pre><span class="pl-k">using</span> LogDensityProblemsAD</pre></div>
<p dir="auto">or equivalent.</p>
<p dir="auto">See the <a href="https://tpapp.github.io/LogDensityProblems.jl/dev" rel="nofollow">documentation</a> for details.</p>
