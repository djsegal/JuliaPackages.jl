<ul dir="auto">
<li>Automatically stores precompile statements for the given project.</li>
<li>Single command build process by <code>build_sysimage()</code>.</li>
<li>Removes old unused sysimages.</li>
<li>Warns if the loaded sysimage contains outdated packages (Julia v1.8+).</li>
</ul>
<p dir="auto">Please see the documentation for more details.</p>
