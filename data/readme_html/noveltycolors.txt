<p>This package is intended to be a lightweight package for non-standard color palettes. Unlike <a href="https://github.com/JuliaLang/Color.jl">Color.jl</a>, there is not (necessarily) any theoretical underpinnings to these palettes. Color palettes provided within this package may represent Internet memes, generalized color palettes seen in advertising or on television/movies, or just aesthetically pleasing colors.</p>
<p>If you are visualizing data for presentation or publication, you should validate that any of these color schemes provide acceptable contrast when printed in Black &amp; White, are appropriate for colorblind individuals and any other considerations.</p>
<p>The color palettes are stored and exported in a plain Julia <code>Dict</code> type, named <code>ColorDict</code></p>

<p>To submit a contribution, submit a pull request where the data file has the same structure as <a href="https://github.com/randyzwitch/NoveltyColors.jl/blob/master/data/wesanderson.json">WesAnderson.json</a>: the file name represents what the color palette will be called, with the values in the text file as JSON where each key is a <code>AbstractString</code> and its value will be an <code>AbstractArray</code> of hex codes.</p>
