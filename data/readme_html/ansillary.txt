<li>REPLectomy: The standard library REPL package is pretty heavyweight, and Ansillary only uses like two things from it (<code>raw!</code> and <code>TTYTerminal</code>) so removing it seems more sensible. It would also be nice if Ansillary could theoretically be used as the base for REPL rather than the other way around.</li>
<li>Mouse support.</li>
<li>A nicer API for style than <code>printstyled</code>.</li>
