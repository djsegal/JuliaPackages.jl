<p dir="auto">An interface for geospatial vector data in <a href="https://julialang.org/" rel="nofollow">Julia</a>.</p>
<p dir="auto">This Package describe a set of traits based on the <a href="https://www.opengeospatial.org/standards/sfa" rel="nofollow">Simple Features standard
(SF)</a> for geospatial vector data, including
the SQL/MM extension with support for circular geometry. Using these traits, it should be
easy to parse, serialize and use different geometries in the Julia ecosystem, without
knowing the specifics of each individual package. In that regard it is similar to
<a href="https://github.com/JuliaData/Tables.jl">Tables.jl</a>, but for geometries instead of tables.</p>
<p dir="auto">Packages which support the GeoInterface.jl interface can be found in
<a href="INTEGRATIONS.md">INTEGRATIONS.md</a>.</p>
