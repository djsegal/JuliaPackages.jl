<p dir="auto">A package for simulating classical spin systems, including the Landau-Lifshitz dynamics of spin dipoles and its generalization to multipolar spin components. In the latter case, Sunny resolves the local quantum structure of individual spins, making it particularly suited for modeling magnetic compounds with strong single-ion anisotropy, e.g., due to crystal field splitting.</p>
<p dir="auto">Sunny additionally provides Monte Carlo algorithms for sampling from thermal equilibrium, as well as tools for measuring dynamical structure factors that can be compared with experimental neutron scattering data. Symmetry analyses are available to facilitate the design and specification of model Hamiltonians. Some interactive 3D visualization tools are available, with more planned.</p>
<p dir="auto">Sunny is currently under heavy development with many <strong>breaking changes</strong>. See the <a href="https://sunnysuite.github.io/Sunny.jl/dev/versions/" rel="nofollow">version history</a> for details.</p>

<p dir="auto">To see Sunny in action, we recommend browsing the <a href="https://sunnysuite.github.io/Sunny.jl/dev/examples/fei2_tutorial/" rel="nofollow">FeI2 case study</a>. This example uses SU(3) spin dynamics to measure the dynamical structure factor. The SU(3) formalism is essential to model the single-ion bound-state, which has been <a href="https://doi.org/10.1038/s41567-020-01110-1" rel="nofollow">observed experimentally</a>.</p>
<p dir="auto">In addition to the examples in the official <a href="https://sunnysuite.github.io/Sunny.jl/dev/" rel="nofollow">documentation</a>, a number of tutorials are available as Jupyter notebooks at the <a href="https://github.com/SunnySuite/SunnyTutorials/tree/main/Tutorials">SunnyTutorials</a> repo.</p>

<p dir="auto">A quantum spin of magnitude <em>S</em> has <em>N</em> = 2 <em>S</em> + 1 distinct angular momentum eigenstates. The traditional classical approximation effectively replaces each spin operator with a dipole expectation value, producing the Landau-Lifshitz spin-dipole dynamics. Alternatively, one can derive a dynamics that retains all multipolar expectation values (dipoles, quadrupoles, etc.) [<a href="https://arxiv.org/abs/2106.14125" rel="nofollow">Zhang and Batista, Phys. Rev. B <strong>104</strong>, 104409 (2021)</a>]. This formalism can be understood as a dynamics of SU(<em>N</em>) coherent states, and coincides with the usual Laudau-Lifshitz dynamics when <em>N</em>=2.</p>
<p dir="auto">Sunny uses recently developed algorithms to simulate this SU(<em>N</em>) spin dynamics in a highly efficient way:</p>
<ul dir="auto">
<li>D. Dahlbom et al., <em>Geometric integration of classical spin dynamics via a mean-field Schr√∂dinger equation</em>, Phys. Rev. B <strong>106</strong>, 054423 (2022) [<a href="https://arxiv.org/abs/2204.07563" rel="nofollow">arXiv:2204.07563</a>].</li>
<li>D. Dahlbom et al., <em>Langevin dynamics of generalized spins as SU(N) coherent states</em>, Phys. Rev. B <strong>106</strong>, 235154 (2022) [<a href="https://arxiv.org/abs/2209.01265" rel="nofollow">arXiv:2209.01265</a>].</li>
</ul>

<p dir="auto">In addition to the generalized SU(<em>N</em>) spin dynamics, Sunny can also simulate spins in the dipole-only approximation. When running in this mode, the capabilities of Sunny are similar to <a href="https://spinw.org/" rel="nofollow">SpinW</a>, which was an inspiration for this project. Sunny supports general single-ion anisotropies, and currently has a focus on classical spin dynamics. The classical spin dynamics allows for simulations at finite temperatures, and (soon) the application of chemical inhomogeneities. Support for the SU(<em>N</em>) version of linear spin wave theory is coming soon; this approach will be faster, and enables arbitrary <em>k</em>-space resolution.</p>

<p dir="auto">Sunny is implemented in the <a href="https://julialang.org/" rel="nofollow">Julia programming language</a>. New Julia users may wish to read our <a href="https://github.com/SunnySuite/Sunny.jl/wiki/Getting-started-with-Julia-and-Sunny">Getting Started</a> wiki page.</p>
<p dir="auto">From the Julia prompt, one can install the latest Sunny using the built-in package manager:</p>
<div class="snippet-clipboard-content notranslate position-relative overflow-auto" data-snippet-clipboard-copy-content="julia&gt; ]
pkg&gt; add Sunny"><pre class="notranslate"><code>julia&gt; ]
pkg&gt; add Sunny
</code></pre></div>
<p dir="auto">New point-releases will include <strong>breaking changes</strong>. To install a specific version of Sunny, say <code>v0.x</code>, use the command <code>add Sunny@0.x</code>.</p>
