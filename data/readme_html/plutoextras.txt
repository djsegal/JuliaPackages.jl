<p dir="auto">This package provides some widgets to be used in Pluto, including an extended version of the TableOfContents from PlutoUI and a new experimental bond container (<code>BondTable</code>).</p>
<p dir="auto">This was formerly a non-registered package named PlutoUtils</p>

<p dir="auto">There is some new under-development experimental feature to simplify the grouping and displaying of many bonds within a given notebook in a floating table (similar to the TableOfContents) that stays on the left and can be hidden, resized and moved around.</p>
<p dir="auto">Have a look at the notebook at <a href="https://rawcdn.githack.com/disberd/PlutoExtras.jl/94fdc84217b5591343ba4671666511d16b1c0eaf/html_exports/test_bondstable.jl.html" rel="nofollow">notebooks/test_bondstable.jl</a> for an example of the features/capabilities. (The link points to a static html export of the notebook. If you want to execute the notebook, remember that it has to be executed from the cloned repository as it currently uses the package environment of the package)</p>

<p dir="auto">The <a href="./src/extended_toc.jl">extended_toc.jl</a> notebook defines and exports <code>ExtendedTableOfContents</code>, which takes the <code>TableOfContents</code> from <code>PlutoUI</code> and adds the following functionalities:</p>
<h3 dir="auto"><a id="user-content-hiding-headingcells" class="anchor" aria-hidden="true" href="#hiding-headingcells"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path d="m7.775 3.275 1.25-1.25a3.5 3.5 0 1 1 4.95 4.95l-2.5 2.5a3.5 3.5 0 0 1-4.95 0 .751.751 0 0 1 .018-1.042.751.751 0 0 1 1.042-.018 1.998 1.998 0 0 0 2.83 0l2.5-2.5a2.002 2.002 0 0 0-2.83-2.83l-1.25 1.25a.751.751 0 0 1-1.042-.018.751.751 0 0 1-.018-1.042Zm-4.69 9.64a1.998 1.998 0 0 0 2.83 0l1.25-1.25a.751.751 0 0 1 1.042.018.751.751 0 0 1 .018 1.042l-1.25 1.25a3.5 3.5 0 1 1-4.95-4.95l2.5-2.5a3.5 3.5 0 0 1 4.95 0 .751.751 0 0 1-.018 1.042.751.751 0 0 1-1.042.018 1.998 1.998 0 0 0-2.83 0l-2.5 2.5a1.998 1.998 0 0 0 0 2.83Z"></path></svg></a>Hiding Heading/Cells</h3>
<p dir="auto">Hiding headings and all connected cells from notebook view can be done via ExtendedTableOfContents</p>
<ul dir="auto">
<li>All cells before the first heading are automatically hidden from the notebook</li>
<li>All hidden cells/headings can be shown by pressing the <em>eye</em> button that appears while hovering on the ToC title.
<ul dir="auto">
<li>When the hidden cells are being shown, the hidden headings in the ToC are underlined</li>
</ul>
</li>
<li>Hidden status of specific headings in the notebook can be toggled by pressing on the eye button that appears to the left each heading when hovering over them</li>
</ul>

<p dir="auto">ToC headings are grouped based on heading level, sub-headings at various levels can be collapsed by using the caret symbol that appears to the left of headings in the ToC upon hover.</p>
<h3 dir="auto"><a id="user-content-scroll-behavior-fix" class="anchor" aria-hidden="true" href="#scroll-behavior-fix"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path d="m7.775 3.275 1.25-1.25a3.5 3.5 0 1 1 4.95 4.95l-2.5 2.5a3.5 3.5 0 0 1-4.95 0 .751.751 0 0 1 .018-1.042.751.751 0 0 1 1.042-.018 1.998 1.998 0 0 0 2.83 0l2.5-2.5a2.002 2.002 0 0 0-2.83-2.83l-1.25 1.25a.751.751 0 0 1-1.042-.018.751.751 0 0 1-.018-1.042Zm-4.69 9.64a1.998 1.998 0 0 0 2.83 0l1.25-1.25a.751.751 0 0 1 1.042.018.751.751 0 0 1 .018 1.042l-1.25 1.25a3.5 3.5 0 1 1-4.95-4.95l2.5-2.5a3.5 3.5 0 0 1 4.95 0 .751.751 0 0 1-.018 1.042.751.751 0 0 1-1.042.018 1.998 1.998 0 0 0-2.83 0l-2.5 2.5a1.998 1.998 0 0 0 0 2.83Z"></path></svg></a>Scroll behavior fix</h3>
<p dir="auto">The weird scrolling issue appearing on chromium-based browsers with the scrolling in the TableOfContents (see <a class="issue-link js-issue-link" data-error-text="Failed to load title" data-id="1455360882" data-permission-text="Title is private" data-url="https://github.com/JuliaPluto/PlutoUI.jl/issues/238" data-hovercard-type="issue" data-hovercard-url="/JuliaPluto/PlutoUI.jl/issues/238/hovercard" href="https://github.com/JuliaPluto/PlutoUI.jl/issues/238">JuliaPluto/PlutoUI.jl#238</a>) is fixed by relying on the <a href="https://github.com/LieutenantPeacock/SmoothScroll/">SmoothScroll</a> library.</p>
<ul dir="auto">
<li><em>SmoothScroll</em> is used by default to replace the scroll upon heading click behavior. To maintain the original behavior, call <code>ExtendedTableOfContents(;use_smoothscroll = false)</code></li>
</ul>

<p dir="auto">Preserving the status of collapsed/hidden heading is supported by writing to the notebook file using notebook and cell metadata, allowing to maintain the status even upon reload of Julia/Pluto</p>
<ul dir="auto">
<li>When the current collapsed/hidden status of each heading is not reflected in the notebook file, a save icon/button appears on the left of the ToC title upon hover. Clicking the icon saves the current state in the notebook file.</li>
</ul>

<p dir="auto">The <code>ExtendedTableOfContents</code> allow to re-order the cell groups identified by each heading within the notebook:</p>
<ul dir="auto">
<li>Each cell group is identified by the cell containing the heading, plus all the cells below it and up to the next heading (excluded)</li>
<li>Holding the mouse on a ToC heading triggers the ability to move headings around
<ul dir="auto">
<li>The target heading is surrounded by a dashed border</li>
<li>While moving the mouse within the ToC, a visual separator appears to indicate the position where the dragged heading will be moved to, depending on the mouse position</li>
<li>Hovering on collapsed headings for at least 300ms opens them up to allow moving headings within collapsed parents</li>
</ul>
</li>
<li>By default, headings can only be moved below or above headings of equal or lower level (H1 &lt; H2 &lt; H3...)
<ul dir="auto">
<li>Holding shift during the dragging process allows to put headings before/after any other heading regardless of the level</li>
</ul>
</li>
</ul>
