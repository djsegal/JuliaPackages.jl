<p dir="auto">Iterate over multinomial series expansions and compute corresponding
multinomial coefficients.</p>
<p dir="auto">Status: In development, ready for public testing and comments.</p>

in a convenient iteration scheme.  Thus, an iterator is provided performing
the expansion</p>

<div class="snippet-clipboard-content notranslate position-relative overflow-auto" data-snippet-clipboard-copy-content="eachmultinomial(m,n)"><pre class="notranslate"><code>eachmultinomial(m,n)
</code></pre></div>
<p dir="auto">Create an iterator over all expanded elements of the multinomial series.
Returns a tuple of the requested dimension <code>m</code>. The iterator type
<code>MultinomialIterator</code> provides length and element type information, see
<code>Base.length</code>, <code>Base.eltype</code>, <code>Base.IteratorSize</code> and <code>Base.IteratorEltype</code>.</p>
<div class="snippet-clipboard-content notranslate position-relative overflow-auto" data-snippet-clipboard-copy-content="multinomial(k)"><pre class="notranslate"><code>multinomial(k)
</code></pre></div>
<p dir="auto">Compute the multinomial coefficient from a tuple k of integers, in the same
way as the elements of the iterator <code>eachmultinomial</code> provides.</p>
