<p>This package defines a <a href="https://github.com/JuliaPlots/Plots.jl"><code>Plots</code></a> recipe to implement the Stata command <a href="https://github.com/michaelstepner/binscatter"><code>binscatter</code></a> in Julia.</p>

<h3><a id="user-content-arguments" class="anchor" aria-hidden="true" href="#arguments"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path fill-rule="evenodd" d="M7.775 3.275a.75.75 0 001.06 1.06l1.25-1.25a2 2 0 112.83 2.83l-2.5 2.5a2 2 0 01-2.83 0 .75.75 0 00-1.06 1.06 3.5 3.5 0 004.95 0l2.5-2.5a3.5 3.5 0 00-4.95-4.95l-1.25 1.25zm-4.69 9.64a2 2 0 010-2.83l2.5-2.5a2 2 0 012.83 0 .75.75 0 001.06-1.06 3.5 3.5 0 00-4.95 0l-2.5 2.5a3.5 3.5 0 004.95 4.95l1.25-1.25a.75.75 0 00-1.06-1.06l-1.25 1.25a2 2 0 01-2.83 0z"></path></svg></a>Arguments</h3>
<ul>
<li><code>df</code>: A DataFrame or a GroupedDataFrame</li>
<li><code>f</code>: A formula created using <a href="@ref"><code>@formula</code></a>. The variable(s) in the left-hand side are plotted on the y-axis. The first variable in the right-hand side is plotted on the x-axis. Add other variables for controls.</li>
<li><code>n</code>: Number of bins (default to 20).</li>
</ul>
<h3><a id="user-content-keyword-arguments" class="anchor" aria-hidden="true" href="#keyword-arguments"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path fill-rule="evenodd" d="M7.775 3.275a.75.75 0 001.06 1.06l1.25-1.25a2 2 0 112.83 2.83l-2.5 2.5a2 2 0 01-2.83 0 .75.75 0 00-1.06 1.06 3.5 3.5 0 004.95 0l2.5-2.5a3.5 3.5 0 00-4.95-4.95l-1.25 1.25zm-4.69 9.64a2 2 0 010-2.83l2.5-2.5a2 2 0 012.83 0 .75.75 0 001.06-1.06 3.5 3.5 0 00-4.95 0l-2.5 2.5a3.5 3.5 0 004.95 4.95l1.25-1.25a.75.75 0 00-1.06-1.06l-1.25 1.25a2 2 0 01-2.83 0z"></path></svg></a>Keyword arguments</h3>
<ul>
<li><code>weights</code>: A symbol for weights</li>
<li><code>seriestype</code>:
<ul>
<li><code>:scatter</code> (default) only plots bins</li>
<li><code>:linearfit</code> plots bins with a regression line</li>
<li><code>:scatterpath</code> plots bins with a connecting line</li>
</ul>
</li>
<li><code>kwargs...</code>: Additional attributes from <a href="http://docs.juliaplots.org/latest/" rel="nofollow"><code>Plots</code></a>.</li>
</ul>
