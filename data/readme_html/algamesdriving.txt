<p dir="auto">To run a simple example of a autonomous driving see script in /examples/intro_example.jl.</p>
<ul dir="auto">
<li>Install AlgamesDriving: <code>add AlgamesDriving#v0.1.4</code></li>
<li>Install Algames: <code>add Algames#v0.1.6</code></li>
<li>Install Plots: <code>add Plots</code></li>
<li>Add other dependencies: <code>add LinearAlgebra Meshcat StaticArrays</code></li>
<li>run the script!</li>
</ul>
