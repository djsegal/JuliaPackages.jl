<p>YaoArrayRegister.jl is a component package in the <a href="https://github.com/QuantumBFS/Yao.jl">Yao.jl</a> ecosystem. It provides the most basic functionality for quantum
computation simulation in Julia and a quantum register type <code>ArrayReg</code>. You will be
able to simulate a quantum circuit alone with this package in principle.</p>
