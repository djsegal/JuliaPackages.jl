<p dir="auto"><em>Random Integrators for many-body quantum systems</em></p>
<p dir="auto">The grand aim is to develop a toolbox for many-body quantum systems that can be represented by a Hamiltonian in second quantisation language. Currently supported features include:</p>
<h3 dir="auto"><a id="user-content-interacting-with-quantum-many-body-models" class="anchor" aria-hidden="true" href="#interacting-with-quantum-many-body-models"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path d="m7.775 3.275 1.25-1.25a3.5 3.5 0 1 1 4.95 4.95l-2.5 2.5a3.5 3.5 0 0 1-4.95 0 .751.751 0 0 1 .018-1.042.751.751 0 0 1 1.042-.018 1.998 1.998 0 0 0 2.83 0l2.5-2.5a2.002 2.002 0 0 0-2.83-2.83l-1.25 1.25a.751.751 0 0 1-1.042-.018.751.751 0 0 1-.018-1.042Zm-4.69 9.64a1.998 1.998 0 0 0 2.83 0l1.25-1.25a.751.751 0 0 1 1.042.018.751.751 0 0 1 .018 1.042l-1.25 1.25a3.5 3.5 0 1 1-4.95-4.95l2.5-2.5a3.5 3.5 0 0 1 4.95 0 .751.751 0 0 1-.018 1.042.751.751 0 0 1-1.042.018 1.998 1.998 0 0 0-2.83 0l-2.5 2.5a1.998 1.998 0 0 0 0 2.83Z"></path></svg></a>Interacting with quantum many-body models</h3>
<ul dir="auto">
<li><strong>Full configuration interaction quantum Monte Carlo (FCIQMC)</strong>, a flavour of projector quantum Monte Carlo for stochastically solving the time-independent Schrödinger equation.</li>
<li><strong>Matrix-free exact diagonalisation</strong> of quantum Hamiltonians (with external package <a href="https://github.com/Jutho/KrylovKit.jl"><code>KrylovKit.jl</code></a>).</li>
<li><strong>Sparse matrix representation</strong> of quantum Hamiltonians for exact diagonalisation with sparse linear algebra package of your choice (fastest for small systems).</li>
</ul>
<h3 dir="auto"><a id="user-content-representing-quantum-many-body-models" class="anchor" aria-hidden="true" href="#representing-quantum-many-body-models"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path d="m7.775 3.275 1.25-1.25a3.5 3.5 0 1 1 4.95 4.95l-2.5 2.5a3.5 3.5 0 0 1-4.95 0 .751.751 0 0 1 .018-1.042.751.751 0 0 1 1.042-.018 1.998 1.998 0 0 0 2.83 0l2.5-2.5a2.002 2.002 0 0 0-2.83-2.83l-1.25 1.25a.751.751 0 0 1-1.042-.018.751.751 0 0 1-.018-1.042Zm-4.69 9.64a1.998 1.998 0 0 0 2.83 0l1.25-1.25a.751.751 0 0 1 1.042.018.751.751 0 0 1 .018 1.042l-1.25 1.25a3.5 3.5 0 1 1-4.95-4.95l2.5-2.5a3.5 3.5 0 0 1 4.95 0 .751.751 0 0 1-.018 1.042.751.751 0 0 1-1.042.018 1.998 1.998 0 0 0-2.83 0l-2.5 2.5a1.998 1.998 0 0 0 0 2.83Z"></path></svg></a>Representing quantum many-body models</h3>
<ul dir="auto">
<li>A composable and efficient type system for representing single- and multi-component <strong>Fock states</strong> of bosons, fermions, and mixtures thereof, to be used as a basis for representing Hamiltonians.</li>
<li>An <strong>interface for defining many-body Hamiltonians</strong>.</li>
<li>Pre-defined models include:
<ul dir="auto">
<li><strong>Hubbard model</strong> in real space for bosons and fermions and mixtures in 1, 2, and 3 spatial dimensions.</li>
<li>Hubbard and related <strong>lattice models in momentum space</strong> for bosons and fermions in one spatial dimension.</li>
<li><strong>Transcorrelated Hamiltonian</strong> for contact interactions in one dimension for fermions, as described in Jeszenski <em>et al.</em> <a href="http://arxiv.org/abs/1806.11268" rel="nofollow">arXiv:1806.11268</a>.</li>
</ul>
</li>
</ul>
<h3 dir="auto"><a id="user-content-statistical-analysis-of-monte-carlo-data" class="anchor" aria-hidden="true" href="#statistical-analysis-of-monte-carlo-data"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path d="m7.775 3.275 1.25-1.25a3.5 3.5 0 1 1 4.95 4.95l-2.5 2.5a3.5 3.5 0 0 1-4.95 0 .751.751 0 0 1 .018-1.042.751.751 0 0 1 1.042-.018 1.998 1.998 0 0 0 2.83 0l2.5-2.5a2.002 2.002 0 0 0-2.83-2.83l-1.25 1.25a.751.751 0 0 1-1.042-.018.751.751 0 0 1-.018-1.042Zm-4.69 9.64a1.998 1.998 0 0 0 2.83 0l1.25-1.25a.751.751 0 0 1 1.042.018.751.751 0 0 1 .018 1.042l-1.25 1.25a3.5 3.5 0 1 1-4.95-4.95l2.5-2.5a3.5 3.5 0 0 1 4.95 0 .751.751 0 0 1-.018 1.042.751.751 0 0 1-1.042.018 1.998 1.998 0 0 0-2.83 0l-2.5 2.5a1.998 1.998 0 0 0 0 2.83Z"></path></svg></a>Statistical analysis of Monte Carlo data</h3>
<ul dir="auto">
<li><strong>Blocking analysis</strong> following Flyvberg &amp; Peterson <a href="http://aip.scitation.org/doi/10.1063/1.457480" rel="nofollow">JCP (1989)</a>, and automated with hypothesis testing by Jonsson
<a href="https://link.aps.org/doi/10.1103/PhysRevE.98.043304" rel="nofollow">PRE (2018)</a>.</li>
<li><strong>Unbiased estimators</strong> for the ground state energy by re-reweighting following Nightingale &amp; Blöte <a href="https://link.aps.org/doi/10.1103/PhysRevB.33.659" rel="nofollow">PRB (1986)</a> and Umrigar <em>et al.</em> <a href="http://aip.scitation.org/doi/10.1063/1.465195" rel="nofollow">JCP (1993)</a>.</li>
</ul>
<p dir="auto">The code supports parallelisation with MPI (harnessing <a href="https://github.com/JuliaParallel/MPI.jl"><code>MPI.jl</code></a>) as well as native Julia threading (experimental). In the future, we may add tools to solve the time-dependent Schrödinger equation and Master equations for open system time evolution.</p>

<p dir="auto"><code>Rimu</code> is a registered package and can be installed with the package manager.
Hit the <code>]</code> key at the Julia REPL to get into <code>Pkg</code> mode and type</p>
<div class="snippet-clipboard-content notranslate position-relative overflow-auto" data-snippet-clipboard-copy-content="pkg&gt; add Rimu"><pre lang="julia-repl" class="notranslate"><code>pkg&gt; add Rimu
</code></pre></div>
<p dir="auto">Alternatively, use</p>
<div class="snippet-clipboard-content notranslate position-relative overflow-auto" data-snippet-clipboard-copy-content="julia&gt; using Pkg; Pkg.add(name=&quot;Rimu&quot;)"><pre lang="julia-repl" class="notranslate"><code>julia&gt; using Pkg; Pkg.add(name="Rimu")
</code></pre></div>
<p dir="auto">in order to install <code>Rimu</code> from a script.</p>
