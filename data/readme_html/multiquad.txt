<p><strong>MultiQuad.jl</strong> is a convenient interface to perform 1D, 2D and 3D numerical integrations.
It uses <a href="https://github.com/JuliaMath/QuadGK.jl">QuadGK</a>, <a href="https://github.com/giordano/Cuba.jl">Cuba</a> and <a href="https://github.com/stevengj/HCubature.jl">HCubature</a> as back-ends.</p>

<p><a target="_blank" rel="noopener noreferrer" href="https://camo.githubusercontent.com/ba752da434783b418877ccc22cb74cebb6db9ced/68747470733a2f2f717569636b6c617465782e636f6d2f6361636865332f61322f716c5f39393438336661613335336661303638366461303163383835333233656261325f6c332e706e67"><img src="https://camo.githubusercontent.com/ba752da434783b418877ccc22cb74cebb6db9ced/68747470733a2f2f717569636b6c617465782e636f6d2f6361636865332f61322f716c5f39393438336661613335336661303638366461303163383835333233656261325f6c332e706e67" title="\int_{x1}^{x2}\int_{y1(x)}^{y2(x)}\int_{z1(x,y)}^{z2(x,y)}f(z,y,x)dzdydx" data-canonical-src="https://quicklatex.com/cache3/a2/ql_99483faa353fa0686da01c885323eba2_l3.png" style="max-width:100%;"></a></p>
<p>The supported integration method are:</p>
<ul>
<li><code>:cuhre</code> (default)</li>
<li><code>:vegas</code></li>
<li><code>:suave</code></li>
<li><code>:divonne</code></li>
</ul>
<p>It is suggested to use <code>:cuhre</code> (the default option)</p>
<p>See <a href="https://giordano.github.io/Cuba.jl/stable/" rel="nofollow">Cuba.jl</a> for all the available keyword arguments.</p>
