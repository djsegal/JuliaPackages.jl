<p><a href="https://travis-ci.com/SeismicJulia/SeisReconstruction.jl" rel="nofollow"><img src="https://camo.githubusercontent.com/62aa52ec843aa2ad49c020a27bb4908e70c99d346452156820b817afd4dacb89/68747470733a2f2f7472617669732d63692e636f6d2f536569736d69634a756c69612f536569735265636f6e737472756374696f6e2e6a6c2e7376673f6272616e63683d6d6173746572" alt="Build Status" data-canonical-src="https://travis-ci.com/SeismicJulia/SeisReconstruction.jl.svg?branch=master" style="max-width:100%;"></a></p>
<p>This package contains Reconstruction tools for SeismicJulia project.</p>
<p>At the moment, it is updated and tested against Julia v1.</p>

<p>For SeisPlot, please refer <a href="https://github.com/SeismicJulia/SeisPlot.jl">here</a>.</p>
<p>For SeisProcessing, please refer <a href="https://github.com/SeismicJulia/SeisProcessing.jl">here</a>.</p>
<p>The following example produces the figure below.</p>
<div class="highlight highlight-source-julia position-relative" data-snippet-clipboard-copy-content="using SeisPlot,PyPlot, SeisReconstruction, SeisProcessing

# Ray Abma and Nurul Kabir, 2006, 3D interpolation of irregular data with a POCS algorithm. 
# GEOPHYSICS, 71(6), E91-E97.

# Gao, J., Stanton, A., Naghizadeh, M., Sacchi, M.D. and Chen, X., 2013, Convergence improvement 
# and noise attenuation considerations for beyond alias projection onto convex sets reconstruction.
# Geophysical Prospecting, 61, 138-151.

# Create linear events

d = SeisLinearEvents(p1 = [-.001, 0.0015],tau=[1, 1/3],dx1=5); 

#Randomly decimate, perc=80 means that 80% of the bins are empty

deci = SeisDecimate(d;perc=80);

param = Dict(:Niter=&gt;100,:fmax=&gt;60,:padt=&gt;2,:padx=&gt;2,:dt=&gt;0.004)
dpocs = SeisPOCS(deci;param...);

subplot(121)
SeisPlotTX(deci,cmap=&quot;seismic&quot;,fignum=1,pclip=200,title=&quot;Decimated data&quot;)
subplot(122)
SeisPlotTX(dpocs[:,:,1,1,1],cmap=&quot;seismic&quot;,fignum=1,pclip=200,title=&quot;After POCS&quot;)
