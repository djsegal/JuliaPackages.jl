<p dir="auto">A Julia package for determining the topology of a net.</p>
<p dir="auto">You can use this package through our website interface without any installation required: <a href="https://progs.coudert.name/topology" rel="nofollow">https://progs.coudert.name/topology</a> !</p>
<p dir="auto">The installation follows the usual procedure. Start by downloading and installing <a href="https://julialang.org/" rel="nofollow">Julia</a>, version 1.6 at least for <code>CrystalNets.jl</code>. This package was optimized with Julia version 1.8 so performance and latency will be better on the more recent versions of Julia. Then, either</p>
<ul dir="auto">
<li>open the Julia REPL and enter the package manager by typing <code>]</code>, then install <code>CrystalNets.jl</code> by entering:
<div class="highlight highlight-source-julia notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="pkg&gt; add CrystalNets"><pre>pkg<span class="pl-k">&gt;</span> add CrystalNets</pre></div>
</li>
<li>alternatively, you can do it from a shell by executing:
<div class="highlight highlight-source-shell notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="julia -e 'import Pkg; Pkg.add(&quot;CrystalNets&quot;)'"><pre>julia -e <span class="pl-s"><span class="pl-pds">'</span>import Pkg; Pkg.add("CrystalNets")<span class="pl-pds">'</span></span></pre></div>
</li>
</ul>
<p dir="auto">Please read <a href="https://coudertlab.github.io/CrystalNets.jl/dev" rel="nofollow">the documentation</a> for more
information on the use of CrystalNets.jl and <a href="https://molsim.info/CrystalNets.jl/dev/#Full-installation" rel="nofollow">alternative installation as an executable</a> to reduce latency.</p>
<p dir="auto">The companion article is published in SciPost Chemistry: <a href="https://doi.org/10.21468/SciPostChem.1.2.005" rel="nofollow">doi: 10.21468/SciPostChem.1.2.005</a>.</p>
