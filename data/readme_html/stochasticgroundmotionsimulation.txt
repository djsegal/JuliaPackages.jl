<p dir="auto"><a href="http://www.julialang.org" rel="nofollow">Julia</a> package to simulate response spectral ordinates via random vibration theory.
The package also provides general functionality for working with Fourier amplitude spectra and duration models.</p>
<p dir="auto">Package defines new custom types:</p>
<ul dir="auto">
<li><code>FourierParameters</code>: representing the parameters of the Fourier amplitude spectrum</li>
<li><code>Oscillator</code>: representing a single degree-of-freedom oscillator, and</li>
<li><code>RandomVibrationParameters</code>: defining methods/models used for random vibration calculations</li>
</ul>
<p dir="auto">The <code>FourierParameters</code> type is constructed from three components:</p>
<ul dir="auto">
<li><code>SourceParameters</code>: representing source properties, such as stress parameter, source velocity and density, <em>etc</em></li>
<li><code>PathParameters</code>: representing the path scaling. This component is itself comprised of three components:
<ul dir="auto">
<li><code>GeometricSpreadingParameters</code>: defines the geometric spreading model</li>
<li><code>NearSourceSaturationParameters</code>: defines the near-source saturation model, and</li>
<li><code>AnelasticAttenuationParameters</code>: defines the anelastic attenuation</li>
</ul>
</li>
<li><code>SiteParameters</code>: defines both the site amplification/impedance function and the site damping (via a kappa filter)</li>
</ul>
<p dir="auto">The package is developed in a manner to enable automatic differentiation operations to be performed via <code>ForwardDiff.jl</code>.
This makes the package suitable for gradient-based inversions of ground-motion data, as well as inversions of published ground-motion models.</p>

<p dir="auto">First, a working version of <a href="http://www.julialang.org" rel="nofollow">Julia</a> needs to be installed.
The relevant binary (or source code) can be downloaded from the <a href="https://julialang.org/downloads/" rel="nofollow">Julia Downloads Page</a>.</p>
<p dir="auto"><code>StochasticGroundMotionSimulation.jl</code> is a registered package and can be installed directly from the package manager.</p>
<p dir="auto">Within a Julia REPL session, access the package manager via <code>]</code>, and then at the <code>pkg&gt;</code> prompt type (below the <code>pkg&gt;</code> component is part of the prompt, so only the <code>add ...</code> portion is necessary).</p>
