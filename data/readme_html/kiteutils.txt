<p dir="auto">Utilities for simulating kite power systems.</p>
<p dir="auto">This package is the foundation of Julia Kite Power Tools, which consist of the following packages:</p>

<p dir="auto">If you want to run simulations and see the results in 3D, please install the meta package  <a href="https://github.com/aenarete/KiteSimulators.jl">KiteSimulators</a> . If you have already KiteSimulators installed, use <code>using KiteSimulators</code> instead of <code>using KiteUtils</code>.
If you just want to learn how this package works quickly just install only this package.</p>
