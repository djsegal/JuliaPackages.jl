<p dir="auto">A package for various forest mensuration and biometrics functions in Julia</p>
<p dir="auto">ForestBiometrics.jl is a package for working with forest measurements data and growth and yield modeling.
This package was inspired by its R counterpart, the <a href="https://CRAN.R-project.org/package=lmfor" rel="nofollow">lmfor package</a> with the addition of equations from the western US. For more info on lmfor, please see <a href="http://cs.uef.fi/~lamehtat/rcodes.htm" rel="nofollow">http://cs.uef.fi/~lamehtat/rcodes.htm</a></p>
<p dir="auto">It is my hope that the package not only ports existing functionality from other languages, but also showcases the abilities of Julia as a programming language and how concepts like multiple dispatch and metaprogramming can be used to solve domain-specific problems.</p>

<p dir="auto"><code>ForestBiometrics</code> exports a <code>Tree</code> type and a <code>Stand</code> type for dispatch. The package was designed with fully expanded treelists in mind and makes no effort to accomodate plot data and plot compilation.</p>
<p dir="auto">A <code>Tree</code> is a minimalistic container of fields that are absolutely needed for basic measurements. Future work will focus on the ability to add arbitrary fields as needed.</p>
<div class="highlight highlight-source-julia notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="struct Tree
