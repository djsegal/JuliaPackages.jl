<div class="snippet-clipboard-content position-relative" data-snippet-clipboard-copy-content="imageterm(x,optional arguments...)
"><pre><code>imageterm(x,optional arguments...)
</code></pre></div>
<p>here x is a 2d array. The following optional arguments are currently supported:</p>
<div class="snippet-clipboard-content position-relative" data-snippet-clipboard-copy-content="col=heatscale/diffscale/bluescale/redscale/greenscale/greyscale
"><pre><code>col=heatscale/diffscale/bluescale/redscale/greenscale/greyscale
</code></pre></div>
<p>picks the colorbar.</p>
<div class="snippet-clipboard-content position-relative" data-snippet-clipboard-copy-content="missval=1.0e32
"><pre><code>missval=1.0e32
</code></pre></div>
<p>defines which values are marked as missing and will get a special missval color.</p>
<div class="snippet-clipboard-content position-relative" data-snippet-clipboard-copy-content="hd=true/false
"><pre><code>hd=true/false
</code></pre></div>
<p>determines the resolution (1 pixel per line for false, 2 pixels per line for true).</p>
