<div class="snippet-clipboard-content notranslate position-relative overflow-auto" data-snippet-clipboard-copy-content="imageterm(x,optional arguments...)"><pre class="notranslate"><code>imageterm(x,optional arguments...)
</code></pre></div>
<p dir="auto">here x is a 2d array. The following optional arguments are currently supported:</p>
<div class="snippet-clipboard-content notranslate position-relative overflow-auto" data-snippet-clipboard-copy-content="col=heatscale/diffscale/bluescale/redscale/greenscale/greyscale"><pre class="notranslate"><code>col=heatscale/diffscale/bluescale/redscale/greenscale/greyscale
</code></pre></div>
<p dir="auto">picks the colorbar.</p>
<div class="snippet-clipboard-content notranslate position-relative overflow-auto" data-snippet-clipboard-copy-content="missval=1.0e32"><pre class="notranslate"><code>missval=1.0e32
</code></pre></div>
<p dir="auto">defines which values are marked as missing and will get a special missval color.</p>
<div class="snippet-clipboard-content notranslate position-relative overflow-auto" data-snippet-clipboard-copy-content="hd=true/false"><pre class="notranslate"><code>hd=true/false
</code></pre></div>
<p dir="auto">determines the resolution (1 pixel per line for false, 2 pixels per line for true).</p>
