<p dir="auto">TensorOperations sometimes modifies memory in-place, which is forbidden for all autodiff engines. The way around this is to re-define the 'basic operations' to no longer work in-place. Things like</p>
<div class="snippet-clipboard-content notranslate position-relative overflow-auto" data-snippet-clipboard-copy-content="@tensor v[-1;-2] = ..."><pre class="notranslate"><code>@tensor v[-1;-2] = ...
</code></pre></div>
<p dir="auto">are automatically rewritten as</p>
<div class="snippet-clipboard-content notranslate position-relative overflow-auto" data-snippet-clipboard-copy-content="@tensor temp[-1;-2] := ...
v = temp"><pre class="notranslate"><code>@tensor temp[-1;-2] := ...
v = temp
</code></pre></div>
<p dir="auto">As long as your code does not explicitly depend on the memory location of v being unchanged, everything should work.</p>
