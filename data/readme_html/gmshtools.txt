<ul>
<li>
<p>Users who would like try the newest version could set the environment variable <code>GMSH_LIB_PATH</code> to your source-compiled Gmsh library
directory and rebuild this package. This needs only to be done once.</p>
</li>
<li>
<p>To come along nicely with Intel MKL, the Julia must be compiled with 32 integer interface, i.e. linking to <strong>lp64</strong> instead of <strong>ipl64</strong>.</p>
</li>
<li>
<p>There is an <a href="https://github.com/shipengcheng1230/GmshTools.jl/issues/5">issue</a> of building on <strong>Windows</strong> for Julia <em>v1.4.0</em>. (Currently I don't have a Windows machine to debug.)</p>
</li>
</ul>
