<p>Integration through <a href="https://github.com/JuliaIO/FileIO.jl">FileIO</a> makes loading and saving easy:</p>
<div class="highlight highlight-source-julia position-relative" data-snippet-clipboard-copy-content="data, fs = load(&quot;input.flac&quot;)
data = data * 2
save(&quot;output.flac&quot;, data, fs)
"><pre>data, fs <span class="pl-k">=</span> <span class="pl-c1">load</span>(<span class="pl-s"><span class="pl-pds">"</span>input.flac<span class="pl-pds">"</span></span>)
data <span class="pl-k">=</span> data <span class="pl-k">*</span> <span class="pl-c1">2</span>
<span class="pl-c1">save</span>(<span class="pl-s"><span class="pl-pds">"</span>output.flac<span class="pl-pds">"</span></span>, data, fs)</pre></div>
<p><code>save()</code> can also take extra keyword parameters such as <code>bits_per_sample</code> and <code>compression_level</code> if such parameters are important.  More advanced usage is possible through the use of the <code>FLACDecoder</code> API, which allows for, among other things, seeking within <code>.flac</code> streams and decoding segments of streams.</p>
