<p dir="auto">Bosonic occupation basis using algorithms from <a href="http://dx.doi.org/10.1016/j.chemphys.2011.10.003" rel="nofollow">Szabados et al., 2012</a> (<a href="http://coulson.chem.elte.hu/surjan/PREPRINTS/181.pdf" rel="nofollow">preprint</a>).</p>
<p dir="auto">Tested with Julia 1.0.</p>
