<p dir="auto">This package aims to generalize <code>Base.Fix1</code> and <code>Base.Fix2</code> for arbitrary function arities and binding patterns with a type <code>Fix</code>.
<code>Fix</code> can also include keyword arguments.
One day, parts of this package may be included in Julia's <code>Base</code> itself; see <a href="https://github.com/JuliaLang/julia/issues/36181" data-hovercard-type="issue" data-hovercard-url="/JuliaLang/julia/issues/36181/hovercard">issue #36181</a>.</p>
<p dir="auto">See the documentation for more detail and examples.</p>
<p dir="auto">Related features in other languages:</p>
<ul dir="auto">
<li><a href="https://en.cppreference.com/w/cpp/utility/functional/bind" rel="nofollow">C++'s std::bind</a></li>
<li><a href="https://docs.python.org/3/library/functools.html#functools.partial" rel="nofollow">Python's functools.partial</a></li>
</ul>
