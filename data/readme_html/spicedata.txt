<p>SpiceData currently supports the following SPICE file formats:</p>
<ul>
<li>9601 (32-bit x-values &amp; 32-bit y-values)</li>
<li>9602 (CppSim-specific format? 64-bit x-values &amp; 64-bit y-values?)</li>
<li>2001 (64-bit x-values &amp; 64-bit y-values)</li>
<li>2013 (64-bit x-values &amp; 32-bit y-values)</li>
</ul>
