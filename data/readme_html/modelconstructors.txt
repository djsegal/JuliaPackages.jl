<p>This package contains the building blocks of model objects, such as <code>Parameter</code>, <code>Observable</code>, <code>Setting</code>, and <code>State</code> types. You may define any custom model, so long as it has parameters. The model object is used in both <a href="https://github.com/FRBNY-DSGE/DSGE.jl">DSGE.jl</a> and <a href="https://github.com/FRBNY-DSGE/SMC.jl">SMC.jl</a>.</p>
<p>See <code>/docs/examples</code> for examples of how to construct your custom model, then use <a href="https://github.com/FRBNY-DSGE/SMC.jl">SMC.jl</a> to estimate it (available for any type of model) and <a href="https://github.com/FRBNY-DSGE/DSGE.jl">DSGE.jl</a> to construct forecasts, shock decompositions, impulse responses, etc. (available for DSGE models).</p>
