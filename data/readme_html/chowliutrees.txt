<p dir="auto">This package provides an efficient implementation for learning <a href="https://en.wikipedia.org/wiki/Chow%E2%80%93Liu_tree" rel="nofollow">Chow-Liu Trees</a> from binary and categorical data.</p>
<p dir="auto">As such, this package also provides
functions to estimate a mutual information matrix from data, both on CPU and on GPU, and
functions to compute (top-k) spanning trees.</p>
