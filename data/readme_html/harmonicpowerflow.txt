<p dir="auto">An open source module to solve the harmonic power flow problem using Norton representation of nonlinear devices.</p>
<p dir="auto">Feel free to contribute and get in contact and check out the <a href="https://pweigmann.github.io/HarmonicPowerFlow.jl/dev" rel="nofollow">documentation</a>, which is still a work in progess.</p>
