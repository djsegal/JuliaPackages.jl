<p>This implements single algebra and evaluation on Multivariate functions.
There are a few ways in which it can be used.</p>
<ul>
<li>This can be used for approximation functions. It can currently implement OLS functions, Chebyshev polynomials, the Schumaker shape preserving spline and basic interpolation schemes. It can also do Recursive Partitioning and create Multivariate Adaptive Regression (MARS) Splines. It could be extended to implement other approximation schemes.</li>
<li>As in the <a href="https://github.com/s-baumann/StochasticIntegrals.jl">StochasticIntegrals.jl</a> package this package can be used to define functions that will be the integrands in stochastic integrals. This has the benefit that the means, variances &amp; covariances implied by these stochastic integrals can be found analytically.</li>
<li>All basic algebra and calculus on a MultivariateFunction can be done analytically.</li>
<li>Newton's method is implemented so that roots and optima can be found using analytical Jacobians and Hessians.</li>
</ul>
