<p dir="auto">This small package supports the representation of images as
<a href="https://github.com/mbauman/AxisArrays.jl">AxisArrays</a> to endow the
axes with "meaning," and makes programming with such arrays easy via
<a href="https://github.com/mauro3/SimpleTraits.jl">traits</a>.</p>
<p dir="auto">See also the main documentation for <a href="http://juliaimages.github.io/latest/" rel="nofollow">JuliaImages</a>.</p>
