<p dir="auto">CircularArrays.jl is a small package adding the <code>CircularArray</code> type which can be backed by any <code>AbstractArray</code>. A <code>CircularArray{T,N,A}</code> is an <code>AbstractArray{T,N}</code> backed by a data array of type <code>A</code>. It has a fixed size and features circular indexing (periodic boundary conditions) across all dimensions: Indexing and assigning beyond its bounds in both directions is possible, as the end of the array is considered adjacent to its start. <code>CircularArray</code>s have the same <code>axes</code> as the underlying backing array, and iterators only iterate over these indices.</p>
<p dir="auto">The <code>CircularVector{T}</code> type is added as an alias for <code>CircularArray{T, 1}</code>.</p>
<p dir="auto">The following constructors are provided.</p>
