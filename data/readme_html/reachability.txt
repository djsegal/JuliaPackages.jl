<p><code>Reachability</code> is a software for reachability analysis and safety property checking that performs flowpipe computation of dynamical systems given by ordinary differential equations models (ODEs) in continuous or discrete time.
It is written in <a href="http://julialang.org" rel="nofollow">Julia</a>, a modern high-performance language
for scientific computing.</p>
<p>Currently this package implements algorithms for flowpipe approximation of:</p>
<ul>
<li>linear and nonlinear purely continuous or purely discrete ODEs with nondeterministic inputs</li>
<li>hybrid dynamical systems</li>
</ul>

<p>The set computations depend on the core library <a href="https://github.com/JuliaReach/LazySets.jl"><code>LazySets.jl</code></a>, which is also part of the <a href="https://github.com/JuliaReach/">JuliaReach</a> framework. <code>LazySets.jl</code> exploits the principle of lazy (on-demand) evaluation and uses support functions to represent lazy sets.</p>
<p>The latest stable release of <a href="https://github.com/JuliaReach/LazySets.jl"><code>LazySets.jl</code></a> is installed automatically when you install <code>Reachability.jl</code> (see the installation instructions below). See the <a href="https://juliareach.github.io/LazySets.jl/dev/man/getting_started/" rel="nofollow">installation section of <code>LazySets.jl</code></a> for further details.</p>
