<p>Any logging events that are compatible with
<a href="http://docs.junolab.org/latest/man/juno_frontend/#Progress-Meters-1" rel="nofollow"><code>Juno.progress</code></a>
specification are displayed using <code>ProgressMeter.Progress</code>.</p>
