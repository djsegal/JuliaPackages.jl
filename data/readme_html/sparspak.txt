<p dir="auto">Translation of the well-known sparse matrix software Sparspak (Waterloo Sparse Matrix Package), solving
large sparse systems of linear algebraic equations. Sparspak is composed of the
subroutines from the book "Computer Solution of Large Sparse Positive Definite
Systems" by Alan George and Joseph Liu. Originally written in Fortran 77, later
rewritten in Fortran 90. Here is the software translated into Julia.</p>
<p dir="auto">The Julia rewrite is released  under the MIT license with an express permission from the authors (George and Liu). Many thanks!</p>

<ul dir="auto">
<li>09/18/2022: Jürgen Fuhrmann implemented dispatch to allow for arbitrary-precision floating point numbers and ForwardDiff.Dual to be routed around BLAS routines. This then should allow Automatic Differentiation (AD) of a sparse-matrix solve.</li>
<li>09/01/2022: Configuration now allows for both Julia 1.6 (LTS) and the subsequent Julia versions.</li>
<li>06/03/2022: The sparse LU solver has been now rewritten and tested.</li>
