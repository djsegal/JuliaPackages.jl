<p dir="auto">This Julia package currently contains the programs in chapters 4, 5 and early sections of 6 as described in "Programming the Finite Element Method" by I M Smith, D V Griffiths and L. Margetts (<a href="http://www.wiley.com/WileyCDA/WileyTitle/productCd-1119973341.html" rel="nofollow">PtFEM</a>).</p>
<p dir="auto">I use PtFEM when referring to the book and PtFEM.jl when referring to the Julia package.  The authors and publisher have given permission to publish the Julia version of the PtFEM toolkit. Please refer to <a href="https://github.com/PtFEM/PtFEM.jl/blob/master/LICENSE.md">LICENSE</a> for more details.</p>

<p dir="auto">Fundamental and great development work related to solving (partial) differential equations is done in several other Julia packages, e.g. <a href="https://github.com/JuliaApproximation/ApproxFun.jl">ApproxFun.jl</a>, <a href="https://github.com/JuliaDiffEq/DifferentialEquations.jl">DifferentialEquations.jl</a>, <a href="http://www.juliafem.org" rel="nofollow">JuliaFEM.jl</a> and  <a href="https://github.com/KristofferC/JuAFEM.jl">JuaFEM.jl</a> to name a few.</p>
<p dir="auto">Outside of Julia at least 2 other toolkits should be mentioned, i.e.  <a href="http://dealii.org" rel="nofollow">deal.II</a> and <a href="https://fenicsproject.org" rel="nofollow">FEniCS</a>.</p>
