<p>PowerGraphics.jl is a Julia package for plotting results from <a href="https://github.com/NREL/PowerSimulations.jl">PowerSimulations.jl</a>.</p>

<h2><a id="user-content-usage" class="anchor" aria-hidden="true" href="#usage"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path fill-rule="evenodd" d="M4 9h1v1H4c-1.5 0-3-1.69-3-3.5S2.55 3 4 3h4c1.45 0 3 1.69 3 3.5 0 1.41-.91 2.72-2 3.25V8.59c.58-.45 1-1.27 1-2.09C10 5.22 8.98 4 8 4H4c-.98 0-2 1.22-2 2.5S3 9 4 9zm9-3h-1v1h1c1 0 2 1.22 2 2.5S13.98 12 13 12H9c-.98 0-2-1.22-2-2.5 0-.83.42-1.64 1-2.09V6.25c-1.09.53-2 1.84-2 3.25C6 11.31 7.55 13 9 13h4c1.45 0 3-1.69 3-3.5S14.5 6 13 6z"></path></svg></a>Usage</h2>
<p><code>PowerGraphics.jl</code> uses <a href="https://github.com/NREL/PowerSystems.jl">PowerSystems.jl</a> and <a href="https://github.com/NREL/PowerSimulations.jl">PowerSimulations.jl</a> to handle the data and execution power system simulations.</p>
<div class="highlight highlight-source-julia"><pre><span class="pl-k">using</span> PowerGraphics
<span class="pl-c"><span class="pl-c">#</span> where "res" is a PowerSimulations.SimulationResults object</span>
<span class="pl-c1">stack_plot</span>(res)</pre></div>
<p><code>PowerGraphics.jl</code> creates figures using a number of optional backends using <code>Plots.jl</code>. For interactive figures, it is recommended to use the <code>PlotlyJS.jl</code> backend, which requires the <code>PlotlyJS.jl</code> and <code>ORCA.jl</code> packages:</p>
<div class="highlight highlight-source-julia"><pre><span class="pl-k">using</span> Pkg
Pkg<span class="pl-k">.</span><span class="pl-c1">add</span>(<span class="pl-s"><span class="pl-pds">"</span>PlotlyJS<span class="pl-pds">"</span></span>)
Pkg<span class="pl-k">.</span><span class="pl-c1">add</span>(<span class="pl-s"><span class="pl-pds">"</span>ORCA<span class="pl-pds">"</span></span>)</pre></div>
<p>When using <code>PowerGraphics.jl</code> within a jupyter notebook, <code>WebIO.jl</code> is also required (*and depending on the local configuration of nbextensions, some <a href="https://juliagizmos.github.io/WebIO.jl/latest/troubleshooting/not-detected/" rel="nofollow">additional steps</a> may be required):</p>
<div class="highlight highlight-source-julia"><pre>Pkg<span class="pl-k">.</span><span class="pl-c1">add</span>(<span class="pl-s"><span class="pl-pds">"</span>WebIO<span class="pl-pds">"</span></span>)</pre></div>
<p>An additional command (<code>plotlyjs()</code>) to startup the <code>PlotlyJS</code> backend from <code>Plots</code> is required:</p>
<div class="highlight highlight-source-julia"><pre><span class="pl-k">using</span> PowerGraphics
<span class="pl-c1">plotlyjs</span>()
<span class="pl-c"><span class="pl-c">#</span> where "res" is a PowerSimulations.SimulationResults object,</span>
<span class="pl-c"><span class="pl-c">#</span> and "sys" is a PowerSystems.System object</span>
<span class="pl-c1">fuel_plot</span>(res, sys)</pre></div>
