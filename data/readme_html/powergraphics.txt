<p dir="auto">PowerGraphics.jl is a Julia package for plotting results from <a href="https://github.com/NREL-Sienna/PowerSimulations.jl">PowerSimulations.jl</a>.</p>

<p dir="auto"><code>PowerGraphics.jl</code> uses <a href="https://github.com/NREL-Sienna/PowerSystems.jl">PowerSystems.jl</a> and <a href="https://github.com/NREL-Sienna/PowerSimulations.jl">PowerSimulations.jl</a> to handle the data and execution power system simulations.</p>
<div class="highlight highlight-source-julia notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="using PowerGraphics
# where &quot;res&quot; is a PowerSimulations.SimulationResults object
gen = get_generation_data(res)
plot_powerdata(gen)"><pre><span class="pl-k">using</span> PowerGraphics
<span class="pl-c"><span class="pl-c">#</span> where "res" is a PowerSimulations.SimulationResults object</span>
gen <span class="pl-k">=</span> <span class="pl-c1">get_generation_data</span>(res)
<span class="pl-c1">plot_powerdata</span>(gen)</pre></div>
<p dir="auto"><code>PowerGraphics.jl</code> creates figures using a number of optional backends using <code>Plots.jl</code>. For interactive figures, it is recommended to use the <code>PlotlyJS.jl</code> backend, which requires the <code>PlotlyJS.jl</code>:</p>
<div class="highlight highlight-source-julia notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="using Pkg
Pkg.add(&quot;PlotlyJS&quot;)"><pre><span class="pl-k">using</span> Pkg
Pkg<span class="pl-k">.</span><span class="pl-c1">add</span>(<span class="pl-s"><span class="pl-pds">"</span>PlotlyJS<span class="pl-pds">"</span></span>)</pre></div>
<p dir="auto">When using <code>PowerGraphics.jl</code> within a jupyter notebook, <code>WebIO.jl</code> is also required:</p>
<div class="highlight highlight-source-julia notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="Pkg.add(&quot;WebIO&quot;)"><pre>Pkg<span class="pl-k">.</span><span class="pl-c1">add</span>(<span class="pl-s"><span class="pl-pds">"</span>WebIO<span class="pl-pds">"</span></span>)</pre></div>
<p dir="auto">An additional command (<code>plotlyjs()</code>) to startup the <code>PlotlyJS</code> backend from <code>Plots</code> is required:</p>
<div class="highlight highlight-source-julia notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="using PowerGraphics
plotlyjs()
# where &quot;res&quot; is a PowerSimulations.SimulationResults object
plot_fuel(res)"><pre><span class="pl-k">using</span> PowerGraphics
<span class="pl-c1">plotlyjs</span>()
<span class="pl-c"><span class="pl-c">#</span> where "res" is a PowerSimulations.SimulationResults object</span>
<span class="pl-c1">plot_fuel</span>(res)</pre></div>
