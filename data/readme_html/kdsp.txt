<div id="readme" class="md" data-path="README.md"><article class="markdown-body entry-content container-lg" itemprop="text"><p dir="auto">DSP.jl provides a number of common DSP routines in Julia.  So far, the following functions are
implemented:</p>
<p dir="auto">digital filtering:</p>
<ul dir="auto">
<li>filt</li>
</ul>
<p dir="auto">correlation and convolution:</p>
<ul dir="auto">
<li>conv</li>
<li>conv2</li>
<li>deconv</li>
<li>xcorr</li>
</ul>
<p dir="auto">FFTs provided by FFTW interface:</p>
<ul dir="auto">
<li>bfft</li>
<li>bfftn</li>
<li>brfft</li>
<li>brfftn</li>
<li>fft</li>
<li>fft2</li>
<li>fft3</li>
<li>fftn</li>
<li>ifft</li>
<li>ifft2</li>
<li>ifft3</li>
<li>ifftn</li>
<li>irfft</li>
<li>irfftn</li>
<li>rfft</li>
<li>rfftn</li>
</ul>
<p dir="auto">FFT utilities:</p>
<ul dir="auto">
<li>fftshift</li>
<li>ifftshift</li>
</ul>
<p dir="auto">periodogram estimation:</p>
<ul dir="auto">
<li>periodogram</li>
<li>welch_pgram</li>
<li>bartlett_pgram</li>
</ul>
<p dir="auto">window functions:</p>
<ul dir="auto">
<li>rect</li>
<li>hanning</li>
<li>hamming</li>
<li>tukey</li>
<li>cosine</li>
<li>lanczos</li>
<li>triang</li>
<li>bartlett</li>
<li>gaussian</li>
<li>bartlett_hann</li>
<li>blackman</li>
<li>kaiser</li>
</ul>
<p dir="auto">common DSP mathematics:</p>
<ul dir="auto">
<li>sinc</li>
</ul>
<p dir="auto">auxiliary functions:</p>
<ul dir="auto">
<li>arraysplit</li>
</ul>
</article></div>
