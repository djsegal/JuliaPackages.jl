<p dir="auto">First you should define a function which takes named  arguments weight = kg, height = cm, and age = years.</p>
<p dir="auto">See the file examples/medication_formulas.jl for examples of formulas.</p>
<p dir="auto">This type of app does not consider concomitant medications. If there are such issues, consider mention in the text of <code>rlabel</code>.

<p dir="auto">Create a <code>Gtk</code> widget with entries for <code>weight</code>, <code>height</code>, <code>age</code>, and dose interval.
Arguments:
<code>func</code> is a function which takes named arguments age = years, weight =  kg, height = cm and returns total 24-hour dosage in mg.
<code>title</code> is the title for the app, and
<code>rlabel</code> is the label to be given the results
<br><br></p>
