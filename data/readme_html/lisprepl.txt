<p>This package provides REPL functionality with Lisp syntax on top of julia.  This is really Michael Hatherly's contribution factored out of LispSyntax.jl.</p>
<h2><a id="user-content-usage" class="anchor" aria-hidden="true" href="#usage"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path fill-rule="evenodd" d="M4 9h1v1H4c-1.5 0-3-1.69-3-3.5S2.55 3 4 3h4c1.45 0 3 1.69 3 3.5 0 1.41-.91 2.72-2 3.25V8.59c.58-.45 1-1.27 1-2.09C10 5.22 8.98 4 8 4H4c-.98 0-2 1.22-2 2.5S3 9 4 9zm9-3h-1v1h1c1 0 2 1.22 2 2.5S13.98 12 13 12H9c-.98 0-2-1.22-2-2.5 0-.83.42-1.64 1-2.09V6.25c-1.09.53-2 1.84-2 3.25C6 11.31 7.55 13 9 13h4c1.45 0 3-1.69 3-3.5S14.5 6 13 6z"></path></svg></a>Usage</h2>
<p>The lisp REPL mode is entered using the <code>)</code> key in a same way as other REPL modes such as
help (<code>?</code>) and shell (<code>;</code>). Unlike those modes the lisp mode is "sticky". After pressing
return to evaluate the current line the mode will <em>not</em> switch back to the <code>julia&gt;</code> mode,
but instead will remain in lisp mode. To return to Julia mode press backspace.</p>
<h2><a id="user-content-customization" class="anchor" aria-hidden="true" href="#customization"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path fill-rule="evenodd" d="M4 9h1v1H4c-1.5 0-3-1.69-3-3.5S2.55 3 4 3h4c1.45 0 3 1.69 3 3.5 0 1.41-.91 2.72-2 3.25V8.59c.58-.45 1-1.27 1-2.09C10 5.22 8.98 4 8 4H4c-.98 0-2 1.22-2 2.5S3 9 4 9zm9-3h-1v1h1c1 0 2 1.22 2 2.5S13.98 12 13 12H9c-.98 0-2-1.22-2-2.5 0-.83.42-1.64 1-2.09V6.25c-1.09.53-2 1.84-2 3.25C6 11.31 7.55 13 9 13h4c1.45 0 3-1.69 3-3.5S14.5 6 13 6z"></path></svg></a>Customization</h2>
<p>The lisp mode prompt text and color may be set via your <code>ENV</code> settings. For example adding
the following to your <code>.juliarc.jl</code>:</p>
<div class="highlight highlight-source-julia"><pre>ENV[<span class="pl-s"><span class="pl-pds">"</span>LISP_PROMPT_TEXT<span class="pl-pds">"</span></span>]  <span class="pl-k">=</span> <span class="pl-s"><span class="pl-pds">"</span>Î» <span class="pl-pds">"</span></span>
ENV[<span class="pl-s"><span class="pl-pds">"</span>LISP_PROMPT_COLOR<span class="pl-pds">"</span></span>] <span class="pl-k">=</span> <span class="pl-s"><span class="pl-pds">"</span>red<span class="pl-pds">"</span></span></pre></div>
<p>will set the prompt for lisp mode to a red lambda.</p>
