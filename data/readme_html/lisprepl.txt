<p>This package provides REPL functionality with Lisp syntax on top of julia.  This is really Michael Hatherly's contribution factored out of LispSyntax.jl.</p>
<h2><a id="user-content-usage" class="anchor" aria-hidden="true" href="#usage"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path fill-rule="evenodd" d="M7.775 3.275a.75.75 0 001.06 1.06l1.25-1.25a2 2 0 112.83 2.83l-2.5 2.5a2 2 0 01-2.83 0 .75.75 0 00-1.06 1.06 3.5 3.5 0 004.95 0l2.5-2.5a3.5 3.5 0 00-4.95-4.95l-1.25 1.25zm-4.69 9.64a2 2 0 010-2.83l2.5-2.5a2 2 0 012.83 0 .75.75 0 001.06-1.06 3.5 3.5 0 00-4.95 0l-2.5 2.5a3.5 3.5 0 004.95 4.95l1.25-1.25a.75.75 0 00-1.06-1.06l-1.25 1.25a2 2 0 01-2.83 0z"></path></svg></a>Usage</h2>
<p>The lisp REPL mode is entered using the <code>)</code> key in a same way as other REPL modes such as
help (<code>?</code>) and shell (<code>;</code>). Unlike those modes the lisp mode is "sticky". After pressing
return to evaluate the current line the mode will <em>not</em> switch back to the <code>julia&gt;</code> mode,
but instead will remain in lisp mode. To return to Julia mode press backspace.</p>
<h2><a id="user-content-customization" class="anchor" aria-hidden="true" href="#customization"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path fill-rule="evenodd" d="M7.775 3.275a.75.75 0 001.06 1.06l1.25-1.25a2 2 0 112.83 2.83l-2.5 2.5a2 2 0 01-2.83 0 .75.75 0 00-1.06 1.06 3.5 3.5 0 004.95 0l2.5-2.5a3.5 3.5 0 00-4.95-4.95l-1.25 1.25zm-4.69 9.64a2 2 0 010-2.83l2.5-2.5a2 2 0 012.83 0 .75.75 0 001.06-1.06 3.5 3.5 0 00-4.95 0l-2.5 2.5a3.5 3.5 0 004.95 4.95l1.25-1.25a.75.75 0 00-1.06-1.06l-1.25 1.25a2 2 0 01-2.83 0z"></path></svg></a>Customization</h2>
<p>The lisp mode prompt text and color may be set via your <code>ENV</code> settings. For example adding
the following to your <code>.juliarc.jl</code>:</p>
<div class="highlight highlight-source-julia"><pre>ENV[<span class="pl-s"><span class="pl-pds">"</span>LISP_PROMPT_TEXT<span class="pl-pds">"</span></span>]  <span class="pl-k">=</span> <span class="pl-s"><span class="pl-pds">"</span>Î» <span class="pl-pds">"</span></span>
ENV[<span class="pl-s"><span class="pl-pds">"</span>LISP_PROMPT_COLOR<span class="pl-pds">"</span></span>] <span class="pl-k">=</span> <span class="pl-s"><span class="pl-pds">"</span>red<span class="pl-pds">"</span></span></pre></div>
<p>will set the prompt for lisp mode to a red lambda.</p>
