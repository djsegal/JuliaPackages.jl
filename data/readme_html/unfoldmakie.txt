<p dir="auto">UnfoldMakie allows many visualizations for ERP and "Unfolded"-models.
Building on the <a href="https://github.com/unfoldtoolbox/unfold.jl/">Unfold</a> and <a href="https://makie.juliaplots.org/stable/" rel="nofollow">Makie</a>, it grants users highly customizable plots.</p>
<p dir="auto">The toolbox requires a Makie-Backend to be loaded, e.g. <code>using CairoMakie</code></p>
<p dir="auto">Some plotting functions require data-frames, with fields like <code>:estimate</code> or <code>:time</code>.</p>
<p dir="auto">The supportet visualizations are:</p>
<ul dir="auto">
<li>Designmatrices</li>
<li>Line Plots</li>
<li>Butterfly Plots</li>
<li>ERP Images</li>
<li>Topo Plots</li>
<li>Parallel Coordinates Plot</li>
</ul>

<p dir="auto">The documentation can be found in the <code>doc</code> folder. We use <code>Documenter.jl</code> and plan to use <code>Literate.jl</code> for future usage.
In the <code>src</code> folder contained within it exists as "raw" <code>.md</code> files and in the <code>build</code> folder contained within it exists as <code>.html</code> after running the <code>make.jl</code> file.
