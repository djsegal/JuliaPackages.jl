<p>A Julia package for calculating hypergeometric functions</p>
<p>This package implements the generalized hypergeometric function <code>pFq([a1,…,am], [b1,…,bn], z)</code>. In particular, the Gauss hypergeometric function is available as <code>_₂F₁(a, b, c, z)</code>, and also <code>_₃F₂([a1, a2, a3], [b1, b2], z)</code>.</p>
