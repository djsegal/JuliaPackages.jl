<p dir="auto">Image registration with subpixel precision using an upsampled discrete Fourier transform cross-correlation. This uses an efficient matrix-multiplication algorithm for upsampling the cross-correlation following Guizar-Sicairos, Thurman, and Fienup (2008).<sup><a href="#user-content-fn-1-dad92c98d1e60034f81d15c516229725" id="user-content-fnref-1-dad92c98d1e60034f81d15c516229725" data-footnote-ref="" aria-describedby="footnote-label">1</a></sup></p>
