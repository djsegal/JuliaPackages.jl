<p dir="auto">This repo contains the code associated with the paper <a href="https://arxiv.org/abs/2210.16294" rel="nofollow">Learning Modular Simulations for Homogenous Networks</a>. Also see the <a href="https://www.microsoft.com/en-us/research/group/autonomous-systems-group-robotics/articles/towards-modular-data-driven-simulations/" rel="nofollow">blogpost</a>.</p>

<ul dir="auto">
<li><code>src/systems</code>: contains the systems we evaluate the MP-NODE and other baselines on.</li>
<li><code>src/datagen.jl</code>: contains utilities to generate data from the systems.</li>
<li><code>src/empde.jl</code>: contains the pieces required to setup MP-NODE models.</li>
<li><code>src/gnnode.jl</code>: contains the pieces required to setup L2S models.</li>
<li><code>src/nde.jl</code>: contains the pieces required to setup NODE models.</li>
<li><code>src/learn.jl</code> and <code>src/learn_utils.jl</code>: contains the model training utilities.</li>
<li><code>exps/models.jl</code>: contains the specific models used in the paper.</li>
<li><code>exps/train_pipeline.jl</code>: contains orchestration for full training pipeline.</li>
<li><code>exps/tasks.jl</code>: contains the specific systems used in the paper.</li>
</ul>
