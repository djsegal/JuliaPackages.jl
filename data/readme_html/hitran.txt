<p>HITRAN.jl is a Julia package for calculating spectral features using the HITRAN database. The package follows a very similar approach to the <a href="https://github.com/hitranonline/hapi">HITRAN Application Programming Interface (HAPI)</a>. In fact the basic workflow and methods work almost the same which should make switching for HAPI users easy.
The package fetches spectral line data from <a href="https://hitran.org" rel="nofollow">HITRANOnline</a> and stores it in a local SQLite database for structured access. This data can be used to calculate spectral lineshapes while taking into account the gas mixture and environmetal conditions (pressure, temperature).</p>
