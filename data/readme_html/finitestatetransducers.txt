<p dir="auto">Play with Weighted Finite State Transducers (WFSTs) using the Julia language.</p>
<p dir="auto">WFSTs provide a powerful framework that assigns a weight (e.g. probability) to conversions of symbol sequences.
WFSTs are used in many applications such as speech recognition, natural language processing and machine learning.</p>
<p dir="auto">This package takes a lot of inspiration from <a href="http://openfst.org/twiki/bin/view/FST/DeterminizeDoc" rel="nofollow">OpenFST</a>.</p>
<p dir="auto">FiniteStateTransducers is still in an early development stage, see the documentation for currently available features and the issues for the missing ones.
An application of WFSTs can be found in the data preparation of the <a href="https://github.com/idiap/TIDIGITSRecipe.jl">TIDIGITS recipe</a>.</p>
