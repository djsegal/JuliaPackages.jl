<p dir="auto">This project facilitates a codegen workflow when consuming GraphQL apis in Julia.</p>
<p dir="auto">This project exports:</p>
<ul dir="auto">
<li>the <code>gql</code> macro which helps the code generator to find the GraphQL queries</li>
<li>The <code>parse_nt</code> function which parses <code>Dict</code> into Julia Named Tuples.</li>
</ul>
