<p dir="auto">Warning.. very basic.. More of an experiment.. Concatenates characters from pre-rendered lookup images, then resizes and colors. 100% julia, no libs. Gets blurry if font height is greater than ~90 pixels.</p>
<p dir="auto">Provides <code>rendertext</code> for generating a rendered image of a string and
<code>overlaytext!</code> for directly overlaying text on a loaded image.</p>
<p dir="auto">Currently built to support truly monospaced fonts only, and Courier is the only font that is provided.
(other open source licensed fonts welcome via PR, see <code>gen/makeCharLookups.jl</code>)</p>

character lookup table, but subsequent calls for the same scale and font will be fast.</p>
