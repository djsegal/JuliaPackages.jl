<p dir="auto">We have made the following (breaking) changes:</p>
<ul dir="auto">
<li><code>SimpleGraph</code> has been renamed <code>UndirectedGraph</code>.</li>
<li><code>SimpleDigraph</code> has been renamed <code>DirectedGraph</code>.</li>
<li><code>SimpleHypergraph</code> has been renamed <code>HyperGraph</code> (note the captial G).</li>
</ul>
<p dir="auto">These may be abbreviated as <code>UG</code>, <code>DG</code>, and <code>HG</code>, respectively</p>
<p dir="auto">These changes were made to help this <code>SimpleGraphs</code> module be interoperable with Julia's <a href="https://github.com/JuliaGraphs/Graphs.jl"><code>Graphs</code></a> module that defines the type <code>SimpleGraph</code> (formerly <code>Graph</code> in the <code>LightGraphs</code> module).</p>
<p dir="auto">Conversion between these two types (<code>UndirectedGraph</code> in this module and <code>SimpleGraph</code> in the <code>Graphs</code> module) is supported; see the file <code>graph_converter.jl</code> in the <code>extras</code> directory.</p>
<hr>
<h2 dir="auto"><a id="user-content-overview" class="anchor" aria-hidden="true" href="#overview"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path d="m7.775 3.275 1.25-1.25a3.5 3.5 0 1 1 4.95 4.95l-2.5 2.5a3.5 3.5 0 0 1-4.95 0 .751.751 0 0 1 .018-1.042.751.751 0 0 1 1.042-.018 1.998 1.998 0 0 0 2.83 0l2.5-2.5a2.002 2.002 0 0 0-2.83-2.83l-1.25 1.25a.751.751 0 0 1-1.042-.018.751.751 0 0 1-.018-1.042Zm-4.69 9.64a1.998 1.998 0 0 0 2.83 0l1.25-1.25a.751.751 0 0 1 1.042.018.751.751 0 0 1 .018 1.042l-1.25 1.25a3.5 3.5 0 1 1-4.95-4.95l2.5-2.5a3.5 3.5 0 0 1 4.95 0 .751.751 0 0 1-.018 1.042.751.751 0 0 1-1.042.018 1.998 1.998 0 0 0-2.83 0l-2.5 2.5a1.998 1.998 0 0 0 0 2.83Z"></path></svg></a>Overview</h2>
<p dir="auto">This module defines three data types for working with graphs:</p>
<ul dir="auto">
<li>The <code>UndirectedGraph</code> type represents <em>undirected</em> graphs without loops
or multiple edges.</li>
<li>The <code>DirectedGraph</code> type represents <em>directed</em> graphs in which there
may be at most one directed edge <code>(u,v)</code> from a vertex <code>u</code> to a
vertex <code>v</code>. There may also be a directed edge in the opposite
direction, <code>(v,u)</code>.</li>
<li>The <code>HyperGraph</code> type representing <em>hypergraphs</em> in which
edges may be any subset of the vertex set.</li>
</ul>
<p dir="auto">Additional functionality can be found in these modules:</p>
<ul dir="auto">
<li><a href="https://github.com/scheinerman/DrawSimpleGraphs.jl"><code>DrawSimpleGraphs</code></a> for visualization.</li>
<li><a href="https://github.com/scheinerman/SimpleGraphAlgorithms.jl"><code>SimpleGraphAlgorithms</code></a>
for functions relying on [integer] linear programming.</li>
</ul>
