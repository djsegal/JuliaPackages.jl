<p dir="auto">CodeTools.jl is a collection of tools for handling Julia code – evaluation, autocompletion etc. – designed to be used as a backend library for IDE support.</p>
<p dir="auto">It handles things such as:</p>
<ul dir="auto">
<li>Extensible autocompletion</li>
<li>Pulling code blocks out of files (given a cursor position)</li>
<li>Finding relevant documentation or method definitions at the cursor</li>
<li>Detecting the module a file belongs to</li>
<li>Evaluation of code blocks with correct file, line and module data</li>
</ul>
<p dir="auto">If you want any info on using this package as support for another IDE feel free to drop me a line.</p>
