<h2 dir="auto"><a id="user-content-quick-start" class="anchor" aria-hidden="true" href="#quick-start"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path d="m7.775 3.275 1.25-1.25a3.5 3.5 0 1 1 4.95 4.95l-2.5 2.5a3.5 3.5 0 0 1-4.95 0 .751.751 0 0 1 .018-1.042.751.751 0 0 1 1.042-.018 1.998 1.998 0 0 0 2.83 0l2.5-2.5a2.002 2.002 0 0 0-2.83-2.83l-1.25 1.25a.751.751 0 0 1-1.042-.018.751.751 0 0 1-.018-1.042Zm-4.69 9.64a1.998 1.998 0 0 0 2.83 0l1.25-1.25a.751.751 0 0 1 1.042.018.751.751 0 0 1 .018 1.042l-1.25 1.25a3.5 3.5 0 1 1-4.95-4.95l2.5-2.5a3.5 3.5 0 0 1 4.95 0 .751.751 0 0 1-.018 1.042.751.751 0 0 1-1.042.018 1.998 1.998 0 0 0-2.83 0l-2.5 2.5a1.998 1.998 0 0 0 0 2.83Z"></path></svg></a>Quick Start</h2>
<p dir="auto">Make sure you have an up-to-date version of both <a href="https://julialang.org/" rel="nofollow">Julia</a> and <a href="https://github.com/fonsp/Pluto.jl">Pluto.jl</a> installed first!</p>
<p dir="auto">Next, open up a Julia REPL with the <code>julia</code> command in a terminal. Enter Pkg mode by pressing <code>]</code> and install PlutoPublish by typing <code>add PlutoPublish</code>.</p>
<div class="highlight highlight-source-julia notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="(@v1.7) pkg&gt; add PlutoPublish"><pre>(<span class="pl-c1">@v1</span>.<span class="pl-c1">7</span>) pkg<span class="pl-k">&gt;</span> add PlutoPublish</pre></div>
<p dir="auto">Now start a Pluto server <strong>with the publisher enabled</strong>. This <em>will not</em> publish any notebooks without explicit action later.</p>
<div class="highlight highlight-source-julia notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="using Pluto, PlutoPublish
Pluto.run(; on_event=publisher())"><pre><span class="pl-k">using</span> Pluto, PlutoPublish
Pluto<span class="pl-k">.</span><span class="pl-c1">run</span>(; on_event<span class="pl-k">=</span><span class="pl-c1">publisher</span>())</pre></div>
<p dir="auto">By passing in the <code>publish</code> function exported by <code>PlutoPublish</code> we listen for publishing requests inside notebooks and will automatically push updates when we do get requests.</p>
<p dir="auto">Once you've got a notebook open that you want to publish, just add a cell with the following code in it:</p>
<div class="highlight highlight-source-julia notranslate position-relative overflow-auto" dir="auto" data-snippet-clipboard-copy-content="PUBLISH = true;"><pre>PUBLISH <span class="pl-k">=</span> <span class="pl-c1">true</span>;</pre></div>
<p dir="auto"><span><b>NOTE:</b> This code <strong>must</strong> be in <strong>its own cell</strong></span></p>
<p dir="auto">The <code>PlutoPublish</code> package will detect this cell and automatically upload and publish your notebook to this site every time you make a change!</p>
