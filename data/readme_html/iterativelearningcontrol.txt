<p dir="auto"><strong>IterativeLearningControl.jl</strong> is a package for solving optimal control problems using the iterative learning control (ILC) algorithm. It is designed to be used with <a href="https://github.com/aarontrowbridge/QuantumCollocation.jl">QuantumCollocation.jl</a> and <a href="https://github.com/aarontrowbridge/NamedTrajectories.jl">NamedTrajectories.jl</a>.</p>
