<p>ArviZ.jl is a Julia interface to the <a href="https://arviz-devs.github.io/arviz/" rel="nofollow">ArviZ</a> (pronounced "AR-<em>vees</em>") package for exploratory analysis of Bayesian models.
It includes functions for posterior analysis, model checking, comparison and diagnostics.</p>
<p>See the <a href="https://arviz-devs.github.io/ArviZ.jl/stable" rel="nofollow">documentation</a> for details.</p>
