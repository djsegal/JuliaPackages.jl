<p dir="auto">ArviZ.jl (pronounced "AR-<em>vees</em>") is a Julia package for exploratory analysis of Bayesian models.
It includes functions for posterior analysis, model checking, comparison and diagnostics.</p>
<p dir="auto">It is part of the <a href="https://arviz.org/" rel="nofollow">ArviZ project</a>, along with the Python package <a href="https://python.arviz.org/" rel="nofollow">ArviZ</a>.</p>
<p dir="auto">See the <a href="https://julia.arviz.org/ArviZ" rel="nofollow">documentation</a> for details.</p>
