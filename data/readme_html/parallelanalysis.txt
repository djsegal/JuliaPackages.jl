<p dir="auto">Parallel analysis <a href="https://link.springer.com/article/10.1007/BF02289447" rel="nofollow">(Houts, 1965)</a> diagnoses the number of dimension to approximate data.</p>
<p dir="auto"><code>parallel</code> simulate eigen values of the random matrix.</p>
<div class="snippet-clipboard-content notranslate position-relative overflow-auto" data-snippet-clipboard-copy-content="fit = parallel(data, 1000, f = fa); # Return `Parallel` struct"><pre class="notranslate"><code>fit = parallel(data, 1000, f = fa); # Return `Parallel` struct
</code></pre></div>
<p dir="auto">Type of <code>data</code> is Matrix or DataFrame and its elements, by the default argument,  are assumed to be ordered categorical. If <code>data</code> has continuous variables, <code>...f = x -&gt; fa(x, cor_method = :Pearson)</code> has to be used.</p>
